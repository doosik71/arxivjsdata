# UNSUPERVISED DEFECT DETECTION FOR SURGICAL INSTRUMENTS

Joseph Huang, Yichi Zhang, Jingxi Zhang, Wei Chen, Seunghyun Hwang, Qiang Qiu, Amy R. Reibman, Edward J. Delp, Fengqing Zhu (2025)

## 🧩 Problem to Solve

본 논문은 수술 도구의 시각적 결함을 자동으로 감지하여 환자의 안전을 보장하고 수술 결과를 개선하는 것을 목표로 한다. 현재 대부분의 의료 환경에서는 숙련된 기술자가 수동으로 결함을 검사하고 있으나, 이는 노동 집약적이며 주관적일 뿐만 아니라, 미세한 결함을 놓칠 가능성이 높은 인적 오류에 취약하다.

이를 자동화하기 위해 기존의 비지도 결함 탐지(Unsupervised Defect Detection) 또는 이상 탐지(Anomaly Detection) 방법론들을 적용할 수 있으나, 주로 자연 이미지나 일반 산업용 이미지로 학습된 기존 모델들은 수술 도구 도메인으로 전이될 때 다음과 같은 세 가지 주요 문제점에 직면한다.

1. **배경 간섭(Background Interference):** 수술용 타월이나 멸균 랩과 같이 텍스처가 있는 배경이 가짜 결함 신호를 생성하여 높은 False Positive(오탐)율을 유발한다.
2. **미세 결함(Subtle Defects):** 미세한 스크래치, 작은 균열 또는 경미한 오염과 같은 임상적으로 중요한 결함들이 산업용 결함에 비해 매우 미세하여 모델이 이를 쉽게 놓친다.
3. **도메인 시프트(Domain Shift):** Vision Transformer(ViT)와 같은 사전 학습된 인코더들은 주로 ImageNet과 같은 자연 이미지 데이터셋으로 최적화되어 있어, 금속 표면이나 특유의 기하학적 구조를 가진 수술 도구의 특수 기능을 효과적으로 캡처하지 못한다.

따라서 본 연구의 목표는 이러한 도메인 특화 문제들을 해결하여 수술 도구 이미지에서 미세한 결함을 신뢰성 있게 탐지할 수 있는 범용적인 비지도 학습 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 기존의 비지도 결함 탐지 모델을 그대로 사용하는 대신, 수술 도구 도메인의 특성에 맞게 **전처리와 어댑테이션 과정을 추가하는 모델 불가지론적(Model-agnostic) 프레임워크**를 설계한 것이다. 주요 기여 사항은 다음과 같다.

- **SAM 기반 배경 마스킹(Background Masking):** Segment Anything Model(SAM)을 활용하여 수술 도구 영역만을 정밀하게 분리함으로써 배경으로 인한 오탐을 억제한다.
- **패치 기반 분석 전략(Patch-based Strategy):** 이미지를 작은 패치 단위로 나누어 분석함으로써, 이미지 크기 조정으로 인한 정보 손실을 방지하고 미세한 결함에 대한 민감도를 높인다.
- **LoRA를 이용한 도메인 적응(Low-Rank Adaptation):** 사전 학습된 인코더에 LoRA를 적용하여, 적은 파라미터 업데이트만으로 수술 도구 특유의 시각적 특징을 효율적으로 학습하게 함으로써 도메인 시프트 문제를 해결한다.

## 📎 Related Works

본 논문은 일반적인 산업용 이상 탐지(Anomaly Detection) 연구들을 관련 연구로 다룬다. 특히 MVTec AD와 같은 벤치마크에서 우수한 성능을 보인 비지도 학습 방법론들이 언급된다.

비지도 학습 방식은 정상 데이터의 분포만을 모델링하고 여기서 벗어나는 데이터를 이상치로 간주하므로, 결함 데이터의 레이블을 대량으로 수집하기 어려운 의료 분야에 매우 적합하다. 그러나 기존 연구들은 정형화된 산업 환경(고정된 배경, 명확한 결함 형태)을 가정하므로, 배경이 가변적이고 결함이 매우 미세한 수술 도구 검사 환경에 그대로 적용하기에는 한계가 있음을 지적하며 본 연구의 차별점을 제시한다.

## 🛠️ Methodology

제안된 방법론은 특정 모델에 종속되지 않으며, Dinomaly나 DRAEM과 같은 다양한 베이스라인 모델에 통합될 수 있는 파이프라인 구조를 가진다.

### 1. Background Masking

배경 텍스처로 인한 False Positive를 제거하기 위해 SAM을 이용한 전처리를 수행한다.

- 계산 효율성을 위해 입력 이미지의 너비를 512 픽셀로 조정(종횡비 유지)한 후 SAM으로 세그멘테이션 마스크를 생성한다.
- 생성된 마스크를 원본 이미지 크기로 확대 적용하며, 마스크 외부(배경)의 모든 픽셀 값은 $0$으로 처리하여 모델이 오직 도구 영역에만 집중하게 한다.

### 2. Patch-based Strategy

전체 이미지를 한 번에 처리할 때 발생하는 해상도 저하 문제를 해결하기 위해 패치 기반 분석을 도입한다.

- 배경 마스킹이 완료된 고해상도 이미지를 $256 \times 256$ 크기의 겹치지 않는 패치들로 분할한다.
- 각 패치 수준에서 결함 탐지를 수행한 후, 이를 원래의 공간적 위치에 따라 다시 결합하여 전체 이미지 결함 맵(Full-image defect map)을 재구성한다.
- 이후 사전 정의된 임계값(Threshold)을 적용하여 노이즈를 제거하고, 최종적으로 SAM 마스크를 다시 적용하여 도구 외부의 잔여 노이즈를 완전히 제거한다.

### 3. Low-Rank Adaptation (LoRA) for Domain Shift

자연 이미지로 학습된 인코더의 가중치를 수술 도구 도메인에 맞게 효율적으로 조정하기 위해 LoRA를 사용한다.

- 사전 학습된 인코더의 가중치는 고정(Freeze)하고, Self-attention 층의 Query-Key-Value(qkv) 투영 및 Output projection(proj) 모듈에 학습 가능한 저차원 행렬을 삽입한다.
- 기존 가중치 행렬을 $W_0 \in \mathbb{R}^{d \times k}$라고 할 때, 적응된 가중치 $W$는 다음과 같이 정의된다.
$$W = W_0 + \Delta W, \quad \Delta W = BA$$
여기서 $A \in \mathbb{R}^{r \times k}$와 $B \in \mathbb{R}^{d \times r}$는 학습 가능한 행렬이며, 랭크 $r$은 $d$와 $k$보다 훨씬 작은 값($r \ll \min(d, k)$)을 가진다.
- 이 방식은 학습 파라미터 수를 $O(dk)$에서 $O(r(d + k))$로 획기적으로 줄여 계산 효율성을 높이고 과적합(Overfitting)을 방지하면서도 도메인 특화 특징을 캡처할 수 있게 한다.

## 📊 Results

### 실험 설정

- **데이터셋:** 20종의 수술 도구 클래스로 구성된 데이터셋을 직접 구축하였다. 총 80장의 정상 이미지와 식용 색소를 사용하여 실제 오염이나 녹을 모사한 15장의 결함 이미지를 포함한다. 모든 이미지는 iPhone 16 Pro를 이용해 라이트박스 내부에서 통제된 조건으로 촬영되었다.
- **비교 모델(Baselines):** 특징 기반 방식인 Dinomaly(ViT 기반)와 재구성 기반 방식인 DRAEM(Denoising Autoencoder 기반)을 사용하였다.
- **비교 조건:** (1) 사전 학습된 가중치 그대로 사용(Pretrained), (2) 정상 데이터로 전체 미세 조정(Finetuned), (3) 제안된 프레임워크 적용(+ Our Method).
- **평가 지표:** 픽셀 수준의 AUROC (P-AUROC)를 사용하였다.

### 주요 결과

실험 결과, 제안된 방법론을 적용했을 때 두 베이스라인 모델 모두에서 비약적인 성능 향상이 관찰되었다.

| Method | Pretrained | Finetuned | + Our Method |
| :--- | :---: | :---: | :---: |
| Dinomaly | 0.606 | 0.872 | **0.946** |
| DRAEM | 0.453 | 0.727 | **0.951** |

특히 DRAEM의 경우 사전 학습 모델의 성능이 매우 낮았으나(0.453), 제안 방법 적용 시 0.951까지 상승하여 도메인 적응의 중요성을 입증하였다.

### Ablation Study (Dinomaly 기준)

각 구성 요소가 성능에 미치는 영향을 분석한 결과는 다음과 같다.

| Configuration | P-AUROC |
| :--- | :---: |
| Baseline Dinomaly | 0.606 |
| Dinomaly + fine-tuning | 0.872 |
| Dinomaly + masking | 0.913 |
| Dinomaly + masking + patching | 0.933 |
| Full proposed method (Masking + Patching + LoRA) | **0.946** |

분석 결과, 배경 마스킹이 가장 즉각적인 성능 향상을 가져왔으며, 패칭과 LoRA가 순차적으로 정밀도를 보완하여 최종 성능에 도달함을 알 수 있다.

## 🧠 Insights & Discussion

본 연구는 기존의 일반적인 이상 탐지 모델이 의료 도메인에서 왜 실패하는지를 정확히 진단하고, 이를 해결하기 위한 실용적인 파이프라인을 제시하였다는 점에서 강점이 있다. 특히 모델의 구조를 변경하지 않고 전처리와 효율적인 어댑터(LoRA)만을 추가함으로써, 향후 새로운 베이스라인 모델이 등장하더라도 쉽게 적용할 수 있는 확장성을 갖추었다.

다만, 몇 가지 한계점과 논의 사항이 존재한다. 첫째, 결함 데이터를 실제 결함이 아닌 식용 색소 등을 이용해 시뮬레이션하였다는 점이다. 실제 임상 현장에서 발생하는 더욱 다양하고 복잡한 형태의 결함에 대해서도 동일한 성능이 유지될지는 추가 검증이 필요하다. 둘째, 사용된 데이터셋의 규모가 매우 작아(정상 80장, 결함 15장) 결과의 일반화 성능을 확신하기에는 통계적 근거가 부족할 수 있다.

결론적으로, 본 논문은 단순한 모델 튜닝보다 도메인 특성을 반영한 데이터 전처리(Masking, Patching)와 효율적인 가중치 적응(LoRA)의 조합이 도메인 시프트 해결에 결정적임을 보여주었다.

## 📌 TL;DR

본 논문은 수술 도구의 결함 탐지를 위해 **배경 마스킹(SAM), 패치 기반 분석, 그리고 저차원 적응(LoRA)**을 결합한 비지도 학습 프레임워크를 제안한다. 이를 통해 배경 소음 제거, 미세 결함 탐지, 도메인 시프트 문제를 동시에 해결하였으며, Dinomaly와 DRAEM 같은 기존 모델의 P-AUROC 성능을 최대 0.95 수준까지 끌어올렸다. 이 연구는 향후 병원 내 수술 도구 품질 보증 공정을 자동화하는 데 중요한 기반 기술이 될 것으로 기대된다.
