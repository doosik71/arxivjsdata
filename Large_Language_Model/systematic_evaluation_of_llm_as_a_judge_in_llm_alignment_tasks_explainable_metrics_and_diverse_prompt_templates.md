# SYSTEMATIC EVALUATION OF LLM-AS-A-JUDGE IN LLM ALIGNMENT TASKS: EXPLAINABLE METRICS AND DIVERSE PROMPT TEMPLATES

Hui Wei, Shenghua He, Tian Xia, Fei Liu, Andy Wong, Jingyang Lin, Mei Han (2025)

## 🧩 Problem to Solve

본 논문은 LLM 정렬(Alignment) 작업에서 인간 평가자를 대체하여 사용되는 **LLM-as-a-Judge** 방식의 신뢰성 문제를 해결하고자 한다. LLM-as-a-Judge는 RLHF나 DPO와 같은 정렬 알고리즘의 성능을 비교하는 데 널리 쓰이지만, 다음과 같은 치명적인 한계점들이 존재한다.

첫째, LLM 판별자는 응답의 위치에 따라 선호도가 달라지는 **Position Bias**와 더 긴 응답을 선호하는 **Length Bias**와 같은 체계적인 편향을 보인다. 둘째, 동일한 입력에 대해서도 비결정론적(non-deterministic)인 출력으로 인해 결과가 달라지는 **내부적 불일치(Internal Inconsistency)** 문제가 발생한다. 셋째, 기존의 신뢰성 평가 지표들은 이론적 설명력이 부족하며, 특히 LLM의 내부 노이즈를 고려하지 않은 채 결정론적인 판단을 내린다고 가정하는 경향이 있다. 마지막으로, 판별 결과에 지대한 영향을 미치는 **프롬프트 템플릿(Prompt Templates)**의 영향력에 대한 체계적인 분석이 부족하다.

따라서 본 연구의 목표는 이론적으로 해석 가능한 평가 지표를 정의하고, LLM의 내부 불일치를 명시적으로 완화하여 LLM-as-a-Judge의 신뢰성과 인간 선호도와의 정렬 수준을 체계적으로 평가하는 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 LLM 판별자의 편향을 측정하는 지표에 이론적 근거를 부여하고, 모델의 비결정론적 특성인 '노이즈'를 분리하여 순수한 편향을 측정하려 한 점에 있다.

1. **이론적 설명력이 강화된 평가 지표 정의**: Position Bias와 Length Bias를 통합된 정확도(Accuracy) 기반 프레임워크 내에서 정의하여 해석 가능성을 높였다.
2. **Flipping Noise 모델링 및 완화**: LLM이 동일한 사례에 대해 판단을 뒤집는 현상을 **Flipping Noise**로 정의하고, 이를 수학적으로 모델링하여 실제 측정치에서 노이즈를 제거(De-noising)하는 방법을 제시하였다.
3. **오픈소스 평가 프레임워크 개발**: 다양한 LLM과 사용자 정의 프롬프트 템플릿을 적용하여 판별자의 정렬 수준과 신뢰성을 비교 및 시각화할 수 있는 도구를 제공한다.
4. **프롬프트 템플릿의 영향력 분석**: 다양한 템플릿이 판별자의 성능과 편향에 미치는 영향을 실험적으로 입증하여, 판별자 선정 시 템플릿 선택의 중요성을 강조하였다.

## 📎 Related Works

기존 연구들은 GPT-4나 PaLM 2와 같은 모델을 판별자로 사용하여 Pairwise 평가를 수행해 왔으며, Position Bias와 Length Bias의 존재를 보고하였다. 특히 응답의 순서를 바꾼 후 판단이 바뀌는 비율로 Position Bias를 측정하거나, 응답 길이에 따른 정확도 차이로 Length Bias를 측정하는 방식이 사용되었다.

그러나 본 논문은 기존 연구들이 다음과 같은 한계를 가진다고 지적한다.

- **이론적 기반 부족**: 편향의 정의가 경험적인 관찰에 의존하며 이론적 해석이 부족하다.
- **결정론적 가정의 오류**: LLM이 동일한 실험에서 항상 동일한 결정을 내린다고 가정하여, 시스템 내부의 노이즈(Self-inconsistency)가 편향 측정치에 섞여 들어가는 문제를 간과하였다.
- **템플릿 탐색 부족**: 주로 어떤 LLM 모델을 사용할지에 집중했을 뿐, 동일 모델 내에서 프롬프트 템플릿 변화가 주는 영향은 충분히 다뤄지지 않았다.

## 🛠️ Methodology

### 1. 정확도 정의 (Accuracy)

LLM 판별자가 인간의 선호도와 얼마나 일치하는지를 측정하기 위해 두 가지 정확도 지표를 정의한다. 데이터 케이스 $h = (x, y_c, y_r)$ (여기서 $y_c$는 인간 선호 응답)에 대해 두 가지 순서 $(y_c, y_r)$와 $(y_r, y_c)$로 평가를 진행한다.

- $\text{Acc}_{\text{both}}$: 두 가지 순서 모두에서 인간 선호 응답 $y_c$를 선택한 비율이다.
$$\text{Acc}_{\text{both}} = \frac{1}{N} \sum_{n=1}^{N} \mathbb{1}(y^{(n)} = y_c^{(n)} \wedge y'^{(n)} = y_c^{(n)})$$
- $\text{Acc}_{\text{random}}$: 두 순서의 결과 중 하나를 무작위로 선택했을 때 $y_c$를 선택한 비율이다.

### 2. Flipping Noise 모델링

LLM의 비결정론적 출력으로 인해 발생하는 판단의 뒤집힘 현상을 모델링한다. 이상적인 결정 변수를 $X \in \{0, 1\}$ (1이면 $y_c$ 선택), 실제로 관찰되는 노이즈 섞인 결정을 $Z \in \{0, 1\}$라고 할 때, 관계식은 다음과 같다.
$$Z = \begin{cases} 1-X, & p[1-X|X] = q \\ X, & p[X|X] = 1-q \end{cases}$$
여기서 $q$는 결정이 뒤집힐 확률인 **Flipping Noise**를 의미한다.

### 3. Position Bias (PB)

Position Bias는 응답 순서에 따른 정확도의 차이로 정의한다.
$$\text{PB} = p[X=1|(y_c, y_r)] - p[X=1|(y_r, y_c)]$$
실제로는 $X$를 직접 알 수 없고 노이즈 섞인 $Z$만 관찰 가능하므로, 다음과 같은 De-noising 식을 통해 원래의 정확도를 추정한다.
$$p[X=1|(y_c, y_r)] = \frac{p[Z=1|(y_c, y_r)] - q_{cr}}{1 - 2q_{cr}}$$
여기서 $q_{cr}$은 $(y_c, y_r)$ 순서에서의 flipping 확률이다.

### 4. Length Bias (LB)

인간 평가자의 길이 편향 대비 LLM 판별자의 **상대적 길이 편향**을 측정한다. $\Delta l = l_c - l_r$ (선호 응답과 비선호 응답의 길이 차이)라고 할 때, LB는 다음과 같이 정의된다.
$$\text{LB} = p[X=1|\Delta l > 0] - p[X=1|\Delta l \le 0]$$
이때, PB의 영향을 제거하기 위해 정확도 측정 시 $\text{Acc}_{\text{both}}$를 사용한다.

## 📊 Results

### 실험 설정

- **데이터셋**: TL;DR Summarization 및 HH-RLHF-Helpfulness 데이터셋.
- **LLM 판별자**: GPT-4o, GPT-4o-mini, GPT-3.5-turbo.
- **프롬프트 템플릿**: 기존 문헌에서 추출한 총 8~10종의 다양한 템플릿.
- **지표**: $\text{Acc}_{\text{both}}$, $\text{Acc}_{\text{random}}$, PB, LB.

### 주요 결과

1. **Temperature의 영향**: 온도가 높을수록 자기 일관성(Self-consistent rate, $1-q$)은 낮아지지만, 정확도 자체에는 큰 영향이 없었다. 본 연구에서는 일관성이 가장 높은 $0.1$을 기본값으로 사용하였다.
2. **프롬프트 템플릿의 민감도**: 동일한 모델이라도 템플릿에 따라 정확도가 크게 달라졌으며, 일부 템플릿에서는 정확도가 $0.2$ 미만으로 매우 낮게 나타났다. 이는 판별자 구축 시 템플릿 선정의 중요성을 시사한다.
3. **정확도와 PB의 상관관계**: $|PB|$와 $\text{Acc}_{\text{both}}$ 사이에는 강한 음의 상관관계가 발견되었다. 즉, 판단 능력이 뛰어난(정확도가 높은) 모델일수록 위치 편향에 덜 휘둘리는 경향이 있다.
4. **Length Bias의 보편성**: 모든 테스트 모델이 인간보다 더 강하게 긴 응답을 선호하는 경향($LB > 0$)을 보였다. 특히 대화 작업(HH-RLHF)에서 요약 작업보다 더 심한 길이 편향이 관찰되었다.
5. **전반적인 성능**: 모든 LLM 판별자의 정확도가 $0.7$ 미만으로 나타나, LLM 판별자만으로는 부족하며 여전히 인간 평가가 필수적임을 보여주었다.

## 🧠 Insights & Discussion

본 논문은 LLM-as-a-Judge의 신뢰성을 단순한 관찰이 아닌 수학적 모델링을 통해 분석했다는 점에서 강점이 있다. 특히 **Flipping Noise**를 도입하여 LLM의 비결정론적 특성을 제거하고 순수한 편향을 측정하려 한 접근은 매우 정교하다.

실험 결과에서 주목할 점은 **모델의 능력(Internal Capacity)과 편향의 관계**이다. GPT-4o 계열이 GPT-3.5보다 높은 정확도를 보인 것은 템플릿과 무관하게 모델 자체의 기본 능력이 판별 성능을 결정함을 의미한다. 또한, PB는 모델의 능력과 상관관계가 있는 반면, LB는 모델 능력보다는 상용 모델들이 인간의 선호도에 과하게 정렬(Over-alignment)되어 나타나는 현상일 가능성이 크다고 분석하였다.

한계점으로는 상용 모델(Closed-source)에만 집중했다는 점이 있으며, 향후 Llama 3.1과 같은 오픈소스 모델 및 전용 Reward Model(RM)로 확장하여 검증할 필요가 있다.

## 📌 TL;DR

본 연구는 LLM-as-a-Judge의 신뢰성을 평가하기 위해 **Flipping Noise를 고려한 이론적 편향 측정 지표(PB, LB)**와 이를 실행할 수 있는 **오픈소스 프레임워크**를 제안하였다. 실험을 통해 프롬프트 템플릿이 판별 성능에 막대한 영향을 미치며, 모델의 능력이 높을수록 위치 편향은 감소하지만 길이 편향은 여전히 강하게 나타남을 확인하였다. 이 연구는 향후 LLM 정렬 작업에서 신뢰할 수 있는 판별자를 선택하고 구축하는 데 중요한 기준을 제공한다.
