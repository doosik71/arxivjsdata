# Object Tracking with Correlation Filters using Selective Single Background Patch

Lasitha Mekkayil, Hariharan Ramasangu (2018)

## 🧩 Problem to Solve

본 논문은 컴퓨터 비전의 핵심 과제 중 하나인 객체 추적(Object Tracking)에서 상관 필터(Correlation Filter, CF) 기반 추적기들이 겪는 고질적인 문제들을 해결하고자 한다. 기존의 CF 기반 추적기들은 높은 속도와 강건한 성능을 보여주었으나, 순환 가정(Circulant Assumption)으로 인한 경계 억제 문제와 추적 중 발생하는 표류(Drift) 현상이라는 한계를 가지고 있다.

특히, 최근 제안된 문맥 인식(Context-Aware, CA) 추적기들은 타겟 주변의 여러 배경 패치(Background Patches)를 사용하여 배경 노이즈를 억제하려 시도하였다. 그러나 많은 수의 배경 패치를 사용하는 것은 계산 효율성을 떨어뜨릴 수 있으며, 모든 배경 패치가 추적 성능 향상에 동일하게 기여하는 것은 아니다. 따라서 본 연구의 목표는 다수의 배경 패치 대신 가장 유효한 **단일 배경 패치(Single Background Patch)**를 선택적으로 사용하고, **복원 필터(Restoration Filter)**를 도입하여 추적 정확도를 높이는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 타겟을 가장 잘 구분할 수 있는 최적의 배경 샘플 하나만을 선택하여 학습에 활용함으로써 효율성과 정확성을 동시에 잡는 것이다. 주요 기여 사항은 다음과 같다.

1.  **단일 배경 패치 선택 전략**: 배경 풀(Background Pool)에서 타겟의 예측 위치와 가장 가까운 거리의 패치를 선택하여, 불필요한 배경 정보를 제거하고 변별력을 높였다.
2.  **복원 기반 상관 필터 도입**: 위너 필터(Wiener Filter) 또는 제약 최소 제곱 필터(Constrained Least Square Filter)를 통해 필터를 수정함으로써, 필터의 응답 특성을 개선하고 추적 정밀도를 향상시켰다.
3.  **표류 억제 메커니즘**: 타겟 패치에 특정 가중치(0.25)를 부여하여 모델 업데이트 시 발생할 수 있는 표류 현상을 완화하였다.

## 📎 Related Works

상관 필터(CF) 기반 추적기는 타겟의 외관을 학습하고 이를 다음 프레임에 적용하여 최대 응답 지점을 찾는 방식으로 동작한다. 본 논문에서 언급한 주요 관련 연구는 다음과 같다.

-   **MOSSE**: 적응형 상관 필터를 사용하여 출력의 제곱 오차 합을 최소화하는 최적화 방식을 제안하였다.
-   **CSK 및 KCF**: 이미지 패치의 순환 구조(Circulant Structure)를 활용하여 커널 기반의 Ridge Regression을 푸는 방식을 통해 계산 속도를 획기적으로 높였다.
-   **DSST**: 별도의 필터를 사용하여 스케일 추정(Scale Estimation) 기능을 추가하였다.
-   **Context-Aware (CA) Tracker**: 타겟 주변의 배경 정보를 명시적으로 학습에 포함시키기 위해 여러 배경 패치를 정규화 항으로 추가하여 배경 클러터(Background Clutter) 문제를 해결하려 하였다.

기존의 CA 추적기가 다수의 하드 네거티브(Hard Negative) 패치를 사용하여 계산 복잡도를 높인 반면, 본 논문의 제안 방식은 최적의 단일 패치만을 선택함으로써 효율적인 구분을 가능하게 한다는 점에서 차별점을 가진다.

## 🛠️ Methodology

본 논문에서 제안하는 RCACF(Restoration-based Context-Aware Correlation Filter)의 전체 파이프라인은 다음과 같다.

### 1. 기본 상관 필터 학습
먼저 $t$번째 프레임에서 타겟의 순환 이동 패치 $P_0$를 식별하고, 가우시안 형태의 타겟 응답 $y$를 생성한다. 이때 학습되는 필터 $w$는 다음과 같은 Ridge Regression 문제를 푸는 것으로 정의된다.

$$\min_{w} \|P_0 w - y\|^2_2 + \lambda_1 \|w\|^2_2$$

여기서 $P_0$는 타겟 패치의 모든 순환 이동을 포함하는 행렬이며, $\lambda_1$은 정규화 매개변수이다.

### 2. 필터 복원 (Restoration)
학습된 필터 $w$를 개선하기 위해 복원 필터 프레임워크를 적용한다. 위너 필터(Wiener filter) 또는 제약 최소 제곱 필터(Constrained Least Square filter)를 사용하여 수정된 필터 $w_1$을 생성함으로써 필터의 정밀도를 높인다.

### 3. 선택적 단일 배경 패치 추출 및 선택
문맥 인식 기능을 수행하기 위해 다음 단계를 거친다.
-   $(t-1)$번째 프레임에서 배경 패치들 $P_i$를 계산한다.
-   수정된 필터 $w_1$을 사용하여 타겟의 위치를 예측한다.
-   예측된 타겟 위치와 배경 패치들 $P_i$ 사이의 거리를 측정한다.
-   **최소 거리**를 가진 단일 배경 패치를 선택하고, 기존의 다수 배경 패치 집합을 이 하나의 패치로 대체한다.

### 4. 최종 필터 최적화
최종적으로 선택된 단일 배경 패치를 사용하여 다음과 같은 목적 함수를 최소화하는 필터를 학습한다. 이는 기존 CA 추적기의 목적 함수에서 배경 패치의 합을 단일 패치로 단순화한 형태이다.

$$\min_{w} \|P_0 w - y\|^2_2 + \lambda_1 \|w\|^2_2 + \lambda_2 \|P_{selected} w\|^2_2$$

여기서 $\lambda_2$는 배경 패치를 0으로 회귀시키기 위한 정규화 파라미터이며, 본 논문에서는 $\lambda_1 = 20$을 사용하였다. 또한, 타겟 패치에 0.25의 가중치를 부여하여 추적 중 발생하는 표류(Drift)를 억제한다.

## 📊 Results

### 실험 설정
-   **데이터셋**: OTB (Object Tracking Benchmark) 50개 시퀀스.
-   **비교 대상**: CT, LSST, MIL, ONNDLT, RET, TLD, SCM, SPT, Struck, VTD, MosseCA 등 11종의 최신 추적기.
-   **평가 지표**: 정밀도(Precision plot)와 성공률(Success plot).

### 정량적 결과
-   **전체 정밀도 (Overall Precision)**: 제안하는 RCACF가 **64.9%**를 기록하여, MosseCA(62.3%), Struck(62.3%), SPT(60.6%) 등 비교 대상 중 가장 높은 성능을 보였다.
-   **전체 성공률 (Overall Success)**: RCACF가 **57.8%**를 기록하여, MosseCA(56%), Struck(53.6%)보다 우수한 성능을 입증하였다.

### 속성별 분석 (Attribute-based Analysis)
RCACF는 특히 다음과 같은 환경에서 강점을 보였다.
-   **정밀도 기준**: 저해상도(89.6%), 배경 클러터(69.2%), 시야 이탈(76.4%) 등에서 높은 정밀도를 보였다.
-   **성공률 기준**: 저해상도(69.8%), 시야 이탈(66.4%), 폐쇄/오클루전(57.6%) 등에서 강건함을 보였다.
-   **취약점**: 스케일 변화(Scale Variations)에 대해서는 정밀도 31.7%, 성공률 40.1%로 상대적으로 낮은 성능을 보였다.

## 🧠 Insights & Discussion

본 논문은 다수의 배경 패치를 사용하는 기존의 문맥 인식 방식보다, **전략적으로 선택된 단일 패치**를 사용하는 것이 효율성과 정확도 면에서 더 유리할 수 있음을 보여주었다. 특히 복원 필터(Restoration Filter)의 도입이 필터의 응답 성능을 개선하여 추적 정밀도를 높이는 데 기여한 것으로 분석된다.

다만, 논문 내에서 복원 필터의 구체적인 수학적 업데이트 식이나 위너 필터의 파라미터 설정 과정이 명확하게 서술되지 않아, 실제 구현 시 재현 가능성에 한계가 있을 수 있다. 또한, 스케일 변화에 대한 성능 저하가 뚜렷하게 나타나는데, 이는 본 제안 알고리즘이 고정된 크기의 패치를 사용하는 CF의 기본 한계를 완전히 극복하지 못했음을 시사한다.

## 📌 TL;DR

본 논문은 상관 필터(CF) 기반 추적기에서 배경 노이즈를 효과적으로 제거하기 위해 **복원 필터(Restoration Filter)**를 적용하고, 배경 풀에서 가장 유의미한 **단일 배경 패치만을 선택적으로 학습**에 사용하는 RCACF 알고리즘을 제안하였다. OTB 데이터셋 평가 결과, 기존의 MosseCA 및 Struck보다 향상된 정밀도(64.9%)와 성공률(57.8%)을 달성하였으며, 특히 저해상도 및 폐쇄 상황에서 강건한 성능을 보였다. 이 연구는 배경 정보의 양보다 질적인 선택이 추적 성능 향상에 중요함을 시사하며, 향후 효율적인 실시간 추적기 설계에 기여할 가능성이 크다.
