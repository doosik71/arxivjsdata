# Large Language Model Agent: A Survey on Methodology, Applications and Challenges

Junyu Luo, Weizhi Zhang, Ye Yuan, Yusheng Zhao, Junwei Yang, Yiyang Gu, Bohan Wu, Binqi Chen, Ziyue Qiao, Qingqing Long, Rongcheng Tu, Xiao Luo, Wei Ju, Zhiping Xiao, Yifan Wang, Meng Xiao, Chenwu Liu, Jingyang Yuan, Shichang Zhang, Yiqiao Jin, Fan Zhang, Xian Wu, Hanqing Zhao, Dacheng Tao, Philip S. Yu, and Ming Zhang

## 🧩 Problem to Solve

LLM 에이전트 분야는 혁신적인 발전에도 불구하고 연구가 파편화되어 있어, 에이전트 설계, 협업, 진화 방식에 대한 통일된 체계가 부족합니다. 기존 AI 시스템은 사용자 입력에 수동적으로 반응하는 반면, LLM 에이전트는 환경을 인식하고, 목표에 따라 추론하며, 행동을 실행하는 능동적인 지능형 주체입니다. 이 논문은 이러한 LLM 에이전트의 급속한 발전에 발맞춰, 그들의 방법론, 응용 및 과제에 대한 체계적인 분류법을 제시하여, 연구자들이 이 복잡하고 빠르게 진화하는 분야를 이해할 수 있는 통합적인 관점을 제공하는 것을 목표로 합니다. 궁극적으로 LLM 에이전트가 인공 일반 지능(AGI)을 향한 중요한 경로라는 관점에서, 그 잠재력과 현재의 한계를 명확히 하고자 합니다.

## ✨ Key Contributions

이 설문 조사는 LLM 에이전트 분야에 대한 다음과 같은 주요 기여를 합니다:

* **방법론 중심의 분류 체계 제시:** LLM 에이전트 시스템을 역할 정의, 메모리 메커니즘, 계획 능력, 행동 실행과 같은 근본적인 방법론적 구성 요소로 체계적으로 분류합니다.
* **통합된 '구축-협업-진화' 프레임워크:** 에이전트가 어떻게 구축되고, 어떻게 협력하며, 시간이 지남에 따라 어떻게 진화하는지에 대한 상호 연결된 세 가지 차원(건설, 협업, 진화)을 분석하여 총체적인 이해를 제공합니다. 이는 개별 에이전트 설계와 협업 시스템 간의 연속성을 강조합니다.
* **최첨단 애플리케이션 및 실제 문제 집중:** 이론적 개념을 넘어 LLM 에이전트의 최신 도구, 통신 프로토콜, 다양한 애플리케이션을 검토하며, 보안, 개인 정보 보호, 윤리적 문제 등 실제 발생하는 중요한 과제에 대한 포괄적인 분석을 제공합니다.
* **미래 연구 방향 제시:** 빠르게 발전하는 이 분야에서 연구자들이 LLM 에이전트를 이해하고 연구를 발전시키는 데 필요한 구조화된 분류법과 유망한 미래 연구 방향을 제시합니다.

## 🛠️ Methodology

본 논문은 LLM 에이전트 시스템을 **구축(Construction), 협업(Collaboration), 진화(Evolution)**라는 세 가지 상호 연결된 차원을 중심으로 하는 방법론 중심의 분류 체계로 분석합니다.

* **구축 (Agent Construction):** 에이전트의 목표 지향적 행동을 가능하게 하는 핵심 구성 요소를 설계합니다.
  * **프로필 정의:** 에이전트의 운영 정체성 및 행동 패턴을 설정합니다. (예: 휴먼-큐레이티드 정적 프로필 [25, 26], 배치-생성 동적 프로필 [30]).
  * **메모리 메커니즘:** 정보를 저장, 구성 및 검색하는 능력을 제공합니다.
    * 단기 메모리 ($M_{short}$): 즉각적인 작업 실행을 위한 임시 데이터 유지 (예: ReAct [33]).
    * 장기 메모리 ($M_{long}$): 구조화된 경험적 지식 저장, 재사용 가능한 도구 생성 (예: Voyager [35], MemGPT [42]).
    * 지식 검색 기반 메모리 (RAG): 외부 지식 저장소 통합 (예: RAG [43], GraphRAG [44]).
  * **계획 능력:** 복잡한 작업을 탐색하는 능력을 가능하게 합니다.
    * 작업 분해 전략: 복잡한 문제를 더 작은 하위 작업으로 분해 (예: Single-path chaining [50], Multi-path tree expansion [55]).
    * 피드백 기반 반복: 환경, 인간, 모델 자체 또는 다중 에이전트 협업으로부터의 피드백을 통해 계획을 개선 (예: AdaPlanner [64]).
  * **행동 실행:** 계획된 행동을 실제 세계에서 수행합니다.
    * 도구 활용: 외부 도구 호출 (예: TRICE [66], GPT4Tools [67]).
    * 물리적 상호작용: 로봇 하드웨어 등 실제 환경에서 행동 수행 (예: DriVLMe [70]).

* **협업 (Agent Collaboration):** 여러 에이전트가 함께 작업하여 문제 해결 능력을 확장합니다.
  * **중앙 집중식 제어:** 중앙 컨트롤러가 작업 할당 및 결정 통합을 통해 에이전트 활동을 조직 (예: Coscientist [73], MetaGPT [27]).
  * **분산형 협업:** 에이전트 간 직접적인 노드-투-노드 상호작용 (예: Revision-based systems [78], Communication-based systems [30, 26]).
  * **하이브리드 아키텍처:** 중앙 집중식 조정과 분산형 협업을 전략적으로 결합 (예: CAMEL [25], AFlow [29]).

* **진화 (Agent Evolution):** 에이전트가 시간이 지남에 따라 스스로 개선됩니다.
  * **자율 최적화 및 자기 학습:** 광범위한 감독 없이 능력 향상 (예: 자기 감독 학습 [86], 자기 성찰 및 자기 교정 [89], 자기 보상 및 강화 학습 [93]).
  * **다중 에이전트 공동 진화:** 다른 에이전트와의 상호작용을 통해 개선 (예: 협력적 학습 [96, 25], 경쟁적 공동 진화 [98, 82]).
  * **외부 자원 통한 진화:** 구조화된 정보 및 피드백 통합 (예: 지식 강화 진화 [83], 외부 피드백 기반 진화 [100]).

## 📊 Results

이 설문 조사는 LLM 에이전트 분야에 대한 광범위하고 체계적인 분류와 분석을 통해 다음과 같은 주요 결과들을 제시합니다:

* **통합된 분류 체계:** LLM 에이전트의 구축, 협업, 진화라는 세 가지 핵심 차원을 중심으로 파편화된 연구들을 조직화하고, 각 차원 내의 세부 구성 요소와 방법론을 명확히 제시했습니다.
* **평가 및 도구 생태계 분석:** LLM 에이전트의 성능을 평가하는 일반적, 도메인 특화적, 협업적 평가 방법론과 에이전트가 활용하는 도구, 에이전트가 생성하는 도구, 에이전트 배포를 위한 도구를 망라하는 포괄적인 도구 생태계를 설명했습니다.
* **실제 문제에 대한 심층 분석:** LLM 에이전트 시스템이 직면한 주요 현실 문제(보안, 개인 정보 보호, 사회적 영향)에 대해 상세히 다루었으며, 각 문제의 유형(예: 에이전트 중심 공격, 데이터 중심 공격, 메모리 취약성, 저작권 침해)과 이에 대한 방어/대응 전략을 요약했습니다.
* **다양한 응용 분야 개괄:** 과학적 발견 (화학, 재료 과학, 생물학, 의학, 천문학, 데이터셋 구축), 게임, 사회 과학 (경제, 심리학, 사회 시뮬레이션), 생산성 도구 (소프트웨어 개발, 추천 시스템) 등 광범위한 도메인에서의 LLM 에이전트 활용 사례를 정리했습니다.
* **미래 방향성 제시:** 확장성, 메모리 제약, 신뢰성, 동적 평가의 어려움, 규제 조치, 역할 연기 시나리오 등 LLM 에이전트의 현재 과제와 미래 연구 동향을 식별했습니다.

## 🧠 Insights & Discussion

이 설문 조사는 LLM 에이전트 분야에 대한 깊이 있는 통찰을 제공하며, 현재의 한계와 미래 방향성을 제시합니다.

* **통찰:**
  * LLM 에이전트가 단일 모델의 한계를 넘어 지각, 추론, 행동을 통합하는 인간과 유사한 인지 루프를 형성함으로써 인공 일반 지능(AGI)을 향한 중요한 경로가 될 수 있음을 강조합니다.
  * 개별 에이전트 설계 원칙과 다중 에이전트 협업 시스템 간의 간극을 연결하는 통합 아키텍처 관점을 제시하여, 이 복잡한 분야의 통일된 이해를 돕습니다.
  * 빠르게 진화하는 LLM 에이전트 기술이 연구에서 광범위한 구현으로 전환됨에 따라, 보안, 개인 정보 보호, 윤리와 같은 실제 문제에 대한 포괄적인 분석의 중요성을 부각시킵니다.

* **한계 및 향후 과제:**
  * **확장성 및 조정:** LLM 기반 다중 에이전트 시스템은 높은 계산 요구 사항과 비효율적인 조정 메커니즘으로 인해 확장성이 어렵습니다. 계층적 구조화 및 분산형 계획이 필요합니다 [306, 307].
  * **메모리 제약 및 장기 적응:** LLM의 제한적인 컨텍스트 창으로 인해 다중 턴 대화에서 일관성을 유지하고 장기적인 지식 축적이 어렵습니다. 계층적 메모리 아키텍처와 자율적 지식 압축 기술이 중요합니다 [310, 313].
  * **신뢰성 및 과학적 엄격성:** LLM의 확률적 특성과 최신 정보 부족은 환각(hallucinations) 및 불확실성을 초래할 수 있습니다. 지식 그래프 기반 검증, 상호 참조, 인간-AI 검증 루프를 통한 엄격한 검증 메커니즘이 필수적입니다 [315, 317].
  * **다중 턴, 다중 에이전트 동적 평가:** 기존의 정적 평가 프레임워크는 LLM 에이전트의 동적이고 다중 턴, 다중 에이전트 환경의 복잡성을 포착하기에 부적합합니다. 동적 상호작용 시나리오와 적응형 샘플 생성 알고리즘을 통합한 평가 방법론이 요구됩니다 [310, 324].
  * **안전한 배포를 위한 규제 조치:** 알고리즘 편향 완화, 책임성, 투명성, 안전성 보장을 위한 규제 프레임워크가 필요하며, 표준화된 감사 프로토콜과 다학제적 접근 방식이 중요합니다 [248, 326].
  * **역할 연기 시나리오:** 훈련 데이터의 한계와 인간 인지에 대한 불완전한 이해로 인해 LLM 에이전트의 역할 연기 효과가 제한됩니다. 다중 에이전트 조정 개선, 실제 추론 프레임워크 통합, 대화 다양성 향상이 필요합니다 [326, 329].

이러한 과제들을 해결하는 것은 LLM 에이전트 기술의 책임감 있고 신뢰할 수 있는 발전을 위해 필수적이며, 인간-기계 협업의 미래를 근본적으로 재편할 잠재력을 가지고 있습니다.

## 📌 TL;DR

이 논문은 급속도로 발전하는 LLM 에이전트 분야의 파편화된 이해를 해결하기 위해, 에이전트의 **구축, 협업, 진화**라는 세 가지 핵심 차원을 중심으로 하는 체계적인 방법론 분류 체계를 제시합니다. 이를 통해 연구의 통일된 아키텍처 관점을 제공하고, 프로필 정의, 메모리, 계획, 행동 실행 등 에이전트 구축 방법론을 상세히 설명합니다. 또한 중앙 집중식/분산형 협업 모델, 자율 학습 및 공동 진화 메커니즘을 분석하며, LLM 에이전트의 평가 방법, 도구 활용, 그리고 보안, 개인 정보 보호, 사회적 영향과 같은 실제 과제들을 다룹니다. 궁극적으로 과학적 발견부터 게임, 사회 과학, 생산성 도구에 이르는 다양한 응용 분야를 조명하고, 확장성, 메모리 제약, 신뢰성, 동적 평가 등의 미래 도전 과제를 제시하여, 이 분야의 책임감 있는 발전을 위한 기초를 다집니다.
