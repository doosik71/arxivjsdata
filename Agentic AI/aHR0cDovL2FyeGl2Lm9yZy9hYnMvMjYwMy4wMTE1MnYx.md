# DeepResearch-9K: A Challenging Benchmark Dataset of Deep-Research Agent

Tongzhou Wu, Yuhao Wang, Xinyu Ma, Xiuqiang He, Shuaiqiang Wang, Dawei Yin, Xiangyu Zhao (2018 - *Note: The text contains 2018 in the ACM template, but references 2025/2026 papers, indicating it is a recent work*)

## 🧩 Problem to Solve

본 논문은 Deep-Research Agent가 직면한 두 가지 핵심적인 병목 현상을 해결하고자 한다. 첫째는 실제 세계의 난이도를 반영한 대규모의 도전적인 데이터셋이 부족하다는 점이며, 둘째는 데이터 합성 및 에이전트 학습을 위한 접근 가능한 오픈소스 프레임워크가 부재하다는 점이다. 

기존의 Multi-hop QA 데이터셋(예: HotpotQA, MuSiQue)은 정보 통합 능력을 평가하지만, 실제 연구 과정에서 필요한 자율적인 탐색 능력을 측정하기에는 부족하다. 특히, 기존 벤치마크들은 검색 도구 호출 횟수와 난이도 간의 체계적인 상관관계를 제시하지 못하며, 정적인 폐쇄형 코퍼스에 의존하여 동적인 웹 환경에서의 전략적 계획 및 반복적 검증 능력을 평가하는 데 한계가 있다. 따라서 본 연구의 목표는 난이도별로 설계된 대규모 벤치마크 데이터셋인 DeepResearch-9K를 구축하고, 이를 학습시킬 수 있는 오픈소스 프레임워크인 DeepResearch-R1을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 다음과 같다.

1. **다층적 난이도의 데이터셋 구축**: 총 9,000개의 고품질 인스턴스로 구성된 DeepResearch-9K를 제안한다. 이 데이터셋은 난이도를 $L_1$에서 $L_3$까지 세 단계로 나누었으며, 난이도의 기준을 해결에 필요한 검색 도구 호출 횟수(Tool Call Frequency)와 직접적으로 연계하였다.
2. **효율적인 데이터 합성 파이프라인**: 인간의 광범위한 주석 없이도 시드 엔티티 추출, 계층적 관계 그래프 구축, 보정된 추상화(Calibrated Abstraction)를 통해 난이도별 연구 쿼리를 생성하는 저비용 파이프라인을 설계하였다. 이 과정의 전체 비용은 약 200달러에 불과하다.
3. **DeepResearch-R1 학습 프레임워크**: 다회차 웹 상호작용, 다양한 강화학습(RL) 알고리즘(PPO, GRPO), 그리고 규칙 기반의 결과 보상 및 LLM-as-judge 피드백을 지원하는 오픈소스 학습 프레임워크를 개발하였다.
4. **학습 패러다임 분석**: Zero-RL과 SFT+RL 방식을 체계적으로 비교하여, SFT가 콜드 스타트(Cold-start) 문제를 완화하며 특화된 소형 모델이 RL을 통해 대형 모델의 성능을 능가할 수 있음을 입증하였다.

## 📎 Related Works

본 논문은 기존의 웹 기반 QA 및 에이전트 벤치마크를 다룬다. GAIA는 추론 단계와 도구 호출 횟수로 과업을 구분하였고, GPQA나 Humanity's Last Exam은 단순 검색으로 해결 불가능한 전문가 수준의 문제에 집중하였다. BrowseComp는 여러 웹사이트를 능동적으로 탐색하고 비교하는 능력을 테스트한다.

그러나 기존 연구들의 한계는 다음과 같다. 대부분의 벤치마크가 단순 평가 목적으로 설계되어, 에이전트를 처음부터 학습시키거나 반복적으로 개선하는 데 필요한 대규모의 고품질 상호작용 궤적(Interaction Trajectories)을 제공하지 않는다. 즉, 평가 도구는 존재하지만 학습을 위한 데이터 생성 및 훈련 체계가 부족한 상태이다. 본 연구는 이러한 공백을 메우기 위해 검증된 다단계 훈련 궤적을 합성하고 이를 학습에 활용할 수 있는 파이프라인을 제공함으로써 차별점을 갖는다.

## 🛠️ Methodology

### 1. DeepResearch-9K 데이터셋 구축
데이터셋은 $L_1$부터 $L_3$까지 난이도를 점진적으로 높이는 계층적 구성 전략을 따른다.

- **$L_1$ (Direct Attribute Mapping)**: 시드 엔티티의 특정 속성을 찾는 단계이다. 단순한 유의어 대체(Light Obfuscation)를 적용하며, 1~2회의 검색으로 해결 가능하다.
- **$L_2$ (Multi-hop Relational Bridging)**: $A \rightarrow B \rightarrow C$ 형태의 관계 체인을 구축한다. 고유 명사, 날짜, 위치 등을 제거하는 중간 단계의 모호화(Moderate Obfuscation)를 적용하여 단일 검색으로는 해결할 수 없게 설계하였다.
- **$L_3$ (Deep-Research / Long-Chain Reasoning)**: $A \rightarrow B \rightarrow \dots \rightarrow \text{Target}$ 형태의 긴 순차적 체인을 구축한다. 
    - **Hard Link Rule**: 각 홉(hop)은 새로운 독립적인 검색 쿼리를 필요로 해야 한다.
    - **Verification Constraint**: 단일 웹페이지나 문서가 체인 내 연속된 두 개 이상의 엔티티를 포함해서는 안 된다.
    - **High Obfuscation**: 모든 고유 식별자를 제거하고 기능적 역할이나 역사적 시대적 배경으로 대체하며, 복잡한 중첩 문장 구조(Nested sentence structures)를 사용하여 최소 15회 이상의 검색이 필요하도록 설계하였다.

### 2. 데이터 합성 파이프라인
전체 과정은 다음의 4단계로 진행된다.
1. **시드 엔티티 식별**: HotpotQA, 2WikiMultihopQA, MuSiQue에서 샘플을 추출하고 DeepSeek-V3를 통해 핵심 엔티티를 추출한다.
2. **추론 체인 구축**: 추출된 엔티티를 기반으로 관계 그래프를 생성하고 선택적 경로 추출을 통해 추론 깊이를 제어한다.
3. **엔티티 모호화**: 난이도 설정에 따라 단계적으로 식별자를 마스킹한다.
4. **규칙 기반 품질 보증**: LLM 기반 분석을 통해 도출된 가이드라인에 따라 논리적 엄밀성을 검증한다.

### 3. 학습 절차 및 프레임워크 (DeepResearch-R1)
본 연구는 통계적으로 검증된 궤적을 생성하기 위해 교사 모델로 Tongyi-DeepResearch-30B-A3B를 사용하여 정답 궤적을 수집하였다. 학습은 다음 두 가지 패러다임으로 진행된다.

- **Zero-RL**: 지시어 튜닝이 되지 않은 베이스 모델에 직접 강화학습을 적용하여 기초적인 추론 및 도구 사용 능력을 부트스트랩(Bootstrap)한다.
- **SFT + RL**: 교사 모델의 정답 궤적으로 Supervised Fine-Tuning(SFT)을 먼저 수행하여 추론 패턴을 학습시킨 후, 전체 데이터셋으로 RL을 수행하여 행동을 정밀화한다.

학습 알고리즘으로는 GRPO(Group Relative Policy Optimization)와 PPO(Proximal Policy Optimization)를 사용하였으며, DeepSeek-V3를 Reward Model로 활용하여 추론 궤적의 품질과 정답 여부를 평가하였다.

## 📊 Results

### 1. 직접 평가 및 난이도 검증
교사 모델인 Tongyi-DeepResearch-30B-A3B를 DeepResearch-9K에 적용한 결과, 난이도에 따른 정확도 차이가 뚜렷하게 나타났다.
- $L_1$: 72.47%
- $L_2$: 71.33%
- $L_3$: 23.73%

특히 $L_3$의 정확도가 기존의 고난도 벤치마크인 BrowseComp-Plus에서의 성능(24.94%)과 매우 유사하게 나타남으로써, $L_3$가 실제 세계의 복잡한 연구 수준의 난이도를 정확히 반영하고 있음을 입증하였다.

### 2. 학습 패러다임 분석
Qwen2.5-3B와 Llama3.2-3B 모델을 대상으로 실험한 결과는 다음과 같다.
- **Qwen 모델의 SFT 의존성**: Qwen2.5-3B는 Zero-RL 시 정확도가 약 12%에 불과했으나, SFT+RL 적용 시 20% 이상으로 크게 상승하였다. 이는 해당 아키텍처에서 도구 사용 능력을 갖추기 위해 SFT를 통한 '웜 스타트(Warm start)'가 필수적임을 시사한다.
- **소형 특화 모델의 가능성**: Llama3.2-3B 모델에 PPO(Zero-RL)를 적용한 결과, **22.50%**라는 가장 높은 정확도를 기록하였다. 이는 적절한 보상 신호와 RL이 결합되었을 때, 소형 모델이 DeepSeek V3(20.18%)와 같은 거대 모델보다 더 나은 성능을 낼 수 있음을 보여준다.
- **전반적 한계**: 모든 모델의 성능이 20% 부근에서 정체되는 양상을 보였는데, 이는 본 데이터셋이 현재 에이전트 기술의 한계를 엄격하게 측정하고 있음을 의미한다.

## 🧠 Insights & Discussion

본 연구는 Deep-Research Agent의 능력을 향상시키기 위해서는 단순한 데이터의 양보다 **'추론 체인의 길이'**와 **'엔티티 모호화 정도'**가 핵심임을 보여주었다. 특히 검색 도구 호출 횟수를 난이도의 척도로 삼은 설계는 에이전트가 얕은 수준의 검색(Shallow retrieval)으로 문제를 해결하는 '지름길(Shortcut)'을 차단하고, 실제 연구와 유사한 심층적 탐색을 강제하는 효과적인 방법이었다.

실험 결과에서 나타난 흥미로운 점은 Zero-RL이 특정 모델(Llama)에서 매우 강력한 성능을 보였다는 것이다. 이는 정해진 정답 궤적을 모방하는 SFT보다, 보상 함수를 통해 스스로 최적의 경로를 찾아가는 RL 방식이 복잡한 연구 과업에서 더 유연한 전략을 구축하는 데 유리할 수 있음을 시사한다.

다만, 본 논문에서는 LLM-as-judge의 평가 신뢰성에 대한 상세한 분석이나, 생성된 데이터셋의 다양성(Diversity)에 대한 정량적 분석이 부족하다는 한계가 있다. 또한, 실시간 웹 환경의 동적인 변화가 에이전트의 성능에 미치는 영향에 대해서는 추가적인 논의가 필요하다.

## 📌 TL;DR

본 논문은 실제 연구 수준의 복잡한 탐색 능력을 평가하고 학습시키기 위한 대규모 벤치마크 데이터셋 **DeepResearch-9K**와 학습 프레임워크 **DeepResearch-R1**을 제안한다. 9,000개의 인스턴스는 검색 횟수에 따라 $L_1, L_2, L_3$로 구분되며, 특히 $L_3$는 고도의 엔티티 모호화와 긴 추론 체인을 통해 현존하는 최신 모델들도 20%대의 낮은 정확도를 보일 만큼 도전적인 난이도를 제공한다. 실험을 통해 SFT와 RL의 조합이 에이전트 성능 향상에 핵심적이며, 적절한 RL 최적화를 거친 소형 모델이 거대 모델을 능가할 수 있음을 입증하였다. 이 연구는 향후 자율적인 심층 연구 에이전트를 개발하기 위한 데이터 생성 및 학습의 표준 가이드라인을 제공할 가능성이 높다.
