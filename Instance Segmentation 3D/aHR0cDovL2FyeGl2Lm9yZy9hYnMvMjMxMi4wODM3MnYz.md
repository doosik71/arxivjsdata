# SAM-guided Graph Cut for 3D Instance Segmentation

Haoyu Guo, He Zhu, Sida Peng, Yuang Wang, Yujun Shen, Ruizhen Hu, Xiaowei Zhou (2024)

## 🧩 Problem to Solve

본 논문은 3D 장면의 인스턴스 분할(3D Instance Segmentation) 문제를 해결하고자 한다. 기존의 3D 인스턴스 분할 방법론들은 주로 딥러닝 기술을 3D 포인트 클라우드에 직접 적용해 왔으나, 레이블링된 3D 데이터셋의 희소성과 낮은 다양성으로 인해 학습 데이터 외의 다양한 장면으로 일반화(Generalization)하는 데 한계가 있었다.

최근에는 2D 인스턴스 분할 결과를 3D로 확장하는 Bottom-up 방식의 접근법들이 제안되었다. 그러나 이러한 방식은 서로 다른 뷰(view)에서 얻은 2D 마스크들 사이에 일관성이 떨어지는 문제가 있으며, 특정 뷰에서는 객체가 검출되지만 다른 뷰에서는 누락되는 현상이 발생하여 최종적인 3D 분할 성능을 저하시키는 원인이 된다. 따라서 본 논문의 목표는 2D 분할 모델의 강력한 일반화 성능을 효율적으로 활용하면서도, 뷰 간의 불일치 문제를 해결할 수 있는 새로운 3D 인스턴스 분할 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **3D-to-2D Query 프레임워크**를 통해 SAM(Segment Anything Model)을 활용하여 3D 슈퍼포인트 그래프(Superpoint Graph)를 구축하고, 이를 Graph Cut 문제로 변환하여 해결하는 것이다.

주요 기여 사항은 다음과 같다.
1. **SAM 기반 그래프 구축**: 3D 장면을 슈퍼포인트로 먼저 분할하고, SAM의 Prompt 메커니즘을 이용해 슈퍼포인트 간의 관계(Edge weight)와 특성(Node feature)을 정의함으로써 다양한 장면에 대해 높은 일반화 성능을 확보하였다.
2. **GNN 기반 정교화 및 Pseudo-label 학습**: Graph Neural Network(GNN)를 도입하여 그래프의 강건성을 높였으며, 2D 분할 모델인 CropFormer를 통해 생성한 Pseudo 3D labels를 활용하여 수동 3D 어노테이션 없이도 모델을 학습시킬 수 있는 전략을 제안하였다.
3. **SOTA 성능 입증**: ScanNet200, ScanNet++, KITTI-360 등 다양한 데이터셋에서 기존 최신 방법론들을 뛰어넘는 성능을 입증하였다.

## 📎 Related Works

### 3D Scene Segmentation
전통적인 3D 분할 방식은 대규모 레이블링 데이터셋을 사용하여 포인트별 특징을 추출하고 레이블을 할당하는 지도 학습 방식을 취한다. 최근 Mask3D와 같은 Transformer 기반 모델이 우수한 성능을 보였으나, 앞서 언급했듯이 데이터 의존성이 높아 새로운 도메인에 대한 일반화 능력이 부족하다. 또한, 장면을 기하학적으로 균질한 영역인 슈퍼포인트(Superpoints)로 나누어 그래프 구조로 표현하고 클러스터링하는 접근 방식이 효율적인 대안으로 제시되어 왔다.

### 2D-to-3D Lifting
2D 이미지의 풍부한 데이터셋과 모델 성능을 3D로 전이하려는 시도가 많다. Semantic-NeRF와 같은 방식은 2D 시맨틱 분할 결과를 3D 필드로 통합한다. 인스턴스 분할의 경우, 뷰 간 인스턴스 ID를 매칭하는 문제가 핵심이며, SAM3D나 Panoptic Lifting 등이 제안되었다. 하지만 이들은 여전히 2D 마스크의 불일치 문제나 복잡한 최적화 과정으로 인한 한계를 가지고 있다. 본 논문은 이러한 Bottom-up 방식 대신 3D 슈퍼포인트를 쿼리로 사용하는 Top-down 형태의 쿼리 프레임워크를 제안하여 차별점을 둔다.

## 🛠️ Methodology

### 전체 파이프라인
본 논문의 시스템은 크게 세 단계로 구성된다: **1) 슈퍼포인트 그래프 구축 $\rightarrow$ 2) SAM 기반 그래프 어노테이션 $\rightarrow$ 3) Deep Graph Cut**.

### 1. Building the Superpoint Graph
먼저 입력된 3D 메시(Mesh) 또는 포인트 클라우드를 전통적인 기하학적 방법(예: 법선 방향 기반의 Graph Cut 또는 Potts energy segmentation)을 통해 과분할(Over-segmentation)하여 슈퍼포인트 집합 $V$를 생성한다. 두 슈퍼포인트 사이의 거리가 미리 정의된 임계값 이내일 때 에지 $E$를 연결하여 그래프 $G=(V, E)$를 구성한다. 이는 연산량을 줄이고 3D 공간의 프록시(Proxy)로서 2D 이미지 정보를 결합하기 위함이다.

### 2. Constructing Edge Weights and Node Features
SAM의 프롬프트 메커니즘을 사용하여 그래프의 노드와 에지에 의미론적 정보를 부여한다.

**에지 가중치(Edge Weights):**
두 슈퍼포인트가 공통으로 보이는 뷰를 선택하고, 각 슈퍼포인트의 2D 투영 영역에서 $k$개의 점을 균일하게 샘플링하여 SAM의 프롬프트로 입력한다. 이를 통해 얻은 두 마스크 $A$와 $B$에 대해 다음과 같이 가중치 $w$를 계산한다.
$$w = \max\left(\frac{A^T B}{A}, \frac{A^T B}{B}\right)$$
여기서 $A^T B$는 두 마스크의 교집합 영역이다. 여러 뷰에 대해 가중치를 계산한 후, 뷰의 신뢰도와 2D 투영 거리를 고려한 계수 $c_i$를 곱해 가중 평균을 낸다: $w = \sum c_i w_i$.

**노드 특징(Node Features):**
슈퍼포인트가 투영된 영역에서 샘플링된 포인트들의 SAM 인코더 특징(Encoder features)을 추출하고, 이를 모든 가시 뷰에 대해 평균 내어 노드의 속성으로 사용한다.

### 3. Deep Graph Cut
구축된 그래프를 GNN에 입력하여 에지의 친화도(Affinity score)를 예측하고 이를 기반으로 최종 분할을 수행한다.

**GNN 아키텍처:**
- **GCN 레이어**: 5층의 Graph Convolutional Network를 통해 각 정점의 특징을 추출한다.
- **MLP 레이어**: 두 정점의 GCN 특징과 SAM 기반 에지 가중치를 결합(Concatenate)하여 3층의 MLP에 입력하고, 최종적으로 두 정점 간의 친화도 점수를 예측한다.

**Pseudo Labels 및 학습 절차:**
3D 정답 데이터 없이 학습하기 위해 2D 분할 모델인 CropFormer를 사용하여 의사 레이블(Pseudo labels)을 생성한다. 여러 뷰에서 일관되게 동일한 마스크에 속하는 슈퍼포인트 쌍을 양성 샘플로 간주한다. 손실 함수는 다음과 같다.
1. **Binary Cross-Entropy Loss ($L_{BCE}$)**: 의사 레이블이 존재하는 에지에 대해 예측값 $s$와 레이블 $s_{ps}$ 사이의 오차를 계산한다.
   $$L_{BCE} = -[s_{ps} \log(s) + (1-s_{ps}) \log(1-s)]$$
2. **Regularization Loss ($L_{reg}$)**: 의사 레이블이 없는 희소한 에지들에 대해, SAM이 예측한 가중치 $w_{SAM}$과 GNN의 예측값 $s$가 일관성을 유지하도록 강제한다.
   $$L_{reg} = |w_{SAM} - 0.5| L_1(s, w_{SAM})$$
최종 손실 함수는 $L = L_{BCE} + L_{reg}$가 된다.

## 📊 Results

### 실험 설정
- **데이터셋**: ScanNet200 (실내), ScanNet++ (고해상도 실내), KITTI-360 (대규모 실외).
- **지표**: 클래스 불가지론적(Class-agnostic) Average Precision (mAP, $\text{AP}_{50}$, $\text{AP}_{25}$)을 사용한다.
- **비교 대상**: 전통적 방법(Felzenszwalb, Guinard), 2D-to-3D lifting(SAM3D), 지도 학습 기반 SOTA(Mask3D).

### 정량적 결과
표 1에 따르면, 제안 방법은 모든 데이터셋에서 baseline들을 크게 상회한다. 특히 GNN을 적용했을 때 성능이 더욱 향상되며, ScanNet200에서 학습된 GNN 모델이 ScanNet++ 및 KITTI-360에서도 성능 향상을 보였다는 점은 GNN이 일반적인 분할 사전 지식(General prior)을 학습했음을 시사한다.

### 일반화 능력 비교 (Mask3D vs Ours)
표 2의 결과가 매우 중요하다. 지도 학습 기반의 Mask3D는 ScanNet200 내에서는 매우 높은 성능을 보이지만, 데이터 수집 방식이 다른 ScanNet++나 완전히 다른 환경인 KITTI-360에서는 성능이 급격히 하락한다(KITTI-360의 mAP는 거의 0에 수렴). 반면, 본 제안 방법은 별도의 학습 없이도(w/o GNN) 모든 데이터셋에서 준수한 성능을 보이며, GNN 적용 시 더욱 강력한 성능과 일반화 능력을 입증하였다.

## 🧠 Insights & Discussion

### 강점 및 해석
본 연구의 가장 큰 강점은 3D 데이터의 부족 문제를 2D 파운데이션 모델(SAM)의 제로샷(Zero-shot) 능력을 통해 해결했다는 점이다. 특히 3D-to-2D 쿼리 방식을 통해 뷰 간의 불일치 문제를 완화하였고, CropFormer를 이용한 의사 레이블링 전략으로 지도 학습 없이도 정교한 GNN 모델을 구축하였다. 이는 3D 장면 이해 분야에서 데이터 획득 비용을 획기적으로 줄일 수 있는 방향성을 제시한다.

### 한계 및 비판적 분석
1. **입력 데이터의 제약**: 본 방법은 3D 기하 구조뿐만 아니라 정밀하게 캘리브레이션된 다중 뷰 이미지가 필수적이다. 따라서 이미지 정보가 없거나 캘리브레이션이 부정확한 환경에서는 적용이 어렵다.
2. **슈퍼포인트 의존성**: 분할의 기본 단위가 슈퍼포인트이므로, 만약 객체가 슈퍼포인트 생성 단계에서 잘못 묶였다면(예: 벽에 붙은 포스터가 벽과 함께 하나의 슈퍼포인트가 된 경우) 이를 다시 분리해낼 방법이 없다.
3. **연산 비용**: 모든 에지에 대해 SAM 프롬프트를 실행하고 마스크를 생성하는 과정은 연산량이 많을 것으로 예상되나, 이에 대한 구체적인 추론 시간 분석은 본문에 명시되지 않았다.

## 📌 TL;DR

본 논문은 3D 인스턴스 분할을 위해 **SAM의 프롬프트 메커니즘을 활용한 슈퍼포인트 그래프 컷(Graph Cut)** 방식을 제안한다. 3D 장면을 슈퍼포인트로 나누고 SAM을 통해 에지 가중치와 노드 특징을 추출한 뒤, GNN으로 이를 정교화하여 최종 분할을 수행한다. 특히 3D 정답 레이블 없이 2D 모델(CropFormer)의 의사 레이블만으로 학습하여, 지도 학습 모델인 Mask3D보다 훨씬 뛰어난 **범용적 일반화 성능(Generalization)**을 달성하였다. 이 연구는 2D-3D 통합 파이프라인을 통해 데이터 희소성 문제를 해결하는 효율적인 경로를 제시하며, 향후 다양한 실내외 환경의 3D 장면 이해에 중요한 역할을 할 것으로 보인다.
