# MASKGROUP: HIERARCHICAL POINT GROUPING AND MASKING FOR 3D INSTANCE SEGMENTATION

Min Zhong, Xinghao Chen, Xiaokang Chen, Gang Zeng, Yunhe Wang (2022)

## 🧩 Problem to Solve

본 논문은 3D 환경에서 각 객체를 개별적으로 분리하고 식별하는 **3D Instance Segmentation** 문제를 해결하고자 한다. 3D 장면은 매우 복잡하며, 특히 다음과 같은 구체적인 문제들이 존재한다.

1.  **객체 스케일의 다양성**: 실제 환경의 객체들은 크기가 매우 다양하다. 기존의 PointGroup과 같은 방식은 단일한 클러스터링 반경(clustering radius)을 사용하는데, 이는 매우 작은 객체를 분리하기 위한 좁은 간격과 큰 객체의 포인트들을 놓치지 않기 위한 넓은 간격 사이의 트레이드-오프 문제를 야기한다.
2.  **노이즈 포인트의 존재**: 3D 포인트 클라우드 데이터의 특성상, 객체를 그룹화한 이후에도 해당 인스턴스에 포함되지 않아야 할 노이즈 포인트들이 섞여 들어가는 경우가 많아 세그멘테이션의 정밀도가 떨어진다.

따라서 본 연구의 목표는 다양한 크기의 객체를 효과적으로 그룹화하고, 그룹화된 결과에서 노이즈를 제거하여 정밀한 인스턴스 마스크를 생성하는 프레임워크를 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **계층적 그룹화(Hierarchical Grouping)**와 **마스킹 기반 정제(Masking-based Refinement)**를 결합하는 것이다.

- **Hierarchical Point Grouping (HPG)**: 단일 반경 대신, 작은 반경에서 시작해 점진적으로 반경을 넓혀가며 포인트를 그룹화한다. 이를 통해 작은 객체부터 큰 객체까지 다양한 스케일의 정보를 캡처할 수 있다.
- **MaskScoreNet**: 그룹화된 결과물(proposals) 내의 노이즈를 제거하기 위해 이진 마스크(binary mask)를 생성하는 Mask branch와, 해당 인스턴스의 품질을 평가하는 Score branch로 구성된 경량 네트워크를 제안한다.

## 📎 Related Works

기존의 3D 인스턴스 세그멘테이션 방식은 크게 두 가지로 나뉜다.

1.  **Detection-based Methods**: 3D Bounding Box를 먼저 예측하고 그 내부의 포인트를 마스킹하는 방식(예: 3D-RCNN, BoNet)이다.
2.  **Embedding-based Methods**: 각 포인트의 공간적 또는 특징적 임베딩 벡터를 학습하고 클러스터링하는 방식(예: PointGroup, OccuSeg)이다.

특히 **PointGroup**은 포인트들을 인스턴스 중심으로 이동시킨 후 void space를 고려해 그룹화하는 bottom-up 방식을 사용하지만, 앞서 언급한 것처럼 **단일한 거리 임계값**을 사용한다는 한계가 있다. 본 논문은 이러한 단일 스케일 접근법의 한계를 극복하기 위해 계층적 구조를 도입함으로써 차별점을 갖는다.

## 🛠️ Methodology

### 1. 전체 파이프라인
전체 시스템은 입력 포인트 클라우드 $P$에 대해 다음과 같은 순서로 동작한다.
- **Backbone**: Voxel-based 네트워크를 통해 포인트 특징 $F^b$를 추출한다.
- **Sub-branches**: 특징 $F^b$를 바탕으로 각 포인트의 **Semantic Label $S$**와 인스턴스 중심점(centroid)으로 향하는 **Offset Vector $D$**를 예측한다.
- **Centroid Estimation**: 원래 좌표 $\mu_i$에 예측된 오프셋 $d_i$를 더해 추정 중심점 $o_i = \mu_i + d_i$를 구한다.
- **HPG**: 추정된 중심점을 기준으로 계층적 그룹화를 수행하여 다양한 스케일의 그룹 $G$를 생성한다.
- **MaskScoreNet**: 생성된 각 그룹에 대해 노이즈를 제거하는 마스크 $M$과 품질 점수 $E$를 예측한다.
- **NMS**: 최종적으로 Non-Maximum Suppression을 통해 중복된 인스턴스를 제거하고 최종 결과를 도출한다.

### 2. Hierarchical Point Grouping (HPG)
HPG는 다단계 클러스터링 과정을 거친다.
- **1단계**: 매우 작은 반경 $r_1$을 사용하여 포인트들을 초기 소그룹으로 묶는다.
- **h단계**: 이전 단계($h-1$)에서 생성된 그룹들을 입력으로 받아, 더 큰 반경 $r_h$ ($r_{h-1} < r_h$)를 사용하여 그룹들을 병합한다.
- **결과**: 각 단계에서 얻어진 그룹들을 모두 수집하여 $\mathcal{G} = \{G_1 \cup \dots \cup G_H\}$를 형성한다. 작은 반경에서 생성된 그룹은 작은 객체에, 큰 반경에서 생성된 그룹은 큰 객체에 최적화되어 멀티스케일 정보를 확보할 수 있다.

### 3. MaskScoreNet
그룹화된 결과에 포함된 노이즈를 제거하기 위해 다음 두 가지 브랜치를 사용한다.
- **Mask Branch**: 그룹의 포인트 특징들을 U-Net에 통과시켜 각 포인트가 실제 객체에 속할 확률을 나타내는 이진 마스크 $M$을 예측한다.
- **Score Branch**: 예측된 마스크 $M$을 사용하여 그룹 특징을 가중 평균하는 **MaskPooling** 레이어를 거친 후, 해당 그룹의 최종 품질 점수 $E$를 예측한다.

### 4. 손실 함수 (Loss Functions)
전체 네트워크는 다음의 합산 손실 함수로 학습된다:
$$L = L_{sem} + L_{off} + L_{dir} + L_{mask} + L_{score}$$

- **Semantic Loss ($L_{sem}$)**: 교차 엔트로피(Cross Entropy)를 사용한 클래스 분류 손실이다.
- **Offset Loss ($L_{off}$)**: 예측된 중심점과 실제 중심점 사이의 $L_1$ 거리 손실이다.
  $$L_{off} = \frac{1}{|\hat{I}|} \sum_{\hat{i}=1}^{|\hat{I}|} \frac{1}{N_{\hat{i}}} \sum_{i=1}^{N_{\hat{i}}} ||d_i + \mu_i - \hat{o}_i||$$
- **Direction Loss ($L_{dir}$)**: 오프셋 벡터의 방향이 실제 중심점 방향과 일치하도록 강제하는 손실이다.
  $$L_{dir} = -\frac{1}{|\hat{I}|} \sum_{\hat{i}=1}^{|\hat{I}|} \frac{1}{N_{\hat{i}}} \sum_{i=1}^{N_{\hat{i}}} \frac{d_i}{||d_i||^2} \cdot \frac{\hat{o}_i - \mu_i}{||\hat{o}_i - \mu_i||^2}$$
- **Mask Loss ($L_{mask}$)**: 예측 마스크와 Ground-truth 마스크 간의 Binary Cross Entropy(BCE) 손실이다.
- **Score Loss ($L_{score}$)**: 예측된 품질 점수 $E$와 실제 IoU 기반 점수 $\hat{E}$ 간의 BCE 손실이다.

## 📊 Results

### 1. 실험 설정
- **데이터셋**: ScanNetV2, S3DIS
- **평가 지표**: $AP$, $AP_{50}$, $AP_{25}$, mean Precision(mPrec), mean Recall(mRec)
- **주요 설정**: 클러스터링 반경을 $\{0.01m, 0.03m, 0.05m\}$로 설정하고, 최소 포인트 수 $N_\theta$는 50으로 설정하였다.

### 2. 정량적 결과
- **ScanNetV2**: $AP_{50}$ 기준으로 **66.4%**를 달성하여 기존 SOTA 모델인 PE(64.5%)보다 1.9% 높고, PointGroup(63.6%)보다 큰 폭으로 향상되었다. 특히 18개 클래스 중 10개 클래스에서 1위를 기록하였다.
- **S3DIS**: Area5와 6-Fold 교차 검증 모두에서 경쟁력 있는 성능을 보였으며, 특히 $AP_{50}$ 기준 PointGroup 대비 각각 7.2%, 5.9% 높은 성능을 기록하였다.

### 3. Ablation Study
- **MaskScoreNet의 효과**: 마스크 브랜치와 MaskPool을 제거했을 때 $AP_{50}$이 62.7%에서 58.3%로 크게 하락하여, 노이즈 제거 과정이 필수적임을 확인하였다.
- **HPG의 효과**: 단일 단계 그룹화를 사용했을 때보다 계층적 그룹화를 사용했을 때 성능이 향상되었다.
- **멀티스케일 반경의 영향**: 단일 반경($0.01$ 또는 $0.03$)을 사용할 때보다 여러 반경 $\{0.01, 0.03, 0.05\}$을 조합하여 사용할 때 가장 높은 성능을 보였다. 이는 다양한 크기의 객체를 동시에 잡아내기 위함이다.

## 🧠 Insights & Discussion

### 강점
- **멀티스케일 대응 능력**: HPG를 통해 단일 임계값 설정의 딜레마를 해결하고, 다양한 크기의 객체를 유연하게 포착할 수 있음을 입증하였다.
- **정밀한 정제 과정**: 단순히 그룹화에서 끝내지 않고 MaskScoreNet을 통해 사후 정제를 수행함으로써 3D 데이터 특유의 노이즈 문제를 효과적으로 해결하였다.

### 한계 및 비판적 해석
- **기하학적 유사성 문제**: 논문의 failure case 분석에서 언급되었듯, 문(door)과 캐비닛(cabinet)처럼 모양이 비슷하고 위치가 매우 인접한 객체들을 완벽하게 분리하는 데에는 여전히 어려움이 있다. 이는 단순히 중심점 기반의 그룹화와 마스킹만으로는 해결하기 어려운, 고차원의 기하학적 특징 분석이 필요함을 시사한다.
- **계산 복잡도**: 계층적으로 여러 번 그룹화를 수행하고 각 그룹에 대해 MaskScoreNet을 적용하는 과정이 추가되므로, 단일 단계 방식보다 추론 시간이 증가했을 가능성이 크나 이에 대한 정량적 분석은 제시되지 않았다.

## 📌 TL;DR

본 논문은 3D 인스턴스 세그멘테이션에서 발생하는 **객체 크기 다양성**과 **노이즈 포인트** 문제를 해결하기 위해 **MaskGroup** 프레임워크를 제안한다. 점진적으로 반경을 넓혀가며 그룹화하는 **Hierarchical Point Grouping (HPG)**과 그룹의 품질을 높이고 노이즈를 제거하는 **MaskScoreNet**을 통해 ScanNetV2와 S3DIS 데이터셋에서 SOTA 성능을 달성하였다. 이 연구는 3D 장면 분석에서 멀티스케일 정보 활용의 중요성을 입증하였으며, 향후 정밀한 3D 객체 분리가 필요한 로보틱스 및 AR 분야에 기여할 가능성이 높다.
