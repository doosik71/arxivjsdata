# AutoInst: Automatic Instance-Based Segmentation of LiDAR 3D Scans

Cedric Perauer, Laurenz Adrian Heidrich, Haifan Zhang, Matthias Nießner, Anastasiia Kornilova, and Alexey Artemov (2024)

## 🧩 Problem to Solve

본 논문은 자율 주행 시스템 및 3D 환경 이해의 핵심 과제인 LiDAR 3D 스캔 데이터의 인스턴스 분할(Instance Segmentation) 문제를 다룬다. 기존의 딥러닝 기반 인스턴스 분할 방식은 대규모의 정밀한 주석(Annotation) 데이터셋을 필요로 하지만, 3D 데이터의 특성상 이러한 레이블을 생성하는 과정은 매우 비용이 많이 들고 노동 집약적이다. 또한, 기존의 지도 학습 모델은 센서 구성이나 데이터 획득 조건이 달라질 경우 성능이 저하되는 도메인 적응(Domain Adaptation) 문제를 겪는다.

따라서 본 연구의 목표는 정답 레이블(Ground-truth) 없이도 3D 장면에서 개별 객체를 분리해낼 수 있는 비지도 학습(Unsupervised Learning) 기반의 자동 인스턴스 분할 프레임워크를 구축하는 것이다. 특히, 특정 클래스에 국한되지 않는 클래스 불가지론적(Class-agnostic) 분할을 수행하여, 이후 다양한 하위 작업에 적용할 수 있도록 하는 것을 지향한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **'비지도 방식의 마스크 제안(Proposal Generation)'**과 **'자기 학습 기반의 정제(Self-trained Refinement)'**라는 2단계 파이프라인을 통해 노이즈가 섞인 초기 예측치를 정교한 인스턴스 마스크로 발전시키는 것이다.

중심적인 설계 직관은 다음과 같다.

1. **멀티모달 특징 통합:** 단순한 기하학적 정보뿐만 아니라, 사전 학습된 자기 지도 학습(Self-supervised) 모델의 포인트 특징(Point-based features)과 이미지 특징(Image-based features)을 결합하여 포인트 간의 유사도를 정밀하게 측정한다.
2. **그래프 컷을 통한 초기 분할:** 포인트들을 정점으로, 유사도를 엣지 가중치로 하는 프록시 그래프(Proxy-graph)를 구축하고 Normalized Cuts(NCut) 알고리즘을 적용하여 초기 인스턴스 후보들을 생성한다.
3. **신경망을 이용한 노이즈 제거:** 비지도 방식으로 생성된 초기 마스크는 노이즈가 많으므로, 이를 유사-레이블(Pseudo-labels)로 사용하여 신경망을 학습시킨다. 이때 신경망이 단순하고 깨끗한 패턴을 먼저 학습한다는 성질을 이용하여, 노이즈가 제거된 정제된 인스턴스를 얻어낸다.

## 📎 Related Works

논문에서는 3D 인스턴스 분할 연구를 세 가지 방향으로 분류하여 설명한다.

- **지도 학습 기반 방법:** MaskPLS와 같은 최신 모델들은 높은 성능을 보이지만, 대규모의 정밀한 주석 데이터셋이 필수적이라는 한계가 있다.
- **약지도/자기지도 학습 방법:** 희소한 레이블을 사용하거나 사전 학습된 표현(Representation)을 활용하지만, 여전히 어느 정도의 수동 입력이나 미세 조정(Fine-tuning) 단계에서 레이블이 필요하며, 추론 단계에서 처음 보는 객체 타입을 처리하는 데 한계가 있다.
- **비지도 학습 방법:** HDBSCAN과 같은 클러스터링 기반 방식이나 2D SAM(Segment Anything Model)의 결과를 3D로 투영하는 방식(SAM3D)이 존재한다. 그러나 단순 클러스터링은 복잡한 장면에서 정밀도가 떨어지며, 2D-to-3D 투영 방식은 투영 과정에서의 손실과 정렬 문제로 인해 한계가 있다.

AutoInst는 이러한 기존 접근 방식과 달리, 비지도 방식으로 생성한 후보군을 다시 딥러닝 모델로 정제하는 'Self-training' 루프를 도입하여 비지도 학습의 한계인 '낮은 정밀도'를 극복하고자 한다.

## 🛠️ Methodology

### 1. 전체 파이프라인 및 전처리

시스템은 입력된 LiDAR 스캔과 RGB 이미지들을 세계 좌표계로 정렬하여 조밀한 3D 맵을 생성하는 것으로 시작한다. 계산 복잡도를 줄이기 위해 전체 맵을 겹치는 영역이 있는 국소적인 **포인트 청크(Local point chunks)** 단위로 나누어 처리하며, 지면 제거(Ground segmentation)와 복셀 다운샘플링(Voxel downsampling)을 통해 연산 효율을 높인다.

### 2. 3D 인스턴스 마스크 제안 (Proposal Generation)

비지도 방식으로 초기 인스턴스를 찾기 위해 멀티모달 특징을 활용한 그래프 컷 방식을 사용한다.

**A. 멀티모달 특징 추출**
각 포인트 $p$에 대해 세 가지 특징을 부여한다.

- **공간 특징 ($S$):** 포인트의 3D 좌표 $x^S_p = p$.
- **포인트 임베딩 ($P$):** 사전 학습된 TARL 모델 $\phi^P$를 통해 얻은 특징 $x^P_p$.
- **이미지 임베딩 ($I$):** 사전 학습된 DINOv2 모델 $\phi^I$를 사용하여 포인트 $p$가 투영된 이미지 픽셀들의 특징 평균값 $x^I_p$를 계산한다.

**B. 프록시 그래프 구축 및 유사도 계산**
반경 $R=1\text{m}$ 내의 포인트들을 연결하여 그래프를 생성하며, 각 엣지의 가중치는 다음과 같은 유사도 함수로 결정된다. 각 특징 공간 $\mu \in \{S, P, I\}$에 대해:
$$w^\mu_{pq} = e^{-\theta^\mu ||x^\mu_p - x^\mu_q||^2}$$
최종 가중치 $w_{pq}$는 모든 모달리티의 유사도를 곱하여 산출한다:
$$w_{pq} = \prod_{\mu} w^\mu_{pq}$$
이는 모든 특징 공간에서 유사도가 높아야만 같은 인스턴스로 묶일 가능성이 높음을 의미한다.

**C. Normalized Cuts (NCut) 적용**
구축된 그래프에 NCut 알고리즘을 적용하여 포인트들을 분할한다. NCut은 인스턴스 내부의 유사도는 최대화하고 인스턴스 간의 유사도는 최소화하는 지점을 찾아 컷(Cut)을 수행한다.

### 3. 자기 학습 기반의 정제 (Self-Training Refinement)

NCut으로 생성된 초기 마스크는 과분할(Over-segmentation) 또는 과소분할(Under-segmentation) 문제가 발생할 수 있다. 이를 해결하기 위해 다음과 같은 정제 과정을 거친다.

- **학습 절차:** NCut으로 생성된 마스크들을 유사-레이블(Pseudo-labels)로 간주하여 인스턴스 분할 신경망을 학습시킨다.
- **네트워크 구조:** MaskPLS 구조(Sparse CNN Encoder + Transformer Decoder)를 채택하여 입력 좌표로부터 인스턴스 마스크와 신뢰도 값을 예측한다.
- **손실 함수:** Dice 손실과 Cross-Entropy 손실의 가중 합을 최소화한다.
$$\mathcal{L} = \lambda_{dice}\mathcal{L}_{dice} + \lambda_{bce}\mathcal{L}_{bce}$$
이 과정에서 신경망은 노이즈가 섞인 레이블 속에서도 더 단순하고 일반적인 패턴(깨끗한 객체 형태)을 먼저 학습하는 성질을 이용하여, 결과적으로 초기 제안보다 더 정밀한 마스크를 생성하게 된다.

### 4. 맵 레벨 병합

개별 청크에서 예측된 인스턴스들은 바운딩 박스의 IoU(Intersection over Union)를 기준으로 병합되어 최종적인 전체 장면의 인스턴스 분할 맵을 구성한다.

## 📊 Results

### 실험 설정

- **데이터셋:** SemanticKITTI 벤치마크 사용.
- **비교 대상 (Baselines):**
  - 비지도 학습: HDBSCAN (밀도 기반 클러스터링), 3DUIS (포인트 특징+그래프 컷), SAM3D (2D SAM 투영).
  - 지도 학습: MaskPLS (최신 지도 학습 모델).
- **평가 지표:** Precision (P), Recall (R), F1-score, Average Precision (AP), $S_{assoc}$ (S).

### 정량적 결과

- **비지도 모델 간 비교:** AutoInst는 모든 지표에서 기존 비지도 모델들을 압도하였다. 특히 최적 설정(NCut(S+P) + Refine)에서 AP@0.25 기준 79.8%를 기록하며, 베이스라인 대비 AP는 13.3%, F1-score는 9.1% 향상되었다.
- **지도 학습 모델과의 비교:** 지도 학습 기반의 MaskPLS보다는 성능이 낮지만(AP@0.25: 79.6% vs 76.9%), 레이블 없이 달성한 결과라는 점에서 그 격차가 상당히 좁음을 확인하였다.

### 절제 연구 (Ablation Study)

- **특징 조합:** 공간 특징($S$)과 포인트 임베딩($P$)의 조합이 가장 우수한 성능을 보였다. 놀랍게도 이미지 임베딩($I$)을 추가했을 때 오히려 성능이 하락하는 경향이 관찰되었다.
- **정제 단계의 효과:** NCut으로 얻은 초기 제안만 사용했을 때보다, Self-training 정제 단계를 거쳤을 때 인스턴스 분리 능력이 크게 향상되었으며, 특히 서로 뭉쳐있던 인스턴스들을 정확히 분리해내는 효과가 있었다.

## 🧠 Insights & Discussion

**강점:**
본 연구는 비지도 학습의 고질적인 문제인 '노이즈'를 해결하기 위해 딥러닝 모델의 학습 특성(단순 패턴 우선 학습)을 영리하게 이용하였다. 또한, 단순한 클러스터링을 넘어 사전 학습된 자기지도 표현(TARL)을 그래프 가중치에 통합함으로써 특징 공간의 변별력을 높였다.

**한계 및 논의:**

1. **이미지 특징의 역효과:** 이론적으로는 멀티모달 정보가 도움이 되어야 하지만, 실제 실험에서는 이미지 임베딩($I$)이 성능을 저하시켰다. 이는 LiDAR-이미지 간의 정렬 오차나, DINOv2의 특징이 3D 인스턴스 경계와 완벽히 일치하지 않기 때문일 가능성이 크다.
2. **계산 비용:** NCut 알고리즘의 복잡도로 인해 복셀 다운샘플링과 청크 단위 처리가 필수적이며, 이는 매우 정밀한 소형 객체 분할에서 손실이 발생할 수 있는 지점이다.
3. **가정:** 본 모델은 정적인 환경에서 등록된(Registered) 3D 맵을 전제로 한다. 동적 객체가 많은 환경에서의 강건성은 어느 정도 확인되었으나, 완전히 실시간으로 움직이는 단일 스캔 데이터에 적용하기에는 한계가 있다.

## 📌 TL;DR

AutoInst는 레이블 없는 LiDAR 3D 데이터에서 인스턴스를 분할하기 위해 **'멀티모달 그래프 컷(NCut) 기반의 마스크 제안 $\rightarrow$ 신경망 기반의 자기 학습 정제'** 파이프라인을 제안한다. 특히 사전 학습된 포인트 임베딩과 Self-training 기법을 결합하여, 비지도 학습임에도 불구하고 지도 학습 모델에 근접한 높은 정밀도를 달성하였다. 이 연구는 수동 주석 비용을 획기적으로 줄이면서도 고품질의 3D 장면 이해를 가능케 하여, 향후 자동 데이터 레이블링 도구 및 자율 주행 시스템의 환경 인식 능력을 높이는 데 기여할 가능성이 크다.
