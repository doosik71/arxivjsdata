# Trace3D: Consistent Segmentation Lifting via Gaussian Instance Tracing

Hongyu Shen, Junfeng Ni, Yixin Chen, Weishuo Li, Mingtao Pei, Siyuan Huang (2025)

## 🧩 Problem to Solve

본 논문은 2D 시각적 세그멘테이션(Visual Segmentation) 결과를 3D Gaussian Splatting(3DGS) 표현으로 확장(Lifting)할 때 발생하는 문제들을 해결하고자 한다. 구체적으로는 다음과 같은 두 가지 핵심 문제에 집중한다.

첫째, **다시점 간 2D 마스크의 불일치(Multi-view Inconsistency)** 문제이다. SAM(Segment Anything Model)과 같은 기존 2D 예측 모델들은 동일한 객체에 대해서도 시점에 따라 서로 다른 수준의 추상화(Abstraction)를 가진 마스크를 생성한다. 이러한 불일치는 3D 특징 학습 과정에서 모호성을 유발하여 정밀한 세그멘테이션을 방해한다.

둘째, **모호한 가우시안(Ambiguous Gaussians)으로 인한 경계 노이즈** 문제이다. 기존 방식들은 3D 장면을 구성하는 가우시안들이 서로 다른 객체에 걸쳐 있는 경우를 적절히 처리하지 못한다. 이는 결과적으로 3D 세그멘테이션 경계에 심각한 아티팩트(Artifact)를 생성하며, 객체 추출 시 품질을 저하시키는 원인이 된다.

따라서 본 논문의 목표는 가우시안의 기하학적 구조를 활용하여 다시점 간 일관성을 확보하고, 가우시안 표현 자체를 정교하게 정제하여 깨끗하고 정밀한 3D 세그멘테이션을 구현하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Gaussian Instance Tracing (GIT)**이라는 메커니즘을 도입하여 각 가우시안이 어떤 인스턴스에 속하는지를 다시점 관점에서 추적하는 것이다.

중심적인 설계 직관은 3D 공간의 가우시안은 고정되어 있지만, 이를 렌더링하는 2D 뷰는 다양하다는 점을 이용하는 것이다. 가우시안 하나하나에 대해 모든 입력 뷰에서의 인스턴스 소속 확률을 저장하는 가중치 행렬(Weight Matrix)을 구축함으로써, 2D 마스크의 불일치를 3D 수준에서 다수결(Majority Vote) 방식으로 해결하고, 여러 객체에 걸쳐 있는 모호한 가우시안을 식별하여 분할(Split)하거나 제거(Prune)할 수 있다.

## 📎 Related Works

### 관련 연구 및 한계

최근 연구들은 SAM과 같은 2D 파운데이션 모델의 결과를 3D로 확장하는 방식을 취하고 있다.

- **NeRF 기반 방식**: 특징 필드를 학습시켜 다시점 일관성을 부여하려 하지만, 렌더링 속도가 느리고 가우시안 기반 방식에 비해 최적화 효율이 떨어진다.
- **Gaussian Splatting 기반 방식**: 빠른 렌더링 속도를 활용하지만, 대부분 기존 NeRF의 아이디어를 그대로 옮겨온 수준이다. 예를 들어, 비디오 객체 추적기(Video Object Tracker)를 사용하여 마스크를 연결하려 하지만 추적이 끊기거나 계층 구조의 불일치 문제를 완전히 해결하지 못한다.

### 기존 접근 방식과의 차별점

Trace3D는 단순히 2D 마스크를 3D로 투영하는 것이 아니라, **역-래스터화(Reverse-rasterization)** 과정을 통해 가우시안과 인스턴스 간의 관계를 명시적으로 정의한다. 또한, 기존 방법들이 모호한 가우시안을 단순히 무시하거나 사후 처리로 제거하여 디테일 손실을 초래한 것과 달리, 본 논문은 학습 과정 중에 **GIT 가이드 기반의 적응형 밀도 제어(Adaptive Density Control)**를 통해 가우시안을 능동적으로 최적화한다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 1. Gaussian Instance Tracing (GIT)

GIT는 각 가우시안 $G_i$에 대해 가중치 행렬 $W_i \in \mathbb{R}^{T \times L}$를 추가하는 작업이다. 여기서 $T$는 한 뷰에서 나타날 수 있는 최대 인스턴스 패치의 수이고, $L$은 전체 뷰의 수이다.

- **작동 원리**: 역-래스터화 단계를 통해 구현된다. 각 뷰의 픽셀이 어떤 가우시안들에 의해 렌더링되었는지 추적하고, 해당 픽셀의 인스턴스 라벨을 기여도에 따라 가우시안에 할당한다.
- **결과**: 이를 통해 각 가우시안이 각 뷰의 어떤 인스턴스 패치에 영향을 주는지에 대한 확률 분포를 얻게 된다.

### 2. 일관된 2D 인스턴스 맵 생성

GIT를 이용해 다시점에서 불일치하는 2D 패치들을 통합한다. 두 패치 $P^\nu_a, P^\nu_b$가 동일한 객체인지 판단하기 위해, 이들과 연관된 가우시안 집합 $G_a, G_b$를 추적하고 다른 뷰들에서의 일관성을 확인한다.

패치 간 유사도는 다음과 같은 수식으로 계산된다:
$$\text{sim}(P_a, P_b) = \frac{1}{|\Omega|} \sum_{(G_a, G_b, \nu) \in \Omega} \langle W^\nu_{G_a}, W^\nu_{G_b} \rangle$$
여기서 $\Omega$는 두 가우시안이 공통으로 보이는 뷰의 집합이며, $\langle \cdot, \cdot \rangle$는 내적(Inner product)을 의미한다. 유사도가 임계값 $\theta=0.5$를 초과하면 두 패치를 하나로 병합한다.

### 3. GIT 가이드 적응형 밀도 제어 (Adaptive Density Control)

객체 경계에서 발생하는 모호한 가우시안을 해결하기 위한 절차이다.

- **모호한 가우시안 탐지**: 가우시안 $G_i$가 가시적인 뷰들($V_i$)에서 두 개 이상의 객체에 대해 유의미한 가중치를 가질 때 모호하다고 판단한다. 모호성 점수 $As_i$는 다음과 같다:
$$As_i = \frac{1}{|V_i|} \sum_{\nu \in V_i} \mathbb{I}(\max_j (W^\nu_{i,j}) < \gamma)$$
여기서 $\mathbb{I}$는 지시 함수이며, $\gamma=0.8$인 경우 최대 확률이 0.8 미만이면 모호한 것으로 간주한다.

- **최적화 절차**: 탐지된 모호한 가우시안을 단순히 제거하지 않고, 크기를 $1/2$로 줄여 두 개의 작은 가우시안으로 분할(Split)한다. 분할 후에도 계속 모호한 상태로 남는 가우시안은 최종적으로 제거(Prune)한다. 이 과정은 학습 중 매 1000회 반복 수행되어 경계를 정교하게 만든다.

### 4. 3D Segmentation Lifting

정제된 가우시안과 일관된 마스크를 바탕으로 두 가지 설정에서 리프팅을 수행한다.

- **Contrastive Lifting (오프라인)**: 각 가우시안에 특징 벡터 $f_i$를 부여하고, 동일 객체 픽셀 간의 특징은 가깝게, 서로 다른 객체는 멀어지게 하는 대조 손실(Contrastive Loss)을 사용한다:
$$\mathcal{L}_{\text{contr}} = -\frac{1}{|U|} \sum_{u \in U} \log \frac{\sum_{u' \in U^+} \exp(\text{sim}(F[u], F[u']; \tau))}{\sum_{u' \in U} \exp(\text{sim}(F[u], F[u']; \tau))}$$
- **Self-Prompting (온라인)**: 단일 뷰의 포인트 프롬프트로 시작하여, GIT를 통해 새로운 뷰의 프롬프트를 생성하고 SAM을 반복 호출하여 3D 가우시안 집합을 완성한다.

## 📊 Results

### 실험 설정

- **데이터셋**: Replica (실내 장면), NVOS (객체 선택 벤치마크)
- **비교 대상**: Gaussian Grouping, FlashSplat, Egolifter, OmniSeg3D, SA3D 등
- **평가 지표**: mIoU (평균 교집합 대비 합집합 비율), PSNR (신호 대 잡음비), mAcc (평균 정확도)

### 주요 결과

1. **3D 객체 추출**: Replica 데이터셋에서 mIoU와 PSNR 모두 기존 baseline들을 크게 상회하였다. 특히 가우시안을 직접 추출하여 렌더링했을 때, 배경과 전경 모두에서 아티팩트가 현저히 적고 경계가 날카로웠다.
2. **새로운 뷰 2D 세그멘테이션**:
   - Replica에서 GS 기반 방법 중 처음으로 NeRF 기반 방법(OmniSeg3D 등)보다 높은 mIoU를 달성하였다.
   - NVOS 벤치마크에서도 mIoU 92.5%로 최고 성능을 기록하며, 특히 GIT가 self-prompting 과정에서의 False Positive(오탐지)를 유의미하게 줄임을 확인하였다.
3. **Ablation Study**: 적응형 밀도 제어(Ambiguous Gaussian Control)가 3D 객체 추출의 mIoU를 크게 향상시켰으며, 일관된 2D 마스크가 3D 리프팅의 기초로서 필수적임을 입증하였다.

## 🧠 Insights & Discussion

### 강점

본 연구는 2D 세그멘테이션 모델의 고질적인 문제인 '시점 간 불일치'를 3D 가우시안이라는 명시적 표현의 특성을 이용해 해결하였다. 특히, 단순히 마스크를 합치는 것이 아니라 가우시안의 밀도 제어 단계에 세그멘테이션 제약 조건을 통합함으로써, 렌더링 품질과 세그멘테이션 정밀도를 동시에 높인 점이 인상적이다.

### 한계 및 비판적 해석

논문에서도 언급되었듯이, 본 방법은 여전히 알파 블렌딩(Alpha-blending) 기반의 암시적 렌더링 프레임워크에 의존한다. 이로 인해 실제 3D 표면 기하구조와 렌더링 기반의 연관성 사이에 간극이 존재한다. 특히 건담 모델의 세부 부품과 같이 극도로 세밀한 계층적 세그멘테이션에서는 여전히 아티팩트가 발생하며, 이는 렌더링 기반 방식의 근본적인 한계일 가능성이 크다. 또한, 패치 병합 과정의 계산 복잡도가 높으나 이를 계층적 정보로 완화하려 했다는 점은 향후 더 효율적인 데이터 구조 도입이 필요함을 시사한다.

## 📌 TL;DR

Trace3D는 Gaussian Instance Tracing(GIT)을 통해 2D 세그멘테이션 마스크의 다시점 불일치 문제를 해결하고, 모호한 가우시안을 적응적으로 분할/제거하여 정밀한 3D 세그멘테이션을 구현하는 방법론이다. 이 연구는 3D 가우시안 기반의 객체 추출 품질을 획기적으로 높였으며, 향후 3D 에셋 추출, 정밀한 장면 편집 및 로보틱스 인터랙션 등의 분야에서 핵심적인 역할을 할 가능성이 높다.
