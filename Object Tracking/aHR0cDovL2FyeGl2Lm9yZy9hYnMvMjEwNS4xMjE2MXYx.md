# Utilization and Experimental Evaluation of Occlusion Aware Kernel Correlation Filter Tracker using RGB-D

Srishti Yadav (2021)

## 🧩 Problem to Solve

본 논문은 시각적 객체 추적(Visual Tracking) 분야에서 널리 사용되는 Kernelized Correlation Filter (KCF) 추적기의 치명적인 한계점인 **폐색(Occlusion), 스케일 변화, 모델 드리프트(Model Drift)** 문제를 해결하고자 한다.

KCF는 계산 효율성이 매우 높아 실시간 추적에 유리하지만, 추적 대상이 다른 객체에 의해 가려지는 폐색 상황이 발생하면 타겟을 놓치게 되며, 한 번 손실된 타겟을 다시 찾아내는 **재검출(Re-detection)** 능력이 부족하다는 문제가 있다. 이러한 문제는 특히 군중이 많은 환경이나 복잡한 실내 공간에서 작동해야 하는 로봇 공학 및 감시 시스템에서 매우 중요하다. 따라서 본 연구의 목표는 RGB 데이터에 깊이(Depth) 정보를 추가한 RGB-D 기반의 KCF 추적기를 설계하여 폐색을 감지하고, 타겟이 다시 나타났을 때 이를 성공적으로 재검출할 수 있는 강건한 장기 추적(Long-term tracking) 시스템을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 다음과 같다.

1. **KCF 추적기의 이론적 및 실험적 상세 분석**: KCF의 수학적 원리(Circulant Matrix, Kernel Trick 등)를 시각화 도구와 함께 상세히 분석하여 구현 및 이해를 돕는 튜토리얼 성격의 가이드를 제공한다.
2. **폐색 인식 RGB-D KCF 추적기 제안**: 깊이 정보를 활용하여 타겟의 폐색 여부를 판단하고, 폐색 시 추적을 일시 중단한 후 타겟이 재등장하면 이를 재검출하는 새로운 파이프라인을 제안한다.
3. **Self-Adjusting Depth Patch 기법**: 타겟의 이동에 따라 깊이 패치(Depth Patch)의 경계에서 발생하는 배경 노이즈를 제거하여, 폐색 판정의 정확도를 높이는 적응형 깊이 패치 알고리즘을 도입한다.
4. **Particle Filter (PF) 통합을 통한 최적화**: 기존의 전수 조사 방식인 슬라이딩 윈도우(Sliding Window) 검색의 높은 계산 비용을 줄이기 위해, 베이지안 프레임워크 기반의 Particle Filter를 도입하여 타겟의 예상 위치를 더 효율적으로 국소화(Localization)한다.

## 📎 Related Works

논문에서는 기존의 추적기를 크게 생성적 추적기(Generative Tracker)와 판별적 추적기(Discriminative Tracker)로 구분하며, 특히 **Discriminative Correlation Filter (DCF)** 계열의 연구를 중점적으로 다룬다.

- **RGB 기반 추적기**: MOSSE, KCF, SRDCF 등이 있으며, 이들은 푸리에 도메인(Fourier Domain)에서의 연산을 통해 매우 빠른 속도를 자랑한다. 하지만 타겟의 외형 변화나 폐색 상황에서 모델이 배경을 학습하게 되는 드리프트 현상이 빈번하며, 재검출 능력이 결여되어 있다.
- **RGB-D 기반 추적기**: 깊이 센서(Kinect 등)를 활용해 전경과 배경을 명확히 구분함으로써 모델 드리프트를 방지한다. Princeton RGB-D 데이터셋 등을 활용한 연구들이 진행되었으나, 많은 경우 단순한 폐색 감지에 그치거나 실시간성(Speed)과 정확도(Accuracy) 사이의 트레이드-오프 문제가 존재한다.
- **차별점**: 본 연구는 단순한 성능 향상을 넘어, KCF의 수학적 구조를 깊이 있게 분석하고, '폐색 감지 $\rightarrow$ 추적 중단 $\rightarrow$ 재검출'로 이어지는 장기 추적 프로세스를 구체화했다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

### 1. 전체 파이프라인 구조

제안하는 RGB-D 추적기는 다음과 같은 단계로 작동한다.

- **특징 추출**: RGB 이미지에서는 HOG(Histogram of Oriented Gradients) 특징을, 깊이 채널에서는 거리 정보를 추출한다.
- **폐색 감지**: 현재 프레임과 이전 프레임의 타겟 중심부 깊이 패치를 비교하여, 중심 영역에서 급격한 깊이 변화(Peak)가 감지되면 폐색 상태로 판단한다.
- **추적 및 재검출**:
  - 폐색이 아닐 경우: KCF의 상관 필터를 통해 타겟의 위치를 업데이트한다.
  - 폐색일 경우: 모델 업데이트를 중단하여 오염을 방지하고, 검색 공간 내에서 타겟의 재등장을 감시한다.
  - 재검출: 상관 응답(Correlation Response) 값이 임계치($0.5$)를 넘으면 타겟이 재등장한 것으로 간주하고 추적을 재개한다.

### 2. 핵심 알고리즘 및 방정식

#### (1) KCF의 기본 원리 및 학습

KCF는 타겟 패치의 모든 가능한 순환 이동(Cyclic Shift)을 포함하는 **순환 행렬(Circulant Matrix)** 구조를 활용한다. 학습 단계에서 최적의 필터 $\alpha$는 다음과 같이 계산된다.
$$\alpha = \mathcal{F}^{-1} \left( \frac{\hat{y}}{\hat{k}_{xx} + \lambda} \right)$$
여기서 $\mathcal{F}$는 푸리에 변환, $\hat{y}$는 가우시안 회귀 라벨, $\hat{k}_{xx}$는 특징의 자기상관(Auto-correlation) 결과, $\lambda$는 정규화 계수이다.

#### (2) Self-Adjusting Depth Patch

타겟이 움직일 때 윈도우 경계에 포함되는 배경 깊이 값을 제거하기 위해, 이전 프레임과 현재 프레임의 깊이 차이를 계산한다.

- 경계 부분의 높은 피크는 배경 정보이므로 패치 크기를 줄여 제거한다.
- 하지만 조정 후에도 **중심 영역**에 높은 피크가 존재한다면, 이는 타겟 앞에 다른 객체가 들어온 **폐색 상황**으로 정의한다.

#### (3) Particle Filter (PF) 통합

슬라이딩 윈도우의 비효율성을 해결하기 위해 PF를 도입한다. 타겟의 상태 벡터를 $\mathbf{s} = \{x, y, \dot{x}, \dot{y}, w, h, \alpha\}$로 정의하고, 다음과 같은 과정으로 위치를 추정한다.

- **예측(Prediction)**: 등속도 모델을 가정하여 파티클을 전파시킨다.
- **가중치 업데이트**: 컬러 히스토그램과 Bhattacharyya 거리 $\text{dist}(p, q) = \sqrt{\sum \sqrt{p_i q_i}}$를 사용하여 타겟과의 유사도를 계산하고 가중치를 부여한다.
- **추정**: 가중 평균을 통해 타겟의 최적 위치 $\mathbf{s}_{est} = \sum w_i \mathbf{s}_i / \sum w_i$를 산출한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: Princeton RGB-D 데이터셋 및 Microsoft Kinect V2를 통해 직접 수집한 실시간 데이터셋.
- **비교 대상**: 기본 KCF, Dhog, Struck, VTD 등.
- **측정 지표**: 성공률(Success Rate, SR), 평균 정밀도(Average Precision, AP), 혼동 행렬(Confusion Matrix - TP, FP, FN, TN).

### 2. 주요 결과

- **Princeton 데이터셋**: 폐색(Occlusion) 시나리오에서 제안 방법의 성공률은 $47\%$로, 기본 KCF($35.2\%$) 대비 약 $12\%$ 향상되었다. 특히 대형 타겟($46\%$)에서 높은 성능을 보였다.
- **실시간 데이터셋**:
  - **평균 정밀도(AP)**: 제안한 RGB-D 추적기는 $66.35\%$를 기록하여, KCF($39.99\%$)보다 압도적으로 높은 정밀도를 보였다.
  - **혼동 행렬**: KCF는 타겟의 부재(Absence)를 판단할 수 없어 TN, FN을 계산할 수 없었으나, 제안 방법은 TN($52.7\%$)과 TP($92.5\%$) 모두에서 높은 수치를 기록하며 폐색 감지 능력을 입증하였다.
- **Particle Filter 효과**: PF를 적용했을 때, 폐색 직후 타겟이 재등장하는 시점에서 단순 KCF-RGBD보다 더 빠르게 타겟을 국소화하고 재검출하는 양상을 보였다.

## 🧠 Insights & Discussion

### 강점

본 연구는 단순한 특징 결합을 넘어, **깊이 정보의 동적 특성을 이용해 폐색을 명시적으로 감지**하고 모델 업데이트를 제어함으로써 모델 드리프트 문제를 효과적으로 억제하였다. 또한, PF를 통해 계산 효율성과 재검출 정확도를 동시에 개선하려는 시도가 돋보인다.

### 한계 및 비판적 해석

- **스케일 불변성 부족**: 본 모델은 스케일 변화에 대응하는 메커니즘이 부족하여, 타겟의 크기가 급격히 변하는 상황에서는 성능이 저하된다.
- **빠른 움직임에 취약**: 매우 빠른 속도의 타겟은 모션 블러(Motion Blur)를 유발하며, 이는 깊이 값의 급격한 변화와 특징 왜곡으로 이어져 PF와 KCF 모두 실패하는 원인이 된다.
- **PF의 초기화 의존성**: PF가 효과적으로 작동하려면 폐색 전 타겟의 속도와 위치 정보가 충분히 학습되어야 하며, 너무 빠르게 폐색이 일어날 경우 파티클 분포가 최적화되지 않아 성능이 떨어진다.

## 📌 TL;DR

본 논문은 KCF 추적기의 폐색 취약성을 해결하기 위해 **RGB-D 데이터를 결합하고 Particle Filter를 통합한 장기 추적기**를 제안한다. 특히 **Self-Adjusting Depth Patch**를 통해 폐색을 정확히 감지하여 모델 오염을 막고, 타겟 재등장 시 효율적으로 재검출하는 구조를 설계하였다. 실험 결과, 일반 KCF 대비 폐색 상황에서의 성공률과 정밀도가 크게 향상되었으며, 이는 실시간 실내 로봇의 객체 추적 시스템에 유용하게 적용될 수 있을 것으로 평가된다.
