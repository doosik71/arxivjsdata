# Deep Tracking: Seeing Beyond Seeing Using Recurrent Neural Networks

Peter Ondruska and Ingmar Posner (2016)

## 🧩 Problem to Solve

본 논문은 로봇이 복잡하고 동적인 환경에서 주변 객체를 추적할 때 발생하는 **부분 관측성(Partial Observability)** 문제를 해결하고자 한다. 일반적으로 로봇의 센서는 주변 환경의 일부만을 관측할 수 있으며, 객체가 다른 물체에 가려지는 **폐색(Occlusion)** 현상이 빈번하게 발생한다.

기존의 객체 추적 방식은 객체 탐지(Detection)와 추적(Tracking) 단계를 분리하여 처리하는 파이프라인을 사용하며, 이는 상당한 수준의 수작업 기반 특성 공학(Feature Engineering)을 필요로 한다. 또한, 고전적인 추적 방법론은 시스템의 물리적 특성을 정의하는 Plant 모델과 센서의 특성을 정의하는 Observation 모델을 명시적으로 설계해야 하며, 데이터 연관(Data Association) 문제에 크게 의존한다는 한계가 있다.

따라서 본 연구의 목표는 센서 모델이나 물리 모델의 사전 정의 없이, **원시 센서 데이터(Raw Sensor Input)로부터 객체의 궤적(Object Tracks)을 직접 매핑하는 엔드-투-엔드(End-to-End) 딥러닝 프레임워크**를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 RNN을 활용하여 센서 데이터로부터 환경의 전체 상태를 추정하는 필터링 시스템을 제안한 것이며, 세부적인 설계 아이디어는 다음과 같다.

1. **엔드-투-엔드 필터링 구조**: RNN을 사용하여 부분적인 관측값으로부터 가려진 영역을 포함한 전체 씬(Full, unoccluded scene)을 추정하는 구조를 제안하였다.
2. **비지도 학습을 위한 Input Dropout**: 정답(Ground-truth) 데이터 없이 원시 데이터만으로 학습이 가능하도록 하는 새로운 형태의 Dropout 기법을 제안하였다. 이는 네트워크가 현재 상태를 단순히 복제하는 것이 아니라, 미래의 상태를 예측하도록 강제함으로써 객체의 동역학(Dynamics)을 스스로 학습하게 만든다.
3. **실시간 성능 검증**: 합성 데이터셋을 통해 다수의 동적 객체가 존재하는 환경에서 폐색과 노이즈가 있음에도 불구하고 실시간으로 객체 위치를 예측할 수 있음을 입증하였다.

## 📎 Related Works

기존의 객체 추적은 주로 **베이지안 필터링(Bayesian Filtering)** 기반의 접근 방식을 사용하였다. 
- **Kalman Filter**: 상태 분포를 다변량 정규분포로 가정하여 예측 및 업데이트 방정식을 계산하지만, 표현력이 제한적이다.
- **Particle Filter**: 분포에 대한 가정을 없애고 샘플링 기반으로 접근하여 표현력을 높였으나, 다수 객체 추적 시 데이터 연관 문제가 매우 중요하다.

본 논문은 이러한 수작업 설계 파이프라인과 달리, 신경망이 스스로 적절한 **Belief State** 표현과 업데이트 연산을 학습하도록 한다. 또한, 과거의 Temporal Restricted Boltzmann Machines (TRBM)와 같은 생성 모델이 샘플링으로 인해 계산 비용이 높았던 반면, 본 연구는 **Feed-forward RNN**을 사용하여 추론과 학습의 계산 효율성을 높이고 정확한 그래디언트 계산을 가능하게 하였다.

## 🛠️ Methodology

### 1. 확률 모델 (Probabilistic Model)
본 연구는 센싱 과정을 다음과 같은 생성 모델(Generative Model)로 가정한다. 여기서 $h_t$는 세계의 완전한 상태를 캡처하는 숨겨진 마르코프 과정(Hidden Markov Process)이며, $y_t$는 관측 가능한 전체 씬, $x_t$는 실제 센서 측정값이다.

$$P(y_{1:N}, x_{1:N}, h_{1:N}) = \prod_{t=1}^{N} P(x_t|y_t)P(y_t|h_t)P(h_t|h_{t-1})$$

이 모델에서 목표는 $P(y_t | x_{1:t})$를 추정하는 것이며, 이는 재귀적 베이지안 추정을 통해 Belief $Bel(h_t) = P(h_t | x_{1:t})$를 업데이트함으로써 달성된다.

### 2. RNN을 이용한 필터링 구현
위의 복잡한 확률 분포를 직접 계산하는 대신, 본 논문은 이를 신경망으로 근사한다. Belief 상태 $B_t$를 벡터 $\mathbb{R}^M$으로 정의하고, 다음과 같은 두 함수를 학습한다.

- **Belief Update 함수 ($F$):** 이전 상태와 현재 관측값을 통해 새로운 상태를 업데이트한다.
  $$B_t = F(B_{t-1}, x_t)$$
- **Prediction 함수 ($P$):** 현재의 Belief 상태를 해석 가능한 씬 상태로 매핑한다.
  $$P(y_t | B_t)$$

### 3. 네트워크 아키텍처
제안된 네트워크는 총 4개의 층으로 구성된 CNN 기반 RNN 구조이다.
- **Encoder**: 원시 센서 데이터를 분석하여 객체를 탐지하고 $50 \times 50 \times 8$ 크기의 임베딩으로 변환한다.
- **Belief Tracker**: Encoder의 출력과 이전 Belief $B_{t-1}$을 결합하여 새로운 Belief $B_t$ ($50 \times 50 \times 16$)를 생성한다.
- **Decoder**: $B_t$를 바탕으로 최종적인 점유 그리드(Occupancy Grid) $p_t$를 출력한다. 출력값은 베르누이 분포로 해석되며, 손실 함수로는 **Binary Cross-Entropy**를 사용한다.

### 4. 비지도 학습 (Unsupervised Training)
정답 데이터 $y_t$가 없는 경우, 네트워크가 단순히 $x_t$를 $y_t$로 복사하는 trivial한 해에 빠지는 것을 방지하기 위해 **Input Dropout** 기법을 사용한다.

- **핵심 아이디어**: 현재 상태 $P(y_t | x_{1:t})$가 아니라 미래의 상태 $P(y_{t+n} | x_{1:t})$를 예측하도록 학습시킨다.
- **절차**: $t$부터 $t+n$ 시점까지의 모든 입력 $x$를 0으로 설정(Dropout)하여 관측 불가능한 상태로 만든다. 이렇게 하면 네트워크는 미래의 관측값을 정확히 예측하기 위해, 객체가 가려진 동안의 동역학(Dynamics)을 내부의 $F$ 함수(Belief Tracker)를 통해 시뮬레이션하는 방법을 강제로 학습하게 된다.

## 📊 Results

### 실험 설정
- **데이터셋**: 로봇 주변에 2~12개의 원형 객체가 무작위 방향과 속도로 이동하는 합성 2D 레이저 스캐너 데이터.
- **입력 데이터**: $50 \times 50$ 픽셀의 2채널 바이너리 이미지 (가시성 및 장애물 존재 여부).
- **지표 및 환경**: 표준 노트북에서 1회 추론에 10ms 소요되어 실시간 처리가 가능함을 확인하였다.

### 주요 결과
- **학습 진행**: 초기에는 무작위 출력을 보이다가, 점차 가시적인 객체의 형태를 완성하고, 최종적으로는 완전한 폐색 상태에서도 객체의 위치를 추적하는 능력을 습득하였다.
- **강건성**: 객체의 모양을 원형에서 사각형으로 변경하거나, 센서 데이터에 1%의 노이즈를 추가한 환경에서도 안정적으로 동작하였다.
- **내부 분석**: Encoder는 가시적 객체의 중심에 스파이크(Spike)를 생성하도록 학습되었으며, Belief $B_t$는 서로 다른 이동 패턴을 가진 객체에 대해 서로 다른 활성화 패턴을 할당하여 추적하는 것이 확인되었다.

## 🧠 Insights & Discussion

본 연구의 가장 큰 강점은 **모델 설계에 대한 도메인 지식(물리 모델, 센서 모델 등) 없이도 데이터로부터 동역학을 스스로 학습**했다는 점이다. 특히 비지도 학습 방식의 Input Dropout은 정답 레이블이 없는 실제 환경에서도 적용 가능성을 높여주는 매우 효율적인 장치이다.

다만, 실험이 합성 데이터셋으로만 진행되었다는 점은 한계로 지적될 수 있다. 실제 환경에서는 객체의 이동 경로가 단순한 등속 직선 운동이 아니며, 센서 노이즈의 특성 또한 훨씬 복잡할 것이다. 또한, 본 논문에서 사용한 단순 RNN 구조보다 LSTM과 같은 더 고도화된 유닛을 사용한다면 더 긴 시간의 폐색 상황에서도 안정적인 추적이 가능할 것으로 판단된다.

비판적으로 해석하자면, 본 모델은 씬을 $50 \times 50$ 그리드로 표현하므로 해상도가 낮아 정밀한 위치 추정에는 한계가 있을 수 있으며, 객체의 수가 급격히 증가할 경우 Belief 상태의 차원이 이를 충분히 표현할 수 있을지에 대한 분석이 부족하다.

## 📌 TL;DR

본 논문은 RNN을 사용하여 원시 센서 데이터로부터 가려진 객체까지 추적하는 엔드-투-엔드 딥러닝 프레임워크를 제안하였다. 특히 미래 상태를 예측하게 만드는 **Input Dropout** 기법을 통해 정답 데이터 없이도 객체의 동역학을 학습하는 비지도 학습 방법을 제시하였다. 이 연구는 로봇이 정교한 물리 모델 없이도 복잡한 동적 환경에서 주변 상황을 인식(Situational Awareness)할 수 있는 새로운 패러다임을 제시하며, 향후 자율주행이나 보행자 추적 등 실시간 센싱 분야에 응용될 가능성이 높다.
