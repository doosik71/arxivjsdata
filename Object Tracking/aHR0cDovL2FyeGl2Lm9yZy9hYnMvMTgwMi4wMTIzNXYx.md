# Tracking Multiple Moving Objects Using Unscented Kalman Filtering Techniques

Xi Chen, Xiao Wang and Jianhua Xuan (N/A)

## 🧩 Problem to Solve

본 논문은 비디오 감시 및 모니터링 시스템에서 여러 개의 움직이는 객체를 안정적으로 탐지하고 추적하는 문제를 해결하고자 한다. 특히, 실제 환경에서는 카메라 노이즈, 조명 조건의 변화, 그리고 객체 간의 가려짐(occlusion) 현상이 빈번하게 발생하며, 객체의 움직임이 항상 선형적이지 않은 비선형 모션(nonlinear motion) 시나리오가 나타난다.

기존의 많은 추적 방법들은 이러한 비선형 움직임과 가려짐 현상이 동시에 발생할 때 객체를 지속적으로 추적하는 데 실패하는 경향이 있다. 따라서 본 연구의 목표는 가려짐이 발생하는 비선형 모션 환경에서도 다중 객체를 정확하게 탐지하고 추적할 수 있는 효과적인 알고리즘을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 단순하고 효율적인 블록 매칭(block matching) 기반의 탐지와 비선형 추적에 강점이 있는 Unscented Kalman Filter (UKF)를 결합하는 것이다. 

가장 중심적인 설계 직관은 객체가 서로 겹치거나 가려지더라도 각 객체는 서로 다른 방향과 속도로 움직이며, 이 속도 정보는 짧은 프레임 구간 동안 어느 정도 연속성을 유지한다는 점이다. UKF를 통해 추정된 각 객체의 속도 정보를 탐지 알고리즘에 피드백함으로써, 가려짐 이후 객체들이 다시 분리될 때 어떤 탐지 결과가 어떤 객체에 해당하는지를 정확하게 매칭(data association)하여 모호성을 해결한다.

## 📎 Related Works

논문에서는 움직이는 객체 탐지를 위해 배경 제거(background subtraction)와 광학 흐름(optical flow) 기술을 언급한다. 특히 블록 매칭 기술은 복잡한 광학 흐름 방식에 비해 계산 속도가 빨라 실시간 응용 분야에 유리하며, 추적을 위한 대략적인 위치 정보를 제공하는 데 적합하다고 설명한다.

추적 알고리즘 측면에서는 Kalman Filter (KF)가 널리 사용되고 있음을 밝힌다. 그러나 KF는 상태 모델과 측정 모델이 선형적이라는 가정 하에 최적의 추정치를 제공하므로, 객체가 갑작스럽게 방향을 트는 비선형 움직임을 보일 경우 추적 성능이 크게 저하되는 한계가 있다. 본 논문은 이러한 KF의 선형성 제약을 극복하기 위해 Unscented Transform (UT)을 사용하는 UKF를 대안으로 제시하며, 이를 통해 선형 및 비선형 모션 모두를 효과적으로 처리하고자 한다.

## 🛠️ Methodology

### 1. 움직이는 객체 탐지 (Moving Object Detection)
먼저 Three-Step Search (TSS) 방식을 사용하여 움직이는 객체를 탐지한다. TSS는 탐색 영역을 효율적으로 좁혀 계산량을 줄이는 빠른 검색 기법이다. 두 블록 간의 유사도를 측정하기 위한 기준으로는 SAD (Sum of Absolute Difference)를 사용하며, 식은 다음과 같다.

$$M(p, q) = \sum_{i=1}^{n} \sum_{j=1}^{n} |I_c(i, j) - I_r(i+p, j+q)|$$

여기서 $I_c$는 현재 프레임 블록, $I_r$은 참조 프레임 블록이며, $(p, q)$는 변위 벡터이다. SAD 값이 최소가 되는 블록을 최적의 매칭 블록으로 선정하여 모션 벡터를 산출한다.

### 2. Unscented Kalman Filtering (UKF) 추적
UKF는 비선형 시스템의 상태 추정을 위해 Unscented Transform (UT)을 도입한다. UT는 평균과 공분산을 가진 가우시안 분포를 대표하는 $2N+1$개의 시그마 샘플(sigma samples) $\mathcal{X}_i$를 생성하여 비선형 함수에 통과시킨 뒤, 그 결과의 가중 평균과 공분산을 통해 후험 확률 분포를 계산한다.

#### 시스템 방정식 설계
본 논문은 2D 이미지 상의 움직임을 $x$축과 $y$축의 독립적인 성분으로 처리한다. $M$개의 객체가 있을 때, 상태 변수 $X$는 다음과 같이 위치, 속도, 가속도를 모두 포함하는 벡터로 정의된다.

$$X = [x_1, v_{x1}, a_{x1}, y_1, v_{y1}, a_{y1}, \dots, x_M, v_{xM}, a_{xM}, y_M, v_{yM}, a_{yM}]^T$$

각 객체의 비선형 운동 모델은 다음과 같이 정의된다.

$$\begin{cases} x_i(n+1) = x_i(n) + v_{xi}(n)dt + \frac{1}{2}a_{xi}(n)dt^2 \\ y_i(n+1) = y_i(n) + v_{yi}(n)dt + \frac{1}{2}a_{yi}(n)dt^2 \end{cases}$$

이 모델을 바탕으로 상태 전이 행렬 $F$와 측정 행렬 $H$를 구성하여 UKF의 예측 및 업데이트 과정을 수행한다.

### 3. 전체 파이프라인 및 추론 절차
전체 시스템은 다음과 같은 루프로 동작한다.
1. **탐지 단계**: 블록 매칭(TSS)을 통해 각 객체의 중심 위치를 측정값으로 얻는다.
2. **추적(예측) 단계**: UKF의 예측 모델을 사용하여 다음 프레임의 예상 위치와 속도를 계산한다.
3. **피드백 단계**: UKF에서 추정된 각 객체의 속도 정보를 탐지 알고리즘으로 전달한다. 가려짐이 발생했을 때, 이 속도 정보를 이용해 객체의 정체성을 유지하며 모호성을 해결한다.
4. **추적(업데이트) 단계**: 실제 탐지된 위치 측정값을 이용해 UKF의 상태 벡터를 업데이트하고 다시 2단계로 돌아간다.

## 📊 Results

### 1. 시뮬레이션 연구 (Simulation Study)
비선형 회전 운동 시나리오에서 UKF와 KF의 추적 오차(MSE)를 비교하였다.
- **결과**: 노이즈 수준 $\sigma=1$일 때, UKF의 오차는 $0.79$로 KF의 $1.51$보다 현저히 낮았다. 
- **분석**: 노이즈가 낮을수록 비선형 모션이 KF의 성능에 더 치명적인 영향을 미치며, UKF가 비선형 움직임을 훨씬 더 잘 처리함을 확인하였다. 노이즈가 매우 높아지면($\sigma=10$) 두 필터의 성능 차이는 줄어들지만, 여전히 UKF가 우위에 있다.

### 2. 실 데이터 탐지 및 추적 실험
실제 이미지 시퀀스를 통해 제안 기법을 검증하였다.
- **객체 탐지**: TSS 기반 블록 매칭이 움직이는 객체를 효과적으로 찾아냈으나, 나뭇잎의 흔들림 같은 배경 노이즈가 일부 탐지되는 한계가 있었다. 이는 연속적인 움직임이 없는 영역을 삭제하는 방식으로 보완하였다.
- **다중 객체 추적 (가려짐 해결)**: 
    - 두 명의 보행자가 겹치는 상황에서, 단순 탐지만으로는 가려짐 이후 객체를 분리할 수 없으나, UKF가 추정한 속도 패턴(Velocity Pattern)을 이용해 각 객체를 정확하게 매칭하여 추적을 유지하였다.
    - 세 명의 보행자가 복잡하게 얽히고 배경이 복잡한 환경에서도 UKF의 속도 피드백을 통해 가려짐 전후의 객체 경로를 정확하게 복원하는 성과를 보였다.

## 🧠 Insights & Discussion

본 논문의 가장 큰 강점은 단순히 정교한 필터를 사용한 것에 그치지 않고, **추적 단계에서 얻은 속도 정보를 탐지 단계로 피드백**하여 '데이터 연관(Data Association)' 문제를 해결했다는 점이다. 이는 다중 객체 추적의 고질적인 문제인 가려짐(occlusion) 상황에서 객체의 ID를 유지하는 실질적인 메커니즘을 제공한다.

다만, 다음과 같은 한계점과 논의 사항이 존재한다.
1. **탐지 의존성**: UKF가 강력하더라도 기초가 되는 블록 매칭 탐지 단계에서 배경 노이즈(예: 흔들리는 나뭇잎)를 완벽히 제거하지 못하면 잘못된 측정값이 필터에 입력될 수 있다.
2. **가정의 단순함**: 객체의 속도가 프레임 간에 어느 정도 연속적이라는 가정을 기반으로 한다. 만약 객체가 가려진 상태에서 급격하게 속도나 방향을 바꾼다면, 예측된 위치와 실제 나타난 위치의 괴리가 커져 매칭에 실패할 가능성이 있다.
3. **계산 복잡도**: UKF는 시그마 샘플을 생성하고 전파시켜야 하므로 일반 KF보다 계산량이 많다. 실시간 시스템 적용 시 이 오버헤드가 어느 정도인지에 대한 상세 분석이 부족하다.

## 📌 TL;DR

이 논문은 비선형 움직임과 가려짐이 발생하는 환경에서 다중 객체를 추적하기 위해 **TSS 기반 블록 매칭 탐지**와 **Unscented Kalman Filter (UKF)**를 결합한 시스템을 제안한다. 특히 UKF가 추정한 객체별 속도 정보를 탐지 단계에 피드백함으로써, 객체가 서로 겹쳤다 분리될 때 발생하는 정체성 모호성 문제를 효과적으로 해결한다. 이 연구는 비디오 감시 시스템에서 복잡한 모션 패턴을 가진 다수 객체를 안정적으로 추적하는 데 중요한 방법론을 제시한다.
