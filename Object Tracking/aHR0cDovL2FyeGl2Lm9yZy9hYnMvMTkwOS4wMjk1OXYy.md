# Discriminative and Robust Online Learning for Siamese Visual Tracking

Jinghao Zhou, Peng Wang, Haoyang Sun (2020)

## 🧩 Problem to Solve

본 논문은 시각적 객체 추적(Visual Object Tracking) 분야에서 기존의 두 가지 주요 패러다임이 가진 고유한 한계점을 해결하고자 한다.

첫째, 온라인 학습 기반의 접근 방식(Online-only approaches)은 학습된 모델의 일반화 능력이 부족하여 타겟 회귀(Target regression) 성능이 떨어진다는 문제가 있다. 둘째, 오프라인 학습 기반의 시아미즈 네트워크(Siamese networks)는 타겟 특유의 컨텍스트 정보가 부족하여 주변의 유사한 방해 요소(Distractors)를 구별하는 판별력(Discriminative capability)이 낮고, 타겟의 변형(Deformation)에 취약하여 강건성(Robustness)이 떨어진다는 한계가 있다.

따라서 본 논문의 목표는 오프라인으로 학습된 시아미즈 네트워크에 타겟 특화 특징을 추출할 수 있는 온라인 모듈을 통합함으로써, 방해 요소에 강건한 판별력과 객체 변형에 대응하는 강건성을 동시에 확보하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 시아미즈 네트워크와 상호 독립적이면서도 체계적으로 통합된 **독립형 온라인 모듈(Standalone Online Module)**을 추가하는 것이다. 주요 기여 사항은 다음과 같다.

1. **어텐션 메커니즘 기반 온라인 모듈 제안**: 타겟 특화 특징을 추출하기 위해 압축 모듈(Compression module)과 듀얼 어텐션 모듈(Attention module)로 구성된 분류 서브넷을 제안하였다.
2. **판별적 학습(Discriminative Learning)**: 온라인 모듈에서 생성된 응답 맵을 시아미즈 네트워크의 분류 점수와 융합(Score fusion)하여 방해 요소와 배경 소음을 효과적으로 처리한다.
3. **강건한 학습(Robust Learning)**: 온라인 분류기의 점수를 지표로 삼아 고품질의 프레임을 선택하고, 이를 통해 시아미즈 네트워크의 템플릿을 업데이트함으로써 타겟의 변형, 회전, 조명 변화에 대응한다.
4. **범용적 통합 가능성**: 제안된 온라인 모듈은 기존의 시아미즈 추적기(SiamFC, SiamRPN++, SiamMask 등)를 재학습시키지 않고도 그대로 통합하여 성능을 향상시킬 수 있다.

## 📎 Related Works

논문에서는 시각적 객체 추적의 흐름을 크게 시아미즈 네트워크 기반 방식과 온라인 학습 기반 방식으로 구분하여 설명한다.

- **Siamese Visual Tracking**: 오프라인 학습을 통해 타겟의 묘사를 풍부하게 하여 성능과 속도를 모두 잡았으나, 배경 정보를 무시하기 때문에 유사 객체(Distractor)가 나타나거나 타겟의 외형이 크게 변할 때 성능이 급격히 저하되는 경직성(Rigidity)을 보인다.
- **Online Learning Approach**: KCF와 같은 상관 필터(Correlation filter) 기반 추적기나 MDNet 같은 분류기 기반 추적기는 타겟과 배경을 구별하는 온라인 모델을 학습하여 판별력이 뛰어나지만, 타겟 회귀의 정확도가 상대적으로 낮다.
- **Target Regression and Classification**: 최근 ATOM과 같은 연구는 온라인 판별 분류기와 고효율 최적화 전략을 IoUNet과 결합하여 회귀와 분류 문제를 동시에 해결하려 시도하였다.

본 논문은 시아미즈 네트워크의 강력한 회귀 성능(Generative ability)을 유지하면서, 병렬 구조의 온라인 분류기를 통해 판별력과 강건성을 보완한다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

시스템은 크게 두 개의 서브넷으로 구성된다.

- **Siamese Matching Subnet ($M$):** 오프라인으로 학습되며, 바운딩 박스 회귀(Bounding box regression)를 주로 담당한다.
- **Online Classification Subnet ($C$):** 온라인으로 학습되며, 타겟과 배경을 구별하는 분류 점수(Classification score)를 생성한다.

### 2. 온라인 모듈의 구성 및 특징 추출

온라인 모듈은 입력 특징 맵을 처리하기 위해 세 단계의 모듈을 거친다.

- **Compression Module**: 백본 네트워크의 특징 맵을 분류에 적합하도록 차원을 축소한다.
- **Attention Module**: 데이터 불균형 문제를 해결하기 위해 공간 어텐션($\phi_s$)과 채널 어텐션($\phi_c$)을 적용하여 타겟 특화 특징을 추출한다.
- **Filter Module**: 최종적으로 분류 점수를 출력하며, 추적 과정에서 지속적으로 업데이트된다.

### 3. 판별적 학습 및 필터 업데이트

온라인 분류기는 $L_2$ 손실 함수를 사용하여 최적화된다.
$$r^C(f, y_1) = \|f^C - y_1\|^2$$
여기서 $y_1$은 타겟 바운딩 박스를 중심으로 하는 가우시안 분포이다. 학습 효율을 높이기 위해 일반적인 SGD 대신 2차 문제 해결에 적합한 **공액 기울기 하강법(Conjugate Gradient Descent)**을 사용하여 최적화 속도를 높였다.

최종 분류 점수 $\dot{f}^C$는 다음과 같이 온라인 점수와 시아미즈 점수의 가중 합으로 계산된다.
$$\dot{f}^C(x; w) = \lambda f^C(x; w) + (1 - \lambda) f_{cls}^M(x, z; w)$$
여기서 $\lambda$는 온라인 신뢰도 점수의 영향력을 조절하는 가중치이다.

### 4. 강건한 학습 및 템플릿 업데이트

타겟의 외형 변화에 대응하기 위해 단기 메모리 템플릿 $z_s$를 도입한다. 템플릿 업데이트를 위한 고품질 프레임 선택 기준은 온라인 분류기의 최대 점수 $\hat{f}^C$를 사용하며, 임계값 $\tau_c$를 넘는 경우에만 업데이트를 수행한다.
$$z_s = \text{warp}(\arg \max_x \hat{f}^C(x; w) * 1, \hat{f}^C > \tau_c)$$

업데이트된 단기 템플릿 $z_s$를 실제 추적에 사용할지 여부는 첫 프레임 템플릿 $z_1$과의 IoU 및 분류 점수 차이를 비교하여 결정함으로써, 템플릿 드리프트(Template drift) 현상을 방지한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: OTB100, VOT2018, VOT2018-LT, UAV123, TrackingNet, LaSOT 등 6개 벤치마크 사용.
- **기준선(Baselines)**: SiamFC, SiamRPN++, SiamMask를 기반으로 DROL-FC, DROL-RPN, DROL-Mask를 구현하였다.
- **지표**: AUC(Area Under Curve), Precision(Pr), EAO(Expected Average Overlap) 등을 사용하였다.

### 2. 정량적 결과

- **OTB2015**: DROL-RPN은 AUC와 Precision 모두에서 최상위 성적을 기록하였으며, SiamRPN++ 대비 AUC는 1.9%, Precision은 2.2% 향상되었다.
- **VOT2018**: EAO 지표에서 ATOM과 SiamRPN++ 대비 각각 8.0%, 6.6%의 절대적 성능 향상을 보였다.
- **장기 추적(VOT2018-LT)**: 타겟이 시야에서 사라지거나 장기간 가려지는 상황에서도 SiamRPN++ 대비 Precision, Recall, F-score 모두에서 약 3.8%의 성능 향상을 달성하였다.
- **기타 벤치마크**: UAV123, TrackingNet, LaSOT에서도 기존 SOTA 모델들을 능가하거나 대등한 성능을 보였으며, 특히 DROL-RPN(ResNet50) 모델은 실시간 이상의 속도로 동작하면서도 높은 정확도를 유지하였다.

### 3. 절제 연구(Ablation Study)

- **점수 융합 ($\lambda$)**: $\lambda$ 값이 0(오직 시아미즈)이거나 1(오직 온라인)일 때보다 $0.8$ 부근에서 최적의 성능이 나타났다. 이는 온라인 점수가 대략적인 위치를 잡고, 시아미즈 점수가 세밀한 회귀를 돕는 상호보완적 관계임을 시사한다.
- **어텐션 모듈**: 어텐션 모듈을 제거했을 때보다 포함했을 때 방해 요소와 폐쇄(Occlusion) 상황에서 더 명확한 피크(Peak) 점수를 생성함을 확인하였다.
- **업데이트 간격 ($T$)**: 매 프레임 업데이트($T=1$) 하는 것보다 적절한 간격($T=5$)을 두고 신뢰도가 높은 템플릿을 선택하는 것이 더 안정적인 추적 성능을 보였다.

## 🧠 Insights & Discussion

### 강점

본 논문의 가장 큰 강점은 **플러그 앤 플레이(Plug-and-play)** 방식의 설계이다. 기존의 강력한 시아미즈 네트워크를 전혀 수정하거나 재학습시키지 않고도, 독립적인 온라인 모듈을 추가하는 것만으로 판별력과 강건성을 동시에 개선하였다. 특히 듀얼 어텐션 메커니즘을 통해 온라인 학습 시 발생하는 데이터 불균형 문제를 효과적으로 해결한 점이 돋보인다.

### 한계 및 가정

본 방법론은 첫 프레임에서 타겟의 정확한 위치가 주어진다는 가정하에 온라인 모듈을 초기화한다. 만약 초기 바운딩 박스에 심한 노이즈가 섞여 있다면, 초기 $L_2$ 학습 단계에서 잘못된 특징이 학습될 위험이 있다. 또한, 온라인 업데이트를 위해 공액 기울기 하강법을 사용하고 템플릿을 관리하는 추가 연산이 발생하므로, 순수 시아미즈 네트워크보다는 연산 오버헤드가 존재한다.

### 비판적 해석

논문에서는 실시간 동작이 가능하다고 언급하였으나, 정확한 FPS 수치는 백본 네트워크와 모듈 구성에 따라 편차가 크다. 특히 ResNet50 기반의 DROL-RPN은 약 30 FPS로 동작하는데, 이는 단순한 SiamRPN++보다는 느리다. 따라서 성능 향상분과 연산 속도 저하 사이의 트레이드-오프(Trade-off)에 대한 더 상세한 분석이 필요해 보인다.

## 📌 TL;DR

본 논문은 오프라인 시아미즈 네트워크의 고질적인 문제인 **판별력 부족(방해 요소 취약)**과 **강건성 부족(외형 변형 취약)**을 해결하기 위해, 어텐션 메커니즘 기반의 **독립형 온라인 분류 모듈**을 제안하였다. 온라인 학습을 통한 점수 융합으로 판별력을 높이고, 신뢰도 기반의 템플릿 업데이트로 강건성을 확보하여 6개 주요 벤치마크에서 SOTA 성능을 달성하였다. 이 연구는 기존 시아미즈 추적기의 구조를 유지하면서도 온라인 적응 능력을 부여할 수 있는 효율적인 프레임워크를 제시했다는 점에서 향후 실시간 고정밀 추적 연구에 중요한 기여를 할 것으로 보인다.
