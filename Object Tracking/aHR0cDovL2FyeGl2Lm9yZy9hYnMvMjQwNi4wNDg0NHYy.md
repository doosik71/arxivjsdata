# Multi-Granularity Language-Guided Training for Multi-Object Tracking

Yuhao Li, Jiale Cao, Muzammal Naseer, Yu Zhu, Jinqiu Sun, Yanning Zhang, and Fahad Shahbaz Khan (2024/2025)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 Multi-Object Tracking (MOT)에서 시각적 정보(visual features)에만 의존하여 객체를 식별하고 추적할 때 발생하는 한계이다. 기존의 MOT 방법론들은 주로 서로 다른 인스턴스 간의 비유사성을 최대화하고 동일 인스턴스 간의 유사성을 최소화하는 방식으로 특징 학습을 수행한다.

그러나 이러한 시각 기반 학습 방식은 다음과 같은 상황에서 성능이 크게 저하되는 취약성을 가진다.
- **환경적 간섭:** 가림(occlusion), 흐림(blur) 등의 현상이 발생할 경우 시각적 특징만으로는 객체를 일관되게 식별하기 어렵다.
- **도메인 변이(Domain Variance):** 야외 환경(outdoor)에서 학습된 모델을 실내 환경(indoor)과 같은 다른 도메인에 적용할 때 일반화 능력이 떨어진다.

따라서 본 논문의 목표는 다중 모달리티의 언어 정보(language information)를 학습 과정에 도입하여, 환경 변화에 강건하고 변별력 있는 시각적 표현(discriminative representations)을 학습함으로써 MOT의 추적 성능과 도메인 일반화 능력을 향상시키는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **다양한 수준의 입도(Multi-granularity)**를 가진 언어 가이드를 통해 시각적 특징 학습을 유도하는 것이다. 구체적인 기여 사항은 다음과 같다.

1. **LG-MOT 프레임워크 제안:** 인스턴스 수준(instance-level)과 장면 수준(scene-level)의 언어 정보를 명시적으로 활용하여 시각적 특징 학습을 가이드하는 새로운 MOT 프레임워크를 제안한다.
2. **다중 입도 언어 데이터셋 구축:** 기존 MOT 데이터셋(MOT17, DanceTrack, SportsMOT)에 장면 및 인스턴스 수준의 언어 묘사를 추가하여 MOT17-L, DanceTrack-L, SportsMOT-L 데이터셋을 구축하였다.
3. **학습 시에만 언어 정보 활용 (Training-only Guidance):** CLIP의 텍스트 인코더를 통해 얻은 언어 임베딩을 시각적 특징과 정렬(align)시키되, 추론(inference) 단계에서는 언어 정보 없이 표준 시각적 특징만을 사용한다. 이를 통해 실용성을 유지하면서 학습 효율을 높였다.

## 📎 Related Works

기존의 MOT 방법론은 크게 Detection-based MOT, Joint detection and tracking, Prediction-based MOT로 분류된다. 최근에는 Transformer 기반의 예측 방식이 우수한 성능을 보이고 있으나, 여전히 소형 객체나 밀집된 장면에서의 성능 저하 문제가 존재한다.

최근 일부 연구들이 언어 힌트를 활용하여 MOT를 개선하려는 시도를 하였으나, 다음과 같은 한계가 있다.
- **추론 시 언어 의존성:** 많은 기존 언어 기반 MOT 방법론들이 학습뿐만 아니라 추론 단계에서도 언어 정보가 필요하다는 제약이 있다.
- **단일 입도 정보:** 주로 인스턴스 수준의 텍스트 정보만을 활용하며, 전체 장면의 맥락을 반영하는 장면 수준의 가이드는 부족했다.

LG-MOT는 사전 학습된 시각-언어 모델(VLM)의 텍스트 인코더로부터 지식을 증류(distillation)하는 방식을 통해, 추론 시에는 추가 정보 없이도 강력한 시각적 특징을 사용할 수 있도록 설계되었다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 파이프라인
LG-MOT는 SUSHI(SUsper-HIerarchy)라는 그래프 기반 MOT 프레임워크를 베이스라인으로 채택한다. SUSHI는 객체 탐지 결과를 노드로, 객체 간의 연관성 가설을 엣지로 구성하여 GNN(Graph Neural Network) 메시지 패싱을 통해 엣지를 분류(동일 객체 여부 판단)하는 구조이다. LG-MOT는 이 과정에 언어 가이드를 추가한다.

### 언어 데이터 생성 (Annotation Pipeline)
- **인스턴스 수준:** 객체 크롭 이미지를 VLM에 입력하여 태그, 속성, 캡션을 생성하고, LLM을 통해 "성별은 무엇인가?", "셔츠 색상은 무엇인가?", "바지 색상은 무엇인가?"와 같은 질문에 답하는 형태로 정교한 묘사를 생성한다.
- **장면 수준:** 카메라 상태(정적/동적), 시점(viewpoint), 날씨 조건 등을 수동 또는 기존 메타데이터를 통해 라벨링한다.

### 핵심 구성 요소 및 학습 절차

#### 1. 인스턴스 수준 시맨틱 가이드 (Instance-level Semantic Guidance, ISG)
노드 임베딩(시각적 특징)이 해당 객체의 언어적 묘사와 일치하도록 유도한다. CLIP 텍스트 인코더를 통해 인스턴스 묘사 $T^{inst}$를 임베딩 $\phi_{i}$로 변환하고, 노드 임베딩 $\phi(b_{i}^{k})$와의 분포 일치(distribution matching)를 통해 학습한다. 손실 함수는 다음과 같다.

$$L_{ISG} = \frac{1}{V} \sum_{i=1}^{V} \sigma(\phi_{i}) \log \frac{\sigma(\phi_{i})}{\sigma(\phi_{i})}$$

여기서 $V$는 노드 수, $\sigma$는 softmax 연산을 의미한다.

#### 2. 장면 수준 인지 가이드 (Scene-level Perception Guidance, SPG)
장면의 전역적 정보(조명, 시점 등)가 데이터 연관성 판단에 영향을 준다는 점에 착안하여, GNN 메시지 패싱 후의 엣지 임베딩 $\hat{E}(u,v)$를 장면 묘사 임베딩 $\phi_{s}$와 정렬시킨다.

$$L_{SPG} = \frac{1}{E} \sum_{u=1}^{V} \sum_{v=1}^{V} \sigma(\hat{E}(u,v)) \log \frac{\sigma(\hat{E}(u,v))}{\sigma(\phi_{s})}, \quad \hat{E} \subset \{(u,v) \in V \times V | u \neq v\}$$

#### 3. 전체 손실 함수 및 학습
최종 손실 함수는 엣지 분류를 위한 이진 교차 엔트로피 손실($L_{c}$)과 두 가지 언어 가이드 손실의 합으로 정의된다.

$$L = L_{c} + \alpha L_{ISG} + \beta L_{SPG}$$

여기서 $\alpha, \beta$는 하이퍼파라미터이다.

#### 4. 추론 절차
추론 시에는 언어 임베딩이나 텍스트 인코더를 전혀 사용하지 않으며, 학습된 시각적 특징 추출기와 SUSHI의 그래프 기반 연관성 모델만을 사용하여 추적을 수행한다.

## 📊 Results

### 실험 설정
- **데이터셋:** MOT17, DanceTrack, SportsMOT.
- **평가 지표:** IDF1 (추적 연속성/ID 유지), HOTA (탐지 및 연관성 종합), MOTA (탐지 정확도 중심), IDSW (ID 스위치 횟수).
- **비교 대상:** ByteTrack, OC-SORT, StrongSORT++, MOTRv3, SUSHI 등 최신 SOTA 모델.

### 주요 결과
1. **정량적 성능 향상:**
   - **MOT17:** IDF1 81.7%, HOTA 65.6%를 달성하며 SUSHI 및 Deep OC-SORT 대비 우수한 성능을 보였다.
   - **DanceTrack:** IDF1 60.5%, HOTA 61.8%를 기록하였으며, 특히 베이스라인인 SUSHI 대비 IDF1에서 2.2%의 절대적 이득을 얻었다.
   - **SportsMOT:** IDF1 77.1%, HOTA 75.0%로 기존 방법론들을 상회하였다.

2. **도메인 일반화 능력 (Cross-domain):**
   - MOT17(야외)에서 학습하고 DanceTrack(실내)에서 테스트한 결과, 시각적 특징만 사용한 베이스라인(IDF1 42.5%)에 비해 LG-MOT는 **IDF1 53.7%**로 매우 큰 성능 향상을 보였다. 이는 언어 가이드가 도메인 불변적인(domain-invariant) 특징 학습을 도왔음을 시사한다.

3. **Ablation Study:**
   - 장면 수준 가이드와 인스턴스 수준 가이드 모두 성능 향상에 기여하며, 두 가지를 모두 사용했을 때 최적의 성능이 나타났다. 특히 도메인 간 테스트(Cross-domain)에서 언어 가이드의 영향력이 훨씬 극명하게 나타났다.

## 🧠 Insights & Discussion

### 강점
- **도메인 불변성 학습:** 옷의 색상이나 성별과 같은 언어적 묘사는 환경(배경, 조명)이 바뀌어도 변하지 않는 특성이다. 이를 통해 모델이 환경 노이즈에 덜 민감하고 본질적인 객체 특징에 집중하도록 학습되었다.
- **추론 효율성:** 복잡한 언어 모델을 추론 과정에 포함하지 않으면서도, 학습 단계에서의 지식 증류를 통해 성능을 끌어올린 전략이 매우 효율적이다.
- **다중 입도 접근:** 객체 개별의 특징(ISG)뿐만 아니라 장면 전체의 맥락(SPG)을 모두 고려하여 엣지 임베딩을 보정함으로써 연관성 예측의 정확도를 높였다.

### 한계 및 논의사항
- **데이터 구축 비용:** 비록 VLM과 LLM을 활용해 자동화하려 했으나, 초기 데이터셋 구축 및 검증 과정에서 상당한 노력이 필요했을 것으로 보인다.
- **언어 묘사의 정확성:** VLM이 생성한 텍스트 묘사가 부정확할 경우(예: 색상 오인식), 오히려 잘못된 가이드를 제공하여 학습을 방해할 가능성이 있다. 논문에서는 가시성 0.5 이상의 슬라이스만 사용하고 통계적으로 최빈값을 선택하는 방식으로 이를 완화하려 했다.

## 📌 TL;DR

본 논문은 MOT에서 시각적 특징의 취약성(가림, 도메인 변화)을 극복하기 위해 **인스턴스 및 장면 수준의 다중 입도 언어 가이드(LG-MOT)**를 제안한다. 학습 단계에서 CLIP의 텍스트 임베딩을 이용해 시각적 특징을 정렬시키고, 추론 시에는 언어 정보 없이 시각적 특징만 사용하는 전략을 취한다. 실험 결과 MOT17, DanceTrack 등 주요 벤치마크에서 SOTA 성능을 달성했으며, 특히 야외 $\to$ 실내와 같은 도메인 전이 상황에서 탁월한 일반화 능력을 입증하였다. 이 연구는 향후 다중 모달리티 정보를 활용한 강건한 객체 추적 연구에 중요한 방향성을 제시한다.
