# Unsupervised Deep Representation Learning for Real-Time Tracking

Ning Wang, Wengang Zhou, Yibing Song, Chao Ma, Wei Liu, Houqiang Li (2020)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 시각적 객체 추적(Visual Object Tracking) 모델을 학습시키기 위해 필수적으로 요구되는 막대한 양의 레이블링된 데이터(labeled data)에 대한 의존성을 제거하는 것이다. 기존의 딥러닝 기반 추적기들은 대규모의 정밀한 ground-truth 레이블이 포함된 데이터셋을 통해 지도 학습(supervised learning) 방식으로 사전 학습되어 왔다. 그러나 이러한 수동 주석(manual annotation) 과정은 비용이 매우 많이 들고 시간이 오래 걸리며, 추적 대상이 될 수 있는 임의의 객체들에 대해 모든 레이블을 확보하는 것은 사실상 불가능하다.

따라서 본 연구의 목표는 레이블이 없는 비디오 시퀀스만을 활용하여, 임의의 객체를 추적할 수 있는 강건한 특징 표현(feature representation)을 학습하는 비지도 학습(unsupervised learning) 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 **'강건한 추적기라면 전방향 추적(forward tracking)과 후방향 추적(backward tracking)이 모두 일관되어야 한다'**는 양방향 모션 분석(bidirectional motion analysis)의 직관에 기반한다.

핵심 기여 사항은 다음과 같다:

1. **비지도 학습 기반 Siamese Correlation Filter 네트워크**: 정답 레이블 없이 전방향 및 후방향 궤적의 일관성을 측정하여 네트워크를 학습시키는 체계를 제안한다.
2. **다중 프레임 검증(Multi-frame Validation) 기법**: 단순히 두 프레임만 비교할 경우 발생할 수 있는 우연한 일관성(coincidental consistency) 문제를 해결하기 위해, 추적 경로를 확장하여 추적 실패 시 불일치성이 더 명확하게 드러나도록 설계하였다.
3. **비용 민감 손실 함수(Cost-sensitive Loss) 및 엔트로피 기반 샘플 선택**: 학습 과정에서 발생하는 노이즈 샘플의 영향을 줄이기 위해, 이미지 엔트로피(image entropy)를 통해 정보량이 많은 영역을 선택하고, 모션 크기에 따라 가중치를 부여하는 손실 함수를 도입하였다.

## 📎 Related Works

### 기존 연구 및 한계

- **Deep Visual Tracking**: VGG와 같은 사전 학습된 CNN 특징을 사용하거나, SiamFC, SiamRPN과 같이 엔드투엔드(end-to-end) 학습 네트워크를 구축하는 방식이 주를 이루어 왔다. 하지만 이들은 모두 대규모 지도 학습 데이터셋에 의존한다.
- **Forward-Backward Analysis**: 주로 광학 흐름(optical flow) 학습이나 추적 실패 감지(TLD 등)를 위해 사용되었다. 기존의 추적 분야에서는 주로 출력의 신뢰도를 측정하는 용도로 쓰였으며, 이를 통해 네트워크 자체를 학습시키는 접근은 드물었다.
- **Unsupervised Representation Learning**: 비디오 프레임을 셔플링하거나 오토인코더(auto-encoder)를 사용하는 방식이 제안되었으나, 이는 일반적인 표현 학습일 뿐 시각적 추적 작업의 특성에 최적화된 방식은 아니었다.

### 차별점

본 논문은 기존의 비지도 학습이 단순한 특징 추출에 그친 것과 달리, 추적 알고리즘 자체를 비지도 학습 루프에 통합하였다. 특히 Siamese correlation filter 구조를 활용하여 실시간 성능을 유지하면서도 추적 작업에 특화된 특징 표현을 학습한다는 점이 기존 접근 방식과의 주요 차별점이다.

## 🛠️ Methodology

### 전체 시스템 구조

본 모델은 **Siamese Correlation Filter Network** 구조를 기반으로 한다. 두 개의 가중치 공유 브랜치(template branch, search branch)가 CNN을 통해 특징을 추출하고, 이를 상관 필터(Correlation Filter)를 통해 결합하여 응답 맵(response map)을 생성한다.

### 비지도 학습 절차 (Unsupervised Learning Pipeline)

학습은 다음과 같은 전방향-후방향 루프로 진행된다:

1. **전방향 추적(Forward Tracking)**: 첫 번째 프레임 $P_1$에서 템플릿 $T$를 설정하고, 다음 프레임 $P_2$의 탐색 영역 $S$에서 타겟의 위치를 예측하여 응답 맵 $R^S$를 생성한다.
2. **의사 레이블 생성(Pseudo Labeling)**: $R^S$의 최댓값 지점을 중심으로 가우시안 분포의 의사 레이블 $Y^S$를 생성한다.
3. **후방향 추적(Backward Tracking)**: $P_2$의 $S$를 템플릿으로, $P_1$의 $T$를 탐색 영역으로 설정하여 다시 전방향의 역과정으로 추적을 수행한다. 이를 통해 첫 번째 프레임에서의 응답 맵 $R^T$를 얻는다.
4. **일관성 손실 계산**: 이상적인 경우, 후방향 추적으로 돌아온 위치 $R^T$는 최초의 초기 레이블 $Y^T$와 일치해야 한다.

### 주요 방정식 및 손실 함수

- **기본 일관성 손실(Consistency Loss)**:
  $$L_{un} = \|R^T - Y^T\|^2_2$$
  이 식은 후방향 추적 결과와 초기 위치 사이의 $L_2$ 거리를 최소화함으로써 네트워크 파라미터 $\theta$를 업데이트한다.

- **다중 프레임 일관성 손실**:
  단일 프레임의 한계를 극복하기 위해 $M$개의 프레임을 사용하여 궤적을 확장하며, 각 경로의 손실을 합산한다.
  $$L_{un} = \sum_{k=1}^{M} L_k, \quad \text{where } L_k = \|R^{(S_k \to T)} - Y^T\|^2_2$$

- **비용 민감 손실(Cost-sensitive Loss)**:
  노이즈 샘플을 제거하는 이진 가중치 $A_{drop}$과, 타겟의 움직임이 큰 샘플에 더 많은 가중치를 주는 모션 가중치 $A_{motion}$을 결합한 정규화 가중치 $A_{norm}$을 사용한다. 최종 손실 함수는 다음과 같다.
  $$L_{final} = \frac{1}{N} \sum_{k=1}^{M} \sum_{i=1}^{N} A_{norm}^i \cdot L_k^i$$
  여기서 $A_{norm}$은 학습 데이터 중 상위 10%의 고손실 샘플을 제외하고, 모션이 큰 샘플에 더 높은 가중치를 부여하여 학습의 안정성을 높인다.

### 학습 및 추론 절차

- **RoI 선택**: 학습 데이터에서 배경(하늘, 잔디 등)을 배제하기 위해 이미지 엔트로피(image entropy)를 측정하여 가장 정보량이 많은 영역을 초기 타겟으로 자동 선택한다.
- **추론(Inference)**: 학습이 완료된 후, 테스트 단계에서는 일반적인 Siamese 추적기처럼 전방향 추적만을 수행하며, 실시간 성능을 위해 DCF의 이동 평균(moving average) 업데이트 방식을 사용하여 타겟의 외형 변화에 대응한다.

## 📊 Results

### 실험 설정

- **데이터셋**: OTB-2013, OTB-2015, Temple-Color, VOT2016, VOT2017/2018, LaSOT, TrackingNet 등 7개의 벤치마크를 사용하였다.
- **지표**: 거리 정밀도(Distance Precision, DP), 성공률(Success Plot, AUC), EAO(Expected Average Overlap) 등을 측정하였다.
- **비교 대상**: SiamFC, CFNet과 같은 지도 학습 기반 추적기, HOG와 같은 수동 특징 기반 추적기, 그리고 오토인코더 기반 비지도 학습 모델과 비교하였다.

### 주요 결과

- **지도 학습과의 격차 최소화**: ILSVRC 2015 데이터셋을 사용했을 때, 비지도 학습으로 학습된 LUDT의 AUC(60.0%)는 완전 지도 학습(62.6%)과 단 2.6%p의 차이만을 보였다.
- **성능 우위**: LUDT는 기본 성능만으로도 SiamFC 및 CFNet과 대등하거나 더 나은 성능을 보였으며, 온라인 업데이트 기법을 추가한 LUDT+는 최신 SOTA 추적기들과 경쟁 가능한 수준의 성능을 달성하였다.
- **데이터 확장성**: unlabeled 데이터(OxUvA, LaSOT 등)를 추가하여 학습시킬수록 추적 정확도가 꾸준히 향상됨을 확인하였다. 이는 비지도 학습 프레임워크가 인터넷상의 방대한 unlabeled 비디오를 활용할 수 있는 잠재력이 있음을 시사한다.
- **속도**: LUDT와 LUDT+는 각각 약 70 FPS와 55 FPS의 속도로 동작하여 실시간 추적이 가능하다.

## 🧠 Insights & Discussion

### 강점

본 연구는 레이블링이라는 고비용 작업 없이도 딥러닝 기반의 강력한 특징 표현을 학습할 수 있음을 입증하였다. 특히, 단순한 특징 추출이 아니라 '추적'이라는 작업의 본질(양방향 일관성)을 학습 목표로 설정함으로써, 일반적인 비지도 학습 모델(예: 오토인코더)보다 추적 작업에 훨씬 적합한 특징을 생성한다.

### 한계 및 비판적 해석

- **시맨틱 정보의 부족**: 시각화 결과, 비지도 학습 모델은 지도 학습 모델에 비해 객체의 경계(boundary) 표현 능력이 다소 떨어진다. 이는 정답 레이블이 제공하는 강력한 구분 정보(discriminative information)가 없기 때문이며, 이로 인해 가려짐(occlusion)이나 급격한 외형 변화가 발생할 때 드리프트(drift) 현상이 더 쉽게 나타나는 한계가 있다.
- **학습 비용**: 오프라인 학습 단계에서 전방향 및 후방향 추적을 반복적으로 수행해야 하므로, 학습 시의 계산 부하가 크다는 점이 단점으로 지적된다.

## 📌 TL;DR

본 논문은 **전방향-후방향 추적의 일관성**을 이용해 레이블 없는 비디오에서 특징 표현을 학습하는 비지도 학습 기반의 Siamese Correlation Filter 추적기를 제안한다. 다중 프레임 검증과 비용 민감 손실 함수를 통해 학습 안정성을 높였으며, 실험적으로 지도 학습 기반의 베이스라인 모델과 대등한 성능 및 실시간 속도를 달성하였다. 이 연구는 향후 인터넷상의 무수히 많은 unlabeled 비디오 데이터를 활용하여 추적 모델의 성능을 극대화할 수 있는 가능성을 제시하였다.
