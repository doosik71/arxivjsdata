# UHP-SOT: An Unsupervised High-Performance Single Object Tracker

Zhiruo Zhou, Hongyu Fu, Suya You, Christoph C. Borel-Donohue and C.-C. Jay Kuo (2021)

## 🧩 Problem to Solve

본 논문은 단일 객체 추적(Single Object Tracking, SOT)에서 발생하는 세 가지 주요 도전 과제, 즉 1) 객체 외형의 급격한 변화, 2) 추적 실패(tracking loss) 시의 복구 어려움, 3) 객체의 위치 및 형태의 급격한 변동을 해결하고자 한다.

특히 기존의 비지도 학습 기반 추적기, 그중에서도 Discriminative Correlation Filters (DCF) 기반 방법론들은 전경(foreground)의 상관관계에만 의존하는 경향이 있어, 객체를 놓쳤을 때 이를 다시 찾아내는 복구 능력이 부족하고 객체의 박스 크기를 유연하게 조정하는 능력이 떨어진다는 한계가 있다. 본 연구의 목표는 전경뿐만 아니라 배경(background)의 상관관계를 함께 활용하여, 지도 학습 기반의 딥러닝 추적기에 근접한 성능을 내면서도 별도의 학습 데이터가 필요 없는 고성능 비지도 학습 추적기인 UHP-SOT를 개발하는 것이다.

## ✨ Key Contributions

UHP-SOT의 핵심 아이디어는 **전경 모델의 외형 업데이트, 배경 움직임 모델링, 그리고 궤적 기반의 박스 예측**이라는 세 가지 모듈을 결합하여 상호 보완적인 추적 메커니즘을 구축하는 것이다.

가장 중심적인 직관은 DCF 기반 추적기가 전경의 특징을 통해 객체를 찾는 것과 대조적으로, 배경 모델링을 통해 배경 영역을 제거함으로써 객체가 존재할 가능성이 높은 영역을 역으로 추론할 수 있다는 점이다. 여기에 객체 운동의 관성(inertia)을 유지하는 궤적 예측 모듈을 더해, 외형 모델만으로는 포착하기 어려운 급격한 형태 변화나 위치 변동에 대응하고 추적의 안정성을 높였다.

## 📎 Related Works

본 논문은 크게 두 가지 관련 연구 흐름을 언급한다.

1. **DCF-based Trackers**: DCF 기반 추적기들은 객체 패치 주변에서 조밀한 샘플링을 수행하고 회귀 문제를 풀어 템플릿을 학습한다. 특히 푸리에 도메인(Fourier domain)에서 연산을 수행하여 속도가 매우 빠르다는 장점이 있다. 본 논문에서 베이스라인으로 채택한 STRCF(Spatial-Temporal Regularized Correlation Filters)는 공간-시간적 정규화를 추가하여 성능을 개선한 모델이다.
2. **DL-based Trackers**: 딥러닝 기반 추적기들은 대규모 데이터셋으로 사전 학습된 네트워크를 특징 추출기로 사용하거나, End-to-End 모델을 사용한다. 최근에는 Siamese 네트워크 기반의 추적기들이 단순성과 효과성으로 주목받고 있다. 하지만 이들은 대량의 정답 박스(annotated boxes)가 포함된 지도 학습 데이터가 필수적이라는 제약이 있다.

UHP-SOT는 이러한 지도 학습의 의존성을 없애면서도, 기존 비지도 학습 방법론들이 해결하지 못한 '추적 실패 후 복구'와 '박스 크기 적응' 문제를 배경 모델링과 궤적 예측이라는 새로운 접근 방식으로 해결하여 차별성을 가진다.

## 🛠️ Methodology

UHP-SOT는 **Tracking-by-Detection** 패러다임을 따르며, HOG(Histogram of Gradients)와 Color Name (CN) 특징을 사용한다. 전체 시스템은 다음 세 가지 모듈의 제안(proposal)을 융합하여 최종 위치를 결정한다.

### 1. Appearance Model Update (STRCF)

베이스라인으로 사용되는 STRCF는 프레임 $t$에서의 객체 외형을 템플릿 $f_t$로 모델링한다. 템플릿 업데이트는 다음의 회귀 방정식을 통해 수행된다.

$$\arg \min_{f} \frac{1}{2} \left\| \sum_{d=1}^{D} x_{t}^{d} * f^{d} - y \right\|^{2} + \frac{1}{2} \sum_{d=1}^{D} \| w \cdot f^{d} \|^{2} + \frac{\mu}{2} \| f - f_{t-1} \|^{2}$$

여기서 각 항의 의미는 다음과 같다.

- 첫 번째 항: 새로운 관측 특징 $x_t$가 할당된 레이블 $y$(가우시안 형태)와 일치하도록 한다.
- 두 번째 항: 공간적 정규화(spatial regularization)로, 박스 외부 영역의 기여도를 낮춘다.
- 세 번째 항: 자기 정규화(self-regularization)로, 외형의 급격한 변화를 막고 부드러운 변화를 유도한다.

UHP-SOT는 여기서 개선을 더해, 뚜렷한 움직임이 없을 때는 업데이트를 건너뛰며, 모든 모듈의 예측이 일치할 때는 $\mu$ 값을 낮추어 변형이 심한 객체에 빠르게 적응하도록 한다.

### 2. Background Motion Modeling

객체의 움직임을 객체 모션과 배경 모션으로 분리하여 생각한다. 배경 모션은 파라미터 모델로 단순하게 모델링할 수 있으며, 본 논문은 다음과 같은 6-파라미터 모델을 제안한다.

$$x_{t+1} = \alpha_1 x_t + \alpha_2 y_t + \alpha_0$$
$$y_{t+1} = \beta_1 x_t + \beta_2 y_t + \beta_0$$

살리언트 포인트(salient points)들을 이용해 최소제곱법으로 파라미터를 구한 뒤, 현재 프레임 $I_{t+1}$과 예측된 배경 이미지 $\hat{I}_{t+1}$의 차이 맵 $\Delta I$를 계산한다.

$$\Delta I = \hat{I}_{t+1}(x,y) - I_{t+1}(x,y)$$

배경 영역은 값이 작고 전경(객체) 영역은 값이 크게 나타나므로, 이를 통해 객체의 잠재적 위치를 파악한다. 이는 전경 상관관계에만 의존하는 DCF가 객체를 놓쳤을 때, 배경을 제거함으로써 객체를 다시 찾아내는 복구 메커니즘으로 작용한다.

### 3. Trajectory-based Box Prediction

최근 $N$ 프레임 동안의 박스 중심점 이동 벡터들에 대해 PCA(주성분 분석)를 적용하여 다음 프레임의 중심 위치 $(\hat{x}_t, \hat{y}_t)$를 예측함으로써 운동 관성을 유지한다.

또한, 배경 제거 후 남은 잔차 이미지(residual image)에서 픽셀 값의 합을 수평/수직 방향으로 계산하고 임계값을 적용하여 박스의 너비 $\hat{w}$와 높이 $\hat{h}$를 추정한다.
$$\hat{w} = x_{\max} - x_{\min}, \quad \hat{h} = y_{\max} - y_{\min}$$
이 추정치는 궤적의 급격한 변동이 있을 경우 기각되어 강건성을 확보한다.

### 4. Fusion Strategy

최종 박스는 다음 세 가지 제안 중 하나를 선택한다.

- $B_{app}$: 외형 변화 포착 (STRCF)
- $B_{trj}$: 위치/형태의 관성 유지 (Trajectory Predictor)
- $B_{bgd}$: 부적절한 영역 제거 (Background Predictor)

최신 모델 $f_{t-1}$과 모든 모듈이 합의했던 과거의 안정적인 모델 $f_i$를 사용하여 상관 점수를 계산한다. 기본적으로는 점수가 가장 높은 제안을 선택하되, $B_{app}$의 점수가 가장 높더라도 $B_{trj}$의 점수가 근접하고 $B_{bgd}$와 일치하며 위치의 급격한 점프가 관찰될 경우 $B_{trj}$를 선택하는 특수 규칙을 적용한다.

## 📊 Results

### 실험 설정

- **데이터셋**: TB-50 및 TB-100 (One Pass Evaluation 프로토콜 사용)
- **평가 지표**: Precision plot (20-pixel threshold 기준), Success plot (AUC score 기준)
- **속도**: CPU(Intel i5-9400F)에서 22.7~32.0 FPS로 동작하여 실시간성 확보.

### 주요 결과

- **정량적 성능**: TB-100에서 STRCF 대비 Precision 4%, Overlap 3.03% 향상되었으며, TB-50에서도 유사한 향상을 보였다. 특히 비지도 학습 방식임에도 불구하고 ResNet-50 기반의 SiamRPN++와 같은 최신 지도 학습 기반 딥러닝 추적기와 대등한 성능을 달성하였다.
- **정성적 분석**: 객체가 가려지거나(occlusion) 급격히 변형되는 상황에서도 배경 모델링을 통한 빠른 복구와 궤적 예측을 통한 안정성 덕분에 정밀한 추적이 가능함을 확인하였다. 다만, 영화 편집 등으로 인해 전경과 배경이 동시에 급변하는 시퀀스(Ironman, Matrix 등)에서는 한계를 보였으나, 이는 실제 환경보다는 특수 효과에 의한 것이므로 실용적 영향은 적다고 판단한다.
- **속성별 평가**: Deformation(DEF), In-plane Rotation(IPR), Low Resolution(LR) 속성에서 다른 비지도 DCF 추적기들보다 월등한 AUC 점수를 기록하였다.

## 🧠 Insights & Discussion

본 논문은 객체 추적이라는 문제가 반드시 고차원적인 지도 학습(supervision)이 필요한 영역이 아닐 수 있다는 가설을 제시한다. 배경 모션 모델링과 궤적 관성이라는 고전적인 기하학적/통계적 접근을 현대적인 DCF 프레임워크에 통합함으로써, 딥러닝 모델이 제공하는 일반화 성능에 필적하는 결과를 낼 수 있음을 보여주었다.

특히, **'관성(Inertia)'**의 개념을 위치뿐만 아니라 박스의 크기와 형태(aspect ratio)에 적용한 점이 인상적이다. DCF 기반 추적기들이 주로 고정된 비율의 다중 스케일 서치 영역을 사용하는 한계를 극복하고, 배경 제거 잔차 이미지를 통해 유연한 박스 적응(box adaptation)을 구현한 것이 성능 향상의 핵심 요인으로 분석된다.

다만, 제안된 방법론은 배경 모션이 파라미터 모델로 단순화될 수 있다는 가정을 전제로 한다. 배경 자체가 매우 복잡하게 움직이거나 전경과 유사한 움직임을 보일 경우, 배경 모델링 모듈의 신뢰도가 떨어질 가능성이 있다.

## 📌 TL;DR

UHP-SOT는 비지도 학습 기반의 단일 객체 추적기로, 기존 DCF 추적기의 한계인 '추적 복구 능력 부족'과 '형태 적응력 저하'를 해결하기 위해 **배경 모션 모델링**과 **궤적 기반 박스 예측** 모듈을 도입하였다. 실험 결과, CPU 환경에서 실시간으로 동작하면서도 최신 지도 학습 기반 딥러닝 추적기에 근접하는 고성능을 달성하였다. 이 연구는 저사양 플랫폼에서 효율적인 실시간 추적 시스템을 구축하는 데 중요한 기반이 될 수 있으며, 비지도 학습만으로도 충분한 추적 성능을 낼 수 있음을 시사한다.
