# ZERO-SHOT AUDIO CLASSIFICATION BASED ON CLASS LABEL EMBEDDINGS

Huang Xie, Tuomas Virtanen (2019)

## 🧩 Problem to Solve

본 논문은 지도 학습(Supervised Learning) 기반의 오디오 분류 시스템이 가진 데이터 의존성 문제를 해결하고자 한다. 일반적으로 높은 성능의 오디오 분류 모델을 구축하기 위해서는 타겟 클래스에 대한 대량의 어노테이션(annotation) 데이터가 필요하다. 그러나 현실적으로 새로운 오디오 인식 작업을 수행할 때마다 매번 새로운 데이터셋을 수집하고 레이블을 지정하는 과정은 시간과 비용 측면에서 매우 비효율적이다.

따라서 본 연구의 목표는 학습 단계에서 한 번도 접하지 못한 클래스의 오디오 샘플이 들어오더라도, 해당 클래스의 텍스트 레이블이 가진 세만틱 정보(semantic information)만을 이용하여 분류를 수행하는 Zero-Shot Learning(ZSL) 기반의 오디오 분류 시스템을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 오디오 특징 공간과 클래스 레이블의 세만틱 공간을 연결하는 Bilinear Model을 사용하여 두 임베딩 간의 호환성(compatibility)을 측정하는 것이다. 구체적으로는 다음과 같은 설계를 제안한다.

1.  **VGGish**를 사용하여 오디오 신호로부터 고차원 특징 임베딩을 추출한다.
2.  **Word2Vec**을 사용하여 텍스트 형태의 클래스 레이블을 세만틱 임베딩 공간으로 투영한다.
3.  두 임베딩을 입력으로 받는 **Bilinear Model**을 통해, 특정 오디오 샘플이 특정 클래스 레이블과 얼마나 일치하는지를 수치화하고, 가장 높은 호환성을 보이는 클래스로 분류한다.

## 📎 Related Works

기존의 Zero-Shot Learning 연구는 주로 컴퓨터 비전 분야에서 활발히 진행되었다. 대부분의 시각적 ZSL 접근 방식은 샘플과 클래스 모두를 중간 단계의 세만틱 표현(intermediate-level semantic representations)으로 변환하여 두 도메인 간의 관계를 학습하는 방식을 취한다. 이때 클래스의 세만틱 표현으로는 속성 임베딩(attribute embeddings)이나 단어 벡터(word vectors)가 주로 사용되었다.

하지만 저자들은 이러한 연구들이 주로 이미지 인식에 치중되어 있으며, 오디오 분류 작업에 Zero-Shot Learning을 적용한 연구는 거의 없다는 점을 지적하며 본 연구의 차별성을 강조한다.

## 🛠️ Methodology

### 전체 파이프라인
시스템의 전체 구조는 오디오 입력과 클래스 레이블 입력을 각각 임베딩 벡터로 변환한 뒤, 이를 Bilinear Model에 입력하여 최종 클래스를 결정하는 구조이다.

### 1. Audio Feature Embedding
오디오 기록에서 특징을 추출하기 위해 Google에서 공개한 VGGish 모델을 사용한다.
- **전처리**: 입력 오디오를 $16\text{kHz}$ 모노(mono) 신호로 리샘플링하고, 1초 단위의 세그먼트마다 $96 \times 64$ 크기의 $\log$ mel spectrogram 텐서를 추출한다.
- **임베딩 생성**: VGGish는 1초의 오디오당 하나의 임베딩 벡터를 생성한다. 전체 오디오 신호를 대표하기 위해 각 1초 단위 임베딩 벡터들의 평균값을 최종 오디오 특징 임베딩으로 사용한다.

### 2. Class Label Embedding
클래스 레이블의 텍스트 정보를 수치화하기 위해 사전 학습된 Word2Vec 모델을 사용한다.
- **벡터화**: Google의 사전 학습된 모델을 통해 각 단어를 $300$차원의 벡터로 변환한다.
- **다중 단어 처리**: 레이블이 단일 단어인 경우 해당 단어 벡터를 그대로 사용하며, 여러 단어로 구성된 레이블의 경우 각 단어 벡터들의 평균을 계산하여 클래스 레이블 임베딩을 생성한다.

### 3. Bilinear Model 및 학습 절차
Bilinear Model은 오디오 특징 임베딩 $\phi(x)$와 클래스 레이블 임베딩 $\psi(y)$ 사이의 호환성을 측정한다.

**호환성 함수 (Compatibility Function):**
파라미터 행렬 $W$가 주어졌을 때, 호환성 $\mathcal{F}$는 다음과 같이 정의된다.
$$\mathcal{F}(x, y; W) = \phi(x)^T \cdot W \cdot \psi(y)$$

**예측 (Prediction):**
입력 샘플 $x$에 대해 가장 높은 호환성 점수를 가진 클래스 $y$를 최종 결과로 선택한다.
$$f(x; W) = \text{argmax}_{y \in \mathcal{Y}} \mathcal{F}(x, y; W)$$

**학습 및 손실 함수 (Training & Loss Function):**
모델은 확률적 경사 하강법(SGD)을 통해 학습되며, 랭킹 손실(ranking loss)을 최소화하는 것을 목표로 한다. 개별 샘플에 대한 손실 함수 $\ell$은 다음과 같다.
$$\ell(x_n, y_n, y; W) = \Delta(y_n, y) + \mathcal{F}(x_n, y; W) - \mathcal{F}(x_n, y_n; W)$$
여기서 $\Delta(y_n, y)$는 $y_n = y$일 때 $0$, 다를 때 $1$의 값을 가지는 타겟 손실이다. 이는 정답 클래스의 호환성 점수가 오답 클래스보다 최소한 $1$ 이상 높게 유지되도록 강제하는 역할을 한다.

최종적으로 모델은 다음과 같은 가중 랭킹 손실(weighted ranking loss)의 경험적 위험(empirical risk)을 최소화하도록 학습된다.
$$\frac{1}{N} \sum_{n=1}^{N} \beta_{r(x_n, y_n; W)} \sum_{y \in \mathcal{Y}} \max \{0, \ell(x_n, y_n, y; W)\}$$
여기서 $\beta_r$은 클래스의 랭크에 따른 가중치로, 정답 클래스의 순위가 낮을수록 더 큰 페널티를 부여한다.

## 📊 Results

### 실험 설정
- **데이터셋**: ESC-50 (50개 클래스, 2000개 샘플, 5개 대분류 카테고리).
- **측정 지표**: Top-1 Accuracy (%).
- **비교 대상**: 무작위 추측(Random Guess)의 정확도인 $10\%$ (50개 클래스 기준).

### 주요 실험 결과
논문은 네 가지 설정(Setting)을 통해 성능을 평가하였다.

1.  **Setting 1 (카테고리 내 교차 검증)**: 동일 카테고리 내 8개 클래스로 학습하고 2개 클래스로 평가했을 때, 무작위 추측($50\%$, 5개 클래스 중 선택 시)보다 높은 성능을 보였다.
2.  **Setting 2 (카테고리 간 ZSL)**: 한 카테고리로 학습하고 다른 카테고리로 평가한 결과, 평균 정확도는 $10\%$를 약간 상회하였다. 특히 학습 카테고리에 따라 성능 편차가 크게 나타나, Bilinear Model이 학습 설정에 민감함을 확인하였다.
3.  **Setting 3 (확장된 ZSL)**: 4개 카테고리(40개 클래스)로 학습하고 나머지 1개 카테고리로 평가하였다. 전체 평균 정확도는 약 $26\%$를 기록했으며, 특히 'Natural' 오디오 카테고리에서는 최대 $39.7\%$의 정확도를 달성하였다.
4.  **Setting 4 (Few-shot Learning)**: ZSL 설정에 타겟 클래스당 5개의 샘플을 추가하여 학습시켰을 때, 성능이 비약적으로 향상되었다 (예: Natural 카테고리 $68.4\%$).

### 분석
Natural 카테고리의 혼동 행렬(Confusion Matrix) 분석 결과, "sea waves", "rain", "wind"와 같이 세만틱하게 유사한 클래스들 사이에서 오분류가 주로 발생함을 확인하였다. 이는 모델이 Word2Vec의 세만틱 정보를 어느 정도 유의미하게 활용하고 있음을 시사한다.

## 🧠 Insights & Discussion

**강점:**
본 논문은 오디오 분야에서 시도되지 않았던 Zero-Shot Learning 프레임워크를 제안하였으며, VGGish와 Word2Vec이라는 검증된 임베딩 도구를 결합하여 추가적인 오디오 데이터 없이도 무작위 추측보다 높은 성능을 낼 수 있음을 입증하였다.

**한계 및 비판적 해석:**
1.  **학습 데이터 민감도**: Setting 2의 결과에서 알 수 있듯이, 학습에 사용되는 클래스 구성에 따라 성능 변동이 매우 심하다. 이는 Bilinear Model이 범용적인 매핑 함수를 학습하기보다 특정 데이터셋의 분포에 과적합(overfitting)되기 쉬운 구조임을 보여준다.
2.  **성능의 절대적 수치**: 평균 $26\%$라는 정확도는 무작위 추측보다는 높지만, 실제 서비스에 적용하기에는 여전히 매우 낮은 수준이다. 이는 단순한 Bilinear Model만으로는 오디오의 복잡한 특징과 텍스트의 추상적 의미 사이의 간극(semantic gap)을 완전히 메우기 어려움을 의미한다.
3.  **데이터셋 규모**: ESC-50이라는 상대적으로 작은 데이터셋에서 실험이 진행되었으므로, AudioSet과 같은 대규모 데이터셋에서의 일반화 성능은 검증되지 않았다.

## 📌 TL;DR

본 논문은 VGGish(오디오 특징)와 Word2Vec(클래스 세만틱)을 Bilinear Model로 연결하여, 학습 데이터가 없는 클래스도 분류할 수 있는 Zero-Shot Audio Classification 시스템을 제안하였다. ESC-50 데이터셋 실험 결과, 무작위 추측($10\%$)보다 높은 평균 $26\%$의 정확도를 보였으며, 특히 자연음 카테고리에서 강점을 보였다. 이 연구는 향후 대규모 오디오 데이터셋을 활용한 세만틱 기반 분류 연구의 가능성을 제시하였다.
