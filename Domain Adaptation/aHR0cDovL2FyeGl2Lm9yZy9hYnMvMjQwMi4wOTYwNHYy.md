# Medical Image Segmentation with InTEnt: Integrated Entropy Weighting for Single Image Test-Time Adaptation

Haoyu Dong, Nicholas Konz, Hanxue Gu, Maciej A. Mazurowski (2024)

## 🧩 Problem to Solve

본 논문은 의료 영상 세그멘테이션 분야에서 발생하는 **Single Image Test-Time Adaptation (SITTA)** 문제를 해결하고자 한다. 일반적인 Test-Time Adaptation (TTA) 기법들은 테스트 단계에서 타겟 도메인의 데이터가 충분히 존재하거나, 데이터가 연속적으로 들어오는 온라인(online) 환경을 가정한다. 그러나 실제 의료 현장에서는 데이터 획득 비용이 매우 높고, 개인정보 보호 문제 및 스캐너 설정의 가변성으로 인해 단 한 장의 라벨 없는 이미지(single unlabeled image)만으로 모델을 적응시켜야 하는 상황이 빈번하게 발생한다.

기존의 TTA 방식들은 주로 예측값의 Entropy를 최소화하는 방향으로 모델 파라미터를 업데이트하지만, 단일 이미지 환경에서는 다음과 같은 문제가 발생한다:

1. **통계적 불안정성**: 단 한 장의 이미지만으로는 타겟 도메인의 Batch Normalization (BN) 층 통계량(평균 및 표준편차)을 정확하게 추정하기 어려우며, 이는 모델 성능의 심각한 불안정성으로 이어진다.
2. **최적화의 한계**: 단일 이미지에 대해 그래디언트 기반의 최적화를 수행하는 것은 성능 향상 폭이 적거나, 오히려 오버피팅을 유발할 위험이 있다.

따라서 본 논문의 목표는 단일 타겟 이미지 데이터만으로도 도메인 시프트(domain shift)에 강건하게 대응할 수 있는 세그멘테이션 적응 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **최적의 BN 통계량을 찾으려는 시도 대신, 가능한 여러 통계량 조합을 가진 모델들의 앙상블을 구축하고 이를 엔트로피 기반으로 통합**하는 것이다.

구체적인 기여 사항은 다음과 같다:

- **BN 통계량 기반 앙상블**: 소스 도메인 통계량($S^{train}$)과 타겟 도메인 통계량($S^{test}$) 사이를 보간(interpolation)하여 여러 개의 적응된 모델 후보군을 생성한다.
- **통합 엔트로피 가중치(Integrated Entropy Weighting)**: 각 모델의 예측 결과에 대해, 단순한 엔트로피가 아닌 전경(foreground)과 배경(background)의 균형을 맞춘 엔트로피 가중치를 부여하여 최종 예측값을 산출한다.
- **SITTA를 위한 실용적 접근**: 반복적인 파라미터 최적화 없이 BN 통계량 수정만으로 적응을 수행함으로써 계산 효율성을 높이고 단일 이미지에서의 불안정성을 극복하였다.

## 📎 Related Works

### 관련 연구 및 한계

- **General TTA**: Tent, SAR 등은 테스트 시점에 예측 엔트로피를 최소화하여 모델을 업데이트한다. 하지만 이들은 배치(batch) 단위의 데이터가 필요하며, 단일 이미지 상황에서는 타겟 도메인 통계량 추정이 불안정하여 성능이 저하된다.
- **Single Image TTA**: 기존의 SITTA 연구들은 학습 단계에서 보조 네트워크(Denoising Autoencoder, Domain Encoder 등)를 함께 학습시키거나, 테스트 이미지에 강한 데이터 증강(Augmentation)을 적용하는 방식을 사용했다. 이는 사전 학습된 임의의 모델에 적용하기 어렵거나 증강 함수 선택에 민감하다는 한계가 있다.

### 차별점

InTEnt는 추가적인 보조 네트워크 학습 없이, 기존 모델의 BN 층 통계량을 조절하는 것만으로 적응을 수행한다. 특히 단일 최적의 통계량을 찾는 대신 Bayesian 관점에서 여러 가능성을 통합(Integration)함으로써 강건성을 확보했다는 점에서 기존 방식과 차별화된다.

## 🛠️ Methodology

### 1. 전체 파이프라인

InTEnt의 전체 과정은 다음과 같다:

1. 소스 도메인에서 학습된 모델 $f$와 단일 테스트 이미지 $x$를 입력으로 받는다.
2. BN 통계량을 다양하게 변형하여 모델 앙상블 $\mathcal{F}$를 생성한다.
3. 각 모델 $f_k \in \mathcal{F}$를 통해 세그멘테이션 확률 맵 $\hat{P}_k$를 예측한다.
4. 전경-배경 균형 엔트로피를 계산하여 각 예측값에 대한 가중치 $w_k$를 부여한다.
5. 가중 평균을 통해 최종 예측 맵 $\hat{P}$를 도출한다.

### 2. Batch Normalization 통계량 수정

BN 층의 연산은 다음과 같다:
$$\text{BN}(h) = \gamma \frac{h - \mu^{train}}{\sigma^{train}} + \beta$$
여기서 $\mu^{train}, \sigma^{train}$은 소스 도메인의 통계량이다. InTEnt는 이를 타겟 통계량 $S^{test} = \{\mu^{test}, \sigma^{test}\}$와 보간하여 혼합 통계량 $S^{mix}_\lambda$를 생성한다:
$$S^{mix}_\lambda := \lambda \times S^{train} + (1 - \lambda) \times S^{test}$$
하이퍼파라미터 $C$ (기본값 0.2)를 사용하여 $\lambda \in \{0.2, 0.4, 0.6, 0.8\}$ 및 양 끝단 $\{0, 1\}$을 샘플링하여 총 6개의 모델 후보군 $\mathcal{F}$를 구성한다.

### 3. 전경-배경 균형 엔트로피 가중치 (Foreground-Background-Balanced Entropy)

단순 평균 엔트로피는 전경/배경 중 한쪽의 예측이 지배적일 때 왜곡될 수 있다. 이를 방지하기 위해 전경($S$)과 배경($S^c$)의 엔트로피를 각각 계산한다:

- 전경 엔트로피: $H^{FG}[q_\theta(Y|x)] = -\mathbb{E}_{Y \sim q_\theta} \sum_{i,j \in S} \ln q_\theta(y_{i,j}|x)$ (단, $S = \{(i,j) | q_\theta \ge 0.5\}$)
- 배경 엔트로피: $H^{BG}[q_\theta(Y|x)]$ (전경의 여집합에 대해 동일하게 계산)

최종 가중치 $w_k$는 이 두 값의 합에 음수를 취한 값으로 결정된다:
$$w_k := - (H^{FG}[q_{\theta_k}(Y|x)] + H^{BG}[q_{\theta_k}(Y|x)])$$

### 4. 최종 예측 통합

각 모델의 예측 $\hat{P}_k$를 앞서 구한 가중치 $w_k$로 통합한다. 가중치는 먼저 정규화된 후 Softmax를 통해 확률 값으로 변환되며, 최종 결과는 다음과 같이 계산된다:
$$\hat{P} = \sum_{f_k \in \mathcal{F}} w'_k \hat{P}_k$$

## 📊 Results

### 실험 설정

- **데이터셋**: 3가지 의료 영상 데이터셋 사용
  - Spinal Cord (SC) MRI: Gray Matter 세그멘테이션
  - Retinal Fundus Imaging: 혈관 세그멘테이션
  - Chest X-ray: 폐 세그멘테이션
- **비교 대상**: UNet (Baseline), Tent, SAR, FSeg, MEMO, SITA 등 최신 TTA 기법들.
- **평가 지표**: Dice Similarity Coefficient (DSC).

### 주요 결과

- **정량적 성능**: 3개 데이터셋의 24가지 도메인 시프트 설정에서 평균적으로 **71.6% DSC**를 달성하여, 기존 SOTA 방법론(최대 68.7%)보다 평균 **2.9% 향상**된 성능을 보였다.
- **BN 통계량의 영향**: 실험 결과, 단순한 파라미터 업데이트보다 $\lambda$ 값(BN 통계량의 혼합 비율)의 선택이 성능에 훨씬 더 결정적인 영향을 미침을 확인하였다. 특히 최적의 $\lambda$ 값은 도메인 시프트의 종류에 따라 매우 가변적이었다.
- **통합 전략 분석**: 최소 엔트로피를 가진 단일 모델만 선택하는 방식(Ent.-Min)보다, 여러 모델을 가중 평균하여 통합하는 방식이 훨씬 안정적이고 높은 성능을 보였다.

## 🧠 Insights & Discussion

### 강점 및 분석

본 연구는 SITTA 환경에서 **"단일 최적해를 찾는 것보다 가능한 해들의 집합을 통합하는 것이 더 강건하다"**는 통찰을 제공한다. 특히 의료 영상에서 BN 통계량이 모델 성능에 미치는 지대한 영향력을 정량적으로 증명하였으며, 전경-배경 균형 엔트로피라는 세그멘테이션 특화 지표를 도입하여 가중치 산정의 정확도를 높였다.

### 한계 및 논의사항

- **엔트로피의 신뢰성**: 단일 이미지에서 엔트로피를 통해 모델의 정답 가능성을 추정하는 것은 본질적으로 불확실성이 존재한다. 저자들 또한 가중치 전략 간의 성능 차이가 아주 크지는 않다는 점을 언급하며, 이는 단일 데이터 포인트만으로 엔트로피를 추정하는 한계에서 기인한다고 분석한다.
- **범용성**: 본 방법론은 BN 층을 사용하는 대부분의 모델에 적용 가능하므로, 세그멘테이션 외의 다른 의료 영상 분석 태스크로의 확장 가능성이 높다.

## 📌 TL;DR

이 논문은 단 한 장의 라벨 없는 의료 영상만으로 모델을 적응시켜야 하는 **SITTA (Single Image Test-Time Adaptation)** 문제를 해결하기 위해 **InTEnt** 방법론을 제안한다. 소스-타겟 도메인의 BN 통계량을 보간하여 여러 모델 후보군을 만들고, 전경-배경 균형 엔트로피를 가중치로 사용하여 이들의 예측값을 통합하는 방식이다. 이를 통해 기존 TTA 기법들의 불안정성을 극복하고 평균 2.9%의 DSC 향상을 이루었으며, 의료 영상의 도메인 불일치 문제를 해결하는 효율적이고 강건한 프레임워크를 제시하였다.
