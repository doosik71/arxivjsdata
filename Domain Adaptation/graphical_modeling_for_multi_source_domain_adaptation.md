# Graphical Modeling for Multi-Source Domain Adaptation

Minghao Xu, Hang Wang, Bingbing Ni (2022)

## 🧩 Problem to Solve

본 논문은 **Multi-Source Domain Adaptation (MSDA)** 문제를 해결하고자 한다. 일반적인 Single-Source Domain Adaptation (UDA)이 하나의 소스 도메인에서 타겟 도메인으로 지식을 전이하는 것과 달리, MSDA는 여러 개의 소스 도메인에서 얻은 지식을 타겟 도메인으로 전이하는 것을 목표로 한다.

이 문제는 실제 환경에서 데이터가 단일 분포가 아닌 다양한 환경(예: 시간대, 날씨, 조명, 배경의 차이)에서 수집된다는 점을 고려할 때 훨씬 더 현실적이고 중요하다. 하지만 여러 소스 도메인을 동시에 다루어야 하므로, 각 도메인 간의 관계를 어떻게 모델링할 것인지와 효율적인 도메인 조합(domain combination) 방식을 어떻게 설계할 것인지가 핵심적인 난제이다. 따라서 본 논문의 목표는 여러 소스 도메인과 타겟 도메인을 공동으로 모델링하고, 학습 가능한 도메인 조합 체계를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 도메인 간의 그래프 구조(graphical structure)를 활용하여 인스턴스와 카테고리 간의 상호 의존성을 효과적으로 모델링하는 것이다. 이를 위해 저자들은 두 가지 유형의 그래프 모델을 제안한다.

1. **CRF-MSDA (Conditional Random Field for MSDA):** 쿼리 샘플과 각 도메인의 세만틱 프로토타입(semantic prototypes)을 노드로 하는 그래프를 구축하고, GNN(Graph Neural Network)을 통한 로컬 메시지 패싱을 통해 레이블의 조건부 분포 $p(Y|X)$를 학습한다.
2. **MRF-MSDA (Markov Random Field for MSDA):** 에너지 기반 공식(energy-based formulation)을 사용하여 마르코프 네트워크 상에서 관측치들의 결합 분포 $p(X|G)$를 모델링한다. 이는 양성(positive) 네트워크와 음성(negative) 네트워크를 대조 학습함으로써 더 높은 표현력과 낮은 계산 비용을 달성한다.

## 📎 Related Works

기존의 MSDA 접근 방식들은 주로 타겟 분포가 여러 소스 분포의 가중치 조합으로 근사될 수 있다는 이론적 보장에 기반하여, 각 소스 도메인별로 독립적인 분류기를 학습시킨 후 그 예측값들을 가중 평균(weighted average)하는 방식을 사용하였다. 하지만 이러한 방식은 다음과 같은 한계가 있다.

* **도메인 간 공동 모델링의 부재:** 각 도메인의 조건부 분포를 독립적으로 학습하기 때문에, 서로 다른 도메인 간의 통계적 상호 의존성을 활용하지 못한다.
* **비학습적 도메인 조합:** 대부분의 가중치 조합 방식이 휴리스틱(heuristics)이나 고정된 유사도 점수에 의존하며, 모델 학습 과정에서 직접적으로 최적화되지 않는다.

본 논문은 이를 해결하기 위해 그래프 구조를 도입하여 도메인 간의 관계를 명시적으로 정의하고, 메시지 패싱이나 결합 확률 최적화를 통해 도메인 조합 과정을 학습 가능하게 만들었다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

### 1. 공통 개념: 프로토타입 (Prototypes)

두 모델 모두 각 도메인의 카테고리를 대표하는 특징 임베딩인 프로토타입 $C = \{c_m^k\}$를 사용한다. 여기서 $m$은 도메인 인덱스($1 \dots M+1$, 타겟 포함), $k$는 카테고리 인덱스($1 \dots K$)이다. 프로토타입은 지수 이동 평균(EMA) 방식으로 업데이트되어 학습의 안정성을 높인다.

### 2. CRF-MSDA

**구조 및 절차:**

* **그래프 구축:** 쿼리 샘플 $z_q$와 모든 프로토타입들을 노드로 하는 완전 그래프를 생성한다. 두 노드 $u, v$ 사이의 인접 가중치 $A_{uv}$는 RBF 커널을 통해 계산된다:
    $$A_{uv} = \exp\left(-\frac{||X_u - X_v||^2}{2\sigma^2}\right)$$
* **메시지 패싱:** GNN을 사용하여 인접 노드들로부터 정보를 집계하여 노드 표현 $H$를 생성하고, 선형 분류기를 통해 각 노드의 레이블을 예측한다.
* **손실 함수:**
  * **Classification Loss ($L_{cls}$):** 프로토타입, 소스 쿼리, 타겟 쿼리에 대해 각각 교차 엔트로피 및 엔트로피 최소화 손실을 적용한다.
  * **Global Alignment ($L_{global}$):** 도메인 간 카테고리 관계의 일관성을 위해 인접 행렬의 블록 행렬 간 Frobenius norm을 최소화한다:
        $$L_{global} = \frac{1}{(M+1)^4} \sum_{i,j,m,n=1}^{M+1} ||\tilde{A}_{i,j} - \tilde{A}_{m,n}||_F$$
  * **Local Alignment ($L_{local}$):** 샘플이 해당 카테고리 프로토타입 근처에 밀집되도록 유클리드 거리를 제한한다.

### 3. MRF-MSDA

**구조 및 절차:**

* **에너지 기반 모델링:** 관측치들의 결합 확률을 다음과 같은 에너지 함수 $f_E$로 정의한다:
    $$p(X|G) = \frac{1}{Z} \exp(-f_E(X, G)), \quad f_E(X, G) = \frac{1}{\tau} \sum_{(u,v) \in E} ||X_u - X_v||^2$$
* **네트워크 구성:**
  * **Positive Network ($G^+$):** 동일 카테고리의 프로토타입끼리 연결하고, 쿼리 샘플을 해당 도메인/카테고리의 프로토타입과 연결한다.
  * **Negative Networks ($G^-$):** $G^+$의 엣지를 무작위로 수정하여 생성한다.
* **학습 방법 (NCE):** Noise Contrastive Estimation (NCE) 아이디어를 차용하여, 분할 함수 $Z$를 계산하는 대신 양성 네트워크와 음성 네트워크의 비정규화된 확률 $\tilde{p}(X|G)$를 대조하여 학습하는 $L_{MLE}$를 사용한다.
* **추론 (Inference):** 쿼리 샘플 $q$가 카테고리 $k$일 확률 $\hat{y}_q[k]$는 쿼리를 각 도메인의 $k$번째 프로토타입과 연결한 여러 마르코프 네트워크들의 결합 확률 합으로 계산한다.

## 📊 Results

### 실험 설정

* **데이터셋:** Digits-five, Office-31, PACS, DomainNet 등 4개의 벤치마크 사용.
* **비교 대상:** Source-only, Single Best, Source Combine 및 기존 MSDA 방법론(MDAN, DCTN, $M^3SDA$, MDDA, CMSS).
* **지표:** 분류 정확도 (Classification Accuracy %).

### 주요 결과

* **정량적 성능:** 모든 데이터셋에서 제안 모델들이 기존 방법론보다 우수한 성능을 보였다. 특히 **MRF-MSDA**가 가장 높은 성능을 기록했다.
* **Digits-five:** MRF-MSDA는 기존 최신 방법론 대비 평균 정확도를 약 12.1% 향상시켰다.
* **PACS:** 도메인 간 스타일 차이가 큰 데이터셋임에도 불구하고, MRF-MSDA는 모든 태스크에서 최고 성능을 냈으며 특히 $\rightarrow S$ 태스크에서 4.7%의 이득을 보였다.
* **DomainNet:** 가장 복잡한 데이터셋에서도 MRF-MSDA가 5/6 태스크에서 최고 성능을 기록하며 평균 정확도를 1.4% 향상시켰다.

## 🧠 Insights & Discussion

### 강점 및 분석

* **표현력의 차이:** MRF-MSDA가 CRF-MSDA보다 성능이 좋은 이유는 CRF가 레이블 간의 의존성 $p(Y|X)$만 모델링하는 반면, MRF는 관측치와 레이블의 결합 분포 $p(X, Y)$를 모두 캡처하여 더 풍부한 관계 패턴을 학습하기 때문이다.
* **계산 효율성:** MRF-MSDA는 GNN의 무거운 메시지 패싱 과정이 필요 없고, 추론 시 단순 산술 계산만 수행하므로 CRF-MSDA보다 계산 비용이 낮다. 특히 학습 단계에서 그래프 구축 복잡도가 $O(M^2K^2)$(CRF)에서 $O(M^2K)$(MRF)로 감소한다.
* **정렬의 효과:** Ablation study를 통해 Global alignment($L_{global}$)와 Local alignment($L_{local}$)가 상호 보완적이며, 두 제약 조건을 모두 사용했을 때 가장 높은 성능이 나옴을 확인하였다.

### 한계 및 논의

* **Office-31 결과:** Office-31 데이터셋에서는 성능 향상 폭이 적었는데, 이는 일부 태스크의 성능이 이미 포화 상태(95% 이상)이거나 도메인 간 유사도가 너무 높아 공동 모델링의 이점이 적었기 때문으로 해석된다.
* **가정:** 본 연구는 도메인 간 카테고리 분포(marginal distribution of category)가 동일하다는 가정을 전제로 하고 있다.

## 📌 TL;DR

본 논문은 여러 소스 도메인을 타겟 도메인으로 전이하는 MSDA 문제를 해결하기 위해 **CRF-MSDA**와 **MRF-MSDA**라는 두 가지 그래프 모델을 제안하였다. 핵심은 도메인 간의 관계를 그래프로 구조화하여 **공동 모델링**을 수행하고, 이를 통해 **학습 가능한 도메인 조합** 방식을 구현한 것이다. 실험 결과, 특히 에너지 기반의 MRF-MSDA 모델이 기존 SOTA 방법론들을 뛰어넘는 성능과 더 높은 계산 효율성을 보였다. 이 연구는 복잡한 실제 환경에서의 도메인 적응 문제를 해결하는 데 중요한 방법론적 토대를 제공한다.
