# 의료 영상 분석을 위한 도메인 적응: 한 가지 조사

Hao Guan, and Mingxia Liu,Senior Member, IEEE

## 🧩 Problem to Solve

의료 영상 분석에 사용되는 머신러닝 기술은 훈련 데이터셋(원본 도메인)과 테스트 데이터셋(타겟 도메인) 간의 데이터 분포 차이, 즉 "도메인 시프트(Domain Shift)" 문제로 인해 성능 저하를 겪습니다. 특히 의료 영상 데이터는 다양한 스캐너, 스캔 프로토콜, 환자군 등으로 인해 도메인 이질성(Domain Heterogeneity)이 심하며, 대규모의 레이블링된 데이터가 부족하여 신뢰할 수 있고 견고한 모델을 구축하기 어렵습니다. 이 논문은 이러한 도메인 시프트 문제를 해결하기 위한 도메인 적응(Domain Adaptation, DA) 방법론들을 의료 영상 분석 관점에서 종합적으로 분석하고, 현재의 연구 동향, 도전 과제 및 미래 방향을 제시합니다.

## ✨ Key Contributions

* 의료 영상 분석에서 도메인 이질성 문제를 해결하기 위한 도메인 적응(DA) 기술 도입의 필요성을 제시합니다.
* 최신 도메인 적응 모델들을 심층 모델(Deep Models)과 얕은 모델(Shallow Models)로 분류하고, 각 모델 유형 내에서 지도(Supervised), 준지도(Semi-Supervised), 비지도(Unsupervised) 학습 방식에 따라 세분화하여 체계적으로 검토합니다.
* 현재 도메인 적응 연구에 활용되는 벤치마크 의료 영상 데이터셋들을 종합적으로 정리하여 제시합니다.
* 의료 영상 분석 분야에서 도메인 적응의 현재 상태, 주요 도전 과제 및 유망한 미래 연구 방향을 논의하여 연구자들이 이 분야를 더 잘 이해할 수 있도록 돕습니다.

## 📎 Related Works

이 조사는 기존의 일반적인 도메인 적응 [9]–[15] 및 전이 학습(Transfer Learning) [16]–[21] 조사 논문들과 차별화됩니다.

* **Cheplygina et al. [22]**: 의료 영상 분석을 위한 준지도, 멀티 인스턴스, 전이 학습을 다루지만, 도메인 적응에 초점을 맞추기보다는 일반적인 전이 학습 방법을 폭넓게 다룹니다.
* **Morid et al. [23]**: ImageNet [6] 사전 학습 기반의 전이 학습 방법을 검토하지만, 최신 3D CNN 기반의 의료 영상 분석 도메인 적응 모델은 포함하지 않습니다.

이 논문은 의료 영상 분석에 특화된 도메인 적응 방법론들을 심층적이고 체계적으로 분류하여 분석한다는 점에서 기존 연구들과 구별됩니다.

## 🛠️ Methodology

이 조사는 도메인 적응 방법론을 크게 두 가지 모델 유형으로 분류하고, 각 유형 내에서 레이블 가용성에 따라 다시 세분화합니다.

### 1. 도메인 적응 설정 (Domain Adaptation Settings)

DA는 원본 도메인과 타겟 도메인이 동일한 학습 작업을 공유하지만, 데이터 분포가 다른 상황을 가정합니다.

* **수학적 정의**: 원본 도메인 $S$와 타겟 도메인 $T$는 각각 특징 공간 $X \times Y$에 대해 다른 분포 $P_s$와 $P_t$를 가집니다. $n_s$개의 레이블링된 원본 샘플 $D_S=\{(x_{S_{i}}, y_{S_{i}})\}_{i=1}^{n_{s}}$과 $n_t$개의 (레이블 유무와 무관한) 타겟 샘플 $D_T=\{(x_{T_{j}})\}_{j=1}^{n_{t}}$이 있을 때, DA의 목표는 $S$에서 학습된 지식을 $T$로 전이하여 $T$에서 특정 작업을 수행하는 것입니다.

### 2. 도메인 적응 방법론 분류 (Categorization of DA Methods)

방법론은 주로 **모델 유형(Model Type)**을 기준으로 분류되며, **레이블 가용성(Label Availability)**에 따라 하위 분류됩니다.

* **모델 유형**:
  * **얕은 도메인 적응(Shallow DA)**: 수작업으로 추출된 특징과 전통적인 머신러닝 모델을 사용합니다.
  * **깊은 도메인 적응(Deep DA)**: CNN과 같은 딥러닝 모델을 사용하여 특징 학습과 모델 훈련을 통합합니다.

* **레이블 가용성**:
  * **지도 도메인 적응(Supervised DA)**: 타겟 도메인에 소수의 레이블링된 데이터가 있는 경우.
  * **준지도 도메인 적응(Semi-Supervised DA)**: 타겟 도메인에 소수의 레이블링된 데이터와 다수의 레이블 없는 데이터가 있는 경우.
  * **비지도 도메인 적응(Unsupervised DA)**: 타겟 도메인에 레이블 없는 데이터만 있는 경우.

### 3. 얕은 도메인 적응 방법론 (Shallow Domain Adaptation Methods)

* **인스턴스 가중치 부여(Instance Weighting)**: 원본 도메인의 샘플들에 타겟 샘플들과의 관련성에 따라 다른 가중치를 부여하여 도메인 시프트를 줄입니다 (예: [40]–[44]).
* **특징 변환(Feature Transformation)**: 원본 및 타겟 샘플들을 공통의 새로운 특징 공간으로 변환하여 분포 간극을 줄입니다 (예: [46]–[48]).

### 4. 깊은 도메인 적응 방법론 (Deep Domain Adaptation Methods)

CNN 특징을 활용하거나 ImageNet 사전 학습 모델을 미세 조정하는 등의 접근 방식이 포함됩니다.

* **지도 깊은 DA**:
  * CNN 특징 추출 후 얕은 DA 방법 적용 (예: [73], [79]).
  * 사전 학습된 모델 미세 조정 (예: ImageNet 사전 학습 후 의료 데이터로 미세 조정 [31], [82]–[85]).
  * 다단계 적응(Multi-step DA) 및 작업별 3D CNN 설계 (예: 3D CNN 사전 학습 후 미세 조정 [87]–[92]).
* **준지도 깊은 DA**:
  * Y-Net과 같은 아키텍처로 재구성 디코더를 활용하거나 [96],
  * GAN 기반으로 레이블된 원본, 레이블 없는 타겟, 생성된 이미지를 사용하여 반지도 학습을 수행 [97].
* **비지도 깊은 DA**:
  * **특징 정렬(Feature Alignment)**: DANN (Domain Adversarial Neural Network) [100] 같은 적대적 학습을 통해 도메인 불변 특징을 학습 (예: [33], [101]–[106]).
  * **이미지 정렬(Image Alignment)**: CycleGAN [118] 같은 딥 생성 모델을 사용하여 이미지-투-이미지 변환을 통해 도메인 간 이미지를 정렬 (예: [121]–[123]).
  * **이미지 + 특징 정렬(Image + Feature Alignment)**: CycleGAN으로 이미지 변환 후, 적대적 학습으로 특징 정렬을 수행하여 두 수준에서 도메인 간극을 줄임 (예: [36], [37]).
  * **분리 표현(Disentangled Representation)**: 이미지를 도메인 불변 내용 공간과 도메인 특정 스타일 공간으로 분리하여 적응 수행 [35].
  * **앙상블 학습(Ensemble Learning)**: 자가 앙상블(Self-ensemble) 기법을 사용하여 교사-학생 네트워크(Teacher-Student Network)의 예측 비교를 통해 도메인 시프트 최소화 [24].
  * **소프트 레이블(Soft Labels)**: NCC(Normalized Cross Correlation)를 이용해 타겟 도메인에 대한 소프트 레이블을 생성하여 모델 훈련에 활용 [132].
  * **특징 학습(Feature Learning)**: 합성곱 오토인코더(Convolutional Auto-encoder)를 사용하여 도메인 적응을 위한 특징 학습 [133], [134].
* **다중 타겟 깊은 DA(Multi-Target Deep DA)**: 여러 타겟 도메인으로의 적응을 목표로 하며, 배치 정규화(Batch Normalization) 파라미터 미세 조정을 통한 라이프롱 학습(Lifelong Learning) 등이 연구됨 [29].

## 📊 Results

이 조사는 자체적인 실험 결과를 제시하기보다는, 의료 영상 분석 분야에서 다양한 도메인 적응 방법론들이 도메인 시프트 문제를 성공적으로 완화하고 있음을 보여줍니다. 주요 결과는 다음과 같습니다.

* **얕은 DA 모델**: 인스턴스 가중치 부여 및 특징 변환 전략을 통해 알츠하이머병 분류, 뇌종양 분할, 폐 질환 분류 등 다양한 의료 영상 분석 작업에서 도메인 시프트를 효과적으로 줄일 수 있음을 확인했습니다.
* **깊은 DA 모델**:
  * **ImageNet 사전 학습 및 미세 조정**: 자연 이미지로 사전 학습된 CNN 모델을 의료 영상 작업에 미세 조정하는 것이 초기 전이 학습에 유용하며, 특히 레이블링된 타겟 데이터가 소량일 때 효과적임을 보여줍니다.
  * **3D CNN 기반 모델**: 의료 영상의 고유한 3D/4D 구조 정보를 활용하는 3D CNN 모델이 2D 모델보다 우수한 성능을 보이며, 도메인 적응에도 성공적으로 적용되고 있습니다.
  * **비지도 DA의 부상**: 레이블 부족 문제를 해결하기 위해 DANN, CycleGAN, 자가 앙상블 등 비지도 도메인 적응 방법론들이 특징 정렬 및 이미지 변환을 통해 상당한 성능 향상을 달성하며 활발히 연구되고 있습니다.
  * **복합 전략의 효과**: 이미지 수준 및 특징 수준 정렬을 결합하거나, 분리 표현 학습, 앙상블 기법 등을 통해 단일 전략보다 더 강력한 도메인 적응 능력을 보여줍니다.

전반적으로 도메인 적응 방법은 의료 영상 데이터의 이질성으로 인한 성능 저하를 극복하는 데 필수적이며, 특히 레이블링된 데이터가 부족한 환경에서 비지도 및 준지도 깊은 도메인 적응 모델들이 유망한 해결책으로 부상하고 있음을 강조합니다.

## 🧠 Insights & Discussion

### 도전 과제 (Challenges)

* **3D/4D 체적 표현 (3D/4D Volumetric Representation)**: 의료 영상은 고차원적이며 3D 또는 4D(시간 차원 포함) 형식으로 저장됩니다. 이러한 풍부한 구조적 맥락 정보를 효과적으로 포착하고 적응하는 모델 설계가 어렵습니다.
* **제한된 훈련 데이터 (Limited Training Data)**: 의료 영상 레이블링은 비용과 시간이 많이 소요되어 레이블링된 데이터가 부족합니다. ImageNet으로 사전 학습된 2D CNN 모델은 3D 의료 영상에 최적화되지 않았으며, 고차원 의료 영상에서 전이 학습의 이점이 크지 않을 수 있습니다.
* **모달리티 간 이질성 (Inter-Modality Heterogeneity)**: CT, MRI, PET 등 다양한 영상 기법은 상호 보완적인 정보를 제공하지만, 모달리티 간 큰 차이는 도메인 적응의 효율적인 지식 전이를 방해합니다.

### 미래 연구 방향 (Future Research Trends)

* **작업별 3D/4D 도메인 적응 모델 (Task-Specific 3D/4D Models for Domain Adaptation)**: 2D CNN보다 강력한 3D/4D CNN 모델 개발이 필요하며, 작업별 관심 영역(ROI) 정의를 통해 불필요하거나 노이즈가 많은 영역을 걸러내는 것이 중요합니다.
* **비지도 도메인 적응 (Unsupervised Domain Adaptation)**: 레이블 부족 문제를 해결하기 위해 비지도 DA 방법 연구가 필수적이며, 나아가 타겟 데이터를 전혀 사용하지 않는 도메인 일반화(Domain Generalization) [34] 및 제로샷 학습(Zero-shot Learning) [154] 연구가 유망합니다.
* **다중 모달리티 도메인 적응 (Multi-Modality Domain Adaptation)**: 이질적인 다중 모달리티 데이터를 활용하기 위해 한 모달리티에서 훈련된 모델이 다른 모달리티로 잘 일반화될 수 있는 DA 모델 개발이 필요합니다 (예: CycleGAN [36], 분리 학습 [35]).
* **다중 원본/다중 타겟 도메인 적응 (Multi-Source/Multi-Target Domain Adaptation)**: 실제 환경에서는 여러 원본 도메인(예: 여러 영상 센터)이 존재하고, 모델을 여러 타겟 도메인으로 전이해야 할 수 있으므로, 이러한 복합적인 시나리오에 대한 연구가 더욱 중요해집니다.

## 📌 TL;DR

이 조사는 의료 영상 분석에서 **도메인 시프트 문제**를 해결하기 위한 **도메인 적응(DA)** 방법론들을 다룹니다. 논문은 DA 모델을 **얕은 모델**과 **깊은 모델**로 분류하고, 각 모델 내에서 **지도, 준지도, 비지도 학습** 방식에 따라 세분화하여 **인스턴스 가중치 부여, 특징 변환, 적대적 학습, 이미지-투-이미지 변환, 앙상블 학습** 등 다양한 기법을 검토합니다. **레이블링된 데이터의 부족**과 **3D/4D 의료 영상의 복잡성**, **다중 모달리티 이질성**이 주요 도전 과제이며, **비지도 학습, 작업별 3D/4D 모델, 다중 모달리티/다중 원본/타겟 적응**이 미래 연구 방향으로 제시됩니다.
