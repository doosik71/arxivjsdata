# Reinforced Domain Selection for Continuous Domain Adaptation

Hanbing Liu, Huaze Tang, Yanru Wu, Yang Li, and Xiao-Ping Zhang (2025)

## 🧩 Problem to Solve

본 논문은 소스 도메인(source domain)과 타겟 도메인(target domain) 사이의 간극이 매우 클 때, 이를 완화하기 위해 여러 중간 도메인(intermediate domains)을 거쳐 점진적으로 적응하는 Continuous Domain Adaptation (CDA) 환경에서의 도메인 선택 문제를 다룬다.

전통적인 CDA 방식은 중간 도메인들을 통해 전이 경로를 구축하여 성능을 높이지만, 명시적인 메타데이터가 없는 비지도 학습(unsupervised) 설정에서 어떤 중간 도메인을 어떤 순서로 선택해야 최적의 전이 경로(transfer path)가 되는지를 결정하는 것은 매우 어려운 문제이다. 특히 전이 경로가 너무 희소(sparse)하면 도메인 간의 간극이 커져 Negative Transfer가 발생할 가능성이 높고, 반대로 너무 조밀(dense)하면 학습 시간이 증가하고 누적 오차가 발생할 위험이 있다. 따라서 본 연구의 목표는 비지도 환경에서 타겟 도메인에 대한 예측 정확도를 최대화할 수 있는 최적의 중간 도메인 선택 전략을 동적으로 학습하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 강화 학습(Reinforcement Learning, RL)과 특징 분리(Feature Disentanglement)를 결합하여 최적의 도메인 전이 경로를 자동으로 탐색하는 프레임워크를 제안하는 것이다.

가장 중심적인 설계는 특징을 도메인 불변 특징(domain-invariant features)과 도메인 특화 특징(domain-specific features)으로 분리하는 것이다. 이를 통해 도메인 불변 특징으로는 타겟 태스크의 성능을 높이고, 저차원으로 임베딩된 도메인 특화 특징으로는 도메인 간의 거리를 계산하여 강화 학습의 보상(reward)으로 활용함으로써, 별도의 레이블 없이도 최적의 전이 경로를 찾을 수 있도록 설계하였다.

## 📎 Related Works

기존의 CDA 연구들은 Self-training, Pseudo-labeling, Adversarial algorithms, 그리고 Optimal Transport 기반의 방법론들을 통해 발전해 왔다. 최근에는 Wasserstein 거리를 이용해 중간 도메인을 정렬하는 Transfer Curriculum 방식 등이 제안되었으나, 이러한 방법들은 주로 주어진 도메인들을 정렬(sorting)하는 데 집중할 뿐, 어떤 도메인이 정말로 필수적인지를 선택하는 동적인 경로 학습에는 한계가 있다.

또한, 강화 학습을 지속 학습(Continual Learning)에 적용한 기존 연구들이 존재하지만, 이들은 주로 네트워크 모델의 구조를 동적으로 확장하는 데 초점을 맞추어 연산 비용이 매우 높다는 단점이 있다. 한편, 데이터 선택을 위해 정책 경사(Policy Gradient) 방법을 사용한 연구가 있었으나, 이는 보상 계산을 위해 타겟 레이블에 의존하므로 비지도 CDA 설정에서는 적용이 불가능하다. 본 논문은 이러한 한계를 극복하기 위해 레이블 없이 특징 분포 간의 거리를 이용한 대리 보상 함수(surrogate reward function)를 도입하여 차별성을 갖는다.

## 🛠️ Methodology

### 전체 시스템 구조
본 논문이 제안하는 프레임워크는 특징 추출기(Feature Extractor $F$), 불변 네트워크(Invariant Network $I$), 특화 네트워크(Specific Network $S$), 그리고 정책 생성기(Policy Generator $P$)로 구성된다. 전체 파이프라인은 다음과 같은 절차로 진행된다.
1. 모든 도메인의 입력 데이터가 특징 추출기 $F$를 통해 공통 특징으로 변환된다.
2. 이후 $I$와 $S$ 네트워크가 각각 도메인 불변 특징 $f_{di}$와 도메인 특화 특징 $f_{ds}$를 추출한다.
3. 추출된 $f_{ds}$는 정책 생성기 $P$의 상태(state)로 입력되어, 현재 중간 도메인을 전이 경로에 포함할지 여부를 결정하는 액션을 수행한다.
4. 도메인 특화 특징 간의 거리를 기반으로 계산된 보상이 정책 생성기의 학습에 사용된다.

### 특징 분리 (Feature Disentanglement)
도메인 간의 정렬 효율을 높이기 위해 Mutual Information Neural Estimator (MINE)를 사용하여 두 특징의 상호 정보량(Mutual Information, MI)을 추정한다. MI의 하한선은 다음과 같이 정의된다.

$$I(X;Z) = \frac{1}{n} \sum_{i=1}^{n} T_{\theta}(x^{(i)}, z^{(i)}) - \log \left( \frac{1}{n} \sum_{i=1}^{n} e^{T_{\theta}(x^{(i)}, z^{(i)})} \right)$$

여기서 $T_{\theta}$는 신경망 기반의 추정기이다. 이를 바탕으로 다음과 같은 손실 함수를 정의하여 특징을 분리한다.
- **불변 네트워크 손실 ($L_{mi}$)**: 소스, 중간, 타겟 도메인 간의 불변 특징들 사이의 MI를 최대화하여 도메인에 관계없이 일관된 특징을 추출하도록 한다.
  $$L_{mi} = -[I(f_{s}^{di}; f_{i}^{di}) + I(f_{s}^{di}; f_{t}^{di}) + I(f_{i}^{di}; f_{t}^{di})]$$
- **특화 네트워크 손실 ($L_{ms}$)**: 도메인 특화 특징들 사이의 MI를 최소화하여 각 도메인만의 고유한 특성이 잘 드러나도록 분리한다.
  $$L_{ms} = I(f_{s}^{ds}; f_{i}^{ds}) + I(f_{s}^{ds}; f_{t}^{ds}) + I(f_{i}^{ds}; f_{t}^{ds})$$
- **분류 손실 ($L_{ce}$)**: 소스 도메인의 불변 특징 $f_{s}^{di}$를 사용하여 기존의 교차 엔트로피 손실을 통해 분류기를 학습시킨다.

### 정책 생성 (Policy Generation)
강화 학습을 통해 최적의 중간 도메인 선택 정책 $\pi(a|s; \theta)$를 학습한다.

- **상태 (State, $s$):** 이전 선택된 중간 도메인의 특화 특징 $\Phi_{T-1}^{i}$, 현재 고려 중인 중간 도메인의 특화 특징 $\Phi_{T}^{i}$, 그리고 타겟 도메인의 특화 특징 $\Phi_{T}^{t}$의 조합으로 정의된다.
- **액션 (Action, $a$):** 현재 중간 도메인을 전이 경로에 포함할지 여부를 결정하는 이진 결정(1: 선택, 0: 제외)이다.
- **보상 (Reward, $R_T$):** Wasserstein 거리를 사용하여 도메인 간의 거리를 측정한다. 현재 선택한 도메인이 이전 도메인에서 타겟 도메인으로 가는 거리를 실질적으로 단축시켰을 때 양의 보상을 부여한다.
  $$R_T = \begin{cases} 2D_{sit} - D_{s'ii} - D_{s'it} & \text{if } a=1, D_{s'ii} < D_{sit} \text{ and } D_{s'it} < D_{sit} \\ -\inf & \text{if } a=1, D_{s'ii} \geq D_{sit} \text{ or } D_{s'it} \geq D_{sit} \\ 0 & \text{if } a=0 \end{cases}$$
  여기서 $D_{it}$는 현재 중간 도메인과 타겟 간의 거리, $D_{ii}$는 현재와 이전 중간 도메인 간의 거리이다.

최종적으로 정책 경사 상승법(Policy Gradient Ascent)을 통해 기대 보상 $J(\theta)$를 최대화하는 방향으로 정책 네트워크 $\theta$를 업데이트한다.

## 📊 Results

### 실험 설정
- **데이터셋:** Rotated MNIST(0~180도 회전, 11개 도메인)와 ADNI(알츠하이머 MRI 이미지, 연령대별 도메인)를 사용하였다.
- **비교 대상:** 기존 CDA 방법론(EAML, AGST, Gradual ST, CDOT, CMA, W-MPOT, CIDA) 및 RL 기반의 CDA 방법론(SDG, GRADIENT, CLEAS, RCL)과 비교하였다.
- **평가 지표:** 분류 정확도(Accuracy)를 측정하였다.

### 정량적 결과
실험 결과, 제안 방법은 두 데이터셋 모두에서 가장 높은 정확도를 기록하였다.
- **Rotated MNIST:** 기존 최상위 RL-CDA 모델인 CLEAS(92.8%)보다 높은 **93.4%**를 달성하였다.
- **ADNI:** 복잡도가 더 높은 의료 영상 데이터셋에서 **90.5%**의 정확도를 보이며, 기존 CDA 방법론들보다 월등한 성능 향상을 보였다.

### 정성적 및 분석 결과
- **경로 선택 전략:** 학습이 진행됨에 따라 보상이 안정화되면서, 모델이 불필요한 중간 도메인을 제외하고 효율적인 전이 경로를 선택하는 것이 확인되었다. (예: Rotated MNIST에서 $\hat{h}=(18, 54)$ 또는 $(18, 54, 90)$ 선택)
- **특징 시각화:** t-SNE 분석 결과, 특화 네트워크는 도메인별로 뚜렷하게 구분되는 특징을 추출하였고, 불변 네트워크는 모든 도메인이 유사한 분포를 갖도록 특징을 추출함으로써 특징 분리가 성공적으로 이루어졌음을 입증하였다.
- **Ablation Study:** 특징 분리만 적용했을 때보다 정책 생성기까지 함께 적용했을 때 정확도가 $81.5\% \rightarrow 93.4\%$로 크게 상승하여, 동적 경로 선택의 중요성을 증명하였다.

## 🧠 Insights & Discussion

본 논문은 CDA에서 단순히 도메인을 정렬하는 것을 넘어, 강화 학습을 통해 '최적의 경로'를 동적으로 탐색함으로써 성능을 극대화할 수 있음을 보여주었다. 특히 특징 분리(Disentanglement)를 통해 도메인 불변 특징으로는 분류 성능을 높이고, 도메인 특화 특징으로는 정밀한 거리 측정이 가능하게 하여 비지도 환경에서도 유효한 보상 체계를 구축한 점이 매우 영리한 접근이다.

또한, 중간 도메인의 풀(pool) 크기 $K$가 증가하더라도 정확도가 안정적으로 유지되거나 오히려 약간 상승하는 경향을 보였는데, 이는 제안된 정책 생성기가 불필요한 도메인을 효과적으로 필터링하여 Negative Transfer를 방지하고 있음을 시사한다. 다만, 보상 함수에서 $-\inf$와 같은 극단적인 값을 사용한 설정이 실제 학습 안정성에 어떤 영향을 미치는지에 대한 상세한 분석은 부족해 보이며, Wasserstein 거리 계산의 연산 복잡도가 매우 큰 도메인 풀 환경에서도 효율적으로 작동할지는 추가적인 검토가 필요하다.

## 📌 TL;DR

이 논문은 비지도 Continuous Domain Adaptation 환경에서 최적의 전이 경로를 찾기 위해 **강화 학습과 특징 분리(Feature Disentanglement)를 결합한 프레임워크**를 제안한다. 특징을 불변/특화 성분으로 나누어 각각 분류 성능 향상과 도메인 거리 기반 보상 계산에 활용함으로써, Rotated MNIST와 ADNI 데이터셋에서 기존 CDA 및 RL-CDA 방법론들을 뛰어넘는 최고의 정확도를 달성하였다. 이 연구는 향후 복잡한 도메인 시프트가 존재하는 실제 환경에서 데이터 효율적인 전이 학습 경로를 설계하는 데 중요한 기여를 할 것으로 보인다.
