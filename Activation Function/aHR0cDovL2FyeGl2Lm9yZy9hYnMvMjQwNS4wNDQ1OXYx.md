# A Significantly Better Class of Activation Functions Than ReLU Like Activation Functions

Mathew Mithra Noel, Yug Oswal (2024)

## 🧩 Problem to Solve

본 논문은 딥러닝 모델에서 거의 보편적으로 사용되는 ReLU 계열 및 Sigmoidal 계열의 활성화 함수들이 가진 근본적인 한계를 해결하고자 한다. 기존의 활성화 함수들은 단일 뉴런이 입력 공간을 하나의 하이퍼플레인(hyperplane)으로 나누어 한쪽 방향의 반공간(half-space)만을 양수 영역으로 정의하는 특성을 가진다.

이러한 구조는 복잡한 비선형 데이터셋을 구분하기 위해 더 많은 수의 뉴런을 필요로 하며, 결과적으로 모델의 크기가 커지는 원인이 된다. 따라서 본 연구의 목표는 기존의 반공간 결정 경계에서 벗어나, 더 적은 수의 뉴런만으로도 복잡한 입력 공간을 효율적으로 분할할 수 있는 새로운 클래스의 활성화 함수를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 결정 경계를 단일 하이퍼플레인이 아닌, 두 개의 평행한 하이퍼플레인 사이의 영역인 하이퍼스트립(hyper-strip) 형태로 설계하는 것이다.

이를 위해 저자들은 Cone 및 Parabolic-Cone이라는 새로운 활성화 함수를 제안하였다. 이 함수들은 특정 유한 구간에서만 양수 값을 가지고, 그 외의 구간에서는 음수 값을 갖는다. 이러한 설계를 통해 단일 뉴런이 입력 공간을 더 세밀하게 분할할 수 있게 되며, 결과적으로 기존 ReLU 계열 함수보다 훨씬 적은 수의 뉴런만으로도 동일하거나 더 높은 성능을 달성할 수 있게 된다. 특히, 기존에는 여러 개의 뉴런이 필요했던 XOR 문제를 단 하나의 뉴런만으로 해결할 수 있음을 입증하였다.

## 📎 Related Works

논문은 ReLU 계열의 활성화 함수가 등장한 이후, Sigmoidal 및 ReLU 계열과 근본적으로 다른 더 나은 함수가 존재하는지에 대한 질문이 지속적으로 제기되어 왔음을 언급한다. 기존의 Leaky ReLU, Swish, GELU 등의 함수들은 원점에서 단 하나의 제로 포인트(zero point)를 가지며, 이에 따라 결정 경계가 단일 하이퍼플레인으로 형성되는 한계를 공유한다.

본 연구는 이러한 기존 접근 방식과 달리, 결정 경계를 두 개의 하이퍼플레인으로 구성함으로써 입력 공간을 분할하는 방식 자체를 차별화하였다.

## 🛠️ Methodology

### 전체 구조 및 핵심 아이디어

단일 뉴런의 출력은 $a = g(w^T x + b)$로 정의된다. 여기서 $g$는 활성화 함수이다. 기존 함수들은 $g(z) = 0 \iff z = 0$이 성립하여 결정 경계가 $w^T x + b = 0$이라는 단일 하이퍼플레인이 되지만, 제안된 Cone-like 함수들은 $g(z) = 0 \iff z = 0 \text{ or } z = \delta$가 성립하여 두 개의 하이퍼플레인을 결정 경계로 가진다.

### 제안하는 활성화 함수

본 논문에서 제안하는 주요 활성화 함수의 수식은 다음과 같다.

1. **Cone**:
   $$g(z) = 1 - |z - 1|$$
2. **Parabolic-Cone**:
   $$g(z) = z(2 - z)$$
3. **Parameterized-Cone**:
   $$g(z) = \beta - |z - \gamma|^\alpha$$
   여기서 $\alpha, \beta, \gamma$는 활성화 함수의 모양을 결정하는 학습 가능한 파라미터이다.

### 결정 경계의 변화: Half-space $\to$ Hyper-strip

Cone-like 활성화 함수를 사용하는 뉴런의 양수 출력 영역 $C^+$는 다음과 같이 정의된다.
$$C^+ = \{x \in \mathbb{R}^n : 0 < w^T x + b < \delta\}$$
이는 무한히 넓은 반공간(half-space)이 아니라, 두 평행한 하이퍼플레인 사이의 좁은 영역인 하이퍼스트립(hyper-strip)이 된다. 이러한 특성 덕분에 단일 뉴런이 훨씬 더 복잡한 비선형 분리를 수행할 수 있다.

### 학습 및 추론 절차

- **최적화 도구**: Adam optimizer 사용 (학습률 $10^{-4}$, 표준 모멘텀 $\beta_1, \beta_2$ 적용).
- **손실 함수**: Categorical Cross-Entropy loss 사용.
- **학습 기간**: 모든 모델은 30 epoch 동안 학습되었다.

## 📊 Results

### 실험 설정

- **데이터셋**: CIFAR-10 및 Imagenette 벤치마크 사용.
- **모델 구조**: 표준 CNN 아키텍처를 사용하며, 컨볼루션 층 이후에 테스트 대상인 활성화 함수가 적용된 단일 Dense 층을 배치하고 최종적으로 Softmax 층으로 마무리하였다.
- **평가 지표**: 5회 독립 시행의 평균, 중앙값, 표준편차, 최댓값, 최솟값 테스트 정확도를 측정하였다.

### 정량적 결과

1. **CIFAR-10**:
   - Dense 층의 뉴런 수가 적을수록(예: 10개, 32개) Cone 및 Parabolic-Cone의 우위가 뚜렷하게 나타났다.
   - 특히 뉴런 10개 설정에서 Parabolic-Cone의 평균 정확도는 $0.6998$로, ReLU($0.6157$) 및 Leaky ReLU($0.6292$)보다 월등히 높았다.
2. **Imagenette**:
   - 뉴런 수가 적은 환경에서 Cone 및 Parabolic-Cone이 더 높은 정확도를 보였으며, 특히 10개 뉴런 설정에서 Parabolic-Cone이 가장 우수한 성능($0.8421$)을 기록하였다.

### 학습 속도 및 효율성

- **학습 속도**: Cone-like 함수들은 ReLU보다 미분 값이 크기 때문에 학습 속도가 유의미하게 빨라짐이 확인되었다(Figure 6 참조).
- **모델 효율성**: 하이퍼스트립 구조 덕분에 훨씬 적은 수의 뉴런만으로도 기존 모델과 유사하거나 더 높은 정확도를 달성할 수 있음을 입증하였다.

## 🧠 Insights & Discussion

### 강점 및 기여

본 연구는 활성화 함수의 형태를 변경함으로써 뉴런 하나가 수행할 수 있는 결정 경계의 기하학적 성질을 반공간에서 하이퍼스트립으로 확장하였다. 이는 딥러닝 모델의 파라미터 효율성을 극대화할 수 있는 방향성을 제시하며, 특히 XOR 문제와 같은 비선형 문제를 단일 뉴런으로 해결할 수 있다는 점은 이론적으로 매우 중요한 진전이다.

### 한계 및 비판적 해석

- **기울기 폭주 문제**: 저자들은 Cone-like 함수가 큰 미분 값을 가져 학습 속도가 빠르다고 주장하지만, 이것이 잠재적으로 기울기 폭주(exploding gradient) 문제를 야기할 가능성에 대해서는 명확한 답변을 내놓지 않았다.
- **일반화 성능**: CIFAR-10과 Imagenette라는 비교적 작은 규모의 데이터셋에서 실험이 이루어졌으므로, 초대형 모델이나 더 거대한 데이터셋에서도 동일한 효율성이 유지될지는 추가 검증이 필요하다.
- **파라미터 학습**: Parameterized-Cone의 경우 $\alpha, \beta, \gamma$를 학습 가능하게 설정하였으나, 실제 실험 결과 테이블에서는 주로 기본 Cone과 Parabolic-Cone 위주로 결과가 제시되어 있어 파라미터 최적화의 구체적인 영향력 분석이 부족하다.

## 📌 TL;DR

본 논문은 결정 경계를 하이퍼플레인이 아닌 **하이퍼스트립(hyper-strip)**으로 생성하는 **Cone** 및 **Parabolic-Cone** 활성화 함수를 제안하였다. 이를 통해 단일 뉴런의 표현력을 높여 **XOR 문제를 단일 뉴런으로 해결**할 수 있게 하였으며, CIFAR-10 및 Imagenette 실험을 통해 **더 적은 수의 뉴런으로 더 높은 정확도**를 달성하고 **학습 속도를 향상**시켰음을 입증하였다. 이 연구는 모델 경량화와 효율적인 신경망 설계에 있어 새로운 가능성을 제시한다.
