# Learn-able parameter guided Activation Functions

S. Balaji, T. Kavya, and Natasha Sebastian (2019)

## 🧩 Problem to Solve

딥러닝 네트워크의 각 레이어는 서로 다른 특성을 가지며, 이에 따라 요구되는 활성화 함수(Activation Function)의 특성 또한 다를 수 있다. 그러나 일반적인 관행은 네트워크의 모든 레이어 출력에 동일한 활성화 함수를 사용하는 것이다.

기존의 ReLU 및 그 변형들(Leaky ReLU, PReLU 등)은 단순성과 계산 효율성 덕분에 널리 사용되어 왔으나, 고정된 형태의 함수는 각 레이어의 구체적인 요구사항에 유연하게 대응하지 못한다는 한계가 있다. 따라서 본 논문은 활성화 함수의 파라미터를 학습 가능하게(learn-able) 만들어, 주변 레이어의 요구사항에 맞게 활성화 함수가 동적으로 적응하도록 함으로써 전체 네트워크의 성능을 향상시키는 것을 목표로 한다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 활성화 함수의 **양수 영역 기울기(Positive Slope)**와 **평균 이동(Mean Shift)** 파라미터를 도입하고, 이를 학습 가능하게 설계하는 것이다.

1. **DP ReLU (Dual Parametric ReLU)**: 기존 PReLU가 음수 영역의 기울기만 학습 가능하게 했던 것과 달리, 양수 영역의 기울기까지 독립적으로 학습 가능하게 설계하여 응답 영역(Response Region)을 확장하였다.
2. **Dual Line**: DP ReLU에 학습 가능한 평균 이동 파라미터($m$)를 추가하여, 활성화 결과의 평균을 0에 가깝게 유지함으로써 편향 제거 및 학습 속도 향상을 도모하였다.
3. **범용적 확장 방법론**: 제안한 학습 가능 파라미터 개념을 기존의 다른 활성화 함수 $G(x)$에도 적용할 수 있는 일반화된 수식을 제시하였다.

## 📎 Related Works

논문에서는 다음과 같은 기존 활성화 함수들의 특성과 한계를 언급한다.

- **ReLU 및 변형들**: ReLU, Leaky ReLU, PReLU 등은 계산 효율성이 높다. 특히 PReLU는 음수 영역에 학습 가능한 파라미터를 도입하여 응답 영역을 넓혔다.
- **PELU (Parametric ELU)**: 학습 가능한 파라미터를 사용하지만, 양수 영역의 파라미터가 다른 파라미터의 비율로 정의되어 있어 독립적이지 못하다는 한계가 있다.
- **Flexible ReLU 및 General ReLU**: 평균 이동(Mean Shift) 파라미터를 도입하여 활성화 평균을 0으로 밀어내는 시도가 있었다.

본 논문이 제안하는 방식은 양수 영역의 기울기 파라미터가 다른 파라미터와 완전히 **독립적**이며, 이를 평균 이동 파라미터와 결합하여 더 역동적인 적응력을 갖게 했다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 1. DP ReLU (Dual Parametric ReLU)
DP ReLU는 양수와 음수 두 축 모두에 학습 가능한 기울기 파라미터를 둔다. 수식은 다음과 같다.

$$
X = 
\begin{cases} 
\alpha \cdot x, & \text{if } x < 0 \\ 
\beta \cdot x, & \text{if } x > 0 
\end{cases}
$$

- $\alpha$: 음수 영역 기울기 파라미터 (초기값 0.01)
- $\beta$: 양수 영역 기울기 파라미터 (초기값 1)

### 2. Dual Line
Dual Line은 DP ReLU에 학습 가능한 평균 이동 파라미터 $m$을 추가하여 직선의 방정식 형태로 구성한 함수이다.

$$
X = 
\begin{cases} 
\alpha \cdot x + m, & \text{if } x < 0 \\ 
\beta \cdot x + m, & \text{if } x > 0 
\end{cases}
$$

- $m$: 평균 이동 파라미터 (초기값 -0.22)
- 이 파라미터는 활성화 함수의 평균을 0으로 밀어내어 오버피팅 위험을 줄이고 학습 속도를 높이는 역할을 한다.

### 3. 타 활성화 함수로의 확장
기존의 임의의 활성화 함수 $G(x)$가 있을 때, 이를 다음과 같이 변형하여 제안된 개념을 적용할 수 있다.

$$
X = 
\begin{cases} 
G(x) + m, & \text{if } x < 0 \\ 
\beta \cdot x + m, & \text{if } x > 0 
\end{cases}
$$

즉, $x > 0$인 영역을 학습 가능한 기울기와 평균 이동이 적용된 선형 함수로 대체하는 방식이다.

## 📊 Results

### 실험 설정
- **데이터셋**: MNIST, CIFAR10
- **모델 아키텍처**: LeNet-4, LeNet-5, WideResNet
- **비교 대상**: 총 43종의 활성화 함수
- **평가 지표**: 정확도(Accuracy), 평균 정확도(Mean Accuracy), Train/Validation Loss, 학습 시간

### 주요 결과
- **MNIST (LeNet-5)**: Dual Line이 평균 정확도 기준 1위를 기록하였으며, 전체 정확도에서는 2위를 차지하였다. 반면 DP ReLU는 18~19위권에 머물러, 평균 이동 파라미터 $m$의 영향력이 매우 큼을 시사한다.
- **MNIST (LeNet-4)**: PELU가 가장 좋은 성능을 보였으나, Dual Line과 DP ReLU 역시 각각 정확도 및 평균 정확도 기준 4~5위권의 상위 성능을 기록하였다.
- **CIFAR10 (WideResNet)**: 
    - 평균 정확도 기준 Dual Line이 2위를 기록하였다 (1위는 Aria2).
    - Train 및 Validation Loss 측면에서는 DP ReLU와 Dual Line이 각각 1, 2위를 차지하며 매우 우수한 수렴 성능을 보였다.
    - 특히 Dual Line은 43개의 함수 중 평균 정확도 기준 Top-5 내에 진입하였다.

## 🧠 Insights & Discussion

### 1. 파라미터의 적응성 분석
WideResNet 모델의 블록 내에서 파라미터 분포를 분석한 결과, 동일 블록 내에서도 첫 번째 활성화 함수와 두 번째 활성화 함수의 $\beta$ 값에 유의미한 차이가 발견되었다. 이는 활성화 함수가 주변 레이어의 특성에 따라 스스로 적응하고 있음을 입증하는 증거이다.

### 2. 학습률(Learning Rate)과의 관계
사용하는 활성화 함수에 따라 최적의 학습률 범위가 달라짐을 확인하였다. Dual Line과 DP ReLU는 각각 특정 범위의 학습률을 선호하며, 이는 활성화 함수의 형태가 최적화 경로에 영향을 주기 때문이다.

### 3. 강점 및 한계
- **강점**: 복잡한 지수 함수나 나눗셈 연산 없이 단순한 선형 연산과 학습 가능 파라미터만으로 성능을 크게 향상시켰다. 또한 fast.ai 리더보드 4개 중 3개에서 성능 향상을 입증하며 실용성을 증명하였다.
- **한계**: 학습 가능한 파라미터가 추가됨에 따라 학습 단계에서의 계산 시간이 일부 증가하는 병목 현상이 발생한다. 저자들은 향후 연구에서 이 계산 시간을 줄이는 것을 목표로 하고 있다.

## 📌 TL;DR

본 논문은 활성화 함수의 양수 영역 기울기와 평균 이동 파라미터를 학습 가능하게 만든 **DP ReLU**와 **Dual Line**을 제안한다. 43종의 활성화 함수를 대상으로 한 광범위한 실험 결과, 특히 **Dual Line**이 MNIST와 CIFAR10 데이터셋에서 최상위권의 정확도와 낮은 손실 값을 기록하였다. 이는 활성화 함수가 네트워크 내부 레이어의 요구사항에 맞춰 동적으로 적응할 때 모델의 성능이 극대화될 수 있음을 보여주며, 향후 새로운 활성화 함수 설계에 있어 중요한 가이드라인을 제공한다.
