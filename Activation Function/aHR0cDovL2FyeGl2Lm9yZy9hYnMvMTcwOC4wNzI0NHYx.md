# On the Compressive Power of Deep Rectifier Networks for High Resolution Representation of Class Boundaries

Senjian An, Mohammed Bennamoun, Farid Boussaid (2017)

## 🧩 Problem to Solve

본 논문은 고차원 데이터 공간에서 클래스 경계(Class Boundary)를 정밀하게 표현하기 위해 필요한 신경망의 구조적 효율성 문제를 다룬다. 특히, 부드러운 곡선 형태의 클래스 경계를 Piecewise Linear (PWL) 분류기로 근사할 때 발생하는 '차원의 저주(Curse of Dimensionality)' 문제를 해결하고자 한다.

일반적으로 고차원 공간에서 구형(Spherical) 경계와 같은 단순한 기하학적 표면을 PWL로 정밀하게 근사하려면 지수적으로 많은 수의 면(Facet)이 필요하다. 이러한 고해상도 경계를 생성하기 위해 단층(Shallow) 네트워크를 사용할 경우, 필요한 파라미터의 수와 학습 데이터의 양이 지수적으로 증가하여 과적합(Overfitting) 위험이 커지고 일반화 성능이 떨어진다. 따라서 본 논문의 목표는 깊은(Deep) Rectifier 네트워크가 단층 네트워크보다 훨씬 적은 수의 파라미터로 고해상도 경계를 표현할 수 있음을 이론적으로 증명하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 깊은 네트워크가 고차원 클래스 경계를 압축적으로 표현할 수 있는 '압축 능력(Compressive Power)'을 가지고 있음을 기하학적 관점에서 분석한 것이다.

1. **Boundary Resolution 개념 도입**: PWL 분류기의 품질을 측정하는 척도로 '경계 해상도(Boundary Resolution)', 즉 경계를 구성하는 노출된 면(Facet)의 개수를 정의하였다. 이는 분류 오류의 하한선을 추정하는 지표로 활용될 수 있다.
2. **단층 네트워크의 한계 증명**: 단층 Rectifier 네트워크가 생성할 수 있는 면의 개수가 유닛 수 $m$이 입력 차원 $d$보다 클 때 다항식 수준으로만 증가함을 보여, 고해상도 경계 구현을 위해 천문학적인 수의 유닛이 필요함을 증명하였다.
3. **깊은 네트워크의 효율성 제시**: 동일한 수의 유닛을 여러 층으로 배치했을 때 경계 해상도가 훨씬 더 빠르게 증가함을 보였으며, 특히 구형 경계 근사를 위한 구체적인 깊은 네트워크 구조를 제안하여 파라미터 효율성을 입증하였다.

## 📎 Related Works

기존 연구들은 주로 신경망의 깊이가 함수 근사(Function Approximation) 능력이나 입력 공간을 선형 영역(Linear Regions)으로 분할하는 능력에 미치는 영향에 집중해 왔다. 예를 들어, Montúfar 등(2014)은 깊은 네트워크가 단층 네트워크보다 입력 공간을 더 많은 선형 영역으로 나눌 수 있음을 보였다.

그러나 본 논문은 다음과 같은 차별점을 제시한다. 기존 연구들은 '함수' 자체의 표현력에 집중했지만, 분류 문제의 본질은 함수 값이 아니라 $f(x)=0$이 되는 '클래스 경계'를 찾는 것이다. 동일한 경계를 표현하는 함수는 무수히 많기 때문에, 단순히 함수 공간의 복잡도를 분석하는 것은 실제 분류 경계의 효율성을 설명하기에 부족하다. 본 연구는 함수 표현력이 아닌 기하학적 경계 근사 관점에서 깊이의 이점을 분석하였다는 점에서 독창성을 가진다.

## 🛠️ Methodology

### 1. PWL 분류기의 경계 해상도 정의
이진 PWL 분류기 $f(x)$의 경계는 $B_f = \{x : f(x) = 0\}$로 정의된다. 이 경계는 여러 개의 선형 면(Facet) $\Omega_k = \{x : a_k^T x + b_k = 0, f(x) = 0\}$의 합집합으로 구성된다. 본 논문에서는 중복되지 않는 면의 개수를 **경계 해상도(Boundary Resolution)**라고 정의한다.

특히 볼록(Convex) PWL 분류기는 다음과 같은 Maxout 표현식으로 나타낼 수 있다.
$$f(x) = \max_{1 \le i \le m} (w_i^T x + b_i)$$
여기서 $m$은 유닛의 수이며, 이 표현이 irreducible(더 이상 줄일 수 없는 상태)일 때 경계 해상도는 정확히 유닛 수 $m$과 일치한다.

### 2. 부드러운 볼록 경계 근사의 필요 조건
본 논문은 구형 경계를 근사하는 경우를 예로 들어, 근사 오차 $\epsilon$에 대해 필요한 면의 개수 $m$이 다음과 같은 관계를 가짐을 제시한다.
$$m \ge \left( \frac{Cd}{\epsilon} \right)^{\frac{d-1}{2}}$$
이는 차원 $d$가 증가함에 따라 필요한 면의 수가 지수적으로 증가함을 의미하며, 일반적인 볼록 경계 또한 최소한 구형 경계만큼의 해상도가 필요함을 시사한다.

### 3. 단층 Rectifier 네트워크의 한계
단층 Rectifier 네트워크 $f(x) = a^T \max(0, Wx + b) + c$가 생성할 수 있는 최대 면의 개수는 공간 분할 영역의 수 $G(d, m)$에 의해 제한된다.
$$G(d, m) = \sum_{k=0}^d \binom{m}{k}$$
분석 결과, 유닛 수 $m$이 차원 $d$보다 작을 때는 $2^m$으로 지수적으로 증가하지만, $m \ge d$가 되면 증가율이 $O(m^d)$의 다항식 수준으로 급격히 떨어진다. 따라서 고해상도 경계를 만들려면 $m$을 매우 크게 잡아야 하며, 필요한 파라미터 수 $N_{sp}$는 다음과 같이 계산된다.
$$N_{sp} = (d+1) \left( \frac{Cd}{\epsilon} \right)^{\frac{1}{2}} \frac{d-1}{e}$$

### 4. 깊은 Rectifier 네트워크의 구성 (구형 경계 근사)
본 논문은 기하학적 대칭성을 이용해 적은 파라미터로 구형 경계를 근사하는 재귀적 구조를 제안한다.

- **2D 기본 블록**: $|x|$와 $|y|$를 입력받아 회전 변환($\cos\theta, \sin\theta$)과 Rectifier를 반복 적용하여 원형을 근사한다. $\theta_k = \pi / 2^k$를 사용하여 층이 깊어질수록 원에 더 가깝게 근사하는 구조이다.
- **고차원 확장**: 2D 블록을 재귀적으로 연결하여 $d$차원 노름(Norm)을 근사한다. 먼저 $x_1, x_2$의 노름을 근사하고, 그 결과값과 $x_3$를 다시 2D 블록에 넣어 3차원 노름을 근사하는 방식을 $d-1$번 반복한다.

최종적으로 제안된 깊은 네트워크의 유닛 수 $N_d$는 다음과 같다.
$$N_d = (d-1) \{ 3 \log(d) + 1.5 \log(\epsilon^{-1}) + 3 \log(\pi) - 2.5 \}$$

## 📊 Results

본 논문은 정량적인 실험 수치보다는 이론적인 상한과 하한 분석을 통해 결과를 제시한다.

- **파라미터 효율성**: 구형 경계를 동일한 정밀도 $\epsilon$으로 근사할 때, 단층 네트워크($N_s$)와 깊은 네트워크($N_d$)의 유닛 수 비율은 다음과 같다.
$$\frac{N_s}{N_d} \approx \frac{(Cd/\epsilon)^{1/2}}{e(3 \log d + 1.5 \log \epsilon^{-1})}$$
- **결론**: 차원 $d$가 작을 때는 정밀도 $\epsilon$에 의해 결정되는 면의 개수 때문에 깊은 네트워크가 지수적으로 더 효율적이며, 차원 $d$가 클 때도 최소 $O(d^{1.5}/\log d)$ 이상의 효율성을 가진다.
- **구조적 이점**: 제안된 깊은 네트워크의 각 노드는 매우 적은 수(3~4개)의 노드와만 연결되며, 유닛당 최대 5개의 파라미터만 가지는 반면, 단층 네트워크의 유닛은 $(d+1)$개의 파라미터를 가져야 한다.

## 🧠 Insights & Discussion

본 논문은 딥러닝의 '깊이'가 단순히 함수 표현력을 높이는 것이 아니라, **데이터의 기하학적 대칭성을 활용해 복잡한 경계를 압축적으로 표현**하게 함으로써 일반화 성능을 높인다는 통찰을 제공한다.

- **강점**: 기존의 '선형 영역 개수' 분석에서 한 단계 나아가 '클래스 경계의 해상도'라는 새로운 관점을 제시하여, 왜 깊은 모델이 적은 파라미터로도 복잡한 결정 경계를 더 잘 학습하는지를 이론적으로 뒷받침하였다.
- **한계**: 본 분석은 구형(Spherical) 경계라는 특수한 사례를 중심으로 전개되었다. 실제 데이터의 클래스 경계가 구형이나 단순 볼록 형태가 아닐 경우, 제안된 재귀적 구조가 얼마나 유효할지에 대해서는 명시적으로 언급되지 않았다.
- **비판적 해석**: 이론적으로는 깊은 네트워크가 훨씬 효율적이지만, 실제 학습 과정에서 이러한 최적의 가중치($\cos\theta, \sin\theta$ 등)를 경사하강법(SGD)으로 찾아낼 수 있는지는 별개의 문제이다. 즉, '표현 가능성(Expressibility)'과 '학습 가능성(Learnability)' 사이의 간극에 대한 논의가 추가될 필요가 있다.

## 📌 TL;DR

이 논문은 고차원 공간에서 부드러운 클래스 경계를 표현할 때, 단층 네트워크는 '차원의 저주'로 인해 지수적으로 많은 파라미터가 필요하지만, **깊은 Rectifier 네트워크는 기하학적 대칭성을 이용해 이를 매우 효율적으로 압축 표현할 수 있음**을 이론적으로 증명하였다. 이는 딥러닝의 깊이가 단순한 성능 향상을 넘어, 모델의 복잡도를 낮추고 일반화 능력을 높이는 핵심 기제임을 시사한다.
