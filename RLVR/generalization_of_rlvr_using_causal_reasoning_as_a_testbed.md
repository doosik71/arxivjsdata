# Generalization of RLVR using Causal Reasoning as a Testbed

Brian Lu, Hongyu Zhao, Shuo Sun, Hao Peng, Rui Ding, Hongyuan Mei (2025)

## 🧩 Problem to Solve

본 논문은 검증 가능한 보상을 이용한 강화학습(Reinforcement Learning with Verifiable Rewards, RLVR)이 거대 언어 모델(LLM)의 복잡한 추론 작업에서 어떠한 조건 하에 강건한 일반화(Generalization) 성능을 보이는지 분석하는 것을 목표로 한다. 최근 RLVR은 수학, 코드 생성 등 정답이 명확한 도메인에서 큰 성과를 거두었으나, 학습 데이터 범위를 벗어난 새로운 문제에 대해 모델이 어떻게 일반화하는지에 대한 이해는 여전히 부족한 상태이다.

이를 위해 저자들은 인과 그래프(Causal Graphical Models) 상의 확률적 추론이라는 구조화된 환경을 테스트베드로 설정하였다. 인과 추론은 '인과 사다리(Causal Ladder)'라고 불리는 계층적 구조(연관, 개입, 반사실적 추론)를 가지고 있어, 동일 레벨 내의 일반화(Within-level generalization)뿐만 아니라 상위 또는 하위 레벨로의 전이 학습(Across-level generalization)을 정밀하게 측정하기에 적합하기 때문이다.

## ✨ Key Contributions

본 연구의 핵심 기여는 RLVR의 일반화 능력이 모델의 초기 추론 역량(Initial reasoning competence)에 강하게 의존한다는 점을 실험적으로 밝혀낸 것이다. 주요 설계 아이디어와 발견점은 다음과 같다.

- **RLCausal 데이터셋 구축**: 자연어 시나리오의 모호성을 제거하고 순수 추론 능력을 측정하기 위해, 완전히 명시된 구조적 인과 모델(SCM)과 쿼리를 포함하는 합성 데이터셋을 구축하였다.
- **일반화 조건의 규명**: RLVR이 SFT(Supervised Fine-Tuning)보다 뛰어난 일반화 성능을 보이기 위해서는 모델의 크기가 일정 수준 이상($\ge 7\text{B}$)이어야 하며, 기반 모델이 최소한의 추론 성공률을 가지고 있어야 함을 보였다.
- **추론 전략의 최적화 분석**: RLVR이 모델의 주변화 전략(Marginalization strategy)을 효율적인 '점진적 주변화(Incremental marginalization)' 방식으로 변화시키고, 확률 유도 과정에서의 추상적 오류를 줄여 복잡한 쿼리에 대한 정답률을 높인다는 점을 분석하였다.

## 📎 Related Works

기존 연구들은 RLVR이 수학 및 코드 생성 분야에서 추론 능력을 크게 향상시킨다는 점을 보여주었으며, SFT와 RL의 일반화 성능 차이를 비교하는 연구들이 진행되어 왔다. 특히 `CLadder`와 같은 벤치마크가 인과 추론 능력을 평가하기 위해 사용되었다.

본 논문은 기존 접근 방식과 다음과 같은 차별점을 가진다. 첫째, `CLadder`가 자연어 시나리오를 사용하여 언어 이해 능력이 함께 측정되는 것과 달리, 본 연구는 SCM 파라미터를 직접 제공하여 오직 '추론 및 계산' 능력만을 분리하여 측정한다. 둘째, 단순히 성능 향상을 보고하는 것이 아니라, 모델 크기와 쿼리 레벨이라는 두 가지 축을 통해 RLVR의 일반화 경계를 체계적으로 탐색하였다.

## 🛠️ Methodology

### 전체 파이프라인 및 작업 정의

입력값 $x$는 인과 그래프의 구조, 변수 정의, 메커니즘 파라미터(CPT) 및 쿼리로 구성된다. 모델은 이에 대해 중간 추론 단계(Thought process)를 거쳐 최종 확률 분포 $\hat{p}$를 출력한다.

### 인과 추론의 세 가지 레벨

본 연구는 인과 사다리에 따라 세 가지 쿼리 수준을 정의한다.

1. **연관(Association)**: 관찰된 값 $v_j$가 주어졌을 때 $v_i$의 분포 $p(v_i | v_j = v_j)$를 구하는 작업이다.
2. **개입(Intervention)**: 외부 개입으로 $v_j$를 $c$로 고정했을 때 $v_i$의 분포 $p(v_i | \text{do}(v_j = c))$를 구하는 작업이다.
3. **반사실적(Counterfactual)**: $v_k = v_k$를 관찰한 세계에서 $v_j$를 $c$로 고정했다면 $v_i$가 어떻게 되었을지 $p(v_i (\text{do}(v_j = c)) | v_k = v_k)$를 구하는 작업이다.

### 학습 목표 및 손실 함수

**RLVR (Reinforcement Learning)**:
모델이 생성한 답변 $\hat{p}_y$와 정답 $p^*_x$ 사이의 정확도와 출력 형식을 보상으로 사용하여 최적화한다. 보상 함수 $r(y)$는 다음과 같이 정의된다.
$$r(y) = 0.8 \cdot r^{ans}(\hat{p}_y, p^*_x) + 0.2 \cdot r^{format}(y)$$
여기서 $r^{ans}$는 Total Variation Distance(TVD)를 이용하여 계산하며, $D(\hat{p}_y, p^*_x) < 0.01$일 때 1, 그렇지 않으면 0을 부여한다. TVD의 정의는 다음과 같다.
$$D(p, q) := \frac{1}{2} \int |p(x) - q(x)| dx$$
학습 알고리즘으로는 GRPO와 DAPO 변형 모델을 사용하였다.

**SFT (Supervised Fine-Tuning)**:
정답 분포 $p^*_x$에 대한 조건부 가능도(Conditional Likelihood)를 최대화하는 방식으로 학습한다.
$$\mathbb{E}_{x \sim D} \log p_\theta(y^*_x | x)$$

### 실험 설계

- **모델 규모**: Qwen-2.5-Instruct 모델의 3B, 7B, 32B 버전을 사용하였다.
- **난이도 측정**: 쿼리에 영향을 주는 관련 서브그래프의 크기 $|V_{rel}|$를 통해 복잡도를 측정하여 Small, Medium, Large로 구분하였다.

## 📊 Results

### 주요 정량적 결과

- **레벨 내 일반화 (Within-level)**: 7B 이상의 모델에서 RLVR은 연관 및 개입 쿼리에 대해 SFT보다 유의미하게 높은 정확도를 보였다. 반면, 3B 모델과 모든 규모의 반사실적 쿼리에서는 RLVR의 효과가 나타나지 않았거나 SFT보다 저조했다.
- **레벨 간 일반화 (Across-level)**: 학습 시 보지 못한 다른 레벨의 쿼리를 평가했을 때, 7B 이상의 모델에서 RLVR이 SFT보다 훨씬 강력한 일반화 성능을 보였다.
- **복잡도에 따른 성능**: 쿼리의 복잡도 $|V_{rel}|$가 증가할수록 SFT의 성능은 급격히 하락하는 반면, RLVR은 복잡한 쿼리에서도 상대적으로 높은 정확도를 유지하였다.
- **정밀도**: RLVR로 학습된 모델은 SFT 모델보다 정답에 더 가까운(TVD가 더 작은) 정밀한 값을 출력하는 경향을 보였다.

### 정성적 분석 결과

- **3B 모델의 실패**: 3B 모델은 RLVR 학습 후 오히려 명시적인 주변화(Marginalization) 시도를 포기하고 정답을 직접 예측하려는 경향(Regression)을 보였다.
- **전략의 변화**: 7B, 32B 모델의 경우, RLVR 학습 후 'Brute-force' 방식의 합산에서 효율적인 'Incremental marginalization(점진적 주변화)' 방식으로 전략이 전이됨이 확인되었다.

## 🧠 Insights & Discussion

본 논문은 RLVR의 성공이 단순한 데이터 학습이 아니라, 모델이 이미 가지고 있는 '추론 잠재력'을 효율적으로 끌어올리는 과정임을 시사한다.

**1. Reasoning Prior의 중요성 (Cold Start Problem)**
RLVR은 기반 모델의 초기 추론 성공률이 너무 낮을 경우 효과가 없다. 3B 모델이 실패한 이유는 학습 과정에서 올바른 추론 경로를 통해 보상을 받을 확률이 극히 낮았기 때문이며, 이는 RLVR의 고질적인 'Cold Start' 문제와 일치한다.

**2. 추론 전략의 정교화**
RLVR은 단순히 정답을 외우게 하는 SFT와 달리, 모델이 확률 유도 과정에서 범하는 추상적 오류(예: 독립성 가정 오류, 개입과 관찰의 혼동)를 줄이고, 계산 단계에서의 실수를 최소화하도록 유도한다. 특히 복잡한 문제일수록 점진적 주변화 전략을 채택하게 함으로써 계산 오류의 가능성을 낮춘다.

**3. 반사실적 추론의 한계**
반사실적 쿼리는 '가정-관찰-개입'의 복잡한 단계가 필요하며, 모델이 Twin-network를 구성하는 등의 고차원적 추론을 수행해야 한다. 본 연구에서 시스템 프롬프트에 힌트를 제공했음에도 성능 향상이 미비했던 점은, 현재의 RLVR 방식만으로는 매우 높은 수준의 추론 도약을 이끌어내기에 한계가 있음을 보여준다.

## 📌 TL;DR

본 연구는 인과 추론이라는 정밀한 테스트베드를 통해 RLVR의 일반화 메커니즘을 분석하였다. RLVR은 기반 모델이 일정 수준 이상의 추론 역량(Reasoning Prior)을 갖추었을 때, SFT보다 훨씬 강력한 일반화 성능을 보이며, 이는 효율적인 주변화 전략의 습득과 추론 오류의 감소를 통해 이루어진다. 이 결과는 향후 LLM의 추론 능력을 강화하기 위해 무조건적인 RL 적용보다는 기반 모델의 역량과 도메인 특성에 맞는 전략적 접근이 필요함을 시사한다.
