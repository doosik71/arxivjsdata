# Studying the Korean Word-Chain Game with RLVR: Mitigating Reward Conflicts via Curriculum Learning

Donghwan Rho (2025)

## 🧩 Problem to Solve

본 논문은 Reinforcement Learning with Verifiable Rewards (RLVR)를 활용하여 한국어 끝말잇기 게임을 수행하는 대규모 언어 모델(LLM)을 학습시키는 것을 목표로 한다.

전형적인 RLVR 연구들은 수학이나 코딩과 같은 논리적 퍼즐에 집중해 왔으나, 언어 중심적인 퍼즐, 특히 한국어와 같은 비영어권 언어의 퍼즐에 대한 연구는 부족한 실정이다. 한국어 끝말잇기는 단순히 마지막 글자로 시작하는 단어를 제시하는 것을 넘어, '두음법칙'이라는 복잡한 음운 규칙과 표준국어대사전 기반의 명사 사용 제약 등 여러 규칙을 동시에 준수해야 하는 과제이다.

이 과정에서 연구자는 서로 다른 규칙들로부터 유도된 보상(Reward)들이 서로 충돌(Conflict)하여, 모델이 더 어렵고 복잡한 규칙(예: 두음법칙)을 학습하지 못하고 쉬운 규칙에만 안주하게 되는 현상을 발견하였다. 따라서 본 논문은 이러한 보상 충돌을 완화하고 모델의 추론 능력을 향상시키기 위한 학습 전략을 제안한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Curriculum Learning(커리큘럼 학습)**을 도입하여 보상 충돌을 해결하는 것이다. 일반적인 커리큘럼 학습이 쉬운 단계에서 어려운 단계로 나아가는 것과 달리, 본 연구에서는 **더 어려운 규칙(두음법칙)을 먼저 학습하게 함으로써** 이후 전체 규칙을 학습할 때 발생할 수 있는 보상 간의 간섭을 최소화한다.

주요 기여 사항은 다음과 같다:

1. 한국어 끝말잇기 규칙에서 유도된 보상들이 내재적으로 충돌할 수 있음을 규명하였다.
2. 단순한 RLVR 적용으로는 모델 학습이 효과적이지 않음을 보였으며, 제안한 커리큘럼 학습 스킴이 보상 충돌을 완화하여 학습 효율을 높임을 입증하였다.
3. Initial-Sound Rule Forcing (ISR) 및 Other Syllable (OS) 패널티와 같은 구체적인 보상 설계 방안을 제시하였다.

## 📎 Related Works

**1. Reinforcement Learning with Verifiable Rewards (RLVR)**
RLVR은 학습된 보상 모델(Reward Model) 대신, 정답 여부를 명확히 확인할 수 있는 검증 가능한 보상(Verifiable Rewards, 주로 이진 보상)을 사용하여 LLM의 추론 능력을 강화하는 방식이다. 주로 수학, 코딩 및 Sudoku와 같은 논리 퍼즐에 적용되었으나, 본 논문은 이를 규칙 기반이면서도 정답이 열린(Open-ended) 한국어 끝말잇기에 적용하였다.

**2. Curriculum Learning 및 보상 충돌**
복잡한 보상 함수를 가진 RL 태스크에서 여러 보상이 동시에 존재할 때 서로 충돌하는 현상이 발생한다. 기존 연구들은 주로 문제의 난이도를 조절하거나 목표 상태까지의 거리를 점진적으로 늘리는 방식을 사용하였다. 본 논문은 이와 차별되게, 성공적인 궤적(Trajectory)을 찾기 위해 쉬운 보상을 먼저 학습하는 일반적 접근과 달리, 규칙 습득을 위해 **더 어려운 보상을 먼저 학습**시키는 전략을 취한다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조

본 연구는 `beomi/Qwen2.5-7B-Instruct-kowiki-qa` 모델을 베이스 모델로 사용하며, LoRA(Low-Rank Adaptation)와 GRPO(Group Relative Policy Optimization) 알고리즘을 통해 학습을 진행한다.

### 보상 함수 설계 (Reward Shaping)

모델이 준수해야 할 규칙은 다음과 같다: (i) 이전 단어의 마지막 음절로 시작, (ii) 두음법칙 적용 가능, (iii) 표준국어대사전의 명사여야 함, (iv) 중복 단어 사용 금지.

**1. Baseline Reward**
가장 단순한 형태의 보상 체계이다.

- (a) 마지막 음절과 시작 음절이 일치하면 $+1$
- (b) (a)를 만족하고 명사라면 $+1$
- (c) 중복 단어 사용 시 $-1$

**2. Initial-Sound Rule Forcing (ISR)**
두음법칙이 적용 가능한 상황에서 이를 무시하고 기본 규칙만 따르는 경우 보상을 낮추어, 두음법칙 학습을 강제한다.

- (a-i) 기본 일치 시 $+1$
- (a-ii) 두음법칙 적용 후 일치 시 $+1$
- (b) (a-i) 또는 (a-ii)를 만족하고 명사라면 $+1$
- (c) 중복 단어 사용 시 $-1$
- *핵심은 두음법칙 적용이 가능한 음절일 때, 반드시 이를 적용해야 전체 보상을 얻을 수 있도록 강제하는 것이다.*

**3. Other Syllable (OS) Penalty**
모델이 이전 단어의 마지막 음절이 아닌, 다른 음절로 시작하는 단어를 생성하는 오류를 방지하기 위한 패널티이다.

- (d) 답변이 이전 단어의 마지막 음절이 아닌 다른 음절로 시작할 경우 $-0.5$ 추가.

### Curriculum Learning (Data Reordering, DR)

보상 충돌을 해결하기 위해 2단계 학습 과정을 거친다.

- **1단계 (Hard Rule First):** 두음법칙이 적용되는 예제들로만 구성된 데이터셋으로 학습한다. 모델이 초기에 두음법칙이라는 어려운 규칙을 먼저 체득하게 한다.
- **2단계 (Full Dataset):** 모든 예제를 포함한 전체 데이터셋으로 학습한다. 이미 1단계에서 두음법칙을 익혔으므로, 기본 규칙과의 보상 충돌 없이 안정적으로 학습이 가능하다.

## 📊 Results

### 실험 설정

- **데이터셋:** 표준국어대사전에서 추출한 명사들을 활용하여 생성.
- **평가 방법:** 사전(Dictionary)과의 대국을 통해 성능 측정.
- **평가 지표:**
  - Average Model Turns: 모델이 규칙을 어기지 않고 얼마나 오래 버텼는가.
  - Win Rate: 사전이 더 이상 단어를 제시할 수 없어 모델이 승리한 비율.

### 주요 결과

1. **성능 향상:** Baseline 대비 ISR, DR, OS 방법론을 결합했을 때 평균 턴 수와 승률이 모두 증가하였다. 특히 OS 패널티는 평균 턴 수를 소폭 상승시키는 효과가 있었다.
2. **두음법칙 습득 속도:** Baseline이나 ISR 단독 적용 시에는 두음법칙 습득 속도가 매우 느리거나 실패하는 경우가 많았으나, DR(커리큘럼 학습)을 적용했을 때 매우 빠르게 규칙을 습득함을 확인하였다.
3. **길이 일반화 (Length Generalization):** 학습 데이터의 최대 체인 길이는 7단어(4턴)였으나, 실제 테스트에서는 평균 턴 수가 이를 훨씬 상회하는 결과가 나타나, 모델이 학습 데이터 이상의 길이에서도 규칙을 유지하는 일반화 능력을 보였다.

### 실패 사례 분석

- **중복 단어 금지 규칙:** 이 규칙은 학습 데이터에서 중복 사례가 드물게 발생하여 모델이 충분히 학습하지 못한 것으로 분석되었으며, 향후 과제로 남겨두었다.
- **기타 오류:** OS 패널티 적용 시 '다른 음절로 시작하는 오류'는 줄어들었으나, '기본 규칙(i) 위반' 사례가 소폭 증가하는 경향이 관찰되었다.

## 🧠 Insights & Discussion

**강점 및 성과**
본 연구는 RLVR 환경에서 서로 다른 보상 간의 충돌이 학습을 방해할 수 있음을 실증적으로 보여주었으며, 이를 해결하기 위해 "어려운 규칙 우선 학습"이라는 역설적인 커리큘럼 전략이 유효함을 증명하였다. 이는 LLM이 단순한 패턴 매칭을 넘어 언어의 음운론적 규칙과 같은 복잡한 제약 조건을 RL을 통해 학습할 수 있음을 시사한다.

**한계 및 비판적 해석**

1. **데이터 편향:** 중복 단어 규칙 학습 실패에서 알 수 있듯, RL의 성능은 보상 함수뿐만 아니라 학습 과정에서 모델이 경험하는 샘플의 분포에 크게 의존한다.
2. **보상 간의 Trade-off:** OS 패널티가 다른 규칙 위반을 소폭 증가시킨 점은, 특정 오류를 억제하는 보상이 모델의 전체적인 정책 공간(Policy Space)에 영향을 주어 다른 형태의 오류를 유발할 수 있음을 보여준다.
3. **일반화 범위:** 한국어 끝말잇기라는 특정 도메인에서의 성공이 다른 언어의 복잡한 규칙 기반 퍼즐에도 동일하게 적용될지는 추가적인 연구가 필요하다.

## 📌 TL;DR

본 논문은 한국어 끝말잇기 게임을 통해 LLM의 추론 능력을 강화하는 RLVR 연구이다. 특히 기본 규칙과 두음법칙 사이의 보상 충돌 문제를 해결하기 위해, **어려운 규칙을 먼저 학습시키는 커리큘럼 전략(DR)**과 **두음법칙 강제 보상(ISR)**, **잘못된 음절 시작 패널티(OS)**를 제안하였다. 실험 결과, 제안 방법론은 모델의 규칙 준수 능력을 크게 향상시켰으며, 학습 데이터보다 더 긴 대화에서도 규칙을 유지하는 길이 일반화 성능을 보였다. 이 연구는 비영어권 언어의 특수성을 반영한 퍼즐 태스크가 LLM의 추론 능력을 정교하게 다듬는 효율적인 도구가 될 수 있음을 시사한다.
