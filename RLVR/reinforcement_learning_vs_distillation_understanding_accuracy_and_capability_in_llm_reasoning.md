# Reinforcement Learning vs. Distillation: Understanding Accuracy and Capability in LLM Reasoning

Minwu Kim, Anubhav Shrestha, Safal Shrestha, Aadim Nepal, Keith Ross (2025)

## 🧩 Problem to Solve

본 논문은 대규모 언어 모델(LLM)의 추론 능력 향상을 위해 널리 사용되는 두 가지 방법론인 Verifiable Rewards를 이용한 강화학습(Reinforcement Learning with Verifiable Rewards, RLVR)과 지식 증류(Distillation)가 모델의 성능에 미치는 서로 다른 영향을 분석한다.

특히, 단순히 한 번의 시도에서 정답을 맞힐 확률인 **정확도(Accuracy, pass@1)**와 모델의 출력 분포 내에 정답이 존재할 확률인 **능력(Capability, pass@k)**의 차이에 주목한다. 기존 연구들은 RLVR이 정확도는 높이지만 능력은 확장하지 못하는 반면, 증류는 두 가지 모두를 개선할 수 있다고 주장해 왔다. 본 연구의 목표는 RLVR이 왜 능력을 확장하지 못하는지, 그리고 증류가 어떤 조건에서 능력을 확장시키는지에 대한 메커니즘을 심층적으로 규명하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여 및 직관은 다음과 같다.

1. **RLVR의 능력 확장 실패 원인 규명**: RLVR이 쉬운 문제의 정확도는 크게 높이지만, 정작 가장 어려운 문제의 정확도는 오히려 떨어뜨리는 '어려운 문제의 희생(sacrificing-difficult-problems)' 현상이 발생함을 발견하였다.
2. **RLVR의 응답 품질 변화 분석**: RLVR이 단순히 기존 정답의 생성 확률을 높이는 것이 아니라, 기존 분포에 없던 고품질의 응답을 생성할 수 있게 함을 self-distillation 실험을 통해 증명하였다. 또한, 이러한 품질 향상이 응답 길이(Length)나 성찰 키워드(Reflection keywords)의 증가와는 무관함을 밝혔다.
3. **증류를 통한 능력 확장의 조건 제시**: 증류가 항상 능력을 확장하는 것은 아니며, 새로운 도메인 지식(New Knowledge)이 주입될 때만 능력이 확장된다는 가설을 제시하였다. 단순히 추론 패턴(Reasoning patterns)만 학습하는 증류는 RLVR과 유사하게 어려운 문제를 희생시키며 정확도만 높이는 경향을 보인다.

## 📎 Related Works

본 논문은 LLM의 추론 성능을 높이기 위한 두 가지 주요 흐름을 다룬다.

* **RLVR 기반 학습**: DeepSeek-R1이나 OpenAI o1과 같이 정답 여부에 기반한 보상을 통해 모델이 스스로 긴 사고 체인(Chain-of-Thought, CoT)을 형성하도록 유도하는 방식이다. 최근 RLVR이 실제로 새로운 능력을 창출하는지, 아니면 잠재된 능력을 증폭시키는 것인지에 대한 학술적 논쟁이 있으며, 본 논문은 전형적인(Typical) 설정에서 RLVR이 능력을 확장하지 못한다는 관점에 무게를 둔다.
* **지식 증류(Distillation)**: 강력한 Teacher 모델의 CoT 응답을 Student 모델이 학습하는 방식이다. 기존 연구들은 증류가 정확도와 능력을 모두 높인다고 보고하였으나, 본 논문은 여기서 '지식 주입'과 '추론 패턴 학습'의 효과를 분리하여 분석함으로써 기존 접근 방식의 한계를 지적한다.

## 🛠️ Methodology

### 1. 정확도와 능력의 정의

본 논문은 모델 $M$과 데이터셋 $D$에 대해 다음과 같이 지표를 정의한다.

* **정확도(Accuracy)**: 모델이 단일 시도에서 문제를 맞힐 확률의 평균이다.
$$\text{Acc}(M) = \frac{1}{N} \sum_{i \in D} p_i^M$$
* **능력(Capability, pass@k)**: $k$번의 독립적인 시도 중 적어도 한 번 정답을 맞힐 확률의 평균이다.
$$\text{Cap}_k(M) = \frac{1}{N} \sum_{i \in D} (1 - (1 - p_i^M)^k)$$

여기서 $p_i^M$은 모델 $M$이 문제 $i$를 맞힐 확률이다.

### 2. RLVR 분석 파이프라인

RLVR의 영향을 분석하기 위해 모델의 출력 분포를 기반으로 문제를 난이도별 Bin(예: 정답 생성 횟수 0회, 1-4회 등)으로 나누어 분석한다. 또한, GRPO(Group Relative Policy Optimization) 알고리즘의 특성상, 배치 내 모든 응답이 오답인 어려운 문제는 가중치 업데이트에 기여하지 못하며, 제한된 모델 용량으로 인해 쉬운 문제에 최적화되는 과정에서 어려운 문제의 정답 확률이 낮아지는 메커니즘을 분석한다.

### 3. Self-Distillation 및 Teacher Distillation

* **Self-Distillation**: 모델 자신의 정답 응답만을 추출하여 SFT(Supervised Fine-Tuning)를 수행한다. $\text{Base} \leftarrow \text{Base}$, $\text{RL} \leftarrow \text{RL}$, $\text{Base} \leftarrow \text{RL}$의 세 가지 설정을 통해 RLVR이 생성하는 응답의 질적 차이를 검증한다.
* **Teacher Distillation**: 지식 주입의 효과를 분리하기 위해 두 종류의 증류 모델을 비교한다.
  * **DeepSeek model**: 방대한 양의 외부 데이터를 통해 새로운 지식과 추론 패턴을 모두 학습한 모델.
  * **Reasoning-only model**: Base 모델이 이미 한 번이라도 맞힌 문제(In-distribution)에 대해서만 Teacher(QwQ-32B)의 응답을 학습하여, 새로운 지식 없이 추론 패턴만 학습한 모델.

## 📊 Results

### 1. RLVR의 난이도별 성능 변화

실험 결과, RLVR은 Base 모델의 성공률이 중간 정도인 Bin([17-64], [65-128])에서는 정확도를 비약적으로 높였으나, 성공률이 0에 가까운 최하위 Bin에서는 개선이 거의 없거나 오히려 성능이 하락하는 양상을 보였다. 이는 RLVR이 전체적인 능력을 확장하기보다 기존에 가능했던 문제들의 성공 확률을 높이는 데 집중함을 의미한다.

### 2. 응답 품질 및 정성적 분석

Self-distillation 실험에서 $\text{Base} \leftarrow \text{RL}$ 설정이 $\text{Base} \leftarrow \text{Base}$보다 훨씬 높은 테스트 정확도 향상을 보였다. 이는 RLVR이 단순히 확률을 높인 것이 아니라, 기존에 없던 **고품질 응답**을 생성할 수 있게 했음을 시사한다. 반면, 정성 분석 결과 RLVR 이후 응답 길이가 길어지거나 "Wait", "Alternatively" 같은 성찰 키워드가 증가하는 현상은 관찰되지 않았으며, 오히려 더 직접적인 응답을 생성하는 경향을 보였다.

### 3. 증류와 능력 확장

Pass@k 실험 결과, 새로운 지식이 주입된 DeepSeek 모델은 $k$값이 커질수록 Base 모델과의 격차가 유지되거나 벌어지며 능력이 확장되었음을 보여주었다. 반면, 추론 패턴만 학습한 Reasoning-only 모델은 작은 $k$에서는 정확도가 높았으나, $k$가 커질수록 Base 모델의 성능 곡선과 수렴하거나 역전되는 현상이 발생하였다. 이는 지식 주입 없이는 증류만으로 능력을 확장하는 것이 불가능하며, 오히려 RLVR처럼 어려운 문제를 희생시키는 양상이 나타남을 보여준다.

## 🧠 Insights & Discussion

본 논문은 LLM 추론 학습에서 발생하는 매우 중요한 트레이드-오프를 밝혀냈다.

첫째, RLVR의 성능 향상은 '능력의 확장'이 아니라 '분포의 집중'에 가깝다. GRPO와 같은 정책 경사 알고리즘은 보상을 받을 가능성이 높은 샘플에 가중치를 두기 때문에, 모델은 효율적으로 정답을 맞힐 수 있는 문제에 최적화된다. 이 과정에서 모델 용량이 제한적일 경우, 정답 확률이 매우 낮은 어려운 문제들에 할당되었던 확률 질량이 쉬운 문제로 이동하며 결과적으로 어려운 문제의 해결 능력은 더욱 상실될 수 있다.

둘째, 기존의 '긴 사고 체인(Long CoT) $\rightarrow$ 고품질 추론'이라는 일반적인 믿음이 소형 모델(1.5B, 3B)에서는 반드시 성립하지 않음을 보여주었다. RLVR을 통해 품질이 향상되었음에도 불구하고 응답 길이나 특정 키워드가 변하지 않았다는 점은, 표면적인 특징(Surface-level traits)만으로 추론 품질을 평가하는 기존 방식의 위험성을 경고한다.

셋째, 증류의 핵심은 '패턴'이 아니라 '지식'에 있다. 추론 방식(Reasoning pattern)은 일종의 최적화 전략이지만, 문제를 풀기 위한 근본적인 재료는 지식이다. 지식 없이 패턴만 학습하는 것은 이미 알고 있는 내용을 더 효율적으로 출력하게 만들 뿐, 풀지 못하던 문제를 풀게 만드는 '능력 확장'으로는 이어지지 않는다.

## 📌 TL;DR

본 연구는 RLVR이 정확도는 높이지만 능력을 확장하지 못하는 이유가 **쉬운 문제의 성능을 높이기 위해 어려운 문제를 희생시키는 메커니즘** 때문임을 밝혀냈다. 또한, RLVR이 응답 길이를 늘리지 않고도 내부적인 응답 품질을 개선함을 증명하였으며, 증류를 통한 능력 확장은 오직 **새로운 도메인 지식이 주입될 때만 가능함**을 확인하였다. 이 결과는 향후 LLM의 근본적인 추론 능력을 확장하기 위해서는 단순한 보상 최적화나 패턴 모방을 넘어, 정교한 지식 주입 전략이 필수적임을 시사한다.
