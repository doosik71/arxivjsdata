# Modeling the Uncertainty in Electronic Health Records: a Bayesian Deep Learning Approach

Riyi Qiu, Yugang Jia, Mirsad Hadzikadic, Michael Dulin, Xi Niu, Xin Wang (2019)

## 🧩 Problem to Solve

본 논문은 전자 건강 기록(Electronic Health Records, EHR)을 활용한 딥러닝 기반 예측 모델의 투명성과 신뢰성 부족 문제를 해결하고자 한다. 딥러닝 모델은 EHR 데이터의 복잡한 패턴을 학습하여 높은 예측 성능을 보이지만, 내부 동작 과정을 알 수 없는 '블랙박스' 구조로 인해 의료 현장의 실제 의사결정 과정에서 신뢰받기 어렵다. 

특히 의료 데이터는 센서의 결함, 잘못된 입력, 누락된 값의 대체(imputation) 등으로 인해 데이터 노이즈가 빈번하게 발생하며, 이는 모델의 예측 불확실성으로 이어진다. 따라서 모델이 내린 예측값이 얼마나 확신할 수 있는지를 나타내는 불확실성(Uncertainty)을 정량화함으로써, 잘못된 예측으로 인한 의료 사고(AI Failure)를 방지하고 모델의 신뢰도를 높이는 것이 본 연구의 핵심 목표이다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 Bayesian Neural Network(BNN)를 사용하여 데이터 노이즈로 인해 발생하는 '입력 데이터 의존적 불확실성'인 heteroscedastic aleatoric uncertainty를 모델링하는 것이다. 

주요 기여 사항은 다음과 같다.
1. EHR 예측 작업에 Convolutional Neural Network(CNN)와 Bayesian learning 기법을 결합하여, 각 예측 사례별로 서로 다른 불확실성을 계산하는 모델을 제안하였다.
2. 계산된 불확실성이 모델의 예측 성능과 밀접한 상관관계가 있음을 입증하고, 불확실성이 높은 환자군이 모델 성능을 저하시키는 요인임을 확인하였다.
3. 불확실성과 예측 확률의 분포를 분석하여, 데이터 수집 빈도를 높이거나 노이즈를 줄였을 때 성능 향상 이득이 가장 큰 특정 환자 그룹을 식별하는 방법을 제시하였다.

## 📎 Related Works

기존의 딥러닝 기반 EHR 분석 연구들은 주로 예측 정확도를 높이는 데 집중하였으며, 1-D CNN 등을 통해 시계열 데이터의 국소적 상관관계를 학습하는 방식을 취해왔다. 그러나 이러한 방식들은 모델이 예측 결과에 대해 얼마나 확신하는지에 대한 정보를 제공하지 않는다.

논문에서는 불확실성을 두 가지 유형으로 구분하여 설명한다.
- **Aleatoric Uncertainty**: 데이터 자체의 노이즈(잘못된 레이블, 정밀하지 못한 측정 등)로 인해 발생하며, 다시 입력값과 무관한 homoscedastic과 입력값에 따라 변하는 heteroscedastic으로 나뉜다.
- **Epistemic Uncertainty**: 모델 구조나 파라미터의 부족으로 인해 발생하며, 더 많은 데이터를 학습함으로써 줄일 수 있다.

기존의 BNN 연구들이 주로 컴퓨터 비전이나 자연어 처리 분야에 적용되었던 것과 달리, 본 연구는 이를 EHR의 ICU 사망률 예측이라는 의료 도메인에 적용하고 특히 heteroscedastic aleatoric uncertainty를 캡처하는 데 집중함으로써 차별점을 둔다.

## 🛠️ Methodology

### 전체 시스템 구조
본 모델은 EHR 데이터의 시간적 상관관계를 학습하기 위해 1-D CNN을 기본 아키텍처로 사용하며, 그 위에 Bayesian 학습 프레임워크를 얹어 불확실성을 추정한다.

### heteroscedastic Aleatoric Uncertainty 모델링
일반적인 딥러닝 모델은 입력 $x$를 통해 가중치 $W$를 거쳐 확률 벡터 $\hat{p}$를 예측한다:
$$\hat{p} = \text{Softmax}(Wx)$$

본 제안 모델은 가중치 $W$에 노이즈 항 $k$를 추가하여 Gaussian 분포를 정의한다. 이때 노이즈의 표준편차 $\sigma_x$는 고정된 값이 아니라, 입력 $x$에 따라 결정되는 별도의 dense layer를 통해 계산된다:
$$\hat{x} = (W + k)x = Wx + kx, \quad kx \sim N(0, \sigma_x^2 I)$$
여기서 $I$는 단위 행렬이다. 최종 예측 확률 $\hat{p}$는 노이즈가 섞인 출력 $\hat{x}$를 통해 계산된다:
$$\hat{p} = \text{Softmax}(\hat{x})$$

### 학습 절차 및 손실 함수
Gaussian 분포를 직접 통합하여 cross entropy loss를 계산하는 분석적 해가 없기 때문에, 본 논문은 Monte Carlo(MC) 시뮬레이션을 통해 이를 근사한다. $T$번의 MC 샘플링을 수행하며, 각 샘플 $\hat{x}_t$는 다음과 같이 생성된다:
$$\hat{x}_t = Wx + \sigma_x \epsilon_t, \quad \epsilon_t \sim N(0, I)$$

이에 따른 Bayesian categorical cross entropy 손실 함수는 다음과 같이 정의된다:
$$\text{Loss} = \log \frac{1}{T} \sum_{t=1}^{T} \exp \left( \hat{x}_t - \log \sum_{c} \exp \hat{x}_{t,c} \right)$$

모델은 일반적인 categorical cross entropy와 위에서 정의한 Bayesian 손실 함수, 그리고 $\sigma_x$에 대한 regularizer의 가중 합을 최소화하는 방향으로 최적화된다. 특히 $\sigma_x^2$ (aleatoric uncertainty)는 지도 학습 대상이 아니며, 전체 손실 함수를 최소화하는 과정에서 함께 학습된다.

## 📊 Results

### 실험 설정
- **데이터셋**: MIMIC-III (ICU 입원 환자 21,139건)
- **작업**: 입원 후 48시간 데이터를 이용한 ICU 내 사망률(Mortality) 예측
- **특징**: 17개 기본 특징을 통해 생성된 76개 특징 사용, 결측치는 imputation 처리 및 마스킹 적용
- **비교 대상**: 일반 1-D CNN (Benchmark CNN)
- **지표**: AUC (Area Under the ROC Curve)

### 정량적 결과
- **전체 성능**: Benchmark CNN의 AUC는 $0.8439 \pm 0.0097$이며, Bayesian CNN은 $0.8507 \pm 0.0116$으로 나타났다. 불확실성을 예측하도록 설계했음에도 성능 저하 없이 유사하거나 약간 높은 성능을 보였다.
- **불확실성과 성능의 관계**: 테스트 셋을 불확실성 중앙값을 기준으로 두 그룹으로 나누었을 때, 저불확실성 그룹의 AUC($0.8593 \pm 0.0170$)가 고불확실성 그룹의 AUC($0.7923 \pm 0.0078$)보다 유의미하게 높았다.

### 데이터 노이즈 실험
원본 데이터의 일부(90%, 70%, 50%, 30%, 10%)만 남기고 나머지를 imputation으로 채워 노이즈를 인위적으로 증가시켰을 때, 데이터가 적어질수록(노이즈가 커질수록) 모델의 불확실성 중앙값이 가파르게 상승하며 AUC는 점진적으로 하락하는 경향을 보였다.

### 의사결정 활용 가능성
불확실성과 예측 확률을 기준으로 환자군을 16개 그룹으로 나누어, 데이터 양을 50%에서 100%로 늘렸을 때의 AUC 변화량을 측정하였다. 실험 결과, 불확실성이 상위 50%~100%에 속하면서 예측 확률이 중간 범위(25%~75%)에 있는 환자들에게 데이터를 더 많이 수집했을 때 성능 향상 폭이 가장 컸다.

## 🧠 Insights & Discussion

본 연구는 BNN을 통해 EHR 데이터의 불확실성을 정량화할 수 있으며, 이 값이 실제 모델의 예측 신뢰도를 반영하는 지표로 활용될 수 있음을 입증하였다. 특히 불확실성 지표가 단순한 성능 측정에 그치지 않고, 어떤 환자에게 더 정밀한 모니터링(데이터 수집)이 필요한지를 알려주는 임상적 가이드라인으로 활용될 수 있다는 점이 강점이다.

다만, 다음과 같은 한계점이 존재한다.
1. **데이터 노이즈의 단순화**: 단순히 원본 데이터를 삭제하고 imputation 하는 방식이 실제 의료 현장에서 발생하는 복잡한 노이즈나, 데이터 수집 빈도가 미래의 치료 결정 및 환자 결과에 미치는 영향을 완전히 반영하지 못한다.
2. **불확실성 범위의 제한**: 본 연구는 데이터 노이즈에 의한 aleatoric uncertainty만을 다루었으며, 모델 구조나 파라미터에서 오는 epistemic uncertainty는 고려하지 않았다.

비판적으로 해석하자면, 제안된 모델이 실제 임상 환경에서 유효하려면 '불확실성이 높은 환자'를 식별했을 때 의료진이 구체적으로 어떤 추가 조치를 취해야 성능이 실제로 향상되는지에 대한 실무적인 프로토콜 연구가 병행되어야 할 것이다.

## 📌 TL;DR

본 논문은 BNN과 1-D CNN을 결합하여 ICU 사망률 예측 시 데이터 노이즈로 인한 **heteroscedastic aleatoric uncertainty**를 정량화하는 방법을 제안한다. 실험을 통해 불확실성이 높은 사례일수록 모델 성능이 낮음을 확인하였으며, 불확실성과 예측 확률을 조합해 **추가적인 데이터 수집이 가장 절실한 환자군을 식별**할 수 있음을 보였다. 이는 향후 제한된 의료 자원을 효율적으로 배분하고 예측 모델의 신뢰성을 높이는 데 기여할 가능성이 크다.
