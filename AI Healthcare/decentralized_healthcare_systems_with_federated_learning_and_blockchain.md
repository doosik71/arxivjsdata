# Decentralized Healthcare Systems with Federated Learning and Blockchain

Abdulrezzak Zekiye and Öznur Özkasap (2023)

## 🧩 Problem to Solve

인공지능(AI)과 딥러닝 기술은 다양한 분야에서 높은 성능을 보이고 있으나, 의료 분야에서는 모델 학습에 필요한 대규모 데이터셋을 확보하는 데 심각한 어려움이 존재한다. 의료 데이터는 병원, 클리닉, 웨어러블 기기 등 여러 소스에 분산되어 있으며, 민감한 개인정보 보호 문제로 인해 각 기관이 데이터를 외부로 공유하는 것을 꺼리기 때문이다.

결과적으로 연구자들은 매우 적은 수의 레코드만으로 모델을 학습시켜야 하는 한계에 직면하며, 이는 모델의 일반화 성능과 정확도를 저하시키는 원인이 된다. 따라서 본 논문은 데이터의 프라이버시를 보존하면서도 분산된 의료 데이터를 효과적으로 활용하여 고성능 AI 모델을 학습시킬 수 있는 체계를 탐색하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 의료 분야에서 **Federated Learning(연합 학습, FL)**과 **Blockchain(블록체인, BC)**을 통합하여 적용한 최신 연구 동향을 체계적으로 검토하고 리뷰했다는 점이다.

중심적인 아이디어는 다음과 같다.

- **Federated Learning**을 통해 원본 데이터(raw data)를 전송하는 대신 모델의 파라미터(parameters)만을 공유함으로써 데이터 프라이버시 문제를 해결한다.
- **Blockchain**을 도입하여 기존 연합 학습의 중앙 서버가 가질 수 있는 단일 장애점(Single Point of Failure) 문제를 해결하고, 시스템의 탈중앙화, 투명성, 불변성 및 스마트 계약(Smart Contracts)을 통한 자율 제어 기능을 구현한다.

## 📎 Related Works

논문에서는 블록체인과 연합 학습을 각각 의료 분야에 적용한 기존 리뷰 논문들을 언급한다. 블록체인을 의료 분야에 적용한 연구[4, 5]와 연합 학습을 스마트 헬스케어에 적용한 연구[6]들이 존재하지만, 두 기술을 **통합적으로** 결합하여 의료 시스템에 적용한 사례를 다룬 리뷰는 부족하다고 지적한다.

기존의 단일 접근 방식과 비교하여, 본 논문이 다루는 통합 접근 방식은 다음과 같은 차별점을 가진다.

- 단순히 데이터의 무결성을 보장하는 블록체인이나, 서버 중심의 프라이버시 보호를 제공하는 연합 학습의 한계를 넘어, '탈중앙화된 신뢰 기반의 협력 학습 체계'를 구축한다는 점이다.

## 🛠️ Methodology

본 논문은 특정 알고리즘을 제안하는 대신, 여러 최신 연구들의 아키텍처를 분석하여 통합적인 파이프라인을 제시한다. 전체적인 시스템 구조는 다음과 같은 흐름을 가진다.

### 1. 전체 시스템 구조

- **데이터 소스**: 병원, 클리닉, 웨어러블 기기 등이 데이터 공급원이 된다.
- **로컬 학습(Local Training)**: 각 엔티티(Entity)는 자신의 로컬 데이터셋을 사용하여 모델을 학습시킨다. 이때 데이터는 외부로 유출되지 않는다.
- **모델 집계(Model Aggregation)**: 학습된 로컬 모델의 파라미터를 수집하여 글로벌 모델(Global Model)로 업데이트한다. 이 과정은 중앙 서버를 통하거나, 블록체인의 스마트 계약을 통해 탈중앙 방식으로 수행된다.

### 2. 주요 구성 요소 및 기술적 장치

분석된 연구들에서 사용된 핵심 메커니즘은 다음과 같다.

- **Differential Privacy (차분 프라이버시)**: 연합 학습 과정에서 파라미터 공유 시 발생할 수 있는 정보 유출을 방지하기 위해 노이즈를 추가하여 보안을 강화한다 [7].
- **SMPC (Secure Multi-Party Computation, 안전한 다자간 계산)**: 중앙 노드가 모델을 집계할 때, 데이터를 암호화하여 전송함으로써 집계 과정에서의 프라이버시를 극대화한다 [9].
- **IPFS (Interplanetary File System)**: 블록체인에 모든 데이터를 저장하는 것은 비효율적이므로, 실제 모델 파일은 IPFS에 저장하고 블록체인에는 그 경로(Hash)만을 기록하여 효율성을 높인다 [7].
- **Non-IID 데이터 해결책**: 데이터가 독립적으로 동일하게 분포되지 않은(Non-IID) 문제를 해결하기 위해, 블록체인을 통해 균일하게 분포된 데이터 서브셋을 공유하여 로컬 학습에 함께 사용하는 2단계 방식을 제안한다 [8].

## 📊 Results

본 논문은 리뷰한 여러 연구들의 정량적 성과를 다음과 같이 제시한다.

- **COVID-19 탐지**: 블록체인 기반 연합 학습 프레임워크를 적용하여 약 $85\%$의 분류 성공률을 달성하였다 [7].
- **유방암 진단**: Breast Cancer Wisconsin 데이터셋을 여러 노드에 분산시켜 학습시킨 결과, $f1\text{-score}$ 기준 약 $99\%$의 매우 높은 성능을 기록하였다 [9].
- **데이터 분포 문제**: Non-IID 데이터 환경에서 소량의 원본 데이터를 공유하며 연합 학습을 진행했을 때, 일반적인 연합 학습보다 정확도가 향상됨을 확인하였다 [8].
- **중앙 집중식 모델과의 비교**: 분산된 소규모 데이터셋으로 학습시킨 모델이 대규모 단일 데이터셋으로 학습시킨 중앙 집중식 솔루션과 유사한 성능을 낼 수 있음을 입증하였다 [11].

## 🧠 Insights & Discussion

### 강점

본 논문은 의료 데이터의 고질적인 문제인 '프라이버시 보호'와 '데이터 부족'이라는 두 마리 토끼를 잡기 위해 FL과 BC의 시너지를 잘 분석하였다. 특히, 블록체인이 단순한 저장소가 아니라 모델 집계의 탈중앙화, 참여자의 평판 관리, 접근 권한 제어(Authorization)의 수단으로 활용될 수 있음을 보여주었다.

### 한계 및 논의사항

- **명시되지 않은 부분**: 각 연구에서 사용된 구체적인 블록체인 합의 알고리즘(Consensus Algorithm)의 종류와 그로 인한 오버헤드(Latency)에 대한 상세 분석은 부족하다.
- **현실적 제약**: 블록체인과 연합 학습을 동시에 운용할 때 발생하는 계산 복잡도와 네트워크 트래픽 증가 문제가 실제 의료 현장의 저사양 기기(웨어러블 등)에서 어떻게 작동할지에 대한 논의가 필요하다.
- **데이터 품질**: Non-IID 문제를 해결하기 위해 일부 데이터를 공유하는 방식[8]은, 엄격한 프라이버시 정책을 가진 기관에서는 수용하기 어려울 수 있다는 비판적 해석이 가능하다.

## 📌 TL;DR

본 논문은 프라이버시 보호를 위한 **Federated Learning**과 탈중앙 신뢰 구조를 위한 **Blockchain**을 결합한 의료 시스템의 최신 연구들을 리뷰한 보고서이다. 이 통합 체계는 데이터 유출 없이 분산된 의료 데이터를 활용해 고성능 AI 모델을 구축할 수 있게 하며, 실제 COVID-19 및 유방암 진단에서 높은 정확도를 입증하였다. 향후 데이터 주권 보장과 보안한 협력 학습이 중요한 의료 AI 분야에서 핵심적인 아키텍처가 될 가능성이 매우 높다.
