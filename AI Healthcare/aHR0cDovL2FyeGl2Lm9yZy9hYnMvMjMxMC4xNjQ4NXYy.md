# A Comprehensive Python Library for Deep Learning-Based Event Detection in Multivariate Time Series Data and Information Retrieval in NLP

Menouar Azib, Benjamin Renard, Philippe Garnier, Vincent Génot, Nicolas André (2023)

## 🧩 Problem to Solve

본 논문은 다변량 시계열(Multivariate Time Series) 데이터에서 특정 이벤트(Event)를 정확하게 탐지하는 문제를 다룬다. 금융, 의료, 사이버 보안, 과학 등 다양한 분야에서 시계열 데이터 내의 이벤트를 식별하는 것은 이상 징후 탐지 및 미래 트렌드 예측을 통해 정보에 기반한 의사결정을 내리는 데 필수적이다.

기존의 딥러닝 기반 이벤트 탐지 방법들은 주로 이진 분류(Binary Classification) 방식에 의존한다. 하지만 이러한 접근 방식은 다음과 같은 한계가 존재한다.
1. **레이블링 비용**: 모든 타임스텝(Time step)에 대해 이벤트 여부를 레이블링해야 하므로 막대한 노동력과 시간이 소모된다.
2. **데이터 불균형**: 실제 환경에서 이벤트는 비이벤트(Non-event)에 비해 매우 드물게 발생하며, 이는 클래스 불균형 문제를 야기하여 분류 모델의 성능을 저하시킨다.
3. **노이즈 및 모호성**: 복잡하고 노이즈가 많은 데이터에서는 이벤트와 비이벤트의 경계가 불분명하여 이진 분류만으로는 이벤트의 강도나 지속 시간과 같은 세부 정보를 포착하기 어렵다.

따라서 본 연구의 목표는 레이블링 부담을 줄이면서도 노이즈에 강건하고, 다양한 도메인에 적용 가능한 새로운 지도 학습 기반의 이벤트 탐지 방법론과 이를 구현한 Python 패키지를 제공하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 기존의 지도 학습 기반 방법론과 차별화되는 네 가지 설계 아이디어에 있다.

1. **회귀 기반 접근법(Regression-based Approach)**: 이진 분류 대신 회귀 모델을 사용하여 이벤트의 존재 여부를 연속적인 값으로 예측한다. 이를 통해 이벤트의 강도와 세부 특성을 더 정밀하게 파악할 수 있으며, 데이터 불균형 문제에 더 유연하게 대응할 수 있다.
2. **단순화된 참조 이벤트 정의**: 모든 시점에 대한 레이블 대신, 이벤트가 발생한 시점(Time point)이나 시간 구간(Interval)으로 정의된 '참조 이벤트(Reference events)' 리스트만을 요구함으로써 데이터 준비 과정을 효율화한다.
3. **Stacked Ensemble Learning 메타-모델**: 단순한 단일 모델이 아니라, Feed-forward Neural Networks(FFN)부터 최신 Transformer 아키텍처에 이르기까지 다양한 딥러닝 모델을 결합한 스택드 앙상블 구조를 채택하여 개별 모델의 편향을 줄이고 예측의 강건성을 높인다.
4. **실용적인 Python 패키지 제공**: 제안한 방법론을 실제 데이터에 쉽게 적용할 수 있도록 `eventdetector-ts`라는 PyPI 설치 가능한 패키지를 개발하여 배포한다.

## 📎 Related Works

논문은 시계열 이상 탐지(Anomaly Detection)를 위한 최신 딥러닝 및 그래프 기반 기술들에 대한 기존 리뷰 논문들을 언급하며, 머신러닝 기술이 시계열 이벤트 탐지에 잠재력이 있음을 밝힌다. 

기존의 일반적인 접근 방식은 각 타임스텝을 '이벤트' 또는 '비이벤트'로 구분하는 이진 분류 방식이다. 본 연구는 이러한 방식이 요구하는 세밀한 레이블링의 한계와 클래스 불균형 문제를 지적하며, 이를 해결하기 위해 연속적인 값을 예측하는 회귀 방식과 앙상블 학습을 결합한 차별점을 제시한다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조
본 방법론은 시계열 데이터를 슬라이딩 윈도우(Sliding Window)로 분할하고, 각 윈도우와 참조 이벤트 간의 겹침 정도를 나타내는 스칼라 값을 예측하는 회귀 모델을 학습시킨 후, 예측값의 피크(Peak)를 통해 이벤트를 추출하는 구조를 가진다.

### 1. 데이터 정의 및 슬라이딩 윈도우
- **입력 데이터**: 시계열 데이터 $S$와 참조 이벤트 리스트 $E$를 사용한다. 시계열 $S$는 시간 $t$에 대해 $f$개의 특징을 가진 벡터 $S(t)$로 정의된다.
- **참조 이벤트**: 각 이벤트 $e_j \in E$는 시작 시간과 종료 시간의 튜플 $(t_{start}, t_{end})$로 표현된다.
- **슬라이딩 윈도우**: 고정된 너비(width)와 스텝(step)을 가진 윈도우 $w_i$가 시계열을 따라 이동하며 데이터를 분할한다.

### 2. Overlapping Parameter ($op$)
회귀 모델의 목표 변수(Target variable)가 되는 $op$는 윈도우 $w_i$와 이벤트 $e_j$ 사이의 겹침 정도를 측정하며, 다음과 같이 계산된다.

$$op(w_i, e_j) = \frac{\text{duration}(w_i \cap e_j)}{\text{duration}(w_i \cup e_j)}$$

여기서 $\text{duration}(w_i \cap e_j)$는 윈도우와 이벤트가 겹치는 구간의 길이이며, $\text{duration}(w_i \cup e_j)$는 두 구간의 합집합 길이이다. 윈도우 $w_i$에 대해 할당될 최종 $op$ 값은 모든 이벤트 중 최대값을 선택한다.

$$op(w_i) = \max_{e_j \in E} op(w_i, e_j)$$

결과적으로 $op$ 값은 0(겹침 없음)에서 1(완벽하게 일치) 사이의 값을 가지며, 윈도우가 이벤트의 중심에 위치할 때 최대값이 된다.

### 3. 모델 학습 및 이벤트 추출
- **모델 $M$**: 딥러닝 모델 $M$은 주어진 윈도우 $w_p$를 입력받아 예측된 $\hat{op}_p$ 값을 출력하도록 학습된다.
- **이벤트 추출**: 예측된 $\hat{op}$ 시퀀스에서 피크(Peak) 지점을 찾고, 해당 지점 $t_q$를 중심으로 설정된 `width_events`를 적용하여 이벤트 구간 $e_q$를 복원한다.
$$e_q = \left[ t_q - \frac{\text{width}_{events}}{2}, t_q + \frac{\text{width}_{events}}{2} \right]$$
- **최적화**: 예측값의 노이즈를 제거하기 위해 가우시안 필터(Gaussian filter)를 사용하여 스무딩(Smoothing)을 수행하며, 특정 임계값(Threshold)을 적용하여 가짜 피크를 제거한다.

### 4. Stacked Ensemble Learning
단일 모델의 약점을 보완하기 위해 여러 베이스 모델(Base models)의 예측값을 메타-모델(Meta-model)의 입력으로 사용하는 스택드 앙상블 구조를 사용한다. 이를 통해 다양한 아키텍처(FFN, CNN, LSTM, Transformer 등)의 강점을 통합하여 최종 $op$ 값을 더 정확하게 예측한다.

## 📊 Results

### 실험 설정
- **데이터셋**:
    1. **신용카드 부정 결제 탐지(Credit Card Fraud Detection)**: 284,807건의 트랜잭션 중 492건의 부정 결제가 포함된 데이터 (매우 심한 클래스 불균형, 0.172%).
    2. **NLP 정보 검색(Information Retrieval in NLP)**: 위키피디아의 Autism 및 Anarchism 텍스트. Word2Vec을 통해 텍스트를 300차원의 시계열 데이터로 변환 후, 키워드 추출(Keys) 및 품사 태깅(POS, 형용사 탐지) 작업을 수행한다.
- **지표**: F1-Score, Precision, Recall을 사용하여 평가한다.
- **비교 설정**: 윈도우 너비(width), 스텝(step), 사용된 스택 모델 아키텍처를 변경하며 성능을 측정한다.

### 주요 결과 (Table I 요약)
- **신용카드 부정 결제**: F1-Score 약 0.82 ~ 0.84 수준을 기록하였다. 특히 FFN 기반 모델만으로도 높은 정밀도(Precision 1.00)를 달성한 사례가 확인되었다.
- **NLP 키워드 추출**: Autism(Keys)과 Anarchism(Keys)에서 각각 F1-Score 0.98, 0.99라는 매우 높은 성능을 보였다.
- **NLP 품사 태깅**: 형용사 탐지(POS)의 경우 F1-Score 0.77 ~ 0.80 수준으로 키워드 추출보다는 낮았으나, 여전히 강건한 성능을 보였다.

실험 결과, 제안된 방법론은 금융 보안과 NLP라는 매우 상이한 도메인에서도 일관되게 유효한 성능을 보였으며, 이는 메타-모델이 $op$ 값의 기저 패턴을 성공적으로 학습했음을 시사한다.

## 🧠 Insights & Discussion

### 강점 및 유효성
본 연구는 단순히 성능 수치를 높이는 것보다 방법론의 **범용성(Versatility)**과 **실용성**을 입증하는 데 집중하였다. 특히 텍스트 데이터를 Word2Vec을 통해 시계열로 변환하여 이벤트 탐지 문제로 치환함으로써, 시계열 분석 도구가 NLP 영역의 정보 검색에도 적용될 수 있음을 보여준 점이 흥미롭다. 또한 가우시안 필터와 피크 임계값 최적화를 통해 회귀 모델 특유의 예측 노이즈 문제를 효과적으로 해결하였다.

### 한계 및 향후 과제
1. **다중 클래스 탐지의 어려움**: 현재 구조로는 단일 종류의 이벤트를 탐지하는 데 최적화되어 있다. 여러 종류의 이벤트를 동시에 탐지하려면 각 클래스별로 별도의 메타-모델을 학습시켜야 하므로 구현 복잡도가 증가한다.
2. **SOTA 성능 검증 미흡**: 저자들은 범용성 입증이 주 목적이었기에 특정 벤치마크에서 최신 기술(SOTA)과 정밀하게 비교하지 않았다. 향후 더 많은 데이터셋에서 하이퍼파라미터 최적화를 통해 타 방법론 대비 우위성을 입증할 필요가 있다.
3. **계산 효율성**: FFN과 같은 가벼운 모델이 좋은 성능을 낸다는 점을 언급하며, 이는 환경적으로 지속 가능한(Sustainable) 솔루션이 될 수 있음을 시사한다.

## 📌 TL;DR

본 논문은 다변량 시계열 데이터에서 이벤트를 탐지하기 위해, 기존의 이진 분류 방식 대신 **회귀(Regression)** 기반의 접근법을 제안한다. 윈도우와 이벤트 간의 겹침 정도를 나타내는 **Overlapping Parameter ($op$)**를 예측 목표로 설정하고, 다양한 딥러닝 모델을 결합한 **Stacked Ensemble Learning**을 통해 예측의 강건성을 확보하였다. 특히 `eventdetector-ts`라는 Python 패키지를 통해 금융 사기 탐지부터 NLP 키워드 추출까지 폭넓은 도메인에 적용 가능한 범용적인 프레임워크를 제공한 것이 핵심이다. 이 연구는 세밀한 레이블링 없이도 참조 이벤트만으로 효율적인 이벤트 탐지가 가능함을 보여주어, 향후 시계열 분석의 실무 적용 가능성을 높였다.
