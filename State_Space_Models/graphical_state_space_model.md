# Graphical State Space Model

ShaoLin Lü (2022)

## 🧩 Problem to Solve

본 논문은 비선형 상태 공간 모델(Nonlinear State Space Model)의 실시간 최적 추정(Optimal Estimation) 문제를 해결하고자 한다. 일반적으로 상태 추정 분야에서는 칼만 필터(Kalman Filter, KF)와 그 확장형인 확장 칼만 필터(Extended Kalman Filter, EKF)가 널리 사용되어 왔다. 그러나 칼만 필터는 시스템의 모든 상태 변수를 시간 순서에 따른 시계열(Time-series) 형태로 취급해야 한다는 제약이 있다.

특히, 시스템 내에 시간에 따라 변하지 않는 상수 상태(Constant state)가 존재함에도 불구하고, 이를 시계열 변수로 모델링하여 처리함으로써 발생하는 추정 정확도의 한계와 계산 효율성 저하가 문제로 지적된다. 따라서 본 논문의 목표는 특정 종류의 비선형 상태 공간 모델에 대해, 상수를 시계열이 아닌 고정된 변수로 처리하는 새로운 프레임워크인 GSSM(Graphical State Space Model)을 제안하여 추정 성능을 높이고 계산 복잡도를 줄이는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **분산 이산화(Distributed Discretization)**이다. 기존의 칼만 필터가 모든 상태 변수를 통합적으로 이산화하여 시계열 모델로 만드는 것과 달리, GSSM은 상태 변수를 변화하는 변수($x_c$)와 상수 변수($x_b$)로 분리하여 모델링한다.

중심적인 설계 직관은 다음과 같다. 상수 변수 $x_b$를 시간 인덱스 $k$에 독립적인 단일 변수로 취급함으로써, 팩터 그래프(Factor Graph) 최적화 시 그래프의 위상 구조(Topology)를 변경하고 최적화해야 할 변수의 차원을 획기적으로 줄일 수 있다는 점이다. 이를 통해 칼만 필터가 해결할 수 없는 형태의 방정식을 구성하고, 팩터 그래프 최적화를 통해 더 높은 정확도의 실시간 추정치를 얻을 수 있다.

## 📎 Related Works

논문은 상태 추정을 위한 기존의 두 가지 주요 접근 방식을 설명한다.

1. **칼만 필터 (Kalman Filter):** 베이지안 필터의 일종으로, 예측과 업데이트 과정을 재귀적으로 수행한다. 논문은 칼만 필터가 동적 베이지안 네트워크(Dynamic Bayesian Network)의 특수한 단순 사례라고 분석한다. 하지만 칼만 필터는 매 단계에서 주변화(Marginalization)를 수행하므로, 미래의 정보가 과거의 상태로 전달되지 않는 전방향 메시지 전달(Forward directional message passing)만 가능하다는 한계가 있다.
2. **팩터 그래프 최적화 (Factor Graph Optimization):** 로보틱스 분야(SLAM, SFM 등)에서 주로 사용되며, 시스템을 이분 그래프(Bipartite Graph)로 표현하여 최적화한다. 이는 전역 최적화(Global Optimization)와 슬라이딩 윈도우 최적화(Sliding Window Optimization)로 나뉜다. 팩터 그래프는 다중 연결 그래프(Multi-connected graph)를 모델링할 수 있다는 강력한 장점이 있으나, 기존의 상태 공간 모델을 그대로 적용할 경우 단순히 슬라이딩 윈도우 필터와 유사한 결과만을 낸다는 한계가 있었다.

## 🛠️ Methodology

### 전체 시스템 구조 및 이산화 방법

GSSM은 연속 시간 시스템을 다음과 같이 정의하고 이산화한다. 상태 벡터를 변화하는 성분 $x_c \in \mathbb{R}^{n_c}$와 상수 성분 $x_b \in \mathbb{R}^{n_b}$로 분리한다.

**연속 시간 모델:**
$$\begin{bmatrix} \dot{x}_c \\ \dot{x}_b \end{bmatrix} = \begin{bmatrix} A_c & A_b \\ 0 & 0 \end{bmatrix} \begin{bmatrix} x_c \\ x_b \end{bmatrix} + \begin{bmatrix} B \\ 0 \end{bmatrix} u + \begin{bmatrix} q \\ 0 \end{bmatrix}$$
$$y = \begin{bmatrix} C_c & C_b \end{bmatrix} \begin{bmatrix} x_c \\ x_b \end{bmatrix} + r$$

**GSSM의 분산 이산화 모델:**
상수 변수 $x_b$를 시계열 $x_b(k)$가 아닌 단일 변수로 취급하여 다음과 같이 이산화한다.
$$x_c(k+1) = F_c(k)x_c(k) + F_b x_b + B_k u_k + q_c(k)$$
$$y_{k+1} = C_c(k+1)x_c(k+1) + C_b(k+1)x_b + r_k$$
$$E(x_b) = x_b^+$$

여기서 $F_c(k) \approx I + A_c T$이며, $F_b = A_b T$이다. 이 방식은 상수 상태를 시계열 형태가 아닌 고정된 팩터로 모델링하는 것이 핵심이다.

### 최적화 절차 및 방정식

슬라이딩 윈도우 최적화 시, GSSM은 다음과 같은 가중 최소 제곱(Weighted Least Squares) 문제를 해결한다.

**목표 변수:** $X_w = [x_b, x_c(k), x_c(k+1), \dots, x_c(k+w)]^T$

**최적화 방정식:**
$$A_w X_w = b_w$$
$$\hat{X}_w = (A_w^T P_w^{-1} A_w)^{-1} A_w^T P_w^{-1} b_w$$

여기서 $A_w$는 더 이상 블록 삼대각 행렬(Block tridiagonal matrix) 구조가 아니며, 이는 칼만 필터의 재귀적 해결 방식으로는 풀 수 없음을 의미한다. 대신 팩터 그래프 최적화를 통해 해결한다.

**차원 비교:**

- **칼만 필터:** 변수 $x_b$를 매 시점 $k$마다 포함시키므로 변수 개수가 $(n_b + n_c)(w+1)$개가 된다.
- **GSSM:** 변수 $x_b$를 단 한 번만 포함시키므로 변수 개수가 $n_b + n_c(w+1)$개로 줄어든다.

## 📊 Results

### 실험 설정

- **작업:** 2차원 레이더 추적(Radar Tracking) 문제.
- **상태 변수:** 거리(distance), 속도(velocity), 고도(altitude).
- **비교 대상:** 확장 칼만 필터(EKF).
- **주요 파라미터:** 샘플링 시간 $T=0.05$, 측정 노이즈 $R=9$, 프로세스 노이즈 $Q=0.005^2$, 윈도우 크기 $w=10$.
- **측정 지표:** 속도 및 고도 추정치의 정성적/정량적 궤적 비교 (Figure 9, 10).

### 결과 분석

실험 결과, GSSM이 칼만 필터보다 속도와 고도 추정에서 더 우수한 성능을 보였다. 이는 GSSM이 상수 상태를 고정된 변수로 처리함으로써 시스템의 정보를 더 효율적으로 추출하고, 윈도우 내의 모든 정보를 동시에 최적화하는 팩터 그래프의 특성을 활용했기 때문으로 분석된다.

## 🧠 Insights & Discussion

### 강점

GSSM은 시스템 모델의 물리적 특성(특정 변수가 상수라는 점)을 그래프 구조에 직접 반영함으로써 계산 효율성과 추정 정확도를 동시에 개선하였다. 특히 변수 차원을 줄여 최적화 속도를 높이면서도, 칼만 필터의 주변화 과정에서 손실되는 정보를 보존할 수 있다는 점이 큰 강점이다.

### 한계 및 비판적 해석

1. **범용성 부족:** 저자 스스로 언급했듯이, 모든 시스템에 적용 가능한 것은 아니다. 상수 상태가 명확히 정의되는 시스템에서만 유효하며, 실제 적용 전 시뮬레이션이 필수적이다.
2. **이론적 분석 부족:** 실험을 통해 성능 향상을 보여주었으나, 어떤 수학적 조건에서 GSSM이 EKF보다 반드시 우월한지에 대한 엄밀한 오차 분석(Error Analysis)이나 수렴성 증명이 부족하다.
3. **가정의 위험성:** 실제 시스템에서 '상수'라고 가정하는 변수가 미세하게 변동할 경우, 이를 고정 변수로 처리하는 것이 오히려 편향(Bias)을 유발할 가능성이 있다.

## 📌 TL;DR

본 논문은 상태 공간 모델에서 상수 변수를 시계열이 아닌 고정 변수로 취급하는 **Graphical State Space Model(GSSM)**을 제안하였다. 이를 통해 팩터 그래프 최적화 시 변수의 차원을 줄이고 그래프 위상을 변경하여, 기존 확장 칼만 필터(EKF)보다 정확하고 효율적인 실시간 상태 추정이 가능함을 레이더 추적 예제를 통해 입증하였다. 이 연구는 시스템의 구조적 특성을 그래프 모델에 반영하는 것이 실시간 최적 추정 성능 향상에 중요한 역할을 할 수 있음을 시사한다.
