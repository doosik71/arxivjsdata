# An algorithm for non-parametric estimation in state-space models

Thi Tuyet Trang Chau, Pierre Ailliot, Valérie Monbet (2020)

## 🧩 Problem to Solve

본 논문은 상태 공간 모델(State-Space Models, SSMs)에서 동적 모델(dynamical model) $m$이 알려지지 않았거나 수치적으로 계산하기 어려운 경우, 이를 비매개변수적(non-parametric) 방식으로 추정하는 문제를 다룬다.

일반적으로 SSM은 관측 노이즈가 존재하는 시계열 데이터를 분석하는 데 유연하고 해석 가능한 프레임워크를 제공한다. 하지만 기존의 매개변수적(parametric) 모델링 방식은 현상의 복잡성을 모두 반영하기 위해 상당한 모델링 노력이 필요하거나, 모델 자체가 충분히 유연하지 않아 실제 데이터를 제대로 재현하지 못하는 한계가 있다.

비매개변수적 접근 방식의 가장 큰 문제는 관측 오차(observational error)의 처리이다. 기존 연구들은 대개 노이즈가 없는 '완벽한' 관측치(perfect observations)가 존재한다고 가정하고 동적 모델 $m$을 추정했다. 그러나 실제 환경에서는 오직 노이즈가 섞인 관측치 $y_{1:T}$만이 제공된다. 만약 관측 오차를 무시하고 $y_{1:T}$를 직접 사용하여 비매개변수적 추정을 수행할 경우, 회귀 분석에서의 '변수 오차 모델(errors-in-variables models)'과 유사하게 추정치가 0으로 편향되는(bias towards zero) 현상이 발생한다. 따라서 본 논문의 목표는 관측 노이즈가 존재하는 상황에서 동적 모델 $m$에 대한 비매개변수적 추정치 $\hat{m}$을 편향 없이 정확하게 구축하는 알고리즘을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Stochastic Expectation-Maximization (SEM) 알고리즘**과 **Sequential Monte Carlo (SMC) 기법**을 결합하여, 관측 노이즈로 인한 편향을 순차적으로 보정하는 비매개변수적 추정 프레임워크를 구축하는 것이다.

중심적인 설계 직관은 다음과 같다. 관측치 $y_{1:T}$에서 직접 $m$을 추정하는 대신, 현재의 모델 추정치를 바탕으로 잠재 상태(latent state)의 평활화 분포(smoothing distribution)를 시뮬레이션하고, 이렇게 얻어진 '정제된' 잠재 상태 궤적들을 사용하여 다시 $m$을 업데이트하는 반복적인 과정을 거치는 것이다. 이를 통해 관측 노이즈가 제거된 샘플을 기반으로 동적 모델을 업데이트함으로써 추정치의 편향을 점진적으로 줄일 수 있다.

## 📎 Related Works

본 논문에서 언급하는 관련 연구와 기존 방식의 한계는 다음과 같다.

1. **비매개변수적 추정 기법**: Nadaraya-Watson 방식이나 Local Linear Regression (LLR) 등이 사용되었다. 특히 LLR은 국소적으로 $m$을 1차 테일러 전개로 근사하여 추정하는 방식으로, 단순한 근접 이웃(nearest neighbors) 방식보다 성능이 우수하다고 알려져 있다. 하지만 이들 모두 '노이즈 없는 데이터'가 필요하다는 전제가 있었다.
2. **EM 및 SEM 알고리즘**: 잠재 변수가 포함된 모델의 최대 가능도 추정(MLE)을 위해 널리 사용된다. 다만, 비선형 SSM에서는 평활화 분포를 분석적으로 구할 수 없어 이를 근사하기 위한 시뮬레이션 기법이 필요하다.
3. **SMC 및 CPF-BS**: Particle Filter와 같은 SMC 기법은 비선형 모델의 분포를 근사하는 데 유용하다. 특히 Conditional Particle Filter-Backward Simulation (CPF-BS) 알고리즘은 샘플 고갈(sample impoverishment) 문제를 완화하고 정확한 평활화 분포를 생성할 수 있어, 파라미터 추정의 효율성을 높인다.

기존 방식과의 차별점은 기존의 SEM이 주로 고정된 구조의 매개변수 $\theta$만을 최적화했다면, 제안된 방법은 매개변수 $\theta$와 비매개변수적 함수 $m$을 동시에, 그리고 반복적으로 업데이트한다는 점에 있다.

## 🛠️ Methodology

### 전체 시스템 구조

본 논문이 고려하는 일반적인 SSM은 다음과 같은 두 방정식으로 정의된다.
$$ X_t = m(X_{t-1}, Z_t) + \eta_t \quad \text{(잠재 상태 방정식)} $$
$$ Y_t = H_t(X_t) + \epsilon_t \quad \text{(관측 방정식)} $$
여기서 $X_t$는 잠재 상태, $Y_t$는 관측치, $Z_t$는 외부 공변량(covariates)이다. 노이즈 항 $\eta_t$와 $\epsilon_t$는 각각 독립적인 가우시안 분포 $\mathcal{N}(0, Q_t(\theta))$와 $\mathcal{N}(0, R_t(\theta))$를 따른다고 가정한다.

### npSEM 알고리즘 (Non-parametric SEM)

제안된 `npSEM` 알고리즘은 다음과 같은 반복 절차를 통해 $\theta$와 $\hat{m}$을 추정한다.

**1. E-step (Expectation Step):**
현재의 파라미터 $\hat{\theta}_{r-1}$과 동적 모델 추정치 $\hat{m}_{r-1}$을 사용하여, 관측치 $y_{1:T}$가 주어졌을 때의 잠재 상태 궤적 샘플 $\{ \tilde{x}_{0:T}^{(i)} \}_{i=1}^N$을 생성한다. 이때 **CPF-BS(Conditional Particle Filter-Backward Simulation)** 알고리즘을 사용하여 정확한 평활화 분포를 근사한다.

**2. M-step (Maximization Step):**

- **Parameter Update**: 생성된 샘플들을 사용하여 완전 로그 가능도(complete log-likelihood) 함수를 최대화하는 $\hat{\theta}_r$를 찾는다.
  $$ \hat{\theta}_r = \arg \max_\theta \frac{1}{N} \sum_{i=1}^N \log p(\tilde{x}_{0:T,r}^{(i)}, y_{1:T}; \theta, \hat{m}_{r-1}) $$
- **Catalog Update**: 생성된 잠재 상태 궤적 $\{ \tilde{x}_{0:T,r}^{(i)} \}$를 학습 데이터(catalog)로 사용하여, **Local Linear Regression (LLR)**을 통해 동적 모델 $\hat{m}_r$을 새롭게 추정한다.

### 과적합(Over-fitting) 방지 및 세부 사항

잠재 상태 샘플 $\tilde{x}_t$가 동일 시간의 관측치 $y_t$에 의존하므로, 이를 그대로 $\hat{m}$ 추정에 사용하면 과적합이 발생할 수 있다. 이를 방지하기 위해 **Lag $\ell$** 개념을 도입한다. 특정 시점 $t$의 $\hat{m}(\tilde{x}_{t-1})$을 추정할 때, 학습 데이터에서 $[t-\ell+1, t+\ell-1]$ 구간의 샘플을 제외함으로써 시점 간의 직접적인 상관관계를 끊어 과적합을 방지한다.

## 📊 Results

### 1. Toy Model 실험 (Lorenz-63)

3차원 혼돈 시스템인 Lorenz-63 모델을 사용하여 제안 방법의 성능을 검증하였다.

- **비교 대상**: `npSEM` (update), `npSEM` (no update), `EnKS` 기반 방식, 완벽한 데이터 사용 방식, 그리고 실제 모델 $m$을 알고 있는 경우.
- **결과**:
  - **파라미터 추정**: `npSEM` (update) 방식은 반복 횟수가 증가함에 따라 $\sigma_Q^2$와 $\sigma_R^2$의 추정치 편향과 분산이 유의미하게 감소하여 실제 값에 근접하였다. 특히 $\hat{m}$을 업데이트하지 않은 방식(`no update`)보다 월등한 성능을 보였다.
  - **상태 재구성**: Root Mean Square Error (RMSE) 측정 결과, `npSEM` (update)는 완벽한 데이터를 사용한 경우나 실제 모델을 알고 있는 경우와 매우 유사한 낮은 오차를 기록하였다. 이는 관측 노이즈가 심한 상황에서도 동적 모델을 효과적으로 복원할 수 있음을 시사한다.

### 2. 실제 사례 연구 (해양 파고 데이터)

프랑스 브레스트 만(Bay of Brest)의 유의 파고($H_s$) 데이터를 사용하여 결측치 보간(imputation) 및 상태 재구성 실험을 수행하였다.

- **설정**: 수심($D$), 풍속($U$), 외해 파고($H_s^{off}$)를 공변량 $Z_t$로 입력하여 비매개변수적으로 $m$을 추정하였다.
- **결과**:
  - **선형 모델 대비 성능**: 기존의 선형 가우시안 SSM (Kalman Smoother 사용)보다 `npSEM`이 훨씬 낮은 RMSE를 기록하였다.
  - **결측치 보간**: 데이터가 누락된 구간(gap)에서도 공변량 정보를 활용한 `npSEM`이 실제 파고의 주기성(조석 주기 등)을 잘 재현하였으며, 보간 오차가 현저히 낮았다.

## 🧠 Insights & Discussion

**강점 및 성과**
본 연구는 관측 노이즈가 존재하는 시계열 데이터에서 비매개변수적으로 동적 모델을 추정할 때 발생하는 편향 문제를 SEM 프레임워크 내에서 '반복적 카탈로그 업데이트'라는 방식으로 우아하게 해결하였다. 특히 CPF-BS를 결합함으로써 비선형성이 강한 시스템에서도 안정적인 평활화 샘플을 얻을 수 있었고, 이를 통해 데이터 기반의 모델 구축이 가능함을 입증하였다.

**한계 및 비판적 해석**

1. **차원의 저주 (Curse of Dimensionality)**: 본 논문의 방법론은 LLR(근접 이웃 기반)과 Particle Filter를 사용한다. 두 기법 모두 상태 공간의 차원이 높아질수록 계산 비용이 기하급수적으로 증가하고 성능이 급격히 저하되는 특성이 있다. 따라서 고차원 시스템으로의 확장 가능성에 대해서는 추가적인 연구(예: 딥러닝 기반의 $m$ 추정)가 필요하다.
2. **노이즈 가정의 제약**: 시스템 노이즈와 관측 노이즈가 가우시안 분포를 따른다고 가정하였다. 비록 부록에서 Student-t 분포에 대한 강건성(robustness)을 확인했으나, 실제 환경의 비가우시안 노이즈에 대해 얼마나 일반화될 수 있는지는 더 엄밀한 분석이 필요하다.
3. **계산 복잡도**: 매 반복마다 모든 입자에 대해 근접 이웃 탐색을 수행해야 하므로, 데이터의 양과 입자 수 $N$이 증가할 때 실시간 처리가 어려울 수 있다.

## 📌 TL;DR

본 논문은 관측 노이즈가 포함된 시계열 데이터에서 동적 모델 $m$을 편향 없이 추정하기 위해, **CPF-BS 평활화**와 **Local Linear Regression(LLR)**을 **SEM 알고리즘**의 반복 구조로 결합한 `npSEM`을 제안한다. 이 방법은 관측 오차로 인해 발생하는 추정치의 편향을 순차적으로 보정하며, Lorenz-63 모델과 실제 해양 파고 데이터 실험을 통해 기존 선형 모델 및 단순 비매개변수 추정 방식보다 뛰어난 상태 재구성 및 결측치 보간 성능을 보였다. 이 연구는 물리적 모델이 불명확한 복잡한 동적 시스템의 데이터 기반 모델링 및 상태 추정에 중요한 방법론적 토대를 제공한다.
