# Backdoor Attacks on Federated Meta-Learning

Chien-Lun Chen, Leana Golubchik, Marco Paolieri (2020)

## 🧩 Problem to Solve

본 논문은 연합 학습(Federated Learning, FL) 환경에서 모델의 유연성을 높이기 위해 도입된 연합 메타 학습(Federated Meta-Learning, FML)의 보안 취약성을 분석한다. 연합 학습은 여러 사용자가 데이터를 공유하지 않고 모델을 공동으로 학습시켜 프라이버시를 보호하지만, 악의적인 사용자가 오염된 데이터를 통해 모델을 조작하는 포이즈닝 백도어 공격(Poisoning Backdoor Attack)에 취약하다는 점이 알려져 있다.

메타 학습은 적은 수의 예제(few-shot)만으로도 새로운 작업에 빠르게 적응(adaptation)할 수 있는 능력을 갖추고 있다. 이러한 적응 능력은 원칙적으로 새로운 벤더 데이터가 깨끗할 경우 백도어 공격에 더 강건한 구조를 가질 수 있다는 가설을 세울 수 있게 한다. 따라서 본 연구의 목표는 연합 메타 학습 환경에서 백도어 공격이 실제로 가능한지, 그리고 공격이 성공했을 때 그 효과가 얼마나 지속되는지를 분석하고, 이를 방어할 수 있는 메커니즘을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 연합 메타 학습의 취약성을 실험적으로 증명하고, 모델의 결정 로직(decision logic)을 제거함으로써 공격을 무력화하는 새로운 방어 전략을 제시한 것이다.

1. **백도어 공격의 유효성 및 지속성 입증**: 단 한 번의 악의적인 업데이트(1-shot attack)만으로도 높은 공격 성공률을 달성할 수 있으며, 이후 다수의 선량한 사용자가 벤더 데이터를 통해 메타 학습을 지속하거나 특정 작업에 대해 파인 튜닝(fine-tuning)을 수행하더라도 공격 효과가 쉽게 사라지지 않고 지속됨을 보였다.
2. **Matching Networks 기반의 방어 메커니즘 제안**: 모델 내부에 고정된 분류기(classifier)를 두는 대신, 입력 데이터의 특징(feature)과 서포트 세트(support set) 간의 유사도를 측정하여 클래스를 예측하는 비파라메트릭(non-parametric) 구조를 제안하였다. 이를 통해 공유 모델에서 공격자가 조작할 수 있는 결정 로직을 제거하여 백도어 공격의 성공률을 획기적으로 낮추었다.

## 📎 Related Works

본 논문은 다음과 같은 기존 연구들을 배경으로 한다.

- **Federated Learning (FL)**: 중앙 서버가 각 클라이언트의 업데이트를 집계하는 구조로, 데이터 프라이버시를 보호하지만 포이즈닝 공격에 취약하다는 한계가 있다.
- **Backdoor Attacks**: 특정 트리거(trigger)가 포함된 입력이 들어왔을 때만 공격자가 의도한 타겟 클래스로 잘못 분류하게 만드는 공격이다. 기존 연구들은 FL에서 이러한 공격이 단일 업데이트만으로도 가능함을 보였다.
- **Gradient-based Meta-Learning**: MAML이나 Reptile과 같이 새로운 작업에 빠르게 적응하기 위한 초기 파라미터를 학습하는 방법론이다. 연합 학습에 메타 학습을 접목하면 각 사용자가 서로 다른 데이터 분포를 가지고 있더라도 공통의 메타 모델을 학습시켜 개별 작업에 최적화할 수 있다는 장점이 있다.

기존의 백도어 방어 기법들은 주로 서버 측에서 사용자 업데이트를 검사하는 방식이었으나, 이는 암호화된 업데이트를 사용하는 Secure Aggregation 환경에서는 적용 불가능하며 프라이버시 침해 우려가 있다. 반면, 본 논문이 제안하는 방식은 사용자가 로컬에서 적용하는 방어 기법이라는 점에서 차별점을 갖는다.

## 🛠️ Methodology

### 전체 시스템 구조 및 학습 절차

본 논문은 $M$명의 사용자가 참여하는 연합 메타 학습 환경을 가정한다. 서버는 선택된 사용자들에게 공유 모델 $\theta_t^G$를 전송하고, 각 사용자는 로컬에서 학습 후 업데이트 $\delta_t^i = \theta_t^i - \theta_t^G$를 서버에 전송한다. 서버는 이를 집계하여 다음 라운드의 모델 $\theta_{t+1}^G = \theta_t^G + \sum_{i=1}^{M_{min}} \alpha_i \delta_t^i$를 생성한다.

메타 학습 알고리즘으로는 1차 근사 방식인 Reptile을 사용한다. 각 사용자는 $B$개의 에피소드로 구성된 메타 배치(meta-batch)를 학습하며, 각 에피소드 $j$에서 $N$개의 클래스와 클래스당 $K$개의 예제를 사용하여 SGD를 수행해 $\theta_{t,j}^i$를 얻는다. 최종 로컬 업데이트는 다음과 같이 계산된다.
$$\theta_t^i = (1-\epsilon)\theta_t^i + \frac{\epsilon}{B} \sum_{j=1}^{B} \theta_{t,j}^i$$
여기서 $\epsilon$은 outer learning rate이다.

### 백도어 공격 방법

공격자는 데이터 포이즈닝을 통해 백도어를 삽입한다. 공격자는 백도어 클래스($X_B$)의 데이터를 타겟 클래스($X_T$)의 레이블로 지정하여 학습시킨다. 또한 공격의 효과를 극대화하기 위해 서버로 전송하는 업데이트에 부스팅 팩터 $\lambda$를 곱하여 전송한다.
$$\delta_a^t = \lambda(\theta_a^t - \theta_t^G)$$
이를 통해 단 한 번의 업데이트만으로도 전역 모델 $\theta_t^G$에 강력한 영향력을 행사한다.

### 방어 메커니즘: Matching Networks 기반 접근

제안된 방어 기법은 모델이 직접 클래스를 예측하는 대신, 임베딩 모델 $f_\theta(x)$를 통해 특징을 추출하고 서포트 세트 $S = \{(x_i, y_i)\}_{i=1}^k$와의 유사도를 측정하는 방식을 취한다. 입력 $\hat{x}$의 예측 클래스 $\hat{y}$는 다음과 같이 결정된다.
$$\hat{y} = \sum_{i} a(\hat{x}, x_i)y_i$$
여기서 $a(\hat{x}, x_i)$는 어텐션 메커니즘(attention mechanism)으로, 본 논문에서는 스케일링된 코사인 거리(scaled cosine distance)를 기반으로 한 소프트맥스 함수를 사용한다.
$$a(\hat{x}, x_i) = \frac{e^{c(\alpha_l f_\theta(\hat{x}), f_\theta(x_i))\beta_l}}{\sum_{j=1}^k e^{c(\alpha_l f_\theta(\hat{x}), f_\theta(x_j))\beta_l}}$$
여기서 $\alpha_l$은 게이트 변수, $\beta_l$은 스케일링 팩터이다.

**방어 절차**:

1. 오염된 모델의 영향력을 줄이기 위해 무작위 Glorot 초기화 값 $\theta_g$를 섞어 새로운 초기값 $\theta' = \delta \theta + (1-\delta)\theta_g$를 설정한다 ($\delta=0.3$).
2. 고정된 $\theta'$ 상태에서 $\alpha_l$과 $\beta_l$을 먼저 학습시킨다.
3. 최종적으로 $\theta', \alpha_l, \beta_l$을 공동으로 학습(joint training)시켜 모델을 정제한다.

## 📊 Results

### 실험 설정

- **데이터셋**: Omniglot (문자 인식), mini-ImageNet (일반 이미지)
- **설정**: 사용자 $M=4$, 공격자 1명, 1-shot attack 수행 ($\lambda=3$).
- **지표**: 메인 작업 정확도(main-task accuracy), 백도어 정확도(backdoor accuracy - 백도어 이미지가 타겟으로 분류되는 비율), 메타 테스트 정확도(meta-testing accuracy).

### 주요 결과

1. **공격의 성공 및 지속성**:
    - **Omniglot**: 단 한 번의 공격으로 백도어 정확도가 약 80%에 달했으며, 메타 테스트 정확도는 단 1%만 하락하여 탐지가 매우 어렵다. 50라운드의 추가 벤더 학습 후에도 백도어 정확도는 50% 수준으로 유지되었다.
    - **mini-ImageNet**: 공격 학습셋에서 75%, 검증셋에서 50%의 성공률을 보였으며, 100라운드의 추가 학습 후에도 성공률이 69%/42%로 유지되어 지속성이 확인되었다.
    - **파인 튜닝**: 벤더 데이터로 파인 튜닝을 10배 더 많이 수행해도 공격 성공률은 10% 미만으로만 감소하여, 단순한 적응 과정으로는 백도어를 제거할 수 없음이 드러났다.

2. **방어 기법의 성능**:
    - Matching Networks 기반 방어 적용 시, 백도어 정확도가 $\approx 20\%$ (무작위 추측 수준)까지 급격히 감소하였다.
    - 특히 벤더 데이터에 백도어 클래스가 포함되어 있을 경우, Omniglot에서는 $\approx 0\%$, mini-ImageNet에서는 $\approx 10\%$까지 공격 성공률이 떨어졌다.
    - 다만, mini-ImageNet의 경우 메타 테스트 정확도가 기존 $\approx 50\%$에서 $\approx 35\%$로 낮아지는 성능 저하가 관찰되었다.

## 🧠 Insights & Discussion

본 연구는 연합 메타 학습의 '빠른 적응력'이 보안 측면에서는 오히려 독이 될 수 있음을 시사한다. 공격자가 심어놓은 백도어는 메타 모델의 파라미터 속에 깊게 각인되어, 이후의 선량한 학습이나 적응 과정으로도 쉽게 지워지지 않는 '지속성'을 갖는다. 이는 메타 학습 모델이 단순한 분류 성능뿐만 아니라, '어떻게 분류하는가'에 대한 로직 자체를 학습하기 때문이다.

제안된 Matching Networks 방어 기법이 효과적인 이유는 **결정 경계(decision boundary)를 모델 파라미터에 고정시키지 않고, 런타임에 서포트 세트와의 유사도 비교를 통해 동적으로 결정**하기 때문이다. 즉, 공격자가 오염시킨 $\theta$ 내의 분류 로직을 무력화하고 특징 추출기($f_\theta$)로서의 기능만 활용함으로써 백도어 트리거의 영향을 최소화한 것이다.

다만, mini-ImageNet에서 나타난 성능 저하는 Matching Networks의 비파라메트릭 구조가 복잡한 이미지 데이터셋에서 기존의 파라메트릭 분류기보다 표현력이 부족할 수 있음을 의미한다. 이는 향후 더 정교한 임베딩 모델이나 유사도 측정 방식의 도입이 필요함을 시사한다.

## 📌 TL;DR

본 논문은 연합 메타 학습(FML)이 단 한 번의 악의적인 업데이트만으로도 강력하고 지속적인 백도어 공격에 취약함을 입증하였다. 특히 메타 학습 특유의 적응 능력이 백도어의 지속성을 높이는 결과를 초래한다. 이를 해결하기 위해 모델의 결정 로직을 제거하고 유사도 기반의 예측을 수행하는 Matching Networks 구조를 도입함으로써, 메인 작업의 성능 저하를 일부 감수하면서도 백도어 공격을 효과적으로 방어할 수 있음을 보여주었다. 이 연구는 프라이버시 보호와 보안성을 동시에 확보해야 하는 연합 학습 환경에서 로컬 단의 방어 메커니즘이 중요함을 시사한다.
