# TAO: A Large-Scale Benchmark for Tracking Any Object

Achal Dave, Tarasha Khurana, Pavel Tokmakov, Cordelia Schmid, and Deva Ramanan (2020)

## 🧩 Problem to Solve

본 논문은 기존의 다중 객체 추적(Multi-Object Tracking, MOT) 벤치마크가 가진 심각한 제한성을 해결하고자 한다. 기존의 데이터셋들은 주로 감시 카메라나 자율 주행 분야에 집중되어 있어, 추적 대상이 사람, 차량, 동물과 같은 소수의 카테고리에 국한되어 있었다. 이러한 편향된 데이터셋으로 학습된 모델들은 실제 세상의 다양한 객체를 추적해야 하는 'in-the-wild' 시나리오에서 성능이 급격히 저하되는 문제가 발생한다.

반면, 객체 검출(Object Detection) 분야에서는 MS COCO나 LVIS와 같이 대규모이며 다양한 카테고리를 포함하는 데이터셋의 도입으로 인해 비약적인 발전이 있었다. 저자들은 MOT 분야에서도 이와 유사하게 광범위한 객체 종류를 다룰 수 있는 대규모 벤치마크가 필요하다고 판단하였으며, 이를 통해 모든 객체를 추적할 수 있는 'Tracking Any Object (TAO)' 데이터셋을 구축하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 단순한 데이터 양의 확장이 아니라, 추적 가능한 객체의 범위를 획기적으로 넓힌 대규모 벤치마크인 TAO를 제안한 것이다.

1. **방대한 카테고리 확장**: 기존 MOT 벤치마크보다 한 자릿수 더 많은 833개의 객체 카테고리를 포함한다.
2. **Bottom-up 카테고리 발견**: 미리 정의된 카테고리 리스트를 따르는 Top-down 방식이 아니라, 비디오 내에서 움직이는 객체를 먼저 찾고 나중에 이름을 붙이는 Bottom-up 방식을 채택하여 큐레이터의 편향을 최소화하였다.
3. **Federated Annotation 기법**: 모든 비디오의 모든 객체를 완벽하게 라벨링하는 것은 비용 면에서 불가능하므로, 정밀도(Precision)와 재현율(Recall)을 효율적으로 측정할 수 있는 연합 라벨링(Federated Annotation) 프로토콜을 추적 도메인에 도입하였다.
4. **SOTA 추적기 분석**: 구축된 데이터셋을 통해 최신 단일/다중 객체 추적기들이 대규모 어휘(Large vocabulary) 환경에서 어떻게 작동하는지 심층 분석하였다.

## 📎 Related Works

### 기존 벤치마크의 한계

- **MOT17, KITTI, UA-DETRAC**: 주로 사람과 차량에 집중되어 있으며, 배경이 매우 유사한 소수의 비디오로 구성되어 모델이 벤치마크에 과적합(Overfitting)되는 경향이 있다.
- **ImageNet-Vid, YTVIS**: 기존보다는 다양하지만, 여전히 카테고리 수가 적고(YTVIS의 경우 40개), 비디오 길이가 짧아 추적 문제의 난이도가 상대적으로 낮다.
- **VidOR**: 다양한 객체를 다루지만, 추적(Track)보다는 객체 간의 관계(Relation)에 집중하여 궤적 정보가 불완전하다.

### 기존 접근 방식과의 차별점

TAO는 단순히 데이터의 양을 늘린 것이 아니라, **카테고리 분포의 질적 변화**를 꾀하였다. 기존 데이터셋이 '사람-차량' 위주였다면, TAO는 Bottom-up 방식을 통해 일상생활의 작은 소품부터 희귀한 동물까지 포괄하는 분포를 가진다. 또한, 픽셀 단위의 마스크(Mask)보다는 박스(Box) 단위의 라벨링을 선택함으로써 규모와 다양성을 우선적으로 확보하였다.

## 🛠️ Methodology

### 1. 데이터 수집 및 필터링

다양한 도메인의 데이터를 확보하기 위해 Charades(실내), AVA(영화), LaSOT(일반), ArgoVerse(도로), YFCC100M(웹) 등 여러 기존 데이터셋에서 비디오를 샘플링하였다. 데이터 품질을 위해 최소 해상도 480p 이상, 최소 2개 이상의 움직이는 객체가 포함된 비디오만을 선정하였다.

### 2. Bottom-up 카테고리 발견

어떤 객체를 라벨링할지 미리 정하지 않고, 작업자에게 비디오 내에서 **스스로 움직이거나 사람에 의해 움직여지는 모든 객체**를 먼저 마킹하도록 요청하였다. 이후 마킹된 객체들에 이름을 부여하였으며, 이때 LVIS 데이터셋의 어휘집을 활용하여 일관성을 유지하되, LVIS에 없는 객체는 자유 형식의 텍스트로 이름을 붙였다.

### 3. Federated Annotation (연합 라벨링)

비용 효율성을 위해 모든 비디오를 전수 조사하는 대신, 카테고리 $c$에 대해 비디오 집합을 세 가지로 구분한다.

- $P_c$: 카테고리 $c$의 모든 인스턴스가 라벨링된 비디오 (Exhaustive)
- $N_c$: 카테고리 $c$의 인스턴스가 전혀 없는 것으로 확인된 비디오 (Negative)
- $U_c$: 카테고리 $c$의 일부 인스턴스만 라벨링된 비디오 (Partial)

평가 시, 정밀도(Precision)는 $P_c$와 $N_c$ 집합만을 사용하여 계산하고, 재현율(Recall)은 $P_c$와 $U_c$ 집합을 사용하여 계산함으로써 적은 비용으로도 신뢰할 수 있는 성능 지표를 도출한다.

### 4. 평가 지표: 3D IoU

본 논문은 시간축을 포함한 3D 공간에서의 Intersection over Union (IoU)를 기본 지표로 사용한다.
지면 진실(Ground Truth) 궤적 $G = \{g_1, \dots, g_T\}$와 예측 궤적 $D = \{d_1, \dots, d_T\}$가 있을 때, 3D IoU는 다음과 같이 정의된다.

$$ \text{IoU}_{3d}(D, G) = \frac{\sum_{t=1}^{T} g_t \cap d_t}{\sum_{t=1}^{T} g_t \cup d_t} $$

이 지표는 단순한 프레임별 정확도를 넘어, 전체 시간 동안 객체를 얼마나 일관되게 유지했는지를 측정한다.

## 📊 Results

### 실험 설정

- **데이터셋**: TAO val set
- **모델**: Mask R-CNN 기반 검출기 (LVIS + COCO 학습), SORT, Viterbi, Tracktor++, 그리고 최신 User-initialized 추적기들.
- **지표**: Tracking mAP (3D IoU $\ge 0.5$ 기준)

### 주요 결과

1. **검출의 어려움**: SOTA 검출기를 사용했음에도 TAO에서의 mAP는 27.1에 그쳐, 다양한 환경에서의 단일 프레임 검출 자체가 매우 도전적인 과제임을 확인하였다.
2. **MOT 성능 분석**:
   - SORT(13.2 mAP)가 Viterbi(6.3 mAP)보다 우수한 성능을 보였다.
   - **Oracle 실험**: 만약 분류(Class)와 연결(Track)이 완벽하다고 가정(Oracle)하면 mAP가 83.6%까지 상승한다. 이는 현재 추적기의 성능 저하가 검출-분류-연결의 복합적인 실패에서 기인함을 시사한다.
3. **특수 카테고리(사람) 추적**: 사람 전용 추적기인 Tracktor++는 36.7 AP를 기록하며 일반 추적기보다 훨씬 뛰어난 성능을 보였다. 이는 사람 카테고리에 대한 집중적인 연구 결과가 반영된 것이나, 동시에 TAO의 다양성이 기존 사람 추적기들에게도 여전히 도전적임을 보여준다.
4. **User-initialized vs MOT**: 사용자로부터 초기 박스를 제공받는 추적기들이 SORT보다 약간 높은 성능을 보였으나, 그 차이가 예상보다 크지 않았다. 이는 대규모 어휘 검출기가 발전함에 따라, 굳이 사용자 초기화 없이도 검출 기반의 MOT가 경쟁력을 가질 수 있음을 의미한다.

## 🧠 Insights & Discussion

### 강점 및 분석

TAO는 기존의 편향된 벤치마크에서 벗어나, 실제 세상의 복잡성을 반영한 평가 환경을 제공한다. 특히 **'Semantic Flicker'**(유사한 클래스 간의 분류 결과가 프레임마다 바뀌는 현상)가 추적 성능을 크게 떨어뜨린다는 점을 발견하였으며, 이는 향후 연구가 단순히 박스를 잘 맞추는 것을 넘어 시간적 일관성을 가진 분류(Classification)를 해결해야 함을 시사한다.

### 한계 및 비판적 해석

- **라벨링 정밀도**: 데이터 규모를 위해 마스크(Mask) 대신 박스(Box)를 사용하고, 1 FPS 단위로 라벨링하였다. 이는 정밀한 세그멘테이션 기반 추적 성능을 평가하기에는 부족함이 있다.
- **데이터 불균형**: Bottom-up 방식으로 수집했음에도 불구하고, 결과적으로 롱테일(Long-tail) 분포가 나타나 희귀 카테고리의 성능이 매우 낮게 측정되는 경향이 있다.

### 결론적 고찰

저자들은 이제 대규모 어휘 검출기의 등장으로 인해, '어떤 객체든 추적할 수 있는(Category-agnostic)' 능력은 더 이상 사용자 초기화(User-initialization)에만 의존할 필요가 없다고 주장한다. 즉, 강력한 일반 객체 검출기와 효율적인 연결 알고리즘의 결합이 오픈월드 추적의 현실적인 해답이 될 수 있다는 통찰을 제시한다.

## 📌 TL;DR

본 논문은 833개의 방대한 카테고리를 포함하는 대규모 MOT 벤치마크인 **TAO**를 제안한다. Bottom-up 방식의 카테고리 발견과 연합 라벨링(Federated Annotation) 기법을 통해 효율적으로 대규모 데이터를 구축하였으며, 실험을 통해 기존 SOTA 추적기들이 대규모 어휘 환경에서 고전하고 있음을 밝혀냈다. 특히, 강력한 일반 객체 검출기가 뒷받침된다면 검출 기반의 다중 객체 추적기가 사용자 초기화 기반 추적기에 필적하는 성능을 낼 수 있음을 입증하여, 향후 오픈월드 추적 연구의 방향성을 제시하였다.
