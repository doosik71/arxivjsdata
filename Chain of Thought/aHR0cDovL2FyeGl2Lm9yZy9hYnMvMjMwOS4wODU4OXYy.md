# CHAIN-OF-THOUGHT REASONING IS A POLICY IMPROVEMENT OPERATOR

Hugh Zhang, David C. Parkes (2023)

## 🧩 Problem to Solve

본 논문은 거대 언어 모델(Large Language Models, LLMs)이 외부의 인간 생성 데이터에 의존하지 않고, 스스로 새로운 기술을 습득할 수 있는가라는 근본적인 문제에 집중한다. 현재의 LLM들은 방대한 양의 학습 데이터에 의존하고 있으나, 고품질의 텍스트 데이터가 고갈되는 '데이터 소진(data exhaustion)' 문제가 제기되고 있으며, 동일한 데이터로 반복 학습할 경우 모델의 성능이 퇴화하는 현상이 보고되고 있다.

따라서 본 연구의 목표는 언어 모델이 스스로 생성한 데이터를 통해 학습하는 '자기 교육(self-education)'의 가능성을 증명하는 것이다. 이를 위해 저자들은 덧셈(addition)이라는 기초적인 수학 과제를 벤치마크로 설정하여, 모델이 초기 단계에서 학습한 수준보다 훨씬 더 복잡한 문제(더 긴 자릿수의 덧셈)를 스스로 학습할 수 있는지 확인하고자 한다.

## ✨ Key Contributions

본 논문의 핵심적인 설계 아이디어는 **Chain-of-Thought(CoT) 추론을 '정책 개선 연산자(Policy Improvement Operator)'로 활용**하는 것이다. 이는 AlphaZero가 Monte-Carlo Tree Search(MCTS)를 통해 현재의 정책을 개선하여 더 나은 수(move)를 찾아내고, 이를 다시 신경망에 학습시키는 방식에서 영감을 얻었다.

중심적인 직관은 다음과 같다. 모델이 CoT 추론을 사용할 때(Slow thinking)는 직접 정답을 출력할 때(Fast thinking)보다 더 높은 성능을 보인다는 점을 이용한다. 즉, $\text{Model}_t$가 CoT를 통해 해결한 어려운 문제와 그 정답을 데이터셋으로 구축하고, $\text{Model}_{t+1}$이 이 정답을 CoT 없이 직접 출력하도록 학습시킨다. 이렇게 개선된 $\text{Model}_{t+1}$은 다시 CoT와 결합했을 때, 이전 버전인 $\text{Model}_t$가 CoT를 사용해서도 풀지 못했던 더 어려운 문제를 해결할 수 있게 된다. 이 루프를 반복함으로써 모델의 능력을 점진적으로 확장한다.

## 📎 Related Works

논문은 다음과 같은 관련 연구들을 언급하며 SECToR의 차별점을 제시한다.

- **산술 학습 연구:** 기존 연구들(Nye et al., 2021; Liu & Low, 2023)은 스크래치패드(scratchpad)나 미세 조정(fine-tuning)을 통해 덧셈 능력을 향상시켰으나, 모델이 스스로를 가르쳐 능력을 확장하는 방식은 제시하지 않았다.
- **LLM의 자기 학습(Self-Learning):** StAR(Zelikman et al., 2022)나 Impossible Distillation(Jung et al., 2023) 같은 연구들이 자기 생성 데이터를 활용한 학습을 시도했다. 그러나 이러한 접근 방식들은 대부분 '에러 아발란체(Error Avalanching)' 현상으로 인해 몇 단계의 개선 이후 성능이 급격히 저하되는 한계를 보였다.
- **AlphaZero와 MCTS:** AlphaZero는 MCTS라는 강력한 정책 개선 연산자를 사용해 초인적인 성능에 도달했다. SECToR은 MCTS 대신 CoT 추론을 이 연산자로 대체하여 일반적인 언어 모델 환경에 적용하고자 한다.

## 🛠️ Methodology

### 전체 파이프라인
SECToR(Self-Education via Chain-of-Thought Reasoning)의 전체 과정은 크게 **지도 학습 단계(Supervised Training)**와 **자기 학습 단계(Self-Training)**로 나뉜다.

### 1. 지도 학습 단계 (Supervised Training)
모델이 초기 기초 능력을 갖추도록 하는 단계이다.
- **커리큘럼 학습(Curriculum Learning):** 1자리 덧셈부터 시작하여 $N$자리 덧셈까지 단계적으로 학습한다. 모델이 $N$자리 문제에서 충분한 정확도를 보여야만 $N+1$자리 문제로 넘어간다.
- **Fast 및 Slow 학습:** 모델은 정답만 바로 출력하는 'Fast' 방식과, 한 단계의 단순화 과정을 거치는 'Slow' 방식(CoT) 모두를 학습한다.
- **종료 조건:** 모델이 CoT를 사용했을 때, 학습하지 않은 $N+1$자리 문제에 대해 거의 완벽한 일반화 성능(length generalization)을 보일 때 지도 학습을 종료하고 자기 학습으로 전환한다.

### 2. 자기 학습 단계 (Self-Training)
외부 데이터 없이 모델이 스스로 생성한 데이터로 학습하는 루프이다.
- **데이터 생성:** 현재 모델이 CoT 추론을 사용하여 $N+1$자리 덧셈의 정답을 생성한다.
- **모델 업데이트:** 생성된 $(문제, 정답)$ 쌍을 사용하여 모델이 CoT 없이 정답을 직접 출력하도록 미세 조정한다.
- **반복:** 개선된 모델이 다시 CoT를 사용하여 $N+2$자리 문제를 풀 수 있게 되면, 이 과정을 계속 반복한다.

### 3. 에러 아발란체(Error Avalanching) 방지 전략
자기 생성 데이터의 오류가 누적되어 모델이 붕괴하는 것을 막기 위해 두 가지 핵심 체크 메커니즘을 도입했다.

- **Simplify-then-guess:** 
  단순히 CoT를 끝까지 수행하는 것이 아니라, 문제를 $K$번 단순화하며 각 단계마다 최종 정답을 추측(guess)하게 한다. 최종 정답은 이 $K$개의 추측 값들에 대한 다수결(majority vote)로 결정한다. 이는 중간 추론 과정의 단일 오류가 전체 정답을 오염시키는 것을 방지한다.
- **교환법칙 체크(Commutativity Checks):** 
  문제 $a + b$와 그 교환 쌍인 $b + a$를 독립적으로 풀어 두 결과가 일치하는지 확인한다. 일치하지 않는 데이터는 학습셋에서 제거한다.

## 📊 Results

### 실험 설정
- **모델:** ByT5 (300M 및 582M 파라미터 모델). 토큰화 문제를 피하기 위해 바이트 단위 모델을 사용했다.
- **데이터셋:** 동일한 자릿수를 가진 두 수의 덧셈 문제.
- **지표:** 정확도(Accuracy) 및 자릿수 일반화 능력.

### 주요 결과
- **성능 확장:** 582M 모델은 1~6자리 덧셈에 대해서만 지도 학습을 받았음에도 불구하고, SECToR를 통해 **최대 29자리 덧셈**까지 스스로 학습하는 데 성공했다. (총 22단계의 자기 개선 수행)
- **정량적 수치:** 최종 모델은 CoT 없이도 30자리 덧셈을 88%의 정확도로 해결할 수 있었다. (표 1 참조)
- **자릿수 일반화:** 지도 학습 단계에서는 CoT 없이 자릿수를 확장하는 능력이 매우 낮았으나, CoT를 사용할 때는 학습한 범위 밖의 자릿수($N+1$)에 대해 높은 정확도를 보였다. 이 특성이 자기 학습 루프의 원동력이 되었다.

## 🧠 Insights & Discussion

### 강점 및 의의
본 연구는 LLM이 적절한 '정책 개선 연산자(CoT)'와 '오류 필터링(Consistency checks)'만 있다면, 인간의 데이터 없이도 자신의 능력을 획기적으로 확장할 수 있음을 증명했다. 특히 데이터 중심의 스케일링 법칙(Scaling Laws)에서 벗어나 계산 중심의 스케일링으로 나아갈 수 있는 가능성을 제시했다는 점에서 학술적 가치가 높다.

### 한계 및 비판적 해석
- **계산 효율성:** 학습 데이터 한 세트를 생성하기 위해 엄청난 양의 추론 계산이 필요하므로 매우 계산 집약적이다.
- **범용성 의문:** 덧셈은 정답이 명확하고 교환법칙과 같은 강력한 일관성 체크가 가능하지만, 정답이 모호한 일반적인 자연어 처리 작업이나 더 복잡한 수학 문제(곱셈 등)에도 동일하게 적용될 수 있을지는 미지수이다.
- **종료 시점:** 결국 29자리 이후에는 에러가 누적되어 학습이 중단되었다. 이는 완벽한 필터링이 불가능하며, 모델의 크기나 필터링 정교함에 따라 한계치가 결정됨을 시사한다.

## 📌 TL;DR

본 논문은 **Chain-of-Thought(CoT) 추론을 정책 개선 연산자로 활용하여 LLM이 스스로 능력을 확장하는 SECToR 프레임워크를 제안**한다. 582M 규모의 모델이 6자리 덧셈 학습만으로 **최대 29자리 덧셈까지 스스로 습득**하게 함으로써, LLM의 자기 교육 가능성을 입증했다. 이는 향후 데이터 고갈 문제를 해결하고, 모델이 스스로 논리적 능력을 강화하는 자율 학습 시스템 구축에 중요한 기여를 할 것으로 보인다.
