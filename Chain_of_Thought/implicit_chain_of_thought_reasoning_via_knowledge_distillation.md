# IMPLICIT CHAIN OF THOUGHT REASONING VIA KNOWLEDGE DISTILLATION

Yuntian Deng, Kiran Prasad, Roland Fernandez, Paul Smolensky, Vishrav Chaudhary, Stuart Shieber (2023)

## 🧩 Problem to Solve

본 논문은 대규모 언어 모델(Large Language Models, LLMs)의 추론 능력을 향상시키기 위해 널리 사용되는 Chain-of-Thought (CoT) 방식의 한계를 해결하고자 한다. 일반적인 CoT 방식은 모델이 최종 정답을 내놓기 전에 중간 추론 단계(intermediate reasoning steps)를 텍스트 형태로 명시적으로 생성하도록 유도한다.

이러한 명시적 CoT 방식은 다음과 같은 두 가지 주요 문제를 야기한다. 첫째, 추론 과정을 텍스트로 하나씩 생성해야 하므로 추론 시간이 매우 오래 걸리는 '수평적(horizontal)' 계산 비용이 발생한다. 둘째, 모델의 추론이 인간의 자연어 체계에 종속되어, 언어 모델이 가질 수 있는 잠재적인 계산 효율성을 충분히 활용하지 못할 가능성이 있다.

따라서 본 연구의 목표는 명시적인 텍스트 생성 없이 모델의 내부 은닉 상태(hidden states)만을 이용하여 추론을 수행하는 'Implicit Chain-of-Thought (Implicit CoT)' 방법을 제안하고, 이를 통해 추론 속도를 높이면서도 CoT 수준의 성능을 유지하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 명시적 CoT를 수행하는 교사 모델(Teacher Model)의 추론 과정을 학생 모델(Student Model)의 내부 레이어 간 '수직적(vertical)' 계산 과정으로 증류(distillation)하는 것이다.

핵심적인 직관은 인간이 복잡한 개념을 완전히 내면화하면 명시적인 단계 없이 직관적으로 정답에 도달하는 것과 유사하게, 언어 모델 역시 은닉 상태 내에서 추론 과정을 압축하여 수행할 수 있다는 점이다. 이를 위해 저자들은 교사 모델의 은닉 상태를 '읽고(Mind-Reading)', 이를 '에뮬레이션(Emulation)'하며, 최종적으로 '최적화(Optimize)'하는 3단계 전략을 제안한다.

## 📎 Related Works

기존의 CoT 연구들은 주로 프롬프팅이나 파인튜닝을 통해 모델이 정답 전 단계들을 생성하도록 유도하는 데 집중해 왔다. 최근 연구에서는 모델의 크기와 데이터 양이 충분히 커지면 CoT 없이도 내부적으로 추론하는 '창발적 능력(Emergent Capabilities)'이 나타난다는 보고가 있었다. 그러나 이러한 방식은 막대한 계산 자원을 필요로 하며, 효율적인 학습 방법론에 대한 연구는 부족했다.

또한, 지식 증류(Knowledge Distillation)는 주로 모델 압축이나 비-자기회귀(non-autoregressive) 번역 등에 사용되어 왔다. 본 논문은 이를 확장하여 교사 모델의 '수평적 추론 과정(텍스트 생성)'을 학생 모델의 '수직적 추론 과정(레이어 간 상태 전이)'으로 변환하는 새로운 관점을 제시하며 기존 접근 방식과 차별화를 둔다.

## 🛠️ Methodology

본 논문에서 제안하는 Implicit CoT 구현을 위한 3단계 전략은 다음과 같다.

### 1. Mind-Reading the Teacher

첫 번째 단계에서는 명시적 CoT를 수행하는 교사 모델의 은닉 상태를 추출하여, 이를 입력받아 정답을 내놓는 학생 모델을 학습시킨다.

- **정보 추출**: 교사 모델의 레이어 수 $L$과 중간 토큰 수 $T$로 구성된 $L \times T$ 은닉 상태 행렬 $z$에서 대각선 요소(diagonal elements)를 선택하여 $L$개의 벡터를 추출한다. 이는 레이어가 깊어질수록 더 많은 추론 정보가 누적된다는 직관에 기반한다.
- **가변 길이 처리**: 중간 단계의 토큰 수가 가변적일 경우, 하이퍼파라미터 $\Delta$를 사용하여 균등한 간격으로 벡터를 추출한다.
- **학습**: 추출된 $L$개의 벡터를 학생 모델의 입력 직후 은닉 상태에 대체하여 삽입하고, 교사 모델은 고정한 채 학생 모델이 최종 정답을 예측하도록 학습시킨다.

### 2. Thought Emulation

테스트 시간에는 교사 모델이 없으므로, 입력값 $x$로부터 교사 모델의 은닉 상태 $z$를 직접 예측하는 에뮬레이터(Emulator)를 학습시킨다.

- **기본 손실 함수**: 예측된 상태 $\hat{z}_l$과 실제 상태 $z_l$ 사이의 평균 제곱 오차(MSE)를 최소화한다.
$$\min \sum_{l=1}^{L} \|z_l - \hat{z}_l\|_2^2$$
- **혼합 모델(Mixture Model)**: 동일한 문제에 대해 여러 추론 경로가 존재할 수 있으므로, 단일 가우시안 분포 대신 혼합 분포를 사용한다.
$$P(\hat{z}_l) = \sum_{c_l} P(\hat{z}_{c_l}^l|c_l)P(c_l)$$
여기서 $c_l$은 서로 다른 추론 모드를 캡처하는 구성 요소이며, 이를 통해 모드 붕괴(mode collapsing)를 방지하고 다양한 추론 경로를 학습할 수 있게 한다.

### 3. Couple and Optimize

마지막으로 에뮬레이터와 학생 모델을 결합하여 전체 시스템을 엔드-투-엔드로 최적화한다.

- **절차**: 에뮬레이터가 예측한 $\hat{z}_l$을 학생 모델의 입력으로 전달하여 정답을 생성하게 하며, 최종 출력의 확률을 최대화하도록 전체 파라미터를 미세 조정(fine-tuning)한다.
- **미분 가능성 확보**: 혼합 모델의 $\text{argmax}$ 연산은 미분이 불가능하므로, 낮은 온도를 가진 소프트맥스(softmax with low temperature)를 사용하여 근사함으로써 역전파가 가능하게 한다.

## 📊 Results

### 실험 설정

- **데이터셋**: 다자리수 곱셈(4x4, 5x5) 및 초등학교 수학 문제(GSM8K-Aug)
- **모델**: GPT-2 Small, Medium, Large
- **비교 대상**: No CoT (직접 정답 예측), Explicit CoT (중간 단계 생성 후 정답 예측)
- **평가 지표**: 정확도(Exact Match Accuracy), 처리량(Throughput, 초당 처리 예제 수)

### 주요 결과

- **성능 향상**: No CoT 설정에서 GPT-2 Medium의 5x5 곱셈 정확도는 2%에 불과했으나, Implicit CoT를 적용했을 때 **96%**로 비약적으로 상승했다. GSM8K-Aug에서도 No CoT(17%)보다 높은 22%의 정확도를 기록했다.
- **추론 속도**: Implicit CoT는 Explicit CoT보다 압도적으로 빠르다. 5x5 곱셈 기준, Implicit CoT의 처리 속도는 No CoT의 73% 수준이었으나, Explicit CoT는 No CoT의 14% 수준에 그쳤다.
- **레이어 수의 영향**: GPT-2 Small은 4x4 곱셈(97%)은 잘 수행했으나 5x5 곱셈(10%)에서는 성능이 급락했다. 이는 Implicit CoT의 성능이 복잡한 계산을 수행할 수 있는 충분한 레이어 수에 의존함을 시사한다.

## 🧠 Insights & Discussion

### 강점 및 성과

본 연구는 명시적 CoT의 성능 이점을 유지하면서도 추론 속도를 No CoT 수준으로 끌어올릴 수 있음을 증명했다. 특히, 텍스트 기반의 수평적 추론을 모델 내부의 수직적 추론으로 변환함으로써 LLM의 계산 효율성을 극대화할 수 있는 가능성을 보여주었다.

### 한계 및 비판적 해석

1. **투명성 및 해석 가능성 결여**: Explicit CoT의 가장 큰 장점은 추론 과정을 인간이 읽을 수 있다는 점이다. 반면 Implicit CoT는 모든 과정이 은닉 상태에서 일어나므로 '블랙박스' 형태가 되어 모델이 왜 그런 결론에 도달했는지 알 수 없다.
2. **교사 모델 의존성**: 제안된 방법론은 여전히 교사 모델의 추론 궤적에서 시작한다. 모델이 완전히 독자적인 효율적 추론 경로를 찾도록 하는 데에는 한계가 있을 수 있다.
3. **성능 격차**: Implicit CoT가 No CoT보다는 훨씬 뛰어나지만, 여전히 Explicit CoT의 정확도에는 미치지 못한다. 이는 사용된 기반 모델(GPT-2)이 기본적으로 수평적 추론에 최적화되어 학습되었기 때문으로 분석된다.

특히 흥미로운 점은 'Couple and Optimize' 단계 이후, 모델의 내부 상태를 다시 텍스트로 매핑했을 때 더 이상 인간이 이해할 수 없는 단어들의 나열로 나타난다는 것이다. 이는 모델이 최적화 과정을 통해 인간의 언어적 추론 방식과는 다른, **모델만의 독자적인 내부 추론 체계**를 구축했음을 시사한다.

## 📌 TL;DR

본 논문은 언어 모델의 명시적 Chain-of-Thought(CoT) 과정을 내부 은닉 상태의 수직적 계산으로 변환하는 **Implicit CoT** 방법론을 제안한다. 교사 모델의 추론 상태를 추출하여 에뮬레이터를 통해 증류하는 3단계 전략을 통해, **Explicit CoT의 추론 능력과 No CoT의 추론 속도라는 두 마리 토끼를 잡고자 했다.** 실험 결과, 단순한 정답 예측으로는 불가능했던 복잡한 수학 문제들을 빠른 속도로 해결할 수 있음을 확인했으며, 이는 향후 LLM이 인간의 언어적 제약에서 벗어나 더 효율적인 내부 추론 메커니즘을 가질 수 있음을 보여준다.
