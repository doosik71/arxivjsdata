# Anomaly Detection in Human Language via Meta-Learning: A Few-Shot Approach

Saurav Singla, Aarav Singla, Advik Gupta, Parnika Gupta (2024/2025)

## 🧩 Problem to Solve

본 논문은 인간 언어 데이터에서 발생하는 이상치 탐지(Anomaly Detection) 문제를 해결하고자 한다. 텍스트 데이터에서의 이상치 탐지는 스팸 필터링, 가짜 뉴스 식별, 혐오 표현 탐지 등 현대 NLP의 핵심적인 응용 분야에서 매우 중요하다.

그러나 텍스트 이상치 탐지에는 다음과 같은 세 가지 주요 난제가 존재한다.
첫째, 텍스트는 비정형 데이터로서 고차원적인 특성과 언어적 다양성을 가지고 있어 강건한 탐지가 어렵다.
둘째, 정의상 이상치(Anomaly)는 매우 희귀하게 발생하므로, 학습에 사용할 수 있는 레이블링된 이상치 데이터가 극도로 부족하다.
셋째, 새로운 유형의 이상치가 지속적으로 등장하므로, 기존에 학습되지 않은 새로운 이상치 패턴에 빠르게 적응해야 하는 일반화 능력이 요구된다.

따라서 본 연구의 목표는 매우 적은 수의 레이블링된 예시(Few-shot)만으로도 새로운 유형의 이상치를 효과적으로 탐지할 수 있는 메타 학습(Meta-learning) 기반의 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 모델이 특정 도메인의 이상치 패턴을 암기하는 것이 아니라, 여러 태스크를 거치며 '이상치란 무엇인가'에 대한 일반적인 개념을 학습하게 하는 것이다. 이를 위해 다음과 같은 기여를 제시한다.

1. **Few-shot 텍스트 이상치 탐지 프레임워크**: MAML(Model-Agnostic Meta-Learning)과 Prototypical Networks라는 두 가지 메타 학습 알고리즘을 텍스트 이상치 탐지에 적용하여, 소수의 샘플만으로 빠르게 적응할 수 있는 능력을 부여하였다.
2. **교차 도메인 샘플링 전략(Cross-Domain Sampling Strategy)**: 학습 과정에서 의도적으로 서로 다른 도메인의 정상 데이터와 이상치 데이터를 조합하여 에피소드를 구성하는 전략을 제안하였다. 이는 모델이 특정 도메인의 지엽적인 특징(예: 특정 단어)에 의존하지 않고, 스타일의 불일치나 문맥적 이질성과 같은 일반적인 이상치 특성을 학습하도록 강제하여 일반화 성능을 높인다.
3. **포괄적인 실험적 검증**: 스팸, 가짜 뉴스, 혐오 표현이라는 세 가지 서로 다른 NLP 도메인에서 기존의 비지도 학습(One-Class SVM) 및 일반 지도 학습(Fine-tuning) 베이스라인보다 뛰어난 성능을 보임을 입증하였다.

## 📎 Related Works

기존의 텍스트 이상치 탐지 연구는 크게 두 가지 방향으로 진행되었다.
첫째는 One-Class SVM이나 Isolation Forest와 같은 비지도 학습 기반 방법으로, 정상 데이터의 표현을 학습하고 여기서 벗어난 데이터를 이상치로 간주한다. 그러나 이러한 방법은 특징 추출의 품질에 크게 의존하며, 소수의 레이블링된 이상치 데이터가 제공될 때의 성능 향상 가능성을 활용하지 못한다는 한계가 있다.
둘째는 BERT와 같은 사전 학습된 언어 모델(PLM)의 임베딩을 활용하는 방법이다. 최근 연구들은 고성능 임베딩과 단순한 탐지기를 결합했을 때 강력한 성능이 나옴을 보였으나, 여전히 대량의 데이터가 필요하거나 새로운 도메인으로의 전이가 어렵다는 문제가 있다.

본 논문은 이러한 한계를 극복하기 위해 '학습하는 법을 학습(Learning to learn)'하는 메타 학습을 도입하였다. 특히 이미지나 센서 데이터 분야에서 사용되던 메타 학습 기법을 텍스트의 언어적 가변성이라는 특성에 맞춰 조정하고, Outlier Exposure 기법에서 영감을 얻은 교차 도메인 샘플링을 통해 텍스트 도메인 특유의 오버피팅 문제를 해결하고자 하였다.

## 🛠️ Methodology

### 전체 시스템 구조
본 프레임워크는 사전 학습된 BERT-base 인코더를 특징 추출기로 사용한다. 입력 텍스트 $x$는 BERT를 통해 768차원의 벡터 $\mathbf{z} = f_\theta(x)$로 변환되며, 이후 메타 학습 알고리즘에 따라 분류 또는 거리 계산이 수행된다.

### 메타 학습 알고리즘

#### 1. Model-Agnostic Meta-Learning (MAML)
MAML은 새로운 태스크에 대해 몇 번의 경사 하강법(Gradient step)만으로 빠르게 최적화될 수 있는 초기 파라미터 $\theta$를 찾는 최적화 기반 접근법이다.
- **내부 루프(Inner-loop)**: 소수의 서포트 셋 $\mathcal{D}_{\text{support}}$에 대해 손실 함수 $\mathcal{L}_{\text{support}}$를 계산하고 파라미터를 업데이트한다.
  $$\theta' = \theta - \alpha \nabla_\theta \mathcal{L}_{\text{support}}(\theta)$$
- **외부 루프(Outer-loop)**: 업데이트된 $\theta'$를 쿼리 셋 $\mathcal{D}_{\text{query}}$에 적용하여 쿼리 손실 $\mathcal{L}_{\text{query}}(\theta')$를 계산하고, 이를 최소화하도록 원래의 $\theta$를 업데이트한다.
  $$\theta \leftarrow \theta - \beta \nabla_\theta \mathcal{L}_{\text{query}}(\theta')$$

#### 2. Prototypical Networks (ProtoNet)
ProtoNet은 각 클래스의 중심점(Prototype)을 학습하고, 쿼리 데이터와 중심점 간의 거리를 측정하는 메트릭 학습 기반 접근법이다.
- **프로토타입 생성**: 서포트 셋의 정상 데이터 임베딩 평균을 $\mathbf{c}_{\text{norm}}$, 이상치 데이터 임베딩 평균을 $\mathbf{c}_{\text{anom}}$으로 정의한다.
- **분류 결정**: 쿼리 데이터 $\mathbf{z}$와 각 프로토타입 간의 유클리드 거리 $d(\mathbf{z}, \mathbf{c})$를 계산하며, 이상치일 확률은 다음과 같이 정의된다.
  $$P(\text{anomaly}) = \sigma\big(d(\mathbf{z}, \mathbf{c}_{\text{norm}}) - d(\mathbf{z}, \mathbf{c}_{\text{anom}})\big)$$

### 교차 도메인 에피소드 샘플링 (Cross-Domain Sampling)
본 논문의 핵심 제안으로, 메타 학습 과정의 일부 에피소드(약 25%)를 다음과 같이 구성한다.
- **구성 방식**: 정상 데이터는 도메인 $A$에서 샘플링하고, 이상치 데이터는 도메인 $B$에서 샘플링하여 하나의 태스크로 묶는다.
- **목적**: 예를 들어 '정상 트윗'들 사이에 '스팸 메시지'를 섞어 놓음으로써, 모델이 도메인 특유의 주제나 단어에 매몰되지 않고 "정상적인 문맥에서 벗어난 이질적인 텍스트"라는 일반적인 속성을 학습하게 한다.

## 📊 Results

### 실험 설정
- **데이터셋**: SMS-Spam (스팸), COVID-Fake (가짜 뉴스), HateSpeech (혐오 표현) 총 3종을 사용하였다.
- **조건**: 학습 데이터의 이상치 비율을 약 3%로 낮게 설정하여 실제 환경의 불균형을 모사하였다.
- **지표**: ROC-AUC, Average Precision (AP), F1-score를 사용하였다.

### 주요 결과
실험 결과, 메타 학습 기반 방법론이 모든 데이터셋에서 베이스라인을 압도하였다.

| Method | SMS-Spam (AUC) | COVID-Fake (AUC) | HateSpeech (AUC) |
| :--- | :---: | :---: | :---: |
| OC-SVM (BERT) | 0.883 | 0.701 | 0.795 |
| Fine-tune BERT | 0.927 | 0.764 | 0.821 |
| ProtoNet | 0.951 | 0.807 | 0.854 |
| **ProtoNet + X-domain** | **0.972** | **0.853** | **0.901** |

- **비지도 학습 대비**: OC-SVM보다 월등한 성능을 보여, 소수의 레이블링된 이상치 데이터가 텍스트 탐지에 매우 결정적인 역할을 함을 확인하였다.
- **일반 지도 학습 대비**: 단순 Fine-tuning보다 메타 학습이 높은 AUC를 기록하였다. 특히 COVID-Fake 데이터셋에서 Fine-tune(0.764) 대비 ProtoNet+X-domain(0.853)은 약 9포인트의 큰 성능 향상을 보였다.
- **교차 도메인 전략의 효과**: 일반 ProtoNet보다 X-domain 전략을 적용했을 때 모든 지표가 상승하였다. 이는 모델이 도메인 독립적인 이상치 특징을 성공적으로 학습했음을 시사한다.

## 🧠 Insights & Discussion

### 강점 및 분석
본 연구는 메타 학습이 단순히 데이터 부족 문제를 해결하는 것을 넘어, 서로 다른 도메인 간의 '이상치라는 개념'을 전이(Transfer)시킬 수 있음을 보여주었다. 특히 Leave-one-out 실험(한 도메인을 완전히 제외하고 학습 후 테스트)에서도 베이스라인보다 높은 성능을 기록한 점은, 본 모델이 새로운 유형의 이상치에 대해서도 매우 강건한 사전 지식(Prior)을 가지고 있음을 입증한다.

### 한계 및 비판적 해석
1. **인위적인 에피소드 구성**: 교차 도메인 샘플링에서 '트윗들 사이의 스팸'과 같은 구성은 현실적으로 발생하기 어려운 인위적인 시나리오이다. 다만, 저자들은 이러한 인위성이 오히려 모델의 결정 경계를 보수적으로 만들어 재현율(Recall)을 높이는 효과를 가져왔다고 주장한다.
2. **오탐지(False Positive) 가능성**: 교차 도메인 학습을 통해 이상치에 민감해진 모델은, 정상 데이터임에도 불구하고 학습 데이터의 정상 분포에서 벗어난 희귀한 주제의 텍스트를 이상치로 오인할 가능성이 있다.
3. **최소 레이블 의존성**: 여전히 적은 수(5-10개)의 레이블이 필요하므로, 완전히 레이블이 없는 zero-shot 환경에서의 성능에 대해서는 논의가 부족하다.

## 📌 TL;DR

본 논문은 레이블링된 데이터가 극도로 부족한 텍스트 이상치 탐지 문제를 해결하기 위해 **MAML과 Prototypical Networks 기반의 메타 학습 프레임워크**를 제안하였다. 특히, 서로 다른 도메인의 데이터를 섞어 학습시키는 **교차 도메인 샘플링 전략**을 통해 특정 도메인에 오버피팅되지 않는 일반적인 이상치 탐지 능력을 확보하였다. 실험 결과, 제안 방법론은 기존 비지도/지도 학습 방식보다 성능이 크게 향상되었으며, 특히 새로운 유형의 이상치를 빠르게 탐지해야 하는 실무 환경(가짜 뉴스, 신종 혐오 표현 탐지 등)에 매우 유용한 접근 방식이 될 것으로 기대된다.
