# Meta-Learning Curiosity Algorithms

Ferran Alet, Martin F. Schneider, Tomás Lozano-Pérez & Leslie Pack Kaelbling (2020)

## 🧩 Problem to Solve

강화학습(Reinforcement Learning, RL) 에이전트가 효율적으로 학습하기 위해서는 환경을 탐색(Exploration)하고 보상을 활용(Exploitation)하는 균형이 필수적이다. 이를 위해 연구자들은 외부 보상(Extrinsic Reward)을 보완하거나 대체하는 호기심(Curiosity) 또는 내재적 동기(Intrinsic Motivation) 기반의 보상 신호를 설계해 왔다.

그러나 기존의 호기심 메커니즘은 주로 인간 연구자가 수동으로 설계(Hand-designing)했다는 한계가 있다. 인간이 모든 가능한 전략의 공간을 체계적으로 탐색하거나, 에이전트가 마주할 수 있는 다양한 환경 분포에 맞게 전략을 최적화하는 것은 매우 어렵다. 따라서 본 논문은 특정 환경에 국한되지 않고 광범위한 환경에 일반화될 수 있는 호기심 알고리즘을 자동으로 발견하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 호기심 메커니즘의 생성 문제를 메타 학습(Meta-Learning) 문제로 정의하는 것이다. 기존의 Meta-RL이 주로 신경망의 가중치(Weights)를 전이하는 방식에 집중하여 유사한 작업 간의 일반화에 그쳤던 것과 달리, 본 연구는 **알고리즘 그 자체(프로그램 코드)**를 메타 학습한다.

이를 위해 신경망, 버퍼(Buffer), 최근접 이웃(Nearest-neighbor) 모듈, 커스텀 손실 함수 등을 결합할 수 있는 풍부한 도메인 특화 언어(Domain-Specific Language, DSL)를 제안한다. 이 DSL을 통해 생성된 프로그램은 상태(State)와 행동(Action) 공간의 차이에 상관없이 적용될 수 있는 다형성(Polymorphic) 타입을 가지며, 이를 통해 완전히 다른 성격의 환경들 사이에서도 일반화 가능한 호기심 알고리즘을 탐색할 수 있다.

## 📎 Related Works

기존의 연구들은 크게 신경망 구조 탐색(NAS)이나 하이퍼파라미터 최적화와 유사한 접근 방식을 취해왔으나, 대부분 단일 작업의 성능 향상에 초점을 맞추었다. 또한, 가중치 전이 기반의 Meta-RL은 환경의 변동성이 적은 경우(예: 미로의 형태만 바뀌는 경우)에는 효과적이지만, 입력/출력 공간이 완전히 다른 환경 간의 전이는 어렵다는 한계가 있다.

내재적 보상 설계 분야에서는 RND(Random Network Distillation)나 Inverse Features와 같은 기법들이 제안되었으나, 이들 역시 인간의 직관에 의존하여 설계된 것이다. 본 논문은 이러한 수동 설계의 한계를 극복하기 위해 프로그램 공간에서의 탐색이라는 접근 방식을 취하며, 기존의 보너스 기반 호기심 알고리즘들이 새로운 환경으로 일반화되는 데 어려움이 있다는 점을 해결하고자 한다.

## 🛠️ Methodology

### 전체 시스템 구조

본 시스템은 외부 루프(Outer loop)와 내부 루프(Inner loop)로 구성된다.

- **외부 루프(Meta-level):** 호기심 메커니즘을 정의하는 프로그램 공간을 탐색하여, 에이전트의 생애 전체 보상을 최대화하는 최적의 프로그램 $C$를 찾는다.
- **내부 루프(RL-level):** 선택된 호기심 모듈 $C$가 제공하는 프록시 보상 $\hat{r}_t$를 사용하여 표준 강화학습(PPO 등)을 수행하며 정책 $\pi$를 학습한다.

최종 목적 함수는 다음과 같이 정의된다:
$$\max_{C} [ \mathbb{E}_{E} [ \mathbb{E}_{r_t \sim e(A,C;E)} [ \sum_{t=0}^{T} r_t ] ] ]$$
여기서 $E$는 환경 분포, $A$는 RL 알고리즘, $C$는 호기심 모듈을 의미한다.

### 프로그램 기반 호기심 모듈 $C$

호기심 모듈은 두 가지 구성 요소로 나뉜다:

1. **내재적 보상 생성기 $I$:** 현재의 전이 $(s_t, a_t, s_{t+1})$와 메모리를 기반으로 내재적 보상 값 $i_t$를 출력한다.
2. **보상 결합기 $\chi$:** 현재 시간 $t$, 외부 보상 $r_t$, 내재적 보상 $i_t$를 입력받아 최종 프록시 보상 $\hat{r}_t$를 계산한다.

프로그램은 유향 비순환 그래프(DAG) 형태의 모듈로 표현되며, 다음과 같은 모듈 클래스를 포함한다:

- **입력 모듈:** $s_t, a_t, s_{t+1}, i_t, r_t$ 등.
- **버퍼 및 파라미터 모듈:** FIFO 큐, 최적화 가능한 신경망 가중치.
- **함수 모듈:** 입력값에 기반한 연산 수행.
- **업데이트 모듈:** 버퍼에 변수를 추가하거나 전역 손실(Global Loss)을 추가하여 경사 하강법(Gradient Descent)을 유도함.

특히, 신경망의 가중치 업데이트 과정 자체를 일종의 **메모리**로 활용하여, 온라인 학습을 통해 방문한 상태의 예측 품질을 측정하고 이를 보상에 반영하는 구조를 가진다.

### 탐색 효율화 전략

방대한 프로그램 공간을 효율적으로 탐색하기 위해 세 가지 전략을 사용한다:

1. **정적 가지치기(Pruning):** 중복 프로그램 제거 및 입력 데이터와 무관하게 손실 함수가 최소화될 수 있는(즉, 데이터에 반응하지 않는) 무의미한 알고리즘을 사전에 제거한다.
2. **저비용 환경 필터링:** 계산 비용이 적은 단순한 환경에서 먼저 테스트하고, 성능이 낮은 프로그램은 조기에 제거하는 순차적 할당(Sequential Halving) 방식을 적용한다.
3. **성능 예측(Meta-meta-RL):** 프로그램의 구조적 특징(사용된 연산의 종류와 횟수)을 바탕으로 $k$-최근접 이웃($k$-NN) 회귀 모델을 사용하여 유망한 프로그램을 우선적으로 평가한다.

## 📊 Results

### 실험 설정

- **RL 알고리즘:** PPO (Proximal Policy Optimization).
- **평가 환경:** GridWorld(이미지 기반), Lunar Lander, Acrobot, Ant, Hopper (MuJoCo).
- **메타 학습 환경:** 초기 탐색은 단순한 이미지 기반 GridWorld에서 수행되었으며, 이후 다른 환경으로 전이(Transfer) 가능성을 테스트하였다.

### 주요 결과

1. **새로운 알고리즘 발견:** 탐색 결과, **FAST (Fast Action-Space Transition)**라는 매우 단순하지만 효과적인 알고리즘이 발견되었다. 이 알고리즘은 $s_{t+1}$로부터 행동 $a$를 예측하는 신경망을 학습시키고, $s_t$와 $s_{t+1}$ 기반의 예측값 차이가 클 때 높은 보상을 부여한다.
2. **일반화 성능:** GridWorld에서 발견된 상위 2,000개의 프로그램들이 Lunar Lander와 Acrobot에서도 높은 상관관계를 보이며 성능을 유지함을 확인하였다.
3. **정량적 비교:** MuJoCo 환경(Ant, Hopper)에서 메타 학습된 알고리즘들은 기존의 수동 설계된 알고리즘(RND 등)과 통계적으로 동등하거나 더 나은 성능을 보였으며, 단순 베이스라인보다는 월등히 높은 성능을 기록하였다.

| Class                   |       Ant        |      Hopper      |
| :---------------------- | :--------------: | :--------------: |
| Baseline algorithms     | $[-95.3, -39.9]$ | $[318.5, 525.0]$ |
| Meta-learned algorithms | $[+67.5, +80.0]$ | $[589.2, 650.6]$ |
| Published algorithms    | $[+67.4, +98.8]$ | $[627.7, 692.6]$ |

*(표의 수치는 평균 에피소드 보상의 신뢰 구간을 나타냄)*

## 🧠 Insights & Discussion

본 논문의 가장 큰 강점은 **다형성 데이터 타입(Polymorphic Data Types)**을 도입하여 알고리즘의 '구조'와 '구현'을 분리했다는 점이다. 예를 들어, 상태 $S$가 이미지인 경우에는 CNN이, 벡터인 경우에는 MLP가 자동으로 할당되도록 설계하여, 단 하나의 프로그램 구조가 서로 다른 입력/출력 공간을 가진 다양한 환경에 그대로 적용될 수 있게 하였다.

또한, 인간이 설계한 알고리즘보다 더 복잡하거나, 혹은 반대로 너무 단순해서 간과되었을 법한 알고리즘(FAST 등)을 기계가 스스로 찾아낼 수 있음을 입증하였다. 이는 ML 알고리즘 설계 영역에서도 AutoML과 같은 자동화된 접근 방식이 유효함을 시사한다.

한계점으로는, 보상 결합기($\chi$)를 최적화하는 과정이 내재적 보상 생성기($I$)의 탐색만큼 정밀하게 이루어지지 않았다는 점이 언급된다. 향후 연구에서는 두 구성 요소를 동시에 공동 최적화(Co-adapt)하는 방향으로 확장될 가능성이 있다.

## 📌 TL;DR

이 논문은 인간이 수동으로 설계하던 호기심 보상 메커니즘을 **메타 학습 기반의 프로그램 탐색**으로 대체하여, 환경에 구애받지 않고 일반화 가능한 호기심 알고리즘을 자동으로 발견하는 프레임워크를 제안한다. 풍부한 DSL과 효율적인 가지치기 전략을 통해 FAST와 같은 새로운 알고리즘을 찾아냈으며, 이는 이미지 기반 환경에서 학습되었음에도 불구하고 로봇 제어와 같은 완전히 다른 환경에서도 기존의 최신 알고리즘들과 대등한 성능을 보였다. 이 연구는 향후 최적화 알고리즘이나 더 고차원의 메타 학습 알고리즘을 자동 설계하는 데 중요한 기반이 될 수 있다.
