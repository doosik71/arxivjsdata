# Deep Online Learning via Meta-Learning: Continual Adaptation for Model-Based RL

Anusha Nagabandi, Chelsea Finn, Sergey Levine (2019)

## 🧩 Problem to Solve

본 논문은 딥 뉴럴 네트워크(Deep Neural Network) 모델이 실시간 데이터 스트림으로부터 빠르게 적응하며 학습하는 능력이 부족하다는 점을 해결하고자 한다. 인간이나 동물은 환경의 갑작스러운 변화에 매우 빠르게 적응하고, 이전에 겪었던 유사한 상황을 기억하여 다시 활용하는 능력을 갖추고 있다. 그러나 일반적인 딥러닝 모델은 학습을 위해 중간 크기의 미니배치(minibatches)와 여러 번의 에포크(epochs)가 필요하며, 이는 데이터가 끊임없이 들어오는 온라인 스트리밍 환경에 적합하지 않다.

특히, 실제 환경은 정체되어 있지 않은 비정상성(non-stationary) 특성을 가진다. 예를 들어, 로봇이 이동하며 지형이 계속 변하거나 모터에 결함이 발생하는 등의 상황이 발생할 수 있다. 이러한 환경에서 모델은 새로운 작업(task)에 빠르게 적응해야 함과 동시에, 과거에 학습한 지식을 잃지 않고(catastrophic forgetting 방지), 이전에 보았던 작업이 다시 나타났을 때 이를 즉각적으로 회상하여 적용할 수 있어야 한다. 따라서 본 논문의 목표는 딥러닝 모델을 사용하여 연속적인 온라인 학습을 가능하게 하고, 작업의 경계가 명시적으로 주어지지 않은 상황에서도 스스로 작업 분포를 관리하며 적응하는 방법을 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 메타 학습(Meta-learning)과 확률적 작업 관리 메커니즘을 결합한 MOLe(Meta-learning for Online Learning) 알고리즘을 제안한 것이다. 

중심적인 아이디어는 두 가지이다. 첫째, 모델-불가지론적 메타 학습(Model-Agnostic Meta-Learning, MAML)을 통해 온라인 SGD(Stochastic Gradient Descent) 업데이트가 매우 적은 횟수의 단계만으로도 효과적으로 작동할 수 있도록 최적의 초기 가중치(prior initialization)를 학습하는 것이다. 둘째, 기대 극대화(Expectation Maximization, EM) 알고리즘과 중국식 레스토랑 프로세스(Chinese Restaurant Process, CRP) 사전 확률을 결합하여, 현재 입력 데이터가 어떤 작업에 해당하는지 추론하고, 필요에 따라 새로운 작업을 생성하거나 기존 작업을 회상하는 혼합 모델(mixture of models) 구조를 유지하는 것이다.

## 📎 Related Works

기존의 온라인 학습 연구들은 주로 온라인 경사 하강법(online gradient updates)이나 확률적 필터링(probabilistic filtering) 방식을 사용했다. 하지만 이러한 방식들은 딥 뉴럴 네트워크와 같은 고차원 모델에 적용했을 때, 배치 모드 학습의 부재로 인해 성능이 크게 제한되는 한계가 있었다.

지속 학습(Continual Learning) 또는 평생 학습(Lifelong Learning) 분야에서는 주로 '치명적 망각(catastrophic forgetting)'을 방지하는 데 집중해 왔으나, 이는 주로 정해진 소수의 작업 세트를 다루는 경우가 많아 실시간 스트리밍 데이터에서의 빠른 적응(fast adaptation) 문제는 충분히 다루지 않았다.

메타 학습, 특히 MAML은 소량의 데이터로 빠르게 적응하는 $k$-shot 학습 설정에서 탁월한 성능을 보였으나, 이는 항상 고정된 사전 모델(prior)에서부터 적응을 시작하는 배치 모드 방식이다. 즉, 이전 단계에서 적응한 결과가 다음 단계로 이어지지 않고 계속 초기화된다는 한계가 있다. MOLe는 이러한 $k$-shot 설정을 넘어, 시간에 따라 지식이 계속 축적되고 진화하는 연속적인 온라인 적응 체계를 구축함으로써 기존 연구들과 차별화된다.

## 🛠️ Methodology

### 전체 시스템 구조
MOLe는 메타 학습을 통해 얻은 사전 모델 $\theta^*$를 시작점으로 하여, 실시간으로 들어오는 데이터 스트림에 대해 여러 개의 작업 전용 모델 $\theta(T_i)$를 유지하고 업데이트하는 구조를 가진다.

### 온라인 추론 및 업데이트 (EM 알고리즘)
모델은 매 시간 단계 $t$마다 입력 $x_t$에 대해 예측 $\hat{y}_t$를 내놓고, 실제 정답 $y_t$를 받아 모델을 업데이트한다. 이 과정은 EM 알고리즘의 형태로 이루어진다.

**1. E-step: 작업 확률 추론**
현재 데이터가 어떤 작업 $T_i$에 속하는지에 대한 사후 확률 $P(T_t = T_i | x_t, y_t)$를 계산한다. 이는 다음과 같이 표현된다.
$$P(T_t = T_i | x_t, y_t) \propto p_{\theta_t(T_i)}(y_t | x_t, T_t = T_i) P(T_t = T_i)$$
여기서 $P(T_t = T_i)$는 작업에 대한 사전 확률이며, 이를 위해 중국식 레스토랑 프로세스(CRP)를 사용한다. CRP에 따르면, 특정 작업 $T_i$가 선택될 확률은 해당 작업에 할당되었던 데이터의 양에 비례하며, 새로운 작업을 생성할 확률은 하이퍼파라미터 $\alpha$에 의해 결정된다.
$$P(T_t = T_i) = \frac{\sum_{t'=1}^{t-1} P(T_{t'} = T_i)}{t-1 + \alpha}, \quad P(T_t = T_{new}) = \frac{\alpha}{t-1 + \alpha}$$

**2. M-step: 모델 파라미터 업데이트**
추론된 작업 확률(responsibility)을 가중치로 사용하여 모든 작업 모델의 파라미터를 업데이트한다. 각 작업 모델 $\theta(T_i)$는 다음과 같이 온라인 SGD를 통해 업데이트된다.
$$\theta_{t+1}(T_i) = \theta_t(T_i) - \beta P_t(T_t = T_i | x_t, y_t) \nabla_{\theta_t(T_i)} \log p_{\theta_t(T_i)}(y_t | x_t)$$
이 식은 현재 데이터가 해당 작업일 확률이 높을수록 그 작업 모델을 더 많이 업데이트함을 의미한다. 만약 새로운 작업 $T_{new}$가 생성된다면, 이는 메타 학습된 사전 모델 $\theta^*$에서 시작하여 현재 데이터로 한 번 업데이트된 후 작업 집합에 추가된다.

### 메타 학습을 통한 사전 모델(Prior) 획득
온라인 SGD가 딥러닝 모델에서 효과적으로 작동하게 하기 위해 MAML을 사용하여 $\theta^*$를 학습한다. MAML의 목표는 소수의 그래디언트 단계만으로도 다양한 작업에서 낮은 손실(loss)을 달성할 수 있는 최적의 초기화 지점을 찾는 것이다.
$$\min_{\theta} \sum_{T} L(\theta - \eta \nabla_{\theta} L(\theta, D_{tr}^T), D_{val}^T)$$
이렇게 학습된 $\theta^*$는 MOLe의 온라인 단계에서 모든 새로운 작업 모델의 출발점이 되며, 빠른 적응을 가능케 하는 기초 체력이 된다.

### Model-Based RL 적용 세부 사항
본 논문은 이를 모델 기반 강화학습의 동역학 모델(dynamics model) 학습에 적용하였다.
- **입력 $x_t$**: 과거 $K$개의 상태-액션 쌍의 연결(concatenation)
- **출력 $y_t$**: 다음 상태들의 시퀀스
- **모델 구조**: 평균을 예측하는 뉴럴 네트워크(3개 은닉층, 각 500차원, ReLU)와 고정된 분산을 가진 가우시안 분포
- **제어**: 학습된 모델을 사용하여 미래 보상을 최대화하는 액션 시퀀스를 찾는 Random-shooting MPC(Model Predictive Control)를 사용한다.

## 📊 Results

### 실험 설정
- **에이전트**: MuJoCo 엔진의 Half-Cheetah 및 Hexapedal Crawler.
- **테스트 시나리오**: 지형 경사 변화(terrain slopes), 모터 결함(motor malfunctions), 다리 마비(crippling of end-effectors).
- **비교 대상**: 
    - $k$-shot adaptation (메타 학습 $\theta^*$에서 매번 새로 적응)
    - Continued adaptation (메타 학습 $\theta^*$에서 시작하여 계속 그래디언트 업데이트)
    - Standard Model-based RL (적응 없음)
    - Model-based RL with online gradient updates (메타 학습 없이 온라인 SGD만 적용)

### 주요 결과
1. **지형 경사 변화**: 메타 학습 기반 방법들이 전반적으로 우수했다. 특히 단순 온라인 SGD는 메타 학습된 초기화 없이는 성능이 매우 낮았다.
2. **모터 결함**: 비정상적으로 변화하는 작업 분포에서 MOLe가 압도적인 성능을 보였다. Continuous adaptation 방식은 최신 데이터에 과적합(overfitting)되어 이전 기술을 잊어버리는 현상이 나타났으나, MOLe는 작업별 모델을 분리 유지함으로써 이를 극복했다.
3. **다리 마비 (Crawler)**: 다리가 마비되었다가 다시 정상으로 돌아오는 반복적인 상황에서, MOLe는 작업의 전환을 정확히 인식하고 과거의 정상 상태 모델을 회상(recall)하여 빠르게 복구하는 모습을 보였다.
4. **정량적 지표**: 누적 보상(cumulative rewards) 측면에서 MOLe는 $k$-shot 및 Continuous adaptation보다 높은 성능을 기록했으며, 특히 작업이 반복될수록 성능이 점진적으로 향상되는 특성을 보였다.

## 🧠 Insights & Discussion

본 연구는 딥러닝 모델이 실시간 스트리밍 데이터 환경에서 겪는 고질적인 문제인 '빠른 적응'과 '망각' 사이의 트레이드오프를 성공적으로 해결하였다.

가장 주목할 점은 **MAML을 통한 메타 학습이 단순히 소량의 데이터를 위한 것이 아니라, 연속적인 온라인 SGD 업데이트의 효율성을 극대적으로 높여주는 기반**이 된다는 사실을 발견한 것이다. 또한, CRP를 이용한 동적 작업 할당 방식은 사람이 직접 작업의 경계를 정해주지 않아도 모델이 스스로 "지금은 새로운 상황이다" 혹은 "예전에 겪었던 상황이다"를 판단하게 함으로써 자율적인 작업 전문화(task specialization)를 가능케 했다.

한계점으로는, EM 알고리즘의 E-step과 M-step을 매 단계 한 번씩만 수행하는 근사 방식을 사용했다는 점이 있다. 비록 실험적으로는 충분했으나, 더 복잡한 환경에서는 EM의 완전한 수렴을 위해 반복 계산이 필요할 수 있다. 또한, 모델 기반 RL이라는 특정 도메인에서 검증되었으나, 시계열 예측이나 능동적 온라인 학습(active online learning) 등 더 넓은 분야로의 확장 가능성이 열려 있다.

## 📌 TL;DR

본 논문은 딥러닝 모델의 실시간 온라인 적응을 위해 **MAML 기반의 사전 모델 $\theta^*$**와 **CRP-EM 기반의 작업 혼합 모델**을 결합한 **MOLe 알고리즘**을 제안한다. 이를 통해 모델은 비정상적인 환경 변화 속에서도 새로운 작업을 스스로 정의하고, 과거의 지식을 보존하며, 빠르게 적응하는 능력을 갖추게 된다. 특히 모델 기반 강화학습 환경에서 로봇의 모터 결함이나 지형 변화와 같은 극한 상황에서도 효과적으로 동작함을 입증하였으며, 이는 향후 실시간 적응이 필수적인 지능형 시스템 구축에 중요한 기여를 할 것으로 보인다.
