# Introduction to Coding Quantum Algorithms: A Tutorial Series Using Pyquil

Daniel Koch, Laura Wessing, Paul M. Alsing (2019)

## 🧩 Problem to Solve

본 논문(튜토리얼 시리즈)은 양자 컴퓨팅 분야의 급격한 성장에도 불구하고, 일반인이 공개된 양자 컴퓨터를 테스트하거나 관련 알고리즘을 학습하는 데 있어 느끼는 진입 장벽을 해결하고자 한다. 특히 양자 물리학에 대한 배경지식이 부족하거나 Python 코딩 경험이 적은 입문자들에게는 이론적 배경과 실제 구현 사이의 간극이 매우 크다는 점이 문제로 제기된다.

따라서 본 연구의 목표는 Rigetti사의 Pyquil 라이브러리를 활용하여, 양자 알고리즘의 기초부터 학술적으로 중요한 주요 알고리즘의 구현까지 단계적으로 학습할 수 있는 상세한 가이드를 제공하는 것이다. 이를 통해 사용자가 양자 시스템의 동작 원리를 이해하고, 실제 코드를 통해 이를 검증하며, 최종적으로 복잡한 양자 알고리즘을 설계할 수 있는 능력을 배양하는 것을 목적으로 한다.

## ✨ Key Contributions

본 튜토리얼의 핵심적인 기여는 이론적 개념을 실제 구현 가능한 코드 수준으로 구체화하여 제공한다는 점이다. 주요 설계 아이디어는 다음과 같다.

1. **단계적 학습 경로 설계**: 단순한 큐비트 생성 및 측정(Lesson 1) $\rightarrow$ 복잡한 프로그램 구조 및 제어(Lesson 2) $\rightarrow$ 표준 게이트 셋의 이해(Lesson 3) $\rightarrow$ 커스텀 함수 및 고차 제어 게이트 구현(Lesson 4) $\rightarrow$ 주요 양자 알고리즘 적용(Lesson 5, 6) 순으로 구성하여 학습 곡선을 완만하게 설계하였다.
2. **실용적 도구 제공**: `Wavefunction` 및 `Measurement`와 같은 커스텀 함수를 통해, 시뮬레이터(QVM)의 복잡한 출력을 사용자가 이해하기 쉬운 형태로 시각화하여 학습 효율을 높였다.
3. **고차 제어 게이트 구현 전략**: 실제 양자 하드웨어의 제약을 고려하여, $n$-control NOT 게이트와 같은 고차 연산을 보조 큐비트(Ancilla qubits)와 CCNOT(Toffoli) 게이트의 체인 구조를 통해 구현하는 구체적인 방법론을 제시하였다.
4. **주요 알고리즘의 체계적 분석**: Deutsch, Deutsch-Jozsa, Bernstein-Vazirani, Simon, Grover, QFT 등 학술적으로 중요한 알고리즘들을 'Hadamard 변환'이라는 공통 분모를 중심으로 분석하여 제공한다.

## 🛠️ Methodology

### 1. 시스템 구조 및 Pyquil 프레임워크

본 튜토리얼은 Python 기반의 Pyquil 라이브러리를 사용하며, 실제 양자 처리 장치(QPU) 대신 가상 머신인 QVM(Quantum Virtual Machine)을 통해 알고리즘을 시뮬레이션한다.

- **Program**: 양자 시스템에 내릴 명령어들의 리스트이다.
- **QVMConnection**: 작성된 프로그램을 실행하고 상태를 시뮬레이션하는 클라우드 기반 가상 머신이다.

### 2. 주요 양자 게이트 및 연산

- **Single Qubit Gates**: $I$(Identity), $X$(NOT), $H$(Hadamard), $Y, Z$, 그리고 위상 게이트($S, T, R_\phi$)와 회전 게이트($R_x, R_y, R_z$)를 사용한다.
- **Multi Qubit Gates**: CNOT(CX), CZ, CPHASE, SWAP, CSWAP(Fredkin), CCNOT(Toffoli) 등을 통해 큐비트 간의 얽힘과 제어 연산을 수행한다.

### 3. 고차 제어 게이트 구현 (Higher Order Control Gates)

$n$개의 제어 큐비트가 모두 $|1\rangle$일 때만 타겟 큐비트에 연산을 적용하는 고차 게이트는 다음과 같은 전략으로 구현한다.

- **전략**: $n-1$개의 보조 큐비트를 사용하여, CCNOT 게이트를 계층적으로 배치함으로써 제어 조건을 축소시킨다.
- **절차**:
    1. 제어 큐비트들을 2개씩 묶어 CCNOT를 적용, 결과를 보조 큐비트에 저장한다.
    2. 이 과정을 반복하여 최종적으로 2개의 보조 큐비트만 남긴다.
    3. 최종 보조 큐비트들을 제어로 하여 타겟 연산을 수행한다.
    4. 연산 후, 보조 큐비트들을 원래 상태($|0\rangle$)로 되돌리기 위해 CCNOT 연산을 역순으로 적용한다.

### 4. 핵심 양자 알고리즘 구현 방법

- **Deutsch/Deutsch-Jozsa/Bernstein-Vazirani**:
  - 입력 상태를 $|0\rangle^{\otimes n}|1\rangle$로 준비하고 Hadamard 변환을 통해 모든 상태의 중첩을 만든다.
  - 오라클 함수 $g$를 통해 상태의 부호를 반전시키며, 다시 Hadamard 변환을 적용하여 간섭 효과를 유도한다.
  - 결과적으로 상쇄 간섭과 보강 간섭을 통해 정답 상태의 확률을 극대화한다.
- **Simon's Algorithm**:
  - 하이브리드 접근 방식을 취하며, 양자 회로를 통해 $s \cdot y = 0 \pmod 2$를 만족하는 $y$값들을 샘플링한다.
  - 이후 고전적인 선형 연립방정식을 풀어 숨겨진 비트스트링 $s$를 찾아낸다.
- **Grover's Search**:
  - **Oracle ($U_w$)**: 마킹된 상태 $|w\rangle$의 부호만 반전시킨다.
  - **Diffusion Operator ($U_s$)**: 모든 상태의 진폭을 평균값에 대해 반전(Reflection about the average)시켜 마킹된 상태의 진폭을 증폭시킨다.
  - 반복 횟수 $\approx \frac{\pi}{4}\sqrt{N}$를 적용하여 성공 확률을 극대화한다.
- **Quantum Fourier Transform (QFT)**:
  - Hadamard 게이트와 제어 위상 게이트($R_\phi$)를 조합하여 상태를 푸리에 도메인으로 변환한다.
  - 수식으로는 다음과 같은 유니터리 변환을 수행한다.
    $$\text{QFT}|x\rangle = \frac{1}{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{2\pi i x k / 2^n} |k\rangle$$

## 📊 Results

본 보고서는 실험 논문이 아닌 튜토리얼이므로, 특정 데이터셋에 대한 성능 측정보다는 알고리즘의 **정확한 구현 및 동작 검증**에 초점을 맞춘다.

- **검증 방법**: QVM 시뮬레이터를 통해 각 알고리즘의 최종 상태(Wavefunction)와 측정 결과(Measurement)를 이론적 예측치와 비교한다.
- **주요 결과**:
  - **Deutsch-Jozsa**: 단 1회의 오라클 호출로 함수가 constant인지 balanced인지 100% 확률로 판별함을 확인하였다.
  - **Bernstein-Vazirani**: 단 1회의 실행으로 숨겨진 비트스트링 $a$를 정확히 추출함을 확인하였다.
  - **Simon's Algorithm**: 여러 번의 측정과 고전적 선형 방정식 풀이를 통해 $s$를 성공적으로 찾아냄을 확인하였다.
  - **Grover Search**: 반복 횟수가 $\frac{\pi}{4}\sqrt{N}$에 가까워질수록 마킹된 상태의 측정 확률이 비약적으로 상승하며, 과도한 반복 시 확률이 다시 낮아지는 주기적 특성을 확인하였다.
  - **QFT**: QFT와 Inverse QFT를 순차적으로 적용했을 때 원래 상태로 복구됨을 확인하여 변환의 유니터리 성질을 검증하였다.

## 🧠 Insights & Discussion

### 1. Hadamard 변환의 핵심적 역할

본 튜토리얼에서 다룬 거의 모든 알고리즘의 핵심은 Hadamard 변환이다. Hadamard 게이트는 단순한 중첩 생성을 넘어, 다음과 같은 역할을 수행한다.

- **병렬 처리**: 모든 가능한 입력 상태를 동시에 처리할 수 있는 기반을 제공한다.
- **기저 변환**: 계산 기저(Computational basis)와 푸리에 기저 사이를 이동시켜, 전역적 특성(함수의 성질, 주기 등)을 국소적 측정값으로 변환한다.

### 2. 보조 큐비트(Ancilla)의 트레이드오프

고차 제어 게이트를 구현할 때 보조 큐비트를 다수 사용하는 전략은 구현의 단순함을 제공하지만, 실제 QPU에서는 큐비트 자원의 희소성으로 인해 매우 비효율적이다. 이는 시뮬레이션 환경과 실제 하드웨어 환경 사이의 큰 간극을 보여주며, 향후 연구에서 큐비트 효율적인 회로 최적화의 필요성을 시사한다.

### 3. 확률적 특성과 하이브리드 접근

Simon's 알고리즘과 같이 결과가 확률적으로 도출되는 경우, 양자 컴퓨팅 단독으로는 해결이 불가능하며 고전 컴퓨팅과의 협력이 필수적이다. 이는 현대 양자 알고리즘의 주류인 VQE나 QAOA와 같은 하이브리드 알고리즘의 기초적인 형태라고 해석할 수 있다.

## 📌 TL;DR

본 논문은 Pyquil 프레임워크를 통해 양자 알고리즘의 이론을 실제 코드로 구현하는 체계적인 튜토리얼을 제공한다. 기초 게이트부터 시작하여 고차 제어 게이트 구현 전략을 제시하고, Deutsch, Grover, QFT 등 학술적으로 중요한 알고리즘들을 단계적으로 구현함으로써 입문자가 양자 프로그래밍의 전 과정을 경험하게 한다. 특히 Hadamard 변환을 통한 상태 제어와 위상 반전(Phase kickback)의 원리를 실제 코드로 검증했다는 점에서 교육적 가치가 크며, 이는 향후 더 복잡한 양자 알고리즘 설계 및 실제 QPU 적용을 위한 필수적인 기초 단계가 될 가능성이 높다.
