# Anomaly Detection with Inexact Labels

Tomoharu Iwata, Machiko Toyoda, Shotaro Tora, Naonori Ueda (2019)

## 🧩 Problem to Solve

본 논문은 **불완전한 이상치 레이블(Inexact Anomaly Labels)**이 주어진 상황에서의 이상치 탐지(Anomaly Detection) 문제를 해결하고자 한다. 일반적으로 이상치 탐지에서는 각 데이터 포인트가 정상인지 이상인지 명확한 레이블이 필요하지만, 실제 환경에서는 이러한 정확한 레이블을 얻기 어려운 경우가 많다.

예를 들어, 서버 시스템의 장애 탐지 시 운영자는 정확한 장애 발생 시점은 모르지만, 특정 시간 범위 내에 장애가 발생했다는 사실만 알 수 있다. 이 경우 해당 시간 범위 내의 모든 데이터 세트에 레이블을 부여하게 되는데, 여기에는 정상 데이터가 포함될 수밖에 없다. 이처럼 **"데이터 세트 내에 적어도 하나 이상의 이상치가 포함되어 있다"**는 정보만 주어진 상태를 불완전한 레이블 상황이라고 정의한다.

논문의 목표는 이러한 불완전한 레이블을 효과적으로 활용하여, 적은 양의 레이블 정보만으로도 높은 탐지 성능을 내는 지도 학습 기반의 이상치 탐지 방법을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Inexact AUC**라는 새로운 평가 지표를 정의하고, 이를 최대화하는 방향으로 모델을 학습시키는 것이다.

1. **Inexact AUC 정의**: 기존의 AUC(Area Under the ROC Curve)를 불완전한 레이블 상황으로 확장하여, 무작위로 선택된 불완전 이상치 세트 내의 최소 한 개의 인스턴스가 정상 인스턴스보다 높은 이상치 점수를 가질 확률로 정의하였다.
2. **하이브리드 학습 구조**: 비지도 학습 기반의 이상치 탐지 메커니즘(예: Autoencoder)과 Inexact AUC 최대화라는 지도 학습 목표를 결합하였다. 이를 통해 레이블이 매우 적은 상황에서도 비지도 학습의 특성을 이용해 강건한 성능을 유지하도록 설계하였다.
3. **클래스 불균형 대응**: AUC 최대화 방식은 본질적으로 클래스 불균형 문제에 강건하므로, 이상치가 매우 드물게 발생하는 이상치 탐지 작업의 특성을 잘 반영한다.

## 📎 Related Works

논문에서는 기존의 접근 방식들을 다음과 같이 분류하고 한계를 지적한다.

- **Multiple Instance Learning (MIL)**: 데이터 세트 단위로 레이블이 부여되는 방식(Positive set은 적어도 하나 이상의 positive instance 포함)을 다룬다. 하지만 기존 MIL 방법들은 클래스 불균형(Class Imbalance) 문제에 취약하다는 한계가 있다.
- **비지도 이상치 탐지 (Unsupervised Anomaly Detection)**: LOF, Isolation Forest, One-Class SVM 등이 있으며, 레이블 정보가 전혀 없을 때 유용하지만 주어진 레이블 정보를 활용할 수 없다는 단점이 있다.
- **지도 이상치 탐지 (Supervised Anomaly Detection)**: 정확한 레이블이 있을 때는 성능이 뛰어나지만, 본 논문에서 다루는 불완전한 레이블(Inexact labels)을 처리할 수 있는 메커니즘이 없다.

## 🛠️ Methodology

### 1. Inexact AUC의 정의
불완전한 이상치 세트를 $B$라고 할 때, **Inexact True Positive Rate (iTPR)**를 다음과 같이 정의한다.

$$iTPR(h) = E_{B \sim p_S} [I(\max_{x \in B} a(x) > h)]$$

여기서 $a(x)$는 이상치 점수 함수, $h$는 임계값, $I(\cdot)$는 지시 함수이다. 이를 바탕으로 **Inexact AUC (iAUC)**는 다음과 같이 정의된다.

$$iAUC = E_{B \sim p_S, x_N \sim p_N} [I(\max_{x \in B} a(x) > a(x_N))]$$

즉, 무작위로 뽑은 불완전 이상치 세트 내의 최대 점수가 무작위로 뽑은 정상 인스턴스의 점수보다 클 확률을 의미한다. 학습을 위한 경험적(empirical) iAUC는 다음과 같이 계산된다.

$$\hat{iAUC} = \frac{1}{|S||N|} \sum_{B_k \in S} \sum_{x_{Nj} \in N} I[\max_{x \in B_k} a(x) > a(x_{Nj})]$$

### 2. 이상치 점수 함수 (Anomaly Score Function)
본 논문에서는 Deep Autoencoder의 **재구성 오차(Reconstruction Error)**를 이상치 점수로 사용한다.

$$a(x; \theta) = \|x - g(f(x; \theta_f); \theta_g)\|^2$$

여기서 $f$는 Encoder, $g$는 Decoder이며, $\theta = \{\theta_f, \theta_g\}$는 모델의 파라미터이다. 정상 데이터와 유사한 데이터는 재구성 오차가 낮고, 본 적 없는 이상치 데이터는 오차가 높다는 특성을 이용한다.

### 3. 목적 함수 (Objective Function)
모델은 정상 데이터의 점수는 낮추고(비지도 학습), $\hat{iAUC}$는 높이는(지도 학습) 방향으로 학습된다. 지시 함수 $I(\cdot)$는 미분 불가능하므로 시그모이드 함수 $\sigma(z) = \frac{1}{1+\exp(-z)}$로 근사하여 다음과 같은 손실 함수 $E$를 최소화한다.

$$E = \frac{1}{|N|} \sum_{x_{Nj} \in N} a(x_{Nj}) - \lambda \frac{1}{|S||N|} \sum_{B_k \in S} \sum_{x_{Nj} \in N} \sigma(\max_{x \in B_k} a(x) - a(x_{Nj}))$$

- **첫 번째 항**: 모든 정상 데이터 $x_{Nj}$의 이상치 점수를 최소화한다. (Unsupervised mechanism)
- **두 번째 항**: 불완전 이상치 세트의 최대 점수와 정상 데이터 점수의 차이를 최대화하여 iAUC를 높인다. (Supervised mechanism)
- **$\lambda$**: 두 항 사이의 균형을 조절하는 하이퍼파라미터이다.

## 📊 Results

### 1. 실험 설정
- **데이터셋**: 합성 데이터셋(Synthetic 2D Gaussian) 및 9개의 공개 이상치 탐지 데이터셋(Annthyroid, KDDCup99 등)을 사용하였다.
- **비교 대상**: 비지도 학습(LOF, OSVM, IF, AE)과 지도 학습(KNN, SVM, RF, NN, MIL, SIF, SAE) 등 총 11가지 방법론과 비교하였다.
- **평가 지표**: 테스트 데이터셋에 대해 정확한 레이블을 사용하여 AUC를 측정하였다.

### 2. 주요 결과
- **합성 데이터셋 결과**: 제안 방법은 $\text{AUC} = 0.982$를 기록하여 AE(0.919), SAE(0.790), MIL(0.905)보다 월등한 성능을 보였다. 특히, 비지도 학습(AE)이 놓치는 이상치와, 불완전 레이블을 정확한 레이블로 오인하는 지도 학습(SAE)의 문제를 동시에 해결하였다.
- **실제 데이터셋 결과**: 9개 데이터셋의 평균 AUC에서 제안 방법이 가장 높은 성능($0.844$)을 기록하였다.
- **레이블 수에 따른 영향**: 훈련에 사용된 불완전 이상치 세트의 수가 증가할수록 성능이 향상되었으며, 세트 내 인스턴스 수가 많아질수록(즉, 레이블이 더 불완전해질수록) AUC가 낮아지는 경향을 보였다.
- **$\lambda$의 영향**: 데이터셋마다 최적의 $\lambda$ 값이 달랐으며, 검증 데이터를 통해 이를 튜닝함으로써 비지도 학습과 지도 학습의 적절한 균형을 찾을 수 있었다.

## 🧠 Insights & Discussion

본 연구의 가장 큰 강점은 **비지도 학습의 일반화 능력과 지도 학습의 정밀함을 결합**했다는 점이다. 

- **강점**: 단순히 레이블을 최대화하는 MIL 방식은 학습 데이터에 없는 새로운 유형의 이상치를 탐지하는 능력이 떨어지지만, 제안 방법은 Autoencoder의 기본 구조를 유지함으로써 "정상 데이터의 분포"를 학습한다. 따라서 레이블로 주어진 이상치뿐만 아니라, 분포상에서 멀리 떨어진 미지의 이상치까지 효과적으로 잡아낼 수 있다.
- **한계 및 가정**: 본 모델은 이상치 점수 함수로 Autoencoder를 가정하였으나, 논문에서는 다른 미분 가능한 함수(VAE, Energy-based models 등)로 대체 가능하다고 언급한다. 다만, 실제 성능은 선택한 기본 비지도 모델의 성능에 의존적일 가능성이 크다.
- **비판적 해석**: 하이퍼파라미터 $\lambda$에 따라 모델이 완전히 비지도 학습(AE)처럼 동작하거나 완전히 MIL처럼 동작할 수 있다. 따라서 $\lambda$를 최적화하기 위한 충분한 검증 데이터가 확보되어야 하며, 이에 대한 민감도 분석이 더 상세히 이루어졌다면 더 견고한 결론을 낼 수 있었을 것이다.

## 📌 TL;DR

본 논문은 불완전한 이상치 레이블(세트 내 적어도 하나가 이상치인 조건)을 처리하기 위해 **Inexact AUC**라는 지표를 제안하고, 이를 최대화하는 **지도-비지도 하이브리드 학습 프레임워크**를 구축하였다. Autoencoder의 재구성 오차를 기본 점수로 사용하고, 정상 데이터 점수 최소화와 iAUC 최대화를 동시에 수행함으로써, 매우 적은 양의 불완전한 레이블만으로도 기존 비지도/지도/MIL 방법론보다 뛰어난 이상치 탐지 성능을 입증하였다. 이 연구는 레이블 획득 비용이 높거나 정확한 시점 파악이 어려운 실제 산업 현장의 이상치 탐지 시스템에 적용될 가능성이 매우 높다.
