# Deep Active Learning for Anomaly Detection

Tiago Pimentel, Marianne Monteiro, Adriano Veloso, Nivio Ziviani (2020)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 비지도 이상치 탐지(Unsupervised Anomaly Detection)의 본질적인 한계인 '이상치에 대한 사전 정의(prior)의 부재'이다. 일반적으로 이상치는 인간 전문가에게는 직관적으로 이해하기 쉬우나, 이를 수학적으로 정의하는 것은 매우 어렵다.

비지도 학습 기반의 이상치 탐지 모델은 데이터의 분포를 학습하여 저확률 영역을 이상치로 간주하지만, 이는 모델의 용량(capacity)에 따라 결과가 달라지는 문제를 가진다. 예를 들어, 모델의 용량이 너무 높으면 밀집된 형태의 이상치(clustered anomalies)까지 정상 패턴으로 학습해버릴 수 있고, 반대로 너무 낮으면 저밀도 이상치(low density anomalies)를 구분하지 못할 수 있다.

따라서 본 논문의 목표는 기존의 비지도 딥러닝 모델에 전문가의 피드백을 효율적으로 통합할 수 있는 Active Learning 체계를 구축하여, 적은 양의 레이블만으로도 정상 데이터와 이상치를 효과적으로 분리하는 방법을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 가장 중심적인 기여는 기존의 비지도 이상치 탐지 모델을 능동형(active) 모델로 변환해주는 **UAI(Unsupervised to Active Inference) 레이어**의 제안이다.

UAI 레이어의 핵심 직관은 비지도 모델이 이미 학습한 **잠재 표현(latent representation)**과 **이상치 점수(anomaly score)**라는 두 가지 강력한 자산을 결합하여 분류기의 입력으로 사용하는 것이다. 이를 통해 모델은 완전히 처음부터 학습하는 것이 아니라, 비지도 학습으로 구축된 데이터 매니폴드(manifold) 위에서 전문가의 피드백을 통해 이상치 영역을 빠르게 식별할 수 있게 된다.

## 📎 Related Works

이상치 탐지 연구는 크게 세 가지 방향으로 나뉜다:
1. **Supervised**: 정상과 이상치가 모두 레이블링된 데이터셋을 사용하지만, 실제 환경에서는 이상치 데이터 확보가 매우 어렵다.
2. **Semi-Supervised (Novelty Detection)**: 정상 데이터만으로 학습하여 이와 다른 데이터를 탐지한다.
3. **Unsupervised**: 레이블 없이 데이터 자체의 분포를 통해 탐지하며, 본 논문이 집중하는 영역이다.

기존의 Active Anomaly Detection 연구들(LODA-AAD, Tree-AAD 등)은 존재하지만, 본 논문의 저자들은 딥러닝의 잠재 표현 학습 능력과 Active Learning을 결합한 end-to-end 아키텍처를 제안했다는 점에서 차별성을 갖는다고 주장한다. 특히, DAGMM과 같은 최신 비지도 모델들이 가진 하이퍼파라미터 민감도 문제를 전문가의 피드백을 통해 완화하고자 한다.

## 🛠️ Methodology

### 1. UAI Layer
UAI 레이어는 비지도 모델의 상단에 추가되는 단순한 분류기(Classifier)이다. 이 레이어는 모델이 생성한 잠재 표현 $l(x)$와 비지도 방식의 이상치 점수 $s(x)$를 결합하여 입력으로 받는다. 본 논문에서는 로지스틱 회귀(Logistic Regression)를 사용하였으며, 최종 이상치 확률 $\hat{p}(y|x)$는 다음과 같이 계산된다:

$$\hat{p}(y|x) \propto s_{uai}(x) = \sigma(W_{act}[l(x); s(x)] + b_{act})$$

여기서 $W_{act}$는 선형 변환 행렬, $b_{act}$는 편향(bias), $\sigma(\cdot)$는 시그모이드 함수이다. 이때 학습 과정에서 기울기(gradient)는 잠재 표현 $l$로는 흐르지만, 이상치 점수 $s$로는 흐르지 않도록 설정한다. 이는 $s$가 미분 불가능할 수 있기 때문이다.

### 2. 기반 아키텍처 (Base Architectures)
UAI 레이어를 적용하기 위해 두 가지 기본 모델을 사용한다:

- **Denoising Autoencoder (DAE)**:
  입력 $x$에 노이즈 $\epsilon$을 추가하여 잠재 공간 $l$로 인코딩한 후 다시 복원하는 구조이다. 손실 함수는 복원 오차의 $L_2$ 노름을 최소화하며, 이상치 점수는 다음과 같이 정의된다:
  $$s_{DAE}(x) = ||x - \hat{x}||_2^2$$

- **Multi-Layer Perceptron (Class)**:
  데이터의 일부 특징을 통해 클래스 레이블을 예측하도록 학습하며, 이때 발생하는 교차 엔트로피(cross entropy) 손실 값을 이상치 점수로 사용한다:
  $$s_{Class}(x) = H(x_y, \hat{x}_y)$$

### 3. 전체 학습 절차 및 손실 함수
전체 모델의 손실 함수 $L_{full}$은 기반 모델의 손실 $L_{base}$와 UAI 레이어의 손실 $L_{uai}$의 합으로 구성된다:

$$L_{full} = L_{uai} + L_{base}$$
$$L_{uai} = \mathbb{1} \cdot H(y, s_{uai}(x))$$

여기서 $\mathbb{1}$은 전문가에 의해 레이블링된 데이터인 경우에만 1이 되고, 그렇지 않으면 0이 되는 지시 함수이다.

**학습 프로세스:**
1. 기반 모델을 비지도 방식으로 사전 학습(pre-train)한다.
2. 현재 모델이 가장 이상치일 가능성이 높다고 판단하는 $k$개의 샘플을 선택하여 전문가에게 전송한다(Most-likely positive sampling).
3. 전문가의 피드백(정상/이상)을 받아 $L_{full}$을 통해 모델을 업데이트한다.
4. 정해진 예산(budget $b$)이 소진될 때까지 이 과정을 반복한다.

## 📊 Results

### 1. 합성 데이터 실험 (MNIST, Fashion-MNIST)
저자들은 이상치의 성격에 따라 세 가지 데이터셋을 구축하여 실험하였다:
- **$\text{MNIST}_0, \text{MNIST}_{0-2}$**: 특정 숫자를 제거하고 일부만 남겨 '밀집된 이상치(clustered anomalies)'를 모사함.
- **$\text{MNIST}_{hard}$**: 약한 분류기가 오분류한 샘플들을 이상치로 정의하여 '저밀도 이상치(low density anomalies)'를 모사함.

**결과:**
- 순수 비지도 모델인 $\text{Class}$는 밀집된 이상치는 잘 찾지만 저밀도 이상치에 취약했고, $\text{DAE}$는 그 반대의 경향을 보였다.
- 반면, UAI 레이어가 결합된 $\text{Class}_{uai}$와 $\text{DAE}_{uai}$는 모든 유형의 이상치에서 강건하게 높은 성능을 보였다. 이는 전문가의 피드백이 모델의 편향을 보정했음을 의미한다.

### 2. 실제 데이터 실험
UCI 머신러닝 저장소의 실제 데이터셋(KDDCUP, Arrhythmia, Covtype 등)을 사용하여 F1-score를 측정하였다. 비교 대상으로는 DAGMM(Clean/Dirty), LODA-AAD, Tree-AAD 등이 포함되었다.

**결과:**
- $\text{DAE}_{uai}$는 대부분의 데이터셋에서 최상위권의 성능을 기록하였다. 특히 이상치 비율이 매우 낮은 Covtype 데이터셋에서 다른 모델들보다 월등한 성능 향상을 보였다.
- 이는 단순한 기반 모델을 사용하더라도 UAI 레이어를 통한 능동적 학습이 강력한 효과를 낸다는 것을 입증한다.

## 🧠 Insights & Discussion

본 논문의 핵심 통찰은 **잠재 표현 $l(x)$로 흐르는 기울기의 중요성**에 있다. 시각화 분석 결과, 초기에는 정상 데이터와 이상치가 잠재 공간에서 섞여 있으나, 전문가의 레이블이 소량 제공됨에 따라 UAI 레이어의 오차가 기반 모델의 인코더로 전달되어 잠재 표현 자체가 이상치를 더 잘 분리하도록 재구성됨을 확인하였다.

**한계점 및 논의사항:**
- 현재 UAI 레이어는 단순한 로지스틱 회귀를 사용하고 있어, 더 복잡한 분류기 아키텍처를 적용했을 때의 성능 향상 여부는 확인되지 않았다.
- 전문가가 잘못된 레이블을 제공했을 때(label noise) 모델이 얼마나 강건하게 대응할 수 있는지에 대한 분석이 누락되어 있다.
- 또한, 단순히 '가장 이상치일 것 같은' 샘플만 뽑는 방식 외에, 모델이 불확실해하는 샘플을 뽑는 전략(Uncertainty sampling)과의 비교가 이루어지지 않았다.

## 📌 TL;DR

본 논문은 비지도 이상치 탐지 모델의 상단에 잠재 표현과 이상치 점수를 입력으로 받는 **UAI 레이어**를 추가하여, 소량의 전문가 피드백만으로 성능을 극대화하는 **Deep Active Learning** 방법을 제안한다. 이 방법은 기반 모델이 가진 고유의 편향(밀집형 vs 저밀도형 탐지 능력 차이)을 극복하고, 실제 데이터셋에서 SOTA 모델들과 대등하거나 더 뛰어난 성능을 보였다. 향후 이상치 탐지 분야에서 고비용의 레이블링 문제를 해결하고 모델의 해석 가능성을 높이는 데 기여할 가능성이 크다.
