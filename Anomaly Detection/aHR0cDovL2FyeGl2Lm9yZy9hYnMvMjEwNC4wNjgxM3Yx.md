# Global Information Guided Video Anomaly Detection

Hui Lv, Chunyan Xu, Zhen Cui (2020)

## 🧩 Problem to Solve

비디오 이상 탐지(Video Anomaly Detection, VAD)는 감시 시스템이나 결함 탐지 시스템과 같은 실세계 응용 분야에서 매우 중요하다. 그러나 이상 징후(Anomaly)의 정의가 주관적이고 복잡하며, 비디오 내의 정확한 이상 시점을 표시하는 시계열 어노테이션(Temporal Annotation)을 생성하는 데 막대한 비용과 시간이 소요된다는 점이 주요 문제로 지적된다.

기존의 많은 VAD 접근 방식은 특징 추출과 이상 탐지를 분리하는 2단계(Two-stage) 방식을 채택하고 있어 절차가 번거롭고 효율성이 떨어진다. 또한, 기존 모델들은 Global Pooling 연산을 통해 공간 해상도를 $1 \times 1$로 축소함으로써, 객체 간의 관계나 외형적 특성과 같은 중요한 공간 정보(Spatial Information)를 상당 부분 손실하는 한계가 있다. 따라서 본 논문은 비디오 수준의 약한 라벨(Video-level weak labels)만을 활용하여 이상 징후를 효율적으로 분류하고 국지화(Localization)할 수 있는 end-to-end 프레임워크를 구축하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 비디오 전체의 전역적 패턴을 먼저 파악하고, 이를 가이드로 삼아 공간적 특징 중에서 이상 징후와 가장 관련이 깊은 정보를 선택적으로 추출하는 것이다.

1. **End-to-End VAD 프레임워크**: 특징 추출부터 이상 탐지까지 하나의 네트워크로 통합하여 편의성과 효율성을 높였다.
2. **Global Information Guided (GIG) 모듈**: 비디오 레벨의 라벨을 활용해 전역 패턴 큐(Global Pattern Cue, GPC)를 생성하고, 이를 채널 주의 집중(Channel-wise Attention) 메커니즘에 적용하여 특징 표현을 강화한다.
3. **Spatial Reasoning (SR) 모듈**: 전역 패턴 큐와 공간 특징 벡터 간의 유사도를 측정하여, 이상 탐지에 가장 중요한 공간적 단서를 동적으로 선택한다.
4. **Video-Segment 목적 함수 ($\ell_{vs}$)**: 비디오 수준의 감독과 세그먼트 수준의 감독을 동시에 수행할 수 있는 새로운 손실 함수를 제안하여 전역 정보의 마이닝 능력을 극대화했다.

## 📎 Related Works

기존의 약지도 학습 기반 VAD 연구들은 주로 다중 인스턴스 학습(Multiple Instance Learning, MIL) 기법을 도입하였다. Sultani 등은 랭킹 손실 함수를 통해 이상 탐지를 MIL 문제로 정의하였으며, Zhu 등은 주의 집중(Attention) 메커니즘을 통해 이상 부위를 더 잘 국지화하려 시도하였다. 또한 Zhong 등은 반복적인 정제를 통해 이상 위치에 대한 감독 신호를 생성하는 방법을 제안하였다.

그러나 이러한 기존 연구들은 전처리 단계에서 Global Pooling을 수행하여 공간 해상도를 완전히 제거하는 경향이 있다. 이로 인해 강도(Robbery), 절도(Theft), 총기 사고(Shooting)와 같이 객체 간의 상호작용이나 특정 사물의 외형이 결정적인 역할을 하는 이상 상황을 구분하는 데 한계가 있다. 본 논문은 이러한 공간 정보의 부재를 해결하기 위해 GPC를 통한 공간 추론 과정을 도입함으로써 기존 방식과 차별화된다.

## 🛠️ Methodology

### 전체 파이프라인
본 모델은 Backbone 네트워크(TPN)를 통해 시공간 특징 맵(Spatio-temporal feature maps)을 추출하고, 이를 GIG 모듈과 SR 모듈에 순차적으로 통과시켜 최종적으로 이상 점수를 산출하는 구조를 가진다.

### Global Information Guided (GIG) Module
GIG 모듈은 비디오 전체의 맥락을 파악하여 특징을 강화하는 역할을 한다.

1. **전역 패턴 큐(GPC) 생성**: 입력된 세그먼트 특징 맵 $X$에 대해 3D 전역 최대 풀링(Global Max Pooling-3D) 연산을 적용하여 전역 패턴 큐 벡터 $g$를 얻는다.
   $$g = \psi(X)$$
2. **특징 강화**: 생성된 $g$를 채널 주의 집중 메커니즘의 가이드로 사용하여 입력 특징 $X$를 강화한다.
   $$\hat{X} = \sigma(g) \odot X + X$$
   여기서 $\sigma$는 정규화 함수이며, $\odot$은 채널별 곱셈을 의미한다. Skip connection을 통해 원래의 정보를 보존한다.
3. **비디오 수준 감독**: GPC 벡터 $g$를 분류 헤드 $fc_1$에 통과시켜 비디오에 이상 징후가 존재하는지 여부를 판단하는 이진 점수 $s^*_g$를 계산하고, 이를 통해 $\ell^*_g$ 손실을 구한다.
   $$s^*_g = \max_{i=1}^C (\sigma(fc_1(g))_i)$$
   $$\ell^*_g = -(y^* \log s^*_g + (1-y^*) \log(1-s^*_g))$$

### Spatial Reasoning (SR) Module
SR 모듈은 GPC를 활용해 공간 도메인에서 가장 관련성이 높은 특징을 선택한다.

1. **관계 점수 계산**: 전역 패턴 큐 $g$와 강화된 시공간 특징 $\hat{X}$의 각 벡터 간의 코사인 유사도(Cosine Similarity)를 측정하여 관계 점수 $r_{i,j}$를 산출한다.
   $$r_{i,j} = R(g, \hat{x}_{i,j})$$
2. **공간 벡터 선택**: 상위 $k$개의 관계 점수를 가진 공간 벡터들을 선택하고 이를 평균내어 해당 세그먼트의 공간 패턴 벡터 $\hat{x}_s$를 생성한다.
   $$\hat{x}_s = \frac{1}{k} \sum_{i,j \in \text{top\_k}} \text{top}_k(\hat{x}_{i,j}, r_{i,j})$$
3. **세그먼트 점수 산출**: $\hat{x}_s$를 분류 헤드 $fc_2$에 통과시켜 세그먼트별 이상 점수 $s$를 얻는다.
   $$s = \sigma(fc_2(\hat{x}_s))$$

### 학습 목표 및 손실 함수
최종 손실 함수 $\ell$은 비디오-세그먼트 손실($\ell_{vs}$)과 희소성 제약(Sparse constraint)의 합으로 정의된다.

1. **Video-Segment Loss ($\ell_{vs}$)**:
   $$\ell_{vs} = \ell_s + \mu_1 \ell^*_s + \mu_2 \ell^*_g$$
   - $\ell_s$: 멀티 클래스 이상 분류 손실.
   - $\ell^*_s$: 세그먼트 기반의 비디오 수준 이진 분류 손실.
   - $\ell^*_g$: GPC 기반의 비디오 수준 이진 분류 손실.
2. **Sparse Loss ($\ell_{sparse}$)**: 이상 징후가 비디오 내에서 짧은 시간 동안만 발생한다는 특성을 반영한다.
   $$\ell_{sparse} = \sum_{i=1}^T s_i$$
3. **최종 손실 함수**:
   $$\ell = \ell_{vs} + \mu_3 \ell_{sparse}$$

## 📊 Results

### 실험 설정
- **데이터셋**: CityScene 챌린지 데이터셋을 사용하였다. 12가지의 실제 도시 이상 상황(사고, 싸움, 화재 등)을 포함하며, 훈련 데이터는 비디오 레벨의 라벨만 제공된다.
- **구현 상세**: Backbone으로 Temporal Pyramid Network (TPN)의 ResNet50 버전을 사용하였으며, Kinetics-400으로 사전 학습된 가중치를 사용하였다. Adagrad 옵티마이저를 통해 100 에폭 동안 학습하였다.
- **평가 지표**: 일반 이상 탐지(General Anomaly Detection)는 AUC를, 특정 이상 탐지(Specific Anomaly Detection)는 MF1-score를 사용하여 측정하였다.

### 실험 결과
- **일반 이상 탐지 (Task 1)**: AUC 84.09%를 기록하며 ActionLab(70.92%)과 Orange-Control(31.65%)보다 우수한 성능을 보였다. 최상위 모델과는 다소 차이가 있으나 경쟁력 있는 수치이다.
- **특정 이상 탐지 (Task 2)**: MF1 52.33%를 기록하며 챌린지 전체에서 3위를 차지하였다. 특히 4위 모델 대비 약 7%의 성능 향상을 보였다.
- **효율성**: 2080Ti GPU 기준 초당 약 100프레임을 처리할 수 있는 빠른 추론 속도를 보였다.

## 🧠 Insights & Discussion

본 논문은 비디오-레벨의 약한 라벨만을 사용하여 end-to-end로 학습 가능한 효율적인 VAD 프레임워크를 제시하였다. 특히 기존 연구들이 간과했던 공간 정보의 중요성을 인식하고, 전역 패턴 큐(GPC)를 통해 공간적 특징을 필터링하는 전략을 도입함으로써 특정 이상 상황에 대한 분류 성능(MF1)을 크게 향상시킨 점이 고무적이다.

다만, 일반 이상 탐지(AUC) 성능이 최상위권 모델에 비해 다소 낮다는 점은 한계로 남는다. 이는 제안된 모델이 특정 클래스의 구분 능력은 뛰어나지만, 전반적인 '정상 vs 이상'의 이진 구분 능력에서는 여전히 개선의 여지가 있음을 시사한다. 또한, TPN이라는 강력한 사전 학습 모델에 의존하고 있어, Backbone의 성능이 전체 시스템의 상한선을 결정할 가능성이 크다.

## 📌 TL;DR

본 연구는 비디오 수준의 약한 라벨만을 활용하는 **End-to-End 비디오 이상 탐지 프레임워크(GIG)**를 제안한다. 전역 패턴 큐(GPC)를 생성하여 특징 표현을 강화하고, 이를 통해 이상 징후와 가장 관련이 높은 공간적 특징만을 선택적으로 추출하는 **Spatial Reasoning** 메커니즘을 도입하였다. 이를 통해 CityScene 챌린지의 특정 이상 탐지 작업에서 3위의 성적을 거두었으며, 실시간 적용이 가능한 빠른 추론 속도를 입증하였다.
