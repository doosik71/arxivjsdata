# ANOMALY DETECTION WITH SDAE

Benjamin Smith, Kevin Cant, Gloria Wang (2020)

## 🧩 Problem to Solve

본 연구에서 해결하고자 하는 문제는 건물 에너지 데이터셋에서 발생하는 이상치(Anomaly)를 자동으로 탐지하고 제거하는 것이다. 실세계 데이터, 특히 센서로부터 수집되는 대규모 데이터셋에는 센서의 결함, 수동 입력 오류, 프로세스 오류 등으로 인한 이상치가 포함될 가능성이 높다. 이러한 이상치는 데이터 전처리 단계에서 적절히 제거되지 않을 경우, 머신러닝 모델이 데이터의 근본적인 분포를 학습하는 것을 방해하여 모델의 성능을 저하시키거나, 의료 분야와 같은 민감한 사례에서는 치명적인 잘못된 결과를 초래할 수 있다.

특히 대량의 데이터가 스트림 형태로 수집되는 환경에서 도메인 전문가가 일일이 수동으로 이상치를 찾아내는 것은 노동 집약적이며 비효율적이다. 따라서 본 논문의 목표는 Autoencoder(AE)를 활용하여 이러한 이상치 탐지 과정을 자동화하고, 특히 지도 학습(Supervised Learning) 요소를 결합한 Supervised Deep Autoencoder(SDAE)가 기존의 비지도 학습 기반 모델보다 이상치 탐지 성능을 향상시킬 수 있는지 검증하는 것이다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 Autoencoder의 잠재 공간(Latent Space) 표현력을 높이기 위해 재구성 손실(Reconstruction Loss)과 지도 손실(Supervised Loss)을 동시에 최적화하는 것이다. 일반적으로 Autoencoder는 입력 데이터를 압축했다가 복원하는 과정을 통해 데이터의 주요 특징을 학습하는 비지도 학습 모델이다. 하지만 본 논문에서는 잠재 공간 층(Latent Space Layer)에서 레이블 정보를 활용한 지도 학습 손실을 추가함으로써, 이상치와 정상 데이터 간의 결정 경계를 더욱 명확하게 구축하고자 하였다. 이를 통해 모델이 단순한 데이터 복원을 넘어, 클래스 구분(정상 vs 이상)에 최적화된 특징 표현을 추출하도록 설계하였다.

## 📎 Related Works

기존의 이상치 탐지 분야에서는 PCA(Principal Component Analysis) 시스템이 주로 사용되었으나, 이후 비선형 차원 축소가 가능한 Autoencoder가 PCA보다 우수한 성능을 보임이 입증되었다. 일반적인 Autoencoder 기반의 이상치 탐지 방식은 정상 데이터만을 사용하여 모델을 학습시킨 뒤, 새로운 입력에 대해 재구성 오차(Reconstruction Error)를 계산하고, 이 오차가 특정 임계값(Threshold)을 초과할 경우 이를 이상치로 판별하는 방식을 취한다.

관련 연구들에서는 Denoising AE, Deep AE, Semi-supervised AE, Variational AE 등이 제안되었으며, 얼굴 검증, 위성 데이터 검증, 뇌 구조 패턴 분석 등 다양한 분야에 적용되었다. 또한, 일부 연구에서는 AE에 지도 학습 구성 요소를 추가하는 것이 이상치 식별 능력을 향상시킨다는 점이 제시되었으며, 이는 클래스 레이블에 맞춤화된 더 나은 표현(Representation)을 추출할 수 있게 하기 때문이다.

## 🛠️ Methodology

### 전체 파이프라인 및 구조
본 연구에서는 세 가지 모델인 Simple AE, Deep AE, 그리고 Supervised Deep Autoencoder(SDAE)를 비교 분석한다. 모든 모델은 입력 데이터를 저차원의 잠재 공간으로 압축하는 Encoder와, 이를 다시 원래 차원으로 복원하는 Decoder로 구성된다.

1.  **Simple AE**: 입력 데이터를 6개 특징의 은닉층을 거쳐 2차원의 잠재 공간으로 압축한 후 다시 13차원으로 복원하는 구조이다.
2.  **Deep AE**: Simple AE보다 더 많은 은닉층을 가지며, 입력 차원을 단계적으로 1씩 줄여가며 2차원의 잠재 공간까지 압축한 후 다시 복원하는 심층 구조이다.
3.  **SDAE**: Deep AE와 구조적으로 동일하지만, 잠재 공간 층에서 추출된 인코딩 값이 지도 손실(Supervised Loss) 함수로 전달된다. 즉, 재구성 손실과 지도 손실이 결합된 통합 손실 함수를 사용하여 역전파(Back-propagation)를 수행한다.

### 데이터셋 구성
Kaggle의 ASHRAE 건물 에너지 데이터셋을 사용하였으며, 건물 메타데이터, 날씨 데이터, 미터 읽기 값 등 총 13개의 특징(Feature)을 학습에 사용하였다. 학습을 위해 5개 건물(18, 22, 40, 49, 490)에서 추출한 40,000개의 레이블링된 데이터를 사용하였으며, 이상치 여부는 도메인 전문가에 의해 수동으로 레이블링되었다.

### 손실 함수 및 학습 절차
모델 학습에는 다음과 같은 손실 함수가 사용된다.

**1. 재구성 손실 (Reconstruction Loss)**
입력값 $X$와 복원된 값 $\hat{X}$ 사이의 평균 제곱 오차(MSE)를 측정하여 데이터 복원 능력을 평가한다.
$$MSE = \frac{1}{n} \sum_{i=1}^{n} (X - \hat{X})^2$$

**2. 지도 손실 (Supervised Loss)**
SDAE에서 사용되며, 예측된 클래스 확률과 실제 레이블 $y$ 사이의 이진 교차 엔트로피(BCE)를 계산한다.
$$BCE = -(y \log(p) + (1-y) \log(1-p))$$
이때, 잠재 공간의 2차원 특징을 강조하기 위해 이상치는 $(1, 0)$, 정상 데이터는 $(0, 1)$의 튜플 형태로 레이블링하여 가중치를 부여하였다.

**3. 학습 설정**
- 하드웨어: Microsoft Azure VM (24 CPUs, 4 NVIDIA Tesla K80 GPUs)
- 배치 크기: 32 (큰 배치 크기에서는 학습이 불안정하여 작게 설정함)
- 학습 횟수: 1,000 epochs (SDAE의 경우 추가로 5,000 epochs 실험 수행)

## 📊 Results

### 실험 설정 및 지표
테스트 세트로 Building 408과 Building 733의 데이터를 사용하였다. 이상치 판별은 모델의 손실 분포를 분석하여 수동으로 설정한 상한 및 하한 임계값(Threshold)을 기준으로 수행하였다. 평가지표로는 True Positives(정탐), False Positives(오탐), Missed(미탐) 등을 확인하였다.

### 주요 결과
**1. Building 733 결과**
세 모델 모두 매우 명확한 손실 분포 차이를 보였으며, 적절한 임계값을 설정했을 때 모든 이상치를 100% 정확하게 탐지하였다. 이는 모델들이 해당 건물의 데이터 분포를 효과적으로 학습했음을 의미한다.

**2. Building 408 결과**
Building 733에 비해 이상치 탐지가 훨씬 어려웠다.
- **Simple AE**: 이상치의 53% 탐지.
- **Deep AE**: 이상치의 57.5% 탐지.
- **SDAE**: 이상치의 56% 탐지.

단, SDAE의 하한 임계값을 아주 미세하게($1e-7$) 조정했을 때, 정탐(True Positives) 수는 162개(약 59%)로 세 모델 중 가장 높았으나, 동시에 오탐(False Positives) 수도 455개로 크게 증가하였다.

### 요약 표 (Building 408)
| Criteria | Simple AE | Deep AE | SDAE |
| :--- | :---: | :---: | :---: |
| True Positives | 147 | 157 | 153 |
| False Positives | 0 | 13 | 398 |
| Missed | 126 | 116 | 120 |

## 🧠 Insights & Discussion

### 모델 성능 분석
Building 733에서는 모든 모델이 성공적이었으나, Building 408에서는 성능이 저하되었다. 이는 Building 408이 학습에 사용된 건물들과 데이터 분포가 크게 다르거나, 해당 건물만이 가진 고유한 특성이 존재하기 때문으로 추측된다. 모든 AE 변형 모델에서 공통적으로 성능 저하가 나타난 점이 이를 뒷받침한다.

### SDAE의 가치
수치상으로는 Deep AE가 정탐과 오탐의 균형 측면에서 우수해 보일 수 있다. 하지만 연구자는 이상치 하나를 놓치는 비용이 정상 데이터를 잘못 제거하는 비용보다 훨씬 크다고 가정한다면, 가장 많은 정탐(162개)을 기록할 수 있는 SDAE가 가장 우수하다고 해석한다. 또한 SDAE의 최종 손실 값이 다른 모델보다 낮게 나타난 점은, 지도 학습의 결합이 입력 데이터 분포의 더 정확한 표현을 가능하게 했음을 시사한다.

### 한계 및 비판적 해석
본 연구의 가장 큰 한계는 임계값(Threshold)을 수동(Trial and Error)으로 설정했다는 점이다. 이는 결과의 객관성을 떨어뜨릴 수 있으며, 동적인 임계값 설정 알고리즘이 도입되어야 할 필요가 있다. 또한, 전문가가 레이블링한 데이터 자체에 오류가 있을 수 있으며, 이로 인해 모델이 실제로는 정답을 맞혔음에도 오탐(False Positive)으로 기록되었을 가능성이 있다.

## 📌 TL;DR

본 논문은 건물 에너지 데이터의 이상치 탐지를 위해 단순 AE, Deep AE, 그리고 지도 학습이 결합된 SDAE를 비교 분석하였다. 실험 결과, 일반적인 상황에서는 Deep AE가 안정적이었으나, 이상치 탐지율을 극대화해야 하는 상황에서는 지도 손실을 결합한 **SDAE가 가장 높은 정탐률을 기록하며 우수한 성능**을 보였다. 이 연구는 Autoencoder의 잠재 공간에 지도 학습 요소를 도입하는 것이 복잡한 데이터셋의 이상치 탐지 능력을 향상시키는 유효한 방법임을 입증하였으며, 향후 동적 임계값 설정 및 전체 데이터셋 레이블링을 통한 고도화 가능성을 제시하였다.
