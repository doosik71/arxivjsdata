# ASC-Net: Adversarial-based Selective Network for Unsupervised Anomaly Segmentation

Raunak Dey and Yi Hong (2021)

## 🧩 Problem to Solve

본 논문은 레이블이 없는 데이터셋에서 비정상 영역을 분할하는 **Unsupervised Anomaly Segmentation** 문제를 해결하고자 한다. 의료 영상 분석과 같은 분야에서는 데이터에 수동으로 레이블을 지정하는 비용이 매우 크기 때문에, 비지도 학습 기반의 분할 방식이 매우 중요하다.

기존의 AnoGAN 계열 알고리즘들은 GAN을 이용해 정상 이미지의 매니폴드(manifold)를 학습하고, 쿼리 이미지에 대해 잠재 공간(latent space)에서 가장 유사한 정상 이미지를 복원하여 그 차이를 통해 이상치를 탐지하는 방식을 사용한다. 그러나 이러한 **Reconstruction-based** 접근 방식은 다음과 같은 한계가 있다.

1. 정상 이미지의 정밀한 복원(faithful reconstruction) 품질이 이상치 탐지 성능에 직접적인 영향을 미친다.
2. 복원 과정에서 반복적인 잠재 공간 탐색이 필요하여 효율성이 떨어진다.

따라서 본 논문의 목표는 완벽한 이미지 복원에 의존하지 않고, 사용자가 제공한 정상 이미지 분포(reference distribution)를 기반으로 이미지를 두 개의 영역으로 분리하여 비정상 영역을 직접적으로 찾아내는 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 이미지를 **Selective Cutting**이라는 개념을 통해 두 개의 컷(cut)으로 분할하는 것이다.

- **Selective Cutting**: 입력 이미지를 '정상 분포에 해당하는 부분'과 '그렇지 않은 나머지 부분'으로 나눈다.
- **Adversarial Learning의 도입**: 사용자가 정의한 정상 이미지 집합(reference distribution)을 판별기(Discriminator)가 학습하게 함으로써, 네트워크의 한쪽 브랜치가 강제로 정상 이미지의 특성을 따르도록 유도한다.
- **복원 의존성 탈피**: 기존 방법들이 '정상 이미지로의 완벽한 복원'에 집중한 반면, ASC-Net은 이미지를 의미론적으로 두 부분으로 분리하는 데 집중하며, 복원은 이 두 부분을 다시 합치는 단순한 과정으로 처리한다. 이를 통해 복원 품질이 낮더라도 이상치 분할 성능을 유지할 수 있다.

## 📎 Related Works

논문에서는 **AnoGAN** 및 **f-AnoGAN**과 같은 GAN 기반의 이상치 탐지 연구들을 언급한다. 이들은 정상 데이터의 분포를 학습하여 입력 이미지와 복원된 이미지 사이의 잔차(residual)를 통해 이상치를 식별한다. 하지만 이러한 방법들은 복원 성능이 낮을 경우 정밀한 분할이 어렵다는 한계가 있다.

또한, 본 연구는 **U-Net**의 인코더-디코더 구조와 **CompNet**의 다중 업샘플링 브랜치 구조를 차용하여, 하나의 인코더에서 나온 특징을 바탕으로 두 개의 서로 다른 성격의 이미지 컷을 생성하도록 설계함으로써 기존의 단순 복원 모델과 차별화를 두었다.

## 🛠️ Methodology

### 전체 시스템 구조

ASC-Net은 크게 세 가지 구성 요소로 이루어져 있다: **Main Module ($M$)**, **Discriminator ($D$)**, 그리고 **Thresholding ($T$)**.

1. **Main Module ($M$)**: U-Net과 유사한 구조로, 하나의 인코더($E$)와 두 개의 디코더($C_f, C_w$)로 구성된다.
    - **Fence Cut ($C_f$)**: 사용자가 제공한 정상 이미지 분포($R_d$)를 따르는 이미지를 생성한다. 이 출력물 $I_{fc}$는 판별기 $D$를 속이도록 학습된다.
    - **Wild Cut ($C_w$)**: $I_{fc}$에 포함되지 않은 나머지 이미지 콘텐츠를 캡처하여 $I_{wc}$를 생성한다.
    - **Reconstructor ($R$)**: $I_{fc}$와 $I_{wc}$를 결합(concatenation)한 후 $1 \times 1$ Convolution과 Sigmoid 활성화 함수를 거쳐 원래의 입력 이미지 $I_{in}$으로 복원한 $I_{ro}$를 생성한다.

2. **Discriminator ($D$)**: 생성된 $I_{fc}$가 실제 정상 이미지 분포 $R_d$에 속하는지 아니면 가짜인지를 판별한다. 이를 통해 $C_f$가 정상 이미지의 특성을 학습하도록 강제한다.

3. **Thresholding ($T$)**: 복원된 이미지 $I_{ro}$의 히스토그램을 분석하여 픽셀 수준에서 정상과 비정상을 분리한다. 히스토그램의 피크(peak) 지점을 임계값으로 사용하여 클러스터링을 수행한다.

### 손실 함수 (Loss Functions)

네트워크는 다음의 네 가지 손실 함수를 통해 학습된다.

- **Fence Cut Loss ($\text{Loss}_{C_f}$)**: $C_f$가 $D$를 속여서 가짜 이미지를 진짜라고 믿게 만들 때 최소화된다.
$$\text{Loss}_{C_f} = \frac{1}{n} \sum_{i=1}^{n} |D(I_{fc}^{(i)}) - 1|$$

- **Wild Cut Loss ($\text{Loss}_{C_w}$)**: $I_{fc}$와 $I_{wc}$가 서로 겹치지 않고 상호 보완적인 관계(disjoincy)를 갖도록 하는 positive Dice score를 최소화한다.
$$\text{Loss}_{C_w} = \frac{2|I_{fc} \cap I_{wc}|}{|I_{fc}| + |I_{wc}|}$$

- **Reconstruction Loss ($\text{Loss}_R$)**: 입력 이미지와 최종 복원 이미지 사이의 평균 제곱 오차(MSE)를 최소화한다.
$$\text{Loss}_R = \frac{1}{n} \sum_{i=1}^{n} \| I_{in}^{(i)} - I_{ro}^{(i)} \|^2_2$$

- **Discriminator Loss ($\text{Loss}_D$)**: $D$가 생성된 $I_{fc}$는 가짜($-1$)로, 참조 분포 $R_d$의 이미지는 진짜($1$)로 정확히 판별하도록 MAE를 사용하여 학습한다.
$$\text{Loss}_D = \frac{1}{n+m} \left( \sum_{i=1}^{n} |D(I_{fc}^{(i)}) - (-1)| + \sum_{i=1}^{m} |D(I_{R_d}^{(i)}) - 1| \right)$$

### 학습 절차 (Training Scheme)

학습은 두 단계로 진행된다.

- **1단계**: $D$와 $M$을 주기적으로(cycle) 학습시킨다. $D$를 먼저 학습시킨 후, $D$의 가중치를 고정(frozen)한 상태에서 $M$이 $D$를 속이도록 학습한다.
- **2단계**: $M$과 $D$를 교대로 학습시키되, $D$의 입력 데이터에 $M$이 생성한 정상 이미지들($I_{fc}(R_d)$)을 추가하여 $R_d$와 $I_{in}$ 사이의 차이에 더 집중하게 만든다.

## 📊 Results

### 실험 설정

- **데이터셋**: MS-SEG2015 (다발성 경화증 병변), BraTS-2019 (뇌종양), LiTS (간 병변).
- **지표**: Dice score를 사용하여 분할 성능을 측정하였다.
- **방식**: 2D 슬라이스 단위로 처리를 수행하였으며, BraTS와 LiTS의 경우 2-fold 교차 검증을 실시하였다.

### 주요 결과

- **MS-SEG2015**: post-processing 적용 전 Dice score 32.94%, 적용 후 **48.20%**를 달성하였다. 이는 기존 f-AnoGAN의 최고 성능(27.8%)보다 크게 향상된 수치이다.
- **BraTS-2019**: 평균 Dice score **63.67%**를 기록하였다. f-AnoGAN의 경우 복원 실패로 인해 정상 이미지 생성에 어려움을 겪었으나, ASC-Net은 안정적인 결과를 보였다.
- **LiTS**: 평균 Dice score 32.24%를 기록하였으며, post-processing 적용 시 **50.23%**까지 향상되었다. (지도 학습 기반의 최신 연구 결과인 67.3%보다는 낮지만, 비지도 학습으로는 유의미한 수치이다.)

## 🧠 Insights & Discussion

### 강점 및 해석

ASC-Net의 가장 큰 강점은 **'복원 품질'과 '이상치 탐지 성능'의 결합을 끊어낸 것**이다. 기존 방법들은 이미지를 완벽하게 복원해야만 그 차이를 통해 이상치를 찾을 수 있었지만, ASC-Net은 이미지를 두 개의 의미적 컷으로 분리하는 문제로 재정의함으로써, 복원 결과가 다소 부정확하더라도 비정상 영역을 독립적인 컷으로 분리해낼 수 있는 강건함(stability)을 보여주었다.

### 한계 및 비판적 논의

1. **참조 분포의 한계**: 정상 이미지만을 참조 분포로 사용하기 때문에, 질병 외의 다른 생리적 변이나 동반 질환(co-morbidities)이 있을 경우 이를 모두 이상치로 판단하는 경향이 있다.
2. **학습 종료 시점의 모호성**: 학습 횟수(cycle)에 따라 히스토그램의 피크 분리 정도가 달라지며, 너무 오래 학습할 경우 오히려 결과물에 구멍(hole)이 생기는 현상이 발견되었다. 즉, 최적의 종료 시점을 결정하는 명확한 기준이 부족하다.
3. **성능 격차**: 비지도 학습으로는 훌륭한 성과를 냈으나, 여전히 지도 학습 기반 모델과는 상당한 Dice score 격차가 존재한다.

## 📌 TL;DR

본 논문은 비지도 이상치 분할을 위해 이미지를 정상 분포와 비정상 분포의 두 가지 컷으로 분리하는 **ASC-Net**을 제안한다. GAN의 판별기를 통해 사용자가 정의한 정상 이미지 지식을 네트워크에 주입함으로써, 완벽한 이미지 복원 없이도 비정상 영역을 효과적으로 분리해낼 수 있다. 특히 의료 영상 데이터셋(BraTS, LiTS, MS-SEG2015)에서 기존 AnoGAN 계열보다 우수한 성능을 입증하며, 향후 사용자 정의 지식을 활용한 비지도 학습 기반 의료 영상 분석의 가능성을 제시하였다.
