# ODDObjects: A Framework for Multiclass Unsupervised Anomaly Detection on Masked Objects

Ricky Ma (2021)

## 🧩 Problem to Solve

본 논문은 마스킹된 객체(Masked Objects)를 대상으로 하는 비지도 방식의 다중 클래스 이상치 탐지(Multiclass Unsupervised Anomaly Detection) 프레임워크인 ODDObjects를 제안한다. 일반적인 이상치 탐지는 데이터 포인트가 정상 분포에서 크게 벗어난 경우를 찾아내는 작업으로, 제조 시스템의 품질 관리나 데이터 세트의 정제 작업 등 다양한 분야에서 중요하다.

특히 이미지와 같은 고차원 데이터에서 비지도 학습을 통해 이상치를 탐지하는 것은 인간의 감독(Supervision)이 없기 때문에 매우 어려운 과제이다. 본 연구의 목표는 COCO 스타일의 데이터셋을 활용하여 훈련된 비지도 오토인코더(Autoencoder)를 통해 다양한 카테고리의 객체들 사이에서 이상치를 효과적으로 탐지하는 체계를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 오토인코더의 이미지 재구성(Image Reconstruction) 능력을 활용하여 이상치를 판별하는 것이다. 정상 데이터로만 학습된 오토인코더는 정상 데이터에 대해서는 낮은 재구성 오차(Reconstruction Error)를 보이지만, 학습 시 보지 못한 이상치 데이터에 대해서는 높은 재구성 오차를 보인다는 직관에 기반한다.

특히, 단순한 오토인코더 구조를 넘어 메모리 증강(Memory-augmented) 구조를 적용함으로써, 모델이 정상 데이터의 특징을 보다 명확하게 기억하고 재구성하게 하여 이상치 탐지 성능을 높이는 설계를 제안한다.

## 📎 Related Works

논문에서는 이상치 탐지 방법을 크게 세 가지 범주로 분류하여 설명한다.

1. **확률 기반 모델(Probabilistic models):** 데이터가 특정 확률 분포를 따른다고 가정하며, Gaussian Mixture Models(GMM) 등이 대표적이다. 하지만 데이터에 적합한 분포를 항상 찾기 어렵다는 한계가 있다.
2. **근접성 기반 모델(Proximity based models):** 정상 데이터와 이상치 데이터가 거리나 밀도 면에서 분리되어 있다고 가정한다. K-Nearest Neighbours(KNN), K-means clustering, DBSCAN 등이 이에 해당한다.
3. **정보 이론 기반 모델(Information theoretic models):** 데이터를 저차원으로 요약하고, 요약본과의 차이를 통해 이상치를 탐지한다. PCA와 오토인코더(Autoencoder)가 대표적이며, 재구성 오차를 측정 지표로 사용한다.

기존의 오토인코더 기반 접근 방식은 고차원 데이터를 효율적으로 압축하고 복원하지만, 때로는 이상치까지 너무 잘 복원해버려 재구성 오차가 낮아지는 문제가 발생할 수 있다. 이를 해결하기 위해 본 논문은 메모리 모듈을 추가한 MemCAE 구조를 검토한다.

## 🛠️ Methodology

### 1. 전체 파이프라인

ODDObjects 프레임워크는 크게 세 가지 구성 요소로 이루어져 있다: **데이터 처리 파이프라인 $\rightarrow$ 오토인코더 모델 $\rightarrow$ 이상치 탐지 알고리즘**.

### 2. 데이터 전처리 및 증강

COCO 2017 데이터셋을 사용하며, 객체 세그멘테이션(Segmentation) 정보를 바탕으로 객체를 중심에 맞추고 크롭(Crop)하며 마스킹 처리한다. 이때 배경 문맥(Background context)을 제거하여 모델이 배경이 아닌 객체 자체의 재구성에만 집중하게 함으로써 수렴 속도와 성능을 향상시켰다.
또한, 훈련 데이터에 대해 $\pm 5$도 회전, 1% 이내의 위치 이동, 10% 이내의 밝기 변화, 100%~125% 줌, 수평 뒤집기 등의 데이터 증강(Data Augmentation)을 적용하였다.

### 3. 모델 아키텍처 및 수식

본 논문은 네 가지 모델을 비교 분석한다. 모든 모델의 인코더는 입력 데이터를 32차원의 잠재 표현(Latent representation) $z$로 압축한다.

- **Vanilla Autoencoder (AE):** 두 개의 Fully-connected 레이어로 구성된다.
  $$z = \sigma(W_{xz}x + b_{xz})$$
  $$\hat{x} = \sigma(W_{zx}z + b_{zx})$$
  재구성 오차는 $\|x - \hat{x}\|$로 계산된다.

- **Convolutional Autoencoder (CAE):** 4개의 Convolution 레이어로 인코딩하고, 4개의 Deconvolution 레이어로 디코딩한다.

- **Convolutional Variational Autoencoder (CVAE):** CAE와 구조는 유사하나, 잠재 공간을 평균과 분산으로 정의하는 확률 분포로 취급한다. 학습 시 Evidence Lower Bound (ELBO)를 최대화하는 방향으로 최적화한다.
  $$L(\phi, \theta; x) = \mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x, z) - \log q_\phi(z|x)]$$

- **Memory-augmented Convolutional Autoencoder (MemCAE):** 인코더와 디코더 사이에 메모리 모듈 $M \in \mathbb{R}^{N \times C}$을 배치한다. 인코더가 생성한 $z$를 쿼리로 사용하여 메모리에서 가장 유사한 항목들을 가중합하여 $\hat{z}$를 생성한다.
  $$\hat{z} = \sum_{i=1}^{N} w_i m_i, \quad w_i = \frac{\exp(\text{sim}(z, m_i))}{\sum_{j=1}^{N} \exp(\text{sim}(z, m_j))}$$
  손실 함수는 재구성 오차와 메모리 가중치의 엔트로피(Entropy)를 결합하여 사용한다.
  $$L(\theta_e, \theta_d, M) = \frac{1}{N} \sum_{i=1}^{N} (\|x_i - \hat{x}_i\|_2^2 + \alpha E(\hat{w}_i))$$
  여기서 $E(\hat{w}_i)$는 메모리 주소 지정의 희소성(Sparsity)을 강제하기 위한 엔트로피 항이다.

### 4. 추론 절차

추론 시, 입력 이미지 $x$에 대해 재구성 오차 $\|x - \hat{x}\|_2^2$를 계산한다. 이 오차 값이 미리 설정된 임계값 $\gamma$보다 크면 해당 객체를 이상치(Anomaly)로 판정한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋:** COCO 데이터셋에서 단일 클래스(정상: orange / 이상치: 나머지 79개 클래스) 및 다중 클래스(orange, banana, apple, broccoli, carrot) 실험을 진행하였다.
- **지표:** ROC 곡선의 아래 면적인 AUC(Area Under Curve)를 사용하여 성능을 측정하였다.
- **환경:** NVIDIA RTX 2080Ti, Adam Optimizer (lr=0.001), Tensorflow 프레임워크 사용.

### 2. 정량적 결과

단일 클래스 이상치 탐지 실험 결과, MemCAE가 가장 높은 AUC 성능을 기록하였다.

| Model | AUC (%) |
| :--- | :---: |
| AE | 57.49 |
| CAE | 58.89 |
| CVAE | 57.18 |
| **MemCAE** | **60.94** |

### 3. 정성적 결과

- **재구성 품질:** MemCAE는 다른 모델들에 비해 훨씬 선명한 재구성 이미지를 생성하였으며, 특히 다중 클래스 학습 시 객체의 색상과 세부 디테일을 가장 잘 포착하였다.
- **이상치 식별:** MemCAE가 가장 높은 오차를 부여한 '최상위 이상치' 객체들을 분석한 결과, 실제로 라벨링이 잘못되었거나 여러 객체가 하나의 영역에 묶여 있는 등 데이터 품질이 낮은 샘플들이 탐지되었다. 반면, 재구성 오차가 가장 낮은 '가장 정상적인' 객체들은 매우 전형적인 형태의 객체들이었다.

## 🧠 Insights & Discussion

본 연구는 비지도 학습 기반의 오토인코더가 이미지 데이터의 이상치 탐지에 유효함을 입증하였다. 특히 다음과 같은 인사이트를 얻을 수 있다.

첫째, **배경 제거(Masking)의 중요성**이다. 객체 주변의 배경 문맥을 제거하고 객체 자체에만 집중하게 함으로써 모델의 수렴 속도를 높이고 재구성 성능을 유의미하게 개선하였다.

둘째, **메모리 모듈의 효용성**이다. MemCAE는 단순히 입력을 복제하는 것이 아니라, 메모리에 저장된 '정상 패턴'의 조합으로 재구성을 시도한다. 이로 인해 정상 데이터는 더 정확하게 복원되는 반면, 정상 패턴에서 벗어난 이상치는 복원 능력이 현저히 떨어지게 되어 이상치 판별력이 높아진다.

셋째, **데이터 정제 도구로서의 가능성**이다. 실험 결과에서 나타났듯, 모델이 탐지한 이상치가 실제 데이터셋의 라벨링 오류와 일치한다는 점은 이 프레임워크가 수작업으로 진행하던 데이터 클리닝 과정을 자동화하는 데 기여할 수 있음을 시사한다.

다만, AUC 점수 자체의 절대적인 수치가 아주 높지는 않으며, MemCAE의 성능 향상 폭이 학습 곡선(Training Curve)에서 보인 격차만큼 정량적 지표(AUC)에 드라마틱하게 반영되지 않았다는 점은 향후 개선이 필요한 부분이다.

## 📌 TL;DR

본 논문은 마스킹된 객체 데이터를 이용해 비지도 방식으로 이상치를 탐지하는 **ODDObjects** 프레임워크를 제안한다. 다양한 오토인코더 구조를 비교 분석한 결과, **메모리 증강 컨볼루션 오토인코더(MemCAE)**가 가장 우수한 재구성 성능과 이상치 탐지 성능(AUC 60.94%)을 보였다. 이 연구는 산업 현장의 실시간 품질 관리 및 대규모 데이터셋의 자동 라벨링 오류 수정 등에 활용될 가능성이 높다.
