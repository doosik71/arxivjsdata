# Semantic Segmentation of Thigh Muscle using 2.5D Deep Learning Network Trained with Limited Datasets

Hasnine Haque, Masahiro Hashimoto, Nozomu Uetake, Masahiro Jinzaki (2019)

## 🧩 Problem to Solve

본 연구가 해결하고자 하는 문제는 CT 영상에서 허벅지 근육(Thigh Muscle)을 11개의 클래스로 자동 분류하고 분할(Semantic Segmentation)하는 것이다. 허벅지 근육의 정량적 평가는 질병의 진행에 따른 근육 위축 정도를 파악하고 골절 위험을 예측하는 등 임상적으로 매우 중요한 가치를 가진다.

그러나 다음과 같은 이유로 인해 자동화된 분할이 어렵다. 첫째, 근육 간의 경계가 불분명하고 연부 조직의 대조도(Contrast)가 낮아 구분하기 어렵다. 둘째, 근육과 주변 조직 간의 Hounsfield Unit (HU) 값 범위가 상당 부분 중첩된다. 셋째, 다수의 얇은 CT 슬라이스를 수동으로 라벨링하는 작업은 시간이 매우 오래 걸리며, 작업자 간 또는 작업자 내의 변동성(Variability)이 크다는 문제가 있다. 따라서 본 논문의 목표는 제한된 데이터셋 환경에서도 높은 정확도를 가지며, 계산 효율적인 자동 근육 분할 알고리즘을 개발하는 것이다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 2D U-Net의 낮은 연산 복잡도와 3D U-Net의 공간적 문맥 정보 활용이라는 장점을 결합한 **2.5D U-Net** 구조를 제안한 것이다.

단순히 2D 슬라이스 하나만 사용하는 것이 아니라, 세 가지 직교 방향(Axial, Coronal, Sagittal)에서 각각 학습된 세 개의 2D U-Net을 운용하고, 이들의 결과를 투표(Voting) 알고리즘으로 통합함으로써 제한된 데이터 환경에서도 과적합(Overfitting)을 방지하고 3D 구조적 정보를 효과적으로 반영하도록 설계하였다.

## 📎 Related Works

기존 연구에서는 계층적 멀티 아틀라스 방법(Hierarchical Multi-atlas Method)을 통해 허벅지 근육을 11개 및 9개 클래스로 분할하여 좋은 성과를 거두었으나, 분할에 소요되는 시간이 약 40분으로 매우 길다는 한계가 있었다.

최근에는 Deep Neural Network 기반의 분할 방법이 성공을 거두고 있으며, 특히 U-Net 아키텍처가 의료 영상 분할의 표준으로 자리 잡았다. 하지만 볼륨 데이터 분할 시 2D U-Net은 파라미터가 적어 간단하지만 3D 공간 정보를 충분히 활용하지 못하고, 3D U-Net은 공간 정보 활용 능력은 뛰어나지만 학습 파라미터가 너무 많아 대규모의 학습 데이터가 필요하며 학습이 어렵다는 트레이드오프(Trade-off)가 존재한다.

## 🛠️ Methodology

### 1. 전체 시스템 파이프라인

전체 프레임워크는 다음과 같은 단계로 구성된다.

1. **영역 선택**: AI 기반 분류기를 통해 전체 CT 슬라이스 중 허벅지(Thigh) 영역에 해당하는 슬라이스만 자동으로 추출한다.
2. **데이터 정규화**:
    - **기하학적 정규화**: 환자의 자세가 다양하므로 대퇴골(Femur)의 해부학적 장축을 기준으로 아핀 변환(Affine Transformation)을 수행하여 정렬한다.
    - **해상도 통일**: $2\text{mm}$ 보셀 간격(Voxel-spacing)으로 리샘플링하고, $128 \times 128 \times 180$ 크기로 고정한다.
    - **대칭성 활용**: 왼쪽 허벅지 영상은 좌우 반전을 통해 오른쪽 허벅지와 동일한 방향성을 갖도록 정규화한다.
    - **강도 정규화**: 근육 픽셀이 명확히 구분되는 $100 \text{HU} \sim 250 \text{HU}$ 범위로 정규화한다.
3. **2.5D 분할**: 아래 설명하는 2.5D U-Net을 통해 분할을 수행한다.
4. **후처리**: 분할 결과물을 원래의 CT 볼륨 공간으로 되돌리고, 3D Connected Component Analysis를 적용하여 가장 큰 연결 요소만 남김으로써 노이즈를 제거한다.

### 2. 2.5D U-Net 아키텍처 및 학습 절차

**2.5D U-Net**은 파라미터 수를 줄이기 위해 스킵 연결(Skip connection)을 두 개만 사용한 간소화된 2D U-Net 세 개로 구성된다.

- **입력 데이터**: 각 방향(Axial, Coronal, Sagittal)의 타겟 슬라이스와 그 양옆 $\pm 4\text{mm}$ 거리의 인접 슬라이스를 포함하여 총 3채널의 입력을 구성한다. 이는 2D 네트워크가 국소적인 3D 문맥을 학습할 수 있게 한다.
- **앙상블 전략**: 세 개의 U-Net(Axial-Net, Coronal-Net, Sagittal-Net)이 각각 생성한 확률 맵(Probability map)을 **Winner-takes-all** 투표 규칙을 통해 최종 세그멘테이션 라벨로 결정한다.
- **손실 함수**: 예측된 분할 영역과 실제 정답(Ground Truth) 영역의 겹침 정도를 측정하는 Dice Similarity Coefficient (DSC)를 손실 함수로 사용하여 최소화한다.
$$ \text{DSC} = \frac{2|X \cap Y|}{|X| + |Y|} $$
- **학습 설정**: 48개의 허벅지 볼륨 중 38개를 학습에 사용하였으며, $\pm 10\%$ 랜덤 스케일링, $\pm 10\text{px}$ 랜덤 시프트, 최대 $20^\circ$ 랜덤 회전 등의 데이터 증강(Augmentation)을 적용하였다. 최적화 도구로는 Adam optimizer ($\alpha=0.0001$)를 사용하였다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: 총 48개의 허벅지 볼륨(24명 환자)을 사용하였으며, 이 중 10개를 테스트셋으로 할당하였다.
- **비교 대상**: 2D U-Net, 3D U-Net, 그리고 제안하는 2.5D U-Net.
- **평가 지표**:
  - **DSC (Dice Similarity Coefficient)**: 예측 영역과 실제 영역의 중첩도를 측정하는 지표.
  - **ASD (Average Surface Distance)**: 예측된 근육 표면 점들과 실제 표면 점들 사이의 평균 거리를 측정하는 지표.

### 2. 정량적 및 정성적 결과

- **정확도**: 2.5D U-Net의 평균 DSC는 $91.18\%$를 기록하였다. 이는 2D U-Net보다 $3.3\%$, 3D U-Net보다 $5.7\%$ 더 높은 수치이다.
- **표면 거리**: 2.5D U-Net의 ASD는 $0.84 \pm 0.27\text{mm}$로 나타났으며, 이는 기존의 계층적 멀티 아틀라스 방법($1.61 \pm 0.70\text{mm}$)보다 우수한 성능이다.
- **계산 효율성**: GPU 기반 워크스테이션에서 허벅지 볼륨 하나당 평균 $1.08 \pm 0.007$분의 시간이 소요되어, 기존 방식(40분) 대비 획기적인 속도 향상을 이루었다.
- **정성적 분석**: 2D U-Net 결과와 비교했을 때, 2.5D U-Net은 근육의 경계면과 3D 표면 렌더링 결과에서 훨씬 더 매끄럽고 정확한 분할 성능을 보였다.

## 🧠 Insights & Discussion

본 연구는 제한된 의료 데이터셋 환경에서 3D 볼륨 데이터를 처리할 때 2.5D 접근 방식이 매우 효율적임을 입증하였다. 3D U-Net의 경우 DSC의 IQR(사분위 범위)이 2.5D보다 좁게 나타나 성능의 안정성은 높았으나, 이는 학습 데이터가 충분할 때 3D U-Net이 더 우수할 수 있음을 시사한다. 하지만 현재와 같이 데이터가 부족한 상황에서는 파라미터 수가 적은 2.5D 모델이 과적합을 피하고 더 높은 일반화 성능을 보였다.

한계점으로는 근육의 부피가 작은 클래스에서 DSC 점수가 저하되는 현상이 관찰되었다. 이는 근육 위축(Muscle atrophy)이 심한 환자의 경우 근육의 형태가 변하거나 크기가 매우 작아져 네트워크가 이를 인식하기 어렵기 때문이다. 향후 연구에서는 이러한 소형 근육 및 위축된 근육 케이스를 더 잘 처리할 수 있는 모델 개선이 필요하다.

## 📌 TL;DR

본 논문은 제한된 CT 데이터셋을 사용하여 허벅지 근육을 11개 클래스로 자동 분할하는 **2.5D U-Net**을 제안하였다. 세 가지 직교 방향의 2D U-Net 결과물을 투표 방식으로 결합함으로써, 3D U-Net보다 적은 데이터로도 더 높은 정확도(평균 DSC 91.18%)와 빠른 처리 속도(볼륨당 약 1분)를 달성하였다. 이 연구는 임상 환경에서 근육 위축 정도를 정량적으로 빠르게 평가하는 시스템으로 활용될 가능성이 높다.
