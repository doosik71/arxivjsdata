# Automated Monitoring of Cultural Heritage Artifacts Using Semantic Segmentation

A. Ranieri, G. Palmieri, S. Biasotti (2025)

## 🧩 Problem to Solve

본 연구는 문화유산(조각상 및 기념물)의 보존을 위해 필수적인 균열(crack) 검출 과정을 자동화하는 문제를 다룬다. 문화유산의 균열을 조기에 정확하게 탐지하는 것은 추가적인 훼손을 방지하고 유물의 수명을 연장하는 데 매우 중요하다. 그러나 기존의 수동 점검 방식은 노동 집약적이고 시간이 많이 소요될 뿐만 아니라, 검사자의 주관에 따라 결과가 달라질 수 있으며, 구조물이 크거나 복잡할 경우 접근성 문제로 인해 정밀한 진단이 어렵다는 한계가 있다.

특히 딥러닝을 문화유산 보존에 적용할 때 발생하는 핵심 문제는 도메인 이동(domain shift)과 어노테이션 데이터의 부족이다. 도로 나 건물과 같은 토목 인프라를 대상으로 한 대규모 균열 데이터셋은 존재하지만, 이러한 데이터는 조각상이나 기념물의 정교한 표면 및 균열 형태와는 시각적 특성이 매우 다르다. 예를 들어, 도로 균열은 주로 아스팔트나 콘크리트 위에서 선형 또는 거북등 모양으로 나타나지만, 문화유산의 균열은 석재, 대리석, 금속 등 다양한 재질의 결을 따라 나타나거나 복잡한 텍스처 배경 속에 숨어 있어 탐지가 훨씬 어렵다. 따라서 본 논문은 이러한 데이터 부족 문제를 해결하고, 일반적인 균열 데이터로 학습된 모델이 실제 문화유산 도메인에서 얼마나 잘 작동하는지 분석하는 것을 목표로 한다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 다양한 복잡도를 가진 CNN Encoder를 결합한 U-Net 아키텍처를 사용하여, 문화유산 데이터셋이 부족한 상황에서도 일반적인 균열 세그멘테이션 모델이 어느 정도의 일반화 성능을 보이는지 정량적 및 정성적으로 분석하는 것이다.

주요 기여 사항은 다음과 같다.

- 다양한 CNN 백본(ResNet, ConvNeXt V2)을 사용한 U-Net 구조가 문화유산 유물의 균열 세그멘테이션에 얼마나 유효한지 분석하였다.
- 대규모 벤치마크 데이터셋인 OmniCrack30k를 활용하여 학습하고, 이를 통해 문화유산 도메인으로의 전이 가능성을 탐색하는 방법론을 제시하였다.
- mIoU, Dice coefficient, Jaccard index 등의 지표를 통해 모델 간의 성능을 정량적으로 비교 평가하였다.
- 라벨이 없는 실제 문화유산 이미지 세트를 활용하여 Out-of-distribution(OOD) 정성 평가 프레임워크를 구축함으로써, 모델의 실무적 유용성과 일반화 능력을 검증하였다.
- 연구의 재현성을 위해 모든 코드와 사전 학습된 모델이 포함된 공개 저장소를 제공한다.

## 📎 Related Works

### 기존 접근 방식 및 한계

균열 세그멘테이션 분야는 전통적인 이미지 처리 기법에서 최신 딥러닝 패러다임으로 발전해 왔다.

- **전통적 기법:** Canny, Sobel과 같은 에지 검출(edge detection), 임계값 설정(thresholding), 형태학적 연산(morphological operations) 등이 사용되었다. 이러한 방법은 계산 비용은 낮으나, 복잡한 배경, 조명 변화, 노이즈가 많은 환경에서는 신뢰도가 낮고 효과가 제한적이다.
- **딥러닝 기법:** CNN은 데이터로부터 복잡한 공간 패턴과 특징 표현을 직접 학습할 수 있어 정확도를 크게 향상시켰다.

### 문화유산 분야의 연구 현황

문화유산 분야에서는 주로 균열의 분류(classification)나 객체 탐지(object detection)에 집중해 왔다. 예를 들어, 태국 사원의 균열 분류를 위해 3층 CNN과 SVM, Random Forest를 비교하거나, YOLOv5를 사용하여 다양한 석조 재료의 균열을 탐지하는 연구가 진행되었다. 그러나 픽셀 수준의 정밀한 경계를 획정하는 시맨틱 세그멘테이션(semantic segmentation) 연구는 상대적으로 부족한 실정이다.

## 🛠️ Methodology

### 시스템 구조 및 아키텍처

본 연구에서는 이미지 세그멘테이션에서 널리 사용되는 U-Net 구조를 채택하였다. U-Net은 크게 두 부분으로 구성된다.

1. **Contracting Path (Encoder):** 입력 이미지를 다운샘플링하여 문맥 정보(contextual information)를 캡처하고 공간 해상도를 줄인다.
2. **Expansive Path (Decoder):** 특징 맵을 업샘플링하여 공간 해상도를 복원하고 최종 세그멘테이션 맵을 생성한다.
특히, Encoder의 특징 맵을 Decoder의 대응하는 층에 직접 연결하는 Skip Connection을 통해, 다운샘플링 과정에서 손실될 수 있는 세밀한 공간 정보를 보존하여 정밀한 경계를 예측한다.

본 논문에서는 다음과 같은 네 가지 CNN 백본을 Encoder로 사용하여 성능을 비교하였다.

- **ResNet-50:** 50층 구조, Shortcut connection을 통해 기울기 소실 문제를 해결한 모델 (25.6M 파라미터, 입력 크기 $270 \times 270$ px).
- **ResNet-101:** ResNet-50의 심화 버전 (44.5M 파라미터, 입력 크기 $540 \times 540$ px).
- **ConvNeXt V2 Base:** Masked Autoencoders로 설계된 순수 CNN 구조 (88.7M 파라미터, 입력 크기 $384 \times 384$ px).
- **ConvNeXt V2 Huge:** ConvNeXt V2의 최대 모델 (660M 파라미터, 입력 크기 $512 \times 512$ px).

### 학습 절차 및 목표 함수

- **데이터셋:** 다양한 재질(아스팔트, 세라믹, 콘크리트, 석조, 강철)의 균열 이미지가 포함된 OmniCrack30k 데이터셋을 사용하여 학습하였다.
- **손실 함수:** 픽셀 단위의 이진 분류(균열 vs 배경)를 위해 Binary Cross-Entropy (BCE) Loss를 사용하였다. BCE Loss는 각 픽셀의 예측 확률과 실제 라벨 간의 차이를 계산한다.
- **최적화:** Adam Optimizer를 사용하였으며, 학습률은 Cosine Annealing 스케줄러를 통해 $1\text{e}^{-3}$에서 $1\text{e}^{-4}$ 사이에서 조정되었다.
- **데이터 증강 (Data Augmentation):** Albumentations 라이브러리를 사용하여 기하학적 변환(Flip, Rotate), 왜곡(Blur, ElasticTransform), 광도 변화(CLAHE, HueSaturationValue)를 적용하였다.

### 평가 프로토콜

1. **정량적 평가:** OmniCrack30k 테스트 세트에서 mIoU, Dice Coefficient, Jaccard Index를 측정하였다.
2. **정성적 평가 (OOD):** 학습에 사용되지 않은, 웹에서 수집한 실제 파손된 조각상 및 기념물 이미지(unlabeled set)에 모델을 적용하여 균열의 연속성, 경계 정밀도, 오탐(False Positive/Negative) 여부, 일반화 성능을 분석하였다.

## 📊 Results

### 정량적 결과 (OmniCrack30k Test Set)

데이터 증강을 적용하지 않은 상태에서의 결과가 가장 우수하게 나타났으며, 주요 지표는 다음과 같다.

| U-Net Architecture | Test Loss | mIoU | Dice | Jaccard |
| :--- | :---: | :---: | :---: | :---: |
| ResNet-50 (270px) | 0.024 | 0.652 | 0.853 | 0.771 |
| ResNet-101 (540px) | 0.029 | 0.645 | 0.831 | 0.745 |
| ConvNeXt V2 Base (384px) | 0.026 | 0.662 | 0.852 | 0.770 |
| **ConvNeXt V2 Huge (512px)** | **0.024** | **0.666** | **0.865** | **0.786** |

분석 결과, **ConvNeXt V2 Huge** 모델이 모든 지표에서 가장 높은 성능을 보였다. 이는 모델의 용량, 더 높은 입력 해상도, 그리고 ImageNet-22k 사전 학습의 이점이 작용한 것으로 분석된다. 반면, 데이터 증강을 적용했을 때 오히려 성능이 하락하는 경향이 관찰되었는데, 이는 해당 데이터셋에서 과도한 정규화(over-regularization)가 발생하여 모델이 균열이 없는 곳을 균열로 예측하는 '환각' 현상이 나타났기 때문으로 추측된다.

### 정성적 결과 (Out-of-Distribution Evaluation)

실제 문화유산 이미지에 적용한 결과, 다음과 같은 통찰을 얻었다.

- **ConvNeXt V2 Huge:** 훈련 데이터에 조각상 이미지가 없었음에도 불구하고, 다양한 색상과 텍스처의 재질에서 균열을 매우 정확하게 세그멘테이션하였다. 특히 미세한 헤어라인 균열의 연속성을 잘 유지하였다.
- **ConvNeXt V2 Base:** 전반적으로 양호하나, 어두운 색상의 대리석 이미지에서 오탐(False Positive)이 발생하는 경향을 보였다.
- **ResNet 계열:** 균열 탐지 능력이 상대적으로 떨어졌으며, 특히 ResNet-50은 균열의 일부만을 탐지하거나, 눈이나 입과 같은 유물의 형태를 균열로 오인하는 경우가 많았다.

## 🧠 Insights & Discussion

### 강점 및 일반화 능력

본 연구의 가장 중요한 발견은 **대규모 CNN 아키텍처가 다양한 일반 균열 데이터셋으로 학습되었을 때, 명시적인 도메인 특화 학습 없이도 시각적으로 완전히 다른 문화유산 도메인으로 지식이 효과적으로 전이(transfer)된다**는 점이다. 특히 ConvNeXt V2와 같은 최신 구조는 복잡한 텍스처와 조명 조건에서도 강건한 일반화 능력을 보여주었다.

### 한계 및 비판적 해석

- **데이터 부족:** 문화유산 전용의 대규모 라벨링 데이터셋이 부재하여 정성적 평가에 의존해야 했다는 점이 한계이다.
- **오탐 문제:** 최상위 모델인 ConvNeXt V2 Huge조차 일부 이미지에서 경계가 불분명하거나 과도하게 확장되어 세그멘테이션되는 오탐 문제가 발생하였다.
- **계산 비용:** ConvNeXt V2 Huge는 최고의 성능을 내지만, ResNet-101 대비 학습 시간이 약 5배 더 소요되어 실시간성이나 자원 제한 환경에서의 적용에는 제약이 있을 수 있다.

## 📌 TL;DR

본 논문은 다양한 CNN 백본을 결합한 U-Net을 통해 문화유산 유물의 균열을 자동으로 탐지하는 연구를 수행하였다. 실험 결과, **ConvNeXt V2 Huge** 기반 모델이 가장 우수한 정량적 성능과 도메인 일반화 능력을 보였으며, 이는 일반적인 균열 데이터로 학습된 고성능 모델이 실제 문화유산 보존 현장에서도 유용하게 사용될 수 있음을 시사한다. 향후 3D 재구성이나 확산 모델(Diffusion Models)을 이용한 합성 데이터 생성 및 전문가(보존 과학자)의 라벨링이 결합된다면 더욱 정밀한 자동 모니터링 시스템 구축이 가능할 것으로 기대된다.
