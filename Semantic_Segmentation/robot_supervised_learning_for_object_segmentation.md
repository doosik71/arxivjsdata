# Robot-Supervised Learning for Object Segmentation

Victoria Florence, Jason J. Corso, and Brent Griffin (2020)

## 🧩 Problem to Solve

본 논문은 비정형적이고 끊임없이 변화하는 환경에서 로봇이 새로운 물체를 인식하고 분할(segmentation)하기 위해 필요한 학습 데이터를 어떻게 효율적으로 확보할 것인가에 대한 문제를 다룬다.

딥러닝 기반의 객체 검출 및 분할 기술은 비약적인 발전을 이루었으나, 이는 대량의 사람이 직접 라벨링한(human-annotated) 학습 데이터가 있다는 전제하에 가능하다. 하지만 로봇이 실제 환경에서 새로운 물체나 작업에 직면할 때마다 사람이 매번 데이터를 라벨링하는 것은 비현실적이며, 이는 로봇의 자율성을 저해하는 주요한 장애물이 된다. 따라서 본 연구의 목표는 인간의 감독 없이 로봇이 스스로 물체의 픽셀 단위 분할 라벨을 생성하여 자신의 시각 시스템을 지속적으로 개선할 수 있는 '로봇 감독(Robot-Supervised)' 학습 체계를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 로봇의 물리적 형태(embodiment)와 기구학(kinematics) 정보를 활용하여 스스로 학습 데이터를 생성하는 것이다. 구체적으로, 물체를 쥐고 있는 상태에서 '전경(foreground)'과 '로봇 팔(manipulator)'을 분리함으로써 물체만을 추출해내는 전략을 취한다.

핵심적인 설계 직관은 물체 분할 문제를 다음의 두 가지 하위 문제로 분해하는 것이다:

1. **전경 분할**: 배경으로부터 로봇 팔과 쥐고 있는 물체를 함께 분리한다.
2. **로봇 팔 분할**: 전경 중에서 로봇 팔 영역만을 분리하여 제거함으로써 물체 영역만을 남긴다.

이 과정에서 로봇은 자신의 기구학적 모델을 이용해 전경을 정의하고, 이를 통해 생성된 라벨로 로봇 팔 인식 네트워크(SRN)를 자가 학습시킨다. 결과적으로 인간의 개입 없이 새로운 환경과 물체, 로봇 플랫폼에 적응할 수 있는 자동 라벨링 파이프라인을 제안한다.

## 📎 Related Works

### 기존 연구 및 한계

1. **대화형 객체 분할(Interactive Object Segmentation)**: 환경의 변화나 물체의 움직임을 이용하여 객체를 분할한다. 하지만 정적인 배경 가정에 의존하거나, 물체를 밀어야 하는 등의 물리적 제약이 있으며 전체 포즈나 환경을 완전히 제어하기 어렵다는 한계가 있다.
2. **In-hand 객체 분할**: 로봇이 물체를 쥐고 있는 상태에서 엔코더 피드백을 활용한다. 기존 방법들은 GMM(Gaussian Mixture Model)이나 3D 메쉬 모델을 사용하는데, 이는 로봇 팔의 정교한 3D 기하학적 모델이 필요하거나, 로봇의 물리적 변경 시 모델을 재설계해야 하며, 픽셀 수준의 유사성(물체-배경-로봇 간)에 취약한 경우가 많다.
3. **자기 감독 로봇 인식(Self-supervised Manipulator Recognition)**: 기구학 모델을 통해 학습 라벨을 생성하는 시도가 있었으나, 대부분 전체 기하학적 모델의 투영에 의존한다.

### 차별점

본 논문은 전체 기하학적 모델 없이 **링크 좌표계(link coordinate frames)**와 **RGB-D 센서**만을 사용한다. 이를 통해 로봇 하드웨어의 변경에 자동으로 적응할 수 있으며, 최종 목적이 단순한 자기 인식이 아니라 'In-hand 물체 분할'을 통한 물체 학습에 있다는 점이 기존 연구와 차별화된다.

## 🛠️ Methodology

### 1. 기구학 기반 전경 분할 (Kinematics-based Foreground Segmentation)

로봇 팔과 물체를 배경으로부터 분리하는 단계이다.

**가. Depth 기반 분할**
먼저 depth 이미지 $D \in \mathbb{R}^{H \times W}$를 Felzenszwalb 알고리즘을 사용하여 과분할(over-segmentation)하여 세그먼트 집합 $S(D) = \{s_1, s_2, \dots, s_n\}$을 생성한다. 로봇의 기구학 모델과 엔코더 값을 통해 각 링크의 3D 좌표 $P_i = [x_i, y_i, z_i]^\top$를 구하고, 이를 카메라 내적 행렬 $K$를 이용해 이미지 좌표 $p_i = [h_i, w_i, 1]^\top$로 투영한다.
이때, 투영된 좌표의 실제 측정 depth $D(h_i, w_i)$가 기구학적 예측 값 $z_i$와 임계값 $\lambda$ 이내의 차이를 보이면 해당 세그먼트를 전경으로 간주한다:
$$s_{fg} := \bigcup \{s_j \mid \exists p_i \in s_j, D(h_i, w_i) \le z_i + \lambda\}$$

**나. RGB 기반 정교화**
Depth 기반의 초기 마스크 $M_0$는 노이즈가 많으므로, 모폴로지 연산(hole filling, opening)을 적용하고 Erosion과 Dilation을 통해 정밀도 중심의 $M_p$와 재현율 중심의 $M_r$을 생성한다. 최종적으로 GrabCut 알고리즘을 사용하여 RGB 이미지에서 전경 마스크 $M_{fg}$를 정교하게 추출한다.

### 2. 로봇 감독 자기 인식 네트워크 (Self-Recognition Network, SRN)

전경 분할 기법을 사용하여 로봇 팔만을 인식하는 네트워크를 학습시킨다.

- **데이터 수집**: 물체를 쥐지 않은 상태에서 다양한 관절 포즈를 취하며 이미지를 수집하고, 앞서 설명한 기구학 기반 전경 분할을 통해 로봇 팔 라벨을 자동으로 생성한다.
- **데이터 증강(Augmentation)**: 실전 환경에서의 오클루전(occlusion)을 모사하기 위해 COCO 데이터셋의 배경 이미지로 교체하거나, 임의의 객체 이미지를 로봇 팔 위에 겹쳐 쌓는 Foreground superposition을 수행한다.
- **모델**: Mask R-CNN(R-50-FPN backbone)을 사용하며, COCO 데이터셋으로 사전 학습된 모델을 기반으로 한다.

### 3. 로봇 감독 물체 어노테이션 (Robot-Supervised Object Annotation)

최종적으로 쥐고 있는 물체의 라벨을 생성하는 절차는 다음과 같다.

1. 물체를 쥔 상태에서 기구학 기반 전경 분할을 통해 $M_{fg}$(로봇 팔 + 물체)를 생성한다.
2. 학습된 SRN을 통해 $M_{SRN}$(로봇 팔) 영역을 예측한다.
3. 두 마스크의 차집합을 통해 물체 영역 $M_{object}$를 도출한다:
$$M_{object} = M_{fg} \cap \neg M_{SRN}$$
마지막으로 노이즈 제거를 위해 morphological opening 연산을 수행한다.

## 📊 Results

### 실험 설정

- **플랫폼**: Toyota Human Support Robot (HSR), RGB-D 카메라 사용.
- **데이터셋**: YCB 표준 객체 데이터셋 중 30개 물체 선정, 물체당 20장씩 총 600장의 테스트 이미지 구축.
- **지표**: 픽셀 단위 분할 성능은 $\text{mIoU}$로, 컨텍스트 내 재인식 성능은 $\text{AP}$ (Average Precision)로 측정한다.

### 주요 결과

1. **분할 성능 비교**:
   - 제안 방법의 $\text{mIoU}$는 $0.639$로, 베이스라인인 Browatzki et al. [6]의 $0.362$보다 월등히 높다.
   - 특히 물체의 크기가 크거나 색상이 로봇 팔과 유사한 경우에도 제안 방법이 더 강건한 성능을 보였다.

2. **데이터 증강의 효과 (Ablation Study)**:
   - 배경 증강(BG)과 전경 증강(FG)을 모두 적용했을 때 $\text{mIoU}$가 $0.431 \rightarrow 0.639$로 크게 상승하여, 도메인 랜덤화가 SRN의 일반화 성능을 높이는 데 기여함을 확인했다.

3. **객체 재인식 (Object Re-recognition in Context)**:
   - LVIS 데이터셋으로 학습된 Mask R-CNN 모델을 제안 방법으로 생성한 라벨로 파인튜닝(fine-tuning)한 결과, 분할 AP가 $38.1$에서 $49.8$로 향상되었다. 이는 자동 생성된 라벨이 실제 딥러닝 모델 학습에 유용함을 입증한다.

## 🧠 Insights & Discussion

### 강점

본 연구는 로봇의 기구학적 정보라는 '물리적 자산'을 활용해 딥러닝의 고질적인 문제인 '데이터 라벨링 비용'을 획기적으로 줄였다. 특히 정교한 3D 모델 없이 좌표계 정보만으로 self-recognition을 구현하여 하드웨어 변경에 유연하게 대응할 수 있다는 점이 매우 실용적이다.

### 한계 및 논의

- **작은 물체에 대한 취약성**: 실험 결과에서 물체의 픽셀 수가 매우 적은 경우 $\text{mIoU}$가 급격히 떨어지는 경향이 관찰되었다. 이는 Depth 센서의 해상도 한계와 기구학적 투영 오차가 작은 물체에서는 더 치명적으로 작용하기 때문으로 해석된다.
- **가정**: 본 방법은 로봇이 물체를 쥐고 있다는 전제가 필요하며, 초기 전경 분할 시 $\lambda$ (노이즈 임계값) 설정에 따라 성능이 달라질 수 있다.

### 비판적 해석

본 논문은 '로봇이 스스로 데이터를 만든다'는 개념을 성공적으로 구현했지만, 생성된 라벨의 노이즈가 어느 정도 존재함에도 불구하고 파인튜닝 결과가 좋게 나왔다는 점은 흥미롭다. 이는 딥러닝 모델이 어느 정도의 라벨 노이즈에 강건하며, 소량의 도메인 특화 데이터(in-hand data)가 대량의 일반 데이터(LVIS)보다 더 효과적일 수 있음을 시사한다.

## 📌 TL;DR

본 논문은 로봇의 기구학 정보와 RGB-D 센서를 결합하여 인간의 도움 없이 물체의 픽셀 단위 분할 라벨을 자동으로 생성하는 **Robot-Supervised Learning** 프레임워크를 제안한다. 로봇 팔과 물체를 함께 묶어 전경으로 잡은 뒤, 자가 학습된 SRN을 통해 로봇 팔 영역만 제거하는 방식으로 물체를 추출한다. 이 방법은 기존의 수동 설계 기반 방법보다 높은 $\text{mIoU}$를 달성했으며, 생성된 라벨을 통해 객체 인식 모델의 성능을 실질적으로 향상시켰다. 이는 로봇이 새로운 환경에서 스스로 학습하며 진화할 수 있는 가능성을 보여준다.
