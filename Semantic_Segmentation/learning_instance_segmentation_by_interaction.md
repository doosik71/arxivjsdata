# Learning Instance Segmentation by Interaction

Deepak Pathak, Yide Shentu, Dian Chen, Pulkit Agrawal, Trevor Darrell, Sergey Levine, Jitendra Malik (2018)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 인간의 개입 없이(self-supervised) 환경과의 상호작용만을 통해 시각적 관측치에서 개별 객체를 분리해내는 Instance Segmentation 능력을 학습하는 것이다.

기존의 State-of-the-art segmentation 시스템들은 COCO 데이터셋과 같이 방대한 양의 정밀한 클래스별 픽셀 수준 어노테이션에 의존한다. 그러나 이러한 데이터 구축 과정은 매우 노동 집약적이며 시간이 많이 소요된다. 또한, 정적인 데이터셋을 통한 학습은 에이전트가 실제 환경에서 마주하는 새로운 시나리오나 예측하지 못한 실수에 적응하는 능력을 제공하지 못한다. 따라서 본 연구의 목표는 능동적인 에이전트가 환경을 탐색하고, 상호작용을 통해 얻은 물리적 신호를 바탕으로 클래스에 구애받지 않는(class-agnostic) 객체 분할 모델을 스스로 학습하도록 하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 발달 심리학의 가설, 즉 유아들이 객체가 하나의 덩어리로 함께 움직이는 것을 보고 객체의 경계를 인식한다는 점에 착안하여 '공동 운명 원리(Principle of Common Fate)'를 적용한 것이다. 즉, 함께 움직이는 픽셀들은 하나의 객체로 그룹화될 가능성이 높다는 직관을 이용한다.

에이전트는 현재의 분할 모델을 통해 객체일 가능성이 높은 픽셀 그룹(Hypothesis)을 추론하고, 실제로 그 영역을 집어 옮기는(Pick and Place) 상호작용을 수행한다. 상호작용 전후의 프레임 차이(Frame Difference)를 통해 실제로 움직인 픽셀들을 추출하여 Pseudo ground-truth 마스크를 생성하고, 이를 통해 모델을 반복적으로 정교화한다. 특히, 상호작용 과정에서 발생하는 노이즈 섞인 학습 신호를 처리하기 위해 Robust Set Loss라는 새로운 손실 함수를 제안한 것이 주요한 기술적 기여이다.

## 📎 Related Works

본 연구는 능동적 지각(Active Perception)의 개념을 차용하고 있다. 기존의 상호작용 기반 세그멘테이션 연구들은 주로 특정 장면을 분할하는 것에 집중했으나, 본 논문은 상호작용을 통해 얻은 데이터를 사용하여 새로운 이미지에서도 작동하는 일반화된 모델을 학습시키는 데 목적이 있다.

또한, Optical flow를 이용하여 수동적으로 관찰된 비디오에서 Pseudo mask를 생성하는 연구들이 존재했으나, 본 연구는 에이전트가 직접 환경에 개입하여 데이터를 생성한다는 점에서 차별화된다. 기존의 Self-supervised representation learning 연구들이 주로 ego-motion, audio, colorization 등을 이용해 특징(feature)을 학습했다면, 본 연구는 로봇의 능동적 상호작용을 통해 Instance Segmentation을 학습하려는 첫 번째 시도라는 점에서 독창성을 가진다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조

에이전트는 다음과 같은 루프를 통해 학습을 진행한다:

1. **관측 및 추론**: 현재 이미지 $I_t$를 입력받아 CNN을 통해 객체 후보군 $\{s^t_1, \dots, s^t_K\}$를 생성한다.
2. **능동적 상호작용**: 후보군 중 하나 $s^t_j$를 선택하여 Pick and Place 동작을 수행한다.
3. **Pseudo-mask 생성**: 상호작용 전후의 이미지 차이($I'_{t+1} - I_t$)를 계산하여 움직인 픽셀 영역을 추출하고, 이를 $m^t_j$라는 Pseudo-mask로 정의한다.
4. **모델 업데이트**: 생성된 $m^t_j$를 학습 데이터로 사용하여 Scoring network와 Mask network를 업데이트한다.

### 네트워크 아키텍처

본 연구는 DeepMask 프레임워크를 기반으로 하며, ResNet-18을 백본으로 사용하여 특징을 추출한다. 이후 두 개의 브랜치로 나뉜다:

- **Scoring Network**: 이미지 패치의 중심 픽셀이 전경(foreground)인지 배경(background)인지 판별한다.
- **Mask Network**: Scoring network가 전경이라고 판단한 패치에 대해 세부적인 분할 마스크를 생성한다.

### Robust Set Loss (RSL)

상호작용으로 얻은 Pseudo-mask는 다른 객체를 함께 움직이거나 조명 변화로 인해 노이즈가 심하다. 단순한 픽셀 단위 Cross-entropy loss는 이러한 노이즈까지 학습하여 일반화 성능을 떨어뜨린다. 이를 해결하기 위해 본 논문은 예측값과 노이즈 섞인 타겟 사이의 $\text{IoU}$가 일정 임계값 $b$ 이상이 되도록 강제하는 제약 조건 기반의 최적화를 제안한다.

수식으로 표현하면, 잠재적인 정답 마스크 $\hat{X}$와 모델의 예측 분포 $Q$ 사이의 KL-divergence를 최소화하는 문제로 정의된다:

$$\text{minimize}_{\theta, \hat{X}, \xi} -\sum_i \log q_i(\hat{x}_i) + \lambda T \xi$$
$$\text{subject to } \text{IoU}(\hat{X}, M^I) \geq b - \xi, \quad \xi \geq 0$$

여기서 $M^I$는 상호작용으로 얻은 노이즈 섞인 마스크이며, $\xi$는 슬랙 변수이다. 이는 모델이 노이즈를 완전히 복제하는 대신, 타겟과 충분히 유사한(IoU 기준) 패턴을 찾도록 유도하며, 여러 예제를 통해 공통적인 객체 형태를 학습하게 만든다.

### 학습 가속화: Passive Self-Supervision

초기 모델은 아무런 지식이 없어 배경을 무작위로 집으려 하므로 학습 효율이 매우 낮다. 이를 해결하기 위해 로봇이 객체를 무작위로 미는(pushing) 기존의 수동적 데이터셋을 사용하여 모델을 사전 학습(Pre-train)시킨 후 능동적 상호작용 단계로 진입한다.

## 📊 Results

### 실험 설정

- **데이터셋**: 36개의 학습 객체, 15개의 테스트 객체, 그리고 다양한 텍스처의 배경을 사용하였다.
- **비교 대상**: Bottom-up 방식의 GOP(Geodesic Object Proposals)와 강력한 지도 학습 기반의 DeepMask를 베이스라인으로 설정하였다.
- **지표**: $\text{IoU}$ 임계값 0.3 및 0.5에서의 mean Average Precision ($\text{mAP}$)을 측정하였다.

### 주요 결과

1. **정량적 성능**: 제안 방법은 $\text{IoU} 0.3$ 기준에서 GOP를 크게 상회하였으며, 강력한 감독 학습 기반인 DeepMask(Tuned)보다는 낮았지만, 사람이 라벨링한 데이터 없이 스스로 학습했다는 점에서 매우 고무적인 수치를 보였다.
2. **Robust Set Loss의 효과**: RSL을 적용했을 때 단순 Cross-entropy를 사용한 경우보다 $\text{mAP}$가 향상되었으며, 결과의 분산(variance)이 줄어들어 학습이 더 안정적임을 확인하였다.
3. **데이터 양과 성능**: 상호작용 횟수가 증가함에 따라 성능이 꾸준히 우상향하는 경향을 보였으며, 이는 에이전트가 시간이 지날수록 더 효율적인 실험(더 정확한 객체 가설 설정)을 수행함을 의미한다.
4. **Downstream Task**: 학습된 분할 모델을 사용하여 여러 객체를 목표 설정으로 재배치하는 Task를 수행한 결과, 명시적인 Instance Segmentation을 사용하지 않은 기존 방식보다 우수한 성능을 보였다.

## 🧠 Insights & Discussion

본 논문은 능동적 학습(Active Learning)이 단순히 데이터의 양을 늘리는 것이 아니라, 데이터의 품질을 점진적으로 높이는 과정임을 보여주었다. 에이전트가 모델을 개선하면 더 정확한 객체 가설을 세우게 되고, 이는 다시 더 깨끗한 Pseudo-mask 생성으로 이어지는 선순환 구조를 형성한다.

**강점 및 한계**:

- **강점**: 인간의 라벨링 없이 물리적 상호작용만으로 일반화 가능한 Instance Segmentation 모델을 구축하였다. 특히 RSL을 통해 세그멘테이션 특유의 '세트 단위 노이즈' 문제를 수학적으로 해결하려 노력했다.
- **한계**: 여전히 강력한 지도 학습 모델(DeepMask)의 성능에는 미치지 못하며, 특히 $\text{IoU} 0.5$와 같은 엄격한 기준에서는 성능 하락이 두드러진다. 이는 상호작용으로 생성된 Pseudo-mask 자체가 가진 본질적인 불완전성 때문이다.
- **일반화 분석**: 분석 결과, 모델은 새로운 배경보다는 새로운 객체에 대해 더 잘 일반화되는 경향을 보였다($\text{IoU} 0.3$ 기준). 이는 객체의 '움직임'이라는 공통 특성이 배경의 다양성보다 더 강력한 신호로 작용했음을 시사한다.

## 📌 TL;DR

본 연구는 로봇이 환경과 직접 상호작용하며 "함께 움직이는 픽셀은 하나의 객체다"라는 원리를 통해 스스로 객체 분할(Instance Segmentation)을 학습하는 프레임워크를 제안한다. 노이즈 섞인 상호작용 데이터를 처리하기 위한 **Robust Set Loss**를 도입하여 학습 안정성과 성능을 높였으며, 이를 통해 사람이 라벨링한 데이터 없이도 새로운 객체와 배경에 대해 일반화 가능한 분할 능력을 갖출 수 있음을 입증하였다. 이는 향후 로봇의 시각-운동 제어 시스템이 객체 중심의 표현(object-centric representation)을 스스로 학습하는 데 중요한 기초가 될 수 있다.
