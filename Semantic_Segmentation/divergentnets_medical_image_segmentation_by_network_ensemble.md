# DivergentNets: Medical Image Segmentation by Network Ensemble

Vajira Thambawita, Steven A. Hicks, Pål Halvorsen and Michael A. Riegler (2021)

## 🧩 Problem to Solve

본 논문은 위장관 내시경 영상에서 대장 용종(colon polyps)을 자동으로 분할(segmentation)하는 문제를 해결하고자 한다. 의료 영상 분석에서 단순히 용종의 존재 여부를 판단하는 분류(classification)나 위치를 찾는 객체 탐지(object detection)보다 분할이 더 중요한 이유는, 픽셀 수준의 정밀도로 환부의 상세 영역을 보여줄 수 있기 때문이다.

특히 의료 분야에서는 알고리즘이 왜 그러한 판단을 내렸는지에 대한 설명 가능성(explainability)이 필수적인데, 분할 마스크는 의사가 알고리즘의 판단 근거를 시각적으로 확인할 수 있게 하여 의사 결정 지원 시스템의 신뢰도를 높인다. 그러나 용종은 모양과 크기가 매우 다양하며, 특히 편평한 병변(flat lesions)의 경우 숙련된 내시경 전문의조차 발견하기 어려울 정도로 식별이 까다롭다는 문제가 있다. 따라서 본 연구의 목표는 다양한 형태의 용종에 대해 일반화 성능(generalization performance)이 뛰어난 세그멘테이션 모델을 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 서로 다른 특성을 가진 여러 네트워크의 예측 결과를 결합하여 단일 모델이 가질 수 있는 편향을 줄이고 일반화 성능을 극대화하는 것이다. 이를 위해 다음과 같은 세 가지 주요 기여를 제시한다.

1. **TriUNet 아키텍처 제안**: 세 개의 UNet 모델을 삼각형 구조로 배치하여, 두 개의 병렬 UNet이 추출한 서로 다른 특징 벡터(feature vector)를 세 번째 UNet이 통합하여 최종 마스크를 예측하게 함으로써 데이터에 대한 다양한 해석을 반영한다.
2. **DivergentNets 앙상블 제안**: TriUNet을 포함하여 UNet++, FPN, DeepLabv3, DeepLabv3+와 같이 검증된 5가지의 서로 다른 세그멘테이션 모델을 앙상블하여, 각 모델의 '분산된 시각(divergent views)'을 평균함으로써 더 정밀하고 일반화된 마스크를 생성한다.
3. **Single-channel Dice Loss 도입**: 다중 클래스 분할 시 배경을 제외하고 오직 용종(polyp) 클래스에 대해서만 손실을 계산하는 수정된 Dice loss를 사용하여, 모델이 가장 중요한 타겟 영역에 집중하도록 강제한다.

## 📎 Related Works

기존의 의료 영상 분할 연구는 주로 CT 스캔의 종양 분할, MRI의 이상 징후 발견, 또는 일반적인 장기 및 조직 분할에 집중되어 왔다. 특히 대장 용종 탐지 분야에서는 프레임 단위의 분류(per-frame classification) 연구가 주를 이루었으나, 최근에는 더 정밀한 분석이 가능한 분할(segmentation) 연구로 관심이 이동하고 있다.

본 논문은 기존의 단일 모델 방식이 가진 한계를 극복하기 위해 앙상블 기법을 도입하였다. 기존 접근 방식들이 단일 아키텍처의 최적화에 집중했다면, 본 연구는 서로 다른 구조의 모델들이 학습하는 특징이 서로 다르다는 점에 착안하여 이를 결합함으로써 일반화 성능을 높이는 차별점을 가진다.

## 🛠️ Methodology

### 1. TriUNet 아키텍처

TriUNet은 UNet 아키텍처를 삼각형 형태로 배치한 CNN 구조이다.

- **구조**: 입력 이미지가 먼저 서로 다른 무작위 가중치를 가진 두 개의 UNet($UNet_1, UNet_2$)에 병렬로 입력된다.
- **특징 결합**: 두 모델에서 출력된 특징 벡터 $\text{Feature Vector}_1$과 $\text{Feature Vector}_2$를 하나로 연결(concatenation)한다.
- **최종 예측**: 연결된 벡터는 세 번째 UNet($UNet_3$)을 통과하여 최종 세그멘테이션 마스크를 생성한다.
- **학습**: 전체 네트워크는 하나의 모델처럼 동시에 학습되며, 손실 함수는 최종 출력 마스크와 Ground Truth 사이에서 계산되어 전체 네트워크로 역전파(backpropagation)된다.

### 2. DivergentNets 아키텍처

DivergentNets는 여러 고성능 모델의 누적된 결정이 단일 모델의 예측보다 더 정확할 것이라는 가설에 기반한 앙상블 시스템이다.

- **구성 모델**: UNet++, FPN, DeepLabv3, DeepLabv3+, TriUNet의 5가지 모델을 사용한다.
- **절차**:
    1. 각 모델을 독립적으로 $E$ 에포크(본 연구에서는 $E=200$) 동안 학습시킨 후 최적의 체크포인트를 선택한다.
    2. 입력 이미지를 5개 모델에 각각 통과시켜 개별 마스크 $M_1, \dots, M_5$를 생성한다. 이때 각 모델의 출력에는 $\text{softmax2d}$ 활성화 함수를 적용한다.
    3. 생성된 5개 마스크의 픽셀 값을 평균(average)낸 뒤, 반올림을 통해 최종 0 또는 1의 이진 마스크를 생성한다.

### 3. Single-channel Dice Loss

용종 분할의 핵심 목표가 '용종' 클래스를 정확히 찾는 것에 있으므로, 배경 클래스를 제외하고 용종 클래스 $n$에 대해서만 손실을 계산하는 수식을 사용한다.

$$ \text{Single-Channel Dice Loss} = \frac{2 \cdot |A_n \cap B_n|}{2 \cdot |A_n \cap B_n| + |B_n \setminus A_n| + |A_n \setminus B_n|} $$

여기서 $A_n$은 예측 마스크, $B_n$은 Ground Truth 마스크의 용종 클래스 영역을 의미한다.

### 4. 구현 상세

- **인코더**: 모든 모델은 ImageNet 가중치로 초기화된 $\text{SE-ResNeXt-50-32x4D}$를 인코더로 사용하였다.
- **전처리**: 이미지를 $256 \times 256$ 크기로 리사이즈 후 학습하였으며, 추론 시에는 Bilinear interpolation을 통해 원래 해상도로 복원하였다.
- **최적화**: Adam 옵티마이저를 사용하였으며, 학습률은 $0.0001$에서 시작하여 50 에포크 후 $0.00001$로 감소시켰다.
- **데이터 증강**: Albumentations 라이브러리를 사용하여 Horizontal flip, Gaussian noise, CLAHE, Random brightness 등 광범위한 증강 기법을 적용하여 일반화 성능을 높였다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: EndoCV2021 개발 데이터셋과 일반화 성능 검증을 위한 HyperKvasir 데이터셋을 사용하였다.
- **평가 지표**: IoU, F1-score, Recall, Precision 등을 측정하였다.

### 2. 정량적 결과

- **내부 검증 및 테스트**: DivergentNets가 가장 높은 성능을 보였으며, UNet++가 그 뒤를 이었다. (Table 2, 3 참조)
- **공식 챌린지 결과 (EndoCV 2021)**:
  - **Round I**: TriUNet이 평균 점수 $0.925$를 기록하며 우승하였다.
  - **Round II**: DivergentNets가 평균 점수 $0.823$을 기록하며 우승하였다.

### 3. 정성적 분석

질적 분석 결과, 개별 모델들은 서로 다른 특징을 학습하여 서로 다른 마스크를 생성하는 경향을 보였다. 예를 들어, TriUNet이 용종의 크기를 과대평가(over-estimate)하는 경우, DivergentNets의 평균화 과정에서 다른 모델들의 예측치가 이를 보정하여 더 매끄럽고 정확한 최종 마스크를 생성하는 것이 확인되었다. 다만, 특정 사례에서는 단일 모델(예: DeepLabv3+)이 앙상블 결과보다 더 정확한 예측을 하는 경우도 존재하였다.

## 🧠 Insights & Discussion

본 연구는 서로 다른 아키텍처를 가진 모델들을 앙상블하는 것이 의료 영상 분할의 일반화 성능을 높이는 데 매우 효과적임을 입증하였다. 각 모델이 데이터의 서로 다른 측면을 포착하는 '분산된 시각'을 가지며, 이를 평균화함으로써 개별 모델의 오류를 상쇄하는 효과(smoothing)를 얻을 수 있었다.

하지만 몇 가지 한계점과 논의 사항이 존재한다.
첫째, 하드웨어의 GPU 메모리 제한으로 인해 UNet을 UNet++로 대체한 $\text{TriUNet++}$ 구조를 실험하지 못한 점이 아쉬움으로 남는다.
둘째, 현재의 앙상블 방식은 단순 평균(averaging)에 의존하고 있다. 저자들은 단순 평균 대신 또 다른 신경망을 사용하여 최종 예측을 생성하는 학습 가능한 앙상블 기법이 더 좋은 성능을 낼 수 있을 가능성을 제시하였다.
셋째, 특정 이미지에서는 단일 모델의 성능이 앙상블보다 좋게 나오는 경우가 있는데, 이는 단순 평균 방식이 때로는 정답에 가까운 강한 예측치를 희석시킬 수 있음을 시사한다.

## 📌 TL;DR

본 논문은 대장 용종 분할의 일반화 성능을 높이기 위해 **TriUNet**(3개 UNet의 계층적 구조)과 **DivergentNets**(5개 서로 다른 모델의 앙상블)라는 두 가지 아키텍처를 제안하였다. 또한 용종 클래스에 집중하는 **Single-channel Dice Loss**를 도입하였다. 이 방법론을 통해 EndoCV 2021 챌린지의 두 라운드에서 모두 우승하며 그 성능을 입증하였으며, 이는 다양한 모델의 예측을 결합하는 앙상블 전략이 의료 영상의 복잡한 변동성을 극복하는 데 효과적임을 보여준다. 향후 학습 가능한 앙상블 기법이나 더 발전된 기본 유닛($\text{UNet++}$ 등)의 적용이 중요한 연구 방향이 될 것이다.
