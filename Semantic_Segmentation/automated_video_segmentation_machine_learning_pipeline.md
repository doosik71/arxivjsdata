# Automated Video Segmentation Machine Learning Pipeline

Johannes Merz, Lucien Fostier (2025)

## 🧩 Problem to Solve

현대 시각 효과(Visual Effects, VFX) 제작 공정은 매우 촉박한 일정과 복잡한 애니메이션 워크플로우를 특징으로 한다. 특히 후반 작업의 초기 단계에서 컴퓨터 그래픽(CG) 요소를 배경 플레이트에 합성하여 미리 보여주는 프리뷰 컴포지트(preview composites) 단계가 필수적이며, 이를 위해서는 배경과 전경 요소를 분리하는 마스크(mask) 생성 작업이 선행되어야 한다.

전통적인 방식인 수동 로토스코피(manual rotoscopy)나 블루스크린 클린업(bluescreen cleanup)은 시간과 자원이 매우 많이 소모되는 작업이다. 더욱이 프리뷰 단계에서 생성된 마스크들은 최종 퀄리티 작업 시 요구되는 정밀도와 기준이 달라 대부분 폐기되는 일회성 자산인 경우가 많다. 따라서 숙련된 아티스트들이 단순 반복적이고 일시적인 목적의 작업에 과도한 자원을 낭비하게 되는 문제가 발생한다. 본 논문의 목표는 이러한 수동 마스크 생성 과정을 자동화하여 VFX 제작 효율성을 높이고, 초기 단계의 컴포지트 제작 속도를 획기적으로 개선하는 자동화된 비디오 세그멘테이션 파이프라인을 구축하는 것이다.

## ✨ Key Contributions

본 연구의 핵심 기여는 텍스트 프롬프트 기반의 객체 탐지, 정밀한 프레임별 이미지 세그멘테이션, 그리고 시간적 일관성을 보장하는 비디오 트래킹을 결합한 통합 머신러닝 파이프라인을 제안한 것이다.

가장 중심적인 설계 아이디어는 단순히 최신 모델을 연결하는 것에 그치지 않고, VFX 제작 환경의 특수성을 고려한 후처리 공정과 캐시 관리 시스템을 도입한 점이다. 구체적으로는 GroundingDINO를 이용한 Open-set 객체 탐지로 유연성을 확보하고, SAM2를 활용하여 정밀한 마스크를 생성하며, 전방향(forward) 및 후방향(backward) 트래킹 패스를 통해 시간적 깜빡임(flickering)을 최소화하고 누락된 프레임을 보완하는 구조를 설계하였다.

## 📎 Related Works

논문에서는 객체 탐지를 위해 널리 사용되는 YOLO(You Only Look Once)와 같은 Closed-set 탐지 방식을 언급한다. 이러한 방식은 특정 카테고리에 대해 높은 정확도를 보이지만, VFX 환경처럼 미래지향적 헬멧을 쓴 캐릭터 등 학습 데이터셋에 없는 다양한 외형을 가진 객체를 탐지하는 데 한계가 있다. 또한, 탐지 대상을 변경할 때마다 새로운 데이터셋을 구축하고 미세 조정(fine-tuning)해야 하는 비효율성이 존재한다.

반면, 본 논문에서 채택한 GroundingDINO와 같은 Open-set 탐지 시스템은 자연어 프롬프트를 통해 조건부 입력을 제공하므로 추가 학습 없이도 다양한 객체를 탐지할 수 있다는 차별점이 있다. 세그멘테이션 분야에서는 SAM(Segment Anything Model)과 그 후속 모델인 SAM2를 활용하여, 고정된 클래스 분류가 아닌 프롬프트 기반의 유연한 인스턴스 세그멘테이션(Instance Segmentation)을 수행함으로써 기존의 단순 시맨틱 세그멘테이션(Semantic Segmentation)이 해결하지 못하는 개별 객체 분리 문제를 해결하였다.

## 🛠️ Methodology

전체 파이프라인은 크게 세 단계의 연속적인 공정으로 구성된다.

### 1. Object Detection (Stage 1)

첫 번째 단계에서는 GroundingDINO를 사용하여 샷 내의 특정 객체 인스턴스를 탐지한다. 사용자는 텍스트 프롬프트(예: "person", "car")를 입력하여 탐지 대상을 지정할 수 있으며, 기본 설정은 "person"으로 지정되어 자동 실행된다. 이 단계의 결과물은 각 프레임별 객체의 위치를 나타내는 Bounding Box들이다.

### 2. Image Segmentation (Stage 2)

탐지된 Bounding Box를 입력으로 하여 SAM2 모델이 이진 세그멘테이션 마스크를 생성한다. 하지만 탐지 단계에서 동일 인물이 중복 탐지되거나, 여러 인물이 겹쳐 있는 경우 잘못된 바운딩 박스가 생성될 수 있다. 이를 해결하기 위해 다음과 같은 비대칭 유사도 지표 $\text{Sim}(A, B)$를 정의하여 후처리를 수행한다.

$$\text{Sim}(A, B) = \frac{A \cap B}{A}, \forall (A, B) \in \text{Masks}$$

이 지표는 마스크 $A$가 마스크 $B$에 의해 얼마나 덮여 있는지를 측정하며, 이를 통해 다음과 같은 기준을 적용한다.

- **삭제(Discard):** 마스크 $A$가 $\text{Sim}(A, B) \ge 0.1$인 다른 마스크를 2개 이상 가지고 있다면, 이는 잘못된 탐지로 간주하여 삭제한다.
- **병합(Merge):** 삭제 후, 특정 마스크와 $\text{Sim}(A, B) \ge 0.6$인 단 하나의 마스크만 존재한다면, 두 마스크를 동일 객체의 일부로 판단하여 병합한다.

### 3. Video Tracking (Stage 3)

프레임별로 생성된 마스크의 시간적 불일치와 깜빡임을 해결하기 위해 SAM2의 비디오 트래킹 모드를 사용한다.

**전체 워크플로우 및 알고리즘:**

1. **전방향 패스(Forward Pass):** 첫 프레임($t=0$)의 마스크를 시작점으로 하여 $S_V = 20$ 프레임 간격으로 전파한다. 이때 $S_P = 5$ 프레임마다 Stage 2의 고품질 마스크 정보를 다시 주입(re-prompt)하여 트래킹 정확도 저하를 방지한다.
2. **후방향 패스(Backward Pass):** 전방향 패스 후, 마지막 프레임에서 시작하여 역방향으로 다시 트래킹을 수행한다. 이는 객체가 샷 중간에 등장하거나 초기에 가려져 탐지되지 못한 경우를 보완하기 위함이다.
3. **마스크 캐시(Mask Cache) 업데이트:** 캐시는 각 픽셀의 레이어 ID($M_{ID, t}$)와 확률값($M_{P, t}$)을 저장한다. 새로운 예측값 $V_{P, t}$가 기존 캐시값보다 일정 마진 $\epsilon = 0.1$ 이상 높을 때만 업데이트를 수행하여 안정성을 확보한다.
   $$M_{P, t} + \epsilon \le V_{P, t}$$
4. **새로운 프롬프트 생성:** Intersection over Union (IoU) 지표를 사용하여 캐시 내 마스크와 Stage 2 마스크의 일치 여부를 확인한다.
   $$\text{IoU}(M, U) = \frac{M \cap U}{M \cup U}$$
   $\text{IoU} \ge 0.9$인 경우 직접 일치로 판단하며, 그렇지 않은 경우 앞서 정의한 비대칭 유사도 지표를 통해 부분 일치 여부를 확인하여 레이어를 유지한다.

### 추가 기능 및 통합

- **Interactive Segmentation:** 자동 파이프라인이 해결하지 못하는 엣지 케이스를 위해, 아티스트가 긍정/부정 포인트(Positive/Negative points)를 직접 클릭하여 마스크를 생성하고 이를 트래킹에 전달하는 인터랙티브 도구를 제공한다.
- **Singularity Integration:** ML 라이브러리 의존성 및 GPU 가속 문제를 해결하기 위해 Singularity/Apptainer 컨테이너 기술을 사용하여 배포하였다.
- **ObjectId:** 다중 샘플 저장이 가능한 ObjectId 포맷을 사용하여 복잡한 겹침이나 반투명 영역을 표현하고, 메타데이터를 통해 사용자 친화적인 객체 식별자를 매핑한다.

## 📊 Results

### 실험 설정 및 지표

- **하드웨어:** NVIDIA A4000 GPU (16GB VRAM).
- **제한 사항:** GPU 메모리 한계로 인해 입력 영상의 최대 해상도를 1024 픽셀로 조정하여 처리하였다.
- **성능 지표:** 처리 시간(Processing time) 및 아티스트의 채택률(Adoption rate).

### 정량적 결과

- **처리 속도:** 3개 단계 전체 공정을 거쳐 평균적으로 분당 10프레임을 처리한다. 일반적인 100프레임 분량의 VFX 샷 하나를 처리하는 데 약 10분이 소요된다.
- **실제 적용 사례:** 2024-2025년 동안 12개의 서로 다른 작품(shows)에서 총 1,241개의 샷에 적용되었다.

### 정성적 결과

- **효용성:** 컴포지팅 팀의 피드백에 따르면, 생성된 마스크의 85%~90%가 추가 수정 없이 그대로 사용 가능했으며, 단 10%~15%의 경우에만 미세한 수동 조정이 필요했다. 이는 자동화된 결과물이 실제 제작 공정에 투입될 수 있을 만큼 충분한 품질을 갖추었음을 시사한다.

## 🧠 Insights & Discussion

본 연구는 최신 딥러닝 모델인 GroundingDINO와 SAM2를 VFX라는 특수한 도메인에 맞춰 성공적으로 통합하였다. 특히 단순히 모델을 사용하는 것에 그치지 않고, VFX 아티스트들이 겪는 '일회성 마스크 생성'이라는 병목 현상을 정확히 타격하여 실질적인 생산성 향상을 이끌어냈다.

**강점 및 성과:**

- 컨테이너 기반 배포와 ObjectId라는 구조화된 저장 형식을 통해 ML 모델의 업데이트와 파이프라인 통합을 유연하게 구현하였다.
- 전/후방향 트래킹과 마스크 캐시 시스템을 통해 비디오 세그멘테이션의 고질적인 문제인 시간적 불안정성을 효과적으로 억제하였다.

**한계 및 논의사항:**

- **완전성 vs 안정성:** Stage 2의 후처리 과정에서 안정성을 위해 일부 배경 객체 마스크가 삭제되는 경향이 있다. 이는 사용 가능성을 높이기 위한 선택이었으나, 모든 객체를 보존해야 하는 경우에는 수동 개입이 필요하다.
- **리소스 제약:** GPU 메모리 한계로 인해 해상도를 1024px로 제한한 점은 고해상도 작업이 필수적인 VFX 환경에서 잠재적인 제약 사항이 될 수 있다.

## 📌 TL;DR

본 논문은 VFX 제작의 병목 지점인 마스크 생성을 자동화하기 위해 **GroundingDINO(탐지) $\rightarrow$ SAM2(세그멘테이션) $\rightarrow$ SAM2 Video(트래킹)**로 이어지는 파이프라인을 제안한다. 비대칭 유사도 기반의 후처리와 전/후방향 트래킹 패스를 통해 시간적 일관성이 높은 인스턴스 마스크를 생성하며, 실제 12개 프로젝트의 1,241개 샷에 적용하여 처리 시간을 획기적으로 단축하고 아티스트의 수정 작업을 최소화했다. 이 연구는 향후 단순 픽셀 마스크를 넘어 로토스코피 스플라인(Spline) 자체를 예측하고 트래킹하는 방향으로 확장될 가능성이 크다.
