# Pixel-wise object tracking using a global attention model and a local segmentation model

Yilin Song, Chenge Li, and Yao Wang (2018)

## 🧩 Problem to Solve

본 논문은 노이즈가 많은 배경 속에서 임의의 익명 객체(anonymous object)를 픽셀 단위로 추적하는 문제를 해결하고자 한다. 일반적인 비주얼 객체 추적(Visual Object Tracking)은 대상의 변형, 모션 블러(motion blur), 조명 변화, 부분 가려짐(partial occlusion) 및 배경 클러터(background clutters)와 같은 까다로운 환경에서 강건하게 작동해야 한다.

기존의 CNN 기반 추적기들은 성능을 향상시켰으나, 객체의 공간적-시간적 특성(spatial-temporal traits)을 통합적으로 모델링하는 메커니즘이 부족했다는 한계가 있다. 또한, 픽셀 단위의 세밀한 세그멘테이션 기반 추적은 연산 비용이 매우 높거나, 단순한 픽셀 정보에만 의존하여 객체의 시맨틱 구조를 반영하지 못하는 경우가 많았다. 따라서 본 연구의 목표는 실시간으로 작동하면서도 객체의 외형 변화와 움직임을 효과적으로 추적할 수 있는 픽셀 단위 추적 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 객체 추적 문제를 **전역 주의 모델(Global Attention Model)**과 **지역 세그멘테이션 모델(Local Segmentation Model)**의 두 단계로 분리하여 해결하는 것이다.

1. **계층적 구조 설계**: 전역 모델이 과거의 세그멘테이션 맵을 바탕으로 객체가 존재할 가능성이 높은 관심 영역(Region of Interest, ROI)을 예측하면, 지역 모델은 해당 ROI 내에서만 정밀한 픽셀 단위 세그멘테이션을 수행한다. 이를 통해 전체 이미지 해상도에 따른 연산 부담을 줄이면서도 세밀한 추적이 가능하다.
2. **시간적 역동성 모델링**: 두 모델 모두 Convolutional LSTM(convLSTM) 구조를 채택하여, 전역 모델은 객체의 움직임 패턴(motion dynamics)을, 지역 모델은 외형 변화(appearance dynamics)를 학습하도록 설계하였다.
3. **반복적 업데이트 전략(Iterative Update Strategy)**: 두 모델 간의 데이터 의존성 문제를 해결하기 위해 상호 보완적으로 학습시키는 반복 최적화 방식을 제안하였다.
4. **오프라인 학습 기반 효율성**: 모델이 한 번 학습되면 특정 객체에 대해 온라인 학습(online learning)이나 미세 조정(fine-tuning)을 수행할 필요가 없어 추론 속도가 매우 빠르다.

## 📎 Related Works

논문에서는 기존의 추적 방식들을 다음과 같이 분류하고 한계를 지적한다.

- **Detection-based Tracking**: 최근 CNN 기반의 특징 추출기를 사용하여 성능을 높였으나, 픽셀 단위의 정밀한 레이블링을 제공하기에는 결과물이 너무 거칠다(too coarse).
- **Segmentation-based Tracking**: 비정형 움직임(non-rigid motions) 처리에 유리하지만, 많은 방법이 픽셀 수준 정보에만 의존해 시맨틱 구조를 놓치거나, Markov Random Field(MRF)나 광학 흐름(Optical Flow) 등을 사용하여 속도가 매우 느리다(약 1 fps).
- **RNN/convLSTM 기반 접근**: 시간적 관계를 모델링하려 시도했으나, 대부분 합성 데이터셋에서만 실험되었거나 정밀한 픽셀 레이블링보다는 거친 위치 예측에 그쳤다.

본 연구는 convLSTM을 세그멘테이션 파이프라인에 직접 통합하고, 전역/지역 모델로 역할을 분리함으로써 정밀도와 속도라는 두 마리 토끼를 잡고자 하였다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

시스템은 전역 모델과 지역 모델이 폐루프(closed loop) 형태로 동작한다.

- **전역 모델**: 이전 프레임의 예측 세그멘테이션 맵 $\hat{Z}_{t-1}$을 입력받아, 현재 프레임 $t$에서 객체가 있을 것으로 예상되는 ROI의 위치와 크기를 결정하는 변환 파라미터 $\theta$를 생성한다.
- **지역 모델**: 전역 모델이 지정한 ROI 이미지 $x_t$를 입력받아 픽셀 단위의 세그멘테이션 맵 $\hat{z}_t$를 생성한다.

### 2. 지역 세그멘테이션 모델 (Local Segmentation Model)

ROI 내의 객체를 정밀하게 분리하는 역할을 수행한다.

- **특징 추출**: 사전 학습된 VGG 네트워크의 `pool4` 레이어 특징을 사용한다. 이때 특징의 범위를 제한하기 위해 두 개의 합성곱 층으로 구성된 **F-CNN**을 통해 $\tanh$ 활성화 함수로 정규화한다.
- **시계열 모델링**: 정규화된 특징은 두 층의 **convLSTM**을 통과한다. 첫 번째 층은 전경과 배경의 역동성을, 두 번째 층은 객체의 외형 변화(appearance shift)를 모델링한다.
- **최종 출력**: convLSTM의 출력은 디컨볼루션(Deconvolution) 층을 거쳐 최종 픽셀별 확률 맵(Segmentation Map)을 생성한다.
- **메모리 초기화**: recurrent 네트워크의 수렴 시간을 단축하기 위해, 첫 프레임의 이미지와 마스크를 입력받아 초기 메모리 상태($c$)와 은닉 상태($h$)를 생성하는 **M-CNN** 모듈을 사용한다.

**손실 함수**:
지역 모델은 교차 엔트로피 손실 $\mathcal{L}$과 세그멘테이션 맵의 파편화를 방지하기 위한 Total Variation(TV) 손실 $\mathcal{V}$를 함께 최소화한다.
$$\min_{\theta} \mathcal{L}(\hat{z}_{1:T}, z_{1:T}) + \lambda \min(\beta, \frac{1}{T} \sum_{i=1}^{T} \mathcal{V}(\hat{z}_i))$$
여기서 $\mathcal{V}(y) = \sum_{i,j} |y_{i+1,j} - y_{i,j}| + |y_{i,j+1} - y_{i,j}|$ 이다.

### 3. 전역 주의 모델 (Global Attention Model)

객체의 대략적인 위치를 예측하여 ROI를 설정하는 역할을 한다.

- **구조**: 입력 세그멘테이션 맵 $\rightarrow$ 합성곱 층 $\rightarrow$ convLSTM $\rightarrow$ Fully Connected 층 $\rightarrow$ **Spatial Transformer Network (STN)** 순으로 구성된다.
- **ROI 결정**: STN은 고정된 앵커 박스 $Z_A$에 아핀 변환(affine transform)을 적용하여 $\hat{Z}_A^t$를 생성한다. 이때 원본 이미지의 가로세로 비율(aspect ratio) $\gamma$를 네트워크에 입력하여 비율 왜곡을 방지한다.

**손실 함수**:
예측된 앵커 $\hat{Z}_A^t$와 실제 정답 마스크 $Z_t$ 사이의 가중 $l_2$ 손실과, 시간적 부드러움을 위한 변환 파라미터 $\theta$의 변화량 손실을 최소화한다.
$$\min_{\phi} \sum_{t=i}^{T} (\mathcal{L}(\hat{Z}_A^t, Z_t) + \lambda ||\theta_t - \theta_{t-1}||^2)$$

## 📊 Results

### 실험 설정

- **데이터셋**: VOT2016 세그멘테이션 데이터셋 (매우 도전적인 데이터셋으로, 객체가 매우 작거나 외형 변화가 심함).
- **평가 지표**: Average IoU (Intersection over Union).
- **비교 대상**: 1-stage ConvLSTM (단순 예측기 사용), FCN (첫 프레임에서 미세 조정된 네트워크).
- **환경**: NVIDIA Tesla K80 GPU 사용.

### 주요 결과

1. **정량적 성능 (Table 2, 3)**:
    - **지역 모델 성능**: convLSTM 기반 모델이 FCN보다 외형 변화에 훨씬 강건하며 높은 IoU를 기록하였다.
    - **전체 시스템 성능**: 2-stage ConvLSTM 구조가 1-stage 방식 및 FCN 방식보다 월등한 성능을 보였다. 특히 시퀀스 길이가 길어질수록(20 $\rightarrow$ 160 프레임) FCN의 성능은 급격히 떨어지지만, 제안 모델은 상대적으로 안정적인 성능을 유지했다.
2. **추론 속도**:
    - 지역 모델: 24ms/frame, 전역 모델: 6ms/frame.
    - 전체 프레임워크는 메모리 초기화 및 ROI 보간을 포함하여 **20 fps 이상**으로 실시간 작동이 가능함을 확인하였다.
3. **정성적 결과**: Figure 7을 통해 다양한 비디오에서 정밀하게 객체를 추적함을 보여주었다. 다만, 급격한 카메라 움직임이 있는 경우 추적에 실패하는 사례가 관찰되었다.

## 🧠 Insights & Discussion

### 강점 및 통찰

- **외형 변화 대응**: convLSTM의 메모리 셀이 외형 변화를 학습함으로써, 첫 프레임에 고정된 특징을 사용하는 FCN보다 훨씬 유연하게 객체를 추적할 수 있음을 입증하였다.
- **연산 효율성**: 전역 모델을 통해 ROI를 먼저 설정함으로써 고해상도 이미지 전체를 처리할 필요가 없게 하여 실시간성을 확보하였다.
- **데이터 증강의 중요성**: 학습 시 무작위로 노이즈 프레임(다른 비디오의 ROI)을 삽입하는 전략이 convLSTM이 단순한 saliency detection에 빠지지 않고 실제 추적 능력을 갖추게 하는 데 결정적인 역할을 하였다.

### 한계 및 비판적 해석

- **Translation Equivariance 부재**: convLSTM의 Hadamard product 연산으로 인해 입력 이미지가 약간 이동했을 때 출력 맵이 동일하게 이동하지 않는 특성이 있다. 저자들은 전역 모델의 ROI가 객체를 중앙에 배치하므로 문제가 없다고 주장하지만, 이는 전역 모델의 정확도에 전적으로 의존하게 만드는 위험 요소가 된다.
- **훈련-테스트 분포 차이**: 전역 모델은 학습 시 정답 마스크(GT)를 입력받지만, 테스트 시에는 지역 모델이 예측한 (오차가 있는) 마스크를 입력받는다. 이를 해결하기 위해 반복 최적화 전략을 사용했으나, 여전히 이 분포 차이가 성능 저하의 주요 원인이 될 수 있다.

## 📌 TL;DR

본 논문은 **전역 주의 모델(위치 예측)**과 **지역 세그멘테이션 모델(정밀 분리)**을 결합한 2단계 픽셀 단위 객체 추적 프레임워크를 제안한다. 두 모델 모두 **convLSTM**을 사용하여 움직임과 외형 변화라는 시간적 특성을 모델링하였으며, 반복 학습 전략을 통해 상호 의존성을 해결했다. 결과적으로 VOT2016 데이터셋에서 기존 FCN 기반 방식보다 우수한 추적 성능을 보였으며, 20 fps 이상의 실시간 처리 속도를 달성하였다. 이 연구는 시계열 모델을 세밀한 픽셀 단위 추적에 적용하여 실용적인 실시간 성능을 확보했다는 점에서 향후 실시간 비디오 분석 연구에 기여할 가능성이 높다.
