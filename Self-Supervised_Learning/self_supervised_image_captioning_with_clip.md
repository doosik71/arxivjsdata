# Self-Supervised Image Captioning with CLIP

Chuanyang Jin (2023)

## 🧩 Problem to Solve

본 논문은 이미지 캡셔닝(Image Captioning) 분야에서 발생하는 두 가지 핵심적인 문제점을 해결하고자 한다.

첫째는 고품질의 이미지-캡션 쌍(image-caption pairs)에 대한 과도한 의존성이다. 현재의 주류 방식인 지도 학습(Supervised Learning)은 방대한 양의 레이블링된 데이터셋을 필요로 하며, 이는 특정 도메인에서 데이터를 수집하고 구축하는 데 매우 높은 비용과 어려움이 따른다.

둘째는 참조 캡션(Reference Captions) 자체의 품질 한계이다. 기존의 모델들은 주로 참조 캡션과의 텍스트 유사도를 최대화하는 방향으로 학습된다. 그러나 공공 데이터셋의 참조 캡션들은 대개 가장 눈에 띄는 일반적인 객체만을 묘사하는 경향이 있어, 모델이 이미지의 세부적인 특징을 포착하기보다는 평이하고 일반적인 묘사에 머물게 하는 '품질의 상한선'으로 작용한다.

따라서 본 연구의 목표는 참조 캡션에 대한 의존도를 획기적으로 낮추면서도, 기존의 지도 학습 기반 모델보다 더 구체적이고 정보량이 많은 캡션을 생성하는 자기지도 학습(Self-supervised Learning) 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **'CLIP 유사도 향상'이라는 보조 작업(Auxiliary Task)을 통한 2단계 학습 전략**이다.

먼저, 아주 적은 양의 레이블링된 데이터를 통해 모델에 기본적인 신호를 제공하는 초기 지도 학습 단계를 거친다. 이후, 레이블이 없는 데이터셋에서 모델이 생성한 캡션과 원본 이미지 사이의 CLIP relevance(상호 관련성)를 최대화하는 방향으로 자기지도 학습을 수행한다. 이를 통해 모델은 특정 정답 문장을 모방하는 것이 아니라, 이미지의 시각적 특징을 가장 잘 설명하는 텍스트를 스스로 찾아가게 된다. 특히, 이 과정에서 이산적인(discrete) 토큰 샘플링 문제를 해결하기 위해 Gumbel-Softmax 기법을 도입하여 미분 가능한 최적화를 구현하였다.

## 📎 Related Works

논문은 이미지 캡셔닝의 일반적인 구조가 시각적 인코더(Visual Encoder)와 텍스트 디코더(Textual Decoder)로 구성됨을 언급한다. 특히 CLIP(Contrastive Language-Image Pre-training)을 활용하여 이미지 임베딩을 언어 모델의 프리픽스(prefix)로 변환해 캡션을 생성하는 ClipCap의 구조를 기반으로 한다.

기존의 평가 지표인 BLEU, METEOR, CIDEr, SPICE 등은 생성된 텍스트와 참조 텍스트 간의 n-gram 중첩도나 구조적 유사성에 의존한다. 저자는 이러한 지표들이 "참조 캡션이 항상 고품질이다"라는 가정을 전제로 하기 때문에, 모델이 참조 캡션보다 더 나은 묘사를 하더라도 오히려 낮은 점수를 줄 수 있다는 한계를 지적하며, VLM(Vision-Language Model) 기반의 새로운 평가 방식의 필요성을 강조한다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

본 모델은 ClipCap의 구조를 계승하며, 다음과 같은 파이프라인을 가진다.

- **Visual Encoder**: CLIP의 시각적 인코더를 사용하여 이미지 $\text{v}$를 추출한다.
- **Mapping Network**: CLIP 임베딩을 언어 모델의 공간으로 투영하는 네트워크(MLP 또는 Transformer)이다. 이를 통해 프리픽스 임베딩 $\text{p} = \text{p}_1, \dots, \text{p}_k$를 생성한다.
- **Language Model**: LLaMA 2를 사용하여 프리픽스 $\text{p}$와 이전까지 생성된 토큰들을 입력받아 다음 토큰을 예측한다.

### 2. 학습 절차 및 손실 함수

#### 단계 1: 지도 학습 단계 (Supervised Stage)

전체 COCO 데이터셋의 2% 미만인 약 10,000개의 이미지-캡션 쌍만을 사용하여 초기 신호를 학습한다. 이때의 목표는 주어진 이미지에 대해 정답 캡션 $\text{c} = \text{c}_1, \dots, \text{c}_l$을 정확히 예측하는 것이며, Cross-Entropy 손실 함수를 사용한다.
$$\mathcal{L}_{sup}^v = -\sum_{i=1}^{l} \log p_{\theta}(\text{c}_i | \text{p}_1, \dots, \text{p}_k, \text{c}_1, \dots, \text{c}_{i-1})$$

#### 단계 2: 자기지도 학습 단계 (Self-supervised Stage)

레이블이 없는 데이터에 대해 모델이 직접 캡션 $\text{c}_{model}$을 생성하고, 이 생성된 캡션이 이미지 $\text{v}$와 얼마나 일치하는지를 CLIP 임베딩 공간에서의 코사인 유사도로 측정한다. 손실 함수는 이 유사도의 역수로 정의된다.
$$\mathcal{L}_{unsup}^v = -\cos(\text{v}, \text{c}_{model})$$

### 3. 최적화 기법: Gumbel-Softmax

언어 모델의 토큰 생성 과정(Softmax 이후의 discrete sampling)은 미분이 불가능하여 그래디언트 전파가 끊긴다. 이를 해결하기 위해 본 논문은 **Gumbel-Softmax** 기법을 사용하여 이산 변수를 재매개변수화(reparameterization)함으로써, 자기지도 학습 단계에서도 역전파(backpropagation)를 통한 최적화가 가능하게 하였다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: MS COCO
- **비교 모델(Baselines)**: Oscar, VLP
- **평가 지표**:
  - 전통적 지표: BLEU@4
  - 제안 지표: **RefCompare Score** (생성된 캡션의 CLIP 유사도가 참조 캡션보다 높은 비율을 측정. 0.5 이상이면 참조 캡션과 비슷하거나 더 우수한 품질임을 의미)
  - 인간 평가: Distinctiveness(독특함) 및 Informativeness(정보량)

### 2. 정량적 결과

- **성능**: BLEU@4 점수는 SOTA 모델들과 비슷하거나 약간 낮았으나, **RefCompare Score에서는 압도적으로 높은 수치**를 기록하였다. 이는 모델이 단순히 정답 문장을 흉내 내는 것이 아니라, 이미지의 의미를 더 잘 포착하고 있음을 시사한다.
- **변체 분석**: Mapping Network로 MLP보다 Transformer를 사용했을 때 성능이 더 좋았으며, Transformer를 사용할 경우 LLaMA 2를 추가로 파인튜닝하는 것이 성능 향상에 큰 영향을 주지 않았다(표현력 과잉 가능성).

### 3. 인간 평가 결과

1,000장의 이미지에 대해 5명의 평가자가 Oscar 모델과 본 제안 모델의 결과물을 비교한 결과:

- **Distinctiveness**: 본 모델이 58.6%의 확률로 더 독특한 캡션을 생성함.
- **Informativeness**: 본 모델이 69.2%의 확률로 더 많은 정보를 전달함.

## 🧠 Insights & Discussion

본 연구의 가장 큰 통찰은 **"참조 캡션에 얽매이지 않을 때 모델의 잠재력이 더 발휘된다"**는 점이다. 기존의 지도 학습 방식은 데이터셋에 포함된 평이한 묘사 수준에 맞춰지게 되지만, CLIP의 풍부한 시각-언어 정렬 정보를 활용한 자기지도 학습은 모델이 이미지 속의 더 구체적인 객체와 세부 사항을 묘사하도록 유도한다.

또한, 지도 학습 단계에서 매우 적은 양의 데이터(심지어 100장의 이미지)만으로도 초기 신호를 제공하면, 이후 자기지도 학습이 수렴할 수 있는 가이드라인이 된다는 점을 발견하였다. 이는 완전한 self-supervised learning보다 'weakly-supervised $\rightarrow$ self-supervised' 경로가 효율적임을 보여준다.

다만, 본 논문은 BLEU와 같은 텍스트 일치 기반 지표에서 아주 약간의 하락이 있음을 인정한다. 이는 모델이 참조 캡션에 없는 '더 상세한 정보'를 생성하기 때문에 발생하는 현상으로, 평가 지표의 한계이지 모델의 성능 저하로 보기 어렵다는 해석을 제시한다.

## 📌 TL;DR

본 논문은 CLIP의 유사도를 최대화하는 보조 작업을 통해, 적은 양의 레이블 데이터만으로도 고성능을 내는 자기지도 이미지 캡셔닝 방법론을 제안한다. 특히 Gumbel-Softmax를 통해 미분 가능한 최적화를 구현하였으며, 실험 결과 기존 SOTA 모델보다 더 상세하고 정보량이 많은(informative) 캡션을 생성함을 확인하였다. 이는 향후 데이터 수집이 어려운 특수 도메인에서의 이미지 캡셔닝 연구에 매우 중요한 방향성을 제시한다.
