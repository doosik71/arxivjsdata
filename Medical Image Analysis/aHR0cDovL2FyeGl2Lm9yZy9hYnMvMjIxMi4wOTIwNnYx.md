# Segmentation Ability Map: Interpret deep features for medical image segmentation

Sheng He, Yanfang Feng, P. Ellen Grant, Yangming Ou (2022)

## 🧩 Problem to Solve

본 논문은 의료 영상 분할(Medical Image Segmentation)에 널리 사용되는 심층 합성곱 신경망(Deep CNN)의 내부 동작 원리를 해석하는 문제를 다룬다. 일반적으로 대부분의 연구는 네트워크의 최종 출력층만을 활용하여 분할 결과를 도출하며, 은닉층(Hidden layers)에서 학습된 심층 특징(Deep features)들이 구체적으로 어떤 정보를 담고 있으며 어떻게 최종 결과로 전이되는지는 충분히 이해되지 않은 상태이다.

특히 의료 분야에서는 모델의 결정 근거를 알 수 없는 '블랙박스' 특성이 신뢰성 문제를 야기하므로, 모델의 투명성을 높이는 설명 가능한 AI(XAI) 시스템의 구축이 매우 중요하다. 따라서 본 연구의 목표는 심층 특징의 분할 능력을 정량적으로 측정하고 시각화할 수 있는 도구를 개발하여, 데이터 과학자에게는 모델 개선의 통찰을 제공하고 최종 사용자에게는 모델의 판단 과정을 이해시켜 적절한 신뢰 수준을 설정할 수 있도록 하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 심층 특징으로부터 이진 분할 맵을 생성하는 **Prototype Segmentation (ProtoSeg)** 방법론과 이를 통해 특징의 성능을 정량화하는 **Segmentation Ability (SA) score**를 제안한 것이다.

핵심 아이디어는 동일한 클래스(객체 또는 배경)에 속하는 픽셀들의 특징 벡터 간 거리는 가깝고, 서로 다른 클래스 간의 거리는 멀어야 한다는 직관에 기반한다. 이를 위해 각 클래스의 대표값인 프로토타입(Prototype)을 계산하고, 모든 픽셀을 가장 가까운 프로토타입에 할당함으로써 파라미터 없이도 해당 특징 맵이 가진 분할 능력을 직접적으로 측정할 수 있게 하였다.

## 📎 Related Works

기존의 심층 신경망 해석 방법론으로는 Attention mechanism, Network Dissection, Class Activation Map (CAM) 등이 있다.

1.  **Attention Mechanism**: 특정 영역을 강조하여 시각화할 수 있으나, 각 특징의 중요도를 정량적으로 측정하거나 픽셀 단위의 결정 과정을 이해하는 데 한계가 있다.
2.  **Network Dissection 및 CAM**: 주로 이미지 인식(Image Recognition) 분야에서 제안된 방법들이다. 이미지 인식은 '이미지-to-클래스' 문제인 반면, 분할은 '이미지-to-이미지' 문제이다. 인식 모델의 CAM은 이미지 전체의 중요도를 나타내지만, 분할 모델에서는 각 픽셀이 서로 다른 라벨을 가지므로 절대적인 활성화 값보다 객체와 배경을 분리하는 '분리 능력(Separation ability)'이 더 중요하다.

본 논문의 ProtoSeg는 이러한 한계를 극복하여 분할 작업의 특성에 맞게 특징 맵의 분리 능력을 정량화함으로써 기존 방법론과 차별점을 갖는다.

## 🛠️ Methodology

### 전체 파이프라인 및 ProtoSeg 구조
ProtoSeg는 별도의 학습 파라미터가 필요 없는 **Plug-and-play** 모듈이다. 임의의 심층 특징 $f$가 주어졌을 때, 이를 기반으로 이진 분할 맵인 **Segmentation Ability Map (SAM)**을 생성하고, 이를 정답(Ground-truth)과 비교하여 **SA score**를 산출한다.

### 상세 방법론
1.  **프로토타입 계산**: 
    신경망의 출력값 $B$를 초기 마스크로 사용하여 객체 영역($k=t$)과 배경 영역($k=b$)의 프로토타입 $c_t, c_b$를 각 영역 특징들의 평균으로 계산한다.
    $$c_t = \frac{\sum (B_i \cdot f_i)}{\sum B_i}, \quad c_b = \frac{\sum ((1-B_i) \cdot f_i)}{\sum (1-B_i)}$$
    여기서 $f_i$는 픽셀 $i$에서의 특징 벡터이다.

2.  **분할 맵(SAM) 생성**: 
    각 픽셀의 특징 $f_i$와 프로토타입 간의 유클리드 거리를 측정하고, Softmax 함수를 통해 확률 값을 계산한다. 
    $$p(f_i, c_k) = \frac{\exp(-D(f_i, c_k))}{\sum_{j \in \{t, b\}} \exp(-D(f_i, c_j))}$$
    만약 $p(f_i, c_t) > p(f_i, c_b)$이면 해당 픽셀을 객체(1)로, 그렇지 않으면 배경(0)으로 분류하여 분할 맵 $S_f$를 얻는다.

3.  **SA Score 측정**: 
    생성된 $S_f$와 정답 $G$ 사이의 Dice score를 계산하여 해당 특징의 분할 능력을 정량화한다.
    $$\text{SA Score}(S_f, G) = \frac{2|S_f \cap G|}{|S_f| + |G|}$$

### 운영 모드
-   **Offline ProtoSeg**: 이미 학습된 모델의 특징들을 분석하여 모델을 해석하는 용도로 사용한다.
-   **Online ProtoSeg**: ProtoSeg가 미분 가능하므로, SA score를 학습 손실 함수(Loss function)에 추가하여 은닉층 특징들이 더 높은 분리 능력을 갖도록 유도한다.
    $$\text{Loss} = L_{gt} + \sum \frac{\text{SA Score}}{N}$$

## 📊 Results

### 실험 설정
-   **데이터셋**: BraTS(뇌종양), ISIC(피부 병변), COVID(폐 CT), Prostate(전립선), Pancreas(췌장) 등 5종의 의료 영상 데이터셋을 사용하였다.
-   **모델**: 표준 U-Net 아키텍처를 사용하였으며, 18개의 컨볼루션 레이어를 분석 대상으로 삼았다.

### 주요 결과
1.  **레이어별 분할 능력 전이**: 분석 결과, SA score는 초기 레이어에서 후기 레이어로 갈수록 점진적으로 증가하는 경향을 보였다. 이는 네트워크가 깊어질수록 특징들이 객체와 배경을 더 잘 분리하도록 학습됨을 의미한다.
2.  **Online ProtoSeg의 효과**: SA score를 손실 함수에 포함하여 학습시킨 경우, 최종 출력의 정확도를 떨어뜨리지 않으면서도 중간 레이어 특징들의 SA score가 상승하였다. 이는 특징들의 해석 가능성이 향상되었음을 뜻한다.
3.  **마지막 레이어의 중복성**: 마지막 레이어의 64개 유닛 중 실제 높은 SA score를 갖는 '활성 유닛(Active units)'은 $20\% \sim 50\%$에 불과했다. 이는 U-Net의 마지막 층이 과하게 설계(Oversized)되었으며, 네트워크 프루닝(Pruning)의 필요성을 시사한다.
4.  **결과 품질 추정**: 정답이 없는 테스트 이미지에 대해 마지막 두 레이어 유닛들의 평균 SA score $\mu(x)$를 계산했을 때, 이 값이 실제 Dice 정확도와 강한 상관관계를 보였다. 이를 통해 $\mu(x)$가 낮은 이미지를 '어려운 샘플'로 분류하고 사용자에게 경고를 주는 샘플 거부(Sample rejection) 전략이 유효함을 확인하였다.

## 🧠 Insights & Discussion

본 논문은 U-Net의 내부 동작을 정량적으로 분석함으로써 몇 가지 중요한 통찰을 제시한다.

첫째, U-Net의 인코더 경로(Down-sampling)는 단순히 해상도를 낮추는 것이 아니라, 노이즈를 제거하고 객체의 대략적인 위치를 파악하는 '디노이징 모델'로서의 역할을 수행하며, 디코더 경로에서 세부 경계를 복원한다는 점을 SA score를 통해 정량적으로 입증하였다.

둘째, 심층 특징의 분리 능력이 최종 출력의 선형 결합 결과보다 더 높게 나타나는 경우가 있는데, 이는 각 이미지에 적응적인(Adaptive) 프로토타입을 사용한 ProtoSeg가 전역적으로 학습된 가중치를 사용하는 일반적인 출력층보다 특정 이미지에 대해 더 유연하게 반응하기 때문으로 해석된다.

셋째, 입력 이미지 자체의 강도(Intensity) 기반 분리 능력을 측정함으로써 데이터셋의 특성을 정의할 수 있었다. 예를 들어 BraTS 데이터셋은 입력 값만으로도 분리 능력이 높아 딥러닝의 이득이 상대적으로 적은 반면, 췌장(Pancreas) 데이터셋은 매우 낮아 딥러닝을 통한 특징 추출의 효과가 매우 큼을 확인하였다.

## 📌 TL;DR

본 논문은 심층 신경망의 은닉 특징들을 정량적으로 해석하기 위한 파라미터 없는 방법론인 **ProtoSeg**를 제안한다. 이를 통해 **SA score**라는 지표로 각 레이어와 유닛의 분할 능력을 측정할 수 있으며, 이는 모델의 투명성을 높이는 도구로 활용된다. 특히, 정답이 없는 테스트 데이터의 예측 신뢰도를 추정하여 전문가의 검토가 필요한 위험 샘플을 선별하는 실용적인 기능을 제공하며, U-Net의 구조적 중복성과 노이즈 제거 특성을 정량적으로 밝혀냈다.
