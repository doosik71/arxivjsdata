# Seeing, Listening, Remembering, and Reasoning: A Multimodal Agent with Long-Term Memory

Lin Long et al. (2025)

## 🧩 Problem to Solve

본 논문은 멀티모달 에이전트가 실세계 환경에서 인간과 유사한 지능을 갖추기 위해 필수적인 **장기 기억(Long-Term Memory)** 시스템의 구축과 활용 문제를 다룬다. 기존의 멀티모달 에이전트들은 다음과 같은 두 가지 핵심적인 한계점을 가지고 있다.

첫째, **무한한 정보 처리(Infinite Information Processing)**의 어려움이다. 대부분의 기존 방법론들은 유한한 길이의 오프라인 비디오를 처리하도록 설계되었으나, 실제 에이전트는 실시간으로 들어오는 임의의 길이의 멀티모달 스트림을 지속적으로 처리하고 통합해야 한다.

둘째, **세계 지식 구축(World Knowledge Construction)**의 부재이다. 기존의 비디오 묘사 방식은 저수준의 시각적 세부 사항에 집중하는 경향이 있으며, 인물의 정체성이나 엔티티의 속성과 같은 고수준의 세계 지식을 누적하여 관리하지 못한다. 이는 장기적인 맥락에서 정보의 모호성과 불일치 문제를 야기한다.

따라서 본 연구의 목표는 실시간 시각 및 청각 입력을 통해 엔티티 중심의 멀티모달 장기 기억을 구축하고, 이를 기반으로 복잡한 지시사항을 수행하기 위해 추론할 수 있는 **M3-Agent** 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 멀티모달 입력을 기억하고 추론하는 과정을 인간의 인지 시스템과 유사하게 설계한 점에 있다.

1. **M3-Agent 프레임워크 제안**: 실시간 멀티모달 스트림을 처리하여 **Episodic Memory**(에피소드 기억)와 **Semantic Memory**(의미 기억)를 생성하고, 이를 엔티티 중심의 멀티모달 그래프 형태로 저장하는 시스템을 구축하였다.
2. **엔티티 중심의 일관된 기억 관리**: 얼굴 인식(Facial Recognition) 및 화자 식별(Speaker Identification) 도구를 사용하여 각 인물에게 고유한 ID($face\_id, voice\_id$)를 부여함으로써, 장기간의 입력 스트림 속에서도 인물 정체성의 일관성을 유지하도록 설계하였다.
3. **RL 기반의 반복적 추론 제어**: 단순한 단발성 검색(RAG)이 아닌, 강화학습(RL)을 통해 최적화된 **Multi-turn Reasoning** 프로세스를 도입하여, 필요한 정보를 반복적으로 검색하고 계획을 수정하며 정답에 도달하는 제어 능력을 구현하였다.
4. **M3-Bench 벤치마크 개발**: 에이전트의 장기 기억 및 추론 능력을 평가하기 위해 로봇 시점의 비디오(M3-Bench-robot)와 웹 소스 비디오(M3-Bench-web)로 구성된 새로운 LVQA(Long-Video Question Answering) 데이터셋을 구축하였다.

## 📎 Related Works

### 1. AI 에이전트의 장기 기억

기존의 LLM 에이전트들은 대화 기록이나 실행 궤적을 직접 메모리에 추가하거나, 요약본, 잠재 임베딩, 구조화된 지식 표현을 사용하는 방식을 취해왔다. 그러나 이러한 방식은 주로 텍스트 기반 에이전트에 집중되어 있으며, 멀티모달 입력의 풍부함을 저장하고 장기적인 일관성을 유지하는 데 한계가 있다.

### 2. 온라인 비디오 이해 (Online Video Understanding)

최근의 연구들은 컨텍스트 윈도우를 확장하거나 시각적 토큰을 압축하는 방식을 사용하지만, 이는 무한한 길이의 스트림을 처리하기에는 계산 비용이 너무 높다. 메모리 모듈을 사용하는 방법론들은 시각적 특징(Visual Features)만을 저장하기 때문에, 시간이 지남에 따라 인물의 정체성이나 이벤트의 흐름을 일관되게 추적하는 능력이 부족하다.

## 🛠️ Methodology

M3-Agent는 **Memorization(기억 형성)**과 **Control(제어)**이라는 두 개의 병렬 프로세스로 구성된다.

### 1. Long-Term Memory 구조

기억은 외부 데이터베이스에 **엔티티 중심의 멀티모달 그래프(Entity-centric Multimodal Graph)** 형태로 저장된다. 각 노드는 ID, 모달리티 타입, 콘텐츠, 임베딩, 가중치($weight$), 메타데이터를 포함한다.

- **일관성 유지**: 동일한 엔티티에 해당하는 정보가 들어오면 해당 노드의 가중치를 증가시키고, 서로 충돌하는 정보가 있을 경우 가중치 기반의 투표 메커니즘을 통해 가장 신뢰도 높은 정보를 선택한다.
- **검색 도구**: 텍스트, 이미지, 오디오 쿼리를 지원하는 `search_node`와 특정 시간대 세그먼트를 검색하는 `search_clip` 함수를 제공한다.

### 2. Memorization Workflow

비디오 스트림을 클립 단위로 처리하여 두 종류의 기억을 생성한다.

- **Episodic Memory**: "Alice가 커피를 마시며 아침에는 이것 없이는 못 산다고 말한다"와 같이 관찰된 구체적인 사건을 기록한다.
- **Semantic Memory**: "Alice는 아침에 커피 마시는 것을 선호한다"와 같이 사건으로부터 추출된 일반적인 지식과 속성을 기록한다.
- **정체성 유지 메커니즘**: 얼굴 및 화자 인식 도구를 통해 추출된 특징을 기존 메모리의 노드와 매칭하여 $face\_id$와 $voice\_id$를 부여한다. 이를 통해 서로 다른 클립에서도 동일 인물을 일관되게 참조할 수 있으며, 서로 다른 모달리티(얼굴-목소리) 간의 동등성(Equivalence)을 추론하여 하나의 캐릭터 ID로 통합한다.

### 3. Control Workflow

사용자의 지시가 들어오면 에이전트는 다음과 같은 반복 루프를 통해 작업을 수행한다.

1. **Reasoning**: 현재 보유한 지식이 충분한지 판단한다.
2. **Action**:
    - `[Search]`: 필요한 정보가 부족할 경우 검색 쿼리를 생성하여 메모리에서 관련 정보를 추출한다.
    - `[Answer]`: 충분한 정보가 수집되었을 때 최종 답변을 생성한다.
3. **Iteration**: 최대 $H$ 라운드까지 이 과정을 반복하며 지식을 보완한다.

### 4. Training 절차

- **Memorization 학습**: Qwen2.5-Omni 모델을 기반으로, Gemini-1.5-Pro와 GPT-4o를 통해 생성한 고품질 합성 데이터를 사용하여 모방 학습(Imitation Learning/SFT)을 수행하였다.
- **Control 학습**: Qwen3 모델을 기반으로 **DAPO(Direct Alignment from Preference Optimization)** 알고리즘을 사용하여 강화학습을 수행하였다.
- **보상 함수**: GPT-4o 평가자가 정답 여부를 판단하여 보상 $R_i$를 부여한다.
$$ R_i = \begin{cases} 1, & \text{gpt4o\_evaluator}(q, a, y_i) = \text{True} \\ 0, & \text{otherwise} \end{cases} $$
- **최적화 목표**: 그룹 내 보상의 상대적 이점($\hat{A}_{i,t}$)을 최대화하도록 정책 $\pi_\theta$를 업데이트한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: M3-Bench-robot(100개 영상), M3-Bench-web(920개 영상), VideoMME-long.
- **평가 지표**: 정답률(Accuracy).
- **비교 대상**: Socratic Models(RAG 기반), 기존 온라인 비디오 이해 모델(MovieChat, MA-LMM, Flash-VStream), 프롬프팅 기반 상용 모델 에이전트(Gemini-Agent, Gemini-GPT4o-Hybrid).

### 2. 주요 결과

M3-Agent는 모든 벤치마크에서 기존 베이스라인을 압도하는 성능을 보였다. 특히 가장 강력한 베이스라인인 `Gemini-GPT4o-Hybrid` 대비 다음과 같은 정확도 향상을 기록하였다.

- **M3-Bench-robot**: +6.7%
- **M3-Bench-web**: +7.7%
- **VideoMME-long**: +5.3%

특히 **인물 이해(Person Understanding)**와 **교차 모달 추론(Cross-modal Reasoning)** 분야에서 큰 성능 향상을 보였는데, 이는 엔티티 중심의 일관된 메모리 구조가 효과적으로 작동했음을 시사한다.

### 3. Ablation Study

- **Semantic Memory의 중요성**: 의미 기억을 제거했을 때 정확도가 최대 19.2%까지 급감하여, 고수준의 세계 지식 추출이 추론에 필수적임이 입증되었다.
- **RL 학습의 효과**: DAPO 학습을 적용했을 때 프롬프팅 기반 모델 대비 정확도가 약 8~10% 향상되었다.
- **추론 모드의 영향**: 반복적인 Reasoning 단계와 턴 간 지시사항(inter-turn instruction)을 제거했을 때 성능이 크게 하락하였다.

## 🧠 Insights & Discussion

### 강점

M3-Agent는 단순히 데이터를 저장하는 것을 넘어, 실시간으로 들어오는 멀티모달 데이터를 **구조화된 지식 그래프**로 변환하고, 이를 RL로 최적화된 에이전트가 전략적으로 검색하여 활용한다는 점에서 매우 효율적이다. 특히 인물 ID를 고정함으로써 장기 비디오에서 흔히 발생하는 정체성 혼동 문제를 해결하였다.

### 한계 및 향후 과제

1. **세밀한 세부 사항의 손실**: 모든 세부 정보를 메모리에 저장하는 것은 인지 과부하를 일으키므로, 작업에 필요한 중요한 정보만을 선택적으로 기억하는 **Attention 메커니즘**의 도입이 필요하다.
2. **공간 추론의 한계**: 현재의 텍스트 기반 메모리는 공간적 레이아웃이나 위치 변화를 저장하는 데 한계가 있다. 이를 해결하기 위해 텍스트 외에 시각적 스냅샷(Visual Snapshots)을 메모리에 포함하는 방식이 제안된다.

## 📌 TL;DR

본 논문은 실시간 멀티모달 입력을 통해 **에피소드 기억과 의미 기억을 구축**하고, 이를 **엔티티 중심의 그래프**로 관리하는 **M3-Agent**를 제안한다. 강화학습(DAPO)을 통해 최적화된 반복적 추론 프로세스를 통해 복잡한 지시사항을 성공적으로 수행하며, 새롭게 제안된 **M3-Bench**에서 기존 상용 모델 기반 에이전트보다 뛰어난 성능을 입증하였다. 이 연구는 향후 자율 주행 로봇이나 개인 비서 에이전트가 실세계에서 일관된 세계 지식을 구축하고 활용하는 데 중요한 이정표가 될 것으로 보인다.
