# PRE-TRAINING END-TO-END ASR MODELS WITH AUGMENTED SPEECH SAMPLES QUERIED BY TEXT

Eric Sun, Jinyu Li, Jian Xue, Yifan Gong (Microsoft)

## 🧩 Problem to Solve

본 논문은 End-to-End(E2E) 자동 음성 인식(Automatic Speech Recognition, ASR) 시스템에서 새로운 언어로 확장(Language Expansion)할 때 발생하는 **학습 데이터 부족 문제**를 해결하고자 한다.

일반적으로 E2E ASR 모델의 성능을 높이기 위해서는 대량의 전사된(transcribed) 음성-텍스트 쌍 데이터가 필요하다. 이를 해결하기 위해 Self-Supervised Learning(SSL) 방식의 사전 학습이나, 자원이 풍부한 타 언어의 데이터를 활용하는 Transfer Learning 방식이 사용되어 왔다. 그러나 SSL은 파인튜닝(fine-tuning)에 사용할 레이블링 데이터가 증가할수록 성능 향상 폭이 줄어드는 한계가 있으며, Transfer Learning은 타겟 언어의 음향적 특성을 충분히 커버하기 위해 여전히 방대한 양의 다국어 지도 학습 데이터가 필요하고 계산 비용이 매우 높다는 단점이 있다.

따라서 본 연구의 목표는 추가적인 음성 데이터 수집 없이, 기존의 텍스트 데이터와 소량의 음성 데이터를 활용하여 저비용으로 효과적인 사전 학습용 증강 데이터를 생성하는 방법론을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **텍스트 데이터를 쿼리(Query)로 사용하여 기존 ASR 코퍼스에서 해당 텍스트에 대응하는 음성 특징 세그먼트(Speech Feature Segments)를 추출하고, 이를 조합하여 새로운 음성 샘플을 생성**하는 것이다.

주요 기여 사항은 다음과 같다.

1. **텍스트 기반 음성 데이터 증강 프레임워크**: TTS(Text-to-Speech) 방식과 달리, 실제 음성 데이터에서 세그먼트를 추출하여 조합하므로 더 다양한 화자와 음향 환경을 반영할 수 있으며, TTS 모델 구축 비용이 들지 않아 경제적이다.
2. **다계층 단위(Multi-level Units) 라이브러리 구축**: 단어(Word), 문장 조각(Sentence Piece), 자소(Grapheme letter)의 세 가지 수준으로 라이브러리를 구축하여, 어휘집 외 단어(Out-of-Vocabulary, OOV) 문제를 효과적으로 해결하고 어떤 텍스트에 대해서도 음성 특징을 생성할 수 있게 하였다.
3. **사전 학습 비용 절감**: 방대한 다국어 데이터를 사용하는 Transfer Learning 대비 훨씬 적은 양의 데이터로 유사하거나 더 뛰어난 성능을 달성함으로써 학습 비용을 획기적으로 줄였다.

## 📎 Related Works

기존의 데이터 증강 방식으로는 크게 두 가지 접근법이 제시되었다.

1. **TTS 기반 생성**: 텍스트를 음성으로 합성하여 데이터를 생성하는 방식이다. 하지만 TTS 생성 음성은 실제 환경의 화자 다양성이나 음향적 변동성이 부족하여 품질이 낮으며, 이를 보완하기 위해 인코더를 동결(freeze)하는 등의 추가 조치가 필요하다.
2. **Spliced-data 기반 생성**: 기존 전사 데이터에서 정렬된 오디오 표현을 추출해 조합하는 방식이다. 그러나 이러한 방식은 대규모 텍스트 데이터에 적용하기 어렵고, 특히 OOV 단어를 처리하는 구체적인 해결책을 제시하지 못했다는 한계가 있다.

본 논문은 텍스트 쿼리를 통한 다계층 라이브러리 접근법을 통해 대규모 확장성과 OOV 문제 해결이라는 차별점을 가진다.

## 🛠️ Methodology

### 1. 전체 파이프라인

본 방법론은 크게 **라이브러리 구축**과 **음성 특징 문장 생성**의 두 단계로 구성된다.

### 2. 라이브러리 구축 (Library Establishment)

먼저 Chenone 모델(자소 기반 하이브리드 모델)을 사용하여 음성 특징과 자소(Grapheme letters) 간의 정렬(Alignment) 정보를 생성한다. 이를 바탕으로 다음 세 가지 수준의 단위에 대한 특징 세그먼트 라이브러리를 구축한다.

- **Word Library**: 전체 단어 수준의 특징 세그먼트 수집.
- **Sentence Piece Library**: BPE(Byte Pair Encoding) 모델로 분할된 문장 조각 수준의 세그먼트 수집.
- **Grapheme Library**: 개별 자소 수준의 세그먼트 수집.

각 라이브러리에는 세그먼트 인덱스와 길이를 저장하며, 비정상적인 길이를 가진 세그먼트는 필터링하여 품질을 유지한다.

### 3. 음성 특징 문장 생성 (Speech Feature Utterance Generation)

LM(Language Model) 학습용 텍스트 데이터를 사용하여 새로운 음성 특징 시퀀스를 생성한다. 생성 우선순위는 다음과 같다.
$$\text{Word} \rightarrow \text{Sentence Piece} \rightarrow \text{Grapheme letter}$$
텍스트의 각 단어를 스캔하며, Word 라이브러리에 존재하면 해당 세그먼트를 무작위로 선택한다. 없을 경우 Sentence Piece $\rightarrow$ Grapheme 순으로 내려가며 특징을 추출한다. 최종적으로 추출된 세그먼트들을 단순히 연결(Concatenate)하여 하나의 음성 특징 문장을 완성한다.

이 과정에서 무작위 선택으로 인해 음성의 연속성이 깨질 수 있다는 우려가 있으나, 본 논문은 다음 이유로 이를 정당화한다.

- 오디오 파형이 아닌 필터뱅크(Filterbank) 특징을 사용하므로 연속성 문제에 덜 민감하다.
- 큰 단위(Word)를 우선적으로 사용하여 연결 부위를 최소화한다.
- E2E 모델(Transformer Transducer)은 프레임 단위가 아닌 세그먼트 단위로 예측하므로 이러한 불연속성이 성능에 큰 영향을 주지 않는다.

### 4. 모델 구조 및 학습 절차

- **아키텍처**: Transformer Transducer를 사용한다.
  - **Encoder**: 18개의 Transformer 블록 (320 hidden nodes, 8 attention heads, 2048 feedforward nodes).
  - **Prediction Network**: 2층의 LSTM (1024-dim embedding/hidden).
- **손실 함수**: 입력 음성 특징 $\mathbf{x}$가 주어졌을 때 출력 타겟 레이블 $\mathbf{y}$의 음의 로그 사후 확률(negative log posterior)을 최소화하는 $\text{RNN-T loss}$를 사용한다.
$$\mathcal{L} = -\log P(\mathbf{y}|\mathbf{x})$$
- **학습 절차**: 증강된 데이터와 원본 데이터를 섞어 사전 학습(Pre-training)을 진행한 후, 원본 데이터만으로 파인튜닝(Fine-tuning)을 수행한다.

## 📊 Results

### 1. 실험 설정

- **대상 언어**: 이탈리아어 (Italian)
- **데이터**: 원본 학습 데이터 12,500시간, 테스트 데이터 38시간. LM 학습용 텍스트 9백만 문장을 사용하여 20,000시간의 증강 데이터를 생성하였다.
- **평가 지표**: Word Error Rate (WER)

### 2. 주요 결과

- **도메인 영향 분석**: 단일 도메인(Dictation)에서 추출한 세그먼트로 생성한 데이터가 여러 도메인을 섞어 생성한 데이터보다 더 낮은 WER(28.66% vs 32.18%)을 기록하였다. 이는 증강 데이터 생성 시 원본 데이터의 도메인 일관성이 중요함을 시사한다.
- **사전 학습 효과**:
  - **Baseline**: 원본 데이터만으로 학습 시 $\text{WER} = 14.08\%$.
  - **Mixed Training**: 증강 데이터(20k) + 원본 데이터(12.5k)를 단순히 섞어 학습하면 $\text{WER} = 14.59\%$로 오히려 성능이 하락한다. 이는 증강 데이터의 품질이 원본보다 낮기 때문이다.
  - **Pre-training $\rightarrow$ Fine-tuning**: Mixed 데이터로 사전 학습 후 원본 데이터로 파인튜닝하면 $\text{WER} = 12.85\%$를 달성하며, 베이스라인 대비 **8.7%의 상대적 WER 감소**를 보였다.
- **Transfer Learning과의 비교**: 75,000시간의 다국어 데이터로 사전 학습한 모델을 파인튜닝한 결과($\text{WER} = 12.87\%$)와 본 방법론의 결과가 유사하였다. 이는 훨씬 적은 데이터 양으로도 다국어 Transfer Learning과 대등한 효과를 낼 수 있음을 증명한다.
- **최적 결과**: 다국어 데이터와 본 방법론의 증강 데이터를 모두 사용하여 사전 학습하고 파인튜닝했을 때, $\text{WER} = 12.36\%$로 **최대 12.2%의 상대적 WER 감소**를 달성하였다.

## 🧠 Insights & Discussion

### 1. 강점

본 논문은 TTS와 같은 무거운 생성 모델 없이, 기존의 정렬(Alignment) 정보와 텍스트 데이터만으로 대규모 음성 특징 데이터를 생성할 수 있음을 보여주었다. 특히, 데이터의 '양'보다는 '사전 학습 단계에서의 활용'과 '이후의 파인튜닝'이라는 전략적 접근이 성능 향상의 핵심임을 밝혀냈다.

### 2. 한계 및 논의사항

- **데이터 품질**: 증강 데이터만으로 학습했을 때 성능이 하락한다는 점은, 생성된 데이터가 실제 음성의 자연스러움을 완전히 대체할 수는 없음을 의미한다. 즉, 본 방법론은 '단독 학습용'이 아니라 '사전 학습용'으로 적합하다.
- **언어적 특성**: 실험에 사용된 이탈리아어는 자소-음소(grapheme-to-phoneme) 매핑이 거의 일대일 대응인 언어이다. 이러한 특성이 증강 효율을 높였을 가능성이 있으며, 매핑이 복잡한 언어(예: 영어)에서도 동일한 효과가 나타날지는 추가 검증이 필요하다. 다만, 저자들은 단 1%만이 자소 수준에서 생성된다는 점을 들어 일반화 가능성이 높다고 주장한다.
- **도메인 불일치**: 여러 도메인의 데이터를 섞어 라이브러리를 구축했을 때 성능이 하락하는 결과는, 증강 데이터를 생성할 때 텍스트의 도메인과 매칭되는 음성 세그먼트 라이브러리를 선택적으로 사용하는 전략이 필요함을 시사한다.

## 📌 TL;DR

본 논문은 텍스트 데이터를 쿼리로 하여 기존 ASR 코퍼스에서 단어, 문장 조각, 자소 단위의 음성 특징 세그먼트를 추출하고 조합하는 새로운 데이터 증강 기법을 제안하였다. 이 방법은 TTS 대비 비용이 매우 저렴하며 실제 음향 환경을 더 잘 반영한다. 이탈리아어 실험 결과, 20,000시간의 증강 데이터를 사전 학습에 활용함으로써 베이스라인 대비 WER을 최대 12.2%까지 낮추었으며, 이는 방대한 다국어 데이터를 사용한 Transfer Learning과 대등하거나 더 뛰어난 성능임을 입증하였다. 이 연구는 저자원 언어의 ASR 성능을 높이기 위한 효율적인 사전 학습 전략으로 활용될 가능성이 높다.
