# Your time series is worth a binary image: machine vision assisted deep framework for time series forecasting

Luoxiao Yang, Xinqi Fan, Zijun Zhang (2023)

## 🧩 Problem to Solve

본 논문은 시계열 예측(Time Series Forecasting, TSF) 분야에서 기존 모델들이 주로 수치형 시계열 데이터를 직접 입력으로 사용하는 방식의 한계를 해결하고자 한다. 인간의 뇌는 수치 데이터보다 시각적 정보를 더 효율적으로 처리하며, 시각 피질은 패턴, 모양, 색상을 빠르게 식별하는 능력이 뛰어남이 입증되었다. 딥러닝 아키텍처가 신경과학적 원리와 밀접한 연관이 있다는 점에 착안하여, 본 연구는 수치값에 의존하는 대신 시계열의 시각적 패턴을 활용하는 새로운 프레임워크를 구축하는 것을 목표로 한다. 즉, 수치형 시계열 데이터를 머신 비전(Machine Vision)이 처리하기 용이한 형태로 변환하여 예측 성능을 높이고자 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 수치형 시계열 공간($S$)을 이진 머신 비전 시계열 메트릭 공간($V$)으로 매핑하여, 시계열 예측 문제를 이미지-투-이미지(Image-to-Image) 변환 문제로 재정의하는 것이다. 구체적인 기여 사항은 다음과 같다.

- **MV-DTSA 프레임워크 제안**: 수치 데이터를 이진 이미지 형태로 변환하고, 이를 머신 비전 모델로 분석한 뒤 다시 수치 데이터로 복원하는 'Machine Vision-assisted Deep Time Series Analysis' 프레임워크를 최초로 제시하였다.
- **이진 머신 비전 공간 정의**: 시계열의 상대적 경향성을 보존하면서 수치 데이터를 2차원 이진 텐서로 변환하는 매핑 함수와 그 역함수를 정의하였다.
- **MV-DTSF 모델 설계**: 제안된 프레임워크를 TSF 작업에 적용한 MV-DTSF 모델을 설계하였으며, 복잡한 데이터 분해(Decomposition)나 모델 커스터마이징 없이도 기존 SOTA 모델보다 우수한 성능을 낼 수 있음을 보였다.

## 📎 Related Works

기존의 시계열 예측 모델은 크게 세 가지 단계로 발전해 왔다.

1. **통계적 모델**: AR, MA, ARIMA 등이 있으며 엄격한 수학적 유도에 기반한다.
2. **머신러닝 모델**: SVR, 의사결정 나무, 그리고 이후 CNN, RNN과 같은 딥러닝 모델로 확장되었다.
3. **Transformer 기반 모델**: Informer, Autoformer, FEDformer, PatchTST 등이 있으며, Attention 메커니즘을 통해 장기 의존성(Long-term dependencies)을 포착한다.

**기존 방식의 한계 및 차별점**: 위 모델들은 공통적으로 수치 데이터를 직접 입력으로 사용하며, 성능 향상을 위해 복잡한 데이터 전처리, 분해 기법, 혹은 정교하게 설계된 아키텍처와 손실 함수에 크게 의존한다. 반면, 본 논문에서 제안하는 방식은 데이터를 이진 비전 공간으로 투영하여 분석함으로써, 모델의 구조적 복잡성보다는 데이터의 시각적 표현 방식의 변화를 통해 성능을 개선한다는 점에서 파괴적인 접근 방식을 취한다.

## 🛠️ Methodology

### 전체 파이프라인

MV-DTSA 프레임워크는 크게 세 단계의 과정으로 구성된다.

1. **매핑(Mapping)**: 실제 공간 $S$의 수치 시계열 데이터를 이진 머신 비전 공간 $V$로 변환한다.
2. **분석(Analysis)**: 변환된 $V$ 공간의 데이터를 입력으로 하여 딥러닝 머신 비전 모델을 학습하고 예측 결과 $\hat{v}_y$를 생성한다.
3. **역매핑(Inverse Mapping)**: 필요한 경우, 비전 공간의 출력 결과를 다시 실제 수치 공간 $S$로 복원한다.

### 이진 머신 비전 시계열 메트릭 공간 ($V$)

공간 $V$는 다음과 같이 정의되는 이진 텐서들의 집합이다.
$$V = \{ v \in \mathbb{R}^{c \times h \times t} \mid v_{i,j,k} \in \{0,1\}, i \in [c], j \in [h], k \in [t], \sum_{j=1}^{h} v_{i,j,k} = 1 \}$$
여기서 $c$는 변수의 개수, $t$는 시계열 길이, $h$는 해상도(Resolution)를 의미한다.

**매핑 함수 $f(\cdot)$**: 수치값 $s_{i,k}$를 해상도 $h$와 최대 스케일 하이퍼파라미터 $MS$를 이용하여 이진 벡터로 변환한다.

- $s_{i,k} \ge MS$이면 $j=h$ 위치에 1을 배치한다.
- $s_{i,k} \le -MS$이면 $j=1$ 위치에 1을 배치한다.
- 그 외의 경우, $j = \lfloor \frac{s_{i,k} + MS}{2MS} \rfloor$ (논문 내 식 (3)의 의도는 $h$ 범위 내의 양자화임) 위치에 1을 배치하고 나머지는 0으로 채운다.

**역매핑 함수 $f^{-1}(\cdot)$**: 이진 벡터에서 수치값을 복원하는 식은 다음과 같다.
$$s_{i,k} = f^{-1}(v_{i,1:h,k}) = \sum_{j=1}^{h} \left( (j-0.5) \times \frac{2MS}{h} - MS \right) \times v_{i,j,k}$$

### 시스템 측정 오차 (SME) 및 수렴성

매핑과 역매핑 과정에서 발생하는 오차를 SME(System Measurement Error)라고 정의하며, 이론적으로 $h$가 증가할수록 SME의 상한선이 감소하여 0으로 수렴함을 증명하였다(Proposition 1). 또한, 주어진 $h$에 대해 SME를 최소화하는 최적의 $MS^*$가 존재함을 제시하였다(Proposition 2).

### MV-DTSF 모델 및 학습 절차

- **아키텍처**: 이미지 세그멘테이션 모델인 **U-Net**과 **DeepLabV2**를 백본으로 사용한다. 이는 입력 이미지와 동일한 크기의 출력 이미지를 생성하는 구조이기 때문이다.
- **입력 데이터**: 수치 데이터를 zero-score 정규화한 후 매핑 함수를 통해 이진 이미지로 변환하여 입력한다.
- **손실 함수**: 예측된 이진 이미지 $\hat{v}_y$와 실제 정답 $v_y$ 사이의 거리를 최소화하는 것을 목표로 하며, 이때 거리 함수 $d$는 **Earth Mover's Distance (EMD)**의 변형을 사용한다.
$$\theta^* = \arg\min_{\theta} d(\hat{v}_y, v_y)$$
- **추론**: 학습된 모델 $D$를 통해 $\hat{v}_y$를 얻은 후, 역매핑 함수 $f^{-1}$를 적용하여 최종 수치 예측값 $\hat{s}_y$를 도출한다.

## 📊 Results

### 실험 설정

- **데이터셋**: Electricity, Exchange rate, Traffic, Weather, ILI, ETTm2 총 6종의 공개 데이터셋을 사용하였다.
- **비교 대상**: Informer, Autoformer, FEDformer, PatchTST, DLinear 등 최신 SOTA 모델들과 비교하였다.
- **평가 지표**: MSE(Mean Squared Error)와 MAE(Mean Absolute Error)를 사용하였다.
- **설정**: 모든 MV-DTSF 모델의 해상도 $h$는 계산 자원 제한으로 인해 200으로 설정하였다.

### 주요 결과

- **정량적 결과**: Table 4와 5에 따르면, MV-DTSF는 Group 1(기존 Transformer 계열) 모델들보다 우수한 성능을 보였으며, Group 2 및 3(PatchTST, DLinear 등)과 비교했을 때 대등하거나 경쟁력 있는 성능을 기록하였다.
- **강점**: 특히 **MAE 지표에서 매우 뛰어난 성능**을 보였는데, 이는 제안 방법이 이상치(Anomaly) 처리에 있어 강건함(Robustness)을 가지고 있음을 시사한다.
- **하이퍼파라미터 분석**:
  - $MS$ 값의 변화에 따른 성능 측정 결과, 이론적으로 도출된 최적의 $MS$ 값에서 가장 좋은 성능이 나타났다.
  - $h$ 값이 증가할수록 예측 오차가 감소하는 경향을 확인하여 이론적 수렴성(Proposition 1)을 실험적으로 입증하였다. 다만, $h$가 너무 클 경우(예: 800) 데이터의 희소성(Sparsity)으로 인해 모델 학습이 어려워져 오히려 오차가 증가하는 현상이 관찰되었다.

## 🧠 Insights & Discussion

**강점**:
본 연구는 시계열 데이터를 단순히 수치로 보는 관점에서 벗어나 '시각적 패턴'으로 변환하여 처리하는 완전히 새로운 패러다임을 제시하였다. 특히 복잡한 시계열 분해 기법이나 정교한 모델 설계 없이, 기존의 범용적인 이미지 세그멘테이션 모델을 그대로 활용하여 SOTA급 성능을 냈다는 점이 매우 고무적이다.

**한계 및 논의**:

1. **계산 효율성**: $h$ 값이 커질수록 SME는 줄어들지만, 텐서의 크기가 증가하여 계산 부하가 커진다. 이는 메모리와 정확도 사이의 트레이드-오프(Trade-off) 관계를 형성한다.
2. **데이터 희소성**: $h$가 과도하게 높을 때 발생하는 성능 저하는 딥러닝 모델이 희소한 이진 데이터를 효과적으로 학습하지 못했음을 의미하며, 이를 해결하기 위한 새로운 인코딩 방식이나 모델 수정이 필요할 수 있다.
3. **범용성**: 본 논문은 TSF 작업에 집중하였으나, 저자들은 이 프레임워크가 다른 시계열 분석 작업(분류, 이상탐지 등)에도 적용 가능할 것으로 보고 있다.

## 📌 TL;DR

이 논문은 수치형 시계열 데이터를 이진 이미지 형태의 비전 공간으로 매핑하여 예측하는 **MV-DTSA** 프레임워크를 제안한다. 수치 데이터를 $h$ 해상도의 이진 텐서로 변환하고 U-Net이나 DeepLabV2 같은 머신 비전 모델을 적용함으로써, 복잡한 전처리 없이도 기존 SOTA 모델과 대등하거나 더 우수한 예측 성능(특히 MAE 기준)을 달성하였다. 이 연구는 시계열 분석을 시각적 관점에서 재해석함으로써, 향후 컴퓨터 비전의 강력한 아키텍처들을 시계열 분석 분야에 통합적으로 활용할 수 있는 가능성을 열었다.
