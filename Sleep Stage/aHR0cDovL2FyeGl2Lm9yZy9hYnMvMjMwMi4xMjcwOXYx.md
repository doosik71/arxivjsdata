# Sleep Model – A Sequence Model for Predicting the Next Sleep Stage

Iksoo Choi and Wonyong Sung (2023)

## 🧩 Problem to Solve

본 논문은 수면 장애의 증가로 인해 수면 단계(Sleep Stage)를 분류해야 할 필요성이 커졌으나, 기존의 표준 검사인 수면다원검사(Polysomnography, PSG)가 환자에게 주는 불편함과 높은 비용 문제를 해결하고자 한다.

특히, 단일 채널의 뇌파(EEG), 안구전도(EOG), 근전도(EMG), 심전도(ECG)와 같은 단순 센서를 이용한 수면 단계 분류는 PSG에 비해 정확도가 크게 떨어지는 한계가 있다. 수면 단계는 단순히 개별 에포크(epoch, 보통 30초 단위)의 신호만으로 결정되는 것이 아니라, 시간에 따른 특정한 전이 패턴과 시퀀스 성질을 가지고 있다. 따라서 본 연구의 목표는 수면 단계의 시퀀스 특성을 학습하는 Sleep Model(SLM)을 구축하고, 이를 신호 처리 모델과 결합하여 특히 정보량이 적은 단순 센서 기반의 수면 단계 분류 정확도를 향상시키는 것이다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 자연어 처리(NLP)의 언어 모델(Language Model) 개념을 수면 단계 분류에 도입한 것이다. 문장에서 다음 단어를 예측하듯, 이전 수면 단계들의 시퀀스를 바탕으로 다음 수면 단계가 무엇일지 확률적으로 예측하는 Sleep Model을 설계하였다.

단순히 각 시점의 최적값을 선택하는 Greedy Decoding 방식 대신, 신호 모델(Signal Model)의 결과와 Sleep Model의 사전 확률(Prior Probability)을 결합한 Beam Search Decoding을 적용함으로써, 전역적인 관점에서 가장 확률이 높은 수면 단계 시퀀스를 찾아내어 분류 성능을 최적화하였다.

## 📎 Related Works

기존의 수면 단계 분류는 PSG의 여러 신호원을 사용하는 수동 시각 판독(Manual Visual Scoring)이 표준이다. 최근에는 이를 자동화하기 위해 단일 채널 EEG, EMG, EOG, ECG 또는 PPG 등을 활용한 대리 측정(Surrogate measures) 방식과 딥러닝(DNN) 기반의 접근법이 연구되어 왔다. 그러나 이러한 단순 센서 기반 방식은 PSG 전체를 사용하는 것보다 정확도가 낮을 수밖에 없다.

본 논문은 이러한 한계를 극복하기 위해 언어 모델링 기법을 차용하였다. 특히 $n$-gram 기반의 통계적 모델과 LSTM-RNN 기반의 신경망 모델을 통해 수면 단계의 전이 확률을 모델링함으로써, 기존의 신호 기반 분류 모델이 놓치는 시퀀스 문맥 정보를 보완하고자 하였다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

시스템은 크게 수면 신호로부터 각 에포크의 확률을 계산하는 **Signal Model**과, 수면 단계의 시퀀스 패턴을 학습하여 다음 단계의 확률을 예측하는 **Sleep Model (SLM)**, 그리고 이 두 모델의 출력을 결합하는 **Beam Search Decoding** 과정으로 구성된다.

### 2. Signal Model

신호 모델은 CNN과 Fully-Connected DNN(FC-DNN)으로 구성된다.

- **특징 추출 블록**: 3개의 CNN과 1개의 FC-DNN을 통해 30초 분량의 PSG 데이터 $d_e$를 64차원의 특징 벡터 $f_e$로 변환한다.
- **분류 블록**: 현재 에포크 $e$를 기준으로 앞뒤 2개 에포크를 포함한 총 5개의 특징 벡터 $(f_{e-2}, f_{e-1}, f_e, f_{e+1}, f_{e+2})$를 입력받아 해당 에포크의 수면 단계 $s_e$를 분류한다.
- 본 연구에서는 전체 4개 채널(EEG 2, EMG 1, EOG 1)을 사용하는 모델과, EOG 단일 채널만을 사용하는 모델 두 가지를 구현하여 비교하였다.

### 3. Sleep Model (SLM)

SLM은 이전 단계들의 시퀀스를 바탕으로 다음 단계 $s_e$의 확률 분포를 예측한다.

- **$n$-gram SLM**: 특정 수면 패턴의 출현 빈도를 계산하여 확률을 근사한다. 예를 들어, 3-gram(trigram) 모델에서 $N_1, N_2$ 다음에 $N_3$가 올 확률은 다음과 같이 계산된다.
$$P(N_3 | N_1, N_2) \approx \frac{C(N_1, N_2, N_3)}{C(N_1, N_2)}$$
여기서 $C(\cdot)$는 데이터셋 내 해당 시퀀스의 등장 횟수를 의미한다. 데이터 부족 문제(Data scarcity)를 해결하기 위해 smoothing 기법과 fall-back 옵션을 적용하였다.

- **LSTM-RNN SLM**: 수면 단계 임베딩 층, LSTM 층, 그리고 Softmax 출력 층으로 구성된다. LSTM의 내부 상태를 통해 $n$-gram 모델의 길이 제한 없이 더 긴 문맥 정보를 유지할 수 있다.

### 4. Beam Search Decoding

단순히 매 순간 가장 높은 확률을 선택하는 Greedy Decoding 대신, Signal Model의 결과($P_{sig}$)와 SLM의 결과($P_{SLM}$)를 결합하여 최적의 시퀀스를 찾는다. 결합 식은 로그 영역에서 다음과 같이 정의된다.

$$\log P(s_e) = \log P_{sig}(s_e | d_{e-2}, \dots, d_{e+2}) + \alpha \log P_{SLM}(s_e | s_1, \dots, s_{e-1})$$

여기서 $\alpha$는 두 모델 간의 가중치를 조절하는 하이퍼파라미터이다. Beam Search 알고리즘은 빔 너비(Beam Width) $W$만큼의 상위 후보 시퀀스를 유지하며 전진하며, 최종적으로 전체 시퀀스 확률이 가장 높은 경로를 선택한다.

## 📊 Results

### 1. SLM 성능 평가 (Perplexity)

SLM의 예측 성능은 Perplexity로 측정되었으며, 값이 낮을수록 예측 성능이 좋음을 의미한다.

- **$n$-gram 모델**: HMC 데이터셋 기준 $n=5$까지는 Perplexity가 감소하다가 이후 다시 증가하는 경향을 보였다.
- **LSTM-RNN 모델**: $n$-gram 모델보다 전반적으로 더 낮은 Perplexity를 기록하며 우수한 성능을 보였다.
- **데이터셋 영향**: NCHSDB 데이터셋(소아 대상)이 HMC 데이터셋(성인 대상)보다 크기가 훨씬 커서 SLM 성능이 더 좋게 나타났으며, 두 데이터셋 간의 수면 패턴 차이로 인해 상호 테스트 시 성능 저하가 관찰되었다.

### 2. 수면 단계 분류 성능

LSTM-RNN SLM과 Beam Search Decoding을 적용한 결과는 다음과 같다.

- **4-채널 신호 모델**: SLM을 결합했을 때 성능 향상이 거의 없었다. 이는 이미 4개 채널의 정보량이 충분하여 Signal Model만으로도 충분한 정보를 제공하기 때문으로 분석된다.
- **1-채널(EOG) 신호 모델**: SLM 결합 시 매우 유의미한 성능 향상이 나타났다.
  - **Kappa Score**: 약 6.5% 향상
  - **Accuracy**: 약 4.3% 향상 ($\alpha=0.42$일 때 최적)
- 이는 정보량이 부족한 단순 센서 환경에서 SLM이 제공하는 수면 시퀀스의 사전 정보가 분류 정확도를 높이는 데 결정적인 역할을 함을 보여준다.

## 🧠 Insights & Discussion

본 논문은 수면 단계 분류에 있어 '신호의 특성'뿐만 아니라 '단계 간의 전이 확률'이라는 문맥 정보가 매우 중요하다는 점을 입증하였다. 특히, 고성능의 다채널 센서를 사용할 수 없는 환경(예: 가정 내 장기 모니터링)에서 SLM과 Beam Search Decoding의 결합이 매우 효과적인 보완책이 될 수 있음을 보여주었다.

다만, SLM의 성능이 학습 데이터셋의 규모와 대상(성인 vs 소아)에 민감하게 반응한다는 점은 한계로 지적된다. 이는 개인별 또는 연령별로 수면 패턴이 다를 수 있음을 시사하며, 향후 사용자 맞춤형 SLM이나 범용적인 수면 패턴 모델에 대한 연구가 필요함을 의미한다. 또한, 본 연구에서는 EOG 센서만을 대상으로 검증하였으나, 제안된 구조는 PPG, ECG 등 다른 대리 센서 모델에도 동일하게 적용 가능할 것으로 보인다.

## 📌 TL;DR

본 연구는 수면 단계의 시퀀스 특성을 학습하는 **Sleep Model(SLM)**을 제안하고, 이를 **Beam Search Decoding**을 통해 신호 모델과 결합하여 수면 단계 분류 정확도를 높였다. 특히 정보량이 적은 **단일 채널(EOG) 센서** 환경에서 분류 성능(Kappa, Accuracy)을 유의미하게 향상시켰으며, 이는 향후 웨어러블 기기 등을 이용한 저비용·고효율 수면 모니터링 시스템 구축에 중요한 기여를 할 수 있을 것으로 기대된다.
