# Imitating Language via Scalable Inverse Reinforcement Learning

Markus Wulfmeier, Michael Bloesch, Nino Vieillard, et al. (2024)

## 🧩 Problem to Solve

대부분의 대규모 언어 모델(LLM) 학습은 다음 토큰 예측을 위한 최대 가능도 추정(Maximum Likelihood Estimation, MLE) 방식의 모방 학습(Imitation Learning)에 의존한다. MLE는 단순함과 확장성이라는 강력한 장점이 있으나, 시퀀스 생성 과정에서 모델이 생성한 토큰이 다시 입력으로 들어가는 자기회귀(Autoregressive) 구조의 특성상 여러 한계가 존재한다.

가장 큰 문제는 학습 시에는 정답 데이터만 사용하는 '교사 강요(Teacher Forcing)' 방식을 사용하지만, 추론 시에는 모델 자신의 출력을 기반으로 다음 토큰을 생성해야 한다는 점이다. 이로 인해 학습 분포와 추론 분포가 달라지는 노출 편향(Exposure Bias)이 발생하며, 초기에 발생한 작은 오류가 누적되어 최종 결과물이 학습 분포에서 완전히 벗어나는 복리 오류(Compounding Errors) 문제가 발생한다. 또한, MLE는 정답 데이터를 정확히 복제하는 데 집중하므로 생성물의 다양성(Diversity)을 유지하는 능력이 부족하다.

본 논문의 목표는 언어 생성을 단순한 토큰 예측이 아닌 순차적 의사결정(Sequential Decision Making) 문제로 정의하고, 역강화학습(Inverse Reinforcement Learning, IRL) 관점에서 보상 함수를 추출하고 시퀀스 전체를 최적화함으로써 MLE의 한계를 극복하고 성능과 다양성을 동시에 확보하는 것이다.

## ✨ Key Contributions

본 연구의 핵심적인 기여는 다음과 같다.

첫째, LLM의 파인튜닝 단계에서 개별 토큰의 가능도를 최대화하는 대신, 보상 함수를 추출하고 시퀀스 생성을 직접 최적화하는 RL 중심의 모방 학습 관점을 제시한다.

둘째, 기존의 Inverse Soft Q-Learning(IQLearn)을 MLE의 시간차 정규화(Temporal Difference Regularization) 확장판으로 재구성하였다. 이를 통해 MLE와 IRL 사이의 이론적 연결 고리를 제공하며, 온라인 데이터 생성 없이도 고정된 데이터셋만으로 학습 가능한 효율적인 오프라인 IRL 학습 체계를 구축하였다.

셋째, 다양한 규모의 모델(T5, PaLM2)과 데이터셋(GSM8k, XSUM 등)을 통해 IRL 기반 모방 학습이 MLE 대비 작업 성능을 유지하거나 향상시키면서도, 생성물의 다양성을 유의미하게 높인다는 것을 입증하였다.

넷째, IRL을 통해 추출된 보상 함수가 실제 작업 성능 지표와 높은 상관관계를 가짐을 분석하여, 향후 RLHF(Reinforcement Learning from Human Feedback) 단계에서 더 견고한 보상 함수를 구축하는 데 기여할 가능성을 보여주었다.

## 📎 Related Works

기존의 모방 학습은 크게 행동 복제(Behavior Cloning, BC)와 역강화학습(IRL)으로 나뉜다. LLM의 SFT(Supervised Fine-Tuning) 과정에서 사용하는 MLE는 본질적으로 BC와 동일하며, 이는 전문가의 행동을 직접 회귀 분석으로 흉내 내는 방식이다. BC는 데이터 커버리지가 부족할 경우 추론 시 발생하는 분포 변화에 취약하여 앞서 언급한 복리 오류 문제에 직면한다.

반면 IRL은 전문가의 시연 데이터로부터 내재된 보상 함수(Reward Function)를 추론하고, 이 보상을 최대화하는 정책을 학습한다. IRL은 이론적으로 환경과의 상호작용을 통해 BC의 복리 오류 문제를 해결할 수 있다. 최근에는 GAIL(Generative Adversarial Imitation Learning)과 같은 적대적 학습 방식이나, IQLearn과 같이 분포 매칭(Distribution Matching)을 이용한 비적대적 방식이 제안되었다.

기존의 LLM 적용 연구들은 주로 온라인 샘플링을 필요로 하여 계산 비용이 매우 높았으나, 본 논문은 IQLearn의 재구성을 통해 오프라인 환경에서도 확장 가능한 IRL 적용 방법을 제시함으로써 기존 연구와의 차별점을 갖는다.

## 🛠️ Methodology

### 전체 시스템 구조
본 연구는 언어 생성을 마르코프 결정 과정(MDP)으로 모델링한다. 상태 $s_i$는 현재까지 생성된 토큰들의 집합이며, 액션 $a_i$는 다음에 생성할 토큰 $x_i$이다. 본 논문은 분포 매칭 관점에서 정책 $\pi$의 상태-액션 분포 $\mu^\pi$와 전문가 $\pi^E$의 분포 $\mu^E$ 사이의 발산(Divergence)을 최소화하는 것을 목표로 한다.

### 주요 구성 요소 및 방정식
연구팀은 Inverse Soft Q-Learning을 재구성하여 MLE와의 관계를 명시적으로 정의하였다. 최종적으로 제안된 $J^{IQLearn}$ 목적 함수는 다음과 같다.

$$J^{IQLearn}(v_r^\lambda, \pi_r) = \mathbb{E}_{\mu^E} [ \lambda(v_r^\lambda + \log \pi_r - \mathbb{E}_{s'}[\gamma v_{r}^\lambda])^2 - \log(\pi_r) ]$$

이 식은 두 부분으로 나뉜다.
1.  **MLE 항 ($-\log \pi_r$):** 기존의 MLE와 동일하게 전문가의 토큰 선택 확률을 높이는 역할이다.
2.  **정규화 항 ($\lambda(\dots)^2$):** 시간차(Temporal Difference, TD) 정규화 항으로, 현재 액션의 로그 확률과 상태 가치 함수(State Value Function)의 변화량 사이의 일치성을 강제한다. 여기서 $\lambda$는 정규화의 강도를 조절하는 하이퍼파라미터이다.

### 학습 절차 및 추론
- **학습:** $\lambda=0$일 때는 표준 MLE와 동일하게 작동하며, $\lambda$ 값을 높임으로써 IRL의 특성을 강화한다. 특히 오프라인 버전의 IQLearn은 모델의 온라인 샘플링 없이 고정된 전문가 데이터셋 $\mu^E$만으로 학습하므로 계산 효율성이 매우 높다.
- **추론:** 학습된 정책 $\pi_r$을 사용하여 자기회귀적으로 토큰을 생성한다.
- **가치 함수 구현:** 실제 구현에서는 LLM의 소프트맥스 층 이전의 로짓(logits)을 Q-값으로 취급하여 학습을 진행하며, 상태 가치 함수 $V(s)$는 $\log \sum_a \exp Q(s,a)$로 계산한다.

## 📊 Results

### 실험 설정
- **모델:** T5 (Base, Large, XL), PaLM2 (Gecko)
- **데이터셋:** GSM8k (수학 추론), XSUM (요약), TLDR (요약), WMT22 (번역)
- **평가 지표:** 작업 성능(Accuracy, ROUGE-1/2) 및 다양성(Self-BLEU, 여기서 낮은 점수가 높은 다양성을 의미)
- **비교 대상:** 표준 MLE, 엔트로피 정규화 MLE, GAIL, IQLearn (Online/Offline)

### 주요 결과
1.  **성능과 다양성의 Trade-off:** 실험 결과, IQLearn은 MLE보다 높은 작업 성능을 보이거나 최소한 대등한 수준을 유지하면서, 생성물의 다양성(Self-BLEU 감소)을 훨씬 더 크게 개선하였다. 특히 수학 및 추론 작업(GSM8k)에서 IRL 기반 방식의 성능 향상이 뚜렷하게 나타났다.
2.  **오프라인 IRL의 효율성:** 온라인 IQLearn과 오프라인 IQLearn을 비교했을 때, 작업 성능 면에서는 큰 차이가 없었으나 오프라인 방식이 계산 비용을 획기적으로 줄일 수 있음을 확인하였다. 이는 언어 생성의 특성상 한 번 잘못 생성된 토큰을 되돌릴 수 없는 '연결 구조(Concatenation Dynamics)' 때문에 온라인 샘플링의 이점이 상대적으로 적기 때문으로 분석된다.
3.  **데이터 규모 및 견고성:** 데이터셋의 크기가 작을수록, 혹은 타겟 시퀀스의 길이가 길수록 IRL의 이점이 더 크게 나타났다. 특히 XSUM의 소규모 데이터셋 실험에서 MLE는 심한 과적합(Overfitting)을 보였으나, IQLearn은 TD 정규화를 통해 이를 효과적으로 억제하며 더 견고한 성능을 보였다.
4.  **보상 함수의 유효성:** IQLearn을 통해 학습된 내재적 보상 함수와 실제 성능 지표 간의 스피어만 상관계수(Spearman's rank correlation)를 분석한 결과, MLE보다 훨씬 높은 상관관계를 보였다. 이는 IRL이 데이터로부터 작업의 질적 특성을 효과적으로 추출하고 있음을 의미한다.

## 🧠 Insights & Discussion

본 논문은 LLM 학습에서 단순한 행동 복제(MLE)를 넘어 순차적 의사결정 관점의 모방 학습(IRL)이 가진 잠재력을 입증하였다.

가장 주목할 점은 **다양성과 성능의 공존**이다. 기존 MLE에 단순히 엔트로피 정규화를 추가하는 방식은 성능 저하를 초래하는 경우가 많았으나, IRL은 시퀀스 전체의 가치 구조를 학습함으로써 성능을 유지하면서도 더 다양한 결과물을 생성할 수 있게 한다. 이는 향후 RLHF 단계에서 인간 평가자에게 더 다양한 샘플을 제공하여 더 질 높은 선호도 데이터를 수집하는 데 기여할 수 있다.

또한, **오프라인 IRL의 실용성**을 확인하였다. RL 기반 방법론의 최대 단점인 무거운 샘플링 비용을 제거하고도 성능 이득을 얻을 수 있다는 점은, 이 방법론이 사전 학습(Pre-training) 단계나 매우 거대한 데이터셋으로 확장될 수 있는 가능성을 시사한다.

다만, 본 연구는 Self-BLEU와 같은 정량적 다양성 지표에 의존하고 있으며, 이러한 다양성 증가가 실제 인간이 느끼는 질적 향상이나 후속 RL 최적화 단계의 탐색(Exploration) 효율로 이어지는지에 대해서는 추가적인 검증이 필요하다.

## 📌 TL;DR

이 논문은 LLM의 SFT 과정을 MLE(행동 복제)가 아닌 **역강화학습(IRL)** 관점에서 재구성하여, **"MLE + 시간차(TD) 정규화"**라는 새로운 최적화 목적 함수를 제안한다. 이를 통해 온라인 샘플링 없이도 **작업 성능 향상과 생성 다양성 증대**라는 두 마리 토끼를 잡았으며, 추출된 보상 함수가 실제 성능과 밀접하게 연관되어 있음을 입증하였다. 이 연구는 향후 효율적인 LLM 정렬(Alignment) 및 포스트 트레이닝 파이프라인 구축에 중요한 이론적/실천적 토대를 제공한다.
