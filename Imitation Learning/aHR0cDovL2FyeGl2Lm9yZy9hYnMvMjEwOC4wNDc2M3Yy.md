# Imitation Learning by Reinforcement Learning

Kamil Ciosek (2022)

## 🧩 Problem to Solve

본 논문은 복잡한 작업에서 보상 함수(reward function)를 직접 설계하는 것이 매우 어렵다는 점에서 출발한다. 이를 해결하기 위해 전문가의 시연(demonstration)으로부터 정책을 학습하는 Imitation Learning (IL)이 사용되지만, 기존 방식들은 각각의 한계점을 가지고 있다.

- **Behavioral Cloning (BC):** 전문가 데이터에 대해 지도 학습을 수행하지만, 학습 오류가 누적되는 compounding errors 문제로 인해 데이터가 제한적인 상황에서 취약하다.
- **Adversarial IL (예: GAIL):** 에이전트와 전문가의 데이터 분포 간의 divergence를 최소화하며, 판별자(discriminator) 네트워크를 통해 비정상 보상(non-stationary reward)을 생성한다. 그러나 판별자와 정책 네트워크를 동시에 학습시켜야 하므로 학습 과정이 매우 불안정하고 brittle하다.

따라서 본 연구의 목표는 구현이 더 간단하고 배포가 용이하며, 이론적인 성능 보장이 가능한 새로운 Imitation Learning 알고리즘을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Imitation Learning 문제를 고정된 보상을 가진 Reinforcement Learning (RL) 문제로 환원(reduction)**시키는 것이다.

- **단순한 보상 설계:** 전문가의 궤적에 포함된 상태-행동 쌍$(s, a)$에 대해서는 보상을 $1$로, 그 외에는 $0$으로 설정하는 정적인 내재적 보상(stationary intrinsic reward)을 정의한다.
- **이론적 정당성 부여:** 이전 연구(Wang et al., 2019)에서 휴리스틱하게 사용되었던 이 방식을 이론적으로 분석하여, 전문가가 결정론적(deterministic)일 때 학습된 정책이 전문가의 분포와 얼마나 가까운지를 Total Variation (TV) distance를 통해 증명하였다.
- **Adversarial IL과의 연결:** 제안하는 방식이 이론적으로 Adversarial IL 알고리즘들과 연관되어 있음을 보임으로써, 판별자 학습 없이도 유사한 효과를 얻을 수 있음을 시사한다.

## 📎 Related Works

- **Behavioral Cloning:** 단순하지만 MDP의 순차적 특성을 무시하여 데이터 부족 시 성능이 급격히 저하된다.
- **Apprenticeship Learning:** 보상을 위한 선형 표현을 가정하고 반복적으로 RL 솔버를 호출하지만, 본 논문의 방식은 RL 솔버를 단 한 번만 호출한다.
- **Adversarial IL (GAIL, InfoGAIL 등):** 분포 간의 거리(JS divergence, Wasserstein distance 등)를 최소화한다. 본 논문의 방식은 TV distance를 최소화하는 것과 유사하며, 판별자 학습 과정이 필요 없다는 점에서 훨씬 효율적이다.
- **Random Expert Distillation (Wang et al., 2019):** 제안된 알고리즘과 동일한 보상 구조를 사용했으나, 이론적 분석이 결여되어 있었다. 본 논문은 이 부분의 이론적 공백을 메운다.
- **SQIL:** 유사한 접근 방식을 취하지만, 비정상 보상을 사용하며 이론적 보장이 없다.

## 🛠️ Methodology

### 1. 전체 파이프라인
알고리즘의 절차는 매우 단순하다.
1. 전문가 데이터셋 $D$를 수집한다.
2. 다음과 같은 내재적 보상 함수 $R^{int}$를 정의한다.
   $$R^{int}(s, a) = \mathbb{1}\{(s, a) \in D\}$$
3. 임의의 RL 솔버(예: SAC)를 사용하여 이 보상을 최대화하는 정책 $\pi_I$를 학습한다.

### 2. 이론적 가정 및 보장 (Proposition 1)
본 논문은 다음 세 가지 가정을 전제로 한다.
- **Assumption 1:** 전문가 정책은 결정론적(deterministic)이다.
- **Assumption 2:** 전문가 정책은 irreducible 및 aperiodic한 마르코프 체인을 유도한다.
- **Assumption 3:** 모방 학습자의 정책 또한 irreducible 및 aperiodic한 마르코프 체인을 유도한다.

이 조건 하에서, 전문가 데이터의 크기 $N$이 충분히 클 때, 높은 확률로 모방 학습자의 상태-행동 분포 $\rho_I$와 전문가의 분포 $\rho_E$ 사이의 Total Variation distance는 다음과 같이 제한된다.
$$\| \rho_E - \rho_I \|_{TV} \le \eta$$
또한, 임의의 유계된 외재적 보상 함수 $R$에 대해 학습자의 기대 보상은 전문가의 보상과 $\eta$ 이내의 차이를 보인다.
$$\mathbb{E}_{\rho_I}[R] \ge \mathbb{E}_{\rho_E}[R] - \eta$$

### 3. 연속적 공간으로의 확장 (Continuous Relaxation)
이론적 분석은 이산 공간(finite MDP)을 대상으로 하지만, 실제 실험에서는 연속적 공간을 다루기 위해 보상 함수를 다음과 같이 완화하여 사용한다.
$$R'_I(s, a) = 1 - \min_{(s', a') \in D} d_{L2}((s, a), (s', a'))^2$$
이는 전문가 데이터셋 내의 가장 가까운 점과의 거리 기반으로 보상을 부여하는 방식이다.

## 📊 Results

### 실험 설정
- **환경:** PyBulletgym의 Hopper, Ant, Walker, HalfCheetah.
- **비교 대상:** BC, GAIL, GMMIL, SQIL. (모두 공정한 비교를 위해 SAC를 기본 RL 알고리즘으로 구현함)
- **지표:** 외재적 보상(Extrinsic Return).

### 주요 결과
- **데이터 양에 따른 성능:** 전문가 데이터가 충분할 때는 대부분의 방법이 잘 작동하지만, 데이터가 적은 상황에서는 ILR(본 논문의 방식)이 BC보다 훨씬 우수하며, Ant 환경 등에서는 다른 Adversarial 방식들보다 더 적은 데이터로도 높은 성능을 달성했다.
- **샘플 효율성:** 환경 상호작용 횟수에 따른 학습 곡선 분석 결과, Ant 환경에서 더 높은 샘플 효율성을 보였으며, 다른 환경에서도 경쟁력 있는 성능을 보였다.
- **내재적 보상과 외재적 보상의 관계:** 실험 결과, 내재적 보상($R^{int}$)의 증가와 외재적 보상(실제 성능)의 증가 사이에 강한 선형 관계가 있음이 확인되었다. 이는 이론적 분석(Lemma 7)이 실제 연속 공간 환경에서도 유효함을 시사한다.
- **런타임:** BC를 제외하고는 SQIL과 유사한 수준의 빠른 런타임을 기록했다. 특히 판별자를 학습시켜야 하는 GAIL이나 커널 계산 비용이 큰 GMMIL보다 훨씬 효율적이다.

## 🧠 Insights & Discussion

### 강점
- **단순성:** 판별자 네트워크를 설계하거나 하이퍼파라미터를 튜닝할 필요 없이, 기존의 RL 솔버를 그대로 사용할 수 있어 구현 비용이 매우 낮다.
- **이론적 토대:** 단순히 "잘 된다"는 휴리스틱을 넘어, 결정론적 전문가에 대한 성능 상한선을 수학적으로 증명하여 신뢰성을 높였다.
- **효율성:** Adversarial IL의 이점을 누리면서도 계산 복잡도를 크게 낮추었다.

### 한계 및 비판적 해석
- **결정론적 전문가 가정:** 본 논문의 이론적 증명은 전문가가 deterministic하다는 가정에 의존한다. 저자 또한 stochastic 전문가의 경우 이 환원 방식이 실패함을 명시하였다. 이는 실제 환경에서 전문가의 행동에 무작위성이 포함될 경우 성능 보장이 어려움을 의미한다.
- **에르고딕성(Ergodicity) 가정:** 마르코프 체인이 irreducible 및 aperiodic해야 한다는 가정이 필요하며, 이를 만족하지 못하는 특수한 환경에서는 이론적 보장이 작동하지 않을 수 있다.
- **보상 완화의 이론적 간극:** 이론은 이진 보상($0$ 또는 $1$)을 다루지만, 실제 실험에서는 $L_2$ 거리 기반의 완화된 보상을 사용한다. 저자는 실무적으로 잘 작동함을 보였으나, 이 완화된 형태에 대한 엄밀한 이론적 증명은 제시되지 않았다.

## 📌 TL;DR

본 논문은 Imitation Learning 문제를 **"전문가 데이터에 포함된 상태-행동 쌍에 보상을 주는 단순한 RL 문제"**로 환원하여 해결하는 방법을 제안한다. 결정론적 전문가에 대해 정책 분포의 유사성을 이론적으로 증명하였으며, 실험을 통해 GAIL과 같은 복잡한 Adversarial 방식 없이도 경쟁력 있는 성능과 빠른 학습 속도를 얻을 수 있음을 보였다. 이는 향후 판별자 학습의 불안정성 없이 효율적인 모방 학습을 구현하는 데 중요한 기초가 될 것으로 보인다.
