# Confidence-Aware Imitation Learning from Demonstrations with Varying Optimality

Songyuan Zhang, Zhangjie Cao, Dorsa Sadigh, Yanan Sui (2021)

## 🧩 Problem to Solve

전형적인 모방 학습(Imitation Learning, IL) 방식들은 제공된 시연(Demonstration) 데이터가 항상 최적(Optimal)인 전문가로부터 생성되었다고 가정한다. 그러나 실제 환경에서 수집되는 데이터는 최적의 경로뿐만 아니라 하위 최적(Suboptimal), 무작위(Random), 혹은 작업에 실패한 데이터가 섞여 있는 경우가 많다.

이러한 데이터의 다양성은 더 넓은 범위의 데이터를 활용할 수 있다는 장점이 있지만, 동시에 심각한 문제를 야기한다. 최적이지 않은 데이터를 그대로 학습할 경우, 에이전트는 전문가의 성능에 도달하지 못하는 하위 최적 정책(Suboptimal policy)을 학습하게 된다. 특히 작업에 실패하거나 악의적인(Malicious) 데이터가 포함되어 있을 경우, 모델의 성능은 더욱 크게 저하된다. 따라서 본 논문의 목표는 시연 데이터의 최적성이 서로 다른 상황에서, 유용한 정보는 추출하고 유해한 정보는 필터링하여 고성능 정책을 학습할 수 있는 일반적인 프레임워크를 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 각 시연 데이터의 최적성 정도를 나타내는 **신뢰도 점수(Confidence score)**를 학습하고, 이를 통해 데이터를 재가중치(Reweight)하여 학습하는 것이다.

주요 기여 사항은 다음과 같다.
1. **CAIL (Confidence-Aware Imitation Learning) 프레임워크 제안**: 시연 데이터의 신뢰도 점수와 정책을 동시에 학습하는 일반적인 구조를 제안한다.
2. **Bi-level Optimization 도입**: 신뢰도를 업데이트하는 Outer-level과 정책을 업데이트하는 Inner-level로 구성된 최적화 구조를 통해, 별도의 정답(Ground-truth) 신뢰도 라벨 없이도 자동으로 최적의 신뢰도를 학습한다.
3. **이론적 보장**: 제안된 Bi-level 최적화 알고리즘이 일정 조건 하에서 최적의 신뢰도 설정으로 수렴함을 수학적으로 증명하였다.
4. **실증적 검증**: MuJoCo 시뮬레이션 및 실제 로봇 팔(UR5e) 실험을 통해, 최적의 데이터가 전혀 없는 상황에서도 기존 방법론보다 우수한 성능을 보임을 입증하였다.

## 📎 Related Works

기존의 모방 학습 및 하위 최적 데이터 처리 방식은 다음과 같은 한계를 가진다.

- **표준 모방 학습 (BC, IRL, GAIL, AIRL)**: 모든 시연 데이터가 최적이라는 가정하에 설계되어, 데이터의 질이 낮을 경우 성능이 급격히 저하된다.
- **순위 기반 방식 (T-REX, D-REX, SSRR)**: 시연 데이터 간의 상대적 순위를 사용하여 보상을 학습한다. 하지만 T-REX는 소량의 순위 데이터만으로는 일반화 능력이 떨어지며, D-REX와 SSRR은 데이터에 노이즈를 섞어 자동으로 순위를 생성하는데, 이는 데이터 자체가 매우 무작위이거나 악의적인 경우 잘못된 순위를 생성할 위험이 있다.
- **신뢰도 기반 방식 (2IWIL, IC-GAIL)**: 각 데이터의 신뢰도를 사용하지만, 정답 신뢰도 라벨이 필요하거나, 최적 데이터가 비최적 데이터보다 더 많아야 한다는 강한 가정이 필요하다. 또한 특정 모방 학습 알고리즘에 종속적인 경우가 많다.

CAIL은 이러한 제약에서 벗어나, 어떤 모방 학습 알고리즘과도 결합 가능하며 정답 라벨 없이도 평가 데이터(Evaluation data)를 통해 신뢰도를 자동 학습한다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조 및 파이프라인

CAIL은 정책을 학습하는 **Inner-level optimization**과 신뢰도를 학습하는 **Outer-level optimization**이 중첩된 Bi-level 최적화 구조를 가진다.

1. **신뢰도 정의**: 상태-행동 쌍 $(s, a)$에 대해 스칼라 값 $\beta(s, a)$를 부여한다. 이 값은 해당 행동이 최적 정책에 나타날 가능성을 나타낸다.
2. **데이터 재가중치**: 시연 데이터의 분포 $p_{\pi_d}(s, a)$에 신뢰도 $\beta$를 곱하여 새로운 분포 $p_{\pi_{new}}(s, a) = \beta(s, a) p_{\pi_d}(s, a)$를 생성한다.
3. **학습 루프**: 
    - **Inner-level**: $\beta$가 고정된 상태에서, 재가중치된 데이터를 사용하여 모방 학습 손실 함수 $L_{in}$을 최소화하는 정책 파라미터 $\theta$를 학습한다.
    - **Outer-level**: 학습된 정책의 성능을 평가 손실 함수 $L_{out}$으로 측정하고, 이를 최소화하도록 $\beta$를 업데이트한다.

### 상세 최적화 절차

연산 효율을 위해 논문은 Pseudo-update 방식을 포함한 근사 최적화 식을 제안한다.

- **Outer-level Update ($\beta$ 업데이트)**:
  현재의 $\theta_\tau$와 $\beta_\tau$를 사용하여 정책을 한 단계 가상으로 업데이트한 $\theta'_{\tau+1}$을 구한다.
  $$\theta'_{\tau+1} = \theta_\tau - \mu \nabla_\theta L_{in}(s, a; \theta_\tau, \beta_\tau)$$
  이후 $L_{out}$을 최소화하는 방향으로 $\beta$를 업데이트한다.
  $$\beta_{\tau+1} = \beta_\tau - \alpha \nabla_\beta L_{out}(\theta'_{\tau+1})$$

- **Inner-level Update ($\theta$ 업데이트)**:
  업데이트된 $\beta_{\tau+1}$을 사용하여 실제 정책 파라미터 $\theta$를 업데이트한다.
  $$\theta_{\tau+1} = \theta_\tau - \mu \nabla_\theta L_{in}(s, a; \theta_\tau, \beta_{\tau+1})$$

### 구체적 구현 (AIRL 기반)

본 논문에서는 구체적인 구현체로 **Adversarial Inverse Reinforcement Learning (AIRL)**을 사용하였다.

- **Inner Loss ($L_{in}$)**: AIRL의 판별자($D$)와 생성자($G$)의 손실 함수를 사용한다. 특히 판별자의 손실 함수에 신뢰도 $\beta$가 곱해져, 신뢰도가 높은 데이터에 더 큰 가중치를 두어 학습하게 된다.
- **Outer Loss ($L_{out}$)**: 소량의 궤적 순위 데이터 $D_E$를 활용한 **Ranking Loss**를 사용한다. AIRL의 판별자가 추정한 보상 함수 $R'_{\theta_D}$를 통해 계산된 궤적의 기대 수익 $\eta'_{\xi}$가 실제 순위와 일치하도록 유도한다. 이때 미분 가능성을 위해 Margin Ranking Loss를 수정하여 사용하였다.
  $$L_{out}(\theta_D) = \sum_{i} \sum_{j>i} R_K [ \eta'_{\xi_i}, \eta'_{\xi_j}; I[\eta_{\xi_i} > \eta_{\xi_j}] ]$$

## 📊 Results

### 실험 설정
- **환경**: MuJoCo (Reacher, Ant), Franka Panda Arm (시뮬레이션), UR5e (실제 로봇).
- **데이터**: 최적부터 실패까지 다양한 수준의 시연 데이터를 혼합하여 구성. 평가 데이터로는 전체의 5%에 해당하는 소량의 순위 정보만 사용.
- **비교 대상**: GAIL, AIRL, 2IWIL, IC-GAIL, T-REX, D-REX, SSRR.

### 주요 결과
1. **정량적 성능**: 모든 환경에서 CAIL이 다른 베이스라인보다 가장 높은 기대 수익(Expected Return)을 달성하였다. 특히 실제 로봇 팔 실험에서 AIRL, GAIL 등은 성능 변동이 매우 심했으나, CAIL은 매우 안정적이고 높은 성능을 보였다.
2. **최적 데이터 부재 상황**: Ant 환경에서 최적 데이터를 모두 제거하고 하위 최적 데이터만 제공했을 때에도, CAIL은 다른 방법론보다 월등히 높은 성능을 보였다. 이는 CAIL이 하위 최적 데이터 내에서도 상대적으로 더 나은 정보를 효율적으로 추출할 수 있음을 의미한다.
3. **데이터 효율성**: 순위 라벨의 비율을 1%까지 낮추어도 성능이 거의 저하되지 않았으며, 이는 T-REX와 같은 순위 기반 방법론보다 훨씬 적은 감독 데이터로도 작동함을 보여준다.
4. **신뢰도 시각화**: 학습된 $\beta$ 값을 시각화한 결과, 실제로 기대 수익이 높은 시연 데이터에 더 높은 신뢰도 점수가 부여되었음을 확인하였다.

## 🧠 Insights & Discussion

### 강점 및 의의
CAIL의 가장 큰 강점은 **범용성**과 **자동화**이다. 특정 모방 학습 알고리즘에 종속되지 않고 Outer loss만 정의된다면 어떤 모델에도 적용 가능하며, 정답 신뢰도 라벨 없이 소량의 순위 정보만으로도 데이터의 질을 판별할 수 있다. 또한 Bi-level 최적화라는 수학적 기반을 통해 수렴성을 보장했다는 점이 학술적으로 가치가 있다.

### 한계 및 가정
- **데이터의 최소 조건**: 비록 최적 데이터가 없어도 학습은 가능하지만, 데이터셋 내에 정책 학습에 도움이 될 만한 '유용한 지식'은 최소한 포함되어 있어야 한다. 모든 데이터가 완전히 무작위라면 학습이 불가능하다.
- **동역학 가정**: 시연을 수행한 주체(Demonstrator)와 학습하는 에이전트(Imitator)의 동역학(Dynamics)이 동일하다고 가정한다. 실제 환경에서는 두 주체의 물리적 특성이 다를 수 있으며, 이는 향후 해결해야 할 과제로 남아 있다.

### 비판적 해석
본 논문은 Bi-level 최적화의 복잡도를 줄이기 위해 Pseudo-update라는 근사 방식을 사용하였다. 이론적으로는 수렴성이 증명되었으나, 실제 딥러닝 네트워크에서 이러한 근사가 항상 최적의 $\beta$를 찾아내는지에 대한 추가적인 분석이 필요해 보인다. 하지만 실험 결과가 매우 압도적이므로, 실용적인 관점에서는 매우 효율적인 접근법이라고 판단된다.

## 📌 TL;DR

**요약**: 본 논문은 최적성과 질이 제각각인 시연 데이터로부터 고성능 정책을 학습하기 위해, 데이터의 신뢰도 점수를 자동으로 학습하여 재가중치하는 **CAIL** 프레임워크를 제안한다. Bi-level 최적화를 통해 신뢰도와 정책을 동시에 학습하며, 소량의 순위 정보만으로도 최적 데이터를 선별해내는 효과를 거둔다.

**영향력**: 이 연구는 전문가의 완벽한 데이터만을 고집하던 기존 모방 학습의 제약을 깨고, 현실 세계의 '불완전한 데이터'를 효율적으로 활용할 수 있는 길을 열었다. 특히 최적 데이터가 부족하거나 없는 환경에서도 작동한다는 점은 실제 산업 로봇의 데이터 수집 비용을 획기적으로 줄이는 데 기여할 가능성이 높다.
