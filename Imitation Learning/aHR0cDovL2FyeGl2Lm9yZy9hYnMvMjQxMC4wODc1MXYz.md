# Zero-Shot Offline Imitation Learning via Optimal Transport

Thomas Rupf, Marco Bagatella, Nico Gürtler, Jonas Frey, Georg Martius (2025)

## 🧩 Problem to Solve

본 논문은 단 하나의 전문가 시연(demonstration)만으로도 학습 단계에서 보지 못한 새로운 행동을 재현하는 Zero-shot Imitation Learning(IL) 문제를 다룬다. 특히, 전문가의 시연이 정밀한 타임스텝 정보가 없는 '거친(rough)' 상태이거나, 전체 상태의 일부 정보만 포함하는 '부분적인(partial)' 상태인 현실적인 제약 조건을 고려한다.

기존의 실용적인 접근 방식들은 전문가의 시연을 일련의 목표(goals) 시퀀스로 간주하고, 이를 상위 수준의 목표 선택기(goal selector)와 하위 수준의 목표 조건부 정책(goal-conditioned policy)으로 나누어 해결하는 계층적 구조를 사용한다. 그러나 이러한 방식은 '근시안적 행동(myopic behavior)'이라는 치명적인 문제를 야기한다. 즉, 에이전트가 개별 목표를 달성하기 위해 취하는 즉각적인 행동이 장기적인 전체 목표 달성을 저해하거나, 시스템을 회복 불가능한 상태로 몰아넣을 수 있다. 

따라서 본 논문의 목표는 이러한 근시안적 문제를 해결하여, 오프라인의 하위 최적(suboptimal) 데이터만으로도 복잡하고 연속적인 환경에서 비근시안적(non-myopic)인 Zero-shot 모방 학습을 수행하는 방법론을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 모방 학습의 본질적인 목표인 '점유 매칭(occupancy matching)' 목적 함수를 직접 최적화하는 것이다. 이를 위해 **Optimal Transport(OT, 최적 운송)** 이론을 도입하여, 에이전트가 생성하는 상태 점유 분포와 전문가의 목표 점유 분포 사이의 거리를 최소화한다.

단순히 다음 목표를 빠르게 달성하는 것이 아니라, 계획 호라이즌(planning horizon) 내의 전체 경로가 전문가의 궤적 분포와 기하학적으로 일치하도록 유도함으로써, 개별 목표 달성에 매몰되지 않고 전체 시퀀스를 고려하는 전역적인 최적화를 가능하게 한다.

## 📎 Related Works

기존의 Zero-shot IL 연구들은 주로 전문가의 관찰 시퀀스를 따라가는 방식에 의존하며, 이를 위해 목표 조건부 강화학습(GC-RL)을 이용해 목표를 순차적으로 달성하는 계층적 분해 방식을 사용했다. 하지만 본 논문은 이러한 방식이 목표 추상화(goal abstraction)가 존재하는 상황에서 필연적으로 근시안적 최적화 문제를 일으킴을 이론적(Proposition 3.1)으로 증명하며 차별성을 둔다.

또한, Forward-Backward(FB) 프레임워크와 같은 최신 Zero-shot RL 기법들이 존재하지만, 이들은 주로 가산적 보상(additive rewards) 형태의 목적 함수를 다룬다. 반면, 본 연구가 해결하려는 점유 매칭 문제는 전역적인 분포 일치 문제이므로 기존의 보상 기반 접근 방식으로는 한계가 있다.

## 🛠️ Methodology

### 전체 파이프라인 및 구조
ZILOT(Zero-shot Offline Imitation Learning from Optimal Transport)은 다음과 같은 단계로 구성된다.
1. **오프라인 학습**: 하위 최적 데이터셋 $\mathcal{D}_\beta$를 사용하여 월드 모델 $\hat{P}$, 목표 조건부 가치 함수 $V$ 및 $W$를 학습한다.
2. **추론 및 계획**: 테스트 시간에 제공된 단일 전문가 시연 $\hat{\rho}^E$와 에이전트의 예측 점유 분포 $\hat{\rho}^\pi$ 사이의 OT 거리를 최소화하는 행동 시퀀스를 찾는다.
3. **제어**: MPC(Model Predictive Control) 방식으로 최적의 첫 번째 행동을 실행하고 이를 반복한다.

### 주요 구성 요소 및 상세 설명

#### 1. 점유 분포 추정 (Occupancy Estimation)
에이전트와 전문가의 점유 분포를 이산적인 샘플 기반 근사치로 정의한다.
- **전문가 점유**: 제공된 목표 시퀀스 $(g_0, \dots, g_M)$를 직접 사용한다.
  $$\hat{\rho}^E_M = \frac{1}{M+1} \sum_{j=0}^M \delta_{g_j}$$
- **정책 점유**: 학습된 월드 모델 $\hat{P}$를 사용하여 현재 정책 $\pi$ 하에서 미래의 상태 궤적을 롤아웃(rollout)하여 추정한다.
  $$\hat{\rho}^\pi_N = \frac{1}{N+1} \sum_{t=0}^N \delta_{\phi(s_t)}$$

#### 2. 거리 메트릭 근사 (Metric Approximation)
OT 문제의 비용 함수로 사용될 거리 $h$를 직접 정의하기 어렵기 때문에, 학습된 목표 조건부 가치 함수 $V^\star$를 이용하여 거리 $d$를 다음과 같이 정의한다.
$$d(s, g) = -V^\star(s, g)$$
이는 상태 $s$에서 목표 $g$에 도달하기까지 걸리는 기대 시간(first hit time)을 의미하며, 환경의 동역학(dynamics) 특성을 비용에 반영하는 효과를 준다.

####  uma 3. 목적 함수 및 최적화 (Optimization)
최종 목표는 다음과 같이 Wasserstein-1 거리를 최소화하는 정책 $\pi$를 찾는 것이다.
$$\pi^\star = \arg \min_\pi \text{OT}_d (\hat{\rho}^\pi_N, \hat{\rho}^E_M)$$
이 최적화 문제는 엔트로피 정규화가 적용된 Sinkhorn 알고리즘을 통해 효율적으로 계산되며, iCEM 또는 MPPI와 같은 제로 오더 최적화기(zero-order optimizer)를 사용하여 MPC 방식으로 해결한다.

#### 4. 호라이즌 절단 (Truncation)
계산 비용과 월드 모델의 부정확성으로 인해 유한한 호라이즌 $H$를 사용해야 한다. 이때 현재 시점에서 도달 가능한 목표들만 고려하기 위해, 목표 간 이동 시간을 예측하는 함수 $W$를 학습하여 목표 시퀀스를 적절히 절단한다.
$$t_i \approx -V(s_0; g_0) - \sum_{j=1}^i W(g_{j-1}; g_j)$$

## 📊 Results

### 실험 설정
- **환경**: Fetch (Push, Pick&Place, Slide), HalfCheetah, PointMaze 등 5개 환경에서 30개 작업 수행.
- **비교 대상**: 
    - 계층적 분해 방식 (Policy+Cls, MPC+Cls)
    - 보상 기반 Zero-shot IL (ER$_{FB}$, RER$_{FB}$)
- **지표**:
    - $W_{min}$: 전문가 궤적과 에이전트 궤적 사이의 최소 Wasserstein-1 거리 (낮을수록 좋음).
    - GoalFraction: 주어진 순서대로 달성한 목표의 비율 (높을수록 좋음).

### 주요 결과
- **정량적 성과**: ZILOT은 대부분의 환경에서 기존 베이스라인보다 낮은 $W_{min}$과 높은 GoalFraction을 기록하였다.
- **근시안적 행동 해결**: 특히 `Fetch Slide` 환경에서 그 효과가 극명하게 나타났다. 계층적 방식(MPC+Cls)은 첫 번째 목표를 빠르게 달성하려다 물체를 제어 불가능한 영역으로 밀어내어 실패하는 반면, ZILOT은 전체 경로를 고려하여 안정적으로 전문가의 궤적을 모방하였다.
- **일반화 능력**: 학습 데이터셋에 존재하지 않는 새로운 상태 시퀀스라도, 개별 상태들이 데이터 분포 내에 있다면 이를 조합(stitching)하여 성공적으로 재현해내는 능력을 보여주었다.

## 🧠 Insights & Discussion

### 강점 및 해석
ZILOT의 가장 큰 강점은 문제를 '개별 목표 달성'이 아닌 '분포 일치' 관점으로 전환한 것이다. 이를 통해 에이전트는 현재 상태에서 가장 가까운 목표로 직진하는 것이 아니라, 미래의 전체 궤적이 전문가의 점유 분포와 기하학적으로 얼마나 유사한지를 평가하게 된다. 이는 로봇 제어에서 흔히 발생하는 '과도한 가속으로 인한 제어 상실'과 같은 근시안적 최적화 문제를 원천적으로 억제한다.

### 한계 및 비판적 논의
1. **월드 모델 의존성**: 본 방법론은 월드 모델 $\hat{P}$의 정확도에 전적으로 의존한다. 예측 호라이즌이 길어질수록 모델 오차가 누적되므로, 이론적인 전역 최적화와 실제 유한 호라이즌 계획 사이의 괴리가 존재한다.
2. **계산 복잡도**: Sinkhorn 알고리즘과 비용 행렬 $\text{C}$를 매 스텝 계산해야 하므로, 단순한 정책 실행보다 추론 속도가 느리다 (2~4Hz). 실시간 제어를 위해서는 Anderson acceleration 등의 최적화가 필요해 보인다.
3. **비마르코프 정책**: ZILOT은 상태 이력과 목표 시퀀스를 동시에 고려하는 비마르코프(non-Markovian) 정책을 유도한다. 비록 성능은 우수하나, 이론적으로는 마르코프 정책으로도 충분한 상황에서 불필요하게 복잡한 탐색 공간을 사용할 가능성이 있다.

## 📌 TL;DR

본 논문은 Zero-shot 모방 학습에서 발생하는 근시안적 행동 문제를 해결하기 위해, **Optimal Transport(OT)**를 이용한 점유 매칭(Occupancy Matching) 기반의 새로운 방법론 **ZILOT**을 제안한다. 월드 모델과 가치 함수를 통해 전문가의 궤적 분포를 근사하고, 이를 MPC 프레임워크 내에서 직접 최적화함으로써 복잡한 연속 제어 환경에서 매우 정밀한 모방 성능을 달성하였다. 이 연구는 향후 전문가의 단편적인 시연만으로도 복잡한 로봇 작업을 수행해야 하는 Zero-shot 제어 분야에 중요한 기여를 할 것으로 평가된다.
