# DITTO: Offline Imitation Learning with World Models

Branton DeMoss, Paul Duckworth, Jakob Foerster, Nick Hawes, Ingmar Posner (2025)

## 🧩 Problem to Solve

본 논문은 모방 학습(Imitation Learning, IL)이 실제 환경의 복잡한 문제로 확장되기 위해 해결해야 할 세 가지 핵심 난제인 고차원 관측치(High-dimensional observations), 오프라인 학습(Offline learning), 그리고 정책으로 인한 공변량 변화(Policy-induced covariate shift)를 해결하고자 한다.

가장 단순한 형태의 모방 학습인 행동 복제(Behavior Cloning, BC)는 전문가의 상태-행동 쌍을 직접 학습하지만, 테스트 단계에서 에이전트가 작은 실수를 하여 학습 데이터 분포에서 벗어나게 되면 오류가 누적되는 compounding error 문제가 발생한다. 이는 훈련 시의 상태 분포와 테스트 시의 상태 분포가 일치하지 않는 covariate shift 현상 때문이다. 이를 해결하기 위해 전문가와의 상호작용을 통한 온라인 학습 방법(예: DAgger)이 제시되었으나, 로보틱스와 같은 실제 환경에서는 안전성 문제나 시간 및 비용 문제로 인해 온라인 학습이 불가능한 경우가 많다.

따라서 본 논문의 목표는 환경과의 직접적인 상호작용 없이, 오직 오프라인 데이터만을 사용하여 고차원 픽셀 입력으로부터 전문가의 성능을 복구할 수 있는 강건한 모방 학습 알고리즘을 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **학습된 월드 모델(World Model, WM)의 잠재 공간(Latent Space) 내에서 오프라인 모방 학습을 온라인 강화 학습 문제로 변환**하는 것이다.

주요 기여 사항은 다음과 같다.
1. **DITTO 프레임워크 제안**: 전문가 데이터를 통해 월드 모델을 먼저 학습시킨 뒤, 에이전트가 이 모델의 잠재 공간 내에서 '상상'을 통해 전문가의 궤적을 따라가도록 학습시킨다. 이를 통해 실제 환경 없이도 온-폴리시(on-policy) 학습 효과를 얻어 covariate shift 문제를 완화한다.
2. **단순하고 강건한 거리 기반 보상 설계**: 기존의 GAIL과 같은 적대적 모방 학습(Adversarial IL)에서 사용하는 복잡한 판별자(Discriminator) 대신, 월드 모델의 잠재 공간에서 전문가 궤적과 에이전트 궤적 사이의 단순한 거리 함수를 보상으로 사용하여 학습의 안정성을 높였다.
3. **SOTA 성능 입증**: 픽셀 기반의 Atari 벤치마크 환경에서 완전히 오프라인 상태로 학습하여 전문가의 성능을 일관되게 복구한 최초의 알고리즘임을 보여주었다.
4. **기존 알고리즘의 확장**: BC와 GAIL을 월드 모델 환경에 적응시킨 D-BC와 D-GAIL을 제안하여 비교 분석함으로써 월드 모델이 표현 학습(Representation learning)에 주는 이점을 분석하였다.

## 📎 Related Works

### 모방 학습 (Imitation Learning)
기존의 BC는 앞서 언급한 covariate shift 문제로 인해 한계가 있으며, 이를 해결하기 위해 전문가에게 쿼리를 보내는 온라인 인터랙티브 방식이 제안되었다. 또한, 전문가의 보상 함수를 추정하는 역강화학습(Inverse RL)이나 상태 분포의 다이버전스를 최소화하는 GAIL 등이 제안되었다. 특히 GAIL은 판별자를 통해 전문가와 학습자의 분포 차이를 줄이지만, 고차원 픽셀 환경에서는 판별자의 오버피팅이나 적대적 학습의 불안정성으로 인해 성능이 저하되는 문제가 보고되었다.

### 월드 모델 (World Models)
월드 모델은 환경의 다이내믹스를 학습하여 에이전트가 모델 내부에서 학습할 수 있게 한다. Ha & Schmidhuber(2018)의 초기 모델부터 Dreamer(Hafner et al., 2021)와 같은 최신 모델까지, 잠재 공간 내에서의 상상(Imagination)을 통한 학습은 데이터 효율성을 극대화하고 실제 환경으로의 전이를 돕는 것으로 알려져 있다.

## 🛠️ Methodology

### 전체 파이프라인
DITTO의 학습 과정은 크게 두 단계로 나뉜다.
1. **월드 모델 학습**: 가용한 모든 궤적 데이터를 사용하여 환경의 다이내믹스를 학습한다.
2. **에이전트 학습**: 학습된 월드 모델의 잠재 공간 내에서 전문가의 궤적을 모방하도록 Actor-Critic 에이전트를 학습시킨다.

### 월드 모델 아키텍처
본 연구는 Dreamer v2의 아키텍처를 기반으로 하며, 다음과 같은 구성 요소로 이루어진 Recurrent State-Space Model(RSSM)을 사용한다.
- **Image Encoder**: 픽셀 관측치 $x_t$를 저차원 표현으로 변환한다.
- **RSSM**: 결정론적 상태 $h_t$와 확률적 상태 $z_t$를 결합하여 모델 상태 $\hat{s}_t = (h_t, z_t)$를 생성한다.
- **Decoder**: 잠재 상태 $\hat{s}_t$로부터 원래의 픽셀 $x_t$를 재구성한다.

월드 모델은 다음의 ELBO(Evidence Lower Bound) 손실 함수를 통해 학습된다.
$$\min_{\phi} \mathbb{E}_{q_\phi} \left[ \sum_{t=1}^T -\log p_\phi(x_t | \hat{s}_t) + \beta D_{KL-B}(q_\phi(z_t | \hat{s}_t) \| p_\phi(\hat{z}_t | h_t)) \right]$$
여기서 $D_{KL-B}$는 Prior와 Posterior 사이의 균형을 맞추는 KL Balancing 기법을 사용하여 학습의 안정성을 꾀한다.

### 내재적 보상 함수 (Intrinsic Reward)
DITTO의 핵심은 에이전트가 월드 모델 내에서 전문가의 잠재 궤적을 얼마나 잘 따라가는지를 측정하여 보상으로 주는 것이다. 구체적으로, 전문가의 잠재 상태 $z^E_t$와 에이전트의 잠재 상태 $z^\pi_t$ 사이의 내적 기반 거리 함수를 정의한다.
$$r^{int}_t(z^E_t, z^\pi_t) = 1 - d(z^E_t, z^\pi_t) = \frac{z^E_t \cdot z^\pi_t}{\max(\|z^E_t\|, \|z^\pi_t\|)^2}$$
이 보상 함수는 에이전트가 전문가가 방문한 상태와 유사한 잠재 공간 영역에 머물도록 유도하며, 복잡한 판별자 없이도 밀집된(dense) 보상 신호를 제공한다.

### 에이전트 학습 절차
에이전트는 Actor-Critic 구조를 가지며, 월드 모델 내에서 다음과 같이 학습한다.
1. 전문가 데이터에서 임의의 시작 상태 $\hat{s}^E_t$를 샘플링한다.
2. 현재 정책 $\pi_\theta$를 사용하여 모델 내부에서 $H$ 스텝 동안 궤적을 생성(roll-out)한다.
3. 위에서 정의한 내재적 보상 $r^{int}$를 계산한다.
4. $\lambda$-return을 사용하여 Critic을 업데이트하고, REINFORCE 알고리즘과 엔트로피 정규화 항을 사용하여 Actor를 업데이트한다.
   - **Actor 손실 함수**: 
     $$L(\theta) = \mathbb{E}_{\pi_\theta, p_\phi} \left[ \sum_{t=1}^{H-1} -\log \pi_\theta(a_t | \hat{s}_t) \text{sg}(V^\lambda_t - v_\psi(\hat{s}_t)) - \eta H(\pi_\theta(\hat{s}_t)) \right]$$

## 📊 Results

### 실험 설정
- **데이터셋**: 5개의 Atari 환경(픽셀 입력) 및 BipedalWalker(연속 제어 환경).
- **전문가 데이터**: RL Baselines Zoo의 PPO 에이전트가 생성한 궤적을 사용하며, 데이터 양을 4~1000 에피소드로 다양하게 설정하였다.
- **비교 대상**: 
    - **BC**: 픽셀 기반의 표준 행동 복제.
    - **D-BC**: 월드 모델의 잠재 공간에서 수행하는 BC.
    - **D-GAIL**: 월드 모델의 잠재 공간에서 수행하는 GAIL (적대적 보상 학습).

### 주요 결과
1. **성능 우위**: DITTO는 모든 Atari 환경에서 BC, D-BC, D-GAIL보다 뛰어난 성능과 샘플 효율성을 보였다. 특히, 완전히 오프라인 설정에서 픽셀 입력만으로 전문가의 성능을 일관되게 복구한 최초의 사례임을 입증하였다.
2. **적대적 학습의 불안정성**: D-GAIL은 표준 GAIL보다는 안정적이었으나, 일부 환경(MsPacman)에서 여전히 적대적 붕괴(Adversarial collapse) 현상이 관찰되었다. 반면 DITTO는 고정된 거리 함수를 사용하므로 매우 안정적이었다.
3. **지표의 유효성 분석**: 실험 결과, 전문가의 행동 예측 정확도(Action prediction accuracy)는 실제 환경의 리턴(Extrinsic reward)과 상관관계가 낮거나 심지어 음의 상관관계를 보였다. 반면, DITTO가 최적화하는 **잠재 거리(Latent distance)**는 실제 성능과 매우 높은 상관관계를 보였다. 이는 단순한 행동 복제보다 궤적 수준의 분포 일치가 훨씬 중요하다는 것을 시사한다.

## 🧠 Insights & Discussion

### 강점 및 통찰
- **분포 일치의 중요성**: 단순히 다음 행동을 맞추는 것이 아니라, 월드 모델의 잠재 공간에서 궤적을 일치시키는 것이 covariate shift 문제를 해결하는 핵심임을 보여주었다.
- **단순함의 미학**: 복잡한 적대적 신경망(Discriminator)을 설계하는 대신, 잘 학습된 월드 모델의 잠재 공간에서 단순한 거리 측정만으로도 충분히 강력한 모방 학습이 가능함을 증명하였다. 이는 모델 기반 학습이 표현 학습(Representation learning)의 이점을 제공하기 때문이다.

### 한계 및 논의
- **월드 모델 의존성**: 에이전트의 성능이 월드 모델의 정확도에 의존할 가능성이 있다. 다만, 저자들은 월드 모델 학습 시 데이터의 품질보다 '상태 공간의 커버리지'가 더 중요하다는 점을 발견하였다.
- **계산 비용**: 월드 모델을 먼저 학습시키고 그 내부에서 수많은 시뮬레이션을 수행해야 하므로, 단순 BC에 비해 학습 전반의 계산 비용이 증가한다.

## 📌 TL;DR

DITTO는 고차원 픽셀 환경의 오프라인 모방 학습에서 발생하는 **공변량 변화(Covariate Shift)** 문제를 해결하기 위해, **월드 모델의 잠재 공간 내에서 전문가 궤적과의 거리를 최소화하는 방식의 온라인 강화 학습**으로 문제를 재정의한 알고리즘이다. 복잡한 판별자 없이 단순한 거리 기반 보상을 사용하여 학습의 안정성을 확보하였으며, Atari 벤치마크에서 오프라인 픽셀 학습만으로 전문가 성능을 복구하는 SOTA 성능을 달성하였다. 이 연구는 월드 모델이 단순한 다이내믹스 예측을 넘어, 효율적인 모방 학습을 위한 최적의 표현 공간을 제공할 수 있음을 시사한다.
