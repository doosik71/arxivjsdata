# Imitation Learning Approach for AI Driving Olympics Trained on Real-world and Simulation Data Simultaneously

Mikita Sazanovich, Konstantin Chaika, Kirill Krinkin, Aleksei Shpilman (2020)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 자율주행 시스템에서 발생하는 시뮬레이션과 실제 환경 간의 간극, 즉 **Sim-to-Real transfer** 문제이다. AI Driving Olympics(AIDO)의 Lane Following Challenge와 같은 경쟁 환경에서는 시뮬레이션에서 최적의 성능을 낸 알고리즘이 실제 환경에서는 제대로 작동하지 않거나, 반대로 실제 환경에 맞춰 튜닝된 제어 알고리즘이 시뮬레이션에서는 낮은 성능을 보이는 현상이 빈번하게 발생한다.

특히 기존의 고전적 제어 알고리즘(Classic Control)은 조명, 도로 유형, 카메라 위치 등 특정 주행 조건에 맞춰 세밀하게 튜닝되어야 하므로, 환경이 조금만 바뀌어도 범용적으로 적용하기 어렵다는 한계가 있다. 따라서 본 연구의 목표는 시뮬레이션과 실제 환경 모두에서 일관되게 높은 성능을 발휘할 수 있는 강건한(robust) 주행 모델을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 **시뮬레이션 데이터와 실제 환경 데이터를 동시에 사용하여 모델을 학습시키는 하이브리드 데이터셋 기반의 Imitation Learning(모방 학습)**이다. 

단일 환경의 데이터에 최적화되는 대신, 서로 다른 특성을 가진 네 가지 데이터 소스를 혼합하여 학습함으로써 모델이 특정 도메인의 특성에 과적합(overfitting)되지 않고, 다양한 환경에서도 공통적으로 적용될 수 있는 핵심 특징(relevant features)을 추출하도록 강제하는 전략을 취했다. 이를 통해 별도의 복잡한 도메인 적응(Domain Adaptation) 기법 없이도 시뮬레이션과 실제 환경 모두에서 우수한 성능을 확보하였다.

## 📎 Related Works

논문에서는 Duckietown 플랫폼을 기반으로 한 자율주행 연구와 Sim-to-Real 전이 문제를 다룬 기존 연구들을 언급한다. 일반적으로 시뮬레이션에서 학습된 신경망이 실제 환경에서 실패하는 이유는 두 환경 간의 시각적, 물리적 차이(Domain Gap) 때문이다. 

이를 해결하기 위해 기존 연구들에서는 **Domain Randomization**(시뮬레이션의 환경 변수를 무작위로 변경하여 모델의 강건성을 높이는 방법) 등이 제안되었다. 하지만 본 논문은 특정 환경에 튜닝된 여러 전문가(인간 운전자 및 튜닝된 PD 제어기)의 데이터를 직접적으로 통합 학습시키는 방식을 통해, 전이 학습의 어려움을 해결하고 범용적인 제어 정책을 학습시키고자 하였다.

## 🛠️ Methodology

### 1. 데이터셋 생성 및 구성
모델은 이미지 입력을 받아 두 바퀴의 전압 값 $\langle V_{left}, V_{right} \rangle$ (범위 $[-1, 1]$)을 예측하도록 설계되었다. 학습을 위해 다음과 같은 네 가지 소스에서 데이터를 수집하였다.

- **SIM-LP**: 단순한 루프 형태의 시뮬레이션 맵에서 튜닝된 PD 제어기가 주행하며 수집한 데이터이다.
- **SIM-IS**: 교차로가 포함된 복잡한 시뮬레이션 맵에서 수집한 데이터이다.
- **REAL-DT**: Duckietown 주최 측에서 제공한 공개 데이터셋으로, 다양한 환경 조건이 포함되어 있다.
- **REAL-IH**: 연구진이 직접 구축한 환경에서 수집한 데이터로, 도로의 굴곡, 회전 방향, 조명 조건 등을 다양화하여 수집하였다.

특히, 실제 환경 데이터 수집 시 완벽한 궤적뿐만 아니라 차선을 약간 벗어난 **불완전한 궤적(imperfect trajectories)**을 의도적으로 포함하였다. 이는 모델이 예상치 못한 상황에서 차선으로 복귀하는 방법을 학습하게 하여 시스템의 강건성을 높이기 위함이다.

### 2. 전처리 및 네트워크 아키텍처
입력 영상은 $640 \times 480$ RGB 이미지이며, 다음과 같은 전처리를 거친다.
1. 배경이 많이 포함되어 제어 결정에 영향이 적은 이미지의 상단 1/3을 제거한다.
2. 이미지 크기를 $64 \times 32$ 픽셀로 축소한다.
3. RGB 색 공간을 **YUV 색 공간**으로 변환하여 조명 변화에 대한 영향을 줄인다.

네트워크 구조는 Bojarski et al. (2016)의 CNN 구조를 참고하여 설계되었으며, CPU 단일 환경에서 구동되어야 한다는 제약 조건을 충족하기 위해 경량 모델로 구성하였다.
- **Convolutional Layers**: 5개의 합성곱 층으로 구성되며, 필터 수를 적게 유지하여 과적합을 방지하였다.
- **Fully-Connected Layers**: 2개의 전결합 층이 뒤따르며 최종적으로 $V_{left}, V_{right}$ 두 값을 출력한다.
- **IC (Independent-Component) Layers**: BatchNorm과 Dropout을 결합한 형태의 IC 레이어를 도입하여 각 층의 활성화 값이 서로 독립적이 되도록 유도하였다. 
- **Spatial Dropout**: 합성곱 층에서는 일반 Dropout 대신 맵 전체를 0으로 만드는 Spatial Dropout을 사용하였다. (첫 번째 층 확률 0.01, 이후 층 0.05)

### 3. 학습 절차
학습 목표는 전문가의 동작을 모방하는 것이므로, 예측값과 실제 전문가 값 사이의 **Mean Square Error (MSE)**를 손실 함수로 사용하였다.

$$ \text{MSE} = \frac{1}{N} \sum_{i=1}^{N} (y_{true} - y_{pred})^2 $$

네 가지 데이터 소스를 동일한 비율로 혼합하여 총 20,294장의 이미지로 학습을 진행하였으며, 검증 세트(8,698장)에서 평균 MSE가 가장 낮은 모델을 최종 선택하였다.

## 📊 Results

### 1. 정량적 평가 (MSE)
각 데이터 소스별 검증 세트에 대해 MSE를 측정한 결과, 특정 데이터셋만으로 학습한 모델보다 **HYBRID** 방식이 모든 데이터셋에 걸쳐 평균적으로 가장 낮은 MSE를 기록하였다. 이는 하이브리드 접근 방식이 특정 도메인에 치우치지 않고 일반화된 성능을 보임을 입증한다.

### 2. Closed-loop 제어 평가
실제 주행 성능을 측정하기 위해 차량이 차선을 이탈하지 않고 주행한 거리(Tiles, 1타일 $\approx 60\text{cm}$)를 측정하였다.

- **시뮬레이션 환경**: SIM 전용 모델이 약간 더 높은 성능을 보였으나, HYBRID 모델 역시 매우 근소한 차이로 우수한 성능을 보였다.
- **실제 환경 (Robotarium)**: SIM 전용 모델은 거의 작동하지 않은 반면, HYBRID 모델은 REAL 전용 모델보다 더 많은 타일을 주행하며 우위를 점했다.
- **최종 테스트 (Unknown Environment)**: 완전히 새로운 환경에서 진행된 최종 평가에서, **HYBRID 모델만이 유일하게 모든 시나리오를 이탈 없이 완주**하였으며, 속도 면에서도 Classic Control(CC)이나 REAL 전용 모델보다 빨랐다. 결과적으로 이 방법론을 통해 대회 1위를 차지하였다.

## 🧠 Insights & Discussion

본 연구의 핵심 통찰은 **데이터의 다양성이 곧 모델의 강건성으로 이어진다**는 점이다. 시뮬레이션 데이터는 기본적인 차선 추종 능력을 제공하고, 실제 데이터는 현실의 노이즈와 시각적 특성을 보완한다. 특히 불완전한 주행 데이터를 학습에 포함시킨 것이 실제 환경에서 모델이 복구 능력을 갖추게 하는 데 결정적인 역할을 했다고 분석된다.

또한, 저자들은 MSE라는 오프라인 지표가 실제 주행 성능(Online performance)과 완전히 일치하지 않는다는 점을 지적하였다. 이는 자율주행 모델 평가에서 단순한 예측 오차보다는 closed-loop 시뮬레이션 및 실차 테스트가 훨씬 중요하다는 점을 시사한다. 

한계점으로는 현재 모델이 단일 프레임만을 입력으로 사용한다는 점이 있다. 시퀀스 데이터를 사용하려는 시도가 있었으나 성능 향상이 없었다고 명시되어 있는데, 이는 향후 더 복잡한 동적 환경에서는 여전히 해결해야 할 과제로 남는다.

## 📌 TL;DR

본 논문은 시뮬레이션과 실제 환경의 데이터를 혼합하여 학습시킨 Imitation Learning 기반의 자율주행 모델을 제안하였다. 경량 CNN 구조와 YUV 색 공간 변환, 그리고 의도적으로 포함된 불완전한 궤적 데이터를 통해 Sim-to-Real 전이 문제를 해결하였으며, 그 결과 AI Driving Olympics Lane Following Challenge에서 1위를 달성하였다. 이 연구는 다양한 환경에 특화된 전문가 데이터를 통합 학습시키는 것만으로도 강력한 일반화 성능을 가진 단일 모델을 구축할 수 있음을 보여주었다.
