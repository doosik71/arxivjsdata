# Injective State-Image Mapping facilitates Visual Adversarial Imitation Learning

Subhajit Chaudhury, Daiki Kimura, Asim Munawar, Ryuki Tachibana (2019)

## 🧩 Problem to Solve

본 논문은 가상 자율 에이전트의 제어 정책(control policy)을 학습시키기 위해 전문가의 로우 비디오(raw video) 데모를 직접 모방하는 방법을 다룬다. 기존의 모방 학습(Imitation Learning) 방식은 주로 전문가의 관절 각도나 속도와 같은 저수준 상태(low-level state) 정보를 필요로 하며, 이는 실제 환경에서 획득하기 어렵다는 한계가 있다.

비디오를 이용한 기존의 시각적 모방 학습 방법들은 이미지 프레임에서 고수준 특징(high-level feature)을 추출한 후, 수동으로 설계된(hand-crafted) 보상 함수를 통해 에이전트와 전문가 사이의 거리를 측정하는 방식을 사용했다. 이러한 방식은 보상 함수 설계에 대한 의존도가 높고, 에이전트의 궤적을 전문가의 비디오 시간축에 맞춰 동기화(time-synchronization)해야 한다는 제약 조건이 존재한다. 따라서 본 연구의 목표는 수동 설계된 보상 함수와 시간 동기화의 필요성을 제거하고, 로우 비디오로부터 직접 효과적인 제어 정책을 학습하는 프레임워크를 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 저수준 상태 궤적(low-level state trajectories)과 그에 대응하는 로우 비디오 스트림 사이에 **단사 매핑(injective mapping)**이 존재한다면, 비디오 데모를 통한 적대적 모방 학습(adversarial imitation learning)이 상태 궤적을 통한 학습과 이론적으로 동일하다는 것을 증명한 점이다.

이를 바탕으로 저자들은 특징 추출과 보상 추정 단계를 통합하여 공동으로 학습하는 **VIGAN(Video Imitation Generative Adversarial Network)** 프레임워크를 제안하였다. 이 방식은 생성적 적대 신경망(GAN) 구조를 활용하여 판별자(discriminator)가 에이전트와 전문가의 비디오 궤적을 구분하게 함으로써, 별도의 보상 설계 없이도 데이터로부터 직접 보상 신호를 추출할 수 있게 한다.

## 📎 Related Works

기존의 모방 학습 연구들은 주로 전문가의 상태-행동(state-action) 쌍이 주어진 상황에서 최대 가능도 추정(maximum likelihood estimation)을 통해 행동을 복제하는 방식에 집중하였다. 행동 정보가 없는 상태에서의 학습 시도들도 있었으나, 주로 저수준 모션 캡처 데이터에 의존하였다.

비디오 기반의 기존 접근 방식들은 이미지 프레임 간의 시간적 거리를 기반으로 임베딩 공간에서 특징을 추출하고, 이들 간의 유클리드 거리 등을 이용해 보상을 계산하는 방식을 취했다. 그러나 이러한 방법들은 다음과 같은 한계가 있다.

1. **수동 보상 설계:** 도메인 지식에 기반한 보상 함수 설계가 필요하며, 이는 최적화가 어렵다.
2. **시간 동기화 제약:** 에이전트의 상태를 전문가의 특정 시점과 일치시켜야만 보상 계산이 가능하다.

본 논문은 GAN 기반의 적대적 학습을 통해 보상 함수를 자동으로 학습함으로써 위 두 가지 문제를 동시에 해결하고자 한다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

VIGAN은 정책 네트워크(Policy Network)를 생성자(Generator)로, CNN 기반의 판별자(Discriminator)를 비평가로 사용하는 GAN 구조를 가진다. 정책 네트워크는 현재 상태 $s$를 입력받아 행동 $a$를 결정하고, 이에 따른 에이전트의 상태 변화는 렌더링 함수 $g(s)$를 통해 비디오 프레임 $I$로 변환된다. 판별자는 에이전트가 생성한 비디오 프레임 전이(transition)와 전문가의 비디오 프레임 전이를 구분하여 이진 분류를 수행한다.

### 2. 이론적 배경 및 방정식

본 논문은 저수준 상태 $s$와 이미지 $I$ 사이의 매핑 $g: S \rightarrow I$가 단사(injective)일 때, 이미지 공간에서의 분포 매칭이 상태 공간에서의 점유 측정치(occupancy measure) 매칭과 동일함을 증명한다.

판별자 $D(I, I')$는 이미지 전이가 전문가에 의해 생성되었을 확률 $p(E|I, I')$을 나타내며, 이는 다음과 같은 가능도 비율(likelihood ratio) $\beta(I, I')$로 표현될 수 있다.
$$D(I, I') = \frac{1}{1 + \beta(I, I')}, \quad \text{where } \beta(I, I') = \frac{p(I, I'|A)}{p(I, I'|E)}$$
여기서 $A$는 에이전트, $E$는 전문가를 의미한다. 매핑 $g$가 단사라면, 야코비안(Jacobian) 행렬 $J = \frac{dI}{ds}$를 이용하여 이미지 공간의 확률 밀도를 상태 공간으로 변환할 수 있으며, 이때 야코비안 항이 분자와 분모에서 서로 상쇄된다.
$$\beta(I, I') = \frac{p(s, s'|A) | \det J |^{-1}}{p(s, s'|E) | \det J |^{-1}} = \frac{p(s, s'|A)}{p(s, s'|E)}$$
결과적으로 $D(I, I') = D(s, s')$가 성립하며, 이는 비디오 공간에서의 적대적 학습이 저수준 상태 공간에서의 학습과 동일한 효과를 가짐을 의미한다.

### 3. 학습 절차 및 보상 함수

VIGAN의 학습 알고리즘은 다음과 같은 절차를 따른다.

1. **데이터 수집:** 현재 정책 $\pi_\theta$를 통해 상태 전이 $\{s, a, s'\}$를 생성하고, 이를 렌더링하여 이미지 전이 $\{I, I'\}$를 획득한다.
2. **판별자 학습:** 전문가 비디오 $\tau^E$와 에이전트 비디오 $\tau^I$를 사용하여 판별자 $D_\phi$를 최적화한다.
   $$\text{Loss} = -\mathbb{E}_{\tau^E}[\log D_\phi(I, I')] - \mathbb{E}_{\tau^I}[\log(1 - D_\phi(I, I'))]$$
3. **정책 최적화:** 판별자의 출력값을 기반으로 보상 $r_t$를 추정하고, 이를 TRPO(Trust Region Policy Optimization) 알고리즘에 입력하여 정책을 업데이트한다.
   $$r_t = -\log(1 - D_\phi(I_t, I_{t+1}))$$

## 📊 Results

### 1. 실험 설정

- **데이터셋 및 환경:** OpenAI Gym의 CartPole, Pendulum, Hopper, Walker2d, BipedalWalker 등 물리 기반 환경을 사용하였다.
- **비교 대상:**
  - **GAIL:** 저수준 상태-행동 쌍을 사용하는 SOTA 모방 학습 방법.
  - **DeepMimic + PixelLoss:** 이미지 간 유클리드 거리를 보상으로 사용.
  - **DeepMimic + TCN:** Time Contrastive Networks를 통한 임베딩 거리 기반 보상 사용.
- **지표:** 최종 학습된 정책의 누적 보상(Reward).

### 2. 정량적 결과

실험 결과, VIGAN은 기존의 비디오 기반 모방 학습 방법(PixelLoss, TCN)보다 일관되게 높은 성능을 보였다. 특히 Hopper와 Walker2d 환경에서 기존 방법들이 상태의 고수준 의미 정보를 제대로 포착하지 못해 낮은 성능을 보인 반면, VIGAN은 상태-행동 정보를 직접 사용한 GAIL과 유사한 수준의 성능을 달성하였다. 이는 비디오만으로도 저수준 상태 기반 학습에 근접한 성능을 낼 수 있음을 시사한다.

### 3. 강건성 및 실용성 테스트

- **노이즈 내성:** 카메라 흔들림을 모사하여 프레임을 무작위로 크롭(0~5%)한 노이즈 비디오를 사용했음에도 불구하고, GAIL 및 비노이즈 상황과 유사한 성능을 유지하여 강건함을 입증하였다.
- **YouTube 비디오 학습:** BipedalWalker 환경의 YouTube 영상을 사용하여 학습시킨 결과, 실제 환경의 보상 신호(true reward)를 사용하여 TRPO로 학습시킨 에이전트와 대등한 수준의 성능을 보였다. 특히 시간 동기화 단계 없이도 성공적으로 학습되었다는 점이 주목할 만하다.

## 🧠 Insights & Discussion

### 강점

본 논문은 단순한 알고리즘 제안을 넘어, 단사 매핑(injective mapping)이라는 조건을 통해 시각적 모방 학습과 상태 기반 모방 학습 사이의 이론적 동등성을 수학적으로 증명하였다. 이를 통해 딥러닝 모델이 이미지에서 유의미한 특징을 추출하고 보상을 정의하는 과정을 적대적 학습 프레임워크 내에서 자동화할 수 있음을 보였다.

### 한계 및 가정

논문에서 제시한 단사 매핑 가정은 2D 환경에서는 합리적이지만, 3D 환경에서는 다음과 같은 사례에서 깨질 수 있다.

1. **투영 문제:** 카메라 축을 따라 이동하여 이미지상의 크기만 변하고 투영된 위치가 변하지 않는 경우.
2. **폐쇄(Occlusion):** 에이전트가 다른 물체에 가려져 상태가 변해도 이미지에 변화가 없는 경우.
이러한 경우 야코비안 $J$가 특이 행렬(singular matrix)이 되어 이론적 전제가 성립하지 않게 된다. 다만, 실제 실험에서는 이러한 퇴화 사례(degenerate cases)가 드물 경우 정책 회복이 가능함을 확인하였다.

### 비판적 해석

본 연구는 렌더링 함수 $g(s)$가 고정된 시뮬레이션 환경을 전제로 한다. 실제 현실 세계의 비디오에서는 배경의 변화, 조명 조건, 다양한 각도의 카메라 등이 존재하므로, 단순한 단사 매핑 가정을 적용하기 어렵다. 향후 연구에서 언급한 것처럼, 배경 맥락이 변하는 복잡한 환경에서는 시맨틱 비디오 클러스터링이나 계층적 보상 추정 방식이 추가적으로 필요할 것으로 판단된다.

## 📌 TL;DR

본 논문은 **저수준 상태와 이미지 간의 단사 매핑이 존재할 때, 비디오를 통한 적대적 모방 학습이 상태 궤적 학습과 동일함**을 이론적으로 증명하고, 이를 구현한 **VIGAN** 프레임워크를 제안하였다. VIGAN은 수동적인 보상 설계나 시간 동기화 없이 로우 비디오만으로 SOTA 수준의 제어 정책을 학습하며, 노이즈가 섞인 영상이나 YouTube 영상에서도 강건한 성능을 보였다. 이 연구는 향후 실제 영상 데이터를 이용한 자율 에이전트 학습의 이론적 기반을 제공한다는 점에서 중요한 의의가 있다.
