# Stealthy Imitation: Reward-guided Environment-free Policy Stealing

Zhixiong Zhuang, Maria-Irina Nicolae, Mario Fritz (2024)

## 🧩 Problem to Solve

현대 제어 시스템의 핵심인 심층 강화학습(Deep Reinforcement Learning, DeepRL) 정책은 도메인 전문 지식, 고정밀 시뮬레이션, 실세계 검증 등 막대한 자원이 투입되어 개발된 가치 있는 지적 재산이다. 이러한 정책들은 블랙박스 쿼리 접근만으로 기능을 복제하려는 모델 스틸링(Model Stealing) 공격에 취약할 수 있다.

기존의 데이터 프리(Data-free) 모델 스틸링 기법들은 이미지 도메인에서는 효과적이었으나, 제어 시스템에 적용하기에는 결정적인 한계가 존재한다. 대부분의 기존 방식은 공격자가 입력 값의 유효 범위(Valid Input Range)를 알고 있다는 가정하에 작동한다. 하지만 제어 시스템에서는 센서의 종류나 전처리 방식에 따라 상태(State)의 의미와 스케일이 제각각이므로, 입력 범위를 사전에 알기 매우 어렵다. 또한, 공격자가 환경(Environment)에 직접 접근하여 상호작용하는 것이 불가능하거나 위험한 상황이 많다.

따라서 본 논문은 **환경에 대한 접근 권한이 없고 입력 범위에 대한 사전 지식이 없는 상황**에서도 타겟 정책을 성공적으로 탈취하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 타겟 정책 $\pi_v$가 학습된 데이터 분포 $S_v$ 내에서는 복잡하게 동작하지만, 학습되지 않은 분포 외 영역에서는 단순하게 동작한다는 직관에서 출발한다. 즉, 공격자 모델 $\pi_a$가 타겟 모델을 모방하기 가장 어려운 지점이 바로 타겟 모델의 실제 상태 분포 $S_v$와 일치하는 지점일 것이라는 가설을 세운다.

이를 위해 본 연구는 다음과 같은 핵심 기여를 한다.
1. **환경 프리 정책 스틸링(Environment-free Policy Stealing)**: 환경 접근이나 입력 범위 지식 없이도 정책을 탈취할 수 있는 최초의 공격 방법인 Stealthy Imitation(SI)을 제안한다.
2. **보상 모델 기반 분포 최적화**: Reward Model $\hat{R}$을 통해 모방의 난이도를 측정하고, 이를 가이드로 삼아 공격자의 쿼리 분포 $S_a$를 타겟 분포 $S_v$에 가깝게 정밀화(Refinement)한다.
3. **분포 평가 지표 제안**: 추정된 분포 $S_a$와 실제 분포 $S_v$ 사이의 유사성을 측정할 수 있는 프록시 메트릭(Proxy Metric)을 도입하고, 이것이 KL-divergence와 높은 상관관계가 있음을 통계적으로 검증한다.
4. **방어 기법 제안**: 유효 범위를 벗어난 쿼리에 대해 무작위 액션을 응답함으로써 공격자의 분포 추정을 방해하는 실질적인 방어책을 제시한다.

## 📎 Related Works

본 논문은 다음과 같은 세 가지 관련 분야를 다룬다.

1. **지식 증류(Knowledge Distillation, KD)**: 교사 모델의 지식을 학생 모델에게 전달하는 기법이다. 일반적으로는 원본 학습 데이터셋이 필요하며, 데이터 프리 KD의 경우 모델의 화이트박스 접근(그레이디언트 계산 등)을 가정하는 경우가 많아 블랙박스 환경인 본 논문의 설정과는 차이가 있다.
2. **모델 스틸링(Model Stealing)**: 블랙박스 모델의 기능을 복제하는 적대적 기법이다. 기존의 데이터 프리 스틸링(DFME 등)은 입력 범위를 안다고 가정하거나, 개별 샘플의 난이도를 높이는 데 집중한다. 반면 본 논문은 '영역(Region)' 단위의 분포 추정에 집중하여 입력 범위 자체를 찾아낸다는 점에서 차별화된다.
3. **모방 학습(Imitation Learning)**: 전문가의 행동을 복제하는 학습법이다. Behavioral Cloning(BC)은 지도 학습 방식으로 동작하며, GAIL은 적대적 학습을 통해 분포를 맞춘다. 그러나 GAIL은 전문가와 환경의 상호작용 데이터가 필요하지만, 본 논문은 환경 접근 없이 쿼리만으로 이를 수행한다.

## 🛠️ Methodology

### 1. 전체 파이프라인
Stealthy Imitation은 공격자의 쿼리 예산이 소진될 때까지 다음의 네 단계를 반복적으로 수행하며 추정 분포 $S_a$를 정밀화한다.

**단계 I: 전이 데이터셋 구축 (Transfer Dataset Construction)**
공격자는 추정 분포 $S_a \sim \mathcal{N}(\mu, \sigma^2)$에서 상태 $s$를 샘플링하여 타겟 정책 $\pi_v$에 쿼리하고, 대응하는 액션 $a^*$를 기록하여 데이터셋 $D_v = \{(s, a^*)\}$를 구축한다. 초기 $\mu$는 $\mathbf{0}$으로, $\sigma$는 $\mathbf{1}$로 설정한다.

**단계 II: 행동 복제 (Behavioral Cloning, BC)**
구축된 $D_v$를 사용하여 타겟 정책을 모방하는 공격자 정책 $\pi_a$를 학습시킨다. 이때 이상치에 강건한 Huber Loss $L_b$를 사용한다.
$$
L_b(\hat{a}, a^*) = 
\begin{cases} 
0.5(\hat{a} - a^*)^2, & \text{if } |\hat{a} - a^*| < 1 \\
|\hat{a} - a^*| - 0.5, & \text{otherwise}
\end{cases}
$$

**단계 III: 보상 모델 학습 (Reward Model Training)**
타겟 정책과 공격자 정책의 행동 차이를 구별하는 판별기(Discriminator)인 보상 모델 $\hat{R}$을 학습시킨다. $\hat{R}$은 $\pi_v$의 데이터와 $\pi_a$의 데이터를 입력받아 타겟의 것인지 여부를 판별하며, 손실 함수는 다음과 같다.
$$
L_r(s, a) = \mathbb{E}_{(s, \hat{a}) \sim D_a} [-\log(\hat{R}(s, \hat{a}))] + \mathbb{E}_{(s, a^*) \sim D_v} [-\log(1 - \hat{R}(s, a^*))]
$$

**단계 IV: 보상 가이드 기반 분포 정밀화 (Reward-guided Distribution Refinement)**
학습된 $\hat{R}$을 이용해 각 샘플의 프록시 보상 $\hat{r}(s, a^*) = -\log(\hat{R}(s, a^*))$를 계산한다. $\hat{r}$ 값이 높을수록 공격자가 모방하기 어려웠던 지점이며, 이는 타겟 분포 $S_v$에 속할 확률이 높음을 의미한다. 이를 가중치로 사용하여 다음 반복 회차의 $\mu'$와 $\sigma'$를 업데이트한다.
$$
\mu' = \frac{\sum \hat{r}(s, a^*) \cdot s}{\sum \hat{r}(s, a^*)}, \quad \sigma'^2 = \frac{\sum \hat{r}(s, a^*) \cdot (s - \mu')^2}{\sum \hat{r}(s, a^*)}
$$

### 2. 정책 탈취 및 평가 (Policy Stealing)
분포 정밀화 과정에서 가장 최적의 분포를 찾기 위해 **분포 평가기(Distribution Evaluator) $\pi_e$**를 사용한다. $\pi_e$는 매 반복마다 BC로 학습되며, 검증 손실 $\bar{L}_b$가 가장 높은 지점을 기록한다. 가설에 따라 $\bar{L}_b$가 높을수록 $S_a$가 $S_v$에 가깝기 때문이다. 최종적으로 예산의 일부를 예약해 두었다가, $\pi_e$가 선택한 최적의 $\mu, \sigma$에서 데이터를 대량 샘플링하여 $\pi_a$를 처음부터 다시 학습시켜 최종 모델을 완성한다.

### 3. 방어 기법 (Countermeasure)
타겟 정책 소유자가 입력 범위(Min/Max)를 알고 있다는 점을 이용한다. 쿼리가 유효 범위를 벗어난 경우, 정답을 알려주거나 무시하는 대신 **무작위 액션(Random Action)**을 응답한다. 이는 공격자가 보상 모델을 통해 분포를 정밀화하는 과정에서 가짜 신호를 주어 $\mu, \sigma$ 추정을 방해한다.

## 📊 Results

### 1. 실험 설정
- **데이터셋 및 환경**: MuJoCo(Hopper, Walker2D, HalfCheetah) 및 Panda-gym(로봇 팔 제어) 환경 사용.
- **비교 대상**: Random sampling(다양한 스케일의 가우시안), DFME(이미지 도메인 기법을 제어 작업에 맞게 변형).
- **평가 지표**:
    - $D_{KL}(S_v \parallel S_a)$: 추정 분포와 실제 분포 사이의 KL 발산도.
    - **Return Ratio (rr)**: $\frac{\text{공격자 정책의 평균 리턴}}{\text{타겟 정책의 평균 리턴}}$.

### 2. 주요 결과
- **분포 추정 능력**: SI는 모든 환경에서 $D_{KL}$ 값을 지속적으로 감소시켰으며, 평균적으로 81%의 감소율을 보였다.
- **탈취 성능**: Hopper 환경에서는 5M 쿼리만으로 타겟 성능의 97%에 도달하는 높은 Return Ratio를 기록했다. 반면 DFME는 적대적 샘플 생성에만 집중하여 광범위한 탐색에 실패함으로써 정책 탈취에 실패했다.
- **실제 로봇 적용**: Panda-gym의 PickAndPlace 및 Reach 작업에서 각각 약 70%와 100%의 Return Ratio를 달성하여 고차원 입력 환경에서도 유효함을 입증했다.
- **방어 효율성**: 제안된 방어 기법을 적용했을 때, $\Delta D_{KL}$이 거의 감소하지 않았으며 Return Ratio가 0%~7% 수준으로 급격히 떨어져 공격이 효과적으로 차단됨을 확인했다.

## 🧠 Insights & Discussion

**강점 및 유효성**
본 연구는 타겟 정책의 '모방 난이도'와 '상태 분포' 사이의 상관관계를 이용해 환경 접근 없이도 분포를 찾아낼 수 있음을 증명했다. 특히, 실제 상태 데이터가 비가우시안(Non-Gaussian) 분포이고 변수 간 상관관계가 높음에도 불구하고, 단순한 대각 가우시안(Diagonal Gaussian) 근사만으로도 충분히 정책 탈취가 가능하다는 점은 시사하는 바가 크다. 이는 RL 정책이 입력 분포가 노출될 경우 지도 학습(BC)만으로도 매우 쉽게 복제될 수 있음을 의미한다.

**한계 및 비판적 해석**
1. **초기 분포 의존성**: 표준 가우시안으로 시작하여 성공했지만, 초기 분포 $S_a$와 실제 분포 $S_v$ 사이의 괴리가 극심할 경우 수렴 가능성에 대한 한계치가 명확히 제시되지 않았다.
2. **컴파운딩 에러(Compounding Errors)**: 결과 분석에서 나타나듯, 분포 추정이 완벽하더라도 아키텍처의 미세한 차이가 리턴 값의 큰 변동을 야기한다. 이는 모방 학습 특유의 에러 누적 문제로, 단순 BC 이상의 정교한 모방 기법이 필요함을 시사한다.
3. **가정의 단순함**: 타겟 정책이 분포 외 영역에서 단순하게 동작한다는 가설에 의존하고 있다. 만약 타겟 정책이 분포 외 영역에서도 복잡하게 동작하도록 설계되었다면 보상 모델의 가이드 능력이 떨어질 가능성이 있다.

## 📌 TL;DR

본 논문은 환경 접근 권한과 입력 범위 지식이 없는 극한의 블랙박스 상황에서도 타겟 RL 정책을 탈취할 수 있는 **Stealthy Imitation(SI)** 공격을 제안한다. 보상 모델을 이용해 모방 난이도가 높은 영역(즉, 실제 상태 분포)을 찾아내고 쿼리 분포를 정밀화하는 방식을 사용하며, 실험을 통해 MuJoCo 및 Panda 로봇 환경에서 매우 높은 복제 성능을 입증했다. 이 연구는 RL 기반 제어 시스템의 지적 재산 보호를 위해 입력 분포 노출을 방지하고, 유효 범위 외 쿼리에 대한 응답을 제어하는 방어 전략이 필수적임을 경고한다.
