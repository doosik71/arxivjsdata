# Neuroprosthetic decoder training as imitation learning

Josh Merel, David Carlson, Liam Paninski, John P. Cunningham (2016)

## 🧩 Problem to Solve

본 논문은 뇌-컴퓨터 인터페이스(Brain-Computer Interface, BCI)에서 신경 활동을 디코딩하여 외부 장치(커서 또는 로봇 팔 등)를 제어하는 디코더(Decoder)의 학습 문제를 다룬다.

일반적으로 BCI 디코더는 사용자의 실제 움직임(overt movements)과 신경 활동을 동시에 관찰하여 그 통계적 매핑을 학습한다. 그러나 사지 마비 환자와 같이 실제 움직임이 불가능한 사용자의 경우, 학습에 필요한 '정답' 데이터인 의도된 움직임을 직접 관찰할 수 없다는 근본적인 문제가 발생한다.

기존의 ReFIT(Recalibrated Feedback Intention-trained Kalman filter)과 같은 방법론은 사용자가 목표 지점을 향해 움직이려 한다는 가정을 바탕으로, 실제 궤적을 목표 방향으로 회전시켜 가상의 의도 데이터를 생성하는 방식을 사용했다. 하지만 이러한 접근 방식은 다음과 같은 한계가 있다:

1. **이론적 근거 부족**: 파라미터 학습 과정에 대한 명확한 이론적 토대가 없다.
2. **확장성 결여**: 단순한 커서 제어에는 유효하나, 다관절 로봇 팔과 같은 고차원 effector의 제어에는 적용하기 어렵다.

따라서 본 논문의 목표는 BCI 디코더 학습을 **모방 학습(Imitation Learning)** 관점에서 재정의하여, 이론적 분석이 가능하고 고차원 장치로 확장이 가능한 일반적인 학습 프레임워크를 제공하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 BCI 디코더 학습을 **DAgger(Dataset Aggregation)**라는 모방 학습 메타-알고리즘의 변형으로 해석하는 것이다.

1. **BCI 학습의 모방 학습 프레임워크화**: 디코더를 일종의 정책(Policy)으로, 신경 활동과 현재 상태를 입력값으로, effector의 업데이트를 행동(Action)으로 정의하였다. 직접적인 관찰이 불가능한 의도된 움직임을 제공하는 '의도-오라클(Intention-Oracle)'을 도입하여 지도 학습 형태로 디코더를 업데이트한다.
2. **이론적 분석(Regret Analysis)**: 온라인 학습 이론의 **Regret(후회도)** 개념을 도입하여, 다양한 파라미터 업데이트 규칙(OGD, MA, FTL)의 성능을 이론적으로 분석하고 최적의 규칙을 제시하였다.
3. **고차원 제어로의 확장**: 단순한 벡터 회전 방식에서 벗어나, **최적 제어(Optimal Control)** 이론을 이용하여 고차원 effector(예: 26 자유도의 로봇 팔)를 위한 의도-오라클을 생성하는 방법을 제안하였다.

## 📎 Related Works

논문은 기존의 **ReFIT**과 **CLDA(Closed-loop Decoder Adaptation)** 프레임워크를 언급한다.

- **ReFIT**: 폐루프(closed-loop) 데이터에서 속도 벡터를 목표 방향으로 회전시켜 학습 데이터를 수정하는 방식을 제안했다. 이는 경험적으로 성능 향상을 보였으나, 커서 제어라는 특수한 설정에만 국한되었으며 이론적 분석이 결여되어 있었다.
- **CLDA**: ReFIT의 아이디어를 확장하여 온라인으로 디코더를 적응시키는 방식을 제안했다. 그러나 여전히 '회전된 속도'라는 단순한 휴리스틱에 의존하고 있어, 다관절 팔과 같은 복잡한 시스템으로의 확장이 불가능했다.

본 연구는 이러한 기존 방식들이 사실상 모방 학습의 특수한 사례임을 밝히고, 이를 일반화하여 이론적 보장과 확장성을 확보했다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 1. 전체 시스템 구성 요소

BCI 시스템을 다음과 같은 구성 요소로 정의한다.

- **Effector**: 제어 대상 (예: 커서, 로봇 팔).
- **Task Objective**: 작업의 성능 측정 기준 (예: 목표 지점까지의 거리 최소화).
- **Intention-Oracle**: 사용자의 의도를 가장 잘 반영하는 정답 행동을 제공하는 전문가.
- **Decoder ($\pi$)**: 신경 활동 $\mathbf{n}$과 현재 상태 $\mathbf{x}$를 입력받아 행동 $\mathbf{o}$를 출력하는 함수.
- **Update Rule**: 오라클 데이터와 현재 디코더의 오차를 이용해 파라미터를 수정하는 규칙.

### 2. DAgger 기반 학습 파이프라인

학습은 다음과 같은 루프로 진행된다:

1. **데이터 수집**: 현재 디코더 $\pi^{(k)}$와 오라클 $\pi^*$를 적절히 혼합($\beta$ 비율로 blending)하여 effector를 움직이며 상태-행동 쌍 $\{\mathbf{x}, \mathbf{n}, \mathbf{o}\}$을 수집한다.
2. **데이터 집계 (Aggregation)**: 수집된 데이터를 기존 데이터셋 $\mathcal{D}$에 추가한다.
3. **디코더 업데이트**: 집계된 데이터셋을 사용하여 새로운 디코더 $\pi^{(k+1)}$를 학습한다.

### 3. 파라미터 업데이트 규칙 및 수학적 정의

손실 함수를 $\ell(\pi(\mathbf{x}, \mathbf{n}), \mathbf{o})$라고 할 때, 세 가지 업데이트 방식을 비교한다.

- **Online Gradient Descent (OGD)**:
  $$\pi^{(k+1)} = \pi^{(k)} - \frac{1}{\eta_k} \nabla_{\pi} L(\pi^{(k)}, \mathcal{D}^{(k+1)})$$
  매 회차마다 그래디언트 방향으로 한 단계 업데이트하며, 학습률 $\eta_k$ 설정이 중요하다.

- **Moving Average (MA)**:
  $$\pi^{(k+1)} = (1-\lambda)\pi^{(k)} + \lambda \arg \min_{\pi} L(\pi, \mathcal{D}^{(k+1)})$$
  최근 데이터로 학습한 파라미터와 이전 파라미터를 지수적으로 가중 평균한다.

- **Follow-the-Leader (FTL)**:
  $$\pi^{(k+1)} = \arg \min_{\pi} L(\pi, \mathcal{D}^{(1:k+1)})$$
  현재까지 수집된 모든 데이터를 사용하여 최적의 파라미터를 다시 추정한다.

### 4. 고차원 Oracle 설계

고차원 effector의 경우, 단순한 방향 회전이 불가능하므로 **MuJoCo** 물리 엔진의 최적 제어 솔버를 사용한다. 목표 함수(예: 손목과 타겟 간의 거리 최소화)를 설정하면, 솔버가 현재 상태에서 목표를 달성하기 위한 최적의 관절 각도 변화량(Joint angular velocities)을 계산하며, 이것이 곧 $\pi^*$ (Oracle)가 된다.

## 📊 Results

### 1. 커서 제어 시뮬레이션

- **설정**: 3D 공간에서 커서를 목표 지점으로 이동시키는 작업.
- **결과**: **FTL** 방식이 가장 빠르게 수렴하고 가장 낮은 손실(SSE)을 기록했다. OGD는 FTL보다 성능이 낮았으며, MA는 가장 낮은 성능을 보였다.
- **정량적 지표**: Sum Squared Error (SSE)를 사용하여 평가하였으며, FTL은 학습 초기 단계에서 매우 가파른 성능 향상을 보였다.

### 2. 26자유도 로봇 팔 제어 시뮬레이션

- **설정**: 원숭이 팔 모델(26 DOF)을 사용하여 타겟 'wand'를 잡는 작업.
- **결과**: 고차원 공간임에도 불구하고 FTL 기반 디코더가 빠르게 수렴함을 확인했다.
- **분석**: 상관계수 분석 결과, 작업 수행에 필수적인 어깨(Shoulder) 관절의 파라미터가 먼저 학습되고, 상대적으로 덜 중요한 손가락 끝 관절의 파라미터는 천천히 학습되는 경향을 보였다. 이는 DAgger가 작업 관련 상태를 우선적으로 샘플링하기 때문이다.

### 3. 모델 불일치(Model Mismatch) 테스트

- **설정**: 사용자의 실제 의도에 무작위 노이즈를 섞어 오라클과 불일치하게 설정.
- **결과**: 의도에 노이즈가 추가되어도 성능 저하가 급격하지 않고 완만하게(graceful) 일어남을 확인했다. 즉, 제안된 프레임워크는 어느 정도의 의도 불일치에 대해 강건(robust)하다.

## 🧠 Insights & Discussion

본 연구는 BCI 디코더 학습을 모방 학습으로 공식화함으로써 다음과 같은 통찰을 제공한다.

1. **이론적 보장과 효율성**: Regret 분석을 통해 FTL 방식이 $O(\log K)$의 후회도를 가져 최적의 성능을 낼 수 있음을 증명했다. 특히 데이터 수집 비용이 비싼 BCI 환경에서는 계산량이 많더라도 모든 데이터를 활용하는 FTL(또는 RLS) 방식이 OGD보다 유리하다.
2. **범용적 확장성**: 최적 제어 기반의 오라클을 도입함으로써, 기존의 ReFIT 방식이 가진 '단순 커서' 제약에서 벗어나 임의의 고차원 effector로 학습을 확장할 수 있는 경로를 제시했다.
3. **디코더 독립성**: 본 프레임워크는 Kalman Filter뿐만 아니라 다른 정적인 결정론적 디코더(예: 신경망)에도 동일하게 적용될 수 있는 일반적인 메타-알고리즘이다.

**한계점**: 본 연구는 신경 부호화(Neural Encoding)가 시간에 따라 변하지 않는 정적 모델이라고 가정했다. 실제 환경에서는 신경 신호의 드리프트(Drift)가 발생할 수 있으며, 이 경우 모든 데이터를 사용하는 FTL보다는 최근 데이터에 가중치를 두는 OGD나 변형된 온라인 학습 알고리즘이 더 적합할 수 있다.

## 📌 TL;DR

이 논문은 BCI 디코더 학습을 **모방 학습(Imitation Learning)** 및 **DAgger** 알고리즘으로 재정의하여, 이론적 근거가 부족했던 기존의 의도 기반 학습법을 체계화했다. **Follow-the-Leader(FTL)** 업데이트 규칙이 이론적/실험적으로 가장 우수함을 보였으며, 특히 **최적 제어(Optimal Control)**를 결합하여 26 자유도의 고차원 로봇 팔 제어까지 성공적으로 확장 가능함을 입증했다. 이는 향후 복잡한 신체 부위를 제어하는 고성능 신경 보조 장치 개발에 중요한 이론적 기반이 될 것으로 보인다.
