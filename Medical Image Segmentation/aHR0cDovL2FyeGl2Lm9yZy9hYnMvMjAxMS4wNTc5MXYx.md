# Interpretable and synergistic deep learning for visual explanations and statistical estimations of segmentation of disease features from medical images

Sambuddha Ghosal, Pratik Shah (2020)

## 🧩 Problem to Solve

본 연구는 의료 영상 분야에서 널리 사용되는 전이 학습(Transfer Learning, TL)의 실질적인 유효성과 한계를 분석하는 것을 목표로 한다. 많은 연구자가 자연 이미지 데이터셋인 ImageNet으로 사전 학습된 모델을 의료 영상 세그멘테이션 작업에 적용하여 성능을 향상시키려 하지만, 이는 단순히 학습 데이터의 양이 증가하면 성능이 좋아질 것이라는 가정에 기반한 경우가 많다.

의료 영상은 자연 이미지와 달리 복잡한 생물학적 텍스처를 가지며, 조명, 광학 특성, 임상 라벨의 다양성 등으로 인해 모델 성능이 급격히 저하될 수 있다. 또한, 의료 데이터셋의 특성상 데이터 양이 적은 저데이터 환경(low data regime)이 빈번하며, 일반적인 80:20 무작위 분할 방식이 라벨의 불균형을 초래하여 모델의 재현성과 복제 가능성을 떨어뜨리는 문제가 존재한다. 따라서 본 논문은 다양한 의료 영상 모달리티에서 전이 학습 모델과 의료 데이터만으로 학습한 모델의 성능을 엄격하게 비교하고, 이를 해석 가능한 방법으로 분석하고자 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 다음과 같다.

1. **전이 학습 모델($T^{II}$)과 의료 영상 전용 학습 모델($L^{MI}$)의 정밀 비교**: 피부암(RGB), 전립선 생검(H&E), 신장 CT(DICOM)라는 세 가지 서로 다른 모달리티를 대상으로 두 학습 전략의 성능을 통계적으로 분석하였다.
2. **해석 가능한 딥러닝 프레임워크 제공**: Grad-CAM을 활용하여 모델이 영상의 어느 부분에 주목하여 세그멘테이션을 수행했는지 시각화함으로써, 성능 차이의 원인을 분석하였다.
3. **시너지(Synergistic) 앙상블 전략 제안**: 두 모델 중 하나를 선택하는 기존 방식에서 벗어나, 각 모델의 강점(민감도 vs 특이도)을 결합하는 합집합(Union) 및 교집합(Intersection) 마스크 연산을 통한 성능 향상 방안을 제시하였다.
4. **통계적 분석 방법론의 정립**: 데이터의 비정규성을 고려하여 Shapiro-Wilk 검정과 Mood's median test와 같은 비모수적 통계 방법을 적용해 결과의 유의성을 검증하였다.
5. **오픈 리소스 공개**: 학습된 모델 30개, 10,000장 이상의 테스트 이미지 및 Grad-CAM 결과, 통계 분석 패키지를 GitHub에 공개하여 후속 연구의 기반을 마련하였다.

## 📎 Related Works

기존 연구들은 주로 자연 이미지(ImageNet)에서 의료 영상으로의 전이 학습이 계산 비용을 줄이고, 적은 데이터로도 오버피팅을 방지하며 성능을 높인다고 주장해 왔다. 특히 MRI나 CT와 같은 동일 모달리티 간의 전이 학습이 효과적이라는 결과가 많았다.

반면, 최근 일부 연구(예: Raghu et al.)에서는 ImageNet 사전 학습이 의료 영상 분류 작업에서 반드시 성능 향상을 가져오지 않으며, 오히려 모델의 과잉 매개변수화(over-parameterization) 문제가 더 중요할 수 있다는 회의적인 시각을 제시하였다. 하지만 이러한 논의는 주로 '분류(Classification)' 작업에 집중되어 있었으며, '세그멘테이션(Segmentation)' 작업에서 모달리티별로 전이 학습이 어떤 영향을 미치는지에 대한 상세한 분석은 부족한 실정이다.

## 🛠️ Methodology

### 1. 시스템 구조 및 모델 아키텍처
본 연구는 세그멘테이션에 최적화된 **VGG-UNet** 구조를 사용한다.
- **Encoder**: Fully-connected 레이어가 제거된 VGG-16 아키텍처를 사용하여 특징을 추출한다.
- **Decoder**: 인코더에서 추출된 특징 맵을 업샘플링하여 최종 클래스 기반의 세그멘테이션 마스크를 생성한다.

### 2. 학습 전략
두 가지 학습 방식을 비교한다.
- **$T^{II}$ (Transfer Learning)**: ImageNet으로 사전 학습된 가중치로 초기화한 후, 특정 의료 영상 데이터로 미세 조정(Fine-tuning)한다.
- **$L^{MI}$ (Learning from Medical Images)**: 사전 학습 없이 무작위 가중치(Random initialization)로 초기화하여 의료 영상 데이터만으로 학습한다.

### 3. 학습 절차 및 손실 함수
- **손실 함수**: 픽셀 단위의 이진 분류를 위해 Categorical Cross-Entropy 손실 함수를 사용한다.
- **최적화 알고리즘**: AdaDelta 옵티마이저를 사용한다.
- **학습 횟수**: 피부 및 전립선 데이터는 40 epoch, 신장 CT 데이터는 20 epoch 동안 학습하였다.

### 4. 평가 지표 및 수학적 정의
모델 성능 평가를 위해 다음과 같은 지표를 사용한다.
- **Sensitivity (민감도)**: $\frac{TP}{TP+FN}$ (실제 병변을 얼마나 잘 찾아냈는가)
- **Specificity (특이도)**: $\frac{TN}{TN+FP}$ (정상 영역을 얼마나 잘 구분했는가)
- **Dice Score (F1 Score)**: 정밀도(Precision)와 민감도의 조화 평균으로, 세그멘테이션의 정확도를 측정한다.
  $$Dice = \frac{2 \cdot Precision \cdot Recall}{Precision + Recall}$$
- **$\Delta_m$ (성능 차이)**: 특정 지표 $m$에 대해 두 모델의 차이를 다음과 같이 정의한다.
  $$\Delta_m = S_{T^{II}}^m - S_{L^{MI}}^m$$
  여기서 $\Delta_m > 0$이면 $T^{II}$가 우수함을 의미한다.

### 5. 해석 및 통계 방법
- **Grad-CAM**: 최종 컨볼루션 레이어의 그라디언트 정보를 활용하여 모델이 집중한 영역을 히트맵으로 시각화한다. 5번의 독립적 실행 결과를 평균하여 최종 히트맵을 생성한다.
- **통계 검정**: 데이터 분포가 정규분포를 따르지 않음을 확인(Shapiro-Wilk test)한 후, 비모수 검정인 **Mood's median test**를 사용하여 중앙값의 유의미한 차이($p < 0.05$)를 검증한다.

## 📊 Results

### 1. 데이터셋 및 실험 설정
- **피부암(ISIC)**: 13,786장 (RGB)
- **전립선 생검(Gleason2019)**: 244장 (RGB, H&E staining)
- **신장 CT(KiTS19)**: 45,937장 (DICOM, 2D slices)

### 2. 모달리티별 성능 결과
- **피부암**: $T^{II}$ 모델이 모든 지표(AUROC, Dice, Sensitivity)에서 $L^{MI}$보다 통계적으로 유의미하게 우수하였다. Grad-CAM 분석 결과, $T^{II}$는 병변과 배경 피부를 구분하는 능력이 훨씬 정밀했다.
- **전립선 생검**: $L^{MI}$ 모델이 $T^{II}$보다 약간 더 높은 성능을 보이거나 유사한 결과를 냈다. 특히 민감도 측면에서 $L^{MI}$가 유의미한 차이를 보였으며, 이는 병리 영상의 특성상 자연 이미지의 특징보다 의료 전용 특징 학습이 더 중요함을 시사한다.
- **신장 CT**: $L^{MI}$ 모델이 민감도와 AUROC에서 약간 더 우수($<5\%$ 차이)했으나, 전반적으로 두 모델의 성능은 유사했다.

### 3. 저데이터 환경(Low Data Regime) 분석
학습 데이터를 80%에서 10%까지 점진적으로 줄였을 때:
- **피부암**에서는 데이터가 적을수록(10-20%) $T^{II}$의 우위가 더욱 뚜렷해졌다.
- **전립선 및 CT** 영상에서는 데이터 감소에 따라 두 모델 모두 성능이 저하되었으나, 특정 지점(10% 등)에서 $T^{II}$가 보완적인 역할을 하는 경우가 관찰되었다.

## 🧠 Insights & Discussion

### 1. 전이 학습의 보편적 효용성에 대한 비판
본 연구는 전이 학습이 모든 의료 영상 작업의 해결책이 아님을 입증하였다. RGB 피부 영상처럼 자연 이미지와 유사한 특성을 가진 경우 $T^{II}$가 매우 효과적이지만, CT나 병리 조직 영상과 같은 특수 모달리티에서는 $L^{MI}$가 더 우수하거나 전이 학습의 이점이 미미하였다.

### 2. 모델 간의 시너지 및 앙상블 전략
두 모델은 서로 다른 오류 특성을 보인다.
- **$T^{II}$**: 일반적으로 민감도가 높아 병변을 잘 찾아내지만, 위양성(False Positive)이 증가하여 특이도가 낮아지는 경향이 있다.
- **$L^{MI}$**: 특이도가 높아 배경을 잘 구분하지만, 위음성(False Negative)이 발생하여 민감도가 낮아질 수 있다.

이를 해결하기 위해 본 논문은 다음과 같은 시너지 전략을 제안한다.
- **Union (합집합)**: 두 모델의 마스크를 합쳐 위음성(FN)을 줄이고 민감도를 높인다. (특히 $T^{II}$가 FN을 많이 생성할 때 유용)
- **Intersection (교집합)**: 두 모델이 공통으로 예측한 영역만 선택하여 위양성(FP)을 줄이고 특이도를 높인다. (특히 $T^{II}$가 FP를 많이 생성할 때 유용)

### 3. 데이터 분할의 위험성
단순한 무작위 80:20 분할은 라벨 분포가 비정규분포(Non-Gaussian)일 때 심각한 편향을 초래할 수 있음을 확인하였다. 따라서 최소 5가지 이상의 다른 비율로 분할하고 반복 학습을 통해 최적의 분포를 찾는 과정이 필수적이다.

## 📌 TL;DR

이 논문은 의료 영상 세그멘테이션에서 ImageNet 기반 전이 학습($T^{II}$)과 의료 데이터 전용 학습($L^{MI}$)을 세 가지 모달리티(피부, 전립선, CT)에서 정밀 비교하였다. 연구 결과, 전이 학습은 RGB 피부 영상과 저데이터 환경에서 매우 효과적이지만, CT나 병리 영상에서는 그 효과가 제한적이었다. 특히, 두 모델이 서로 다른 오류 패턴(민감도 vs 특이도)을 가진다는 점에 착안하여, 마스크의 합집합과 교집합 연산을 통해 두 모델을 시너지 있게 결합하는 앙상블 전략이 단일 모델 사용보다 우수함을 입증하였다. 이 연구는 의료 영상의 특성에 맞는 학습 전략 선택과 해석 가능한 모델 구축의 중요성을 제시한다.
