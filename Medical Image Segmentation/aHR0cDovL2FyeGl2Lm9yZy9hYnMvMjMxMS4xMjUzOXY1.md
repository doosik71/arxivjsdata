# GMISeg: General Medical Image Segmentation without Re-Training

Jing Xu (n.d.)

## 🧩 Problem to Solve

본 연구가 해결하고자 하는 핵심 문제는 의료 영상 분할(Medical Image Segmentation) 모델의 낮은 일반화 성능이다. 기존의 딥러닝 기반 분할 모델들은 새로운 해부학적 구조, 레이블, 또는 이미지 형태를 가진 미지의 작업(unknown tasks)에 직면했을 때 성능이 급격히 저하되는 경향이 있다.

이러한 문제를 해결하기 위해서는 새로운 데이터셋에 대해 모델을 처음부터 다시 학습시키거나 미세 조정(fine-tuning)해야 한다. 그러나 의료 분야의 임상 연구자나 과학자들은 대부분 머신러닝 전문가가 아니며, 모델 학습에 필요한 상당한 시간과 계산 자원, 전문 지식을 갖추기 어렵다. 결과적으로 이러한 실무적 제약은 새로운 의료 영상 분할 모델의 개발과 임상 적용 속도를 늦추는 병목 현상이 된다. 따라서 본 논문의 목표는 추가적인 재학습 없이도 다양한 의료 영상 분할 작업을 수행할 수 있는 범용 모델을 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 재학습 없이 새로운 분할 작업에 일반화될 수 있는 **GMISeg** 프레임워크의 제안이다. 중심적인 설계 아이디어는 다음과 같다.

1. **Loose LoRA (Low-Rank Adaptation) 도입**: 기존의 LoRA 구조를 개선하여, MLP 층을 평균 풀링(average pooling) 연산을 통해 동일한 크기의 벡터로 변환하는 'Loose' 구조를 제안하였다. 이는 모델이 기존에 학습한 지식을 잊어버리는 치명적 망각(catastrophic forgetting) 현상을 방지하는 강성(rigidity)과 새로운 작업에 적응하는 가소성(plasticity) 사이의 균형을 맞추기 위함이다.
2. **효율적인 파라미터 업데이트**: 이미지 인코더의 전체 파라미터를 동결(freeze)하고, 극소수의 파라미터만을 업데이트하는 저차원 행렬 근사 방식을 통해 효율적인 미세 조정을 수행하였다.
3. **범용적 적용 가능성 검증**: 서로 다른 영상 모달리티(MRI, CT, X-ray, Optical camera 등)와 다양한 해부학적 구조(심장, 복부, 뇌, 망막 등)를 포함하는 데이터셋을 통해 모델의 일반화 능력을 입증하였다.

## 📎 Related Works

논문에서는 의료 영상 분할을 위한 기존 접근 방식과 그 한계를 다음과 같이 설명한다.

- **지도 학습 기반 CNN**: 각 작업마다 모델을 처음부터 설계하고 학습시켜야 하므로 오버헤드가 매우 크다.
- **nnU-Net**: 데이터 전처리나 아키텍처 설계를 자동화하였으나, 여전히 상당한 학습 시간이 소요된다.
- **다중 작업 학습 (Multi-task Learning)**: 여러 작업을 동시에 학습하지만, 학습 단계에서 정의된 작업만 수행할 수 있어 추론 단계에서의 동적인 작업 지정이 불가능하다.
- **전이 학습 (Transfer Learning)**: 자연 이미지로 사전 학습된 모델을 활용하지만, 의료 영상과 자연 이미지 간의 특성 차이로 인해 효율이 떨어지며 여전히 각 작업별 미세 조정이 필요하다.
- **메타 학습 및 Few-shot Segmentation**: 적은 데이터로 적응하려 시도하지만, 여전히 미세 조정 과정에서 ML 전문가의 개입과 계산 자원이 필요하며, 주로 동일 도메인 내의 클래스 확장에 치중되어 있어 서로 다른 모달리티나 장기 간의 일반화에는 한계가 있다.

GMISeg는 이러한 한계들을 극복하여, 사용자가 추가 학습 없이 예시 샘플(support set)과 시각적 프롬프트만을 제공함으로써 즉시 새로운 분할 작업을 수행할 수 있게 한다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조

GMISeg는 **SAM (Segment Anything Model)**의 아키텍처를 계승한다. 전체 파이프라인은 사전 학습된 ViT 기반의 이미지 인코더, 프롬프트 인코더, 그리고 마스크 디코더로 구성된다. 본 방법론의 핵심은 이미지 인코더를 동결한 상태에서 각 트랜스포머 블록에 학습 가능한 **Loose LoRA** 바이패스(bypass)를 추가하는 것이다.

### 주요 구성 요소 및 상세 설명

#### 1. LoRA in Image Encoder

ViT의 $Q, K, V$ 투영 층에 LoRA를 적용하여 효율적인 파라미터 업데이트를 수행한다. LoRA는 거대한 가중치 행렬의 변화량을 두 개의 작은 저차원 행렬의 곱으로 근사하는 방식이다.

- **가중치 업데이트 방정식**:
    $$ \hat{W} = W + \Delta W = W + BA $$
    여기서 $W$는 동결된 사전 학습 가중치이며, $A$와 $B$는 학습 가능한 저차원 행렬이다. 입력 토큰 $F$에 대해 업데이트된 결과는 $\hat{F} = \hat{W}F$가 된다.

- **어텐션 메커니즘 적용**:
    $$ \text{attn}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{C_{attn}}} + B\right)V $$
    여기서 $Q$는 다음과 같이 계산된다:
    $$ Q = \hat{W}_q F = W_q F + B_q A_q F $$
    ($K$ 또한 동일한 방식으로 LoRA가 적용된다.)

#### 2. Loose LoRA Embedding

기존 LoRA의 MLP 구조가 대형 모델에서 치명적 망각을 일으킬 수 있다는 점에 착안하여 **Loose LoRA**를 제안하였다.

- **구조**: 인코더와 디코더의 각 MLP 층을 평균 풀링 연산을 통해 동일한 크기의 벡터로 변환한다.
- **역할**: 이 구조는 모델의 강성(anti-forgetting)을 높여 기존 지식을 유지하게 함과 동시에, 새로운 작업에 대한 가소성(learning ability)을 높여 일반화 성능을 향상시킨다.

#### 3. 훈련 전략 및 손실 함수

- **손실 함수**: 클래스 불균형 문제를 해결하기 위한 Focal Loss와 영역 유사도를 측정하는 Dice Loss를 결합하여 사용한다.
    $$ L = \lambda_1 \text{Focal}(\hat{S}_f, \text{down}(S)) + \lambda_2 \text{Dice}(\hat{S}_f, \text{down}(S)) $$
    여기서 $\text{down}(\cdot)$은 정답(Ground Truth)의 해상도를 모델 출력 해상도와 맞추기 위한 다운샘플링 연산이다.
- **프롬프트 생성**: 사용자의 입력을 시뮬레이션하기 위해 정답 데이터에서 무작위로 점(point)을 선택하거나, 최대 경계에 오프셋을 더해 바운딩 박스(box)를 생성하여 프롬프트 인코더에 입력한다.

## 📊 Results

### 실험 설정

- **데이터셋**: ACDC, MICCAI 2018(심장), Synapses(복부), STARE(망막), ISLES 2022(뇌) 등 5개의 데이터셋을 사용하였다.
- **비교 대상**: CycleGAN, Neuralizer, Universeg 등 최신 일반화 분할 모델들과 비교하였다.
- **평가 지표**: Dice score, Jaccard score, ASD (Average Surface Distance)를 사용하였다.

### 주요 결과

GMISeg는 모든 테스트 데이터셋에서 기존 모델들보다 우수한 성능을 보였다.

- **정량적 결과**: Dice score 기준, 기존 모델 대비 $3.2\% \sim 4.0\%$의 향상을 보였으며, Jaccard score는 $2.4\% \sim 13.3\%$ 향상되었다. ASD 값 또한 유의미하게 감소하여 경계 예측의 정확도가 높아졌음을 확인하였다.
- **정성적 결과**: 시각화 결과, GMISeg가 예측한 분할 영역이 타 모델보다 더 매끄럽고 정확한 형태를 띠었다. 이는 SAM의 강력한 특징 추출 능력과 Loose LoRA의 효과적인 미세 조정 전략의 결합 덕분으로 분석된다.

### 분석 결과

1. **작업 다양성**: 학습 시 사용된 작업의 수가 많을수록 테스트 작업에서의 성능이 향상되었다. 특히 학습 데이터셋의 선택이 결과에 큰 영향을 미침을 확인하였다.
2. **서포트 세트 크기 ($N$)**: 추론 시 제공되는 예시 이미지(support size)가 $1 \to 64$로 증가함에 따라 Dice score가 $57.6\% \to 69.1\%$로 빠르게 증가하였다. 특히 $N > 6$일 때 앙상블 예측이 추가적인 성능 향상을 가져왔다.
3. **제한된 데이터 영향**: 매우 적은 수의 레이블된 예시만으로도 추론이 가능하지만, 예시 수가 적을수록 샘플의 다양성에 따라 성능 편차(variance)가 크게 나타났다.

## 🧠 Insights & Discussion

**강점**: GMISeg는 의료 영상 분야에서 가장 큰 문제 중 하나인 '재학습의 번거로움'을 해결하였다. 특히 SAM이라는 거대 모델의 제로샷 능력을 의료 도메인에 맞게 효율적으로 이식하였으며, Loose LoRA를 통해 모델의 안정성과 적응력을 동시에 확보하였다.

**한계 및 가정**:

- 본 연구는 2D 이미지와 이진 분할(binary segmentation) 작업에 집중하였다. 따라서 3D 볼륨 데이터나 다중 클래스 분할 작업에 대한 검증은 이루어지지 않았다.
- 추론 성능이 서포트 세트(support set)의 크기와 품질에 강하게 의존한다는 점은, 완전히 데이터가 없는 상황에서의 성능에는 한계가 있을 수 있음을 시사한다.

**비판적 해석**: 논문에서는 재학습이 필요 없다고 주장하지만, 사실상 'General' 모델을 만들기 위해 초기 단계에서 방대한 양의 다양한 의료 데이터셋으로 사전 학습(fine-tuning)을 거쳤다. 따라서 이는 완전히 새로운 모델을 만드는 비용을 줄인 것이지, 학습 과정 자체가 사라진 것은 아니다. 다만, 최종 사용자(임상 연구자) 관점에서는 추가 학습 없이 프롬프트와 예시만으로 사용할 수 있으므로 실용적 가치가 매우 높다.

## 📌 TL;DR

GMISeg는 SAM의 구조에 **Loose LoRA**라는 효율적인 저차원 파라미터 업데이트 전략을 결합하여, **추가적인 재학습 없이** 새로운 의료 영상 분할 작업을 수행할 수 있게 한 범용 모델이다. 다양한 모달리티와 해부학적 구조에서 기존 일반화 모델보다 우수한 성능(Dice score $\uparrow 3.2\sim4.0\%$)을 입증하였으며, 이는 ML 전문 지식이 없는 의료진이 AI 모델을 실제 임상 연구에 즉각적으로 도입하는 데 기여할 가능성이 매우 높다.
