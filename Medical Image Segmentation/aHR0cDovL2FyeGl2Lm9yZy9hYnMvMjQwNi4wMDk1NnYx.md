# Improving Segment Anything on the Fly: Auxiliary Online Learning and Adaptive Fusion for Medical Image Segmentation

Tianyu Huang, Tao Zhou, Weidi Xie, Shuo Wang, Qi Dou, Yizhe Zhang (2024)

## 🧩 Problem to Solve

본 논문은 의료 영상 분할(Medical Image Segmentation) 분야에서 Segment Anything Model (SAM) 및 그 변형 모델들이 실제 임상 수준의 정밀한 분할 성능을 달성하지 못하고 있다는 문제에서 출발한다. 의료 영상의 특성상 도메인(CT, MRI, 초음파 등)과 대상(장기, 조직, 세포 등)의 다양성이 매우 크며, 이로 인해 범용 모델인 SAM이 생성한 결과물을 의료 전문가가 수동 또는 반수동으로 수정하는 과정이 빈번하게 발생한다.

기존의 해결책으로는 의료 전용 SAM을 새로 학습시키거나, 어댑터(Adapter) 등을 통해 특정 작업에 맞게 미세 조정(Fine-tuning)하는 방식이 제안되었다. 하지만 전자는 방대한 양의 라벨링된 데이터가 필요하며, 후자는 오프라인 학습 단계에서 상당한 계산 비용과 시간이 소요될 뿐만 아니라 모델의 범용성을 잃게 만들 수 있고, 새로운 작업이 추가될 때마다 매번 새로운 오프라인 학습 세션을 거쳐야 하는 경직성이라는 한계가 있다.

따라서 본 연구의 목표는 모델 배포 후 테스트 단계(Test-time)에서 의료 전문가의 피드백을 활용해 실시간으로 모델의 성능을 개선할 수 있는 효율적이고 유연한 온라인 학습(Online Learning) 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 거대한 범용 모델인 SAM(Generalist)과 함께, 작고 가벼운 보조 모델(Specialist)을 운용하여 테스트 시점에 실시간으로 학습시키는 **Auxiliary Online Learning (AuxOL)** 방법론을 제안하는 것이다. 

주요 기여 사항은 다음과 같다:
1. 의료 영상 분할 성능 향상을 위해 SAM에 온라인 머신러닝(OML) 개념을 최초로 도입하였다.
2. SAM의 거대한 파라미터를 직접 업데이트하는 대신, 작은 보조 모델을 생성하여 학습함으로써 학습 효율성, 계산 비용, 그리고 학습 안정성을 동시에 확보하였다.
3. 보조 모델의 출력을 SAM의 출력과 적응적으로 융합(Adaptive Fusion)하는 기법을 통해 최적의 분할 결과를 도출하도록 설계하였다.
4. 다양한 의료 영상 모달리티와 데이터셋을 통해 제안 방법론이 SAM 및 이미 적응된(Adapted) SAM의 성능을 유의미하게 향상시킴을 입증하였다.

## 📎 Related Works

### 1. Medical Segment Anything Model
Medical SAM이나 SAM-Med2D와 같이 의료 영상 데이터셋으로 사전 학습된 모델들이 제안되었다. 하지만 이러한 모델들조차 임상적으로 수용 가능한 수준의 정밀도와는 여전히 간극이 존재하며, 의료 파운데이션 모델의 특징(Feature)이 반드시 작은 전용 모델보다 뛰어난 성능을 보장하지 않는다는 점이 지적되었다.

### 2. Adapting SAM for Specific Tasks
Medical-SAM-Adapter나 SAMed와 같은 연구들은 LoRA 또는 추가 레이어 삽입을 통해 SAM을 특정 도메인에 적응시켰다. 그러나 이러한 방식은 모두 오프라인 전수 학습(Offline full-training)에 의존하므로 계산 비용이 높고, 새로운 태스크가 나타날 때마다 재학습이 필요한 경직된 구조를 가진다.

### 3. Online Machine Learning
실시간 데이터 스트림으로부터 학습하는 온라인 학습은 유연성과 효율성이 높다. 하지만 SAM과 같은 초거대 모델에 이를 직접 적용할 경우, 학습 비용이 너무 크고 가중치 업데이트 과정에서 모델이 붕괴(Model collapse)되거나 불안정해질 위험이 크다.

## 🛠️ Methodology

### 전체 파이프라인
AuxOL은 SAM($f_{SAM}$)과 작은 보조 모델($f_{aux}$, 예: U-Net)을 함께 사용한다. 전체 프로세스는 다음과 같다:
1. **추론 단계**: 이미지 $x_t$와 프롬프트 $P_t$가 입력되면, SAM이 분할 마스크 $s_{t,j}$를 생성한다.
2. **영역 추출**: 생성된 $s_{t,j}$의 좌표를 기반으로 해당 영역을 크롭(Crop)하여 $x_{t,j}$를 얻는다.
3. **보조 모델 추론**: 크롭된 이미지 $x_{t,j}$를 보조 모델 $f_{aux}$에 입력하여 $u_{t,j}$를 얻는다.
4. **융합(Fusion)**: SAM의 출력 $s_{t,j}$와 보조 모델의 출력 $u_{t,j}$를 결합하여 최종 결과 $\hat{y}_{t,j}$를 생성한다.
5. **온라인 학습**: 의료 전문가(HE)가 $\hat{y}_{t,j}$를 수정하여 정답지 $y_{t,j}$를 생성하면, 이를 이용해 보조 모델 $f_{aux}$만 업데이트한다.

### 핵심 방정식 및 알고리즘

**1. 세그멘테이션 융합 (Segmentation Fusion)**
두 모델의 기여도를 조절하는 스칼라 파라미터 $\alpha$를 사용하여 다음과 같이 결합한다.
$$\hat{y}_{t,j} = \tau(\alpha s_{t,j} + (1-\alpha)u_{t,j})$$
여기서 $\tau$는 Sigmoid와 같은 활성화 함수이다.

**2. 적응적 융합 (Adaptive Fusion)**
고정된 $\alpha$ 대신, 이전 샘플들에서 얻은 통계를 바탕으로 $\alpha$를 최적화한다. 구체적으로는 다음과 같은 목적 함수를 최대화하는 $\alpha_{t,j}^*$를 찾는다.
$$\arg \max_{\alpha_{t,j}} \text{DSC}(\tau(\alpha_{t,j}s_{t,j} + (1-\alpha_{t,j})u_{t,j}), y_{t,j})$$
현재 샘플의 $\alpha$ 값은 최근 $K$개(기본값 5)의 최적 $\alpha^*$ 값들의 평균으로 결정된다.

**3. 온라인 배치 기반 업데이트 (Online-batch based Update)**
단일 샘플 학습의 불안정성을 줄이기 위해 크기 $k$의 온라인 배치를 유지한다.
- 배치가 가득 찼을 때 새로운 샘플이 들어오면, 현재 배치 내에서 손실(Loss)이 가장 작은 샘플을 찾아 새로운 샘플의 손실과 비교한다.
- 새로운 샘플의 손실이 더 크다면, 기존의 가장 작은 손실을 가진 샘플을 제거하고 새로운 샘플을 추가한다.
- 업데이트에는 AdamW 옵티마이저를 사용하며, 손실 함수로는 Dice Loss 등을 사용한다.

## 📊 Results

### 실험 설정
- **데이터셋**: 폴립 분할(5개 데이터셋), 유방암 초음파 분할(1개), 조직학 글랜드 분할(1개), OCT 유체 영역 분할(1개) 등 총 8개 데이터셋을 사용하였다.
- **모델**: SAM-H(641.1M), Medical SAM(93.7M)을 기반 모델로 사용하였으며, 보조 모델로는 랜덤 가중치로 초기화된 U-Net(17.3M)을 사용하였다.
- **지표**: Dice Similarity Coefficient (DSC $\uparrow$)와 Hausdorff Distance (HD $\downarrow$)를 측정하였다.

### 주요 결과
- **성능 향상**: 모든 데이터셋에서 AuxOL을 적용했을 때 SAM과 Medical SAM의 DSC가 상승하고 HD가 감소하였다. 특히 Medical SAM의 경우, 포인트 프롬프트 사용 시 DSC가 비약적으로 상승하는 결과(예: Table 1의 CVC-300에서 10.32 $\rightarrow$ 72.34)를 보였다.
- **부분적 피드백의 효과**: 의료 전문가의 수정이 전체의 25%만 이루어진 경우에도 상당한 성능 향상이 관찰되었다. 또한, DSC가 특정 임계값(예: 0.85) 미만인 저품질 샘플에 대해서만 피드백을 제공하는 실용적인 시나리오에서도 효과적이었다.
- **기존 적응 모델과의 결합**: MSA나 SAMed와 같이 이미 특정 태스크에 적응된 SAM에 AuxOL을 적용했을 때도 추가적인 성능 향상이 있었다. 특히 SAM의 파라미터를 직접 업데이트하는 DirectOL 방식보다 AuxOL이 더 안정적이고 우수한 성능을 보였는데, 이는 직접 업데이트 시 발생하는 모델 붕괴 문제를 보조 모델이 해결해주기 때문이다.

## 🧠 Insights & Discussion

### 강점 및 분석
- **안정적인 학습 구조**: 거대 모델인 SAM을 고정하고 작은 보조 모델만 학습시킴으로써, 온라인 학습의 고질적인 문제인 파라미터 불안정성과 계산 비용 문제를 동시에 해결하였다.
- **상호보완적 역할의 변화**: 적응적 융합 파라미터 $\alpha$의 변화를 분석한 결과, 학습 초기에는 $\alpha$가 1에 가까워 SAM(Generalist)이 주도적인 역할을 하지만, 보조 모델이 태스크 특화 지식을 습득함에 따라 $\alpha$가 0으로 수렴하며 보조 모델(Specialist)의 비중이 높아지는 양상을 보였다. 이는 일반적 지식에서 특화 지식으로의 부드러운 전이를 의미한다.
- **실용적 가치**: 전문가가 매번 모든 픽셀을 수정할 필요 없이, 모델이 실시간으로 학습하여 수정량을 줄여줌으로써 의료 전문가의 업무 부담을 경감시킬 수 있는 잠재력을 가진다.

### 한계 및 논의사항
- 본 논문은 주로 지도 학습(Supervised) 설정에서의 온라인 학습을 다루었으나, 실제 환경에서는 전문가의 피드백을 얻는 비용이 여전히 높다. 논문에서 언급된 pseudo-labeling이나 self-supervised learning의 적용 가능성이 향후 연구의 핵심이 될 것으로 보인다.
- 보조 모델의 아키텍처를 U-Net으로 단순하게 설정하였으나, 더 효율적인 경량 구조나 어댑터 구조를 사용할 경우의 성능 변화에 대한 분석은 부족하다.

## 📌 TL;DR

본 논문은 거대 모델인 SAM의 성능을 실시간으로 개선하기 위해, 작은 보조 모델을 함께 운용하며 전문가의 수정을 통해 실시간 학습하는 **AuxOL** 프레임워크를 제안한다. 이 방법은 SAM의 범용성을 유지하면서도 의료 영상 특화 지식을 효율적으로 학습하며, 적응적 융합 기법을 통해 최적의 분할 결과를 도출한다. 이는 막대한 비용이 드는 오프라인 재학습 없이도 테스트 단계에서 모델을 최적화할 수 있는 실용적인 경로를 제시하며, 향후 의료 AI 시스템의 상호작용형 학습 모델로 활용될 가능성이 높다.
