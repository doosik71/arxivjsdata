# EVALUATING UTILITY OF MEMORY EFFICIENT MEDICAL IMAGE GENERATION: A STUDY ON LUNG NODULE SEGMENTATION

Kathrin Khadra, Utku Türkbey (2024)

## 🧩 Problem to Solve

본 연구는 의료 영상 분야에서 딥러닝 모델 개발의 핵심 제약 사항인 고품질 의료 데이터의 희소성 문제를 해결하고자 한다. 의료 데이터는 환자의 개인정보 보호법, 데이터 품질의 불균일성, 그리고 전반적인 데이터 부족으로 인해 확보가 매우 어렵다. 이러한 문제를 해결하기 위해 합성 데이터(Synthetic Data)를 생성하여 학습에 활용하는 방안이 제시되고 있으나, 특히 3D 의료 영상의 경우 높은 해상도로 인해 생성 모델 학습 시 막대한 메모리 자원이 소모된다는 기술적 한계가 존재한다.

따라서 본 논문의 목표는 메모리 효율적인 Patch-wise Denoising Diffusion Probabilistic Model (DDPM)을 통해 폐 결절(Lung Nodule)이 포함된 CT 영상을 생성하고, 이렇게 생성된 합성 데이터가 실제 폐 결절 분할(Segmentation) 모델의 성능 향상에 실질적인 유용성(Utility)을 제공하는지 검증하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 다음과 같다.

1. **메모리 효율적인 이미지 생성 구조 제안**: 기존의 Patch-wise 접근 방식을 이미지 생성에 맞게 수정하여, 전체 영상을 한 번에 처리하지 않고 패치 단위로 학습함으로써 메모리 제약을 극복하고 고해상도 CT 영상을 생성하였다.
2. **조건부 생성 메커니즘**: 생성 모델의 입력으로 Segmentation Mask와 좌표(Coordinates) 정보를 조건(Condition)으로 제공하여, 사용자가 원하는 위치와 형태의 결절을 정확하게 삽입할 수 있도록 설계하였다.
3. **합성 데이터의 유용성에 대한 심층 분석**: 단순히 이미지의 외관이 실제와 유사한지를 평가하는 것을 넘어, 합성 데이터만으로 학습한 모델과 실제 데이터에 합성 데이터를 추가하여 학습한 모델의 성능을 비교함으로써 하위 작업(Downstream Task)에서의 실질적인 효용성을 입증하였다.

## 📎 Related Works

기존의 의료 영상 생성 연구는 주로 Generative Adversarial Networks (GANs)와 Variational Autoencoders (VAEs)를 중심으로 이루어졌다. 하지만 이러한 모델들은 학습의 불안정성이나 생성된 이미지의 다양성 부족이라는 한계가 있었다. 최근에는 Denoising Diffusion Probabilistic Models (DDPM)과 Latent Diffusion Models (LDM)가 고품질 이미지 생성으로 주목받고 있다. 특히 LDM은 압축된 잠재 공간(Latent Space)에서 작동하여 메모리를 절약하지만, 본 논문은 픽셀 공간에서의 효율성을 높이기 위해 Patch-wise 접근 방식을 채택하였다.

기존의 Patch-wise DDPM 연구(Bieder et al. [20])는 주로 Segmentation Mask를 생성하는 데 집중했으나, 본 연구는 이를 반전시켜 CT 이미지 자체를 생성하는 데 적용했다는 점에서 차별점을 가진다. 또한, 많은 확산 모델 연구들이 이미지 품질 지표에만 집중하는 것과 달리, 본 연구는 실제 분할 작업에서의 성능 향상을 정량적으로 분석하였다.

## 🛠️ Methodology

### 1. 전체 파이프라인 및 시스템 구조
본 연구는 LIDC-IDRI 데이터셋에서 3mm 이상의 폐 결절 데이터를 추출하여 사용한다. 생성 모델인 DDPM을 통해 합성 CT 영상을 생성하고, 이를 nnU-Net 기반의 분할 파이프라인에 입력하여 성능을 평가하는 구조를 가진다.

### 2. DDPM의 기본 원리
DDPM은 가우시안 노이즈를 점진적으로 추가하는 순방향 과정(Forward Process)과 이를 다시 제거하는 역방향 과정(Reverse Process)으로 구성된다.

- **순방향 확산 과정 (Forward Diffusion Process)**:
  원본 데이터 $x_0$에 $T$ 단계에 걸쳐 가우시안 노이즈를 추가하여 최종적으로 순수 노이즈 상태로 만드는 과정이다. 각 단계의 전이는 다음과 같이 정의된다.
  $$q(x_t|x_{t-1}) = \mathcal{N}(x_t; \sqrt{\alpha_t}x_{t-1}, (1-\alpha_t)I)$$
  여기서 $\alpha_t$는 시간 가변적 스케일링 인자이다.

- **역방향 확산 과정 (Reverse Diffusion Process)**:
  학습된 신경망 $\theta$를 통해 노이즈가 섞인 데이터 $x_t$로부터 $x_{t-1}$을 복원하는 과정이다.
  $$p_\theta(x_{t-1}|x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))$$
  학습 목표는 예측된 노이즈 $\epsilon_\theta$와 실제 추가된 노이즈 $\epsilon$ 사이의 평균 제곱 오차(MSE)를 최소화하는 것이다.
  $$L = \mathbb{E}_{t, x_0, \epsilon} [\|\epsilon - \epsilon_\theta(x_t, t)\|^2]$$

### 3. 메모리 효율적 이미지 생성 (Memory Efficient Generation)
메모리 문제를 해결하기 위해 본 연구는 다음과 같은 전략을 사용한다.
- **Patch-wise Training**: 전체 영상 대신 무작위로 크롭된 패치(Patch)를 학습하여 공간적 차원과 메모리 요구량을 줄인다.
- **Conditioning**: 패치 단위 학습 시 모델이 현재 위치를 알 수 있도록 **좌표(Coordinates)** 정보를 제공하며, 결절의 위치와 형태를 제어하기 위해 **Segmentation Mask**를 조건으로 입력한다.
- **Generation**: 학습은 패치 단위로 수행하지만, 추론(Sampling) 시에는 전체 좌표계와 전체 Segmentation Mask를 입력하여 전체 CT 영상을 한 번에 생성한다.

## 📊 Results

### 1. 실험 설정
- **데이터셋**: LIDC-IDRI (훈련 553장, 검증 142장, 테스트 138장).
- **분할 모델**: nnU-Net (자가 설정 파이프라인).
- **평가 지표**: Dice Similarity Coefficient (DSC).
  $$DSC = \frac{2TP}{2TP + FP + FN}$$

### 2. 주요 결과
실험은 세 가지 시나리오(실제 데이터만 학습, 합성 데이터만 학습, 실제+타겟팅된 합성 데이터 학습)로 진행되었으며, 결과는 다음과 같다.

| 학습 데이터 구성 | 평균 테스트 Dice Score (DSC) | p-Value |
| :--- | :---: | :---: |
| Real (Benchmark) | $0.4913 \pm 0.02733$ | - |
| Synthetic (General) | $0.5016 \pm 0.0206$ | - |
| Real + Synthetic (Targeted) | $\mathbf{0.5418 \pm 0.03015}$ | $0.024335$ |

- **일반 유용성 테스트 (General Utility)**: 합성 데이터만으로 학습한 모델의 DSC(0.5016)가 실제 데이터 학습 모델(0.4913)과 매우 유사하게 나타났다. 이는 합성 데이터가 실제 데이터와 유사한 유용한 정보를 포함하고 있음을 시사한다.
- **타겟 유용성 테스트 (Targeted Utility)**: 벤치마크 모델이 가장 낮은 성능을 보인 59개의 검증 샘플(예: 흉막에 인접한 결절)의 마스크를 이용해 합성 데이터를 생성하고 이를 추가 학습시켰을 때, DSC가 0.5418로 통계적으로 유의미하게($p < 0.05$) 향상되었다.

## 🧠 Insights & Discussion

### 1. 강점 및 성과
본 연구는 단순히 "그럴듯한" 이미지를 만드는 것을 넘어, 특정 엣지 케이스(Edge Cases)를 타겟팅하여 합성 데이터를 생성함으로써 모델의 약점을 보완할 수 있음을 입증하였다. 특히 흉막(Pleura)에 붙어 있거나 여러 개의 결절이 밀집된 어려운 케이스에서 분할 성능이 개선된 점은 의료 영상 분야에서 합성 데이터의 실질적인 활용 가치를 보여준다.

### 2. 한계 및 비판적 해석
- **과분할(Over-segmentation) 문제**: 합성 데이터를 추가한 후 일부 이미지에서 결절이 없는 곳을 결절로 인식하는 과분할 현상이 관찰되었다. 이는 모델이 합성 데이터의 특정 패턴을 과도하게 학습하여 발생한 것으로 보인다.
- **생성 전략의 정교함 부족**: 단순히 마스크를 조건으로 주는 것 외에, 해부학적 구조의 복잡성을 더 정교하게 반영할 수 있는 생성 전략이 필요하다.
- **성능 지표의 절대적 수치**: DSC 수치 자체가 0.5 수준으로 낮은 편인데, 이는 폐 결절 분할 작업 자체가 매우 어렵기 때문이라고 논문은 설명한다. 하지만 더 높은 기준선(Baseline)을 확보하기 위한 추가적인 최적화가 필요해 보인다.

## 📌 TL;DR

본 논문은 메모리 효율적인 Patch-wise DDPM을 제안하여, 고해상도 3D 폐 CT 영상을 성공적으로 합성하였다. 특히 합성 데이터만으로 학습한 모델이 실제 데이터 학습 모델과 유사한 성능을 보였으며, 모델이 취약한 특정 케이스를 타겟팅하여 데이터를 증강했을 때 분할 성능이 유의미하게 향상됨을 확인하였다. 이 연구는 의료 데이터 부족 문제를 해결하기 위해 확산 모델을 이용한 정교한 데이터 증강 전략이 유효함을 입증하며, 향후 맞춤형 의료 데이터 생성 연구에 중요한 기초를 제공한다.
