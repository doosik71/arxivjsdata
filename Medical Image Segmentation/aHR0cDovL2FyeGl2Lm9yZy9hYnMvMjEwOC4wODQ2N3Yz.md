# Medical Image Segmentation with 3D Convolutional Neural Networks: A Survey

S Niyas, S J Pawan, M Anand Kumar, Jeny Rajan (2022)

## 🧩 Problem to Solve

본 논문은 의료 영상 분석, 특히 의료 영상 분할(Medical Image Segmentation) 분야에서 3D Convolutional Neural Networks(CNN)의 활용 현황을 분석하고 체계적으로 정리하는 것을 목표로 한다. 

의료 영상 데이터(MRI, CT, 초음파 등)는 본질적으로 3차원 볼륨 데이터이지만, 기존의 많은 연구들이 계산 복잡도와 메모리 제약으로 인해 2D CNN 모델을 적용해 왔다. 그러나 2D 접근 방식은 인접한 슬라이스 간의 관계, 즉 슬라이스 간 정보(inter-slice information)를 학습할 수 없다는 치명적인 한계가 있다. 또한, 전통적인 머신러닝 방식은 도메인 전문가의 수동 특징 추출(handcrafted features)에 의존해야 하며, 이는 주관성이 개입될 여지가 크고 일반화 성능이 떨어진다는 문제가 있다.

따라서 본 논문은 3D 볼륨 데이터의 공간적 특징을 완전히 활용할 수 있는 3D 딥러닝 방법론들을 조사하고, 현재의 기술적 성취와 한계점, 그리고 향후 연구 방향을 제시하고자 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 다음과 같이 요약할 수 있다.

1.  **3D 딥러닝 기반 의료 영상 분할의 체계적 분류**: 단순히 논문을 나열하는 것이 아니라, 학습 방식과 구조에 따라 3D CNN 모델을 다섯 가지 주요 카테고리(완전 지도 학습, 준지도 학습, 약지도 학습, 비용 효율적 근사 모델, 다중 작업 학습)로 분류하여 분석하였다.
2.  **2D vs 3D CNN의 심층 비교**: 3D CNN이 왜 의료 영상 분할에 필수적인지를 이론적, 실험적 근거(계산 복잡도, 메모리 요구량, 슬라이스 간 정보 추출 능력 등)를 통해 설명하였다.
3.  **최신 연구 동향 분석 및 연구 공백 제시**: 2015년부터 2021년까지의 최신 연구들을 분석하여, 데이터 부족 문제 해결을 위한 준지도/약지도 학습의 필요성과 클래스 불균형(Class Imbalance) 문제 등의 해결 과제를 명시하였다.

## 📎 Related Works

논문은 기존의 의료 영상 분석 리뷰 논문들이 주로 특정 응용 분야(Application-specific)에 집중되어 있거나, 2D CNN 위주로 다루어지고 있음을 지적한다. 

기존 접근 방식의 한계는 다음과 같다.
- **전통적 머신러닝**: 전문가의 수동 특징 추출 의존성, 수동 개입의 불가피함, 임의적인 파라미터 설정 문제.
- **2D CNN 기반 분할**: 3D 데이터를 2D 슬라이스로 나누어 처리함으로써 볼륨 데이터가 가진 입체적인 공간 정보와 슬라이스 간 상관관계를 상실함.

본 논문은 이러한 한계를 극복하기 위해 3D 커널을 사용하는 3D CNN에 집중하며, 특히 Encoder-Decoder 구조의 FCNN(Fully Convolutional Neural Networks) 및 U-Net 계열의 발전 방향을 중점적으로 다룬다.

## 🛠️ Methodology

본 논문은 서베이 논문으로서 3D CNN의 기본 원리와 분석 대상이 된 모델들의 분류 체계를 방법론으로 제시한다.

### 1. 3D CNN의 기본 원리
3D CNN은 2D CNN의 확장으로, 커널이 $x, y$ 축뿐만 아니라 $z$ 축(깊이) 방향으로도 이동하며 특징을 추출한다. 

**수학적 표현:**
$n$번째 레이어에서 $k$번째 커널에 의해 생성되는 특징 맵(Feature Map) $X^k_n$은 다음과 같이 정의된다.
$$X^k_n = \sigma(w^k_n * X_{n-1} + b^k_n), \quad k=\{1,2,...,K\}$$
여기서 $w^k_n$은 $n$번째 레이어의 $k$번째 3D 커널, $b^k_n$은 바이어스, $\sigma(\cdot)$는 활성화 함수를 의미한다. 

3D CNN의 특징 맵은 (가로, 세로, 깊이, 채널)의 4차원 형태를 띠며, 이를 통해 볼륨 데이터 내의 공간적 특징을 모든 방향에서 학습할 수 있다.

### 2. 3D CNN 모델의 분류 체계
논문은 3D 의료 영상 분할 모델을 다음과 같이 다섯 가지 범주로 나누어 분석한다.

- **Fully Supervised 3D CNN**: 완전히 라벨링된 3D 데이터를 사용한다.
    - **Direct 3D CNN**: 2D 구조를 3D로 직접 확장 (예: V-Net, 3D U-Net).
    - **Patch-wise Segmentation**: 메모리 절약을 위해 이미지를 작은 3D 패치로 나누어 처리 (예: DeepMedic).
    - **Multi-task Learning**: 분할과 분류 등 여러 작업을 동시에 학습하여 일반화 성능을 높임.
- **Semi-supervised Learning (SSL)**: 소량의 라벨링 데이터와 대량의 미라벨링 데이터를 함께 사용 (예: Teacher-Student 구조, GAN 기반 합성).
- **Weakly-supervised Learning**: 이미지 레벨 라벨이나 바운딩 박스 등 정밀도가 낮은 라벨을 사용.
- **Cost-effective Approximations**: 3D CNN의 높은 비용을 줄이기 위해 2D 평면 3개를 융합하거나, Separable 3D Convolution을 사용하는 방식.

## 🧠 Insights & Discussion

### 1. 3D CNN의 강점 및 한계
- **강점**: 3D CNN은 볼륨 데이터의 입체적 문맥(Context)과 슬라이스 간 정보를 직접 학습할 수 있어, 특히 병변이 여러 슬라이스에 걸쳐 나타나는 의료 영상에서 2D 방식보다 월등한 성능을 보인다.
- **한계**: 
    - **데이터 부족**: 3D 볼륨 데이터의 슬라이스별 정밀 라벨링은 막대한 시간과 비용이 소요되어 대규모 데이터셋 구축이 어렵다.
    - **계산 자원**: 높은 GPU 메모리 요구량으로 인해 하드웨어 제약이 크다.
    - **클래스 불균형**: 정상 영역에 비해 병변 영역(Voxel share)이 매우 적어 학습 시 과적합이나 오분할이 발생하기 쉽다.

### 2. 비판적 해석 및 제언
대부분의 3D CNN 연구가 기존 2D 모델의 커널을 단순히 3D로 확장하는 방식에 머물러 있다. 이는 구현은 쉽지만 3D 데이터만이 가진 고유한 특성을 완전히 활용하는 최적화된 구조라고 보기 어렵다. 따라서 향후에는 3D 데이터의 기하학적 특성을 반영한 새로운 아키텍처 설계가 필요하다. 또한, 데이터 부족 문제를 해결하기 위해 GAN을 이용한 고품질 3D 합성 데이터 생성 연구가 매우 유망할 것으로 판단된다.

## 📌 TL;DR

본 논문은 3D 의료 영상 분할을 위한 3D CNN 방법론들을 체계적으로 분석한 서베이 보고서이다. 3D CNN이 슬라이스 간 정보를 추출하여 2D 방식의 한계를 극복함을 보였으며, 모델들을 지도/준지도/약지도 학습 및 효율적 근사 모델로 분류하여 정리하였다. 결론적으로 3D CNN은 높은 메모리 비용과 데이터 확보의 어려움이라는 과제가 있으나, 준지도 학습과 효율적인 커널 설계(Separable 3D 등)를 통해 이를 극복하는 추세이며, 향후 3D 데이터 특화 아키텍처와 합성 데이터 생성 연구가 핵심이 될 것임을 시사한다.
