# IOP-FL: Inside-Outside Personalization for Federated Medical Image Segmentation

Meirui Jiang, Hongzheng Yang, Chen Cheng, Qi Dou (2023)

## 🧩 Problem to Solve

본 논문은 연합 학습(Federated Learning, FL) 기반의 의료 영상 분할(Medical Image Segmentation)에서 발생하는 데이터 이질성(Data Heterogeneity) 문제를 해결하고자 한다. 

의료 영상 데이터는 사용하는 스캐너의 종류나 환자의 인구통계학적 특성에 따라 분포가 매우 다양하며, 이로 인해 모든 클라이언트에게 최적인 단일 글로벌 모델(Global Model)을 구축하는 것은 매우 어렵다. 특히, 연합 학습 과정에 참여하지 않은 새로운 외부 클라이언트(Outside Client)에게 글로벌 모델을 배포할 경우, 학습 시 보지 못한 데이터 분포(Unseen Distribution)로 인해 성능 저하가 더욱 심각하게 나타나는 문제가 발생한다.

따라서 본 연구의 목표는 연합 학습 내부의 참여 클라이언트(Inside Clients)와 외부의 미참여 클라이언트(Outside Clients) 모두를 위해 모델을 최적화할 수 있는 통합된 개인화 프레임워크인 **IOP-FL**을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 연합 학습을 통해 얻은 다양한 로컬 모델들이 서로 다른 데이터 분포의 특성을 담고 있다는 점에 착안하여, 이를 하나의 **라우팅 공간(Routing Space)**으로 활용하는 것이다.

1.  **Inside Personalization**: 글로벌 그래디언트(Global Gradient)와 로컬 그래디언트(Local Gradient)를 누적하여 결합함으로써, 공통 지식을 유지하면서도 각 클라이언트의 특성에 최적화된 로컬 적응 모델(Local Adapted Model)을 생성한다.
2.  **Outside Personalization**: 내부 클라이언트들이 학습한 다양한 개인화 모델들과 글로벌 모델을 결합하여 풍부한 파라미터 공간을 구축한다. 외부 클라이언트가 테스트 데이터를 입력하면, 이 라우팅 공간 내에서 최적의 모델 조합을 동적으로 찾는 **테스트 시간 라우팅(Test-time Routing)** 기법을 통해 개인화된 모델을 생성한다.

## 📎 Related Works

기존의 개인화 연합 학습(Personalized FL) 연구들은 주로 내부 클라이언트에 집중해 왔다.
- **내부 개인화 방식**: Multi-task learning, Meta-learning, 또는 FedRep나 FedBN처럼 네트워크의 일부(예: Classification head, BN layer)만 개인화하는 방식이 제안되었다. 그러나 이러한 방식은 공유 파라미터와 개인화 파라미터를 명확히 구분하기 어렵고, 학습 단계에서만 수행되므로 배포 단계의 외부 클라이언트에게는 적용하기 어렵다는 한계가 있다.
- **외부 일반화 및 적응 방식**: FedDG와 같이 도메인 일반화(Domain Generalization)를 통해 글로벌 모델의 범용성을 높이거나, TTT(Test-Time Training)나 Tent와 같은 테스트 시간 적응(Test-time Adaptation) 기법이 사용되었다. 하지만 TTT나 Tent는 단일 모델을 대상으로 하므로, 연합 학습이 제공하는 다양한 개인화 모델들의 잠재력을 충분히 활용하지 못한다.

IOP-FL은 내부와 외부 개인화를 하나의 프레임워크로 통합하며, 특히 외부 개인화 시 단일 모델이 아닌 내부의 다양한 모델 집합을 기반으로 적응한다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

### 1. 전체 시스템 구조
IOP-FL은 크게 두 단계로 구성된다. 먼저 연합 학습 과정에서 각 내부 클라이언트를 위한 개인화 모델을 학습하고, 이후 이 모델들을 저장소(Routing Space)로 삼아 외부 클라이언트가 테스트 시점에 자신에게 맞는 모델을 조합해 사용하는 구조이다.

### 2. Inside Personalization: Local Adapted Model
내부 클라이언트는 글로벌 지식과 로컬 특성을 모두 반영하기 위해 다음과 같은 방식으로 모델을 업데이트한다. 

전통적인 방식에서는 글로벌 모델과 로컬 모델 사이의 유사도 $\text{sim}(w_k, \bar{w})$를 제약 조건으로 두어 최적화하지만, 본 논문은 하이퍼파라미터 $\lambda$에 민감한 이 방식 대신 그래디언트를 직접 혼합하는 방식을 제안한다. 로컬 적응 모델 $P_k$의 업데이트 식은 다음과 같다.

$$P_{t+1}^k = (1-\tau)P_t^k - \tau (\eta_l \nabla^k L + \eta_g \nabla^G)$$

여기서 $\tau$는 과거의 정보를 유지하며 학습을 안정화하는 누적률(Accumulation rate)이며, $\eta_l \nabla^k L$은 로컬 그래디언트, $\eta_g \nabla^G$는 서버에서 집계된 글로벌 그래디언트를 의미한다. 이를 통해 명시적인 유사도 제약 없이도 공통 패턴과 로컬 분포를 동시에 학습할 수 있다.

### 3. Outside Personalization: Test-time Routing
외부 클라이언트를 위한 개인화는 학습된 내부 모델들 $\{P_1, \dots, P_K\}$과 글로벌 모델 $w_g$를 포함하는 **라우팅 공간 $\tilde{W}$**에서 최적의 가중치 조합을 찾는 과정이다.

**파라미터 라우팅 (Parameter Routing):**
단순한 모델 앙상블(Output ensemble)이 아니라 레이어 수준(Layer-wise)에서 가중치를 조합한다. $l$번째 레이어의 최종 가중치 $W^o_l$은 다음과 같이 계산된다.

$$W^o_l \times x^o_i = \left( \sum_{k=1}^{K+1} r^l_k W^l_k \right) \times x^o_i$$

여기서 $r^l_k$는 라우팅 행렬 $R$의 요소로, 입력 데이터의 특성에 따라 동적으로 결정되는 재가중치 계수이다. 이 계수는 입력 피처 맵을 입력으로 받는 작은 선형 네트워크 $\theta$를 통해 계산된다.

**테스트 시간 손실 함수 (Test-time Loss):**
외부 클라이언트는 라벨이 없는 데이터를 사용하므로, 비지도 학습 기반의 손실 함수 $L_t$를 최소화하도록 $R$을 최적화한다.

$$L_t = L_{cons}(x^o_i, x^o_i + \epsilon; R) + \beta(L_s(x^o_i; R) + L_e(x^o_i; R))$$

- $L_{cons}$ (Consistency Loss): 입력 이미지에 가우시안 노이즈 $\epsilon$을 추가했을 때, 모델의 예측 결과가 일관되게 유지되도록 강제한다.
- $L_s$ (Shape Constraint): 분할 맵의 경계가 매끄럽고 일정한 형태를 유지하도록 하는 형태 제약 조건이다.
- $L_e$ (Entropy Loss): 예측의 불확실성을 줄이기 위해 엔트로피를 최소화한다.

## 📊 Results

### 실험 설정
- **데이터셋**: 전립선 MRI 분할(6개 소스) 및 망막 저저 이미지 분할(4개 기관의 Optic Disc 및 Cup 분할).
- **평가 지표**: Dice Score.
- **비교 대상**: 
    - Inside: FedAvg, FedRep, FedBN, pFedMe, Single-site training.
    - Outside: FedAvg, FedDG, TTT, Tent, Average/Ensemble.

### 주요 결과
1.  **내부 개인화 성능**: IOP-FL은 모든 내부 클라이언트에서 FedAvg 대비 성능 향상을 보였으며, 특히 전립선 분할에서 평균 Dice 점수를 2.63% 높였다. FedRep나 FedBN 같은 최신 SOTA 방법보다도 전반적으로 우수한 성능을 기록했다.
2.  **외부 개인화 성능**: 외부 클라이언트에 대해 TTT, Tent, FedDG보다 월등한 성능을 보였다. 단순 앙상블(Ensemble) 방식보다 전립선 분할에서 Dice 점수가 5.26% 향상되었다. 이는 다양한 내부 모델들로 구성된 라우팅 공간이 외부 데이터의 분포에 유연하게 적응할 수 있음을 시사한다.
3.  **소요 시간**: 테스트 시간 라우팅을 통한 적응 과정은 샘플당 1초 미만의 짧은 시간 내에 성능이 빠르게 수렴하고 안정화됨을 확인하였다.

## 🧠 Insights & Discussion

### 강점 및 가치
본 연구는 연합 학습이 단순히 하나의 글로벌 모델을 만드는 도구가 아니라, 다양한 로컬 모델의 집합을 생성하여 이를 외부 클라이언트의 적응을 위한 '지식 베이스'로 활용할 수 있다는 새로운 관점을 제시했다. 특히 비지도 학습 기반의 테스트 시간 라우팅을 통해 라벨 없이도 외부 도메인에 빠르게 적응할 수 있다는 점이 임상 적용 가능성을 높인다.

### 한계 및 비판적 해석
1.  **통신 및 계산 비용**: 외부 개인화를 위해 모든 내부 모델 파라미터를 전송해야 하므로, 참여 클라이언트 수($K$)가 많아질수록 통신 비용과 라우팅 행렬 최적화를 위한 계산량이 증가하는 문제가 있다.
2.  **형태 제약의 일반성**: 본 논문에서 사용한 $L_s$ (Shape Constraint)는 타원형에 가까운 장기 구조에 최적화되어 설계되었다. 만약 분할 대상이 매우 불규칙한 형태를 가진다면, 이 제약 조건이 오히려 잘못된 가이드를 제공하여 성능을 저하시킬 가능성이 있다.
3.  **프라이버시**: 모델 파라미터 자체를 공유하는 것이 모델 인버전 공격(Model Inversion Attack)에 취약할 수 있다는 우려가 있으나, 논문 내 실험을 통해 그래디언트 정보 없이는 원본 이미지 복구가 어렵다는 점을 보여주어 어느 정도 방어 가능함을 입증하였다.

## 📌 TL;DR

IOP-FL은 연합 학습의 내부 클라이언트를 위한 **그래디언트 기반 로컬 적응 모델**과, 외부 클라이언트를 위한 **테스트 시간 파라미터 라우팅**을 통합한 프레임워크이다. 내부 모델들의 다양성을 라우팅 공간으로 활용하여, 라벨이 없는 외부 데이터에서도 최적의 모델 조합을 찾아내어 높은 분할 정확도를 달성한다. 이 연구는 데이터 이질성이 심한 의료 영상 분야에서 FL 모델의 실제 배포 및 적용 가능성을 크게 확장시킨 연구라고 평가할 수 있다.
