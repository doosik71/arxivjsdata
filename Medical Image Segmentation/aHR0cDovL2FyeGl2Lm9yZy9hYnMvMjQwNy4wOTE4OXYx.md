# Segmenting Medical Images with Limited Data

Zhaoshan Liu, Qiujie Lv, Chau Hung Lee, Lei Shen (2024)

## 🧩 Problem to Solve

본 연구가 해결하고자 하는 핵심 문제는 의료 영상 분할(Medical Image Segmentation) 분야에서 발생하는 **레이블링된 데이터의 부족 문제**이다. 의료 영상 데이터는 수집 과정에서 전문 장비와 전문가의 개입이 필요하여 시간과 비용이 많이 소모될 뿐만 아니라, 환자의 개인정보 보호 문제로 인해 대규모의 공개 데이터셋을 확보하기가 매우 어렵다.

이러한 데이터 부족은 딥러닝 모델의 일반화 성능을 저하시키는 주요 원인이 된다. 기존의 데이터 증강(Data Augmentation, DA)이나 준지도 학습(Semi-Supervised Learning, SSL) 방법들이 제안되었으나, 여전히 계산 비용이 높거나 모델의 변동성(Variability) 및 학습 안정성 측면에서 한계가 존재한다. 따라서 본 논문의 목표는 제한된 레이블 데이터와 다량의 레이블 없는 데이터를 효율적으로 활용하여, 특히 **데이터가 극도로 부족한 상황에서도 높은 분할 성능을 유지하는 데이터 효율적인 의료 영상 분할 모델인 DEMS(Data-Efficient Medical Segmenter)를 개발하는 것**이다.

## ✨ Key Contributions

DEMS의 핵심 아이디어는 **입력 데이터의 다양성(Input Diversity)과 특징의 다양성(Feature Diversity)을 동시에 극대화하여 모델의 일반화 능력을 높이고, 다중 디코더 간의 일관성을 강제하여 학습을 안정화하는 것**이다.

이를 위해 다음과 같은 세 가지 핵심 설계를 도입하였다:
1. **Online Automatic Augmenter (OAA)**: 학습 과정에서 실시간으로 다양한 이미지 변환을 적용하여 입력 데이터의 다양성을 확보한다.
2. **Residual Robustness Enhancement (RRE) block**: 인코더와 디코더 사이에 배치되어 특징 다양성을 풍부하게 하고, 각 디코더에 서로 다른 섭동(Perturbation)을 주입하여 모델의 변동성을 높인다.
3. **Sensitivity Loss**: 메인 디코더와 보조 디코더 간의 예측 결과 차이를 XOR 연산 기반으로 계산하여, 일관성을 높이고 학습 과정을 안정화한다.

## 📎 Related Works

논문에서는 기존의 접근 방식을 크게 데이터 증강(DA)과 준지도 학습(SSL)으로 나누어 설명한다.

**1. 데이터 증강 (Data Augmentation)**
- **Conventional DA**: 회전, 대칭, 밝기 조절 등 단순 변환을 사용하며 직관적이지만, 파이프라인 설계가 설계자의 경험에 크게 의존한다.
- **GAN-based DA**: 가상 이미지를 생성하여 데이터셋을 확장하지만, 생성된 이미지의 품질이 일정하지 않고 학습 시간이 매우 길며 데이터 요구량이 많다.
- **Automatic DA**: 강화학습 등을 통해 최적의 증강 전략을 탐색하지만, 계산 비용이 높고 절차가 복잡하며 모델에 혼란을 줄 가능성이 있다.

**2. 준지도 학습 (Semi-Supervised Learning)**
- **Mean Teacher (MT) 기반 방식**: Teacher-Student 구조를 통해 일관성 규제(Consistency Regularization)를 적용한다. 하지만 Teacher 네트워크의 성능에 종속되며 구조적 공유로 인해 모델의 변동성이 제한된다는 한계가 있다.
- **기타 구조 (Encoder-Decoder, GAN 등)**: 정교한 네트워크 및 손실 함수 설계를 통해 성능을 높였으나, 여전히 변동성 제한과 학습 안정성 문제가 존재한다.

DEMS는 이러한 한계를 극복하기 위해 별도의 사전 학습 단계가 없는 **Training-free OAA**와 특징 수준에서 변동성을 주는 **RRE 블록**, 그리고 안정성을 위한 **Sensitivity Loss**를 결합하여 차별성을 둔다.

## 🛠️ Methodology

### 전체 시스템 구조
DEMS는 기본적으로 U-Net 아키텍처를 기반으로 하는 Encoder-Decoder 구조를 따른다. 하나의 인코더($E$)와 하나의 메인 디코더($D_m$), 그리고 세 개의 보조 디코더($D_{a1}, D_{a2}, D_{a3}$)로 구성된다. 보조 디코더들은 오직 학습 단계에서만 사용되며, 추론 시에는 제거된다.

### 주요 구성 요소 및 역할
1. **Online Automatic Augmenter (OAA)**
   - MedAugment의 오프라인 방식을 온라인 방식으로 변경한 모듈이다.
   - 픽셀 증강 공간($\mathcal{A}_p$: 밝기, 대비, 가우시안 노이즈 등)과 공간 증강 공간($\mathcal{A}_s$: 회전, 플립, 스케일링 등)에서 변환 조합을 샘플링하여 매 에포크마다 입력 데이터에 적용한다.
   - 중복 샘플링을 허용하여 데이터 다양성을 더욱 극대화하며, 기존 DA 파이프라인에 쉽게 통합 가능하다.

2. **Residual Robustness Enhancement (RRE) Block**
   - 인코더의 출력 특징을 받아 각 디코더로 전달하기 전 특징을 가공하는 블록이다.
   - **구성**: Residual Connection, Depthwise Convolution (DwConv), Pointwise Convolution (PwConv), 그리고 **Feature Perturbation Injection (FPI)** 블록으로 이루어져 있다.
   - FPI는 특징 노이즈, 특징 드롭아웃, 일반 드롭아웃을 주입하여 각 디코더가 서로 다른 변형된 특징을 입력받게 함으로써 모델의 변동성을 강제로 높인다.

### 손실 함수 및 학습 절차
학습은 레이블 데이터에 대한 손실과 레이블 없는 데이터에 대한 손실을 동시에 최적화한다.

**1. Fusion Loss ($L_f$)**
레이블 데이터에 대해 픽셀 수준의 정확도를 측정하는 Binary Cross Entropy ($L_{BCE}$)와 객체 수준의 겹침 정도를 측정하는 Dice Loss ($L_{DSC}$)를 결합한다.
$$L_f = 0.5 \cdot L_{BCE}(\hat{y}, y) + L_{DSC}(\hat{y}, y)$$

**2. Sensitivity Loss ($L_s$)**
메인 디코더($\hat{y}_m$)와 보조 디코더($\hat{y}_a$)의 이진화된 예측 결과가 서로 다를 때 발생하는 '민감도 영역'을 줄이는 것이 목적이다. XOR 연산을 통해 두 예측의 불일치 정도를 계산한다. (여기서 $\oplus$는 XOR 연산, $T$는 임계값 $0.5$이다.)
$$L_s = \frac{1}{N} \sum_{i=1}^{N} (\hat{y}_{m,i} > T) \oplus (\hat{y}_{a,i} > T)$$

**3. Unsupervised Loss ($L_u$)**
레이블 없는 데이터에 대해 메인 디코더와 보조 디코더 간의 예측값 차이를 Mean Squared Error (MSE)로 계산하여 일관성을 부여한다.
$$L_u = \sum_{i=1}^{N} (\hat{y}_{m,i} - \hat{y}_{a,i})^2$$

**4. 전체 손실 함수 및 Warm-up**
초기 학습 단계에서 $L_s$와 $L_u$가 학습을 불안정하게 만드는 것을 방지하기 위해 가우시안 웜업 함수 $\lambda(t, t_{max})$를 적용한다.
$$L = \bar{L}_f + \lambda(\bar{L}_s + \bar{L}_u)$$
여기서 $\lambda$는 학습 스텝 $t$가 진행됨에 따라 $0$에서 $1$로 점진적으로 증가한다.

## 📊 Results

### 실험 설정
- **데이터셋**: 자체 수집한 SNGT(위 비강 튜브) 데이터셋과 공개 데이터셋인 BUS(유방 초음파), BUSI(유방 초음파), DDTI(갑상선 초음파) 총 4종을 사용하였다.
- **데이터 구성**: 레이블 데이터의 $20\%$와 $40\%$만 사용하여 준지도 학습 성능을 평가하였다.
- **평가 지표**: Dice Score (DSC), IoU, Sensitivity (SEN), Precision (PRE), Pixel Accuracy (PA).

### 주요 결과
1. **정량적 성능**:
   - SNGT 데이터셋(레이블 $20\%$ 사용)에서 DEMS는 **DSC 54.59%**를 달성하여, U-Net 대비 **16.85%**, 기존 SOTA 방법인 MGCC 대비 **10.37%** 성능 향상을 보였다.
   - 다른 세 가지 데이터셋(BUS, BUSI, DDTI)에서도 전반적으로 SOTA 방법들을 능가하거나 대등한 성능을 보였다.
2. **데이터 효율성**:
   - 데이터가 적을수록(예: 레이블 $20\%$ 상황) U-Net 대비 성능 향상 폭이 더 크게 나타났다. 이는 DEMS가 데이터 부족 상황에서 매우 효율적임을 입증한다.
   - 레이블 $40\%$만 사용한 DEMS가 레이블 $100\%$를 사용한 일반 U-Net보다 더 높은 성능을 보이는 구간이 확인되었다.
3. **일반화 능력**:
   - BUS 데이터셋으로 학습하고 BUSI 데이터셋으로 평가하는 Cross-dataset 실험 결과, 성능 저하가 적어 강력한 일반화 능력을 보여주었다.
4. **계산 효율성**:
   - MGCC가 매우 높은 성능을 보이지만 학습 시간이 약 10시간으로 매우 긴 반면, DEMS는 합리적인 시간(약 2시간 내외) 내에 최상위 성능을 달성하였다.

## 🧠 Insights & Discussion

**강점**
DEMS는 입력-특징-손실 함수로 이어지는 3단계의 다양성 및 일관성 전략을 통해 의료 영상의 고질적인 문제인 데이터 부족을 효과적으로 해결하였다. 특히 단순한 아키텍처 변경만으로 데이터 효율성을 극대화했다는 점이 돋보인다.

**한계 및 논의사항**
- **이진 분할의 한계**: 본 논문의 Sensitivity Loss는 XOR 연산을 기반으로 하므로, 다중 클래스 분할(Multi-object segmentation)에 직접 적용하기 어렵다. 저자들은 이를 여러 개의 이진 분할 작업으로 나누어 해결하는 방안을 제시했으나, 이는 추론 시간 증가와 클래스 간 겹침 처리 문제라는 추가적인 과제를 낳는다.
- **가정**: 모든 초음파 영상 데이터셋에서 일관된 성능 향상을 보였으나, 이는 초음파 영상의 특성(스펙클 노이즈 등)에 최적화된 결과일 수 있으며, CT나 MRI와 같은 다른 모달리티에서도 동일한 효과가 있을지는 추가 검증이 필요하다.

**비판적 해석**
전체적으로 실험 설계가 매우 꼼꼼하며, 특히 Bland-Altman plot을 통해 예측 일관성을 분석하고 상한선(Upper bound)과의 비교를 통해 데이터 효율성을 입증한 점이 학술적으로 설득력이 높다. 다만, $\lambda$ 웜업 함수의 하이퍼파라미터(예: 상수 5)가 결정된 근거가 명시되지 않아, 다른 데이터셋 적용 시 튜닝이 필요할 것으로 보인다.

## 📌 TL;DR

본 논문은 데이터가 부족한 의료 영상 분할 문제를 해결하기 위해 **DEMS**라는 준지도 학습 프레임워크를 제안한다. 실시간 데이터 증강(OAA), 특징 섭동 주입(RRE), 그리고 XOR 기반의 일관성 손실(Sensitivity Loss)을 통해 **극소량의 레이블 데이터만으로도 기존 SOTA 모델 및 전체 데이터를 사용한 U-Net보다 우수한 성능**을 달성하였다. 이 연구는 데이터 수집 및 레이블링 비용이 높은 의료 AI 분야에서 매우 실용적인 데이터 효율적 학습 방향을 제시한다.
