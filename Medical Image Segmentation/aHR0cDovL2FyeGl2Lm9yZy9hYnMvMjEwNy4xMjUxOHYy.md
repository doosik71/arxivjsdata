# Segmentation in Style: Unsupervised Semantic Image Segmentation with Stylegan and CLIP

Daniil Pakhomov, Sanchit Hira, Narayani Wagle, Kemar E. Green, Nassir Navab (2021)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 시맨틱 이미지 세그멘테이션(Semantic Image Segmentation)을 수행할 때 필수적으로 요구되는 대규모의 정밀한 학습 데이터셋 구축 비용과 그 과정에서의 비효율성이다. 특히 딥러닝 기반의 세그멘테이션 모델들은 최적의 성능을 내기 위해 방대한 양의 픽셀 단위 어노테이션(pixel-level annotation)이 필요하다.

이러한 수동 어노테이션 과정은 매우 느릴 뿐만 아니라, 특히 전문 지식이 필요한 의료 영상 분야에서는 도메인 전문가의 개입이 필수적이기에 더욱 까다롭다. 또한, 시맨틱 클래스 간의 시각적 경계가 모호한 경우 어노테이션 작업자 간의 일관성이 떨어지며, 이는 결과적으로 학습 데이터의 불균질성(heterogeneity)을 초래하여 모델의 성능을 저하시키는 원인이 된다. 따라서 본 연구의 목표는 인간의 감독(supervision) 없이도 이미지 내에서 의미 있는 시맨틱 영역을 자동으로 분할하고, 이를 통해 학습된 모델이 실제 이미지에서도 일반화될 수 있도록 하는 비지도 학습 기반의 세그멘테이션 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 사전 학습된 StyleGAN2의 생성 능력과 특성 공간(feature space)의 구조적 특성을 활용하여 시맨틱 클래스를 발견하고, 이를 CLIP(Contrastive Language-Image Pre-training)과 결합하여 자연어 기반으로 클래스를 정의 및 정제하는 것이다.

구체적인 설계 아이디어는 다음과 같다. 첫째, StyleGAN2 생성기의 중간 특성 맵(intermediate feature maps)을 클러스터링함으로써 이미지 내의 일관된 시맨틱 영역을 자동으로 식별한다. 둘째, 클러스터링만으로는 발견하기 어려운 희귀 클래스(예: 안경, 수염)를 찾기 위해, CLIP을 이용해 latent space 내의 특정 방향(manipulation direction)을 찾아내어 해당 속성을 증폭시킨 후 클러스터링을 수행한다. 셋째, 이렇게 생성된 '합성 이미지-세그멘테이션 마스크' 쌍을 통해 세그멘테이션 네트워크를 학습시킨 후, 이를 실제 이미지에 적용하는 지식 증류(Knowledge Distillation) 방식을 채택한다.

## 📎 Related Works

논문에서는 기존의 세그멘테이션 접근 방식을 크게 완전 지도 학습(Fully Supervised Learning)과 준지도 학습(Semi-Supervised Learning, SSL)으로 구분하여 설명한다. 기존의 SSL 방식들은 소량의 어노테이션 데이터와 대량의 비라벨 데이터를 함께 사용하여 성능을 높이려 시도했다.

본 연구는 이러한 기존 방식들과 달리 어떠한 수동 어노테이션도 사용하지 않는다는 점에서 명확한 차별점을 가진다. 기존의 generative model을 이용한 세그멘테이션 연구들이 존재하지만, 본 논문은 StyleGAN2의 특성 공간 클러스터링과 CLIP의 텍스트-이미지 정렬 능력을 결합하여, 단순한 영역 분할을 넘어 자연어로 정의된 구체적인 시맨틱 클래스를 할당하고 희귀 속성까지 포착할 수 있다는 점을 강조한다.

## 🛠️ Methodology

본 연구의 전체 파이프라인은 StyleGAN2의 특성 추출, K-means 클러스터링, CLIP을 이용한 클래스 발견 및 분류, 그리고 최종 세그멘테이션 모델 학습의 순서로 구성된다.

### 1. StyleGAN2 기반 특성 추출 및 클러스터링
StyleGAN2는 정규 분포 $z \in Z$에서 샘플링된 벡터를 매핑 네트워크를 통해 중간 잠재 벡터 $w \in W$로 변환하고, 이를 생성기 네트워크에 입력하여 이미지를 생성한다. 본 방법론에서는 생성 과정 중의 특정 레이어(주로 7번째 또는 9번째 레이어)에서 출력되는 특성 맵(feature maps)을 추출한다. 앞쪽 레이어일수록 거칠지만 시맨틱한 의미가 강하고, 뒤쪽 레이어일수록 세밀하지만 시맨틱 의미는 낮아지는 특성이 있다.

추출된 특성 맵을 평탄화(flatten)하여 데이터 포인트 $x_n$을 생성한 후, 다음과 같은 손실 함수를 최소화하는 K-means 클러스터링을 수행한다.

$$J = \sum_{n=1}^{\bar{N}} \sum_{k=1}^{K} r_{nk} \|x_n - \mu_k\|^2$$

여기서 $x_n$은 특성 공간의 데이터 포인트, $\mu_k$는 $k$번째 클러스터의 중심, $r_{nk} \in \{0, 1\}$은 데이터 포인트 $n$이 클러스터 $k$에 할당되었는지를 나타내는 이진 지표 변수이며, $\bar{N}$은 전체 픽셀 수이다. 이 과정을 통해 이미지 내에서 일관된 시맨틱 영역들이 클러스터로 구분된다.

### 2. CLIP을 이용한 희귀 클래스 발견 (Image Manipulation)
수염, 안경, 모자와 같이 전체 데이터셋에서 빈도가 낮은 클래스는 일반적인 클러스터링으로 발견하기 어렵다. 이를 해결하기 위해 CLIP의 텍스트 프롬프트를 활용하여 StyleGAN2의 잠재 공간 내에서 특정 속성을 변화시키는 방향 벡터 $G$를 학습한다.

잠재 벡터 $w$에 대해 $\alpha G$를 더해 $F(w + \alpha G)$를 생성함으로써, 모든 샘플 이미지에 해당 속성이 강하게 나타나도록 조작한다. 이렇게 속성이 증폭된 이미지 세트를 다시 클러스터링하면, 희귀 클래스가 하나의 독립적인 클러스터로 명확하게 분리되어 발견될 수 있다.

### 3. 클러스터 분류 (Cluster Classification)
발견된 각 클러스터가 실제로 어떤 시맨틱 클래스(예: '머리카락', '턱')에 해당하는지 정의하기 위해 CLIP의 텍스트 인코더와 이미지 인코더를 사용한다. 이때 이미지 인코더는 작은 클러스터의 해상도를 높이기 위해 다운샘플링 레이어를 제거하고 dilated convolution을 추가하여 수정된 형태를 사용한다.

각 클러스터 영역의 임베딩을 여러 이미지에 대해 평균 낸 후, 미리 정의된 텍스트 프롬프트들의 임베딩과 내적(dot product)을 계산한다. 내적 값이 가장 큰 텍스트 프롬프트를 해당 클러스터의 클래스로 할당한다.

### 4. 지식 증류 (Knowledge Distillation)
마지막으로, StyleGAN2를 통해 생성된 대량의 합성 이미지와 그에 대응하는 자동 생성 마스크(pseudo-labels)를 사용하여 ResNet-18 기반의 세그멘테이션 모델(dilated convolution 적용)을 학습시킨다. 이렇게 학습된 모델은 합성 데이터에서 배운 시맨틱 경계를 실제 이미지(real images)에도 적용할 수 있도록 일반화된다.

## 📊 Results

### 실험 설정
- **데이터셋:** CelebA-Mask8(8개 클래스), CelebAMask-HQ(19개 클래스), OpenEDS(안구 세그멘테이션), 그리고 수염 세그멘테이션을 위한 커스텀 데이터셋을 사용하였다.
- **비교 대상:** 준지도 학습(SSL) 모델인 GAN [20], semanticGAN [10] 및 완전 지도 학습 모델인 DRN [19], SegNet [13]과 비교하였다.
- **평가 지표:** Mean IoU (Intersection over Union)를 사용하였다.

### 주요 결과
1. **CelebA-Mask8:** 제안 방법은 수동 어노테이션을 전혀 사용하지 않고도 Mean IoU 73.1을 기록하며, 소량의 라벨을 사용한 최신 SSL 방법론들(70.0, 69.0)보다 우수한 성능을 보였다.
2. **CelebAMask-HQ:** 19개의 복잡한 클래스에 대해 완전 지도 학습(DRN, 70.5)보다는 낮지만 62.5라는 준수한 성능을 보였다.
3. **OpenEDS (안구):** StyleGAN2를 처음부터 학습시켜 적용한 결과, Mean IoU 82.39를 달성하며 완전 지도 학습(SegNet, 84.1)과 매우 근접한 성능을 나타냈다. 이는 안구 구조가 단순하여 비지도 학습으로도 충분히 포착 가능함을 시사한다.
4. **수염 세그멘테이션:** CLIP을 통한 조작으로 발견한 클래스를 평가한 결과, 완전 지도 학습(88.8)에 근접한 84.29의 Mean IoU를 기록하였다.

## 🧠 Insights & Discussion

본 논문은 데이터 라벨링의 고통 없이도 생성 모델의 내재된 특성 공간과 대규모 언어-이미지 정렬 모델(CLIP)을 결합하여 고품질의 세그멘테이션 마스크를 생성할 수 있음을 증명하였다. 특히, 인간조차 일관되게 라벨링하기 어려운 모호한 경계 영역에서도 모델이 일관된 세그멘테이션 결과를 내놓는다는 점은 매우 고무적이다.

다만, 본 방법론의 성능은 사전 학습된 StyleGAN2 모델의 품질과 해당 도메인에 대한 생성 능력에 크게 의존한다. 만약 생성 모델이 특정 클래스를 현실적으로 생성하지 못한다면(예: 귀걸이, 목걸이 등), 이를 기반으로 한 클러스터링과 세그멘테이션 성능 역시 저하될 수밖에 없다. 또한, CLIP을 이용한 클래스 매핑 과정에서 텍스트 프롬프트의 선택이 결과에 영향을 미칠 수 있다는 점이 잠재적인 한계로 판단된다.

그럼에도 불구하고, 합성 데이터를 생성하여 실제 이미지로 전이시키는 지식 증류 파이프라인은 데이터 부족 문제를 겪는 의료 영상 및 특수 도메인 세그멘테이션 연구에 있어 매우 유용한 프레임워크를 제시하였다고 평가할 수 있다.

## 📌 TL;DR

본 논문은 **StyleGAN2의 특성 공간 클러스터링**과 **CLIP의 텍스트 기반 가이드**를 결합하여, 인간의 감독 없이도 시맨틱 이미지 세그멘테이션을 수행하는 비지도 학습 방법론을 제안한다. 생성 모델을 통해 '합성 이미지-마스크' 쌍을 만들고 이를 ResNet-18에 학습시켜 실제 이미지에 적용함으로써, **어노테이션 없이도 준지도 학습 모델을 능가하거나 완전 지도 학습 모델에 근접하는 성능**을 달성하였다. 이 연구는 라벨링 비용이 극심한 의료 영상이나 희귀 클래스 탐지가 필요한 분야에서 매우 중요한 가능성을 제시한다.
