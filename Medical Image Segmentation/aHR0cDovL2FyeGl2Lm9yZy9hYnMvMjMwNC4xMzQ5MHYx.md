# MIXING DATA AUGMENTATION WITH PRESERVING FOREGROUND REGIONS IN MEDICAL IMAGE SEGMENTATION

Xiaoqing Liu, Kenji Ono, Ryoma Bise (2023)

## 🧩 Problem to Solve

본 논문은 의료 영상 분할(Medical Image Segmentation)에서 딥러닝 모델을 학습시킬 때 발생하는 데이터 부족 문제와 이로 인한 과적합(Overfitting) 문제를 해결하고자 한다. 의료 영상 데이터셋은 일반적으로 수집 가능한 이미지 수가 제한적이므로 데이터 증강(Data Augmentation, DA)이 필수적이다.

기존의 데이터 증강 방식들은 크게 두 가지 한계를 가진다. 첫째, 일부 최신 기법들은 파라미터 업데이트를 위해 추가적인 계산 자원을 소모하며 학습 시간을 증가시킨다. 둘째, Gaussian Blur나 Grid Dropout과 같은 단순한 증강 기법들은 이미지 전체 영역에 섭동(Perturbation)을 가하기 때문에, 분할 성능의 핵심인 전경(Foreground, 장기 영역)과 배경(Background) 사이의 경계 특징을 훼손하여 오히려 성능을 저하시킬 위험이 있다.

따라서 본 연구의 목표는 추가적인 파라미터 학습 없이, 전경 영역의 중요한 특징은 보존하면서 배경 영역에만 증강을 적용하여 분할 정확도와 경계 예측 정밀도를 높이는 단순하고 효율적인 데이터 증강 방법을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **전경 영역(장기)의 특징을 보존(Preserving)하면서 배경 영역에만 변형을 가하는 것**이다. 이를 위해 저자들은 두 가지 새로운 증강 기법인 **KeepMask**와 **KeepMix**를 제안한다.

- **KeepMask**: 원본 이미지의 전경 부분은 그대로 유지하고, 배경 부분에만 기존의 증강 기법(GaussianBlur, GridDropout 등)을 적용하여 전경의 세부 특징과 경계를 보존한다.
- **KeepMix**: 장기가 포함된 이미지의 전경과 장기가 포함되지 않은 이미지의 배경을 결합하여, 장기 영역의 특징은 유지하면서 배경의 다양성을 높인 새로운 이미지를 생성한다.

## 📎 Related Works

논문에서는 다음과 같은 관련 연구와 차별점을 제시한다.

- **Keep Augment**: Saliency map을 사용하여 중요한 영역을 보존하는 기법으로 자연 이미지 분류를 위해 설계되었다. 이를 의료 영상 분할에 그대로 적용할 경우 마스크 경계가 불연속적으로 생성되어 성능이 낮아지는 한계가 있다.
- **MedMix**: Cutting and Pasting을 통한 강한 증강을 사용하여 병변 검출(Lesion Detection)을 위한 대조 학습(Contrastive Learning)에 사용된다. 그러나 검출 작업과 달리 분할 작업은 정밀한 엣지(Edge) 생성이 중요하므로 직접 적용하기 어렵다.
- **기존 단순 증강**: GaussianBlur, GridDropout 등은 이미지 전체에 적용되므로 전경 영역의 판별적 특징(Discriminative features)을 훼손하는 문제가 있다.

본 논문은 이러한 한계를 극복하기 위해 의료 영상 분할의 특성인 '정밀한 경계 추출'에 최적화된, 파라미터 업데이트가 필요 없는 단순한 구조의 보존형 증강 방식을 제안한다.

## 🛠️ Methodology

제안된 방법론의 핵심은 전경 마스크(Ground Truth)를 활용하여 이미지의 특정 영역을 선택적으로 변형하는 것이다.

### 1. KeepMask

KeepMask는 전경 영역의 특징을 유지하고 배경 영역에만 섭동을 추가한다. 입력 이미지 $x_i$와 이에 대응하는 전경 마스크 $y_i$가 있을 때, 증강된 이미지 $\tilde{x}_i$는 다음과 같이 정의된다.

$$\tilde{x}_i = y_i \odot x_i + (1 - y_i) \odot x'_i, \quad x'_i = f_{DA}(x_i)$$

여기서 $\odot$은 원소별 곱셈(Element-wise multiplication)을 의미하며, $f_{DA}$는 배경에 적용할 증강 함수이다. 본 논문에서는 서로 다른 장비의 영상 선명도 차이를 모사하기 위해 $\text{GaussianBlur}$와 $\text{GridDropout}$을 사용하였다. 증강 후의 마스크 $\tilde{y}_i$는 원본 마스크 $y_i$와 동일하게 유지된다.

### 2. KeepMix

KeepMix는 서로 다른 두 이미지의 전경과 배경을 섞어 배경의 다양성을 확보하는 방법이다. 전경(장기)이 포함된 이미지 $x_i$와 전경이 포함되지 않은 배경 이미지 $x_j$를 한 쌍으로 사용하여 다음과 같이 증강된 이미지 $\tilde{x}_i$를 생성한다.

$$\tilde{x}_i = (1 - y_i) \odot x_i + y_j \odot x_j$$

*(참고: 논문 텍스트 내 수식 $\tilde{x}_i = (1 - y_i) \odot x_i + y_j \odot x_j$는 설명과 수식이 상충하는 것으로 보이며, 문맥상 $x_i$의 전경을 유지하고 $x_j$의 배경을 가져오는 $\tilde{x}_i = y_i \odot x_i + (1 - y_i) \odot x_j$의 의도로 해석된다. 본 보고서는 논문에 명시된 수식을 기재하되, 저자가 설명한 '전경 영역을 포함한 이미지와 포함하지 않은 이미지의 믹싱'이라는 목적에 집중한다.)*

학습 시 매 배치(Batch)마다 원본 이미지와 증강된 이미지가 섞여 들어갈 수 있도록 파라미터 $p$(배치 내 원본 이미지의 비율)를 통해 적용 여부를 제어한다.

## 📊 Results

### 실험 설정

- **데이터셋**: CT 스캔 데이터인 CHAOS(간), SLIVER07(간), MSD spleen(비장) 세 가지를 사용하였다.
- **아키텍처**: MobileNet v2를 인코더로 사용하는 Unet 및 Unet++를 사용하였으며, 사전 학습(Pre-training)은 수행하지 않았다.
- **지표**: 전경 영역의 겹침 정도를 측정하는 Dice Similarity Coefficient (Dice)를 평가지표로 사용하였다.
- **비교 대상**: Baseline(증강 없음), Basic DA(Shift, Rotation, Elastic), Real-ESRGAN, RandomFog, MaskDrop.

### 정량적 결과

표 1에 따르면 제안 방법(Ours)이 모든 데이터셋과 아키텍처에서 가장 높은 성능을 기록하였다.

- **Unet 사용 시**: CHAOS에서 94.15% (Baseline 대비 +3.04%), MSD spleen에서 74.70% (Baseline 대비 +5.25%)의 Dice 계수를 달성하였다.
- **Unet++ 사용 시**: MSD spleen에서 77.64% (Baseline 대비 +7.41%)로 가장 큰 폭의 향상을 보였다.

### 분석 및 추가 실험

- **단일 연산 비교**: KeepGaussianBlur, KeepGridDropout, KeepMix(diff/same) 각각의 단일 적용만으로도 Baseline보다 성능이 향상되었으며, 이들을 결합했을 때 최적의 성능이 나타났다.
- **데이터 양에 따른 효과**: 학습 데이터의 양이 적을수록(1/4, 1/2 비율) 제안 방법의 성능 향상 폭이 더 크게 나타났다. 이는 데이터가 부족한 의료 영상 분석 환경에서 본 방법론이 매우 유효함을 시사한다.
- **강건성**: 증강 적용 비율인 $p$ 값의 변화에도 불구하고 KeepMask는 Baseline 및 일반 GridDropout보다 일관되게 높은 성능을 유지하였다.

## 🧠 Insights & Discussion

본 논문의 강점은 복잡한 파라미터 최적화 과정 없이, 의료 영상의 도메인 특성(전경-배경의 명확한 구분 및 경계의 중요성)을 잘 이용했다는 점이다. 특히 전경 영역을 보존함으로써 모델이 장기의 정밀한 경계(Boundary)를 학습하는 데 도움을 주었으며, 이는 시각화 결과(Fig 4)를 통해 증명되었다.

다만, KeepMix의 수식 표기에 있어 텍스트 설명과 수식 간의 불일치가 발견되어 명확한 구현 상세에 의문이 남는다. 또한, 제안된 방법이 매우 단순하기 때문에, 더 복잡한 형태의 장기나 다중 장기가 포함된 데이터셋에서도 동일한 효과를 거둘 수 있을지에 대한 추가 검증이 필요할 것으로 보인다.

결론적으로, 본 연구는 데이터 획득이 어려운 의료 영상 분야에서 전경 보존 기반의 단순한 증강만으로도 상당한 성능 향상을 이끌어낼 수 있음을 보여주었다.

## 📌 TL;DR

이 논문은 의료 영상 분할에서 전경(장기) 영역의 특징을 보존하면서 배경에만 변형을 가하는 **KeepMask**와 **KeepMix**라는 두 가지 단순한 데이터 증강 기법을 제안한다. 이 방법은 추가적인 연산 비용이나 파라미터 학습 없이도 전경의 경계 특징을 효과적으로 유지하여, 특히 학습 데이터가 부족한 상황에서 Unet 및 Unet++의 분할 정확도(Dice Coefficient)를 유의미하게 향상시킨다. 향후 데이터 부족 문제가 심각한 희귀 질환 영상 분석이나 정밀한 장기 분할 연구에 유용하게 적용될 가능성이 높다.
