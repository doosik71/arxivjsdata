# SSEGEP: Small SEGment Emphasized Performance evaluation metric for medical image segmentation

Ammu R, Neelam Sinha (2021)

## 🧩 Problem to Solve

본 논문은 의료 영상 분할(Medical Image Segmentation) 성능을 평가할 때, 기존의 지표들이 가진 한계점을 해결하고자 한다. 의료 영상 분석에서 분할 성능의 정확한 정량화는 매우 중요하지만, 다음과 같은 세 가지 주요 문제점이 존재한다.

첫째, **데이터 분포의 불균형(Imbalance in data distribution)** 문제이다. 의료 영상에서는 배경(Background)에 비해 분할 대상인 관심 영역(ROI)의 크기가 매우 작기 때문에, True Negative(TN) 픽셀의 수가 압도적으로 많다. 이로 인해 Accuracy나 Specificity 같은 전통적인 지표들은 실제 분할 성능과 관계없이 매우 높게 측정되는 경향이 있어 부적절하다.

둘째, **영역 크기의 다양성(Variation in size)** 문제이다. 의료 진단에서 아주 작은 병변(Small lesion)을 찾아내는 것은 질병의 조기 발견과 치료에 결정적인 역할을 한다. 그러나 Dice Similarity Coefficient(DSC)나 Intersection Over Union(IOU)와 같은 기존의 중첩 기반 지표들은 모든 픽셀을 동일하게 처리한다. 결과적으로, 작은 세그먼트 전체를 놓치는 것보다 큰 세그먼트의 일부 픽셀을 놓치는 것이 지표상으로는 더 큰 손실로 나타날 수 있어, 임상적 중요성을 제대로 반영하지 못한다.

셋째, **클래스 불균형이 존재하는 다중 라벨 분할(Multi-label segmentation with class imbalance)** 문제이다. 예를 들어 간(Liver)과 종양(Tumor)을 동시에 분할할 때, 간의 크기에 비해 종양의 크기가 매우 작다. Generalized Dice Score(GD)와 같은 지표는 라벨 간의 불균형은 해결하지만, 동일 라벨 내에 존재하는 다양한 크기의 세그먼트(예: 여러 개의 작은 종양들) 간의 중요도 차이는 구분하지 못한다.

따라서 본 논문의 목표는 작은 세그먼트의 검출 성능에 더 높은 가중치를 부여함으로써, 임상적 진단 가치를 더 잘 반영하는 새로운 평가 지표인 **SSEGEP(Small SEGment Emphasized Performance evaluation metric)**를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **세그먼트의 크기에 반비례하는 가중치를 적용**하여, 작은 영역의 분할 성공 여부가 전체 점수에 더 큰 영향을 미치도록 설계하는 것이다.

구체적인 직관은 다음과 같다. 작은 병변은 놓치기 쉽지만 조기 진단에 매우 중요하므로, 작은 세그먼트에 속한 픽셀이 정확하게 예측(True Positive)되었을 때 더 높은 보상을 주고, 반대로 작은 세그먼트를 놓쳤을 때 더 큰 패널티를 부여하는 방식이다. 또한, False Positive(FP) 픽셀에 대해서도 단순히 개수를 세는 것이 아니라, 해당 라벨의 전체 TP 수에 기반한 가중치를 적용하여 평가의 객관성을 높였다.

## 📎 Related Works

논문에서는 기존의 다양한 평가 지표들을 소개하며 각각의 한계를 지적한다.

- **전통적 지표 (Accuracy, Sensitivity, Specificity, PPV):** TN 픽셀의 비중이 너무 커서 의료 영상의 픽셀 레벨 평가에는 부적합하다.
- **중첩 기반 지표 (DSC, IOU):** 데이터 불균형 문제를 일부 해결하지만, 영역의 크기에 상관없이 모든 TP 픽셀을 동일하게 처리한다. 즉, 작은 세그먼트의 소멸을 과소평가한다.
- **경계 기반 지표 (Hausdorff Distance):** 경계 묘사가 중요한 경우 사용되지만, 이상치(Outlier)에 민감하고 계산 시간이 오래 걸린다는 단점이 있다.
- **Generalized Dice Score (GD):** 라벨별로 면적의 역수를 가중치로 사용하여 클래스 불균형을 해결하지만, 동일 클래스 내의 서로 다른 크기를 가진 여러 세그먼트들을 구분하여 평가하지는 못한다.

SSEGEP는 이러한 기존 지표들과 달리, **단일 클래스 내부에서도 세그먼트별 크기 차이를 인식하여 가중치를 차등 부여**한다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 전체 파이프라인
SSEGEP의 계산 과정은 다음과 같은 단계로 이루어진다.
1. **세그먼트 추출:** Ground Truth(GT)와 예측 이미지(Prediction)에서 각 라벨에 해당하는 개별 연결 성분(Connected Components)들을 추출한다.
2. **GT 가중치 할당:** GT의 각 세그먼트에 대해 해당 세그먼트 면적의 역수를 가중치로 부여한다.
3. **Weighted TP 계산:** 각 세그먼트 내에서 예측된 TP 픽셀 수에 해당 세그먼트의 가중치를 곱하여 합산한다.
4. **Weighted FP 계산:** 라벨별로 FP 픽셀 수를 구하고, 이를 해당 라벨의 전체 TP 합의 역수로 가중치를 주어 계산한다.
5. **최종 지표 산출:** 위 값들을 조합하여 0(나쁨)에서 1(좋음) 사이의 값을 갖는 SSEGEP 점수를 계산한다.

### 주요 방정식
SSEGEP의 최종 값은 다음과 같은 식으로 정의된다.

$$SSEGEP = \frac{\sum_{i=1}^{n_s} w_{tp_i} a_i}{n_s} + \sum_j n_{fp_j}$$

여기서:
- $n_s$는 GT 이미지 내의 전체 세그먼트 수이다.
- $w_{tp_i}$는 $i$번째 세그먼트의 가중치이며, 이는 해당 세그먼트 면적의 역수로 정의된다.
- $a_i$는 $i$번째 세그먼트에서 발생한 TP 픽셀 수이다.
- $n_{fp_j}$는 라벨 $j$에 대한 가중치가 적용된 FP 합계이다.

가중치가 적용된 FP ($n_{fp_j}$)는 다음과 같이 계산된다.

$$n_{fp_j} = fp_j \times \left( \frac{1}{\sum_{label(a_i)=j} a_i} \right)$$

여기서 $fp_j$는 라벨 $j$의 전체 FP 픽셀 수이며, 분모는 해당 라벨 $j$에 속한 모든 세그먼트의 TP 픽셀 합계이다. 즉, TP가 적을수록(분할 성능이 낮을수록) FP에 의한 패널티가 커지도록 설계되어 있다.

## 📊 Results

### 실험 설정
- **데이터셋:** 4개의 공개 데이터셋을 사용하였다.
    - DIARETDB1 (안저 영상, exudates)
    - CBIS-DDSM (유방 촬영술 영상, calcifications)
    - MSD Challenge (간 및 췌장 CT 영상, tumor)
- **비교 대상:** DSC, IOU, Accuracy, Sensitivity, MCC 등.
- **검증 방법:**
    1. **Mean Opinion Score (MOS):** 15명의 평가자가 부여한 주관적 점수와 지표 값 사이의 편차를 측정하여 인간의 인지 능력과 얼마나 일치하는지 분석하였다.
    2. **가설 검정 (Hypothesis Testing):** Welch's t-test를 통해 '좋은 분할' 그룹과 '나쁜 분할' 그룹을 통계적으로 유의미하게 구분할 수 있는지 $p\text{-value}$를 통해 측정하였다.

### 주요 결과
1. **합성 이미지 실험:** 동일한 수의 픽셀을 놓치더라도, 큰 세그먼트 일부를 놓친 경우보다 작은 세그먼트 하나를 완전히 놓친 경우에 SSEGEP가 훨씬 더 낮은 점수를 부여함을 확인하였다. 이는 DSC가 구분하지 못하는 사례를 정확히 잡아낸 것이다.
2. **안저 영상 (Fundus):** SSEGEP는 DSC에 비해 MOS(평균 의견 점수)와의 편차가 30% 더 적게 나타났다. 이는 SSEGEP가 인간 전문가의 판단 기준과 더 밀접하게 연관되어 있음을 시사한다.
3. **간 및 유방 영상:** 통계적 유의성 검정 결과, SSEGEP의 $p\text{-value}$가 DSC보다 훨씬 낮게 나타났다 (예: 간 종양의 경우 $10^{-18}$ 수준). 이는 SSEGEP가 좋은 결과와 나쁜 결과를 구분하는 변별력이 더 뛰어남을 의미한다.
4. **췌장 영상:** 각 클래스당 세그먼트가 하나만 존재하는 경우, SSEGEP와 Generalized Dice의 결과가 유사하게 나타났다. 이는 세그먼트 내 크기 변화가 없을 때는 기존 지표와 유사하게 작동함을 보여준다.

## 🧠 Insights & Discussion

### 강점
SSEGEP는 의료 영상의 특성인 '작은 병변의 높은 임상적 중요성'을 수학적으로 모델링하여 평가 지표에 반영하였다. 특히 동일 클래스 내에서도 크기가 다양한 여러 세그먼트가 존재할 때, 기존의 DSC나 GD가 잡아내지 못하는 성능 저하(작은 세그먼트 누락)를 효과적으로 정량화할 수 있다는 점이 가장 큰 강점이다.

### 한계 및 비판적 해석
논문에서 명시한 한계점은 다음과 같다.
- **세그먼트 수의 영향:** GT에 세그먼트가 단 하나만 존재하거나, 모든 세그먼트의 크기가 비슷할 경우에는 DSC와 같은 기존 중첩 지표와 큰 차이가 없으므로 SSEGEP만의 이점이 사라진다.
- **무작위 분할 시의 점수:** 분할이 완전히 무작위로 이루어졌을 때 지표 값이 이론적으로 0이 되어야 하지만, SSEGEP는 반드시 0이 되지 않는 특성이 있다고 언급되었다.

또한, 수식 (1)에서 $\sum n_{fp_j}$ 항이 더하기($+$)로 표기되어 있으나, 텍스트 설명에서는 FP가 많을수록 SSEGEP 값이 작아진다고 기술하고 있다. 일반적으로 성능 지표에서 FP는 감점 요소이므로, 실제 구현 시에는 뺄셈으로 처리되었거나 전체 식의 구조가 다를 가능성이 높다. 이 부분에 대한 명확한 수식적 설명이 부족한 점은 아쉽다.

## 📌 TL;DR

본 논문은 의료 영상 분할에서 작은 병변의 조기 발견이라는 임상적 중요성을 반영하기 위해, **세그먼트 크기의 역수를 가중치로 사용하는 SSEGEP**라는 새로운 평가 지표를 제안하였다. 실험을 통해 SSEGEP가 작은 세그먼트를 놓쳤을 때 더 큰 패널티를 부여함으로써, 기존의 Dice Score보다 인간 전문가의 판단(MOS)에 더 가깝고 통계적으로 더 높은 변별력을 가짐을 증명하였다. 이 연구는 향후 조기 진단을 위한 의료 영상 분할 알고리즘의 최적화 및 성능 평가에 중요한 기준을 제공할 것으로 기대된다.
