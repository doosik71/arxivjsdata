# STYLEAUGMENTATION IMPROVES MEDICAL IMAGE SEGMENTATION

Kevin Ginsburger (2022)

## 🧩 Problem to Solve

의료 영상 분할(Medical Image Segmentation) 분야에서는 딥러닝 모델을 학습시키기 위한 라벨링된 데이터의 양이 매우 제한적이라는 고질적인 문제가 존재한다. 이러한 데이터 부족 현상은 모델의 과적합(Over-fitting)을 유발하며, 특히 합성곱 신경망(Convolutional Neural Networks, CNN)이 이미지의 형태(Shape)보다는 질감(Texture) 정보에 과도하게 의존하는 'Texture Bias' 경향을 보인다는 점이 여러 연구를 통해 밝혀졌다.

의료 영상의 경우, 촬영 장비의 종류, 획득 설정, 또는 조직의 특성에 따라 질감이 달라질 수 있다. 만약 네트워크가 특정 질감에 편향되어 학습된다면, 학습 데이터와 다른 질감을 가진 새로운 데이터에 대해 성능이 급격히 떨어지는 일반화 성능 저하 문제가 발생한다. 따라서 본 논문의 목표는 분류(Classification) 작업에서 효과가 입증된 Style Augmentation을 세그멘테이션 작업에 도입하여, 질감 과적합을 줄이고 분할 성능을 향상시키는 것이다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 이미지의 형태와 에지(Edge) 정보는 유지하면서 질감, 대비, 색상을 무작위로 변경하는 Style Augmentation을 학습 과정에 도입하는 것이다. 이를 통해 모델이 특정 질감 패턴에 의존하지 않고, 객체의 기하학적 형태(Shape)를 중심으로 학습하도록 유도함으로써 데이터가 부족한 환경에서도 강건한(Robust) 세그멘테이션 성능을 확보하고자 하였다.

## 📎 Related Works

기존의 의료 영상 분할 연구에서는 U-Net과 같이 적은 양의 데이터로도 작동하는 아키텍처가 널리 사용되어 왔으며, 회전, 반전, 탄성 변형(Elastic deformation)과 같은 기하학적 변환 중심의 데이터 증강 기법이 주로 활용되었다. 그러나 이러한 전통적인 방식은 이미지의 형태나 색상을 일부 수정할 뿐, CNN의 근본적인 문제인 Texture Bias를 해결하지 못한다는 한계가 있다.

반면, Geirhos et al. (2019) 등의 연구는 CNN이 질감에 편향되어 있음을 정량적으로 입증하였으며, 질감을 무작위화하는 Style Augmentation이 이미지 분류 작업에서 성능과 강건성을 높인다는 것을 보여주었다. 본 논문은 이러한 Style Augmentation의 효용성을 세그멘테이션 작업, 특히 데이터 희소성이 높은 의료 영상 분야로 확장하여 적용했다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 1. 데이터셋 및 네트워크 구조
본 연구에서는 다양한 장기와 환자의 H&E 염색 조직 이미지로 구성된 MoNuSeg 데이터셋을 사용하였다. 총 30장의 이미지 중 25장은 학습에, 5장은 검증에 사용하였으며, 나머지 14장은 테스트셋으로 분리하였다. 모든 이미지는 $512 \times 512$ 크기로 조정되었다.

사용된 모델은 파라미터 수를 줄이면서도 성능을 높인 UNeXt 아키텍처이다. 모델의 복잡도를 낮추어 적은 데이터셋에서도 과적합이 덜 발생하도록 설계되었으며, 구체적인 학습 설정은 다음과 같다.
- **Dropout**: 학습 및 추론 시 모두 10%의 Dropout 비율을 적용한다.
- **학습 설정**: Mini-batch size는 4, 총 2000 epoch 동안 학습을 진행한다.
- **모델 저장**: 검증 세트에서 Intersection over Union (IoU) 점수가 가장 높은 모델을 저장한다.
- **추론 절차**: Dropout이 적용된 네트워크 20개 인스턴스의 결과를 평균내어 최종 성능을 측정한다.

### 2. 데이터 증강 전략
모든 실험에서 기본적으로 90도 무작위 회전(Random 90 degrees rotations)과 반전(Flipping)을 적용하였다. 여기에 추가로 도입된 Style Augmentation은 Jackson et al. (2019)의 방법론을 따르며, 세부 설정은 다음과 같다.
- **Style Transfer Strength**: 최적값인 0.5로 설정하여 형태와 에지를 보존하면서 질감을 변경한다.
- **적용 방식**: 매 mini-batch마다 무작위의 Stylization Ratio를 결정하여, 해당 배치 내에서 무작위 스타일이 적용될 이미지의 수를 결정한다.

## 📊 Results

### 1. 정량적 결과
Style Augmentation 적용 여부에 따른 성능 비교 결과, 모든 지표에서 유의미한 향상이 관찰되었다.

| Metrics | No Style Aug. | Style Aug. |
| :--- | :---: | :---: |
| **IoU** | $0.6072$ | $0.6656$ |
| **Dice Score** | $0.7533$ | $0.7991$ |

특히, 매우 적은 수의 학습 데이터(25장)만을 사용했음에도 불구하고, Style Augmentation을 적용한 UNeXt의 결과가 Medical Transformer와 같은 최신 기법(State-of-the-art)의 정량적 결과에 근접하는 성과를 보였다.

### 2. 정성적 및 학습 곡선 분석
- **학습 곡선**: Style Augmentation을 적용하지 않았을 때는 약 200 epoch 지점에서 검증 손실(Validation loss)이 다시 증가하는 조기 과적합 현상이 나타났다. 반면, Style Augmentation을 적용했을 때는 학습 손실과 검증 손실이 함께 부드럽게 수렴하는 양상을 보였다. 비록 스타일 변형으로 인해 학습 난이도가 높아져 학습 손실 값 자체는 더 높게 형성되었으나, 최종적인 검증 손실은 훨씬 낮아졌다.
- **분할 결과**: 정성적 분석 결과, 기본 증강만 사용한 경우 가양성(False Positive) 오류가 많이 발생하였으나, Style Augmentation을 적용한 경우 가양성이 현저히 줄어들고 세포의 경계선(Edge)이 더 날카롭고 정확하게 분할되는 것이 확인되었다.

## 🧠 Insights & Discussion

본 연구는 Style Augmentation이 의료 영상 분할에서 두 가지 핵심적인 역할을 수행함을 보여준다. 첫째, 학습 과정에 의도적으로 노이즈(질감 변형)를 추가함으로써 모델이 쉬운 지표인 질감에 안주하지 않고 더 본질적인 특징인 형태(Shape)를 학습하게 하여 과적합을 방지한다. 둘째, 이를 통해 결과적으로 분할 마스크의 정밀도를 높이고 오검출을 줄이는 효과를 거두었다.

다만, 본 연구는 MoNuSeg라는 단일 데이터셋과 UNeXt라는 특정 아키텍처를 중심으로 진행되었다는 한계가 있다. 따라서 다른 성격의 의료 데이터셋이나 다른 세그멘테이션 네트워크에서도 동일한 효과가 나타날지는 추가적인 검증이 필요하다. 또한, 저자는 향후 연구 방향으로 SinGAN과 같은 이미지 생성 기법을 통해 합성 데이터를 만들고, 여기에 Style Augmentation을 결합한다면 질감 과적합을 더욱 효과적으로 억제하고 형태 기반의 표현력을 극대화할 수 있을 것이라고 제안한다.

## 📌 TL;DR

본 논문은 CNN의 질감 편향(Texture Bias) 문제를 해결하기 위해 의료 영상 분할 작업에 Style Augmentation을 도입한 연구이다. 적은 양의 데이터셋(MoNuSeg)에서 실험한 결과, 질감의 무작위화가 모델의 과적합을 효과적으로 방지하고 IoU 및 Dice Score를 유의미하게 향상시킴을 입증하였다. 이는 추가적인 하이퍼파라미터 튜닝 없이도 간단히 적용 가능한 기법으로, 데이터가 부족한 의료 영상 분석 분야에서 모델의 일반화 성능을 높이는 실용적인 방법이 될 가능성이 높다.
