# EigenRank by Committee: A Data Subset Selection and Failure Prediction paradigm for Robust Deep Learning based Medical Image Segmentation

Bilwaj Gaonkar, Joel Beckett, Mark Attiah, Christine Ahn, Matthew Edwards, Bayard Wilson, Azim Laiwalla, Banafsheh Salehi, Bryan Yoo, Alex Bui, Luke Macyszyn (2021)

## 🧩 Problem to Solve

본 논문은 완전 자동화된 딥러닝 기반 의료 영상 분할(Medical Image Segmentation) 기술을 실제 임상 워크플로우에 적용할 때 발생하는 두 가지 핵심적인 알고리즘적 난제를 해결하고자 한다.

첫 번째 문제는 **데이터 서브셋 선택(Data Subset Selection, DSS)**의 효율성 문제이다. 의료 영상 분야에서는 전문가의 수동 주석(Manual Annotation) 비용이 매우 높기 때문에, 방대한 데이터베이스 전체를 라벨링하는 것은 비현실적이다. 따라서 최소한의 데이터만으로도 모델의 성능을 극대화할 수 있는 최적의 학습 데이터 서브셋을 선택하는 것이 매우 중요하다.

두 번째 문제는 **실패 예측(Failure Prediction, FP)**의 부재이다. 대부분의 딥러닝 분할 모델은 예측 결과가 잘못되었음에도 불구하고 이를 알리지 않고 침묵하며 실패(fail silently)하는 경향이 있다. 임상 환경에서는 모델이 자신의 예측이 틀렸을 가능성을 의료진에게 경고할 수 있는 능력이 필수적이며, 이는 시스템의 신뢰성과 안전성을 결정짓는 중요한 요소가 된다.

결과적으로 본 논문의 목표는 데이터 선택의 효율성을 높여 학습 비용을 줄임과 동시에, 모델이 실패할 가능성이 높은 케이스를 사전에 식별하여 임상적 강건성(Robustness)을 확보하는 'EigenRank' 알고리즘을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 여러 모델로 구성된 위원회(Committee) 간의 **'불일치 정도'를 정량화**하여, 이를 기반으로 가장 정보 가치가 높은(challenging) 데이터를 선택하고 실패 가능성이 높은 데이터를 예측하는 것이다.

중심적인 설계 직관은 다음과 같다. 만약 서로 다른 서브셋으로 학습된 여러 모델이 특정 영상에 대해 서로 매우 다른 분할 결과를 내놓는다면, 해당 영상은 모델들에게 '어려운(challenging)' 케이스이며, 따라서 학습 데이터로 추가되었을 때 모델의 성능과 강건성을 높이는 데 기여할 가능성이 크다는 것이다. 이를 위해 본 논문은 모델 간의 Dice score를 원소로 하는 행렬을 구성하고, 이 행렬의 **주 고윳값(Principal Eigenvalue)**을 통해 불일치 정도를 측정하는 방식을 제안한다.

## 📎 Related Works

기존의 데이터 서브셋 선택 및 능동 학습(Active Learning) 연구들은 주로 다음과 같은 한계를 가진다.

1.  **픽셀/복셀 수준의 접근:** 많은 기존 방식들이 픽셀 수준의 불확실성(Uncertainty)이나 불일치를 이용하여 전문가에게 쿼리를 보내는 방식을 취한다. 그러나 이는 국소적인 영역의 문제만 다룰 뿐, 영상 전체(Subject/Patient level)의 관점에서 어떤 케이스가 어려운지를 판단하는 데에는 한계가 있다.
2.  **분류 문제 중심:** 대다수의 DSS 알고리즘은 이진 분류(Binary Classification)의 정확도 보존에 초점을 맞추고 있어, 의료 영상 분할과 같은 복잡한 형태의 작업에 직접 적용하기 어렵다.
3.  **정확도와 강건성의 차이:** 기존 연구들은 주로 평균 정확도(Average Accuracy) 향상에 집중하지만, 본 논문은 임상 적용을 위해 결과의 일관성을 의미하는 '강건성(Robustness)'에 더 큰 가치를 둔다.

본 연구는 Query-by-Committee (QBC) 프레임워크를 의료 영상 분할의 전역적(Global) 수준으로 확장하고, 이를 Von Neumann 정보 이론과 연결함으로써 기존의 픽셀 수준 능동 학습과 차별점을 둔다.

## 🛠️ Methodology

### 1. 전체 파이프라인 및 구조
본 연구에서는 **Residual U-Net** 아키텍처를 기본 모델로 사용하며, EigenRank 알고리즘은 크게 초기화 단계와 반복 단계로 구성된다.

### 2. EigenRank 알고리즘 상세

#### 가. 초기화 단계 (Initialization)
1.  전체 데이터셋 $T$에서 크기가 $k$인 두 개의 서브셋 $S_1, S_2$를 무작위로 선택한다.
2.  각 서브셋으로 두 개의 모델 $D_{S_1}, D_{S_2}$를 학습시킨다.
3.  학습에 사용되지 않은 나머지 영상 $I_j \in T \setminus S$들에 대해 두 모델의 분할 결과 $S_{D_{S_1}^j}$와 $S_{D_{S_2}^j}$ 사이의 Dice score를 계산한다.
    $$\Delta_{S_1, S_2}^j = \Delta(S_{D_{S_1}^j}, S_{D_{S_2}^j})$$
4.  Dice score가 가장 낮은(즉, 두 모델이 가장 많이 불일치하는) $k$개의 영상을 선택하여 세 번째 서브셋 $S_3$를 구성한다.

#### 나. 반복 단계 (Iterations)
$t$번째 반복 단계에서는 이미 학습된 $t$개의 모델 $\{D_{S_1}, \dots, D_{S_t}\}$이 존재한다.

1.  각 영상 $I_j$에 대해, 모델 쌍 $(p, q)$ 간의 Dice score를 원소로 하는 $t \times t$ 크기의 **Dice 행렬** $D_j$를 생성한다.
    $$D_j = [D_{pq}^j] \quad \text{where} \quad D_{pq}^j = \Delta(S_{D_{S_p}^j}, S_{D_{S_q}^j})$$
2.  이 행렬 $D_j$의 **최대 고윳값(Maximum Eigenvalue)** $\lambda_{\max}^j$를 계산한다.
    $$\lambda_{\max}^j = \max(\text{eig}[D_j])$$
3.  $\lambda_{\max}^j$ 값이 가장 작은 $k$개의 영상을 선택하여 다음 서브셋 $S_{t+1}$을 구성한다. (값이 작을수록 모델들 간의 전반적인 합의도가 낮음을 의미한다.)
4.  이 과정을 목표로 하는 데이터 양이 채워질 때까지 반복한다.

### 3. 실패 예측 (Failure Prediction)을 위한 활용
학습된 앙상블 모델들이 특정 영상에 대해 강한 합의를 이루지 못한다면(즉, $\lambda_{\max}$ 값이 낮다면), 해당 영상은 딥러닝 모델이 실패할 가능성이 높은 케이스로 판단한다. 이러한 케이스들을 사전에 선별하여 인간 전문가에게 전달함으로써 '침묵하는 실패'를 방지할 수 있다.

### 4. 이론적 배경: Von Neumann 정보
본 논문은 Dice 행렬 $D_j$가 양의 준정부호(Positive Semi-definite) 행렬임을 수학적으로 증명한다. 이를 통해 $D_j$의 고윳값들을 이용하여 **Von Neumann 엔트로피** $H_j$를 정의할 수 있다.
$$H_j = -\sum_{r=1}^{t} \lambda_r \log(\lambda_r)$$
실험적으로 주 고윳값 $\lambda_1$이 다른 고윳값들에 비해 압도적으로 크기 때문에($\lambda_1 \gg \lambda_2, \dots, \lambda_t$), $\lambda_{\max}$를 사용하는 것이 Von Neumann 엔트로피를 통해 불일치 정도를 측정하는 것과 실질적으로 동일한 효과를 낸다고 설명한다.

## 📊 Results

### 1. 실험 설정
- **데이터셋:** UCLA PACS에서 수집한 요추 MRI 데이터 (척수관 및 추간판 분할 작업).
- **비교 대상:** 무작위 선택(Random Selection) 방식.
- **평가 지표:** Dice score의 평균(Mean) 및 표준편차(Std. Dev). 표준편차가 낮을수록 모델의 강건성(Robustness)이 높다고 판단한다.

### 2. 척수관(Spinal Canal) 분할 결과
- **정성적 결과:** EigenRank가 선택한 영상들은 무작위 선택된 영상들에 비해 척추측만증, 금속 임플란트, 낮은 대비(Contrast) 등 훨씬 복잡하고 도전적인 케이스들을 포함하고 있었다.
- **정량적 결과:**
    - EigenRank로 선택된 서브셋으로 학습한 모델은 무작위 선택 모델보다 **표준편차가 일관되게 낮게** 나타났다.
    - 특히, EigenRank가 선별한 어려운 케이스들을 제외하고 검증했을 때보다, 전체 데이터에서 검증했을 때 무작위 모델과의 강건성 차이가 더 극명하게 드러났다. 이는 EigenRank가 실제로 '어려운' 데이터를 정확히 짚어냈음을 시사한다.

### 3. 추간판(Intervertebral Disk) 분할 결과
- 척수관 실험과 마찬가지로, EigenRank를 통해 선택된 데이터로 학습한 모델이 무작위 선택 모델보다 더 낮은 표준편차를 보이며 높은 강건성을 입증하였다.

### 4. 실패 예측 성능
- 무작위 서브셋으로 학습된 모델 $D_m$이 있을 때, EigenRank를 사용하여 검증 셋에서 '어려운' 케이스들을 반복적으로 제거하였다.
- 제거된 케이스들의 평균 Dice score는 매우 낮았으나, 남은 케이스들의 평균 Dice score는 점진적으로 상승하고 표준편차는 감소하였다. 이는 EigenRank가 모델이 실패할 가능성이 높은 케이스를 정확하게 예측해낼 수 있음을 보여준다.

## 🧠 Insights & Discussion

### 1. 강점 및 의의
- **강건성 중심의 접근:** 단순히 평균 정확도를 높이는 것이 아니라, 표준편차를 줄임으로써 임상 환경에서 필수적인 '일관된 성능(Robustness)'을 확보하는 전략을 취했다.
- **전역적 불일치 측정:** 픽셀 단위의 불확실성이 아닌, 영상 전체의 분할 결과 간의 관계를 행렬의 고윳값으로 정량화하여 계산 효율성과 판단의 정확성을 동시에 잡았다.
- **이론적 토대:** 단순한 휴리스틱이 아니라 Von Neumann 엔트로피라는 정보 이론적 근거를 제시하여 알고리즘의 타당성을 뒷받침했다.

### 2. 한계 및 논의사항
- **모델 의존성:** 본 연구는 Residual U-Net을 사용했으나, 다른 아키텍처나 하이퍼파라미터 설정에서도 동일한 효과가 나타날지는 추가 검증이 필요하다.
- **지표의 확장성:** Dice score 외에도 Jaccard index, Hausdorff distance 등 다른 분할 지표를 Dice 행렬 대신 사용할 수 있는 가능성을 언급하였으나, 이에 대한 구체적인 비교 실험은 수행되지 않았다.
- **계산 비용:** 반복적으로 모델을 학습시켜야 하므로, 데이터셋이 매우 크거나 모델이 무거울 경우 학습 시간이 상당히 증가할 수 있다는 점이 잠재적 한계로 보인다.

## 📌 TL;DR

본 논문은 의료 영상 분할에서 **최소한의 데이터로 최대의 강건성을 확보**하기 위한 데이터 선택 및 실패 예측 프레임워크인 **EigenRank**를 제안한다. 앙상블 모델 간의 Dice score 행렬의 주 고윳값($\lambda_{\max}$)을 통해 데이터의 '난이도'를 정량화하며, 이를 통해 **(1) 모델의 강건성을 극대화하는 최적의 학습 서브셋을 선택**하고, **(2) 모델이 실패할 가능성이 높은 영상을 사전에 예측**하여 전문가에게 알리는 기능을 수행한다. 이는 고비용의 의료 데이터 라벨링 문제를 해결하고, 딥러닝 모델의 임상 적용 시 신뢰성을 높이는 데 중요한 기여를 할 것으로 기대된다.
