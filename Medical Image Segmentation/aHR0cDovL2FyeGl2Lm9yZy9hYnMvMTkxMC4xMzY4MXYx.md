# The Domain Shift Problem of Medical Image Segmentation and Vendor-Adaptation by Unet-GAN

Wenjun Yan, Yuanyuan Wang, Shengjia Gu, Lu Huang, Fuhua Yan, Liming Xia, and Qian Tao (2018)

## 🧩 Problem to Solve

본 논문은 의료 영상 분할(Medical Image Segmentation) 분야에서 발생하는 **Domain Shift** 문제를 해결하고자 한다. 일반적으로 U-Net과 같은 합성곱 신경망(CNN)은 학습 데이터와 테스트 데이터가 동일한 분포를 가질 때 매우 뛰어난 성능을 보이지만, 실제 임상 환경에서는 서로 다른 제조사(Vendor)나 의료 센터에서 획득한 영상들이 사용된다. 

이러한 데이터의 분포 차이는 모델의 성능을 예기치 않게 저하시키며, 새로운 도메인이 추가될 때마다 대량의 어노테이션(Annotation) 데이터를 수집하여 모델을 재학습시키는 것은 비용과 시간이 너무 많이 소요되어 사실상 불가능에 가깝다. 따라서 본 연구의 목표는 추가적인 어노테이션 없이도 서로 다른 제조사 간의 영상 특성을 맞추어 분할 성능을 유지할 수 있는 범용적인 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Unpaired Generative Adversarial Network (GAN)**를 이용하여 타겟 도메인의 영상을 소스 도메인의 영상으로 변환한 뒤, 이미 학습된 U-Net을 적용하는 것이다. 

단순한 이미지 변환을 넘어, 본 연구는 **특징 수준(Feature level)에서의 일관성**을 강제하는 설계를 도입하였다. 즉, GAN이 생성한 이미지가 단순히 시각적으로 비슷해 보이는 것을 넘어, 분할 작업에 핵심적인 특징(Segmentation-specific features)을 유지하도록 U-Net의 인코더 경로를 특징 추출기로 활용하여 손실 함수에 반영한 것이 핵심적인 직관이다.

## 📎 Related Works

기존의 의료 영상 분할에서는 U-Net 아키텍처가 인간의 속도-정확도 균형을 뛰어넘는 성과를 거두었으며, 이미지 간 변환을 위해서는 CycleGAN과 같은 Unpaired image-to-image translation 기법이 널리 사용되어 왔다. 

또한, 일반적인 딥러닝 모델의 일반화 능력을 높이기 위해 데이터 증강(Data Augmentation) 기법이 사용되지만, 본 논문은 Domain Shift가 우연한 노이즈가 아니라 체계적인(Systematic) 분포 차이에서 기인하기 때문에, 단순한 무작위 노이즈 추가와 같은 증강 방식으로는 일반화 격차를 완전히 해소할 수 없다는 한계를 지적한다.

## 🛠️ Methodology

### 전체 파이프라인
제안된 **Unet-GAN** 프레임워크는 다음의 3단계 절차로 동작한다.
1. 어노테이션이 충분한 소스 도메인 데이터로 **LV-Unet**을 먼저 학습시킨다.
2. 소스 도메인과 타겟 도메인의 비정형(Unannotated) 데이터를 사용하여 **Unet-GAN**의 생성자와 판별자를 학습시킨다. 이때 U-Net의 특징 정보를 손실 함수에 통합한다.
3. 타겟 도메인의 영상을 Unet-GAN을 통해 소스 도메인 영상으로 변환한 후, 학습된 LV-Unet에 입력하여 최종 분할 결과를 얻는다.

### 주요 구성 요소 및 손실 함수
본 모델은 CycleGAN의 구조를 기반으로 하며, 이미지 수준과 특징 수준의 손실을 결합하여 학습한다.

**1. 이미지 수준: SSIM Loss**
의료 영상에서는 밝기나 대비가 변하더라도 해부학적 구조는 보존되어야 한다. 따라서 평균 제곱 오차(MSE) 대신 구조적 유사도 지수(SSIM)를 사용한다. SSIM은 휘도(Luminance), 대비(Contrast), 구조(Structure) 세 가지 요소를 측정하며, 본 논문에서는 구조 항에 가장 높은 가중치를 부여한다. 이미지 수준의 손실 함수 $\mathcal{L}_{SSIM}$은 다음과 같이 정의된다.
$$\mathcal{L}_{SSIM} = \sum ||1 - \text{SSIM}(x, y)||^2$$

**2. 특징 수준: Unet Loss**
시각적인 유사성만으로는 적대적 노이즈(Adversarial noise)에 취약할 수 있다. 이를 해결하기 위해 LV-Unet의 인코더를 특징 추출기로 활용한다. 원본 이미지와 변환된 이미지 사이의 특징 맵 $f$의 MSE를 측정하여, GAN이 분할 작업에 필수적인 특징을 보존하도록 강제한다.
$$\mathcal{L}_{Unet} = \sum ||f_1 - f_2||^2$$

**3. 전체 손실 함수**
최종 손실 함수는 GAN의 적대적 손실($\mathcal{L}_{GAN}$)과 위에서 정의한 두 가지 손실을 가중치 $\lambda_1, \lambda_2$와 함께 결합한 형태이다.
$$\mathcal{L}_{Total} = \mathcal{L}_{GAN} + \lambda_1 \mathcal{L}_{Unet} + \lambda_2 \mathcal{L}_{SSIM}$$

## 📊 Results

### 실험 설정
- **데이터셋**: 3개 제조사(Philips, GE, Siemens)의 심장 Cine MRI 영상 144명의 데이터를 사용하였다.
- **도메인 설정**: Philips 데이터를 소스 도메인으로 설정하여 LV-Unet을 학습시켰으며, Siemens와 GE를 타겟 도메인으로 설정하였다.
- **평가 지표**: 정답지(Ground Truth)와 분할 결과 사이의 **Dice overlap index**를 측정하였다.
- **비교 대상**: 
    1. Clean Unet: 소스 도메인으로 학습된 모델을 타겟 데이터에 직접 적용.
    2. Noisy Unet: 소스 데이터에 무작위 노이즈를 추가하여 증강 학습한 모델을 적용.
    3. Unet-GAN: 제안 방법론 적용.

### 실험 결과
실험 결과, 소스 도메인(Philips)에서만 학습된 Clean Unet은 타겟 도메인에서 성능이 급격히 저하되었다. 특히 Siemens 데이터의 경우 Dice 지수가 $0.892$에서 $0.474$로 크게 떨어졌다. 무작위 노이즈를 통한 증강(Noisy Unet)은 성능을 소폭 향상시켰으나($0.474 \rightarrow 0.502$), 한계가 명확하였다.

반면, **Unet-GAN**을 적용했을 때 성능이 비약적으로 향상되었다. Siemens 데이터의 경우 Dice 지수가 $0.805$까지 상승하였으며, GE 데이터 또한 $0.727$에서 $0.867$로 크게 개선되었다. 이는 Unet-GAN이 타겟 도메인의 영상을 소스 도메인의 특성으로 성공적으로 변환하였음을 시사한다.

## 🧠 Insights & Discussion

본 논문은 CNN 기반의 의료 영상 분할 모델이 데이터의 분포 변화에 매우 취약하다는 점을 실증적으로 보여주었으며, 이를 해결하기 위해 GAN을 이용한 도메인 적응(Domain Adaptation) 방식을 제안하였다. 

특히 주목할 점은 단순히 이미지의 외형을 맞추는 것이 아니라, **분할 모델의 내부 특징(Feature)을 보존하도록 GAN을 학습**시킨 점이다. 이는 딥러닝 모델이 학습한 통계적 분포의 차이를 효과적으로 메워주는 장치가 된다. 다만, 본 연구는 소스 도메인에 이미 매우 정교하게 학습된 U-Net 모델이 존재한다는 가정 하에 작동하므로, 초기 소스 모델의 품질이 전체 시스템의 성능 상한선을 결정한다는 한계가 있다. 또한, 서로 다른 제조사 간의 변환 외에 완전히 다른 모달리티 간의 전이 가능성에 대해서는 추가적인 논의가 필요하다.

## 📌 TL;DR

본 연구는 의료 영상의 제조사별 특성 차이로 인한 성능 저하(Domain Shift) 문제를 해결하기 위해, **U-Net의 특징 보존 손실을 결합한 Unet-GAN** 프레임워크를 제안하였다. 타겟 도메인의 영상을 소스 도메인으로 변환하여 기존 모델을 그대로 사용할 수 있게 함으로써, 추가적인 어노테이션 없이도 다기관·다제조사 환경에서 높은 분할 정확도를 달성할 수 있음을 입증하였다. 이 연구는 실제 임상 현장에서 딥러닝 모델의 범용적인 배포 가능성을 높이는 데 중요한 역할을 할 수 있다.
