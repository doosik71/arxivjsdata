# Self-Loop Uncertainty: A Novel Pseudo-Label for Semi-Supervised Medical Image Segmentation

Yuexiang Li, Jiawei Chen, Xinpeng Xie, Kai Ma, and Yefeng Zheng (2020)

## 🧩 Problem to Solve

본 논문은 의료 영상 분할(Medical Image Segmentation) 분야에서 딥러닝 모델을 학습시키기 위해 필요한 대량의 픽셀 단위 주석(Pixel-wise annotation) 데이터가 부족하다는 문제를 해결하고자 한다. 의료 영상의 주석 작업은 전문의의 많은 시간과 비용을 요구하기 때문에, 실제 환경에서 충분한 양의 레이블링된 데이터를 확보하는 것은 매우 어렵다.

따라서 본 연구의 목표는 한정된 양의 레이블링된 데이터($D_L$)와 대량의 레이블링되지 않은 데이터($D_U$)를 함께 활용하는 준지도 학습(Semi-supervised learning) 접근 방식을 제안하여, 데이터 부족 문제를 극복하고 분할 정확도를 높이는 것이다. 특히, 레이블링되지 않은 데이터에 대해 신뢰할 수 있는 의사 레이블(Pseudo-label)을 효율적으로 생성하는 방법에 집중한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Self-loop uncertainty**라는 새로운 형태의 의사 레이블 생성 방식이다. 이는 자기지도 학습(Self-supervised learning) 태스크를 통해 신경망을 반복적으로 최적화하는 과정에서 발생하는 모델의 상태 변화를 활용한다.

핵심 직관은 다음과 같다. 자기지도 학습 태스크(예: Jigsaw puzzles)를 수행하며 가중치가 계속 업데이트되는 신경망의 서로 다른 단계(stage)를 서로 다른 모델로 간주하는 것이다. 이를 통해 다수의 모델을 별도로 학습시켜 앙상블하는 방식의 높은 신뢰성을 유지하면서도, 추론 시간과 계산 비용을 획기적으로 줄여 앙상블 불확실성(Ensemble uncertainty)을 근사할 수 있다.

## 📎 Related Works

기존의 준지도 학습 기반 의료 영상 분할에서는 레이블링되지 않은 데이터의 의사 레이블을 생성하기 위해 주로 다음과 같은 불확실성 추정(Uncertainty estimation) 방법들을 사용해 왔다.

1. **Softmax Probability Map**: 모델의 최종 출력층에서 나오는 확률 지도를 그대로 사용하는 방식이다.
2. **Monte Carlo (MC) Dropout**: 추론 시 드롭아웃을 활성화하여 여러 번 예측을 수행하고 그 변동성을 측정하는 방식으로, 베이지안 불확실성의 근사치로 간주된다.
3. **Network Ensemble**: 서로 다른 초기값으로 학습된 $K$개의 네트워크를 앙상블하여 평균 확률 지도를 생성하는 방식이다.

관련 연구에 따르면, 앙상블 방법이 가장 신뢰도 높은 결과를 제공하지만 계산 비용이 매우 높고, MC Dropout은 드롭아웃 설정에 따라 성능 의존도가 높다는 한계가 있다. 본 논문은 이러한 기존 방식들의 장점인 '신뢰성'과 '효율성'을 동시에 잡기 위해 Self-loop uncertainty를 제안하며 차별점을 둔다.

## 🛠️ Methodology

### 전체 시스템 구조

본 프레임워크는 Fully Convolutional Network (FCN)를 기반으로 하며, 레이블링된 데이터($D_L$)와 레이블링되지 않은 데이터($D_U$)를 동시에 활용하여 세 가지 손실 함수($L_{SEG}, L_{UG}, L_{SS}$)를 통해 학습한다.

### 주요 구성 요소 및 절차

#### 1. Self-supervised Sub-task (Jigsaw Puzzles)

신경망의 인코더가 이미지의 구조적 정보를 깊게 학습하도록 하기 위해 **Jigsaw Puzzles** 태스크를 도입한다.

- 이미지를 $3 \times 3$ 형태의 9개 타일로 나누고, 이를 무작위로 섞고(translation) 회전(rotation)시킨다.
- 인코더는 섞인 타일들의 배치를 보고 원래의 순서를 맞추는 분류 작업을 수행하며, 이때의 손실 함수 $L_{SS}$는 Cross-entropy loss를 사용한다.

#### 2. Self-loop Uncertainty 생성 과정

의사 레이블 $y_{sl}$을 생성하는 절차는 다음과 같다 (Algorithm 1 참조).

1. 이미지 $x$에 대해 $Q$개의 서로 다른 퍼뮤테이션(permutation) $P'_i$를 선택한다.
2. 각 퍼뮤테이션에 대해 다음을 반복한다:
    - 퍼뮤테이션된 이미지를 네트워크에 통과시켜 퍼즐 정답을 예측($p_i$)하고, 동시에 분할 결과($S_i$)를 얻는다.
    - 퍼즐 예측 오차인 self-supervised loss $l_i$를 계산하고, 이를 통해 인코더 가중치 $\theta_e$를 업데이트한다.
3. 업데이트된 각 단계의 가중치는 서로 다른 모델의 역할을 수행하게 된다.
4. 최종 의사 레이블 $y_{sl}$은 각 단계의 분할 결과 $S_i$를 가중 평균하여 생성한다:
    $$y_{sl} = \sum_{i=1}^{Q} T^{-1}_{P'_i}(S_i) \times \text{norm}(\omega_i)$$
    여기서 가중치 $\omega_i$는 퍼즐 풀이의 난이도(손실 $l_i$)에 반비례하여 결정된다. 즉, 퍼즐을 잘 맞춘(손실이 낮은) 상태의 분할 결과에 더 높은 가중치를 부여한다.

#### 3. 손실 함수 및 학습 목표

- **Supervised Segmentation Loss ($L_{SEG}$)**: 레이블링된 데이터에 대해 실제 정답(Ground-truth)과 예측값 사이의 Binary Cross-entropy loss를 계산한다.
- **Uncertainty-guided Loss ($L_{UG}$)**: 레이블링되지 않은 데이터에 대해, 생성된 의사 레이블 $y_{sl}$ 중 신뢰도가 높은 영역(임계값 $th$ 이상)만을 대상으로 Mean Squared Error (MSE) loss를 계산한다:
    $$L_{UG}(S_x, y_{sl}) = \frac{\sum_{H \times W} I(y_{sl} > th) \| S_x - y_{sl} \|^2}{\sum_{H \times W} I(y_{sl} > th)}$$
- **Self-supervised Loss ($L_{SS}$)**: 퍼즐 맞추기 작업의 정확도를 높이기 위한 손실 함수이다.

### 최종 목적 함수 (Objective Function)

전체 손실 함수 $L$은 다음과 같이 정의된다:
$$L = \sum_{j=1}^{N} L_{SEG}(x_j, y_j) + \sum_{j=N+1}^{N+M} L_{UG}(x_j, y_{sl}) + \sum_{j=1}^{N+M} \sum_{i=1}^{Q} L_{SS}(T_{P'_i}(x_j), g_i)$$

**학습 절차의 특이점**: 레이블링되지 않은 데이터의 경우, 먼저 디코더를 고정하고 인코더를 $L_{SS}$로 반복 업데이트하여 $y_{sl}$을 생성한 뒤, 전체 네트워크를 $L_{UG}$로 최적화하는 비동기적(unsynchronized) 최적화 방식을 사용한다.

## 📊 Results

### 실험 설정

- **데이터셋**: MoNuSeg (세포핵 분할), ISIC (피부 병변 분할)
- **평가 지표**: F1 score
- **백본 네트워크**: ResUNet-18
- **비교 대상 (Baselines)**: Softmax probability map, MC Dropout, Network Ensemble

### 주요 결과

1. **의사 레이블 품질**: MoNuSeg 데이터셋에서 레이블링 데이터의 비율을 20%, 50%로 설정하여 실험한 결과, Self-loop uncertainty가 모든 베이스라인보다 높은 F1 score를 기록하였다. 특히 MC Dropout 대비 20% 데이터 환경에서 +2.27%, 50% 환경에서 +2.88% 성능 향상을 보였다.
2. **분할 성능**:
    - **MoNuSeg**: 레이블링 데이터가 20%일 때, 제안 방법은 Fully-supervised(100% 데이터 사용) 대비 성능 하락을 최소화하며, 오히려 기본 Fully-supervised(20% 데이터)보다 5.6% 높은 성능을 보였다. 50% 데이터 사용 시에는 100% 데이터 사용 결과(79.30%)와 거의 유사한 79.10%를 달성하였다.
    - **ISIC**: 마찬가지로 50% 데이터 사용 시 86.17%의 F1 score를 기록하여, 100% 데이터 사용 결과(86.58%)에 근접하는 성능을 보였다.
3. **효율성**: Ensemble 및 MC Dropout 방식은 테스트 단계에서도 여러 번의 추론이 필요하지만, Self-loop uncertainty는 단 한 번의 추론으로 결과를 낼 수 있어 계산 비용이 매우 낮다.

## 🧠 Insights & Discussion

본 논문은 자기지도 학습의 '전이 학습' 효과와 '모델 앙상블' 효과를 동시에 활용했다는 점에서 매우 영리한 설계를 보여준다.

**강점 및 해석**:

- **효율적인 앙상블 근사**: 물리적으로 여러 모델을 학습시키는 대신, 하나의 모델이 학습 과정에서 거치는 서로 다른 가중치 상태를 이용해 앙상블 효과를 낸 점이 혁신적이다.
- **데이터 효율성**: 50%의 데이터만으로도 100% 데이터 학습 결과에 근접한 성능을 낸 것은, 의료 현장에서 주석 비용을 절반으로 줄일 수 있는 실질적인 가능성을 제시한다.
- **상호 보완적 학습**: $L_{SS}$가 단순히 의사 레이블을 만드는 도구가 아니라, 인코더가 이미지의 일반적인 특징을 더 잘 추출하게 함으로써 분할 성능 자체를 끌어올리는 역할을 수행한다.

**한계 및 논의사항**:

- **퍼뮤테이션 수($Q$)의 영향**: 실험 결과 $Q$가 커질수록 성능이 향상되지만, 6에서 10으로 늘릴 때의 이득은 매우 적었다. 이는 최적의 $Q$ 값을 찾는 하이퍼파라미터 튜닝이 필요함을 시사한다.
- **태스크 의존성**: Jigsaw puzzle 외에 다른 자기지도 학습 태스크(예: 회전 예측, 색상화)를 적용했을 때도 동일한 효과가 나타날지에 대한 추가 검증이 필요하다.

## 📌 TL;DR

이 논문은 의료 영상 분할을 위해 **자기지도 학습(Jigsaw puzzle) 과정을 통해 모델의 불확실성을 추정하고, 이를 기반으로 고품질의 의사 레이블(Self-loop uncertainty)을 생성하는 준지도 학습 프레임워크**를 제안한다. 이 방법은 계산 비용이 큰 모델 앙상블의 성능을 단일 모델의 반복 최적화 과정으로 근사하여, 매우 적은 양의 레이블링된 데이터만으로도 풀-슈퍼바이즈드 학습에 근접하는 성능을 달성했다. 향후 의료 영상 분야에서 데이터 주석 비용을 획기적으로 줄이는 실용적인 방법론으로 활용될 가능성이 높다.
