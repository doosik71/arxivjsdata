# Virtual Agents in Live Coding: A Short Review

Anna Xambó (2021)

## 🧩 Problem to Solve

본 논문은 음악 라이브 코딩(Live Coding) 환경에서 가상 에이전트(Virtual Agents, VAs)의 역할과 가능성을 탐구한다. 라이브 코딩은 실시간으로 프로그램을 작성하여 음악을 생성하는 예술적 행위이나, 다음과 같은 현실적인 제약과 위험 요소가 존재한다.

첫째, 가청 가능한 코드를 생성하는 데 상당한 시간이 소요된다. 둘째, 즉흥 연주 과정에서 항상 영감이 존재하지 않는다는 의존성 문제가 있다. 셋째, 실시간으로 코드를 수정하는 특성상 코드 오류(Error)가 발생할 위험이 크다. 이러한 한계를 극복하기 위해 협력적 음악 라이브 코딩(Collaborative Music Live Coding, CMLC)이 대안으로 제시되지만, 항상 인간 협업자를 찾을 수 없는 상황이 발생한다.

따라서 본 연구의 목표는 과거부터 현재까지 라이브 코딩에 적용된 가상 에이전트들의 사례를 분석하고, 이를 통해 인간 라이브 코더의 한계를 보완할 수 있는 가상 에이전트의 방향성을 제시하는 것이다. 이는 저자의 후속 프로젝트인 'MIRLCAuto' 연구를 위한 비판적 맥락을 제공하는 데 목적이 있다.

## ✨ Key Contributions

본 논문의 핵심 기여는 라이브 코딩 내 가상 에이전트를 체계적으로 분류하고 분석하기 위한 이론적 프레임워크를 제공한 점이다. 특히 가상 에이전트를 단순히 '도구'로 보는 것이 아니라, 다음과 같은 관점에서 재정의한다.

1. **가상 에이전시(Virtual Agency)의 정립**: 에이전트를 환경을 감지하고 독립적으로 행동하는 자율적 시스템으로 정의하며, 이를 라이브 코딩의 'Liveness(생동감)' 개념과 연결한다.
2. **뮤지컬 사이보그(Musical Cyborg) 메타포**: 인간의 창의성과 디지털 알고리즘의 결합을 통해 인간 단독으로는 불가능한 사운드와 순간을 창조하는 '인간-에이전트'의 일대일 협력 관계를 제안한다.
3. **분석 차원의 제시**: 기존 가상 에이전트들을 **학습 가능성(Learnability)**과 **사회적 상호작용성(Social Interactivity)**이라는 두 가지 축으로 분류하여 각 시스템의 특성을 명확히 규명하였다.

## 📎 Related Works

본 논문은 가상 에이전트와 관련된 여러 학문적 배경을 검토한다.

- **Machine Musicianship**: AI 기술을 컴퓨터 음악에 적용하여 시스템이 스스로 학습하고 진화하게 하는 분야이다. Voyager, The Continuator, Shimon 등이 대표적이나, 라이브 코딩과의 접점은 상대적으로 덜 탐구되었다.
- **Musical Metacreation**: 음악적 창의성의 계산적 시뮬레이션을 연구하며, 인간과 창의적 가상 에이전트 간의 협업을 다룬다.
- **Computational Creativity Frameworks**: 창의성을 '새롭고 가치 있는 아이디어를 생성하는 능력'으로 정의하며, 이를 평가하기 위한 SPECs나 CSF와 같은 프레임워크가 존재한다.
- **Tanimoto의 Liveness 레벨**: 프로그래밍 과정의 생동감을 6단계로 구분한다. 특히 레벨 5(Tactically Predictive)와 레벨 6(Strategically Predictive)는 머신러닝을 통해 프로그래머의 다음 행동이나 의도를 예측하는 단계로, 이는 에이전트의 자율성과 직접적으로 연결된다.

## 🛠️ Methodology

본 연구는 2003년부터 2020년까지의 문헌 30건을 대상으로 NVivo 소프트웨어를 활용한 주제 분석(Thematic Analysis)을 수행하였다. 분석 대상이 된 가상 에이전트들은 다음의 두 가지 핵심 차원을 기준으로 평가된다.

### 1. 학습 가능성 (Learnability)

- **Offline Learning**: 미리 준비된 데이터셋을 통해 학습하고, 추론 단계에서는 고정된 모델을 사용한다.
- **Online Learning**: 실시간 상호작용 과정에서 환경이나 사용자로부터 지속적으로 학습하며 행동을 수정한다.

### 2. 사회적 상호작용성 (Social Interactivity)

상호작용의 방식은 다음과 같이 두 가지 범주로 나뉜다.

- **Summative Actions (합산적 작용)**: 개별 에이전트들의 기여가 단순히 합쳐지는 형태로, 상호 간의 영향력이 적은 독립적인 프로세스이다.
- **Multiplicative Actions (곱셈적 작용)**: 개별 기여분이 서로 밀접하게 의존하며, 상호작용을 통해 새로운 결과물이 도출되는 고도로 상호의존적인 프로세스이다.

## 📊 Results

논문은 분석 대상이 된 주요 가상 에이전트들의 특성을 다음과 같이 정리한다.

| 시스템 이름 | 학습 방식 | 상호작용 특성 | 핵심 기술 및 특징 |
| :--- | :--- | :--- | :--- |
| **Betablocker** | Evolutionary (Online) | Multiplicative | 유전 알고리즘, 게임패드 기반 제어, 자가 수정 프로그램 |
| **ixi lang (Autocode)** | No Learning | Summative | 결정론적 랜덤 행동, 단순 구문 기반 에이전트 생성 |
| **LOLbot** | No Learning | Multiplicative | 패턴 매칭 알고리즘, 인간 연주자의 패턴 분석 및 반응 |
| **Flock** | Online | Multiplicative | Machine Listening (SCMIR), 투표 기반 오디오 믹스 제어 |
| **Cacharpo** | Offline | Summative | ANN (Haskell), 쿰비아 소니데라 장르 특화, 오디오 분석 |
| **Cibo / Cibo v2** | Offline | Solo (N/A) | Seq2Seq (PyTorch), TidalCycles 코드 생성, 전술적 예측 |
| **Autopia** | Genetic (Online) | Multiplicative | 유전 알고리즘, Machine Listening 기반 적합도 평가 |
| **Mégra** | Online | Cyborg-style | Probabilistic Finite Automata, 실시간 데이터 학습 및 시각화 |

### 주요 결과 분석

- **학습 경향**: 많은 시스템이 오프라인 학습(Cibo, Cacharpo)에 의존하고 있으나, 최근에는 Mégra와 같이 학습 과정 자체를 퍼포먼스의 일부로 포함하는 온라인 학습으로의 전환이 관찰된다.
- **상호작용 수준**: 단순한 합산적 작용에서 벗어나, 환경과 타 에이전트의 상태를 반영하는 곱셈적 작용(Betablocker, Autopia)으로 발전하고 있다.

## 🧠 Insights & Discussion

### 강점 및 가치

본 논문은 파편화되어 있던 라이브 코딩 에이전트 사례들을 '자율성'과 '학습'이라는 일관된 기준으로 정리하여 학술적 지도를 제공하였다. 특히 '뮤지컬 사이보그'라는 개념을 통해 인간과 AI의 관계를 단순한 대체재가 아닌 확장된 정체성으로 해석한 점이 돋보인다.

### 한계 및 미해결 과제

- **평가 방법의 부재**: 제시된 시스템들이 음악적으로나 계산적으로 어떻게 우수한지 측정할 수 있는 공식적인 평가 방법론(Formal Evaluation)이 부족하다.
- **재현성 문제**: 코드와 데이터의 문서화가 미비하여 연구의 재현성이 떨어진다는 점이 지적된다.
- **가독성 문제**: 가상 에이전트가 생성한 코드가 관객이나 인간 코더에게 어떻게 읽혀야 하는지에 대한 '코드 가독성(Legibility)' 문제가 여전히 남아 있다.

### 비판적 해석

현재의 가상 에이전트들은 주로 인간의 스타일을 모방하거나(Cibo), 단순한 규칙에 따라 반응하는 수준에 머물러 있다. 진정한 의미의 '전략적 예측(Level 6 Liveness)'에 도달하기 위해서는 단순한 패턴 매칭을 넘어 음악적 문맥과 의도를 이해하는 고도의 모델링이 필요할 것이다. 또한, VA-VA 간의 완전한 자율 협업이 가능해질 때 발생할 기계 윤리(Machine Ethics) 문제에 대한 선제적 논의가 필요하다.

## 📌 TL;DR

본 논문은 음악 라이브 코딩 분야의 가상 에이전트(VA)들을 **학습 가능성**과 **사회적 상호작용성**이라는 두 축으로 분석한 리뷰 논문이다. 저자는 VA가 인간 코더의 한계를 보완하는 '뮤지컬 사이보그'로서 기능할 수 있음을 시사하며, 향후 연구 방향으로 온라인 학습의 확대, 코드 가독성 개선, 그리고 VA 간의 자율적 협업 가능성을 제시한다. 이 연구는 향후 실시간 음악 생성 AI의 자율성과 협업 모델을 설계하는 데 중요한 이론적 기초가 될 것으로 보인다.
