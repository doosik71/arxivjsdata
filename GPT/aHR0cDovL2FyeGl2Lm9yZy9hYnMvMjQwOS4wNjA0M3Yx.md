# Identifying the sources of ideological bias in GPT models through linguistic variation in output

Christina P. Walker, Joan C. Timoneda (2024)

## 🧩 Problem to Solve

본 연구는 GPT-3.5와 GPT-4와 같은 생성형 AI 모델에서 나타나는 **이데올로기적 편향(Ideological Bias)**의 근원을 식별하는 것을 목표로 한다. 기존 연구들은 생성형 AI가 사회적 고정관념과 편향을 영속시킨다는 점을 지적해 왔으나, 구체적으로 어떤 경로를 통해 정치적·이데올로기적 편향이 발생하는지에 대한 탐구는 부족했다.

특히, 모델의 출력 결과가 단순히 방대한 양의 **학습 데이터(Training Data)**에서 기인한 것인지, 아니면 모델 출시 전후에 적용된 **필터링 알고리즘(Filtering Algorithm)**에 의한 것인지를 구분하는 것이 핵심 문제이다. 이러한 편향은 AI를 활용한 사회과학 연구의 타당성을 훼손할 뿐만 아니라, AI를 통해 정보를 습득하는 개인의 정치적 행동과 의사결정에 영향을 미칠 수 있다는 점에서 매우 중요하다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 특정 국가의 정치적 성향과 밀접하게 연관된 **언어적 변이(Linguistic Variation)**를 활용하여 편향의 소스를 추적하는 것이다.

특정 이슈에 대해 사회적 합의나 정치적 성향이 뚜렷하게 갈리는 언어 쌍(예: 보수적인 폴란드어 vs 진보적인 스웨덴어)을 선택하여 모델의 응답을 비교함으로써, 모델이 학습한 데이터의 편향성을 측정한다. 또한, 필터링 정책이 강화된 GPT-4와 그렇지 않은 GPT-3.5를 비교 분석하여, 학습 데이터에 의한 편향(Training Bias)과 알고리즘에 의해 강제된 편향(Algorithmic Bias)을 분리하여 식별하는 방법론을 제시하였다.

## 📎 Related Works

기존의 LLM 편향 연구들은 주로 성별, 인종과 같은 사회적 고정관념에 집중해 왔으며, GPT-3가 특정 인종이나 종교에 대해 폭력적인 출력을 생성하거나 성별에 따른 직업 고정관념을 보인다는 점을 밝혀냈다.

그러나 본 논문은 기존 연구들이 다음과 같은 한계가 있다고 주장한다.

1. **영어 중심의 분석**: 대부분의 연구가 영어 데이터에 치중되어 있어, 전 세계의 다양한 사회적 가치와 태도를 모델 출력과 매칭하기 어려웠다.
2. **근원 식별 불가**: 편향이 발견되었다 하더라도, 그것이 학습 데이터의 특성 때문인지 아니면 OpenAI의 후처리 필터링 때문인지 구분하는 방법론이 부재했다.
3. **블랙박스 특성**: GPT 모델의 폐쇄적인 구조로 인해 내부 작동 방식이나 구체적인 학습 데이터셋을 알 수 없어, 출력 결과만으로 편향을 추론해야 하는 한계가 있다.

## 🛠️ Methodology

### 전체 파이프라인 및 실험 설계

연구진은 정치적으로 민감하며 지리적·언어적으로 제약된 두 가지 이슈를 선정하여 실험을 설계하였다.

1. **낙태(Abortion)**: 사회적으로 보수적인 **폴란드어**, 진보적인 **스웨덴어**, 그리고 미국 중심의 데이터가 많은 **영어**를 사용하여 응답을 수집하였다.
2. **카탈루냐 독립(Catalan Independence)**: 독립 찬성 성향이 강한 **카탈루냐어**와 반대 성향이 강한 **스페인어**를 사용하여 응답을 수집하였다.

각 모델(GPT-3.5, GPT-4)과 언어별로 500개의 독립적인 응답을 생성하였으며, 프롬프트는 문장 완성형(예: "낙태를 한 여성은 $\dots$")으로 구성하였다. 생성된 텍스트는 두 명의 코더(Coder)가 수동으로 레이블링(예: 낙태 찬성 여부, 독립 반대 여부)하여 정량화하였다.

### 분석 방법: 다층 모델링 (Multilevel Modeling, MLM)

데이터가 '텍스트'와 '모델'이라는 계층적 구조로 이루어져 있기 때문에, 연구진은 일반적인 로지스틱 회귀 분석 대신 **다층 모델링(MLM)**을 사용하였다. 이는 모델 유형에 따른 집단 간 이질성을 캡처하고 표준 오차의 왜곡을 방지하기 위함이다.

주요 방정식은 다음과 같다:
$$Y_i = \beta X_{ij} + \gamma_0 + \gamma_1 Z_j + \epsilon_{ij} + \mu_j$$

여기서 각 변수의 의미는 다음과 같다:

- $Y_i$: 범주형 결과 변수 (예: 진보적 응답 여부).
- $X_{ij}$: 텍스트 수준의 예측 변수 (예: 사용된 언어).
- $Z_j$: 그룹 수준의 공변량 (예: GPT 모델 버전).
- $\beta$: 텍스트 수준 회귀 계수.
- $\gamma_0$: 전체 모델 수준의 절편 (Fixed effect).
- $\gamma_1$: 그룹 수준의 효과 (Random effect).
- $\epsilon_{ij}, \mu_j$: 각각 텍스트 수준과 그룹 수준의 오차항.

최종적으로 로짓 링크 함수(Logit-link function)를 적용한 모델을 통해, 특정 언어와 모델 조합이 진보적/보수적 응답을 생성할 확률을 추정하였다.

## 📊 Results

### 1. 낙태 이슈 (Abortion)

- **학습 데이터 편향**: GPT-3.5와 GPT-4 모두에서 스웨덴어 응답이 폴란드어 응답보다 훨씬 더 진보적이었다. 특히 GPT-3.5에서 진보적 응답 확률은 폴란드어 0.434, 스웨덴어 0.534로 약 23%의 차이를 보였다.
- **알고리즘 편향**: GPT-4는 모든 언어에서 GPT-3.5보다 훨씬 진보적인 경향을 보였다. 특히 영어의 경우 GPT-4는 95.9%의 확률로 낙태 찬성(Pro-choice) 응답을 생성하며, 이는 OpenAI의 강력한 필터링 정책이 반영된 결과로 해석된다.

### 2. 카탈루냐 독립 이슈 (Catalan Independence)

- **학습 데이터 편향**: 카탈루냐어 응답은 독립에 긍정적인 반면, 스페인어 응답은 독립에 부정적인 경향이 뚜렷했다. GPT-3.5에서 스페인어의 독립 반대 응답 확률은 61.15%였으나, 카탈루냐어는 31.08%에 불과했다.
- **알고리즘 편향**: GPT-4에서는 두 언어 모두에서 극단적인 가치 판단(예: "불법", "재앙")이 사라지고 훨씬 중립적이거나 민주적 절차를 강조하는 방향으로 이동하였다. 결과적으로 GPT-4는 GPT-3.5보다 더 친-독립(Pro-independence)적인 편향을 보였다.

### 정량적 결과 요약

| 분석 대상 | 결과 (GPT-3.5 $\rightarrow$ GPT-4 변화) | 결론 |
| :--- | :--- | :--- |
| **낙태 (폴란드어/스웨덴어)** | 언어 간 격차 유지 $\rightarrow$ 전체적으로 진보화 | 학습 데이터 편향 + 알고리즘 편향 모두 존재 |
| **낙태 (영어)** | 보수/중립 $\rightarrow$ 거의 완전한 진보(95.9%) | 강력한 알고리즘 편향 |
| **카탈루냐 독립** | 언어 간 격차 유지 $\rightarrow$ 중립 및 친-독립화 | 학습 데이터 편향 + 알고리즘 편향 모두 존재 |

## 🧠 Insights & Discussion

본 연구는 생성형 AI의 편향이 단일한 소스가 아닌, **학습 데이터**와 **필터링 알고리즘**이라는 두 가지 경로를 통해 생성됨을 입증하였다.

**강점 및 시사점:**

- **데이터 편향의 영속성**: OpenAI가 GPT-4에서 정교한 필터링을 도입했음에도 불구하고, 폴란드어와 스웨덴어 사이의 이데올로기적 격차가 여전히 존재한다는 점은 학습 데이터에 내재된 편향이 쉽게 제거되지 않음을 의미한다.
- **필터링의 역설**: 후처리 필터링(Post-training filtering)은 기존의 편향을 완전히 제거하기보다는, 오히려 특정 방향(본 연구에서는 진보적/민주적 방향)으로 새로운 알고리즘 편향을 주입하는 결과를 초래한다.

**한계 및 비판적 해석:**

- **블랙박스 접근의 한계**: 모델 내부의 가중치나 구체적인 필터링 로직을 직접 확인하지 못하고 출력값만으로 추론했기 때문에, 추정치에 의존할 수밖에 없다.
- **언어 선택의 제약**: 분석에 사용된 언어들이 해당 국가의 정치적 성향을 완벽하게 대표하는지에 대한 추가 검증이 필요할 수 있다.

결론적으로, 모델의 편향을 줄이기 위해서는 출시 후의 알고리즘적 보정보다, 학습 단계에서부터 **고품질의 큐레이션된 데이터셋(High-quality, curated datasets)**을 사용하는 것이 근본적인 해결책임을 시사한다.

## 📌 TL;DR

본 논문은 서로 다른 정치적 성향을 가진 국가의 언어(예: 폴란드어 vs 스웨덴어)를 이용해 GPT 모델의 이데올로기적 편향을 분석하였다. 연구 결과, 모델은 학습 데이터에 내재된 사회적 편향을 그대로 반영(Training Bias)할 뿐만 아니라, OpenAI의 후처리 필터링으로 인해 특정 정치적 방향으로 편향(Algorithmic Bias)되는 경향이 있음을 확인하였다. 이는 단순히 필터링을 강화하는 것만으로는 편향을 제거할 수 없으며, 오히려 새로운 편향을 생성할 수 있음을 경고한다. 따라서 향후 AI 연구 및 개발은 데이터 큐레이션 단계에서의 품질 관리에 집중해야 한다.
