# Deep Learning for Image Denoising: A Survey

Chunwei Tian, Yong Xu, Lunke Fei, and Ke Yan (2018)

## 🧩 Problem to Solve

본 논문은 이미지 처리의 핵심 분야인 이미지 디노이징(Image Denoising) 기술, 특히 최근 급격히 발전한 딥러닝 기반의 접근 방식들을 체계적으로 검토하고 분석하는 것을 목표로 한다.

이미지 디노이징은 잡음이 섞인 이미지에서 원래의 깨끗한 이미지를 복원하는 작업이다. 이는 이미지 세그멘테이션, 분류, 객체 탐지 등 다양한 후속 이미지 처리 작업의 성능을 결정짓는 중요한 전처리 단계이다. 기존의 전통적인 머신러닝 기반 방법들은 우수한 성능을 보였으나, 다음과 같은 두 가지 주요 문제점이 존재하였다.

첫째, 상당수의 전통적 방법들이 비볼록(non-convex) 최적화 문제에 기반하고 있어 사용자가 수동으로 하이퍼파라미터를 설정해야 하는 번거로움이 있다. 둘째, 테스트 단계에서 복잡한 최적화 과정을 거쳐야 하므로 계산 비용이 매우 높다는 단점이 있다. 따라서 본 논문은 이러한 한계를 극복하기 위해 등장한 딥러닝 기반 디노이징 기술들의 원리와 성능, 차이점을 분석하고 향후 연구 방향을 제시하고자 한다.

## ✨ Key Contributions

본 논문의 중심적인 기여는 최신 딥러닝 기반 이미지 디노이징 기술들을 종합적으로 리뷰하여 연구자들에게 가이드라인을 제공하는 것이다. 특히 다음과 같은 설계 아이디어와 흐름을 강조한다.

- **수동 설계에서 자동 학습으로의 전환**: 사람이 직접 설계한 이미지 사전 정보(Image Prior) 대신, 딥러닝의 깊은 구조를 통해 데이터로부터 최적의 특징(feature)을 자동으로 학습하는 방식의 이점을 분석한다.
- **잔차 학습(Residual Learning)의 도입**: 이미지 전체를 복원하는 대신 잡음 성분만을 예측하여 제거하는 잔차 학습 개념이 디노이징 성능 향상에 어떻게 기여했는지 설명한다.
- **연산 효율성 증대**: GPU 가속과 Batch Normalization(BN) 등의 기법을 통해 전통적인 최적화 기반 방법들보다 추론 속도를 획기적으로 개선한 점을 조명한다.

## 📎 Related Works

### 전통적인 이미지 디노이징 방법
논문에서는 다음과 같은 전통적인 접근 방식들을 언급한다.
- **Non-local algorithm**: 이미지 내의 유사한 패치들을 찾아 활용하는 방식이다.
- **BM3D**: 유사한 2차원 이미지 조각들을 3차원 데이터 배열로 변환하여 희소성(sparsity)을 이용해 잡음을 제거한다.
- **MRF(Markov Random Fields)**: 신념 전파(belief propagation) 추론 방식을 융합하여 문제를 해결한다.

### 기존 방식의 한계 및 차별점
전통적인 방법들은 특정 형태의 사전 정보(prior)에 국한되어 있으며, 앞서 언급한 수동 파라미터 설정 문제와 높은 계산 비용이라는 명확한 한계가 있다. 반면, 딥러닝 기반 방식은 End-to-End 구조를 통해 특징 추출과 복원을 동시에 수행하며, 한 번 학습된 모델은 테스트 단계에서 단순한 순전파(forward pass)만으로 빠르게 결과를 낼 수 있다는 점에서 차별화된다.

## 🛠️ Methodology

### 기본 수학적 모델
이미지 디노이징의 기본 목표는 잡음이 섞인 이미지 $y$로부터 깨끗한 이미지 $x$를 얻는 것이다. 이는 다음과 같은 방정식으로 정의된다.
$$y = x + n$$
여기서 $n$은 분산이 $\sigma^2$인 가산 백색 가우시안 잡음(Additive White Gaussian Noise, AWGN)을 의미한다.

### 핵심 네트워크 구조: ResNet (Residual Network)
딥러닝 디노이징의 기반이 되는 ResNet의 핵심은 잔차 블록(Residual Block)이다. 네트워크의 깊이가 깊어질 때 발생하는 기울기 소실(vanishing gradient) 및 폭주(exploding gradient) 문제를 해결하기 위해 다음과 같은 구조를 사용한다.
$$\text{Output} = f(x) + x$$
여기서 $x$는 입력값, $f(x)$는 가중치 층과 활성화 함수를 거친 결과물이다. 입력 $x$를 출력에 직접 더해주는 identity mapping을 통해 더 깊은 네트워크에서도 안정적인 학습이 가능하며, 오버피팅을 억제하고 학습 속도를 높일 수 있다.

### 주요 딥러닝 디노이징 모델
논문에서는 구체적인 디노이징 모델들의 특징을 다음과 같이 설명한다.

1. **DnCNN**:
   - BN(Batch Normalization)과 ResNet 구조를 활용한다.
   - 깨끗한 이미지를 직접 예측하는 것이 아니라, 잡음 성분(residual image)을 예측하도록 설계되었다. 테스트 단계에서는 입력 이미지 $y$에서 예측된 잡음을 빼서 $x$를 얻는다.

2. **FFDNet**:
   - 입력으로 잡음 이미지뿐만 아니라 잡음 수준 맵(noise level map)을 함께 전달한다.
   - 이를 통해 하나의 모델로 다양한 잡음 수준($\sigma$)을 처리할 수 있는 유연성을 확보하였다.

3. **IRCNN**:
   - 모델 기반 최적화 방법과 CNN을 융합한 구조이다.
   - Dilated Convolution(확장 컨볼루션)을 도입하여 수용 영역(receptive field)을 넓힘으로써 디노이징 성능을 향상시켰다.

## 📊 Results

### 실험 설정 및 지표
- **데이터셋**: BSD68 데이터셋을 주로 사용하였다.
- **평가 지표**: PSNR(Peak Signal-to-Noise Ratio)을 사용하여 복원 성능을 정량적으로 측정하였다.
- **비교 대상**: BM3D, WNNM, TNRD 등 전통적 방법과 DnCNN, FFDNet, IRCNN 등 딥러닝 방법들을 비교하였다.

### 주요 결과
잡음 수준 $\sigma = 25$인 조건에서 각 방법론의 PSNR 결과는 다음과 같다 (Table 1 참조).

- **전통적 방법**: BM3D(28.57), WNNM(28.83), TNRD(28.92) 순으로 성능이 나타났다.
- **딥러닝 방법**: DnCNN(29.23), FFDNet(29.19), IRCNN(29.15), DDRN(29.18) 순으로 나타났다.

결과적으로 모든 딥러닝 기반 방법들이 전통적인 방법들보다 높은 PSNR 수치를 기록하며 우수한 성능을 입증하였다. 특히 DnCNN이 해당 실험 조건에서 가장 높은 성능을 보였다. 또한 FFDNet의 경우, 정성적 결과(Fig 3)에서 CBM3D보다 시각적으로 더 깨끗한 복원 성능을 보여주었다.

## 🧠 Insights & Discussion

### 강점 및 성과
딥러닝 기반 디노이징은 깊은 아키텍처를 통해 복잡한 이미지 특징을 효과적으로 추출하며, GPU를 이용한 병렬 연산 덕분에 추론 속도가 매우 빠르다. 특히 잔차 학습의 도입은 학습의 안정성과 성능을 동시에 잡은 핵심적인 전략으로 평가된다.

### 한계 및 미해결 과제
논문은 현재 기술의 한계점을 다음과 같이 지적한다.
1. **잡음 종류의 제한**: 대부분의 연구가 가산 백색 가우시안 잡음(AWGN)에만 집중되어 있어, 저조도 이미지에서 발생하는 실제 잡음(real-world noise) 제거에는 효과적이지 않다.
2. **작업의 파편화**: 디노이징, 초해상도(super-resolution), 블러 제거(deblurring) 등의 저수준 비전 작업들을 하나의 통합된 모델로 처리하지 못하고 각각 별도의 모델을 사용하고 있다.
3. **Blind Denoising의 어려움**: 잡음의 수준($\sigma$)을 모르는 상태에서 처리하는 blind Gaussian noise 제거 능력이 여전히 부족하다.

### 비판적 해석
본 논문은 다양한 최신 기법들을 잘 정리하고 있으나, 개별 모델의 손실 함수(loss function)나 구체적인 하이퍼파라미터에 대한 상세 분석보다는 결과적인 성능 수치 비교에 치중한 경향이 있다. 하지만 서베이 논문으로서 딥러닝 디노이징의 흐름을 파악하고 진입 장벽을 낮추는 데 유용한 지표를 제공한다.

## 📌 TL;DR

본 논문은 전통적인 최적화 기반 이미지 디노이징의 한계를 극복하기 위한 딥러닝 기술들을 종합적으로 분석한 서베이 보고서이다. ResNet의 잔차 학습 구조가 디노이징 성능 향상의 핵심임을 밝히고, DnCNN, FFDNet 등 주요 모델들이 기존 BM3D 등의 방법보다 높은 PSNR과 빠른 속도를 보임을 입증하였다. 향후 연구는 실제 환경의 잡음 처리와 여러 저수준 비전 작업을 통합 수행하는 범용 모델 개발 방향으로 나아가야 함을 시사한다.
