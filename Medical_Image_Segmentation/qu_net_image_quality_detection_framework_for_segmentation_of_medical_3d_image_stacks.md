# QU-net++: Image Quality Detection Framework for Segmentation of Medical 3D Image Stacks

Sohini Roychowdhury (2022)

## 🧩 Problem to Solve

의료 3D 이미지 스택(image stacks)에서 병변 부위를 자동으로 분할(segmentation)하는 것은 진단 및 사후 관리에 매우 중요하다. 하지만 정밀한 분할 모델을 학습시키기 위해서는 고품질의 어노테이션(annotation) 데이터가 필요하며, 의료 전문가가 직접 수행하는 이 과정은 막대한 비용과 시간이 소요된다.

본 논문의 목표는 전체 이미지 데이터셋 중에서 모델 학습에 반드시 필요한 최소한의 이미지 서브셋(minimal training subset)을 자동으로 탐지하는 프레임워크를 제안하는 것이다. 즉, 모든 데이터를 어노테이션하는 대신, 학습 효율이 높은 핵심 데이터만을 선별하여 어노테이션 비용을 획기적으로 줄이면서도 높은 분할 성능을 유지하고자 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 이미지의 통계적 특성과 모델의 내부 예측 불일치(disagreement)를 결합한 2단계 데이터 선별 방식이다.

1. **2단계 이미지 품질 분석 방법론**: 먼저 무감독(unsupervised) 방식으로 초기 학습 세트를 구성하고, 이후 U-Net++ 모델의 딥 슈퍼비전(deep supervision) 구조를 활용하여 모델이 아직 학습하지 못한 특성을 가진 이미지를 식별해 내는 방식을 제안한다.
2. **모달리티 간 전이 학습 가능성 입증**: 제안된 QU-Net++ 모델이 CT(Computed Tomography)와 OCT(Optical Coherence Tomography)라는 서로 다른 의료 영상 모달리티 간에도 커널 가중치가 확장(scale)될 수 있음을 보여줌으로써, 미세 조정(fine-tuning)에 필요한 샘플 수를 더욱 줄일 수 있음을 증명하였다.

## 📎 Related Works

기존의 의료 영상 분할 연구들은 주로 수동 선택(manual selection), 무작위 샘플링(random sampling), 또는 이미 학습된 ML 모델을 사용하여 학습 데이터를 선정해 왔다. 하지만 이러한 방식은 데이터의 실제 품질이나 모델이 필요로 하는 정보의 다양성을 충분히 반영하지 못한다는 한계가 있다. 본 연구는 전이 학습(transfer learning)의 흐름을 따라 소량의 데이터로 최적의 성능을 내는 것에 집중하며, 단순 샘플링이 아닌 '이미지 품질'과 '모델의 예측 불분명성'을 기준으로 데이터를 선별한다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 1. 초기 학습 이미지 세트($S_0$) 선정

모델을 처음 학습시키기 위해, 어노테이션이 필요한 최소한의 초기 이미지 세트를 무감독 방식으로 선정한다.

- **Blurriness 측정**: Laplacian 연산자의 픽셀 분산을 이용하여 이미지의 흐림 정도를 측정한다. 분산 값이 낮을수록 흐린 이미지로 간주한다.
- **Contrast 측정**: 이미지에서 중간값 필터링(median filtering)을 적용한 이미지와의 차이 이미지 분산을 최대 픽셀 강도로 나눈 $\text{PSNR-inv}$ 지표를 사용한다. 값이 높을수록 대비가 낮음을 의미한다.
- **ROI 품질 분석**: 위 두 지표를 통해 1차 선별된 이미지 중, 관심 영역(ROI) 내의 변동 계수(Coefficient of Variation, CoV)와 평균 픽셀 값을 분석하여 서로 너무 유사한 샘플을 제거함으로써, 최종적으로 10장 미만의 매우 적은 초기 세트 $S_0$를 구성한다.

### 2. QU-Net++ 아키텍처

본 연구는 U-Net++ 모델을 기반으로 한다. U-Net++는 기존 U-Net의 skip connection을 중첩된 밀집 블록(nested dense blocks)으로 재설계하여 다중 스케일 특징을 더 효과적으로 활용한다.

- **구조**: 4레벨의 인코더-디코더 구조를 가지며, 각 레벨의 출력이 서로 연결되어 신호 강도를 증폭시킨다.
- **손실 함수**: 학습에는 다음과 같은 Negative Dice Coefficient Loss를 사용한다.
$$\text{Loss} = -\sum_{k=1}^{l_p} \left( \frac{2 \cdot [P(k) \cup Y(k)]}{[P(k) + Y(k) + 1]} \right)$$
여기서 $l_p$는 전체 픽셀 수, $P$는 예측 결과, $Y$는 정답(ground-truth)을 의미한다.

### 3. 최소 학습 데이터셋($S_m$) 탐지 절차

초기 세트 $S_0$로 학습된 모델을 사용하여 나머지 이미지들에 대해 '학습 품질'을 평가한다.

- **딥 슈퍼비전 활용**: U-Net++의 마지막 두 단계인 레벨 3($L_{3,r}$)과 레벨 4($L_4$)의 출력 마스크를 비교한다.
- **품질 지수($Q_j$) 계산**: 두 레이어의 출력 간의 Jaccard Score(Intersection-over-Union)를 통해 이미지 콘텐츠의 품질을 측정한다.
$$Q_j = \frac{L(3,r) \cap L(4)}{L(3,r) \cup L(4)} = J(L(3,r), L(4))$$
- **데이터 추가 조건**: 만약 $Q_j$ 값이 특정 임계값 $q_0$보다 낮다면, 모델이 해당 이미지의 특성을 제대로 학습하지 못한 것으로 간주하여 이를 학습 세트 $S_m$에 추가하고 다시 학습시킨다.

## 📊 Results

### 실험 설정

- **데이터셋**: Lung CT (COVID-19 GGO 분할) 및 OCT (낭종 분할) 3D 이미지 스택.
- **비교 대상**: 무작위 샘플링(Random Sampling, 전체의 25%)을 통한 U-Net 및 U-Net++ 학습 결과와 기존 문헌의 결과와 비교하였다.
- **평가 지표**: Precision(Pr), Recall(Re), Jaccard Score(J), Dice Score(D), Accuracy(Acc).

### 주요 결과

- **데이터 효율성**: 제안된 방법론은 전체 이미지의 약 8~15%만을 학습에 사용하여도 매우 높은 성능을 보였다.
- **정량적 성과**:
  - **Lung-CT-med**: QU-net++는 단 15장의 이미지로 Dice Score 0.56을 달성하여, 무작위 25% 샘플링을 사용한 U-Net++(0.55)보다 우수한 성능을 보였다.
  - **Lung-CT-rad**: 40장의 이미지로 Dice Score 0.64를 기록하며, 무작위 샘플링(0.62) 대비 성능 향상을 이루었다.
  - **OCT Stacks**: 67장의 이미지로 Dice Score 0.72를 기록하여, 무작위 샘플링(0.60) 대비 압도적인 성능 향상을 보였다.
- **결론적으로**, 무작위로 많은 데이터를 쓰는 것보다 이미지 품질과 모델의 불확실성에 기반해 선별된 적은 양의 데이터를 학습시키는 것이 더 효과적임을 입증하였다.

## 🧠 Insights & Discussion

### 강점

본 연구는 의료 데이터 어노테이션의 고비용 문제를 해결하기 위해 매우 실용적인 접근 방식을 취하였다. 특히 모델 내부의 서로 다른 레벨 간의 예측 불일치를 '학습 부족'의 지표로 활용한 점이 독창적이다. 또한, 서로 다른 모달리티(CT, OCT)에서도 동일한 프레임워크가 작동함을 보여줌으로써 범용성을 입증하였다.

### 한계 및 논의사항

- **이진 분할의 한계**: 현재 제안된 모델은 단일 클래스 분할(binary segmentation)에 집중되어 있다. 저자 또한 향후 연구에서 다중 클래스 분할(multi-class segmentation)로 확장할 필요가 있음을 언급하였다.
- **임계값 설정**: 품질 지수 $Q_j$를 결정하는 임계값 $q_0$가 성능에 미치는 영향에 대한 상세한 분석은 부족하며, 이 값이 어떻게 최적화되었는지 명시되지 않았다.
- **데이터셋 규모**: 사용된 데이터셋의 규모가 아주 크지 않아, 대규모 데이터셋에서도 동일한 비율(8~15%)의 데이터만으로 충분한 성능이 나올지는 추가 검증이 필요하다.

## 📌 TL;DR

이 논문은 의료 3D 영상 분할을 위해 어노테이션 비용을 최소화하는 **QU-net++** 프레임워크를 제안한다. 이미지의 통계적 품질 분석과 U-Net++의 딥 슈퍼비전 레이어 간 예측 불일치(Jaccard score)를 이용해 **전체 데이터의 8~15%만으로도 최적의 학습 세트를 구성**할 수 있음을 보였다. 이는 향후 다양한 의료 영상 모달리티에서 적은 비용으로 고성능 분할 모델을 구축하는 데 기여할 수 있을 것으로 기대된다.
