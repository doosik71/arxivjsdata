# Foundation AI Model for Medical Image Segmentation

Rina Bao, Erfan Darzi, Sheng He, Chuan-Heng Hsiao, Mohammad Arafat Hussain, Jingpeng Li, Atle Bjørnerud, Ellen Grant, and Yangming Ou (2024)

## 🧩 Problem to Solve

현재 의료 영상 분할(Medical Image Segmentation) 분야에서는 각 장기나 질병, 혹은 특정 데이터셋에 맞춘 수천 개의 태스크 특화형(task-specific) AI 모델을 개발하는 것이 일반적인 관행이다. 그러나 이러한 접근 방식은 다음과 같은 심각한 효율성 문제를 야기한다.

첫째, 새로운 태스크가 등장할 때마다 매번 새로운 모델을 설계하고 학습시켜야 하므로 자원 소모가 극심하다. 둘째, 의료 데이터의 특성상 충분한 양의 학습 데이터를 확보하기 어려운 경우가 많아, 데이터셋이 너무 작은 태스크에서는 효과적인 모델 학습이 불가능하다.

따라서 본 논문의 목표는 자연어 처리의 ChatGPT나 일반 이미지 분할의 SAM(Segment Anything Model)과 같이, 방대한 데이터를 통해 학습되어 다양한 태스크에 광범위하게 적용될 수 있는 '하나의 모델로 다수를 해결하는(one-for-many 또는 one-for-all)' 의료 영상 분할용 파운데이션 모델(Foundation Model)의 구축 방향성과 가능성을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 의료 영상 분할을 위한 파운데이션 모델을 구축하기 위한 두 가지 전략적 경로와 모델의 분류 체계를 제안한 점이다.

중심적인 아이디어는 이미 검증된 자연 이미지 파운데이션 모델을 의료 영상에 맞게 적응(Adaptation)시키거나, 혹은 의료 영상 데이터만을 사용하여 처음부터 의료 전용 파운데이션 모델을 구축하는 것이다. 또한, 모든 의료 영상을 통합하는 '제너럴리스트(Generalist)' 모델과 특정 속성을 공유하는 영상들로 범위를 좁힌 '스페셜리스트(Specialist)' 모델이라는 개념을 도입하여, 의료 영상의 높은 다양성을 체계적으로 해결하려는 설계 방향을 제시하였다.

## 📎 Related Works

본 논문에서는 자연 이미지 분할의 파운데이션 모델인 SAM(Segment Anything Model)을 주요 관련 연구로 다룬다. SAM은 1,100만 개의 자연 이미지와 10억 개 이상의 마스크 데이터로 학습되어, 새로운 이미지에 대해서도 추가 학습 없이 분할이 가능한 zero-shot 성능을 보여주었다.

그러나 이를 의료 영상에 직접 적용했을 때의 한계가 명확히 제시되었다. 의료 영상에서는 태스크 특화형 모델에 비해 Dice coefficient(분할 정확도 지표)가 $0.5 \sim 0.7$ 정도 낮게 측정되는 결과가 보고되었다. 이는 의료 영상 특유의 낮은 대비(low-contrast), 불규칙한 형태, 3D 데이터의 특성 및 자연 이미지와는 완전히 다른 텍스처와 대비 특성 때문이다. 이를 해결하기 위해 DeSAM, AutoSAM, All-in-SAM과 같이 프롬프트 입력을 자동화하거나 SAM의 일부 가중치를 미세 조정(fine-tuning)하려는 시도들이 이어지고 있다.

## 🛠️ Methodology

논문은 의료 영상 분할 파운데이션 모델을 구축하고 적용하기 위한 방법론적 프레임워크를 다음과 같이 제안한다.

### 1. 데이터 소스에 따른 구축 경로

- **자연 이미지 기반 경로:** SAM과 같은 기존 모델을 활용한다. 직접 적용, 의료 데이터로의 미세 조정(Fine-tuning), 혹은 프롬프트 자동화 기술을 통해 의료 영상에 적응시킨다.
- **의료 이미지 전용 경로:** 의료 영상 데이터만을 사용하여 모델을 처음부터 학습시킨다.

### 2. 모델의 유형 분류 (Generalist vs Specialist)

- **Generalist Model:** 영상의 모달리티, 차원, 장기, 태스크에 관계없이 모든 의료 영상 데이터를 통합하여 학습한 통합 모델이다.
- **Specialist Model:** 다음과 같이 유사한 속성을 가진 그룹으로 한정하여 학습한 모델이다.
  - **동일 장기:** 예) 뇌 또는 심장 전용 모델 (다양한 질병 및 모달리티 포함).
  - **동일 모달리티:** 예) 모든 X-ray 영상 또는 모든 초음파 영상 전용 모델.
  - **유사 태스크:** 예) 장기에 상관없이 '작고 확산된 병변' 분할 전용 모델.
  - **동일 차원:** 2D 의료 영상 전용 또는 3D 의료 영상 전용 모델.
  - **동일 획득 소스:** 카메라 기반(내시경, 조직학 등), 방사선 기반(CT, PET 등), 비방사선/비카메라 기반(MRI, 초음파 등)으로 구분.

### 3. 적용 및 평가 절차

구축된 모델은 다음과 같은 세 가지 설정으로 적용되어 성능을 평가한다.

- **Zero-shot:** 타겟 태스크의 주석 데이터 없이 모델을 직접 적용하는 방식이다.
- **Few-shot:** 타겟 태스크의 매우 적은 양(통상 20개 미만 샘플)의 데이터를 학습에 사용하는 방식이다.
- **Fine-tuning:** 충분한 양(통상 100개 이상 샘플)의 데이터를 사용하여 모델을 최적화하는 방식이다.

## 📊 Results

본 논문은 새로운 실험 결과보다는 기존 연구들의 결과 분석을 통해 의료 영상 분할의 현주소를 진단한다.

- **SAM의 의료 영상 적용 결과:** 병리 영상, 간 종양 분할, 뇌 MRI 분할, 복부 CT 장기 분할 등 다양한 데이터셋에서 SAM의 성능은 태스크 특화형 모델보다 현저히 낮았다. 특히 타겟 영역이 매우 작거나, 대비가 낮고, 형태가 불규칙한 경우에 취약함을 보였다.
- **데이터 특성 분석:** 의료 영상은 획득 방식, 해상도, 채널 수, 차원(2D/3D)이 매우 다양하며, 분할 대상이 전체 이미지의 $1\%$ 미만인 경우가 많아 일반적인 자연 이미지 모델의 일반화 능력이 그대로 전이되지 않음을 확인하였다.

## 🧠 Insights & Discussion

본 논문은 의료 영상 파운데이션 모델이 직면한 세 가지 주요 도전 과제와 통찰을 제시한다.

첫째, **샘플 사이즈의 문제**이다. SAM이 자연 이미지에서 높은 성능을 내기 위해 100만 개 이상의 이미지를 사용했음을 고려할 때, 더 복잡한 의료 영상 분야에서는 이보다 더 많은 데이터가 필요할 수 있다. 다만, 최근 공개 데이터셋이 증가하고 있어 실험적 검증이 가능해진 상태이다.

둘째, **데이터 준비의 복잡성**이다. 자연 이미지는 $1024 \times 1024$ 해상도와 RGB 3채널로 표준화가 쉽지만, 의료 영상은 모달리티마다 해상도, 차원, 강도(intensity) 범위가 천차만별이다. 따라서 서로 다른 데이터셋을 통합하기 위한 표준화 공정이 모델의 성능과 일반화 능력에 결정적인 영향을 미칠 것이다.

셋째, **프롬프트 설계의 전문성**이다. 일반 사용자를 위한 SAM과 달리 의료 AI는 의료 전문가가 사용한다. 따라서 단순한 포인트나 박스 형태의 프롬프트를 넘어, 임상 기록이나 텍스트 채팅, 혹은 전문가의 인터랙티브한 수정 요청을 어떻게 모델에 반영할 것인가가 핵심적인 연구 과제가 될 것이다.

결론적으로, 모든 의료 영상을 아우르는 Generalist 모델보다는, 현실적인 데이터 표준화와 성능 확보를 위해 특정 모달리티나 장기에 특화된 Specialist 모델의 구축이 더 효율적인 접근 방식이 될 가능성이 높다.

## 📌 TL;DR

이 논문은 의료 영상 분할 분야에서 태스크 특화형 모델의 한계를 극복하기 위한 파운데이션 모델의 구축 전략을 제안한다. 자연 이미지 모델(SAM 등)을 적응시키는 경로와 의료 전용 모델을 처음부터 구축하는 경로를 제시하며, 모델을 제너럴리스트와 스페셜리스트로 구분하는 체계적인 프레임워크를 제공한다. 특히 데이터 표준화와 전문적인 프롬프트 설계가 향후 의료 AI의 일반화 능력을 결정짓는 핵심 요소가 될 것임을 시사하며, 이는 향후 의료 영상 분석의 패러다임을 개별 모델 개발에서 통합 모델 적응으로 전환하는 데 중요한 이정표가 될 수 있다.
