# Improving CT Image Segmentation Accuracy Using StyleGAN Driven Data Augmentation

Soham Bhosale, Arjun Krishna, Ge Wang, and Klaus Mueller (2023)

## 🧩 Problem to Solve

본 논문은 의료 영상 분석의 핵심 과제인 CT 영상 분할(Segmentation)에서 발생하는 데이터 부족 및 데이터 간의 특성 차이 문제를 해결하고자 한다. 딥러닝 기반의 분할 모델은 높은 성능을 보이지만, 의료 데이터의 특성상 개인 정보 보호 문제와 전문의의 레이블링 비용으로 인해 대규모의 정답지(Annotation)가 포함된 데이터셋을 확보하기가 매우 어렵다.

더욱이, 공개적으로 사용 가능한 데이터셋이라 할지라도 촬영 당시의 방사선 조사량(Dosage)이나 스캐너 장비의 특성(Scanner properties)에 따라 영상의 텍스처(Texture)가 서로 다르게 나타난다. 이러한 텍스처 차이는 학습 데이터셋과 실제 테스트 데이터셋 사이의 도메인 간극을 발생시키며, 결과적으로 학습된 모델을 다른 데이터셋에 적용했을 때 분할 정확도가 현저히 떨어지는 결과를 초래한다. 따라서 본 연구의 목표는 소규모의 정답지가 있는 데이터셋을 활용하여, 텍스처가 다른 대규모의 정답지 없는 데이터셋에 대해서도 높은 분할 정확도를 달성하는 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 StyleGAN을 이용한 텍스처 전이(Texture Transfer) 기반의 데이터 증강(Data Augmentation)이다. 단순히 이미지의 수를 늘리는 기존의 증강 방식과 달리, 분할하고자 하는 대상 데이터셋(Large non-annotated dataset)에서 추출한 텍스처를 정답지가 있는 소규모 데이터셋(Small annotated dataset)에 입히는 방식을 취한다. 이를 통해 정답지의 해부학적 구조(Anatomy)는 유지하면서, 대상 데이터셋의 장비 특성과 노이즈 특성이 반영된 합성 영상을 생성한다. 결과적으로 모델이 실제 테스트할 데이터와 매우 유사한 텍스처를 가진 데이터로 학습하게 함으로써 도메인 불일치 문제를 해결하고 분할 성능을 향상시킨다.

## 📎 Related Works

논문은 의료 영상 분할에서 널리 사용되는 U-Net 구조와 기존의 텍스처 학습 연구(Krishna et al. [2])를 기반으로 한다. 기존의 접근 방식들은 대규모 레이블링 데이터셋을 확보하는 것에 집중하거나 일반적인 데이터 증강 기법을 사용하였으나, 본 논문은 서로 다른 CT 스캐너나 조사량으로 인해 발생하는 '텍스처 차이'라는 구체적인 문제에 집중한다. 특히, StyleGAN을 활용하여 특정 스타일(텍스처)을 학습하고 이를 다른 이미지의 콘텐츠(해부학적 구조)에 결합하는 방식을 도입함으로써 기존의 단순한 증강 기법보다 정교하게 도메인 간극을 줄였다는 점에서 차별점을 가진다.

## 🛠️ Methodology

본 연구의 전체 파이프라인은 크게 StyleGAN 학습, 데이터셋 증강, 그리고 U-Net 분할 모델 학습의 세 단계로 구성된다.

### 1. StyleGAN을 이용한 텍스처 학습

저자들은 인코더-디코더 기반의 StyleGAN 아키텍처를 사용하여 각 장기별 텍스처를 학습한다. 대규모 비정답 데이터셋에서 서로 다른 스캐너 특성을 대표하는 6개의 '스타일 이미지'를 선정하고, 이에 대응하는 6개의 StyleGAN 모델을 개별적으로 학습시킨다. 이때, 스타일 이미지의 정답지(Segmentation map)는 방사선 전문의가 수동으로 작성하여 학습에 사용한다.

학습의 핵심은 segment-wise style loss($L_s$)와 content loss($L_c$)를 사용하는 것이다.

- **Style Loss ($L_s$):** 생성된 이미지의 텍스처가 스타일 이미지의 텍스처와 유사하도록 강제한다. Graham Matrices를 사용하여 특징 맵의 상관관계를 계산하며 식은 다음과 같다.
$$L_s = \sum_{sg \in SG} \sum_{l \in SL} \frac{1}{4N_l^2 M_l^2} \|G(R_{sg}^l) - G(S_{sg}^l)\|_F^2$$

- **Content Loss ($L_c$):** 생성된 이미지가 원본 정답지의 해부학적 구조를 유지하도록 강제한다.
$$L_c = \sum_{sg \in SG} \sum_{l \in CL} \frac{1}{2N_l M_l} \|R_{sg}^l - O_{sg}^l\|_2^2$$

여기서 $O^l, S^l, R^l$은 각각 원본 이미지, 스타일 이미지, 스타일이 적용된 결과 이미지에서 추출된 VGG 네트워크의 특징 맵을 의미하며, $SG$는 심장, 흉부, 폐, 척수, 식도 등의 세그먼트 집합을 나타낸다.

### 2. 데이터셋 증강 (Augmentation)

학습된 6개의 StyleGAN에 소규모 정답 데이터셋의 모든 세그먼트 맵을 입력으로 넣는다. 각 세그먼트 맵에 대해 6가지 서로 다른 스타일의 이미지가 생성되므로, 원본 데이터셋을 포함하여 총 7배의 데이터 증강이 이루어진다. 예를 들어 원본 700장의 이미지는 약 4,900장의 이미지로 확장된다. 이렇게 생성된 이미지들은 대상 데이터셋의 저선량(low-dose) 텍스처를 가지면서도 정확한 정답지를 보유하게 된다.

### 3. U-Net 분할 모델 학습

최종적으로 증강된 데이터셋을 사용하여 U-Net 기반의 분할 네트워크를 학습시킨다. U-Net은 인코더(Down-sampling)와 디코더(Up-sampling) 구조로 이루어져 있으며, 각 층의 특징 맵을 연결하는 skip-connection을 통해 세부 정보를 보존한다. 증강된 텍스처 데이터로 학습된 U-Net은 실제 대규모 비정답 데이터셋의 이미지들을 입력받아 분할 맵을 생성한다.

## 📊 Results

### 실험 설정

- **데이터셋:** 소규모 고선량(high-dose) 정답 데이터셋(환자 30명)과 대규모 저선량(low-dose) 비정답 데이터셋(환자 약 14k명)을 사용하였다.
- **평가 방법:** 대규모 데이터셋에서 무작위로 선정된 100장의 이미지를 테스트 셋으로 구성하였으며, 정답지는 방사선 전문의가 수동으로 수정하여 생성하였다.
- **비교 대상:** 원본 소규모 데이터셋으로만 학습한 U-Net 1과 StyleGAN으로 증강된 데이터셋으로 학습한 U-Net 2를 비교하였다.

### 정량적 및 정성적 결과

- **정량적 지표:** 테스트 정확도(Testing Accuracy) 측정 결과, U-Net 1은 $93.1\%$를 기록한 반면, U-Net 2는 $95.7\%$를 기록하여 약 $2.6\%$의 성능 향상을 보였다.
- **정성적 결과:** 시각적 분석 결과, U-Net 2가 폐(Lung)와 흉부(Torso) 영역을 훨씬 더 정확하게 분류하는 것으로 나타났다. 이는 증강된 데이터셋이 대상 데이터셋의 텍스처 특성을 효과적으로 모사하여 모델의 일반화 능력을 높였음을 시사한다.

## 🧠 Insights & Discussion

### 강점 및 성과

본 연구는 데이터 부족 문제와 도메인 불일치(Texture difference) 문제를 동시에 해결하기 위해 StyleGAN을 데이터 증강 도구로 활용하였다는 점에서 매우 실용적인 접근 방식을 보여준다. 특히, 정답지가 없는 대규모 데이터셋에서 스타일만 추출하여 기존의 소규모 정답 데이터셋에 이식함으로써, 추가적인 대규모 레이블링 비용 없이도 성능을 향상시킬 수 있음을 입증하였다.

### 한계 및 미해결 과제

그럼에도 불구하고 식도(Esophagus)나 척수(Spinal cord)와 같은 작은 세그먼트에서는 여전히 분할 오류가 관찰되었다. 이는 다음 두 가지 원인으로 분석된다.

1. **해부학적 다양성 부족:** 소규모 데이터셋이 포함하는 환자의 수가 적어 대규모 데이터셋에 존재하는 다양한 해부학적 구조를 모두 커버하지 못했다.
2. **스캐너 위치 차이:** 데이터셋 간의 스캐너 렌즈 위치 차이로 인해 생성된 이미지의 공간적 세부 정보(Spatial detail) 수준이 실제 데이터와 차이가 있었다.

### 향후 연구 방향

저자들은 이러한 한계를 극복하기 위해 세그먼트 맵의 PCA 공간을 탐색하여 해부학적 구조가 다양한 합성 영상을 생성하는 방법, StyleGAN과 U-Net을 순환적으로 학습시키는 cyclic approach, 그리고 폐 및 뼈 윈도우(Window) 영상을 함께 사용하는 다중 윈도우 접근 방식을 제안하였다.

## 📌 TL;DR

본 논문은 CT 영상 분할에서 발생하는 데이터 부족 및 장비별 텍스처 차이 문제를 해결하기 위해 StyleGAN 기반의 데이터 증강 기법을 제안한다. 대상 데이터셋의 텍스처를 소규모 정답 데이터셋에 전이하여 학습 데이터를 7배로 확장함으로써, U-Net의 분할 정확도를 $93.1\%$에서 $95.7\%$로 향상시켰다. 이 연구는 레이블링 데이터가 부족한 의료 영상 분야에서 생성 모델을 이용해 도메인 간극을 줄이는 효과적인 전략을 제시하였으며, 향후 다양한 의료 영상 분석 파이프라인에 적용될 가능성이 높다.
