# Medical Image Segmentation via Unsupervised Convolutional Neural Network

Junyu Chen and Eric C. Frey (2020)

## 🧩 Problem to Solve

본 논문은 의료 영상 분할(Medical Image Segmentation)에서 발생하는 데이터 어노테이션(Annotation)의 비용 문제와 전통적인 영상 처리 알고리즘의 계산 효율성 문제를 동시에 해결하고자 한다.

현재 대부분의 딥러닝 기반 분할 방법론은 고품질의 대규모 학습 데이터를 필요로 하며, 이러한 데이터를 생성하는 과정은 수개월에서 일 년까지 소요될 정도로 매우 많은 시간이 걸리는 작업이다. 반면, 클러스터링(Clustering)이나 레벨 셋(Level-set) 기반의 전통적인 방식들은 학습 데이터 없이 영상의 강도 통계량(Intensity statistics)만을 이용하므로 데이터 의존성은 낮다. 그러나 이러한 방식들은 입력 영상마다 통계량을 다시 모델링하고 최적화해야 하므로, 계산 부담이 매우 크다는 치명적인 단점이 있다.

따라서 본 연구의 목표는 딥러닝의 빠른 추론 속도와 일반화 능력, 그리고 전통적인 에너지 기반 모델의 비지도 학습 특성을 결합하여, 지도 학습 데이터가 거의 없거나 전혀 없는 환경에서도 효과적으로 작동하는 분할 모델을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 전통적인 **Active Contour Without Edges (ACWE)** 프레임워크를 **Convolutional Neural Network (ConvNet)**를 통해 파라미터화하는 것이다.

기존의 ACWE 방식은 개별 이미지에 대해 에너지 함수를 최소화하는 최적의 경계선을 찾는 반복적인 최적화 과정을 거치지만, 본 연구에서는 이 과정을 ConvNet의 학습 과정으로 대체하였다. 즉, 네트워크가 영상의 강도 통계량을 기반으로 분할 마스크를 직접 예측하도록 학습시킴으로써, 학습 단계에서는 ACWE의 에너지 함수를 손실 함수(Loss function)로 사용하고, 추론 단계에서는 최적화 과정 없이 단 한 번의 순전파(Feed-forward)만으로 분할 결과를 얻을 수 있게 설계하였다.

## 📎 Related Works

논문에서는 다음과 같은 기존 접근 방식과 그 한계를 언급한다.

1. **Deep Learning 기반 방법론**: 높은 정확도를 보이지만, 대량의 정밀한 어노테이션 데이터가 필수적이라는 한계가 있다.
2. **전통적 방법론 (Clustering, Level-set)**: Chan and Vese(2001) 등이 제안한 레벨 셋 방식은 데이터 없이 강도 통계량만으로 작동하지만, 매 이미지마다 최적화 과정을 수행해야 하므로 계산 비용이 매우 높다.

본 논문의 차별점은 ACWE의 에너지 최소화 원리를 딥러닝의 손실 함수로 통합함으로써, 학습 시에는 비지도 방식으로 보편적인 표현(Universal representation)을 학습하고, 추론 시에는 레벨 셋 방식보다 훨씬 빠른 속도로 결과를 도출한다는 점에 있다.

## 🛠️ Methodology

### 전체 시스템 구조

본 모델은 2D SPECT 영상 슬라이스를 입력받아 이진 마스크(Binary mask)를 출력하는 구조를 가진다. 학습 설정에 따라 비지도 학습(Unsupervised) 또는 반지도 학습(Semi-supervised) 방식으로 운영될 수 있다.

### ConvNet 아키텍처

네트워크는 Recurrent Convolutional Neural Network (RCNN)를 기반으로 설계되었다.

- **구성**: 3개의 Recurrent 레이어와 2개의 일반 Convolutional 레이어로 구성된다.
- **세부 구조**: 각 Recurrent 레이어는 $T=3$의 타임 스텝을 가지며, 이는 결과적으로 깊이가 $T+1=4$인 피드포워드 서브 네트워크와 동일한 효과를 낸다.
- **활성화 함수 및 정규화**: 각 레이어 이후에는 Batch Normalization과 PReLU (Parametric Rectified Linear Unit)가 적용된다.
- **출력층**: Softmax 대신 PReLU를 사용하여 0보다 큰 값은 전경(Foreground), 0 이하의 값은 배경(Background)으로 분류하는 방식을 채택하였다.
- **입출력 크기**: 입력과 출력 모두 $192 \times 192 \times 1$의 차원을 공유한다.

### 손실 함수 (Loss Function)

#### 1. 비지도 학습 손실 ($L_{ACWE}$)

Chan and Vese 모델의 에너지 함수를 기반으로 하며, 정답 라벨 없이 영상의 강도 통계량만으로 계산된다. 기본 에너지 함수는 다음과 같다.

$$F(c_1, c_2, C) = \mu \cdot \text{Length}(C) + \nu \cdot \text{Area}(\text{inside}(C)) + \lambda_1 \int_{\text{inside}(C)} |g_i - c_1|^2 di + \lambda_2 \int_{\text{outside}(C)} |g_i - c_2|^2 di$$

여기서 $C$는 경계선, $g$는 영상, $c_1$과 $c_2$는 각각 경계선 내부와 외부의 평균 강도를 의미한다. 본 논문에서는 단순화를 위해 $\mu=0, \lambda_1=1, \lambda_2=1$로 설정하였으며, 이를 ConvNet $f_\theta$에 적용한 손실 함수 $L_{ACWE}$는 다음과 같이 정의된다.

$$L_{ACWE} = \nu \cdot \text{Area}(f_\theta(g) > 0) + \sum_{f_\theta(g)>0} |g - c_1|^2 + \sum_{f_\theta(g) \le 0} |g - c_2|^2$$

#### 2. 지도 학습 손실 ($L_{label}$)

정답 라벨 $u$가 존재할 경우, 구조적 정보를 학습하기 위해 다음과 같은 손실 함수를 추가로 사용한다.

$$L_{label} = \sum_{f_\theta(g)} |\nabla(f_\theta(g))| + \sum_{\Omega} ((1-u)^2 - (0-u)^2) f_\theta(g)$$

여기서 $\Omega$는 영상 도메인을 나타낸다.

#### 3. 최종 통합 손실 함수

두 손실 함수를 결합하여 다음과 같이 최종 손실 함수를 구성한다.

$$L = L_{ACWE} + \alpha L_{label}$$

실험적으로 $\alpha=0.4$, $\nu=0.004$로 설정하였다.

## 📊 Results

### 실험 환경

- **데이터셋**: XCAT 팬텀을 이용해 생성한 고정밀 시뮬레이션 $^{99\text{m}}\text{Tc SPECT}$ 영상.
- **데이터 구성**: 140개의 3D 볼륨에서 총 12,000개의 2D 슬라이스를 추출하여 학습(8,000장)과 테스트(4,000장)에 사용하였다.
- **평가 지표**: Dice Similarity Coefficient (DSC) 및 추론 시간(Computation Time).

### 비교 설정 (Modes)

- **Mode 1**: $L_{ACWE}$만을 이용한 완전 비지도 학습.
- **Mode 2**: Mode 1 학습 후 10개의 GT(Ground Truth) 라벨로 미세 조정(Fine-tuning).
- **Mode 3**: Mode 1 학습 후 80개의 GT 라벨로 미세 조정.
- **Mode 4**: 처음부터 $L_{ACWE}$와 $L_{label}$을 동시에 사용하여 학습.
- **Baseline**: 전통적인 레벨 셋 기반 ACWE 방식.

### 결과 분석

1. **정량적 성능 (DSC)**:
    - Mode 4와 Mode 3가 가장 높은 DSC(각각 $0.856\pm0.09$, $0.732\pm0.12$)를 기록하였다.
    - 특히, 비지도 학습 모델을 80개의 적은 라벨로 미세 조정하는 것만으로도 성능이 크게 향상됨을 확인하였다.
    - 전통적인 레벨 셋 ACWE(DSC $0.518\pm0.337$)보다 월등한 성능을 보였다.
2. **계산 효율성**:
    - 제안 방법의 추론 시간은 $0.006\pm0.022$초인 반면, 레벨 셋 ACWE는 $2.698\pm0.085$초가 소요되었다. 이는 약 450배 이상의 속도 향상을 의미한다.

## 🧠 Insights & Discussion

본 연구는 딥러닝 모델이 반드시 대량의 라벨링 데이터를 필요로 한다는 고정관념을 깨고, 물리적/통계적 기반의 에너지 함수를 손실 함수로 활용함으로써 데이터 부족 문제를 해결할 수 있음을 보여주었다.

**강점**:

- 매우 적은 양의 라벨(80개)만으로도 상당한 성능 향상을 이끌어낼 수 있어, 실제 의료 현장에서 어노테이션 비용을 획기적으로 줄일 수 있다.
- 추론 속도가 압도적으로 빨라 실시간 영상 분석에 적용 가능하다.

**한계 및 논의**:

- 본 실험은 시뮬레이션 데이터(XCAT phantom)를 기반으로 수행되었으므로, 실제 환자의 복잡한 노이즈가 포함된 실제 임상 데이터에서도 동일한 성능이 유지될지는 추가 검증이 필요하다.
- $\nu$나 $\alpha$와 같은 하이퍼파라미터가 경험적 실험(Empirical experiments)에 의해 결정되었으므로, 다른 모달리티나 다른 장기 분할에 적용할 때 최적 값을 찾는 과정이 필요할 것이다.

## 📌 TL;DR

본 논문은 전통적인 ACWE 에너지 함수를 ConvNet의 손실 함수로 통합하여, 라벨 없이도 학습 가능한 비지도/반지도 의료 영상 분할 모델을 제안하였다. 실험 결과, 제안 모델은 전통적인 레벨 셋 방식보다 훨씬 빠르고 정확한 성능을 보였으며, 특히 소량의 정답 데이터만으로도 성능을 극대화할 수 있음을 입증하였다. 이는 데이터 수집이 어려운 의료 영상 분야에서 효율적인 학습 전략을 제공한다는 점에서 큰 의의가 있다.
