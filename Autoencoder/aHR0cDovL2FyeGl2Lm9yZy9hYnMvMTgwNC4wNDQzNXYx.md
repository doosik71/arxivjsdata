# Variational Composite Autoencoders

Jiangchao Yao, Ivor Tsang, and Ya Zhang (2018)

## 🧩 Problem to Solve

본 논문은 잠재 변수 모델(latent variable model)에서 복잡한 데이터 구조가 존재하거나 잠재 변수(latent variable)를 처리하기 어려운 경우 발생하는 학습의 난해함을 해결하고자 한다. 기존의 Variational Autoencoder(VAE)는 인코더와 디코더가 직접적으로 연결된 단순한 구조를 가지고 있어, 매우 정교한 데이터 구조를 처리하거나 잠재 변수의 최적화가 불안정한 상황에서 효율성이 떨어진다는 한계가 있다. 따라서 본 연구의 목표는 인코딩 효율을 높이고 최적화 부담을 완화할 수 있는 더 표현력이 풍부한 구조를 탐색하여, 기존 VAE의 단순한 구조적 한계를 극복하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 잠재 변수($z$)와 관측 데이터($x$) 사이에 **대리 변수(surrogate variable, $s$)**를 도입하여 계층적 잠재 변수 모델(hierarchical latent variable model)을 구축하는 것이다. 이를 통해 표현 학습(representation learning) 과정을 두 개의 하위 절차로 분리(decouple)함으로써, 복잡한 데이터 구조에서 오는 학습 난이도나 잠재 변수 최적화의 불안정성을 대리 공간(surrogate space)에서 완화할 수 있도록 설계하였다.

## 📎 Related Works

기존의 잠재 변수 모델 연구는 크게 두 가지 방향으로 진행되었다. 첫째는 그래프 구조 데이터나 제어 가능한 이미지 표현, 문법 구문 트리와 같은 다양한 데이터 구조를 처리하기 위한 연구들(예: Variational Graph Autoencoder, Variational Lossy Autoencoder, Grammar VAE)이다. 둘째는 연속적 또는 이산적 잠재 변수의 최적화 어려움을 해결하기 위한 연구들로, 제어 변수를 이용한 일반적인 score function 방법이나 이산 분포의 연속적 완화 버전인 Concrete distribution 등이 제안되었다.

그러나 이러한 개별적인 시도들에도 불구하고, 기존 VAE들의 기본적인 인코더-디코더 구조 자체가 너무 단순하여, 인코더와 디코더의 구현만으로는 정교한 데이터나 불안정한 최적화 문제를 완전히 해결하기 어렵다는 점이 본 논문이 지적하는 차별점이다.

## 🛠️ Methodology

### 전체 시스템 구조

VCAE는 데이터를 직접 잠재 공간으로 보내는 대신, 대리 변수를 통해 단계적으로 학습하는 구조를 가진다. 이 모델은 두 개의 추론 네트워크(inference networks)와 두 개의 생성 네트워크(generative networks)로 구성된다.

1. **추론 과정 (Encoding):**
   - $q_\theta^1(s|x)$: 입력 $x$로부터 대리 변수 $s$를 추론하는 네트워크이다.
   - $q_\theta^2(z|x)$: 입력 $x$로부터 잠재 변수 $z$를 추론하는 네트워크이다.

2. **생성 과정 (Decoding):**
   - $p_\psi(s|z)$: 잠재 변수 $z$로부터 대리 변수 $s$를 생성하는 네트워크이다.
   - $p_\phi(x|s)$: 대리 변수 $s$로부터 최종 관측 데이터 $x$를 복원하는 네트워크이다.

### 목적 함수 및 학습 절차

VCAE는 다음과 같은 변분 하한(variational lower bound)을 최대화하는 것을 목표로 한다.

$$ \ln p(x) \ge \mathbb{E}_{q_\theta^1(s|x)}[\ln p(x|s)] - \mathbb{E}_{q_\theta^2(z|x)}[D_{KL}(q_\theta^1(s|x) || p_\psi(s|z))] - D_{KL}(q_\theta^2(z|x) || p(z)) $$

위 식의 각 항은 다음과 같은 의미를 갖는다.

- $\mathbb{E}_{q_\theta^1(s|x)}[\ln p(x|s)]$: 대리 변수 $s$를 통한 데이터 $x$의 복원 성능을 나타내는 기대 로그 가능도이다.
- $\mathbb{E}_{q_\theta^2(z|x)}[D_{KL}(q_\theta^1(s|x) || p_\psi(s|z))]$: 잠재 변수 $z$로부터 생성된 $s$의 분포와 $x$로부터 추론된 $s$의 분포 사이의 유사도를 측정하는 KL 발산 항이다.
- $D_{KL}(q_\theta^2(z|x) || p(z))$: 잠재 변수 $z$의 추론 분포와 사전 분포 사이의 유사도를 측정하는 KL 발산 항이다.

이 구조의 장점은 다양한 최적화 기법을 유연하게 적용할 수 있다는 점이다. 예를 들어 $z$와 $s$가 모두 재매개변수화(reparameterizable) 가능하다면 확률적 역전파(stochastic backpropagation)를 직접 사용할 수 있으며, $z$가 이산 변수이고 $s$가 재매개변수화 가능하다면 score function 방법과 재매개변수화 기법을 동시에 적용하여 최적화할 수 있다.

## 📊 Results

### 실험 설정

- **데이터셋:** 정적으로 이진화된 MNIST 데이터셋을 사용하였으며, 학습/검증/테스트 셋을 $50,000/10,000/10,000$으로 분할하였다.
- **모델 구성:** 모든 모듈은 Sigmoid Belief Network로 매개변수화되었다. 인코딩 채널($q_\theta^1, q_\theta^2$)과 디코딩 채널($p_\phi$)은 200개의 확률적 유닛과 선형 층 또는 두 개의 $\tanh$ 유닛 층(Nonlinear)으로 구성되었다. $p_\psi$는 하나의 sigmoid 층으로 고정되었다.
- **최적화:** ADAM 옵티마이저를 사용하였으며, 학습률은 $3 \times 10^{-4}$로 설정하였다. 평가는 중요도 샘플링(importance-sampled) 기반의 로그 가능도 추정치를 사용하였다.

### 주요 결과

1. **생성 모델링 및 구조적 예측:**
   - 생성 모델링(Generative modeling) 작업에서 VCAE는 $\text{Linear}$ 케이스 기준 $-95.7$의 로그 가능도를 달성하여, VAE($-112.8$)나 Concrete-s($-105.3$)보다 우수한 성능을 보였다.
   - 구조적 예측(Structured prediction) 작업에서도 VCAE($-62.1$)가 VAE($-77.1$) 및 다른 베이스라인들보다 뛰어난 성능을 기록하였다.

2. **이산 변수 모델 최적화:**
   - 이산 변수 모델 학습 시, VCAE는 NVIL, REBAR, Concrete-z와 같은 최신 기법들과 비교되었다.
   - 생성 모델링(Linear)에서 VCAE는 $-97.9$를 기록하여 REBAR($-107.6$)와 Concrete-z($-107.5$)를 크게 상회하였다.
   - 구조적 예측에서는 연속 변수를 사용한 Concrete-s보다도 높은 성능을 보였다.

3. **기울기 분산(Gradient Variance) 분석:**
   - 학습 과정에서의 기울기 분산을 추적한 결과, VCAE는 NVIL보다 낮은 분산을 보였으며, 특히 $\text{Nonlinear}$ 케이스에서는 Concrete-s보다도 분산이 낮았다. 이는 VCAE가 더 빠른 수렴 속도를 가질 수 있음을 시사한다.

## 🧠 Insights & Discussion

본 논문은 VAE의 단순한 구조가 데이터의 복잡성이나 잠재 변수의 최적화 문제를 해결하는 데 한계가 있음을 지적하고, 이를 계층적 구조(대리 변수 도입)로 해결할 수 있음을 입증하였다.

특히 주목할 점은 VCAE가 단순한 모델 확장이 아니라, 기존의 score function 방법과 재매개변수화 기법을 통합할 수 있는 프레임워크 역할을 한다는 것이다. 실험 결과에서 나타난 낮은 기울기 분산은 대리 변수를 통해 학습 부담을 분산시킨 결과로 해석되며, 이는 이산 잠재 변수를 포함한 복잡한 모델의 최적화를 훨씬 효율적으로 만들어준다.

다만, 본 논문에서는 MNIST라는 비교적 단순한 데이터셋을 사용하여 성능을 검증하였으므로, 더 복잡하고 고차원인 실제 이미지 데이터셋에서도 동일한 효율성이 나타날지에 대해서는 추가적인 검증이 필요할 것으로 보인다.

## 📌 TL;DR

본 연구는 VAE의 단순한 인코더-디코더 구조를 개선하기 위해 잠재 변수와 데이터 사이에 **대리 변수(surrogate variable)**를 도입한 **VCAE(Variational Composite Autoencoder)**를 제안한다. 이 계층적 구조는 표현 학습의 부담을 줄이고, 특히 이산 잠재 변수 최적화 시 발생하는 기울기 분산 문제를 완화하여 기존 VAE 및 최신 최적화 기법들보다 우수한 성능과 수렴 속도를 보여준다. 이 연구는 향후 복잡한 데이터 구조를 가진 생성 모델이나 이산 변수 기반의 잠재 모델을 설계할 때 효율적인 구조적 가이드를 제공할 가능성이 높다.
