# FedQuad: Federated Stochastic Quadruplet Learning to Mitigate Data Heterogeneity

Özgü Göksu, Nicolas Pugeault (2025)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 연합 학습(Federated Learning, FL) 환경에서 클라이언트 간의 데이터 이질성(Data Heterogeneity)으로 인해 발생하는 **표현 붕괴(Representational Collapse)** 현상이다. 

일반적으로 FL에서는 데이터가 각 클라이언트에 분산되어 있으며, 실제 환경에서는 클래스 비율, 샘플 크기, 특징 분포가 서로 다른 Non-i.i.d. 특성을 가진다. 이러한 데이터 이질성은 각 클라이언트가 서로 상충하는 특징 공간(Feature Space)을 학습하게 만들며, 서버에서 모델을 집계(Aggregation)했을 때 글로벌 모델의 임베딩이 붕괴되어 클래스 간 변별력을 상실하게 만든다. 특히 데이터셋의 크기가 작거나 클래스 불균형이 심한 경우 이 문제가 더욱 두드러지며, 결과적으로 글로벌 모델의 일반화 성능이 크게 저하된다.

따라서 본 연구의 목표는 데이터 이질성 환경에서도 표현 붕괴를 방지하고, 클래스 내 분산(Intra-class variance)은 줄이며 클래스 간 분산(Inter-class variance)은 높게 유지하는 강건한 표현 학습 방법을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 중심 아이디어는 **Metric Learning** 기법을 연합 학습의 로컬 학습 단계에 도입하여, 특징 공간 상에서 데이터 간의 상대적 거리를 직접 최적화하는 것이다. 구체적인 기여 사항은 다음과 같다.

1.  **FedQuad 프레임워크 제안**: 데이터 이질성 하에서 표현 붕괴를 완화하기 위해 설계된 Metric Learning 기반의 연합 학습 프레임워크를 제안한다.
2.  **새로운 Quadruplet Loss 설계**: 기존의 Quadruplet Loss를 수정하여, 앵커(Anchor)와 포지티브(Positive) 샘플 간의 거리는 좁히고, 서로 다른 클래스에서 추출된 두 개의 네거티브(Negative) 샘플과의 거리는 동시에 멀어지게 함으로써 클래스 간 분별력을 극대화한다.
3.  **Stochastic Quadruplet Sampling 전략**: 클래스 불균형 및 Non-i.i.d. 분포를 고려하여, 각 배치 내에 반드시 정답 포지티브 쌍이 포함되고 네거티브 샘플들이 서로 다른 클래스에서 추출되도록 하는 오프라인 샘플링 전략을 설계하였다.
4.  **표현 붕괴 완화 입증**: 제안 방법이 로컬 및 글로벌 모델 모두에서 변별력 있는 표현을 유지함으로써, 집계로 인한 성능 저하를 효과적으로 방지함을 실험적으로 증명하였다.

## 📎 Related Works

논문에서는 데이터 이질성을 해결하기 위한 기존 접근 방식을 크게 두 가지 방향으로 설명한다.

1.  **정규화 기반 접근 방식 (Regularization-based)**:
    *   **FedProx**: 글로벌 모델과 로컬 모델 파라미터 간의 유클리드 거리를 목적 함수에 추가하여 로컬 업데이트가 글로벌 모델에서 너무 멀어지지 않도록 제한한다.
    *   **SCAFFOLD**: 제어 변수(Control variables)를 사용하여 클라이언트 드리프트(Client Drift) 문제를 해결하려 한다.
    *   **한계**: 이러한 방식은 로컬 업데이트의 안정성을 높이지만, 특징 공간의 구조적 변별력을 직접적으로 최적화하지는 않는다.

2.  **대조 학습 기반 접근 방식 (Contrastive Learning-based)**:
    *   **MOON**: 로컬 모델의 표현을 글로벌 모델의 표현과 일치시키도록 유도하여 로컬 모델의 강건성을 높인다.
    *   **FedRCL**: 동일 클래스의 데이터 포인트들이 특징 공간에서 충분히 잘 분리되도록 하여 다양성을 보존한다.
    *   **한계**: 지나치게 공격적인 정렬(Alignment)은 오히려 클래스 내 다양성을 훼손하여 표현 붕괴를 야기할 수 있으며, 클래스 불균형이 심한 경우 성능이 급격히 저하되는 경향이 있다.

**FedQuad의 차별점**은 추가적인 글로벌 모델이나 프로토타입(Prototypes)에 의존하지 않고, 로컬 학습 단계에서 Metric Learning 손실 함수를 통해 특징 공간의 구조(Intra-class $\downarrow$, Inter-class $\uparrow$)를 직접 최적화함으로써 표현 붕괴를 원천적으로 방지한다는 점이다.

## 🛠️ Methodology

### 전체 파이프라인
FedQuad는 기본적으로 $\text{FedAvg}$ 알고리즘의 구조를 따른다. 서버는 글로벌 모델을 브로드캐스팅하고, 각 클라이언트는 로컬 데이터로 학습한 후 업데이트된 파라미터를 서버에 전송하며, 서버는 이를 평균 내어 새로운 글로벌 모델을 생성한다. 차이점은 로컬 학습 시 사용하는 손실 함수에 있다.

### 로컬 학습 목적 함수
각 클라이언트는 교차 엔트로피 손실($\ell_{ce}$)과 제안된 Quadruplet 손실($\ell_{quad^*}$)의 결합 형태로 학습을 진행한다.

$$\ell = \ell_{ce}(w_t^i; (x_a, y_a)) + \beta \ell_{quad^*}(w_t^i; (x_a, x_p, x_{n1}, x_{n2}))$$

여기서 $\beta$는 Quadruplet Loss의 영향을 조절하는 하이퍼파라미터이다.

### 수정된 Quadruplet Loss ($\ell_{quad^*}$)
기존의 Quadruplet Loss가 네거티브 쌍 간의 거리를 제한하는 것에 집중했다면, 본 논문의 $\ell_{quad^*}$는 앵커를 중심으로 여러 네거티브 샘플을 동시에 밀어내는 구조를 가진다.

$$\ell_{quad^*} = \left[ \|f(x_a) - f(x_p)\|^2 - \|f(x_a) - f(x_{n1})\|^2 + m_1 \right]_+ + \left[ \|f(x_a) - f(x_p)\|^2 - \|f(x_a) - f(x_{n2})\|^2 + m_2 \right]_+$$

*   $f(\cdot)$: 특징 추출기(Encoder)를 통해 임베딩을 생성하는 함수이다.
*   $x_a$: 앵커(Anchor) 샘플이다.
*   $x_p$: 앵커와 동일한 클래스인 포지티브(Positive) 샘플이다.
*   $x_{n1}, x_{n2}$: 앵커 및 서로와도 다른 클래스인 네거티브(Negative) 샘플들이다.
*   $m_1, m_2$: 각 네거티브 쌍에 대해 적용되는 마진(Margin) 값이다. 두 마진을 다르게 설정함으로써 임베딩 공간에 유연성을 부여하고, 모든 네거티브를 일률적으로 밀어내지 않고 세밀하게 분리하도록 유도한다.
*   $[x]_+ = \max(0, x)$는 Hinge loss 형태로, 조건이 만족되지 않을 때만 손실이 발생하게 한다.

### Stochastic Quadruplet Sampling
데이터 불균형 문제를 해결하기 위해 다음과 같은 샘플링 전략을 사용한다.
1.  데이터셋에서 앵커 $x_a$를 무작위로 선택한다.
2.  $x_a$와 동일한 클래스에서 포지티브 $x_p$를 선택한다.
3.  현재 클라이언트가 보유한 클래스 중 $x_a$의 클래스를 제외한 나머지 클래스들 중에서 서로 다른 두 클래스를 선택하고, 각각에서 네거티브 $x_{n1}, x_{n2}$를 하나씩 추출한다.

이 과정을 통해 각 배치는 의미론적으로 적절한 쿼드러플렛(Quadruplet)을 구성하게 되며, 이는 특히 데이터가 희소한 Non-i.i.d. 환경에서 효율적인 최적화를 가능하게 한다.

## 📊 Results

### 실험 설정
*   **데이터셋**: CIFAR-10, CIFAR-100
*   **모델 구조**: CNN Backbone $\rightarrow$ FC Layer (128차원 임베딩 생성)
*   **데이터 분포**: Dirichlet 분포 ($\alpha=0.5, 0.3$)를 사용하여 Non-i.i.d. 환경 조성
*   **클라이언트 수**: 10, 50, 200명
*   **비교 대상**: FedAvg, SupConFL (Supervised Contrastive), TripletFL, QuadrupletFL (기존 방식), MOON

### 주요 결과
1.  **정확도 향상**: CIFAR-10과 CIFAR-100 모두에서 FedQuad가 모든 베이스라인보다 우수한 성능을 보였다. 특히 데이터 이질성이 심한 $\alpha=0.3$ 환경에서 다른 방법론들에 비해 압도적인 성능 유지력을 보였다.
2.  **클라이언트 수 증가에 따른 강건성**: 클라이언트 수가 200명으로 증가하여 데이터 희소성이 심해진 상황에서도 FedQuad는 상대적으로 높은 성능을 유지하였다. 특히 MOON의 경우 클라이언트 수가 증가함에 따라 성능이 급격히 저하되는 반면, FedQuad는 안정적인 모습을 보였다.
3.  **시각적 분석 (t-SNE)**: 학습 초기 단계(Round 5)의 임베딩 시각화 결과, $\text{FedAvg}$는 클래스 간 경계가 모호하고 뭉쳐 있는 반면, $\text{FedQuad}$는 클래스별로 명확하게 군집화된 분포를 보였다.
4.  **Ablation Study**: $\ell_{ce}$ 없이 $\ell_{quad^*}$만 사용했을 때는 성능이 크게 하락하였으며, 적절한 마진 값($m_1=1.0, m_2=0.5$)과 $\beta=0.5$일 때 최적의 성능을 기록하였다.

## 🧠 Insights & Discussion

### 강점 및 해석
FedQuad의 성공 요인은 로컬 학습 단계에서 **클래스 내 분산을 최소화하고 클래스 간 분산을 최대화**하는 명시적인 제약 조건을 부여했다는 점에 있다. 이는 글로벌 모델 집계 과정에서 발생하는 특징 공간의 충돌을 방지하고, 집계 후에도 변별력 있는 구조가 유지되도록 만드는 '완충 작용'을 한다. 특히 기존의 Contrastive Loss가 클래스 불균형 상황에서 노이즈가 많은 임베딩을 생성하는 한계를, 두 개의 서로 다른 네거티브 샘플을 이용한 강한 제약 조건으로 극복하였다.

### 한계 및 비판적 논의
1.  **확장성(Scalability) 문제**: 저자들도 언급하였듯이, 클라이언트 수가 극도로 많아지고(예: 1,000명 이상) 클라이언트당 보유 데이터가 극소수가 될 경우, 유의미한 네거티브 샘플을 추출하는 것이 어려워져 성능이 저하될 가능성이 있다.
2.  **계산 비용**: 기존 $\text{FedAvg}$에 비해 로컬 학습 시 샘플링 과정과 추가적인 손실 함수 계산 비용이 발생한다. 비록 이는 모델 파라미터 전송 비용에 비해 작지만, 리소스가 극도로 제한된 엣지 디바이스에서는 고려 대상이 될 수 있다.
3.  **데이터셋의 한계**: CIFAR와 같은 정형화된 이미지 데이터셋 외에, 실제 의료 영상이나 사용자 행동 데이터와 같이 도메인 특성이 강하고 더 복잡한 데이터셋에서의 검증이 추가적으로 필요하다.

## 📌 TL;DR

본 논문은 연합 학습에서 데이터 이질성으로 인해 발생하는 **표현 붕괴(Representational Collapse)** 문제를 해결하기 위해, 수정된 Quadruplet Loss를 도입한 **FedQuad** 프레임워크를 제안한다. 앵커-포지티브-네거티브(2개)로 구성된 확률적 샘플링과 손실 함수를 통해 로컬 특징 공간의 변별력을 극대화함으로써, 글로벌 모델 집계 후에도 높은 일반화 성능을 유지한다. 특히 클래스 불균형이 심하고 클라이언트 수가 많은 극한의 Non-i.i.d. 환경에서 기존 방식(MOON, FedAvg 등)보다 탁월한 성능을 보였으며, 이는 향후 의료 영상 등 데이터 프라이버시와 이질성이 동시에 중요한 실제 서비스 적용에 기여할 가능성이 높다.
