# Federated Self-Supervised Learning in Heterogeneous Settings: Limits of a Baseline Approach on HAR

Sannara Ek, Romain Rombourg, François Portet, and Philippe Lalanda (2022)

## 🧩 Problem to Solve

본 논문은 모바일 기기 환경에서의 인간 활동 인식(Human Activity Recognition, HAR)을 위한 연합 학습(Federated Learning, FL) 시스템에서 발생하는 데이터 라벨링의 비용 문제와 데이터의 이질성(Heterogeneity) 문제를 해결하고자 한다.

일반적인 지도 학습(Supervised Learning) 기반의 연합 학습은 모든 클라이언트가 정확하게 라벨링된 데이터를 보유하고 있다고 가정하지만, 실제 모바일 환경에서 사용자가 직접 자신의 활동 데이터를 라벨링하는 것은 매우 어렵고 비용이 많이 드는 작업이다. 이로 인해 방대한 양의 데이터가 라벨 없이 방치되는 문제가 발생한다.

따라서 본 연구의 목표는 라벨이 없는 데이터를 활용하는 자기 지도 학습(Self-Supervised Learning, SSL)을 연합 학습 체계에 도입하여, 특히 데이터의 이질성이 강한 실제 환경에서도 효과적인 특징 표현(Representation)을 학습할 수 있는지 검증하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 실제와 유사한 이질적 환경(Heterogeneous setting)에서 연합 반지도 학습(Federated Semi-Supervised Learning)의 기준점(Reference)이 되는 평가 체계를 구축하고, 기본 접근 방식의 한계를 분석한 것이다. 

구체적으로는 가벼운 구조의 오토인코더(Autoencoder, AE)와 표준 연합 평균(Federated Averaging, FedAvg) 알고리즘을 결합하여 HAR 태스크에 적용하였으며, 이를 통해 단순한 AE 기반의 특징 학습이 이질적인 데이터셋 환경에서 강건한 표현을 학습하는 데 한계가 있음을 정량적으로 증명하였다.

## 📎 Related Works

기존의 연합 자기 지도 학습 연구들은 주로 다음과 같은 한계를 가지고 있다고 지적한다.

1. **데이터의 동질성 가정**: 많은 연구가 데이터셋이 균일한(Homogeneous) 상황을 가정하여 실험을 진행하였으며, 실제 환경의 복잡한 이질성을 충분히 반영하지 않았다.
2. **모델 복잡도**: SimCLR를 변형한 대조 학습(Contrastive Learning) 네트워크 등 고성능 모델을 제안한 연구들이 있으나, 이는 계산 자원이 제한적인 모바일 기기에서 실행하기에는 너무 무겁고 복잡한 구조(예: Dual-stream architecture)를 가진다.
3. **실무적 제약 무시**: 기기의 계산 능력이나 메모리 제약, 그리고 실제 필드에서 발생하는 데이터 불균형 문제를 간과한 경우가 많다.

본 논문은 이러한 한계를 극복하기 위해 매우 가벼운 모델인 AE를 선택하고, 여러 공개 데이터셋을 조합하여 극단적인 이질성을 구현한 환경에서 실험을 수행함으로써 기존 연구와 차별화를 둔다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조
본 연구는 다음과 같은 2단계 학습 절차를 따른다.
1. **연합 특징 학습 단계 (Unsupervised Feature Learning)**: 80명의 클라이언트가 각자 보유한 라벨 없는 데이터를 사용하여 AE를 연합 학습 방식으로 학습시킨다.
2. **서버 측 미세 조정 단계 (Supervised Fine-tuning)**: 학습된 AE의 인코더(Encoder) 부분만을 추출하여 분류기(Classifier)를 부착한 뒤, 서버가 보유한 소량의 라벨링된 데이터를 사용하여 지도 학습 방식으로 미세 조정을 수행한다.

### 데이터 구성 및 전처리
- **데이터셋**: UCI, HHAR, REALWORLD, SHL 총 4개의 공개 데이터셋을 결합하여 80명의 가상 클라이언트를 생성하였다.
- **대상 활동**: Walk, Upstairs, Downstairs, Sit, Stand, Lay, Jump, Run, Bike, Car, Bus, Train, Subway 등 총 13가지 활동을 정의하였다.
- **입력 데이터**: 3축 가속도계 및 3축 자이로스코프 데이터를 사용하며, $50\text{Hz}$로 리샘플링하였다.
- **윈도우 설정**: 윈도우 크기는 $128$ (약 $2.56\text{s}$)이며, $50\%$의 오버랩을 적용하였다. 데이터는 채널별 Z-정규화(z-normalization)를 거쳤다.
- **데이터 분할**: 각 데이터셋의 $20\%$는 테스트셋으로, $80\%$는 훈련셋으로 사용한다. 훈련셋 중 다시 $80\%$는 라벨을 제거하여 클라이언트의 무감독 학습에 사용하고, 나머지 $20\%$의 라벨링 데이터는 서버의 미세 조정에 사용한다.

### 모델 아키텍처 및 학습 절차
- **Autoencoder (AE)**: 
    - **Encoder**: 4개의 1D Convolution 레이어로 구성되며, 각 레이어는 32개의 필터와 커널 크기 5를 가진다. 최종 잠재 공간(Latent Space)의 크기는 $128$이다.
    - **Decoder**: 인코더와 대칭되는 구조를 가지며, Transposed Convolution 레이어를 사용한다.
- **연합 학습 설정**: 
    - 알고리즘: $\text{FedAvg}$
    - 통신 라운드: $200$회 / 클라이언트 로컬 에포크: $5$회
    - 최적화 도구: $\text{SGD}$, 학습률(Learning Rate) $\text{LR} = 0.01$
- **분류기 및 미세 조정**:
    - 구조: $\text{Encoder} \rightarrow \text{Dense layer (32)} \rightarrow \text{Softmax (13)}$
    - 최적화 도구: $\text{ADAM}$, $\text{LR} = 0.00005$, 에포크 $200$회
    - 클래스 불균형 문제를 해결하기 위해 Class-weighted learning을 적용하였다.

## 📊 Results

### 실험 결과 비교
주요 지표로 Macro F-score를 사용하였으며, 결과는 다음과 같다.

| 학습 방법 | 전체 합산(Combined) | UCI | HHAR | REALWORLD | SHL |
| :--- | :---: | :---: | :---: | :---: | :---: |
| **FL + Autoencoder** | $71.29\%$ | $73.51\%$ | $82.84\%$ | $77.98\%$ | $68.81\%$ |
| **Conventional (Supervised)** | $\mathbf{71.96\%}$ | $\mathbf{78.92\%}$ | $\mathbf{82.92\%}$ | $76.84\%$ | $\mathbf{70.44\%}$ |
| **Conventional + AE** | $69.04\%$ | $73.06\%$ | $78.58\%$ | $75.48\%$ | $66.52\%$ |

### 분석 결과
1. **AE의 효과 부재**: 연합 학습(FL + AE)을 적용했음에도 불구하고, 단순히 라벨링된 데이터만 사용한 전통적인 지도 학습(Conventional) 방식보다 성능이 높지 않았다. 특히 중앙 집중식으로 AE를 사전 학습시킨 경우(Conventional + AE) 오히려 성능이 하락하는 결과가 나타났다.
2. **데이터 이질성의 영향**: FedAvg의 가중 평균 방식 특성상, 데이터 양이 많은 클라이언트(SHL, REALWORLD)의 영향력이 커져 데이터 양이 적은 클라이언트(UCI, HHAR)의 성능이 상대적으로 낮게 측정되었다.
3. **분류 오류 분석**: Confusion Matrix 분석 결과, 'Bus', 'Train', 'Subway'와 같은 이동 수단 클래스들이 서로 빈번하게 혼동되었으며, 특히 대중교통 이용 시 서 있는 상태가 많아 'Stand' 활동과도 자주 혼동되는 양상을 보였다.
4. **특징 공간의 중첩**: T-SNE 시각화 결과, AE를 통해 학습된 특징 표현들이 클래스별로 명확히 구분되지 않고 서로 얽혀 있는(intertwined) 모습이 확인되었다.

## 🧠 Insights & Discussion

본 연구를 통해 다음과 같은 학술적 통찰을 얻을 수 있다.

첫째, **FedAvg의 한계**이다. 단순한 가중 평균 방식은 데이터 분포가 극도로 불균형하고 이질적인 환경에서 특정 클라이언트에 편향된 모델을 생성할 위험이 크다. 이는 데이터 양이 적은 클라이언트의 특성을 반영하지 못하는 결과를 초래한다.

둘째, **특징 공간의 정렬(Alignment) 문제**이다. 각 클라이언트가 로컬 학습을 통해 데이터를 잠재 공간으로 투영할 때, 동일한 데이터라도 클라이언트마다 서로 다른 방향과 스케일로 투영하게 된다. 단순한 FedAvg 방식으로는 이러한 투영 규칙의 불일치를 해결할 수 없으므로, 글로벌한 정렬 규칙이 필요하다.

셋째, **모델의 표현력 부족**이다. 사용된 AE 구조는 매우 가벼워 연산 효율성은 높지만, 복잡한 HAR 데이터의 특징을 추출하기에는 너무 단순(decrepit)하다. 따라서 하드웨어 제약을 고려하면서도 더 강력한 특징 추출 능력을 가진 $\text{CPC}$, $\text{SimCLR}$, $\text{MoCo}$와 같은 최신 대조 학습 아키텍처의 도입이 필요하다.

## 📌 TL;DR

본 논문은 이질적인 HAR 데이터 환경에서 가벼운 Autoencoder와 FedAvg를 이용한 연합 자기 지도 학습의 가능성을 탐색하였다. 실험 결과, 단순한 AE 기반 접근법은 라벨 없는 데이터를 충분히 활용하지 못하며, 오히려 전통적인 지도 학습보다 성능이 낮거나 비슷한 수준에 그쳤다. 이는 실제 환경의 데이터 이질성을 극복하기 위해서는 단순한 평균 기반의 집계 방식과 기초적인 AE 구조를 넘어, 더 정교한 SSL 아키텍처와 고도화된 연합 학습 집계 알고리즘이 필수적임을 시사한다.
