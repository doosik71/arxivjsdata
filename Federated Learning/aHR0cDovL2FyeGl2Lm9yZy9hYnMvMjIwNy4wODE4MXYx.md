# Federated Continual Learning through distillation in pervasive computing

Anastasiia Usmanova, François Portet, Philippe Lalanda, and German Vega (2022)

## 🧩 Problem to Solve

본 논문은 퍼베이시브 컴퓨팅(Pervasive Computing) 환경의 연합 학습(Federated Learning, FL)에서 발생하는 **치명적 망각(Catastrophic Forgetting)** 문제를 해결하고자 한다.

퍼베이시브 컴퓨팅 환경의 모바일 기기들은 저장 공간이 매우 제한적이며, 수집되는 데이터의 특성이 시간에 따라 급격하게 변하는 동적인 특성을 가진다. 기존의 FL 솔루션들은 서버에서 전송된 모델을 미세 조정(Fine-tuning)하기 위해 클라이언트 측에 대량의 데이터를 저장하고 있어야 한다는 가정을 전제로 한다. 그러나 이는 실제 모바일 환경에서 비현실적이며, 새로운 데이터로 모델을 단순히 업데이트할 경우 이전에 학습했던 지식을 잃어버리는 치명적 망각 현상이 발생한다.

따라서 본 연구의 목표는 대량의 데이터 저장 없이도 새로운 데이터에 적응하면서 이전의 지식을 유지할 수 있는 **연합 지속 학습(Federated Continual Learning, FCL)** 접근 방식을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **지식 증류(Knowledge Distillation)** 기술을 연합 학습 프레임워크에 도입하여 모델의 지식을 보존하는 것이다. 특히, 모델이 새로운 작업을 학습할 때 과거의 모델이나 서버의 전역 모델을 '교사(Teacher)'로 삼아 현재의 '학생(Student)' 모델이 이전의 지식을 유지하도록 유도한다.

주요 기여 사항은 다음과 같다:
1. **FLwF-1 (Federated Learning without Forgetting - 1)**: 클라이언트의 이전 라운드 모델을 단일 교사로 사용하여 로컬 지식을 보존한다.
2. **FLwF-2 (Federated Learning without Forgetting - 2)**: 클라이언트의 이전 모델과 서버의 전역 모델이라는 두 개의 교사를 활용한다. 이를 통해 클라이언트의 개별적 특성(Specificity)과 서버의 일반적 지식(Generality)을 동시에 학습함으로써 망각을 더 효과적으로 억제한다.
3. **컨텍스트 인식 전략**: 모든 상황에 증류를 적용하는 대신, 데이터 분포가 불균형할 때만 증류를 사용하고 균형 잡힌 경우에는 미세 조정을 사용하는 하이브리드 전략의 효율성을 입증하였다.

## 📎 Related Works

기존의 연합 학습(FL)은 데이터 프라이버시를 보호하고 통신 비용을 줄이기 위해 모델 파라미터만을 교환하는 방식을 취한다. 하지만 CL(Continual Learning) 관점에서 보면, 각 클라이언트가 순차적으로 서로 다른 작업(Task)을 학습할 때 발생하는 치명적 망각 문제에 취약하다.

기존의 지속 학습 접근 방식들은 주로 대량의 과거 데이터를 저장하는 Replay 기반 방식이나, 복잡한 정규화(Regularization) 방식을 사용한다. 그러나 퍼베이시브 컴퓨팅 환경에서는 메모리 제약으로 인해 대규모 데이터 저장이 어렵다. 본 논문은 이러한 한계를 극복하기 위해 모델의 출력값(Logits)을 이용하는 지식 증류 방식을 채택하여, 데이터 저장량을 최소화하면서도 성능을 유지하는 차별점을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조
본 시스템은 하나의 서버와 여러 개의 클라이언트로 구성된다. 서버는 $\text{FedAvg}$ 알고리즘을 통해 클라이언트들의 모델을 집계하여 전역 모델을 생성하고 이를 다시 배포한다. 클라이언트는 로컬에서 데이터를 수집하여 모델을 업데이트하며, 이때 제안된 $FLwF$ 손실 함수를 사용하여 학습한다.

### 학습 방법론 및 손실 함수

모델의 출력 로짓(Logits)을 $o(x)$라 할 때, 온도 스케일링(Temperature Scaling)이 적용된 소프트맥스 확률 분포 $\pi$는 다음과 같이 정의된다.

$$\pi'_i(x) = \frac{e^{o'_i(x)/T}}{\sum_{j=1}^n e^{o'_j(x)/T}}$$

여기서 $T$는 온도 파라미터이며, $n$은 클래스의 수이다.

#### 1. FLwF-1 (단일 교사 방식)
FLwF-1은 클라이언트의 이전 라운드 모델($\theta_{r-1}^k$)을 교사로 사용한다. 최종 손실 함수는 분류 손실(Classification Loss)과 증류 손실(Distillation Loss)의 가중 합으로 구성된다.

- **증류 손실 ($L_{dis\_cl}$)**: 이전 모델의 예측값과 현재 모델의 예측값 사이의 차이를 측정한다.
$$L_{dis\_cl}(D^k_r; \theta^k_r, \theta^k_{r-1}) = \sum_{x \in X^k_r} \sum_{i=1}^n -\pi^{r-1}_i(x) \log [\pi^r_i(x)]$$
- **분류 손실 ($L_{class}$)**: 현재 데이터의 정답 라벨(Ground-truth)과의 교차 엔트로피를 측정한다.
- **최종 손실**: 
$$L_{FLwF-1} = \alpha L_{class} + (1-\alpha) L_{dis\_cl}$$

#### 2. FLwF-2 (이중 교사 방식)
FLwF-2는 이전 로컬 모델뿐만 아니라 서버의 전역 모델($\theta_{r-1}$)을 추가 교사로 도입한다.

- **서버 증류 손실 ($L_{dis\_serv}$)**: 서버 모델의 예측값 $\hat{\pi}$와 현재 모델의 예측값 사이의 차이를 측정한다.
$$L_{dis\_serv}(D^k_r; \theta^k_r, \theta_{r-1}) = \sum_{x \in X^k_r} \sum_{k=1}^n -\hat{\pi}^{r-1}_k(x) \log [\pi^r_k(x)]$$
- **최종 손실**:
$$L_{FLwF-2} = \alpha L_{class} + \beta L_{dis\_cl} + (1-\alpha-\beta) L_{dis\_serv}$$

여기서 $\alpha$와 $\beta$는 각 항의 영향력을 조절하는 하이퍼파라미터이다.

### 추론 및 학습 절차 (Algorithm 1)
1. 서버는 초기 모델 $\theta_0$를 생성한다.
2. 각 라운드 $r$마다 클라이언트는 서버로부터 $\theta_{r-1}$을 받고, 자신의 이전 모델 $\theta_{r-1}^k$와 함께 로컬 학습을 수행한다.
3. 클라이언트는 $L_{FLwF-2}$를 최소화하는 방향으로 가중치를 업데이트한 후 서버에 전송한다.
4. 서버는 $\text{FedAvg}$를 통해 모델을 집계하여 $\theta_r$을 생성하고 다시 배포한다.

## 📊 Results

### 실험 설정
- **데이터셋**: UCI Human Activity Recognition (HAR) 데이터셋 (6개 활동 클래스).
- **모델 아키텍처**: CNN (16x1 Conv $\rightarrow$ Max Pooling $\rightarrow$ Dense $\rightarrow$ Softmax).
- **시나리오**: 5명의 클라이언트 중 1명(Client 1)은 치명적 망각이 일어나기 쉬운 불균형한 작업 순서(Walking upstairs $\rightarrow$ Walking downstairs)를 학습하고, 나머지 4명은 균형 잡힌 데이터를 학습하는 환경을 조성하였다.
- **평가 지표**: 일반 정확도($A_{gen}$), 개인 정확도($A_{per}$), 평균 정확도($A_t$), 망각 측정치($F_t$).

### 주요 결과
1. **증류 기반 방식의 효과 (Exp 1)**: 단순 미세 조정(Fine-tuning)은 매우 심한 치명적 망각을 보였으나, FLwF-1과 FLwF-2는 이를 효과적으로 완화하였다. 특히 FLwF-2가 FLwF-1보다 더 나은 성능을 보였다.
2. **하이브리드 전략의 우수성 (Exp 2)**: 모든 경우에 증류를 사용하는 것보다, 불균형 데이터일 때만 증류를 사용하고 균형 데이터일 때는 미세 조정을 사용하는 전략이 가장 높은 정확도와 가장 낮은 망각 수치를 기록하였다.
3. **Exemplars 활용 (Exp 3)**: 소량의 과거 데이터 샘플(Exemplars)을 저장하여 함께 학습시켰을 때, FLwF-2 하이브리드 전략의 망각 수치가 $0.212$에서 $0.12$로 더욱 감소하여 최적의 성능을 보였다.

## 🧠 Insights & Discussion

### 강점 및 분석
- **서버 모델의 역할**: FLwF-2의 결과는 서버 모델이 단순한 집계체가 아니라, 클라이언트에게 일반적인 지식을 전달하여 오버피팅을 방지하고 망각을 줄이는 '지식의 저장소' 역할을 할 수 있음을 시사한다.
- **자원 효율성**: 본 제안 방식은 추가적인 대량의 데이터 저장 없이 기존에 존재하는 모델 파라미터만을 활용하므로, 메모리가 제한된 퍼베이시브 기기에 매우 적합하다.
- **맥락 인식의 중요성**: 실험 결과, 데이터가 균형 잡힌 상황에서는 증류가 오히려 성능을 저하시킬 수 있음이 밝혀졌다. 이는 무조건적인 지식 보존보다 현재 데이터의 특성에 맞는 학습 전략(Fine-tuning vs Distillation)을 선택하는 것이 중요함을 의미한다.

### 한계 및 비판적 해석
- **가정의 단순함**: 실험에서 '일반화된 클라이언트(Generalized Client)'를 단일 모델로 단순화하여 모델링하였으나, 실제 환경에서는 각 클라이언트의 데이터 분포가 훨씬 더 복잡하고 다양할 것이다.
- **하이퍼파라미터 의존성**: $\alpha, \beta, T$와 같은 하이퍼파라미터가 그리드 서치(Grid Search)를 통해 결정되었는데, 실제 환경에서 이러한 최적값을 실시간으로 찾는 방법은 제시되지 않았다.

## 📌 TL;DR

본 논문은 퍼베이시브 컴퓨팅 환경의 연합 학습에서 발생하는 치명적 망각 문제를 해결하기 위해 **지식 증류(Knowledge Distillation) 기반의 연합 지속 학습(FCL)** 프레임워크를 제안하였다. 특히 이전 로컬 모델과 서버의 전역 모델을 동시에 교사로 사용하는 **FLwF-2** 방식이 단일 교사 방식보다 지식 보존 능력이 뛰어남을 입증하였다. 또한, 데이터 분포에 따라 미세 조정과 증류를 선택적으로 사용하는 하이브리드 전략이 가장 효과적임을 보였다. 이 연구는 메모리 제약이 큰 IoT 및 웨어러블 기기에서 지속적인 학습을 구현하는 데 중요한 기여를 할 것으로 평가된다.
