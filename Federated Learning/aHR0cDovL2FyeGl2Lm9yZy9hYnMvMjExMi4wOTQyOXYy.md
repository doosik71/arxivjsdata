# Federated Learning with Superquantile Aggregation for Heterogeneous Data

Krishna Pillutla, Yassine Laguel, Jérôme Malick, Zaid Harchaoui (2022)

## 🧩 Problem to Solve

연합 학습(Federated Learning, FL) 환경에서 발생하는 핵심적인 문제는 클라이언트 간의 데이터 분포가 서로 다른 통계적 이질성(Statistical Heterogeneity)이다. 기존의 표준적인 연합 학습 방식인 FedAvg는 전체 클라이언트의 평균 손실(Average Loss)을 최소화하는 것을 목표로 한다.

그러나 평균 손실을 최소화하는 접근 방식은 데이터 분포가 전체 인구 집단과 크게 다르거나, 보유한 데이터의 양이 매우 적은 '꼬리 클라이언트(Tail Clients)'에 대해 매우 낮은 예측 성능을 보이는 결과를 초래한다. 즉, 평균적인 성능은 높을 수 있으나, 특정 소수 클라이언트에게는 모델이 제대로 작동하지 않는 성능 불균형 문제가 발생한다.

본 논문의 목표는 데이터 이질성이 심한 환경에서도 모든 개별 클라이언트에게 최소한의 예측 성능을 보장할 수 있는 강건한(Robust) 연합 학습 프레임워크를 설계하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 손실 분포의 꼬리 부분 통계량을 캡처하는 **Superquantile**(또는 Conditional Value at Risk, CVaR) 기반의 학습 목적 함수를 도입하는 것이다.

$\Delta$-FL이라 명명된 이 프레임워크는 단순히 평균을 최적화하는 대신, 손실이 특정 임계값(Tail Threshold)보다 큰 클라이언트들의 평균 손실을 최소화함으로써, 성능이 가장 낮은 클라이언트들의 성능을 끌어올리는 데 집중한다. 또한, 이를 구현하기 위해 차분 프라이버시(Differential Privacy, DP)가 적용된 분위수(Quantile) 추정 알고리즘과 연합 평균(Federated Averaging) 단계를 결합한 최적화 알고리즘을 제안한다.

## 📎 Related Works

기존의 연구들은 크게 두 가지 방향으로 이질성 문제를 해결하려 했다.

1.  **알고리즘적 개선**: Proximal term을 추가한 FedProx나 제어 변수(Control Variates)를 사용하는 방식 등은 수렴 속도를 개선하고 이질성의 영향을 완화하려 했으나, 여전히 평균 손실 최소화라는 기본 목적 함수를 유지한다.
2.  **목적 함수 설계**: AFL(Agnostic Federated Learning)은 모든 클라이언트 중 최악의 오류(Worst-case error)를 최소화하려 하며, q-FFL은 손실의 $L_p$ 노름을 최소화하여 공정성을 높이려 한다. Tilted-ERM은 엔트로피 위험 측정치(Entropic Risk Measure)를 사용한다.

$\Delta$-FL은 최악의 경우만을 고려하는 AFL과 달리, 꼬리 부분의 평균을 고려하는 Superquantile을 사용하여 더 유연하고 효율적인 최적화를 수행한다. 또한, 본 논문은 제안된 알고리즘의 수렴성을 수학적으로 증명하고, 실제 구현 시 필수적인 차분 프라이버시(DP) 적용 방안을 구체적으로 제시했다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 1. Superquantile 목적 함수
본 논문은 $(1-\theta)$-superquantile $S_\theta$를 사용하여 다음과 같은 목적 함수를 정의한다.
$$F_\theta(w) := S_\theta(F_1(w), \dots, F_n(w)) + \frac{\lambda}{2}\|w\|^2$$
여기서 $S_\theta(Z) = E[Z | Z > Q_\theta(Z)]$이며, $Q_\theta(Z)$는 $(1-\theta)$-분위수이다. 즉, 손실 분포에서 상위 $\theta$ 비율에 해당하는 클라이언트들의 평균 손실을 최소화하는 것이다. $\theta \to 1$이면 일반적인 FedAvg(평균 최소화)가 되고, $\theta \to 0$이면 AFL(최악의 경우 최소화)이 된다.

### 2. $\Delta$-FL 파이프라인
알고리즘은 다음과 같은 절차로 진행된다.
1.  **모델 배포**: 서버가 선택된 $m$개의 클라이언트에 현재 모델 $w^{(t)}$를 전송한다.
2.  **분위수 추정**: 클라이언트와 서버가 협력하여 차분 프라이버시가 보장된 방식으로 손실의 $(1-\theta)$-분위수 $\hat{Q}_\theta$를 추정한다.
3.  **클라이언트 필터링**: 자신의 손실 $F_i(w^{(t)}) \ge \hat{Q}_\theta$인 '꼬리 클라이언트'들만 업데이트에 참여하도록 가중치 $\tilde{\pi}_i$를 설정한다.
4.  **로컬 업데이트**: 선택된 클라이언트들은 $\tau$번의 로컬 경사 하강법(SGD)을 수행한다.
5.  **가중 합산**: 서버는 꼬리 클라이언트들의 업데이트만을 평균 내어 전역 모델 $w^{(t+1)}$을 갱신한다.

### 3. DP 기반 분위수 추정 (Algorithm 2)
클라이언트의 개별 손실 값이 서버에 직접 노출되는 것을 방지하기 위해 다음과 같은 기법을 사용한다.
-   **Hierarchical Histogram**: 손실 범위를 이진 트리 구조의 히스토그램으로 나누어 표현한다.
-   **Distributed Discrete Gaussian Mechanism**: 각 클라이언트가 히스토그램 빈(bin)에 노이즈를 추가하고, Secure Aggregation을 통해 합산된 히스토그램만을 서버가 확인하게 하여 개별 데이터를 보호한다.

### 4. 이론적 수렴성
본 논문은 다음과 같은 수렴 속도를 증명하였다.
-   **비볼록(Non-convex) 케이스**: $T$번의 통신 라운드 동안 $O(1/\sqrt{T})$의 속도로 수렴한다.
-   **강볼록(Strongly Convex) 케이스**: $O(\exp(-T/\kappa^{3/2}) + \kappa/T)$의 속도로 수렴하며, 여기서 $\kappa$는 로컬 컨디션 넘버이다.

## 📊 Results

### 1. 실험 설정
-   **데이터셋**: EMNIST(문자 인식), Sent140(감성 분석).
-   **모델**: Linear Model, ConvNet, RNN(LSTM).
-   **비교 대상**: FedAvg, FedProx, q-FFL, AFL, Tilted-ERM.
-   **평가 지표**: 평균 오차(Mean Error) 및 90번째 백분위수 오차(90th percentile error, 꼬리 성능 지표).

### 2. 주요 결과
-   **꼬리 성능 개선**: $\Delta$-FL은 모든 벤치마크에서 90번째 백분위수 오차를 가장 효과적으로 낮추었다. 특히 EMNIST-ConvNet에서 기존 Vanilla FL 대비 약 3.3%의 절대적 성능 향상을 보였다.
-   **평균 성능 유지**: 꼬리 성능을 개선하면서도 평균 오차 측면에서는 FedAvg와 경쟁 가능한 수준을 유지하였다.
-   **데이터 부족 클라이언트 지원**: 분석 결과, $\Delta$-FL은 특히 보유 데이터 양이 적은(Data-poor) 클라이언트들의 예측 오차를 크게 줄여주는 효과가 있음을 확인하였다.
-   **프라이버시-효용 트레이드오프**: DP를 적용했을 때도 $\Delta$-FL은 FedAvg보다 꼬리 오차 측면에서 우수한 성능을 보였으며, 평균 오차에 대해서는 FedAvg와 유사한 프라이버시-효용 효율성을 나타냈다.

## 🧠 Insights & Discussion

### 1. 강점 및 해석
$\Delta$-FL은 "모두를 위한 최적화"가 아닌 "가장 뒤처진 이들을 위한 최적화"를 수행함으로써 시스템 전체의 하한선(Lower bound)을 끌어올린다. 특히 단순히 최악의 케이스 하나만을 쫓는 AFL보다 꼬리 부분의 평균을 최적화하는 것이 수렴 안정성과 실제 성능 면에서 훨씬 유리함을 입증하였다.

### 2. 한계 및 논의사항
-   **클라이언트 가용성 문제**: $\Delta$-FL의 가중치 업데이트는 매우 희소(Sparse)하게 일어난다. 즉, $\theta m$명의 클라이언트만 업데이트에 기여하므로, 동일한 수의 유효 업데이트를 얻기 위해서는 FedAvg보다 더 많은 수의 클라이언트를 처음에 샘플링해야 하며, 이는 클라이언트 가용성이 낮은 환경에서 제약이 될 수 있다.
-   **개인화(Personalization)와의 관계**: 실험 결과, 모델 개인화(Fine-tuning)를 거치면 $\Delta$-FL과 FedAvg의 성능 차이가 사라지는 경향을 보였다. 이는 $\Delta$-FL이 전역 모델 자체의 강건성을 높이는 데 효과적이지만, 개별 최적화 단계에서는 그 효과가 희석될 수 있음을 시사한다.

## 📌 TL;DR

본 논문은 연합 학습에서 데이터 이질성으로 인해 발생하는 성능 불균형 문제를 해결하기 위해, 손실 분포의 꼬리 부분을 최적화하는 **Superquantile** 기반의 $\Delta$-FL 프레임워크를 제안한다. DP 기반의 분위수 추정을 통해 프라이버시를 보호하면서 꼬리 클라이언트만을 선별적으로 학습에 참여시키며, 이를 통해 평균 성능을 유지하면서도 하위 10% 클라이언트의 성능을 획기적으로 개선하였다. 이 연구는 향후 공정한 머신러닝(Fair ML) 및 강건한 분산 학습 시스템 설계에 중요한 기초를 제공한다.
