# Memory-Augmented SAM2 for Training-Free Surgical Video Segmentation

Ming Yin, Fu Wang, Xujiong Ye, Yanda Meng, and Zeyu Fu (2025)

## 🧩 Problem to Solve

본 논문은 컴퓨터 보조 수술에서 필수적인 수술 기구 분할(Surgical Instrument Segmentation, SIS) 작업에서 발생하는 성능 저하 문제를 해결하고자 한다. 최근 이미지 및 비디오 분할에서 뛰어난 성능을 보인 Segment Anything Model 2 (SAM2)가 등장하였으나, 수술 비디오의 특성상 기구의 급격한 움직임, 빈번한 가려짐(occlusion), 그리고 복잡한 기구-조직 간의 상호작용이 빈번하게 발생한다.

SAM2의 기존 메모리 설계는 Greedy selection 방식(FIFO 큐 기반의 순차적 메모리 처리)을 사용하는데, 이는 중간 프레임에서 불확실하거나 신뢰도가 낮은 마스크가 생성될 경우 불안정성이 누적되는 결과를 초래한다. 구체적으로는 기구가 화면에서 사라졌다가 다시 나타날 때 추적에 실패하거나, 여러 기구가 서로 겹칠 때 분할 결과에 모호함이 발생하는 문제가 있다. 따라서 본 논문의 목표는 추가적인 학습 없이(training-free) 이러한 메모리 한계를 극복하여 복잡한 수술 비디오에서도 강건한 분할 성능을 유지하는 MA-SAM2 전략을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 SAM2의 고정된 메모리 구조를 개선하여, 문맥 인식 능력과 가려짐에 대한 회복력을 높인 새로운 메모리 증강 전략을 도입하는 것이다.

1. **MA-SAM2 프레임워크 제안**: 추가 학습 없이 SAM2의 성능을 향상시키는 training-free 전략을 제안하며, 특히 단일 프롬프트만으로 다중 타겟을 동시에 추적하는 one-prompt 메커니즘을 통해 효율성을 높였다.
2. **새로운 메모리 뱅크 아키텍처**: Context-Aware Memory (CAM)와 Occlusion-Resilient Memory (ORM)를 동적으로 통합하는 메모리 구조를 설계하여, 장기적인 시간적 모델링 능력을 강화하고 가려짐 간섭에 대한 강건성을 확보하였다.
3. **성능 입증**: EndoVis2017 및 EndoVis2018 데이터셋에서 SAM2 대비 각각 6.1%와 4.36%의 Challenge IoU 향상을 달성하여 실제 수술 환경에서의 적용 가능성을 증명하였다.

## 📎 Related Works

기존의 SAM(Segment Anything Model)은 정적 이미지에 최적화되어 있어 비디오 분할에 필요한 시간적 일관성(temporal consistency)을 유지하는 데 한계가 있었다. 이를 해결하기 위해 SAM2가 제안되었으며, 최근 Surgical SAM2와 같은 연구들이 수술 비디오 분할에 이를 적용하기 위해 프레임 프루닝(frame pruning) 등의 전략을 도입하였다.

하지만 기존의 의료 영상 적응형 SAM 모델들은 여전히 추가적인 학습(training)이나 미세 조정(fine-tuning)이 필요한 경우가 많으며, SAM2의 근본적인 문제인 Greedy memory selection으로 인한 추론 오류를 완전히 해결하지 못했다. MA-SAM2는 이러한 기존 방식과 달리 **학습이 전혀 필요 없는(training-free)** 구조이면서도, 메모리 관리 전략을 수정함으로써 SAM2의 추론 오류를 직접적으로 해결한다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조

MA-SAM2는 입력 프레임의 후보 마스크들을 CAM과 ORM이라는 두 개의 서로 다른 메모리 경로로 분리하여 관리한다. 고정된 용량의 메모리 뱅크 내에서 ORM을 우선적으로 채우고, 남은 용량이 있을 때 CAM을 추가하는 적응형 설계를 채택한다.

### Occlusion-Resilient Memory (ORM)

ORM은 기구의 움직임, 소멸, 또는 가려짐으로 인해 발생하는 분할 아티팩트를 억제하기 위해 설계되었다.

1. **마스크 선택**: 현재 프레임의 여러 후보 마스크 중 평균 IoU 점수가 가장 높은 마스크를 기본 예측값 $M_s$로 선택한다. 이때 신뢰도를 보장하기 위해 $\theta_{IoU} \ge 0.8$의 임계값을 적용한다.
2. **간섭 마스크 생성**: 기본 예측값 $M_s$ 외의 나머지 후보 마스크들을 대안 가설 $M_a$로 간주한다. $M_s$와 겹치는 영역을 제거하고 가장 큰 연결 성분(Largest Connected Component, CC)만을 남겨 정제된 간섭 마스크 $M_f$를 생성한다.
   $$M_f = CC(M_s \setminus (M_s \cap M_a)) \cup (M_s \cap M_a)$$
3. **저장 결정**: $M_f$와 $M_s$의 바운딩 박스 겹침 비율이 $0.6 \sim 0.9$ 범위 내에 있으면 간섭이 적은 것으로 판단하여 결과를 수용한다. 강한 간섭이 식별된 프레임들만을 추출하여 ORM에 저장하며, 최대 저장 용량은 5프레임으로 제한하여 추론 속도를 유지하고 불필요한 과거 정보의 간섭을 방지한다.

### Context-Aware Memory (CAM)

CAM은 SAM2의 단기 최적화(greedy) 방식이 초래하는 누적 오류를 해결하고, 기구가 재등장했을 때의 모호성을 제거하기 위해 고품질의 과거 마스크를 유지한다.

1. **필터링**: ORM에 저장된 프레임과 초기 프롬프트 프레임을 제외한 나머지 프레임 중, 평균 신뢰도와 IoU 점수가 설정된 임계값을 모두 초과하는 신뢰할 수 있는 마스크만을 저장한다.
2. **누적 점수 기반 선택**: 디코더가 생성한 3개의 후보 마스크에 대해 다음과 같은 누적 점수 $S_k(t)$를 계산하여 최적의 가설을 선택한다.
   $$S_k(t) = S_k(t-1) + \log(IoU_k(t) + \epsilon)$$
   여기서 $k$는 후보 마스크의 인덱스, $t$는 현재 프레임, $\epsilon$은 로그 계산 시 0이 되는 것을 방지하는 작은 상수이다. 가장 높은 누적 점수를 가진 마스크를 최종 결과로 선택하고 다음 타임스텝으로 전달한다.

### 추론 절차: One-Prompt Strategy

실시간성을 위해 각 카테고리의 기구가 처음 등장할 때 단 한 번의 프롬프트만 제공하는 mask-based one-prompt 전략을 사용한다. 이후에는 추가적인 수동 개입 없이 MA-SAM2의 메모리 증강 전략을 통해 자동으로 추적을 수행한다.

## 📊 Results

### 실험 설정

- **데이터셋**: EndoVis2017 (8개 시퀀스, 각 225프레임) 및 EndoVis2018 (15개 시퀀스, 각 149프레임)을 사용하였다.
- **평가 지표**: Challenge IoU, IoU, mean class IoU (mcIoU)를 사용하였다.
- **비교 대상**: TrackAnything, PerSAM, Surgical SAM2, SAM2.
- **구현 세부사항**: 모든 실험은 training-free 설정으로 수행되었으며, 입력 이미지는 $512 \times 512$로 리사이즈되었다.

### 정량적 결과

MA-SAM2는 모든 데이터셋에서 baseline인 SAM2보다 우수한 성능을 보였다.

- **EndoVis2017**: Challenge IoU 62.49%를 기록하여 SAM2(56.39%) 대비 **6.10% 향상**되었으며, mcIoU 또한 59.89%로 SAM2(55.38%)보다 4.51% 높았다.
- **EndoVis2018**: Challenge IoU 64.40%를 기록하여 SAM2(60.04%) 대비 **4.36% 향상**되었으며, mcIoU는 62.13%로 SAM2(58.40%)보다 3.73% 높았다. 특히 MSC(75.35%) 및 PF(44.91%)와 같은 복잡한 카테고리에서 강점을 보였다.

### 절제 실험 (Ablation Study)

CAM과 ORM의 기여도를 분석한 결과, SAM2에 CAM만 추가했을 때보다 ORM을 함께 추가했을 때 성능 향상 폭이 훨씬 컸다. (EndoVis2018 기준: SAM2(60.04%) $\rightarrow$ +CAM(60.93%) $\rightarrow$ +ORM(63.98%) $\rightarrow$ +Both(64.40%)) 이는 가려짐 및 간섭 제거(ORM)가 수술 비디오 분할에서 매우 중요한 요소임을 시사한다.

## 🧠 Insights & Discussion

본 연구의 강점은 별도의 학습 과정 없이 기존 SAM2의 메모리 관리 방식만 수정하여 수술 비디오라는 특수 환경의 난제(가려짐, 재등장)를 효과적으로 해결했다는 점이다. 특히 ORM을 통한 간섭 마스크 정제와 CAM의 누적 점수 기반 가설 선택은 SAM2의 Greedy selection으로 인한 불안정성을 체계적으로 보완한다.

다만, 저자들은 결과 분석에서 신뢰도 점수와 IoU 값에 의존하여 메모리 품질을 유지하기 때문에, 매우 복잡한 상호작용이 일어나는 일부 상황(예: LND 기구)에서는 신뢰도가 낮은 마스크들이 필터링되어 일부 누락될 수 있음을 인정하였다. 이는 전체적인 강건성을 얻기 위한 합리적인 트레이드-오프(trade-off)로 해석된다.

비판적으로 보았을 때, 본 논문은 $\theta_{IoU}$나 $\epsilon$, 메모리 저장 용량(5프레임) 등의 하이퍼파라미터를 설정하였으나, 이 값들이 어떻게 최적화되었는지에 대한 상세한 분석이나 다양한 값에 대한 민감도 분석은 명시되지 않았다.

## 📌 TL;DR

MA-SAM2는 SAM2의 Greedy 메모리 설계로 인한 추적 실패와 모호성 문제를 해결하기 위해 **Context-Aware Memory(CAM)**와 **Occlusion-Resilient Memory(ORM)**를 도입한 **Training-free** 수술 기구 분할 전략이다. 단일 프롬프트만으로 다중 타겟을 효율적으로 추적하며, EndoVis 데이터셋에서 SAM2 대비 최대 6.1%의 IoU 향상을 달성하였다. 이 연구는 추가 학습 없이도 고성능의 의료 비디오 분할이 가능함을 보여주어, 실제 수술 현장의 실시간 분석 시스템에 적용될 가능성이 높다.
