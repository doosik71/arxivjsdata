# Energy-Based Models for Continual Learning

Shuang Li, Yilun Du, Gido M. van de Ven, Igor Mordatch (2022)

## 🧩 Problem to Solve

본 논문은 지속 학습(Continual Learning, CL) 과정에서 발생하는 치명적 망각(Catastrophic Forgetting) 문제를 해결하고자 한다. 특히, 새로운 클래스가 계속해서 추가되는 클래스 증분 학습(Class-Incremental Learning, Class-IL) 상황에서 기존의 소프트맥스 기반 분류기(Softmax-Based Classifier, SBC)가 보이는 한계를 지적한다.

SBC는 정규화된 확률 분포를 사용하여 학습하며, 교차 엔트로피(Cross-Entropy) 손실 함수를 통해 학습된다. 이러한 구조는 새로운 태스크를 학습할 때 현재 관찰되는 클래스의 확률은 높이지만, 학습 데이터에 포함되지 않은 이전 클래스들의 확률을 과도하게 억제하는 경향이 있다. 결과적으로 소프트맥스 연산의 '승자 독식(Winner-take-all)' 역학이 이전 지식을 빠르게 삭제하여 성능 저하를 초래한다. 본 논문의 목표는 이러한 정규화된 확률 분포의 제약에서 벗어나, 비정규화된 확률 분포를 학습하는 에너지 기반 모델(Energy-Based Models, EBMs)을 통해 지속 학습 성능을 향상시키는 것이다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 분류 문제를 각 클래스에 대한 에너지 값을 학습하는 문제로 재정의하는 것이다. 중심적인 기여 사항은 다음과 같다.

1. **분류를 위한 EBM 프레임워크 제안**: 입력을 클래스 레이블과 결합하여 스칼라 에너지 값으로 매핑하는 에너지 함수를 도입하여, 정규화 과정 없이도 클래스 간 구분을 가능하게 한다.
2. **Contrastive Divergence(CD) 기반 학습 목표 설계**: 모든 클래스의 에너지를 동시에 조정하는 대신, 정답 레이블의 에너지는 낮추고 현재 배치에서 샘플링된 단일 부정 레이블(Negative Label)의 에너지만을 높이는 단순하고 효율적인 학습 목적 함수를 제안한다. 이는 이전 클래스들에 대한 간섭을 최소화한다.
3. **레이블 조건화 아키텍처(Label Conditioning Architecture)**: 레이블을 어텐션 필터(Attention Filter) 또는 게이트(Gate)로 활용하여 입력 데이터에서 해당 클래스와 가장 관련 있는 정보만을 선택적으로 추출하도록 설계하였다.
4. **범용적 적용 가능성 입증**: 제안 방법이 경계 인식(Boundary-aware) 설정뿐만 아니라, 태스크 경계가 모호한 경계 자유(Boundary-free) 설정에서도 효과적임을 보였으며, 기존의 리플레이(Replay) 기반 방법론과 결합하여 성능을 더욱 높일 수 있음을 입증하였다.

## 📎 Related Works

지속 학습은 크게 다음과 같은 설정과 접근 방식으로 나뉜다.

- **학습 설정**:
  - **Task-IL vs Class-IL**: Task-IL은 추론 시 태스크 ID가 제공되어 해당 태스크 내 레이블만 예측하면 되지만, Class-IL은 지금까지 학습한 모든 클래스 중에서 정답을 찾아야 하므로 훨씬 더 어렵다.
  - **Boundary-aware vs Boundary-free**: 전자는 태스크 간 경계가 명확히 구분되어 제공되는 환경이며, 후자는 데이터 분포가 점진적으로 변하여 경계가 없는 환경이다. 기존의 많은 CL 방법론은 경계 인식 설정에 의존하여 통합 단계(Consolidation step)를 수행하므로 경계 자유 설정에서는 작동하기 어렵다.

- **기존 접근 방식의 한계**:
  - **Task-specific/Growing models**: 태스크별로 네트워크 일부를 할당하거나 모델을 확장하지만, 추론 시 태스크 ID가 필요하며 Class-IL에 부적합하다.
  - **Regularization-based (e.g., EWC, SI)**: 중요 파라미터의 변화를 억제하지만, 학습이 진행될수록 새로운 태스크를 위한 모델 용량이 줄어들며 실제 Class-IL 환경에서는 성능이 낮다.
  - **Replay methods (e.g., iCaRL, DGR)**: 이전 데이터를 저장하거나 생성하여 재학습시키지만, 메모리 비용이 높고 데이터 저장 및 생성 모델 학습 자체가 어려운 문제다.

## 🛠️ Methodology

### 1. 에너지 기반 모델(EBM)의 정의

EBM은 데이터 $x$의 가능도(Likelihood)를 다음과 같은 볼츠만 분포(Boltzmann distribution)로 정의한다.
$$p_\theta(x) = \frac{\exp(-E_\theta(x))}{Z(\theta)}, \quad Z(\theta) = \int_{x \in X} \exp(-E_\theta(x)) dx$$
여기서 $E_\theta(x)$는 입력 $x$를 스칼라 값으로 매핑하는 에너지 함수이다.

본 논문에서는 이를 분류 문제로 확장하여, 입력 $x$가 주어졌을 때 레이블 $y$의 조건부 확률을 다음과 같이 정의한다.
$$p_\theta(y|x) = \frac{\exp(-E_\theta(x, y))}{Z(\theta; x)}, \quad Z(\theta; x) = \sum_{y' \in Y} \exp(-E_\theta(x, y'))$$

### 2. Contrastive Divergence (CD) 학습 목표

전통적인 최대 가능도 추정(MLE) 방식은 모든 $y'$에 대해 분할 함수 $Z$를 계산해야 하므로, 새로운 클래스를 배울 때 기존 클래스들의 에너지를 모두 높여 치명적 망각을 유발한다. 이를 해결하기 위해 본 논문은 Contrastive Divergence 근사치를 사용하여 다음과 같은 손실 함수를 제안한다.
$$L_{CD}(\theta; x, y) = \mathbb{E}_{(x,y) \sim p_D} [E_\theta(x, y) - E_\theta(x, y^-)]$$
여기서 $y$는 정답 레이블이고, $y^-$는 현재 학습 배치 $Y_B$에서 무작위로 선택된 정답이 아닌 레이블($y^- \neq y$)이다. 이 방식은 정답의 에너지는 낮추고 단 하나의 부정 샘플 에너지분만 높이므로, 학습되지 않은 다른 이전 클래스들에 주는 영향(Interference)을 최소화한다.

### 3. 에너지 네트워크 아키텍처

단순히 $x$와 $y$를 결합하는 것이 아니라, 레이블 $y$를 어텐션 필터로 활용하는 구조를 제안한다.

1. $x$를 신경망에 통과시켜 특징 $f(x)$를 추출한다.
2. $y$를 작은 학습 가능 네트워크나 고정 투영(Fixed projection)을 통해 $g(y)$로 변환한다.
3. 게이팅 블록 $G$를 통해 $m(x, y) = G(f(x), g(y))$를 계산하여 가장 관련 있는 정보를 선택한다.
4. 최종적으로 가중치 레이어를 통해 에너지 값 $E_\theta(x, y)$를 출력한다.

### 4. 추론 절차

추론 시에는 입력 $x$에 대해 모든 가능한 클래스 $y' \in Y$에 대한 에너지를 계산하고, 가장 낮은 에너지 값을 가진 클래스를 최종 예측값으로 선택한다(MAP 추정).
$$\hat{y} = \arg \min_{y' \in Y} E_\theta(x, y')$$

## 📊 Results

### 1. 실험 설정

- **데이터셋**: split MNIST, permuted MNIST, CIFAR-10, CIFAR-100
- **비교 대상**: SBC(Baseline), EWC, Online EWC, SI, LwF, MAS, BGD 등
- **지표**: Class-IL 설정에서의 평균 정확도(Accuracy)

### 2. 주요 정량적 결과

- **Non-replay 설정**: EBM은 리플레이를 사용하지 않는 설정에서 SBC 및 기존 정규화 기반 방법론(EWC, SI 등)보다 압도적인 성능 향상을 보였다. 예를 들어, split MNIST에서 SBC는 $19.90\%$에 그친 반면 EBM은 $53.12\%$를 달성하였다.
- **Replay 결합**: EBM을 Exact Replay(ER)와 결합한 `EBM+ER`은 `SBC+ER`보다 뛰어난 성능을 보였으며, 특히 CIFAR-100에서는 리플레이를 사용하지 않은 EBM($30.28\%$)이 리플레이를 사용한 SBC($28.57\%$)보다 더 높은 성능을 기록하는 놀라운 결과를 보였다.
- **경계 자유(Boundary-free) 설정**: 태스크 경계가 없는 스트리밍 데이터 환경에서도 EBM은 SBC나 Online EWC보다 월등히 높은 정확도를 기록하여, 범용적인 CL 모델로서의 가능성을 입증하였다.

### 3. 분석 결과

- **학습 목표의 영향**: SBC 모델에 EBM의 CD 학습 목표만 적용(SBC*)해도 성능이 크게 향상되었으나, 레이블 조건화 아키텍처까지 갖춘 EBM이 가장 높은 성능을 보였다.
- **에너지 랜드스케이프**: 시각화 결과, SBC는 새로운 태스크를 학습하면 모든 데이터의 예측값이 최신 클래스로 쏠리는 경향이 있으나, EBM은 이전 태스크의 정답 레이블에 대해 낮은 에너지 값을 유지하며 망각을 방지함을 확인하였다.

## 🧠 Insights & Discussion

### 강점 및 유효성

본 논문은 지속 학습의 핵심 문제가 '분류기 아키텍처'와 '학습 목적 함수'의 상호작용에 있음을 밝혀냈다. 특히 정규화된 확률 분포를 강제하는 소프트맥스 층이 오히려 치명적 망각을 가속화한다는 통찰을 제시하였으며, 이를 비정규화된 에너지 기반 모델로 대체함으로써 해결하였다. 또한, CD 손실 함수가 부정 샘플의 선택 범위를 현재 배치로 제한함으로써 이전 지식에 대한 간섭을 물리적으로 차단한다는 점이 효과적이었다.

### 한계 및 논의사항

1. **추론 비용**: 추론 시 모든 클래스에 대해 에너지 값을 계산해야 하므로, 클래스 수가 극도로 많아질 경우 계산 비용이 증가할 수 있다. 다만, 실험 결과 실제 구현 상의 시간 차이는 미미했다.
2. **모델 용량**: 추가 분석을 통해 EBM이 동일한 파라미터 수의 SBC보다 더 큰 유효 용량(Effective Capacity)을 가지고 있음을 확인하였다. 이는 EBM의 성공이 단순히 망각 방지뿐만 아니라 더 많은 데이터를 기억할 수 있는 능력에서 기인했을 가능성을 시사한다.
3. **평가 지표의 단순함**: 평균 정확도 위주로 평가되었으며, Forward/Backward Transfer와 같은 전이 학습 능력을 보다 세밀하게 분석할 필요가 있다.

## 📌 TL;DR

본 논문은 지속 학습(CL), 특히 클래스 증분 학습(Class-IL)에서 발생하는 치명적 망각 문제를 해결하기 위해 **에너지 기반 모델(EBM)**을 도입하였다. 기존 소프트맥스 기반 분류기의 '승자 독식' 구조가 망각을 유발한다는 점에 착안하여, **Contrastive Divergence(CD) 손실 함수**와 **레이블 조건화 어텐션 구조**를 통해 간섭을 최소화하고 기억 용량을 극대화하였다. 실험 결과, 리플레이 없이도 기존 방법론을 크게 상회하는 성능을 보였으며, 태스크 경계가 없는 환경에서도 강건하게 작동함을 증명하였다. 이는 향후 지속 학습 모델 설계에 있어 정규화된 확률 분포 대신 에너지 기반의 접근 방식이 매우 유용한 빌딩 블록이 될 수 있음을 시사한다.
