# Continual Learning with Self-Organizing Maps

Pouya Bashivan, Martin Schrimpf, Robert Ajemian, Irina Rish, Matthew Riemer, Yuhai Tu (2019)

## 🧩 Problem to Solve

현대 신경망은 대규모의 통제된 데이터셋을 통해 한 번 학습된 특정 도메인의 정적인 환경에서는 매우 뛰어난 성능을 보이지만, 데이터 분포가 시간에 따라 변하는 비정상 상태(non-stationary) 학습 환경에서는 심각한 문제를 겪는다. 특히, 새로운 작업을 학습할 때 이전에 학습했던 지식을 급격히 망각하는 '치명적 망각(catastrophic forgetting)' 현상이 발생한다.

기존의 연구들은 이를 해결하기 위해 과거 작업의 샘플을 저장하는 메모리 리플레이 버퍼(memory replay buffers)를 사용하거나, 파라미터를 정규화하는 방식을 사용해 왔다. 그러나 메모리 기반 방식은 실제 환경의 복잡한 문제로 확장될 때 요구되는 메모리 용량이 지나치게 커진다는 확장성(scalability) 문제가 존재한다. 따라서 본 논문의 목표는 메모리 리플레이 버퍼 없이도 과거의 학습 내용을 유지하면서 새로운 작업을 학습할 수 있는 메모리리스(memoryless) 방식의 지속 학습(continual learning) 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 비지도 학습 메커니즘인 Self-Organizing Map(SOM)을 활용하여 입력 데이터를 적절한 작업 컨텍스트(task contexts)로 적응적으로 클러스터링하고, 이에 따라 네트워크 자원을 할당하는 것이다.

SOM은 명시적인 작업 레이블 없이도 입력 데이터의 다중 모드 분포(multimodal data distributions)를 캡처하여, 각 입력 샘플을 네트워크의 가장 관련성 높은 부분으로 라우팅(routing)하는 역할을 수행한다. 이를 통해 유사한 작업 간에는 네트워크 노드를 공유하고, 서로 다른 작업 간에는 분리된 노드를 사용하게 함으로써 과거의 학습 내용이 현재의 학습을 방해하지 않도록 선택적으로 자원을 할당한다.

## 📎 Related Works

지속 학습의 치명적 망각을 해결하기 위해 제안된 기존 방식들은 크게 세 가지로 분류된다. 첫째는 구조적 강건성(structural robustness)을 도입하는 방법, 둘째는 파라미터 변화를 제약하는 정규화(regularization) 방법(예: EWC), 셋째는 과거 데이터를 저장하여 재학습하는 메모리 버퍼(memory buffers) 방법(예: GEM)이다.

특히 메모리 기반 방법들은 가장 성공적인 성능을 보이지만, 샘플을 저장할 때 명시적인 작업 레이블이 필요하며 데이터 양이 증가함에 따라 메모리 요구량이 급격히 증가하는 한계가 있다. 본 논문에서 제안하는 SOMLP는 이러한 명시적 레이블이나 샘플 메모리 버퍼 없이도 유사한 수준의 성능을 달성함으로써 기존 접근 방식과 차별화된다.

## 🛠️ Methodology

본 논문에서 제안하는 **Self-Organized Multi-Layer Perceptron (SOMLP)**는 표준의 완전 연결 계층(Fully-Connected layer)과 SOM 계층을 병렬로 배치한 구조를 가진다.

### 1. 시스템 구조 및 동작 원리

입력 데이터 $x$는 MLP의 은닉층과 SOM 계층에 동시에 입력된다. SOM은 비지도 학습을 통해 입력 데이터의 분포를 학습하며, 각 입력에 대해 출력 마스크 $\Gamma$를 생성한다. 이 마스크는 MLP 은닉층의 출력값에 곱해져, 특정 작업에 불필요한 노드를 마스킹하고 필요한 노드만을 활성화하는 라우팅 역할을 수행한다.

### 2. SOM 업데이트 절차 (Algorithm 2.1)

SOM은 경쟁 학습(competitive learning)을 통해 고차원 입력을 저차원 표현으로 매핑한다.

1. **Best Matching Unit (BMU) 탐색**: 배치 데이터의 평균 $\bar{x}$와 가장 유사한(유클리드 거리가 가장 가까운) SOM 노드 $i^*$를 찾는다.
   $$i^* = \text{argmin}_i (\| \bar{x} - \theta^i \|)$$
2. **이웃 마스크(Neighborhood Mask) 계산**: BMU와 다른 노드들 간의 거리 $D_i$를 기반으로 영향력 $\phi$를 계산한다.
   $$\phi = e^{D/\sigma^2}$$
3. **출력 마스크(Output Mask) 생성**: 입력 $x$와 SOM 가중치 $\theta$ 사이의 거리를 이용하여 최종 마스크 $\Gamma$를 생성하며, 이는 MLP 출력에 적용된다.
   $$\Gamma = e^{-\|x-\theta\|/\epsilon}$$
4. **가중치 업데이트**: 학습률 $\alpha$와 이웃 마스크 $\phi$를 사용하여 BMU와 그 주변 노드의 가중치를 업데이트한다.
   $$\theta^i \leftarrow \theta^i + \alpha \phi (x - \theta^i)$$
   이때 $\alpha$와 $\sigma$는 학습이 진행됨에 따라 지수적으로 감소하여 학습을 안정화한다.

### 3. 학습 절차 (Algorithm 2.2)

학습은 두 단계로 진행될 수 있다.

- **사전 학습(Pretraining)**: 가용한 경우, 레이블이 없는 데이터를 사용하여 SOM 가중치를 순차적으로 사전 학습시킨다.
- **본 학습(Sequential Training)**: 레이블이 있는 데이터를 사용하여 SOM 가중치(비지도 학습)와 MLP 가중치(SGD를 이용한 지도 학습)를 동시에 업데이트한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: MNIST-permutations(20개 작업), MNIST-rotations(20개 작업)
- **비교 대상**: Naive MLP, Elastic Weight Consolidation (EWC), Gradient Episodic Memory (GEM)
- **평가 지표**: 평균 정확도 (%)
- **제약 사항**: 파라미터 수를 동일하게 맞추기 위해 MLP/EWC/GEM은 은닉 유닛 3200개를, SOMLP는 1600개를 사용하였다(SOM 가중치 포함 시 총 파라미터 수 유사).

### 2. 정량적 결과

- **MNIST-permutations**: $\text{SOMLP}_m$(사전 학습 모델)은 EWC와 대등한 성능을 보였으며, 대용량 메모리를 사용한 GEM보다는 낮았다.
- **MNIST-rotations**: $\text{SOMLP}_m$은 EWC보다 우수한 성능을 보였으며, GEM보다는 낮았으나 준수한 성능을 기록했다.
- **메모리 효율성**: 메모리 버퍼 크기를 줄였을 때, $\text{SOMLP}_m$은 모든 베이스라인 방법보다 더 높은 성능을 기록하였다. 이는 SOMLP가 샘플 메모리 없이도 효율적으로 작동함을 시사한다.

### 3. 정성적 분석

- **작업별 마스크 특성**: MNIST-permutations에서는 각 작업의 픽셀 순서가 무작위이므로 작업 간 마스크가 서로 독립적으로 형성되었다. 반면, MNIST-rotations에서는 회전 각도가 유사한 작업끼리는 SOM 노드를 공유하고, 각도가 크게 다른 작업들은 서로 다른 노드를 사용하는 경향이 확인되었다.

## 🧠 Insights & Discussion

본 연구의 강점은 명시적인 작업 레이블이나 과거 샘플의 저장 없이도 SOM의 적응적 클러스터링을 통해 치명적 망각을 억제했다는 점이다. 특히 메모리 자원이 제한적인 상황에서 기존의 리플레이 기반 방법론보다 강건한 성능을 보인다는 점이 인상적이다.

다만, GEM과 같은 고용량 메모리 기반 방식에 비해 절대적인 성능이 낮게 나타나는 한계가 있다. 저자들은 이를 (1) SOM 구조로 인한 실질적인 네트워크 용량의 감소, (2) 작업별 마스크 적용으로 인해 각 작업이 사용할 수 있는 가용 자원이 제한되기 때문으로 분석하고 있다.

비판적으로 해석하자면, 본 연구에서 제안한 $\text{SOMLP}_m$의 성능 향상이 상당 부분 사전 학습(pretraining) 단계에서 입력 데이터의 분포를 미리 파악한 것에 기인했을 가능성이 크다. 완전한 온라인 환경에서 사전 학습 없이 $\text{SOMLP}_s$가 보인 성능 저하는 SOM이 실시간으로 데이터 분포를 충분히 캡처하는 데 시간이 걸리거나 초기화에 민감할 수 있음을 시사한다.

## 📌 TL;DR

본 논문은 비지도 학습 기반의 **Self-Organizing Map(SOM)**을 MLP에 결합하여, 메모리 리플레이 버퍼 없이도 작업을 적응적으로 분리하고 라우팅하는 **SOMLP** 구조를 제안한다. 이 방법은 메모리 사용량을 획기적으로 줄이면서도 기존의 정규화 및 메모리 기반 지속 학습 방법론들과 경쟁 가능한 성능을 보이며, 특히 저메모리 환경에서 매우 효율적이다. 향후 자원 제한적인 임베디드 시스템에서의 지속 학습 구현에 중요한 영감을 줄 수 있는 연구이다.
