# Rethinking Overlooked Aspects in Vision-Language Models

Yuan Liu, Le Tian, Xiao Zhou, Jie Zhou (2024)

## 🧩 Problem to Solve

본 논문은 최근 급격히 발전하고 있는 대규모 시각-언어 모델(Large Vision-Language Models, LVLMs)의 성능 향상 동력이 주로 데이터의 양적 팽창(Scaling)에 의존하고 있다는 점에 주목한다. 특히, 많은 연구가 사전 학습(Pre-training) 데이터의 규모를 웹 스케일(Web-scale)로 확장하고, 지도 미세 조정(Supervised Fine-Tuning, SFT) 데이터셋의 다양성과 양을 늘리는 방향으로 진행되고 있다.

연구진은 여기서 두 가지 핵심적인 의문을 제기한다. 첫째, LLM(Large Language Models)에서 관찰되는 'Scaling Law'가 LVLM의 사전 학습 단계에서도 동일하게 적용되어, 데이터 양이 증가함에 따라 성능이 지속적으로 향상되는가 하는 점이다. 둘째, 현재 사용되는 방대한 SFT 데이터셋들이 실제로 모두 모델의 일반적 능력을 향상시키는 데 기여하는지, 아니면 불필요한 중복과 저품질 데이터가 포함되어 있는지를 분석하고자 한다.

결과적으로 본 논문의 목표는 LVLM 개발 과정에서 간과되었던 데이터 효율성을 정밀하게 분석하여, 단순한 양적 증가가 아닌 질적 선택의 중요성을 입증하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 LVLM의 데이터 효율성에 대한 비판적 분석과 최적의 SFT 데이터셋 선택 전략을 제시한 것에 있다.

1. **사전 학습 데이터의 효율성 분석**: 단순하게 사전 학습 데이터의 양을 늘리는 것이 LVLM의 성능 향상으로 이어지지 않으며, 오히려 특정 임계치를 넘어서면 성능이 저하될 수 있음을 실험적으로 증명하였다.
2. **모델 크기와 데이터의 상관관계 규명**: 모델의 성능 향상은 사전 학습 데이터의 양적 증가보다는 LLM 백본(Backbone)의 크기를 키우는 것에 더 큰 영향을 받는다는 점을 밝혀냈다.
3. **Individual Select 전략 제안**: 수많은 공개 SFT 데이터셋 중 모델 성능 향상에 실질적으로 기여하는 데이터셋만을 정밀하게 선별하는 파이프라인을 구축하였다. 이를 통해 데이터의 양은 줄이면서도 성능은 높이는 최적의 SFT 조합을 찾아냈다.

## 📎 Related Works

본 논문은 LVLM의 구조적 기반이 되는 LLaVA-1.5를 중심으로 관련 연구를 살펴본다. 일반적인 LVLM은 시각 인코더(Vision Encoder), 어댑터(Adapter), 그리고 LLM의 세 가지 구성 요소로 이루어진다.

기존 연구들은 시각-언어 정렬(Vision-Language Alignment)을 위해 시각 인코더와 LLM을 동결시키고 가벼운 어댑터만을 학습시키거나, 일부는 시각 인코더까지 함께 학습시키는 방식을 취해왔다. 또한, LLaVA와 같은 선구적인 연구는 GPT-4를 이용해 합성된 명령어 데이터셋을 사용하여 모델이 인간의 지시를 따르게 하는 Instruction Tuning 기법을 도입하였다.

최근의 InternVL-1.5나 DeepSeek-VL과 같은 모델들은 수십억 개의 웹 스케일 데이터를 사용하여 성능을 높였으나, 본 논문은 이러한 접근 방식이 데이터의 질적 분석 없이 단순히 양에만 의존하고 있다는 점을 지적하며 기존 연구와의 차별성을 둔다.

## 🛠️ Methodology

### 전체 시스템 구조 및 베이스라인

본 연구는 가장 단순하고 효율적인 구조인 LLaVA-1.5를 베이스라인으로 채택하였다. LLaVA-1.5는 CLIP-Large-336px 시각 인코더, MLP 어댑터, 그리고 Vicuna/Qwen/Yi와 같은 LLM으로 구성된다.

### 개선된 사전 학습 설정 (Improved Pre-training)

기존 LLaVA는 효율성을 위해 시각 인코더를 동결하였으나, 본 논문은 시각 인코더와 어댑터를 동시에 학습시키는 방식을 사용한다. 이때 시각 인코더와 MLP 어댑터에 서로 다른 학습률(Learning Rate)을 적용하여 특징 분포를 생성 작업에 맞게 최적화하였다.

- 시각 인코더 학습률: $\text{lr}_v$
- MLP 어댑터 학습률: $\text{lr}_a$

### 사전 학습 데이터 스케일링 실험

사전 학습 데이터의 효율성을 측정하기 위해 LAION-5B-en 데이터셋에서 $1\text{M}$부터 $100\text{M}$까지 7가지 규모의 서브셋을 추출하여 학습을 진행하였다. 이를 Vicuna-7B/13B, Qwen1.5-7B/14B, Yi-Chat-6B/34B 등 다양한 크기의 LLM에 적용하여 Scaling Law의 존재 여부를 확인하였다.

### SFT 데이터 선택 전략: Individual Select

연구진은 방대한 SFT 데이터셋에서 최적의 조합을 찾기 위해 다음과 같은 **Individual Select** 프로세스를 제안한다.

1. **카테고리 정의 및 필터링**: SFT 데이터를 12가지 카테고리(Captioning, General QA, Science, Chart, Mathematics, Knowledge, OCR, Document, Grounding, Conversation, Text-only, Screen)로 분류하고 저품질 데이터를 제거한다.
2. **점진적 선택 과정**:
    - 기본 데이터셋(Baseline, 예: LLaVA-1.5-665K)에서 시작한다.
    - 특정 카테고리 내의 후보 데이터셋을 하나씩 추가하여 모델을 미세 조정한다.
    - 새로운 모델의 성능이 기존 베이스라인과 비슷하거나 더 높을 경우, 해당 데이터셋을 '후보 풀(Candidate Pool)'에 포함시킨다.
    - 선택된 데이터셋들을 통합하여 새로운 베이스라인을 구축하고, 다음 카테고리로 넘어가 과정을 반복한다.
3. **최종 데이터셋 구성**: 이 과정을 통해 총 37개의 데이터셋 중 실질적으로 도움이 되는 17개의 데이터셋만을 최종적으로 선정하여 통합한다.

## 📊 Results

### 사전 학습 데이터 스케일링 결과

실험 결과, 사전 학습 데이터의 양을 $1\text{M}$에서 $100\text{M}$으로 늘려도 성능 향상은 미미했으며, 특히 $50\text{M}$을 초과하는 시점부터는 오히려 성능이 저하되는 경향이 관찰되었다. 예를 들어, Qwen1.5-7B 모델은 SEED-Bench에서 데이터 양이 $20\text{M}$에서 $100\text{M}$으로 늘어날 때 성능이 $3.3$ 포인트 하락하였다. 반면, 동일한 데이터 양에서 LLM의 크기를 키웠을 때(예: Vicuna-7B $\rightarrow$ 13B)는 뚜렷한 성능 향상이 나타났다. 이는 LVLM에서 단순한 데이터 스케일링보다 모델 파라미터 확장이 더 효과적임을 시사한다.

### SFT 데이터 선택 결과

Individual Select 전략을 통해 선정된 데이터셋으로 학습한 모델은 기존 LLaVA-1.5-665K 기반 모델보다 모든 벤치마크에서 우수한 성적을 거두었다.

- **정량적 결과**: Vicuna-7B 기반 모델의 경우, MMBench(en)에서 $64.3 \rightarrow 72.6$, MathVista에서 $25.6 \rightarrow 42.6$으로 비약적인 상승을 보였다.
- **비교 분석**: 동일한 LLM을 사용한 다른 SOTA 모델들과 비교했을 때도 본 연구의 모델이 더 높은 성능을 기록하였으며, 고해상도 이미지를 사용하는 LLaVA-Next보다도 높은 성능을 보여 SFT 데이터 구성의 중요성을 입증하였다.
- **데이터 효율성**: 모든 SFT 데이터셋을 무분별하게 사용했을 때보다, 선별된 데이터셋만을 사용했을 때 성능이 더 높았으며 계산 비용 또한 약 6배 감소하였다.

## 🧠 Insights & Discussion

### 사전 학습 스케일링의 한계 원인

본 논문은 사전 학습 데이터 양의 증가가 성능 향상으로 이어지지 않는 이유를 두 가지로 분석한다.
첫째, LAION-5B와 같은 웹 크롤링 데이터셋에 포함된 텍스트의 문법적 오류나 노이즈가 모델 학습에 부정적인 영향을 줄 수 있다.
둘째, 이미 시각 인코더가 사전 학습 단계에서 학습한 데이터 분포와 LVLM의 정렬 단계에서 사용하는 데이터 분포가 매우 유사하기 때문에, 중복된 정보를 더 많이 학습시키는 것이 추가적인 지식 주입으로 이어지지 않는다는 것이다.

### SFT 데이터 구성의 교훈

SFT 단계에서는 단순히 데이터의 양을 늘리는 것보다, 모델이 수행해야 할 작업(Task)과 유사한 분포를 가진 고품질 데이터를 정밀하게 선택하는 것이 훨씬 중요하다. 특히 특정 도메인(예: Science)의 성능을 높이기 위해 해당 도메인의 데이터를 추가하는 것이 효과적이지만, 이때 일반적인 능력이 저하되지 않도록 균형을 맞추는 것이 필수적이다.

### 한계 및 향후 연구

연구진은 본 연구가 데이터셋 단위의 개별 선택이라는 다소 단순한(rudimentary) 방식을 사용했다는 점과, 세 가지 벤치마크를 중심으로 ablation study를 진행하여 일부 과적합(overfitting)의 가능성이 있다는 점을 인정한다. 향후에는 데이터의 세부적인 품질과 분포 분석, OCR과 같은 특수 데이터의 주입, 그리고 고해상도 이미지 처리 기법의 통합을 연구할 계획이다.

## 📌 TL;DR

본 논문은 LVLM의 성능 향상이 단순히 데이터 양의 증가(Scaling)에 비례한다는 통념에 의문을 제기한다. 실험을 통해 **단순한 사전 학습 데이터의 양적 확대는 효율적이지 않으며 오히려 성능 저하를 초래할 수 있음**을 밝혀냈으며, 대신 **Individual Select**라는 전략을 통해 불필요한 중복을 제거하고 최적의 SFT 데이터셋 조합을 구성함으로써 모델 성능을 획기적으로 향상시킬 수 있음을 증명하였다. 이 연구는 향후 LVLM 연구가 '데이터의 양'보다는 '데이터의 질과 정교한 구성'에 집중해야 함을 시사한다.
