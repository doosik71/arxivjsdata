# Using Vision Language Models to Detect Students’ Academic Emotion through Facial Expressions

Deliang Wang, Chao Yang, Gaowei Chen (2025)

## 🧩 Problem to Solve

학생들이 학습 과정에서 느끼는 학업적 감정(Academic emotions)은 사회적 행동과 학습 성과에 지대한 영향을 미친다. 긍정적인 감정은 학업 성취도와 정적 상관관계를 가지며, 반대로 심각한 부정적 감정은 자해나 자살과 같은 극단적인 결과로 이어질 수 있어 이를 정확하게 감지하고 개입하는 것이 매우 중요하다.

기존의 자동화된 감정 분석 방식은 주로 지도 학습(Supervised Machine Learning) 알고리즘에 의존해 왔다. 그러나 이러한 모델들은 학습 데이터에 과적합되는 경향이 있어 서로 다른 맥락(Context)에서의 일반화 성능이 떨어지며, 새로운 환경에 적용할 때마다 데이터 수집, 어노테이션(Annotation), 재학습이라는 반복적이고 비용이 많이 드는 과정을 거쳐야 한다. 또한, 특정 인종(예: 아시아인)의 데이터로 학습된 모델은 안면 특징의 차이로 인해 타 인종에게 적용 시 성능이 저하되는 한계가 있다. 따라서 본 논문은 파인튜닝(Fine-tuning) 없이 제로샷 프롬프팅(Zero-shot Prompting)만으로 시각적 인식 작업의 일반화가 가능한 Vision-Language Models(VLMs)를 활용하여 온라인 학습 환경에서 학생들의 학업적 감정을 탐지할 수 있는지 그 가능성을 조사하는 것을 목표로 한다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 대규모 이미지-텍스트 쌍으로 사전 학습된 VLMs의 제로샷 능력을 활용하여, 추가적인 학습 데이터 구축 없이도 학생의 안면 표정을 통해 학업적 감정을 분류하는 것이다. 특히, 모델이 단순히 이미지만 보는 것이 아니라 각 감정 상태에 대한 정의가 포함된 텍스트 컨텍스트(Context)를 함께 제공받음으로써, 복잡한 학업적 감정 상태를 더 정확하게 추론하도록 설계하였다. 이를 통해 기존 지도 학습 모델이 가진 데이터 의존성과 일반화 부족 문제를 해결하고자 하였다.

## 📎 Related Works

학업적 감정을 측정하기 위한 전통적인 방법으로는 설문조사(Questionnaires)가 사용되었으나, 이는 학생이 타인의 시선을 의식해 감정을 숨길 수 있고 학습 중 발생하는 동적인 감정 변화를 포착하지 못한다는 한계가 있다. 이를 극복하기 위해 생체 신호(Physiological signals)를 측정하는 다중 모달(Multimodal) 접근 방식이 제안되었으나, 센서의 높은 비용과 착용의 불편함이 실질적인 보급을 가로막았다.

이후 음성, 텍스트, 이미지, 비디오를 활용한 분석이 등장하였으며, 초기에는 SVM, 결정 트리, 랜덤 포레스트와 같은 전통적인 머신러닝이 사용되었다. 이후 CNN, LSTM, Transformer 기반의 딥러닝 모델들이 도입되어 인식 성능이 향상되었으나, 여전히 모델이 학습한 특정 도메인이나 인종적 특성에 종속되는 일반화 문제가 지속적으로 제기되었다. 본 연구는 이러한 한계를 극복하기 위해 최신 VLM을 도입하여 제로샷 기반의 일반화된 감정 인식 가능성을 탐색한다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

### 데이터셋

본 연구에서는 온라인 학습 환경에서의 자발적 안면 표정 데이터베이스인 OLSFED(Online Learning Spontaneous Facial Expression Database)를 사용하였다. 해당 데이터셋은 82명의 아시아 학생이 교육 영상을 시청하는 동안의 표정을 캡처한 것으로, 총 31,115장의 이미지를 포함하고 있다. 분류 대상이 되는 다섯 가지 감정 카테고리는 다음과 같다:

- Confused (혼란스러움)
- Distracted (주의 산만)
- Happy (행복함)
- Tired (피곤함)
- Neutral (중립)

본 예비 연구에서는 각 카테고리당 1,000장씩 무작위로 추출하여 총 5,000장의 이미지를 분석에 사용하였다.

### 사용 모델

오픈 소스 기반의 최신 VLM 두 가지를 선정하여 성능을 비교하였다:

1. Llama-3.2-11B-Vision-Instruct: Llama 3.1 언어 모델을 기반으로 시각적 인식 및 추론에 최적화된 모델이다.
2. Qwen2.5-VL-7B-Instruct: Qwen2-VL을 기반으로 객체 인식, 이미지 콘텐츠 분석 및 비디오 이해에 최적화된 모델이다.

### 프롬프트 설계 (Prompting Strategy)

VLMs가 정확한 분류 결과를 내놓을 수 있도록 다음과 같은 세 가지 구성 요소로 이루어진 프롬프트를 설계하였다:

1. **Instruction (지시어)**: 모델이 수행해야 할 구체적인 작업(이미지 속 학생의 표정을 5가지 카테고리로 분류)을 명시한다. 특히, 결과 추출의 효율성을 위해 "추가 단어 없이 카테고리만 답변하라"는 제약 조건을 부여하였다.
2. **Context (문맥)**: 각 감정 카테고리에 대한 정의를 제공하여 모델의 판단 기준을 명확히 한다. 예를 들어, 'Confused'는 미간을 찌푸리거나 당혹스러운 기색을 보이는 상태로, 'Distracted'는 화면을 보지 않고 다른 곳을 보거나 다른 활동을 하는 상태로 정의하였다.
3. **Input (입력)**: 분석 대상이 되는 학생의 이미지이다.

전체적인 추론 절차는 $\text{Image} + \text{Prompt} \rightarrow \text{VLM} \rightarrow \text{Category Label}$의 형태로 진행되며, 별도의 가중치 업데이트 없이 제로샷 방식으로 수행되었다.

## 📊 Results

### 평가 지표 및 전반적 성능

모델의 성능 평가는 $F1 \text{ Score}$, $\text{Accuracy}$, $\text{Precision}$, $\text{Recall}$을 통해 측정되었다. 5가지 클래스 분류 작업이므로 $F1 \text{ Score}$와 $\text{Accuracy}$를 주요 지표로 삼았다.

- **Qwen2.5-VL-7B-Instruct**: $F1 \text{ Score} = 0.5197$, $\text{Accuracy} = 0.5546$
- **Llama-3.2-11B-Vision-Instruct**: $F1 \text{ Score} = 0.4519$, $\text{Accuracy} = 0.5118$

전반적으로 Qwen2.5-VL 모델이 Llama-3.2 모델보다 우수한 성능을 보였으며, 두 모델 모두 학업적 감정 인식에서 보통 수준의 성능을 나타냈다.

### 카테고리별 상세 결과

- **Happy**: 두 모델 모두 가장 높은 인식 성능을 보였다. (Llama $F1=0.92$, Qwen $F1=0.83$)
- **Distracted**: 두 모델 모두 탐지하는 데 실패하였다. 특히 $\text{Recall}$ 수치가 극히 낮아(0.01~0.02), 모델이 이 상태를 거의 예측하지 못함을 의미한다. 다만 $\text{Precision}$은 매우 높게 나타나, 드물게 예측한 경우는 매우 정확했음을 시사한다.
- **Confused**: 두 모델 간 성능 차이가 가장 뚜렷했다. Llama는 인식 능력이 현저히 낮았으나($F1=0.27$), Qwen은 상대적으로 높은 성능($F1=0.65$)을 보여 실용적인 활용 가능성을 입증하였다.
- **Neutral & Tired**: 두 모델 모두 중간 정도의 성능을 기록하였다.

## 🧠 Insights & Discussion

본 연구는 VLM이 별도의 학습 없이도 제로샷 프롬프팅을 통해 학업적 감정을 어느 정도 수준에서 인식할 수 있음을 보여주었다. 특히 Qwen2.5-VL-7B-Instruct 모델이 'Confused' 상태를 비교적 잘 포착한다는 점은 매우 고무적이다. 이는 교육 현장에서 학생이 어느 부분에서 어려움을 느끼는지 실시간으로 파악하여 적절한 학습 콘텐츠를 제공하는 시스템에 응용될 가능성이 높음을 시사한다.

그러나 몇 가지 한계점이 존재한다. 첫째, OLSFED 데이터셋의 일부(subset)만을 사용한 예비 연구이므로, 더 광범위하고 다양한 데이터셋에서의 검증이 필요하다. 둘째, 안면 표정이라는 단일 모달리티(Unimodal)에만 의존하였다. 실제 학업 감정은 제스처, 시선, 음성 등 다양한 신호가 복합적으로 작용하므로 다중 모달 분석으로 확장해야 한다. 셋째, 오픈 소스 모델 두 가지만을 비교하였기에, GPT-4o나 Gemini 1.5 Pro와 같은 폐쇄형(Proprietary) 고성능 모델과의 비교 분석이 이루어지지 않았다.

결론적으로, VLM은 기존 지도 학습 모델의 고질적인 문제인 데이터 의존성과 일반화 문제를 해결할 수 있는 유망한 대안이며, 프롬프트 엔지니어링을 통해 특정 도메인의 감정 인식 능력을 최적화할 수 있음을 확인하였다.

## 📌 TL;DR

본 논문은 지도 학습 기반 감정 인식 모델의 일반화 한계를 극복하기 위해, Llama-3.2와 Qwen2.5-VL이라는 두 가지 Vision-Language Model을 활용하여 제로샷 기반의 학업적 감정 탐지 가능성을 실험하였다. 실험 결과, Qwen2.5-VL 모델이 전반적으로 우수했으며 특히 '혼란스러움(Confused)' 상태를 잘 감지하는 능력을 보였다. 이 연구는 향후 추가 학습 없이도 학생의 학습 상태를 실시간으로 모니터링하고 맞춤형 교육 지원을 제공하는 지능형 학습 시스템 구축에 중요한 기초 자료가 될 것으로 기대된다.
