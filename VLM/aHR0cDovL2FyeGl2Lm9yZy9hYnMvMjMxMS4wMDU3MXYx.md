# LLaVA-Interactive: An All-in-One Demo for Image Chat, Segmentation, Generation and Editing

Wei-Ge Chen, Irina Spiridonova, Jianwei Yang, Jianfeng Gao, Chunyuan Li (2023)

## 🧩 Problem to Solve

본 연구가 해결하고자 하는 핵심 문제는 기존의 거대 다중모달 모델(Large Multimodal Models, LMMs)이 가진 상호작용의 한계이다. 대표적으로 GPT-4V와 같은 모델들은 뛰어난 성능을 보여주지만, 다음과 같은 제약 사항이 존재한다. 첫째, 주로 텍스트 기반의 상호작용에 치우쳐 있어 사용자의 입력 이미지는 맥락 제공용으로만 쓰이며, 시스템의 출력은 텍스트로 제한되는 경우가 많다. 둘째, 모델의 훈련 세부 사항이나 시스템 아키텍처가 공개되지 않아 오픈소스 커뮤니티에서 이를 확장하거나 개선하는 데 어려움이 있다.

따라서 본 논문은 텍스트 프롬프트뿐만 아니라 시각적 프롬프트(Visual Prompt)를 통해 사용자의 의도를 정밀하게 반영할 수 있고, 이미지 채팅, 세그멘테이션(Segmentation), 생성 및 편집을 모두 수행할 수 있는 통합적인 다중모달 인간-AI 상호작용 프로토타입을 구축하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 중심적인 설계 아이디어는 추가적인 모델 훈련 없이, 이미 검증된 세 가지 서로 다른 다중모달 AI 모델을 유기적으로 결합하여 시스템 레벨의 시너지를 창출하는 것이다.

가장 큰 기여는 'Visual Prompting'의 도입이다. 사용자는 단순한 텍스트 입력을 넘어, 이미지 위에 직접 선을 긋는 스트로크(Stroke), 드래그 앤 드롭, 바운딩 박스(Bounding Box) 등을 사용하여 자신의 의도를 정확하게 전달할 수 있다. 이를 통해 LMM 단독 모델보다 사용자의 의도를 더 잘 따를 수 있으며, 보다 몰입감 있는 상호작용 경험을 제공한다. 또한, 이 전체 시스템을 오픈소스로 공개함으로써 향후 다중모달 AI 에이전트 연구를 위한 확장 가능한 프레임워크를 제시하였다.

## 📎 Related Works

기존의 LMM 연구는 크게 두 가지 방향으로 진행되었다. 하나는 GILL, Emu, Kosmos-G와 같이 이미지 생성 및 편집 기능을 갖추도록 직접 모델을 훈련시키는 방향이고, 다른 하나는 Visual ChatGPT, ViperGPT와 같이 LLM이 추론 단계에서 전문가 비전 모델(Expert Vision Models)을 도구로서 호출하게 하는 프롬프트 엔지니어링 방식이다.

LLaVA-Interactive는 다음과 같은 점에서 기존 연구와 차별화된다. 첫째, 개발 비용이 매우 저렴하다. 새로운 모델을 훈련시키거나 복잡한 LLM 프롬프트 엔지니어링을 수행하지 않고, 기존 모델들의 추론 단계(Inference stages)를 결합한 시스템이다. 둘째, 시각적 상호작용을 극대화하였다. 특히 사용자가 직접 스트로크를 그려 세그멘테이션과 편집 영역을 지정하는 기능은 기존의 비주얼 어시스턴트 시스템에서 찾아보기 힘든 독특한 기능이다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조

LLaVA-Interactive는 별도의 추가 학습 없이 세 가지 사전 학습된 모델을 웹 서비스 형태로 연결한 시스템이다. 전체적인 워크플로우는 다음과 같은 순서로 진행된다.

1. **이미지 입력**: 사용자가 이미지를 업로드하거나, 텍스트 캡션과 바운딩 박스를 통해 새로운 이미지를 생성한다.
2. **Visual Chat**: LLaVA 모델을 통해 이미지에 대해 질문하고, 수정 제안을 받는다.
3. **Interactive Segmentation**: SEEM 모델을 사용하여 스트로크나 텍스트 프롬프트로 객체의 마스크(Mask)를 생성한다.
4. **Grounded Editing**: GLIGEN 모델을 사용하여 바운딩 박스와 텍스트 기반으로 객체를 추가, 삭제 또는 교체한다.
5. **반복 상호작용**: 위 단계들을 반복하며 최종 결과물을 정교하게 다듬는다.

### 주요 구성 요소 및 역할

- **LLaVA (Large Language and Vision Assistant)**: 시각적 이해와 추론을 담당한다. 사용자와의 대화를 통해 이미지의 내용을 분석하고 편집 방향을 제안하는 역할을 수행한다.
- **SEEM (Segment Everything Everywhere with Multi-modal prompts)**: 다중모달 프롬프트(점, 박스, 스크리블, 텍스트 등)를 통해 이미지 내 객체를 정밀하게 분할(Segmentation)한다.
- **GLIGEN (Grounded-Language-to-Image Generation)**: 텍스트와 시각적 프롬프트(바운딩 박스)를 조건으로 하여 이미지를 생성하거나 특정 영역을 편집한다.
- **LaMA**: GLIGEN의 인페인팅(Inpainting) 모델이 배경의 빈 공간을 완벽하게 채우지 못하는 문제를 해결하기 위해 배경 채우기 전용으로 도입되었다.

### 추론 절차 및 상호작용 로직

- **객체 제거**: 사용자가 스트로크를 그려 SEEM으로 마스크를 생성 $\rightarrow$ 마스크를 이미지 밖으로 드래그하여 제거 표시 $\rightarrow$ GLIGEN과 LaMA를 통해 해당 영역을 배경으로 채움.
- **객체 추가**: 사용자가 바운딩 박스를 그림 $\rightarrow$ 해당 영역에 들어갈 객체의 텍스트를 입력 $\rightarrow$ GLIGEN이 해당 위치에 객체를 생성.
- **객체 교체**: 세그멘테이션 마스크 생성 $\rightarrow$ 교체할 객체의 텍스트 입력 $\rightarrow$ 해당 영역을 새로운 객체로 생성.

## 📊 Results

본 논문은 정량적인 수치 비교보다는 다양한 실제 응용 시나리오를 통한 정성적 평가(Case Study)에 집중하였다.

### 실험 설정 및 지표

- **데이터셋 및 작업**: 특정 벤치마크 데이터셋 대신 사진 예술가 보조, 핼러윈 포스터 디자인, 아동복 패션 디자인, 요리 준비, 스토리텔링, 과학 교육, 인테리어 디자인, 위험 요소 식별 등 8가지 이상의 실전 시나리오를 설정하였다.
- **측정 방법**: 사용자가 시스템과 주고받는 멀티턴(Multi-turn) 대화와 그에 따른 이미지 변화 과정을 분석하여 시스템의 유용성을 평가하였다.

### 주요 결과

- **반복적 공동 창작**: 사용자가 텍스트로 풍경화를 생성한 후, LLaVA의 제안을 받아 특정 부분을 수정하고(예: 해변 제거, 거위 추가), 최종적으로 해당 이미지에 맞는 시를 작성하게 함으로써 시각-언어 간의 일관성을 확인하였다.
- **정밀한 디자인 제어**: 핼러윈 포스터 제작 시, 단순히 "무서운 분위기"라고 요청하는 것이 아니라 박스를 그려 박쥐, 유령, 해골 등을 정확한 위치에 배치함으로써 정밀한 레이아웃 제어가 가능함을 보여주었다.
- **분석 기반 합성(Analysis-by-Synthesis)**: 위험 요소 식별 시나리오에서, '움직이는 차 위에서 다림질하는 사람' 이미지에서 사람을 제거하거나 차를 꽃으로 바꿨을 때 LLaVA가 인식하는 '이상함'의 정의가 어떻게 변하는지를 실험하여, 시각적 편집이 모델의 추론에 미치는 영향을 분석하였다.

## 🧠 Insights & Discussion

### 강점

본 시스템은 모델 훈련에 드는 막대한 비용 없이도 기존의 강력한 모델들을 적절히 조합함으로써 범용적인 다중모달 에이전트를 구현할 수 있음을 입증하였다. 특히 텍스트에만 의존하던 기존 LMM의 한계를 넘어, 직접적인 시각적 조작(Visual Prompting)을 통해 인간의 의도를 훨씬 더 정확하고 효율적으로 전달할 수 있다는 점이 큰 강점이다.

### 한계 및 비판적 해석

- **개별 모델 의존성**: 시스템의 성능이 LLaVA, SEEM, GLIGEN이라는 개별 모델의 성능에 완전히 종속되어 있다. 즉, 특정 모듈의 성능이 낮으면 전체 시스템의 결과물 퀄리티가 제한된다.
- **창발적 능력의 부재**: 여러 모델을 파이프라인으로 연결한 형태이므로, 단일 거대 모델에서 나타나는 잠재 공간(Latent space) 상의 '창발적 능력'이나 '태스크 합성 능력'은 기대하기 어렵다. 단순히 기존 기능을 순차적으로 실행하는 수준에 머물러 있다.
- **정량적 평가 부족**: 다양한 시나리오를 제시하였으나, 객관적인 지표(예: 사용자 만족도 설문, 생성 이미지의 품질 점수 등)를 통한 정량적 검증이 부족하여 시스템의 우월성을 객관적으로 입증하는 데 한계가 있다.

## 📌 TL;DR

LLaVA-Interactive는 LLaVA(채팅), SEEM(분할), GLIGEN(생성/편집)이라는 세 가지 오픈소스 모델을 결합하여, 추가 훈련 없이 이미지 채팅과 정밀한 시각적 편집을 가능하게 한 다중모달 상호작용 프로토타입이다. 특히 스트로크와 바운딩 박스 같은 '시각적 프롬프트'를 통해 사용자의 의도를 정밀하게 반영할 수 있다는 점이 핵심이며, 이는 향후 더 통합된 형태의 범용 다중모달 AI 에이전트 개발에 중요한 기초 가이드라인을 제공한다.
