# Large Language Models are Visual Reasoning Coordinators: Can VLMs be used on videos for action recognition?

Harsh Lunia (2024)

## 🧩 Problem to Solve

본 논문은 개별적으로는 뛰어난 성능을 보이는 여러 Vision-Language Models (VLMs)의 능력을 어떻게 효율적으로 결합하여 시너지 효과를 낼 수 있을 것인가라는 문제를 다룬다. 특히, 단순한 예측값의 결합이나 가중치 조절과 같은 전통적인 앙상블 방식의 한계를 극복하고자 한다.

더 나아가, 저자는 이러한 LLM 기반의 조정(Coordination) 프레임워크를 비디오 기반의 행동 인식(Human Action Recognition, HAR) 작업으로 확장하여 적용할 수 있는지 탐구한다. 구체적으로는 매우 제한적인 수의 핵심 프레임(Keyframes)과 약한 시간적 정보(Temporal information)만 주어진 극한의 상황에서도 LLM이 여러 VLM의 출력을 조정하여 정확한 행동을 추론할 수 있는지를 검증하는 것을 목표로 한다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 Large Language Model (LLM)을 단순한 생성기가 아니라, 여러 VLM 전문가들의 의견을 종합하고 판단하는 '조정자(Coordinator)'로 활용하는 것이다.

1. **LLM 조정자 개념의 검증**: LLM이 여러 VLM의 출력물을 자연어 형태로 입력받아 조정함으로써, 기존의 전통적인 앙상블 방식보다 더 나은 결과를 낼 수 있음을 실험적으로 증명하였다.
2. **행동 인식(HAR)으로의 확장**: 이미지 기반의 시각적 추론 프레임워크인 Cola를 비디오 데이터셋에 적용하여, 비디오 내 행동 인식 작업에서도 LLM의 조정 능력이 유효함을 보여주었다.
3. **제한적 환경에서의 가능성 제시**: 최대 10장의 무작위 프레임과 약한 시간적 신호만 제공되는 까다로운 설정에서도 행동 인식이 어느 정도 가능하다는 점을 확인하여, 지능적 대안으로서의 잠재력을 제시하였다.

## 📎 Related Works

논문은 시각적 추론을 위해 복잡한 아키텍처에 의존하거나 학습 데이터 외의 일반화 능력이 떨어지는 전통적인 모델들의 한계를 언급한다. 이를 해결하기 위해 다음과 같은 최신 접근 방식들을 소개한다.

- **Socratic Models**: 프롬프트 엔지니어링을 통해 VLM과 LLM 간의 통신을 가능하게 하여 제로샷(Zero-shot) 멀티모달 추론을 구현한 연구이다.
- **Cola Framework (Chen et al., 2024)**: 본 논문의 기반이 되는 프레임워크로, LLM을 조정자로 사용하여 여러 VLM의 출력물을 결합하는 새로운 앙상블 패러다임을 제안하였다.
- **전통적 앙상블 방식**: 모델 가중치를 조정하거나(Weight averaging) 최종 예측값을 단순히 결합하는 방식이다. 그러나 이러한 방식은 GPT-4나 Claude와 같은 블랙박스 API 기반 모델들에 적용하기 어렵다는 한계가 있다.

## 🛠️ Methodology

### 전체 시스템 구조 (Cola Framework)

Cola 프레임워크는 여러 개의 VLM이 시각적 맥락과 가능한 답변을 생성하고, LLM이 이를 종합하여 최종 결정을 내리는 구조이다.

1. **VLM 구성**: 본 연구에서는 $\text{OFA-Large}$와 $\text{BLIP}$을 VLM으로 사용하였다.
2. **LLM 구성**: 인코더-디코더 트랜스포머 구조인 $\text{FLAN-T5-Base}$를 조정자로 사용하였다.
3. **추론 절차**:
    - 각 VLM이 독립적으로 이미지에 대한 캡션과 예상 답변을 생성한다.
    - 생성된 캡션, 답변, 질문, 선택지가 포함된 지시문(Instruction prompt)이 LLM에 입력된다.
    - LLM은 이 모든 문맥을 융합하여 최종 답변을 도출한다.

### 행동 인식(HAR) 적용 파이프라인

비디오 데이터를 처리하기 위해 다음과 같은 특수 절차를 수행한다.

**1. 핵심 프레임 선택 (Key Frame Selection)**
비디오 전체를 처리하는 것은 비효율적이므로 $\text{Katna}$ 라이브러리를 통해 최대 10장의 프레임을 추출한다. 특히 모델의 성능을 엄격하게 테스트하기 위해 프레임을 시간 순서대로 배치하지 않아 시간적 정보를 의도적으로 약화시켰다. 추출 과정은 다음과 같다.

- LUV 색 공간에서의 절대 차이를 이용해 이전 프레임과 충분히 다른 프레임을 1차 선택한다.
- 밝기(Brightness)와 엔트로피(Entropy) 점수를 기준으로 필터링한다.
- 이미지 히스토그램 기반의 K-Means 클러스터링을 수행한다.
- 각 클러스터에서 라플라시안 분산(Variance of the Laplacian)을 계산하여 블러(Blur)가 적은 최적의 프레임을 최종 선택한다.

**2. VLM 쿼리 및 LLM 학습**

- 추출된 모든 핵심 프레임에 대해 각 VLM에 "What action is happening in the frame?"이라는 질문을 던진다.
- 각 VLM의 응답 결과들을 모아 커스텀 템플릿을 구성한다.
- 이 템플릿을 사용하여 LLM이 정답 행동 이름(Target action name)을 예측하도록 학습(Fine-tuning)시킨다.

## 📊 Results

### 1. LLM 조정자 성능 검증 (A-OKVQA 데이터셋)

전통적인 앙상블 방식과 Cola-FT(Fine-tuned) 방식을 비교하여 LLM의 조정 능력을 검증하였다. 기본 앙상블은 VLM 출력과 정답 선택지 간의 코사인 유사도를 평균 내어 계산하는 방식이다.

$$P(v, q) = \frac{1}{n} \sum_{i=1}^{n} P_i(v, q)$$

실험 결과, 논문 원본의 $\text{FLAN-T5-xxl}$ (11B) 모델은 $77.7\%$의 정확도를 보였으며, 저자가 훨씬 작은 $\text{FLAN-T5-base}$ (248M) 모델로 재현했을 때도 $59.48\%$를 기록하여 기본 앙상블($56.6\%$)보다 높은 성능을 보였다. 이는 모델 크기가 매우 작아지더라도 LLM이 VLM들을 조정하는 것이 단순 결합보다 효과적임을 시사한다.

### 2. 행동 인식 성능 (SPHAR 데이터셋)

SOTA 모델(2D-CNN + SPBD LSTM)과 본 제안 방식을 비교한 결과는 다음과 같다.

| Model | Recall | Precision | F1 score |
| :--- | :---: | :---: | :---: |
| **Ours (Cola)** | 0.5941 | 0.5608 | 0.5660 |
| **SOTA (Manaf and Singh, 2021)** | 0.9090 | 0.9757 | 0.9064 |

SOTA 모델 대비 성능 차이가 크게 나타나는데, 이는 SOTA 모델이 24장 이상의 프레임을 사용하고 강력한 시간적 순서 정보를 처리하는 반면, 본 방식은 10장 이하의 무작위 프레임만 사용했기 때문이다. 그럼에도 불구하고 매우 제한적인 정보만으로 유의미한 행동 인식이 가능함을 확인하였다.

## 🧠 Insights & Discussion

### 강점 및 성과

본 모델은 '훔치기(stealing)', '기물 파손(vandalizing)', '걷기(walking)', '앉기(sitting)', '때리기(hitting)', '불 붙이기(igniting)'와 같은 행동들을 상당히 높은 정확도로 식별해냈다. 이는 LLM과 VLM의 결합된 지식 베이스가 정적인 이미지 정보만으로도 특정 행동의 패턴을 추론할 수 있음을 보여준다.

### 한계 및 분석

- **동적 행동 인식의 어려움**: '공황 상태(panicking)', '살인(murdering)', '달리기(running)'와 같이 시간적 흐름이 중요한 행동의 인식률은 낮게 나타났다.
- **핵심 프레임 추출의 문제**: 자동차 사고(car-crash) 카테고리에서 낮은 성능을 보였는데, 이는 핵심 프레임 추출기가 정작 중요한 사고 순간의 프레임을 놓쳤을 가능성이 크다.
- **의미적 중첩(Semantic Overlap)**: 하이파이브(high-five) 동작이 '때리기(hitting)'로, 짐을 들고 이동하는 모습이 '걷기(walking)'로 오인되는 등, 행동 간의 미묘한 차이를 구분하는 데 한계가 있다.

### 결론적 해석

본 연구는 LLM이 시각적 전문가(VLM)들의 의견을 조정하는 효율적인 컨트롤러가 될 수 있음을 입증하였다. 비디오 작업에서 SOTA 수준에 도달하기 위해서는 단순히 프레임 수를 늘리는 것뿐만 아니라, 강력한 시간적 신호(Temporal signal)의 통합과 핵심 프레임 추출 알고리즘의 정교화가 필수적이다.

## 📌 TL;DR

본 논문은 LLM을 여러 VLM의 출력을 종합하는 '조정자'로 활용하는 Cola 프레임워크를 제안하고 검증하였다. 이미지 기반의 VQA 작업뿐만 아니라, 매우 제한적인 프레임(최대 10장)과 약한 시간 정보만 주어진 비디오 행동 인식(HAR) 작업에 적용하여 그 가능성을 확인하였다. 비록 시간적 정보의 부족으로 인해 SOTA 모델보다는 성능이 낮으나, LLM의 조정 능력이 단순 앙상블보다 우수함을 증명하였으며, 향후 더 강력한 시간적 신호와 더 큰 모델을 통합한다면 효율적인 비디오 이해 솔루션이 될 가능성이 높다.
