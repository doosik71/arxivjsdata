# A Review of 3D Object Detection with Vision-Language Models

RANJAN SAPKOTA, KONSTANTINOS I. ROUMELIOTIS, RAHUL HARSHA CHEPPALLY, MARCO FLORES CALERO, MANOJ KARKEE (2025)

## 🧩 Problem to Solve

본 논문은 전통적인 3D 객체 검출(3D Object Detection) 방식이 가진 한계를 극복하기 위해 Vision-Language Models(VLMs)를 접목한 최신 연구 동향을 분석한다.

전통적인 3D 검출 시스템은 LiDAR 포인트 클라우드나 복셀(Voxel) 그리드와 같은 기하학적 정보에 주로 의존한다. 그러나 이러한 접근 방식은 다음과 같은 심각한 문제점을 안고 있다. 첫째, 대규모의 정밀한 3D 데이터셋에 대한 수동 어노테이션(Annotation) 비용이 매우 높다. 둘째, 학습 데이터에 포함되지 않은 새로운 객체 카테고리를 인식하지 못하는 폐쇄형 어휘(Closed-set vocabulary) 문제로 인해 일반화 능력이 떨어진다. 셋째, 센서 보정(Calibration)에 대한 의존도가 높고, "내 팔이 닿는 거리 내의 잘 익은 사과만 찾아라"와 같은 고차원적인 인간의 자연어 명령을 이해하고 수행할 수 있는 시맨틱 추론 능력이 결여되어 있다.

따라서 본 논문의 목표는 시각적 인식과 자연어 이해를 결합한 VLM이 어떻게 3D 공간 인식의 한계를 극복하고, 제로샷(Zero-shot) 일반화 및 오픈 보캐블러리(Open-vocabulary) 검출을 가능하게 하는지 체계적으로 분석하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 3D 객체 검출 분야에서 VLM의 역할을 정의하고, 기존 기하학 중심의 패러다임에서 시맨틱 중심의 패러다임으로의 전환을 분석한 점이다.

중심적인 직관은 텍스트와 이미지, 그리고 3D 기하학적 특징을 하나의 공통 임베딩 공간(Shared Embedding Space)으로 정렬(Alignment)함으로써, 모델이 명시적인 3D 학습 데이터 없이도 사전 학습된 언어-시각 지식을 활용해 3D 공간 내 객체를 추론할 수 있게 하는 것이다. 이를 통해 프롬프트 기반의 제어, 제로샷 일반화, 그리고 인간의 의도에 정렬된(Human-aligned) 쿼리 수행이 가능해진다.

## 📎 Related Works

논문은 3D 객체 검출의 발전 과정을 크게 두 단계로 나누어 설명한다.

**1. 전통적인 딥러닝 기반 방식:**
PointNet++, VoxelNet, PV-RCNN, VoteNet 등이 대표적이다. 이들은 포인트 클라우드나 복셀 데이터를 직접 처리하여 기하학적 구조를 학습한다. 하지만 이 방식들은 어노테이션 의존도가 높고, 새로운 환경이나 객체에 대해 유연하게 대응하지 못하는 경직된 재학습 구조를 가진다는 한계가 있다.

**2. VLM 기반 방식:**
CLIP, PaLM-E, RoboFlamingo-Plus와 같은 모델들이 등장하며 시각적 인식을 언어적 추론과 결합하기 시작했다. 기존 방식이 "어디에 무엇이 있는가"라는 기하학적 질문에 집중했다면, VLM 기반 방식은 "사용자가 원하는 특정한 속성을 가진 객체가 어디에 있는가"라는 시맨틱 질문에 답할 수 있게 한다. 이는 기존의 고정된 클래스 분류 방식에서 벗어나 임의의 텍스트 쿼리를 통해 객체를 찾는 오픈 보캐블러리 방식으로의 진화를 의미한다.

## 🛠️ Methodology

본 논문은 리뷰 논문으로서 100편 이상의 최신 논문을 분석하였으며, VLM 기반 3D 객체 검출의 일반적인 아키텍처와 파이프라인을 다음과 같이 설명한다.

### 1. VLM의 기본 구조

VLM은 기본적으로 세 가지 주요 구성 요소로 이루어진다.

- **Image Encoder:** Vision Transformer(ViT)나 CNN을 통해 시각적 입력을 풍부한 임베딩으로 인코딩한다.
- **Multimodal Projector:** 시각적 임베딩을 텍스트 토큰 공간으로 매핑하여 두 모달리티 간의 정렬을 수행한다.
- **Text Decoder:** 통합된 표현을 디코딩하여 자연어 답변이나 객체의 좌표(Bounding Box)를 생성하는 LLM이다.

### 2. 학습 절차

- **Pre-training:** 이미지 엔코더와 텍스트 디코더를 고정한 상태에서 Projector만을 학습시켜 시각적 특징을 텍스트 공간에 정렬시킨다. 이후 디코더까지 함께 미세 조정(Fine-tuning)하여 교차 모달 추론 능력을 높인다.
- **Fine-tuning:** 특정 3D 검출 작업(예: "모든 자동차의 바운딩 박스를 출력하라")을 위해 텍스트 지시어와 시각적 토큰을 결합하여 학습시킨다.

### 3. VLM 기반 3D 객체 검출 파이프라인

텍스트 쿼리를 기반으로 3D 공간에서 객체를 찾는 과정은 다음과 같은 4단계로 진행된다.

1. **2D Object Proposals:** VLM의 시각-언어 정렬 능력을 이용해 이미지에서 2D 바운딩 박스와 클래스 라벨을 먼저 생성한다.
2. **2D-to-3D Projection:** 깊이 맵(Depth map)이나 LiDAR-카메라 보정 행렬을 사용하여 2D 제안 영역을 3D 공간의 프러스텀(Frustum, 절두체) 형태로 투영한다.
3. **Hierarchical Feature Alignment:** 투영된 3D 영역 내의 포인트 클라우드 특징과 2D 시각 토큰을 Cross-attention 메커니즘을 통해 정렬한다.
4. **Refinement and Filtering:** 기하학적 제약 조건과 VLM의 시맨틱 피드백을 통해 3D 바운딩 박스의 위치와 크기를 정밀하게 조정하고, 신뢰도가 낮은 제안을 필터링한다.

*참고: 구체적인 손실 함수(Loss Function)의 수식은 본문에 명시되지 않았으나, 3DVLP와 같은 모델에서 OID(IoU-guided Detection) loss, OCC(Cross-Contrastive) alignment, OSC(Self-Contrastive) learning 등의 기법이 사용됨을 언급하고 있다.*

## 📊 Results

본 논문은 다양한 벤치마크와 모델의 정성적/정량적 결과를 분석하여 VLM의 효용성을 입증한다.

### 1. 주요 모델 및 데이터셋 결과

- **OMNI3D:** 234,000장의 이미지와 300만 개 이상의 인스턴스를 포함한 대규모 벤치마크이다. 여기서 Cube R-CNN 모델은 가상 깊이(Virtual Depth) 개념을 도입하여 카메라 사양 변화에 관계없이 강건한 3D 검출 성능을 보였다.
- **CoDA:** 3D Novel Object Discovery(3D-NOD) 전략과 교차 모달 정렬을 통해, SUN-RGBD 및 ScanNet 데이터셋에서 기존 방식 대비 mAP(mean Average Precision)를 최대 80% 향상시키며 오픈 보캐블러리 검출 능력을 입증했다.
- **G3-LQ:** Poincaré Semantic Alignment 손실을 사용하여 ScanRefer 데이터셋에서 복잡한 언어 쿼리(예: "오른쪽 벽에서 다섯 번째 의자")에 대해 매우 높은 정밀도의 시각적 그라운딩(Visual Grounding) 성능을 보였다.

### 2. 전통적 방식 vs VLM 방식 비교 분석

분석 결과, 두 패러다임 간의 뚜렷한 트레이드-오프(Trade-off)가 존재함이 밝혀졌다.

| 비교 항목 | 전통적 3D NN (PointNet, VoxelNet 등) | VLM 기반 3D 검출 (3D-LLM, PaLM-E 등) |
| :--- | :--- | :--- |
| **어휘 범위** | 폐쇄형(Closed-set), 학습된 클래스만 가능 | 오픈 보캐블러리(Open-vocabulary) 가능 |
| **학습 패러다임** | 지도 학습 (Dense 3D labels 필요) | 제로샷/퓨샷 학습 (사전 학습 모델 활용) |
| **시맨틱 이해** | 낮음 (기하학적 특징에 집중) | 높음 (언어적 맥락 및 속성 이해) |
| **추론 속도** | 매우 빠름 (실시간 배포 가능, 50+ FPS) | 상대적으로 느림 (계산 복잡도 높음, 8-15 FPS) |
| **일반화 능력** | 낮음 (도메인 변경 시 성능 저하) | 높음 (다양한 환경 및 객체에 대응 가능) |

## 🧠 Insights & Discussion

### 1. 강점 및 가능성

VLM 기반 3D 검출은 더 이상 수만 장의 3D 데이터를 일일이 라벨링 하지 않아도 된다는 점에서 혁신적이다. 특히 CLIP과 같은 거대 모델의 2D-텍스트 지식을 3D 공간으로 전이(Transfer)함으로써, 학습하지 않은 객체도 텍스트 설명만으로 찾아낼 수 있는 능력을 갖추게 되었다. 이는 로봇 공학, AR, 자율 주행 등 인간과 상호작용이 필요한 분야에서 필수적인 기능이다.

### 2. 한계 및 미해결 과제

그럼에도 불구하고 다음과 같은 기술적 병목 현상이 존재한다.

- **공간 추론 능력의 부족:** VLM은 여전히 상대적 위치나 객체의 방향성을 정확히 파악하는 데 어려움을 겪으며, 이는 바운딩 박스의 부정확한 배치로 이어진다.
- **교차 모달 미정렬(Misalignment):** 고차원 3D 특징을 언어 임베딩 공간으로 투영하는 과정에서 기하학적 세부 정보가 손실되는 현상이 발생한다.
- **실시간성 결여:** 트랜스포머 기반의 무거운 아키텍처로 인해 실제 로봇 시스템에 적용하기에는 추론 속도가 너무 느리다.
- **시맨틱 환각(Hallucination):** 텍스트 프롬프트가 모호할 경우, 실제로는 존재하지 않는 객체를 검출하는 환각 현상이 발생하여 안전 중심 시스템(Safety-critical) 적용에 제약이 있다.

### 3. 향후 발전 방향

논문은 이러한 문제를 해결하기 위해 **3D 씬 그래프(3D Scene Graphs)**의 통합, **합성 데이터 파이프라인(Synthetic Data Pipeline)**을 통한 라벨링 비용 절감, 그리고 **강화 학습(RL)**을 이용한 추론 최적화 등을 제안한다.

## 📌 TL;DR

본 논문은 3D 객체 검출의 패러다임이 기하학 중심의 CNN 방식에서 시맨틱 중심의 VLM 방식으로 이동하고 있음을 체계적으로 분석한 리뷰 보고서이다. VLM은 **오픈 보캐블러리 검출과 제로샷 일반화**라는 강력한 이점을 제공하여 로봇과 Embodied AI의 인지 능력을 획기적으로 높일 수 있지만, **추론 속도 저하와 공간 추론의 부정확성**이라는 과제를 안고 있다. 향후 3D 씬 그래프와 합성 데이터 활용 기술이 성숙해진다면, 인간의 복잡한 명령을 실시간으로 수행하는 지능형 3D 인식 시스템이 실현될 가능성이 매우 높다.
