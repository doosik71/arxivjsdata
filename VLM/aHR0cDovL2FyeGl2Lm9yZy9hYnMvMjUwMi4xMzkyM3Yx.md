# Qwen2.5-VL Technical Report

Qwen Team, Alibaba Group (2025)

## 🧩 Problem to Solve

현재의 대규모 시각-언어 모델(Large Vision-Language Models, LVLMs)들은 다양한 작업에서 준수한 성능을 보이지만, 특정 영역에서 압도적인 성능을 내지 못하는 '샌드위치 쿠키의 중간층'과 같은 상태에 머물러 있다. 구체적으로는 세밀한 시각적 인식(Fine-grained perception) 능력이 부족하며, 계산 복잡도가 높고, 문맥 이해 능력이 제한적이며, 입력 시퀀스 길이에 따라 성능이 일관되지 않은 병목 현상을 겪고 있다.

본 논문의 목표는 이러한 한계를 극복하여 정밀한 객체 위치 추정(Localization), 강력한 문서 파싱(Document Parsing), 그리고 장시간 비디오 이해(Long-video comprehension) 능력을 갖춘 플래그십 모델 Qwen2.5-VL을 개발하는 것이다. 이를 통해 단순한 인식을 넘어 컴퓨터나 모바일 기기를 조작할 수 있는 대화형 시각 에이전트(Interactive Visual Agent)로서의 기반을 마련하고자 한다.

## ✨ Key Contributions

Qwen2.5-VL의 핵심 아이디어는 시각적 인식의 정밀도를 극대화하면서 계산 효율성을 동시에 확보하는 것이다. 이를 위한 주요 기여 사항은 다음과 같다.

1. **계산 효율성 최적화**: Vision Transformer(ViT)에 Window Attention을 도입하여, 입력 해상도가 높아져도 계산 비용이 기하급수적(Quadratic)으로 증가하지 않고 선형적(Linear)으로 증가하도록 설계하였다.
2. **시공간적 동적 처리**: 이미지의 해상도뿐만 아니라 비디오의 FPS(Frame Per Second)를 동적으로 샘플링하는 Dynamic FPS sampling을 도입하여 다양한 길이와 속도의 비디오를 효율적으로 처리한다.
3. **절대 시간 기반 위치 임베딩**: Multimodal Rotary Position Embedding(MRoPE)의 시간 성분을 절대 시간(Absolute Time)에 정렬하여, 비디오 내 이벤트의 속도와 정확한 발생 시점을 더 정밀하게 학습하게 하였다.
4. **데이터 스케일 및 품질 확장**: 사전 학습 코퍼스를 1.2조 개에서 4.1조 개의 토큰으로 대폭 확장하였으며, 특히 문서 파싱 및 에이전트 작업을 위한 고품질 합성 데이터를 구축하였다.

## 📎 Related Works

기존의 LVLM 패러다임은 일반적으로 시각 인코더(Visual Encoder), 크로스 모달 프로젝터(Cross-modal Projector), 그리고 거대 언어 모델(LLM)의 결합으로 구성된다. 최근 연구들은 시각 인코더의 성능 개선, 해상도 스케일링, 그리고 MoE(Mixture of Experts) 구조 등을 통해 성능을 높여왔다.

그러나 기존 모델들은 좌표를 정규화(Normalization)하여 처리하는 경향이 있어 실제 물리적 스케일 정보를 손실하는 문제가 있었고, 비디오 처리 시 단순한 프레임 수에 의존하는 위치 임베딩을 사용하여 실제 시간 흐름을 제대로 반영하지 못하는 한계가 있었다. Qwen2.5-VL은 이러한 기존 방식과 달리 네이티브 해상도와 절대 시간 인코딩을 사용하여 실제 세계의 공간적, 시간적 역동성을 직접적으로 인식하도록 차별화하였다.

## 🛠️ Methodology

### 1. 전체 시스템 아키텍처

Qwen2.5-VL은 크게 세 가지 구성 요소로 이루어져 있다.

* **Vision Encoder**: 네이티브 해상도를 지원하는 재설계된 ViT이다. 2D-RoPE와 Window Attention을 사용하여 계산 효율을 높였으며, RMSNorm과 SwiGLU 활성화 함수를 채택하여 LLM과의 구조적 호환성을 맞추었다.
* **Vision-Language Merger**: ViT에서 추출된 방대한 양의 패치 특징을 효율적으로 압축하기 위해 MLP 기반의 머저를 사용한다. 공간적으로 인접한 4개의 패치 특징을 그룹화하여 연결(Concatenate)한 뒤, 2층 MLP를 통해 LLM의 임베딩 차원으로 투영한다.
* **Large Language Model (LLM)**: Qwen2.5 LLM을 기반으로 하며, 다중 모달 입력을 처리하기 위해 1D RoPE를 절대 시간에 정렬된 MRoPE로 수정하여 적용하였다.

### 2. 핵심 기술 상세 설명

#### $\text{Window Attention}$ 및 $\text{ViT}$ 효율화

입력 이미지의 크기가 다양하기 때문에 발생하는 계산 부하를 줄이기 위해, 대부분의 레이어에 Window Attention을 적용하였다. 윈도우 크기는 최대 $112 \times 112$ (패치 기준 $8 \times 8$)로 제한하며, 오직 4개의 레이어만 Full Self-Attention을 수행한다. 이를 통해 계산 복잡도를 패치 수에 비례하는 선형 복잡도로 낮추었다.

#### $\text{Native Dynamic Resolution}$ 및 $\text{Absolute Time Encoding}$

이미지와 비디오를 처리할 때 강제로 크기를 조정하지 않고 네이티브 해상도를 유지한다. 특히 bounding box나 포인트 좌표를 표현할 때 정규화된 좌표 대신 이미지의 실제 픽셀 치수를 그대로 사용하여 모델이 실제 스케일 정보를 학습하게 한다.
비디오의 경우, MRoPE의 시간 ID를 실제 타임스탬프와 정렬시킨다. 이를 통해 서로 다른 FPS로 샘플링된 비디오에서도 일관된 시간적 정렬을 학습할 수 있으며, 별도의 추가 헤드 없이도 이벤트의 정확한 시점을 로컬라이제이션 할 수 있다.

#### $\text{Document Omni-Parsing}$

문서 이해 능력을 극대화하기 위해 표, 차트, 수식, 악보 등을 포함한 다양한 요소를 HTML 태그 구조로 표준화하여 학습시켰다. 예를 들어, 문단은 `<p>`, 표는 `<table>`, 수식은 `<div class="formula">` 형태로 정의하며, 각 요소에 실제 좌표(`data-bbox`)를 함께 부여하여 레이아웃과 내용을 동시에 파악하도록 하였다.

### 3. 학습 절차 (Training Pipeline)

학습은 크게 세 단계의 사전 학습과 사후 학습(Post-training)으로 진행된다.

1. **사전 학습 (Pre-training)**:
    * **1단계**: ViT만 학습시켜 언어 모델과의 기본 정렬(Alignment)을 수행한다.
    * **2단계**: ViT와 LLM의 모든 파라미터를 해제하여 복잡한 시각적 추론과 다중 작업 학습을 수행한다.
    * **3단계**: 시퀀스 길이를 32,768까지 확장하여 장문 맥락, 장편 비디오, 복잡한 에이전트 데이터를 학습한다.
2. **사후 학습 (Post-training)**:
    * **SFT (Supervised Fine-Tuning)**: 약 200만 개의 고품질 지시어 데이터를 통해 모델을 미세 조정한다. 이때 **Rejection Sampling**을 사용하여 정답과 일치하는 고품질의 Chain-of-Thought(CoT) 추론 과정만 선별하여 학습시킨다.
    * **DPO (Direct Preference Optimization)**: 인간의 선호도 데이터를 활용하여 응답의 질을 높이고 정렬을 수행한다.

## 📊 Results

### 1. 종합 성능 비교

Qwen2.5-VL-72B 모델은 GPT-4o 및 Claude 3.5 Sonnet과 대등하거나 일부 능가하는 성능을 보였다. 특히 MMMU(70.2), MathVista(74.8) 등 대학 수준의 문제 해결 및 수학적 추론에서 매우 강력한 성능을 입증하였다.

### 2. 세부 능력 평가

* **문서 이해 및 OCR**: OCRBench_v2에서 영어와 중국어 트랙 모두 Gemini 1.5 Pro를 크게 상회하는 성능을 보였으며, 특히 복잡한 차트와 도표 파싱에서 SOTA 성능을 달성하였다.
* **공간 이해 (Grounding)**: Refcoco 및 ODinW 등에서 최상위 성능을 기록하였다. 특히 포인트 기반의 정밀한 위치 추정과 CountBench에서의 객체 개수 세기(93.6%) 능력이 탁월하다.
* **비디오 이해**: LVBench와 MLVU 등 장편 비디오 이해 벤치마크에서 GPT-4o를 유의미하게 앞섰으며, Charades-STA에서 이벤트 로컬라이제이션 성능(mIoU 50.9)이 매우 높게 나타났다.
* **에이전트 능력**: ScreenSpot Pro에서 43.6%의 정확도를 기록하며 기존 모델들을 압도하였고, OSWorld 등 실제 환경에서도 보조 도구(SoM) 없이 동작 가능한 에이전트로서의 가능성을 보여주었다.

### 3. 텍스트 전용 작업 성능

시각 능력을 강화했음에도 불구하고, Qwen2.5-VL-72B는 MMLU-Pro, MATH, HumanEval 등 텍스트 전용 벤치마크에서 기반 모델인 Qwen2.5-72B와 거의 동일하거나 더 나은 성능을 유지하여 언어 능력이 훼손되지 않았음을 증명하였다.

## 🧠 Insights & Discussion

Qwen2.5-VL의 성공 요인은 단순히 모델 크기를 키운 것이 아니라, **'데이터의 정밀한 큐레이션'**과 **'물리적 실제 값의 반영'**에 있다. 기존 모델들이 좌표를 $0 \sim 1$ 사이로 정규화하여 추상적인 공간을 학습하게 한 것과 달리, 실제 픽셀 좌표와 절대 시간을 사용함으로써 모델이 실제 세계의 물리적 스케일을 체감하게 한 점이 주효하였다.

또한, 단순한 데이터 확장이 아닌 4단계 스코어링 시스템(텍스트 품질, 이미지-텍스트 관련성, 상호 보완성, 정보 밀도 균형)을 통한 데이터 필터링과 Rejection Sampling을 이용한 CoT 데이터 구축은 모델의 추론 능력을 비약적으로 향상시켰다.

다만, 논문에서 언급되었듯이 CoT 추론 과정에서 시각적 정보와 텍스트 정보를 완벽하게 정렬하는 것은 여전히 도전적인 과제로 남아 있으며, 이는 향후 연구의 핵심 방향이 될 것으로 보인다.

## 📌 TL;DR

Qwen2.5-VL은 Window Attention을 통한 효율적인 ViT 설계, 절대 시간/좌표 기반의 네이티브 동적 해상도 처리, 그리고 4.1조 토큰의 고품질 데이터 학습을 통해 구축된 최첨단 시각-언어 모델이다. 특히 문서 파싱, 정밀한 객체 위치 추정, 장편 비디오 이해 및 GUI 에이전트 제어 능력에서 GPT-4o 수준의 성능을 구현하였으며, 이는 향후 실제 모바일 및 PC 환경에서 동작하는 지능형 시각 에이전트 구현에 핵심적인 역할을 할 것으로 기대된다.
