# Visual Instruction Tuning

Haotian Liu, Chunyuan Li, Qingyang Wu, Yong Jae Lee (2023)

## 🧩 Problem to Solve

본 논문은 대규모 언어 모델(LLM) 분야에서 입증된 **Instruction Tuning**(지시어 튜닝) 기법을 멀티모달(시각-언어) 영역으로 확장하고자 한다.

기존의 시각-언어 모델들은 주로 특정 작업(분류, 검출, 캡셔닝 등)을 독립적으로 해결하거나, 언어를 단순히 이미지 내용을 묘사하는 용도로만 사용했다. 이로 인해 모델의 인터페이스가 고정되어 사용자의 복잡하고 다양한 지시사항에 유연하게 대응하는 상호작용 능력이 부족했다.

따라서 본 연구의 목표는 일반적인 시각적 지시사항을 따를 수 있는 **범용 시각-언어 어시스턴트(General-purpose visual assistant)**를 구축하는 것이며, 이를 위해 텍스트 기반의 GPT-4를 활용하여 멀티모달 지시어 데이터를 생성하고 모델을 학습시키는 방법론을 제안한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **"텍스트 전용 LLM(GPT-4)을 사용하여 멀티모달 지시어 데이터를 생성하고, 이를 통해 시각-언어 모델을 정렬(Alignment)시킨다"**는 것이다. 주요 기여 사항은 다음과 같다.

1. **멀티모달 지시어 데이터 생성 파이프라인**: 이미지-텍스트 쌍(Caption, Bounding Box)을 텍스트 형태의 상징적 표현으로 변환하여 GPT-4에 입력함으로써, 인간의 의도가 반영된 고품질의 지시어-응답 쌍을 자동으로 생성했다.
2. **LLaVA (Large Language and Vision Assistant)**: CLIP의 시각 인코더와 Vicuna LLM을 연결하고, 생성된 데이터를 통해 엔드-투-엔드로 학습시킨 대규모 멀티모달 모델(LMM)을 개발했다.
3. **LLaVA-Bench 벤치마크**: 모델의 지시어 이행 능력을 평가하기 위해 다양하고 도전적인 작업으로 구성된 두 가지 평가 벤치마크를 구축했다.
4. **오픈소스 공개**: 생성된 데이터셋, 코드, 모델 체크포인트를 공개하여 후속 연구의 기반을 마련했다.

## 📎 Related Works

### 관련 연구 및 한계

- **멀티모달 에이전트**: 기존 연구들은 특정 도메인(예: 내비게이션, 이미지 편집)에 특화된 엔드-투-엔드 모델이거나, LangChain 등을 통해 여러 모델을 조합하는 시스템 방식이었다. 범용적인 시각-언어 지시어 수행 능력에 집중한 엔드-투-엔드 모델은 부족했다.
- **Instruction Tuning**: NLP 분야에서는 InstructGPT, ChatGPT 등이 지시어 튜닝을 통해 제로샷(Zero-shot) 일반화 능력을 크게 향상시켰으나, 이를 시각-언어 영역에 적용하려는 시도는 적었다.
- **LMMs**: Flamingo, BLIP-2, OpenFlamingo 등이 제안되었으나, 이들은 주로 이미지-텍스트 쌍으로 학습되었을 뿐, 명시적인 시각-언어 지시어 데이터로 튜닝되지 않아 멀티모달 작업 성능이 언어 전용 작업에 비해 낮았다.

### 차별점

LLaVA는 단순히 이미지를 텍스트로 매핑하는 것을 넘어, 사용자의 복잡한 지시를 이해하고 수행할 수 있도록 **명시적인 Visual Instruction Tuning**을 수행했다는 점에서 기존 LMM과 차별화된다.

## 🛠️ Methodology

### 1. GPT 보조 시각 지시어 데이터 생성

텍스트 전용 GPT-4를 사용하여 이미지 내용을 포함한 지시어 데이터를 생성한다. 이때 이미지를 직접 입력하는 대신, 다음과 같은 **상징적 표현(Symbolic Representation)**을 사용한다.

- **Captions**: 이미지의 장면을 설명하는 텍스트.
- **Bounding Boxes**: 객체의 종류와 공간적 위치 정보.

이를 바탕으로 세 가지 유형의 데이터를 생성했다.

- **Conversation**: 시각적 내용에 대한 질의응답 형태의 대화.
- **Detailed Description**: 이미지에 대한 풍부하고 종합적인 묘사.
- **Complex Reasoning**: 엄격한 논리를 따르는 단계별 추론 과정이 포함된 질문과 답변.

최종적으로 총 158K개의 고유한 언어-이미지 지시어 샘플을 수집했다.

### 2. LLaVA 아키텍처

LLaVA는 시각 인코더(Vision Encoder)와 언어 모델(LLM)을 단순한 선형 투영 층(Linear Projection Layer)으로 연결한 구조이다.

- **시각 인코더**: CLIP ViT-L/14를 사용하여 이미지 $X_v$로부터 시각적 특징 $Z_v = g(X_v)$를 추출한다.
- **연결 층**: 학습 가능한 투영 행렬 $W$를 통해 시각 특징 $Z_v$를 LLM의 워드 임베딩 공간으로 변환하여 시각 토큰 $H_v$를 생성한다.
$$H_v = W \cdot Z_v$$
- **언어 모델**: Vicuna를 사용하여 입력된 시각 토큰 $H_v$와 텍스트 지시어 $X_q$를 바탕으로 응답 $X_a$를 생성한다.

### 3. 학습 절차

학습은 다음의 2단계 과정으로 진행된다.

**Stage 1: 특징 정렬을 위한 사전 학습 (Pre-training for Feature Alignment)**

- **목적**: 시각 특징 $H_v$를 LLM의 워드 임베딩 공간에 정렬시키는 것.
- **데이터**: 필터링된 CC3M 데이터셋(595K 쌍)을 사용하며, 단순한 이미지 캡셔닝 형태로 변환한다.
- **방법**: 시각 인코더와 LLM의 가중치는 고정(Frozen)하고, **투영 행렬 $W$만 학습**시킨다.

**Stage 2: 엔드-투-엔드 미세 조정 (Fine-tuning End-to-End)**

- **목적**: 모델이 사용자의 지시사항을 따르도록 능력을 최적화하는 것.
- **데이터**: GPT-4로 생성한 158K의 지시어 데이터셋을 사용한다.
- **방법**: 시각 인코더는 고정하되, **투영 층 $W$와 LLM의 가중치 $\phi$를 함께 업데이트**한다.
- **학습 목표**: 자동회귀(Auto-regressive) 목적 함수를 사용하여 정답 토큰의 확률을 최대화한다.
$$p(X_a | X_v, X_{\text{instruct}}) = \prod_{i=1}^{L} p_{\theta}(x_i | X_v, X_{\text{instruct}, <i}, X_{a, <i})$$

## 📊 Results

### 1. 멀티모달 챗봇 성능

정성적 평가 결과, LLaVA는 BLIP-2나 OpenFlamingo와 달리 단순히 장면을 묘사하는 것이 아니라 사용자의 지시를 정확히 따라 응답했다. 특히 학습 데이터에 없었던 이미지에 대해서도 GPT-4와 유사한 추론 능력을 보여주었다.

### 2. 정량적 평가 (LLaVA-Bench)

GPT-4를 판별자(Judge)로 사용하여 모델의 응답 품질을 1~10점으로 평가했다.

- **LLaVA-Bench (COCO)**: 모든 유형의 데이터를 사용하여 튜닝했을 때 가장 높은 성능(상대 점수 85.1%)을 보였으며, 특히 복잡한 추론 데이터의 추가가 대화 능력까지 향상시킴을 확인했다.
- **LLaVA-Bench (In-the-Wild)**: 외부의 다양한 이미지셋으로 평가한 결과, LLaVA가 BLIP-2(+29%) 및 OpenFlamingo(+48%)보다 월등히 높은 성능을 기록했다.

### 3. ScienceQA 벤치마크

멀티모달 과학 질문 답변 데이터셋인 ScienceQA에서 LLaVA는 90.92%의 정확도를 기록했다. 특히, **텍스트 전용 GPT-4를 판별자로 사용하여 LLaVA의 결과와 앙상블(Ensemble)** 했을 때, 새로운 SOTA(State-of-the-art) 기록인 **92.53%**를 달성했다.

## 🧠 Insights & Discussion

### 강점 및 emergent behaviors

LLaVA는 학습 과정에서 명시적으로 학습되지 않은 능력들이 발현되는 '창발적 행동'을 보였다.

- **OCR 능력**: 텍스트 인식 능력이 뛰어나 웹사이트 목업(Mock-up)을 HTML/JS 코드로 변환하는 작업이 가능했다.
- **일반화 능력**: 학습 데이터에 없었던 인물(예: Elon Musk)을 인식하거나, 밈(Meme)의 유머 코드를 이해하여 설명하는 능력을 보여주었다.

### 한계 및 비판적 해석

- **시각적 환각(Hallucination)**: 모델이 이미지를 세밀한 의미 구조로 파악하지 못하고 '패치들의 집합(Bag of patches)'으로 인식하는 경향이 있다. 예를 들어, 냉장고에 '요거트'와 '딸기'가 각각 따로 있음에도 '딸기맛 요거트'가 있다고 답변하는 오류가 발견되었다.
- **해상도 및 지식 범위**: 매우 세밀한 브랜드 명칭을 인식하거나 전문적인 외부 지식이 필요한 질문에는 여전히 한계가 있다.

## 📌 TL;DR

본 논문은 GPT-4를 활용해 대량의 멀티모달 지시어 데이터를 생성하고, 이를 통해 CLIP-Vision Encoder와 Vicuna-LLM을 연결한 **LLaVA** 모델을 학습시켰다. LLaVA는 단순한 이미지 묘사를 넘어 복잡한 지시사항을 수행하고 추론하는 능력을 갖추었으며, 특히 ScienceQA에서 SOTA 성능을 달성했다. 이 연구는 LMM의 학습 방향을 '단순 캡셔닝'에서 '지시어 이행(Instruction Following)'으로 전환했다는 점에서 향후 범용 시각 어시스턴트 개발에 중요한 이정표가 될 것으로 보인다.
