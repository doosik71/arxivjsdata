# Are Unified Vision-Language Models Necessary: Generalization Across Understanding and Generation

Jihai Zhang, Tianle Li, Linjie Li, Zhengyuan Yang, Yu Cheng (2025)

## 🧩 Problem to Solve

최근 비전-언어 모델(Vision-Language Models, VLMs)은 시각적 이해(Understanding)와 생성(Generation)이라는 두 가지 핵심 능력을 하나의 프레임워크로 통합하려는 경향을 보이고 있다. 기존의 가설은 이해와 생성 작업을 동시에 학습하는 통합 아키텍처가 두 작업 간의 상호 보완적 향상을 이끌어낼 수 있다는 것이었으나, 이에 대한 체계적인 검증은 부족한 상태였다.

특히, 이해 전용 모델과 생성 전용 모델이 각각의 영역에서 이미 매우 뛰어난 성능을 보이고 있는 상황에서, 굳이 이를 통합한 Unified VLM이 필요한지에 대한 근본적인 의문이 제기된다. 본 논문은 Unified VLM에서 이해와 생성 작업 간의 일반화(Generalization)가 실제로 발생하는지, 그리고 어떤 요인이 이 현상에 영향을 미치는지를 정량적으로 분석하는 것을 목표로 한다.

## ✨ Key Contributions

본 연구의 핵심적인 기여는 통합 비전-언어 모델에서 이해와 생성 작업 사이의 상호 작용을 체계적으로 분석하여 통합 모델의 필요성을 입증한 것이다. 주요 직관은 "생성할 수 없는 것은 이해한 것이 아니다"라는 철학에 기반하며, 다음과 같은 핵심 발견을 제시한다.

1. **상호 이익(Mutual Benefits)의 존재**: 이해와 생성 데이터를 혼합하여 학습한 모델이 각 작업만 학습한 모델보다 두 작업 모두에서 더 나은 성능을 보이며, 이는 데이터 양이 증가함에 따라 확장(Scale-up)된다.
2. **입출력 공간 정렬(Alignment)의 중요성**: 시각적 입력 공간(Understanding)과 출력 공간(Generation)이 잘 정렬되어 있을수록 작업 간 일반화 성능이 향상된다.
3. **생성 $\rightarrow$ 이해로의 지식 전이**: 생성 작업에서 습득한 지식이 이해 작업으로 전이될 수 있으며, 이러한 전이는 모달리티 어댑터가 아닌 기반 언어 모델(LLM) 내부에서 일어난다.

## 📎 Related Works

기존의 통합 VLM 연구들은 주로 아키텍처 혁신이나 학습 전략에 집중해 왔다.

- **토큰 기반 접근법**: Chameleon이나 LWM과 같이 VQ-기반 이미지 토크나이저를 사용하여 텍스트와 이미지를 동일한 autoregressive 프레임워크 내에서 처리하는 방식이다.
- **디퓨전 결합 방식**: LLM이 잠재 코드를 생성하고 이를 사전 학습된 Diffusion 모델로 전달하거나, Transfusion 및 Show-o와 같이 연속적/이산적 디퓨전을 LLM과 통합하는 방식이다.
- **구조적 분리**: Janus-pro와 같이 이해와 생성 경로를 분리하여 두 작업 간의 충돌을 완화하려는 시도가 있었다.

그러나 이러한 연구들은 특정 아키텍처에 국한된 분석만을 제공했으며, 작업 간의 양방향 일반화 능력을 포괄적으로 평가하지 않았다. 특히 일부 연구에서는 통합 학습이 오히려 성능 저하를 일으킨다는 상충하는 결과가 보고되기도 하여, 본 논문은 통제된 실험 환경을 통해 이를 명확히 규명하고자 한다.

## 🛠️ Methodology

### 1. Synthetic Dataset: Smart Watch UI Dataset

정밀한 통제와 효율적인 실험을 위해 본 논문은 'Smart Watch UI 데이터셋'을 자체 설계하였다. 이 데이터셋은 시간, 날씨, 날씨 위치, 배터리 잔량, 배터리 위치, 시계 배경색이라는 6가지 속성으로 제어된다.

- **이해 작업**: VQA(Visual Question Answering) 및 이미지 캡셔닝 데이터를 생성한다.
- **생성 작업**: 텍스트 지시어로부터 이미지를 생성하는 Text-to-Image 데이터를 구성한다.

### 2. Unified VLM 아키텍처

본 연구에서 사용된 모델은 LLM 기반의 통합 구조를 가진다.

- **이해 파이프라인**: $\text{Image} \rightarrow \text{Image Encoder} \rightarrow \text{Understanding Vision Adapter} \rightarrow \text{LLM}$ 순으로 흐르며, 시각적 특징을 LLM의 hidden space로 투영한다.
- **생성 파이프라인**: LLM이 `<image>` 심볼을 생성하면 $\text{Image Generation Head}$가 활성화되어 LLM의 hidden state를 시각 토큰으로 변환한다. 이후 $\text{Generation Vision Adapter}$가 이 토큰을 다시 LLM의 입력 공간으로 매핑하여 autoregressive하게 이미지를 생성한다.

### 3. 실험 설정 및 구성

다양한 인코더와 토큰 조합을 평가하기 위해 네 가지 설정을 사용하였다.

- **SigLIP-VQ**: SigLIP 인코더 $\rightarrow$ VQ 토큰 생성 (Janus 스타일)
- **VQ-VQ**: VQ-VAE 인코더 $\rightarrow$ VQ 토큰 생성 (Liquid 스타일)
- **SigLIP-SigLIP**: SigLIP 인코더 $\rightarrow$ SigLIP 임베딩 생성 (MetaMorph 스타일)
- **VQ-SigLIP**: VQ-VAE 인코더 $\rightarrow$ SigLIP 임베딩 생성 (비교용 baseline)

학습 시 $\text{Vicuna-7B-v1.5}$를 기반 LLM으로 사용하였으며, LoRA(Low-Rank Adaptation)를 통해 효율적으로 파인튜닝하였다. 손실 함수로는 VQ 토큰의 경우 Cross-entropy loss를, SigLIP 임베딩의 경우 Cosine similarity loss를 사용하였다.

## 📊 Results

### 1. 통합 학습의 상호 이익 검증

혼합 데이터로 학습한 Unified VLM이 이해 전용($\text{U-only}$) 또는 생성 전용($\text{G-only}$) 모델보다 모든 지표에서 우수한 성능을 보였다. 이는 이해 작업이 생성 능력을 돕고, 반대로 생성 학습이 이해 능력을 향상시키는 상호 보완적 관계가 성립함을 의미한다.

### 2. 입출력 공간 정렬의 영향

시각적 입력 공간과 출력 공간이 동일하거나 유사한 모델(예: $\text{SigLIP-SigLIP}$, $\text{VQ-VQ}$)이 그렇지 않은 모델보다 더 높은 성능을 보였다. 이를 검증하기 위해 입력 공간에 가역적인 Affine transformation을 추가하여 의도적으로 정렬을 깨뜨렸을 때, 이해 전용 모델은 영향이 없었으나 통합 모델의 성능은 급격히 저하되었다. 이는 작업 간 일반화가 입력-출력 공간의 근접성에 의존함을 보여준다.

### 3. 데이터 스케일링 효과

- 이해 데이터를 늘리면 생성 성능(FID)이 개선된다.
- 생성 데이터를 늘리면 이해 성능(VQA Accuracy)이 개선된다.
특히 $\text{SigLIP-VQ}$ 모델의 경우, 이해 데이터를 180K 늘리는 것보다 생성 데이터를 120K 늘리는 것이 이해 성능 향상에 더 효율적이라는 결과가 관찰되었다.

### 4. 지식 전이(Knowledge Transfer) 분석

특정 속성(예: 날씨)의 이해 데이터 비중을 극단적으로 낮추고 생성 데이터만 유지했을 때, 이해 전용 모델은 해당 속성을 인식하지 못했으나 통합 모델은 거의 100%의 정확도를 보였다. t-SNE 분석 결과, 이해 전용 모델의 비전 토큰에도 이미 날씨 정보가 포함되어 있었으나 LLM이 이를 활용하지 못한 것이었다. 즉, 생성 학습이 LLM으로 하여금 비전 토큰과 텍스트 간의 관계를 더 잘 파악하게 함으로써 지식 전이가 일어난 것이다.

### 5. 실제 사례 실험 (LLaVA-1.5 확장)

LLaVA-1.5-7B에 생성 기능을 추가하여 350K의 생성 데이터를 함께 학습시킨 결과, MME, MM-Bench 등 8개의 벤치마크에서 기존 이해 전용 LLaVA-1.5보다 성능이 향상됨을 확인하였다.

## 🧠 Insights & Discussion

본 논문은 Unified VLM이 단순히 두 기능을 합쳐놓은 것이 아니라, 서로의 성능을 끌어올리는 시너지 효과가 있음을 입증하였다. 특히 주목할 점은 **지식 전이가 비전 어댑터가 아닌 기반 LLM 내부에서 발생한다**는 점이다. 이는 LLM이 입력 공간과 출력 공간의 암시적 정렬(Implicit alignment)을 수행할 능력이 있음을 시사한다.

**강점 및 한계**:

- **강점**: 통제 가능한 합성 데이터셋을 통해 변수를 엄격히 제어하여 인과 관계를 명확히 분석하였다.
- **한계**: Diffusion 기반의 생성 컴포넌트를 사용하는 최신 아키텍처까지는 포괄하지 못했으며, 더 발전된 LLM 베이스를 사용했을 때의 잠재적 성능 향상 폭은 확인하지 않았다.

비판적으로 해석하자면, 본 연구는 합성 데이터셋이라는 단순한 환경에서 강력한 효과를 보였으나, 실제 세계의 극도로 복잡한 이미지-텍스트 쌍에서도 동일한 수준의 상호 이익이 발생하는지는 추가적인 검증이 필요하다. 하지만 LLaVA-1.5 실험을 통해 실데이터에서도 긍정적인 신호가 있음을 보여주었다.

## 📌 TL;DR

본 논문은 시각적 이해와 생성 능력을 통합한 Unified VLM이 각 기능을 분리한 모델보다 우수하며, 두 작업이 서로의 성능을 향상시키는 **상호 보완적 시너지**를 가진다는 것을 입증하였다. 특히 비전 입출력 공간의 정렬이 잘 될수록 일반화 성능이 높아지며, 생성 학습을 통해 얻은 지식이 LLM 내부에서 이해 작업으로 전이된다는 점을 밝혀내어, 향후 VLM 설계 시 통합 구조의 필요성과 방향성을 제시하였다.
