# Spa-VLM: Stealthy Poisoning Attacks on RAG-based VLM

Lei Yu, Yechao Zhang, Ziqi Zhou, Yang Wu, Wei Wan, Minghui Li, Shengshan Hu, Pei Xiaobing, Jing Wang (2025)

## 🧩 Problem to Solve

본 논문은 Vision-Language Model (VLM)의 최신 지식 부족으로 인한 환각(Hallucination) 현상을 해결하기 위해 도입된 Retrieval-Augmented Generation (RAG) 기반 VLM 시스템의 보안 취약점을 다룬다.

기존의 LLM 기반 RAG 시스템에서는 외부 지식 베이스에 악의적인 텍스트를 삽입하는 데이터 포이즈닝(Data Poisoning) 공격이 가능함이 알려져 있다. 하지만 VLM 기반의 RAG는 이미지와 텍스트라는 다중 모달리티(Multi-modality)를 동시에 처리하는 복잡한 파이프라인을 가지고 있다. 연구진은 기존의 단일 모달(텍스트 전용) 포이즈닝 공격을 다중 모달 RAG 시나리오에 적용했을 때 공격 성공률이 0%에 수렴한다는 점을 발견하였다. 이는 VLM RAG 시스템이 텍스트를 검색하기 전, 이미지 기반의 시각적 검색(Visual Retrieval) 단계를 먼저 거치기 때문에 텍스트만으로는 검색 상위 결과에 진입할 수 없기 때문이다.

따라서 본 논문의 목표는 시각적 검색 단계와 텍스트 재순위화(Reranking) 단계를 모두 통과하여 LLM이 공격자가 의도한 잘못된 답변을 생성하도록 유도하는 새로운 다중 모달 포이즈닝 공격 방법론인 **Spa-VLM**을 제안하는 것이다.

## ✨ Key Contributions

Spa-VLM의 핵심 아이디어는 **시각적 검색(Visual Retrieval)과 텍스트 재순위화(Reranking)라는 두 가지 관문을 동시에 통과할 수 있도록 '악의적인 이미지-텍스트 쌍'을 정교하게 설계하여 주입**하는 것이다.

1. **시각적 관문 통과**: 타겟 쿼리 이미지와 임베딩 공간에서 매우 유사하도록 적대적 노이즈(Adversarial Noise)가 추가된 이미지를 생성하여, 시각적 검색기(Visual Retriever)가 해당 악성 항목을 상위 $k_1$개 결과에 포함시키게 만든다.
2. **텍스트 관문 및 생성 제어**: 재순위화 단계에서 필터링되지 않도록 타겟 쿼리와의 유사도를 높이는 동시에, LLM이 이를 컨텍스트로 읽었을 때 공격자가 설정한 타겟 답변(Target Answer)을 출력하도록 텍스트의 '공격성(Aggressiveness)'을 최적화한다.

## 📎 Related Works

본 논문은 다음과 같은 관련 연구들을 검토하고 차별점을 제시한다.

- **RAG-based VLM**: 외부 지식 베이스를 통해 VQA 성능을 높이는 연구들이 진행되었으나, 이에 대한 보안 분석은 거의 이루어지지 않았다.
- **Data Poisoning Attacks**: 머신러닝 모델의 학습 데이터나 RAG의 지식 베이스를 오염시키는 공격들이 제안되었다. 특히 `PoisonedRAG`와 같은 연구는 LLM의 지식 베이스를 공격하지만, 이는 텍스트 모달리티에만 국한되어 있다.
- **기존 방식과의 차별점**: 기존의 텍스트 기반 공격은 VLM-RAG의 시각적 검색 단계에서 차단된다. 반면, 이미지로만 공격할 경우 검색은 되더라도 LLM을 유도할 구체적인 텍스트 가이드가 없어 실패한다. Spa-VLM은 이미지와 텍스트를 동시에 최적화함으로써 이 두 가지 한계를 극복한 최초의 다중 모달 RAG 포이즈닝 공격이다.

## 🛠️ Methodology

### 1. 시스템 구조 및 공격 흐름

RAG 기반 VLM 시스템은 `지식 베이스 $\rightarrow$ 시각적 검색기(Visual Retriever) $\rightarrow$ 재순위화기(Reranker) $\rightarrow$ LLM` 순으로 작동한다. 공격자는 이 과정의 각 단계에서 필터링되지 않고 최종 LLM까지 도달하여 잘못된 답을 내게 하는 것을 목표로 한다.

### 2. 악성 이미지 생성 (Visual Retrieval 최적화)

시각적 검색기가 타겟 이미지 $Q_p$와 유사한 악성 이미지 $P_p$를 검색하게 하기 위해 다음과 같은 최적화를 수행한다.

- **타겟 임베딩 근사**: 공격자는 정확한 사용자 이미지를 알 수 없으므로, 동일 카테고리의 이미지들을 수집하고 $k$-means 클러스터링을 통해 중심 벡터 $V_j$를 구하여 타겟 임베딩 $E_v(Q_p)$를 근사한다.
- **적대적 섭동 추가**: PGD(Projected Gradient Descent) 방식을 사용하여 초기 이미지에 미세한 노이즈를 추가한다. 손실 함수는 다음과 같이 음의 코사인 유사도로 정의된다.
  $$L(P_{pi}) = -\cos(E_v(P_{pi}), V_j)$$
- **제약 조건**: 노이즈가 인간의 눈에 띄지 않도록 섭동 범위를 $\epsilon = 0.05$로 제한하는 Clipping 과정을 거친다.
  $$P_{t+1}^{pi} = \text{clip}(P_{t+1}^{pi}, P_0^{pi} - \epsilon, P_0^{pi} + \epsilon)$$

### 3. 악성 텍스트 생성 (Reranker 및 LLM 최적화)

텍스트 $P_t$는 재순위화기를 통과해야 하며, LLM이 타겟 답변 $R$을 생성하게끔 유도해야 한다.

- **초기화**: GPT-4o와 같은 VLM을 사용하여, 컨텍스트로 제공되었을 때 타겟 답변 $R$을 생성하게 만드는 초기 텍스트를 생성한다.
- **유사도 최적화**: 재순위화기의 fused embedding(이미지+텍스트 결합 임베딩) $E_Q$와 악성 텍스트 임베딩 $E_{P_t}$ 간의 유사도를 높인다.
  $$L_{\text{total}} = L_{\text{similarity}} + \lambda L_{\text{regularization}}$$
  여기서 $L_{\text{similarity}} = -\cos(E_Q, E_{P_t})$이며, $L_{\text{regularization}}$은 초기 텍스트에서 너무 멀어지지 않도록 제한하는 MSE 손실이다.
- **공격성(Aggressiveness) 최적화**: VLM을 통해 텍스트를 반복적으로 다시 작성(Rewrite)하며, 실제 LLM에 넣어 타겟 답변 $R$이 나오는지 확인한다. 성공할 때까지 또는 최대 횟수 $L$까지 이 과정을 반복한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: Encyclopedic-VQA (2M entries), InfoSeek (100K entries)
- **지표**: 공격 성공률 (ASR), 정밀도 (Precision)
- **시스템 구성**: Eva-CLIP (검색기), Q-Former (재순위화기), Mistral-7B / LLaMA-8B (LLM)
- **공격 규모**: 타겟 질문당 단 5개의 악성 이미지-텍스트 쌍 주입

### 2. 정량적 결과

- **ASR 및 Precision**: Spa-VLM은 두 데이터셋 모두에서 **ASR 0.8 이상**을 기록하였다. 이는 매우 낮은 포이즈닝 비율(10만~200만 개 중 단 5개)만으로도 높은 성공률을 보인 것임을 의미한다.
- **비교 분석**:
  - **Naive Attack / Corpus Poisoning / PoisonedRAG**: ASR이 거의 0에 가깝거나 매우 낮았다. 이는 시각적 검색 조건을 만족시키지 못했기 때문이다.
  - **Prompt Injection**: 어느 정도의 ASR(0.46~0.63)을 보였으나, Spa-VLM보다 현저히 낮았다.
- **효율성**: 이미지 최적화에 약 4.4초, 텍스트 최적화에 텍스트당 평균 3회 미만의 VLM 쿼리와 약 4초의 시간이 소요되어 공격 생성 비용이 매우 낮다.

### 3. 분석 (Ablation Study)

- **주입 개수 ($N$)**: $N$이 검색 상위 개수 $k_1$에 도달할 때까지 ASR이 상승하며, 그 이후에는 포화 상태가 된다.
- **검색 상위 개수 ($k_1$)**: $k_1$이 커질수록(더 많은 깨끗한 데이터가 함께 검색될수록) ASR이 서서히 감소하여 약 0.65에서 안정화되는 경향을 보였다.
- **재순위화 개수 ($k_2$)**: $k_2$ 값의 변화는 ASR에 거의 영향을 주지 않았다.

## 🧠 Insights & Discussion

### 1. 강점 및 시사점

Spa-VLM은 다중 모달 RAG 시스템이 가진 파이프라인의 특성(이미지 검색 $\rightarrow$ 텍스트 재순위화)을 정확히 공략하였다. 특히, 매우 적은 양의 데이터 주입만으로도 시스템 전체를 오염시킬 수 있다는 점에서 매우 위협적인 공격임을 입증하였다.

### 2. 방어 기제의 한계

논문에서 제안된 세 가지 방어 기제는 모두 효과가 없었다.

- **무작위 입력 전처리(Randomized Input Preprocessing)**: 이미지 크기 조정 및 패딩을 적용했으나, 공격자가 카테고리 중심 벡터를 사용하여 임베딩을 근사했기 때문에 우회되었다.
- **의역(Paraphrasing)**: 질문을 의역하여 유사도를 낮추려 했으나, 재순위화 점수를 유의미하게 낮추지 못했다.
- **중복 텍스트 필터링**: VLM의 온도(Temperature) 설정으로 인해 생성된 악성 텍스트들이 서로 다양하여 필터링되지 않았다.

### 3. 비판적 해석 및 한계

- **White-Box 설정**: 본 연구는 검색기와 재순위화기의 파라미터를 알고 있다는 가정하에 수행된 화이트박스 공격이다. 실제 환경에서는 이러한 파라미터에 접근하기 어려울 수 있으므로, 블랙박스 시나리오에서의 전이성(Transferability)에 대한 검증이 추가로 필요하다.
- **방어 전략의 단순함**: 평가된 방어 기제들이 매우 기초적인 수준이다. 더 정교한 적대적 학습(Adversarial Training)이나 확산 모델 기반의 이미지 정화(Purification) 기법에 대해서는 논의가 부족하다.

## 📌 TL;DR

본 논문은 RAG 기반 VLM 시스템을 겨냥한 최초의 다중 모달 포이즈닝 공격인 **Spa-VLM**을 제안한다. 공격자는 적대적 노이즈가 포함된 이미지와 최적화된 텍스트를 쌍으로 묶어 지식 베이스에 주입하며, 이를 통해 시각적 검색과 재순위화 단계를 모두 통과하여 LLM이 잘못된 답변을 내도록 유도한다. 실험 결과, 단 5개의 악성 샘플만으로도 80% 이상의 공격 성공률을 보였으며, 기존의 단순 방어 기제로는 이를 막을 수 없음을 확인하였다. 이는 향후 다중 모달 RAG 시스템 구축 시 지식 베이스의 무결성 검증과 강력한 보안 대책이 필수적임을 시사한다.
