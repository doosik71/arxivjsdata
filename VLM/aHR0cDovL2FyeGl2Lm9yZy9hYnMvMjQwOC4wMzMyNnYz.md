# LLaVA-OneVision: Easy Visual Task Transfer

Bo Li, Yuanhan Zhang, Dong Guo, Renrui Zhang, Feng Li, Hao Zhang, Kaichen Zhang, Peiyuan Zhang, Yanwei Li, Ziwei Liu, Chunyuan Li (2024)

## 🧩 Problem to Solve

현재 오픈 소스 대규모 멀티모달 모델(LMM)들은 일반적으로 단일 이미지, 다중 이미지, 또는 비디오라는 개별 시나리오에 특화되어 개발되는 경향이 있다. 특히 비디오 이해 능력을 높이기 위해 튜닝된 모델들은 종종 단일 이미지 처리 성능이 저하되는 Trade-off 문제가 발생한다.

본 논문은 이러한 파편화된 접근 방식을 해결하고, 단일 모델이 단일 이미지(Single-image), 다중 이미지(Multi-image), 그리고 비디오(Video)라는 세 가지 핵심 컴퓨터 비전 시나리오 모두에서 동시에 최첨단(SoTA) 성능을 달성하는 것을 목표로 한다. 특히, 이미지 학습을 통해 얻은 능력이 비디오나 다중 이미지 작업으로 전이되는 'Task Transfer' 메커니즘을 통해 새로운 창발적 능력(Emerging Capabilities)을 구현하고자 한다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 **"강력한 단일 이미지 모델은 다중 이미지 및 비디오 작업으로의 전이 능력이 뛰어나다"**는 직관에 기반한다. 이를 위해 다음과 같은 설계 전략을 채택하였다.

1. **통합된 시각적 표현(Unified Visual Representation):** AnyRes 전략을 확장하여 서로 다른 모달리티(이미지, 비디오)가 LLM 관점에서 유사한 토큰 시퀀스로 인식되도록 설계함으로써 모달리티 간 전이를 용이하게 하였다.
2. **커리큘럼 학습 기반의 훈련 파이프라인:** 언어-이미지 정렬 $\rightarrow$ 고품질 지식 학습 $\rightarrow$ 시각적 지시 튜닝(단일 이미지 $\rightarrow$ OneVision 통합) 순으로 학습 난이도를 높여 모델의 범용성을 확보하였다.
3. **대규모 고품질 합성 데이터 활용:** 모델 스스로 생성한 캡션과 GPT-4V 등을 이용한 고품질 합성 데이터를 대량으로 학습시켜 데이터 스케일링의 효율성을 극대화하였다.

## 📎 Related Works

기존의 상용 LMM인 GPT-4V, GPT-4o, Gemini, Claude-3.5 등은 이미 다양한 비전 시나리오에서 뛰어난 성능을 보이고 있다. 반면, 오픈 소스 커뮤니티에서는 각 시나리오별로 특화된 모델들이 개발되어 왔다.

- **한계점:** 대부분의 오픈 소스 모델은 단일 이미지 성능에 치중해 있으며, 비디오 모델의 경우 이미지 성능을 희생하는 경우가 많았다.
- **차별점:** LLaVA-OneVision은 LLaVA-NeXT-Interleave의 레시피를 계승하면서도, 더 확장된 데이터셋과 강력한 LLM(Qwen-2)을 결합하여 세 가지 시나리오 모두에서 높은 성능을 내는 최초의 단일 오픈 모델을 지향한다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

LLaVA-OneVision은 전형적인 LLaVA 시리즈의 미니멀리즘 설계를 따른다.

- **Vision Encoder:** SigLIP ($\text{g}_\psi(\cdot)$)를 사용하여 입력 이미지 $\text{X}_v$를 시각적 특징 $\text{Z}_v$로 인코딩한다.
- **Projector:** 2-layer MLP ($\text{p}_\theta(\cdot)$)를 통해 시각적 특징을 LLM의 워드 임베딩 공간으로 투영하여 시각적 토큰 $\text{H}_v$를 생성한다.
- **LLM:** Qwen-2 ($\text{f}_\phi(\cdot)$)를 사용하여 최종 응답을 생성한다.

최종 정답 $\text{X}_a$에 대한 확률은 다음과 같은 수식으로 계산된다.
$$p(\text{X}_a | \text{X}_v, \text{X}_q) = \prod_{i=1}^{L} p(x_i | \text{X}_v, \text{X}_q, <i, \text{X}_{a,<i})$$
여기서 $\text{X}_v$는 시나리오에 따라 단일 이미지 크롭, 다중 이미지, 또는 비디오 프레임 시퀀스가 된다.

### 2. 시각적 표현 전략 (Higher AnyRes)

해상도와 토큰 수 사이의 균형을 맞추기 위해 **Higher AnyRes** 전략을 사용한다.

- **토큰 수 조절:** 총 토큰 수 $L$이 임계값 $\tau$를 초과할 경우, Bilinear Interpolation을 사용하여 각 크롭당 토큰 수 $T_{new}$를 다음과 같이 줄인다.
$$T_{new} = \begin{cases} \frac{\tau}{(a \times b + 1)} & \text{if } L > \tau \\ T & \text{if } L \le \tau \end{cases}$$
- **시나리오별 토큰 할당:**
  - **단일 이미지:** 고해상도 유지 및 많은 토큰을 할당하여 비디오 표현과 유사하게 만들어 전이 능력을 높인다.
  - **다중 이미지:** 계산 비용 절감을 위해 기본 해상도(Base resolution)만 사용한다.
  - **비디오:** 각 프레임을 기본 해상도로 리사이즈하고 Bilinear Interpolation으로 토큰을 줄여 더 많은 프레임을 처리한다.

### 3. 학습 데이터 구성

- **고품질 지식 학습(Knowledge Learning):** 재캡셔닝된 상세 설명 데이터(3.5M), 문서/OCR 데이터(1.1M), 중국어 및 언어 데이터(235K) 등 주로 합성 데이터를 사용하여 지식을 주입한다.
- **시각적 지시 튜닝(Instruction Tuning):**
  - **Single-Image Stage:** 3.2M 개의 고품질 단일 이미지 데이터를 통해 기초 능력을 학습한다.
  - **OneVision Stage:** 비디오, 다중 이미지, 단일 이미지가 혼합된 1.6M 개의 데이터를 통해 범용 능력을 확장한다.

### 4. 훈련 절차 (Curriculum Learning)

학습은 총 세 단계로 진행된다.

1. **Stage-1 (Language-Image Alignment):** Projector만 학습시켜 시각 특징을 언어 공간에 정렬한다.
2. **Stage-1.5 (High-Quality Knowledge Learning):** 전체 모델을 학습시켜 새로운 고품질 지식을 주입한다.
3. **Stage-2 (Visual Instruction Tuning):**
    - 먼저 **단일 이미지 훈련**을 통해 강력한 기초 능력을 갖춘다.
    - 이후 **OneVision 훈련**을 통해 비디오 및 다중 이미지 시나리오로 능력을 확장하고 전이시킨다.

## 📊 Results

### 1. 실험 설정

- **모델 사이즈:** 0.5B, 7B, 72B 세 가지 버전으로 제공하여 에지 디바이스부터 클라우드 서버까지 대응한다.
- **평가 지표:** LMMs-Eval 프레임워크를 사용하여 0-shot 설정으로 평가하였다.
- **비교 대상:** GPT-4V, GPT-4o 등 상용 모델 및 InternVL-2, Cambrian-1 등 오픈 모델과 비교하였다.

### 2. 주요 결과

- **단일 이미지:** LLaVA-OneVision-72B는 대부분의 벤치마크에서 GPT-4V를 능가하며 GPT-4o에 근접하는 성능을 보였다. 특히 차트, 다이어그램, 문서 이해 분야에서 강점을 보인다.
- **다중 이미지:** 단일 이미지 단계만 거친 모델(SI)조차 기존의 다중 이미지 특화 모델보다 우수한 성능을 보였으며, OneVision 훈련 후에는 GPT-4V를 상당한 차이로 앞질렀다.
- **비디오:** EgoSchema 및 VideoMME와 같은 복잡한 벤치마크에서 기존 오픈 소스 모델보다 월등한 성능을 보였으며, 상용 모델인 GPT-4V와 경쟁 가능한 수준에 도달하였다.

## 🧠 Insights & Discussion

### 1. 창발적 능력 (Emerging Capabilities)

본 모델은 명시적으로 학습되지 않은 작업에서도 다음과 같은 전이 능력을 보여주었다.

- **S1 (다이어그램+차트 통합 이해):** 단일 이미지에서 배운 개별 능력이 다중 이미지 시나리오에서 결합되어 복합 추론을 수행한다.
- **S3 (Set-of-mark Prompting):** 시각적 참조(Referring)와 OCR 능력이 결합되어 숫자 라벨이 붙은 객체를 정확히 식별한다.
- **S5 (비디오 간 차이 분석):** 다중 이미지의 '틀린 그림 찾기' 능력이 비디오 시나리오로 전이되어 두 비디오의 차이점을 상세히 분석한다.

### 2. 분석 및 비판적 해석

- **강점:** 단일 이미지 모델의 성능을 극대화하는 것이 비디오/다중 이미지 모델로 가기 위한 가장 효율적인 경로임을 입증하였다. 또한, 합성 데이터의 대규모 활용이 실제 성능 향상으로 이어짐을 보여주었다.
- **한계:** 실세계의 시각적 채팅(Visual Chat) 시나리오에서는 여전히 GPT-4o와 같은 최신 상용 모델과 간극이 존재한다. 이는 더 강력한 LLM 백본이나 더 정교한 선호도 학습(Preference Learning)이 필요함을 시사한다.
- **가정:** 본 연구는 시각적 토큰의 수를 비슷하게 맞추는 것이 모달리티 간 전이에 도움이 된다고 가정하였으며, 실험 결과 이는 유효한 전략이었음이 확인되었다.

## 📌 TL;DR

LLaVA-OneVision은 단일 이미지, 다중 이미지, 비디오 시나리오를 모두 아우르는 통합 오픈 LMM으로, **"강력한 이미지 모델 $\rightarrow$ 비디오/다중 이미지 모델"**로의 Task Transfer 전략을 통해 성능을 극대화하였다. 특히 72B 모델은 상용 모델인 GPT-4V를 능가하는 수준의 범용 시각 이해 능력을 보여주었으며, 이는 향후 LMM의 스케일링 및 범용 비전 어시스턴트 개발에 중요한 이정표가 될 것으로 보인다.
