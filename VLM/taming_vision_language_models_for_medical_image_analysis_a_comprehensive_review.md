# Taming Vision-Language Models for Medical Image Analysis: A Comprehensive Review

Haoneng Lin, Cheng Xu, Jing Qin (2025)

## 🧩 Problem to Solve

본 논문은 일반적인 목적의 Vision-Language Models (VLMs)를 의료 영상 분석(Medical Image Analysis) 분야에 적용할 때 발생하는 기술적 격차와 한계를 해결하고자 한다.

의료 영상 분석은 전통적으로 영상 데이터에만 의존해 왔으나, 실제 임상 환경에서는 임상 노트, 영상 판독문, 환자의 과거 기록과 같은 텍스트 데이터가 진단과 의사 결정에 필수적이다. 따라서 시각 정보와 텍 uma-modal 정보를 통합하는 VLM의 도입은 매우 중요하다. 하지만 일반 도메인에서 학습된 VLM을 의료 도메인에 그대로 적용하기에는 다음과 같은 심각한 문제들이 존재한다.

1. **도메인 격차(Domain Gap):** 일반 이미지와 의료 영상 간의 시각적 특징 차이가 크며, 의료 전문 용어의 복잡성으로 인해 직접적인 전이가 어렵다.
2. **병리학적 변이의 복잡성:** 의료 데이터는 미세한 병리학적 변화가 진단 결과를 바꾸는 경우가 많아, 높은 신뢰성과 정밀도가 요구된다.
3. **블랙박스 모델의 불투명성:** 고위험 환경인 의료 현장에서 VLM의 의사 결정 과정이 투명하지 않은 점은 규제 준수 및 임상 적용의 큰 걸림돌이 된다.
4. **데이터 부족 및 다양성:** 고품질의 의료 영상-텍스트 쌍 데이터셋을 구축하는 데 드는 비용이 매우 높으며, 기관마다 프로토콜이 달라 데이터의 이질성이 크다.

결과적으로 본 논문의 목표는 일반 VLM을 의료 영상 분석 작업에 맞게 최적화하는 방법론을 체계적으로 정리하고, 이를 위한 다섯 가지 적응 전략(Adaptation Strategies)을 제안하여 향후 연구 방향을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 일반 목적의 VLM을 의료 도메인으로 전이시키기 위한 **통합적 적응 프레임워크(Unified Framework of Five Adaptation Strategies)**를 제안하고 이를 체계적으로 분석했다는 점이다.

단순한 모델 아키텍처의 나열이 아니라, 다음과 같은 다섯 가지 전략적 관점에서 VLM의 의료 분야 적응 방법을 분류하였다.

- **입력 증강(Input Augmentation):** 부족한 데이터를 보완하기 위해 VLM을 이용해 이미지나 텍스트를 생성/정제하는 방법.
- **특징 추출(Feature Extraction):** 사전 학습된 VLM의 인코더를 활용해 고차원 특징을 뽑아내고 이를 의료 작업에 맞게 융합하는 방법.
- **감독 강화(Supervision Enhancement):** VLM의 지식을 손실 함수(Loss Function)나 유사도 측정에 활용하여 학습 효율을 높이는 방법.
- **작업별 미세 조정(Task-specific Fine-tuning):** PEFT(Parameter-Efficient Fine-Tuning) 등을 통해 적은 비용으로 도메인 적응을 수행하는 방법.
- **직접 배포(Direct Deployment):** 추가 학습 없이 제로샷(Zero-shot)이나 프롬프트 엔지니어링만으로 VLM을 활용하는 방법.

또한, 이러한 전략들이 11가지 주요 의료 영상 분석 작업(보고서 생성, VQA, 세그멘테이션 등)에서 어떻게 구현되는지를 분석하여 실질적인 가이드라인을 제공한다.

## 📎 Related Works

논문은 기존의 의료 영상 분석 접근 방식과 최신 VLM의 흐름을 비교 분석한다.

1. **전통적인 딥러닝 방식:** 주로 단일 모달리티(영상 전용)에 집중했으며, 특정 작업(종양 검출, 장기 분할 등)에서는 성능이 좋지만 다면적인 임상 맥락을 파악하지 못해 실제 임상 워크플로우에 통합되기 어려웠다.
2. **기존 멀티모달 융합 방식:** 수작업으로 설계된(Hand-crafted) 아키텍처에 의존하여 확장성이 부족했고, 주석이 달린 데이터셋의 부족으로 인해 성능 향상에 한계가 있었다.
3. **최신 VLM (CLIP, BLIP, GPT-4V 등):** 대규모 이미지-텍스트 쌍으로 사전 학습되어 모달리티에 구애받지 않는 세만틱 임베딩을 획득하였다. 이를 통해 제로샷 학습, 교차 모달 검색 등이 가능해졌으나, 의료 도메인 특유의 정밀함과 신뢰성 부족 문제가 여전히 남아 있다.

본 논문은 기존 리뷰 논문들이 주로 모델 구조나 벤치마크 비교에 치중한 것과 달리, **'일반 VLM을 어떻게 의료 전용으로 길들이는가(Taming)'**라는 방법론적 경로(Methodological Pathways)에 집중하여 차별성을 둔다.

## 🛠️ Methodology

본 논문은 VLM을 의료 영상 분석에 적용하기 위한 학습 패러다임과 구체적인 적응 전략을 상세히 설명한다.

### 1. 학습 패러다임 (Training Schemes)

- **Pretraining:** 대규모 의료 데이터셋을 사용하여 기초 모델을 구축한다. 데이터 부족 문제를 해결하기 위해 LLM으로 이미지-텍스트 쌍을 생성(UniMed-CLIP)하거나, VQA 기반의 사전 학습(VQA-Pretraining)을 통해 임상적 특징을 학습시킨다.
- **Fine-tuning:** 사전 학습된 모델을 의료 데이터로 조정한다. 계산 비용을 줄이기 위해 LoRA와 같은 PEFT 기술을 사용하거나, 노이즈에 강건한 RAN(Rectify Adversarial Noise) 프레임워크를 적용한다.
- **Prompt Learning:** 모델 가중치를 고정하고 입력 프롬프트를 최적화한다. SAMPOT와 같이 다운스트림 성능에 따라 프롬프트를 동적으로 조정하거나, 외부 의료 지식 그래프를 임베딩하여 해석력을 높인다.

### 2. 다섯 가지 적응 전략 (Adaptation Strategies)

#### ① 입력 증강 (Input Augmentation)

- **이미지 생성:** Diffusion 모델과 정밀한 텍스트 프롬프트를 결합하여 고충실도 합성 이미지를 생성, 데이터 다양성을 확보한다.
- **텍스트 생성 및 정제:** GPT-4와 같은 LLM을 이용해 영상에서 임상적 개념(Concept)을 추출하거나, 스페인어 보고서를 영어로 번역하여 데이터셋의 가용성을 높인다.

#### ② 특징 추출 (Feature Extraction)

VLM 인코더에서 뽑은 특징을 융합하는 방법으로, 크게 세 가지 메커니즘을 사용한다.

- **Attention Mechanism:** Cross-attention을 통해 시각적 영역과 텍스트 쿼리 간의 정렬을 수행한다. (예: Bilinear Attention, Gated Attention)
- **Matrix Operation:** 단순 연결(Concatenation), 원소별 곱셈(Element-wise multiplication), 덧셈을 통해 효율적으로 특징을 통합한다.
- **Text-Parameterized Convolution (TPC):** 텍스트 특징을 이용해 합성곱 계층의 가중치($W$)와 편향($b$)을 동적으로 생성하여 시각 특징 추출에 반영한다.

#### ③ 감독 강화 (Supervision Enhancement)

- **출력 레벨 감독(Output-level):** VLM을 이용해 전문가 수준의 유사-라벨(Pseudo-labels)이나 구조화된 주석을 자동으로 생성하여 학습 데이터로 활용한다.
- **특징 레벨 감독(Feature-level):** VLM이 추출한 시각-텍스트 특징 간의 코사인 유사도나 대조 손실(Contrastive Loss)을 설계하여 정밀한 정렬을 유도한다.

#### ④ 작업별 미세 조정 (Task-specific Fine-tuning)

- **Prompt Tuning:** 입력 공간에 학습 가능한 임베딩을 추가하여 최적화한다.
- **Adapter-based Tuning:** 기존 모델 층 사이에 가벼운 어댑터 모듈을 삽입하여 효율적으로 도메인 적응을 수행한다.
- **Prefix Tuning:** 트랜스포머의 여러 층에 학습 가능한 접두사(Prefix)를 추가하여 심층적인 상호작용을 유도한다.

#### ⑤ 직접 배포 (Direct Deployment)

- 추가 학습 없이 제로샷(Zero-shot) 능력을 활용하거나, 전문가의 피드백을 반영한 멀티뷰 프롬프팅(Multi-view prompting) 및 인컨텍스트 학습(In-context learning)을 통해 진단 작업을 수행한다.

### 3. 주요 작업별 수학적 정의

- **의료 보고서 생성 (Medical Report Generation):**
  $$S^* = \arg \max_S P(S|I; \theta)$$
  영상 $I$가 주어졌을 때, 모델 파라미터 $\theta$를 통해 가장 확률이 높고 임상적으로 유의미한 보고서 $S^*$를 생성하는 것이 목표이다.
- **의료 VQA (Medical VQA):**
  $$A = f(I, Q)$$
  영상 $I$와 질문 $Q$를 입력받아 정답 $A$를 도출하는 매핑 함수 $f$를 학습한다.
- **의료 멀티태스크 학습 (Medical Multi-Task Learning):**
  $$\mathcal{L}_{MMTL} = \sum_{i=1}^{m} w_i \mathcal{L}_i$$
  여러 작업(분류, 분할 등)의 손실 함수 $\mathcal{L}_i$를 가중치 $w_i$와 함께 통합 최적화하여 작업 간 시너지를 도모한다.

## 📊 Results

본 논문은 특정 실험 결과보다는 수많은 기존 문헌을 분석한 **메타 분석 결과**를 제시한다.

1. **전략별 분포 (Figure 3):**
    - **보고서 생성:** 특징 추출(56.2%)과 미세 조정(31.2%) 전략이 주를 이룬다.
    - **VQA:** 특징 추출(31.2%)과 미세 조정(26.6%)이 많이 사용된다.
    - **세그멘테이션:** 특징 추출(12.6%)보다 미세 조정과 직접 배포의 비중이 낮으며, 하이브리드 전략의 활용도가 높다.
    - **멀티태스크 학습:** 특징 추출(100%) 전략이 압도적으로 많이 사용되어, 통합된 특징 공간을 구축하는 것이 핵심임을 보여준다.

2. **시간적 진화 (Figure 4):**
    - 2021~2022년에는 단순한 특징 추출과 미세 조정이 주류였으나, 2023년 이후부터는 프롬프트 튜닝, 어댑터 기반 튜닝, 그리고 LLM을 결합한 직접 배포 전략이 급격히 증가하는 추세이다.

3. **작업별 한계 분석:**
    - **보고서 생성:** 미세한 병변(microcalcifications) 탐지 능력이 부족하고, 보고서의 구조적 연속성이 떨어지는 문제가 확인되었다.
    - **VQA:** 희귀 질환에 대한 클래스 불균형 문제와 복잡한 추론 능력이 부족하며, 다회차 대화 시 일관성이 떨어지는 한계가 있다.
    - **세그멘테이션:** CT/MRI 같은 방사선 영상에서의 성능이 일반 영상(피부과 등)보다 낮게 나타나며, 모호한 경계 처리 능력이 부족하다.

## 🧠 Insights & Discussion

### 강점 및 기회

- VLM은 단순한 영상 분석을 넘어, 의료 전문가의 지식을 텍스트 형태로 주입함으로써 **해석 가능성(Interpretability)**과 **일반화 능력**을 획기적으로 높일 수 있다.
- 특히 데이터가 극도로 부족한 희귀 질환 분야에서 제로샷/퓨샷 학습을 통한 진단 가능성을 열어주었다.

### 한계 및 비판적 해석

- **데이터의 질적 문제:** VLM을 이용한 데이터 증강(Augmentation) 시, 실제 의료적으로 불가능한 '환각(Hallucination)' 이미지가 생성될 위험이 있으며, 이는 오진으로 이어질 수 있는 치명적인 결함이다.
- **계산 효율성:** 고해상도 3D 의료 영상을 처리하기 위한 VLM의 계산 비용이 너무 커서, 실시간 진단이 필요한 응급 상황이나 자원이 제한된 환경에서의 배포는 여전히 어렵다.
- **도메인 불일치:** 일반 이미지-텍스트 쌍으로 학습된 CLIP 계열 모델들이 의료 영상의 미세한 텍스트-이미지 정렬(Fine-grained alignment)을 완벽히 수행하지 못하고 있음을 알 수 있다.

## 📌 TL;DR

본 논문은 일반 목적의 Vision-Language Model을 의료 영상 분석에 최적화하기 위한 **5가지 적응 전략(입력 증강, 특징 추출, 감독 강화, 작업별 미세 조정, 직접 배포)**을 제안하고 체계적으로 분류한 종합 리뷰 보고서이다.

VLM은 의료 데이터 부족 문제를 해결하고 진단의 해석력을 높일 수 있는 강력한 도구이지만, **데이터의 환각 문제, 높은 계산 비용, 그리고 의료 도메인 특유의 정밀한 정렬 부족**이라는 과제가 남아 있다. 향후 연구는 **경량화 모델(LoRA, Distillation)** 개발과 **의료 지식 그래프(Knowledge Graph)** 통합, 그리고 **안전한 연합 학습(Federated Learning)** 기반의 데이터 확보 방향으로 나아가야 할 것으로 보인다.
