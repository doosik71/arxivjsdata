# Multivariate Time Series Anomaly Detection via Dynamic Graph Forecasting

Katrina Chen, Mingbin Feng, Tony S. Wirjanto (2023)

## 🧩 Problem to Solve

본 논문은 다변량 시계열(Multivariate Time Series) 데이터에서 발생하는 이상치 탐지 문제를 다룬다. 일반적으로 단변량 시계열에서의 이상치는 과거의 패턴에서 벗어난 비정상적인 값이나 템포의 변화를 의미하지만, 다변량 시계열에서는 각 변수 간의 관계(inter-series relationship), 예를 들어 상관관계가 시간에 따라 비정상적으로 변하는 상황이 발생한다.

기존의 그래프 신경망(GNN) 기반 연구들은 이러한 변수 간 관계를 모델링하려 했으나, 대부분의 경우 정적인 그래프(static graph)를 학습하거나 특정 컨텍스트 윈도우 내에서만 그래프를 유지한다. 이는 동적으로 진화하는 비정상적 관계를 명시적으로 탐지하는 데 한계가 있다. 또한, 그래프 재구성(reconstruction) 기반의 접근 방식은 그래프의 이산적인 특성으로 인해 세밀한 시간적 패턴(temporal patterns)을 포착하지 못하고 데이터를 거칠게(coarse) 만드는 경향이 있다.

따라서 본 연구의 목표는 변수 간의 동적인 관계 변화(inter-series)와 개별 변수의 시간적 패턴(intra-series) 모두를 효과적으로 포착하여, 정상 상태에서 비정상 상태로의 전이를 정밀하게 탐지할 수 있는 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **동적 상관관계 그래프(Dynamic Correlation Graphs)**의 리스트를 활용하여, 그래프 예측(Graph Forecasting) 작업과 시계열 예측(Time Series Forecasting) 작업을 동시에 수행하는 DyGraphAD 프레임워크를 제안하는 것이다.

주요 기여 사항은 다음과 같다:

1. **동적 그래프 예측 프레임워크 제안**: 단기적인 관계 변동과 장기적인 정적 관계를 모두 모델링할 수 있는 그래프 인코더 모듈을 설계하였다.
2. **이종 예측 작업의 결합**: 그래프 예측과 시계열 예측 작업을 결합하여, 각 작업이 가진 한계(그래프의 이산성으로 인한 세밀한 패턴 손실 vs 변수 간 관계 파악 부족)를 상호 보완하도록 설계하였다.
3. **DTW 기반의 강건한 관계 추출**: 시간적 밀림(time shift) 현상이 빈번한 실제 산업 데이터의 특성을 고려하여, 유클리드 거리 대신 Dynamic Time Warping (DTW) 거리를 사용하여 상관관계 그래프를 구축하였다.

## 📎 Related Works

기존의 시계열 이상치 탐지는 크게 예측 기반(forecasting-based)과 재구성 기반(reconstruction-based) 방법으로 나뉜다. RNN이나 TCN 기반의 모델들은 시간적 의존성은 잘 포착하지만, 변수 간의 명시적인 관계를 학습하도록 설계되지 않아 해석력이 떨어지고 관계 기반 이상치를 놓치는 경향이 있다.

최근에는 GNN을 활용하여 변수 간 관계를 모델링하는 시도가 늘고 있다. GDN이나 MTAD-GAT 같은 모델들은 그래프 구조를 학습하거나 고정된 인접 행렬을 사용하지만, 이는 전역적으로 정적인 구조를 가정하므로 시간에 따라 변하는 동적인 관계의 편차를 탐지하는 데 적합하지 않다. 한편, MSCRED와 같이 유사도 행렬을 재구성하는 방식은 ConvLSTM 등을 사용하지만, 그래프 데이터의 특성상 인접 픽셀 간의 상관관계를 가정하는 CNN 기반 구조가 부적절하며, 재구성 과정에서 세밀한 시간적 이상치를 놓치는 문제가 있다.

## 🛠️ Methodology

### 1. 동적 상관관계 그래프 구축 (Construction of Dynamic Correlation Graph)

입력 샘플 $S_t$를 $m$개의 불연속적인 세그먼트로 나누고, 각 세그먼트 내에서 변수 쌍 간의 유사도를 계산하여 $m$개의 인접 행렬 $\{A_{t,s}\}_{s=1}^m$을 생성한다. 시간적 밀림에 강건하도록 DTW 거리를 사용하며, 행렬의 각 원소 $A_{t,s}^{i,j}$는 다음과 같이 정규화된 유사도로 계산된다:
$$A_{t,s}^{i,j} = \exp \left( -\frac{\text{DTW}^2(S_{t,s}^{i\cdot}, S_{t,s}^{j\cdot})}{\tau} \right)$$
여기서 $\tau$는 하이퍼파라미터이다.

### 2. 그래프 인코더 모듈 (Graph Encoder Module)

그래프 인코더는 동적 그래프의 시간적 역동성과 각 그래프 내부의 구조적 특징을 추출한다.

- **특징 추출**: 2D Convolution과 Dilated Inception Layer (DIL)를 통해 시간적 패턴을 학습한 표현 $Z_t$를 생성한다. DIL은 다양한 커널 크기를 통해 DTW와 유사하게 시간적 편차를 어느 정도 수용하는 효과를 준다.
- **가중치 인접 행렬**: 정적 관계를 나타내는 학습 가능한 행렬 $Q$와 동적 관계 행렬 $A_{t,s}$를 결합하여 최종 인접 행렬 $\tilde{A}_{t,s}$를 생성한다:
$$\tilde{A}_{t,s} = \sigma(W_1) \odot Q + (1 - \sigma(W_1)) \odot A_{t,s}$$
- **그래프 합성**: MixHop 그래프 합성곱(Graph Convolution) 모듈을 적용하여 각 그래프의 은닉 상태 $\{H_{t,s}\}_{s=1}^m$을 얻는다.

### 3. 그래프 예측 모듈 (Graph Forecasting Module)

과거의 그래프 은닉 상태들을 입력받아 다음 단계의 그래프 $\hat{A}_{t,m}$을 예측한다.

- **구조**: Transformer 블록과 인과적 어텐션 마스크(causal attention mask)를 사용하여 그래프 시퀀스의 시간적 역동성을 학습한다.
- **디코더**: Transformer의 출력 $O_t$를 FC 레이어와 $l_2$ 정규화를 거쳐 $E_t$를 생성하고, 이를 외적 $\text{E}_t = J_t J_t^T$를 통해 인접 행렬 형태로 변환한다.
- **최근 그래프 보정**: 예측된 $E_t$와 가장 최근의 실제 그래프 $A_{t,m-1}$을 결합하여 최종 예측치 $\hat{A}_{t,m}$을 도출함으로써 정보 손실을 보완한다.

### 4. 시계열 예측 모듈 (Time Series Forecasting Module)

그래프 인코더를 통해 변수 간 정보가 전파된 상태 $M_t$를 활용하여 다음 시점의 값 $\hat{y}_{t+1}$을 예측한다. 이는 그래프 예측 작업이 놓칠 수 있는 세밀한 시간적 이상치를 포착하기 위함이며, 2D Convolution 레이어와 풀링을 통해 최종 값을 출력한다.

### 5. 학습 목표 및 이상치 점수

**손실 함수**: 시계열 예측 오차(MSE)와 그래프 예측 오차(Frobenius norm)의 합을 최소화한다.
$$L = \sum_{t \in T_{\text{train}}} \frac{1}{N} \|y_{t+1} - \hat{y}_{t+1}\|_2^2 + \frac{1}{N^2} \|A_{t,m} - \hat{A}_{t,m}\|_F^2$$

**이상치 점수**: 각 변수 $i$에 대해 시계열 예측 오차($\text{Err}_{t,i,ts}$)와 그래프 예측 오차($\text{Err}_{t,i,graph}$)의 조화 평균(harmonic mean)을 계산하여 최종 점수를 산출한다.

## 📊 Results

### 실험 설정

- **데이터셋**: SWaT, WADI, SMD, SMAP, MSL 등 5개의 실제 산업 벤치마크 데이터셋을 사용하였다.
- **비교 대상**: Deep SVDD, DAGMM, LSTM-VAE, MAD-GAN, USAD, OMNI, MSCRED, MTAD-GAT, GDN 등.
- **평가 지표**: F1-score, Precision, Recall을 사용하였다.

### 주요 결과

- **성능 우위**: DyGraphAD는 모든 데이터셋에서 베이스라인 모델들보다 높은 F1-score를 기록하였다. 특히 변수가 많고 관계가 복잡한 WADI 데이터셋에서 다른 모델들이 낮은 Recall을 보인 반면, 본 모델은 현저한 성능 향상을 보였다.
- **상호 보완성**: Ablation Study 결과, 시계열 예측만 수행하거나 그래프 예측만 수행했을 때보다 두 작업을 결합했을 때 성능이 더 높게 나타났다. 이는 두 작업이 서로 다른 차원의 이상치(변수 간 관계 vs 개별 변수 패턴)를 포착하기 때문이다.
- **그래프 기반 모델 대비 우수성**: GDN, MTAD-GAT와 같은 기존 GNN 기반 모델보다 높은 성능을 보였으며, 이는 정적 그래프가 아닌 동적으로 진화하는 관계를 명시적으로 활용했기 때문으로 분석된다.

## 🧠 Insights & Discussion

본 논문은 다변량 시계열 이상치 탐지에서 **"관계의 동적 변화"**와 **"시간적 세부 패턴"**이라는 두 마리 토끼를 동시에 잡으려 했다는 점에서 강점이 있다.

특히 Case Study를 통해 밝혀진 점은 다음과 같다:

1. **상보적 탐지**: 사이버 공격으로 인해 변수 간의 관계가 무너지는 경우(예: SWaT의 수위-유량 관계)는 그래프 예측 작업이 매우 민감하게 반응하며, 아주 짧은 순간 발생하는 급격한 값의 변화(예: SMD의 스파이크)는 시계열 예측 작업이 더 잘 포착한다.
2. **재구성 vs 예측**: MSCRED와 같은 재구성 기반 모델은 오토인코더가 정체 함수(identity mapping)를 학습하는 경향이 있어 이상치 상황에서도 그래프를 너무 잘 재구성해버리는 문제가 있다. 반면, 본 모델은 '예측' 프레임워크를 채택하여 정상 패턴의 진화 방향과 실제 변화의 괴리를 정확히 측정함으로써 이 문제를 해결하였다.

다만, DTW 기반의 그래프 구축 과정이 전처리 단계에서 계산 비용을 발생시킬 수 있으며, 하이퍼파라미터 $\tau$나 윈도우 크기 $w$ 등의 설정이 데이터셋마다 달라져야 한다는 점은 실무 적용 시 고려해야 할 사항이다.

## 📌 TL;DR

DyGraphAD는 DTW 기반의 동적 상관관계 그래프를 구축하고, 이를 통해 **그래프 구조의 진화 예측**과 **시계열 값 예측**을 동시에 수행하는 프레임워크이다. 변수 간의 관계 변화와 개별 변수의 시간적 패턴을 모두 고려함으로써, 기존의 정적 그래프 모델이나 단순 재구성 모델이 놓치던 복잡한 산업 현장의 이상치를 매우 효과적으로 탐지할 수 있음을 증명하였다. 이 연구는 향후 고차원 센서 데이터의 실시간 모니터링 및 사이버-물리 시스템(CPS)의 보안 강화에 중요한 역할을 할 것으로 기대된다.
