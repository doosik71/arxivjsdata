# CNTS: Cooperative Network for Time Series

Jinsheng Yang, Yuanhai Shao, Chunna Li (2023)

## 🧩 Problem to Solve

본 논문은 시계열 데이터에서 비지도 방식의 이상치 탐지(Unsupervised Anomaly Detection) 문제를 해결하고자 한다. 시계열 데이터의 이상치 탐지는 금융 사기 탐지, 침입 탐지, IoT 보안 등 다양한 분야에서 매우 중요하지만, 실제 환경에서는 고품질의 레이블링된 데이터를 확보하기가 매우 어렵다는 문제가 있다.

기존의 비지도 학습 기반 방법론, 특히 재구성 기반(Reconstruction-based) 방법들은 데이터의 정상적인 패턴을 학습하여 재구성 오차(Reconstruction Error)를 통해 이상치를 판별한다. 그러나 이러한 방식은 다음과 같은 두 가지 핵심적인 한계가 존재한다. 첫째, 학습 과정 중에 포함된 이상치가 모델 학습에 그대로 반영되어 결과적으로 모델이 이상치까지 재구성하도록 학습되는 '오염' 문제가 발생하며, 이는 탐지 성능을 저하시킨다. 둘째, 네트워크의 최적화 목표가 '이상치 탐지'가 아닌 '데이터 재구성' 자체에 집중되어 있어, 탐지 성능 측면에서 최적의 결과를 내지 못하는 하위 최적(Suboptimal) 결과가 도출되는 경향이 있다.

따라서 본 논문의 목표는 재구성과 탐지라는 두 가지 역할을 분리하고 서로 협력하게 함으로써, 이상치의 영향을 최소화하고 탐지 성능을 극대화하는 협력 네트워크인 CNTS(Cooperative Network for Time Series)를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 재구성 네트워크(Reconstructor, $R$)와 탐지 네트워크(Detector, $D$)를 분리하고, 이들이 상호 피드백을 주고받는 '협력적 학습 구조'를 설계한 것이다.

1. **역할의 분리 및 상호 보완**: $R$은 정상 데이터의 특징을 학습하여 재구성하는 데 집중하고, $D$는 $R$이 제공한 재구성 오차를 바탕으로 이상치를 직접 탐지하는 데 집중한다.
2. **이상치 영향을 배제하는 학습 전략**: $D$가 판별한 이상치 정보를 $R$에게 전달하여, $R$이 학습 시 상위 일부의 이상치 데이터를 제외하고 학습하게 함으로써 이상치에 의한 모델 오염을 방지한다.
3. **탐지 성능의 직접적 최적화**: $R$의 재구성 오차를 $D$의 학습 목표(Pseudo-label)로 활용함으로써, 재구성 모델의 부수적인 결과물로 탐지하는 것이 아니라 탐지 네트워크를 직접 학습시킨다.
4. **다목적 최적화 문제로의 정식화**: 재구성과 탐지의 목표를 동시에 달성하기 위해 협력적 솔루션 전략을 기반으로 한 다목적 프로그래밍 문제를 해결한다.

## 📎 Related Works

기존의 재구성 기반 이상치 탐지 모델들은 정상 데이터는 정확하게 재구성하고, 이상치는 재구성에 실패하도록 학습하는 것을 목표로 한다. 일반적인 목적 함수는 다음과 같다.
$$\min_{\theta_M} \frac{1}{n} \sum_{i=1}^{n} ||x_i - M(x_i, \theta_M)||^2$$
이러한 접근 방식을 사용하는 대표적인 연구로는 DAGMM, MAD-GAN, USAD, MTAD-GAT, CAE-M 등이 있다.

하지만 이러한 기존 방식들의 한계는 다음과 같다.

- **이상치 민감성**: 모든 데이터에 동일한 가중치를 부여하여 학습하므로, 훈련 세트에 포함된 소수의 이상치가 재구성 모델의 학습 방향을 왜곡할 수 있다.
- **간접적 탐지**: 모델의 목표가 재구성이기 때문에, 재구성 오차가 반드시 이상치 탐지에 최적화된 지표가 된다는 보장이 없다.

본 논문은 이러한 한계를 극복하기 위해 GAN(Generative Adversarial Networks)의 적대적 학습 구조에서 영감을 얻었으나, 서로 경쟁하는 것이 아니라 동일한 목표를 향해 서로를 돕는 '협력적(Cooperative)' 관계를 구축했다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조

CNTS는 두 개의 딥러닝 네트워크인 재구성기(Reconstructor, $R$)와 탐지기(Detector, $D$)로 구성된다. 두 네트워크 모두 기본 아키텍처로 **FEDFormer**를 사용한다. FEDFormer는 시계열 데이터의 계절성-추세 분해(Seasonal-Trend Decomposition)와 주파수 영역 분석을 결합하여 장기 시계열 예측 및 분석에 효율적인 구조이다.

### 학습 절차 및 상세 방법론

CNTS의 학습은 $R$과 $D$가 번갈아 가며 업데이트되는 협력 과정으로 이루어진다.

#### 1. 탐지기($D$)의 학습 (Detection Phase)

$D$의 목표는 $R$이 생성한 재구성 오차를 바탕으로 이상치를 판별하는 능력을 키우는 것이다.

- **입력 및 경로**: 입력 데이터 $W$는 $R$을 통과하여 재구성된 윈도우 $W^r$와 재구성 오차 $e_r$을 생성한다. 동시에 $W$는 $D$를 통과하여 이상치 점수($\text{Score}$)를 생성한다.
- **데이터 선택**: 클래스 불균형으로 인한 학습 붕괴를 막기 위해, $\text{Score}$ 값이 가장 높은 상위 20%의 데이터 포인트만을 학습에 사용한다.
- **손실 함수**: $R$의 재구성 오차($e_r$)와 $D$의 예측 점수($\text{Score}$) 사이의 CrossEntropy 손실을 최소화한다.
$$\theta^*_D = \text{argmin}_{\theta_D} L_C(\text{Softmax}(e^s_r), \text{Softmax}(\text{Score}^s))$$
이 과정에서 $R$의 파라미터 $\theta_R$은 고정된다.

#### 2. 재구성기($R$)의 학습 (Reconstruction Phase)

$R$의 목표는 이상치의 영향을 받지 않고 정상 데이터의 분포를 정확하게 학습하는 것이다.

- **입력 및 경로**: 데이터 $W$가 먼저 $D$를 통과하여 이상치 점수($\text{Score}$)를 얻는다.
- **이상치 배제**: $D$가 예측한 점수 중 상위 10%에 해당하는 데이터를 '이상치'로 간주하여 학습 대상에서 제외(Drop)한다.
- **손실 함수**: 선택된 정상 데이터 $W^s$와 재구성된 데이터 $W^r_s$ 사이의 MSE(Mean Squared Error)를 최소화한다.
$$\theta^*_R = \text{argmin}_{\theta_R} L_M(W^s, W^r_s(W, \theta_R))$$
이 과정에서 $D$의 파라미터 $\theta_D$는 고정된다.

### 요약된 알고리즘 흐름

1. $R$과 $D$를 초기화한다.
2. 정해진 에포크(epoch) 동안 다음을 반복한다.
   - $R$을 학습시킨다: $D$의 점수를 이용해 상위 10%를 제외한 데이터로 MSE 손실을 최소화하여 $\theta_R$을 업데이트한다.
   - $D$를 학습시킨다: $R$의 재구성 오차를 레이블로 삼아 상위 20% 데이터에 대해 CrossEntropy 손실을 최소화하여 $\theta_D$를 업데이트한다.

## 📊 Results

### 실험 설정

- **데이터셋**: NAB, NASA-MSL, NASA-SMAP 총 3개의 실세계 시계열 데이터셋(128개 데이터 포함)을 사용하였다.
- **비교 대상 (Baselines)**: MERLIN, LSTM-NDT, OmniAnomaly, MSCRED, MAD-GAN, USAD, MTAD-GAT, CAE-M, GDN 등 9가지 딥러닝 기반 방법론과 비교하였다.
- **평가 지표**: Accuracy(ACC), F1-score, AUC를 사용하였으며, 임계값($\delta$)은 F1-score를 최대화하는 지점을 선택하였다.

### 정량적 결과

실험 결과, CNTS는 모든 데이터셋에서 9개의 베이스라인 모델보다 우수한 성능을 보였다.

- **탐지 성능**: ACC, F1, AUC 모든 지표에서 가장 높은 수치를 기록했으며, 베이스라인 대비 ACC는 약 15%, F1은 17%, AUC는 6% 향상되었다.
- **재구성 변별력**: 정상 데이터와 이상 데이터의 재구성 오차 차이를 나타내는 $\text{Dis}$ 지표($\text{Dis} = \frac{\text{MSE}_a - \text{MSE}_n}{\text{MSE}_n}$)를 측정했을 때, CNTS가 단일 재구성 모델($R$)보다 훨씬 높은 변별력을 보였으며, 특히 NASA-MSL 데이터셋에서는 약 3배 이상의 성능 향상을 보였다.

### 협력 학습의 효과 분석

학습 과정에서 정상 데이터의 재구성 손실($\text{MSE}_n$)은 지속적으로 감소하는 반면, 탐지기의 F1-score는 점진적으로 상승하는 모습이 확인되었다. 이는 $R$이 정상 데이터를 더 잘 학습함에 따라 이상치의 재구성 오차가 상대적으로 커지고, 이를 통해 $D$가 더 정확하게 이상치를 식별하게 되는 선순환 구조(Cooperative relationship)가 형성되었음을 입증한다.

## 🧠 Insights & Discussion

### 강점

본 논문은 단순히 모델의 구조를 변경한 것이 아니라, 학습 프로세스 자체에 '협력'이라는 개념을 도입하여 비지도 학습의 고질적인 문제인 '이상치 오염' 문제를 효과적으로 해결하였다. 특히 재구성과 탐지라는 서로 다른 두 태스크를 분리함으로써, 각각의 네트워크가 자신의 목적 함수에 최적화될 수 있도록 설계한 점이 성능 향상의 핵심 요인으로 분석된다.

### 한계 및 논의사항

- **계산 복잡도**: 두 개의 네트워크($R, D$)를 동시에 운용하고 번갈아 학습시켜야 하므로, 단일 모델 대비 시간 및 메모리 복잡도가 증가한다. 저자 또한 이를 언급하며 향후 효율적인 프레임워크 통합이 필요함을 시사하였다.
- **하이퍼파라미터 의존성**: $D$의 학습을 위해 상위 20%를 선택하고, $R$의 학습을 위해 상위 10%를 제외하는 등의 비율 설정이 결과에 영향을 미칠 수 있으나, 이에 대한 최적의 비율을 찾는 체계적인 분석은 부족한 상태이다.
- **기반 모델의 일반성**: FEDFormer라는 범용 시계열 모델을 기반으로 사용하였는데, 이상치 탐지나 재구성에 특화된 전용 네트워크 구조를 적용했을 때의 성능 변화에 대한 추가 연구가 필요하다.

## 📌 TL;DR

CNTS는 재구성 네트워크($R$)와 탐지 네트워크($D$)가 서로 피드백을 주고받는 협력 구조를 통해 시계열 이상치 탐지 성능을 높인 모델이다. $D$는 $R$의 오차를 통해 이상치를 배우고, $R$은 $D$의 판별 결과를 통해 이상치 데이터를 배제하며 정상 패턴만을 정교하게 학습한다. 실험 결과, 기존의 재구성 기반 모델들이 겪던 이상치 오염 문제를 해결하여 ACC, F1, AUC 등 주요 지표에서 SOTA 성능을 달성하였다. 이 연구는 비지도 학습에서 태스크 분리와 상호 협력적 학습이 매우 효과적일 수 있음을 보여주며, 향후 실시간 이상치 탐지 시스템이나 복합 시계열 분석 연구에 중요한 기초가 될 것으로 보인다.
