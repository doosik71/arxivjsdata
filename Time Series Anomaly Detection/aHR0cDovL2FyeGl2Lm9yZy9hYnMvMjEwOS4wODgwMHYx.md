# A ROBUST AND EFFICIENT MULTI-SCALE SEASONAL-TREND DECOMPOSITION

Linxiao Yang, Qingsong Wen, Bo Yang, Liang Sun (2021)

## 🧩 Problem to Solve

실제 세계의 시계열 데이터는 서로 다른 길이를 가진 여러 개의 계절성(Multiple Seasonality)을 보이는 경우가 많다. 예를 들어, 트래픽 데이터는 일일(Daily) 주기와 주간(Weekly) 주기를 동시에 가질 수 있다. 이러한 계절성 성분을 제거하는 Seasonal-Trend Decomposition은 시계열 예측(Forecasting)이나 이상치 탐지(Anomaly Detection)와 같은 다양한 응용 분야에서 매우 중요한 전처리 단계이다.

그러나 기존의 분해 알고리즘들은 여러 계절성 성분이 존재할 때, 특히 주기(Periodic length)가 길어질수록 막대한 계산 비용이 발생하며 분석을 위해 매우 많은 양의 데이터를 요구한다는 한계가 있다. 본 논문의 목표는 다중 계절성을 가진 시계열 데이터를 위해 계산 효율적이면서도 강건한(Robust) 멀티스케일(Multi-scale) 계절성-추세 분해 알고리즘을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **서로 다른 해상도(Resolution)에서 성분을 모델링하는 멀티스케일 접근법**을 계절성-추세 분해에 최초로 적용한 것이다.

핵심 직관은 짧은 주기 성분(Short seasonal component)은 고해상도 데이터에서 정밀하게 표현되어야 하지만, 긴 주기 성분(Long seasonal component)은 저해상도 데이터만으로도 충분히 근사할 수 있다는 점이다. 따라서 저해상도에서 먼저 대략적인 추세와 긴 주기 성분을 추정한 뒤, 이를 가이드로 삼아 고해상도에서 최적화 문제를 통해 정밀한 성분을 복구하는 방식을 취한다. 이를 통해 필요한 데이터의 양을 획기적으로 줄이면서도 계산 효율성을 극대화한다.

## 📎 Related Works

논문에서는 MSTL, STR, TBATS, RobustSTL과 같은 기존의 계절성-추세 분해 알고리즘을 언급한다. 이러한 방법들은 일반적으로 원본 데이터를 사용하여 각 성분을 반복적으로 추정하는데, 이로 인해 주기 길이가 길어질수록 계산 비용이 급격히 증가하는 문제가 있다.

반면, 최근 시계열 저장, 쿼리 처리, 로컬 패턴 발견 등을 위해 멀티스케일 방법들이 제안되었으나, 이를 계절성-추세 분해 분야에 적용한 연구는 본 논문이 처음이다. 본 연구는 기존의 단일 스케일 반복 추정 방식에서 벗어나, 해상도 간의 관계를 최적화 문제로 정의함으로써 효율성을 확보하여 기존 방식과 차별화된다.

## 🛠️ Methodology

### 전체 파이프라인

본 알고리즘은 크게 두 단계로 구성된다. 첫째, 저해상도 시계열에서 대략적인 성분을 추출하고, 둘째, 이 결과를 바탕으로 고해상도 시계열의 성분을 복구하는 최적화 문제를 해결한다.

### 1. 저해상도 시계열 분해 (Low-Resolution Decomposition)

입력 시계열 $\{y_t\}$는 추세 $\tau_t$, 짧은 주기 성분 $s_{d,t}$, 긴 주기 성분 $s_{w,t}$, 그리고 나머지 성분 $r_t$의 합으로 정의된다:
$$y_t = \tau_t + s_{d,t} + s_{w,t} + r_t$$

저해상도 데이터 $\hat{y}^l_t$를 얻기 위해 다운샘플링 팩터 $N$을 사용하여 데이터를 집계한다. 저해상도 데이터에서도 여전히 두 가지 계절성이 존재하므로, 짧은 주기 길이 $T^l_d$를 이용한 계절성 차분(Seasonal Difference)을 계산하여 짧은 주기 성분을 제거한다:
$$g^l_t = \nabla_{T^l_d} \hat{y}^l_t = \hat{y}^l_{t+T^l_d} - \hat{y}^l_t = \nabla_{T^l_d} \tau^l_t + \nabla_{T^l_d} s^l_{w,t} + \tilde{r}^l_t$$

이렇게 변환된 $g^l_t$는 단일 계절성 시계열이 되므로, RobustSTL 알고리즘을 직접 적용하여 저해상도에서의 추세 $\tau^l$과 긴 주기 성분 $s^l_w$의 대략적인 추정치를 얻는다.

### 2. 고해상도 시계열 분해 (High-Resolution Decomposition)

저해상도에서 얻은 결과를 바탕으로 고해상도 성분을 복구하기 위해 다음과 같은 목적 함수를 최소화하는 최적화 문제를 정의한다:
$$\min \|g_h - \hat{D}_{T_d} x\|_1 + \lambda_1 \|z - Bx\|_2^2 + \lambda_2 \|\hat{D}x\|_1 + \lambda_3 \|\hat{D}^2x\|_1$$

각 항의 의미는 다음과 같다:

- $\|g_h - \hat{D}_{T_d} x\|_1$: 실제 고해상도 차분 데이터 $g_h$와 모델 간의 경험적 손실(Empirical loss)을 최소화한다.
- $\lambda_1 \|z - Bx\|_2^2$: 고해상도 분해 결과가 저해상도에서 얻은 결과 $z$와 일관성을 유지하도록 강제한다.
- $\lambda_2 \|\hat{D}x\|_1, \lambda_3 \|\hat{D}^2x\|_1$: 추세와 긴 주기 성분이 국소적으로 매끄럽게(Smooth) 유지되도록 1차 및 2차 차분을 이용한 정규화를 적용한다.

여기서 $x = [(\tau_h)^T, (s_{h,w})^T]^T$는 우리가 찾고자 하는 고해상도의 추세와 긴 주기 성분이다.

### 3. 효율적인 구현: ADMM

위의 비볼록(non-smooth) 최적화 문제를 해결하기 위해 ADMM(Alternating Direction Multiplier Method)을 사용한다. 보조 변수 $\bar{p}, p', p''$를 도입하여 문제를 분리하고, 다음과 같은 절차로 업데이트한다:

1. **보조 변수 업데이트**: Soft-thresholding 연산자 $S(x, \rho) = \text{sign}(x) \max(|x|-\rho, 0)$를 사용하여 업데이트한다.
2. **변수 $x$ 업데이트**: 행렬 $Q$의 역행렬을 이용하여 $x = Q^{-1}h$ 형태로 계산한다. $Q$ 행렬의 역행렬은 오프라인에서 미리 계산 가능하므로, 실제 계산 복잡도는 행렬-벡터 연산인 $O(T_r^2)$ 수준으로 낮아진다.
3. **라그랑주 승수 업데이트**: 경사 상승법(Gradient Ascent)을 통해 업데이트한다.

최종적으로 $\tau_h$와 $s_{h,w}$가 구해지면, 원본 데이터에서 이를 뺀 후 Non-local seasonal filtering을 통해 짧은 주기 성분 $s_{d,t}$를 추출한다.

## 📊 Results

### 실험 설정

- **합성 데이터**: 길이 30,240의 시계열을 생성하였으며, 주기 길이는 각각 1,440과 10,080으로 설정하였다. 추세의 갑작스러운 변화(Abrupt changes)와 이상치(Outliers)를 추가하였으며, $N=60$으로 다운샘플링하였다.
- **실제 데이터**: Yahoo! A1 데이터셋(총 길이 504, 주기 24 및 168)을 사용하였다.
- **비교 대상**: STL, RobustSTL.
- **지표**: 평균 제곱 오차(MSE), 실행 시간(Running Time).

### 정량적 결과 (합성 데이터)

| Method | Trend MSE | Seasonality MSE | Running Time (s) |
| :--- | :--- | :--- | :--- |
| STL | 3.1012 | 1.7916 | 275.6 |
| RobustSTL | 0.1090 | 0.0993 | 138.23 |
| **Proposed** | **0.0017** | **0.0019** | **17.4** |

제안 방법은 MSE 측면에서 가장 낮은 오차를 보였으며, 특히 실행 시간을 RobustSTL 대비 약 8배, STL 대비 약 15배 이상 단축시켰다.

### 정성적 결과

- **합성 데이터**: 제안 방법은 짧은 주기 성분을 정확하게 분리하였으며, 추세의 급격한 변화와 이상치를 효과적으로 캡처하였다.
- **실제 데이터**: RobustSTL과 마찬가지로 이상치를 잘 잡아냈으나, 분해된 나머지 성분(Remainder)에 포함된 노이즈가 RobustSTL보다 적어 더 깨끗한 결과를 보였다.

## 🧠 Insights & Discussion

본 논문은 멀티스케일 구조를 통해 계산 복잡도를 획기적으로 줄이면서도 정확도를 유지했다는 점에서 강점이 있다. 특히 고해상도 데이터를 전부 사용하는 대신, 저해상도 결과를 사전 정보(Prior)로 활용하여 최적화 문제를 푸는 방식은 대규모 시계열 시스템에서 매우 실용적인 접근이다.

다만, 본 논문에서는 두 개의 계절성 성분을 가정하고 분석을 진행하였다. 비록 알고리즘을 반복 적용하여 확장 가능하다고 언급하였으나, 성분의 개수가 늘어날 때의 복잡도 증가나 수렴 속도에 대한 구체적인 분석은 제시되지 않았다. 또한, 추세가 국소적으로 매끄럽다는 가정과 긴 주기 성분이 천천히 변한다는 가정이 전제되어 있으므로, 매우 변동성이 큰 시계열 데이터에서는 성능이 저하될 가능성이 있다.

## 📌 TL;DR

이 논문은 다중 계절성을 가진 시계열 분해의 계산 비용 문제를 해결하기 위해, 저해상도에서 대략적인 성분을 먼저 찾고 이를 이용해 고해상도 성분을 복구하는 **멀티스케일 분해 알고리즘**을 제안한다. ADMM 기반의 최적화 기법을 통해 계산 효율성을 극대화하였으며, 실험 결과 기존 RobustSTL보다 훨씬 빠른 속도와 높은 정확도를 달성하였다. 이 연구는 대용량 IoT 데이터나 시스템 모니터링 트래픽과 같이 주기성이 길고 복잡한 시계열 데이터를 실시간으로 분석해야 하는 환경에 중요한 기여를 할 수 있다.
