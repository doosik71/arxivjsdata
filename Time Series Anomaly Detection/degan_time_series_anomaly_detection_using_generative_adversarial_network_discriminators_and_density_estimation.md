# DEGAN: Time Series Anomaly Detection Using Generative Adversarial Network Discriminators and Density Estimation

Yueyan Gu, Farrokh Jazizadeh (2022)

## 🧩 Problem to Solve

본 논문은 산업 시스템의 서비스 품질 유지와 조기 경보를 위한 효율적인 시계열 이상치 탐지(Time Series Anomaly Detection) 방법론을 제안한다. 시계열 데이터는 다양한 프로세스의 동적 특성을 측정하고 분석하는 보편적인 형태이지만, 복잡한 시간적 의존성(Temporal Dependence)과 확률적 특성(Stochastic Nature)으로 인해 이상치 탐지가 어렵다.

특히, 산업 현장에서 이상치 데이터에 대한 레이블(Label)을 확보하는 것은 매우 어렵고 비용이 많이 들기 때문에, 레이블이 없는 데이터를 사용하는 비지도 학습(Unsupervised Learning) 방식의 중요성이 크다. 본 연구의 목표는 정상 데이터만을 사용하여 훈련된 Generative Adversarial Network(GAN)의 판별자(Discriminator, $D$)를 독립적인 이상치 예측기로 활용하는 비지도 학습 프레임워크인 DEGAN을 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 GAN의 생성자($G$)와 판별자($D$)를 함께 훈련시키되, 최종적으로는 잘 훈련된 **독립적인 판별자($D$) 모델만을 추출하여 이상치 탐지기로 사용**한다는 점이다. 기존의 GAN 기반 이상치 탐지 연구들이 주로 생성자의 재구성 손실(Reconstruction Loss)이나 $G$와 $D$의 손실 합계를 이상치 점수로 활용한 것과 달리, DEGAN은 판별자의 분류 능력 자체에 집중한다.

또한, 판별자가 식별한 의심 지점들의 불확실성을 해결하기 위해 **커널 밀도 추정(Kernel Density Estimation, KDE)**을 도입하여 확률 밀도 함수를 생성하고, 이를 통해 실제 이상치의 위치를 정교하게 식별하는 파이프라인을 구축하였다.

## 📎 Related Works

시계열 이상치 탐지 방법론은 크게 통계적 접근법(ARIMA 등), 전통적 머신러닝(k-means, Isolation Forest, OC-SVM), 딥러닝(MLP, CNN, LSTM), 그리고 생성 모델(Autoencoder, GAN)로 구분된다.

기존의 GAN 기반 접근 방식은 크게 두 가지 방향으로 진행되었다. 첫째, 이상치 데이터를 오버샘플링하여 불균형 문제를 해결하는 지도 학습 방식이다. 둘째, $G$와 $D$의 손실 함수를 결합하여 이상치 점수(Anomaly Score)를 정의하는 비지도 학습 방식이다. 하지만 이러한 방식들은 대부분 생성자($G$)의 역할에 의존하거나 두 모델을 동시에 사용한다. 본 논문은 독립적인 판별자($D$)의 잠재력을 활용한다는 점에서 기존 연구들과 차별화된다.

## 🛠️ Methodology

### 전체 파이프라인

DEGAN의 전체 프로세스는 **데이터 전처리 $\rightarrow$ GAN 훈련 $\rightarrow$ $D$ 모델 선택 $\rightarrow$ 확률적 이상치 탐지**의 단계로 구성된다.

### 1. 데이터 전처리

시계열 데이터 $\text{TS}$를 슬라이딩 윈도우(Sliding Window) 방식을 통해 하위 시퀀스(Subsequence)로 추출하여 2D 텐서 형태로 변환한다. 윈도우 길이를 $wl$이라 할 때, 특징 행렬 $W$는 다음과 같이 정의된다.

$$W := (W_1, W_2, \dots, W_N)^T = ((x_1, \dots, x_{wl}), (x_2, \dots, x_{wl+1}), \dots, (x_N, \dots, x_{N+wl-1}))^T$$

추출된 각 시퀀스는 제로 평균 정규화(Zero-mean normalization)를 거쳐 학습에 사용된다.

### 2. GAN 아키텍처 및 훈련

DEGAN은 생성자($G$)와 판별자($D$)가 서로 경쟁하는 min-max 게임을 통해 학습된다.

$$\min_G \max_D V(D, G) = \mathbb{E}_{x \sim p_{\text{data}}(x)} [\log D(x)] + \mathbb{E}_{z \sim p_z(z)} [\log(1 - D(G(z)))]$$

- **생성자($G$):** 가우시안 분포에서 추출된 128차원의 랜덤 신호를 입력받아 정상 시퀀스를 생성하는 2층의 Dense 네트워크 구조이다.
- **판별자($D$):** 입력 데이터가 실제(Real)인지 가짜(Fake)인지를 판별한다. 본 논문에서는 1D-Convolutional 레이어 하나와 두 개의 Dense 레이어로 구성된 **CNN-D** 아키텍처를 기본으로 사용한다.

### 3. $D$ 모델 선택 (Model Selection)

레이블이 없는 상황에서 최적의 $D$ 모델을 선택하기 위해 **N/A ratio (Normal/Abnormal ratio)** 지표를 제안한다. 이는 깨끗한 검증 데이터셋($\text{TS}_B$)에서 $D$가 얼마나 많은 데이터를 정상으로 예측하는지를 측정하는 지표이다.

$$\text{N/A} = \frac{\text{# Predicted Normal Points}}{\text{# Predicted Anomalous Points}}$$

이상적으로 $D$는 정상 데이터셋에서 이상치를 탐지하지 않아야 하므로, N/A ratio가 높을수록 성능이 좋다고 판단한다. 다만, 모든 데이터를 정상으로 예측하여 N/A가 무한대가 되는 경우는 너무 포괄적인 모델로 간주하여 제외하며, 무한대가 아니면서 가장 높은 N/A ratio를 보이는 지점의 모델을 최종 선택한다.

### 4. 확률적 이상치 탐지 및 KDE

선택된 $D$ 모델이 점수를 0.5보다 크게 출력하면 해당 윈도우를 이상치 후보로 간주한다. 하지만 슬라이딩 윈도우 특성상 하나의 이상치가 여러 개의 연속적인 후보 지점을 생성하므로, 정확한 위치를 특정하기 위해 커널 밀도 추정(KDE)을 적용한다.

$$\hat{f}_h(x) = \frac{1}{nh} \sum_{i=1}^n K \left( \frac{x - x_i}{h} \right)$$

여기서 $K$는 가우시안 커널 함수이며, $h$는 평활화 정도를 결정하는 대역폭(Bandwidth)이다. 생성된 확률 밀도 함수에서 피크 선택 알고리즘(Peak Selection Algorithm)을 통해 국소 최댓값을 찾아 최종 이상치 위치로 확정한다.

## 📊 Results

### 실험 설정

- **데이터셋:** 미국 Class I 철도 선로의 5마일 구간에서 수집된 수직 가속도(Vertical Acceleration) 시계열 데이터.
- **평가 지표:** 실제 운영자가 레이블링한 이상치와 비교하여 Recall, Precision, F1-score를 측정한다.
- **허용 오차($r_t$):** 실제 이상치 위치 $d$와 예측 위치 $p_{\text{closest}}$ 사이의 거리가 $r_t$ 이내이면 True Positive(TP)로 간주한다.

### 주요 결과

1. **윈도우 크기의 영향:** 윈도우 크기가 100일 때 가장 높은 Precision(0.86)과 F1-score(0.83)를 보였으며, 200일 때 가장 높은 Recall(0.88)을 기록하였다.
2. **아키텍처 비교:** $D$ 모델의 구조 중 **CNN-D**가 Dense-D나 LSTM-D보다 우수한 성능을 보였다. 특히 LSTM-D는 본 케이스 스터디에서 낮은 성능을 보였는데, 이는 이상치가 문맥적 의존성보다는 국소적인 패턴 변화에 기인했기 때문으로 분석된다.
3. **모델 선택 방식:** 레이블이 없는 데이터를 이용한 비지도 모델 선택 방식이 레이블을 이용한 지도 방식과 비교했을 때 F1-score 면에서 유사한 성능을 보였다. 비지도 방식은 Recall이 높고, 지도 방식은 Precision이 높은 경향을 보였다.
4. **클러스터링 효과:** k-means 클러스터링을 통해 모티프(Motif)별로 GAN을 훈련시키는 방법은 윈도우 크기가 클 때는 일부 도움이 되었으나, 작은 윈도우 크기에서는 오히려 성능이 저하되었다.

## 🧠 Insights & Discussion

본 논문은 GAN의 판별자($D$)가 정상 데이터의 분포를 학습함으로써, 학습 과정에서 보지 못한 패턴(이상치)을 효과적으로 구분해낼 수 있음을 입증하였다. 특히 KDE를 결합하여 슬라이딩 윈도우로 인한 위치 불확실성을 해결한 점이 실용적인 기여라고 평가할 수 있다.

**강점 및 한계:**

- **강점:** 레이블이 전혀 없는 환경에서도 정상 데이터만으로 신뢰할 수 있는 이상치 탐지기를 구축할 수 있으며, 특히 CNN 기반 판별자가 시계열의 국소적 특징을 잘 잡아낸다.
- **한계:** 모델의 성능이 하이퍼파라미터(학습률, 윈도우 크기, KDE 대역폭 등)와 훈련 에포크(Epoch)에 매우 민감하게 반응한다. 또한, 단일 채널 가속도 데이터만 사용했기에 다변량 시계열 데이터로의 확장 가능성에 대한 검증이 추가로 필요하다.

**비판적 해석:**
제안된 N/A ratio 기반의 모델 선택 방식은 매우 단순하지만 효과적이다. 그러나 이는 검증 데이터셋($\text{TS}_B$)이 훈련 데이터셋($\text{TS}_A$)과 통계적으로 매우 유사하다는 가정하에 성립한다. 만약 데이터의 분포가 시간에 따라 변하는 Concept Drift가 발생한다면, 이 지표의 신뢰도는 떨어질 가능성이 크다.

## 📌 TL;DR

본 논문은 정상 시계열 데이터만으로 학습된 **GAN의 판별자($D$)를 독립적인 이상치 탐지기로 활용하는 DEGAN 프레임워크**를 제안한다. 슬라이딩 윈도우 전처리와 CNN 기반 판별자, 그리고 최종 위치 정밀화를 위한 KDE를 결합하여, 철도 선로 가속도 데이터에서 높은 Recall(80%)과 Precision(86%)을 달성하였다. 이 연구는 레이블이 부족한 산업 현장에서 비지도 학습 기반의 시계열 이상치 탐지 시스템을 구축하는 데 중요한 방법론적 근거를 제공한다.
