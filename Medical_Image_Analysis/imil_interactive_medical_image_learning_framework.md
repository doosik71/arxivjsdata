# IMIL: Interactive Medical Image Learning Framework

Adrit Rao, Andrea Fisher, Ken Chang, John Christopher Panagides, Katherine McNamara, Joon-Young Lee, Oliver Aalami (2024)

## 🧩 Problem to Solve

의료 영상 분석을 위한 딥러닝 모델 학습 시, 데이터 부족 문제를 해결하기 위해 다양한 데이터 증강(Data Augmentation) 기법이 사용된다. 그러나 CutOut과 같은 일반적인 증강 기법은 무작위로 이미지 영역을 제거하기 때문에, 임상적으로 매우 중요한 정보(Clinically relevant information)까지 삭제할 위험이 있다. 이는 모델이 잘못된 시각적 특징에 집중하게 만들어 추론 시 오답을 내놓거나, 임상적 근거가 없는 엉뚱한 영역을 근거로 판단하는 'Shortcut learning' 문제를 야기한다.

본 논문의 목표는 임상 전문가의 피드백을 학습 과정에 직접 통합하여, 모델이 임상적으로 유의미한 영역에 집중하도록 유도하는 **Interactive Medical Image Learning (IMIL)** 프레임워크를 제안하는 것이다. 이를 통해 데이터 증강의 안전성을 높이고, 적은 양의 데이터로도 모델의 성능과 신뢰도(Calibration)를 향상시키고자 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 학습 중간 단계에서 모델이 가장 어려워하는 샘플(Misprediction outliers)을 추출하고, 이에 대해 전문의가 직접 '집중해야 할 영역'을 지정하게 하여 데이터를 증강하는 것이다.

중심적인 설계 직관은 다음과 같다.

- **전문가 가이드 기반 증강**: 무작위 제거가 아닌, 전문의가 지정한 핵심 영역만 남기고 나머지를 제거하는 'Blackout' 증강을 수행함으로써 모델의 시각적 주의(Attention)를 강제로 교정한다.
- **중간 학습 단계 개입 (Intermediate Training)**: 학습 초기부터 증강을 적용하는 것이 아니라, 모델이 어느 정도 학습된 후(예: 70 epoch) 발생하는 고신뢰도 오답 샘플에 개입하여 효율성을 극대화한다.
- **낮은 어노테이션 부담**: 전체 데이터셋이 아닌, 소수의 고신뢰도 오답 샘플(본 연구에서는 20개)에 대해서만 피드백을 받으므로 전문의의 시간 소모를 최소화한다.

## 📎 Related Works

- **의료 영상 신뢰도 교정 (Medical Confidence Calibration)**: 표준 CNN은 과잉 확신(Overconfidence) 경향이 있어 신뢰도 추정치가 부정확한 경우가 많다. MixUp, CutMix, CutOut 등의 현대적 증강 기법이 정규화 및 교정 효과가 있다고 알려져 있으나, 의료 영상의 특수성(임상적 유의 영역)을 고려하지 않은 무작위성은 오히려 독이 될 수 있다.
- **시각적 설명 가능성 (Visual Explainability)**: 많은 모델이 높은 정확도를 보이더라도, 실제로는 병변이 아닌 텍스트 표시나 주변부 조직 같은 무관한 특징(Spurious correlations)을 통해 예측하는 문제가 보고되었다.
- **Human-in-the-loop Training**: 능동 학습(Active Learning)과 같이 전문가가 레이블링에 참여하는 연구는 많았으나, 학습 중간에 증강 전략을 수정하기 위해 전문가의 피드백을 반영하는 기법은 이전 연구에서 확인되지 않았다.

## 🛠️ Methodology

### 전체 파이프라인

IMIL은 CNN 학습 과정 중에 호출되는 콜백(Callback) 모듈 형태로 작동하며, 전체 흐름은 다음과 같다:

1. **오답 아웃라이어 선택**: 설정된 `IMIL_epoch`에 도달하면 전체 학습셋에 대해 예측을 수행한다. 정답과 다른 예측값 중 신뢰도(Confidence)가 가장 높은 샘플들을 $\text{num\_outliers}$만큼 추출한다. 이는 모델이 강하게 확신하면서도 틀린, 즉 '가장 교정이 시급한' 사례에 집중하기 위함이다.
2. **사용자 해석**: 선택된 아웃라이어들에 대해 원본 이미지, Grad-CAM 히트맵, 모델의 예측 레이블 및 신뢰도, 실제 정답(Ground-truth)을 전문의에게 제공한다.
3. **로컬라이제이션 그리드 피드백**: 이미지 위에 $4 \times 4$ 등의 그리드(Grid)를 덮어씌우고, 전문의가 모델이 집중해야 할 임상적 유의 영역에 해당하는 그리드 칸을 선택한다.
4. **Blackout 증강 적용**: 전문의가 선택한 영역만 남기고 나머지 영역은 모두 검은색으로 지우는 'Blackout' 처리를 한다.
5. **재학습**: 원본 이미지를 이 증강된 이미지로 교체한 후, 남은 에포크 동안 학습을 계속 진행하여 모델이 올바른 특징과 레이블 간의 관계를 재학습하게 한다.

### 비교 대상 증강 기법의 수식

IMIL의 효과를 검증하기 위해 사용된 기존 기법들의 수식은 다음과 같다.

- **MixUp**: 두 이미지 $x_i, x_j$와 레이블 $y_i, y_j$를 볼록 조합(Convex combination)으로 섞는다.
  $$\tilde{x} = \lambda x_i + (1-\lambda) x_j, \quad \tilde{y} = \lambda y_i + (1-\lambda) y_j$$
  여기서 $\lambda$는 $\text{Beta}(\alpha, \alpha)$ 분포에서 샘플링된 값이다.
- **CutMix**: 이미지의 일부 패치를 제거하고 다른 이미지의 패치로 대체한다.
  $$\tilde{x} = M x_i + (1-M) x_j, \quad \tilde{y} = \mu y_i + (1-\mu) y_j$$
  여기서 $M$은 바이너리 마스크이며, $\mu$는 마스크의 면적 비율을 나타낸다.
- **CutOut**: 이미지의 무작위 위치에 고정된 크기의 제로 마스크(Zero-masking)를 적용하여 해당 영역을 완전히 가린다.

### 평가 지표: Expected Calibration Error (ECE)

모델의 신뢰도 교정 수준을 측정하기 위해 ECE를 사용한다.
$$\text{ECE} = \sum_{b=1}^B \frac{n_b}{N} |\text{acc}(b) - \text{conf}(b)|$$
여기서 $B$는 빈(Bin)의 개수이며, $\text{acc}(b)$는 해당 빈의 정확도, $\text{conf}(b)$는 평균 신뢰도이다. ECE 값이 낮을수록 모델의 확신 정도가 실제 정확도와 일치함을 의미한다.

## 📊 Results

### 실험 설정

- **데이터셋**: NIH Shenzhen No. 3 People's Hospital의 결핵(TB) 흉부 X-ray(CXR) 데이터셋 (총 662장, 학습 530장/테스트 132장).
- **모델**: ResNet-50 기반 전이 학습.
- **IMIL 설정**: 70 epoch에서 콜백 실행, $4 \times 4$ 그리드 사용, 20개의 아웃라이어 추출.
- **피드백 제공자**: 스탠퍼드 의과대학의 방사선과 레지던트 3명.

### 정량적 결과

| 모델 | 정확도(Accuracy) | AUROC | ECE |
| :--- | :---: | :---: | :---: |
| Baseline (ResNet-50) | 0.804 | 0.832 | 0.2 |
| CutOut | 0.797 ($\downarrow$) | 0.817 ($\downarrow$) | 0.201 ($\approx$) |
| MixUp | 0.834 ($\uparrow$) | **0.878 ($\uparrow$)** | 0.161 ($\downarrow$) |
| CutMix | 0.812 ($\uparrow$) | 0.853 ($\uparrow$) | **0.15 ($\downarrow$)** |
| **IMIL (Resident 1)** | **0.846 ($\uparrow \uparrow$)** | 0.857 ($\uparrow$) | 0.179 ($\downarrow$) |

- **성능**: IMIL(Resident 1)은 baseline 대비 정확도를 4.2% 향상시켜 모든 모델 중 가장 높은 정확도를 기록하였다.
- **신뢰도**: IMIL은 baseline 및 CutOut보다 낮은 ECE를 보여 신뢰도 교정 성능이 개선되었음을 확인하였다. 특히 CutOut은 성능과 교정 모두에서 baseline보다 하락하거나 정체된 모습을 보였다.

### 정성적 결과 및 시각화

- **CAM 분석**: Grad-CAM 결과, CutOut은 어깨나 흉부 연조직 등 무관한 영역에 주의를 기울이는 경향이 있었으나, IMIL은 폐 영역과 폐문 림프절(Hilar lymph nodes) 등 임상적으로 중요한 영역에 정확히 집중하는 모습을 보였다.
- **전문의 피드백**: 전문의들은 "모델이 폐 주변부의 연조직에 집중하고 있어, 폐 실질과 림프절 쪽으로 포커스를 옮겼다"는 식의 구체적인 근거를 바탕으로 피드백을 제공하였으며, 이것이 실제 모델의 성능 향상으로 이어졌다.

## 🧠 Insights & Discussion

### 강점 및 해석

본 연구는 무작위 데이터 증강이 의료 영상에서 가질 수 있는 위험성을 지적하고, 이를 '인간-인-더-루프(Human-in-the-loop)' 방식으로 해결할 수 있음을 입증하였다. 특히 전체 데이터의 약 3.77%에 불과한 20장의 이미지만으로도 유의미한 성능 향상을 이끌어냈다는 점은 전문의의 업무 부담을 최소화하면서도 모델의 품질을 높일 수 있는 실용적인 접근법임을 시사한다.

### 한계 및 미해결 질문

- **데이터셋 규모**: 상대적으로 작은 규모의 데이터셋에서 검증되었으므로, 대규모 데이터셋으로 확장했을 때 전문가 피드백의 최적 비율(Outlier-to-dataset ratio)이 얼마인지에 대한 연구가 필요하다.
- **단일 시점 개입**: 본 실험에서는 70 epoch라는 단일 시점에서만 콜백을 실행하였으나, 학습 과정 중 여러 번 개입하는 것이 더 효과적일 가능성이 있다.
- **전문가 간 편차**: 레지던트별로 정확도 결과에 차이가 있었는데, 이는 전문가 간의 판단 차이가 모델 성능에 직접적인 영향을 미칠 수 있음을 의미한다. 다수의 전문가 피드백을 어떻게 통합할 것인지에 대한 논의가 필요하다.

### 비판적 해석

IMIL은 CutOut의 '제거' 개념을 차용했으나, 이를 전문가의 지식으로 제어함으로써 CutOut의 단점(중요 정보 소실)을 극복하고 장점(정규화 효과)을 취했다. 다만, MixUp이나 CutMix가 AUROC나 ECE 면에서 여전히 강력한 성능을 보인다는 점을 고려할 때, 단순 'Blackout' 방식이 아니라 MixUp/CutMix의 메커니즘에 전문가의 가이드를 결합한다면 더욱 비약적인 성능 향상이 가능할 것으로 보인다.

## 📌 TL;DR

본 논문은 의료 영상 학습 시 무작위 증강으로 인한 임상 정보 손실 문제를 해결하기 위해, 학습 중간에 전문의가 오답 샘플의 핵심 영역을 지정하고 나머지를 제거하는 **IMIL(Interactive Medical Image Learning)** 프레임워크를 제안한다. 결핵 진단 실험 결과, 단 20장의 이미지에 대한 전문가 피드백만으로 정확도를 4.2% 향상시키고 모델의 시각적 주의를 임상적 유의 영역으로 교정하는 데 성공하였다. 이 연구는 데이터가 부족한 의료 AI 분야에서 전문가의 지식을 효율적으로 주입하여 모델의 성능과 신뢰도를 동시에 높이는 실용적인 가이드라인을 제시한다.
