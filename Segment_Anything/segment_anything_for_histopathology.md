# Segment Anything for Histopathology

Titus Griebel, Anwai Archit, Constantin Pape (2025)

## 🧩 Problem to Solve

디지털 병리학(Digital Pathology)에서 핵 분할(Nucleus Segmentation)은 암 등급 결정, 종양 미세환경 분석, 생존율 예측 등을 위해 필수적인 분석 작업이다. 그러나 기존의 자동 분할 방법들은 데이터 분포가 다른 새로운 데이터셋에 적용했을 때 성능이 급격히 떨어지는 도메인 시프트(Domain Shift) 문제에 취약하다. 이를 해결하기 위해서는 사용자가 새로운 데이터에 대해 직접 핵을 주석(Annotation) 처리하고 모델을 재학습시켜야 하는데, 이 과정은 전문 지식이 필요하며 매우 많은 시간과 자원이 소모되는 지루한 작업이다.

최근 자연어 이미지 분야에서 Segment Anything Model (SAM)과 같은 시각 기반 파운데이션 모델(Vision Foundation Models, VFMs)이 등장하여 강건한 분할 성능을 보여주고 있으나, 조직병리학 이미지의 특수성으로 인해 기존 SAM을 그대로 적용하는 데에는 한계가 있다. 따라서 본 논문의 목표는 다양한 조직병리학 데이터셋을 통해 학습되어, 자동 분할과 인터랙티브 분할 모두에서 뛰어난 성능을 보이는 핵 분할 전용 파운데이션 모델인 **PathoSAM**을 개발하는 것이다.

## ✨ Key Contributions

본 연구의 핵심 기여는 SAM을 기반으로 조직병리학의 특성에 맞게 미세 조정(Finetuning)하여, 특정 데이터셋에 국한되지 않고 범용적으로 사용할 수 있는 핵 분할 파운데이션 모델인 PathoSAM을 제안한 것이다.

중심적인 설계 아이디어는 다음과 같다.

1. **범용적 학습(Generalist Training):** 6개의 다양한 H&E 염색 데이터셋을 통합하여 학습함으로써, 학습에 사용되지 않은 외부 데이터(Out-of-domain)에서도 높은 일반화 성능을 갖도록 설계하였다.
2. **자동 및 인터랙티브 분할의 결합:** $\mu$SAM의 구조를 채택하여 사용자의 프롬프트에 반응하는 인터랙티브 분할과, 별도의 디코더를 통한 효율적인 자동 인스턴스 분할(Automatic Instance Segmentation, AIS)을 동시에 지원한다.
3. **확장성:** 인스턴스 분할뿐만 아니라, 추가적인 디코더 학습을 통해 시맨틱 분할(Semantic Segmentation) 작업으로도 확장 가능함을 보였다.
4. **실용성 제공:** QuPath 및 $\mu$SAM과 같은 기존 주석 도구와의 호환성을 확보하고, 전체 슬라이드 이미지(Whole-Slide Image, WSI) 처리를 위한 스크립트를 제공하여 실제 의료 현장에서의 활용도를 높였다.

## 📎 Related Works

기존의 핵 분할 접근 방식은 주로 UNet 계열의 지도 학습(Supervised Learning)에 의존해 왔다. StarDist나 HoVerNet과 같은 방법들은 핵의 겹침(Overlapping) 문제나 분류 작업을 동시에 처리하는 등 도메인 특화된 해결책을 제시하였으며, 최근의 HoVerNeXt나 InstanSeg는 정확도를 유지하면서 효율성을 높이는 데 집중하였다.

그러나 이러한 모델들은 앞서 언급한 일반화 성능 부족 문제가 치명적이다. SAM을 의료 영상에 적용하려는 시도(예: CellViT)가 있었으나, CellViT는 특정 데이터셋(PanNuke)에만 학습되어 범용성이 부족하며, 오직 자동 분할만을 지원하여 사용자가 직접 수정하는 인터랙티브 분할 기능을 제공하지 않는다. BioMedParse와 같은 모델은 텍스트 프롬프트 기반의 시맨틱 분할을 지원하지만, 개별 객체를 구분하는 인스턴스 분할 기능이 결여되어 있다. PathoSAM은 이러한 한계들을 극복하여 인터랙티브와 자동 인스턴스 분할을 모두 지원하는 최초의 조직병리학 핵 분할 파운데이션 모델을 지향한다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

PathoSAM은 SAM의 기본 구조(Image Encoder, Prompt Encoder, Mask Decoder)를 유지하면서, 자동 분할을 위한 별도의 디코더를 추가한 형태를 띤다. 이미지 인코더로는 ViT-B, ViT-L, ViT-H 세 가지 크기의 Vision Transformer를 사용하여 각각의 Generalist 모델을 구축하였다.

### 2. 인터랙티브 분할 (Interactive Segmentation)

사용자가 점(Point), 바운딩 박스(Bounding Box), 또는 저해상도 마스크(Mask)와 같은 프롬프트를 제공하면 모델이 해당 객체의 마스크를 예측한다. 예측이 부정확할 경우 사용자가 추가적인 긍정/부정 프롬프트를 제공하여 마스크를 반복적으로 수정할 수 있다.

### 3. 자동 인스턴스 분할 (Automatic Instance Segmentation, AIS)

기존 SAM의 자동 마스크 생성(AMG) 방식(그리드 형태의 점 프롬프트를 배치하는 방식)은 계산 효율이 낮고 품질이 불안정하다. 이를 해결하기 위해 $\mu$SAM의 방식을 도입하여 UNETR 스타일의 별도 디코더를 추가하였다. 이 디코더는 다음 세 가지 변수를 예측한다.

- 전경 확률(Foreground probabilities)
- 가장 가까운 객체 중심까지의 거리(Distance to the closest object center)
- 가장 가까운 객체 경계까지의 거리(Distance to the closest object boundary)

추론 단계에서는 예측된 거리 맵을 바탕으로 **Seeded Watershed** 알고리즘을 적용하여 개별 핵 인스턴스를 분리한다.

### 4. 시맨틱 분할 확장 (Semantic Segmentation)

인스턴스 분할 모델을 기반으로 시맨틱 분할을 수행하기 위해, 클래스별 확률을 출력하는 별도의 컨볼루션 디코더를 추가하였다. 학습 시에는 Cross Entropy Loss를 사용하며, 이미지 인코더를 동결(Freeze)하거나 미세 조정(Finetune)하는 등 네 가지 전략을 실험하였다.

### 5. 학습 절차 및 데이터

- **데이터:** 14개의 공개 데이터셋을 사용하였다. 6개의 H&E 데이터셋으로 Generalist 모델을 학습시키고, 나머지 8개 데이터셋은 외부 데이터 테스트(OOD evaluation)에 사용하였다.
- **학습 목표:** 인터랙티브 분할과 자동 분할을 동시에 최적화하는 $\mu$SAM의 목적 함수를 사용하였다.
- **최적화:** AdamW 옵티마이저를 사용하였으며, 초기 학습률은 $1e-5$로 설정하였다.

## 📊 Results

### 1. 자동 인스턴스 분할 성능

PathoSAM(특히 ViT-L 모델)은 HoVerNet, StarDist, CellViT, HoVerNeXt, InstanSeg 등 기존 SOTA 모델들과 비교하여 전반적으로 가장 높은 성능을 기록하였다.

- **일반화 능력:** 학습에 사용되지 않은 외부 데이터셋(OOD)에서도 매우 강력한 성능을 보였으며, 특히 H&E 외의 IHC-TMA 데이터셋에서도 우수한 결과가 나타났다.
- **효율성:** AIS 방식이 기존의 AMG 방식보다 분할 품질과 계산 효율성 면에서 모두 뛰어남을 확인하였다.

### 2. 인터랙티브 분할 성능

기본 SAM 및 $\mu$SAM과 비교했을 때, 단일 점(Point)이나 박스(Box) 프롬프트만으로 예측한 초기 마스크의 정확도가 훨씬 높았다. 반복적인 수정 과정을 거친 후에는 PathoSAM과 $\mu$SAM 모두 거의 완벽한 분할 성능에 도달하였다.

### 3. 기타 작업 및 실용성 평가

- **특수 구조 분할:** 림프구(Lymphocytes)나 샘(Glands) 분할의 경우, 범용 모델보다는 해당 데이터로 학습된 Specialist 모델이 더 우수하였다. 이는 범용 모델이 모든 핵을 검출하는 반면, Specialist는 특정 클래스만 구분하기 때문이다.
- **시맨틱 분할:** PanNuke 데이터셋에서 CellViT보다는 낮지만, HoverNet이나 BioMedParse보다는 우수한 2위의 성능을 기록하였다.
- **WSI 적용:** 타일 기반 예측(Tile-and-stitch) 방식을 통해 거대한 전체 슬라이드 이미지(WSI)에도 적용 가능하며, GPU 사용 시 1시간 이내에 처리가 가능함을 보였다.

### 4. 평가 지표

인스턴스 분할 성능은 다음과 같은 **Mean Segmentation Accuracy**를 통해 측정하였다.
$$\text{Mean Segmentation Accuracy} = \frac{1}{|\# \text{thresholds}|} \sum_{t} \frac{TP(t)}{TP(t) + FP(t) + FN(t)}$$
여기서 $t$는 $0.5$부터 $0.95$까지의 IoU 임계값이다.

## 🧠 Insights & Discussion

### 강점

본 논문은 조직병리학 분야에서 최초의 핵 분할 파운데이션 모델을 제시하였다는 점에서 의의가 크다. 특히 단일 모델로 다양한 도메인의 데이터를 처리할 수 있는 일반화 능력을 확보하였으며, 인터랙티브 인터페이스를 통해 전문가의 수정 작업을 획기적으로 단축시킬 수 있는 실용적인 도구를 제공하였다.

### 한계 및 비판적 해석

1. **범용성의 범위:** 본 모델은 '핵 분할'에 대해서는 파운데이션 모델이라 할 수 있으나, 샘(Glands)이나 조직 수준의 분할까지 확장된 '조직병리학 전체'의 파운데이션 모델은 아니다. 구조와 크기가 다른 객체들에 대해서는 여전히 Specialist 모델이 필요하다.
2. **시맨틱 분할의 한계:** CellViT에 비해 성능이 낮은 이유는 소수 클래스(예: dead cells)에 대한 오버샘플링이나 손실 함수 밸런싱 전략이 부족했기 때문으로 분석된다.
3. **가정:** 본 연구는 H&E 염색 데이터 위주로 학습되었으며, 다른 염색법에 대한 성능 향상은 주로 SAM 자체의 강건함에 의존하고 있을 가능성이 있다.

## 📌 TL;DR

PathoSAM은 SAM을 기반으로 다양한 조직병리학 데이터셋을 학습시킨 **핵 분할 전용 시각 파운데이션 모델(VFM)**이다. 이 모델은 자동 인스턴스 분할(AIS)과 인터랙티브 분할 모두에서 기존 SOTA 성능을 경신하였으며, 특히 새로운 데이터셋에 대한 일반화 능력이 매우 뛰어나다. QuPath 등 기존 도구와 통합되어 실제 의료 데이터 주석 처리 시간을 크게 단축할 수 있으며, 향후 SAM 2 기반의 확장이나 다중 스케일 분할 모델로의 발전 가능성이 높다.
