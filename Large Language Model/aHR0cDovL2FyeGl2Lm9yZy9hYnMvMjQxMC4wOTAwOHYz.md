# SUPERCORRECT: Advancing Small LLM Reasoning with Thought Template Distillation and Self-Correction

Ling Yang, Zhaochen Yu, Tianjun Zhang, Minkai Xu, Joseph E. Gonzalez, Bin Cui, Shuicheng Yan (2025)

## 🧩 Problem to Solve

본 논문은 소형 거대언어모델(Small LLMs)이 복잡한 수학적 추론 작업에서 겪는 한계를 해결하고자 한다. GPT-4나 DeepSeek-R1과 같은 대형 모델들은 뛰어난 추론 능력을 보여주지만, 소형 모델들은 여전히 복잡한 수학 문제에서 추론 오류를 스스로 식별하고 수정하는 능력이 부족하다.

기존의 반추(reflection) 기반 방법론들은 모델이 스스로 오류를 찾아내어 수정하도록 유도하지만, 많은 경우 소형 모델은 외부의 도움 없이 독립적으로 추론 단계의 오류를 탐지하는 데 어려움을 겪는다. 특히, 오류 위치가 제공되더라도 이전의 잘못된 추론 맥락에 매몰되어 편향된 결과를 내놓는 경향이 있다. 따라서 본 연구의 목표는 대형 교사 모델(Teacher Model)의 지식을 활용하여 소형 학생 모델(Student Model)의 추론 과정과 반추 과정을 동시에 감독하고 교정함으로써, 소형 모델의 추론 정확도와 자가 수정(self-correction) 능력을 근본적으로 향상시키는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 교사 모델의 고도화된 추론 경로와 교정 흔적(correction traces)을 학생 모델에게 전이시키는 2단계 프레임워크인 SUPERCORRECT를 설계한 것이다.

첫 번째 핵심 기여는 **계층적 사고 템플릿(Hierarchical Thought Template)**의 도입이다. 이는 단순한 생각의 흐름(Chain-of-Thought)을 넘어, 일반화된 해결 전략을 담은 '고수준 사고'와 핵심 추론 단계의 상세 분석을 담은 '세부 솔루션'을 결합하여 학생 모델이 더 정밀한 추론을 수행하도록 유도한다.

두 번째 핵심 기여는 **교차 모델 협업 직접 선호도 최적화(Cross-model Collaborative DPO)** 방법론이다. 단순히 정답과 오답을 구분하는 기존 DPO와 달리, 학생 모델이 생성한 잘못된 자가 수정 시도(rejected)와 교사 모델이 제시한 정확한 교정 경로(chosen)를 쌍으로 구성하여 학습시킨다. 이를 통해 학생 모델은 자신의 사고 병목(thought bottleneck)을 깨고, 오류를 정확히 위치시키고 해결하는 새로운 기술과 지식을 습득하게 된다.

## 📎 Related Works

기존의 LLM 성능 향상 방법은 크게 전통적인 미세 조정(SFT) 및 정렬(RLHF, DPO) 방식과 반추(reflection) 기반 방식으로 나뉜다. SFT와 RLHF는 주로 최종 답변의 정확성이나 단순한 추론 근거에 집중하기 때문에, 추론 과정 중 어디서 오류가 발생했는지 구체적으로 식별하고 수정하는 능력을 기르기 어렵다는 한계가 있다.

반면, Reflexion이나 RCI와 같은 반추 기반 방법들은 모델이 스스로의 출력을 수정하도록 하지만, 외부 피드백 없이는 추론 오류를 찾아내지 못하거나 오히려 정답을 오답으로 바꾸는 문제가 발생한다. 또한 CoT, ToT, GoT와 같은 사고 확장 기법들은 프롬프팅 수준에서의 개선에 치중하여 모델 내부의 근본적인 추론 능력을 증진시키는 데는 한계가 있다. SUPERCORRECT는 이러한 한계를 극복하기 위해 교사 모델의 명시적인 오류 분석과 교정 경로를 학습 데이터로 활용함으로써, 자가 수정의 병목 현상을 해결하고자 한다.

## 🛠️ Methodology

SUPERCORRECT는 다음과 같은 두 단계의 파이프라인으로 구성된다.

### 1단계: 계층적 사고 템플릿을 이용한 지도 미세 조정 (HSFT)

먼저, 교사 모델(o1-preview 등)을 통해 수학 문제에 대한 계층적 사고 템플릿을 추출한다. 이 템플릿은 XML 형식으로 구성되며 다음과 같은 구조를 가진다.
- $\langle Step \rangle$: 단계별 추론 과정.
- $\langle Key \rangle$: 해당 단계가 까다롭거나 중요한 경우, 학생 모델이 이해할 수 있도록 제공하는 상세 설명 및 분석.
- $\langle Generalized \rangle$: 유사한 문제를 해결하는 데 적용할 수 있는 일반화된 해결 전략.
- $\langle Answer \rangle$: 최종 정답.

학습 목표는 학생 모델이 이러한 계층적 구조를 따르며 추론하도록 하는 것이며, 손실 함수는 다음과 같은 표준적인 최대 가능도 추정(MLE) 방식을 따른다.
$$\mathcal{L}_{sft} = \text{argmax} \sum_{(P^{stu}, x, T^{tea}, s^{tea}) \in D_{sft}} \log \pi((T^{tea}, s^{tea}) | (P^{stu}, x))$$
여기서 $P^{stu}$는 프롬프트, $x$는 문제, $T^{tea}$와 $s^{tea}$는 각각 교사 모델의 계층적 사고와 세부 솔루션을 의미한다.

### 2단계: 교차 모델 협업 DPO (Cross-model Collaborative DPO)

1단계에서 학습된 모델 $\pi_{ref}$가 생성한 오답 데이터셋 $D_{err}$을 수집한다. 이후 교사 모델을 통해 각 오답의 첫 번째 오류 지점 $k$를 찾아내고, 이에 대한 오류 원인 분석 $a_k$와 교정 가이드 $c_k$를 생성한다.

이때, **선호 데이터 쌍**을 다음과 같이 구성한다.
- **Chosen ($\mathbf{y^+}$):** 교사 모델이 생성한 정확한 분석과 교정 경로 $(a^+_k, c^+_k)$.
- **Rejected ($\mathbf{y^-}$):** 학생 모델이 스스로 시도한 잘못된 분석과 교정 경로 $(a^-_k, c^-_k)$.

최적화 목표는 학생 모델이 교사 모델의 교정 흔적을 따르도록 하는 것이며, 손실 함수는 다음과 같다.
$$\mathcal{L}_{Cross-DPO}(\theta) = -\mathbb{E}_{(x, s_{1 \sim k-1}, (a^+_k, c^+_k)) \in D_{corr}} \left[ \log \sigma \left( \beta \log \frac{\pi_\theta((a^+_k, c^+_k) | x; s_{1 \sim k-1})}{\pi_{ref}((a^+_k, c^+_k) | x; s_{1 \sim k-1})} - \beta \log \frac{\pi_\theta((a^-_k, c^-_k) | x; s_{1 \sim k-1})}{\pi_{ref}((a^-_k, c^-_k) | x; s_{1 \sim k-1})} \right) \right]$$
이 과정에서 $s_{1 \sim k-1}$은 오류가 발생하기 전까지의 올바른 추론 단계들을 의미하며, 이를 조건으로 주어 모델이 정확히 오류 지점부터 교정을 수행하도록 학습시킨다.

## 📊 Results

### 실험 설정
- **베이스 모델:** Qwen2.5-Math-7B, Meta-Llama3.1-8B, DeepSeek-Math-7B.
- **데이터셋:** MATH, GSM8K, GaoKao Bench 등.
- **평가 지표:** Chain-of-Thought 기반의 정확도(%).

### 주요 결과
SUPERCORRECT는 7B 파라미터 모델 중 SOTA 성능을 달성하였다. 특히 **SUPERCORRECT-Qwen-7B** 모델은 MATH 벤치마크에서 $70.2\%$, GSM8K에서 $89.5\%$의 정확도를 기록하였다. 이는 기존의 강력한 모델인 DeepSeekMath-7B보다 MATH 기준 $7.8\%$, Qwen2.5-Math-7B보다 $15.1\%$ 향상된 결과이다. 또한 70B 규모의 Llama3-Instruct 모델보다 GSM8K와 MATH에서 더 높은 성능을 보여, 모델 크기의 한계를 효율적인 학습 방법으로 극복했음을 입증하였다.

### 자가 수정 능력 및 안정성
실험 결과, SUPERCORRECT는 추론 후 스스로 오류를 수정하는 단계에서 정확도를 $5 \sim 6\%$ 추가로 향상시켰으나, 타 모델들은 오히려 정확도가 떨어지거나 정체되는 모습을 보였다. 또한 MATH 데이터셋의 가장 어려운 레벨(Level-5) 문제들에 대해 반복 실험을 수행한 결과, 베이스 모델 대비 평균 정확도는 상승하고 분산(Variance)은 크게 감소하여 추론의 안정성이 확보되었음을 확인하였다.

## 🧠 Insights & Discussion

본 연구의 가장 큰 강점은 단순히 정답 데이터를 학습시키는 것이 아니라, **'어떻게 틀렸고 어떻게 고쳐야 하는가'**에 대한 고차원적인 통찰(Insight)을 전이시켰다는 점이다. 

**분석 및 논의:**
1. **사고 병목(Thought Bottleneck)의 타파:** 실험 결과, 모델이 원래 어려워하던 주제(예: 기하학, 정수론 등)에서 성능 향상 폭이 더 크게 나타났다. 이는 교차 모델 DPO를 통해 학생 모델이 기존에 갖지 못했던 새로운 문제 해결 기법과 팁을 습득했음을 시사한다.
2. **계층적 사고의 효과:** 단순 CoT보다 $\langle Key \rangle$와 $\langle Generalized \rangle$ 태그를 포함한 계층적 구조가 모델의 환각(Hallucination)을 줄이고 추론의 일관성을 높이는 데 기여하였다.
3. **한계 및 가정:** 본 방법론은 매우 강력한 교사 모델(o1 등)의 존재를 전제로 한다. 또한, 데이터 생성 과정에서 Inspector LLM을 통해 품질을 검증하는 반복적인 파이프라인이 필요하므로 데이터 구축 비용이 높을 수 있다는 점이 한계로 지적될 수 있다.

## 📌 TL;DR

SUPERCORRECT는 교사 모델의 **계층적 사고 템플릿(HTT)**과 **교차 모델 교정 경로**를 활용하여 소형 LLM의 수학적 추론 능력을 극대화하는 2단계 학습 프레임워크이다. SFT를 통해 정밀한 추론 구조를 학습시키고, DPO를 통해 교사 모델의 오류 수정 능력을 전이함으로써 7B급 모델로 SOTA 성능을 달성하였다. 이 연구는 소형 모델이 스스로 해결하지 못하는 '사고의 병목'을 외부 모델의 가이드를 통해 효과적으로 해결할 수 있음을 보여주며, 향후 소형 모델의 추론 효율화 연구에 중요한 방향성을 제시한다.
