# Detecting Mode Collapse in Language Models via Narration

Sil Hamilton (2024)

## 🧩 Problem to Solve

본 연구는 대규모 언어 모델(LLM)의 정렬(Alignment) 과정이 모델의 창의적 표현 능력, 특히 다양한 '가상 저자(Virtual Author)'를 구현하는 능력에 미치는 영향을 분석하고자 한다.

문학 이론에서 가상 저자란 텍스트 내의 어휘나 수사적 장치를 통해 암시되는 저자의 페르소나를 의미한다. 초기 LLM들은 필터링되지 않은 방대한 데이터를 학습하여 일관되지 않은 인격을 가졌으나, 이는 오히려 다양한 관점에서 문학적 샘플링을 수행하는 데 유리했다. 그러나 최근의 모델들은 지시어 튜닝(Instruction Tuning)과 인간 피드백 기반 강화학습(RLHF)을 통해 주관적으로 일관된 페르소나를 갖도록 정렬되었다.

이 과정에서 발생하는 문제는 모델이 특정 인간 가치나 답변 스타일에 과적합(Overfitting)되어, 사용자가 요청하는 임의의 가상 저자의 관점을 일반화하여 생성하는 능력을 상실했을 가능성이 있다는 점이다. 따라서 본 논문의 목표는 정렬된 모델들이 여전히 다수의 관점을 유지하고 있는지, 아니면 정렬으로 인해 '모드 붕괴(Mode Collapse)' 현상을 겪고 있는지를 검증하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 LLM의 정렬 과정이 모델의 생성 다양성을 저해하여, 결과적으로 특정 패턴만 반복 생성하는 '모드 붕괴' 현상을 유발할 수 있음을 실증적으로 제시한 것이다.

특히, 모델이 최신 버전으로 업데이트되고 더 많은 정렬 과정을 거칠수록, 요청된 페르소나(인종, 성별, 교육 수준 등)에 관계없이 매우 유사하고 반복적인 서사 구조와 특정 명칭(예: 특정 이름이나 트로프)을 사용하는 경향이 강해진다는 점을 밝혀냈다. 이는 정렬이 가져오는 '정렬 세금(Alignment Tax)'이 단순한 성능 저하를 넘어, 모델의 일반화 능력 및 창의적 생성 능력의 상실로 이어질 수 있음을 시사한다.

## 📎 Related Works

논문은 롤랑 바르트(Roland Barthes)의 '저자의 죽음' 이론과 계산 스타일 분석(Computational Stylometry)을 이론적 배경으로 삼는다. 스타일 분석은 텍스트의 스타일적 특징을 통해 저자를 식별하는 기술로, 이는 텍스트가 암시하는 '가상 저자'를 찾는 작업과 일맥상통한다.

기존 연구들은 LLM이 다양한 페르소나를 호출하는 데 능숙하며, 이를 통해 사회과학자들이 인간의 의사소통을 시뮬레이션하는 도구로 활용해 왔음을 언급한다. 하지만 정렬된(Aligned) 최신 모델들이 여전히 이러한 다중 관점 생성 능력을 유지하고 있는지에 대해서는 충분한 조사가 이루어지지 않았으며, 본 연구는 이 공백을 메우고자 한다.

## 🛠️ Methodology

### 실험 설계 및 데이터 생성

연구자는 세 가지 서로 다른 정렬 단계의 OpenAI 모델을 대상으로 실험을 설계하였다. 각 모델에 대해 사회문화적 인구통계학적 특성을 반영한 프롬프트를 입력하여 총 4,374개의 고유한 이야기를 생성하게 하였다.

프롬프트에 포함된 독립 변수는 다음과 같다:

- **교육 수준(Education):** 무교육, 교육받음, 명시되지 않음
- **성적 지향(Orientation):** 이성애자, 퀴어, 명시되지 않음
- **인종(Ethnicity):** 백인 미국인, 흑인 미국인, 명시되지 않음
- **성별(Gender):** 시스젠더 남성, 시스젠더 여성, 명시되지 않음
- **암시적 독자(Implied Reader):** 개인, 그룹
- **이야기 유형(Type of Story):** 일반 이야기, 정치적 알레고리, 민담

### 분석 대상 모델

모든 모델은 1,750억 개의 파라미터를 가진 디코더 전용 모델이며, 정렬의 정도에 따라 구분된다:

1. `davinci-instruct-beta`: 지시어 튜닝(Instruction Tuning)만 수행한 모델.
2. `text-davinci-003`: 지시어 튜닝에 RLHF 단계가 추가된 모델.
3. `gpt-3.5-turbo`: RLHF 이후 대화형 작업으로 추가 미세 조정(Fine-tuning)이 이루어진 최신 모델.

### 측정 방법: 토픽 분석(Topic Analysis)

가상 저자의 구현 능력을 측정하기 위해 생성된 텍스트에 대해 `BERTopic`을 이용한 토픽 분석을 수행하였다. `BERTopic`은 BERT 임베딩을 활용하여 텍스트 내의 어휘적 규칙성을 클러스터링하는 기법이다. 만약 모델이 요청된 페르소나와 관계없이 유사한 단어와 주제를 반복해서 사용한다면, 이는 높은 밀도의 특정 토픽 클러스터로 나타나게 된다.

실험 시 설정값은 온도(Temperature) $1.0$, 최대 토큰 수 400으로 설정하여 생성의 무작위성을 확보하였다.

## 📊 Results

### 정량적 결과 (토픽 밀도 분석)

분석 결과, `gpt-3.5-turbo` 모델에서 검출된 토픽의 밀도가 이전 모델들에 비해 현저히 높게 나타났다(Figure 2 참조). 이는 `gpt-3.5-turbo`가 생성한 이야기들이 서로 매우 유사한 어휘적 특징을 공유하고 있음을 의미한다.

반면, `davinci-instruct-beta`와 `text-davinci-003`은 뚜렷한 토픽 클러스터가 적게 발견되었으며, 발견된 토픽들 또한 "said", "I'm", "know"와 같은 일반적인 단어들로 구성되어 있어 특정 패턴에 매몰되지 않았음을 보여준다.

### 정성적 결과 (반복적 패턴 발견)

`gpt-3.5-turbo`에서 발견된 고밀도 토픽들은 매우 구체적인 내용을 담고 있었다. 예를 들어 "kofi, tree, village, man" 또는 "people, chosen ones, leader"와 같은 특정 키워드가 반복적으로 등장하였다. 또한, 프롬프트에서 요구한 인구통계학적 특성과 상관없이 'Amara', 'Rachel', 'Mary'와 같은 특정 이름들이 빈번하게 사용되는 경향이 확인되었다.

이는 모델이 가상 저자의 특성을 반영하기보다는, 학습 과정에서 과적합된 특정 서사 템플릿을 반복적으로 출력하고 있음을 보여준다.

## 🧠 Insights & Discussion

본 연구는 생성적 적대 신경망(GAN)에서 주로 발견되던 '모드 붕괴(Mode Collapse)' 현상이 LLM의 과도한 정렬(Over-alignment) 과정에서도 발생할 수 있음을 시사한다. 모드 붕괴란 모델이 타겟 분포의 다양성을 학습하지 못하고 매우 제한된 일부 샘플만을 생성하는 상태를 말한다.

저자는 `gpt-3.5-turbo`가 이전 모델들보다 더 많은 정렬 단계를 거치면서, 인간의 선호도에 맞춘 정형화된 답변 방식에 과적합되었고, 이로 인해 텍스트의 고유한 속성인 '저자의 다양성'을 일반화하는 능력을 상실했다고 분석한다.

이는 사회과학적 시뮬레이션을 위해 LLM을 사용하는 연구자들에게 중요한 경고를 준다. 최신 모델이라고 해서 반드시 더 정확한 페르소나 구현 능력을 갖춘 것은 아니며, 오히려 과도한 정렬로 인해 특정 인구통계학적 특성을 반영한 데이터 샘플링 시 왜곡된(정형화된) 결과를 얻을 위험이 크다.

## 📌 TL;DR

본 논문은 OpenAI의 세 가지 모델을 비교 분석하여, 모델의 정렬(Alignment) 수준이 높아질수록(특히 `gpt-3.5-turbo`에서) 생성되는 이야기의 다양성이 급격히 감소하고 특정 패턴만 반복하는 '모드 붕괴' 현상이 나타남을 증명하였다. 이는 과도한 RLHF와 미세 조정이 모델의 창의적 일반화 능력을 저해하는 '정렬 세금(Alignment Tax)'으로 작용할 수 있음을 보여주며, 향후 사회과학적 시뮬레이션이나 창의적 글쓰기에 LLM을 적용할 때 모델 선택에 주의가 필요함을 시사한다.
