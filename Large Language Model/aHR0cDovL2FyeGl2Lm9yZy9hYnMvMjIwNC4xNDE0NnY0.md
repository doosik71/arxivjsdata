# Training Language Models with Language Feedback

Jérémy Scheurer, Jon Ander Campos, Jun Shern Chan, Angelica Chen, Kyunghyun Cho, Ethan Perez (2022)

## 🧩 Problem to Solve

사전 학습된 언어 모델(Pretrained Language Models, LMs)은 다양한 NLP 작업에서 뛰어난 성능을 보이지만, 공격적인 텍스트 생성이나 사실과 다른 요약문 생성과 같이 인간의 선호도(Human Preferences)에 반하는 결과물을 만들어내는 고질적인 문제가 있다. 이를 해결하기 위해 최근에는 인간이 두 개의 모델 출력물 중 더 나은 것을 선택하는 쌍체 비교(Pair-wise comparison) 방식의 피드백을 통해 모델을 학습시키는 방법이 주로 사용되었다.

그러나 쌍체 비교 방식은 한 번의 인간 평가당 전달되는 정보의 양이 매우 제한적이라는 한계가 있다. 본 논문의 목표는 단순한 비교가 아닌, 구체적인 수정 방향을 제시하는 자연어 피드백(Natural Language Feedback)을 활용하여 더 적은 양의 데이터로도 모델을 효율적으로 인간의 선호도에 맞게 정렬(Alignment)하는 알고리즘을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 인간이 작성한 상세한 자연어 피드백을 이용하여 모델의 출력을 직접적으로 개선하고, 그렇게 생성된 고품질의 데이터를 통해 모델을 미세 조정(Finetuning)하는 세 단계의 학습 알고리즘을 설계한 것이다. 특히, 대규모 언어 모델(Large Language Models, LLMs)이 지시사항을 따르는 능력(Instruction following)을 갖추고 있다는 점에 착안하여, 피드백을 반영한 여러 수정안을 생성하게 한 뒤 그중 가장 적절한 것을 선택해 학습 데이터로 사용하는 전략을 취한다.

## 📎 Related Works

기존의 연구들은 주로 인간의 선호도를 학습시키기 위해 강화 학습(Reinforcement Learning) 기반의 방법론이나 보조 손실 함수(Auxiliary losses)를 사용하였다. 이러한 방식은 단순한 선호도 점수나 랭킹을 학습하는 데는 유용하지만, 구체적인 자연어 피드백을 직접적으로 활용하여 모델을 업데이트하는 데에는 한계가 있다.

또한, 텍스트 분류 작업에서 설명(Explanation)을 활용하는 연구들이 존재했으나, 이는 정답 레이블이 주어진 상태에서 '왜' 이것이 정답인지를 설명하는 것에 가깝다. 반면 본 논문에서 다루는 피드백은 정답 레이블 없이 기존 출력물을 '어떻게' 개선해야 하는지에 집중한다는 점에서 차별점을 가진다. 일부 자연어 피드백을 활용한 연구들이 있었으나, 대부분은 출력물의 순위를 매기는 랭킹 작업에 국한되었으며, 본 연구처럼 생성 모델의 직접적인 학습을 목표로 한 경우는 드물다.

## 🛠️ Methodology

본 논문은 입력 $x$에 대해 인간의 선호도에 부합하는 출력 $y$를 생성하기 위해 다음과 같은 3단계 알고리즘을 제안한다.

**1. 수정안 생성 (Generating Refinements)**
먼저, 언어 모델에게 입력 $x$, 모델이 처음에 생성한 출력 $y$, 그리고 인간이 작성한 자연어 피드백 $f$를 조건으로 제공한다. 모델은 이 정보들을 바탕으로 피드백이 반영된 여러 개의 수정안 $y'_1, \dots, y'_N$을 샘플링한다.

**2. 최적의 수정안 선택 (Choosing the Best Refinement)**
생성된 $N$개의 수정안 중 피드백 $f$를 가장 잘 반영한 하나를 선택해야 한다. 이를 위해 본 논문은 대조 학습으로 사전 학습된 텍스트 임베딩 함수 $E$를 사용하여 피드백과 수정안 간의 코사인 유사도(Cosine Similarity)를 계산하는 점수 함수 $S$를 정의한다.
$$y'_{best} = \text{argmax}_{y'_i} \text{cosine\_similarity}(E(f), E(y'_i))$$
피드백은 보통 이상적인 텍스트가 어떤 모습이어야 하는지를 묘사하므로, 피드백 임베딩과 가장 유사한 수정안이 가장 품질이 높을 것이라고 가정한다.

**3. 모델 미세 조정 (Finetuning)**
마지막으로, 선택된 최적의 수정안 $y'_{best}$를 정답(Target)으로 하여 입력 $x$가 주어졌을 때 $y'_{best}$가 생성될 확률을 최대화하도록 언어 모델을 미세 조정한다.

이 전체 프로세스는 강화 학습과 같은 복잡한 절차 없이, 피드백을 통해 생성된 고품질의 데이터를 통한 지도 학습(Supervised Learning) 형태로 수행된다는 특징이 있다.

## 📊 Results

### 실험 설정
- **합성 작업 (Synthetic Task):** 문장에서 특정 공격적인 단어를 제거하는 제어된 환경의 작업을 통해 모델이 피드백을 정확히 반영하는지 평가하였다.
- **실제 작업 (Real-world Task):** Reddit 포스트를 요약하는 텍스트 요약(Text Summarization) 작업을 수행하였다.
- **데이터:** 단 100개의 인간 작성 피드백 샘플만을 사용하였다.
- **평가 지표:** 인간 평가자가 생성된 요약문을 랭킹하는 방식의 Win Rate를 측정하였다.

### 주요 결과
- **모델 크기의 영향:** 합성 작업 결과, 오직 175B 파라미터 규모의 대형 모델(GPT-3 Davinci 등)만이 피드백을 유의미하게 반영하여 수정안을 생성할 수 있음을 확인하였다. (Table 1 참조)
- **요약 성능:** 제안된 알고리즘으로 학습된 GPT-3 모델은 인간이 작성한 요약문(Human Summaries)과 비교했을 때 약 $51.0 \pm 5.0\%$의 Win Rate를 기록하며 인간 수준의 요약 성능에 도달하였다.
- **베이스라인 대비 성능:** 단순 초기 요약문으로 미세 조정했을 때보다 Win Rate가 $57.0 \pm 5.0\%$ 더 높게 나타나, 피드백 기반의 수정안이 학습 데이터로서 훨씬 가치 있음을 입증하였다.
- **피드백 반영도:** 분석 결과, 'Best of N' 전략을 사용했을 때 요약문의 $72.0 \pm 4.5\%$가 피드백에 언급된 최소 한 가지 이상의 포인트를 실제로 반영하고 있었다.

## 🧠 Insights & Discussion

본 연구는 매우 적은 양의 데이터(100개)만으로도 자연어 피드백을 통해 모델을 인간의 선호도에 맞게 효율적으로 정렬할 수 있음을 보여주었다. 특히, 복잡한 RLHF(Reinforcement Learning from Human Feedback) 과정 없이도 유사도 기반의 필터링과 지도 학습만으로 유사한 효과를 낼 수 있다는 점이 강점이다.

다만, 본 방법론은 수정안을 생성하는 단계에서 매우 거대한 모델(175B)이 필요하다는 전제가 있다. 작은 모델들은 피드백을 정확히 반영하지 못하는 경향이 있어, 실제 적용 시 계산 비용의 문제가 발생할 수 있다.

또한, 초기 요약문의 품질이 낮을수록 피드백의 효과가 더 크게 나타난다는 점이 관찰되었다. 이는 모델이 이미 높은 품질의 결과물을 내놓고 있을 때는 자연어 피드백이 줄 수 있는 정보의 이득이 적지만, 품질이 낮을 때는 구체적인 가이드라인이 성능 향상에 결정적인 역할을 함을 시사한다.

## 📌 TL;DR

본 논문은 인간의 구체적인 자연어 피드백을 사용하여 언어 모델을 학습시키는 3단계 알고리즘(**수정안 생성 $\rightarrow$ 유사도 기반 최적안 선택 $\rightarrow$ 지도 학습 기반 미세 조정**)을 제안한다. 실험을 통해 175B 규모의 대형 모델이 피드백을 효과적으로 수용할 수 있음을 확인하였으며, 단 100개의 피드백 샘플만으로도 모델의 요약 능력을 인간 수준까지 끌어올릴 수 있음을 증명하였다. 이는 향후 RLHF의 데이터 효율성 문제를 해결하고, 더 직관적인 방법으로 모델을 정렬하는 새로운 방향성을 제시한다.
