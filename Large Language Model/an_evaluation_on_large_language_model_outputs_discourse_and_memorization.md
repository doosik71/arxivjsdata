# An Evaluation on Large Language Model Outputs: Discourse and Memorization

Adrian de Wynter, Xun Wang, Alex Sokolov, Qilong Gu, Si-Qing Chen (2026)

## 🧩 Problem to Solve

본 논문은 대규모 언어 모델(Large Language Models, LLMs)이 학습 데이터를 어느 정도까지 암기(Memorization)하고 있으며, 이러한 암기 현상이 생성된 텍스트의 품질과 어떤 상관관계가 있는지를 분석하는 것을 목표로 한다.

현재 많은 LLM은 내부 가중치(weights)나 학습 데이터셋이 공개되지 않은 서비스 형태로 제공되고 있어, 과학 커뮤니티가 모델을 직접적으로 검증하거나 분석하는 데 한계가 있다. 특히 LLM의 출력물은 문법적으로는 매우 완벽해 보이지만, 사실과 다른 정보를 제공하는 환각(Hallucination), 독성 콘텐츠 생성, 그리고 학습 데이터의 단순 암기 출력이라는 고질적인 문제를 가지고 있다. 이에 저자들은 모델의 내부 구조에 접근하지 않고도 외부에서 관찰 가능한 출력물만으로 암기 정도와 담론 품질(Discourse Quality) 사이의 관계를 규명하고자 한다.

## ✨ Key Contributions

본 연구의 핵심적인 기여는 LLM의 암기 현상과 출력 품질 사이의 상관관계를 실증적으로 분석했다는 점에 있다. 주요 설계 아이디어와 발견점은 다음과 같다.

1. **범용 도구를 이용한 분석 체계 구축**: 모델의 가중치나 특수 데이터셋 없이, 시중의 표절 검사 도구(Plagiarism detection tools)와 인간의 정성적 어노테이션(Human annotation)만을 사용하여 분석을 수행함으로써 재현 가능성을 높였다.
2. **암기와 품질의 상관관계 발견**: 일반적인 통념과 달리, 많은 경우 학습 데이터를 더 많이 암기하여 출력하는 모델이 담론 품질(문법, 논리, 사실 관계 등) 면에서 더 높은 성능을 보이는 경향이 있음을 발견하였다.
3. **암기 완화 전략 검증**: 모델에게 창의적인 작가가 되어 기존 내용을 그대로 출력하지 말라고 명시적으로 지시하는 프롬프팅 전략이 암기된 텍스트의 출력 비율을 유의미하게 낮출 수 있음을 입증하였다.

## 📎 Related Works

논문에서는 기존의 암기 관련 연구들을 다음과 같이 설명하며 본 연구와의 차별점을 제시한다.

- **기존 연구의 한계**: Carlini 등은 모델의 파라미터 크기와 암기 경향성 사이의 상관관계를 정량화하였고, Lee 등은 암기와는 구별되는 표절(Plagiarism)의 개념을 다루었다. 그러나 이러한 연구들은 주로 모델의 가중치 분석이나 고정된 태스크 코퍼스(Fixed-task corpora)에 의존하는 경향이 있었다.
- **본 연구의 차별점**: LLM의 학습 데이터가 인터넷 전체에서 수집되므로, 고정된 벤치마크 데이터셋 자체가 이미 학습 데이터에 포함되었을 가능성이 높다. 따라서 본 연구는 자동화된 벤치마크보다는 사용자가 체감하는 텍스트의 품질과 암기 여부를 직접 분석하는 수동 분석(Manual analysis)에 집중하며, 특히 암기가 사실적 정확성(Factuality) 및 전체적인 담론 품질과 어떻게 연결되는지를 심층적으로 탐구한다.

## 🛠️ Methodology

### 전체 파이프라인

연구진은 9개의 주요 LLM(BLOOM, ChatGPT, Galactica, GPT-3, GPT-4, OPT, OPT-IML, LLaMA)을 대상으로 실험을 진행하였다. 전체 프로세스는 **프롬프트 입력 $\rightarrow$ 텍스트 생성 $\rightarrow$ 표절 검사 도구를 통한 암기 확인 $\rightarrow$ 인간에 의한 담론 품질 평가** 순으로 이루어진다.

### 데이터 생성 및 프롬프팅

- **도메인 설정**: 과학 논문(Scientific papers), 블로그 포스트(Blog posts), 지식 검색(Knowledge Retrieval, KR), 긴 텍스트 자동 완성(Long-text autocompletion), 공통 텍스트 시작구(Baseline) 등 5가지 도메인을 설정하였다.
- **생성 절차**: 각 도메인당 15개의 프롬프트를 작성하였으며, 모델당 75개의 텍스트를 생성하였다. 동일 프롬프트에 대해 5번 생성 후 가장 긴 텍스트를 선택하는 'Best-of-five' 접근 방식을 사용하였다.

### 평가 지표 및 기준

텍스트 분석은 크게 두 가지 범주로 나뉜다.

1. **텍스트 범주(Textual categories)**: 5개의 온라인 표절 검사 서비스를 이용하여 암기된 텍스트의 비율과 고유 텍스트(Original text)의 비율을 측정하였다.
2. **담론 범주(Discourse categories)**: 인간 평가자가 다음 4가지 항목을 기준으로 점수를 매겼다.
    - **개인 식별 정보(PII)**: 실존 인물의 이름, 이메일, 웹사이트 등이 잘못 할당되었는지 확인한다.
    - **사실적 오류(Factual errors)**: 신뢰할 수 있는 출처로 검증 불가능한 주장을 포함하는지 확인한다.
    - **논리적 모순(Logical inconsistencies)**: 자기모순적 진술이나 잘못된 수학적 증명 등이 포함되었는지 확인한다.
    - **담론 일관성(Discourse coherence)**: 문법 오류, 잘못된 서식, 부적절한 논증 흐름 등을 평가한다.

### 분석 방법

변수 간의 상관관계는 피어슨 상관계수(Pearson's $\rho$)를 사용하여 계산하였다.

## 📊 Results

### 정량적 결과

- **암기 빈도**: 분석 대상 출력물의 약 $80.0\%$가 어느 정도의 암기 징후를 보였으며, 평균적으로 $82.3\%$의 고유 텍스트 비율을 나타냈다.
- **오류 발생률**: 사실적 오류 $46.4\%$, 담론 결함 $52.0\%$, 논리적 오류 $31.3\%$, PII 포함 $15.4\%$로 나타났다.
- **모델별 성능**:
  - **GPT-3.51**은 가장 높은 담론 품질(결함률 $18.7\%$)을 보였으나, 동시에 암기율($89.3\%$)도 매우 높았다.
  - **Galactica**는 사실적 오류($73.3\%$)와 논리적 모순이 가장 빈번하게 발생하였다.
  - **ChatGPT**는 PII 오류율 $0.0\%$를 기록하며 개인정보 보호 측면에서 가장 우수했다.

### 주요 발견 및 상관관계

1. **암기와 품질의 역설**: 암기 비율이 높을수록 담론 결함이 적게 나타나는 경향(약한 반비례 관계, $\rho = -0.40$)이 발견되었다. 즉, 학습 데이터를 더 많이 암기하여 출력하는 모델이 결과적으로 더 고품질의 텍스트를 생성하는 경향이 있다.
2. **프롬프트 길이의 영향**: 프롬프트의 길이가 짧을수록 모델이 암기된 내용을 출력할 확률이 높아지는 일반적인 경향이 모든 모델에서 공통적으로 관찰되었다.
3. **완화 전략의 효과**: "당신은 창의적인 작가이며, 기존 내용과 다르게 작성하라"는 제약 조건을 추가했을 때, 암기된 텍스트의 출력 비율이 유의미하게 감소하였다. 특히 GPT-4의 경우 암기 발생률이 $40\%$ 감소하였다.

## 🧠 Insights & Discussion

### 암기와 학습의 경계

본 논문은 "모델이 사실적으로 정확하고 품질 높은 텍스트를 생성하기 위해 어느 정도의 암기가 필수적인가?"라는 근본적인 질문을 던진다. 실험 결과, 사실적 정확성과 암기 능력 사이에 강한 상관관계가 있음이 드러났으며, 이는 LLM이 단순한 통계적 패턴 학습을 넘어 데이터를 '기억'함으로써 지식을 출력하고 있음을 시사한다.

### 비판적 해석

저자들은 현재 LLM의 성능 평가 방식에 대해 비판적인 시각을 제시한다. 많은 모델이 문법적으로는 완벽한 텍스트를 생성하여 높은 평가를 받지만, 실제로는 논리적 추론 능력이 결여되어 있거나 단순 암기에 의존하고 있을 가능성이 크다. 따라서 텍스트의 '유창함(Grammaticality)'이 아닌 '추론 및 논증 능력(Argumentative and reasoning capabilities)'을 중심으로 평가 기준이 옮겨가야 한다고 주장한다.

### 한계점

- 분석에 사용된 프롬프트의 수가 인간 어노테이션의 한계로 인해 상대적으로 적다.
- GPT-4와 같은 폐쇄형 모델의 경우 내부 처리 과정(Upstream/Downstream components)을 알 수 없어 공정한 비교가 어려울 수 있다.
- 표절 검사 서비스가 모든 형태의 암기(예: 서식이 변경된 텍스트)를 잡아내지 못하는 한계가 있다.

## 📌 TL;DR

본 연구는 9개의 주요 LLM을 대상으로 암기 현상과 출력 품질의 상관관계를 분석하였으며, **높은 암기율이 오히려 더 높은 담론 품질 및 사실적 정확성과 연결된다는 점**을 발견하였다. 또한, 명시적인 프롬프팅을 통해 이러한 암기 출력을 완화할 수 있음을 입증하였다. 이 결과는 LLM의 성능을 평가할 때 단순한 문법적 유창함보다는 논리적 추론 능력을 검증하는 것이 훨씬 중요하다는 시사점을 제공하며, 향후 모델 학습 시 데이터 중복 제거(Deduplication)와 학습 전략의 중요성을 강조한다.
