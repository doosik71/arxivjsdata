# INTERACTIVE LEARNING FOR LLM REASONING

Hehai Lin, Shileii Cao, Sudong Wang, Haotian Wu, Minzhi Li, Linyi Yang, Juepeng Zheng, Chengwei Qin (2025)

## 🧩 Problem to Solve

본 논문은 대규모 언어 모델(LLM)의 추론 능력을 향상시키기 위한 기존 학습 방식의 한계를 해결하고자 한다. 현재의 학습 패러다임은 주로 지도 미세 조정(SFT), 선호도 학습(PL), 강화 학습(RL)과 같은 단일 에이전트 학습(Single-agent learning)에 의존하고 있으며, 이는 모델이 스스로 시행착오를 거치며 학습하는 방식이다.

최근 다중 에이전트 학습(Multi-agent learning) 연구들이 등장하여 모델 간의 협력을 통해 시스템 전체의 성능을 높이는 다중 에이전트 시스템(MAS)을 구축하고 있으나, 이러한 방식은 추론 시에도 반드시 MAS를 재실행해야만 최종 답안을 얻을 수 있다는 치명적인 단점이 있다. 이는 타인과의 상호작용을 통해 능력을 배양한 뒤, 이후에는 독립적으로 문제를 해결하는 인간의 인지 과정과 상충된다. 따라서 본 연구의 목표는 다중 에이전트 간의 상호작용을 통해 LLM 개별 모델의 독립적인 문제 해결 능력을 강화하는 학습 프레임워크를 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 다중 에이전트 상호작용을 통해 개별 LLM의 추론 능력을 높이는 **ILR (Interactive Learning for LLM Reasoning)** 프레임워크를 제안한 것이다. ILR의 중심 설계 아이디어는 다음 두 가지 핵심 구성 요소로 요약된다.

1. **Dynamic Interaction (동적 상호작용):** 문제의 난이도와 모델의 능력에 따라 협력(Cooperation) 또는 경쟁(Competition) 전략을 적응적으로 선택한다. 또한, 인간의 토론 방식을 모방하여 아이디어 공유, 분석, 융합의 3단계로 구성된 **Idea3** 상호작용 패러다임을 도입하여 정보 교환의 효율성을 높인다.
2. **Perception Calibration (인지 보정):** 다른 에이전트의 보상 분포 특성을 자신의 보상 함수에 통합함으로써, 모델이 동료 에이전트의 수행 능력을 인지하고 그에 맞춰 자신의 추론을 조정하도록 유도한다. 이를 위해 GRPO(Group Relative Policy Optimization) 알고리즘을 사용하여 모델을 최적화한다.

## 📎 Related Works

기존의 다중 에이전트 학습 연구는 크게 두 가지 관점에서 진행되었다.

* **Multi-Agent Communication:** Debate(다수결 기반)나 Actor-Critic(피드백 기반), 혹은 SiriuS와 같은 역할 분담 방식이 제안되었다. 그러나 이러한 방식들은 개별 에이전트를 시스템의 부품으로 간주하며, 목표 또한 시스템 전체의 성능 최적화에 집중되어 있다.
* **Multi-Agent Training:** MALT나 Multiagent-FT 등은 에이전트들이 샘플링한 경험을 기반으로 SFT나 DPO를 적용하여 독립적으로 업데이트하는 방식을 사용한다. 최근에는 MARL(Multi-Agent Reinforcement Learning)을 통해 실시간 상호작용을 구현하려는 시도가 있었으나, 대부분 수동으로 설정된 하이퍼파라미터에 의존하여 보상을 형성한다는 한계가 있다.

본 논문은 에이전트를 독립적인 엔티티로 취급하여 개별 능력을 강화한다는 점과, 보상 체계를 자동화된 보정 메커니즘으로 구현했다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

ILR 프레임워크의 전체 파이프라인은 문제 난이도 추정 $\rightarrow$ 동적 상호작용 전략 선택 $\rightarrow$ Idea3 기반 정보 교환 $\rightarrow$ 인지 보정 기반 보상 계산 $\rightarrow$ GRPO 최적화 순으로 진행된다.

### 1. 문제 난이도 추정 (Question Difficulty Estimation)

LLM의 self-ranking 능력을 활용하여 문제의 상대적 난이도를 측정한다. 전체 데이터를 $B$개의 무작위 배치로 나누고, 모델이 각 배치 내에서 문제를 쉬운 순서대로 정렬하게 한다. 이를 $S$번 반복하여 평균을 낸 최종 난이도 점수 $D_q$를 다음과 같이 계산한다.

$$D_q = \frac{1}{m} \sum_{i=1}^{m} M_i \left( \frac{1}{S} \sum_{j=1}^{S} \frac{r_{q,j}}{N'} \right)$$

여기서 $m$은 LLM의 수, $r_{q,j}$는 $j$번째 분할에서의 순위, $N'$은 배치 당 문제 수이다.

### 2. 동적 상호작용 (Dynamic Interaction)

문항 반응 이론(Item Response Theory, IRT)을 사용하여 모델 $i$가 난이도 $D_q$인 문제를 맞힐 확률 $P_{q,i}$를 계산한다.

$$P_{q,i} = \frac{1}{1 + e^{-1.7 \times (\gamma_i - D_q)}}$$

여기서 $\gamma_i$는 모델의 추론 능력 지표이다. 모든 모델의 평균 확률 $P_q$가 $0.5$ 미만이면 **협력(Cooperation)** 모드를, $0.5$ 이상이면 **경쟁(Competition)** 모드를 선택한다. 선택된 모드에 따라 다음과 같은 **Idea3** 과정을 거친다.

* **Idea Sharing:** 각 모델이 자신의 해결 전략과 추론 과정을 제시한다.
* **Idea Analysis:** 협력 모드에서는 동료의 아이디어에서 보완할 점을 찾고, 경쟁 모드에서는 상대 전략의 약점과 강점을 비판적으로 분석한다.
* **Idea Fusion:** 분석된 통찰을 종합하여 최종 수정 답안을 생성한다.

### 3. 인지 보정 (Perception Calibration)

단순한 보상이 아닌, 동료 모델의 성과를 반영한 보정된 보상 $\bar{R}$을 사용한다. 각 모델이 동일 문제에 대해 $n$번 샘플링하여 얻은 보상의 최대값($R_{\max}$), 최소값($R_{\min}$), 평균값($R_{\text{avg}}$)을 활용한다. 모델 $i$의 $k$번째 응답에 대한 최종 보상은 다음과 같이 계산된다.

$$\bar{R}_{i,k} = R_{i,k} + \sum_{l \in M \setminus \{M_i\}} \text{clip} \left( \frac{R_{i,k} - R_{l,\text{avg}}}{R_{l,\max} - R_{l,\min}}, -\frac{1}{m-1}, \frac{1}{m-1} \right)$$

이 보정된 보상을 기반으로 GRPO를 적용하여 모델의 정책을 업데이트함으로써, 모델이 동료의 수준을 인지하고 더 나은 방향으로 학습하도록 유도한다.

## 📊 Results

### 실험 설정

* **대상 모델:** Llama-3.1-8B-Instruct, Qwen2.5-7B-Instruct, Qwen2.5-14B-Instruct를 서로 다른 조합(Group 1, 2, 3)으로 짝지어 학습시켰다.
* **데이터셋:** MATH 데이터셋으로 학습하고, GSM8K, MATH-500, Minerva Math, Olympiad Bench, AIME 24&25, MBPP(코딩) 등 6개 벤치마크에서 평가하였다.
* **비교 대상:** SFT, DPO, PPO, GRPO, Reinforce++(단일 에이전트) 및 DebateFT(다중 에이전트)와 비교하였다.

### 주요 결과

* **성능 향상:** ILR은 거의 모든 벤치마크에서 기존 베이스라인을 압도하였다. 특히 Llama-3.1-8B의 경우, 가장 강한 베이스라인인 GRPO 대비 평균 성능이 3.12% 향상되었다.
* **고난도 문제 해결:** 경쟁 수준의 벤치마크인 AIME 24&25에서 비약적인 성능 향상을 보였다. Llama-3.1-8B는 GRPO(5.00%) 대비 2배 높은 10.00%의 정확도를 기록하였다.
* **일반화 성능:** 수학 외 영역인 MBPP 코딩 벤치마크에서도 DPO, GRPO보다 높은 Pass@1 성능을 보여, 추론 능력의 일반적인 향상을 입증하였다.
* **에이전트 조합의 영향:** 초기 능력이 서로 비슷한 모델끼리 짝을 이뤘을 때(Group 1) 학습 효과가 더 좋았으며, 이는 능력 차이가 너무 크면 상호작용이 한쪽으로 쏠려 학습 효율이 떨어지기 때문으로 분석된다.

## 🧠 Insights & Discussion

본 논문은 다중 에이전트 학습이 단순한 시스템 구축을 넘어 개별 모델의 지능을 높이는 도구가 될 수 있음을 보여주었다.

**주요 통찰:**

1. **Idea3의 강건성(Robustness):** 추론 단계에서 Idea3 방식은 강한 모델이 약한 모델의 낮은 품질 응답에 휘둘리지 않게 돕는다. 비판적 분석 과정이 일종의 필터 역할을 하여 노이즈를 제거하고 유용한 정보만 선택적으로 수용하기 때문이다.
2. **동적 전략의 필요성:** 순수 협력이나 순수 경쟁 전략보다 난이도에 따라 전략을 바꾸는 것이 훨씬 효과적이다. 쉬운 문제는 경쟁을 통해 효율성을 높이고, 어려운 문제는 협력을 통해 맹점을 극복하는 인간의 학습 방식이 LLM에서도 유효함을 시사한다.

**한계 및 논의:**

* 본 연구는 계산 비용 문제로 인해 두 개의 모델 간 상호작용만을 다루었다. 3개 이상의 모델이 상호작용할 때 성능이 확장(Scale)되는지는 여전히 미해결 과제로 남아 있다.
* IRT를 통한 난이도 추정이 실용적이지만 항상 최적의 비율을 찾아내지는 못하므로, 향후 더 정교한 난이도 측정 메커니즘이 필요하다.

## 📌 TL;DR

본 논문은 LLM이 타 모델과의 상호작용을 통해 독립적인 추론 능력을 키울 수 있도록 하는 **ILR** 프레임워크를 제안한다. 문제 난이도에 따라 협력/경쟁 모드를 선택하는 **Dynamic Interaction**과 동료의 성과를 보상에 반영하는 **Perception Calibration**을 통해, 모델들이 서로를 자극하며 성장하게 한다. 실험 결과, ILR은 특히 고난도 수학 문제와 코딩 작업에서 단일 에이전트 학습보다 월등한 성능 향상을 보였으며, 이는 다중 에이전트 학습이 LLM의 개별 지능을 높이는 효과적인 패러다임이 될 수 있음을 증명한다.
