# Large language models have learned to use language

Gary Lupyan (2025)

## 🧩 Problem to Solve

본 논문은 대규모 언어 모델(Large Language Models, 이하 LLMs)이 실제로 언어를 '사용'하는 능력을 습득했는가에 대한 학술적 논쟁을 다룬다. 특히, LLM이 언어의 형식적 구조를 단순히 모방하는 '확률적 앵무새(stochastic parrots)'에 불과하다는 시각과, 생성 언어학(generative linguistics)의 엄격한 이론적 틀(예: Minimalist syntax)에 부합하지 않기 때문에 언어 모델로서 부적합하다는 회의론적 관점을 해결하고자 한다.

저자는 언어 능력의 평가 기준을 '특정 언어 이론에 대한 부합 여부'나 '메타언어적 판단 능력'이 아닌, 실제적인 '언어 사용(use)'의 관점에서 재정의해야 한다고 주장한다. 즉, LLM이 인간 수준의 언어 사용 능력을 보여준다면, 이는 LLM이 언어를 학습했음을 의미하며, 만약 이것이 기존의 언어학 이론과 충돌한다면 수정되어야 할 것은 LLM이 아니라 기존의 이론이라는 점을 강조한다.

## ✨ Key Contributions

본 논문의 핵심적인 직관은 언어적 유능함(competence)의 척도를 '형식적 규칙의 명시적 수행'이 아닌 '실제적인 언어 운용 능력'으로 보아야 한다는 것이다. 주요 기여 사항은 다음과 같다.

첫째, LLM을 단순한 통계적 모방 장치로 치부하는 'stochastic parrots' 관점이 실제 사용자의 경험과 배치됨을 지적하며, 예측 기반의 언어 학습(learning language through prediction)이 실제로 유효함을 주장한다.

둘째, 문법성 판단(grammaticality judgment)과 실제 언어 사용 능력을 분리하여 생각해야 함을 제안한다. 예를 들어, 5세 어린이가 구문 분석 트리(parse tree)를 그리지 못한다고 해서 언어 능력이 부족하다고 하지 않듯, LLM이 특정 언어학적 규칙을 명시적으로 출력하지 못하는 것이 언어 사용 능력의 부재를 의미하지 않는다는 논리를 제시한다.

셋째, Base model과 Fine-tuned model 간의 차이를 통해, 의미론적 내용과 문법적 구조를 분리하여 인식하는 능력이 미세 조정(fine-tuning) 과정을 통해 발현될 수 있음을 시사한다.

## 📎 Related Works

저자는 다음과 같은 기존 연구 및 관점들을 언급하며 자신의 주장과 대조한다.

1. **생성 언어학 및 Minimalist Syntax (Murphy et al., 2025; Bolhuis et al., 2024):** LLM이 최소주의 구문론(Minimalist syntax)에 따른 트리 구조를 생성하지 못하므로 언어의 기본 원리를 체득하지 못했다는 주장이다. 저자는 이에 대해 실제 언어 사용자가 반드시 이러한 이론적 구조를 명시적으로 알 필요는 없다고 반박한다.
2. **확률적 앵무새 (Bender & Hanna, 2025):** LLM이 이해나 의사소통 의도 없이 언어의 형태만을 조작한다는 시각이다. 저자는 8억 명의 사용자가 체감하는 언어적 유능함을 고려할 때, 이러한 주장은 지나치게 단순화된 해석이라고 비판한다.
3. **예측 기반 학습 (Elman, 1990):** 데이터로부터의 예측을 통해 언어 구조를 학습할 수 있다는 수십 년 된 과학적 프로그램의 연장선상에서 LLM의 성공을 해석한다.

## 🛠️ Methodology

본 논문은 정식 연구 논문이라기보다 논평(Commentary)의 성격을 띠고 있어 상세한 시스템 아키텍처보다는 실험적 관찰과 논리적 추론에 집중한다.

**1. 무의미하지만 문법적인 문장 생성 실험 (Informal Experiment)**
저자는 인간과 LLM(GPT-5)이 '최대한 무의미하지만 문법적으로는 옳은 문장'을 생성하게 하고, 이를 서로 평가하게 하는 실험을 수행하였다.
- **측정 지표:** 문장의 의미론적 무의미함(meaningfulness)에 대한 평점과, 해당 문장이 인간에 의해 생성되었는지 LLM에 의해 생성되었는지를 판별하는 정확도($d'$)를 측정하였다.
- **결과:** GPT-5가 생성한 문장은 인간이 생성한 무의미한 문장과 통계적으로 구분되지 않았으며, 이는 LLM이 의미(meaning)와 문법(grammaticality)을 분리하여 처리할 수 있음을 보여준다.

**2. Base Model vs. Instruction-tuned Model 비교**
언어 모델의 학습 단계에 따른 문법 판단 능력의 차이를 분석하였다.
- **Base Model:** 자기 지도 학습(self-supervision)만 거친 모델은 surprisal(의외성)을 통해 문법성을 간접적으로 나타낼 수는 있으나, 명시적인 문법성 판단은 수행하지 못한다. 또한, 비현실적인 문장(예: "고양이가 개에게 짖는다")은 생성하지만, 완전히 문법에 어긋나거나 무의미한 문장을 생성하는 데 어려움을 겪는다.
- **Fine-tuned Model:** 소규모 데이터셋(예: Databricks-dolly-15k)으로 미세 조정을 거친 모델은 명시적인 문법성 판단 능력이 발현된다. 특이한 점은 이 데이터셋에 문법 판단에 관한 예시가 포함되어 있지 않음에도 불구하고 이러한 능력이 나타난다는 것이다.

## 📊 Results

실험의 주요 결과는 다음과 같다.

- **정량적 결과:** Figure 1의 결과에 따르면, 인간과 GPT-5가 생성한 '무의미한 문장'의 의미론적 평점 차이는 유의미하지 않았으며($t < 1$), 생성 주체(인간 vs LLM)를 판별하는 정확도 역시 무작위 수준($t < 1$)에 가까웠다.
- **정성적 예시:** LLM이 생성한 "Let velvet measure tomorrow with raisins"와 같은 문장은 인간의 "Burpy trees monkey in mountain poodle"과 유사하게, 구조적으로는 문법적이나 의미상으로는 무의미한 특성을 보였다.
- **모델 간 비교:** Base model은 유능한 대화 상대가 되지 못하지만, 미세 조정을 거친 모델은 언어의 잠재적 지식을 활용해 실제적인 언어 사용(Conversational use)이 가능해진다.

## 🧠 Insights & Discussion

저자는 LLM의 능력을 인정하는 것이 언어 과학의 새로운 돌파구가 될 수 있다고 주장한다. 

**강점 및 시사점:**
- LLM은 인간과 매우 다른 아키텍처를 가졌음에도 불구하고, 언어 학습에 필수적인 계층적 구조(hierarchical structure)를 내부적으로 학습했음을 보여주는 증거들이 존재한다(Goldstein et al., 2025; Liu et al., 2025).
- 언어의 목적은 구문 트리를 그리는 것이 아니라, 상대의 심리 상태를 추론하고 협력하며 목표를 달성하는 '행위'에 있다. LLM이 이를 성공적으로 수행한다면, 그것이 곧 언어 학습의 성공이다.

**한계 및 비판적 해석:**
- 본 논문에서 제시된 GPT-5 관련 실험은 'informal tests'로 명시되어 있어, 엄격한 통제 하에 이루어진 정식 실험 결과로 보기에는 한계가 있다.
- Base model에서 Fine-tuned model로 넘어갈 때 문법 판단 능력이 어떻게 발현되는지에 대한 인과적 메커니즘은 아직 명확히 규명되지 않았으며, 이는 향후 연구 과제로 남아 있다.

**결론적 논의:**
저자는 곰이 불을 발견한 상황을 가정한 소설 "Bears Discover Fire"를 인용하며, LLM이 언어를 사용하게 된 현재의 상황을 '포스트 튜링 테스트 시대(post-Turing test era)'라고 정의한다. 이제는 LLM이 인간의 언어 이론에 맞는지 따지는 것이 아니라, LLM이라는 새로운 시스템이 어떻게 언어를 구사하는지를 연구함으로써 언어의 본질을 다시 이해해야 할 때라는 관점을 제시한다.

## 📌 TL;DR

본 논문은 LLM을 '확률적 앵무새'로 치부하거나 특정 언어학 이론(Minimalist syntax)의 잣대로 평가하는 회의론을 비판하며, **실제적인 '언어 사용 능력(use)'이야말로 언어 학습의 진정한 척도**임을 주장한다. GPT-5 등이 인간과 구분 불가능한 수준의 문법적 무의미 문장을 생성할 수 있다는 점과 미세 조정을 통해 문법 판단 능력이 발현된다는 관찰을 통해, LLM이 언어의 핵심 구조를 실제로 학습했음을 시사한다. 이 연구는 언어학의 평가 기준을 형식주의에서 기능주의/사용주의로 전환해야 한다는 중요한 방향성을 제시하며, 향후 LLM을 통한 언어 과학의 새로운 패러다임을 예고한다.
