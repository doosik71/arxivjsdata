# EXPLORING LARGE LANGUAGE MODELS TO GENERATE EASY TO READ CONTENT

Paloma Martínez, Lourdes Moreno, Alberto Ramos (2024)

## 🧩 Problem to Solve

본 연구는 인지 장애 및 지적 장애를 가진 사람들이 웹 페이지, 신문, 행정 문서, 건강 관련 문서 등 다양한 매체의 정보에 접근할 때 겪는 인지적 접근성(Cognitive Accessibility) 문제를 해결하고자 한다. 복잡한 문장 구조, 생소한 단어, 어려운 언어적 표현은 이러한 사용자들에게 심각한 정보 격차를 유발하며, 이는 단순히 읽기 능력의 문제를 넘어 사회적 소외로 이어진다.

현재 'Easy to Read' 및 'Plain Language'와 같은 텍스트 단순화 가이드라인이 존재하지만, 이를 표준화하여 적용하는 과정은 대부분 수동으로 이루어져 시간이 많이 소요되고 비용이 높다는 한계가 있다. 따라서 본 논문의 목표는 거대 언어 모델(Large Language Models, LLMs)과 자연어 처리(NLP) 기법을 활용하여 스페인어 텍스트를 체계적으로 Easy to Read 형식으로 변환하는 자동화 방안을 탐색하고, 그 가능성을 검증하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 다음과 같다.

1.  **스페인어 Easy to Read 병렬 코퍼스 구축**: 원문 텍스트와 지적 장애인을 위해 적응된 Easy to Read 버전이 쌍을 이루는 병렬 코퍼스를 구축하여, 향후 텍스트 단순화 시스템의 학습 및 테스트를 위한 자원을 제공하였다.
2.  **자동 문장 정렬(Sentence Alignment) 프로세스 제안**: 원문과 단순화된 텍스트 간의 의미적 유사성을 측정하여 최적의 쌍을 찾는 정렬 알고리즘을 개발하였다.
3.  **LLM 기반의 단순화 전략 탐색 및 비교**: Llama2 모델을 대상으로 한 파인튜닝(Fine-tuning) 방식과 프롬프트 엔지니어링(Prompt Engineering) 방식을 비교하여, 인지적 접근성을 높이기 위한 최적의 모델 구성과 전략을 분석하였다.
4.  **전문가 기반의 정성적 평가 체계 도입**: BLEU나 SARI와 같은 전통적인 자동 지표 대신, Easy to Read 전문가가 실제 가이드라인 준수 여부를 평가하는 인간 중심의 평가 방식을 적용하였다.

## 📎 Related Works

### Easy to Read vs Plain Language
논문은 두 가지 주요 단순화 전략을 구분하여 설명한다. **Easy to Read**는 주로 지적 장애인을 대상으로 하며, 구조적·언어적 단순함뿐만 아니라 시각적 보조 도구의 활용과 실제 대상자의 피드백 반영을 강조한다. 반면, **Plain Language**는 일반 대중, 읽기 능력이 낮은 사람, 비원어민 등 더 넓은 층을 대상으로 하며, 명확성, 간결성, 논리적 조직화에 집중한다.

### 기존 텍스트 단순화 연구 및 한계
기존의 NLP 기반 단순화 연구는 크게 두 가지 접근법으로 나뉜다.
- **모듈형 접근법(Modular Approach)**: 복잡한 단어를 교체하고, 긴 문장을 나누며, 수동태를 능동태로 바꾸는 등의 개별 단계를 파이프라인으로 구성한다.
- **통합형 접근법(All-in-one Approach)**: 생성형 모델을 사용하여 한 번에 단순화된 문장을 생성한다.

스페인어 환경에서는 Simplext나 FIRST와 같은 프로젝트가 있었으나, 스페인 표준(UNE 15310:2018)을 완전히 준수하거나 실제 장애인의 참여를 통한 검증이 부족했다는 한계가 지적되었다. 또한, 대부분의 연구가 어휘적 단순화(Lexical Simplification)에 치중되어 있어, 문장 구조와 문서 설계를 아우르는 종합적인 Easy to Read 생성 연구는 부족한 실정이다.

## 🛠️ Methodology

### 전체 파이프라인
시스템의 전체 구조는 **문장 정렬 $\rightarrow$ QLoRA를 이용한 모델 경량화 $\rightarrow$ LLM 파인튜닝 $\rightarrow$ 추론 및 테스트** 순으로 구성된다.

### 1. 문장 정렬 (Sentence Alignment)
원문과 단순화된 텍스트는 내용의 삭제나 변경이 많아 직접적인 매칭이 어렵다. 이를 해결하기 위해 Sentence Transformers 프레임워크와 LaBSE(Language-agnostic BERT Sentence Embedding) 모델을 사용하여 문장을 벡터로 임베딩하고, 코사인 유사도(Cosine Similarity)를 계산하여 가장 유사한 문장 쌍을 선택한다. 두 벡터 $\mathbf{u}$와 $\mathbf{v}$ 사이의 유사도는 다음과 같이 정의된다.

$$\text{similarity} = \cos(\theta) = \frac{\mathbf{u} \cdot \mathbf{v}}{\|\mathbf{u}\| \|\mathbf{v}\|}$$

### 2. 모델 아키텍처 및 학습 절차
에너지 효율성과 비용 절감을 위해 오픈소스 모델인 **Llama2-7B**를 기본 모델로 선택하였다.

- **QLoRA (Quantitative Low-Rank Adaptation)**: 32비트 부동 소수점 가중치를 4비트로 양자화(Quantization)하여 GPU 메모리 사용량을 줄이고, 저차원 행렬을 추가하여 학습 가능한 파라미터 수를 최적화하는 기법을 적용하였다. 이를 통해 NVIDIA GeForce RTX 3060(12GB) 환경에서도 학습이 가능하도록 하였다.
- **학습 설정**: 1,941개의 문장으로 구성된 코퍼스를 사용하여 4 에포크(Epoch) 동안 학습하였으며, 학습률은 $2 \times 10^{-5}$로 설정하였다.
- **프롬프트 설계**: 모델이 단순화의 목적과 대상(지적 장애인)을 명확히 인지하도록 "지적 장애인이 이해하기 쉽게 단순하고 짧으며 능동태인 문장으로 변환하라"는 지침을 포함한 템플릿을 사용하였다.

### 3. 평가 방법
전통적인 정량적 지표(BLEU, SARI 등)는 실제 이해도를 반영하지 못하므로, 전문가가 Table 3에 제시된 21가지 Easy to Read 가이드라인(예: 상형 문자 사용 자제, 능동태 사용, 한 문장에 하나의 아이디어만 포함 등)을 기준으로 정성 평가를 수행하였다.

## 📊 Results

### 실험 구성 및 결과
본 연구는 총 5가지 실험(E1~E5)을 통해 모델의 성능을 비교하였다.

- **E1 (Llama2-7B + Fine-tuning)**: 특정 도메인(스포츠)에 대해 파인튜닝된 모델은 해당 도메인의 텍스트를 매우 효과적으로 단순화하였다.
- **E2 (Llama2-7B + FT + Translation)**: 스페인어$\rightarrow$영어$\rightarrow$단순화$\rightarrow$스페인어 경로를 거치는 방식은 파인튜닝된 모델에서 오히려 성능이 저하되는 결과를 보였다.
- **E3 (Llama2-70B + Zero-shot)**: 파인튜닝 없는 대형 모델은 단순한 단어 교체 수준에 머물렀으며, 문장 구조의 근본적인 단순화는 이루어지지 않았다.
- **E4 (Llama2-70B + Translation)**: 영어 번역 과정을 거친 경우 E3보다 우수한 결과를 보였으며, 짧고 직접적인 문장을 생성하였다.
- **E5 (Llama2-70B + Detailed Guidelines)**: 21가지 상세 가이드라인을 프롬프트에 포함시킨 경우, 파인튜닝된 7B 모델보다 더 우수한 결과를 보였으며, 도메인 지식 없이도 높은 수준의 단순화가 가능함을 확인하였다.

### 오류 분석
전문가 평가 결과, 다음과 같은 주요 오류가 발견되었다.
- **성-수 일치 오류**: 다국어 LLM 특성상 스페인어의 성-수 일치(gender and number agreement)가 완벽하지 않은 경우가 발생하였다.
- **용어 일관성 부족**: 동일한 대상에 대해 '선수(players)'와 '운동선수(athletes)'를 혼용하여 사용하는 등 가이드라인 G11(동일 대상에 동일 단어 사용)을 위반하는 사례가 있었다.
- **전문 용어 설명 부족**: 'acta(score sheet)'와 같은 도메인 특화 용어를 설명 없이 그대로 사용하거나 부적절한 동의어로 대체하는 문제가 있었다.

## 🧠 Insights & Discussion

### 모델 규모와 도메인 지식의 관계
실험 결과, 모델의 크기가 작을 때는 특정 도메인 데이터로의 파인튜닝이 성능 향상에 결정적인 역할을 한다. 하지만 Llama2-70B와 같은 거대 모델의 경우, 정교하게 설계된 지침(Guidelines)만으로도 파인튜닝 없이 매우 높은 수준의 단순화 성능을 낼 수 있음을 시사한다. 이는 거대 모델이 이미 방대한 언어적 패턴을 학습하고 있어, 명확한 제약 조건만 주어진다면 이를 잘 수행할 수 있기 때문이다.

### 언어적 특성의 영향
스페인어는 영어에 비해 형태론적으로 복잡하고 동사 변화가 다양하며 종속절이 많은 특성이 있다. 이로 인해 단순한 단어 교체만으로는 Easy to Read 수준의 접근성을 확보하기 어려우며, 문장 구조 자체를 재구성하는 능력이 필수적이다.

### 한계 및 제언
본 연구는 상대적으로 작은 규모의 코퍼스를 사용하였으며, 일부 도메인에 치우쳐 있다. 향후 더 방대한 양의 Easy to Read 병렬 코퍼스를 구축하고, 도메인별 전문 용어 사전(Terminology Dictionary)을 학습 과정에 통합한다면, 전문 용어 설명 부족 문제를 해결하고 더욱 정밀한 단순화 시스템을 구축할 수 있을 것이다.

## 📌 TL;DR

본 논문은 지적 장애인을 위한 스페인어 Easy to Read 텍스트 자동 생성을 위해 **Llama2 모델을 활용한 단순화 방안**을 탐색하였다. 이를 위해 **스페인어 병렬 코퍼스와 LaBSE 기반의 문장 정렬 도구**를 구축하였으며, QLoRA를 통한 효율적인 파인튜닝과 상세 가이드라인 기반의 프롬프팅 전략을 비교 분석하였다. 결과적으로 **상세 가이드라인을 적용한 Llama2-70B 모델이 가장 우수한 성능**을 보였으며, 자동화 시스템이 인지적 접근성을 획기적으로 높일 수 있음을 입증하였다. 다만, 최종 결과물은 반드시 전문가의 검토를 거쳐야 함을 강조한다.
