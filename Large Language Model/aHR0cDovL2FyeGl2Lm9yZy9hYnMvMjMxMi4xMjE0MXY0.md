# Neuron-Level Knowledge Attribution in Large Language Models

Zeping Yu, Sophia Ananiadou (2024)

## 🧩 Problem to Solve

본 연구는 거대 언어 모델(LLM)의 최종 예측에 결정적인 역할을 하는 중요한 뉴런(Neuron)을 식별하는 문제를 다룬다. LLM이 사실적 지식(Factual Knowledge)을 저장하는 메커니즘을 이해하는 것은 모델의 해석 가능성을 높이고, 향후 지식 편집(Knowledge Editing) 연구를 수행하는 데 있어 필수적이다.

기존의 지식 속성 분석(Knowledge Attribution) 기술들은 주로 Causal Tracing이나 Integrated Gradients와 같은 방법론에 의존해 왔다. 그러나 이러한 방식들은 수백만 개의 뉴런이 존재하는 LLM 구조에서 순전파(Forward pass)와 역전파(Backward pass) 연산을 반복적으로 수행해야 하므로, 계산 복잡도가 매우 높아 뉴런 수준(Neuron-level)의 정밀한 분석을 수행하기에는 현실적인 제약이 크다. 또한, 기존의 정적(Static) 분석 방법들은 서로 간의 성능 비교가 부족하거나, Attention 모듈과 Feed-Forward Network(FFN) 모듈 중 어느 한쪽으로만 치우쳐 분석하는 경향이 있다는 한계가 있다.

따라서 본 논문의 목표는 계산 효율성이 높으면서도 정확하게 중요 뉴런을 찾아낼 수 있는 정적 속성 분석 방법을 제안하고, 이를 통해 지식이 모델 내부의 Attention 및 FFN 레이어에 어떻게 저장되고 흐르는지 분석하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 뉴런이 최종 예측의 확률 분포에 미치는 영향력을 정량화하기 위해 **로그 확률 증가량(Log probability increase)**을 중요도 점수로 사용하는 것이다.

단순히 뉴런의 가중치나 활성화 값만을 보는 것이 아니라, 특정 뉴런의 출력이 추가되었을 때 정답 토큰의 로그 확률이 얼마나 상승하는지를 측정함으로써, 최종 예측에 직접적으로 기여하는 '가치 뉴런(Value Neurons)'을 식별한다. 더 나아가, 이러한 가치 뉴런을 활성화시키는 '쿼리 뉴런(Query Neurons)'을 식별하기 위해 서브키(Subkey)와 잔차 연결(Residual stream) 간의 내적(Inner product)을 계산하는 정적 방법을 제안한다.

## 📎 Related Works

기존의 속성 분석 방법론은 크게 두 가지 방향으로 나뉜다. 첫째는 Gradient 기반 방법 및 Causal Tracing 방법으로, 내부 모듈이 최종 예측에 미치는 영향을 직접 계산하므로 정확도가 높지만 계산 비용이 매우 크다. 이로 인해 뉴런 단위보다는 은닉 상태(Hidden states) 단위의 분석에 주로 사용되었다. 둘째는 Attention 가중치나 FFN 뉴런의 계수(Coefficient)를 이용하는 Saliency score 기반의 정적 방법이다. 이 방법들은 단 한 번의 순전파만으로 분석이 가능해 효율적이지만, 실제 예측 결과와의 충실성(Faithfulness)에 대해 학술적 논쟁이 지속되고 있다.

본 연구는 Mechanistic Interpretability 분야의 Logit Lens(내부 벡터를 어휘 공간으로 투영하여 해석하는 기법) 개념을 활용하여, 정적 방법론의 효율성을 유지하면서도 로그 확률 변화라는 더 정밀한 지표를 도입함으로써 기존 Saliency score 방법들의 한계를 극복하고자 한다.

## 🛠️ Methodology

### 1. 기본 정의 및 뉴런의 정의
모델의 출력은 각 레이어의 입력 $h^{l-1}_i$에 Attention 출력 $A^l_i$와 FFN 출력 $F^l_i$가 더해진 형태로 계산된다.
$$h^l_i = h^{l-1}_i + A^l_i + F^l_i$$

본 논문에서는 뉴런을 다음과 같이 정의한다.
- **FFN 뉴런**: FFN 출력 $F^l_i$를 구성하는 가중치 행렬 $W^l_{fc2}$의 각 열(Column)인 subvalue $fc2^l_k$를 하나의 뉴런으로 간주한다.
- **Attention 뉴런**: Attention 출력 $A^l_i$에서 각 헤드의 출력 행렬 $W^o_{j,l}$의 각 열을 subvalue로, $W^v_{j,l}$의 각 행을 subkey로 정의하여 뉴런 단위로 분석한다.

### 2. 가치 뉴런(Value Neurons) 식별 방법
뉴런 $v$가 추가되었을 때의 확률 변화를 분석하기 위해, 저자들은 **bs-value(Before-softmax value)** 개념을 도입한다. 특정 토큰 $w$에 대한 bs-value는 $e_w \cdot x$로 계산되며, 이는 소프트맥스 함수 적용 전의 로짓(Logit) 값에 해당한다.

가치 뉴런의 중요도 점수 $\text{Imp}(v^l)$는 다음과 같이 로그 확률의 차이로 정의한다.
$$\text{Imp}(v^l) = \log(p(w|v^l + h^{l-1})) - \log(p(w|h^{l-1}))$$
여기서 $v^l$은 분석 대상이 되는 뉴런의 벡터이며, $h^{l-1}$은 해당 레이어의 입력 벡터이다. FFN 레이어의 경우 $h^{l-1}$ 대신 $h^{l-1} + A^l$을 사용하여 Attention의 영향을 포함시킨다.

### 3. 쿼리 뉴런(Query Neurons) 식별 방법
식별된 가치 뉴런들이 어떻게 활성화되는지를 찾기 위해 쿼리 뉴런을 추적한다. FFN의 활성화 값(coefficient score)은 subkey와 입력 벡터의 내적으로 결정된다. 따라서 가치 뉴런의 subkey와 잔차 연결 내의 다른 뉴런/서브벡터 간의 내적 값을 계산하여, 내적 값이 클수록 해당 가치 뉴런을 활성화시키는 데 기여도가 높은 쿼리 뉴런으로 판단한다.

## 📊 Results

### 1. 실험 설정
- **데이터셋**: TriviaQA에서 추출한 6가지 지식 유형(언어, 수도, 국가, 색상, 숫자, 월)을 사용한다.
- **모델**: GPT2-large 및 Llama-7B를 대상으로 실험을 진행하였다.
- **비교 대상**: 제안 방법(로그 확률 증가량)을 포함하여 총 8가지의 정적 속성 분석 방법을 비교하였다 (예: Norm, Coefficient score, Vocabulary space ranking 등).
- **평가 지표**: 식별된 상위 10개 뉴런을 제거(Zero-out)했을 때, 정답 토큰의 MRR, 확률($prob$), 로그 확률($logp$)이 얼마나 크게 감소하는지를 측정하여 분석 방법의 정확성을 평가하였다.

### 2. 주요 결과
- **정량적 성능**: 제안된 로그 확률 증가량 방법이 모든 지표에서 가장 큰 감소를 이끌어냈다. 이는 본 방법이 실제 예측에 가장 영향력이 큰 핵심 뉴런을 정확하게 식별했음을 의미한다.
- **지식 저장 위치**: 
    - 분석 결과, 지식을 저장하는 가치 뉴런들은 주로 **깊은 레이어(Deep layers)**에 분포한다.
    - 유사한 의미론적 특성을 가진 지식(예: 국가, 수도, 언어)은 동일한 레이어나 헤드에 저장되는 경향이 있으며, 서로 다른 특성(예: 색상, 숫자)은 서로 다른 위치에 저장된다.
- **뉴런 간의 상호작용 (Information Flow)**:
    - **얕은/중간 레이어의 FFN 뉴런** $\rightarrow$ **중간/깊은 레이어의 Attention 뉴런** $\rightarrow$ **깊은 레이어의 FFN/Attention 뉴런** 순으로 정보가 흐르는 메커니즘을 발견하였다.
    - 구체적으로, 얕은 층의 FFN 쿼리 뉴런들이 특징을 추출하여 깊은 층의 가치 뉴런들을 활성화시키는 구조를 보인다.

## 🧠 Insights & Discussion

본 연구는 계산 효율적인 정적 방법만으로도 LLM 내부의 지식 저장 위치와 흐름을 뉴런 수준에서 정밀하게 추적할 수 있음을 입증하였다. 특히, 단순한 가중치 분석이 아니라 로그 확률의 변화라는 동적인 관점을 정적 분석에 도입함으로써 분석의 충실성을 높였다는 점이 강점이다.

다만, 본 연구는 6가지의 특정 지식 유형에 한정하여 분석을 진행하였으며, GPT2와 Llama라는 두 가지 모델만을 대상으로 했다는 한계가 있다. 또한, 정적 방법이 효율적이긴 하지만 Causal Mediation Analysis와 같은 동적/인과적 분석 방법과 직접적으로 비교하여 어느 정도의 정확도 차이가 발생하는지에 대한 추가 연구가 필요하다.

비판적으로 해석하자면, 본 방법론을 통해 특정 지식 뉴런을 정확히 찾아낼 수 있다는 것은 역으로 모델의 특정 지식을 임의로 수정하거나 삭제하는 '지식 편집'이나 '공격'에 악용될 위험이 있음을 시사한다. 하지만 저자들은 이를 통해 환각(Hallucination) 제거, 독성(Toxicity) 및 편향성 완화 등 모델의 안전성을 높이는 방향으로 활용될 수 있다고 논의한다.

## 📌 TL;DR

본 논문은 LLM에서 특정 사실적 지식을 저장하는 뉴런을 효율적으로 찾기 위해 **로그 확률 증가량** 기반의 정적 속성 분석 방법을 제안하였다. 실험을 통해 이 방법이 기존의 단순 Saliency score 방식보다 정확하게 핵심 뉴런을 식별함을 보였으며, 지식이 **[얕은 FFN(Query) $\rightarrow$ 중간 Attention $\rightarrow$ 깊은 FFN/Attention(Value)]** 순으로 흐른다는 메커니즘을 밝혀냈다. 이 연구는 향후 LLM의 지식 편집 및 모델 해석 가능성 연구에 중요한 기초 자료가 될 것으로 기대된다.
