# Beneath the Surface: Unveiling Harmful Memes with Multimodal Reasoning Distilled from Large Language Models

Hongzhan Lin, Ziyang Luo, Jing Ma, Long Chen (2023)

## 🧩 Problem to Solve

현대 사회의 소셜 미디어에서 밈(meme)은 매우 빠르게 확산되는 다중 모달 콘텐츠이다. 그러나 일부 밈은 이미지와 텍스트의 결합을 통해 정치적, 사회문화적 분열을 조장하거나 특정 집단에 대한 혐오를 퍼뜨리는 '유해한 밈(harmful memes)'으로 작용한다.

유해한 밈 탐지의 핵심적인 어려움은 유해성이 텍스트나 이미지 단일 모달리티의 표면적인 정보가 아니라, 두 정보의 상호작용을 통한 **암시적 의미(implicit meaning)**에 숨겨져 있다는 점이다. 예를 들어, 이미지와 텍스트를 각각 개별적으로 보았을 때는 무해하지만, 함께 결합되었을 때만 특정 인종에 대한 스테레오타입을 강화하는 유해성이 드러나는 경우가 많다. 기존의 탐지 방식들은 주로 엔드투엔드(end-to-end) 분류 방식으로 접근하여 표면적인 신호만을 인식하며, 깊이 있는 인지적 추론 과정을 생략한다는 한계가 있다. 따라서 본 논문은 LLM(Large Language Models)의 복잡한 추론 능력을 활용하여 밈의 내면에 숨겨진 유해성을 밝혀내는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 LLM이 가진 풍부한 상식과 배경 지식을 활용하여 밈의 유해성에 대한 **가추법적 추론(abductive reasoning)** 과정을 생성하고, 이를 작은 언어 모델(Small LM)에 전이(distill)시키는 것이다.

주요 기여 사항은 다음과 같다.
1. **LLM 기반 추론 지식 추출**: 유해한 밈 탐지 분야에서 LLM의 상식 지식을 명시적으로 활용하여 표면적 이해의 한계를 극복하려 시도한 첫 번째 연구이다.
2. **생성적 프레임워크 제안**: LLM으로부터 추출한 추론 과정(rationale)을 학습하여 다중 모달 융합 능력을 높이고 경량화된 미세 조정을 가능하게 하는 새로운 생성 프레임워크인 **MR.HARM**을 제안한다.
3. **2단계 학습 파이프라인**: '추론 증류(Reasoning Distillation)'와 '유해성 추론(Harmfulness Inference)'이라는 두 단계의 학습 과정을 통해 모델이 단순 분류가 아닌 인지적 추론을 통해 유해성을 판단하도록 설계하였다.

## 📎 Related Works

### 기존 유해 밈 탐지 연구
기존 연구들은 주로 텍스트와 이미지 인코더를 통해 특징을 추출한 후, 어텐션 메커니즘이나 다중 모달 융합 기술을 사용하는 2-스트림 모델을 채택하였다. 최근에는 VisualBERT, ViLBERT와 같은 사전 학습된 다중 모달 모델을 미세 조정하거나, 이미지 캡셔닝 결과를 프롬프트로 사용하는 MaskPrompt와 같은 방식이 제안되었다. 그러나 이러한 방식들은 여전히 인식 수준(recognition-level)의 표면적 신호 포착에 집중하며, 시각적 요소와 텍스트 요소 간의 복잡한 관계를 이해하기 위한 명시적인 연역적 추론 과정을 결여하고 있다.

### LLM 및 지식 증류(Knowledge Distillation)
LLM은 Chain-of-Thought(CoT)와 같은 기법을 통해 복잡한 추론 능력을 보여주었으나, 모델의 거대한 크기로 인해 실제 밈 탐지 서비스에 직접 배치하기에는 효율성이 떨어진다. 기존의 지식 증류 연구는 주로 언어 모달리티에만 집중되어 있어, 이미지와 텍스트의 시너지 효과가 중요한 유해 밈 탐지 작업에는 부적합하다. 본 논문은 이를 해결하기 위해 LLM을 교사 모델로, 작은 LM을 학생 모델로 하여 다중 모달 추론 지식을 전이하는 방식을 취한다.

## 🛠️ Methodology

본 논문은 유해 밈 탐지 작업을 자연어 생성 패러다임으로 전환하여, 이미지 $I$와 텍스트 $T$를 입력받아 유해성 여부 $y$를 포함한 텍스트 시퀀스를 생성하도록 설계하였다.

### 1. LLM을 이용한 가추법적 추론 (Abductive Reasoning)
학습 데이터셋의 각 샘플 $\{y, I, T\}$에 대해, 먼저 이미지 $I$를 텍스트 캡션 $\tilde{I}$로 변환한다. 이후 LLM(ChatGPT)에게 다음과 같은 프롬프트를 제공하여 유해성 판단의 근거가 되는 **추론 과정(rationale, $r$)**을 생성하게 한다.
- **프롬프트 구성**: "텍스트 $[T]$가 이미지 $[\tilde{I}]$에 포함되어 있고, 유해성 라벨이 $[y]$일 때, 라벨을 직접 언급하지 말고 이 밈이 왜 $[y]$로 판단되는지에 대한 간결한 추론 과정을 제공하라."
이렇게 생성된 $r$은 학생 모델이 학습해야 할 정답지 역할을 한다.

### 2. 추론 증류 (Reasoning Distillation) - 1단계 학습
작은 언어 모델(T5 기반)이 LLM의 추론 지식을 학습하는 단계이다.

**인코딩 과정:**
텍스트 $T$와 이미지 $I$를 각각 임베딩 벡터로 변환한다.
$$H^0_T = TE(T), \quad H_I = VE(I)$$
여기서 $TE$는 텍스트 임베딩 층, $VE$는 고정된 CLIP ViT-B/32 Vision Extractor이다. 이후 교차 어텐션(Cross-Attention) 메커니즘을 통해 텍스트 쿼리($Q_T$)와 이미지 키/값($K_I, V_I$)을 융합하여 다중 모달 표현을 얻는다.
$$H^i_I = \text{softmax}\left(\frac{Q_T K_I^\top}{\sqrt{d_k}}\right) V_I$$
최종적으로 융합된 표현 $\hat{H}$는 LM Encoder의 각 층을 거치며 업데이트된다.

**디코딩 및 손실 함수:**
융합된 표현 $\hat{H}$를 LM Decoder에 입력하여 추론 과정 $r$을 생성하며, 교차 엔트로피(Cross-Entropy) 손실 함수를 통해 학습한다.
$$\mathcal{L}_{distill} = CE(f(I, T), r)$$

### 3. 유해성 추론 (Harmfulness Inference) - 2단계 학습
1단계에서 학습된 모델을 기반으로, 최종적으로 유해성 라벨 $y$를 예측하도록 미세 조정하는 단계이다. 모델 구조와 인코딩 과정은 동일하며, 출력 대상만 라벨 $y$로 변경된다.
$$\mathcal{L}_{infer} = CE(f(I, T), y)$$
이 과정을 통해 모델은 1단계에서 습득한 '추론하는 방법'을 실제 유해성 판단에 활용하게 된다.

## 📊 Results

### 실험 설정
- **데이터셋**: Harm-C (코로나19 관련), Harm-P (미국 정치 관련), FHM (Facebook Hateful Memes Challenge)의 3가지 벤치마크를 사용하였다.
- **비교 대상**: Text BERT, Image-Region, Late Fusion, MMBT, VisualBERT, ViLBERT, MOMENTA, MaskPrompt 등 최신 SOTA 모델들과 비교하였다.
- **평가 지표**: Accuracy와 Macro-F1 score를 사용하였다.

### 주요 결과
실험 결과, **MR.HARM**은 모든 데이터셋에서 기존 베이스라인을 상회하는 성능을 보였다. 특히 Macro-F1 기준, 최적의 베이스라인 대비 Harm-C에서 2.63%, Harm-P에서 1.31%, FHM에서 9.86%의 성능 향상을 기록하였다.

- **데이터셋 난이도와의 관계**: FHM과 같이 더 복잡하고 도전적인 데이터셋일수록 MR.HARM의 성능 향상 폭이 컸다. 이는 인식 수준의 모델보다 추론 수준의 모델이 복잡한 밈을 더 잘 처리함을 시사한다.
- **소형 모델의 효율성**: T5-Base(약 220M 파라미터)를 사용한 MR.HARM이 더 큰 모델인 MaskPrompt(300M 이상)보다 뛰어난 성능을 보였다.

### 절제 실험 (Ablation Study)
- **추론 증류 제거**: 1단계 과정 없이 2단계만 수행했을 때 성능이 크게 하락하여, LLM으로부터 전이된 추론 지식의 중요성이 입증되었다.
- **2단계 학습 제거**: 추론 과정과 라벨을 한 번에 학습시키는 방식은 두 작업 간의 상호 간섭으로 인해 성능과 안정성이 저하되었다.

## 🧠 Insights & Discussion

### 인지 관점의 추론 분석 (Cognition-view Analysis)
본 모델은 최종 라벨뿐만 아니라 중간 추론 과정을 생성할 수 있다. 분석 결과, 모델은 단순히 단어를 매칭하는 것이 아니라 "이미지의 정치인 모습이 텍스트의 'AMERICA'와 연결되어 특정 정치적 의도를 가진다"는 식의 배경 지식 기반 추론을 수행하고 있었다. 이는 모델의 판단 근거를 인간이 확인 가능하게 함으로써 설명 가능성(Explainability)을 제공한다.

### 한계점 및 오류 분석
오류 분석 결과, 모델이 여전히 **매우 깊은 배경 지식**이 필요한 이미지를 인식하는 데 어려움이 있음이 밝혀졌다. 예를 들어, KKK(Ku Klux Klan) 단원들의 복장을 단순한 '해변의 여성들'로 오인하여 유해한 밈을 무해하다고 판단하는 사례가 발견되었다. 이는 시각적 특징 추출 단계에서 더 정교한 배경 지식 통합이 필요함을 의미한다.

### 비판적 해석
본 연구는 LLM의 능력을 경량 모델로 성공적으로 전이시켰으나, 추론 과정의 품질을 정량적으로 평가하는 지표가 부족하며 주로 정성적 분석에 의존하고 있다. 또한, 텍스트 캡션을 통해 LLM에 입력하는 방식은 캡셔닝 과정에서 정보 손실이 발생할 가능성이 크므로, 향후 Vision-Language LLM(VLM)을 직접 활용한 증류 방식이 더 효과적일 것으로 판단된다.

## 📌 TL;DR

본 논문은 유해 밈의 암시적 의미를 파악하기 위해 **LLM의 추론 능력을 작은 언어 모델에 전이하는 MR.HARM 프레임워크**를 제안한다. LLM을 통해 밈의 유해성 근거(Rationale)를 생성하고, 이를 1단계(추론 증류)와 2단계(유해성 추론)의 순차적 학습을 통해 소형 모델에 학습시켰다. 실험 결과, 기존 SOTA 모델들을 뛰어넘는 성능을 보였으며, 특히 복잡한 추론이 필요한 데이터셋에서 강점을 보였다. 이 연구는 향후 다중 모달 콘텐츠의 유해성 탐지 및 설명 가능한 AI 연구에 중요한 기여를 할 가능성이 높다.
