# Capability Instruction Tuning: A New Paradigm for Dynamic LLM Routing

Yi-Kai Zhang, De-Chuan Zhan, Han-Jia Ye (2025)

## 🧩 Problem to Solve

본 논문은 거대 언어 모델(Large Language Models, LLMs)의 높은 추론 비용과 계산 자원 소모 문제를 해결하고자 한다. 일반적으로 1,000억 개 이상의 파라미터를 가진 거대 모델이 뛰어난 성능을 보이지만, 이는 막대한 비용을 수반한다. 저자들은 여러 개의 작은 규모 모델(Smaller LLMs)들이 각자 서로 다른 강점을 가지고 있으며, 이를 적절히 조합하여 최적의 모델에 지시문을 할당하는 Model Routing을 통해 거대 모델에 필적하는 성능을 낼 수 있다는 점에 주목한다.

기존의 Model Routing 방식, 특히 Re-ranking 기반 방법론들은 후보 모델들의 추론 결과(inference outputs)를 먼저 얻은 뒤 최적의 답변을 선택해야 하므로, 후보 모델의 수가 늘어날수록 추론 지연 시간이 선형적으로 증가하는 심각한 오버헤드 문제가 존재한다. 따라서 본 연구의 목표는 후보 모델의 실제 추론 결과나 정답(ground truth) 없이도, 모델의 능력 표현(capability representation)만을 이용해 최적의 모델을 빠르게 선택할 수 있는 새로운 라우팅 패러다임을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 **Capability Instruction Tuning**이라는 새로운 패러다임을 제안한 것이다. 이 아이디어는 모델의 능력을 마치 '적성 검사(Aptitude Test)' 결과처럼 텍스트 형태의 표현으로 정량화하고, 이를 기반으로 특정 지시문을 처리할 수 있을지 예측하도록 라우터를 학습시키는 것이다.

주요 기여 사항은 다음과 같다:
1. **Capability Instruction Tuning 패러다임 제안**: 모델의 능력 표현, 사용자 지시문, 성능 질의 프롬프트를 결합하여 모델의 적합성을 판단하는 새로운 학습 방식을 정의하였다.
2. **MODEL-SAT 프레임워크 개발**: Capability Encoder와 경량 LLM을 결합한 엔드-투-엔드 라우터를 설계하여, 후보 모델의 추론 없이도 최적의 모델을 선택할 수 있게 하였다.
3. **포괄적인 벤치마크 구축**: 소규모, 혼합 규모, 고성능 모델군 및 멀티모달 시나리오를 포함하는 다양한 Model Routing 벤치마크를 설정하여 제안 방법의 유효성을 검증하였다.
4. **동적 적응 능력 입증**: 새로운 모델이 출시되었을 때, 단 50개의 핵심 과제에 대한 간단한 테스트만으로 빠르게 능력 표현을 생성하여 라우팅 시스템에 통합할 수 있음을 보여주었다.

## 📎 Related Works

기존의 모델 선택 및 라우팅 연구는 크게 세 가지 방향으로 진행되었다.
첫째, 특정 다운스트림 태스크에 최적화된 학습 초기값(initialization)을 찾는 연구들이 있었으나, 이는 주로 미세 조정(fine-tuning)이 필요한 상황을 가정한다.
둘째, LogME와 같은 전이 가능성(transferability) 측정 지표를 통해 모델을 선택하는 방식이 있었으나, 이는 레이블링된 데이터나 역전파(backpropagation) 단계가 필요한 경우가 많다.
셋째, 최근의 Re-ranking 기반 방식들은 모델들의 추론 결과 간의 대조적 관계를 학습하여 최적의 답변을 선택한다. 그러나 이는 앞서 언급했듯이 모든 후보 모델의 추론을 수행해야 하므로 모델 풀이 커질수록 실시간 서비스에 적용하기 어렵다는 한계가 있다.

본 논문은 후보 모델을 Black-box로 취급하며, 추론 결과가 아닌 모델의 특성을 나타내는 '능력 표현'을 입력으로 사용함으로써 기존 Re-ranking 방식의 지연 시간 문제를 근본적으로 해결하고 차별화를 꾀하였다.

## 🛠️ Methodology

### 1. Capability Instructions 구성
모델 라우팅을 위해 다음과 같은 세 가지 요소로 구성된 **Capability Instruction** $z_i$를 정의한다:
- **Capability Representation ($c_m$)**: MMLU 데이터셋에서 추출한 50개의 핵심 과제(task)에 대해 20-shot 테스트를 수행한 결과이다. 이 결과는 "수학-기하학에서 85% 정확도 달성"과 같이 자연어 형태로 기술된다.
- **User Instruction ($x_i$)**: 라우팅 대상이 되는 실제 사용자 지시문이다.
- **Performance Inquiry Prompt ($p$)**: 해당 모델이 지시문을 처리할 수 있는지 묻는 질의문이다 (예: "이 모델이 지시문을 처리할 수 있는지 'Yes' 또는 'No'로 예측하라").

### 2. MODEL-SAT 아키텍처
MODEL-SAT은 모델의 능력 표현과 지시문 사이의 세만틱 맵핑을 학습하기 위해 다음과 같은 구조를 가진다:
- **Capability Encoder ($\psi$)**: E5-Large ($\sim 0.5$B) 모델을 사용하여 텍스트 형태의 능력 표현 $c_m$을 벡터화한다.
- **Connector**: 단일 레이어 MLP를 사용하여 인코더의 출력 차원을 LLM의 입력 임베딩 차원과 맞춘다.
- **Lightweight LLM ($\phi$)**: Phi-3-Mini (3.8B) 모델을 사용하여 최종 판단을 내린다.

전체 과정은 다음과 같이 수식화된다. 정렬된 모델 능력 벡터 $e^c_m$은 다음과 같이 계산된다:
$$e^c_m = W \cdot \psi(c_m)$$
최종 입력 시퀀스 $z_i$는 모델 능력 벡터, 지시문 임베딩, 프롬프트 임베딩의 결합으로 구성된다:
$$z_i = [e^c_m, e_{x_i}, e_p]$$

### 3. 학습 절차 및 손실 함수
라우터는 특정 모델이 특정 지시문을 잘 수행할지 여부를 'Yes'라는 토큰의 생성 확률로 예측한다.
$$\text{Pr} ('\text{Yes}' | z_i)$$

학습 시에는 **Homogeneous In-Batch Negative Sampling**을 적용하여, 하나의 모델에 대해 잘 수행한 지시문(positive)과 그렇지 못한 지시문(negative)을 함께 배치에 넣어 변별력을 높인다. 손실 함수로는 Cross-Entropy Loss를 사용하여 배치 내에서 어떤 지시문이 positive인지 정확히 찾아내도록 최적화한다:
$$L_{CE} = \mathbb{E}_{Z \in D_{test}} [-\log \text{Pr} (h_{\phi, 'Yes'}(Z) = y_{pos} | Z)]$$
여기서 $h_{\phi, 'Yes'}(Z)$는 배치 $Z$ 내에서 'Yes' 확률이 가장 높은 지시문을 선택하는 함수이다.

학습은 2단계로 진행된다. 1단계에서는 Connector만 미세 조정하여 기본적인 맵핑을 형성하고, 2단계에서는 전체 파라미터를 미세 조정하되 인코더와 커넥터에 더 높은 학습률을 적용하여 정렬 성능을 극대화한다.

## 📊 Results

### 1. 실험 설정
- **후보 모델군 (Zoo)**: 10B 미만 소규모 모델군, 10B~70B 혼합 모델군, 70B 이상 고성능 모델군, 그리고 멀티모달 모델군으로 나누어 실험하였다.
- **평가 데이터셋**: MMLU, WinoGrande (In-domain) 및 ARC-Challenge, TruthfulQA, BoolQ, MRPC, MNLI (Out-of-domain) 등을 사용하였다.
- **비교 대상**: Random Selection, Re-ranking 기반(Cappy, BGE, GTE), 그리고 모델군 내에서 단일 최고의 성능을 내는 모델(Best-Performing)과 비교하였다.

### 2. 주요 결과
- **성능 향상**: 소규모 모델군(10B 미만)에서 MODEL-SAT을 이용한 라우팅 결과, 평균 성능이 $75.28\%$를 기록하며, 단일 최강 모델($74.16\%$)보다 높은 성능을 보였다. 특히 이는 70B 규모의 Llama-3-Instruct 성능($76.90\%$)에 근접하는 수치이다.
- **효율성**: Re-ranking 방식은 후보 모델 수에 따라 비용이 증가하지만, MODEL-SAT은 모델 수와 무관하게 일정한 추론 비용을 유지하며 지연 시간을 획기적으로 줄였다.
- **일반화 능력**: 학습에 사용되지 않은 새로운 모델이 추가되었을 때, 단 50개의 20-shot 과제 수행 결과만으로도 즉각적으로 효과적인 라우팅이 가능함을 확인하였다(Figure 5).
- **멀티모달 확장**: 멀티모달 LLM  zoo에서도 MMMU-VAL, AI2D-TEST 등에서 우수한 평균 성능을 기록하며 방법론의 범용성을 입증하였다.

## 🧠 Insights & Discussion

### 강점
본 연구의 가장 큰 강점은 **추론 오버헤드의 제거**와 **동적 확장성**이다. 기존의 라우터들이 모든 모델의 답변을 들어보고 결정하는 '사후 판단' 방식이었다면, MODEL-SAT은 모델의 적성 검사 결과를 보고 결정하는 '사전 판단' 방식을 도입함으로써 실시간 서비스 적용 가능성을 극대화하였다. 또한, 새로운 모델이 출시될 때마다 라우터를 재학습시킬 필요 없이 능력 표현(텍스트)만 추가하면 된다는 점은 매우 실용적이다.

### 한계 및 비판적 해석
논문에서 제시한 '50개의 핵심 과제'가 모델의 전체 능력을 충분히 대표할 수 있는가에 대한 의문이 남는다. 특정 도메인에서 매우 특이한 강점을 가진 모델의 경우, 일반적인 MMLU 기반의 적성 검사로는 그 능력이 포착되지 않을 가능성이 있다. 또한, 능력 표현을 자연어로 기술하는 방식이 LLM의 내부 임베딩 공간에서 얼마나 정교하게 작동하는지에 대한 심층적인 분석이 부족하며, 단순히 'Yes/No' 확률로 판단하는 것이 정교한 성능 예측(예: 구체적인 정확도 수치 예측)보다 항상 유리한지에 대한 논의가 필요하다.

## 📌 TL;DR

본 논문은 LLM 라우팅 시 발생하는 막대한 추론 비용을 해결하기 위해, 모델의 능력을 텍스트로 정량화한 '능력 표현(Capability Representation)'을 기반으로 하는 **Capability Instruction Tuning** 패러다임과 **MODEL-SAT** 프레임워크를 제안한다. 이 방법은 후보 모델들의 실제 추론 없이도 최적의 모델을 선택할 수 있게 하며, 소규모 모델들의 조합만으로 거대 모델에 필적하는 성능을 낼 수 있음을 보였다. 특히 새로운 모델의 추가에 매우 유연하게 대응할 수 있어, 향후 동적으로 변하는 LLM 생태계에서 효율적인 모델 오케스트레이션을 가능하게 하는 핵심 기술이 될 가능성이 높다.
