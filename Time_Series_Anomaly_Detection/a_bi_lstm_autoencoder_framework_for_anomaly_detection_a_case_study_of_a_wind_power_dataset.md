# A Bi-LSTM Autoencoder Framework for Anomaly Detection – A Case Study of a Wind Power Dataset

Ahmed Shoyeb Raihan and Imtiaz Ahmed (n.d.)

## 🧩 Problem to Solve

본 논문은 시계열 데이터에서 발생하는 이상치(Anomaly)를 효과적으로 탐지하는 문제를 해결하고자 한다. 이상치란 정상적이고 균질한 이벤트에서 벗어난 데이터 포인트나 이벤트로, 금융 사기, 네트워크 침입, 장비 고장, 프로세스 변경 등 드물지만 중요한 사건들을 포함한다. 이러한 이상치를 신속하게 탐지하는 것은 재정적 손실, 정보 유출 및 인적 자원 피해를 방지하는 데 매우 중요하다.

특히 시계열 데이터의 이상치 탐지는 시간 차원으로 인한 복잡성이 추가되기 때문에 일반적인 데이터보다 더 까다롭다. 따라서 본 연구의 목표는 Bidirectional Long Short Term Memory (Bi-LSTM) 아키텍처와 Autoencoder를 결합한 새로운 프레임워크를 제안하여, 다변량 시계열 데이터에서 정교한 이상치 탐지 성능을 확보하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 Bi-LSTM의 양방향 문맥 파악 능력과 Autoencoder의 데이터 재구성(Reconstruction) 능력을 결합하는 것이다.

1. **Bi-LSTM을 통한 양방향 특징 추출**: 기존의 단방향 LSTM은 과거에서 미래 방향으로만 데이터를 처리하지만, Bi-LSTM은 데이터를 정방향과 역방향 모두에서 분석한다. 이를 통해 시퀀스 데이터에 숨겨진 장기 의존성(Long-term dependencies)을 더 효과적으로 발견하고 풍부한 문맥 정보를 추출할 수 있다.
2. **Autoencoder 기반의 이상치 판별**: 정상 데이터만을 사용하여 학습된 Autoencoder는 정상 패턴의 압축된 표현을 학습한다. 새로운 데이터가 입력되었을 때, 모델이 이를 제대로 재구성하지 못해 발생하는 재구성 오차(Reconstruction Error)를 측정하고, 이를 통해 설정된 임계값(Threshold)을 초과하는 데이터를 이상치로 분류한다.

## 📎 Related Works

논문에서는 이상치 탐지를 위한 기존 접근 방식들의 한계를 다음과 같이 설명한다.

- **전통적 통계 방식**: 통계적 방법이나 시간 불변(Time-invariant) 방식들은 이상치의 복잡하고 끊임없이 변하는 특성을 처리하기에 부족하다.
- **기존 머신러닝 및 딥러닝 방식**: 많은 연구가 일반적/이상적 행동을 학습하는 데 치중하여, 새로운 이상치를 식별할 때 시간적 패턴(Temporal patterns)을 통합적으로 고려하지 못하는 한계가 있다.
- **단방향 LSTM**: RNN의 기울기 소실(Vanishing gradient) 문제를 해결하여 장기 의존성을 학습할 수 있으나, 데이터가 길고 복잡할 경우 한 방향으로만 처리하는 특성 때문에 성능 저하가 발생한다.

본 연구는 이러한 한계를 극복하기 위해 Bi-LSTM을 도입하여 양방향 문맥을 모두 활용하고, 이를 Autoencoder 구조에 통합하여 비지도 학습 기반의 이상치 탐지 시스템을 구축함으로써 기존 방식들과 차별화를 꾀하였다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

제안된 프레임워크는 **[시계열 데이터 입력 $\rightarrow$ Bi-LSTM Encoder $\rightarrow$ Bi-LSTM Decoder $\rightarrow$ 이상치 탐지 모듈]**의 4단계 파이프라인으로 구성된다.

### 2. 주요 구성 요소 및 역할

- **Bi-LSTM Encoder**: 고차원의 입력 시퀀스를 저차원의 고정 크기 벡터(Latent space)로 매핑한다. 본 모델에서는 64개 셀의 첫 번째 층과 32개 셀의 두 번째 층으로 구성된 두 개의 Bi-LSTM 층을 사용하여 데이터를 압축한다.
- **Bi-LSTM Decoder**: Encoder가 생성한 저차원 표현을 다시 원래의 고차원 입력 시퀀스로 복원한다. Encoder의 역순으로 32개 셀, 64개 셀의 Bi-LSTM 층을 배치하여 재구성된 데이터를 생성한다.
- **이상치 탐지 모듈**: 입력 데이터와 복원된 데이터 사이의 차이인 재구성 오차를 계산하여, 미리 설정된 임계값 $\tau$와 비교해 이상 여부를 판별한다.

### 3. 상세 메커니즘 및 방정식

#### (1) LSTM Unit

LSTM은 Forget, Input, Output 게이트를 통해 정보의 흐름을 조절한다.

- **Forget Gate**: 이전 은닉 상태 $h_{t-1}$과 현재 입력 $x_t$를 받아 어떤 정보를 버릴지 결정한다.
  $$f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f)$$
- **Input Gate**: 새로운 정보 중 무엇을 메모리 셀에 저장할지 결정한다.
  $$\tilde{C}_t = \tanh(W_C \cdot [h_{t-1}, x_t] + b_C)$$
  $$i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i)$$
  $$C_t = f_t * C_{t-1} + i_t * \tilde{C}_t$$
- **Output Gate**: 업데이트된 셀 상태를 바탕으로 최종 은닉 상태 $h_t$를 출력한다.
  $$o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o)$$
  $$h_t = o_t * \tanh(C_t)$$

#### (2) Autoencoder의 인코딩 및 디코딩

- **Encoding**: 입력 $x$를 저차원 표현 $h$로 변환한다.
  $$h = \phi(W_e x + b_e)$$
- **Decoding**: $h$를 통해 원래 입력과 유사한 $\hat{x}$를 재구성한다.
  $$\hat{x} = \psi(W_d h + b_d)$$

#### (3) 재구성 손실 함수 및 이상치 판별

모델은 평균 제곱 오차(MSE) 형태의 재구성 손실 $L$을 최소화하도록 학습된다.
$$L = \frac{1}{n} \sum_{i=1}^{n} (x_i - \hat{x}_i)^2$$
추론 단계에서 특정 샘플의 재구성 오차가 임계값 $\tau$를 초과하면 이를 이상치로 간주한다.
$$\text{If } L(x, \hat{x}) > \tau \implies \text{Anomaly}$$

## 📊 Results

### 1. 실험 설정

- **데이터셋**: 풍력 발전소(Wind farm)에서 수집된 2015년도 다변량 시계열 데이터 (총 30,997개 레코드).
- **입력 변수**: 평균 풍속($V$), 풍속 표준편차($V_s$), 평균 풍향($D$), 온도($T$)의 4가지 특징.
- **데이터 분할**: 학습 및 검증에 70%, 테스트에 30% 사용. 특히 모델은 **정상 데이터(Negatively labeled)로만 학습**되어 정상 패턴을 익히도록 설계되었다.
- **하이퍼파라미터**: Lookback period = 10, Learning rate = 0.0001, Batch size = 128.

### 2. 정량적 결과

제안된 Bi-LSTM Autoencoder 모델을 일반 LSTM Autoencoder 모델과 비교한 결과는 다음과 같다.

| Model | Accuracy | Precision | Recall | F1 Score |
| :--- | :---: | :---: | :---: | :---: |
| **Bi-LSTM Autoencoder** | **0.9679** | **0.7205** | **0.8846** | **0.7942** |
| LSTM Autoencoder | 0.9427 | 0.5598 | 0.8415 | 0.6724 |

- **정확도(Accuracy)**: 96.79%를 달성하여 LSTM AE보다 우수한 성능을 보였다.
- **AUC-ROC**: 0.97의 높은 점수를 기록하여 모델이 정상과 이상치를 매우 효과적으로 구분하고 있음을 입증하였다.

## 🧠 Insights & Discussion

### 1. 강점 및 해석

본 연구는 Bi-LSTM을 통해 시계열 데이터의 앞뒤 문맥을 모두 파악함으로써, 단순한 단방향 모델보다 훨씬 정교한 특징 추출이 가능함을 보여주었다. 특히 풍력 발전 데이터와 같이 환경 변수 간의 복잡한 상호작용과 시간적 흐름이 중요한 도메인에서 Bi-LSTM의 효용성이 입증되었다.

### 2. 한계 및 논의 사항

- **임계값($\tau$) 설정의 민감도**: 논문에서 언급되었듯, 임계값 설정은 모델 성능의 핵심이다. 임계값이 너무 낮으면 탐지율은 올라가지만 오탐(False Alarm)이 증가하고, 너무 높으면 미탐(Miss detection)이 증가하는 트레이드-오프 관계가 존재한다. 본 논문에서는 도메인 지식과 재구성 오차의 분포를 함께 활용하여 이를 결정하였다.
- **학습 데이터의 순수성**: 모델이 정상 데이터로만 학습되어야 한다는 가정이 필요하며, 학습 데이터에 이상치가 섞여 있을 경우 정상 패턴 학습에 방해가 되어 성능이 저하될 가능성이 있다.

### 3. 비판적 해석

실험 결과에서 Precision(0.7205)이 Recall(0.8846)보다 낮게 나타나는데, 이는 모델이 이상치를 놓치지 않고 잡아내는 능력(Recall)은 뛰어나지만, 정상 데이터를 이상치로 오인하는 경우(Precision 저하)가 상대적으로 더 많음을 의미한다. 이는 앞서 언급한 임계값 설정의 어려움과 관련이 있을 것으로 판단된다.

## 📌 TL;DR

본 논문은 풍력 발전 데이터의 이상치 탐지를 위해 **Bi-LSTM 기반의 Autoencoder** 프레임워크를 제안하였다. 양방향 LSTM을 통해 시계열의 문맥적 특징을 극대화하여 학습하고, 재구성 오차를 기반으로 이상치를 판별하는 방식을 취했다. 실험 결과, 기존 LSTM Autoencoder 대비 정확도(96.79%)와 F1 Score(0.7942) 등 모든 지표에서 우수한 성능을 보였으며, 이는 복잡한 다변량 시계열 데이터의 모니터링 및 고장 진단 시스템에 효과적으로 적용될 수 있음을 시사한다. 향후 연구로 Variational Autoencoder(VAE)와의 결합을 통한 성능 향상을 예고하고 있다.
