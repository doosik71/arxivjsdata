# mTSBench: Benchmarking Multivariate Time Series Anomaly Detection and Model Selection at Scale

Xiaona Zhou, Constantin Brif, Ismini Lourentzou (2025)

## 🧩 Problem to Solve

다변량 시계열 이상치 탐지(Multivariate Time Series Anomaly Detection, MTS-AD)는 헬스케어, 사이버 보안, 산업 모니터링 등 다양한 도메인에서 필수적인 기술이다. 그러나 다변량 시계열 데이터는 변수 간의 복잡한 의존성, 비선형적 시간적 역동성, 그리고 이상치 레이블의 희소성으로 인해 탐지가 매우 어렵다.

특히, 기존의 다양한 이상치 탐지 알고리즘 중 특정 데이터셋에 대해 어떤 모델이 최적인지를 결정하는 모델 선택(Model Selection) 문제는 매우 중요하다. 현재까지 어떤 단일 탐지기가 모든 시나리오에서 우수한 성능을 보이지 않는다는 점이 여러 연구를 통해 밝혀졌으나, 레이블이 없는 비지도 학습(Unsupervised) 환경에서 최적의 모델을 선택하는 표준화된 방법론과 이를 평가할 대규모 벤치마크가 부족한 상황이다. 따라서 본 논문은 MTS-AD 및 비지도 모델 선택 기법을 체계적으로 평가할 수 있는 통합 벤치마크인 mTSBench를 제안하여 이 문제를 해결하고자 한다.

## ✨ Key Contributions

본 연구의 핵심 기여는 다음과 같다.

1. **대규모 다변량 시계열 벤치마크 구축**: 12개 애플리케이션 도메인, 19개 데이터셋에서 추출한 344개의 레이블링된 다변량 시계열을 포함하는 mTSBench를 구축하였다. 이는 현재까지 공개된 가장 크고 다양한 MTS-AD 벤치마크이다.
2. **광범위한 모델 평가**: 재구성(Reconstruction), 예측(Prediction), 통계(Statistics) 및 최신 LLM 기반 탐지기를 포함한 24가지 이상치 탐지 방법론을 체계적으로 평가하였다.
3. **비지도 모델 선택 벤치마킹**: 최초로 MTS-AD를 위한 비지도 모델 선택 방법론들을 통합하여 표준화된 조건에서 벤치마킹하였으며, 최적 성능(Oracle) 및 사소한 기준선(Random)과의 간극을 분석하였다.
4. **통합 평가 체계 제공**: 포인트 기반(Point-based) 및 범위 기반(Range-based) 지표를 모두 포함하는 평가 스위트를 도입하여, 연구자들이 재현 가능하고 엄격한 비교 분석을 수행할 수 있도록 하였다.

## 📎 Related Works

### 기존 연구 및 한계

- **시계열 이상치 탐지**: 기존 연구들은 비지도 학습, 설명 가능성, 트랜스포머 기반 접근 방식 등을 제안해 왔으며, 최근에는 파운데이션 모델(Foundation Models)을 활용하려는 시도가 늘고 있다. 그러나 대부분의 포괄적 평가는 단변량(Univariate) 설정에 치중되어 있어, 다변량 데이터의 복잡한 의존성을 다루는 능력에 대한 이해가 부족하다.
- **모델 선택 전략**: 메타 학습(Meta-learning), 분류기 기반, 강화 학습, Human-in-the-loop 등 다양한 전략이 제안되었다. 하지만 이러한 방법들은 서로 다른 데이터셋과 작업에서 평가되어 일관성이 부족하며, 특히 다변량 시계열 환경에서의 성능 검증이 미비하다.
- **기존 벤치마크**: TODS, Timesead, TimeEval 등이 존재하지만, 다변량 시계열의 비중이 낮거나 모델 선택 성능을 평가하는 기능이 결여되어 있다.

### 차별점

mTSBench는 단순히 탐지기의 성능을 측정하는 것을 넘어, **"어떤 모델을 선택해야 하는가"**라는 모델 선택 문제에 집중하며, 이를 위해 다변량 특화 데이터셋과 모델 선택 전용 지표(Precision@k, Recall@k, NDCG)를 통합 제공한다는 점에서 기존 벤치마크와 차별화된다.

## 🛠️ Methodology

### 문제 정의 (Problem Definition)

다변량 시계열 $\mathcal{T}$는 다음과 같이 정의된다.
$$\mathcal{T} = \{(x_1, t_1), (x_2, t_2), \dots, (x_n, t_n)\}$$
여기서 $x_i \in \mathbb{R}^d$는 타임스탬프 $t_i$에서 관측된 $d$-차원 특징 벡터이며, $n$은 시계열의 길이이다. 이상치 탐지 모델 $m_j: \mathbb{R}^d \to \mathbb{R}$는 각 관측치 $x_i$에 대해 이상치 점수 $s_i$를 할당하며, 임계값 $\tau_j$를 통해 이진 예측 $\hat{y}_i$를 결정한다.
$$\hat{y}_i = I(s_i > \tau_j)$$

비지도 모델 선택의 목표는 레이블이 없는 테스트 시계열 $\mathcal{T}_{test}$에 대해, 모델 집합 $\mathcal{M} = \{m_1, \dots, m_k\}$ 중 최적의 모델 $m^*$를 찾는 것이다.
$$m^* = \arg \max_{m_j \in \mathcal{M}} \hat{P}(m_j, \mathcal{T}_{test})$$
여기서 $\hat{P}$는 데이터의 내재적 특성과 모델의 속성을 기반으로 성능을 추정하는 프록시 성능 함수이다.

### mTSBench 구성 요소

1. **이상치 탐지기 (24종)**:
    - **비지도(Unsupervised)**: PCA, IForest, LOF, KNN, COPOD, EIF 등 (레이블 없이 직접 적용).
    - **준지도(Semi-supervised)**: AutoEncoder, LSTM-AD, USAD, OmniAnomaly, TranAD 등 (정상 데이터로 학습).
    - **파운데이션 모델(LLM-based)**: ALLM4TS, OFA (사전 학습된 언어 모델 활용).
2. **모델 선택기 (3종)**:
    - **MetaOD**: PCA로 데이터 표현을 추출하고 Random Forest를 통해 모델 성능을 매핑하는 메타 학습 방식이다.
    - **FMMS**: Factorization Machine을 사용하여 데이터의 메타 특징과 성능 행렬 간의 2차 관계를 회귀 분석하는 방식이다.
    - **Orthus**: 22개의 단변량 메타 특징을 추출하고, 데이터가 기존 학습 데이터와 유사한지 여부에 따라 URegression 또는 CFact 방식을 선택하여 추천한다.
3. **평가 지표**:
    - **탐지 지표**: Precision, Recall, $F_1$, AUC-ROC, AUC-PR 및 범위 기반 지표인 VUS-ROC, VUS-PR 등을 사용한다.
    - **선택 지표**: 추천된 상위 $k$개 모델 중 실제 고성능 모델이 포함되었는지를 측정하는 Precision@k, Recall@k, NDCG를 사용한다.

## 📊 Results

### 이상치 탐지 성능 분석

- **단일 모델의 한계**: 특정 모델이 모든 데이터셋에서 우세하지 않음을 확인하였다. PCA는 단순함에도 불구하고 전반적으로 안정적인 성능을 보였으며, OmniAnomaly는 시간적 의존성 모델링 능력이 뛰어나 높은 점수를 기록하였다.
- **LLM 기반 탐지기**: OFA는 다양한 지표에서 높은 중앙값을 기록하며 전이 학습의 가능성을 보여주었으나, ALLM4TS는 데이터셋의 노이즈에 민감하여 성능 변동성이 크게 나타났다.
- **성능 하위 모델**: KNN, Transformer, LOF, HBOS 등은 특히 불균형 데이터셋(AUC-PR 지표)에서 매우 낮은 성능을 보였다.

### 모델 선택 성능 분석

- **선택기 비교**: Orthus와 FMMS가 MetaOD보다 일관되게 우수한 성능을 보였다. 특히 Orthus는 AUC-ROC와 R-based-$F_1$에서 강점을 보였으며, FMMS는 Recall과 Event-based-$F_1$에서 더 효과적이었다.
- **성능 간극(Gap)**: 가장 우수한 선택기(Orthus/FMMS)조차 'Near-optimal'(두 번째로 좋은 모델을 선택하는 기준선)과 비교했을 때 주요 지표에서 약 $15\% \sim 30\%$의 성능 차이가 발생하였다. 이는 현재의 비지도 모델 선택 기법이 여전히 최적의 모델을 찾는 데 한계가 있음을 시사한다.
- **차원별 분석**: 데이터 차원이 높을수록($d > 30$) FMMS의 성능이 향상되는 경향을 보였으며, Orthus는 모든 차원 범위에서 안정적인 성능을 유지하였다.

### 효율성 및 런타임

- **런타임**: FMMS가 대부분의 데이터셋에서 2초 미만의 처리 시간을 기록하며 가장 빨랐다. 반면, AnomalyTransformer와 OFA는 계산 비용이 가장 높았다.

## 🧠 Insights & Discussion

### 강점 및 발견

본 논문은 다변량 시계열 이상치 탐지 분야에서 가장 방대한 벤치마크를 구축함으로써, 단순한 모델 성능 비교를 넘어 **'적응형 모델 선택(Adaptive Model Selection)'**의 필요성을 정량적으로 입증하였다. 특히 LLM 기반 모델들이 잠재력은 있으나 전통적인 통계 모델(PCA 등)보다 항상 우월하지 않다는 점을 밝혀, 도메인 특화 어댑터의 필요성을 제시하였다.

### 한계 및 비판적 해석

- **단일 모델 선택의 제약**: 현재 벤치마크는 단 하나의 최적 모델을 선택하는 것에 집중하고 있다. 하지만 실제 환경에서는 여러 모델의 결과를 결합하는 앙상블(Ensemble) 방식이나 전문가 혼합(Mixture of Experts) 방식이 더 효과적일 수 있다.
- **하이퍼파라미터 튜닝**: 현실적인 배포 상황을 반영하기 위해 기본 설정(Default settings)을 사용하였으나, 이는 튜닝을 통해 성능을 극대화할 수 있는 모델들의 잠재력을 과소평가했을 가능성이 있다.
- **모델 선택기의 성능 부족**: Oracle과 Near-optimal 기준선과의 큰 간극은, 현재의 메타 특징(Meta-features) 추출 방식이 다변량 시계열의 복잡한 시간적/공간적 특성을 충분히 캡처하지 못하고 있음을 의미한다.

## 📌 TL;DR

본 논문은 19개 데이터셋, 344개 시계열로 구성된 최대 규모의 다변량 시계열 이상치 탐지 및 모델 선택 벤치마크인 **mTSBench**를 제안한다. 24종의 탐지기와 3종의 모델 선택기를 평가한 결과, 단일 모델로 모든 데이터를 커버할 수 없으며 기존 모델 선택 방법론들은 최적 성능에 도달하기에 여전히 부족함이 드러났다. 이 연구는 향후 문맥 인식(Context-aware) 모델 선택 및 파운데이션 모델의 적응형 튜닝 연구를 위한 표준화된 평가 기반을 제공한다.
