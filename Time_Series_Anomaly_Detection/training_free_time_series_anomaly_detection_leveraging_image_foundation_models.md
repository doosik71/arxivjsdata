# Training-Free Time-Series Anomaly Detection: Leveraging Image Foundation Models

Nobuo Namura, Yuma Ichikawa (2024)

## 🧩 Problem to Solve

본 논문은 시계열 이상치 탐지(Time-Series Anomaly Detection, TAD) 분야에서 딥러닝 모델들이 겪고 있는 실용적인 한계점을 해결하고자 한다. 기존의 딥러닝 기반 TAD 모델들은 데이터의 다양성으로 인해 학습 과정이 불안정하며, 각 케이스마다 방대한 하이퍼파라미터 튜닝이 필요하다는 단점이 있다. 이는 결국 전문적인 딥러닝 지식과 상당한 인적 비용을 요구하게 된다.

최근 대규모 데이터로 학습된 파운데이션 모델(Foundation Models)이 대안으로 제시되고 있으나, 시계열 분야에서는 여전히 높은 성능을 내기 위해 파인튜닝(Fine-tuning)이 필수적이며, 이는 앞서 언급한 학습의 불안정성과 튜닝 문제를 완전히 해결하지 못한다. 따라서 본 연구의 목표는 추가적인 신경망 학습이나 파인튜닝 없이도 높은 성능을 낼 수 있는 **학습이 필요 없는(Training-free) 이미지 기반 시계열 이상치 탐지(ITF-TAD)** 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 시계열 데이터를 이미지 형태로 변환하여, 이미 검증된 **이미지 파운데이션 모델(Image Foundation Model)**의 강력한 표현 능력을 그대로 활용하는 것이다.

중심적인 설계 직관은 다음과 같다.

1. **시간-주파수 변환**: 연속 웨이블릿 변환(Continuous Wavelet Transform, CWT)을 통해 시계열의 특성을 스칼로그램(Scalogram) 이미지로 변환하여 단순한 시점 기반 분석을 넘어선 패턴 분석을 가능하게 한다.
2. **차원 압축 및 매핑**: 다변량 시계열에서 발생하는 방대한 데이터를 PCA(주성분 분석)나 랜덤 행렬을 통해 효율적으로 압축하여 이미지 파운데이션 모델이 처리 가능한 입력 형태로 변환한다.
3. **학습 없는 탐지**: 학습된 ResNet 기반의 이미지 특징 추출기와 PatchCore 알고리즘을 결합하여, 별도의 재학습 없이 정상 데이터의 특징 분포(Coreset)만을 저장함으로써 이상치를 탐지한다.

## 📎 Related Works

기존의 비지도 학습 기반 TAD 방법론은 크게 세 가지로 분류된다.

- **유사도 기반(Similarity-based)**: 정상 데이터의 분포를 학습하고 거리나 확률 밀도를 측정하는 방식이다 (예: DAGMM, Deep-SVDD).
- **예측 기반(Prediction-based)**: 미래 값을 예측하고 실제 값과의 차이를 측정하는 방식이다 (예: ARIMA, LSTM, DiffAD).
- **재구성 기반(Reconstruction-based)**: 데이터를 압축 후 재구성하여 재구성 오차를 측정하는 방식이다 (예: LSTM-VAE, OmniAnomaly, Anomaly Transformer).

이미지 기반 접근 방식으로는 Fourier 변환이나 웨이블릿 변환을 사용하여 시계열을 2차원 이미지로 변환하는 연구들이 존재한다(예: TimesNet). 하지만 이러한 기존 방식들은 여전히 CNN 등의 신경망을 직접 학습시켜야 하며, 이 과정에서 발생하는 하이퍼파라미터 튜닝의 불안정성이라는 근본적인 문제가 남아 있었다. 본 논문은 이러한 '학습 과정' 자체를 제거하고 프리트레인된(Pre-trained) 모델을 그대로 사용한다는 점에서 차별점을 가진다.

## 🛠️ Methodology

ITF-TAD의 전체 파이프라인은 **스칼로그램 생성 $\rightarrow$ 스칼로그램 집계 $\rightarrow$ 이미지 기반 이상치 탐지**의 세 단계로 구성된다.

### 1. Dual Mother Wavelets를 이용한 스칼로그램 생성

다변량 시계열 $X = (x^d) \in \mathbb{R}^{D \times T}$ (여기서 $D$는 차원, $T$는 길이)에 대해, 각 차원 $x^d$를 CWT를 통해 시간-주파수 특성인 스칼로그램으로 변환한다.

$$S^d_{i,t,\omega} = \text{CWT}_{\phi_i}(x^d) \in \mathbb{R}^{T \times \Omega}$$

여기서 $\phi_i$는 마더 웨이블릿(Mother Wavelet)이며, 본 연구에서는 **Complex Morlet wavelet**과 **Ricker wavelet** 두 가지를 동시에 사용한다. Complex Morlet은 주파수 특성에 민감하고, Ricker는 위상 변화(Phase shift)에 민감하므로, 두 가지를 함께 사용함으로써 탐지 민감도를 극대화한다. 이후 각 스칼로그램은 최대 절대값으로 정규화된다.

### 2. 스칼로그램 집계 (Scalogram Aggregation)

다변량 데이터의 방대한 채널 수($D$)를 이미지 모델의 입력 채널 수(보통 3채널)로 맞추기 위해 압축 과정이 필요하다.

- **Component-Wise PCA Mapping**: 각 성분별로 PCA를 수행하여 상위 $\hat{\Omega}$개의 주성분을 추출한다. 이는 정보 손실을 최소화하며 시간 방향의 일관성을 유지한다.
- **Random Mapping**: PCA가 주성분(Backbone)에 집중하여 미세한 이상치를 놓칠 수 있다는 점을 보완하기 위해, 랜덤 텐서 $\Gamma$를 이용한 선형 변환을 수행한다.
  $$S^{\text{agg}}_{i,t,\omega} = \frac{\sum_{d=1}^{D} \gamma_{i,d,\omega} \bar{S}^d_{i,t,\omega}}{\sum_{d=1}^{D} \gamma_{i,d,\omega}}$$
- **Channel Embedding (RGB 구성)**: 최종적으로 RGB 이미지를 생성한다.
  - **Red, Green 채널**: 집계된 두 종류의 스칼로그램(Morlet, Ricker)을 배치한다.
  - **Blue 채널**: 주파수 인덱스 $\hat{\omega}$를 입력하여 **Frequency Encoding**을 수행한다. 이는 이미지 모델이 패치 단위로 분석할 때 주파수 위치 정보를 잃어버려 발생하는 오탐(False Positive)을 방지하기 위함이다.

### 3. 이미지 파운데이션 모델을 이용한 이상치 탐지

이미지로 변환된 데이터는 **PatchCore** 모델에 입력된다. PatchCore는 학습이 필요 없는 semi-supervised 모델로, 다음과 같은 절차를 따른다.

1. **특징 추출**: 프리트레인된 Wide ResNet-50-2를 사용하여 이미지의 특징 벡터를 추출한다.
2. **Coreset 구축**: 학습 데이터(정상)의 특징 벡터 중 대표적인 포인트들만을 선별하여 Coreset을 생성한다.
3. **점수 계산**: 테스트 이미지의 특징 벡터와 Coreset 간의 최단 거리를 측정하여 이상치 점수를 산출한다.

최종 시계열 이상치 점수 $\hat{y}_t$는 이미지 내 주파수 방향의 최대값으로 결정된다: $\hat{y}_t = \max_{\hat{\omega}} A_{t, \hat{\omega}}$.

## 📊 Results

### 실험 설정

- **데이터셋**: UCR(단변량), PSM, SMAP, MSL, SMD(다변량) 총 5종의 벤치마크 데이터셋을 사용하였다.
- **비교 대상**: GPT2-FPT, TimesNet, DiffAD, TranAD, Anomaly Transformer, GDN, LSTM 등 최신 딥러닝 기반 모델 7종과 비교하였다.
- **평가 지표**: 기존 Point Adjustment(PA)의 결함을 보완하기 위해 **Score Partitioning(SP)**을 적용한 $F1^*$-SP와 $\text{AUCPR-SP}$를 사용하였다.

### 주요 결과

- **성능**: ITF-TAD는 신경망 학습 없이도 대부분의 데이터셋에서 기존 딥러닝 모델들과 대등하거나 더 우수한 성능을 보였다. 특히 단변량 데이터셋인 **UCR에서 압도적인 성능 향상**을 보였으며, 이는 스칼로그램이 원본 시계열보다 더 풍부한 정보를 제공함을 시사한다.
- **효율성**: 하이퍼파라미터 튜닝 시간이 전혀 필요 없다는 점에서 실용성이 매우 높다. 다만, CWT 연산 및 PatchCore의 Coreset 생성 시간으로 인해 순수 추론 시간은 기존 모델과 비슷하거나 약간 더 소요될 수 있다.
- **정성적 분석**: 시각화 결과, ITF-TAD는 이상 구간에서 매우 날카로운 피크(Spike)를 생성하여 오탐이 적고 정확도가 높음을 확인하였다. 반면, 다른 모델들은 정상 구간에서도 많은 오탐을 생성하는 경향이 있었다.

## 🧠 Insights & Discussion

### 강점 및 해석

본 연구는 "시계열 $\rightarrow$ 이미지 $\rightarrow$ 이미지 파운데이션 모델"이라는 파이프라인을 통해 TAD의 고질적인 문제인 **학습 불안정성과 튜닝 비용**을 완전히 제거하였다. 특히, 서로 다른 특성을 가진 두 종류의 마더 웨이블릿을 사용하고 주파수 인코딩을 Blue 채널에 삽입한 설계가 성능 향상에 결정적인 역할을 했음을 Ablation Study를 통해 입증하였다.

### 한계 및 비판적 논의

1. **SMD 데이터셋에서의 성능 저하**: SMD 데이터셋에서는 상대적으로 낮은 성능을 보였는데, 이는 스칼로그램의 특성상 이상치 점수의 분포가 넓게 퍼지는 경향이 있어 지표 계산 시 오탐이 증가하기 때문으로 분석된다. 이는 모델의 탐지 능력 자체보다는 평가 지표와의 상성 문제일 가능성이 크다.
2. **연산 비용**: 학습은 필요 없지만, CWT 변환과 이미지 생성 과정의 연산량이 적지 않다. 특히 다변량 데이터의 차원 $D$가 클수록 PCA 연산 비용이 증가하며, 이를 해결하기 위해 제안된 Random Mapping(RM)이 더 실용적인 대안이 될 수 있다.
3. **범용성**: 이미지 파운데이션 모델의 입력 해상도(256x256)에 맞춰 데이터를 분할해야 하므로, 윈도우 크기 설정에 따른 제약이 존재한다.

## 📌 TL;DR

본 논문은 시계열 데이터를 CWT를 통해 이미지로 변환하고, 이를 프리트레인된 이미지 파운데이션 모델(PatchCore)로 분석하여 **학습과 튜닝이 전혀 필요 없는(Training-free)** 이상치 탐지 방법론인 ITF-TAD를 제안하였다. 실험 결과, 기존의 복잡한 딥러닝 모델들에 비해 실용성이 월등히 높으면서도 대등하거나 더 우수한 탐지 성능을 보였으며, 특히 단변량 시계열에서 탁월한 성과를 거두었다. 이 연구는 시계열 분석에 이미지 파운데이션 모델을 직접적으로 활용하는 새로운 방향성을 제시하며, 향후 더 효율적인 정보 압축 방법과 다양한 파운데이션 모델의 적용 가능성을 열어두었다.
