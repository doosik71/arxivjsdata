# Feedforward Neural Network for Time Series Anomaly Detection

ZHANG Rong, NIE Xin, DONG Shandong, XIAO Shiguang (2018)

## 🧩 Problem to Solve

본 논문은 대규모 산업 환경에서 발생하는 수백만 개의 시계열 데이터, 즉 KPI(Key Performance Indicators, 핵심 성과 지표)에서 이상치(Anomaly)를 탐지하는 문제를 해결하고자 한다. CPU 사용량, 주문 수, 온라인 사용자 수와 같은 KPI 데이터는 각각 서로 다른 형태(seasonal, timed tasks 등)와 추세를 가지기 때문에, 단순한 통계적 모델 하나만으로는 모든 종류의 시계열 데이터에 대해 효과적으로 이상치를 탐지하기 어렵다.

기존의 지도 학습 기반 모델들은 숙련된 엔지니어의 경험에 의존하는 복잡한 Feature Engineering(특징 공학) 과정을 필요로 한다. 하지만 이는 엔지니어마다 구성하는 특징이 달라 일관성이 떨어지며, 특징의 수가 많아질수록 실시간 탐지 시스템의 효율성이 저하되는 문제가 발생한다. 따라서 본 연구의 목표는 Feature Engineering 과정을 최소화하거나 완전히 배제한 End-to-End 형태의 딥러닝 기반 시계열 이상 탐지 시스템을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 심층 피드포워드 신경망(Deep Feedforward Neural Network, DNN)이 시계열 데이터의 복잡한 특징들을 스스로 학습할 수 있다는 점에 착안하여, 원본 데이터를 직접 입력으로 사용하는 모델을 설계한 것이다.

주요 기여 사항은 다음과 같다.

1. **End-to-End 모델 제안**: 정규화된 원본 시계열 데이터를 입력으로 하여 레이블을 직접 예측하는 DNN 모델을 제안함으로써, 수동 Feature Engineering의 필요성을 제거하였다.
2. **이론적 뒷받침**: DNN이 기존의 통계적 특징(평균, 최대/최소, 이동 평균 등)들을 계산하는 함수를 근사할 수 있음을 수학적 정리(Theorem)를 통해 증명하였다.
3. **성능 향상**: 대량의 수동 특징을 사용한 XGBoost 모델보다 DNN 기반 모델이 정밀도(Precision)와 재현율(Recall) 측면에서 더 우수한 성능을 보임을 입증하였다.
4. **TimeSeries2Vec 개념 도입**: 학습된 신경망의 은닉층(Hidden Layer) 출력을 시계열 데이터의 특징 벡터로 활용하여, 시계열 간의 유사도 측정 및 클러스터링에 활용할 수 있음을 보여주었다.

## 📎 Related Works

논문에서는 시계열 이상 탐지를 위해 사용되는 기존 접근 방식들을 다음과 같이 설명한다.

1. **통계적 방법**: 3-sigma 방법, Control Chart 이론, ARIMA 및 Seasonal ARIMA, Holt-Winters 모델 등이 사용되어 왔다. 이러한 방법들은 모델이 단순하지만, 데이터의 형태가 다양해질수록 모든 케이스를 커버하기 어렵다.
2. **머신러닝 기반 방법**: Isolation Forest, One-class SVM, 클러스터링 기반 방법 등이 있으며, 주로 정상 패턴을 구축한 뒤 여기서 벗어나는 점을 이상치로 간주한다.
3. **기존 지도 학습 시스템**: Tsinghua와 Baidu의 'Opprentice' 시스템은 64개의 검출기를 통해 133개의 특징을 추출하고 Random Forest로 학습시키는 방식을 사용한다. 하지만 이러한 방식은 매우 복잡한 Feature Engineering 과정을 수반한다.
4. **비지도 학습 및 RNN**: Variational Auto-Encoder(VAE)나 LSTM 등이 제안되었으나, LSTM의 경우 단일 모델이 하나의 시계열에 대해서만 작동한다는 한계가 있다.

## 🛠️ Methodology

### 1. 데이터 구성 및 전처리 (Input Pipeline)

특정 시점 $t$의 데이터 $x_t$가 이상치인지 판단하기 위해, 본 모델은 현재 시점뿐만 아니라 과거의 패턴을 함께 고려하는 세 가지 하위 시퀀스(Subsequences)를 구성한다.

- **오늘의 시퀀스 ($x$):** $[x_{t-k}, \dots, x_t]$
- **어제의 시퀀스 ($y$):** $[y_{t-k}, \dots, y_{t+k}]$ (여기서 $y_i = X_{i-1440}$)
- **지난주의 시퀀스 ($z$):** $[z_{t-k}, \dots, z_{t+k}]$ (여기서 $z_i = X_{i-10080}$)

여기서 $k$는 윈도우 크기로, 본 연구에서는 $k=180$(3시간)을 사용한다. 이 세 시퀀스를 결합하면 총 $5k+3 = 903$ 길이를 가진 하나의 조인트 시퀀스가 생성된다. 입력 데이터는 모델의 안정성을 위해 Min-Max 정규화를 거치며, 수식은 다음과 같다.
$$x'_i = \frac{x_i - a}{b - a}$$
($a$는 조인트 시퀀스의 최솟값, $b$는 최댓값이다.)

### 2. 모델 아키텍처

모델은 단순한 구조의 Deep Feedforward Neural Network를 사용한다.

- **구조**: 입력층 $\rightarrow$ 2개의 은닉층 $\rightarrow$ 출력층으로 구성된 완전 연결(Fully Connected) 구조이다.
- **활성화 함수**: 은닉층에서는 $\text{Leaky ReLU}(x) = \max(x, \alpha x)$ (단, $\alpha=0.2$)를 사용하며, 출력층에서는 Softmax 함수를 사용하여 이상치(0)와 정상(1)일 확률을 계산한다.
- **손실 함수**: 두 클래스 간의 확률 분포 차이를 줄이기 위해 Cross Entropy 손실 함수를 사용한다.

### 3. 학습 절차 및 클래스 불균형 해결

실제 데이터에서는 정상 데이터가 이상치보다 압도적으로 많은 Class Imbalance 문제가 발생한다. 이를 해결하기 위해 **Under-sampling** 기법을 적용하였다. 모든 이상치 샘플을 선택하고, 정상 샘플 중 일부를 랜덤하게 선택하여 이상치와 정상 샘플의 비율을 약 $2:1$로 맞추어 학습시켰다. 저자들은 DNN이 효과적으로 학습되기 위해 최소 5만 개 이상의 레이블된 데이터가 필요하다고 명시하였다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: 총 학습 데이터 78,120개, 테스트 데이터 14,785개를 사용하였다.
- **비교 대상 (Baselines)**: 3-Sigma, EWMA Control Chart, Polynomial Regression, Isolation Forest, XGBoost(243개 특징 사용).
- **평가 지표**: Precision, Recall, F1-Score를 사용하였다.

### 2. 정량적 결과

실험 결과, DNN 기반 모델이 기존의 모든 모델보다 우수한 성능을 보였다. 특히 수동 특징 공학을 적용한 XGBoost와 비교했을 때 다음과 같은 결과가 나타났다.

| Algorithm | Recall | Precision | F1-Score |
| :--- | :---: | :---: | :---: |
| XGBoost | $77.0\%$ | $96.0\%$ | $85.5\%$ |
| DNN-1 (기본) | $80.2\%$ | $96.4\%$ | $87.6\%$ |
| DNN-2 (Dropout 적용) | $\mathbf{81.1\%}$ | $\mathbf{96.7\%}$ | $\mathbf{88.2\%}$ |

DNN-2 모델(Dropout 적용)이 가장 높은 F1-Score를 기록하며, 수동 특징 추출 없이도 더 높은 탐지 성능을 달성할 수 있음을 입증하였다.

### 3. 시계열 특징 벡터화 (TimeSeries2Vec)

학습된 DNN의 은닉층 출력을 특징 벡터로 추출하여 KMeans 클러스터링과 Cosine Similarity 측정을 수행하였다. 그 결과, 유사한 형태를 가진 시계열 데이터들이 벡터 공간 상에서 가깝게 위치함을 확인하였으며, 이는 DNN이 시계열의 고유한 특징을 효과적으로 압축하여 표현하고 있음을 시사한다.

## 🧠 Insights & Discussion

본 논문의 가장 큰 강점은 시계열 이상 탐지 분야에서 고질적인 문제였던 **'Feature Engineering의 의존성'**을 DNN의 표현 학습(Representation Learning) 능력을 통해 해결하려 했다는 점이다. 특히, 단순히 모델을 제안하는 것에 그치지 않고, DNN이 기존 통계적 특징들을 수학적으로 구현할 수 있음을 증명함으로써 모델 선택의 이론적 근거를 제시하였다.

하지만 몇 가지 한계점과 논의 사항이 존재한다.
첫째, **데이터 의존성**이다. 논문에서는 DNN이 성능을 내기 위해 최소 5만 개의 레이블된 데이터가 필요하다고 언급하였다. 이는 현업에서 정교하게 레이블링된 대량의 데이터를 확보하는 것이 매우 어렵다는 점을 고려할 때 상당한 진입장벽이 될 수 있다.
둘째, **모델의 해석 가능성(Explainability)**이다. XGBoost와 같이 특징 기반 모델은 어떤 특징이 이상치 판단에 기여했는지 알 수 있지만, DNN은 블랙박스 모델이므로 왜 특정 포인트가 이상치로 판별되었는지 설명하기 어렵다.
셋째, **시간적 복잡도**이다. 입력 시퀀스의 길이가 903으로 상당히 길고 수백만 개의 KPI를 실시간으로 처리해야 하므로, 추론 단계에서의 연산 효율성에 대한 구체적인 분석이 추가될 필요가 있다.

## 📌 TL;DR

본 연구는 수동 Feature Engineering 없이 원본 시계열 데이터를 직접 입력으로 사용하는 **End-to-End Deep Feedforward Neural Network 기반의 이상 탐지 시스템**을 제안한다. DNN이 기존 통계적 특징들을 근사할 수 있음을 이론적으로 증명하였으며, 실험을 통해 XGBoost 등 기존 모델보다 높은 F1-Score를 달성하였다. 또한 학습된 모델의 은닉층을 활용한 **'TimeSeries2Vec'**을 통해 시계열 유사도 분석이 가능함을 보여주었다. 이 연구는 향후 대규모 시계열 모니터링 시스템에서 인간의 개입을 최소화한 자동화된 이상 탐지 프레임워크 구축에 중요한 기여를 할 가능성이 크다.
