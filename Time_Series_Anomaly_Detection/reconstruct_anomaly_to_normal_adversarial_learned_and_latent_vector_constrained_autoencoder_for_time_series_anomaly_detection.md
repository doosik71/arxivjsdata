# Reconstruct Anomaly to Normal: Adversarial Learned and Latent Vector-constrained Autoencoder for Time-series Anomaly Detection

Chunkai Zhang, Wei Zuo, Xuan Wang (2020)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 시계열 데이터의 이상치 탐지(Anomaly Detection)에서 발생하는 **이상치 데이터의 과도한 재구성(Over-reconstruction)** 현상이다.

기존의 딥러닝 기반 생성 모델들은 주로 정상 데이터만을 학습하여 정상 데이터의 분포를 캡처하고, 테스트 단계에서 재구성 오차(Reconstruction Error)가 큰 샘플을 이상치로 판별한다. 그러나 생성 모델이 학습 과정에서 이상치 데이터를 접하지 못했기 때문에, 때때로 이상치 데이터조차 매우 정교하게 재구성하여 낮은 재구성 오차를 생성하는 경우가 발생한다. 이는 결국 이상치를 정상으로 오판하는 미탐지(Omission) 문제로 이어진다.

또한, 기존 알고리즘들이 시계열 데이터의 특성인 인접 데이터 포인트 간의 시간적 상관관계(Temporal Correlation)를 충분히 활용하지 못한다는 점을 지적한다. 따라서 본 논문의 목표는 이상치 데이터가 정상 데이터의 분포를 따르도록 강제하여 재구성 오차를 의도적으로 극대화함으로써, 정상과 이상치를 더 명확하게 구분하는 RAN(Reconstruct Anomalies to Normal) 모델을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **"이상치를 정상으로 재구성하게 하여 오차를 극대화한다"**는 직관에 기반한다. 주요 기여 사항은 다음과 같다.

1. **모사된 이상치 데이터(Imitated Anomaly Data) 도입**: 모델이 정상 데이터뿐만 아니라 인위적으로 생성한 이상치 데이터를 함께 학습하게 함으로써 모델의 시야를 넓혔다.
2. **잠재 벡터 제약(Latent Vector-constraint)**: 이상치 데이터의 잠재 벡터가 정상 데이터의 잠재 벡터와 최대한 유사해지도록 강제하여, 디코더가 이상치를 정상적인 형태로 복원하도록 유도하였다.
3. **적대적 학습(Adversarial Learning) 적용**: Autoencoder와 판별자(Discriminator)를 결합한 적대적 네트워크를 구성하여, 모든 재구성 결과물이 정상 데이터의 분포를 따르도록 강제하였다.
4. **다중 커널 크기의 1D-CNN 사용**: 서로 다른 크기의 커널을 가진 Fully Convolutional Network를 통해 시계열 데이터의 인접 정보와 풍부한 지역적 특징을 추출하였다.

## 📎 Related Works

### 기존 접근 방식 및 한계

- **전통적 방법론**: PAA, SAX와 같은 차원 축소 기반 방식이나 LOF, RDOS 같은 밀도 기반 방식, 그리고 iForest 등이 있다. 그러나 이러한 방법들은 차원 축소 과정에서 중요한 정보를 손실하거나, 고차원 데이터에서 '차원의 저주' 문제로 인해 성능이 저하되는 한계가 있다.
- **딥러닝 기반 방법론**: AnoGAN, ALAD, DAGMM, MemAE 등이 재구성 오차를 기반으로 이상치를 탐지한다. 특히 MemAE는 메모리 모듈을 통해 이상치의 재구성을 방해하려 했으나, 여전히 이상치가 잘 재구성될 가능성이 존재한다.
- **시계열 특화 방법론**: LSTM-AD나 EncDec-AD 같은 모델이 제안되었으나, 일부 예측 불가능한 시계열 데이터에서는 한계가 있다.

### 차별점

본 논문은 단순히 정상 데이터를 잘 복원하는 것에 그치지 않고, **이상치 데이터의 복원 결과가 정상 데이터의 분포 내에 들어오도록(Reconstruct Anomalies to Normal)** 강제한다는 점에서 기존의 재구성 기반 모델들과 근본적으로 차별화된다.

## 🛠️ Methodology

### 1. 모사된 이상치 데이터 생성 (Imitate Anomaly Subsequences)

정상 데이터 세트 $\mathcal{S}_{norm}$에서 무작위로 일부 인덱스 집합 $R$을 선택하고, 해당 위치의 값을 다음과 같이 변경하여 이상치 데이터를 생성한다.
$$a_{norm\_x_j} = \mu_j + 4 \cdot \sigma_j \quad (j \in R)$$
여기서 $\mu_j$와 $\sigma_j$는 정상 데이터 세트 내 해당 위치 값들의 평균과 표준편차이다. 이를 통해 실제 이상치의 특성(정상 데이터 분포를 벗어남)을 모사한 $\mathcal{S}_{imi}$를 구축한다.

### 2. 시스템 구조 및 구성 요소

전체 시스템은 1D-CNN 기반의 Autoencoder(AE)와 판별자($D_x$)로 구성된다.

- **Encoder**: 다양한 커널 크기를 가진 1D-CNN을 사용하여 데이터 포인트와 그 이웃 간의 관계를 추출하고 잠재 벡터 $z$를 생성한다.
- **Decoder**: 1D-Deconvolutional Network를 사용하여 잠재 벡터 $z$로부터 재구성 데이터 $\mathcal{S}_{rec}$를 생성한다.
- **Discriminator ($D_x$)**: 재구성된 데이터가 실제 정상 데이터인지 아니면 생성된 가짜(재구성된) 데이터인지 판별한다.

### 3. 학습 목표 및 손실 함수

모델의 목표는 정상 데이터는 잘 복원하고, 이상치 데이터는 '정상처럼' 복원하여 결과적으로 원본 이상치 데이터와의 오차를 크게 만드는 것이다.

**Autoencoder (Generator)의 손실 함수**:
$$\mathcal{L}_{AE} = \lambda \cdot \mathcal{L}_{latent} + \mathcal{L}_{adversarial}$$
$$\mathcal{L}_{latent} = z - z_{imi}$$
$$\mathcal{L}_{adversarial} = \mathbb{E}_{\mathcal{S}_{imi} \sim \mathcal{S}} [\log(1 - D_x(AE(\mathcal{S}_{imi})))]$$

- $\mathcal{L}_{latent}$는 정상 데이터의 잠재 벡터 $z$와 모사된 이상치의 잠재 벡터 $z_{imi}$ 사이의 거리를 최소화하여, 인코더가 이상치를 정상으로 인식하게 만든다.
- $\mathcal{L}_{adversarial}$은 판별자 $D_x$를 속여 재구성 결과물이 정상 데이터의 분포를 따르도록 강제한다. ($\lambda=10$으로 설정)

**판별자 ($D_x$)의 손실 함수**:
$$\mathcal{L}_{Dx} = \mathbb{E}_{\mathcal{S}_{rec} \sim \mathcal{S}} [\log(D_x(\mathcal{S}_{rec}))] + \mathbb{E}_{\mathcal{S}_{norm} \sim \mathcal{S}} [\log(1 - D_x(\mathcal{S}_{norm}))]$$
판별자는 재구성된 데이터를 0(fake)으로, 실제 정상 데이터를 1(real)로 구분하도록 학습된다.

### 4. 추론 및 이상치 점수 계산

테스트 단계에서는 고정된 모델을 통해 재구성 오차 $rec\_error$를 계산하고, 이를 다음과 같이 정규화하여 최종 이상치 점수(Anomaly Score)를 산출한다.
$$Ano\_Score_i = \frac{rec\_error[i] - \min(rec\_error)}{\max(rec\_error) - \min(rec\_error)}$$

## 📊 Results

### 실험 설정

- **데이터셋**: UCR Repository, MIT-BIH, BIDMC 데이터베이스에서 수집된 ECG, Sensor, Motion, Image 데이터 등 총 15종의 시계열 데이터셋을 사용하였다.
- **비교 대상**: ESAX, SAX_TD, iForest 등의 고전적 방법과 AnoGAN, DAGMM, ALAD, MemAE, LSTMAD 등의 딥러닝 방법론을 비교 대상으로 설정하였다.
- **평가 지표**: AUC-ROC를 사용하여 정상과 이상치를 구분하는 능력을 측정하였다.

### 주요 결과

- **정량적 성과**: RAN은 15개 데이터셋 중 9개에서 가장 높은 AUC-ROC를 기록하며 최상위 성능을 보였다. 특히 MemAE가 두 번째로 좋은 성능을 보였는데, 이는 이상치의 재구성을 제어하는 것이 중요하다는 점을 시사한다.
- **정성적 분석**: MIT-BIH210 데이터셋 실험 결과, RAN은 정상 데이터는 매우 정확하게 복원하는 반면, 이상치 데이터는 정상 데이터와 유사한 형태로 복원함으로써 매우 높은 재구성 오차를 생성함을 확인하였다.
- **점수 분포**: 이상치 점수의 히스토그램 분석 결과, RAN이 다른 알고리즘들에 비해 정상 데이터의 점수 분포와 이상치 데이터의 점수 분포 사이의 겹침(Overlap) 영역이 가장 작아 변별력이 매우 높음을 입증하였다.
- **Ablation Study**: $\text{RAN} > \text{LAE-FCN} > \text{AE-FCN} > \text{AE}$ 순으로 성능이 나타났으며, 이는 잠재 공간 제약과 적대적 학습, 그리고 CNN 구조가 각각 성능 향상에 기여했음을 보여준다.

## 🧠 Insights & Discussion

### 강점

본 논문은 단순히 "정상을 학습한다"는 기존의 패러다임을 넘어, "이상치를 어떻게 처리할 것인가"에 대한 명확한 전략을 제시하였다. 특히 잠재 공간(Latent Space)과 원본 공간(Original Space) 모두에 제약을 가함으로써, 모델이 이상치를 강제로 정상의 형태로 매핑하게 만든 설계는 매우 효율적인 접근 방식이다.

### 한계 및 논의사항

1. **이상치 모사 방식의 단순함**: $\mu + 4\sigma$를 이용한 이상치 생성 방식은 매우 단순한 형태의 이상치만을 가정한다. 실제 세계의 이상치는 훨씬 복잡하고 다양한 패턴을 가지므로, 더 정교한 모사 전략이 필요할 수 있다.
2. **하이퍼파라미터 민감도**: $\lambda$ 값이나 적대적 학습의 균형(Nash equilibrium)을 맞추는 과정이 실험적으로 설정되었는데, 다양한 데이터셋에 대해 이 설정이 얼마나 일반화될 수 있는지에 대한 추가 논의가 필요하다.
3. **계산 복잡도**: 적대적 학습 네트워크를 사용하므로 일반적인 Autoencoder보다 학습 시간이 더 소요될 가능성이 크다.

## 📌 TL;DR

본 논문은 시계열 이상치 탐지에서 이상치가 너무 잘 복원되어 발생하는 미탐지 문제를 해결하기 위해, **이상치를 정상 데이터의 분포로 강제 재구성하는 RAN 모델**을 제안한다. 모사된 이상치 데이터를 사용하여 잠재 벡터를 제약하고, 적대적 학습을 통해 복원 결과물이 정상 분포를 따르게 함으로써 이상치의 재구성 오차를 극대화하였다. 실험 결과, 다양한 시계열 데이터셋에서 기존 모델들보다 높은 AUC-ROC를 기록하며 탁월한 이상치 변별력을 입증하였다. 이 연구는 생성 모델 기반의 이상치 탐지에서 '이상치 제약'의 중요성을 일깨워주며 향후 고도화된 비지도 학습 기반 탐지 연구에 기여할 가능성이 높다.
