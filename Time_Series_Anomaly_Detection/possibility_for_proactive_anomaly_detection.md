# POSSIBILITY FOR PROACTIVE ANOMALY DETECTION

Jinsung Jeon, Jaehyeon Park, Sewon Park, Jeongwhan Choi, Minjung Kim, Noseong Park (2025)

## 🧩 Problem to Solve

본 논문은 시계열 이상치 탐지(Time-series Anomaly Detection) 분야에서 기존의 **사후 대응적(Reactive) 접근 방식**이 가진 한계를 해결하고자 한다. 기존 모델들은 주로 모델의 출력값과 실제 관측값(Ground Truth) 사이의 오차를 기반으로 이상치를 탐지한다. 이러한 방식은 이상 현상이 이미 발생한 이후에 이를 식별하는 구조이므로, 실제 산업 현장에서 잠재적 피해나 손실을 사전에 방지하는 데 실질적인 도움이 되지 않는다.

따라서 본 연구의 목표는 이상 현상이 발생하기 전, 즉 전조 증상(Precursors)이 나타나는 시점에서 이를 미리 감지하여 조기에 개입할 수 있도록 하는 **선제적(Proactive) 이상치 탐지** 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **이상치 탐지에 특화된 시계열 예측 모델(Forecasting Model)**과 **데이터 기반의 이상치 탐지 모델(Data-driven Anomaly Detection Model)**을 통합하여, 테스트 데이터의 실제 정답(Ground Truth) 없이도 미래의 이상 가능성을 예측하는 파이프라인을 구축하는 것이다.

특히, 다변량 시계열 데이터가 가진 **연속형(Continuous)** 변수와 **이산형(Discrete)** 변수의 서로 다른 특성을 개별적으로 처리하면서도, **Adaptive Graph Convolution (AGC)** 레이어를 통해 변수 간의 상관관계를 동시에 학습함으로써 예측 정확도를 높인 점이 주요한 설계 포인트이다.

## 📎 Related Works

논문에서는 데이터 기반 이상치 탐지 방식을 크게 두 가지 범주로 나누어 설명한다.

1. **분포 기반 방법(Distribution-based methods):** 데이터의 밀도나 분포 특성을 이용한다. Gaussian Mixture Model (GMM)은 가우시안 분포의 혼합을 가정하며, ECOD는 경험적 누적 분포 함수(Empirical Cumulative Distribution)의 꼬리 부분(Tail)에 위치한 데이터를 이상치로 간주한다.
2. **경계 기반 방법(Boundary-based methods):** 데이터가 포함되는 최적의 경계를 학습한다. SVDD와 DeepSVDD는 고차원 특징 공간에서 데이터를 포함하는 최소한의 초구(Hypersphere)를 찾고, 이 경계 밖에 위치한 데이터를 이상치로 정의한다.

기존의 예측 기반 이상치 탐지 모델들은 예측값과 실제값의 차이를 계산하는 사후 방식에 의존했으나, 본 논문은 예측값 그 자체를 데이터 기반 모델의 입력으로 넣어 임계값을 초과하는지 확인하는 선제적 방식으로 차별화를 꾀했다.

## 🛠️ Methodology

### 1. 전체 파이프라인 (Proactive Detection Process)

전체 시스템은 다음과 같은 단계로 작동한다.

1. **임계값 설정:** 훈련 데이터만을 사용하여 데이터 기반 모델(GMM, ECOD, DeepSVDD 등)을 학습시키고, 훈련 데이터의 점수 분포를 통해 이상치 임계값(Threshold)을 설정한다.
2. **미래 값 예측:** 테스트 데이터가 입력되면, 특화된 예측 모델이 미래의 시점 $\hat{T}_{N+1}$ 값을 예측한다.
3. **선제적 탐지:** 예측된 값 $\hat{T}_{N+1}$을 학습된 데이터 기반 모델에 입력하여 점수를 계산한다. 이 점수가 설정된 임계값보다 낮거나(또는 모델에 따라 높거나) 기준을 벗어나면, 실제 해당 시점이 되기 전에 이를 이상치로 판정한다.

### 2. 시계열 예측 모델 아키텍처

본 모델은 연속형과 이산형 데이터를 분리하여 처리한다.

**A. 연속형 특징 처리 (Continuous Features):**
연속형 데이터는 추세(Trend)와 계절성(Seasonality)으로 분해된다.

- 추세 $C^T$는 이동 평균(Moving Average)을 통해 추출하며, 계절성 $C^S$는 원본에서 추세를 뺀 나머지 값으로 정의한다.
- 각각 $\text{Linear}_{\text{trend}}$와 $\text{Linear}_{\text{seasonality}}$ 레이어를 통해 다음 시점의 값을 예측하고, 이를 합산하여 최종 예측값 $\hat{C}_{N+1}$을 생성한다.
$$\hat{C}_{N+1} = \hat{C}_{N+1}^{T} + \hat{C}_{N+1}^{S}$$

**B. 이산형 특징 처리 (Discrete Features):**
이산형 데이터는 원-핫 인코딩(One-hot encoding)을 거쳐 $\text{Linear}_{\text{discrete}}$ 레이어에 입력되어 다음 시점의 원-핫 벡터 $\hat{O}_{N+1}$을 예측한다.

**C. Adaptive Graph Convolution (AGC) 레이어:**
분리된 처리 과정에서 놓칠 수 있는 변수 간 의존성을 학습하기 위해 AGC 레이어를 사용한다.

- 학습 가능한 노드 임베딩 행렬 $E \in \mathbb{R}^{(c+d) \times b}$를 이용하여 적응형 인접 행렬(Adaptive Adjacency Matrix) $A$를 다음과 같이 생성한다.
$$A = I + \text{softmax}(\text{ReLU}(EE^T))$$
- 이를 통해 특징 간의 근접도를 학습하고, 그래프 컨볼루션을 통해 변수 간 관계가 반영된 특징 벡터 $Z$를 추출한다.

**D. 최종 결합 및 손실 함수:**
최종 예측값은 개별 예측값에 AGC 레이어의 출력 $Z$를 더하여 완성한다.

- **연속형 손실:** 평균 제곱 오차(MSE)를 사용한다.
$$\text{Loss}_C = \frac{1}{c} \sum_{i=1}^{c} (C_{N+1}^i - \hat{C}_{N+1}^i)^2$$
- **이산형 손실:** 교차 엔트로피(Cross-Entropy) 손실을 사용한다.
$$\text{Loss}_D = \frac{1}{d} \sum_{i=1}^{d} \sum_{j=1}^{e} -O_{N+1}^{ij} \log(\hat{O}_{N+1}^{ij})$$
- **전체 손실:** $\text{Loss}_{\text{Total}} = \text{Loss}_C + \text{Loss}_D$

## 📊 Results

### 1. 실험 설정

- **데이터셋:** MSL, SMAP (이산형 변수 비중 높음), SMD, PSM (연속형 위주)의 4개 벤치마크 사용.
- **평가 지표:** F1-@K, F1-Composite, F1-Range.
- **비교 대상:** LSTM-P, DeepAnT, TCN-S2S-P, MTAD-GAT, GDN, GTA 등 6개의 이상치 탐지 모델과 NLinear, DLinear, TimesNet, PatchTST 등 4개의 예측 모델.

### 2. 주요 결과

- **정량적 성능:** MSL과 SMAP 데이터셋에서 가장 우수한 성능을 보였다. 이는 해당 데이터셋들이 이산형 변수를 많이 포함하고 있어, 본 모델의 특화된 처리 구조가 효과적이었음을 시사한다.
- **선제적 탐지의 실효성:** SMAP 데이터셋 시각화 결과, 실제 이상 현상이 발생하기 직전 시점(예: 타임스탬프 29304의 이상을 29303에 감지)에 빨간색 선으로 표시된 'Proactively Detected Point'가 나타남을 확인하여 선제적 탐지의 가능성을 입증하였다.
- **예측 정확도:** MSE 기준으로는 최신 예측 모델들과 대등하거나 우수한 성능을 보였으며, 특히 이산형 변수를 정확하게 예측함으로써 데이터 기반 모델의 점수 산출에 긍정적인 영향을 주었다.

## 🧠 Insights & Discussion

### 1. 예측 가능한 이상치 분석 (Forecastable Anomaly)

저자들은 모든 이상치가 선제적으로 탐지될 수 없다는 점을 인정하고, 푸리에 변환(Fourier Transform)과 컨벡스 헐(Convex Hull)을 이용해 '예측 가능성'을 분석하였다.

- **조건:** 테스트 데이터의 이상치 기저(Bases)가 훈련 데이터와 동일해야 하며, 기저 계수(Coefficients)가 훈련 데이터의 컨벡스 헐 내부에 존재해야 한다.
- **분석 결과:** SMAP의 경우 단 1.2%만이 헐 외부에 존재하여 매우 예측 가능했지만, SMD는 약 21%가 외부에 존재했다. 이는 SMD 데이터셋에서 본 모델의 성능이 상대적으로 낮게 나타난 이유를 이론적으로 설명해 준다.

### 2. 비판적 해석 및 강점

- **강점:** 단순한 예측 오차 기반이 아니라, 예측값 자체를 분포 모델에 넣어 판단함으로써 '정답' 없이도 미래를 예측하는 프레임워크를 제시했다는 점이 독창적이다. 또한 연속형/이산형 변수를 분리 처리하고 AGC로 통합한 구조가 실용적이다.
- **한계:** 전조 증상이 없는 '갑작스러운(Sudden)' 이상치의 경우 여전히 탐지가 어렵다는 한계가 있다. 또한, 데이터 기반 모델(GMM, ECOD 등)의 선택과 임계값 설정 방식에 따라 최종 성능이 크게 좌우될 가능성이 있다.

## 📌 TL;DR

본 논문은 기존의 사후 대응적 이상치 탐지에서 벗어나, **특화된 시계열 예측 모델과 데이터 기반 임계값 모델을 결합한 선제적(Proactive) 이상치 탐지 방법론**을 제안한다. 연속형/이산형 변수를 분리하여 처리하고 Adaptive Graph Convolution으로 변수 간 상관관계를 학습한 예측 모델을 통해, 실제 이상이 발생하기 전 시점에서 이를 감지할 수 있음을 입증하였다. 특히 푸리에 분석을 통해 이상치의 예측 가능성 여부를 이론적으로 규명하였으며, 이는 향후 전조 증상이 있는 이상치를 효율적으로 방어하는 시스템 구축에 기여할 것으로 보인다.
