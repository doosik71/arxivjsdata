# Monte Carlo EM for Deep Time Series Anomaly Detection

François-Xavier Aubet, Daniel Zügner, Jan Gasthaus (2021)

## 🧩 Problem to Solve

본 논문은 시계열 데이터에서 레이블이 없는(unlabeled) 상태로 존재하는 이상치(outliers) 또는 이상 징후(anomalies)가 모델 학습에 미치는 부정적인 영향을 해결하고자 한다.

일반적인 딥러닝 기반의 시계열 이상 탐지 및 예측 모델들은 학습 데이터에 포함된 이상치의 비율이 무시할 수 있을 정도로 작다고 가정하며, 레이블이 없는 데이터를 모두 정상 데이터 분포(nominal data distribution)에서 생성된 것으로 간주하고 학습한다. 그러나 실제 환경에서 학습 데이터에 이상치가 포함되어 있을 경우, 모델은 잘못된 데이터 분포를 학습하게 되어 결과적으로 정상 데이터에 대한 모델링 성능이 저하되고 이상 탐지 정확도가 떨어지는 Bias 문제가 발생한다.

따라서 본 연구의 목표는 학습 데이터 내의 이상치를 명시적으로 고려하여, 정상 데이터의 분포를 더 정확하게 추론함과 동시에 이상 지점을 식별할 수 있는 범용적인 학습 기법을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **잠재 이상 지표 변수(Latent Anomaly Indicator Variable)**를 도입하고, 이를 **Monte Carlo Expectation-Maximization (MC EM)** 알고리즘을 통해 학습하는 것이다.

구체적으로, 각 시점 $t$에 대해 해당 데이터가 정상인지 이상인지를 나타내는 이진 잠재 변수 $z_t$를 설정한다. 이를 통해 모델은 학습 과정에서 어떤 데이터 포인트가 정상 분포에서 왔고 어떤 포인트가 이상 분포에서 왔는지를 동시에 추론할 수 있다. 이러한 접근 방식은 특정 모델 구조에 의존하지 않는 래퍼(wrapper) 형태로 설계되어, 기존의 다양한 확률적 시계열 예측 또는 재구성 모델과 쉽게 결합될 수 있다는 점이 주요한 설계 특징이다.

## 📎 Related Works

기존의 시계열 이상 탐지 방법론은 크게 예측(Forecasting) 기반 방법과 재구성(Reconstruction) 기반 방법으로 나뉜다. 예측 방법은 관측값이 예측값에서 벗어나는 정도를 측정하고, 재구성 방법은 재구성된 값과의 차이를 통해 이상치를 판별한다.

이러한 방법들의 공통적인 한계는 모델의 파라미터를 학습할 때 관측 데이터 $\mathbf{x}$를 그대로 사용한다는 점이다. 관측 데이터에 이상치가 섞여 있다면 모델은 필연적으로 오염된 분포를 학습하게 된다. 일부 연구에서 강건한(robust) 모델링을 시도했으나, 여전히 정상과 이상 데이터를 동시에 학습하려는 경향이 있다.

본 논문은 이를 해결하기 위해 Robust Mixture Model이나 고전적 시계열 모델에서 사용되던 잠재 지표 변수 개념을 최신 딥러닝 기반 시계열 모델에 접목하였다. 특히, 복잡한 확률적 추론 과정을 Monte Carlo EM으로 단순화하여 딥러닝 모델의 학습 루프에 통합함으로써 기존 접근 방식과의 차별점을 둔다.

## 🛠️ Methodology

### 1. 전체 시스템 구조 및 확률 모델

본 제안 방법론은 관측 데이터 $x_t$가 두 가지 잠재 경로 중 하나를 통해 생성된다고 가정하는 Mixture Model 구조를 가진다.

- **정상 데이터 모델 $p^+_\theta$**: 파라미터 $\theta$로 정의되는 확률 모델로, 잠재적인 정상 시계열 $y^+_{1:T}$를 모델링한다. 본 논문의 실험에서는 MLP 기반의 확률적 예측 모델을 사용하였으며, $l$-차 마르코프 가정을 통해 다음과 같이 정의한다.
  $$p(y^+_{t+1}|y^+_{t:t-l}) = \mathcal{N}(f_\theta(y_{t:t-l}), g_\theta(y_{t:t-l}))$$
- **이상 데이터 모델 $p^-$**: 이상 데이터 $y^-_{1:T}$를 모델링한다. 특정 분포를 가정하지 않고 학습 데이터 영역 전체에 대한 균등 분포(Uniform Distribution)로 간단히 정의한다.
- **이상 지표 모델 $p_z$**: 각 시점의 정상/이상 여부를 결정하는 잠재 변수 $z_t$의 분포를 모델링한다. 시간적 의존성을 반영하기 위해 2개의 상태(0: 정상, 1: 이상)를 가진 은닉 마르코프 모델(Hidden Markov Model, HMM)을 사용한다.

최종 관측치 $x_t$는 다음과 같은 생성 과정을 거친다.
$$x_t = I[z_t = 0]y^+_t + I[z_t = 1]y^-_t$$
여기서 $I[\cdot]$은 지시 함수(indicator function)이다.

### 2. 학습 절차: Monte Carlo EM

학습은 E-step과 M-step이 반복되는 MC EM 구조를 따른다.

**E-step (Expectation):**
HMM의 Forward-Backward 알고리즘을 사용하여 잠재 변수 $z_{1:T}$의 사후 분포 $p_z(z_{1:T})$를 추론한다. 이때 사용되는 조건부 확률은 다음과 같다.

- $p(x_t|z_t=0) = p^+_\theta(x_t)$
- $p(x_t|z_t=1) = p^-(x_t)$

**M-step (Maximization):**
분석적인 업데이트가 어려운 딥러닝 모델의 특성을 고려하여 Monte Carlo 근사 방식을 사용한다.

1. $p_z(z_{1:T})$로부터 $S$개의 지표 시계열 샘플 $z^s$를 추출한다.
2. 각 샘플 $z^s$에서 $z^s_t=0$인 포인트(정상으로 판단된 포인트)들에 대해서만 $p^+_\theta$를 학습시킨다.
3. 만약 모델 구조상 결측치 처리가 불가능하여 전체 시퀀스가 필요하다면, 이전 M-step에서 학습된 $p^+_\theta$를 이용해 $z^s_t=1$인 지점의 값을 보간(Imputation)하여 학습에 사용한다.
4. 샘플링된 $z^s$들의 전이 횟수를 계산하여 HMM의 전이 행렬(Transition Matrix)을 업데이트한다.

### 3. 추론(Inference) 과정

테스트 단계에서는 HMM 필터링을 통해 새로 들어오는 데이터 $x_t$가 $p^+$와 $p^-$ 중 어디서 생성되었을 확률이 높은지 판단한다. 만약 이상치($p^-$)로 판단될 경우, 해당 값을 그대로 사용하지 않고 $p^+_\theta$의 예측값이나 최빈값(mode)으로 대체하여 모델의 입력으로 사용함으로써 추론의 안정성을 높인다.

## 📊 Results

### 실험 설정

- **모델**: MLP를 사용하여 가우시안 분포의 평균과 분산을 예측하는 단순한 확률적 예측 모델을 사용했다. 입력 윈도우 크기는 25이다.
- **데이터셋 및 지표**:
  - **Yahoo Anomaly Detection Dataset**: 4개의 하위 집합(A1-A4)에 대해 Adjusted F1 score를 측정하였다.
  - **Electricity Dataset**: 370개의 시계열 데이터에 인위적으로 0.4%의 포인트 이상치를 주입하여 MAE(Mean Absolute Error)를 측정하였다.

### 주요 결과

1. **이상 탐지 성능 (Yahoo Dataset)**:
   제안 방법(MLP + LAI)이 기존 MLP 대비 F1 score를 대폭 향상시켰다. 특히 A3 세트의 경우 $63.25 \rightarrow 87.91$로 비약적인 상승을 보였다. 또한 학습 데이터 내에서 추론된 $p(z_t=1)$ 값을 통해 실제 레이블과 비교했을 때 높은 F1 score를 기록하여, 학습 과정에서 이상치를 정확히 식별하고 있음을 입증하였다.

2. **오염된 데이터에서의 예측 성능 (Electricity Dataset)**:
   학습 데이터에 이상치가 주입되었을 때, 일반 MLP는 MAE가 $0.1551 \rightarrow 0.2092$로 크게 증가한 반면, LAI를 적용한 모델은 $0.1558 \rightarrow 0.1704$로 오차 증가 폭이 현저히 적었다. 이는 제안 방법이 학습 데이터의 오염에 강건함을 보여준다.

3. **정성적 분석**:
   합성 데이터(Sinusoidal) 실험 결과, 일반 모델은 이상치에 의해 예측 곡선이 왜곡되는 반면, LAI 적용 모델은 이상치를 무시하고 정상적인 사인 곡선을 정확하게 학습하는 모습을 확인하였다.

## 🧠 Insights & Discussion

본 논문은 딥러닝 모델이 학습 데이터의 오염에 취약하다는 점을 확률적 잠재 변수 모델과 MC EM이라는 전통적인 통계적 기법을 통해 효율적으로 해결하였다.

**강점:**

- **범용성**: $p^+_\theta$ 모델의 아키텍처에 구애받지 않으므로, 더 복잡한 모델(예: OmniAnomaly, 최신 Transformer 기반 모델)에도 쉽게 적용 가능하다.
- **명시적 분리**: 단순히 손실 함수에 가중치를 주는 방식이 아니라, 데이터 생성 과정 자체를 모델링하여 정상과 이상 데이터를 명시적으로 분리해 학습시킨다.

**한계 및 논의사항:**

- **추론 시 결정 문제**: 현재 추론 단계에서는 각 포인트가 정상인지 이상인지 결정하여 대체하는 방식을 취하는데, 이 결정 과정에서 오류가 발생할 경우 예측 성능에 영향을 줄 수 있다. 저자들은 향후 연구로 Particle Filter와 같은 기법을 도입하여 학습 때와 유사한 Monte Carlo 접근 방식을 추론에도 적용할 수 있음을 언급하였다.
- **계산 비용**: MC EM의 특성상 여러 번의 샘플링과 모델 업데이트가 필요하므로, 일반적인 학습보다 계산 시간이 증가할 가능성이 있다.

## 📌 TL;DR

본 연구는 학습 데이터에 포함된 레이블 없는 이상치로 인해 딥러닝 모델이 편향되는 문제를 해결하기 위해, **잠재 이상 지표 변수($z_t$)와 Monte Carlo EM**을 결합한 **LAI(Latent Anomaly Indicator)** 프레임워크를 제안한다. 이 방법은 HMM을 통해 데이터의 정상/이상 여부를 추론하고 정상 데이터만을 선택적으로 학습함으로써, 오염된 데이터셋에서도 강건한 모델 학습을 가능하게 하며 이상 탐지 및 예측 성능을 유의미하게 향상시킨다. 이 연구는 특히 실무에서 깨끗한 학습 데이터를 구하기 어려운 시계열 이상 탐지 분야에 매우 실용적인 적용 가능성을 제시한다.
