# HFN: Heterogeneous Feature Network for Multivariate Time Series Anomaly Detection

Jun Zhan, Chengkun Wu, Canqun Yang, Qiucheng Miao, Xiandong Ma (2022)

## 🧩 Problem to Solve

본 논문은 다변량 시계열(Multivariate Time Series, MTS) 데이터에서 이상치 탐지(Anomaly Detection, AD)를 수행할 때 발생하는 변수 간의 **이질성(Heterogeneity)** 문제를 해결하고자 한다.

일반적인 산업 시스템의 모니터링 데이터는 연속적인 수치형 변수(Continuous numerical variables)와 이산적인 범주형 변수(Discrete categorical variables), 그리고 이들이 혼합된 하이브리드 변수(Hybrid variables)가 동시에 존재한다. 기존의 많은 딥러닝 기반 이상치 탐지 모델들은 이러한 변수들의 타입 차이를 무시하고 모든 변수를 동일한 성격의 데이터(Homogeneous)로 처리하는 경향이 있다. 하지만 서로 다른 타입의 변수들은 엣지 분포(Edge distribution)가 근본적으로 다르기 때문에, 이를 구분하지 않고 네트워크에 입력할 경우 중요한 정보가 손실되어 탐지 정확도가 떨어진다.

따라서 본 논문의 목표는 변수 간의 이질적인 구조 정보를 학습할 수 있는 **HFN(Heterogeneous Feature Network)** 프레임워크를 제안하여, 다변량 시계열 데이터에서의 이상치 탐지 성능을 높이고 탐지된 이상치에 대한 설명 가능성(Interpretability)을 제공하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 변수 간의 관계를 단순한 단일 그래프가 아닌, 데이터 타입에 따른 **이질적 그래프 구조(Heterogeneous Graph Structure)**로 모델링하는 것이다.

1. **이질적 그래프 구조 학습(HGSL):** 센서 임베딩(Global semantic)과 특징 값(Local feature)의 유사도를 동시에 고려하여 이질적 그래프 구조를 학습한다. 특히 기존의 메타-패스(Meta-path) 방식 대신, 데이터 타입 기반의 고정 길이 샘플링 방식을 제안하여 복잡한 MTS 데이터에서 효율적으로 하위 그래프를 추출한다.
2. **이질적 특징 네트워크(HFN) 설계:** 노드 어텐션(Node Attention)과 채널 어텐션(Channel Attention)을 결합하여, 서로 다른 타입의 변수들이 가진 동적 의존성을 함께 최적화하여 학습한다.
3. **이상치 위치 식별(Anomaly Locating):** 예측값과 실제값의 편차를 기반으로 센서별 상태 점수(Condition Score)를 계산함으로써, 시스템의 어느 부분에서 이상이 발생했는지 빠르게 식별할 수 있는 근거를 제공한다.

## 📎 Related Works

### 1. MTS 이상치 탐지 접근 방식

- **비지도 학습(Unsupervised):** K-means와 같은 클러스터링 기반 방법이 주로 사용되나, 정적인 데이터 모델링에 치중되어 동적인 시계열 특성을 잡기 어렵다.
- **지도 학습(Supervised):** 정확도는 매우 높지만, 실제 산업 현장에서 정답 라벨(Positive/Negative samples)이 포함된 고품질의 학습 데이터를 얻는 것이 매우 어렵다.
- **준지도 학습(Semi-supervised):** 정상 상태의 데이터는 대량으로 확보 가능하므로, 정상 데이터만으로 모델을 학습시킨 후 편차를 통해 이상치를 찾는 방식이 효율적이다.

### 2. 이질적 데이터 모델링 및 그래프 학습

- **이질적 데이터 학습:** 행렬 분해(Matrix Decomposition)나 연속 데이터의 이산화(Discretization) 방식이 사용되었으나, 계산 비용이 높거나 변수 간의 상관관계를 무시하는 한계가 있다.
- **그래프 신경망(GNN):** 복잡한 데이터 간의 구조 모델링에 강점이 있으나, 대부분의 MTS 모델은 그래프 구조가 미리 정의되어 있다고 가정하거나 모든 노드를 동일하게 처리하는 동형 그래프(Isomorphic graph)를 학습한다.

본 논문은 이러한 한계를 극복하기 위해 변수의 타입을 구분하여 서로 다른 유사도 서브그래프를 생성하고 이를 통합하는 방식을 채택하여 차별점을 둔다.

## 🛠️ Methodology

### 1. 전체 파이프라인

본 프레임워크는 크게 세 단계로 구성된다.

- **HGSL 단계:** 센서 임베딩 유사도와 특징 값 유사도를 융합하여 이질적 그래프 구조를 학습한다.
- **표현 학습 단계:** 노드 및 채널 어텐션을 통해 최적의 시계열 표현(Representation)을 얻는다.
- **이상치 탐지 및 위치 식별 단계:** 예측값과 실제값의 차이를 분석하여 이상치를 판별하고 위치를 찾는다.

### 2. 이질적 그래프 구조 학습 (HGSL)

변수 간의 관계를 정의하기 위해 전역적 의미론적 상관관계(Global semantic correlation)와 지역적 특징 상관관계(Local feature correlation)를 모두 사용한다.

- **전역적 상관관계:** 각 변수 $i$에 대해 학습 가능한 임베딩 벡터 $E_i \in \mathbb{R}^{1 \times \omega'}$를 도입한다.
- **지역적 상관관계:** 입력 특징 벡터 $x_{type}$를 공통 공간으로 투영하는 매핑 네트워크를 사용한다. 예를 들어 수치형 변수($C$)의 경우 다음과 같이 투영된다.
  $$f_{C,i} = \text{MLP}(x_{C,i} \cdot W_{MLP} + b_{MLP})$$
- **유사도 융합:** 노드 임베딩 유사도 행렬 $M_{NE}$와 노드 특징 유사도 행렬 $M_{NF}$를 계산한 뒤, 학습 가능한 가중치 행렬 $W$와 하다마르 곱($\circ$)을 통해 통합 유사도 행렬 $M_{agg}$를 생성한다.
  $$M_{agg} = M_{NE} \circ W_{NE} + M_{NF} \circ W_{NF}$$
- **인접 행렬 생성:** 학습 가능한 임계값 $\tau$를 사용하여 인접 행렬 $A$를 결정한다.
  $$A_{ij} = \begin{cases} 1 & \text{if } M_{agg}[i,j] \ge \tau \\ 0 & \text{if } M_{agg}[i,j] < \tau \end{cases}$$
이후, 데이터 타입에 따라 이산 특징 서브그래프(DFS), 연속 특징 서브그래프(CFS), 하이브리드 특징 서브그래프(HFS)로 분리하며, 과적합 방지를 위해 랜덤 마스크(Random mask) 연산을 적용한다.

### 3. HFN 표현 학습 (Representation Learning)

추출된 세 개의 서브그래프는 각각 독립적인 Graph Attention Network(GAT)로 입력된다.

- **노드 어텐션:** 각 서브그래프 내에서 이웃 노드 간의 중요도 $\alpha_{ij}$를 계산하여 정보를 집계한다.
  $$\alpha_{ij} = \text{softmax}(\text{LeakyReLU}(\vec{a}^T [W f_i || W f_j]))$$
- **채널 어텐션(Channel Aggregation):** 서로 다른 채널(연속, 이산, 하이브리드)에서 얻은 노드 임베딩 $p'$들을 융합한다. 학습 가능한 파라미터 $\beta$를 통해 가중 합을 수행한다.
  $$p = \beta (p'_C || p'_D) + (1-\beta) p'_{CD}$$
  여기서 $p'_C, p'_D$는 각각 연속 및 이산 채널의 임베딩이며, $p'_{CD}$는 하이브리드 채널의 임베딩이다.

### 4. 이상치 탐지 및 예측 절차

- **예측:** 최종 임베딩 $p$와 전역 임베딩 $E$를 MLP 레이어에 입력하여 다음 시점의 값 $\hat{y}_{t+1}$을 예측한다.
  $$\hat{y}_{t+1} = \text{activation}(\text{MLP}(p \oplus E))$$
- **손실 함수:** 정상 데이터만을 사용하여 평균 제곱 오차(MSE)로 모델을 최적화한다.
  $$\mathcal{L} = \frac{1}{N} \sum (y_{t+1} - \hat{y}_{t+1})^2$$
- **이상치 점수(Condition Score):** 테스트 단계에서 실제값과 예측값의 차이를 정규화하여 점수를 산출한다. 이때 IQR(Interquartile Range)과 중앙값(Median)을 사용하여 차원 간의 스케일 차이를 제거한다.
  $$\text{Score}_i = \frac{|y_{t,i} - \hat{y}_{t,i}| - \text{Median}_i}{\text{IQR}_i + 1}$$
최종적으로 이 점수가 임계값을 초과하면 이상치로 판단한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋:** SWaT (수처리스뮬레이터), WADI (수분배시스템), WTD (풍력 발전소) 등 3종의 실제 데이터를 사용하였다.
- **비교 모델:** LSTM-VAE, USAD, MAD-GAN, MTAD-GAT, GDN 및 전통적인 방식인 PCA, Isolation Forest, LightGBM 등을 사용하였다.
- **평가 지표:** Precision, Recall, F1-score를 측정하였다.

### 2. 주요 결과

- **정량적 결과:** HFN은 SWaT와 WTD 데이터셋에서 기존 모델 대비 F1-score가 약 5%~14% 향상되는 성능을 보였다. 특히 WADI 데이터셋에서도 우수한 성능을 보였으며, 전반적으로 가장 높은 Recall(재현율)을 기록하였다. 이는 실제 산업 현장에서 이상치를 놓치지 않고 탐지하는 것이 중요하다는 점을 고려할 때 매우 유의미한 결과이다.
- **구성 요소 분석 (Ablation Study):**
  - 노드 임베딩($NE$)을 제외했을 때 성능 저하가 가장 심하게 나타나, 전역적 구조 학습이 매우 중요함을 확인하였다.
  - 하이브리드 서브그래프($HFS$)를 제외했을 때도 성능이 하락하여, 서로 다른 타입의 센서 간 상호작용이 보완적인 역할을 함을 입증하였다.
- **이상치 위치 식별:** SWaT 데이터셋의 사례 분석 결과, 실제 공격이 발생한 센서(FIT401, UV401, P501)에서 가장 높은 상태 점수가 산출됨을 확인하여, 모델이 단순히 이상을 탐지하는 것을 넘어 위치를 식별할 수 있음을 보여주었다.

## 🧠 Insights & Discussion

### 1. 강점

- **이질성 고려:** 단순한 시계열 예측을 넘어 변수의 데이터 타입(Continuous vs Discrete)에 따른 이질적 특성을 그래프 구조에 반영함으로써 모델의 강건함(Robustness)을 높였다.
- **설명 가능성:** 각 변수별로 상태 점수를 개별 산출함으로써, 운영자가 어떤 센서의 이상으로 인해 시스템 전체에 문제가 생겼는지 빠르게 파악할 수 있게 한다.
- **동적 상관관계 캡처:** 전역 임베딩과 지역 특징 유사도를 모두 활용하여, 시스템의 운전 상태 변화에 따른 동적인 변수 간 관계를 효과적으로 학습하였다.

### 2. 한계 및 비판적 해석

- **임계값 설정 문제:** 논문에서 이상치 판별을 위한 임계값($\tau$) 설정을 도메인 지식에 의존하며 명확한 자동 선택 방법을 제시하지 않았다. 이는 실제 적용 시 사용자에게 부담이 될 수 있다.
- **데이터셋의 다양성:** 비록 3종의 데이터셋을 사용했으나, 텍스트 데이터나 이미지 데이터와 같은 더 복잡한 형태의 이질적 데이터가 융합된 환경에서의 성능은 검증되지 않았다.
- **계산 복잡도:** 그래프 구조를 매번 학습하고 세 개의 서브그래프를 독립적인 GAT로 처리하는 구조는 실시간성이 극도로 중요한 시스템에서는 오버헤드가 될 가능성이 있다.

## 📌 TL;DR

본 논문은 다변량 시계열 데이터의 변수 타입(수치형, 범주형)에 따른 **이질성(Heterogeneity)**을 고려한 이상치 탐지 모델 **HFN**을 제안한다. 전역적 의미론적 임베딩과 지역적 특징 유사도를 융합하여 이질적 그래프 구조를 학습하고, 노드 및 채널 어텐션을 통해 최적의 특징 표현을 추출한다. 실험 결과, 기존의 동형 그래프 기반 모델이나 단순 시계열 모델보다 높은 탐지 정확도(특히 Recall)를 보였으며, 센서별 점수를 통해 이상치의 위치를 식별할 수 있음을 입증하였다. 이 연구는 복합적인 센서 데이터가 발생하는 실제 산업 제어 시스템(ICS)의 보안 및 유지보수 분야에 유용하게 적용될 가능성이 높다.
