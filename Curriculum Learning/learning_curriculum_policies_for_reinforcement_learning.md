# Learning Curriculum Policies for Reinforcement Learning

Sanmit Narvekar, Peter Stone (2019)

## 🧩 Problem to Solve

강화학습(Reinforcement Learning, RL)에서 매우 어려운 타겟 작업(target task)을 학습시키는 것은 많은 시간과 자원이 소요된다. 이를 해결하기 위해 더 쉬운 일련의 소스 작업(source tasks)들을 먼저 학습하고 그 지식을 전이하는 Curriculum Learning(커리큘럼 학습) 방법론이 사용된다.

본 논문에서 해결하고자 하는 핵심 문제는 **'학습 효율을 극대화할 수 있는 최적의 작업 순서(curriculum)를 어떻게 자동으로 설계할 것인가'**이다. 기존 연구들은 커리큘럼 설계를 위해 휴리스틱을 사용하거나 특정 제약 조건을 두었으나, 학습 에이전트의 상태에 따라 동적으로 다음 작업을 결정하는 '커리큘럼 정책(curriculum policy)' 자체를 데이터로부터 직접 학습시키려는 시도는 부족했다. 따라서 본 논문의 목표는 커리큘럼 설계 문제를 Markov Decision Process(MDP)로 정식화하고, 경험을 통해 학습 가능한 커리큘럼 정책을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 커리큘럼 생성 과정을 상위 수준의 MDP인 **Curriculum MDP(CMDP)**로 정의하고, 이 CMDP 상에서 동작하는 정책 $\pi^C$를 학습시키는 것이다.

주요 기여 사항은 다음과 같다:

1. **커리큘럼 정책의 학습 가능성 증명**: CMDP의 상태 공간을 적절히 표현(representation)함으로써, 커리큘럼 정책이 실제로 학습될 수 있음을 처음으로 입증하였다.
2. **CMDP 모델의 일반화**: 기존의 CMDP 모델을 확장하여 가치 함수 전이(Value Function Transfer)뿐만 아니라 보상 형성(Reward Shaping) 등 다양한 전이 학습(Transfer Learning) 알고리즘을 수용할 수 있도록 설계하였다.
3. **효율적인 상태 표현 제안**: 학습 에이전트의 가중치 벡터 $\theta$를 CMDP의 상태 변수로 활용하고, 이를 Tile Coding 등을 통해 효율적인 특징 공간(feature space)으로 변환하는 방법을 제안하였다.

## 📎 Related Works

기존의 커리큘럼 학습 연구들은 주로 도메인 전문가가 수동으로 순서를 정하거나, 타겟 작업의 초기/종료 상태 분포를 변경하는 식의 단순한 제약 조건을 활용하였다. 최근에는 CMDP를 활용해 작업을 시퀀싱하려는 시도가 있었으나, 이는 주로 휴리스틱을 이용해 단일한 커리큘럼 경로를 찾는 수준이었으며 전체 정책 $\pi^C$를 학습시키지는 않았다.

본 연구는 다음과 같은 차별점을 가진다:

- **데이터 기반 학습**: 사전 정의된 작업 기술자(task descriptors)나 휴리스틱에 의존하지 않고, 환경과의 직접적인 상호작용 경험을 통해 최적의 순서를 학습한다.
- **동적 정책**: 고정된 시퀀스가 아니라, 현재 학습 에이전트가 보유한 지식 상태($s^C$)에 따라 다음 작업을 유연하게 선택하는 정책을 학습한다.

## 🛠️ Methodology

### 전체 시스템 구조

시스템은 두 개의 에이전트로 구성된다.

1. **Learning Agent**: 타겟 작업(MDP)을 해결하려는 하위 수준 에이전트이다.
2. **Curriculum Agent**: 어떤 작업을 학습시킬지 결정하는 상위 수준 에이전트이며, CMDP 환경에서 동작한다.

### Curriculum MDP (CMDP) 정의

CMDP $\mathcal{M}^C$는 6-튜플 $(S^C, A^C, p^C, r^C, S^C_0, S^C_f)$로 정의된다.

- **상태 공간 ($S^C$)**: 학습 에이전트가 표현할 수 있는 모든 정책 $\pi$의 집합이다. 본 논문에서는 이를 학습 에이전트의 가중치 벡터 $\theta$로 표현한다.
- **액션 공간 ($A^C$)**: 학습 에이전트가 훈련할 수 있도록 미리 정의된 소스 작업들의 집합이다.
- **전이 함수 ($p^C$)**: 상태 $s^C$에서 작업 $a^C$를 학습한 후, 에이전트의 정책이 $s'^C$로 변할 확률이다. 이는 학습 알고리즘의 확률적 특성으로 인해 stochastic하게 결정된다.
- **보상 함수 ($r^C$)**: 상태 $s^C$에서 작업 $a^C$를 학습하는 데 걸리는 시간(경험 샘플 수 또는 wall clock time)의 음수 값이다. 즉, 학습 시간이 짧을수록 높은 보상을 얻는다.
- **종료 상태 ($S^C_f$)**: 타겟 작업에서 목표 성능 임계값 $\delta$를 달성한 정책 상태이다.

### CMDP 상태 표현 (State Representation)

가장 핵심적인 도전 과제는 연속적이고 거대한 가중치 공간 $\theta$를 어떻게 효율적인 특징 $\phi^C$로 변환하느냐이다.

1. **이산 상태 표현 (Discrete State Representation)**:
   타겟 작업의 상태 공간이 유한한 경우, 각 상태-액션 쌍에 대한 가치 값을 계산하여 $\theta'$를 생성한다. 이후 이 값들에 Tile Coding을 적용하여 특징을 추출한다.
2. **연속 상태 표현 (Continuous State Representation)**:
   $\theta$가 특정 상태에 국한되지 않는 경우, 각 가중치 $\theta_i$에 대해 개별적인 타일링을 생성하고 가중치 값들을 정규화(normalization)하여 특징을 추출한다.

### 전이 학습 메커니즘

학습 에이전트가 소스 작업에서 타겟 작업으로 지식을 전이하는 방법은 두 가지가 사용된다.

- **Value Function Transfer**: 소스 작업에서 학습한 액션-가치 함수 $Q_{source}$를 타겟 작업의 $Q_{target}$ 초기값으로 사용한다.
- **Reward Shaping**: 타겟 작업의 보상 함수 $r$에 소스 작업의 지식을 활용한 형성 보상 $f$를 추가한다.
  $$r'(s, a, s') = r(s, a, s') + f(s, a, s')$$
  여기서 $f$는 다음과 같은 전위 함수(potential function)의 차이로 정의된다.
  $$f(s, a, s') = \Phi(s', \pi(s')) - \Phi(s, a)$$
  이때 $\Phi$는 학습된 여러 소스 작업들의 가치 함수의 합으로 구성된다: $\Phi(s, a) = \sum_{i \in X} Q_i(s, a)$.

## 📊 Results

### 실험 설정

- **데이터셋/도메인**: Gridworld(기본, 액션 의존, 로프 사용 에이전트) 및 Ms. Pac-Man.
- **기준선(Baselines)**:
  - No Curriculum (타겟 작업 바로 학습)
  - 기존 휴리스틱 기반 정적 커리큘럼 (Narvekar et al. [16], Svetlik et al. [23])
  - Naive 학습 기반 접근 (학습한 작업의 리스트를 상태로 사용)
- **측정 지표**: 타겟 작업에서 목표 성능에 도달할 때까지 소요되는 총 비용(시간/스텝).

### 주요 결과

1. **Gridworld 결과**: 세 종류의 에이전트 모두에서 학습된 커리큘럼 정책이 'No Curriculum'보다 월등히 빠르게 학습되었으며, 기존의 정적 커리큘럼과 비슷하거나 더 나은 성능을 보였다. 특히, 이 방법은 소스 작업에 대한 사전 정보(task descriptors) 없이도 최적의 순서를 찾아냈다.
2. **Ms. Pac-Man 결과**: 훨씬 복잡한 도메인에서도 가치 함수 전이와 보상 형성 방식 모두에서 커리큘럼 정책 학습이 가능함을 보였다. 특히 보상 형성 방식에서는 기존 Svetlik et al.의 방법보다 더 효율적인 커리큘럼을 생성하였다.
3. **학습 중단 시점의 영향**: 소스 작업에서 반드시 수렴할 때까지 학습할 필요가 없으며, 매우 짧은 고정 기간(small fixed episodes)만 학습해도 커리큘럼 정책이 이를 보완하여 최적의 경로를 선택할 수 있음을 확인하였다.

## 🧠 Insights & Discussion

### 강점

본 논문은 커리큘럼 설계 문제를 단순한 시퀀싱이 아닌 '정책 학습'의 관점으로 접근함으로써, 에이전트의 현재 지식 상태에 반응하는 동적인 커리큘럼을 가능하게 했다. 또한, 가중치 $\theta$를 직접 상태로 활용하는 표현 방식을 통해 도메인에 구애받지 않는 일반적인 프레임워크를 제시하였다.

### 한계 및 비판적 해석

- **학습 비용의 문제**: 커리큘럼 정책 $\pi^C$를 학습시키기 위해 필요한 경험 데이터의 양이 타겟 작업을 그냥 학습시키는 것보다 훨씬 많을 수 있다. 이는 '메타 학습' 단계에서 발생하는 오버헤드이다.
- **가정**: 본 연구는 소스 작업의 집합 $A^C$가 미리 제공된다고 가정한다. 만약 소스 작업을 자동으로 생성하는 단계까지 포함된다면 더 강력한 시스템이 될 것이다.
- **확장성**: 현재는 가치 함수 기반(Value-function-based) 방법론에 집중되어 있어, Policy Gradient 방식의 에이전트에게도 동일한 효율성이 나타날지는 미지수이다.

## 📌 TL;DR

본 논문은 커리큘럼 설계를 CMDP(Curriculum MDP) 문제로 정의하고, 에이전트의 가중치 $\theta$를 상태로 사용하여 **학습 가능한 커리큘럼 정책 $\pi^C$**를 제안하였다. 실험을 통해 제안 방법이 다양한 전이 학습 알고리즘과 복잡한 도메인(Gridworld, Ms. Pac-Man)에서 타겟 작업의 학습 속도를 획기적으로 향상시킴을 증명하였다. 이 연구는 향후 유사한 타겟 작업들에 대해 학습된 커리큘럼을 재사용함으로써 메타 학습 비용을 절감하는 연구의 기초가 될 것으로 보인다.
