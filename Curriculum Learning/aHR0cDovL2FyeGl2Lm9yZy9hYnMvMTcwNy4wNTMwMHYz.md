# Reverse Curriculum Generation for Reinforcement Learning

Carlos Florensa, David Held, Markus Wulfmeier, Michael R. Zhang, Pieter Abbeel (2017)

## 🧩 Problem to Solve

본 논문은 강화학습(Reinforcement Learning, RL)에서 특정 목표 상태에 도달하거나 물체를 원하는 구성으로 조작해야 하는 **Goal-oriented tasks**를 해결하고자 한다. 이러한 작업들은 다음과 같은 핵심적인 문제점을 가진다.

1.  **희소 보상(Sparse Reward) 문제**: 목표에 도달했을 때만 보상을 주는 이진 보상(Binary Reward) 함수를 사용할 경우, 에이전트가 우연히 목표에 도달하기 전까지는 아무런 학습 신호를 받지 못해 탐색(Exploration) 비용이 지나치게 높아진다.
2.  **보상 설계(Reward Shaping)의 어려움**: 희소 보상을 해결하기 위해 보상 함수를 세밀하게 설계하는 Reward Shaping 방식을 사용할 수 있으나, 이는 도메인 전문가의 많은 노력과 실험을 필요로 하며, 잘못 설계할 경우 의도치 않은 행동(Undesired behaviors)을 학습하는 부작용이 발생한다.
3.  **전문가 시연(Expert Demonstrations)의 의존성**: 전문가의 데이터를 사용하는 방식은 반드시 외부의 개입이 필요하다는 제약이 있다.

따라서 본 논문의 목표는 전문가의 시연이나 인위적인 Reward Shaping 없이, **단 하나의 목표 상태(Goal state) 정보만을 가지고도 복잡한 목표 지향적 작업을 효율적으로 학습할 수 있는 자동 커리큘럼 생성 방법론을 제시하는 것**이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **"목표 지점으로부터 거꾸로 학습한다"**는 직관에 기반한다.

1.  **Reverse Learning 전략**: 목표 상태와 매우 가까운 상태에서 시작하여 목표에 도달하는 법을 먼저 배우고, 점진적으로 목표에서 더 멀리 떨어진 시작 상태로 학습 범위를 확장하는 방식을 제안한다.
2.  **자동 커리큘럼 생성**: 에이전트의 현재 성능을 관찰하여 학습하기에 적절한 난이도의 시작 상태 분포 $\rho_i$를 자동으로 생성하고 적응시킨다.
3.  **Action-space 기반의 주변 상태 샘플링**: 상태 공간(State-space)에서 단순히 노이즈를 추가하는 대신, 액션 공간(Action-space)에서 무작위 행동을 취하는 **Brownian motion** 방식을 통해 물리적으로 실행 가능한(Feasible) 주변 상태들을 생성한다.

## 📎 Related Works

기존의 커리큘럼 학습 방식들은 주로 다음과 같은 한계점을 가지고 있다.

1.  **수동 설계 커리큘럼**: 미리 정의된 작업 순서에 의존하며, 에이전트의 실시간 성능에 적응하지 못한다.
2.  **내재적 동기(Intrinsic Motivation)**: 학습 진행도에 따라 작업을 분할하는 방식이 있으나, 정밀한 조작 작업(Fine-grained manipulation)과 같이 연속적이고 고차원적인 상태 공간에서는 적용이 어렵다.
3.  **Asymmetric Self-play**: 한 에이전트(Alice)가 다른 에이전트(Bob)를 위해 시작 상태를 제안하는 방식이다. 그러나 연속적인 액션 공간에서는 Alice가 지역 최적점(Local optima)에 빠져 다양한 시작 상태를 제안하지 못하는 문제가 발생한다.

본 연구는 이러한 기존 방식과 달리, **목표 상태로부터의 역방향 확장**과 **성공 확률 기반의 "Good Starts" 필터링**을 통해 더 안정적이고 효율적인 커리큘럼을 구축한다.

## 🛠️ Methodology

### 전체 시스템 구조 및 파이프라인
본 방법론은 에이전트가 목표 상태 $s_g$에서 시작하여 점진적으로 더 넓은 영역의 시작 상태 $S_0$에서도 목표에 도달할 수 있도록 학습시킨다. 전체 과정은 다음과 같은 루프를 따른다.
1.  현재의 "Good starts" 리스트에서 주변 상태들을 샘플링하여 새로운 시작 상태 분포 $\rho_i$를 생성한다.
2.  $\rho_i$를 기반으로 정책 $\pi_i$를 학습시킨다.
3.  학습된 정책의 성공 확률을 평가하여, 너무 쉽거나 너무 어려운 상태를 제외한 새로운 "Good starts"를 선별한다.
4.  이 과정을 반복하며 학습 영역을 확장한다.

### 주요 구성 요소 및 상세 설명

#### 1. 문제 정의 및 보상 함수
에이전트의 목표는 시작 상태 분포 $\rho_0$에서 샘플링된 $s_0$로부터 목표 영역 $S_g$에 도달하는 정책 $\pi_\theta$를 찾는 것이다. 보상 함수는 다음과 같은 단순한 이진 형태를 가진다.
$$r(s_t) = \mathbb{1}\{s_t \in S_g\}$$
이때, 시작 상태 $s_0$에 대한 기대 수익(Expected return) $R(\pi, s_0)$는 시간 $T$ 내에 목표에 도달할 확률과 동일하다.
$$R(\pi, s_0) = P(\bigcup_{t=0}^T s_t \in S_g \mid \pi, s_0)$$

#### 2. Reverse Curriculum 생성을 위한 가정
본 방법론은 다음 세 가지 가정을 전제로 한다.
- **Assumption 1**: 에이전트를 임의의 시작 상태 $s_0 \in S$로 리셋할 수 있다.
- **Assumption 2**: 목표 영역에 속하는 최소 하나의 상태 $s_g \in S_g$가 제공된다.
- **Assumption 3**: 무작위 행동을 통해 생성되는 마르코프 연쇄(Markov Chain)가 모든 시작 상태 $S_0$와 목표 상태 $s_g$를 포함하는 communicating class를 가진다. (즉, 가역성이 존재하여 목표에서 시작점으로, 시작점에서 목표로 이동이 가능하다.)

#### 3. "Good Starts"의 정의 및 샘플링
학습 효율을 극대화하기 위해, 에이전트가 **"가끔 성공하지만 항상 성공하지는 않는"** 상태에서 학습하는 것이 가장 효과적이라고 가정한다.
$$S_0^i = \{s_0 : R_{min} < R(\pi^i, s_0) < R_{max}\}$$
여기서 $R_{min}$과 $R_{max}$는 성공 확률의 하한과 상한을 의미한다.

#### 4. SampleNearby 절차 (Brownian Motion)
상태 공간에서 단순 유클리드 거리 기반으로 샘플링하면 물리적으로 불가능한 상태(예: 물체 겹침)가 생성될 가능성이 높다. 이를 방지하기 위해 액션 공간에서 노이즈를 주는 방식을 사용한다.
- 시드 상태(Seed states)에서 시작하여 $T_B$ 스텝 동안 무작위 액션 $a_t \sim \mathcal{N}(0, \Sigma)$를 취한다.
- 이 경로 상의 모든 상태들은 물리적으로 도달 가능한 상태임이 보장되므로, 이를 새로운 시작 상태 후보로 사용한다.

#### 5. 학습 절차 (Algorithm 1)
- **초기화**: 정책 $\pi_0$와 목표 상태 $s_g$를 설정한다.
- **반복 학습**:
    - `SampleNearby`를 통해 새로운 상태들을 생성하고, 이전의 "Good starts" 리스트에서 일부를 샘플링하여 $\rho_i$를 구성한다.
    - TRPO(Trust Region Policy Optimization)와 같은 On-policy 알고리즘을 사용하여 정책 $\pi_i$를 업데이트한다.
    - 각 상태별 성공 확률 $R(\pi, s_0)$를 추정하여 $[R_{min}, R_{max}]$ 범위에 있는 상태들만 다시 "Good starts"로 저장한다.

## 📊 Results

### 실험 설정
- **데이터셋 및 작업**: MuJoCo 시뮬레이터에서 네 가지 작업 수행.
    - Point-mass maze: G자형 미로 네비게이션.
    - Ant maze: 4족 보행 로봇의 U자형 미로 네비게이션.
    - Ring on Peg: 7-DOF 로봇 팔을 이용해 링을 핀에 끼우는 작업.
    - Key insertion: 7-DOF 로봇 팔을 이용해 열쇠를 구멍에 넣고 돌리는 작업.
- **비교 대상(Baselines)**:
    - Uniform Sampling: 시작 상태를 $\rho_0$에서 균일하게 샘플링 (기존 TRPO).
    - Brownian from All Starts: "Good starts" 필터링 없이 모든 샘플링 상태를 사용하는 방식.
    - Oracle: 모든 상태를 전수 조사하여 정확히 $S_0^i$를 샘플링하는 이상적인 방식.
    - Asymmetric Self-play: Alice와 Bob 두 에이전트를 이용한 커리큘럼 생성 방식.

### 주요 결과
1.  **학습 가능 여부**: Ring on Peg와 Key insertion 같은 정밀 조작 작업에서 Uniform Sampling은 성공률이 매우 낮아(각각 10%, 2%) 사실상 학습에 실패했다. 반면, 제안 방법론은 성공적으로 작업을 완수했다.
2.  **"Good Starts"의 효과**: 모든 샘플링 상태를 사용하는 것보다 성공 확률 기반으로 필터링된 "Good starts"에서 학습했을 때 학습 속도가 훨씬 빨랐다.
3.  **Brownian Motion의 유효성**: 액션 공간 기반의 샘플링이 실제 "Good starts"를 생성하는 비율이 훨씬 높음을 확인하였다.
4.  **Asymmetric Self-play와의 비교**: Point-mass maze에서 Alice 에이전트가 지역 최적점에 빠져 Bob에게 유용한 상태를 제안하지 못하는 현상이 발견되었으며, 제안 방법론이 훨씬 안정적이었다.

## 🧠 Insights & Discussion

### 강점
- **전문가 지식 배제**: Reward Shaping이나 시연 데이터 없이 단 하나의 목표 상태만으로 복잡한 작업을 해결했다.
- **물리적 타당성 확보**: Brownian motion을 통해 고차원 연속 상태 공간에서도 실행 가능한 상태들만 효율적으로 샘플링했다.
- **효율적인 샘플 활용**: "Good starts" 개념을 통해 너무 쉽거나 너무 어려운 샘플에 낭비되는 계산 자원을 줄였다.

### 한계 및 논의사항
- **확장 방향의 균일성**: 현재 방식은 목표 지점에서 모든 방향으로 균일하게 확장한다. 만약 목표 지점과 실제 시작 지점이 매우 멀다면, 특정 방향으로 우선 확장하는 Bias를 주는 것이 더 효율적일 수 있다.
- **가역성 가정**: 본 방법론은 시스템의 가역성(Assumption 3)에 크게 의존한다. 만약 한 번 수행하면 되돌릴 수 없는 동작(Irreversible action)이 많은 환경에서는 적용이 어려울 수 있다.
- **실제 환경 적용**: 시뮬레이션에서는 리셋(Reset)이 자유롭지만, 실제 로봇 환경에서는 임의의 상태로 리셋하는 것이 어렵다. 이를 해결하기 위해 Domain Randomization 등의 기법과 결합할 필요가 있다.

## 📌 TL;DR

본 논문은 희소 보상 환경의 강화학습 문제를 해결하기 위해, 목표 상태로부터 역방향으로 학습 영역을 넓혀가는 **Reverse Curriculum Generation** 방법을 제안한다. 액션 공간에서의 무작위 행보(Brownian motion)를 통해 실행 가능한 주변 상태를 생성하고, 에이전트의 성공 확률에 따라 학습 난이도를 조절하는 "Good Starts" 필터링을 적용함으로써, 기존 RL 방식으로는 해결 불가능했던 정밀한 로봇 조작 작업을 성공적으로 수행하였다. 이는 향후 보상 설계가 어려운 복잡한 물리 제어 시스템의 자동 학습에 중요한 기여를 할 가능성이 높다.
