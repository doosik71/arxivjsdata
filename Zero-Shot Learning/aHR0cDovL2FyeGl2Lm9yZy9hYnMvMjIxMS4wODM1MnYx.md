# Visual Semantic Segmentation Based on Few/Zero-Shot Learning: An Overview

Wenqi Ren, Yang Tang, Qiyu Sun, Chaoqi Zhao, Qing-Long Han

## 🧩 Problem to Solve

기존 시각 의미론적 분할(Visual Semantic Segmentation) 방법은 방대한 양의 고밀도 주석 데이터에 크게 의존하며, 학습 시 보지 못한(unseen) 범주에 대한 정확한 의미론적 레이블 추정에 실패합니다. 이는 동적인 시나리오에서의 유연한 적용을 방해하고, 3D 데이터와 같이 주석 작업이 노동 집약적이고 비용이 많이 드는 문제점을 야기합니다. 이 논문은 이러한 한계를 극복하기 위해 소수의 레이블된 샘플(few-shot) 또는 레이블이 전혀 없는 샘플(zero-shot)만으로도 새로운 범주를 학습할 수 있는 시각 의미론적 분할 방법을 연구합니다.

## ✨ Key Contributions

* 다양한 Few/Zero-Shot 시각 분할 태스크의 문제 설정에 대한 비교와 기술 솔루션 요약 제공.
* Few/Zero-Shot 시각 의미론적 분할의 발전 사항을 검토하고, 이미지 의미론적 분할 (Image Semantic Segmentation, ISS), 비디오 객체 분할 (Video Object Segmentation, VOS), 3D 분할 (3D Segmentation, 3DS) 등 다양한 분할 태스크에서의 기술 솔루션 차이점을 명시.
* Few/Zero-Shot 시각 의미론적 분할이 직면한 데이터, 알고리즘, 응용 분야의 공개 과제들을 논의하여 후속 연구자들에게 통찰력 제공.

## 📎 Related Works

* Few/Zero-Shot 학습 및 시각 분할에 대한 기존 서베이 논문들 [132]–[134].
* 딥러닝(Deep Learning) 기반 시각 의미론적 분할의 발전과 한계점.
* Few-Shot Learning [129]–[131] 및 Zero-Shot Learning [129], [130], [137]의 기본 개념과 정의.
* 약지도 학습(Weakly-Supervised Learning) [126]–[128] 방법론.

## 🛠️ Methodology

이 논문은 Few/Zero-Shot 시각 의미론적 분할 방법론을 크게 **판별적(Discriminative) 접근**과 **생성적(Generative) 접근**으로 분류하고, 이를 ISS, VOS, 3DS 세 가지 주요 태스크에 적용된 사례를 검토합니다.

* **접근 방식 분류:**
  * **판별적 접근**: $P(m|x)$를 최대화하는 강력한 분할 모델 구축에 중점을 둡니다.
    * **거리 학습 기반 (Metric Learning-based)**: 특징 공간에서 유사도를 측정하는 거리 함수를 학습하여 픽셀별 분류를 수행합니다. 프로토타입 네트워크 [145]를 활용하거나, 다중 프로토타입 [27] 또는 조밀한 매칭 [32]을 통해 세밀한 정보 캡처를 시도합니다.
    * **매개변수 예측 기반 (Parameter Prediction-based)**: 특정 태스크의 예측 계층에 필요한 신경망 가중치를 직접 예측하는 생성기를 사용하여 빠른 클래스 간 적응을 달성합니다 [14], [38].
    * **미세 조정 기반 (Fine-tune-based)**: 사전 학습된 분할 네트워크의 매개변수를 소수의 레이블된 샘플에 맞춰 최적화 알고리즘으로 미세 조정하여 빠르게 수렴시킵니다 [40], [41].
    * **메모리 기반 (Memory-based)**: RNN (Recurrent Neural Networks)이나 메모리 네트워크 [73]–[75] 등을 활용하여 이전에 본 정보나 시간적 정보를 캐시하고 인코더와 협력하여 객체 표현을 강화합니다.
  * **생성적 접근**: $P(x|m)P(m)$을 모델링하며, 생성기를 통해 보지 못한 클래스의 레이블된 인스턴스(가상 시각 특징)를 합성하여, 이를 활용해 판별적 모델의 매개변수를 조정합니다 [52]–[55] (그림 5 참조).

* **주요 태스크별 적용:**
  * **Few/Zero-Shot 이미지 의미론적 분할 (ISS)**: Few-shot 환경에서는 거리 학습 기반 방법(프로토타입 네트워크, 다중 프로토타입)이 주류를 이루며, Zero-shot 환경에서는 시각 특징과 의미론적 특징을 공유 공간에 투영하는 거리 학습 기반과 가상 특징을 생성하는 생성 모델 기반이 활용됩니다.
  * **Few/Zero-Shot 비디오 객체 분할 (VOS)**: 비디오의 시간적 연속성을 활용하여 이전 프레임이나 초기 프레임과의 조밀한 매칭을 사용하는 거리 학습 기반과 시공간 메모리 네트워크, GNN (Graph Neural Networks), RNN 등을 활용하는 메모리 기반 방법이 중요하게 다루어집니다 [73], [74], [83], [84]. Fine-tune 기반도 적용되나 실시간 성능에 제약이 있습니다.
  * **Few/Zero-Shot 3D 분할 (3DS)**: 3D 데이터의 불규칙성과 복잡성으로 인해 2D 방법론을 직접 적용하기 어렵습니다. Few-shot에서는 거리 학습 기반(프로토타입 네트워크 [106], 형상 변형 [108]), 매개변수 예측 기반 [110], 미세 조정 기반(메타 학습 [111], 자기 지도/비지도 사전 학습 [112]–[114]) 방법이 연구되며, Zero-shot에서는 생성 모델 기반(의미론적으로 일관된 특징 생성)이 주로 사용됩니다 [116], [117].

## 📊 Results

이 개요 논문은 새로운 실험 결과를 제시하기보다는, 조사된 Few/Zero-Shot 시각 의미론적 분할 방법론들의 전반적인 성과와 특징, 그리고 태스크별 동향을 요약합니다.

* **전반적인 성과:** Few/Zero-Shot 시각 의미론적 분할은 제한된 수의 주석된 샘플만으로 보지 못한 범주에 대한 분할을 가능하게 하여 기존 딥러닝 기반 분할 방법의 주요 한계점(방대한 주석 데이터 요구)을 효과적으로 해결했습니다.
* **기술 솔루션별 특징:**
  * **거리 학습 및 매개변수 예측 기반:** 단일 순방향 전파와 소수 특징 간의 유사도 계산으로 인해 낮은 계산 비용과 짧은 적응 기간을 가지며 효율적입니다.
  * **미세 조정 및 생성 모델 기반:** 다중 순방향/역방향 전파로 인해 더 높은 계산 비용과 긴 적응 기간을 요구하지만, Zero-shot 시나리오에서 생성 모델 기반은 가상 데이터를 통해 보지 못한 클래스 학습을 가능하게 합니다.
  * **메모리 기반:** 시간적 정보 캡처 능력 덕분에 비디오 객체 분할 (VOS)과 같은 순차 학습 문제에서 특히 뛰어난 성능을 보였습니다.
* **태스크별 동향:**
  * **Few-shot ISS:** 특징 표현, 거리 함수 및 특징 매칭 전략의 다양성을 바탕으로 프로토타입 네트워크 기반의 거리 학습 방법이 지배적입니다.
  * **Zero-shot ISS:** 구현 용이성으로 인해 거리 학습 기반 방법이 선호되며, 생성 모델을 통해 Zero-shot 문제를 Few-shot 문제로 전환하는 접근 방식도 인기를 얻고 있습니다.
  * **Few/Zero-shot VOS:** 시간적 정보 활용에 대한 높은 요구사항 때문에 메모리 기반 방법이 선호되며, ISS와 비교하여 데이터 부족, 누적 오류, 실시간 성능 등의 새로운 도전과제를 안고 있습니다.
  * **Few/Zero-shot 3DS:** 3D 샘플의 무질서함과 복잡성으로 인해 2D 공간에서 효과적인 기술 솔루션들을 직접 적용하기 어려워 아직 초기 단계에 있으며, 3D 데이터 구조에 특화된 효율적인 솔루션 개발이 필요합니다.

## 🧠 Insights & Discussion

Few/Zero-Shot 시각 의미론적 분할은 상당한 발전을 이루었지만, 실제 환경 적용을 위해서는 여전히 여러 과제를 안고 있습니다.

* **도메인 간 전이성 (Cross-domain transferability):** 현재 방법론들은 미확인 클래스 샘플이 기본 클래스와 동일한 분포를 따른다고 가정하지만, 실제 환경에서는 이러한 가정이 제한적입니다. 도메인 적응 (domain adaptation) 기술과 클래스 간 적응 (cross-category adaptation) 기술을 통합하여 모델의 실제 시나리오 적용성을 높이는 연구가 필요합니다.
* **일반화된 Few/Zero-Shot 분할 (Generalized few/zero-shot segmentation):** 모델이 학습된(base) 클래스와 보지 못한(unseen) 클래스 모두를 효과적으로 분할할 수 있도록 성능을 개선해야 합니다. 실제 응용 분야에서는 다양한 범주의 객체를 동시에 분할하는 능력이 요구됩니다.
* **약지도 Few/Zero-Shot 분할 (Weakly-supervised few/zero-shot segmentation):** 현재 방법들은 보지 못한 범주를 학습하기 위해 기본 클래스에 대한 많은 양의 레이블된 샘플을 여전히 필요로 합니다. 이를 완화하기 위해 약지도 학습 또는 비지도 학습 [156]을 Few/Zero-Shot 분할과 결합하는 연구가 유망합니다.
* **다중 모달 감독 (Multi-modal supervision):** Few-shot 방법은 단일 모달 (예: 2D 이미지) 지원 세트에만 의존하는 경향이 있습니다. 언어 표현이나 광학 흐름(optical flow)과 같은 다른 모달리티에서 보조 감독 신호를 활용하여 감독 정보를 보강하고, Zero-shot 분할의 정보 풍부함을 더욱 증진시키는 것이 중요합니다.
* **경량 네트워크 (Lightweight network):** 임베디드 장치에 배포하기 위해서는 계산 오버헤드와 런타임 비용을 줄인 경량 네트워크 설계가 필수적입니다. 실시간 성능과 높은 예측 품질 간의 균형을 찾는 것이 이 분야의 중요한 연구 방향입니다.
* **태스크 간 협업 (Cross-task collaboration):** Few/Zero-Shot 시각 의미론적 분할을 Few-shot 객체 탐지 [159]와 같은 다른 컴퓨터 비전 태스크와 결합하여 모델의 능력을 확장하고, 제한된 샘플의 가치를 극대화할 수 있습니다.
* **태스크 간 학습 (Cross-task learning):** ISS, VOS, 3DS 간의 기술 솔루션에서 공통점과 보편성을 활용하여 한 분야에서 제안된 아이디어를 다른 분야로 전이하는 것이 가능하고 상호 유익합니다 (예: Few-shot ISS의 외관 간극 해결 전략을 Few-shot VOS의 장시간 시퀀스 문제에 적용).
* **응용 분야 (Applications):** 석유화학 산업 (예: 파이프라인 누출 감지 [160], 다양한 결함 유형 [161] 처리)과 같은 실제 시나리오에서 레이블된 데이터가 부족하거나 빠른 적응이 필요한 경우 Few/Zero-Shot 의미론적 분할은 중요한 역할을 합니다. 또한 다중 모달 정보를 활용하여 전천후 시나리오 (예: 적외선 이미지 기반 안전 생산 모니터링 [162], [163])에 적용할 수 있습니다.

## 📌 TL;DR

* **문제:** 기존 시각 의미론적 분할은 대규모 레이블 데이터에 의존하며, 보지 못한(unseen) 범주에 대한 분할에 취약합니다.
* **해결책:** 이 논문은 소수의 레이블(Few-shot) 또는 레이블 없는(Zero-shot) 샘플만으로 미확인 범주를 분할하는 시각 의미론적 분할(Visual Semantic Segmentation) 방법론들을 2D 및 3D 공간에 걸쳐 체계적으로 분석합니다.
* **핵심 내용:** 거리 학습, 매개변수 예측, 미세 조정, 메모리 기반 및 생성 모델 기반의 다섯 가지 주요 기술 솔루션을 분류하고, 이미지(ISS), 비디오(VOS), 3D 분할(3DS) 태스크별 적용 사례와 장단점을 비교합니다. 또한 도메인 간 전이성, 일반화, 약지도 학습, 다중 모달 감독, 경량 네트워크, 태스크 간 협업/학습 등 미래 연구의 주요 과제들을 제시합니다.
