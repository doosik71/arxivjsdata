# Zero-Shot Anomaly Detection via Batch Normalization

Aodong Li, Chen Qiu, Marius Kloft, Padhraic Smyth, Maja Rudolph, Stephan Mandt (2023)

## 🧩 Problem to Solve

본 논문은 데이터의 '정상(normal)' 분포가 시간이 지남에 따라 변화하거나 환경에 따라 달라지는 **분포 변화(Distribution Shift)** 상황에서, 새로운 정상 데이터에 대한 추가 학습 없이도 이상치를 탐지해야 하는 **Zero-shot Anomaly Detection (AD)** 문제를 해결하고자 한다.

일반적인 이상 탐지 모델은 학습 시 보았던 정상 분포에 고정되어 있어, 테스트 단계에서 정상 분포가 변하면(예: 의료 영상 장비의 교체, 환자 인구통계학적 변화 등) 정상 데이터를 이상치로 오인하는 문제가 발생한다. 이를 해결하기 위해 기존에는 소수의 샘플을 사용하는 Few-shot AD나 거대 기초 모델(Foundation Models)을 활용한 Zero-shot AD가 제안되었으나, 전자는 지원 세트(Support set)가 필요하고 후자는 주로 자연 이미지에 국한되며 전문가의 프롬프트가 필요하다는 한계가 있다.

따라서 본 연구의 목표는 이미지뿐만 아니라 정형 데이터(Tabular data) 등 도메인에 관계없이 적용 가능하며, 추가 학습이나 인간의 개입 없이 배치의 통계적 특성만을 이용하여 즉각적으로 적응하는 Zero-shot AD 방법론을 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Batch Normalization (BN)**과 **Meta-training**의 결합을 통해 **Adaptive Centered Representations (ACR)**를 학습시키는 것이다.

중심적인 직관은 다음과 같다. 어떤 배치 내에 정상 데이터가 다수를 차지하고 있다면, BN의 중심화(Recentering) 및 스케일링(Scaling) 연산은 정상 샘플들을 원점 근처로 모으는 반면, 소수인 이상치들은 원점에서 멀리 떨어지게 만든다는 점이다. 이를 위해 모델이 다양한 분포의 데이터셋에 대해 이러한 '중심화' 능력을 갖추도록 Meta-training을 수행함으로써, 학습 시 보지 못한 새로운 분포의 데이터가 들어와도 BN 레이어를 통해 자동으로 적응하여 이상치를 구분해낼 수 있게 한다.

## 📎 Related Works

1. **Deep AD**: Autoencoder나 One-class Classification 기반의 딥러닝 모델들이 제안되었으나, 대부분 테스트 시 데이터 분포가 일정하다는 가정을 전제로 하며, 분포 변화가 발생하면 성능이 급격히 저하된다.
2. **Few-shot AD**: MAML과 같은 메타 학습을 통해 소수의 샘플로 빠르게 적응하는 방식이 연구되었다. 하지만 테스트 시점에 레이블이 지정된 소수의 정상 샘플(Support set)이 반드시 필요하며, 적응을 위한 추가적인 그래디언트 업데이트 과정이 필요하여 실시간 배포에 제약이 있다.
3. **Zero-shot AD**: CLIP과 같은 거대 기초 모델을 활용하여 텍스트 설명과 이미지 간의 정렬(Alignment)을 통해 이상치를 탐지하는 방식이 최근 주목받고 있다. 그러나 이러한 모델은 자연 이미지 외의 전문 도메인(의료, 산업용) 데이터나 정형 데이터에는 적용하기 어렵고, 효과적인 텍스트 프롬프트를 작성하기 위한 인간 전문가의 개입이 필수적이다.

ACR은 위 방법론들과 달리 추가 샘플이나 프롬프트 없이 **배치 수준(Batch-level)**의 통계량만을 활용하여 도메인 독립적인 Zero-shot 적응을 수행한다는 점에서 차별화된다.

## 🛠️ Methodology

### 1. 전체 파이프라인 및 구조

ACR은 기존의 딥러닝 기반 이상 탐지 모델(예: Deep SVDD, NTL)에 Batch Normalization 레이어를 적극적으로 배치하고, 이를 특수한 메타 학습 전략으로 훈련시킨 구조이다.

### 2. 주요 구성 요소 및 역할

- **Batch Normalization (BN)**: 입력 배치의 평균을 0, 분산을 1로 변환하여 서로 다른 분포의 데이터를 공통의 참조 프레임으로 가져오는 적응 모듈 역할을 한다.
- **Feature Extractor**: BN 레이어를 포함한 신경망으로, 데이터를 '적응적으로 중심화된 표현(Adaptively Centered Representations)'으로 변환한다.
- **Anomaly Scorer**: 중심화된 표현을 바탕으로 원점으로부터의 거리 등을 계산하여 이상치 점수를 부여한다.

### 3. 훈련 목표 및 손실 함수

모델이 특정 분포에 과적합되지 않고 일반적인 적응 능력을 배우도록 **Meta Outlier Exposure** 전략을 사용한다. $K$개의 상호 연관된 훈련 분포 $P_1, \dots, P_K$가 있을 때, 각 분포 $P_j$에 대해 다른 분포 $\bar{P}_j$의 데이터를 인위적으로 섞어 오염된 분포 $P^\pi_j$를 생성한다.

$$P^\pi_j := \pi P_j + (1-\pi) \bar{P}_j, \quad \bar{P}_j := \frac{1}{K-1} \sum_{i \neq j} P_i$$

여기서 $\pi$는 정상 데이터의 비율을 의미한다. 손실 함수는 정상 샘플 $x_i$에 대해서는 이상치 점수 $S_\theta$를 최소화하고, 인위적으로 섞은 이상치 샘플에 대해서는 역점수 $A_\theta$를 최소화하도록 설계된다.

$$L[S_\theta(x^B)] = \frac{1}{B} \sum_{i \in B} \{ (1-y_i) S^i_\theta(x^B) + y_i A^i_\theta(x^B) \}$$

- $y_i$: 데이터 $x_i$가 분포 $P_j$에 속하지 않으면 1(이상치), 속하면 0(정상)인 지표 변수이다.
- $S_\theta$: 정상 샘플이 낮게, 이상치가 높게 나오도록 하는 점수 함수이다.
- $A_\theta$: $S_\theta$와 반대로 동작하는 함수이다.

### 4. 추론 절차 (Zero-shot Prediction)

테스트 시에는 학습 과정에서 보지 못한 새로운 분포 $P^*$의 데이터 배치를 입력으로 받는다. 모델은 추가 학습 없이 BN 레이어를 통해 해당 배치의 통계량을 즉석에서 계산하여 데이터를 중심화하고, 중심으로부터의 거리를 측정하여 이상치 여부를 판별한다.

### 5. 이론적 근거

논문은 Theorem 2.1을 통해 일반화 오차(Generalization Error)가 변환된 분포들 간의 **Total Variation (TV) distance**에 의해 상한선이 결정됨을 수학적으로 증명하였다. BN이 모든 분포를 평균 0, 분산 1로 재보정하기 때문에 TV 거리가 줄어들며, 결과적으로 보지 못한 분포 $P^*$에 대해서도 낮은 일반화 오차를 가질 수 있음을 시사한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**:
  - 이미지: CIFAR100-C (노이즈 오염), OrganA (의료 영상), MVTec AD (산업용 부품 결함).
  - 정형 데이터: Anoshift (네트워크 트래픽), Malware (악성 코드).
- **지표**: AUROC (Area Under the Receiver Operating Characteristic curve).
- **기준선(Baselines)**: ADIB, OC-MAML, CLIP-AD, WinCLIP, OC-SVM, IForest 등.

### 2. 주요 결과

- **이미지 이상 탐지**: 의료 영상 데이터셋(OrganA) 및 비자연 이미지에서 CLIP-AD를 포함한 모든 베이스라인을 상회하는 성능을 보였다. 특히 노이즈가 섞인 CIFAR100-C에서도 경쟁력 있는 성능을 기록하였다.
- **이미지 이상 세그멘테이션**: MVTec AD 벤치마크에서 기존 SOTA인 WinCLIP보다 평균 7.4% 높은 AUROC를 달성하며 새로운 SOTA 성능을 기록하였다.
- **정형 데이터 이상 탐지**: Anoshift와 Malware 데이터셋에서 기존 딥러닝 및 머신러닝 기반 모델들을 압도하였다. 특히 분포 변화가 심한 상황(Anoshift의 FAR split)에서 대부분의 베이스라인이 무작위 추측(Random guessing)보다 낮은 성능을 보인 반면, ACR은 유의미하게 높은 탐지 성능을 유지하였다.

## 🧠 Insights & Discussion

### 1. 강점

- **범용성**: 특정 데이터 타입에 의존하는 기초 모델과 달리, BN이라는 보편적인 메커니즘을 사용하므로 이미지, 정형 데이터, 시계열 데이터 등 모든 도메인에 적용 가능하다.
- **효율성**: 테스트 시점에 추가적인 그래디언트 업데이트나 지원 세트(Support set)가 필요 없으므로 연산 비용이 낮고 실시간 배포에 유리하다.
- **이론적 토대**: BN의 중심화 작용이 어떻게 일반화 오차를 줄이는지 수학적으로 분석하여 방법론의 타당성을 뒷받침하였다.

### 2. 한계 및 가정

- **배치 수준의 예측**: 단일 샘플이 아닌 배치 단위로 예측해야 하므로, 실시간 단일 샘플 처리를 위해서는 별도의 통계량 저장 전략이 필요하다.
- **정상 데이터 다수 가정**: 배치 내에 정상 데이터가 다수를 차지해야 한다는 가정($A3$)이 필수적이다. 만약 배치 내 이상치 비율이 매우 높다면 BN의 통계량이 오염되어 성능이 저하될 수 있다.
- **메타 학습 세트 필요**: 상호 연관된 여러 분포의 데이터셋($A1$)이 학습 단계에서 준비되어야 한다.

### 3. 비판적 해석

본 연구는 BN의 단순한 특성을 메타 학습과 결합하여 Zero-shot AD라는 난제를 해결했다는 점에서 매우 영리한 접근 방식을 취하고 있다. 특히 기초 모델(Foundation Model)이 커버하지 못하는 전문 영역(Medical, Industrial, Cyber-security)에서의 효용성을 입증한 점이 고무적이다. 다만, '정상 데이터가 다수'라는 가정이 실제 매우 희소한 정상 데이터 환경에서 어떻게 작동할지에 대한 추가적인 분석이 필요해 보인다.

## 📌 TL;DR

본 논문은 **Batch Normalization의 중심화 특성**과 **메타 학습(Meta-training)**을 결합하여, 새로운 데이터 분포에 대해 추가 학습 없이 즉각적으로 적응하는 **Zero-shot 이상 탐지 방법론(ACR)**을 제안한다. 이 방법은 이미지와 정형 데이터 모두에서 탁월한 성능을 보였으며, 특히 의료 영상 및 산업 결함 탐지, 사이버 보안 데이터와 같은 특수 도메인에서 기존 SOTA 모델들을 능가하는 범용적인 적응 능력을 입증하였다.
