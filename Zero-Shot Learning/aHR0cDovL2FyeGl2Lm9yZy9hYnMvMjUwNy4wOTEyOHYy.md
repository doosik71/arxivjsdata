# A Generalization Theory for Zero-Shot Prediction

Ronak Mehta, Zaid Harchaoui (2025)

## 🧩 Problem to Solve

본 논문은 최근 인공지능 분야의 핵심 패러다임인 제로샷 예측(Zero-Shot Prediction, ZSP)의 일반화 능력을 수학적으로 이해하기 위한 이론적 프레임워크를 제공하는 것을 목표로 한다. 구체적으로, CLIP과 같은 파운데이션 모델은 대규모 데이터셋에서 자기지도 학습(Self-Supervised Learning, SSL)을 통해 텍스트와 이미지의 공유 표현 공간을 학습한 뒤, 레이블이 없는 다운스트림 태스크에서 예측을 수행한다.

기존의 통계적 학습 이론은 레이블이 있는 데이터로 모델을 학습시키는 직접적인 방식에 집중되어 있어, 사전 학습(Pre-training), 프롬프팅(Prompting), 평가(Evaluation)라는 다단계 과정을 거치는 ZSP의 메커니즘을 충분히 설명하지 못한다. 따라서 본 연구는 다운스트림 태스크의 성능이 1) 사전 학습 분포, 2) 평가 분포, 3) 프롬프팅 전략이라는 세 가지 요소에 어떻게 의존하는지를 투명하게 분석하고자 한다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 제로샷 예측을 '간접 예측 경로(Indirect Prediction Path)'로 정의하고, 예측 오차를 세 가지 구성 요소로 분해하여 분석한 점이다.

1. **이론적 프레임워크 제안**: ZSP를 직접 예측자(Direct Predictor)와 간접 예측자(Indirect Predictor)의 비교 관점에서 정형화하였다.
2. **오차 분해(Error Decomposition)**: 제로샷 예측의 오차를 다음의 세 가지 항으로 분해하였다.
    * **프롬프트 편향(Prompt Bias)**: 사용자가 설계한 프롬프트 전략이 사전 학습 및 평가 분포와 얼마나 일치하지 않는지를 측정한다.
    * **잔차 의존성(Residual Dependence)**: 텍스트(Z)라는 매개체를 통해 예측할 때, 이미지(X)와 레이블(Y) 사이의 정보 손실, 즉 $Z$가 주어졌을 때 $X$와 $Y$ 사이에 여전히 남아있는 의존성을 측정한다.
    * **추정 오차(Estimation Error)**: 유한한 수의 사전 학습 샘플과 프롬프트 샘플로 인해 발생하는 통계적 오차를 의미한다.
3. **두 가지 학습 관점 제시**: ZSP를 구현하는 방식을 '조건부 평균(Conditional Mean)' 접근법과 '정보 밀도(Information Density)' 접근법으로 나누어 각각의 통계적 성질과 수렴 속도를 분석하였다.

## 📎 Related Works

본 논문은 제로샷 예측과 밀접한 관련이 있는 몇 가지 기존 연구를 언급하며 차별점을 제시한다.

* **Few-Shot Learning (FSL) 및 Meta-Learning**: FSL은 소량의 레이블 데이터를 사용하여 예측자를 학습시키지만, ZSP는 레이블 데이터가 전혀 없는 상태에서 텍스트-이미지 정렬(Alignment)에만 의존한다. 기존 FSL 이론은 사전 학습 데이터의 잠재 변수 모델에 의존하는 경향이 있으나, 본 논문은 다중 모달리티(Multimodal) 맥락에서 더 현실적인 가정(Conditional Independence를 강제하지 않음)을 사용한다.
* **CLIP 및 기반 모델**: CLIP의 성공은 자연어라는 매개체와 거대 데이터셋에 기인한다. 기존 연구(Chen et al., 2024)가 CLIP 기반 인코더의 정확도 경계를 제시했으나, 프롬프트 품질에 대한 명시적 의존성이나 데이터 분포 불일치 문제를 충분히 다루지 않았음을 지적한다.
* **프롬프트 엔지니어링 (CuPL 등)**: LLM을 이용하여 클래스별 맞춤형 프롬프트를 생성하는 경험적 방법들이 존재한다. 본 논문은 이러한 실무적 기법들이 왜 효과적인지를 통계적 학습 이론 관점에서 설명하고자 한다.

## 🛠️ Methodology

### 1. 시스템 구조 및 예측 경로

본 논문은 데이터를 이미지 $X$, 레이블 $Y$, 텍스트 캡션 $Z$의 세 가지 변수로 정의한다.

* **직접 예측자 ($\eta^\star$)**: 이미지 $X$에서 레이블 $Y$를 직접 예측하는 이상적인 함수이다.
    $$\eta^\star(x) := E_{P^{Y,X}}[r(Y)|X](x)$$
* **간접 예측자 ($\eta^\rho$)**: 텍스트 $Z$를 매개체로 하여 예측하는 ZSP의 경로이다.
    $$\eta^\rho(x) = E_{Q^{X,Z}}[g^\rho(Z)|X](x), \quad \text{where } g^\rho(z) = E_{\rho^{Y,Z}}[r(Y)|Z](z)$$
    여기서 $Q^{X,Z}$는 사전 학습 분포, $\rho^{Y,Z}$는 프롬프트 생성 전략을 의미한다.

### 2. 핵심 방정식 및 오차 분해

전체 예측 오차(Mean Squared Error)는 다음과 같이 분해된다:
$$\|\eta^\star - \hat{\eta}^\rho\|^2_{L^2(P^X)} \le 2\underbrace{\|\eta^\star - \eta^\rho\|^2_{L^2(P^X)}}_{\text{Information-theoretic error}} + 2\underbrace{\|\eta^\rho - \hat{\eta}^\rho\|^2_{L^2(P^X)}}_{\text{Estimation error}}$$

이 중 정보 이론적 오차는 다시 다음과 같이 상한선이 정의된다 (Theorem 1):
$$\|\eta^\rho - \eta^\star\|^2_{L^2(P^X)} \lesssim \underbrace{E_{P^Z}[I(X;Y|Z)]}_{\text{Residual Dependence}} + \underbrace{\|g^\rho - g_{P^{Y,Z}}\|^2_{L^2(P^Z)}}_{\text{Prompt Bias}}$$

* **Residual Dependence**: $I(X;Y|Z)$는 조건부 평균 제곱 우연성(Conditional Mean Square Contingency)으로, 텍스트 $Z$가 주어졌을 때 이미지 $X$가 레이블 $Y$에 대해 가지는 추가적인 정보량을 의미한다. 이 값이 0에 가까울수록 텍스트만으로도 충분한 예측이 가능하다.
* **Prompt Bias**: 실제 분포 $P^{Y,Z}$와 사용자의 프롬프트 전략 $\rho^{Y,Z}$ 사이의 차이를 의미한다.

### 3. 두 가지 학습 접근법

* **조건부 평균(Conditional Mean) 접근법**: 사전 학습을 통해 연산자 $M_{Z|X} : g \to E[g(Z)|X]$를 학습하고, 프롬프트를 통해 함수 $g^\rho$를 근사하는 방식이다.
* **정보 밀도(Information Density) 접근법**: 사전 학습을 통해 Radon-Nikodym 도함수인 정보 밀도 $R(x, z) = \frac{dQ^{X,Z}}{dQ^X Q^Z}$를 학습하고, 이를 이용해 기댓값을 계산하는 방식이다.

## 📊 Results

### 1. 잔차 의존성 시뮬레이션

가우시안 분포를 이용한 시뮬레이션 실험을 통해 잔차 의존성 $I(X;Y|Z)$가 제로샷 성능에 미치는 영향을 분석하였다.

* **결과**: $\theta$라는 파라미터를 통해 $I(X;Y|Z) \to 0$이 되도록 조절했을 때, 간접 예측자 $\eta^\rho$뿐만 아니라 실제 CLIP 및 VICReg 모델의 성능이 직접 예측자 $\eta^\star$의 성능에 수렴함을 확인하였다. 이는 텍스트 매개체가 이미지-레이블 간의 정보를 얼마나 잘 보존하느냐가 성능의 상한선을 결정함을 시사한다.

### 2. 프롬프트 편향 실험 (Unbiased Prompting)

ImageNet-Captions 데이터셋을 사용하여, 템플릿 기반 프롬프트와 실제 데이터에서 샘플링한 '편향 없는(Unbiased)' 프롬프트를 비교하였다.

* **결과**: 편향 없는 프롬프트를 사용했을 때, 샘플 수 $M$이 약 10개 이상이 되면 기본 템플릿보다 성능이 우수해졌으며, $M=100$ 수준에서는 성능이 크게 향상되었다(ResNet50의 경우 Top-1 정확도가 최대 15% 상승).

### 3. LLM 기반 프롬프팅 (CuPL) 분석

Llama 3를 이용해 클래스별 상세 묘사를 생성하는 CuPL 방식을 적용하여 다양한 데이터셋(DTD, Flowers 102, SUN397 등)에서 평가하였다.

* **결과**: 데이터셋의 특성에 따라 다르지만, DTD와 같은 텍스트 묘사가 중요한 데이터셋에서는 LLM 기반 프롬프트가 기본 템플릿 대비 Top-5 정확도를 약 20% 가량 향상시켰다.

## 🧠 Insights & Discussion

### 강점 및 이론적 함의

본 논문은 제로샷 예측의 성공을 단순히 '거대 모델의 힘'으로 치부하지 않고, 이를 **사전 학습 분포 $\to$ 프롬프트 분포 $\to$ 평가 분포**로 이어지는 확률적 경로의 일치 문제로 정형화하였다. 특히, 프롬프트의 개수 $M$과 사전 학습 샘플 수 $N$에 따른 수렴 속도를 수학적으로 제시함으로써, 향후 어떤 부분(데이터 증강 vs 프롬프트 최적화)에 집중해야 하는지에 대한 이론적 가이드를 제공한다.

### 한계 및 비판적 해석

* **가정의 단순성**: 이론적 분석을 위해 RKHS(Reproducing Kernel Hilbert Space) 설정을 사용하였으나, 실제 딥러닝 모델의 비선형성과 복잡한 최적화 과정이 완벽하게 반영되었는지는 알 수 없다.
* **분포 시프트 문제**: 사전 학습 분포 $Q^X$와 평가 분포 $P^X$ 사이의 불일치(Distribution Shift)를 분석에 포함했으나, 실무적으로 이를 어떻게 효과적으로 줄일 수 있을지에 대한 구체적인 방법론보다는 현상 분석에 치중되어 있다.

## 📌 TL;DR

본 연구는 제로샷 예측(ZSP)의 일반화 능력을 분석하기 위해 **'간접 예측 경로'**라는 이론적 프레임워크를 제안하였다. ZSP의 오차를 **잔차 의존성(매개체 정보 손실), 프롬프트 편향(전략 불일치), 추정 오차(샘플 부족)**로 분해하였으며, 시뮬레이션과 실험을 통해 텍스트 매개체의 정보량과 프롬프트의 품질이 성능을 결정짓는 핵심 요소임을 입증하였다. 이 연구는 파운데이션 모델의 프롬프팅 전략을 최적화하고 성능 상한선을 예측하는 데 중요한 이론적 토대를 제공한다.
