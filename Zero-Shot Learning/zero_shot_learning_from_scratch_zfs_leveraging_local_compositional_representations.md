# Zero-Shot Learning from scratch (ZFS): leveraging local compositional representations

Tristan Sylvain, Linda Petrini, R Devon Hjelm (2019)

## 🧩 Problem to Solve

본 논문은 Zero-Shot Learning (ZSL) 분야에서 기존 연구들이 가진 근본적인 방법론적 한계를 지적한다. 일반적인 ZSL 설정에서는 학습 단계에서 본 적 없는 클래스를 분류하기 위해 클래스별 semantic 정보(속성 또는 텍스트 설명)를 활용한다. 그러나 최근의 고성능 ZSL 모델들은 대부분 ImageNet으로 사전 학습된(pretrained) 인코더를 백본(backbone)으로 사용한다.

저자들은 이러한 방식이 다음과 같은 문제를 야기한다고 주장한다. 첫째, 전이 학습(transfer learning)의 성능이 사전 학습된 백본의 종류에 과도하게 의존하게 된다. 둘째, 모든 ZSL 태스크에 대해 ImageNet과 같은 거대 데이터셋이 존재하거나 적합하다는 보장이 없다. 셋째, 일부 벤치마크의 테스트 클래스가 이미 ImageNet 학습 데이터에 포함되어 있어, 진정한 의미의 Zero-Shot 능력을 평가하기 어렵게 만드는 데이터 누수(data leakage) 문제가 발생한다.

따라서 본 논문의 목표는 사전 학습된 외부 인코더의 사용을 엄격히 금지하고, 오직 대상 데이터셋의 학습 분할(training split)만을 사용하여 모델을 학습시키는 **Zero-Shot Learning from scratch (ZFS)**라는 더 도전적인 설정을 제안하고, 이를 통해 진정한 일반화 성능을 탐구하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 ZSL의 평가 패러다임을 '절대적 성능 달성'에서 '새로운 개념에 대한 추론 능력 평가'로 전환하려 했다는 점이다. 구체적인 기여 사항은 다음과 같다.

1. **ZFS(Zero-Shot Learning from scratch) 설정 제안**: 타겟 데이터셋 외부의 정보를 사용하는 것을 금지하는 엄격한 ZSL 벤치마크 설정을 정의하였다.
2. **지역적 구성성(Local Compositionality)의 중요성 제시**: 이미지의 전역적 표현뿐만 아니라, 지역적인 부분(part)들의 표현을 학습하고 이를 조합하는 능력이 ZFS 설정에서 일반화 성능을 높이는 핵심 아이디어임을 제안하였다.
3. **지역적 목적 함수(Local Objective) 도입**: 인코더의 초기 레이어에서 지역적 특징(local features)들이 의미론적으로 유용한 정보를 갖도록 유도하는 보조 손실 함수를 도입하여 성능 향상을 입증하였다.

## 📎 Related Works

본 논문은 ZSL을 메타 학습(meta-learning) 관점에서 접근하거나, 데이터 샘플과 클래스 속성을 공통 부분 공간(common subspace)으로 매핑하는 Metric Learning 방식으로 처리하는 기존 연구들을 언급한다.

특히 **구성성(Compositionality)**과 관련하여, 지능적 에이전트가 새로운 개념으로 일반화하기 위해서는 부분들의 조합을 통해 전체를 인식하는 능력이 필요하다는 인지 과학적 관점을 인용한다. 또한, 지역적 특징(local features)을 활용하는 Self-attention이나 Deep InfoMax (DIM)와 같은 연구들이 표현 학습의 성능을 높였음을 언급하며, DIM이 ZFS 설정에서도 유망한 기반 모델이 될 수 있다는 가설을 세운다. 기존 ZSL 연구들이 ImageNet-pretrained 특성에 의존했다면, 본 연구는 모델이 스스로 지역적 특징을 구성하여 일반화하는 능력을 학습시키는 데 집중한다는 점에서 차별점을 갖는다.

## 🛠️ Methodology

### 전체 시스템 구조

본 연구의 파이프라인은 크게 두 단계로 나뉜다. 먼저 이미지 인코더 $E_\phi$를 지도 또는 비지도 학습 방식으로 학습시키고, 이후 고정된(fixed) 인코더의 표현 위에 Prototypical Network $P_\theta$를 올려 Zero-shot 분류를 수행한다.

### 지역적 목적 함수 (Local Objective)

저자들은 인코더가 이미지의 특정 패치(patch)에 해당하는 지역적 특징에서 의미 있는 정보를 추출하도록 강제하기 위해, 인코더의 초기 레이어(예: AlexNet의 세 번째 레이어)에 보조 분류기를 추가한다. 이를 통해 전역적 표현뿐만 아니라 지역적 표현에서도 클래스나 속성 정보가 보존되도록 유도한다.

보조 분류기에는 두 가지 방식이 제안된다:

1. **Local Label-based Classifier (LC)**: 표준 Cross-entropy 손실을 사용하여 지역적 특징 $l_i$를 통해 학습 클래스 $y_i$를 예측한다.
2. **Local Attribute-based Classifier (AC)**: 별도의 Prototypical Network를 사용하여 지역적 특징을 속성 임베딩 공간으로 투영하고, 속성 임베딩과의 일치 여부를 학습한다.

### 학습 절차 및 방정식

전체 학습 목적 함수 $J$는 모델의 기본 손실 함수(예: VAE의 재구성 손실, DIM의 상호 정보량 최대화 손실 등)인 $L_{global}$과 앞서 언급한 지역적 손실 $L_{local}$의 합으로 정의된다.

$$J = L_{local} + L_{global}$$

학습 과정은 다음과 같은 흐름으로 진행된다:

1. 이미지 $x_i$를 인코더 $E$에 통과시켜 지역적 특징 $l_i$와 전역적 특징 $g_i$를 추출한다.
2. $L_{local}$을 계산하여 인코더 $E$와 지역 모델 $L$의 파라미터 $\phi$를 업데이트한다.
3. 전역 특징 $g_i$를 detach하여(기울기 전파 차단), Prototypical Network $P_\theta$가 속성 $a_i$와 레이블 $y_i$ 사이의 관계를 학습하도록 파라미터 $\theta$를 업데이트한다.

## 📊 Results

### 실험 설정

- **데이터셋**: AwA2, CUB, SUN 세 가지 벤치마크를 사용하였으며, 데이터 누수를 방지하기 위해 Proposed Split (PS)을 적용하였다.
- **비교 대상**: Fully supervised (FC), VAE, $\beta$-VAE, AAE, DIM 등 다양한 표현 학습 모델을 비교하였다.
- **인코더**: AlexNet과 기본 CNN(Basic CNN)을 사용하여 모델 용량에 따른 영향을 분석하였다.
- **지표**: Top-1 Accuracy를 측정하였다.

### 정량적 결과

실험 결과, 대부분의 모델에서 지역적 정보(LC 또는 AC)를 추가했을 때 정확도가 상승하는 경향을 보였다.

- **최고 성능 모델**: DIM (Deep InfoMax)이 가장 높은 성능을 기록하였으며, 특히 CUB 데이터셋에서는 지도 학습 기반의 FC 모델과 유사한 성능을, AwA2와 SUN에서는 FC를 상회하는 성능을 보였다.
- **보조 분류기의 효과**: 전역 정보에만 의존하는 VAE, $\beta$-VAE, AAE의 경우 LC가 AC보다 약간 더 나은 성능을 보였으나 차이가 크지 않았다. 반면, DIM은 AC(속성 기반 지역 분류기)를 사용했을 때 매우 유의미한 성능 향상이 나타났다.

이 결과는 지역적 특징을 이미 잘 활용하는 모델(DIM)일수록, 그 지역적 특징에 의미론적 제약(semantic constraint)을 주는 것이 일반화 성능 향상에 훨씬 더 효과적임을 시사한다.

## 🧠 Insights & Discussion

본 논문은 ZFS라는 극한의 설정을 통해, ZSL의 성공 요인이 단순한 데이터의 양이나 사전 학습된 가중치가 아니라 **'지역적 구성성(local compositionality)'**에 있음을 실험적으로 입증하였다.

특히 DIM 모델이 탁월한 성적을 거둔 점은 주목할 만하다. DIM은 본래 지역적 특징과 전역적 특징 사이의 상호 정보량을 최대화하도록 설계된 모델이다. 이러한 구조적 특성이 ZFS에서 요구되는 "부분을 통한 전체의 이해"라는 메커니즘과 일치하며, 여기에 속성 기반의 보조 손실을 더했을 때 시너지가 발생한다는 점은 매우 강력한 통찰을 제공한다.

다만, 본 연구에서는 비교적 작은 규모의 인코더를 사용하였으므로, 매우 거대한 아키텍처에서도 동일한 지역적 제약이 동일한 수준의 효율성을 가질지는 추가적인 연구가 필요하다. 또한, 재구성 기반 모델(VAE 등)에서는 지역적 손실의 효과가 상대적으로 낮았는데, 이는 전역적 재구성이라는 목표 자체가 이미지 전체의 구조를 파악해야 하므로 지역적 제약과 충돌하거나 상쇄될 가능성이 있음을 시사한다.

## 📌 TL;DR

본 논문은 외부 사전 학습 데이터 없이 타겟 데이터셋만으로 학습하는 **Zero-Shot Learning from scratch (ZFS)**라는 새로운 벤치마크 설정을 제안한다. 저자들은 이미지의 지역적 특징을 의미론적으로 학습시키는 **Local Objective**가 새로운 클래스로의 일반화 능력을 크게 향상시킨다는 것을 증명하였다. 특히 지역적 표현 학습에 강점이 있는 **DIM** 모델에 속성 기반 보조 손실을 결합했을 때 최적의 성능을 보였으며, 이는 ZSL 연구가 나아가야 할 방향이 단순한 전이 학습이 아닌 '구성적 표현 학습'에 있음을 시사한다.
