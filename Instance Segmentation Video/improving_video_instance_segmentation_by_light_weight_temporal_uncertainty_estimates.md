# Improving Video Instance Segmentation by Light-weight Temporal Uncertainty Estimates

Kira Maag, Fabian Hüger, Matthias Rottmann, Peter Schlicht, Serin Varghese, Hanno Gottschalk (2021)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 인스턴스 세그멘테이션(Instance Segmentation) 네트워크에서 발생하는 **신뢰도 점수(Confidence Score)의 불일치 문제**이다. 일반적으로 YOLACT나 Mask R-CNN과 같은 신경망은 예측 결과에 대해 점수를 제공하지만, 실제로는 잘못된 예측(False Positive)에 높은 점수를 부여하고 정답(True Positive)에 낮은 점수를 부여하는 경우가 빈번하다.

이러한 문제는 특히 자율 주행과 같이 안전이 최우선인 분야에서 심각한 안전 문제를 야기할 수 있으며, 모델의 해석 가능성을 떨어뜨린다. 기존의 신뢰도 보정(Confidence Calibration) 방법들이 존재하지만, 본 논문은 단순한 점수 임계값(Score Threshold) 적용만으로는 False Positive(FP)를 효과적으로 제거하면서 False Negative(FN)를 유지하기 어렵다는 점에 주목한다. 따라서 본 연구의 목표는 비디오 시퀀스의 시간적 정보를 활용하여 인스턴스별 예측 품질을 정확하게 추정하고, 이를 통해 FP를 효과적으로 탐지하여 전체적인 세그멘테이션 성능을 향상시키는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **시간적 불확실성 추정(Temporal Uncertainty Estimation)**을 통해 인스턴스 수준의 예측 품질을 평가하는 사후 처리(Post-processing) 프레임워크를 제안하는 것이다. 주요 기여 사항은 다음과 같다.

1. **메타 태스크(Meta Tasks)의 도입**: 인스턴스 세그멘테이션 분야에 **메타 분류(Meta Classification)**(IoU $\ge 0.5$ 여부 판단)와 **메타 회귀(Meta Regression)**(IoU 값 직접 예측) 개념을 최초로 도입하였다.
2. **경량 트래킹 알고리즘**: 신경망 모델과 독립적으로 작동하며, 선형 회귀(Linear Regression) 기반의 위치 예측과 겹침(Overlap) 정도를 이용해 인스턴스를 추적하는 경량 알고리즘을 제안하였다.
3. **시간-역동적 지표(Time-dynamic Metrics) 설계**: 단일 프레임 지표를 넘어 형태 보존도, 중심 및 크기 편차, 생존 시간 분석, 높이-너비 비율 등 시간적 흐름을 반영한 새로운 불확실성 지표들을 설계하였다.
4. **성능 향상 입증**: 제안한 메타 분류기를 사용하여 기존의 단순 점수 임계값 방식보다 FP를 최대 44.03%까지 줄이면서도 mAP(mean Average Precision)를 향상시킬 수 있음을 보였다.

## 📎 Related Works

### 관련 연구 및 한계

- **불확실성 정량화**: Bayesian 모델이나 MC Dropout 기반의 방법들이 존재한다. 하지만 MC Dropout은 네트워크를 여러 번 실행해야 하므로 연산 비용이 높고, 인스턴스 세그멘테이션에 특화된 연구는 부족한 상태였다.
- **객체 추적(Multi-Object Tracking)**: 많은 추적 알고리즘이 Bounding Box 기반으로 작동하며, 딥러닝 모델에 통합된 형태가 많다. 이는 특정 네트워크에 종속적이라는 한계가 있다.
- **세그멘테이션 품질 추정**: 세만틱 세그멘테이션(Semantic Segmentation)에서는 메타 분류/회귀 연구가 진행되었으나, 개별 객체를 구분해야 하는 인스턴스 세그멘테이션으로의 확장은 이루어지지 않았다.

### 차별점

본 제안 방법은 특정 신경망 구조에 종속되지 않는 **사후 처리 단계(Post-processing step)**로 설계되어, 이미 학습된 어떤 인스턴스 세그멘테이션 네트워크에도 적용 가능하다. 또한, 단순한 단일 프레임 통계가 아니라 시간적 일관성(Temporal Consistency)을 정량화한 지표를 사용하여 예측 품질을 추정한다는 점에서 기존 방식과 차별화된다.

## 🛠️ Methodology

### 1. 인스턴스 트래킹 알고리즘

신경망이 예측한 인스턴스들을 프레임 간에 연결하기 위해 경량 트래킹을 수행한다.

- **매칭 기준**: 인스턴스 간의 겹침(Overlap) 정도나 기하학적 중심(Geometric Center)의 거리로 매칭한다.
- **위치 예측**: 이전 프레임들의 데이터를 바탕으로 선형 회귀를 수행하여 다음 프레임의 예상 위치 $\hat{\bar{i}}_t$를 계산하고, 이를 기반으로 매칭 효율을 높인다.
- **겹침 정의**: 두 인스턴스 $i$와 $j$의 겹침 $O_{i,j}$는 다음과 같이 정의된다.
$$O_{i,j} = \frac{|i \cap j|}{|i \cup j|}$$
- **기하학적 중심**: 마스크 내 모든 픽셀 $(z_v, z_h)$의 평균 좌표로 계산한다.
$$\bar{i}_t = \frac{1}{|i|} \sum_{(z_v, z_h) \in i} (z_v, z_h)$$

### 2. 시간적 인스턴스 지표 (Temporal Metrics)

모델의 입력으로 사용될 지표 집합 $V_i$를 구성한다.

- **단일 프레임 지표 ($U_i$)**: 인스턴스 크기, 중심, 소프트맥스 출력의 엔트로피(Entropy) 평균 등을 포함한다.
- **형태 보존도 ($f$)**: 인스턴스의 중심을 일치시킨 후 이전 프레임과 현재 프레임 간의 겹침 정도를 측정하여 급격한 형태 변화를 감지한다.
- **중심 및 크기 편차 ($d_c, d_s$)**: 선형 회귀로 예측된 예상 중심/크기와 실제 측정값 사이의 차이를 계산한다.
$$\text{Center Deviation: } d_c = \|\hat{\bar{i}}_t - \bar{i}_t\|^2$$
- **생존 시간 ($v$)**: Cox 회귀 모델을 사용하여 인스턴스가 얼마나 오랫동안 정답과 매칭되어 유지되는지를 예측한다.
- **높이-너비 비율 ($r$)**: 클래스별 정답(GT)의 평균 비율 $r_{GT}^c$와 예측된 인스턴스의 비율을 비교한다.
$$r = (h_i / w_i) / r_{GT}^c$$

### 3. 메타 태스크 학습 및 추론

수집된 지표들의 시계열 데이터 $V_{i,k} (k=t-n_c, \dots, t)$를 입력으로 하여 다음 두 가지 태스크를 수행한다.

- **메타 분류 (Meta Classification)**: $\text{IoU} < 0.5$ (False Positive) 인지 $\text{IoU} \ge 0.5$ (True Positive) 인지를 분류한다.
- **메타 회귀 (Meta Regression)**: 실제 $\text{IoU}$ 값을 직접 예측한다.
- **사용 모델**: Gradient Boosting (GB), 단일 은닉층 신경망 (NN L2), $\ell_1$-정규화 선형 회귀 (LASSO)를 비교 분석하였다.

## 📊 Results

### 실험 설정

- **데이터셋**: KITTI (도로 장면), MOT (보행자 장면)
- **대상 네트워크**: YOLACT (실시간성 강조), Mask R-CNN (정밀도 강조)
- **평가 지표**: 메타 분류(ACC, AUROC), 메타 회귀 ($\sigma, R^2$), 전체 세그멘테이션 성능 (mAP)

### 주요 결과

- **메타 태스크 성능**: Gradient Boosting 모델이 가장 우수한 성능을 보였으며, 메타 분류에서 최대 **AUROC 98.78%**, 메타 회귀에서 최대 **$R^2$ 86.85%**를 달성하였다.
- **시간적 정보의 영향**: 단일 프레임 지표만 사용할 때보다 시계열 데이터를 활용했을 때 성능이 유의미하게 향상되었으며, 특히 10프레임 정도의 윈도우를 사용했을 때 높은 성능을 보였다.
- **FP 제거 효과 (Advanced Score)**:
  - **YOLACT**: 기존 score threshold 방식 대비 FP를 최대 **44.03% 감소**시켰으며, mAP를 $57.04\% \rightarrow 58.22\%$로 향상시켰다.
  - **Mask R-CNN**: FP를 최대 **43.33% 감소**시키는 성과를 거두었다.

## 🧠 Insights & Discussion

### 강점

본 연구는 복잡한 신경망 재학습 없이 **사후 처리만으로** 예측 신뢰도를 획기적으로 높였다는 점이 큰 강점이다. 특히 시간적 일관성이라는 직관을 정량적 지표(편차, 생존 시간 등)로 변환하여 메타 모델에 학습시킨 점이 유효했다. 또한, 모델 독립적인 트래킹 알고리즘을 제안함으로써 다양한 아키텍처에 범용적으로 적용 가능하다는 실용성을 확보하였다.

### 한계 및 비판적 해석

- **데이터 의존성**: MOT 데이터셋에서는 데이터의 중복성으로 인해 오버피팅 문제가 발생하여 KITTI 데이터셋 위주로 실험이 진행되었다. 이는 제안 방법이 데이터의 특성(프레임 레이트, 객체 이동 속도 등)에 영향을 받을 수 있음을 시사한다.
- **트래킹 성능**: 제안한 경량 트래킹 알고리즘이 TrackR-CNN 같은 딥러닝 기반 통합 모델보다는 성능(MOTA 등)이 약간 낮다. 하지만 본 논문의 목적이 '정밀한 추적' 자체가 아니라 '불확실성 추정을 위한 최소한의 연결'이라는 점을 고려하면 납득 가능한 수준이다.
- **GT 비율 사용**: 높이-너비 비율 $r$을 계산할 때 클래스별 GT 평균값을 사용하는데, 이는 학습 데이터셋의 분포가 테스트 데이터셋과 유사하다는 가정이 필요하다.

## 📌 TL;DR

이 논문은 비디오 인스턴스 세그멘테이션에서 신뢰도 점수가 부정확하여 발생하는 False Positive 문제를 해결하기 위해, **시간적 불확실성 지표를 활용한 메타 분류/회귀 프레임워크**를 제안한다. 경량 트래킹을 통해 구축된 인스턴스의 시간적 특성(형태 변화, 위치 편차 등)을 Gradient Boosting 모델로 분석하여 예측 품질(IoU)을 정확히 추정하며, 이를 통해 **FP를 약 44% 감소시키고 전체적인 검출 정밀도(mAP)를 향상**시켰다. 이 방법은 기존 모델을 수정하지 않는 사후 처리 방식이므로, 자율 주행 등 안전이 중요한 실시간 시스템의 신뢰도를 높이는 데 즉시 적용 가능하다.
