# BoxVIS: Video Instance Segmentation with Box Annotations

Minghan Li, Lei Zhang (2023)

## 🧩 Problem to Solve

본 논문은 비디오 인스턴스 분할(Video Instance Segmentation, VIS) 모델 학습에 필요한 픽셀 단위 마스크(pixel-wise mask) 어노테이션의 과도한 비용 문제를 해결하고자 한다. VIS는 비디오 전체에서 인스턴스의 마스크와 카테고리를 예측하는 작업으로, 기존의 최신 모델들은 픽셀 단위의 정밀한 라벨을 필요로 한다. 그러나 비디오 데이터에서 모든 프레임의 픽셀을 라벨링하는 것은 노동 집약적이며 비용이 매우 많이 든다.

이러한 비용 문제로 인해 기존의 VIS 데이터셋(YouTube-VIS, OVIS 등)은 어노테이션의 양이 적으며, 이는 결과적으로 학습된 모델의 일반화 성능을 제한하는 요소가 된다. 따라서 본 연구의 목표는 픽셀 단위 마스크 대신 훨씬 저렴한 비용으로 획득 가능한 바운딩 박스(bounding box) 어노테이션만을 사용하여, 픽셀 지도 기반의 감독 학습 모델에 근접하는 성능의 VIS 모델을 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 바운딩 박스 기반의 약지도 학습(Weakly-supervised learning)을 VIS에 적용하여 어노테이션 비용을 획기적으로 줄이면서도 높은 성능을 유지하는 것이다. 이를 위해 다음과 같은 두 가지 핵심 기여를 제시한다.

1. **STPA Loss (Spatial-Temporal Pairwise Affinity Loss) 제안**: 단일 프레임 내의 공간적 일관성뿐만 아니라 프레임 간의 시간적 일관성을 강제하기 위해, 바운딩 박스의 중심점 이동을 가이드로 사용하는 시공간 쌍방 어피니티 손실 함수를 설계하였다.
2. **BVISD (Box-annotated VIS Dataset) 구축**: 데이터 부족 문제를 해결하기 위해 기존 VIS 벤치마크의 비디오와 COCO 데이터셋의 이미지를 가공한 유사 비디오 클립(pseudo video clips)을 통합하여 대규모 바운딩 박스 기반 VIS 데이터셋을 구축하였다.

## 📎 Related Works

### 기존 VIS 접근 방식

기존의 픽셀 지도 기반 VIS 모델들은 크게 두 가지 방향으로 나뉜다.

- **YTVIS 지향 모델**: 주로 짧은 비디오의 희소한 객체를 분할하는 데 집중하며, 프레임 간 추적기(tracker)나 쿼리 기반(query-based) 트랜스포머 구조를 사용한다.
- **OVIS 지향 모델**: 긴 비디오 내의 밀집된 객체와 가려짐(occlusion) 문제를 해결하는 데 집중하며, 대조 학습(contrastive learning)이나 Mask2Former와 같은 마스크 기반 트랜스포머를 확장하여 사용한다.

### 기존 연구의 한계 및 차별점

기존의 모든 VIS 방법론은 픽셀 단위의 정밀한 어노테이션을 전제로 하기에 일반화 능력이 제한적이며 데이터 구축 비용이 매우 높다. 이미지 인스턴스 분할(IIS) 분야에서는 이미 바운딩 박스만을 이용한 약지도 학습 연구(BoxInst, BoxTeacher 등)가 성과를 거두었으나, 비디오 영역에서는 객체의 움직임과 외형 변화로 인해 이를 적용하기가 더 어렵다. 본 논문은 이러한 IIS의 박스 지도 학습 아이디어를 VIS로 확장하고, 비디오 특유의 시간적 연속성을 고려한 STPA loss를 도입함으로써 차별성을 갖는다.

## 🛠️ Methodology

### 1. BoxVIS Baseline (M2F-VIS-box)

본 연구는 최신 픽셀 지도 기반 모델인 Mask2Former-VIS (M2F-VIS)를 기반으로 하여, 이를 바운딩 박스 지도 기반으로 변경한 `M2F-VIS-box`를 베이스라인으로 설정한다.

- **손실 함수**:
  - **Projection Loss ($L_{proj}$)**: 예측된 마스크의 수평 및 수직 투영(projection)이 정답 바운딩 박스와 일치하도록 Dice loss를 통해 감독한다.
  - **Pairwise Affinity Loss ($L_{pair}$)**: 두 픽셀의 색상 유사도가 임계값보다 높으면 동일한 마스크 라벨을 가져야 한다는 가정을 바탕으로 학습한다.
- **Teacher-Student 구조**: EMA(Exponential Moving Average) 기반의 Teacher Net이 고품질의 의사 마스크(pseudo mask)를 생성하고, Student Net은 이를 통해 학습한다. 헝가리안 매칭(Hungarian matching) 알고리즘을 통해 예측 마스크와 정답 박스를 연결한다.
- **동적 임계값**: 의사 마스크의 품질을 보장하기 위해 학습 진행률($r$)에 따라 변하는 동적 임계값 $\epsilon = 1/(1 + e^{-2r})$을 사용하여 고품질의 마스크만 선택적으로 학습에 사용한다.

### 2. Spatial-Temporal Pairwise Affinity (STPA) Loss

단일 프레임의 색상 유사성만으로는 비디오의 시간적 일관성을 유지하기 어려우므로, 시공간적 쌍방 어피니티를 계산하는 STPA loss를 제안한다.

- **시공간 쌍(Spatio-temporally paired pixels) 생성**:
  - **공간적 이웃 ($E_s$)**: 현재 프레임의 픽셀에서 오른쪽과 아래쪽 이웃 픽셀을 선택한다.
  - **시간적 이웃 ($E_t$)**: 인접한 두 프레임 $t_i$와 $t_j$ 사이의 바운딩 박스 중심점 좌표 차이 $(dx_c, dy_c)$를 계산하고, 이 오프셋만큼 픽셀 위치를 이동시켜 대응되는 픽셀 쌍을 찾는다.
- **어피니티 계산**: 색상 유사도($S_{lab}$)뿐만 아니라 패치 특징 상관관계($S_{corr}$)를 결합하여 더 안정적인 지표를 생성한다.
$$S_e = S_{lab} + 0.5 S_{corr}$$
여기서 $S_{corr}$은 Mask2Former의 마지막 인코더 레이어에서 추출한 특징 벡터들의 코사인 유사도 평균으로 계산된다.
- **최종 손실 함수**:
$$L_{stpa} = -\frac{1}{N} \sum_{e \in E_{stin}} \mathbb{1}\{S_e \ge \tau\} \log P(y_e = 1)$$
이 식은 어피니티 $S_e$가 임계값 $\tau$보다 큰 픽셀 쌍에 대해 동일한 마스크 라벨을 갖도록 강제한다.

### 3. Box-annotated VIS Dataset (BVISD)

데이터 부족 문제를 해결하기 위해 다음과 같이 BVISD를 구축하였다.

- **구성**: YouTube-VIS 2021, OVIS 데이터셋의 비디오를 통합하고, COCO 데이터셋의 정적 이미지를 가공하여 추가하였다.
- **Pseudo Video Clip 생성**: COCO 이미지를 무작위로 리사이징, 크롭, 회전시켜 $T$개의 프레임을 가진 짧은 가상 비디오 클립으로 변환하였다.
- **카테고리 통합**: 서로 다른 데이터셋 간의 시맨틱 충돌을 방지하기 위해 유사 카테고리를 슈퍼 카테고리(예: car, truck $\rightarrow$ vehicle)로 통합하여 총 40개의 카테고리를 정의하였다.

## 📊 Results

### 실험 설정

- **데이터셋 및 지표**: YouTube-VIS 2021 (YTVIS21)과 OVIS 데이터셋을 사용하였으며, 평가지표로는 Mask AP(Average Precision)를 사용하였다.
- **백본**: ResNet50 및 Swin Large를 사용하였다.

### 주요 결과

- **정량적 성과 (ResNet50 기준)**:
  - **YTVIS21**: BoxVIS(BVISD 학습)는 **43.2% Mask AP**를 달성하여, 픽셀 지도 기반의 M2F-VIS(43.1%)와 대등한 성능을 보였다.
  - **OVIS**: BoxVIS(BVISD 학습)는 **29.0% Mask AP**를 달성하여, 베이스라인인 M2F-VIS-box(23.9%)보다 크게 향상되었으며 SOTA 모델인 MDQE(29.2%)에 근접하는 성능을 보였다.
- **효율성**: BVISD의 어노테이션 비용은 픽셀 단위 마스크 라벨링 비용의 약 **16%** 수준에 불과하다.
- **Swin Large 백본 결과**: MDQE의 클립 레벨 추적기(clip-level tracker)를 적용했을 때 YTVIS21에서 53.9%, OVIS에서 40.6% AP를 달성하여 픽셀 지도 기반 모델들과 경쟁 가능한 수준임을 입증하였다.

## 🧠 Insights & Discussion

### 강점 및 의의

본 연구는 픽셀 단위의 정밀한 라벨 없이 바운딩 박스만으로도 고성능의 VIS 모델을 구축할 수 있음을 증명하였다. 특히 STPA loss를 통해 비디오의 시간적 연속성을 효과적으로 학습시켰으며, COCO 이미지를 가공한 유사 비디오 클립이 모델의 일반화 성능을 높이는 데 기여했음을 확인하였다. 어노테이션 비용을 84%나 절감하면서도 유사한 성능을 낸다는 점은 실용적 가치가 매우 높다.

### 한계 및 분석

OVIS 데이터셋과 같이 객체가 매우 밀집되어 있고 가려짐이 심한 경우, 픽셀 지도 기반 모델과의 성능 격차가 일부 존재한다. 이는 바운딩 박스 정보만으로는 복잡한 경계선이나 정밀한 객체 임베딩을 학습하는 데 한계가 있기 때문으로 분석된다. 또한, 추론 속도 측면에서 클립 단위 추론 방식을 사용하므로 프레임 단위 추론보다 느린 37.8 FPS를 기록하였다.

## 📌 TL;DR

본 논문은 매우 비싼 픽셀 단위 마스크 라벨링 대신 저렴한 **바운딩 박스 어노테이션**만을 사용하여 비디오 인스턴스 분할(VIS)을 수행하는 **BoxVIS** 프레임워크를 제안한다. 시공간적 일관성을 강화하는 **STPA loss**와 대규모 박스 기반 데이터셋인 **BVISD**를 도입하여, 픽셀 지도 기반 SOTA 모델 대비 **약 16%의 어노테이션 비용만으로 대등한 수준의 성능**을 달성하였다. 이는 향후 VIS 분야에서 데이터 구축 비용을 획기적으로 줄이면서 모델의 일반화 성능을 높이는 중요한 방향성을 제시한다.
