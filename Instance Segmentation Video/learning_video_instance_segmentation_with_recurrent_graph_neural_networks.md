# Learning Video Instance Segmentation with Recurrent Graph Neural Networks

Joakim Johnander, Emil Brissman, Martin Danelljan, Michael Felsberg (2020)

## 🧩 Problem to Solve

본 논문은 비디오 인스턴스 세그멘테이션(Video Instance Segmentation, VIS) 문제를 해결하고자 한다. VIS는 사전 정의된 클래스에 대해 객체 인스턴스를 동시에 검출(detecting), 세그멘테이션(segmenting), 그리고 추적(tracking)하는 작업이다.

기존의 VIS 접근 방식들은 대부분 '검출 후 추적(tracking-by-detection)' 패러다임을 따르며, 여러 모듈을 휴리스틱(heuristic)하게 결합하여 최종 결과를 도출한다. 이러한 방식은 다음과 같은 두 가지 주요 문제점을 가진다.

1. **유연성 부족**: 학습된 모델이 모든 검출 결과에 대해 전역적으로 추론하거나 시간적 정보(temporal information)에 접근하는 능력이 부족하다.
2. **학습과 추론의 괴리**: 모델 학습 단계가 실제 추론 과정을 밀접하게 반영하지 않는다. 예를 들어, 단지 두 프레임 간의 쌍(pair)만을 이용해 학습하거나 이후의 검출 병합 단계를 무시하는 경우가 많다. 이로 인해 모델은 가벼운 검출기 사용으로 인한 오류나 복잡한 비디오 상황에서 발생하는 다양한 문제들을 학습할 기회를 얻지 못한다.

따라서 본 논문의 목표는 비디오 인스턴스 세그멘테이션 전체 과정을 통합적으로 모델링하여, 추론 단계를 밀접하게 모사하는 순수 학습 기반의 새로운 정식화(learning formulation)를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Graph Neural Network(GNN)**와 **Recurrent Neural Network(RNN)**를 결합하여, 각 프레임에서 발생하는 모든 새로운 정보를 처리함과 동시에 과거의 정보를 재귀적으로 유지하는 유연한 모델을 구축하는 것이다.

주요 기여 사항은 다음과 같다.

- **엔드투엔드(End-to-End) 학습 정식화**: 비디오 인스턴스 세그멘테이션을 위해 추론 과정을 직접 모델링하는 새로운 학습 방식을 제안하여 엔드투엔드 학습이 가능하게 하였다.
- **GNN 및 RNN 기반 모델**: GNN을 통해 현재 프레임의 검출물과 기존 트랙 간의 관계를 전역적으로 추론하고, RNN 연결을 통해 시간적 일관성을 유지하는 아키텍처를 설계하였다.
- **학습 가능한 외형 모델링**: 객체의 외형(appearance)을 가우시안 분포로 모델링하고, 이를 베이지안 업데이트(Bayesian update) 방식으로 갱신하는 학습 가능한 메커니즘을 도입하였다.
- **실시간 성능 달성**: 제안 방법은 30 FPS 이상의 속도로 동작하면서도 기존의 실시간 VIS 방법론들보다 뛰어난 성능(YouTube-VIS 데이터셋에서 mAP 9.0% 상대적 향상)을 보였다.

## 📎 Related Works

기존의 VIS 연구들은 주로 이미지 인스턴스 세그멘테이션 모델을 통해 각 프레임에서 검출물을 얻은 후, 이를 트랙(track)으로 형성하는 방식을 취했다. 대표적으로 마스크 전파(mask propagation), 칼만 필터(Kalman filter) 기반의 바운딩 박스 추적, 또는 인스턴스별 외형 기술자(appearance descriptors)의 유사도 학습 등이 사용되었다.

이러한 기존 방식들의 한계는 다음과 같다.

- **휴리스틱 의존성**: 검출물을 트랙에 할당하는 과정이 주로 휴리스틱에 의존하며, 전체 파이프라인이 엔드투엔드로 학습되지 않는 경우가 많다.
- **트랙 스코어링의 단순함**: 트랙의 클래스와 신뢰도(confidence)를 결정할 때 단순히 다수결(majority vote)이나 평균값을 사용하는 등 단순한 방식에 의존하여, 검출의 일관성이나 불확실성을 충분히 반영하지 못한다.

본 논문은 MOT(Multiple Object Tracking)나 특징 매칭(feature matching)에서 사용되는 GNN 기반의 이분 그래프(bipartite graph) 추론 방식을 VIS에 도입하였다. 기존 GNN 기반 MOT 연구들이 주로 엣지(edge)나 노드(node) 중 하나에만 집중한 반면, 본 연구는 엣지를 통해 할당(association)을 예측하고 노드를 통해 클래스 및 신뢰도를 예측함으로써 두 정보를 모두 활용한다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 파이프라인

본 모델은 프레임 단위로 순차적으로 동작하며, 각 프레임에서 (1) 신규 트랙 생성, (2) 기존 트랙과 검출물의 할당, (3) 기존 트랙의 스코어링을 수행한다. 전체 구조는 '인스턴스 세그멘테이션 $\rightarrow$ GNN $\rightarrow$ Recurrent Connection $\rightarrow$ 출력 및 다음 프레임 전달' 순으로 구성된다.

### 1. 트랙-검출 할당 (Track-Detection Association)

각 프레임에서 검출기(YOLACT)가 생성한 검출물 $\{\delta_n\}$과 기존에 유지되던 트랙 메모리 $\{\tau_m\}$을 입력으로 하여 그래프를 구축한다.

- **그래프 구조**: 트랙 $\tau_m$과 검출물 $\delta_n$을 노드로, 이들 사이의 관계 $e_{mn}$을 엣지로 설정한다. 신규 트랙 가능성을 판단하기 위해 가상의 '빈 트랙(empty track) $\tau_0$'을 도입하여 $e_{0n}$ 엣지를 생성한다.
- **GNN 업데이트**: 정보 교환을 위해 다음과 같은 업데이트 식을 사용한다.
  $$e_{mn}^+ = f_e([e_{mn}, \tau_m, \delta_n])$$
  $$\tau_m^+ = f_\tau([\tau_m, \sum_j g_\tau(e_{mj})e_{mj}])$$
  $$\delta_n^+ = f_\delta([\delta_n, \sum_i g_\delta(e_{in})e_{in}])$$
  여기서 $f$는 ReLU가 포함된 선형 층이며, $g$는 시그모이드 함수로 끝나는 MLP(Multi-Layer Perceptron)이다. 특히 $g$는 게이트(gate) 역할을 하여 어떤 정보만 선택적으로 수집할지 결정한다.
- **할당 결정**: 업데이트된 엣지 임베딩 $e_{mn}$을 로지스틱 모델에 통과시켜 매칭 확률을 예측한다. $e_{0n}$은 해당 검출물이 새로운 트랙을 생성해야 하는지에 대한 확률을 예측한다.

### 2. 트랙 스코어링 (Track Scoring)

각 트랙에 대해 신뢰도와 클래스 예측을 수행한다. 이는 단순 평균이 아니라 GNN을 통해 수집된 정보와 RNN을 통한 시간적 누적 정보가 담긴 트랙 임베딩 $\tau_m$을 기반으로 다항 로지스틱 모델(multinomial logistic model)을 통해 예측된다.

### 3. 재귀적 연결 (Recurrent Connection)

단순한 RNN 연결은 학습이 불안정하고 기울기 소실/폭주 문제가 발생하므로, LSTM의 게이팅 메커니즘을 도입하였다.

- **구조**: GNN의 출력을 LSTM의 입력으로 사용한다.
  $$\{\tilde{\tau}_t^m\}_m = \text{GNN}(\{\tau_{t-1}^m\}_m, \{\delta_t^n\}_n)$$
  $$\tau_t^m = L(\tilde{\tau}_t^m)$$
  여기서 $L$은 LSTM의 게이트 시스템($\alpha_{\text{forget}}, \alpha_{\text{input}}, \alpha_{\text{output}}$ 등)을 의미하며, 이를 통해 갱신된 $\tau_t^m$이 다음 프레임의 GNN 입력으로 전달된다.

### 4. 외형 모델링 (Modelling Appearance)

객체의 외형을 대각 공분산(diagonal covariance)을 가진 다차원 가우시안 분포로 모델링한다.

- **업데이트 방식**: 베이지안 업데이트(Bayesian update)를 사용하여 새로운 검출물의 외형 $x$를 반영한다.
  $$\mu^+ = \kappa x + (1-\kappa)\mu$$
  $$\Sigma^+ = \nu \tilde{\Sigma} + (1-\nu)\Sigma + \frac{\kappa(1-\nu)}{\kappa+\nu}(x-\mu)^2$$
  여기서 업데이트 속도를 결정하는 $\kappa$와 $\nu$는 고정값이 아니라 트랙 임베딩으로부터 예측되는 학습 가능한 파라미터이다.

### 5. 학습 절차 및 손실 함수

학습은 2단계로 진행된다. (1) 인스턴스 세그멘테이션 모델(YOLACT)을 먼저 학습시키고, (2) 이를 동결(freeze)한 후 GNN, RNN, 외형 네트워크를 학습시킨다.
전체 손실 함수는 다음과 같다.
$$L = \lambda_1 L_{\text{score}} + \lambda_2 L_{\text{seg}} + \lambda_3 L_{\text{match}} + \lambda_4 L_{\text{init}}$$

- $L_{\text{score}}$: 클래스 예측 정확도 (Cross-Entropy)
- $L_{\text{seg}}$: 세그멘테이션 결과의 정확도 (Lovasz loss)
- $L_{\text{match}}$: 기존 트랙-검출 매칭 정확도 (BCE)
- $L_{\text{init}}$: 신규 트랙 생성 여부 정확도 (BCE)

## 📊 Results

### 실험 설정

- **데이터셋**: YouTube-VIS (40개 클래스, 2883개 비디오)
- **지표**: video mean Average Precision (mAP)
- **백본**: ResNet50 및 ResNet101
- **속도**: ResNet50 기준 30 FPS 달성

### 정량적 결과

- **SOTA 비교**: 제안 방법(ResNet50)은 35.3 mAP를 기록하여, SipMask(33.7 mAP)나 STEm-Seg(30.6 mAP) 등 기존 실시간/근실시간 방법론들을 능가하였다. 특히 ResNet101을 사용할 경우 37.7 mAP까지 성능이 향상되었다.
- **비실시간 모델과의 비교**: MaskProp나 VIS2019 Winner 모델은 mAP가 더 높지만(44.8~46.6), 처리 속도가 1~2 FPS 수준으로 매우 느려 실시간 적용이 불가능하다.

### 절제 연구 (Ablation Study)

- **GNN의 중요성**: GNN을 제한하고 단순 유사도 기반으로 할당했을 때 mAP가 6.7p 하락하였다. 이는 전역적 추론의 중요성을 입증한다.
- **외형 모델링**: 외형 정보를 제거하거나 단순한 업데이트 방식을 사용할 경우 성능이 저하되었으며, 제안한 학습 가능한 베이지안 업데이트가 가장 효과적이었다.
- **LSTM 게이팅**: LSTM-like 게이팅을 제거했을 때 모델 학습이 발산(diverge)하였다. 이는 시간적 정보를 처리할 때 게이팅 메커니즘이 안정적인 학습을 위해 필수적임을 보여준다.

## 🧠 Insights & Discussion

### 강점

본 논문은 VIS의 전체 파이프라인을 학습 가능한 형태로 정식화함으로써, 검출기의 노이즈를 모델이 스스로 학습하여 억제할 수 있게 하였다. 정성적 결과에서 검출기가 일시적으로 객체를 놓치거나(missed detection) 잘못 검출(false positive)하더라도, GNN의 전역 추론과 RNN의 시간적 메모리를 통해 트랙을 유지하거나 빠르게 제거하는 능력이 확인되었다.

### 한계 및 비판적 해석

- **클래스 외 객체 탐지**: 정성적 분석 결과, 학습 데이터에 없는 클래스(예: 펭귄)를 유사한 다른 클래스(예: 코끼리)로 오인하여 추적하는 경우가 발생하였다. 이는 모델이 외형적 유사성에 의존하는 경향이 있음을 시사한다.
- **그림/사진의 오탐지**: 실제 객체가 아닌, 트럭에 그려진 사람 그림 등을 실제 객체로 인식하여 추적하는 문제가 발견되었다. 이는 2D 이미지 기반의 특징 추출이 가진 근본적인 한계로 보이며, 깊이 정보나 더 고차원적인 맥락 정보가 필요할 것으로 판단된다.
- **2단계 학습의 한계**: 검출기를 먼저 학습시키고 이후 모듈을 학습시키는 greedy한 방식은 완전한 엔드투엔드 학습에 비해 최적해를 찾는 능력이 떨어질 수 있다. 다만, 이는 메모리 제약으로 인한 선택이었음을 명시하고 있다.

## 📌 TL;DR

본 논문은 비디오 인스턴스 세그멘테이션(VIS)을 위해 **GNN(전역적 관계 추론)**과 **RNN(시간적 메모리 유지)**을 결합한 새로운 학습 프레임워크를 제안하였다. 특히 추론 과정을 학습 단계에 밀접하게 반영하고, 외형 모델을 학습 가능한 가우시안 분포로 설계하여 실시간성(30 FPS)과 높은 정확도(mAP 35.3+)를 동시에 확보하였다. 이 연구는 휴리스틱한 모듈 결합 방식에서 벗어나 VIS 전체 과정을 통합적으로 학습할 수 있는 가능성을 제시했다는 점에서 향후 실시간 객체 추적 및 세그멘테이션 연구에 중요한 역할을 할 것으로 보인다.
