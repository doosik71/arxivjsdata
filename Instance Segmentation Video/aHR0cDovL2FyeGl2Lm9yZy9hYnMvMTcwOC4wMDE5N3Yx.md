# Video Object Segmentation with Re-identification

Xiaoxiao Li, Yuankai Qi, Zhe Wang, Kai Chen, Ziwei Liu, Jianping Shi, Ping Luo, Xiaoou Tang, Chen Change Loy (2017)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 문제는 비디오 객체 분할(Video Object Segmentation, VOS)에서 발생하는 객체 손실 및 추적 실패 문제이다. 기존의 VOS 방법론들은 주로 인접 프레임 간의 시간적 연속성(temporal continuity)에 의존하여 마스크를 전파(propagation)하는 방식을 사용한다.

그러나 이러한 가정은 다음과 같은 상황에서 심각한 한계를 보인다:
1. **Drifting**: 전파 과정에서 오차가 누적되어 마스크의 위치가 서서히 벗어나는 현상이다.
2. **Large Displacement**: 객체가 빠르게 움직여 인접 프레임 간의 거리 차이가 클 때 대응하기 어렵다.
3. **Occlusions**: 객체가 다른 물체에 의해 가려졌다가 다시 나타날 때, 연속성이 끊어져 객체를 놓치게 된다.

따라서 본 논문의 목표는 적응형 객체 재식별(adaptive object re-identification) 메커니즘을 도입하여, 객체가 소실되었을 때 이를 효과적으로 복구함으로써 비디오 전체에서 강건하게 객체를 분할하는 VS-ReID 모델을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **단기 기억(Short-term memory)**에 기반한 '마스크 전파'와 **장기 기억(Long-term memory)**에 기반한 '객체 재식별'을 결합하여 상호 보완적인 시스템을 구축하는 것이다.

- **마스크 전파 모듈(Mask Propagation Module)**: 시간적 연속성을 활용하여 정밀한 마스크를 생성하며, 전체 이미지 대신 객체 바운딩 박스(Bounding Box) 기반의 패치를 입력으로 사용하여 다양한 스케일의 객체와 세밀한 구조를 더 잘 포착한다.
- **재식별 모듈(Re-identification Module)**: 전파 과정에서 유실된 객체를 탐색하고 복구한다. 이는 템플릿과의 유사도 비교를 통해 장기적인 관점에서 객체를 다시 찾아내는 역할을 한다.
- **반복적 정제 및 체크포인트 메커니즘**: 두 모듈을 반복적으로 적용하여 마스크를 업데이트하며, 업데이트 시점의 거리를 기록하는 체크포인트 방식을 통해 전파 거리가 길어져 품질이 저하된 마스크가 최신 마스크를 덮어쓰지 않도록 방지한다.

## 📎 Related Works

기존의 비디오 분할 방법론들은 딥러닝과 그래피컬 모델을 결합하거나 고해상도 이미지에서 실시간 추론을 가능하게 하는 방향으로 발전해 왔다. 특히 시간적 상관관계를 이용해 마스크를 전파하는 방식들이 주를 이루었으나, 실제 비디오 환경에서 발생하는 심한 변형(deformation)과 폐색(occlusion) 문제로 인해 앞서 언급한 drifting 및 큰 변위 처리 불가 문제가 지속적으로 제기되었다.

VS-ReID는 이러한 기존의 '전파 중심' 접근 방식에 '재식별'이라는 복구 메커니즘을 추가함으로써, 연속성이 끊긴 상황에서도 객체를 다시 찾아낼 수 있다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 1. Mask Propagation Module
마스크 전파 모듈은 프레임 $i$의 확률 맵 $P_{i,k}$를 이용하여 프레임 $j$의 확률 맵 $P_{j,k}$를 예측한다.

- **Flow Guided Warping**: 먼저 FlowNet2.0을 통해 프레임 $i$와 $j$ 사이의 광학 흐름(optical flow) $f_{i \to j}$를 추출하고, bilinear warping 함수 $W$를 통해 $P_{i,k}$를 $P_{i \to j,k}$로 변형하여 거친 추정치(coarse estimation)를 얻는다.
- **Mask Propagation Network ($N_{mp}$)**: 거친 추정치를 정밀하게 다듬기 위해 두 개의 스트림(RGB 브랜치, Optical Flow 브랜치)으로 구성된 CNN을 사용한다.
    - **입력 방식**: 전체 이미지가 아닌, 객체를 포함하는 바운딩 박스 영역의 패치($I^b_j, f^b_{i \to j}, P^b_{i \to j,k}$)를 정규화된 크기로 잘라 입력한다. 이는 다양한 크기의 객체에 대응하고 세밀한 구조를 캡처하기 위함이다.
    - **구조적 특징**: ResNet-101을 백본으로 사용하며, 예측 해상도를 높이기 위해 컨볼루션 스트라이드를 줄이고 Dilated Convolution과 Atrous Spatial Pyramid Pooling(ASPP)을 적용하였다.
- **최종 출력**: RGB 브랜치와 Flow 브랜치의 예측값을 평균 내어 최종 마스크를 생성한다.

### 2. Re-identification Module
재식별 모듈은 마스크 전파가 실패한 지점에서 유실된 인스턴스를 복구한다.

- **절차**:
    1. 탐지 네트워크 $N_{det}$ (Faster R-CNN)를 통해 프레임 $I_i$에서 후보 바운딩 박스 $X$를 추출한다.
    2. 각 후보 박스 $x_j$와 대상 객체의 템플릿 $t_k$ 간의 특징을 $N_{reid}$ 네트워크로 추출하여 코사인 유사도 $s_j$를 계산한다.
    3. 가장 유사도가 높은 후보 $\hat{x}_j$를 선택한다.
- **수락 조건**: 다음 두 조건이 모두 충족될 때만 재식별 결과로 인정한다.
    - 유사도 점수가 임계값 $\rho_{reid}$보다 높아야 한다: $s_{\hat{j}} > \rho_{reid}$
    - 현재 예측된 마스크의 바운딩 박스와의 IoU가 임계값 $\rho_{occ}$보다 낮아야 한다: $\text{IoU}(\hat{x}_j, b) < \rho_{occ}$ (이미 잘 추적되고 있다면 재식별이 불필요하기 때문)

### 3. VS-ReID 파이프라인 및 알고리즘
시스템은 다음과 같은 단계로 동작한다.

1. **초기화**: 첫 프레임의 마스크를 기반으로 마지막 프레임까지 순방향으로 마스크를 전파한다.
2. **반복적 정제 (Iterative Refinement)**:
    - 모든 프레임과 인스턴스에 대해 ReID 모듈을 적용하여 가장 높은 유사도를 가진 $\hat{x}$를 찾는다.
    - 해당 지점을 시작점으로 하여 첫 프레임의 템플릿을 이용해 확률 맵을 복구한다.
    - 복구된 지점을 기준으로 양방향(bi-directional)으로 다시 마스크 전파를 수행한다.
3. **체크포인트 메커니즘**: 변수 $c_{i,k}$를 통해 $P_{i,k}$가 업데이트된 시작점을 기록한다. 업데이트 시 $\text{dist}(c_{i,k}, i)$보다 새로운 시작점과의 거리가 더 가까울 때만 업데이트를 허용하여, 전파 거리가 길어 품질이 낮은 마스크가 기존의 양질의 마스크를 덮어쓰는 것을 방지한다.
4. **최종 마스크 융합**: 각 픽셀 $l$에서 가장 높은 확률을 가진 클래스를 선택한다.

$$M_i(l) = \arg \max_{0 \le k \le K} \frac{P_{i,k}(l)}{\prod_{j=1}^{K} (1-P_{i,j}(l)) + \sum_{j=1}^{K} P_{i,j}(l)}$$
*(단, 분자는 $k \neq 0$일 때의 확률이며, $k=0$인 배경의 경우 $\prod (1-P_{i,j}(l))$를 사용한다. $Z$는 정규화 인자이다.)*

## 📊 Results

### 실험 설정
- **데이터셋**: DAVIS 2017 (150개 비디오 시퀀스)
- **평가 지표**: Region Jaccard ($\mathcal{J}$), Boundary F-measure ($\mathcal{F}$)
- **기준선**: Perazzi et al. [11] 의 방법론을 baseline으로 설정

### 주요 결과
- **벤치마크 성과**: 2017 DAVIS Challenge의 test-challenge 세트에서 **Global Mean 0.699**를 기록하며 1위를 차지하였다. 2위 방법론 대비 2% 이상의 성능 향상을 보였으며, 특히 $\mathcal{J}\text{-Decay}$와 $\mathcal{F}\text{-Decay}$ 값이 낮게 나타나 장기적인 추적 성능이 우수함을 입증하였다.

### Ablation Study
각 구성 요소가 성능 향상에 기여하는 정도는 다음과 같다 (Global Mean 기준):
- **Baseline [11]**: $0.517$
- **Full-image $\to$ BBox patch 입력**: $0.555$ ($\uparrow 0.038$) $\rightarrow$ 가장 큰 성능 향상을 보였으며, 스케일 변화 대응 및 경계 세부 묘사에 효과적임을 확인하였다.
- **Flow-stream 추가**: $0.584$ ($\uparrow 0.007$)
- **Re-ID 모듈 및 반복 정제 적용**: $0.652$ ($\uparrow 0.068$) $\rightarrow$ 유실된 객체 복구가 성능 향상에 핵심적임을 보여준다.
- **Multi-scale testing 적용**: $0.661$ ($\uparrow 0.009$)

## 🧠 Insights & Discussion

**강점**:
본 연구는 단순한 시간적 전파의 한계를 인지하고, '재식별'이라는 장기 기억 메커니즘을 통합함으로써 VOS의 고질적인 문제인 occlusion과 drifting 문제를 효과적으로 해결하였다. 특히 전체 이미지가 아닌 패치 기반의 입력을 사용함으로써 연산 효율성을 높이면서도 작은 객체와 세밀한 경계를 더 잘 포착할 수 있게 설계된 점이 인상적이다. 또한, 체크포인트 메커니즘을 통해 반복적 업데이트 과정에서 발생할 수 있는 품질 저하 문제를 논리적으로 방어하였다.

**한계 및 논의**:
논문에서 명시적으로 언급되지는 않았으나, ReID 모듈과 마스크 전파 모듈을 반복적으로 적용하는 구조는 비디오의 길이에 따라 추론 시간이 상당히 증가할 가능성이 있다. 또한, 재식별을 위해 사용되는 임계값 $\rho_{reid}$와 $\rho_{occ}$가 검증 세트(validation set)에서 선택되었으므로, 데이터셋의 특성에 따라 이 하이퍼파라미터에 대한 민감도가 존재할 수 있다.

## 📌 TL;DR

본 논문은 비디오 객체 분할에서 객체 유실 문제를 해결하기 위해 **마스크 전파(단기 기억)**와 **객체 재식별(장기 기억)**을 결합한 **VS-ReID** 모델을 제안한다. 패치 기반의 2-스트림 CNN 전파 방식과 반복적인 양방향 복구 프로세스, 그리고 이를 관리하는 체크포인트 메커니즘을 통해 2017 DAVIS Challenge에서 1위를 달성하였다. 이 연구는 VOS 분야에서 단순 전파를 넘어선 '복구' 개념의 중요성을 입증하였으며, 향후 강건한 비디오 객체 추적 및 분할 연구에 중요한 기초가 될 것으로 보인다.
