# R3Net: Random Weights, Rectifier Linear Units and Robustness for Artificial Neural Network

Arun Venkitaraman, Alireza M. Javid, and Saikat Chatterjee (2018)

## 🧩 Problem to Solve

본 논문은 인공 신경망(ANN)이 학습 데이터에 과도하게 적합(overfitting)됨에 따라 발생하는 입력 섭동(input perturbation)에 대한 취약성 문제를 해결하고자 한다. 특히, 이미지 분류 작업 등에서 인간이 인지하지 못할 정도의 미세한 노이즈가 추가되었음에도 불구하고 클래스 레이블이 완전히 바뀌는 현상이 발생하며, 이는 신경망이 적대적 공격(adversarial attacks)에 매우 취약함을 의미한다.

연구의 핵심 목표는 입력의 작은 변화가 출력의 급격한 변화로 이어지지 않는 **노이즈 강건성(noise robustness)**과, 서로 다른 입력이 출력 공간에서도 적절한 거리를 유지하여 구별 가능하게 만드는 **판별 능력(discrimination ability)**을 동시에 갖춘 네트워크 구조를 제안하고 이를 수학적으로 증명하는 것이다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 무작위 가중치(random weights), 구조화된 사인-스플리터(sign-splitter), 그리고 ReLU 활성화 함수를 결합한 **R3Net** 아키텍처를 설계하는 것이다.

단순한 ReLU는 입력이 음수일 때 출력이 0이 되어 정보가 손실되고 판별 능력이 저하되는 한계가 있다. 이를 해결하기 위해 저자들은 입력을 양의 부분과 음의 부분으로 분리하여 동시에 처리하는 사인-스플리터 구조를 도입하였다. 이를 통해 출력 특징 공간에서 거리의 수축(contraction)을 유도하여 강건성을 확보하는 동시에, 입력 간의 최소 거리를 유지함으로써 판별 능력을 보존하는 수학적 메커니즘을 제시하였다.

## 📎 Related Works

기존 연구들은 불변성(invariances)을 구축하여 변형이나 섭동에 강건한 특징을 추출하려 노력해 왔다. 또한, Extreme Learning Machine(ELM)과 같이 가중치의 상당 부분을 무작위로 설정하고 일부만 학습시키거나 전체를 고정하는 방식이 계산 복잡도를 낮추면서도 높은 성능을 낼 수 있음이 알려져 있다.

그러나 기존의 무작위 가중치 기반 네트워크들이 왜 실제로 잘 작동하는지에 대한 수학적 설명은 부족했다. 본 논문은 단순히 성능을 측정하는 것을 넘어, 무작위 가중치와 ReLU의 조합이 어떻게 입력 섭동에 대한 립시츠 연속성(Lipschitz continuity)을 보장하고 강건성을 제공하는지를 분석적으로 규명함으로써 기존 연구들과 차별점을 가진다.

## 🛠️ Methodology

### 1. 전체 파이프라인 및 단일 블록 구조

R3Net의 기본 단위는 단일 블록(layer)으로 구성되며, 입력 벡터 $q \in \mathbb{R}^{m \times 1}$에 대해 다음과 같은 연산을 수행한다.

$$\bar{y} = g(V W q)$$

여기서 $W$는 무작위 가중치 행렬, $V$는 사인-스플리터 행렬, $g(\cdot)$는 성분별 ReLU 활성화 함수이다.

### 2. 주요 구성 요소 및 역할

- **무작위 가중치 행렬 ($W$):** Gaussian 분포 $\mathcal{N}(0, \frac{1}{n})$ 등에서 독립적으로 추출된 가중치를 사용한다. 이는 입력 데이터를 고차원 공간으로 매핑하며, 압축 센싱(compressed sensing)의 RIP(Restricted Isometry Property) 이론에 근거하여 입력 벡터 간의 거리를 보존하는 역할을 한다.
- **사인-스플리터 ($V$):** ReLU의 정보 손실 문제를 해결하기 위해 다음과 같이 정의된다.
$$V = \begin{bmatrix} I_n \\ -I_n \end{bmatrix}$$
이 행렬은 입력 벡터 $z = Wq$를 $[z, -z]^T$ 형태로 확장하여, 이후 ReLU를 통과했을 때 $z$의 양수 부분($z^+$)과 음수 부분($z^-$)이 모두 보존되게 한다.
- **ReLU 활성화 함수 ($g$):** $g(x) = \max(0, x)$를 적용한다. $V$와 결합된 $g(Vz)$는 결과적으로 $[|z^+|, |z^-|]^T$와 같은 형태가 되어, 입력의 절대적인 크기 정보를 유지한다.

### 3. 수학적 분석 및 손실 함수

본 논문은 학습 알고리즘보다는 아키텍처의 해석적 특성에 집중한다. 두 입력 $q_1, q_2$에 대해 출력 $\bar{y}_1, \bar{y}_2$ 사이의 관계는 다음과 같이 정의된다.

$$\kappa(1-\delta)\|q_1 - q_2\|_2 \le \|\bar{y}_1 - \bar{y}_2\|_2 \le (1+\delta)\|q_1 - q_2\|_2$$

여기서 $\delta$는 RIP 상수로, $\kappa$는 입력 섭동의 강도에 따라 결정되는 계수이다.

- **상한선(Upper Bound):** $(1+\delta)$에 의해 출력의 변화가 입력의 변화에 의해 제한되므로 노이즈 강건성(립시츠 연속성)이 보장된다.
- **하한선(Lower Bound):** $\kappa(1-\delta)$에 의해 입력이 충분히 다르면 출력 또한 일정 거리 이상을 유지하여 판별 능력이 보장된다.

### 4. 체인 구조(Blockchain Construction)

이러한 단일 블록을 $L$개 층으로 연결한 전체 네트워크의 출력 $\bar{y}^{(L)}$은 다음과 같은 범위를 가진다.

$$\prod_{l=1}^{L} \kappa_l (1-\delta_l) \|x_1 - x_2\|_2 \le \|\bar{y}^{(L)}_1 - \bar{y}^{(L)}_2\|_2 \le \prod_{l=1}^{L} (1+\delta_l) \|x_1 - x_2\|_2$$

## 📊 Results

본 논문은 수치적인 벤치마크 데이터셋 실험 대신 **분석적 증명과 시뮬레이션** 결과만을 제시한다.

- **실험 설정:** 입력 $q_1$과 섭동 $\Delta$를 각각 이방성 다변량 가우시안 분포 $\mathcal{N}(0, \sigma^2 I)$에서 추출하여 설정하였다. ($\sigma_{q_1}=1, \sigma_{\Delta}=0.25$)
- **정성적 결과:** Figure 1의 산점도(scatter plot) 분석 결과, 모든 출력 섭동 $\|\bar{y}_1 - \bar{y}_2\|^2$ 값이 기준선인 $\|q_1 - q_2\|^2$보다 엄격하게 아래에 위치함을 확인하였다.
- **결론:** 이는 네트워크가 입력의 섭동을 효과적으로 감쇄(attenuation)시키고 있음을 보여주며, 특히 가중치 행렬 $W$의 차원이 커질수록 이러한 수축(contraction) 효과가 더 강하게 나타남을 확인하였다.

## 🧠 Insights & Discussion

### 강점

본 연구는 딥러닝의 블랙박스적인 특성에서 벗어나, 무작위 가중치와 특정 구조(사인-스플리터)의 결합이 어떻게 수학적으로 강건성을 제공하는지를 명확히 규명하였다. 특히 ReLU의 치명적 단점인 '음수 영역 정보 소실'을 $V$ 행렬을 통해 해결하면서도 강건성을 유지한 점이 탁월하다.

### 한계 및 가정

1. **실증적 검증 부족:** 실제 이미지 데이터셋(MNIST, CIFAR-10 등)이나 실제 적대적 공격 시나리오에서의 성능 평가가 포함되지 않았다. 저자들은 향후 확장 논문에서 다루겠다고 언급하였으나, 현재로서는 이론적 가능성 제시 수준에 머물러 있다.
2. **희소성 가정:** RIP 이론을 적용하기 위해 입력 벡터 $q$가 희소(sparse)하다는 가정을 전제하고 있다. 실제 데이터가 이러한 희소성 조건을 항상 만족하는지에 대한 논의가 필요하다.

### 비판적 해석

R3Net은 학습 없이 가중치를 고정하는 구조에 가깝다. 실제 딥러닝의 강점은 데이터로부터 최적의 특징을 '학습'하는 것인데, 본 논문은 '무작위성'이 주는 일반적인 강건성에 집중하고 있다. 따라서 이 구조가 실제 복잡한 태스크에서 학습 가능한 네트워크와 비교해 어느 정도의 성능 저하가 있을지, 혹은 학습 단계에서 $W$를 미세 조정했을 때 강건성이 유지될지가 핵심적인 연구 과제가 될 것이다.

## 📌 TL;DR

R3Net은 **무작위 가중치 $\rightarrow$ 사인-스플리터 $\rightarrow$ ReLU** 순의 구조를 통해 입력 노이즈에 강건하면서도 데이터 판별력을 유지하는 신경망 아키텍처를 제안한다. 수학적으로 출력의 립시츠 연속성을 증명하여 섭동의 감쇄 효과를 입증하였으며, 이는 무작위 특징 추출 기반 네트워크가 왜 실무에서 강건하게 작동하는지에 대한 이론적 근거를 제공한다. 향후 실제 데이터셋에 적용되어 적대적 공격 방어 성능이 입증된다면, 매우 가볍고 강력한 강건한 네트워크 설계 방법론이 될 가능성이 크다.
