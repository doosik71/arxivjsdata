# SurgPose: a Dataset for Articulated Robotic Surgical Tool Pose Estimation and Tracking

Zijian Wu et al. (2025)

## 🧩 Problem to Solve

로봇 보조 수술(Robot-assisted surgery, RAS)에서 수술 도구의 정확한 포즈 추정(Pose Estimation)은 증강 현실(AR) 가이드, 자율 조작(Autonomous manipulation) 및 시각적 서보잉(Visual servoing)과 같은 하위 응용 분야를 위해 필수적이다.

현재 da Vinci와 같은 시스템은 관절 상태와 기구학(Kinematics) 데이터를 제공하지만, 실제 환경에서는 케이블의 유연성, 기구의 휨, 백래시(Backlash) 등으로 인해 기구학적 모델만으로는 정확한 도구 위치를 파악하기 어렵다. 또한, 카메라 뷰와 로봇 좌표계를 맞추는 Hand-eye calibration 과정이 매우 복잡하고 시간이 많이 소요된다. 이러한 한계를 극복하기 위해 카메라 기반의 직접적인 포즈 추정 방식이 대안으로 제시되었으나, 딥러닝 모델을 학습시키기 위한 고품질의 실제 데이터셋이 매우 부족한 상황이다. 특히 합성 데이터(Synthetic data)는 Sim-to-real gap 문제로 인해 성능에 한계가 있으며, 실제 수술 환경의 데이터 수집 비용은 매우 높다.

본 논문의 목표는 관절형 수술 도구의 시각적 포즈 추정 및 추적을 위해 인스턴스 인식 시맨틱 키포인트(Instance-aware semantic keypoints)와 스켈레톤(Skeletons) 정보를 포함하는 대규모 데이터셋인 **SurgPose**를 구축하는 것이다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 **자외선(UV) 반응 페인트**를 사용하여 가시광선 영역에서는 보이지 않지만 UV 광원 아래에서는 형광으로 나타나는 투명 마커를 도구에 도포하는 것이다. 이를 통해 다음과 같은 기여를 한다.

1. **비침습적 고정밀 라벨링**: 수술 도구의 외관을 거의 변화시키지 않으면서도, UV 광원을 통해 정밀한 키포인트 위치를 추출할 수 있는 데이터 수집 파이프라인을 제안한다.
2. **대규모 멀티모달 데이터셋 구축**: 6종의 수술 도구에 대해 약 120k 개의 인스턴스(학습 80k, 검증 40k)를 포함하는 데이터셋을 구축하였다. 여기에는 2D/3D 키포인트뿐만 아니라 엔드이펙터 기구학, PSM 관절 상태, 파트 수준의 세그멘테이션 마스크가 포함된다.
3. **3D 포즈 확장성 제공**: 정류화된 스테레오(Rectified stereo) 영상을 제공하여 2D 포즈를 3D로 리프팅할 수 있도록 하였다.
4. **기준선 평가**: YOLOv8, ViTPose, DeepLabCut과 같은 범용 포즈 추정 모델을 적용하여 데이터셋의 유용성을 검증하였다.

## 📎 Related Works

### 관련 연구 및 한계

- **전통적 방식**: 템플릿 매칭(Template matching)이나 CAD 모델 기반의 가상 템플릿 생성 방식이 사용되었다. 하지만 이러한 방식은 저수준 시각 특징에 의존하므로 시맨틱 정보가 부족하며, 영상 손상이나 배경 변화에 취약하다.
- **딥러닝 방식**: UNet, FCN, Attention 기반 모델 등이 제안되었으나, 대부분 소규모 데이터셋이나 합성 데이터에 의존하여 일반화 성능이 떨어진다.
- **기존 데이터셋**: RMIT, EndoVis15, SurgRIPE 등이 존재하지만, 데이터의 양이 매우 적거나, 단순한 샤프트 포인트(Shaft point)만 제공하거나, 정밀한 관절 키포인트가 부족한 경우가 많다.

### SurgPose의 차별점

SurgPose는 기존 데이터셋보다 훨씬 많은 양의 라벨링된 프레임을 제공하며, 특히 UV 페인트를 이용해 수동 라벨링의 오차를 줄이면서도 실제 수술 도구의 외관을 유지한 채 정밀한 Ground Truth를 확보했다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 1. UV 반응 페인트 라벨링 및 데이터 수집

- **마커 도포**: 가시광선에서는 투명하고 UV 광원에서는 형광을 띠는 락커 기반 UV-reactive 페인트를 사용한다. 빨간색 페인트가 녹색보다 투명도가 높아 최종적으로 선택되었다.
- **궤적 생성**: dVRK(da Vinci Research Kit)를 사용하여 PSM 1과 3이 사전에 정의된 궤적을 따라 움직이게 한다.
  - **위치 궤적**: 작업 공간 내에서 무작위 점을 선택하고 주기적 큐빅 스플라인(Periodic interpolating cubic spline)을 사용하여 부드러운 폐곡선을 생성한다.
  - **관절 궤적**: 관절 4-7 및 그리퍼 각도에 대해 주기적인 사인 함수(Sinusoidal) 궤적을 생성한다.
- **수집 절차**: 동일한 궤적을 **백색광(White light)** 환경과 **UV 광원(Black light)** 환경에서 각각 두 번 실행하여, 실제 영상과 라벨링용 영상을 쌍으로 수집한다.

### 2. 데이터 처리 및 어노테이션

- **키포인트 추출**: UV 광원 아래에서 촬영된 영상에서 형광 마커를 세그멘테이션한다. 이때 **SAMv2 (Segment Anything 2)** 모델을 사용하여 첫 프레임의 프롬프트를 통해 전체 비디오로 마스크를 전파(Propagate)함으로써 라벨링 효율을 높였다.
- **스테레오 매칭 및 3D 리프팅**: 체커보드를 이용해 카메라를 캘리브레이션하고 영상을 정류화(Rectification)한다. 이후 RAFT 알고리즘을 사용하여 시차(Disparity)를 계산하고 다음 수식을 통해 깊이(Depth)를 도출한다.

$$\text{Depth} = \frac{\text{focal length} \times \text{baseline}}{|\text{disparity} + (c_{x1} - c_{x0})|}$$

여기서 $c_{x1} - c_{x0}$는 x축을 따른 주점(Principal points)의 차이이다.

### 3. 마커 가시성 검증

UV 페인트가 실제 영상(백색광)에서 보이지 않는지 확인하기 위해, 페인트 도포 전후의 영상을 뺀 차분 영상(Difference image)을 분석하고, **GLCM(Gray Level Co-occurrence Matrix)** 특징을 추출하여 분류기가 도포 전후의 패치를 구분할 수 있는지 확인하였다. 결과적으로 알고리즘적으로 구분하기 어려울 만큼 가시성이 낮음을 확인하였다.

## 📊 Results

### 실험 설정

- **데이터 구성**: 6종의 도구(Large/Mega Needle Driver, Micro Forceps, Curved Scissor, DeBakey Forceps, Prograsp Forceps)를 사용하였다.
- **배경**: 학습셋은 닭 겐트(Chicken gizzard) 및 돼지 간(Pork liver) ex vivo 배경을, 검증셋은 닭 허벅지 및 소고기 ex vivo 배경을 사용하였다.
- **평가 지표**: COCO 데이터셋의 **OKS (Object Keypoint Similarity)**를 기반으로 한 **mAP (mean Average Precision)**를 측정하였다.

### 정량적 결과

세 가지 Baseline 모델의 성능 결과는 다음과 같다.

| Method | mAP | Inference Latency (ms) |
| :--- | :---: | :---: |
| YOLOv8 | $63.3 \pm 1.2$ | 16.4 |
| ViTPose | $53.6 \pm 1.0$ | 67.7 |
| DeepLabCut | $66.2 \pm 2.0$ | 194.8 |

- **DeepLabCut**이 가장 높은 mAP를 기록하였으나, 추론 속도가 매우 느려 실시간 적용이 어렵다.
- **YOLOv8**은 성능과 속도 사이의 균형이 좋으며 실시간 추론이 가능하다.
- **ViTPose**는 상대적으로 낮은 성능을 보였다.

## 🧠 Insights & Discussion

### 강점

SurgPose는 수동 라벨링의 고질적인 문제인 '시간 소요'와 '인적 오류'를 UV 페인트와 SAMv2의 조합으로 해결하였다. 또한 스테레오 영상과 기구학 데이터를 함께 제공함으로써 2D를 넘어 3D 포즈 추정 연구로 확장할 수 있는 기반을 마련하였다.

### 한계 및 비판적 해석

논문에서 명시했듯이, 데이터셋의 규모는 커졌으나 **배경의 다양성이 여전히 부족**하다. 실험 결과, 학습셋과 다른 배경(검증셋)에서 모델의 성능이 급격히 떨어지는 심각한 **오버피팅(Overfitting)** 현상이 관찰되었다. 이는 현재의 데이터셋만으로는 실제 임상 환경으로의 직접적인 전이(Transfer)가 어렵다는 것을 의미한다. 즉, 실제 적용을 위해서는 더 다양한 in vivo 환경 데이터나 합성 데이터를 결합한 데이터 증강 기법이 필수적이다.

또한, 도구 간의 **상호 가려짐(Occlusion)** 상황이 포함되지 않았다는 점은 실제 수술 환경의 복잡성을 충분히 반영하지 못한 한계로 지적될 수 있다.

## 📌 TL;DR

본 논문은 UV 반응 페인트를 이용해 외관 변화 없이 정밀한 키포인트 라벨링을 수행한 수술 도구 포즈 추정 데이터셋 **SurgPose**를 제안한다. 6종의 도구에 대해 120k 개의 인스턴스와 스테레오 영상, 기구학 정보를 제공하며, 이는 수술 로봇의 시각적 포즈 추정 모델을 학습시키고 평가하는 데 중요한 벤치마크가 될 것이다. 향후 이 데이터셋은 Few-shot learning이나 파운데이션 모델의 미세 조정(Fine-tuning) 연구에 유용하게 활용될 가능성이 높다.
