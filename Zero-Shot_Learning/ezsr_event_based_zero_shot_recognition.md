# EZSR: Event-based Zero-Shot Recognition

Yan Yang, Liyuan Pan, Dongxu Li, Liu Liu (2024)

## 🧩 Problem to Solve

본 논문은 이벤트 카메라(Event Camera) 데이터를 이용한 **Zero-Shot Object Recognition** 문제를 해결하고자 한다. 이벤트 카메라는 전통적인 이미지 센서와 달리 픽셀 단위의 밝기 변화만을 비동기적으로 캡처하여 이벤트 스트림을 생성하므로, 모션 블러가 없고 시간 해상도가 높으며 전력 소모가 적다는 장점이 있다.

그러나 이벤트 데이터는 RGB 이미지에 비해 **공간적 희소성(Spatial Sparsity)**이 매우 높다. 기존 연구들은 RGB 이미지로 사전 학습된 CLIP의 이미지 인코더와 이벤트 인코더 사이의 임베딩 유사도를 최적화하거나, 이벤트 데이터로부터 RGB 프레임을 재구성하여 CLIP에 입력하는 방식을 사용하였다. 하지만 이러한 접근법은 다음과 같은 한계가 있다:

1. **재구성 기반 방식**: 추가적인 네트워크 비용이 발생하며, 재구성 품질 저하 및 오차 누적으로 인해 인식 성능이 떨어진다.
2. **대조 학습(Contrastive Learning) 기반 방식**: 이벤트 데이터의 희소성으로 인해 학습된 이벤트 임베딩들이 서로 지나치게 유사해지는 경향이 있으며, 이는 결과적으로 CLIP의 텍스트 임베딩 공간과의 **시맨틱 미정렬(Semantic Misalignment)** 문제를 야기한다.

따라서 본 연구의 목표는 추가적인 재구성 네트워크 없이, 이벤트 데이터의 희소성 문제를 극복하고 텍스트 임베딩과의 정렬을 강화하여 일반화된 Zero-Shot 인식 성능을 높이는 이벤트 인코더를 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 이벤트 임베딩 공간의 자유도를 제한하여 RGB 임베딩과의 정렬을 강제하는 **Scalar-wise Modulation** 전략과, 데이터 부족 문제를 해결하기 위한 **대규모 합성 데이터 생성 파이프라인**이다.

1. **Scalar-wise Modulation**: 단순한 코사인 유사도 기반의 대조 학습을 넘어, 임베딩의 각 요소(scalar) 수준에서 정렬을 강제함으로써 이벤트 임베딩이 지나치게 유사해지는 현상을 방지하고 텍스트 임베딩과의 정렬을 촉진한다.
2. **대규모 합성 데이터 생성**: 정적인 RGB 이미지에 무작위 아핀 변환(Affine Transformation)을 적용하여 비디오 시퀀스를 만들고, 이를 V2E(Video-to-Event)를 통해 이벤트 데이터로 변환함으로써 대량의 이벤트-RGB 쌍 데이터를 확보하였다.
3. **확장성(Scalability) 입증**: 모델 파라미터 수와 학습 데이터의 양이 증가함에 따라 성능이 지속적으로 향상되는 매력적인 확장성을 확인하였다.

## 📎 Related Works

논문에서는 이벤트 기반 Zero-Shot 학습을 두 가지 그룹으로 분류하여 설명한다.

1. **재구성 기반 접근법(Reconstruction-based approach)**: 이벤트 데이터로부터 그레이스케일 이미지를 복원하여 CLIP 이미지 인코더를 활용하는 방식이다. CLIP의 강력한 RGB 도메인 사전 지식을 활용할 수 있지만, 앞서 언급한 오차 누적 문제로 인해 인식 성능이 낮다.
2. **대조 학습 기반 접근법(Contrastive learning-based approach)**: 이벤트 인코더와 CLIP 이미지 인코더 간의 임베딩 유사도를 최적화하여 지식 증류(Distillation)를 수행하는 방식이다. 그러나 이벤트 데이터의 특성상 임베딩들이 서로 너무 가깝게 배치되어, 텍스트 임베딩과의 정렬에서 자유도가 너무 높아져 실제 Zero-Shot 성능으로 이어지지 않는 한계가 있다.

본 연구는 이러한 기존 방식들과 달리, 합성 데이터를 통해 데이터 부족 문제를 해결하고 Scalar-wise Modulation을 통해 정렬 문제를 직접적으로 해결함으로써 차별성을 갖는다.

## 🛠️ Methodology

### 전체 파이프라인

본 프레임워크의 목표는 사전 학습된 CLIP의 이미지 인코더 $f_{img}(\cdot)$를 대체할 수 있는 이벤트 인코더 $f_{evt}(\cdot)$를 학습시키는 것이다. 학습 과정에서 CLIP 이미지 인코더와 텍스트 인코더 $f_{txt}(\cdot)$는 동결(frozen)된 상태로 유지된다.

### 훈련 목표 및 손실 함수

본 논문은 baseline으로 사용되는 대조 학습 손실 함수에 Scalar-wise Modulation 손실을 추가한 통합 손실 함수를 사용한다.

**1. Baseline: Symmetrized InfoNCE Loss**
이벤트 임베딩 $\hat{x}_{evt}$와 RGB 임베딩 $x_{img}$ 사이의 인스턴스 구별을 위해 다음과 같은 대칭적 InfoNCE 손실을 사용한다.
$$L_{baseline} = \frac{1}{|\{\hat{x}_{evt}\}|} \sum_{\hat{x}_{evt} \in \{\hat{x}_{evt}\}} L_{nce}(\hat{x}_{evt}, \{x_{img}\}) + \frac{1}{|\{x_{img}\}|} \sum_{x_{img} \in \{x_{img}\}} L_{nce}(x_{img}, \{\hat{x}_{evt}\})$$
여기서 $L_{nce}$는 쿼리와 긍정 샘플 간의 유사도는 높이고, 부정 샘플 간의 유사도는 낮추는 일반적인 대조 학습 손실이다.

**2. Scalar-wise Modulation Loss**
이벤트 임베딩들의 과도한 유사성을 억제하고 정밀한 정렬을 위해, RGB 임베딩과의 요소별 차이를 직접 규제하는 손실 함수 $L_{mod}$를 도입한다.
$$L_{mod} = \sum_{\hat{x}_{evt}} \lambda_{\hat{x}_{evt}} \|\hat{x}_{evt} - x_{img}^+\|^2$$
여기서 $\lambda_{\hat{x}_{evt}}$는 가중치 파라미터로, 이벤트 임베딩 $\hat{x}_{evt}$가 다른 부정 샘플들과 얼마나 유사한지를 나타내는 $\lambda_{unf}$ 값을 기반으로 가우시안 분포 함수 $N(\cdot)$를 통해 계산된다. 즉, 임베딩들이 서로 너무 비슷해져서 변별력이 떨어지는 경우 더 강한 페널티를 부여하여 각 요소 수준에서의 정렬을 강제한다.

최종 손실 함수는 다음과 같다:
$$L_{total} = L_{baseline} + L_{mod}$$

### 데이터 합성 절차

학습 데이터 부족을 해결하기 위해 정적 RGB 이미지로부터 다음과 같은 세 가지 모션 패턴을 시뮬레이션하여 이벤트 데이터를 생성한다.

- **Translation**: 이미지를 수평/수직으로 선형 이동시켜 카메라가 직선으로 움직이는 효과를 재현한다.
- **Scaling**: 이미지 크기를 선형적으로 확대/축소하여 줌인/줌아웃 효과를 재현한다.
- **Rotation**: 이미지를 중심축 기준으로 회전시켜 카메라 또는 물체의 회전 움직임을 재현한다.
이렇게 생성된 시퀀스는 V2E 모델을 통해 이벤트 데이터로 변환된다.

## 📊 Results

### 실험 설정

- **데이터셋**:
  - 객체 인식: N-ImageNet, N-Caltech101, CIFAR-10-DVS, N-MINIST.
  - 행동 인식: HARDVS, DailyAction, PAF, Bullying10K, UCF101-DVS.
- **지표**: 정확도(Accuracy, %).
- **백본**: ViT-B/16 (CLIP 사전 학습 가중치로 초기화).

### 주요 결과

1. **객체 인식 성능**: N-ImageNet 데이터셋에서 ViT-B/16 기반 모델이 **47.84%**의 Zero-Shot 정확도를 달성하였다. 이는 기존의 Zero-Shot 프레임워크들보다 월등히 높은 수치이며, 일부 데이터셋 특화(dataset-specific) 모델과 경쟁 가능한 수준이다.
2. **행동 인식 성능**: 행동 인식에서는 단순한 전방 패스 외에 시간적 역학을 추론하기 위한 2-layer ViT를 추가로 사용하여 평가하였다. 특히 가장 복잡한 데이터셋인 HARDVS에서 기존 방식들보다 유의미하게 높은 성능을 보였다.
3. **모델 확장성**: CLIP 이미지 인코더의 크기를 ViT-B에서 ViT-bigE(4.4B 파라미터)까지 확장했을 때, N-ImageNet 정확도가 **47.84%에서 62.28%까지** 선형적으로 증가하는 것을 확인하였다.
4. **데이터 확장성**: 사용되는 합성 데이터의 비율을 20%에서 100%로 늘릴수록 성능이 지속적으로 향상되었다.

## 🧠 Insights & Discussion

**강점 및 통찰**

- 본 연구는 단순히 유사도를 높이는 대조 학습의 한계를 이론적/실험적으로 분석하고, 임베딩 공간의 자유도를 제한하는 Scalar-wise Modulation이라는 구체적인 해결책을 제시하였다.
- 정적 이미지로부터 이벤트를 합성하는 단순하면서도 효율적인 파이프라인을 통해 1.8억 장의 대규모 데이터셋을 구축하여, 이벤트 기반 모델의 고질적인 문제인 데이터 부족 문제를 효과적으로 해결하였다.
- 이벤트 인코더를 LLaVA와 같은 시각-언어 모델(VLM)의 이미지 인코더 자리에 교체 삽입하여 이벤트 데이터만으로도 적절한 캡셔닝이 가능함을 보임으로써, 제안된 인코더가 강력한 시맨틱 이해 능력을 갖추었음을 입증하였다.

**한계 및 논의사항**

- CIFAR-10-DVS나 N-MINIST와 같이 해상도가 매우 낮은 데이터셋에서는 여전히 성능이 낮게 나타난다. 이는 제안된 방식이 고해상도 데이터의 시맨틱 정렬에는 강점이 있으나, 저해상도 데이터의 특징 추출에는 한계가 있음을 시사한다.
- 또한, 현재의 성과에도 불구하고 이벤트 카메라를 위한 범용적인 파운데이션 모델(Generalist Foundation Model)을 구축하기까지는 여전히 간극이 존재하며, 이는 향후 연구 과제로 남아있다.

## 📌 TL;DR

본 논문은 이벤트 카메라 데이터의 공간적 희소성으로 인해 발생하는 시맨틱 미정렬 문제를 해결하여 **Zero-Shot Object Recognition** 성능을 극대화한 **EZSR** 프레임워크를 제안한다. 핵심 기여는 임베딩의 각 요소 수준에서 정렬을 강제하는 **Scalar-wise Modulation** 전략과, 정적 이미지로부터 대량의 이벤트를 생성하는 **합성 데이터 파이프라인**이다. 실험 결과, 제안 방법은 기존 Zero-Shot 방식들을 압도하며 모델 크기와 데이터 양에 따라 성능이 향상되는 강력한 확장성을 보였으며, 이는 향후 이벤트 기반 시각-언어 파운데이션 모델 구축을 위한 핵심 기반 기술이 될 가능성이 높다.
