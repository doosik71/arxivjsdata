# Generative Zero-shot Network Quantization

Xiangyu He, Qinghao Hu, Peisong Wang, Jian Cheng (2021)

## 🧩 Problem to Solve

본 논문은 딥러닝 모델의 추론 속도를 높이고 메모리 사용량을 줄이기 위한 핵심 기술인 네트워크 양자화(Network Quantization) 과정에서 발생하는 데이터 의존성 문제를 해결하고자 한다. 일반적으로 활성화 함수(Activation)의 양자화를 위해서는 데이터의 동적 범위를 파악하기 위한 보정 데이터셋(Calibration set)이 필수적이며, 정밀도 하락을 막기 위해 레이블이 있는 데이터로 재학습(Fine-tuning)을 수행하는 Training-aware quantization 방식이 사용된다.

그러나 실제 산업 현장에서는 의료 데이터나 개인 정보와 같은 민감한 정보로 인해 원본 학습 데이터셋에 접근할 수 없는 경우가 많다. 이에 따라 원본 데이터 없이 모델을 양자화하는 Zero-shot/Data-free quantization의 필요성이 대두되었으나, 기존의 데이터 프리 방식들은 단순한 가우시안 노이즈를 사용하거나 BN(Batch Normalization) 통계치만을 단순하게 매칭하는 방식에 의존하여 생성된 이미지의 품질이 낮고, 결과적으로 양자화 후의 성능 하락이 심하다는 한계가 있다. 따라서 본 논문의 목표는 원본 데이터 없이도 BN 통계치를 활용하여 실제 데이터와 유사한 고품질의 합성 이미지를 생성하고, 이를 통해 양자화 성능을 극대화하는 GZNQ(Generative Zero-shot Network Quantization) 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 합성 이미지의 최적화 과정을 BN 통계치의 분포를 맞추는 '생성 모델링(Generative Modeling)' 관점에서 재해석하고 설계한 것이다. 단순히 특정 값에 매칭하는 것이 아니라, 입력 데이터의 변환(Random augmentation)을 통해 BN 통계치가 확률 변수로서 실제 데이터의 분포를 따르도록 유도한다.

또한, 합성 이미지의 품질과 다양성을 높이기 위해 두 가지 추가적인 설계를 도입하였다. 첫째는 다양한 압축 모델(양자화, 가지치기, 저차원 분해 모델 등)의 앙상블을 통해 의사 레이블(Pseudo-label)을 생성하여 합성 이미지가 네트워크 입장에서 '어려운(Hard)' 샘플이 되도록 유도하는 Prior Regularization을 도입한 것이다. 둘째는 마지막 컨볼루션 레이어에 샘플별 채널 게이팅(Channel Gating)을 적용하여, 각 샘플이 더 얽힌 표현(Entangled representations)을 학습하도록 유도함으로써 생성 이미지의 세밀한 품질을 개선한 것이다.

## 📎 Related Works

기존의 모델 압축 연구는 크게 세 가지 방향으로 나뉜다. Training-aware quantization은 레이블 기반의 미세 조정을 통해 높은 성능을 내지만 방대한 데이터가 필요하다. Label-free quantization은 레이블이 없는 데이터만을 사용하지만 여전히 일부의 실제 데이터에 의존한다. Zero-shot quantization은 모델 내부에 저장된 메타데이터나 BN 통계치를 사용하여 데이터를 복원하려 한다.

특히 최근 연구들은 BN 레이어의 running mean($\mu$)과 variance($\sigma^2$)가 데이터의 사전 분포(Prior distribution)를 캡처하고 있다는 점에 주목하여, 이를 매칭하는 손실 함수를 사용해왔다. 하지만 이러한 방식들이 왜 의미 있는 데이터를 생성하는지에 대한 이론적 해석이 부족했으며, 단순한 거리 기반 매칭은 표본 추출의 부족으로 인해 생성된 이미지의 품질이 낮아 low-bit 양자화에서 성능 저하가 발생하는 한계가 있었다.

## 🛠️ Methodology

### 전체 파이프라인

GZNQ는 크게 두 단계로 구성된다. 첫 번째 단계에서는 생성 모델링을 통해 고품질의 합성 데이터셋을 생성하고, 두 번째 단계에서는 이 합성 데이터를 보정 데이터셋으로 사용하여 양자화를 수행하거나 지식 증류(Knowledge Distillation) 기반의 미세 조정을 수행한다.

### 1. 분포 매칭 (Distribution Matching)

저자들은 BN 통계치의 매칭 과정을 최대 우도 추정(MLE) 기반의 생성 모델링으로 공식화한다. 입력 이미지 $I$에 랜덤한 변환 $z$ (flipping, jitter, shift 등)를 적용한 함수 $f_I(z)$를 정의하고, 이를 통해 계산된 배치 통계치 $\mu_{batch}, \sigma^2_{batch}$가 실제 데이터의 통계치 $\mu, \sigma^2$의 분포와 일치하도록 최적화한다.

중심극한정리(CLT)에 의해 배치 크기 $N$이 충분히 클 때, 배치 평균과 분산은 정규분포를 따른다고 가정하며, 이에 기반한 분포 매칭 손실 함수 $L^{DM}$은 다음과 같이 정의된다.

$$L^{DM} = \frac{1}{2N\sigma^2}(\mu_{batch} - \mu)^2 + \frac{1}{2N Var(F_l - \mu)}(\sigma^2_{batch} - \sigma^2)^2 + \text{term } I + \frac{1}{2}\ln Var(F_l - \mu)$$

이 식은 기존의 단순 MSE 기반 BN 매칭보다 통계적 근거가 명확하며, $\mu$와 $\sigma^2$ 사이의 상관관계를 더 잘 반영한다.

### 2. 의사 레이블 생성기 (Pseudo-label Generator)

단순한 BN 매칭만으로는 이미지의 다양성이 부족하고 그라디언트 분산이 커질 수 있다. 이를 보완하기 위해 저자들은 여러 종류의 압축 모델(Quantized, Pruned, Low-rank models)의 앙상블을 구축한다. 압축 모델들은 일반적으로 복잡한 샘플에 취약하므로, 원본 모델과 압축 모델 앙상블 간의 KL divergence를 최대화함으로써 생성된 이미지가 모든 압축 모델에게 '어려운' 샘플이 되도록 유도하는 $L^{KL}$ 손실을 추가한다.

$$L^{KL} = KL(\phi_\omega(f_I(z)) || \frac{1}{M}\sum_{i=1}^M \phi_{\hat{\omega}_i}(f_I(z)))$$

여기서 $\phi_\omega$는 원본 모델의 확률 분포이며, $\phi_{\hat{\omega}_i}$는 $i$번째 압축 모델의 분포이다.

### 3. 채널 게이팅 (Channel Gating)

합성 이미지의 의미적 표현력을 높이기 위해 마지막 컨볼루션 레이어에 샘플별 채널 프루닝을 적용한다. Gumbel-Softmax trick을 사용하여 이진 게이팅 마스크 $G_{i,j}$를 생성하며, 이를 통해 각 이미지 샘플이 네트워크의 서로 다른 채널 조합을 통해 표현되도록 유도한다. 이는 모델의 중복성을 줄이고 더 구체적인 시각적 특징을 생성하도록 돕는다.

## 📊 Results

### 실험 설정

- **데이터셋:** CIFAR-10, CIFAR-100, ImageNet.
- **모델:** ResNet-18, ResNet-20, ResNet-44, ResNet-50, WRN.
- **측정 지표:** Top-1 Accuracy, Inception Score (IS).
- **비교 대상:** ZeroQ, GDFQ, Knowledge Within, DFNQ 등 기존의 Zero-shot 양자화 방법론.

### 정량적 결과

- **CIFAR 데이터셋:** 4-bit 가중치 및 활성화 함수 양자화 설정에서 GZNQ는 기존 SOTA 방법론들을 일관되게 상회한다. 특히 ResNet-20 (CIFAR-10)에서 ZeroQ의 정확도가 79.30%인 반면, GZNQ는 89.06%를 기록하며 매우 적은 성능 하락을 보였다.
- **ImageNet 데이터셋:** ResNet-50 모델에서 4-bit 양자화 시 72.7%의 정확도를 달성하였으며, 6-bit 설정에서는 75.5%를 기록하여 기존의 OCS, ACIQ 등의 방법보다 우수한 성능을 보였다.
- **보정 데이터셋의 영향:** 합성 데이터셋을 보정셋으로 사용했을 때의 성능이 실제 CIFAR-10 데이터를 사용했을 때와 매우 유사한 수준까지 도달함을 확인하였다 (Table 5).

### 정성적 결과 및 이미지 품질

Inception Score (IS) 측정 결과, GZNQ는 224x224 해상도에서 84.7 $\pm$ 2.8를 기록하여, DeepInversion(60.6)이나 ZeroQ(2.8)보다 압도적으로 높은 시각적 충실도를 보여주었다. 생성된 이미지들은 실제 이미지와 완전히 동일하지는 않지만, 카툰 스타일의 명확한 객체 형태를 띠고 있어 네트워크가 내부적으로 어떤 특징을 기대하는지를 잘 보여준다.

## 🧠 Insights & Discussion

본 논문은 단순한 휴리스틱에 의존하던 BN 매칭 방식을 생성 모델링이라는 수학적 프레임워크로 정립함으로써, 데이터 프리 환경에서도 고품질의 보정 데이터를 생성할 수 있음을 입증하였다. 특히 압축 모델의 앙상블을 통해 'Hard sample'을 유도하고, 채널 게이팅으로 다양성을 확보한 점이 성능 향상의 주요 요인으로 분석된다.

다만, 생성된 이미지가 실제 사진보다는 카툰 형태에 가깝다는 점은 여전히 생성 모델의 한계가 있음을 시사한다. 또한, BN 통계치에 전적으로 의존하기 때문에, BN 레이어가 없는 모델이나 BN 통계치가 오염된 모델에서는 적용이 어려울 수 있다는 가정이 전제되어 있다. 하지만 결과적으로 원본 데이터 없이도 실용적인 수준의 양자화가 가능하다는 점을 보여주었다는 점에서 학술적, 산업적 가치가 높다.

## 📌 TL;DR

본 논문은 원본 데이터 없이 모델을 양자화하기 위해, BN 통계치를 매칭하는 과정을 생성 모델링 관점에서 재구성한 GZNQ를 제안한다. 분포 매칭 손실($L^{DM}$), 압축 모델 앙상블 기반의 의사 레이블 생성, 그리고 채널 게이팅 기법을 통해 고품질의 합성 이미지를 생성하고, 이를 보정셋으로 사용하여 4-bit/6-bit 양자화에서 기존 Zero-shot 방법론보다 월등한 정확도를 달성하였다. 이 연구는 데이터 프라이버시 문제가 중요한 환경에서 고성능 모델 압축을 가능하게 하는 중요한 발판이 될 것으로 보인다.
