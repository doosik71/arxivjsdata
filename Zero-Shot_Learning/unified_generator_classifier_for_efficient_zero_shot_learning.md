# UNIFIED GENERATOR-CLASSIFIER FOR EFFICIENT ZERO-SHOT LEARNING

Ayyappa Kumar Pambala, Titir Dutta, Soma Biswas (2019)

## 🧩 Problem to Solve

본 논문은 Zero-Shot Learning (ZSL) 및 Generalized Zero-Shot Learning (GZSL)에서 발생하는 효율성 및 성능 문제를 해결하고자 한다. ZSL의 핵심 목표는 훈련 단계에서 본 적 없는 unseen classes에 속하는 객체를 분류하는 것이다. 특히 GZSL은 테스트 단계에서 입력 데이터가 seen class와 unseen class 모두에 속할 수 있다는 점에서 더욱 도전적인 과제이다.

기존의 접근 방식은 크게 두 가지로 나뉜다. 첫째, 시각적 공간과 시맨틱 공간 사이의 매핑을 찾는 Embedding 기반 방식은 구조적으로 우아하지만, 생성 기반 방식에 비해 성능이 떨어진다는 한계가 있다. 둘째, GAN이나 VAE와 같은 Generative 모델을 사용하여 unseen class의 가상 특징(pseudo-features)을 생성하고 이를 통해 분류기를 학습시키는 방식은 높은 성능을 보이지만, 새로운 클래스가 추가될 때마다 분류기를 매번 다시 학습시켜야 하는 비효율성이 존재한다.

따라서 본 논문은 생성 모델의 강력한 표현력과 임베딩 방식의 효율성을 결합하여, 새로운 클래스에 대해 별도의 분류기 재학습이 필요 없는 통합 프레임워크인 GenClass를 제안한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 생성기(Generator)와 분류기(Classifier)를 하나의 통합된 구조로 설계하여 end-to-end로 학습시키는 것이다.

주요 기여 사항은 다음과 같다.

1. **통합 프레임워크(GenClass) 제안**: 생성 과정과 분류 과정을 통합하여 end-to-end로 학습함으로써, 새로운 객체 카테고리에 대해 분류기를 다시 학습시킬 필요를 제거하였다.
2. **판별적 특징 생성**: 분류기가 생성기의 학습 과정에 직접 참여함으로써, 단순히 실제 데이터와 유사한 샘플을 만드는 것을 넘어 분류에 최적화된 더욱 판별적인(discriminative) 특징을 생성하도록 유도하였다.
3. **범용성 입증**: 객체 분류뿐만 아니라 행동 인식(Action Classification) 도메인에서도 수정 없이 최첨단 성능(state-of-the-art)을 달성함으로써 제안 방법론의 일반화 능력을 증명하였다.

## 📎 Related Works

논문에서는 ZSL 접근 방식을 두 가지 범주로 설명한다.

- **Embedding-based approaches**: 시각적 공간 $\to$ 시맨틱 공간, 시맨틱 $\to$ 시각적, 또는 공통 중간 공간으로의 매핑을 학습한다. 하지만 이러한 방식은 생성 기반 방식에 비해 성능이 낮게 나타나는 경향이 있다.
- **Generative approaches**: GAN이나 VAE를 통해 unseen class의 가상 데이터를 생성하고, 이를 기반으로 supervised classification 문제로 변환하여 해결한다. 이 방식은 성능은 뛰어나지만, 새로운 클래스가 나타날 때마다 분류기를 재학습해야 한다는 치명적인 단점이 있다.

GenClass는 이러한 두 방식의 장점을 취하여, 생성기의 출력물을 즉시 분류에 활용할 수 있는 통합 구조를 통해 재학습 문제를 해결하고 성능을 유지하고자 한다.

## 🛠️ Methodology

### 전체 시스템 구조

GenClass는 WGAN(Wasserstein GAN)을 기본 생성 네트워크로 채택하며, 여기에 통합 분류기 $C_I$를 결합한 형태이다. 시스템은 생성기 $G$, 판별기 $D$, 그리고 통합 분류기 $C_I$의 세 가지 주요 모듈로 구성된다.

### 훈련 목표 및 손실 함수

전체 프레임워크는 다음과 같은 통합 손실 함수 $L_{GC}$를 최소화하는 방향으로 학습된다.
$$L_{GC} = L_{WGAN} + L_{C_I}$$

#### 1. WGAN 손실 함수 ($L_{WGAN}$)

생성기 $G$가 실제 데이터의 분포를 학습하도록 하며, Gradient Penalty (GP)를 통해 학습의 안정성을 확보한다.
$$L_{WGAN} = E[D(x|a)] - E[D(\tilde{x}|a)] - \lambda(GP)$$
여기서 $x$는 실제 샘플, $\tilde{x} = G(z|a)$는 생성된 가짜 샘플, $a$는 클래스 속성(attribute) 벡터이다.

#### 2. 통합 분류기 손실 함수 ($L_{C_I}$)

통합 분류기 $C_I: X \times X \to \{0, 1\}$는 두 개의 특징 벡터(실제 샘플 하나, 가짜 샘플 하나)를 입력으로 받아, 두 샘플이 동일한 클래스에 속하는지 여부를 판별한다.

- **Seen class 손실 ($L_s$)**: 실제 샘플 $x_i$와 생성된 샘플 $\tilde{x}_j$의 쌍을 이용하여 학습한다.
$$L_s = E_{(x_i, \tilde{x}_j) \sim (D_{similar}^{seen} \cup D_{dissimilar}^{seen})} (C_I(x_i, \tilde{x}_j) - 1(y_i, y_j))^2$$
- **Unseen class 손실 ($L_u$)**: unseen class에 대한 편향을 줄이기 위해, 오직 생성된 가짜 샘플들끼리의 쌍을 이용하여 학습한다.
$$L_u = E_{(\tilde{x}_i, \tilde{x}_j) \sim (D_{similar}^{unseen} \cup D_{dissimilar}^{unseen})} (C_I(\tilde{x}_i, \tilde{x}_j) - 1(y_i, y_j))^2$$
최종 분류기 손실은 $L_{C_I} = L_s + \gamma L_u$로 정의되며, $\gamma$는 가중치를 조절하는 하이퍼파라미터이다.

### 추론 및 테스트 절차

테스트 단계에서는 추가적인 학습 없이 다음과 같은 절차를 따른다.

1. 각 클래스의 속성 $a^{(i)}$를 이용해 $n_g$개의 샘플을 생성하고, 이들의 평균 벡터 $m_i$를 계산하여 해당 클래스의 프로토타입(prototype)으로 사용한다.
$$m_i = \frac{1}{n_g} \sum_{k=1}^{n_g} G(z_k | a^{(i)})$$
2. 테스트 쿼리 $x_t$가 입력되면, 모든 클래스의 프로토타입 $m_i$와 쌍을 이루어 $C_I$에 입력한다.
3. $C_I$의 출력값이 가장 높은 클래스를 최종 예측 결과로 선택한다.
$$\hat{y} = \text{argmax}_{1 \le i \le |C|} C_I(m_i, x_t)$$

## 📊 Results

### 실험 설정

- **데이터셋**: 객체 분류를 위해 AWA, CUB, SUN 데이터셋을 사용하였으며, 행동 인식을 위해 UCF101, HMDB51 데이터셋을 사용하였다.
- **지표**: ZSL에서는 top-1 accuracy를, GZSL에서는 seen/unseen 성능의 조화 평균인 H-factor를 측정하였다.
- **기준선**: DEViSE, ALE, f-CLSWGAN, CVAE 등 기존의 임베딩 및 생성 기반 모델들과 비교하였다.

### 주요 결과

- **객체 분류**: ZSL 시나리오에서 모든 임베딩 기반 방식보다 우수한 성능을 보였으며, 생성 기반 방식과 대등하거나 더 높은 성능을 기록하였다. 특히 GZSL의 H-factor 지표에서 모든 데이터셋에 대해 최첨단 성능을 달성하였다.
- **행동 인식**: 별도의 수정 없이 UCF101과 HMDB51 데이터셋에서도 기존 최신 기법들보다 유의미하게 높은 정확도를 보이며 도메인 일반화 능력을 입증하였다.
- **샘플 효율성**: 기존 생성 모델들이 수백 개의 샘플이 필요했던 것과 달리, GenClass는 20~50개의 샘플만으로도 성능이 포화(saturate)되는 효율성을 보였다.

## 🧠 Insights & Discussion

### 강점 및 분석

본 논문의 가장 큰 강점은 생성기와 분류기를 통합하여 end-to-end로 학습시킨 점이다. 분석 결과, 이렇게 학습된 생성기는 단순히 외형적으로 유사한 샘플을 만드는 것이 아니라, 분류기가 클래스를 더 잘 구분할 수 있도록 하는 '판별적 특징'을 생성하는 법을 배운다. 이는 Table 3의 실험을 통해 입증되었는데, GenClass로 생성한 특징을 기존의 softmax 분류기에 넣었을 때 성능이 향상되는 결과가 나타났다.

### 한계 및 논의

GZSL 상황에서 unseen 클래스에 대한 성능은 매우 뛰어나지만, seen 클래스에 대한 성능은 일부 최신 기법보다 약간 낮게 나타나는 경향이 있다. 이는 생성된 가짜 데이터에 의존하여 unseen 클래스의 영역을 확장하려다 보니 발생하는 트레이드-오프(trade-off)로 해석될 수 있다.

또한, 본 논문은 WGAN을 기본 모델로 사용하였으나, CVAE로 교체했을 때도 성능 향상이 나타남을 보여줌으로써 제안한 통합 분류 구조($C_I$) 자체가 생성 모델의 종류와 관계없이 효과적임을 시사하였다.

## 📌 TL;DR

본 논문은 ZSL/GZSL을 위해 생성기와 분류기가 통합된 **GenClass** 프레임워크를 제안한다. 이 모델은 새로운 클래스가 추가될 때마다 분류기를 재학습해야 하는 기존 생성 모델의 비효율성을 해결하였으며, end-to-end 학습을 통해 훨씬 적은 수의 가상 샘플만으로도 더 판별적인 특징을 생성하여 높은 정확도를 달성한다. 객체 및 행동 인식 데이터셋 모두에서 SOTA 수준의 성능을 보였으며, 이는 향후 실시간성이나 효율성이 중요한 Zero-Shot 학습 시스템 구축에 중요한 기여를 할 것으로 보인다.
