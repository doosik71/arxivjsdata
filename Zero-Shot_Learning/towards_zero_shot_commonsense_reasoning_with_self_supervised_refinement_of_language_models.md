# Towards Zero-shot Commonsense Reasoning with Self-supervised Refinement of Language Models

Tassilo Klein, Moin Nabi (2021)

## 🧩 Problem to Solve

본 논문은 언어 모델(Language Model, LM)이 상식 추론(Commonsense Reasoning), 특히 대명사 해소(Pronoun Resolution) 작업인 Winograd Schema Challenge(WSC)에서 겪는 한계를 해결하고자 한다.

기존의 많은 연구들은 사전 학습된 언어 모델을 특정 데이터셋으로 미세 조정(Fine-tuning)하여 성능을 높이는 방식을 사용했다. 그러나 이러한 지도 학습 기반의 접근 방식은 모델이 실제 추론 능력을 갖추기보다 데이터셋 내의 통계적 편향이나 인위적인 특징(Statistical Data Artifacts)을 이용해 정답을 맞히는 이른바 'Clever Hans' 효과에 취약하다는 문제가 있다.

따라서 본 연구의 목표는 외부의 레이블링 된 데이터에 의존하는 미세 조정 없이, 자기 지도 학습(Self-supervised Learning) 기반의 정제(Refinement) 과정을 통해 언어 모델의 내재적인 상식 추론 능력을 강화하여 제로샷(Zero-shot) 환경에서 성능을 향상시키는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 언어 모델이 특정 개념 관계에 대한 다양한 언어적 변형(Linguistic Perturbations)에 강건해지도록 학습시킴으로써, 보다 일반화된 '개념화(Conceptualization)' 능력을 갖추게 하는 것이다.

단순히 정답을 맞히도록 학습시키는 것이 아니라, 입력 문장에 의미론적 변형을 가한 결과물(Perturbed representation)을 모델이 예측하도록 유도함으로써 손실 함수 지형(Loss landscape)을 정제하고, 이를 통해 모델이 상식적 개념의 본질적인 관계를 더 잘 파악하도록 설계하였다.

## 📎 Related Works

기존의 상식 추론 접근 방식은 크게 두 가지 방향으로 나뉜다. 첫째는 대규모 지도 학습 데이터셋(예: Winogrande)을 사용하여 미세 조정하는 방식이고, 둘째는 GPT-3와 같은 초거대 언어 모델의 방대한 파라미터를 이용해 제로샷 능력을 확보하는 방식이다.

하지만 전자는 앞서 언급한 'Clever Hans' 효과로 인해 실제 추론 능력을 검증하기 어렵고, 후자는 모델의 크기가 너무 커서 자원 소모가 극심하다는 한계가 있다. 본 논문은 표준적인 사전 학습 모델(BERT, RoBERTa 등)을 입력으로 사용하되, 레이블 없는 자기 지도 학습 기반의 정제 단계를 추가함으로써 이 두 가지 접근 방식의 단점을 극복하고 현실적인 대안을 제시한다.

## 🛠️ Methodology

### 전체 파이프라인

본 방법론은 사전 학습된 언어 모델(init-LM)을 가져와 정제(Refinement) 단계를 거친 후, 추가적인 학습 없이 제로샷으로 WSC 작업을 수행하는 구조이다. 정제 과정에서는 문장에 특정 변형을 가하는 '변형 토큰(Perturbation Token)'을 도입하여 모델이 변형된 임베딩을 생성하도록 학습시킨다.

### 변형 생성 프레임워크 (Perturbation Generation)

모델의 입력 앞에 `[CLS]` 토큰 다음에 변형 토큰 $p \in P$를 추가한다. 여기서 $P$는 시제 변경(`[TENSE]`), 동의어 교체(`[SYNONYM]`), 성별 변경(`[GENDER]`) 등의 변형 집합과 아무런 변경을 하지 않는 `[IDENTICAL]`을 포함한다.
모델은 원본 문장 $x$와 변형 토큰 $[k]$가 주어졌을 때, 실제로 변형된 문장 $x^{[k]}$의 임베딩 $\hat{x}^{[k]}$를 예측하도록 훈련된다.

### 자기 지도 정제 (Self-supervised Refinement)

모델의 최적화를 위해 세 가지 손실 함수를 결합하여 사용하며, 전체 목적 함수는 다음과 같다.

$$\min_{\theta_1, \theta_2} L_R(f_{\theta_1}) + L_C(f_{\theta_1}) + L_D(f_{\theta_1}, q_{\theta_2})$$

여기서 $f_{\theta_1}$은 언어 모델이며, $q_{\theta_2}$는 임베딩의 변형 타입을 판별하는 MLP 기반의 판별자(Discriminator)이다.

1. **Reconstruction Loss ($L_R$):** 모델이 예측한 변형 임베딩 $\hat{x}_i^{[k]}$가 실제 변형된 문장의 임베딩 $x_i^{[k]}$와 가까워지도록 유도한다.
   $$L_R = -\alpha \sum_{i=1}^{N} \sum_{k \in P} \phi(x_i^{[k]}, \hat{x}_i^{[k]})$$
2. **Contrastive Loss ($L_C$):** 동일한 변형 타입이라 하더라도 서로 다른 샘플 간의 임베딩은 서로 멀어지게 하여, 모델이 모든 입력을 하나의 단일한 표현으로 뭉뚱그리는 붕괴(Collapse) 현상을 방지한다.
   $$L_C = \beta \sum_{i,j: i \neq j} \sum_{k \in P} \phi(\hat{x}_i^{[k]}, \hat{x}_j^{[k]})$$
3. **Diversity Loss ($L_D$):** 동일한 샘플 내에서 서로 다른 변형 토큰에 의해 생성된 임베딩들이 서로 구분 가능하도록 하여, 변형 간의 변별력을 확보한다. 이는 판별자 $q$를 이용한 교차 엔트로피 손실로 계산된다.
   $$L_D = -\gamma \sum_{i=1}^{N} \sum_{k \in P} \log \frac{q(k|\hat{x}_i^{[k]})}{\sum_{t \in P, t \neq k} q(t|\hat{x}_i^{[k]})}$$

여기서 유사도 측정 함수 $\phi$로는 BERTscore의 변형 버전을 사용하며, 변형이 주로 국소적으로 일어난다는 점을 고려하여 슬라이딩 윈도우(Sliding window) 메커니즘을 적용해 토큰 매칭 범위를 제한하였다.

### 제로샷 대명사 해소 (Zero-shot Pronoun Disambiguation)

정제가 완료된 모델은 Masked Token Prediction 방식을 통해 추론을 수행한다. 대명사 위치를 `[MASK]`로 치환하고, 후보 단어들 중 모델이 예측하는 확률 $p(c|s)$가 더 높은 단어를 최종 답안으로 선택한다.

## 📊 Results

### 실험 설정

- **데이터셋:** 285개의 샘플 문장과 샘플당 최대 10개의 변형이 포함된 enhanced-WSC 코퍼스를 사용하였다.
- **모델:** BERT-large-uncased 및 RoBERTa-large를 사용하였다.
- **평가 지표:** 9개의 서로 다른 벤치마크(DPR, GAP, KnowRef, PDP-60, WSC-273, WinoGender, WinoGrande, WinoBias 등)에서 정확도(Accuracy)를 측정하였다.

### 주요 결과

실험 결과, RoBERTa 모델에서 매우 일관적이고 유의미한 성능 향상이 나타났다.

- **평균 성능 향상:** BERT는 약 $+0.8\%$, RoBERTa는 약 $+4.5\%$의 정확도 상승을 보였다.
- **벤치마크별 상세:**
  - DPR에서 RoBERTa는 $+6.56\%$의 큰 폭으로 상승하였다.
  - WinoGender에서도 RoBERTa는 $+7.6\%$ 향상된 결과를 보였다.
  - 반면, BERT의 경우 PDP-60($-1.67\%$)이나 WSC-273($-1.1\%$)과 같은 일부 벤치마크에서는 오히려 성능이 소폭 하락하는 모습이 관찰되었다.

### 절제 연구 (Ablation Study)

세 가지 손실 함수($L_R, L_C, L_D$)의 기여도를 분석한 결과, 모든 손실 함수가 함께 작용할 때 최적의 성능이 도출되었다. 특히 $L_C$와 $L_D$는 서로 경쟁적인 관계에 있어, 이들 사이의 균형이 깨질 경우 성능이 크게 하락하는 것이 확인되었다. 이는 제안된 방법론의 평형 상태(Equilibrium)가 제로샷 추론 능력 향상에 필수적임을 시사한다.

## 🧠 Insights & Discussion

본 연구는 사전 학습된 언어 모델에 단순한 자기 지도 학습 정제 과정을 추가하는 것만으로도 제로샷 상식 추론 능력을 향상시킬 수 있음을 입증하였다. 특히 지도 학습 기반의 미세 조정이 유발하는 데이터 편향 문제를 회피하면서도, 모델의 강건성(Robustness)을 높여 일반화 능력을 개선했다는 점이 강점이다.

다만, BERT보다 RoBERTa에서 훨씬 더 뚜렷한 성능 향상이 나타난 점은 모델의 초기 사전 학습 상태나 구조에 따라 정제 효과가 다르게 나타날 수 있음을 의미한다. 또한, 사용된 변형 데이터셋의 규모가 상대적으로 작다는 한계가 있으며, 논문에서도 향후 연구를 통해 샘플별 특성에 맞는 정교한 변형 데이터를 생성하는 것이 성능을 더욱 끌어올리는 핵심이 될 것이라고 언급하고 있다.

비판적으로 해석하자면, 본 방법론은 여전히 WSC라는 특정 도메인의 변형 데이터를 학습에 사용하므로, 완전한 의미의 '태스크 불가지론적(Task-agnostic)' 접근인지에 대해서는 추가적인 논의가 필요할 수 있다.

## 📌 TL;DR

본 논문은 언어 모델이 상식 추론 시 겪는 데이터 편향 문제를 해결하기 위해, 언어적 변형(Perturbation)을 활용한 **자기 지도 정제(Self-supervised Refinement)** 프레임워크를 제안한다. 재구성($L_R$), 대조($L_C$), 다양성($L_D$) 손실 함수를 통해 모델이 개념의 본질적인 관계를 학습하도록 하여, 추가적인 미세 조정 없이도 **제로샷 환경에서 상식 추론 성능을 향상**시켰다. 이 연구는 거대 모델을 사용하지 않고도 표준 LM의 내부 표현을 정제함으로써 상식 추론 능력을 강화할 수 있는 실질적인 방향성을 제시한다.
