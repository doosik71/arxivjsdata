# An agent-based intelligent environmental monitoring system

Ioannis N. Athanasiadis and Pericles A. Mitkas

## 🧩 Problem to Solve

급변하는 환경 변화에 대응하여 대기질과 같은 환경 요소를 지속적으로 감시하고 실시간으로 의사결정을 내릴 필요가 있습니다. 기존 환경 정보 시스템(EIS)은 종종 인간 전문가의 개입이나 오프라인 분석에 의존하여, 정확하고 시기적절한 정보 제공에 한계가 있었습니다. 특히, 대기질 모니터링 및 경고 시스템에 데이터 기반의 맞춤형 지능을 통합하여 이러한 한계를 극복하는 것이 주요 연구 문제입니다.

## ✨ Key Contributions

* **다중 에이전트 시스템 개발:** 대기질 속성 모니터링 및 평가를 위한 다중 에이전트 시스템인 O$_{3}$RTAA를 개발했습니다.
* **데이터 마이닝 기반 지능 통합:** 데이터 마이닝 기법을 활용하여 에이전트에 데이터 기반의 맞춤형 지능을 부여했습니다.
* **고급 기능 구현:** 측정값 검증, 누락된 측정값 추정, 맞춤형 경보 식별 등 고급 기능을 포함하며, 데이터 모니터링, 저장 및 접근과 같은 기본적인 작업도 수행합니다.
* **계층적 아키텍처 및 온톨로지 설계:** 시스템의 계층적 아키텍처와 도메인 온톨로지를 제시하고, 에이전트 간 상호작용 및 통신 방식을 명확히 했습니다.
* **실제 사례 적용 및 검증:** 스페인의 실제 기상 관측소 데이터를 사용하여 시스템을 배포하고 높은 정확도로 성능을 입증했습니다.

## 📎 Related Works

* **환경 정보 시스템(EIS):** 환경 모니터링, 데이터 저장, 재난 대응, 보고, 계획, 시뮬레이션 및 의사결정을 포함하는 광범위한 시스템군입니다.
* **기존 대기질 운영 센터(AQOC):** 런던 대기질 네트워크, 텍사스 천연자원 보존 위원회와 같이 대기 변수를 모니터링하고 분석 결과를 게시하지만, 실시간 의사결정은 주로 인간 전문가에 의해 이루어지거나 수학적 모델이 오프라인 연구에 사용됩니다.
* **다른 환경 모니터링 정보 시스템(EMIS):**
  * **New Zealand Distributed Information System (NZDIS):** 분산 환경 정보 시스템 통합을 위한 에이전트 기반 아키텍처.
  * **InfoSleuth 프로젝트:** 분산 환경 데이터 클러스터 질의를 위한 에이전트 사용.
  * **Java Agents for Monitoring and Management:** 모니터링 네트워크에서 다양한 관리 작업을 수행하는 소프트웨어 에이전트.
  * **Knowledge-based Environmental Data Analysis Assistant:** 사용자에게 오프라인 분석 패키지 인터페이스 제공.
  * **DNEMO:** 실시간 대기질 관리를 위한 다중 에이전트 시스템.
* 본 논문의 시스템은 기존 시스템들이 도메인 지식, 전문가 시스템, 사례 기반 추론 등을 활용하는 것과 달리, **데이터 마이닝 기술을 활용하여 맞춤형 지능을 추가한다**는 점에서 차별화됩니다.

## 🛠️ Methodology

본 시스템은 소프트웨어 에이전트 패러다임과 데이터 마이닝 기반 지식 발견 로드맵을 결합하여 지능형 환경 소프트웨어 애플리케이션을 제공합니다.

1. **시스템 목표 및 계층 구조:**
    * O$_{3}$RTAA는 단일 기상 관측소에서 측정된 오염 물질 분포를 포함한 대기질 데이터를 모니터링합니다.
    * 시스템 목표는 세 가지 계층으로 구성됩니다:
        * **기여 계층 (Contribution Layer):** 전처리 활동을 담당합니다. 진단 에이전트(Diagnosis Agents)가 측정값 캡션, 데이터 검증, 오작동 식별 및 복구, 전처리된 측정값 전달을 수행합니다.
        * **관리 계층 (Management Layer):** 전처리된 측정값 조작 및 경보 식별을 담당합니다. 경보 에이전트(Alarm Agent)와 데이터베이스 에이전트(Database Agent)가 법적 임계값에 따른 '공식 경보'와 사용자 요구 사항에 따른 '맞춤 경보'를 식별합니다.
        * **배포 계층 (Distribution Layer):** 식별된 경보를 관심 당사자의 프로필에 따라 적절한 수신자에게 배포합니다. 배포 에이전트(Distribution Agent)가 이메일, SMS 등 지정된 방식으로 경보를 전달합니다.

2. **에이전트 유형별 역할:**
    * **진단 에이전트(Diagnosis Agents):** 각 센서에서 특정 대기질 속성(SO$_{2}$, O$_{3}$, NO, NO$_{2}$, NO$_{X}$, VEL, DIR, TEM, HR, RAD, PRE)을 모니터링합니다. 수신 측정값 유효성 검사(IMV) 및 센서 고장 시 누락값 추정(MVE)을 수행합니다. IMV 및 MVE는 과거 데이터에 대한 데이터 마이닝을 통해 추출된 추론 엔진(결정 모델)을 활용합니다.
    * **경보 에이전트(Alarm Agent):** 진단 에이전트로부터 유효한 측정값을 받아 튜플을 완성하고, 데이터베이스 에이전트에 저장 요청을 보냅니다. 공식 경보 트리거링(FAT) 엔진과 맞춤 경보 식별(ICA) 엔진을 통해 경보를 식별하고 배포 에이전트에 전달합니다. ICA는 데이터 마이닝 기반 결정 모델을 사용합니다.
    * **데이터베이스 에이전트(Database Agent):** 경보 에이전트로부터 수신된 측정값 튜플을 환경 데이터베이스에 저장합니다.
    * **배포 에이전트(Distribution Agent):** 경보 에이전트가 발생시킨 경보를 사용자 프로필에 따라 적절한 수신자와 알림 매체(이메일, SMS)를 선택하여 전달합니다.

3. **의사결정 모델 및 데이터 마이닝:**
    * IMV, MVE, ICA 엔진은 데이터 마이닝 기법으로 생성된 귀납적 결정 모델(데이터 기반 지식)을 사용합니다. FAT 엔진은 연역적 결정 모델을 사용합니다.
    * **Agent Academy**의 Data Miner 컴포넌트(WEKA 확장)를 사용하여 과거 환경 데이터에서 예측 모델을 추출합니다.
    * 추출된 모델은 PMML(Predictive Model Markup Language) 형식으로 표현되며, JESS 규칙으로 변환되어 소프트웨어 에이전트에 통합됩니다.
    * 스페인 Onda 기상 관측소의 2000년, 2001년 분기별 역사 데이터를 사용하여 규칙을 생성했습니다. 특히 Quinlan의 C4.5 알고리즘이 결정 트리 유도에 적용되었습니다.

4. **기술 채택:**
    * **에이전트 플랫폼:** JADE
    * **규칙 엔진:** JESS
    * **온톨로지 편집기:** Protégé 2000
    * **데이터 마이닝:** WEKA 코어
    * **지식 모델 표현:** PMML
    * **에이전트 통신:** FIPA 표준 (FIPA-ACL 사용)

## 📊 Results

* **데이터셋 활용:** 스페인 발렌시아의 Onda 기상 관측소에서 2000년과 2001년에 걸쳐 약 70,000개의 분기별 측정 기록(기상 속성 및 대기 오염 물질 값, 유효성 검증 태그 포함)을 사용하여 데이터 기반 규칙을 생성하고 시스템을 학습시켰습니다.
* **측정값 유효성 검증(IMV) 성능:** C4.5 알고리즘을 사용하여 생성된 결정 트리는 15개의 리프를 가졌습니다. 이 결정 모델은 유효성 검증 세트(2001년 데이터)에서 **99.71%**의 예측 정확도를 달성하여 매우 만족스러운 결과를 보였습니다.
* **기타 고급 기능 입증:** 개발된 시스템은 유효하지 않은 측정값 추정 및 맞춤형 경보 식별에서도 신뢰할 수 있는 결정 모델을 생성할 수 있음을 입증했습니다.
* **실제 배포 준비:** 시스템은 스페인 발렌시아의 지중해 환경 연구 센터 재단(CEAM)에 시범 사례로 설치될 예정입니다.

## 🧠 Insights & Discussion

* **맞춤형 지능의 중요성:** 이 연구는 기존 환경 모니터링 시스템에 데이터 마이닝을 통한 '맞춤형 지능'을 추가하는 것이 지역적 특성과 트렌드를 반영하여 의사결정의 정확도를 크게 향상시킬 수 있음을 보여줍니다. 이는 일반적인 경향이나 경험 법칙에만 의존하는 것보다 효과적입니다.
* **인간 개입 감소 및 효율성 증대:** 다중 에이전트 시스템은 측정값 검증, 누락값 추정, 경보 식별 등 반복적이고 복잡한 작업을 자동화함으로써 인간 전문가의 부담을 줄이고 실시간 모니터링 및 의사결정을 가능하게 합니다.
* **확장성 및 유연성:** 에이전트 기반 아키텍처는 시스템의 모듈성과 분산 특성을 제공하여, 향후 여러 기상 관측소로 네트워크를 확장하고 다양한 유형의 맞춤형 경보(예: 환자, 공공 행정, 산업을 위한 경보)를 포함하는 데 유연성을 제공합니다.
* **한계 및 향후 연구:** 현재 시스템은 단일 기상 관측소에 초점을 맞추고 있습니다. 향후 연구는 전체 네트워크를 포괄하도록 아키텍처를 확장하고, 에이전트 학습 및 재학습 기능을 더욱 강화하는 데 집중할 것입니다.

## 📌 TL;DR

빠른 환경 변화에 대응하기 위해 대기질 모니터링에서 실시간 의사결정의 필요성과 기존 시스템의 한계를 해결하고자, 본 논문은 데이터 마이닝 기반의 맞춤형 지능을 에이전트에 통합한 다중 에이전트 시스템인 O$_{3}$RTAA를 제안합니다. 이 시스템은 진단, 경보, 데이터베이스, 배포 에이전트로 구성된 3계층 아키텍처를 통해 측정값 검증, 누락값 추정, 공식 및 맞춤형 경보 식별 및 배포를 자동화합니다. 실제 기상 관측소 데이터를 활용한 C4.5 알고리즘 기반 학습 결과, 측정값 유효성 검증에서 **99.71%**의 높은 정확도를 달성하여, 인간 전문가의 개입 없이도 효과적인 자율 모니터링 및 의사결정 지원이 가능함을 입증했습니다.
