# How to solve a classification problem using a cooperative tiling Multi-Agent System?

Thibault Fourez, Nicolas Verstaevel, Frédéric Migeon, Frédéric Schettini, and Frédéric Amblard (2022)

## 🧩 Problem to Solve

본 논문은 일반적으로 선형 분리가 불가능한(non-linearly separable) 분류 문제, 특히 클래스의 특성이 시간과 공간에 따라 변화하는 동적 문제(dynamic problems)를 효율적으로 해결하는 방법을 다룬다. 일반적인 비선형 분류기들은 모델 최적화를 위해 방대한 양의 관측 데이터가 필요하며, 이는 새로운 데이터가 제한적으로 발생하는 동적 환경에서 한계로 작용한다. 따라서 본 연구의 목표는 분류 문제를 입력 변수 공간의 협력적 타일링(cooperative tiling) 문제로 변환하여, 단순한 선형 분류기들의 집합만으로도 비선형 분류 문제를 효과적으로 해결할 수 있는 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 Adaptive Multi-Agent Systems (AMAS) 프레임워크를 기반으로 한 `smapy`라는 앙상블 기반 MAS 구현체이다. `smapy`는 입력 공간을 여러 개의 하이퍼큐브(hypercube) 영역으로 나누고, 각 영역을 담당하는 Context 에이전트가 개별적인 기계 학습 모델을 보유하게 한다. 이를 통해 전체 시스템은 전역적인 비선형 경계를 학습하는 대신, 국소적인 영역에서의 선형 분류를 수행하고 에이전트 간의 협력 규칙(cooperation rules)을 통해 전체 공간을 타일링함으로써 결과적으로 복잡한 비선형 결정 경계를 근사하게 된다.

## 📎 Related Works

논문에서는 분류기 결합(Aggregation of classifiers) 방식을 크게 세 가지 관점에서 설명한다.

1. **연결주의(Connectionism):** 인공신경망(ANN)과 같이 내부 분류기들을 직렬로 연결하여 출력값을 도출하는 방식이다. 하지만 가중치 최적화를 위해 매우 많은 데이터가 필요하다는 단점이 있다.
2. **앙상블(Ensemble):** Bagging(예: Random Forest)이나 Boosting(예: AdaBoost, XGBoost)과 같이 여러 분류기를 병렬로 배치하여 결과를 집계하는 방식이다.
3. **구성주의(Constructivism):** 학습 과정에서 분류기의 그래프 구조가 고정되지 않고 스스로 진화하는 자기 조직화(self-organizing) 방식이다.

기존의 Self-Adaptive Context Learning (SACL) 패턴은 입력 공간을 하이퍼큐브로 나누어 탐색하는 방식을 제안했으나, 본 논문은 여기서 더 나아가 각 Context 에이전트 내부에 온라인 학습이 가능한 머신러닝 모델을 탑재함으로써 분류 성능과 적응력을 높였다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조

`smapy`는 세 종류의 에이전트로 구성된 SACL 아키텍처를 따른다.

- **Percept 에이전트:** 새로운 관측치의 입력 변수 값을 수집하여 Context 에이전트들에게 전달하며, 각 변수의 최댓값과 최솟값을 저장한다.
- **Context 에이전트:** 입력 공간 내의 특정 하이퍼큐브 영역을 정의하고, 해당 영역에 속하는 데이터만을 사용하여 내부 분류 모델을 학습시킨다.
- **Head 에이전트:** 활성화된 Context 에이전트들의 예측값 중 가장 신뢰도가 높은 에이전트의 제안을 최종 선택하고, 결과에 대한 피드백을 전달하며, 필요 시 새로운 에이전트를 생성한다.

### 주요 방정식 및 지표

Context 에이전트 $l$의 상태를 정의하는 주요 수식은 다음과 같다.

1. **영역 부피 (Volume):** $p$차원 공간에서 하이퍼큐브의 부피 $v^l(T)$는 다음과 같이 계산된다.
    $$v^l(T) = \prod_{j=1}^{p} (r_{l,j,1}^t - r_{l,j,0}^t)$$
    여기서 $r_{l,j,0}^t$와 $r_{l,j,1}^t$는 각각 $j$번째 차원의 하한과 상한이다.

2. **신뢰도 (Confidence):** 에이전트가 내놓은 예측의 정답 여부에 따라 누적되는 값이다.
    $$c^l(T) = \sum_{t \in H_l^T} (F^+ * \mathbb{1}_{\hat{y}_l^t = y^t} - F^- * \mathbb{1}_{\hat{y}_l^t \neq y^t})$$
    여기서 $F^+$와 $F^-$는 각각 정답과 오답에 대한 가중치이다.

3. **점수 (Score):** 신뢰도 $c^l(T)$에 정규화 함수 $N_c$를 적용하여 산출한다.
    $$s^l(T) = N_c \circ c^l(T)$$

4. **중첩 지수 (Overlapping Index):** 두 에이전트 $l_1, l_2$ 사이의 영역 중첩 정도를 나타낸다.
    $$o_{l_1, l_2} = \frac{v^{l_1 \cap l_2}}{\min(v^{l_1}, v^{l_2})}$$

### 학습 및 추론 절차 (Exploration & Operation)

- **탐색 단계(Exploration):** 새로운 데이터가 들어오면 해당 영역의 Context 에이전트가 활성화되어 예측을 제안한다. Head 에이전트의 피드백이 긍정적이면 에이전트는 부피를 확장($\alpha$배)하고 신뢰도를 높인다. 부정적이면 부피를 축소하거나, 설정에 따라 해당 데이터를 영역에서 제외(Point exclusion)하고 내부 모델을 미세 조정한다.
- **작동 단계(Operation):** 학습 없이 현재 구성된 타일링 상태를 이용하여 새로운 데이터를 분류한다.

### 비협력 상황(Non-cooperative situations, NCS) 해결

시스템은 다음과 같은 세 가지 NCS를 정의하고 해결한다.

- **무능력(Incompetence):** 어떤 에이전트도 활성화되지 않은 경우, 해당 지점에 새로운 에이전트를 생성한다.
- **경쟁(Competition):** 두 에이전트가 같은 클래스를 예측한 경우, 중첩 지수가 높으면 점수가 높은 쪽이 낮은 쪽을 흡수(Absorption)하고, 그렇지 않으면 밀어낸다(Push).
- **충돌(Conflict):** 두 에이전트가 서로 다른 클래스를 예측한 경우, 점수가 높은 에이전트가 낮은 에이전트를 밀어내어 영역을 조정한다.

## 📊 Results

### 실험 설정

- **데이터셋:** scikit-learn의 2차원 이진 분류 토이 데이터셋 3종을 사용하였다.
  - Moons (중간 정도의 비선형성)
  - Circles (높은 비선형성, 원형 구조)
  - Linearly separable (선형 분리 가능)
- **비교 대상:** 단일 선형 분류기 4종(Logistic Regression, Linear SVM, PA-I, PA-II)과 이들을 내부 모델로 사용하는 `smapy` 인스턴스를 비교하였다.
- **측정 지표:** 5-fold 교차 검증을 통한 분류 정확도(Accuracy) 및 결정 경계(Decision boundaries)의 시각화.

### 실험 결과

- **분류 정확도:**
  - 선형 분리 가능 데이터셋에서는 단일 분류기와 MAS 간의 유의미한 차이가 없었다.
  - Moons 데이터셋에서는 MAS 접근 방식이 약간 더 높은 성능을 보였다.
  - **Circles 데이터셋**에서 가장 극적인 성능 향상이 나타났다. 단일 선형 분류기들은 약 50% (무작위 수준)의 정확도를 보인 반면, `smapy`는 약 83% 이상의 정확도를 기록하였다.
- **결정 경계:** 시각화 결과, 단일 분류기들은 항상 직선의 경계를 가졌으나, `smapy`는 여러 선형 에이전트들의 협력적 배치를 통해 Circles 데이터셋의 원형 구조를 근사하는 비선형 결정 경계를 형성함을 확인하였다.

## 🧠 Insights & Discussion

본 연구는 복잡한 비선형 문제를 해결하기 위해 모델 자체의 복잡도를 높이는 대신, 공간을 분할하고 에이전트 간의 협력을 통해 문제를 국소화(localization)하는 전략을 취했다. 이를 통해 내부적으로는 단순한 선형 모델만 사용하더라도 전체적으로는 비선형 경계를 모사할 수 있음을 입증하였다.

다만, 논문에서는 **과적합(Over-specialization)** 문제를 지적한다. 특히 'Point exclusion' 메커니즘이 활성화될 경우, 에이전트가 너무 좁은 영역의 균질한 데이터에만 특화되어 일반화 성능이 떨어질 수 있다. 이는 기하학적 해석 가능성(Interpretability)과 일반화 능력 사이의 트레이드-오프 관계로 해석된다.

## 📌 TL;DR

본 논문은 선형 분류기들을 협력적 Multi-Agent System(`smapy`)의 개별 에이전트로 배치하여 비선형 분류 문제를 해결하는 방법을 제안한다. 입력 공간을 하이퍼큐브로 타일링하고 에이전트 간의 흡수, 밀어내기 등의 협력 규칙을 통해 비선형 결정 경계를 근사하며, 특히 선형 모델로는 해결 불가능한 원형 데이터셋(Circles)에서 매우 높은 성능 향상을 보였다. 이 연구는 향후 해석 가능한 예측이 필요한 스마트 시티 관리와 같은 동적 실세계 문제에 적용될 가능성이 크다.
