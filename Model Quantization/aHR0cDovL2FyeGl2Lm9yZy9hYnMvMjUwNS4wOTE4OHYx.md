# Zero-shot Quantization: A Comprehensive Survey

Minjun Kim, Jaehyeon Choi, Jongkeun Lee, Wonjin Cho, and U Kang (2025)

## 🧩 Problem to Solve

딥러닝 모델을 리소스가 제한된 엣지 디바이스에 배포하기 위해서는 메모리와 연산 요구량을 줄이는 모델 압축 기술이 필수적이다. 그중 네트워크 양자화(Network Quantization)는 고정밀도 모델을 저비트 포맷으로 변환하여 효율성을 극대화하는 강력한 방법이다. 그러나 전통적인 양자화 방식은 모델을 보정(Calibration)하거나 미세 조정(Fine-tuning)하기 위해 원본 학습 데이터에 접근해야 한다.

실제 산업 현장, 특히 의료, 금융, 기업 서비스 분야에서는 개인정보 보호, 보안 및 규제 제약으로 인해 원본 학습 데이터를 사용할 수 없는 경우가 많다. 따라서 본 논문은 학습 데이터 없이도 모델을 양자화할 수 있는 Zero-shot Quantization (ZSQ, 또는 Data-free Quantization)의 최신 동향을 체계적으로 분석하고 정리하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 ZSQ 분야의 첫 번째 심층 서베이로서 다음과 같은 체계적인 분석 프레임워크를 제공하는 것이다.

1.  **ZSQ 문제의 정식화**: 데이터 없이 양자화를 수행해야 하는 문제 상황을 정의하고, 이 과정에서 발생하는 세 가지 주요 도전 과제(사전 학습 모델로부터의 지식 전이, 실제 데이터와 합성 데이터 간의 괴리, 문제 설정의 다양성)를 제시한다.
2.  **방법론적 분류 체계(Taxonomy) 제안**: ZSQ 알고리즘을 데이터 생성 전략에 따라 세 가지 카테고리인 Synthesis-free, Generator-based, Noise-optimization-based 방법으로 분류한다.
3.  **포괄적인 알고리즘 분석**: 각 카테고리 내에서 다시 Quantization-aware Training (QAT)과 Post-training Quantization (PTQ)으로 세분화하여, 최신 기법들의 동기와 핵심 아이디어 및 성과를 분석한다.
4.  **향후 연구 방향 제시**: 합성 데이터 분석의 원리적 접근, 적용 도메인 확장, 이론적 한계 탐색 등 ZSQ의 발전을 위한 구체적인 연구 방향을 제안한다.

## 📎 Related Works

기존의 모델 압축이나 네트워크 양자화 관련 서베이 논문들이 존재하지만, 대부분은 일반적인 양자화 기법을 다루며 ZSQ는 하나의 하위 주제로 짧게 언급하는 수준에 그쳤다.

본 논문은 ZSQ만을 집중적으로 다룸으로써, 연구자들이 ZSQ의 현재 기술적 수준을 정확히 파악하고, 개별 연구들의 핵심 발견 사항을 구분하며, 향후 연구 방향을 설정하는 데 도움을 주고자 한다. 특히 CNN뿐만 아니라 Vision Transformer (ViT)에 적용된 최신 ZSQ 기법들까지 포함하여 분석 범위를 확장하였다.

## 🛠️ Methodology

### 1. 양자화의 기본 원리
네트워크 양자화는 고정밀도 가중치와 활성화 함수를 저비트 포맷으로 인코딩하는 과정이다. 가중치 행렬 $W$를 $B$비트로 양자화한 결과 $W^q$는 다음과 같은 방정식으로 계산된다.

$$W^q = \lfloor \frac{W}{s} + z + \frac{1}{2} \rfloor$$

여기서 $s$는 scaling factor로 $(\beta - \alpha) / (2^B - 1)$이며, $z$는 zero-point로 $-2^{B-1} - \alpha/s$이다. 이때 $[\alpha, \beta]$는 clipping range를 의미하며, 이 범위를 어떻게 설정하느냐가 양자화 정확도에 결정적인 영향을 미친다.

### 2. ZSQ 방법론의 분류

#### (1) Synthesis-free ZSQ
합성 데이터를 전혀 생성하지 않고 모델의 구조적 특성이나 이론적 기초만을 이용하는 방식이다.
- **DFQ**: Cross-layer equalization과 bias correction을 통해 가중치 범위를 조정하여 양자화 오차를 최소화한다.
- **SQuant**: Hessian 행렬의 대각 성분 근사(Diagonal Hessian Approximation)를 통해 계산 효율성을 높이면서 양자화 오차를 줄인다.
- **UDFC**: 가지치기(Pruning)와 양자화를 통합하여, 손상된 채널을 나머지 채널의 가중 조합으로 복구하는 폐쇄형 솔루션(Closed-form solution)을 사용한다.

#### (2) Generator-based ZSQ
독립적인 생성기(Generator, 주로 GAN 기반)를 학습시켜 합성 데이터셋을 구축하는 방식이다.
- **핵심 아이디어**: 사전 학습된 모델의 Batch Normalization Statistics (BNS)를 활용하여 합성 데이터의 분포를 실제 데이터와 유사하게 맞춘다.
- **Zero-shot QAT**: 생성기와 양자화 모델을 교대로 최적화한다. 예를 들어 GDFQ는 모델이 레이블을 정확히 예측하도록 데이터를 생성하고, 모델은 이 데이터를 통해 성능 격차를 줄인다.
- **Zero-shot PTQ**: Stable Diffusion과 같은 기운된 텍스트-이미지 생성 모델을 사용하여 고해상도의 시맨틱 데이터를 생성하고, 에너지 스코어 필터링 등을 통해 품질을 높인다 (예: GenQ).

#### (3) Noise-optimization-based ZSQ
별도의 생성기 없이 무작위 노이즈 자체를 직접 최적화하여 합성 데이터를 생성하는 방식이다.
- **절차**: 1단계에서 노이즈를 최적화하여 데이터셋을 합성하고, 2단계에서 이 데이터를 이용해 모델을 양자화한다.
- **손실 함수**: BNS loss를 통해 통계적 분포를 맞추고, Inception Loss (IL)를 통해 클래스별 세부 특성을 반영한다.
- **주요 기법**:
    - **IntraQ**: 클래스 내 이질성(Intra-class heterogeneity)을 높여 데이터 다양성을 확보한다.
    - **HAST**: 모델이 틀리기 쉬운 'Hard sample'을 의도적으로 생성하여 성능을 높인다.
    - **PSAQ-ViT**: ViT를 위해 Patch Similarity Entropy (PSE) loss를 도입하여 셀프 어텐션의 다양성을 반영한 데이터를 생성한다.

## 📊 Results

본 논문은 ImageNet 데이터셋으로 사전 학습된 ResNet-18 및 DeiT-Tiny 모델을 대상으로 ZSQ 성능을 비교 분석하였다.

- **벤치마크 설정**: 부동 소수점(FP) 모델의 정확도는 71.47%이다.
- **정량적 결과**:
    - **W4A4 (4-bit weights & activations)**: 최신 ZSQ 기법들은 FP 모델 대비 수 퍼센트 포인트 내외의 낮은 성능 저하를 보이며 경쟁력 있는 결과를 낸다. 특히 Noise-optimization 기반의 PTQ 방법들이 높은 정확도를 기록하는 경향이 있다.
    - **W3A3 (3-bit)**: 비트 수가 낮아질수록 성능 저하가 심화되며, 많은 방법론이 급격한 정확도 하락을 겪는다. 다만 IntraQ와 같은 일부 기법은 극저비트 영역에서도 유의미한 성능을 보여준다.
- **ViT 적용**: DeiT-Tiny 모델의 경우 W4A8 설정에서 CLAMP-ViT 등이 우수한 성능을 보였으며, 이는 CNN 기반 방법론을 그대로 적용하기보다 ViT의 특성(Attention)을 반영한 데이터 생성이 중요함을 시사한다.
- **데이터 효율성**: Noise-optimization 방식은 보통 5,120장의 이미지를 생성하여 QAT를 수행하며, PTQ 방식은 CNN의 경우 1,000장, ViT의 경우 32장 정도의 적은 샘플만으로도 보정이 가능하다.

## 🧠 Insights & Discussion

### 강점 및 분석
ZSQ 연구는 단순히 데이터를 대체하는 것을 넘어, 모델 내부의 통계 정보(BNS)나 어텐션 맵, 텍스처 특징 등을 추출하여 데이터의 본질을 재구성하는 방향으로 발전해 왔다. 특히 GAN에서 Diffusion 모델로, 그리고 단순 노이즈 최적화에서 시맨틱 기반 최적화로 진화하며 실제 데이터와의 간극을 줄여가고 있다.

### 한계 및 비판적 해석
1.  **단편적인 해결책(Patchwork solutions)**: 현재의 연구들은 '클래스 내 다양성 부족'이나 '텍스처 부재'와 같은 개별 증상에 집중하여 이를 해결하는 기법을 제안하고 있다. 하지만 합성 데이터가 실제 데이터와 근본적으로 왜 다른지에 대한 원리적 분석은 부족한 상태이다.
2.  **계산 비용**: 데이터 생성 시간이 상당하며, 특히 Noise-optimization 기반 방식은 RTX 4090 GPU로 수 시간이 소요된다. 이는 실시간 양자화나 엣지 디바이스에서의 직접 적용에 걸림돌이 된다.
3.  **평가 환경의 제한**: 대부분의 실험이 ImageNet과 같은 표준 벤치마크에서만 이루어지고 있어, 실제 데이터 접근이 제한된 특수 도메인(의료, 보안 등)에서의 실용성이 완전히 검증되지 않았다.

## 📌 TL;DR

본 논문은 데이터 없이 모델을 양자화하는 **Zero-shot Quantization (ZSQ)**의 전반적인 방법론을 분석한 종합 서베이 논문이다. ZSQ를 **Synthesis-free, Generator-based, Noise-optimization-based**의 세 가지 전략으로 분류하고, 각 기법이 어떻게 사전 학습 모델의 지식을 추출하여 합성 데이터를 만들고 이를 통해 양자화 오차를 줄이는지 상세히 설명한다. 이 연구는 향후 데이터 프라이버시가 중요한 환경에서 모델 압축을 적용하기 위한 이론적 토대와 기술적 가이드라인을 제공하며, 특히 ViT 모델로의 확장 가능성과 합성 데이터의 원리적 분석이라는 새로운 연구 방향을 제시한다.
