# Query-augmented Active Metric Learning

Yujia Deng, Yubai Yuan, Haoda Fu, and Annie Qu (2021)

## 🧩 Problem to Solve

본 논문은 쌍별 제약 조건(pairwise constraints)을 활용한 클러스터링에서 메트릭 학습(metric learning)의 효율성과 정확도를 높이는 문제를 다룬다. 일반적인 지도 학습과 달리, 모든 데이터에 라벨을 붙이는 것은 비용이 매우 많이 들기 때문에, 전문가의 피드백(두 샘플이 같은 클러스터인지 여부)을 통해 거리 측정 방식(metric)을 학습하고 이를 통해 클러스터링을 수행하는 방식이 사용된다.

기존의 접근 방식들은 다음과 같은 한계를 가진다:

1. **데이터 효율성 부족**: 무작위로 선택된 제약 조건은 클러스터 구조에 대한 정보가 적어 학습 효율이 낮으며, 대부분의 기존 방법론은 학습 과정에서 라벨이 없는(unlabeled) 인스턴스 쌍의 정보를 활용하지 않는다.
2. **특징 선택의 부재**: 원본 데이터 공간에는 클러스터링에 불필요한 무관한 특징(irrelevant features)이 많이 포함되어 있어, 이를 적절히 제거하지 않으면 거리 측정의 정확도가 떨어진다.
3. **정보 손실**: 새로운 제약 조건이 추가될 때마다 모델을 처음부터 다시 학습시키는 경우가 많아, 이전 단계에서 학습된 정보가 유실된다.

따라서 본 논문의 목표는 라벨이 없는 데이터의 잠재적 정보를 활용해 제약 조건을 증강하고, 무관한 특징을 적응적으로 제거하며, 정보 획득량을 극대화하는 능동적 쿼리(active query) 전략을 통해 효율적인 클러스터링을 달성하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **쿼리 증강(Query Augmentation)**, **적응적 패널티를 통한 메트릭 응집(Metric Aggregation)**, 그리고 **최소 기대 엔트로피 기반의 능동적 쿼리 전략**의 결합이다.

- **제약 조건 증강**: 사용자가 제공한 소수의 제약 조건뿐만 아니라, 현재의 클러스터 구조에서 추론된 암시적 제약 조건을 함께 활용하여 메트릭 학습의 효율성을 높인다.
- **적응적 특징 선택**: 이전 학습 단계에서 얻은 메트릭 행렬의 고유값(eigenvalue) 순위 통계량을 사용하여, 클러스터링에 기여도가 낮은 특징에 선택적으로 패널티를 부여함으로써 차원을 축소하고 강건성을 높인다.
- **이웃 구조 기반 능동적 쿼리**: 단순히 개별 쌍의 불확실성을 보는 것이 아니라, 이웃 구조(neighborhood structure)와 전이성(transitivity)을 고려하여 전체 데이터셋의 후험 분포(posterior distribution) 엔트로피를 가장 많이 낮출 수 있는 인스턴스를 선택한다.

## 📎 Related Works

기존의 능동적 클러스터링(active clustering) 연구들은 주로 다음과 같은 방식을 취해왔다:

- **Explore-Consolidate 프레임워크**: 서로 다른 점들을 먼저 찾고 경계를 정교화하는 방식.
- **불확실성 기반 쿼리**: 모델의 불확실성이 높은 쌍을 선택하여 쿼리하는 방식.
- **Metric Learning 기반 접근**: Mahalanobis 거리 등을 학습하여 유사한 객체는 가깝게, 다른 객체는 멀게 배치하는 방식.

그러나 본 논문은 기존 연구들이 **라벨이 없는 쌍(unlabeled pairs)의 정보를 메트릭 학습에 통합하지 않았다**는 점과, **인간-기계 상호작용 과정에서 적절한 차원 축소 전략을 사용하지 않았다**는 점을 지적하며 차별점을 둔다.

## 🛠️ Methodology

### 1. 제약 조건 증강 및 메트릭 학습 (Query Augmentation)

먼저, 쿼리된 제약 조건 $S \cup D$를 바탕으로 퍼지 멤버십 행렬(fuzzy membership matrix) $H$를 추정한다. 이때 MDSP(multi-directional separation penalty)를 사용하여 멤버십의 불확실성을 줄인다:
$$\hat{H}= \text{argmin}_{H} \sum_{(i,j) \in S \cup D} (y_{ij} - h_i^T h_j)^2 + \lambda \sum_{i=1}^n \sum_{k=1}^K \min(|h_{ik}|, |h_{ik}-1|)$$

추정된 $\hat{H}$를 이용하여 두 인스턴스 간의 일치도(concordance) $\hat{h}_i^T \hat{h}_j$를 계산하고, 이를 임계값 $1/K$와 비교하여 증강된 유사 집합 $\tilde{S}$와 비유사 집합 $\tilde{D}$를 생성한다. 이후 다음의 최적화 문제를 통해 메트릭 행렬 $A$를 학습한다:
$$\min_{A} \frac{1}{|S|} \sum_{(i,j) \in S} \|x_i - x_j\|_A^2 + \frac{1}{|\tilde{S}|} \sum_{(i,j) \in \tilde{S}} w_{ij} \|x_i - x_j\|_A^2$$
$$\text{s.t. } \frac{1}{|D|} \sum_{(i,j) \in D} \|x_i - x_j\|_A + \frac{1}{|\tilde{D}|} \sum_{(i,j) \in \tilde{D}} w_{ij} \|x_i - x_j\|_A \ge 1, A \succeq 0$$
여기서 $w_{ij}$는 추론된 제약 조건의 확실성을 나타내는 가중치이다.

### 2. 적응적 패널티를 통한 메트릭 응집 (Metric Aggregation)

학습된 메트릭들을 순차적으로 업데이트하며 무관한 특징을 제거한다. 이전 $T-1$ 단계까지의 메트릭 행렬 $A_t$들의 고유값 순위 통계량(rank statistics) 평균 $\bar{r}$을 계산하여, 평균 순위가 가장 낮은 $q$개의 특징 집합 $G^T$를 선정한다. 이후 $T$번째 단계에서는 이들에 대해 선택적 패널티를 부여한다:
$$\hat{A} = \text{argmin}_{A} \text{Loss}(A) + \gamma \sum_{k \in G^T} \sigma_k(A)$$
여기서 $\sigma_k(A)$는 $A$의 $k$번째 고유값이다. 최종적으로 학습된 $\hat{A}$를 이용하여 PCKmeans 클러스터링을 수행한다.

### 3. 최소 기대 엔트로피 기반 능동적 쿼리 (Active Query)

새로운 인스턴스 $x_i$를 선택할 때, 해당 인스턴스의 이웃 멤버십이 결정되었을 때 전체 데이터의 후험 분포 $\rho(l|x)$의 엔트로피가 얼마나 감소하는지를 측정한다. 구체적으로는 다음의 기대 엔트로피를 최소화하는 $x_i$를 선택한다:
$$x^*_i = \text{argmin}_{x_i \notin N^t} - \sum_{m=1}^{L^t} P^t(l_i = m) \sum_{l \in \Omega} \rho^{+im}(l|x) \log \rho^{+im}(l|x)$$
이 과정에서 **이웃 구조(neighborhood structure)**를 활용하여, 한 번의 쿼리로 여러 개의 제약 조건을 동시에 생성하는 전이성(transitivity)을 이용함으로써 쿼리 효율을 극대화한다.

## 📊 Results

### 1. 시뮬레이션 실험

- **증강 제약 조건의 효과**: 무작위 쿼리를 사용했을 때, 증강된 제약 조건($\tilde{S}, \tilde{D}$)을 포함한 경우 ARI(Adjusted Rand Index)가 일관되게 높게 나타났다. 이는 라벨이 없는 쌍의 정보를 활용함으로써 오버피팅을 방지하고 강건성을 높였음을 보여준다.
- **낮은 신호 대 잡음비(SNR) 환경**: 무관한 특징이 매우 많은 환경($p_1=5, p_2=30$)에서 제안 방법(AQM+MEE)이 PCKmeans, MPCKmeans, COBRA 등 기존 방법론보다 압도적인 ARI 성능을 보였다. 특히 MPCKmeans와 비교했을 때 ARI가 50% 이상 향상되었다.
- **고차원 데이터**: $p=500, n=250$인 고차원 설정에서도 제안 방법이 가장 높은 정확도를 유지하며 확장성을 입증하였다.

### 2. 실제 데이터셋 실험

- **데이터셋**: Breast Cancer(유방암 진단), MEU-Mobile(키스트로크), Urban Land Cover(도시 토지 피복) 세 가지 데이터셋을 사용하였다.
- **정량적 결과**: 모든 데이터셋에서 제안 방법이 가장 높은 평균 ARI를 기록하였다. 특히 쿼리 수가 적을 때 MPCKmeans보다 훨씬 빠르게 높은 정확도에 도달하는 효율성을 보였다.
- **특징 해석력**: Urban Land Cover 데이터에서 제안 방법은 NDVI(정규 식생 지수)와 관련된 특징들을 핵심 특징으로 정확히 식별해냈으며, 이는 실제 도메인 지식과 일치함을 확인하였다.

## 🧠 Insights & Discussion

본 논문은 단순한 알고리즘 제안을 넘어, 능동적 쿼리 전략이 무작위 선택보다 이론적으로 우월함을 KL-divergence 관점에서 증명하였다. 특히 **Theorem 2**와 **Theorem 3**를 통해, 제안된 순차적 업데이트 전략이 정보 획득량을 선형적으로 증가시켜, 2단계(two-step) 전략보다 훨씬 빠르게 최적 분포에 수렴함을 보였다.

**강점**:

- 라벨이 없는 데이터를 버리지 않고 메트릭 학습에 통합한 점.
- 메트릭의 고유값 기반 순위 통계량을 사용하여 특징 선택의 강건성을 확보한 점.
- 전역적인 엔트로피 감소를 고려하는 능동적 쿼리 전략으로 쿼리 비용을 획기적으로 줄인 점.

**한계 및 논의**:

- 본 모델은 기본적으로 선형 부분 공간(linear subspace)과 타원형 클러스터 구조를 가정하고 있다. 비선형 구조를 가진 데이터의 경우 커널 방법(kernel method)이나 스펙트럴 클러스터링으로의 확장이 필요하다.
- 메트릭 학습과 능동적 쿼리가 분리되어 진행되는데, 이를 하나의 통합된 프레임워크로 묶어 상호작용하게 한다면 더 높은 강건성을 얻을 수 있을 것이다.

## 📌 TL;DR

본 논문은 능동적 메트릭 학습을 통해 적은 수의 전문가 피드백만으로도 고정밀 클러스터링을 수행하는 방법론을 제안한다. **암시적 제약 조건 증강**, **고유값 기반의 적응적 특징 제거**, **기대 엔트로피 최소화 쿼리**라는 세 가지 핵심 기법을 통해, 특히 무관한 특징이 많은 노이즈 섞인 고차원 데이터 환경에서 기존 방법론 대비 압도적인 성능 향상과 특징 해석력을 보여주었다. 향후 실시간 온라인 학습이나 딥러닝 기반의 비지도 제약 생성 모델로 확장될 가능성이 크다.
