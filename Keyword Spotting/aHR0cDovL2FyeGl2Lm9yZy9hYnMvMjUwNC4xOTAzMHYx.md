# Improving Pretrained YAMNet for Enhanced Speech Command Detection via Transfer Learning

Sidahmed Lachenani, Hamza Kheddar, Mohamed Ouldzmirli (2024)

## 🧩 Problem to Solve

본 연구는 스마트 애플리케이션의 사용자 상호작용을 개선하는 데 핵심적인 요소인 음성 명령 인식(Speech Command Recognition) 시스템의 정확도와 효율성을 높이는 것을 목표로 한다. 전통적인 음성 인식 방법들은 상당한 계산 자원과 방대한 데이터셋을 요구하며, 특히 자원이 제한된 환경에서 실시간 성능을 구현하는 데 어려움이 있다. 따라서 본 논문은 이미 대규모 오디오 데이터로 사전 학습된 YAMNet 모델을 활용하고 전이 학습(Transfer Learning)을 적용함으로써, 적은 양의 데이터와 계산 비용으로도 높은 인식 정확도를 달성할 수 있는 효율적인 음성 명령 인식 방법을 제안한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 사전 학습된 YAMNet의 강력한 특징 추출 능력을 특정 음성 명령 인식 작업에 맞게 최적화한 것이다. 구체적인 기여 사항은 다음과 같다.

- **YAMNet 기반 특징 추출 강화**: 대규모 YouTube 오디오 데이터로 학습된 YAMNet을 특징 추출기로 사용하여, 제한된 데이터셋에서도 특정 음성 명령을 정밀하게 인식할 수 있도록 모델을 적응시켰다.
- **최적 모델 아키텍처 구성**: YAMNet의 출력층을 대상 작업인 12개의 클래스에 맞게 재구성하고, 최적의 분류 성능을 내는 신경망 구조를 실험적으로 식별하였다.
- **하이퍼파라미터 최적화**: 학습률(Learning Rate)과 배치 크기(Batch Size) 등의 변수를 정밀하게 튜닝하여, 학습되지 않은 데이터(Unseen data)에 대한 일반화 성능과 인식 정확도를 극대화하였다.

## 📎 Related Works

음성 명령 인식 분야에서는 과거에 은닉 마르코프 모델(Hidden Markov Models, HMMs)과 가우시안 혼합 모델(Gaussian Mixture Models, GMMs)이 널리 사용되었다. 그러나 이러한 전통적 방식은 정확도가 낮고 계산 자원 소모가 커 실시간 응용 프로그램에 적용하는 데 한계가 있었다.

이후 합성곱 신경망(CNN), 순환 신경망(RNN), 그리고 장단기 메모리(LSTM) 네트워크와 같은 딥러닝(Deep Learning) 기법이 도입되면서 오디오 신호의 시간적, 스펙트럼적 특징을 효과적으로 캡처할 수 있게 되었고, 인식 정확도가 크게 향상되었다. 특히 전이 학습(Transfer Learning)은 라벨링 된 대규모 데이터셋을 확보하기 어려운 음성 인식 분야에서 매우 효과적인 대안으로 부상하였다.

본 연구에서 채택한 YAMNet은 Google에서 개발한 모델로, MobileNetV1 아키텍처를 기반으로 하여 경량화되어 있으며 오디오 이벤트 분류에 최적화되어 있다. 기존 연구들이 YAMNet을 총기 소리 탐지나 환경음 분류에 사용한 것과 달리, 본 논문은 이를 구체적인 음성 명령 인식 작업으로 확장하여 적용했다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 파이프라인
제안된 시스템은 오디오 전처리, 특징 추출, 그리고 분류 단계로 구성된다. 원본 오디오 파형을 Mel-spectrogram으로 변환한 후, 사전 학습된 YAMNet을 통해 고차원 특징(Embedding)을 추출하고, 이를 사용자 정의된 완전 연결 층(Fully Connected Layers)에 입력하여 최종 클래스를 분류한다.

### 데이터 전처리 및 조건화
학습에는 Speech Commands v0.01 데이터셋의 일부인 32,465개 샘플(12개 클래스)이 사용되었으며, 강건성을 높이기 위해 배경 소음을 추가하는 데이터 증강(Data Augmentation)을 수행하였다.

1. **리샘플링(Resampling)**: 모든 오디오 신호는 $16\text{ kHz}$로 리샘플링된다. 샘플링 주파수를 $f'_s$에서 $f_s$로 변경하는 식은 다음과 같다.
   $$y(t) = x\left(\frac{t}{\alpha}\right), \quad \text{where } \alpha = \frac{f'_s}{f_s}, f_s = 16000\text{Hz}$$

2. **프레임 분할**: 신호를 25ms 길이의 프레임으로 나누고 10ms 간격(hop)으로 이동하며 처리한다.
   - 프레임당 샘플 수: $N_f = 0.025 \times 16000 = 400 \text{ samples}$
   - 홉당 샘플 수: $N_h = 0.010 \times 16000 = 160 \text{ samples}$

3. **단시간 푸리에 변환(STFT)**: 각 프레임에 Hanning 윈도우 $w[n]$를 적용하여 스펙트로그램을 생성한다.
   $$X(m,k) = \sum_{n=0}^{N_f-1} x[n+mN_h] \cdot w[n] \cdot e^{-j2\pi kn/N_f}$$
   스펙트로그램 값 $S(m,k)$는 STFT 결과의 제곱 크기로 계산된다: $S(m,k) = |X(m,k)|^2$.

4. **Mel-scale 변환**: 인간의 청각 특성을 반영하기 위해 Bark-scale(Mel-scale) 필터 뱅크를 적용한다. 주파수 $f$를 Mel 주파수 $f_{\text{mel}}$로 변환하는 식은 다음과 같다.
   $$f_{\text{mel}} = 2595 \log_{10}\left(1 + \frac{f}{700}\right)$$

### 모델 아키텍처 및 학습 절차
- **특징 추출기**: 사전 학습된 YAMNet을 사용하여 오디오의 고수준 특징을 추출한다.
- **분류기**: YAMNet의 기본 출력 클래스인 521개를 제거하고, 본 연구의 목표인 12개 클래스 분류를 위한 새로운 Softmax 출력층을 추가하였다.
- **학습 설정**:
    - **데이터 분할**: 훈련 세트 80%, 검증 세트 20%
    - **최적화 알고리즘**: Adam Optimizer
    - **하이퍼파라미터**: 학습률 $0.0003$, 배치 크기 $128$, 총 에폭(Epoch) 수 $15$
    - **입력**: Mel-spectrogram (50개 주파수 밴드)

## 📊 Results

### 실험 환경 및 설정
실험은 Intel Core i5-7500 CPU, 8GB RAM, NVIDIA GeForce GT 710 GPU 환경에서 MATLAB R2022b를 사용하여 수행되었다.

### 정량적 결과
제안된 YAMNet 기반 전이 학습 모델은 매우 높은 성능을 보였으며, 기존의 일반 딥러닝(DL) 벤치마크 모델보다 우수한 성능을 기록하였다.

| Metric | YAMNet (Proposed) | DL [20] (Baseline) |
| :--- | :---: | :---: |
| **Accuracy** | $\mathbf{0.9528}$ | $0.9441$ |
| **Precision** | $\mathbf{0.9508}$ | $0.9446$ |
| **Recall** | $\mathbf{0.9443}$ | $0.9441$ |
| **F1 Score** | $\mathbf{0.9457}$ | $0.9441$ |
| **Specificity** | $0.9949$ | $0.9949$ |

### 상세 분석
- **학습 곡선**: 정확도 곡선은 초기 단계에서 급격히 상승하여 수백 회의 반복 학습(iteration) 내에 약 90%에 도달했으며, 최종적으로 95% 수준에서 안정화되었다. 훈련 및 검증 곡선이 밀접하게 일치하여 오버피팅(Overfitting)이 발생하지 않았음을 확인하였다.
- **혼동 행렬(Confusion Matrix)**:
    - **고성능 클래스**: "background" (100%), "right" (97.27%), "yes" (96.17%)에서 매우 높은 인식률을 보였다.
    - **저성능 클래스**: "go" (89.23%), "on" (89.11%), "no" (92.22%) 등은 상대적으로 낮은 정확도를 보였다.
    - **분석**: 특히 "go"와 "on"이 "no" 또는 "off"와 혼동되는 경향이 있는데, 이는 해당 단어들 간의 음향적 유사성(Acoustic Similarity) 때문인 것으로 분석된다.
- **강건성**: "unknown" 클래스(97.16%)와 "background" 클래스(100%)의 높은 정확도는 모델이 명령어가 아닌 소음을 효과적으로 배제할 수 있음을 시사한다.

## 🧠 Insights & Discussion

본 연구는 전이 학습을 통해 적은 데이터로도 전문적인 오디오 분류 모델을 구축할 수 있음을 입증하였다. 특히 99.49%라는 매우 높은 특이도(Specificity)는 실제 환경에서 발생할 수 있는 오작동(False Positive)을 최소화할 수 있다는 점에서 실용적 가치가 크다.

그러나 일부 단어들 사이의 낮은 분류 정확도는 단순한 특징 추출만으로는 해결하기 어려운 음향적 유사성 문제가 존재함을 보여준다. 이는 향후 데이터 증강 전략을 수정하거나, 더 정밀한 주파수 분석 기법을 도입해야 함을 시사한다. 또한, 실험에 사용된 GPU(GT 710)가 매우 저사양임에도 불구하고 효율적인 학습이 가능했다는 점은 YAMNet의 경량 구조가 자원 제한적 환경(IoT 기기 등)에 매우 적합함을 뒷받침한다.

## 📌 TL;DR

본 논문은 사전 학습된 **YAMNet 모델에 전이 학습(Transfer Learning)을 적용**하여, 12종의 음성 명령을 인식하는 시스템을 제안하였다. Mel-spectrogram 전처리와 하이퍼파라미터 최적화를 통해 **95.28%의 높은 인식 정확도**를 달성하였으며, 특히 배경 소음과 명령어 외 소리를 배제하는 능력이 매우 뛰어남을 확인하였다. 이 연구는 자원이 제한된 IoT 환경에서 효율적인 음성 제어 인터페이스를 구현하는 데 중요한 기초 자료가 될 것으로 기대된다.
