# Integrating HMM-Based Speech Recognition With Direct Manipulation In A Multimodal Korean Natural Language Interface

Geunbae Lee, Jong-Hyeok Lee, Sangeok Kim (1996)

## 🧩 Problem to Solve

본 연구는 그래픽 사용자 인터페이스(GUI)의 핵심 요소인 직접 조작(Direct Manipulation, 이하 DM) 방식과 음성 인식 인터페이스가 각각 가진 한계점을 해결하고자 한다.

DM 인터페이스는 사용자에게 수많은 항목 중 대상(target)을 찾아야 하는 높은 인지적 부하를 주며, 화면 크기의 제한으로 모든 항목을 표시할 수 없다는 단점이 있다. 또한, 유사한 상호작용을 반복해야 하므로 매크로 기능이나 파라미터 지정과 같은 유연성이 부족하며, 정밀한 마우스 컨트롤이 요구되어 근육 피로를 유발하고 장애인에게는 사용이 어렵다는 문제가 있다.

반면, 음성 인터페이스만 사용할 경우 커서 이동과 같은 공간적 제어가 매우 어색하고, 사용자가 명령어를 기억해야 하며, 응답 시간이 느리고 오인식의 가능성이 상존한다는 문제가 있다. 따라서 본 논문의 목표는 DM 인터페이스의 공간적 선택 능력과 음성 인식의 명령 전달 능력을 결합한 다중 모달(multimodal) 한국어 문서 편집기를 개발하여 사용성을 높이는 것이다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 사용자 작업의 성격에 따라 입력 모달리티를 분리하여 배치하는 '작업 분할(split of tasks)' 전략이다.

구체적으로, 공간적 작업(spatial task)인 '객체 선택(object selection)'은 시각적 입력과 수동 응답에 최적화된 DM 인터페이스가 담당하게 하고, 언어적 작업(verbal task)인 '동작 지정(action specification)'은 청각적 입력과 음성 응답에 최적화된 음성 인식 인터페이스가 담당하도록 설계하였다. 이를 통해 DM의 반복적 조작 문제와 음성 인식의 공간 제어 문제를 동시에 해결하고자 하였다.

## 📎 Related Works

논문에서는 다중 모달 인터페이스와 관련된 몇 가지 선행 연구를 언급한다. 음성 기반 스프레드시트 연구나 다중 모달 상호작용의 담화 모델 분석 등이 수행되었으나, 본 연구는 실제 구현과 시스템 통합에 집중한다.

특히 유사한 시스템으로 NEC의 다중 모달 드로잉 툴과 Xspeak II 시스템을 든다. NEC 툴은 키보드, 마우스, 음성을 통합하여 컨텍스트에 따른 포인팅 해석을 제공하며, Xspeak II는 음성 명령을 통한 윈도우 네비게이션을 지원한다. 본 연구의 차별점은 음성 인식기와 문서 편집기 사이의 통신을 위해 UNIX의 공유 메모리(shared memory)와 세마포어(semaphore) 메커니즘을 사용하여 효율적인 시스템 통합을 구현했다는 점과, 전용 음성 서버 없이 일반적인 음성 인식기와 통신할 수 있는 구조를 가졌다는 점이다.

## 🛠️ Methodology

시스템은 음성 인식 엔진인 **ABrain**과 이를 지원하는 한국어 문서 편집기 **SHE(Simple Hearing Editor)**의 두 부분으로 구성된다.

### 1. ABrain: HMM 기반 음성 인식 엔진
ABrain은 음소(phoneme)를 기본 인식 단위로 사용하는 화자 의존형(speaker-dependent) 이산 음성 인식기이다.

- **전처리 및 특징 추출**: 16kHz-16bit 샘플링을 사용하며, 단시간 에너지(STE)와 제로 크로싱 레이트(ZCR) 분석을 통해 엔드포인트(end-point)를 검출한다. 이후 프리엠퍼시스(pre-emphasis)와 해밍 윈도우(Hamming window)를 적용하고, 15개의 Mel-scaled filter-bank 파라미터를 기반으로 특징 벡터를 생성한다. 양자화 오류를 줄이기 위해 3개의 서로 다른 코드북(codebook)을 사용한다.
- **HMM 모델 구조**: 39개의 한국어 음소 각각에 대해 3개의 상태(state)를 가진 단순한 좌-우 전이(left-to-right) 및 자기 전이(self-transition) 구조의 HMM 모델을 구축한다.
- **학습 절차**: 대규모 분절 데이터베이스의 부재를 해결하기 위해 **Segmental k-means training** 알고리즘을 사용한다. 이 알고리즘은 학습 과정에서 자동으로 데이터를 분절한다. HMM 파라미터는 다음과 같은 방정식으로 재추정된다.

전이 확률 $a_{ij}$는 다음과 같다.
$$a_{ij} = \frac{\# \text{ of transitions from state } i \text{ to } j}{\# \text{ of transitions from state } i \text{ to any state}}$$

방출 확률 $b_i(k)$는 다음과 같다.
$$b_i(k) = \frac{\# \text{ of vectors with codeword } k \text{ in state } i}{\# \text{ of vectors in state } i}$$

- **인식 단계**: 음성 사전(phonetic dictionary)을 통해 단어 HMM 모델을 생성하고, 프레임 동기식 비터비 탐색(frame-synchronous Viterbi search)을 통해 최적의 단어 시퀀스를 찾는다. 또한, 문서 편집 명령 시퀀스를 처리하기 위해 유한 상태 언어 모델(FSN language model)을 적용한다.

### 2. SHE: 한국어 문서 편집기 및 통합 구조
SHE는 EuroBridge Xewwidget을 기반으로 구축된 WYSIWYG 스타일의 한국어 편집기로, 텍스트, 이미지, 음성 및 비디오와 같은 멀티미디어를 지원한다.

- **통신 메커니즘**: ABrain과 SHE는 UNIX 공유 메모리와 세마포어를 통해 통신한다. 
- **작동 흐름**: 사용자가 ABrain을 호출(예: "kant")하면 ABrain이 활성화되고 SHE와 핸드쉐이킹을 시작한다. 이후 사용자가 음성 명령(예: "글자색")을 내리면 ABrain이 이를 인식하여 결과를 공유 메모리에 쓰고, SHE가 해당 동작을 수행한 뒤 다시 ABrain을 슬립 상태로 전환시키는 구조이다.

## 📊 Results

실험은 687개의 단어로 학습된 39개 음소 모델을 바탕으로, 언어 모델이 없는 상태에서 198개의 새로운 단어를 테스트하여 초기 성능을 측정하였다.

- **정량적 결과**: 
    - 언어 모델(LM)을 적용하지 않았을 때의 인식률은 $55\%$였다. (Top-3 및 Top-6 인식률은 각각 $78\%$, $90\%$이다.)
    - SHE 명령어를 위한 유한 상태 언어 모델(FSN)을 적용했을 때, 300개 이상의 항목에 대해 **$97\%$의 인식률**을 기록하였다.
    - 음성 명령에 대한 응답 시간은 약 **2초**로, 문서 편집 작업에 적절한 수준임을 확인하였다.

## 🧠 Insights & Discussion

본 연구는 공간적 작업과 언어적 작업을 분리하여 각각 DM과 음성 인식에 할당함으로써, 두 인터페이스의 단점을 상쇄하고 시너지를 낼 수 있음을 보여주었다. 특히 한국어 음성 인식에 있어 대규모 분절 데이터 없이도 Segmental k-means training을 통해 효율적인 모델 학습이 가능함을 입증하였다.

하지만 몇 가지 한계점이 존재한다. 첫째, 3프레임 미만의 매우 짧은 프레임을 가진 음소 처리에 어려움이 있어 모델 개선이 필요하다. 둘째, 현재 시스템은 화자 의존형(speaker-dependent)이며 이산 음성(discrete speech)만을 처리하므로, 향후 화자 독립형(speaker-independent), 연속 음성 인식, 소음 강건성 확보 및 자유로운 자연어 처리 기능이 추가되어야 한다. 마지막으로, 실제 인간 피험자를 대상으로 한 사용성 테스트를 통해 다중 모달 환경의 실제 효율성을 검증하는 과정이 필요하다.

## 📌 TL;DR

본 논문은 한국어 문서 편집 환경에서 **객체 선택은 DM(Direct Manipulation)**으로, **동작 지정은 음성 인식**으로 분리하여 처리하는 다중 모달 인터페이스를 제안한다. HMM 기반의 음성 인식 엔진인 ABrain과 공유 메모리 기반의 통합 구조를 가진 SHE 편집기를 구현하였으며, 언어 모델 적용 시 **$97\%$의 높은 인식률**과 **2초의 응답 시간**을 달성하였다. 이 연구는 향후 GUI의 인지적 부하를 줄이고 접근성을 높이는 다중 모달 입력 체계의 기초적인 구현 사례로서 의미가 있다.
