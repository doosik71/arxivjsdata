# TACOS: LEARNING TEMPORALLY STRUCTURED EMBEDDINGS FOR FEW-SHOT KEYWORD SPOTTING WITH DYNAMIC TIME WARPING

Kevin Wilkinghoff and Alessia Cornaggia-Urrigshardt (2023)

## 🧩 Problem to Solve

본 논문은 **Few-shot Keyword Spotting (KWS)** 시스템에서 발생하는 고정 크기 슬라이딩 윈도우(sliding window)의 한계점을 해결하고자 한다. 

전통적인 KWS 시스템은 오디오 신호를 분석하기 위해 고정된 크기의 윈도우를 사용한다. 하지만 키워드마다 발화 길이가 서로 다르기 때문에 적절한 윈도우 크기를 결정하는 것은 매우 어렵다. 윈도우가 너무 짧으면 키워드 인식에 필요한 정보가 누락될 수 있고, 반대로 너무 길면 불필요한 노이즈나 다른 단어들이 포함되어 키워드의 정확한 시작 지점(on-set)과 종료 지점(off-set)을 탐지하는 성능이 저하된다.

특히 Few-shot 학습 환경에서는 데이터 양이 매우 적어, 기존의 Sequence-to-sequence 손실 함수(예: Connectionist Temporal Classification, CTC)를 사용하기 어렵고, 사전 학습된 ASR(Automatic Speech Recognition) 임베딩을 사용하는 것은 계산 오버헤드가 크다는 문제가 있다. 따라서 본 논문의 목표는 적은 데이터로도 키워드의 시간적 구조를 유지하면서 효율적으로 인식할 수 있는 새로운 임베딩 학습 방법과 KWS 시스템을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 **TACos**라는 새로운 **Angular Margin Loss**를 제안하여, 시간적 특성이 보존된 2차원 임베딩을 학습시킨 점이다. 

핵심 직관은 단순히 "어떤 키워드인가"만을 학습하는 것이 아니라, "키워드 내에서 해당 세그먼트가 어느 상대적 위치에 있는가"를 동시에 학습시킴으로써, 임베딩이 시간에 따라 변화하도록 강제하는 것이다. 이렇게 생성된 시간적 구조를 가진 임베딩을 **Dynamic Time Warping (DTW)**의 템플릿으로 사용함으로써, 고정 윈도우의 한계를 극복하고 가변적인 길이의 키워드를 정밀하게 탐지할 수 있게 한다. 또한, 학습 과정에서 시간축을 반전시킨 세그먼트를 함께 사용하여 모델이 시간적 순서를 더 엄격하게 학습하도록 유도하였다.

## 📎 Related Works

논문은 기존의 KWS 접근 방식과 그 한계를 다음과 같이 설명한다.

1. **슬라이딩 윈도우 기반 신경망:** 고정된 크기의 블록을 추출하여 임베딩을 생성하고 분류하는 방식이다. 앞서 언급한 바와 같이 윈도우 크기 설정 문제와 정밀한 온/오프셋 탐지의 어려움이 있다.
2. **Prototypical Loss 기반 Few-shot 학습:** 적은 샘플로 임베딩 함수를 학습하는 방식이지만, 대개 결과 임베딩이 시간에 따라 일정하게 유지되는 경향이 있어 DTW와 결합했을 때 효율이 낮다.
3. **Hand-crafted 특징 및 DTW:** HFCCs와 같은 수작업 특징을 DTW 템플릿으로 사용하는 무감독 학습 방식이다. 하지만 매우 짧은 단어나 열악한 음향 환경에서는 성능이 급격히 떨어진다.
4. **ASR 기반 접근법:** CTC Loss 등을 사용하지만, 방대한 양의 학습 데이터가 필요하여 Few-shot 환경에 부적합하다.

본 연구는 기존 연구[24]에서 제안한 2차원 임베딩 방식을 계승하되, 임베딩이 시간적으로 정적인 문제를 해결하기 위해 위치 정보 학습($L_{pos}$)을 추가하여 차별화를 꾀하였다.

## 🛠️ Methodology

### 전체 시스템 구조
시스템은 **Frontend $\rightarrow$ Neural Network $\rightarrow$ Backend**의 세 단계로 구성된다.
- **Frontend:** 오디오 신호를 16kHz로 리샘플링하고 50Hz 하이패스 필터를 적용한 후, 0.25초 길이의 세그먼트로 분할하여 64-bin log-Mel spectrogram을 추출한다.
- **Neural Network:** 4개의 Residual Block으로 구성된 수정된 ResNet 아키텍처를 사용하여 2차원 임베딩을 추출한다. 주파수 차원에는 Max-pooling을 적용하지만, 시간 차원에는 풀링을 적용하지 않아 시간적 해상도를 유지한다.
- **Backend:** 추출된 임베딩을 템플릿으로 사용하여 **Sub-sequence DTW**를 적용, 테스트 문장에서 키워드와 가장 유사한 경로를 찾아 온/오프셋을 결정한다.

### TACos Loss Function
TACos 손실 함수는 지도 학습 손실($L_{kw}$)과 자기지도 학습 손실($L_{pos}$)의 합으로 정의된다.

1. **상대적 위치 인코딩 ($y_{pos}$):** 
키워드의 길이가 제각각이므로, 절대적 위치가 아닌 상대적 위치 $[0, 1]$ 범위를 $N_{pos}$개의 클래스로 매핑하여 인코딩한다.

2. **코사인 유사도 ($\theta$):** 
학습 가능한 클러스터 센터 $c$를 도입하여 임베딩 $e$와 키워드 $i_{kw}$, 위치 $i_{pos}$ 사이의 시간적 평균 유사도를 계산한다.
$$\theta(e, i_{kw}, i_{pos}) := \frac{1}{T} \sum_{t=1}^{T} \max_{i_{cluster}} \frac{\langle e(t), c(i_{cluster}, i_{kw}, i_{pos}) \rangle}{\|e(t)\|_2 \|c(i_{cluster}, i_{kw}, i_{pos})\|_2}$$

3. **Softmax 확률 ($s$):** 
동적으로 적응하는 스케일 파라미터 $\hat{s}$를 사용하여 확률을 계산한다.
$$s(e, i_{kw}, i_{pos}) := \frac{\exp(\hat{s} \cdot \theta(e, i_{kw}, i_{pos}))}{\sum_{j_{kw}=1}^{N_{kw}} \sum_{j_{pos}=1}^{N_{pos}} \exp(\hat{s} \cdot \theta(e, j_{kw}, j_{pos}))}$$

4. **최종 손실 함수:** 
키워드 분류 손실 $L_{kw}$와 위치 예측 손실 $L_{pos}$를 합산하여 전체 손실 $\mathcal{L}_{TAC}$를 구성한다.
$$\mathcal{L}_{TAC} := -\frac{1}{K} \sum_{k=1}^{K} \frac{1}{N_{seg}(k)} \sum_{i_{seg}=1}^{N_{seg}(k)} (L_{kw}(e^{k, i_{seg}}) + L_{pos}(e^{k, i_{seg}}))$$

### 시간 반전 세그먼트 (Temporally Reversed Segments)
모델이 단순한 특징의 집합이 아닌 '시간적 순서'를 학습하게 하기 위해, 모든 키워드 세그먼트를 시간축으로 반전시킨 데이터를 추가 학습 데이터로 사용한다. 반전된 세그먼트는 원래 키워드와는 다른 별도의 클래스로 취급하여, 모델이 정방향과 역방향을 구분하도록 강제함으로써 임베딩의 변별력을 높인다.

## 📊 Results

### 실험 설정
- **데이터셋:** 본 논문에서 제시한 **KWS-DailyTalk** (5-shot, 15개 키워드). 훈련 세트는 키워드당 5개 샘플(총 39초)만 제공되며, 테스트 세트는 실제 대화 문장으로 구성되어 open-set 환경을 모사한다.
- **지표:** Event-based F-score (micro-averaged).
- **비교 대상:** HFCCs 기반 DTW, 슬라이딩 윈도우 기반 신경망, 기존 $\mathcal{L}_{kw}$만 사용한 임베딩 기반 DTW.

### 주요 결과
- **TACos의 우수성:** $\mathcal{L}_{kw} + \mathcal{L}_{pos}$를 사용한 방식이 $\mathcal{L}_{kw}$만 사용한 방식이나 슬라이딩 윈도우 방식보다 월등한 성능을 보였다. 특히 테스트 세트에서 높은 F-score를 기록하였다.
- **데이터 효율성:** 단 39초의 학습 데이터만으로도 전통적인 HFCCs 기반 방식보다 더 나은 성능을 달성하였다.
- **시간 반전 데이터의 효과:** 시간 반전 세그먼트를 학습에 활용했을 때, 사용된 임베딩 종류와 상관없이 모든 케이스에서 성능이 향상되었다.
- **임계값 설정의 편의성:** 슬라이딩 윈도우 방식은 키워드별 개별 임계값(individual threshold) 튜닝이 필수적이었으나, 제안된 TACos 방식은 글로벌 임계값만으로도 충분하거나 개별 튜닝 시 오히려 성능이 소폭 하락하는 양상을 보여 실용성이 높음을 입증하였다.

## 🧠 Insights & Discussion

본 논문은 KWS에서 단순히 '분류'를 잘하는 임베딩을 만드는 것보다, **'시간적 흐름'을 간직한 임베딩**을 만드는 것이 DTW와 결합했을 때 훨씬 강력한 도구가 된다는 점을 시사한다. 

특히 $L_{pos}$를 통해 임베딩 공간 상에서 각 세그먼트가 시간적 순서에 따라 궤적을 그리도록 강제한 점이 핵심이다. 이는 기존의 임베딩들이 시간축에 대해 거의 상수(constant) 값에 가까워 DTW의 장점인 '시간적 왜곡 허용'을 제대로 활용하지 못했던 문제를 정면으로 해결한 것이다. 

다만, 클러스터 센터 $c$를 도입함으로써 학습 가능한 파라미터 수가 크게 증가했다는 점이 한계로 작용할 수 있다. 또한, 실험이 비교적 정제된 DailyTalk 데이터셋에서 이루어졌으므로, 극심한 소음 환경에서의 강건성은 추가적인 검증이 필요하다.

## 📌 TL;DR

본 논문은 Few-shot KWS에서 고정 윈도우의 한계를 극복하기 위해, 키워드 종류와 상대적 위치를 동시에 학습하는 **TACos Loss**를 제안한다. 이를 통해 시간적 구조가 보존된 임베딩을 생성하고 이를 DTW 템플릿으로 활용함으로써, 적은 데이터로도 정밀한 키워드 탐지가 가능함을 보였다. 특히 시간 반전 데이터를 통한 학습이 성능 향상에 기여함을 확인하였으며, 이는 향후 저자원 환경의 음성 인식 및 이벤트 탐지 연구에 중요한 기초 아이디어를 제공한다.
