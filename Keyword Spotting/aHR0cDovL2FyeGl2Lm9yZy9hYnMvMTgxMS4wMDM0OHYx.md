# SEQUENCE-TO-SEQUENCE MODELS FOR SMALL-FOOTPRINT KEYWORD SPOTTING

Haitong Zhang, Junbo Zhang, Yujun Wang (2018)

## 🧩 Problem to Solve

본 논문은 모바일 기기의 웨이크업 트리거(wake-up trigger)로 사용되는 Keyword Spotting (KWS) 시스템의 효율성과 정확도를 높이는 문제를 해결하고자 한다. KWS 시스템은 실시간으로 동작해야 하므로 적은 메모리 점유율(small memory)과 낮은 CPU 사용량(low CPU footprint)을 유지하면서도 높은 인식 정확도를 달성해야 하는 까다로운 제약 조건을 가진다.

기존의 End-to-End KWS 모델, 특히 Attention 기반 모델들은 학습 시에는 Sequence-to-one 방식으로 훈련되지만 추론 시에는 슬라이딩 윈도우(sliding window)를 사용하여 데이터를 처리하는 불일치 문제가 존재한다. 또한, 윈도우 크기를 임의로 설정해야 하는 설정의 자의성과 윈도우 이동으로 인한 지연 시간(latency) 발생이 주요 문제로 지적된다. 따라서 본 연구의 목표는 슬라이딩 윈도우 없이도 높은 성능을 내는 소규모 footprint의 Sequence-to-sequence KWS 모델을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 KWS 문제를 Sequence-to-sequence (Seq2Seq) 프레임워크로 재정의하여, 입력 프레임 시퀀스에 대해 프레임 단위의 정렬(alignment)을 통한 레이블링을 수행하고 이를 직접 학습하는 것이다. 이를 통해 다음과 같은 기여를 달성한다.

- **슬라이딩 윈도우 제거**: 프레임 단위의 정렬된 레이블을 사용하여 학습함으로써, 기존 모델의 고질적인 문제였던 임의의 슬라이딩 윈도우 설정 필요성을 제거하고 파이프라인을 단순화하였다.
- **추론 효율성 및 정확도 향상**: Sequence-to-one 구조보다 유연한 Sequence-to-sequence 구조를 채택하여, 더 적은 파라미터 수로도 기존 Attention 기반 모델보다 우수한 성능을 입증하였다.
- **디코딩 평활화(Decoding Smoothing) 도입**: 추론 단계에서 단일 프레임의 확률값에 의존하지 않고, 연속된 $n$개 프레임의 평균 확률을 사용하는 간단하고 계산 비용이 낮은 평활화 기법을 제안하여 강건성을 높였다.

## 📎 Related Works

KWS 분야의 기존 접근 방식은 크게 세 가지로 분류된다. 
첫째, 대규모 어휘 연속 음성 인식(LVCSR) 시스템을 사용하여 오프라인으로 키워드를 검색하는 방식이다. 둘째, 키워드와 필러(filler)에 대해 각각 Hidden Markov Model (HMM)을 훈련하고 Viterbi 디코딩을 통해 검색하는 방식이다. 셋째, 최근의 딥러닝 기반 접근법으로 CNN이나 RNN을 사용하여 키워드의 하위 단어(sub-word) 확률을 예측하거나 전체 키워드를 직접 예측하는 End-to-End 모델이다.

본 논문에서 베이스라인으로 삼은 Attention 기반 모델은 Encoder와 Attention 메커니즘을 통해 입력 시퀀스를 하나의 벡터로 압축하여 확률을 예측한다. 그러나 이 방식은 앞서 언급한 학습과 추론의 불일치 문제와 윈도우 설정의 임의성이라는 한계가 있으며, 본 연구는 이를 Seq2Seq 구조로 전환함으로써 차별화를 꾀하였다.

## 🛠️ Methodology

### 전체 시스템 구조
제안된 모델은 입력 오디오 특징에서 키워드 존재 여부를 프레임 단위로 출력하는 단순한 구조를 가진다. 전체 파이프라인은 **Sequence-to-sequence 훈련**과 **디코딩 평활화**의 두 단계로 구성된다.

### Sequence-to-sequence 훈련
모델은 입력 특징 시퀀스를 받아 각 프레임이 키워드를 포함하고 있는지 나타내는 one-hot 레이블을 예측하도록 학습된다. 

- **레이블링 전략**: TDNN-LSTM 모델을 통해 생성된 phone-state alignment를 기반으로 레이블을 부여한다.
  - 전체 키워드가 완성된 시점의 프레임: $1$
  - 키워드가 포함되지 않은 프레임: $0$
  - 키워드의 약 3.5글자 정도만 포함되어 모호한 프레임: $-1$ (학습 시 가중치를 0으로 설정하여 오차 전파를 방지한다.)

### 디코딩 및 평활화
테스트 단계에서 모델은 단일 프레임 특징을 입력받아 키워드 검출 확률 $y_t$를 출력한다. 단일 프레임의 변동성을 줄이기 위해, 본 논문은 $n$개의 연속된 프레임 확률의 평균을 구하는 smoothing 방법을 적용한다.

$$ \hat{y}_t = \frac{\sum_{i=t-n+1}^{t} y_i}{n} $$

여기서 $\hat{y}_t$는 최종적으로 결정되는 신뢰도 높은 확률값이며, $n$은 평활화 윈도우 크기이다.

### 학습 절차
- **특징 추출**: 40차원 filterbank 특징을 추출하고, 이를 PCEN (Per-Channel Energy Normalization) Mel-spectrogram으로 변환하여 사용한다.
- **손실 함수**: Cross Entropy loss를 사용한다.
- **최적화**: Adam optimizer를 사용하며, 학습률은 $1e-3$, L2 weight decay는 $1e-5$로 설정하였다. Gradient norm clipping을 1로 적용하여 안정성을 높였다.

## 📊 Results

### 실험 설정
- **데이터셋**: 중국어 4글자 키워드("xiao-ai-tong-xue")를 사용하였으며, Xiaomi AI Speaker에서 수집한 실제 데이터를 활용하였다.
- **평가 지표**: 시간당 오경보 횟수(False Alarm, FA)가 0.1회일 때의 오거부율(False Reject Rate, FRR)을 측정하여 성능을 비교하였다.

### 주요 결과
실험 결과, 제안된 Seq2Seq 모델이 모든 아키텍처(LSTM, GRU)에서 베이스라인 모델보다 우수한 성능을 보였다.

- **정량적 성과**: Seq-to-seq GRU 모델은 약 73K의 파라미터만으로 0.1 FA/h에서 **3.05%의 FRR**을 달성하였다. 이는 Baseline GRU(4.47%) 대비 약 20%의 성능 향상을 의미한다.
- **Encoder 구조 분석**: 네트워크를 깊게 쌓는 것보다 넓게 구성하는 것이 더 효과적임을 확인하였다. 예를 들어, 3개 층의 64유닛 모델보다 1개 층의 128유닛 모델이 훨씬 더 좋은 성능을 보였다.
- **Smoothing 효과**: 평활화 프레임 $n=12$를 적용했을 때, 적용하지 않은 경우보다 FRR이 약 0.06%~0.15% 정도 소폭 개선되는 효과가 있었다.

## 🧠 Insights & Discussion

본 연구는 KWS 시스템에서 단순히 복잡한 Attention 메커니즘을 사용하는 것보다, 문제의 정의를 Sequence-to-sequence로 변경하고 데이터의 정렬(alignment)을 정확히 활용하는 것이 더 효율적임을 보여주었다. 

특히, Attention 기반 모델의 가중치가 키워드의 마지막 글자에 과도하게 집중되는 경향이 있는 반면, 제안된 Seq2Seq 모델은 전체 키워드가 인지되는 시점에 확률이 상승하는 형태를 보였다. 이는 인간의 청각적 주의 집중 방식과 더 유사하며, 결과적으로 불필요한 슬라이딩 윈도우 없이도 모델이 암시적으로 시간적 의존성을 학습할 수 있게 한다.

다만, Smoothing 기법에 의한 성능 향상 폭이 매우 적다는 점은 한계로 볼 수 있으나, 저자들은 계산 비용이 매우 저렴하므로 실용적인 관점에서 충분히 도입 가치가 있다고 주장한다.

## 📌 TL;DR

본 논문은 소규모 footprint의 KWS 시스템을 위해 슬라이딩 윈도우를 제거하고 프레임 단위 정렬 기반의 **Sequence-to-sequence 모델**을 제안하였다. 제안 모델은 기존 Attention 기반 모델보다 적은 파라미터(약 73K)로 더 높은 정확도(FRR 3.05% @ 0.1 FA/h)를 달성하였으며, 이는 연산 자원이 제한된 모바일 및 임베디드 기기의 웨이크업 트리거 구현에 매우 실용적인 접근 방식이다.
