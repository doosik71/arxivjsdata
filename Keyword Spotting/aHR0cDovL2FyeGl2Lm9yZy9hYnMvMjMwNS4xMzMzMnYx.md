# Conditional Online Learning for Keyword Spotting

Michel Meneses, Bruno Iwami (2023)

## 🧩 Problem to Solve

본 논문은 키워드 검출(Keyword Spotting, KWS) 시스템이 실제 배포 환경에서 겪는 성능 저하 문제를 해결하고자 한다. 일반적으로 KWS 모델은 대규모의 정적 데이터셋을 통해 훈련되지만, 실제 환경에서는 새로운 발화 방식(prosodies), 환경 소음, 마이크 설정 변경 등 훈련 데이터와는 다른 데이터 분포(Distribution Shift)가 나타난다. 이러한 비정상성(Non-stationary nature)으로 인해 모델의 예측 성능이 저하되는 문제가 발생한다.

기존에는 이러한 문제를 해결하기 위해 훈련 단계에서 가능한 한 방대하고 다양한 데이터셋을 구축하려 노력했으나, 이는 비용이 매우 많이 들 뿐만 아니라 배포 후 발생하는 모든 변수를 예측하는 것이 불가능하다. 따라서 본 논문의 목표는 모델을 배포한 후, 기기 내부(on-device)에서 실시간으로 유입되는 데이터를 통해 모델을 업데이트함으로써 변화하는 환경에 적응시키는 효율적인 Online Continual Learning 방법을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 Stochastic Gradient Descent(SGD)를 이용한 단순한 온라인 업데이트 방식에 **조건부 업데이트(Conditional Update)** 메커니즘을 결합하는 것이다. 

단순히 실시간 데이터로 모델을 업데이트할 경우, 유입되는 데이터가 독립 동일 분포(i.i.d.)를 따르지 않아 편향된 그래디언트(Biased Gradients)가 생성되고, 결과적으로 기존에 학습한 지식을 잃어버리는 **치명적 망각(Catastrophic Forgetting)** 현상이 발생한다. 이를 방지하기 위해 본 논문은 훈련 데이터와 동일한 분포를 가진 소규모의 정적 **Hold-out 데이터셋**을 유지하며, 모델 업데이트가 이 Hold-out 데이터셋의 성능을 저해하지 않을 때만 파라미터를 확정하는 방식을 제안한다.

## 📎 Related Works

최근 KWS 분야의 Continual Learning 연구들은 주로 모델이 학습하지 않은 '새로운 키워드'를 추가로 학습하는 Task-incremental learning에 집중해 왔다. 예를 들어, 지식 증류(Knowledge Distillation)를 사용하거나 새로운 태스크를 위한 서브 네트워크를 추가하는 방식 등이 제안되었다.

그러나 이러한 접근 방식은 연산 복잡도가 높아 저전력 엣지 디바이스(Edge devices)에서 실행하기 어렵다는 한계가 있다. 또한, 실제 상용 애플리케이션에서는 새로운 키워드를 추가하는 것보다, 고정된 키워드 세트를 다양한 환경 소음 속에서 정확하게 검출하는 능력을 유지하고 향상시키는 것이 더 중요하다. 본 논문은 새로운 태스크 학습이 아닌, **동일한 KWS 태스크 내에서의 환경 적응**에 초점을 맞춘다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

### 전체 시스템 구조
본 논문에서 제안하는 알고리즘인 **COOL (Conditional Online Learning)**은 이미 훈련된 모델 $M_0$와 소규모의 Hold-out 데이터셋 $D_v$를 기기에 함께 배포한 상태에서 시작한다. 데이터 스트림 $S = \{(X_i, y_i)\}$가 순차적으로 유입되면, 이를 버퍼에 저장하고 특정 조건이 충족될 때 모델을 업데이트한다.

### 상세 학습 절차 및 알고리즘 (COOL)
COOL의 작동 과정은 다음과 같다.

1. **데이터 버퍼링**: 유입되는 샘플을 타겟 키워드($S_t$)와 비타겟 키워드($S_n$) 버퍼에 각각 저장한다.
2. **배치 구성**: 두 버퍼에 각각 최소 $b/2$개의 샘플이 쌓여 균형 잡힌 배치 $B$를 구성할 수 있을 때 업데이트를 수행한다.
3. **가설 모델 업데이트**: 현재 모델 $M_{j-1}$에 대해 배치 $B$의 손실 함수(Loss)를 계산하고, SGD를 통해 업데이트된 임시 모델 $M_j$를 생성한다.
4. **조건부 검증 (Conditional Validation)**: 생성된 $M_j$가 다음 두 조건을 모두 만족하는지 확인한다.
   - 배치 $B$에 대한 손실 $l'$이 이전 손실 $l$보다 감소했는가? ($l' \le l$)
   - Hold-out 데이터셋 $D_v$에 대한 손실 $l'_v$가 초기 모델 $M_0$의 손실 $l_v$를 초과하지 않는가? ($l'_v \le l_v$)
5. **파라미터 확정**: 위 조건을 만족하면 $M_j$를 새로운 모델로 확정하고, 만족하지 못하면 업데이트를 폐기하고 $M_{j-1}$을 유지한다.

### 주요 방정식 및 수식
모델의 업데이트 결정은 다음과 같은 손실 값 비교를 통해 이루어진다.
- 업데이트 전 배치 손실: $l = \text{loss}(B, M_{j-1})$
- 업데이트 후 배치 손실: $l' = \text{loss}(B, M_j)$
- 업데이트 후 Hold-out 손실: $l'_v = \text{loss}(D_v, M_j)$
- 초기 Hold-out 손실: $l_v = \text{loss}(D_v, M_0)$

최종적으로 $l'_v > l_v$ 이거나 $l' \ge l$ 인 경우, 모델 업데이트는 취소된다.

## 📊 Results

### 실험 설정
- **모델 아키텍처**: 소형 CNN 모델인 `cnn-one-fstride4`를 사용하였다.
- **데이터셋**: Google Speech Commands v2 (GSC)의 8개 단어를 사용하여 8개의 이진 분류 태스크를 생성하였다. 소음 데이터로는 DCASE 데이터셋(아기 울음소리, 총성, 유리 깨지는 소리)을 사용하여 SNR 25dB로 혼합하였다.
- **비교 대상**: Frozen Base Model (업데이트 없음), Naive Online Learner (Hold-out 검증 없는 단순 SGD 업데이트), COOL.
- **평가 지표**: 정확도(Accuracy).

### 정량적 결과
- **평균 정확도 향상**: COOL은 모든 시나리오에서 베이스 모델보다 높은 성능을 보였으며, 특히 Clean 환경에서는 베이스 모델 대비 약 30.13%의 성능 향상을 기록하였다. 전체 시나리오 평균 향상 폭은 19.96%이다.
- **환경 변화 적응력**: 소음 환경(BabyCrying, GlassBreak, GunShot)에서도 베이스 모델 대비 각각 18.29%, 15.21%, 16.20%의 정확도 향상을 보였다.
- **연속 실험 결과**: 여러 환경이 순차적으로 변하는 데이터 스트림에서 실험했을 때, COOL은 이전 환경에서 학습한 지식을 유지하면서 새로운 환경에 적응하는 모습을 보였다.

### 분석 결과
Naive Online Learner의 경우 초기에는 빠르게 성능이 오르지만, 데이터 분포가 변하는 시점에서 그래디언트 편향으로 인해 성능이 급격히 하락하는 양상을 보였다. 반면 COOL은 Hold-out 세트를 통한 정규화 덕분에 성능 하락 없이 안정적으로 우상향하는 곡선을 그렸다.

## 🧠 Insights & Discussion

### 강점 및 의의
본 연구는 복잡한 알고리즘 없이도 단순한 조건부 업데이트만으로 엣지 디바이스에서 실시간 적응형 KWS를 구현할 수 있음을 증명하였다. 특히 Hold-out 데이터셋을 활용한 정규화 전략이 치명적 망각을 효과적으로 억제하며, 이는 저사양 하드웨어에서도 실행 가능한 매우 실용적인 접근 방식이다.

### 한계 및 가정
본 논문은 실시간으로 유입되는 데이터의 라벨($y_i$)이 이미 존재한다는 가정을 전제로 한다. 하지만 실제 환경에서는 라벨이 없는 데이터가 유입되므로, 이를 해결하기 위한 라벨 추정(Label estimation) 모듈이 추가적으로 필요하다. 또한, 테스트에 사용된 오디오 스트림의 길이가 실제 운영 환경에 비해 짧을 수 있다는 점이 한계로 지적된다.

### 비판적 해석
제안된 방법론은 이론적으로 매우 단순하지만, 실제로는 "어느 정도 크기의 Hold-out 세트를 유지해야 하는가"와 "업데이트 주기(Batch size $b$)를 어떻게 설정해야 하는가"에 대한 하이퍼파라미터 최적화 문제가 남아 있다. 그럼에도 불구하고, 본 연구가 제시한 Baseline은 향후 온디바이스 학습 기반 KWS 연구에 중요한 지표가 될 것으로 판단된다.

## 📌 TL;DR

본 논문은 엣지 디바이스의 KWS 모델이 배포 후 환경 변화로 인해 성능이 떨어지는 문제를 해결하기 위해, **소규모 Hold-out 데이터셋을 이용해 업데이트 여부를 결정하는 '조건부 온라인 학습(COOL)'** 방법을 제안한다. 실험 결과, COOL은 단순 SGD 기반 학습보다 치명적 망각을 훨씬 효과적으로 방지하며, 베이스 모델 대비 평균 약 20%의 성능 향상을 이루었다. 이는 상용 KWS 시스템의 오인식률(False Acceptance Rate)을 낮추고 현장 적응력을 높이는 데 매우 실용적인 해결책이 될 가능성이 높다.
