# HIERARCHICAL NEURAL NETWORK ARCHITECTURE IN KEYWORD SPOTTING

Yixiao Qu, Sihao Xue, Zhenyi Ying, Hang Zhou, Jue Sun (2018)

## 🧩 Problem to Solve

Keyword Spotting (KWS)은 자동 음성 인식(ASR) 시스템을 가동시키기 위한 시작 신호를 제공하는 핵심적인 전처리 단계이다. 따라서 KWS 시스템은 높은 재현율(Recall rate)과 낮은 오인식률(Mistake rate)을 보장해야 한다. 그러나 동시에 실시간성(Real-time property)이 필수적이므로, 계산 복잡도가 매우 낮아야 한다는 제약 조건이 존재한다.

기존의 대규모 복잡 모델(예: LSTM)은 높은 성능을 보장하지만 계산 비용이 너무 커서 KWS에 부적합하며, 반대로 단순한 모델은 다양한 배경 소음이 존재하는 실제 환경에서 성능이 급격히 저하되는 문제가 있다. 본 논문의 목표는 이러한 성능과 복잡도 사이의 모순을 해결하기 위해, 적은 계산 복잡도로도 다양한 환경에서 강건하게 동작하는 계층적 신경망(Hierarchical Neural Network, HNN) 구조를 제안하고 검증하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 서로 다른 소음 환경에서 학습된 여러 수준(Level)의 모델을 계층적으로 구성하여 최종 후험 확률(Posterior probability)을 계산하는 것이다. 

주요 설계 직관은 다음과 같다:
1. **환경별 특화 학습**: 각 계층의 네트워크를 서로 다른 환경 데이터(정숙, 비디오 소음, 차량 소음)로 학습시켜 환경 적응력을 높인다.
2. **Bottleneck Feature의 활용**: 하위 계층의 네트워크가 학습한 저차원의 특징(Bottleneck feature)을 상위 계층으로 전달함으로써, 상위 모델이 효율적으로 정보를 통합하게 한다.
3. **효율적인 아키텍처**: DNN이나 CNN 대비 모델 크기와 계산 복잡도를 유사하거나 더 낮게 유지하면서도 성능을 향상시켜 임베디드 기기 배포에 용이하게 설계하였다.

## 📎 Related Works

논문에서는 다음과 같은 관련 연구들을 언급하고 있다:
- **LVSCR 및 BN 특성**: 다국어 MLP 특징을 이용한 대규모 어휘 연속 음성 인식(LVSCR) 시스템이나, 계층적 Bottleneck 특징을 적용한 연구들이 존재한다.
- **KWS 접근 방식**: 오프라인 LVCSR 시스템을 이용한 키워드 검출이나, 온라인 시스템을 위한 은닉 마르코프 모델(HMM) 기반의 접근 방식이 주로 사용되었다.
- **음향 모델의 진화**: HMM 프레임워크 내에서 음향 모델링을 위해 가우시안 혼합 모델(GMM)이 사용되었으나, 점차 심층 신경망(DNN)으로 대체되는 추세이다.

본 연구는 기존의 단일 DNN/CNN 구조에서 벗어나, 환경별로 특화된 모델을 계층적으로 쌓고 Bottleneck 특징을 전달하는 구조를 KWS에 적용했다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 1. KWS 알고리즘: FST 기반 구현
본 연구에서는 유한 상태 트랜스듀서(Finite State Transducer, FST)를 사용하여 KWS를 구현하였다.
- **작동 방식**: 키워드(예: "hi nomi")를 음소(Phone) 단위로 분해하고, 이를 기반으로 상태 전이 규칙을 생성한다.
- **상태 구성**: Tri-phone들을 클러스터링하여 상태를 생성하며, 음성 입력이 FST의 전이 규칙에 따라 최종 상태에 도달하여 출력이 1이 되면 기기가 깨어나는(Wake-up) 구조이다.

### 2. Hierarchical Neural Network (HNN) 구조
HNN은 세 가지 환경 데이터인 정숙(Quiet, Q), 비디오 소음(Video, V), 차량 소음(In-car, C)을 사용하여 단계적으로 학습된다.

**학습 절차:**
1. **1단계 (Low-level)**: 정숙(Q) 데이터로 MLP를 학습시킨다. 이 네트워크는 저차원 특징을 추출하는 Bottleneck 층($BN_1$)을 포함한다.
2. **2단계 (Mid-level)**: 비디오(V) 데이터로 학습시킨다. 입력값으로 원본 특징(Original feature)과 1단계에서 추출된 $BN_1$을 함께 사용하며, 자체적인 Bottleneck 층($BN_2$)을 가진다.
3. **3단계 (High-level)**: 차량(C) 데이터로 학습시킨다. 입력값으로 원본 특징과 이전 단계들의 Bottleneck 특징들을 사용하며, 최종 출력층 역할을 하므로 Bottleneck 구조를 가지지 않는다.

### 3. 후처리 및 웨이크업 결정 (Post Processing)
세 수준의 모델이 각각 출력을 내놓으므로, 이를 통합하는 전략이 필요하다. 논문에서는 두 가지 처리 방식을 제안한다:
- **단일 출력**: 마지막 3단계 모델의 출력만 사용한다.
- **전체 출력**: 세 모델의 출력을 모두 활용하며, 다음 두 가지 전략 중 하나를 선택한다.
    - **OR 전략**: 어느 한 레벨이라도 웨이크업 신호를 보내면 기기를 깨운다.
    - **평균 전략**: 세 모델 출력의 평균값을 계산하여 결정한다.

## 📊 Results

### 실험 설정
- **데이터셋**: 총 830k 발화(긍정 520k, 부정 310k)를 사용하였으며, 테스트 데이터는 180시간 분량의 다양한 환경 소음이 포함된 데이터이다.
- **베이스라인**: 3개 은닉층(각 512노드)을 가진 DNN 모델을 사용하였다.
- **입력 특징**: 40차원 Log-filterbank energy (25ms window, 10ms shift), 컨텍스트 윈도우는 "5-1-5" 형태로 설정하였다.

### 주요 결과
1. **Bottleneck 구조 비교**: 
   - ROC 곡선 분석 결과, HNN이 베이스라인 DNN보다 월등한 성능을 보였다.
   - 모든 Bottleneck 특징을 사용하는 것보다, 선택적인 Bottleneck 구조를 사용하는 것이 더 좋은 성능을 보였는데, 이는 과도한 파라미터 증가가 수렴을 방해했기 때문으로 분석된다.
2. **출력 처리 전략**:
   - 세 레벨의 출력을 모두 활용하는 방식이 단일 출력 방식보다 전반적으로 재현율(Recall)이 높았다.
   - 다만, 오경보(False Alarm, FA)가 매우 낮은 구간에서는 단일 출력 방식이 더 유리했다. 이는 여러 모델의 출력을 통합할 때 어느 한 모델이라도 오작동하면 FA가 발생하기 때문이다.
3. **CNN과의 비교 및 MHNN 제안**:
   - HNN은 대부분의 CNN 베이스라인보다 성능이 좋았으나, 일부 최적화된 CNN보다는 낮았다.
   - 이를 보완하기 위해 앞의 두 레벨은 CNN으로, 마지막 레벨은 DNN으로 구성한 **Multi Hierarchical Neural Network (MHNN)**를 구현하였으며, 이는 실험한 모든 CNN 모델보다 우수한 성능을 기록하였다.

## 🧠 Insights & Discussion

본 연구는 KWS 시스템에서 모델의 복잡도를 크게 늘리지 않으면서도 환경 강건성을 확보할 수 있는 실용적인 방법을 제시하였다. 특히, 데이터를 환경별로 분리하여 계층적으로 학습시키고 Bottleneck 특징을 통해 정보를 전달하는 방식이 효과적임을 입증하였다.

**강점 및 시사점:**
- 계산 복잡도(Computation Complexity)를 베이스라인 DNN 수준으로 유지하면서도(약 $0.95\text{M} \sim 1.28\text{M}$ 연산량), 소음 환경에서의 성능을 크게 개선하였다.
- 단순한 토폴로지 구조 덕분에 다양한 임베디드 장치에 배포하기 용이하다.

**한계 및 논의:**
- 논문에서 제시한 ROC 결과에 따르면, FA가 매우 낮은 엄격한 기준에서는 계층적 통합 방식이 오히려 취약할 수 있다는 점이 드러났다. 이는 각 계층 모델 간의 신뢰도 가중치를 다르게 설정하는 정교한 통합 메커니즘이 필요함을 시사한다.
- CNN의 성능이 일부 구간에서 더 높게 나타난 점을 근거로, 향후 모든 계층을 CNN으로 구성한 HNN 구조에 대한 추가 연구가 필요함을 명시하고 있다.

## 📌 TL;DR

본 논문은 KWS의 실시간성과 환경 강건성이라는 상충하는 문제를 해결하기 위해, 서로 다른 소음 환경에서 학습된 모델들을 쌓아 올린 **계층적 신경망(HNN)** 구조를 제안한다. 각 층에서 추출된 **Bottleneck Feature**를 상위 층으로 전달하여 계산 효율성을 유지하면서도 인식 성능을 높였으며, 특히 CNN과 DNN을 결합한 **MHNN** 구조를 통해 기존 단일 모델들보다 뛰어난 성능을 달성하였다. 이 연구는 저전력/저사양 기기에서 동작해야 하는 웨이크업 워드 감지 시스템의 최적화 방향을 제시한다.
