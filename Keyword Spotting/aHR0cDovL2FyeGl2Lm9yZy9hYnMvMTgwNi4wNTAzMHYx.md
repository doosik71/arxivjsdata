# Visually grounded cross-lingual keyword spotting in speech

Herman Kamper and Michael Roth (2018)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 문제는 **교차 언어 키워드 스포팅(Cross-lingual Keyword Spotting, CLKS)**이다. 이는 특정 언어(쿼리 언어)로 작성된 텍스트 키워드가 주어졌을 때, 다른 언어(검색 언어)로 된 음성 발화 중에서 해당 키워드를 포함하고 있는 발화를 찾아내는 작업이다.

이 문제는 특히 저자원 언어(low-resource language) 환경에서 매우 중요하다. 일반적으로 자동 음성 인식(ASR) 시스템을 구축하려면 방대한 양의 전사(transcription) 데이터가 필요하지만, 많은 언어에서 이러한 데이터를 수집하는 것은 현실적으로 불가능하거나 매우 어렵다. 따라서 본 논문은 전사 데이터나 병렬 번역 데이터 없이, 오직 이미지와 음성 데이터의 쌍(paired image-speech data)만을 활용하여 교차 언어 검색을 수행하는 것을 목표로 한다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 **시각적 접지(Visual Grounding)**를 통해 음성과 텍스트 간의 간극을 메우는 것이다. 즉, 이미지와 음성이 서로 연관되어 있다는 점을 이용하여, 쿼리 언어로 학습된 시각적 태거(Visual Tagger)가 제공하는 텍스트 라벨을 음성 네트워크의 지도 학습 신호로 사용하는 방식이다.

구체적으로, 검색 언어(영어)의 음성과 쌍을 이루는 이미지에 대해 쿼리 언어(독일어)의 시각적 태거를 적용하여 소프트 라벨(soft labels)을 생성하고, 음성 네트워크가 영어 음성을 입력받아 독일어 키워드를 예측하도록 학습시킨다. 이를 통해 병렬 텍스트 데이터 없이도 시각적 정보를 매개체로 하여 서로 다른 언어의 음성과 텍스트를 연결할 수 있다.

## 📎 Related Works

기존의 키워드 스포팅 연구는 주로 쿼리와 검색 언어가 동일한 단일 언어 환경에서 정확한 일치(exact match)를 찾는 것에 집중해 왔다. 교차 언어 환경에서의 검색은 주로 ASR 시스템을 통해 음성을 텍스트로 변환한 뒤, 텍스트 기반의 교차 언어 정보 검색(CLIR)을 수행하는 계층적 방식으로 접근하였다. 그러나 이 방식은 고성능의 ASR 시스템이 전제되어야 한다는 한계가 있다.

최근에는 이미지와 음성을 공통 벡터 공간으로 매핑하여 서로를 검색하는 연구들이 진행되었으나, 이는 텍스트 기반의 예측(labelled predictions)이 불가능하다는 단점이 있다. 본 논문은 외부 시각적 태거를 사용하여 음성에서 텍스트 라벨을 직접 예측하는 방식을 제안함으로써 기존의 단순 임베딩 기반 접근법과 차별화를 둔다.

## 🛠️ Methodology

### 전체 파이프라인
전체 시스템은 학습 단계와 테스트 단계로 나뉜다. 학습 단계에서는 이미지 $I$와 그에 대응하는 영어 음성 $X$의 쌍을 사용한다. 독일어 시각적 태거가 이미지 $I$로부터 독일어 키워드 확률 분포 $\hat{y}_{de}$를 생성하고, 음성 네트워크 $f(X)$는 영어 음성 $X$를 입력받아 이 $\hat{y}_{de}$를 예측하도록 학습된다. 테스트 단계에서는 이미지를 사용하지 않고, 영어 음성 $X$만을 입력하여 특정 독일어 키워드 $w$가 포함되었는지에 대한 점수를 출력한다.

### 주요 구성 요소 및 역할
1. **독일어 시각적 태거(German Visual Tagger):** 
   - 이미지에서 관련 단어(명사, 형용사, 동사)의 집합을 예측하는 다중 라벨 분류기이다. 
   - VGG-16 모델을 기반으로 하며, Multi30k 데이터셋을 사용하여 학습되었다. 
   - 출력은 어휘 사전 크기 $W$에 대한 확률 분포 $\hat{y}_{de} \in [0, 1]^W$이다.
2. **음성 네트워크 $f(X)$ (XVISIONSPEECHCNN):** 
   - 영어 음성(MFCC 특징값)을 입력받아 독일어 키워드 존재 확률을 예측하는 CNN 모델이다.
   - 구조는 1-D ReLU Convolution $\rightarrow$ Max Pooling $\rightarrow$ 1-D ReLU Convolution $\rightarrow$ Max Pooling $\rightarrow$ 1-D ReLU Convolution $\rightarrow$ Global Max Pooling $\rightarrow$ Fully-connected ReLU $\rightarrow$ Sigmoid layer 순으로 구성된다.

### 학습 목표 및 손실 함수
음성 네트워크는 시각적 태거가 생성한 소프트 타겟 $\hat{y}_{de}$를 정답으로 하여 학습한다. 이때 사용되는 손실 함수는 합산 교차 엔트로피 손실(summed cross-entropy loss)이며, 식은 다음과 같다.

$$\ell(f(X), \hat{y}_{de}) = -\sum_{w=1}^{W} \{ \hat{y}_{de,w} \log f_w(X) + (1 - \hat{y}_{de,w}) \log [1 - f_w(X)] \}$$

여기서 $f_w(X)$는 음성 $X$에 대해 독일어 단어 $w$가 존재할 확률을 나타낸다.

## 📊 Results

### 실험 설정
- **데이터셋:** Flickr8k Audio Captions (영어 음성과 이미지 쌍).
- **평가 데이터:** 영어 발화에 대한 독일어 참조 번역이 존재하는 약 1,000개의 발화.
- **지표:** $P@10$ (상위 10개 결과의 정밀도), $P@N$ (정답 개수 $N$만큼의 상위 결과 정밀도), EER (Equal Error Rate), AP (Average Precision).
- **비교 모델:**
    - `DETEXTPRIOR`: 음성을 무시하고 단어의 단순 출현 빈도만 이용하는 baseline.
    - `DEVISIONCNN`: 음성 대신 이미지의 실제 시각적 태그를 그대로 사용하는 모델.
    - `XBOWCNN`: 실제 정답 텍스트 라벨을 사용하는 상한선(Upper bound) 모델.

### 정량적 결과
실험 결과, 제안 모델인 `XVISIONSPEECHCNN`은 전사 데이터 없이도 $P@10$ 기준 $58.2\%$의 성능을 기록하였다. 이는 단순 빈도 기반 모델(`DETEXTPRIOR`, $7.2\%$)보다 월등히 높으며, 흥미롭게도 이미지 기반 모델인 `DEVISIONCNN`($41.5\%$)보다도 높은 성능을 보였다. 이는 모델이 단순히 이미지를 복제하는 것이 아니라, 음성 신호 내에 존재하는 추가적인 정보를 학습했음을 시사한다.

### 정성적 분석 및 오류 분석
오류 분석 결과, 많은 잘못된 검색 결과가 실제로 의미론적으로 관련이 있는 경우였다.
- **유형 (1):** 리터럴한 단어는 없으나 동의어(synonym)가 포함된 경우.
- **유형 (2):** 키워드 자체는 없으나 의미상 밀접하게 관련된 경우 (예: '셔츠' 쿼리에 '자켓'이 포함된 음성이 검색됨).
- **유형 (3):** 완전히 잘못된 검색 결과.

유형 (1)과 (2)를 정답으로 간주할 경우, `XVISIONSPEECHCNN`의 $P@10$은 $91\%$까지 상승하여 지도 학습 모델(`XBOWCNN`)에 근접하는 성능을 보였다.

## 🧠 Insights & Discussion

본 연구를 통해 시각적 접지를 활용한 교차 언어 키워드 스포팅의 가능성이 입증되었다. 특히 다음과 같은 통찰을 얻을 수 있다.

첫째, **멀티태스크 학습(Multitask Learning)**의 효과이다. 테스트에 사용되지 않는 다른 단어들까지 함께 예측하도록 학습시킨 모델이 특정 키워드만 학습시킨 모델(`KEYXVISIONSPEECHCNN`)보다 성능이 약간 더 좋았다. 이는 추가적인 예측 작업이 정규화(regularization) 효과를 주어 일반화 능력을 향상시켰기 때문으로 분석된다.

둘째, **소프트 타겟(Soft Targets)**의 이점이다. 정답 라벨을 0 또는 1로 명확히 준 모델(`ORACLEXVISIONSPEECHCNN`)보다 시각적 태거의 확률값을 그대로 사용한 모델의 성능이 더 좋았다. 이는 지식 증류(Knowledge Distillation) 관점에서 학생 네트워크가 교사 네트워크의 풍부한 확률 분포 정보를 학습하는 것이 더 유리하다는 기존 연구 결과와 일치한다.

한계점으로는 여전히 많은 오류가 의미론적 혼동에서 발생한다는 점이며, 이는 향후 더 정교한 의미론적 매핑 연구가 필요함을 시사한다.

## 📌 TL;DR

본 논문은 병렬 텍스트나 전사 데이터 없이 **이미지를 매개체로 하여 영어 음성에서 독일어 키워드를 찾아내는 교차 언어 키워드 스포팅** 방법을 제안하였다. 독일어 시각적 태거의 소프트 라벨을 학습 목표로 설정하여 $P@10$ 기준 약 $58\%$의 성능을 달성했으며, 의미론적 유사성을 고려할 경우 그 성능은 더욱 향상된다. 이 연구는 데이터가 부족한 저자원 언어의 음성 검색 시스템을 구축하는 새로운 경로를 제시하였다.
