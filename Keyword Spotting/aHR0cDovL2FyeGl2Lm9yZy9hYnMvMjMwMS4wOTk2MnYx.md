# A Comparison of Temporal Encoders for Neuromorphic Keyword Spotting with Few Neurons

Mattias Nilsson, Ton Juny Pina, Lyes Khacef, Foteini Liwicki, Elisabetta Chicca, and Fredrik Sandin (2023)

## 🧩 Problem to Solve

본 연구는 저전력 키워드 검출(Keyword Spotting, KWS) 시스템을 위한 효율적인 시간적 인코딩(Temporal Encoding) 메커니즘을 탐구한다. 현대의 AI 기반 가상 비서들은 복잡한 음성 인식 모델을 사용하는데, 이는 연산 비용이 매우 높다. 따라서 후속의 고비용 연산을 활성화하기 위한 "웨이크업(wake-up)" 메커니즘으로서의 저전력 KWS 시스템이 필수적이다.

뉴로모픽 센서와 스파이킹 신경망(Spiking Neural Networks, SNNs)은 이벤트 기반의 희소한(sparse) 처리가 가능하여 유망한 대안으로 꼽히지만, 실제 뉴로모픽 하드웨어에서 실시간 스트리밍 데이터를 효율적으로 처리하기 위해서는 물리적 시간을 내재적으로 활용하는 자원 효율적인 시간적 인코딩 방법이 필요하다. 본 논문의 목표는 최근 제안된 두 가지 시간적 인코딩 요소인 Time-Difference Encoder(TDE)와 disynaptic Excitatory-Inhibitory(E-I) 요소의 성능을 KWS 작업에서 비교 분석하는 것이다.

## ✨ Key Contributions

본 연구의 핵심 기여는 뉴로모픽 하드웨어 구현에 최적화된 두 가지 서로 다른 시간적 인코딩 방식이 KWS 작업의 특성 추출 및 분류 성능에 미치는 영향을 정량적으로 비교했다는 점이다.

1. **TDE와 E-I 요소의 비교 분석**: 시간 차이를 인코딩하는 TDE와 뉴런의 이질성(heterogeneity) 및 포스트-억제 반동(postinhibitory rebound) 현상을 이용하는 E-I 요소를 동일한 입력 데이터(TIDIGITS 데이터셋의 포먼트)에 적용하여 성능을 평가하였다.
2. **자원 효율적 관점의 평가**: 적은 수의 뉴런을 사용했을 때의 분류 정확도와 스파이크 발생 빈도를 측정하여, 전력 소비와 연산 효율성 측면에서의 가능성을 조사하였다.

## 📎 Related Works

기존의 뉴로모픽 KWS 연구들은 버퍼링(buffering), 전송 지연(transmission delays), 또는 정밀하게 튜닝된 시간 상수(time constants) 등의 방법을 사용하여 시간 정보를 처리하였다. 그러나 이러한 방식들은 혼합 신호(mixed-signal) 뉴로모픽 하드웨어에서 실시간으로 구현하기에 효율적이지 않다는 한계가 있다.

본 논문은 이러한 한계를 극복하기 위해 하드웨어 친화적인 두 가지 대안을 제시한다. TDE는 입력 채널 간의 스파이크 시간 차이를 직접 인코딩하며, E-I 요소는 전용 지연 메커니즘 대신 아날로그 하드웨어의 소자 불일치(device mismatch)로 인한 이질성을 활용하여 시간적 변이(variance)를 표현한다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조
전체 시스템은 **[입력 데이터 $\rightarrow$ 시간적 인코더(TDE 또는 E-I) $\rightarrow$ 스파이크 카운트 변환 $\rightarrow$ 선형 분류기]** 순으로 구성된다.

1. **데이터 전처리**: TIDIGITS 데이터셋의 음성 데이터에서 spectral maxima인 포먼트(formants) 4개를 추출한다. 이를 1ms 시간 빈(time-bin) 단위의 스파이크 트레인으로 변환하여 입력으로 사용한다.
2. **인코딩 레이어**: 입력 데이터는 TDE 레이어와 두 개의 순차적인 E-I 레이어를 각각 통과하며 특성을 추출한다.
3. **분류 단계**: 각 레이어에서 출력된 스파이크 트레인을 전체 발화당 스파이크 횟수(spike counts)로 변환하고, 이를 입력으로 하는 로지스틱 회귀(Logistic Regression) 모델($L_2$ 정규화 적용)을 통해 One-vs-Rest 이진 분류를 수행한다.

### 주요 구성 요소 및 동작 원리

#### 1. Time-Difference Encoder (TDE)
TDE는 두 입력 채널 간의 스파이크 시간 차이를 인코딩한다.
- **구조**: CuBa LIF 뉴런, 이득(gain) 구획, 촉진 시냅스(facilitatory synapse), 트리거 시냅스(trigger synapse)로 구성된다.
- **동작**: 
    - 촉진 시냅스로 들어온 스파이크는 지수적으로 감소하는 이득 계수를 생성한다.
    - 트리거 시냅스로 들어온 스파이크는 흥분성 포스트시냅스 전류(EPSC)를 생성하는데, 이 전류의 진폭은 현재의 이득 계수 값에 비례한다.
    - 결과적으로 두 입력 스파이크 사이의 시간 차이가 뉴런의 스파이크 버스트(burst) 형태로 인코딩된다.
- **토폴로지**: 인접한 포먼트 트랙의 변화를 캡처하기 위해 최대 측면 거리 $d_{max}=3$ 이내의 입력 채널 쌍에 대해 TDE를 배치하여 총 180개의 뉴런을 구성하였다.

#### 2. Excitatory-Inhibitory (E-I) Elements
E-I 요소는 하나의 흥분성 전류와 하나의 억제성 전류의 합산을 통해 효과적인 지연 흥분(delayed excitation)을 생성하며, 이는 생물학적인 포스트-억제 반동 현상을 모사한다.
- **구조**: AdEx 뉴런 모델과 DPI 시냅스 회로를 사용한다.
- **동작**: 흥분성 및 억제성 시냅스의 시간 상수 차이를 이용하여 입력 스파이크가 발생한 후 일정 시간이 지난 뒤 뉴런의 막전위가 상승하도록 설계한다.
- **토폴로지**: $d_{max}=3$ 설정을 통해 각 뉴런이 4개의 인접 입력 채널과 연결되도록 구성하였다. 1층(174개 뉴런)과 2층(156개 뉴런)의 계층 구조를 가져 더 긴 시간 상수를 표현할 수 있게 하였다. 특히, 하드웨어의 소자 불일치를 모사하기 위해 시간 상수와 가중치에 $\sigma=50\%$의 가우시안 분포를 적용하여 이질성을 부여하였다.

## 📊 Results

### 정량적 결과
1. **Permutation Importance**: 뉴런별 중요도를 분석한 결과, TDE 및 E-I 레이어의 상위 뉴런들이 원본 포먼트 입력 채널보다 3~4배 높은 중요도를 보였다. 이는 해당 인코더들이 시공간적 패턴 추출에 유효함을 시사한다.
2. **훈련 데이터 정확도**: TDE와 E-I 레이어를 모두 통과한 데이터는 선형 분류기만으로도 훈련 세트에서 **100%의 정확도**를 기록하였다. (원본 포먼트 직접 분류 시에는 79%~96% 수준)
3. **테스트 데이터 정확도**: 훈련 세트와 달리 테스트 세트에서는 TDE나 E-I 요소를 사용했을 때 원본 포먼트를 직접 사용한 것보다 뚜렷한 성능 향상이 나타나지 않았다.
4. **스파이크 효율성**: TDE는 다른 뉴런들에 비해 발화당 약 100개 더 많은 스파이크를 생성하였다. 이는 뉴로모픽 프로세서의 전력 소모가 스파이크 발생 및 라우팅에 집중된다는 점을 고려할 때, 전력 효율성 측면에서 불리할 수 있음을 의미한다.

### 소수 뉴런 기반 분류 (Few Neurons)
단일 뉴런 혹은 최대 10개의 뉴런만 사용하여 분류했을 때, TDE나 E-I 요소가 원본 포먼트 채널의 일부를 선택해 사용하는 것보다 월등한 성능을 보이지 않았다.

## 🧠 Insights & Discussion

### 강점 및 가능성
본 연구는 TDE와 E-I 요소가 훈련 데이터에 대해 매우 강력한 특성 추출 능력을 갖추고 있음을 입증하였다. 특히 선형 분류기만으로 100% 분류가 가능하다는 점은 이 인코더들이 입력 데이터의 복잡한 시간적 특징을 효과적으로 선형 분리 가능한 공간으로 투영하고 있음을 보여준다.

### 한계 및 비판적 해석
가장 큰 문제는 **일반화(Generalization)의 부족**이다. 훈련 세트에서는 완벽한 성능을 보였으나 테스트 세트에서 향상이 없었다는 것은 모델이 훈련 데이터에 오버피팅(overfitting)되었거나, 설정된 파라미터들이 특정 샘플에만 최적화되었을 가능성이 크다.

저자들은 이러한 이유가 다음과 같다고 분석한다:
- **파라미터의 균일성**: TDE의 경우 모든 유닛이 동일한 시간 상수를 가지며, E-I의 경우 단순히 무작위 가우시안 분포에서 샘플링된 값을 사용하였다.
- **학습 알고리즘의 부재**: 그리드 서치(grid search)나 무작위 설정이 아닌, 특정 키워드에 민감하게 반응하도록 하는 정교한 학습 프로토콜(Learning Protocol)이 적용되지 않았다.

결론적으로, 뉴로모픽 하드웨어의 이질성을 단순히 무작위로 활용하는 것을 넘어, 각 뉴런의 가중치와 시간 상수를 최적화하는 학습 방법이 도입되어야만 실제적인 성능 향상을 기대할 수 있을 것이다.

## 📌 TL;DR

본 논문은 저전력 키워드 검출(KWS)을 위해 뉴로모픽 하드웨어 친화적인 두 가지 시간적 인코더(TDE, E-I)를 비교 분석하였다. 실험 결과, 두 방식 모두 훈련 데이터에서는 완벽한 분류 성능을 보였으나 테스트 데이터에서는 원본 데이터 대비 뚜렷한 이점이 없었으며, 특히 TDE는 스파이크 발생량이 많아 전력 소모 측면의 우려가 확인되었다. 이 연구는 단순한 구조적 도입보다 **뉴런 간의 이질적인 파라미터를 정교하게 최적화하는 학습 방법**이 뉴로모픽 KWS의 성능 향상에 핵심적임을 시사한다.
