# META-AF ECHO CANCELLATION FOR IMPROVED KEYWORD SPOTTING

Jonah Casebeer, Junkai Wu, Paris Smaragdis (2023)

## 🧩 Problem to Solve

본 논문은 적응형 필터(Adaptive Filters, AFs)가 하위 작업(downstream tasks), 특히 키워드 검출(Keyword Spotting, KWS)의 성능을 최적화하지 못하는 문제를 해결하고자 한다. 

전통적인 AF 설계는 평균 제곱 오차(Mean-Squared Error, MSE)와 같은 단순한 신호 수준(signal-level)의 목표를 최소화하는 데 집중한다. 그러나 신호 수준의 손실 함수를 최소화하는 것이 반드시 하위 분류 작업의 정확도 향상으로 이어지지는 않는다. 기존의 딥러닝 기반 접근 방식들은 전처리와 하위 작업을 완전히 분리하여 모듈성을 유지하려 하거나, 반대로 완전히 통합하여 모듈성을 포기하는 극단적인 전략을 취해왔다. 

따라서 본 연구의 목표는 하위 작업의 지식을 AF의 업데이트 규칙에 통합함으로써, 테스트 단계에서의 추가적인 피드백 없이도 하위 작업의 성능을 극대화하는 적응형 필터 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 **CT-Meta-AF (Classification-Trained Meta-AF)** 프레임워크의 제안이다. 이 프레임워크의 중심 아이디어는 다음과 같다.

1.  **하위 작업 피드백의 통합**: AF의 업데이트 규칙을 학습시킬 때, 단순한 신호 복원 성능이 아닌 하위 분류기(Classifier)의 피드백을 직접적으로 반영하는 새로운 학습 방법론을 도입하였다.
2.  **자기 지도 학습과 분류 손실의 결합**: 깨끗한 음성 신호나 임펄스 응답과 같은 오라클(Oracle) 데이터 없이도, 신호 수준의 자기 지도 학습(Self-supervision)과 분류기의 손실 함수를 결합하여 학습할 수 있도록 설계하였다.
3.  **범용적 적용 가능성**: 제안된 방법은 사전 학습된(Pre-trained) 분류 모델과 공동 학습(Joint-trained) 모델 모두에 적용 가능하며, 추론 시 추가적인 연산 복잡도나 튜닝을 요구하지 않는다.

## 📎 Related Works

논문에서는 다음과 같은 관련 연구들을 언급하며 차별점을 제시한다.

*   **전통적 AF 및 DNN 기반 제어**: RLS(Recursive Least Squares)와 같은 전통적 AF는 수동 튜닝에 의존하며, 최근의 DNN 기반 제어 방식들은 AF의 업데이트 규칙을 학습시키려 하지만 여전히 하위 작업의 성능을 충분히 고려하지 않는다.
*   **수동 통합 방식(Manual Unification Schemes)**: 하위 작업의 지식을 AF 제어에 통합하려는 시도가 있었으나, 이는 매우 복잡한 유도 과정이 필요하며 테스트 시점에 하위 모델로부터의 피드백이 필수적이라는 한계가 있다.
*   **종단간(End-to-End) DNN 파이프라인**: 전처리와 하위 작업을 하나의 큰 네트워크로 구성하는 방식은 성능은 좋으나 모듈성이 떨어지고 학습 구조가 복잡해지는 경향이 있다.

**CT-Meta-AF의 차별점**은 Meta-learning을 통해 AF의 업데이트 규칙 자체를 학습시키되, 그 학습 목표에 하위 작업의 분류 손실을 포함시킴으로써 모듈성과 성능이라는 두 마리 토끼를 잡았다는 점이다.

## 🛠️ Methodology

### 1. 전체 시스템 구조
시스템은 크게 세 가지 구성 요소로 이루어진다:
*   **AEC (Acoustic Echo Canceller)**: 자체 재생 음성(own-playback)으로 인한 에코를 제거하는 필터 $h_{\theta[\tau]}$이다.
*   **Optimizer ($g_\phi$)**: Meta-learned된 신경망으로, AEC의 파라미터 $\theta$를 어떻게 업데이트할지 결정하는 규칙을 제공한다.
*   **Classifier ($m_\phi$)**: AEC의 출력 신호를 받아 키워드를 분류하는 KWS 모델이다.

### 2. 적응형 필터 및 업데이트 규칙
AF는 다음과 같은 가산 규칙(additive rule)에 의해 업데이트된다.
$$\theta[\tau+1] = \theta[\tau] + \Delta[\tau]$$
여기서 $\Delta[\tau]$는 옵티마이저 $g_\phi$에 의해 생성된다. 옵티마이저는 주파수 도메인에서의 그래디언트, 입력, 타겟, 에러, 출력 등의 정보를 입력으로 받아 업데이트 값 $\Delta^k[\tau]$와 내부 상태 $\psi^k[\tau]$를 출력한다.

### 3. CT-Meta-AF 학습 목표 (Loss Function)
본 논문의 핵심인 CT-Meta-AF는 다음과 같은 통합 메타 손실 함수 $L_{CM}$을 사용한다.

$$L_{CM}(\bar{e}, c, \hat{c}) = \lambda \cdot L_C(c, \hat{c}) + (1-\lambda) \cdot L_M(\bar{e})$$

*   $L_C(c, \hat{c})$: **Classification Loss**. 실제 클래스 $c$와 예측 클래스 $\hat{c}$ 사이의 오차를 측정하는 분류 손실이다.
*   $L_M(\bar{e})$: **Self-supervised Meta-loss**. 오라클 데이터 없이 AF 출력의 에너지를 최소화하려는 로그-MSE 손실이다.
    $$L_M(\bar{e}) = \ln \mathbb{E}[\|\bar{e}[\tau]\|^2]$$
*   $\lambda$: 두 손실 사이의 가중치를 조절하는 하이퍼파라미터 ($\lambda \in [0, 1]$)이다.

### 4. 학습 절차
*   **Pretrained Classifier 설정**: 분류기 $m_\phi$를 동결(frozen)시킨 상태에서, 분류기의 피드백을 이용하여 옵티마이저 $g_\phi$만을 최적화한다.
*   **Jointly Trained Classifier 설정**: 옵티마이저 $g_\phi$와 분류기 $m_\phi$를 동시에 학습시킨다. 분류기는 AF의 출력 특성에 적응하고, 옵티마이저는 분류가 잘 되도록 신호를 정제하는 방향으로 함께 진화한다.

## 📊 Results

### 1. 실험 설정
*   **데이터셋**: Microsoft AEC Challenge의 재생 데이터와 Google Speech Commands V2의 키워드 데이터를 결합하여 사용하였다.
*   **평가 지표**: 클래스 불균형을 반영하기 위해 Macro 및 Micro averaged F1 score를 사용하였다.
*   **비교 대상**: Regular Meta-AF, Kalman Filter (Diag. KF), No-Echo (에코 없는 환경), No-AEC (에코 제거 없는 환경).

### 2. 주요 결과
*   **분류 성능 향상**: 사전 학습된 KWS 모델을 사용했을 때, CT-Meta-AEC는 합성 데이터(Synthetic)와 실제 데이터(Real) 모두에서 Diag. KF 및 일반 Meta-AEC보다 월등한 F1 score를 기록하였다.
*   **신호 지표와 하위 작업 성능의 괴리**: 흥미로운 점은 일반 Meta-AF가 에코 제거량(dB) 측면에서는 더 우수했음에도 불구하고, KWS 분류 성능(F1)은 CT-Meta-AEC가 더 높았다는 것이다. 이는 신호 수준의 지표가 하위 작업의 성능을 대변하지 못함을 시사한다.
*   **특수화(Specialization) 효과**: 특정 키워드 집합에 맞춰 학습된 CT-Meta-AEC 모델이 범용 모델보다 더 높은 성능을 보였으며, 이는 분류 학습이 데이터 효율적인 성능 향상 방법임을 보여준다.
*   **공동 학습의 이점**: JCT-Meta-AEC(Jointly Trained)는 특히 신호 길이가 긴(12초) 테스트 세트에서 일반 Meta-AEC 대비 유의미한 성능 향상을 보였다.

## 🧠 Insights & Discussion

### 강점 및 통찰
본 연구는 적응형 필터의 최적화 목표를 '신호의 복원'에서 '하위 작업의 성공'으로 전환함으로써 실질적인 성능 향상을 이루어냈다. 특히, 복잡한 모델 통합 설계 없이 손실 함수 수정만으로 이를 달성했다는 점과, 오라클 데이터 없이 자기 지도 학습을 결합하여 현실적인 학습 시나리오를 구축했다는 점이 높게 평가된다.

### 한계 및 논의
*   **데이터 의존성**: 공동 학습(Joint training)의 경우 충분한 데이터와 계산 자원이 필요하다는 전제가 있다.
*   **일반화 가능성**: 본 논문은 AEC와 KWS라는 특정 작업에 집중했으나, 다른 하위 작업(예: 음성 인식, 소리 이벤트 검출)에서도 동일한 효과가 나타날지는 추가적인 검증이 필요하다.
*   **비판적 해석**: 신호 수준의 에코 제거 성능(dB)이 낮아짐에도 불구하고 분류 성능이 올라갔다는 점은, AF가 단순한 에코 제거를 넘어 분류기가 인식하기 좋은 형태의 '특징 추출기' 역할을 수행하도록 학습되었을 가능성을 시사한다.

## 📌 TL;DR

본 논문은 적응형 필터(AF)의 업데이트 규칙을 학습시킬 때 하위 분류 작업의 피드백을 직접 반영하는 **CT-Meta-AF** 프레임워크를 제안한다. 이를 통해 에코 제거(AEC) 후의 키워드 검출(KWS) 성능을 획기적으로 향상시켰으며, 신호 수준의 최적화가 반드시 하위 작업의 성능 향상으로 이어지지 않는다는 점을 입증하였다. 이 연구는 추론 비용의 증가 없이 기존 AF 파이프라인을 대체할 수 있는 플러그 앤 플레이(plug-and-play) 방식의 솔루션을 제공한다는 점에서 향후 스마트 기기의 음성 인식 시스템 최적화에 중요한 기여를 할 것으로 보인다.
