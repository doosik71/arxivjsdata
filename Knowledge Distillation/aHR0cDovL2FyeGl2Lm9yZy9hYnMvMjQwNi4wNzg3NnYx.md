# Small Scale Data-Free Knowledge Distillation

He Liu, Yikai Wang, Huaping Liu, Fuchun Sun, Anbang Yao (2024)

## 🧩 Problem to Solve

본 논문은 데이터 프리 지식 증류(Data-Free Knowledge Distillation, D-KD) 과정에서 발생하는 막대한 계산 비용과 훈련 효율성 저하 문제를 해결하고자 한다. 일반적으로 D-KD는 원본 학습 데이터에 접근할 수 없는 상황에서, 사전 학습된 교사 네트워크(Teacher Network)의 지식을 활용하여 합성 데이터를 생성하고 이를 통해 학생 네트워크(Student Network)를 학습시키는 방식을 취한다. 

기존의 주류 방식인 적대적 역전-증류(Adversarial Inversion-and-Distillation) 패러다임은 신뢰할 수 있는 지식 전수를 위해 원본 데이터셋 크기에 육박하거나 혹은 그보다 더 많은 양의 합성 샘플을 생성해야만 했다. 이는 실제 응용 환경에서 훈련 자원 소모를 극심하게 만들어 D-KD의 실용성을 저해하는 주요 병목 구간이 된다. 따라서 본 연구의 목표는 합성 데이터의 규모를 획기적으로 줄이면서도 모델의 성능을 유지하거나 향상시키는 '소규모 데이터 기반의 효율적인 D-KD' 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 "적은 양의 데이터만 사용하더라도, 그 데이터의 품질(다양성과 난이도)이 높다면 충분한 지식 증류가 가능하다"는 직관에서 출발한다. 저자들은 합성 데이터가 교사 네트워크의 전역적인 분포 정보를 반영하고 있기 때문에, 데이터 규모를 대폭 줄였을 때 오히려 원본 데이터의 일부를 사용하는 것보다 더 좋은 성능을 낼 수 있다는 점을 발견하였다.

이를 구현하기 위해 본 논문은 **SSD-KD (Small Scale Data-free Knowledge Distillation)**를 제안하며, 다음의 두 가지 핵심 장치를 도입한다.
1. **조절 함수(Modulating Function, $\phi(x)$):** 합성 데이터의 클래스 분포를 다양성(Diversity)과 난이도(Difficulty) 관점에서 명시적으로 균형 잡게 하여 고품질의 소규모 데이터셋을 구축한다.
2. **우선순위 샘플링(Priority Sampling):** 강화 학습 전략을 차용하여, 교사와 학생 모델 간의 정보 차이가 큰 샘플을 동적으로 선택해 학습 효율을 극대화한다.

## 📎 Related Works

D-KD 연구는 크게 두 가지 흐름으로 나뉜다. 첫째는 교사 모델의 레이어별 활성화 기록(Activation records)을 사용하는 방식이고, 둘째는 원본 데이터나 메타데이터 없이 적대적 생성 신경망(GAN)을 통해 데이터를 역전(Inversion)시키는 방식이다. 최근의 고성능 D-KD 방법론들은 대부분 후자의 적대적 역전-증류 패러다임을 따른다.

기존의 효율적인 D-KD 방법(예: Fast10)은 주로 제너레이터의 수렴 속도를 높여 데이터 생성 시간을 단축하는 데 집중했다. 그러나 이들은 여전히 학생 모델을 학습시키기 위해 방대한 양의 합성 데이터를 생성해야 한다는 한계가 있다. 반면, SSD-KD는 데이터 생성 단계뿐만 아니라 이후의 지식 증류 과정에서의 샘플링 효율성을 함께 고려한다는 점에서 기존 연구들과 차별화된다.

## 🛠️ Methodology

### 전체 최적화 목표
SSD-KD의 최적화 목표는 다음과 같은 수식으로 정의된다.
$$\min_{f_s} \max_g \mathbb{E}_{x=\delta \circ g(z)} [L_{BN}(x) + L_{KD}(x) + \phi(x)L_{Task}(x)]$$

여기서 $L_{BN}$은 Batch-Normalization 통계량을 맞추는 손실 함수이며, $L_{KD}$는 교사와 학생의 출력 차이를 줄이는 증류 손실, $L_{Task}$는 교사의 예측값을 정답으로 사용하는 태스크 손실이다. $\phi(x)$는 샘플의 중요도를 조절하는 함수이며, $\delta$는 우선순위 샘플링 전략을 의미한다.

### 1. 분포 균형을 고려한 데이터 역전 (Data Inversion with Distribution Balancing)
제너레이터가 단순히 쉬운 샘플만 생성하거나 특정 클래스에 치우치는 현상을 막기 위해 조절 함수 $\phi(x)$를 도입한다. $\phi(x)$는 다음의 두 가지 항으로 구성된다.

- **다양성 인식 균형(Diversity-aware balancing):** 리플레이 버퍼 $B$ 내에서 해당 샘플과 동일한 클래스의 비율이 높을수록 가중치를 낮추어, 희소한 클래스의 샘플 생성을 유도한다.
- **난이도 인식 균형(Difficulty-aware balancing):** 교사 모델이 예측한 확신도(Confidence) $p^T(x)$가 낮을수록(즉, 어려운 샘플일수록) 가중치를 높인다.

최종적인 조절 함수 $\phi(x)$의 식은 다음과 같다.
$$\phi(x) = \left(1 - \frac{1}{|B|} \sum_{x' \in B} \mathbb{I}_{c^T(x')=c^T(x)}\right) (1 - p^T(x))^\gamma$$

### 2. 우선순위 샘플링 기반 증류 (Distillation with Priority Sampling)
모든 합성 데이터를 균일하게 학습하는 대신, 학습에 가장 도움이 되는 샘플을 선택적으로 학습하는 우선순위 샘플링(PS)을 도입한다.

- **우선순위 함수 $\delta^i(x)$:** 교사 모델 $f_t$와 학생 모델 $f_s$ 사이의 KL-divergence를 측정하여, 두 모델 간의 정보 격차가 큰 샘플에 높은 우선순위를 부여한다.
  $$\delta^i(x) = w^{i-1}(x) KL(f_t(x; \theta_t) || f_s(x; \theta_s))$$
- **편향 수정 (Importance Sampling):** 특정 샘플만 집중적으로 학습할 때 발생하는 분포 편향을 해결하기 위해 중요도 샘플링 가중치 $w^i(x)$를 도입하여 학습 과정을 보정한다.
  $$w^i(x) = (N \cdot P^i(x))^{-\beta}$$

## 📊 Results

### 실험 설정
- **데이터셋:** CIFAR-10, CIFAR-100 (이미지 분류), NYUv2 (시맨틱 세그멘테이션)
- **비교 대상:** Fast2, Fast10, DeepInv, CMI, DAFL, DFQ, ZSKT 등
- **측정 지표:** Top-1 Accuracy (분류), mIoU (세그멘테이션), 전체 훈련 시간 (Hours)

### 주요 결과
1. **훈련 효율성:** SSD-KD는 원본 데이터셋의 10% 규모(5,000개 샘플)만으로도 Fast2 대비 1.9배에서 3.92배까지 훈련 속도를 향상시켰으며, 다른 주류 D-KD 방법들보다 1~2 order of magnitude 더 빠르게 학습을 완료하였다.
2. **모델 성능:** 
   - CIFAR-100에서 Fast2 대비 최소 3.29%에서 최대 10.19%의 절대 정확도 향상을 보였다.
   - NYUv2 데이터셋에서 mIoU 0.384를 달성하여, 원본 데이터로 학습한 학생 모델(0.375)이나 기존 D-KD 방법들보다 우수한 성능을 기록하였다.
3. **데이터 규모에 따른 분석:** 합성 데이터 규모를 100%에서 1%까지 줄였을 때, 훈련 시간은 선형적으로 감소하는 반면 정확도 하락은 매우 완만하게 나타났다. 이는 고품질의 소규모 데이터셋만으로도 충분한 학습이 가능함을 입증한다.

## 🧠 Insights & Discussion

본 논문은 D-KD의 효율성을 개선하는 관점을 '빠른 생성'에서 '효율적인 증류'로 확장했다는 점에서 큰 의의가 있다. 특히 합성 데이터가 교사 모델의 지식을 압축적으로 담고 있기 때문에, 적절한 필터링과 샘플링 전략만 있다면 원본 데이터보다 훨씬 적은 양으로도 동일하거나 더 나은 성능을 낼 수 있다는 사실을 실험적으로 보여주었다.

다만, 본 방법론은 여전히 교사 모델의 BN(Batch-Normalization) 통계량에 의존하고 있으며, 조절 함수 $\phi(x)$의 하이퍼파라미터 $\gamma$나 샘플링 가중치 $\beta$ 등에 따라 성능 변동이 있을 수 있다는 점이 잠재적인 한계로 보인다. 그럼에도 불구하고, 데이터 규모를 1/10로 줄이면서 성능을 유지한 점은 실제 리소스가 제한된 환경에서 D-KD를 적용하는 데 매우 강력한 도구가 될 것이다.

## 📌 TL;DR

SSD-KD는 합성 데이터의 **다양성**과 **난이도**를 조절하는 함수($\phi(x)$)와 모델 간 격차를 기반으로 한 **우선순위 샘플링**을 통해, 원본 데이터의 10% 수준인 초소규모 합성 데이터만으로도 고성능 지식 증류를 수행하는 방법론이다. 이를 통해 기존 D-KD 대비 훈련 속도를 수십 배에서 수백 배까지 가속화하면서도 경쟁력 있는 정확도를 달성하였다.
