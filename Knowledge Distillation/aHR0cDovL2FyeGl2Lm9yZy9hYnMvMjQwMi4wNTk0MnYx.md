# Cooperative Knowledge Distillation: A Learner Agnostic Approach

Michael Livanos, Ian Davidson, Stephen Wong (2024)

## 🧩 Problem to Solve

본 논문은 기존의 지식 증류(Knowledge Distillation, KD) 방식이 가진 세 가지 주요 한계점을 해결하고자 한다. 첫째, 기존 KD는 교사 모델(Teacher)의 모든 지식이 유용하다고 가정하여 무분별하게 전달하는 경향이 있다. 둘째, 지식의 흐름이 교사에서 학생(Student)으로 향하는 단방향(Unidirectional) 구조이며, 학생은 학습의 수동적 대상일 뿐이다. 셋째, 일반적으로 단일 교사와 단일 학생 간의 전송으로 제한된다.

이러한 제약은 서로 다른 강점과 약점을 가진 여러 모델이 존재하는 환경에서 효율적인 지식 공유를 방해한다. 특히, 특정 도메인에서 각 모델이 일부 영역에서만 전문성을 가진 '세미 전문가(Semi-expert)'인 경우, 상호 보완적인 지식 교환이 필수적이다. 따라서 본 연구의 목표는 모델의 아키텍처나 알고리즘에 구애받지 않고(Learner Agnostic), 여러 모델이 동시에 교사와 학생의 역할을 수행하며 필요한 지식만을 선택적으로 교환하는 '협력적 증류(Cooperative Distillation)' 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 지식을 로짓(Logits)이나 파라미터 형태로 전달하는 대신, **데이터 자체에 지식을 임베딩하여 가상 인스턴스(Virtual Instance) 형태로 전달**하는 것이다. 이를 위해 저자들은 **정수적 반사실(Quintessential Counterfactuals)**이라는 개념을 도입하였다.

일반적인 반사실(Counterfactual) 생성은 레이블을 변경(Flip)하는 방향으로 작동하지만, 정수적 반사실은 해당 인스턴스가 정답 클래스의 특징을 더 강하게 나타내도록 생성한다. 즉, 교사 모델이 정답을 맞힌 데이터 중 학생 모델이 틀린 데이터를 찾아, 교사 모델의 관점에서 "이 데이터는 이런 점에서 정답 클래스의 전형적인 모습이다"라는 정보를 담은 가상 데이터를 생성하여 학생의 학습 셋에 추가하는 방식이다. 이를 통해 모델 간의 아키텍처 차이와 상관없이 데이터 수준에서 지식을 전송할 수 있다.

## 📎 Related Works

지식 증류(KD) 연구는 크게 전달되는 지식의 종류와 증류 체계에 따라 구분된다.
- **지식의 종류:** 학생이 교사의 출력을 복제하는 Response/Output Distillation, 파라미터나 은닉층의 특징을 모방하는 Feature Distillation, 그리고 데이터 간의 관계나 유사도를 학습하는 Relation Distillation이 있다.
- **증류 체계:** 이미 학습된 교사가 학생을 가르치는 Offline KD와 학습 과정 중에 동시에 일어나는 Online KD로 나뉜다.

기존 연구들과의 차별점은 다음과 같다. 
1. **전이 학습(Transfer Learning)과의 차이:** 전이 학습은 서로 다른 소스-타겟 도메인 간의 전이를 다루지만, 본 연구는 동일 도메인 내의 세미 전문가들 간의 협력을 다룬다.
2. **전통적 KD와의 차이:** 전통적인 KD는 전문가 $\rightarrow$ 초보자 구조의 단방향 전송인 반면, 본 연구는 다방향(Multidirectional) 전송을 수행하며, 모든 인스턴스가 아닌 학생의 결핍이 있는 특정 인스턴스에만 집중(Focused)하여 전송한다.
3. **데이터 프라이버시:** 모델을 공유하되 원본 데이터를 공유하지 않고 가상 인스턴스만을 교환함으로써 데이터 프라이버시 문제를 해결할 수 있는 구조를 가진다.

## 🛠️ Methodology

### 전체 파이프라인
협력적 증류 프로세스는 크게 세 단계로 구성된다.

1. **전문성 식별 (Expertise Identification):** 각 모델 $i$가 정확하게 예측할 수 있는 인스턴스 집합 $S_i$를 식별한다.
2. **결핍 식별 (Deficiency Identification):** 모델 $i$는 맞혔지만 모델 $j$는 틀린 인스턴스 집합 $R_{i \to j}$를 찾는다.
   $$R_{i \to j} = S_i - S_j$$
3. **협력적 증류 (Cooperative Distillation):** $R_{i \to j}$에 속한 각 인스턴스 $x$에 대해, 교사 모델 $i$가 정수적 반사실 $x'$를 생성하여 학생 모델 $j$의 학습 셋에 추가하고 재학습시킨다.

### 정수적 반사실 생성 (Quintessential Counterfactual Generation)
교사 모델 $f_i$는 학생 모델 $f_j$가 이해하기 쉽도록 정답 클래스의 전형적인 특징을 강화한 가상 인스턴스 $x'$를 생성한다. 이때 목표 출력값 $y'$는 다음과 같이 설정된다.
$$y' = f_i(x) + \alpha(y - f_i(x))$$
여기서 $y$는 실제 정답 레이블이며, $\alpha$는 교사의 영향력을 조절하는 하이퍼파라미터이다 ($\alpha=0.5$ 사용). $\alpha$가 1에 가까울수록 완벽한 정답 클래스의 특성을 지향하고, 0에 가까울수록 교사 모델의 현재 예측치에 가깝게 생성된다.

최종적으로 생성되는 반사실 $x'$는 다음의 최적화 목적 함수를 통해 결정된다.
$$\text{argmin}_{x'} d(x', x) + \lambda |f_j(x') - y'|^2$$
- $d(x', x)$: 원본 데이터와 반사실 간의 거리 (Manhattan distance 사용).
- $\lambda$: 거리 유지와 목표 출력 달성 사이의 균형을 맞추는 가중치.
- 모델이 미분 가능할 경우 Adam Optimizer를 사용하며, Decision Tree와 같은 비미분 모델의 경우 입자 군집 최적화(Particle Swarm Optimization, PSO)를 사용하여 $x'$를 찾는다.

## 📊 Results

### 실험 설정
저자들은 총 6가지 실험을 통해 제안 방법의 유효성을 검증하였다.
- **데이터셋 및 작업:** 중고차 가격 예측 (Binary Classification), 심장병 유무 예측, FashionMNIST 이미지 분류 등 다양한 정형 및 비정형 데이터셋을 사용하였다.
- **비교 대상:** Parameter Transfer, SSL (GAN, Mixup), Response-based KD, Feature-based KD 및 모든 데이터를 합쳐 학습시킨 이상적인 케이스(Data Pollution)와 비교하였다.

### 주요 결과
1. **아키텍처 및 알고리즘 독립성 (Exp 1, 2, 4):** MLP $\leftrightarrow$ MLP뿐만 아니라, MLP $\leftrightarrow$ Decision Tree, Decision Tree $\leftrightarrow$ Naive Bayes $\leftrightarrow$ SVM 등 서로 다른 알고리즘 간의 지식 전송이 성공적으로 이루어짐을 확인하였다. 특히 성능이 낮은 모델이 높은 모델을 가르쳐 성능을 향상시키는 결과가 나타났다.
2. **소규모 데이터셋 성능 향상 (Exp 3, 4):** 각 모델이 매우 적은 양의 데이터로 학습되어 기본 성능이 낮은 상황에서도 협력적 증류를 통해 성능이 크게 향상되었으며, 일부 케이스에서는 모든 데이터를 합쳐 학습시킨 경우보다 더 높은 성능을 보였다.
3. **다수 모델 간의 협력 (Exp 5):** FashionMNIST에서 각 모델이 서로 다른 클래스 하나에만 결핍이 있는 상황을 설정했을 때, 상호 보완적인 지식 전송을 통해 기본 성능 76%에서 83%까지 향상시켰다.
4. **특징 중첩 민감도 (Exp 6):** 특징(Feature)의 일부가 누락되어 특징 공간이 완전히 일치하지 않는 상황에서도 성능 향상이 유지됨을 보였으며, 특징 중첩 정도와 성능 향상 사이에는 유의미한 상관관계가 없음을 확인하였다.

## 🧠 Insights & Discussion

### 강점 및 해석
- **학습자 불가지론적(Learner Agnostic) 특성:** 지식을 데이터 형태로 전달하기 때문에 모델의 내부 구조(가중치, 로짓)를 알 필요가 없다. 이는 서로 다른 프레임워크나 알고리즘을 사용하는 모델 간의 협력을 가능케 한다.
- **데이터 다양성 증가:** t-SNE 시각화 결과, 생성된 반사실들이 원본 데이터의 분포를 유지하면서도 새로운 샘플을 제공함으로써 모델의 일반화 성능을 높이는 효과를 준다.
- **프라이버시 보호:** 원본 데이터를 직접 공유하지 않고, 특정 목적을 위해 생성된 '가상 데이터'만을 공유하므로 의료나 금융과 같이 데이터 보안이 중요한 분야에 적용 가능하다.

### 한계 및 비판적 해석
- **반사실 생성의 복잡도:** 정형 데이터(Tabular data)에서는 반사실 생성이 용이하지만, 이미지와 같은 고차원 데이터에서는 생성 품질을 보장하기 어렵다. 논문에서도 이미지 데이터(FashionMNIST)의 경우 픽셀 최적화 범위를 제한하는 추가 작업이 필요했음을 언급하였다.
- **특징 중첩 가정:** 특징 공간이 완전히 다르다면 전송이 불가능하며, 어느 정도의 특징 중첩이 존재해야 한다는 가정이 필요하다.
- **계산 비용:** 모든 모델 쌍에 대해 결핍을 식별하고 반사실을 생성해야 하므로 모델 수가 $k$개일 때 $O(k^2|X|)$의 복잡도가 발생한다. 모델 수가 매우 많아질 경우 계산 비용이 급격히 증가할 수 있다.

## 📌 TL;DR

본 논문은 기존의 단방향/무분별한 지식 증류의 한계를 극복하기 위해, 여러 모델이 서로의 부족한 점을 채워주는 **다방향 협력적 지식 증류(Cooperative Knowledge Distillation)** 방법을 제안한다. 교사 모델이 학생 모델의 결핍을 보완하는 **정수적 반사실(Quintessential Counterfactuals)** 가상 데이터를 생성하여 전달하는 방식을 통해, 모델의 아키텍처나 알고리즘에 상관없이 지식을 전송할 수 있음을 증명하였다. 특히 원본 데이터 공유 없이 모델 간 지식 전송이 가능하다는 점에서 의료 및 금융 분야의 프라이버시 보존형 협력 학습에 기여할 가능성이 크다.
