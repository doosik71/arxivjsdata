# Pre-training Distillation for Large Language Models: A Design Space Exploration

Hao Peng, Xin Lv, Yushi Bai, Zijun Yao, Jiajie Zhang, Lei Hou, Juanzi Li (2024)

## 🧩 Problem to Solve

본 논문은 대규모 언어 모델(Large Language Models, LLMs)의 지식 증류(Knowledge Distillation, KD) 적용 시점을 기존의 사후 학습(post-training) 단계에서 사전 학습(pre-training) 단계로 확장하는 문제를 다룬다. 일반적으로 LLM 분야의 KD는 교사 모델이 생성한 지시어-응답 쌍을 사용하여 학생 모델을 학습시키는 사후 학습 단계에 집중되어 왔다. 그러나 이러한 방식은 이미 사전 학습이 완료된 모델을 대상으로 하므로, 사전 학습 단계에서부터 교사 모델의 지식을 직접 전달받는 것이 가능한지, 그리고 이를 위해서는 어떤 설정이 최적인지에 대한 체계적인 연구가 부족한 상황이다. 따라서 본 연구의 목표는 사전 학습 증류(Pre-training Distillation, PD)의 효용성을 검증하고, 로그릿 처리, 손실 함수 선택, 스케일링 법칙, 로그릿 획득 전략이라는 네 가지 핵심 설계 공간(design space)을 체계적으로 탐색하여 최적의 PD 구성을 찾는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 LLM의 사전 학습 단계에 Logits-based KD를 도입하고, 이에 영향을 미치는 주요 변수들을 정밀하게 분석한 설계 공간 탐색에 있다. 특히, 교사 모델의 방대한 로그릿 데이터를 효율적으로 저장하기 위한 $\text{top-}p\text{-}k$ 절단(truncation) 방식을 제안하였으며, 단순한 손실 함수 적용을 넘어 학습률 및 증류 비중을 동적으로 조절하는 Warmup-Stable-Decay (WSD) 스케줄러의 효과를 입증하였다. 이를 통해 학생 모델의 크기가 클수록 PD의 이점이 커지며, 무조건 큰 교사 모델을 사용하는 것보다 학생-교사 모델 간의 역량 격차(capacity gap)를 고려하는 것이 중요하다는 통찰을 제공한다.

## 📎 Related Works

지식 증류는 거대 모델의 지식을 작은 모델로 전이하여 모델을 압축하는 기법으로, 컴퓨터 비전, 자연어 처리, 음성 인식 등 다양한 분야에서 활용되어 왔다. 최근 LLM 시대에는 GPT-4와 같은 고성능 모델이 생성한 데이터를 통해 학생 모델을 정렬(alignment)시키는 사후 학습 KD(예: Alpaca)가 주류를 이루고 있다. 

사전 학습 단계의 증류와 관련하여, BERT 시대의 소형 모델 증류 연구들이 존재하지만 이는 파라미터 수가 수백만 개 수준으로, 수십억 개의 파라미터를 가진 최신 LLM의 학습 역학과는 차이가 있다. 또한 Gemma 2나 MiniTron과 같은 최신 LLM들이 사전 학습 증류를 사용하고 있으나, 구체적인 증류 과정과 최적 설정에 대한 상세한 공개 정보가 제한적이다. 본 논문은 이러한 공백을 메우기 위해 체계적인 경험적 연구를 수행함으로써 기존의 단순한 데이터 기반 증류와 차별화한다.

## 🛠️ Methodology

### 전체 시스템 구조 및 목표 함수
본 논문은 학생 모델 $\theta_S$와 교사 모델 $\theta_T$가 있을 때, 다음과 같은 통합 목적 함수를 통해 사전 학습을 수행한다.

$$\theta^*_S = \arg \min_{\theta_S} L = \arg \min_{\theta_S} [(1-\alpha)L_{lm} + \alpha L_{kd}]$$

여기서 $L_{lm}$은 기존의 one-hot 레이블을 사용하는 표준 언어 모델링 손실 함수이며, 다음과 같이 정의된다.

$$L_{lm} = \frac{1}{T} \sum_{t=1}^{T} -\log P_{\theta_S}(x_t | x_{<t})$$

$L_{kd}$는 교사 모델의 로그릿 분포와 학생 모델의 분포 사이의 차이를 줄이는 증류 손실 함수로, 다음과 같이 정의된다.

$$L_{kd} = \frac{1}{T} \sum_{t=1}^{T} L(P_{\theta_S}(x_t | x_{<t}), F(P_{\theta_T}(x_t | x_{<t})))$$

이때 $F$는 교사 모델의 로그릿에 적용되는 절단(Truncation) 및 정규화(Normalization) 연산이며, 다음과 같은 수식을 따른다.

$$F(z) = \text{softmax}\left(\frac{\text{Truncate}(z)}{\tau}\right)$$

여기서 $\tau$는 분포의 날카로움을 조절하는 Temperature 하이퍼파라미터이다.

### 주요 설계 공간 (Design Space)

1.  **Logits Processing**: 교사 모델의 어휘 사전 크기가 매우 크기 때문에(약 150k), 모든 로그릿을 저장하는 것은 불가능하다. 이를 해결하기 위해 $\text{top-}p$로 상위 누적 확률을 먼저 거르고, 다시 $\text{top-}k$로 개수를 제한하는 $\text{top-}p\text{-}k$ 절단 방식을 사용한다.
2.  **Loss Selection**: 증류 손실 함수 $L$로 Kullback-Leibler Divergence (KLD), Negative Log-Likelihood (NLL), Mean Squared Error (MSE)를 비교한다. 또한 $\alpha$ 값을 고정하지 않고 WSD 스케줄러를 통해 동적으로 조절한다.
3.  **Scaling Law**: 학생 모델과 교사 모델의 파라미터 크기 변화 및 사전 학습 코퍼스의 크기에 따른 성능 변화를 분석한다.
4.  **Offline vs Online**: 교사 모델이 이미 학습된 상태에서 로그릿을 미리 생성하여 저장하는 Offline 방식과, 교사 모델의 학습 과정에서 실시간으로 로그릿을 저장하는 Online 방식을 비교한다.

## 📊 Results

### 실험 설정
- **Teacher/Student**: GLM-4-9B를 교사 모델로, 1.9B 모델을 기본 학생 모델로 사용하였다.
- **데이터셋**: HellaSwag, WinoGrande, PIQA, MMLU(영어), KBQA, C3, C-Eval(중국어), GSM8k(수학) 등 다양한 벤치마크를 사용하였다.
- **평가 방법**: 사전 학습 후 10B 규모의 고품질 데이터로 Supervised Fine-Tuning (SFT)를 수행한 뒤 Zero-shot 또는 Few-shot으로 평가하였다.

### 주요 결과
1.  **로그릿 처리**: $\text{top-}p\text{-}k$ 절단에서 $p$와 $k$ 값의 변화는 성능에 큰 영향을 주지 않았으나, 저장 공간을 획기적으로 줄일 수 있었다. Temperature $\tau$의 경우 $\tau \leq 2.0$ 범위에서 유사한 성능 향상이 나타났으며, 너무 높은 $\tau$는 성능을 저하시켰다.
2.  **손실 함수**: NLL과 KLD는 효과적이었으나, MSE는 오히려 성능을 크게 떨어뜨렸다. 특히 $\alpha$를 WSD 스케줄러로 조절하고 이를 WSD 학습률 스케줄러와 결합했을 때 최고의 성능을 보였다. 이는 높은 학습률을 유지하는 구간에서 KD 손실의 비중을 높이는 것이 효율적임을 시사한다.
3.  **스케일링 법칙**: 학생 모델의 크기가 커질수록 PD로 인한 이득이 증가하는 경향을 보였다. 반면, 교사 모델의 크기가 커진다고 해서 반드시 성능이 향상되지는 않았는데, 이는 학생과 교사 간의 역량 격차가 너무 크면 오히려 학습이 어려워지기 때문으로 분석된다.
4.  **Offline vs Online**: Offline 로그릿이 더 우수한 성능을 보였으나, Online 로그릿 역시 학습 후반부의 데이터를 사용할 경우 유의미한 향상을 보였다.

### 최종 최적 구성 (PD*) 결과
최적 설정(top-0.95-50 truncation, $\tau=2.0$, KLD loss, WSD-$\alpha$ 및 WSD-LR)을 적용했을 때, 1.9B 학생 모델의 평균 성능이 크게 향상되었으며, 특히 3.8B 및 6.8B 모델에서도 일관된 성능 향상이 관찰되었다.

## 🧠 Insights & Discussion

본 연구는 LLM의 사전 학습 단계에서 지식 증류가 매우 유효한 전략임을 입증하였다. 특히 주목할 점은 **'역량 격차(Capacity Gap)'**에 대한 분석이다. 단순히 가장 큰 교사 모델을 사용하는 것이 정답이 아니라, 학생 모델의 크기가 교사 모델의 약 10% 이상이 될 때 증류 효과가 뚜렷해진다는 점은 향후 모델 압축 및 전이 학습 설계 시 중요한 가이드라인이 될 것이다.

또한, MSE 손실 함수가 이미지 분류 분야와 달리 LLM 사전 학습에서는 성능을 저하시킨다는 발견은 LLM의 학습 역학이 기존의 KD 연구와는 다른 특성을 가지고 있음을 시사한다. 한계점으로는 다양한 설계 변수 간의 상호작용(interaction)을 모두 실험하기에는 계산 자원이 부족하여 개별 변수 통제 실험 위주로 진행되었다는 점이 있다.

## 📌 TL;DR

본 논문은 LLM의 사전 학습 단계에서 교사 모델의 로그릿을 활용하는 **Pre-training Distillation (PD)**의 설계 공간을 체계적으로 탐색하였다. $\text{top-}p\text{-}k$ 로그릿 절단, KLD 손실 함수, 그리고 WSD 스케줄러의 조합이 최적의 성능을 낸다는 것을 발견하였다. 특히 학생 모델의 크기가 클수록 PD의 혜택을 더 많이 받으며, 교사-학생 간의 적절한 크기 비율이 중요함을 밝혔다. 이 연구는 향후 효율적인 소형 LLM을 구축하려는 연구자들에게 실질적인 하이퍼파라미터 설정 및 학습 전략에 대한 기준을 제시한다.
