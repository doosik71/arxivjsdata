# Sentence-Level or Token-Level? A Comprehensive Study on Knowledge Distillation

Jingxuan Wei, Linzhuang Sun, Yichongong Leng, Xu Tan, Bihui Yu, Ruifeng Guo (2024)

## 🧩 Problem to Solve

신경 기계 번역(Neural Machine Translation, NMT) 분야에서 지식 증류(Knowledge Distillation, KD)는 모델 압축 및 학습 대상 단순화를 위해 널리 사용되는 기법이다. KD는 크게 두 가지 방향인 문장 수준 증류(Sentence-level Distillation)와 토큰 수준 증류(Token-level Distillation)로 나뉜다.

문장 수준 증류는 교사 모델(Teacher Model)이 생성한 의사 타겟 문장(Pseudo target sentence)을 학습 목표로 삼아 전체적인 구조와 의미적 일관성을 학습하게 하며, 토큰 수준 증류는 각 토큰의 출력 분포(Output distribution)를 직접 학습하여 세밀한 지식 전이를 가능하게 한다. 그러나 기존 연구들에서 두 방법의 성능이 시나리오에 따라 서로 다르게 나타나는 경향이 있어, 실제 적용 시 어떤 방법을 선택해야 하는지에 대한 명확한 기준이 부족한 상태였다. 본 논문의 목표는 두 KD 방법의 성능 차이가 발생하는 원인을 분석하여 적절한 선택 기준을 제시하고, 이를 통합한 하이브리드 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 다음과 같다.

1. **가설 제시 및 검증**: 토큰 수준 증류는 "단순한(Simple)" 시나리오에 적합하고, 문장 수준 증류는 "복잡한(Complex)" 시나리오에 더 효과적이라는 가설을 세우고 이를 체계적인 실험을 통해 입증하였다.
2. **복잡성 정의**: NMT 시나리오의 복잡성을 결정하는 세 가지 핵심 요소로 학생 모델의 크기(Model size), 텍스트의 복잡도(Text complexity), 디코딩 절차의 난이도(Decoding difficulty)를 정의하였다.
3. **하이브리드 증류 방법론 제안**: 두 방법의 장점을 모두 활용하기 위해 동적 게이팅 메커니즘(Dynamic Gating Mechanism)을 도입하여 문장 수준과 토큰 수준 증류를 적응적으로 결합한 하이브리드 모델을 제안하였다.

## 📎 Related Works

기존의 KD 연구는 모델 압축과 효율성 증대에 집중해 왔다. 특히 NMT에서는 다음과 같은 두 갈래의 접근 방식이 주를 이루었다.

- **토큰 수준 증류**: 개별 단어나 구문의 번역 정확도를 높이는 데 집중한다. 주로 어휘적 다양성이 높은 텍스트에서 유리하며, 세밀한 토큰 간의 차이를 학습하는 데 효과적이다.
- **문장 수준 증류**: 타겟 문장의 분포를 단순화하여 학생 모델의 학습 난이도를 낮추는 데 집중한다. 전체 문장의 의미적 일관성과 긴 시퀀스의 구조를 포착하는 데 유리하다.

최근 연구 중 HKD와 같이 게이트 기반의 KD 방식이 제안되었으나, 이는 문장 수준과 토큰 수준의 게이트를 독립적으로 운영하여 두 방식을 유기적으로 결합하지 못했다는 한계가 있다. 본 논문은 이를 통합하여 하나의 게이팅 메커니즘으로 최적의 학습 경로를 찾고자 한다.

## 🛠️ Methodology

### 1. 시나리오 복잡성 분석 프레임워크
연구진은 다음과 같은 세 가지 관점에서 복잡성을 정의하고 실험하였다.
- **모델 크기**: 학생 모델의 파라미터 수가 작을수록 제한된 용량에 지식을 압축해야 하므로 복잡한 시나리오로 간주한다.
- **텍스트 복잡도**: 문장 구조가 복잡하거나 어휘 다양성이 높고, 노이즈(삭제, 교체, 스와핑 등)가 많을수록 복잡한 시나리오로 간주한다.
- **디코딩 난이도**: 빔 서치(Beam Search)는 여러 가설을 탐색하므로 복잡하며, 교사 강제(Teacher Forcing)는 정답 프리픽스를 직접 제공하므로 단순한 시나리오로 간주한다.

### 2. 하이브리드 증류 방법 (Hybrid Distillation Method)
복잡성을 사전에 정의하기 어렵다는 점을 해결하기 위해, 입력 시퀀스 $x$에 따라 두 손실 함수를 동적으로 조절하는 게이팅 메커니즘 $G$를 제안한다.

**게이트 함수 $g(x)$**:
입력 시퀀스 $x$에 대해 다음과 같이 시그모이드 함수를 사용하여 게이트 값을 결정한다.
$$g(x) = \frac{1}{1 + e^{-z(x)}}$$
여기서 $z(x)$는 입력 시퀀스 $x$에 의해 결정되는 학습 가능한 함수이다.

**손실 함수 구성**:
- **토큰 수준 손실 ($L_{\text{token-level}}$)**: 교사 모델 $P_t$와 학생 모델 $P_s$의 토큰 분포 간의 차이를 계산한다.
$$L_{\text{token-level}}(x) = -\sum_{j=1}^{m} \sum_{y_j \in V} P_t(y_j|x) \log P_s(y_j|x)$$
- **문장 수준 손실 ($L_{\text{sentence-level}}$)**: 교사 모델이 생성한 최적의 문장 $\hat{y}$에 대한 학생 모델의 음의 로그 우도(NLL)를 계산한다.
$$L_{\text{sentence-level}}(x) = -\log P_s(\hat{y}|x)$$

**최종 통합 손실 함수**:
$$L(x) = g(x) \cdot L_{\text{token-level}}(x) + (1 - g(x)) \cdot L_{\text{sentence-level}}(x)$$
이 구조를 통해 모델은 학습 과정에서 $g(x)$ 값을 조정하며 스스로 어떤 증류 방식에 더 집중할지 결정한다.

## 📊 Results

### 1. 가설 검증 실험 (분석 결과)
- **모델 크기**: 학생 모델의 크기가 작을 때는 문장 수준 증류가 우세하며, 모델 크기가 커질수록 토큰 수준 증류의 성능이 상대적으로 높아지는 역전 현상이 관찰되었다.
- **텍스트 복잡도**: 노이즈가 없는 깨끗한 텍스트에서는 토큰 수준 증류가 유리하지만, 노이즈 수준이 높아질수록(High Noise) 문장 수준 증류가 더 높은 회복력(Resilience)과 성능을 보였다.
- **디코딩 난이도**: 단순한 방식인 Teacher Forcing 환경에서는 토큰 수준 증류가 우세했으나, 복잡한 Beam Search 환경에서는 문장 수준 증류가 더 나은 성능을 기록하였다.

### 2. 하이브리드 방법 성능 평가
IWSLT14 (de$\rightarrow$en) 데이터셋에서 다양한 베이스라인 모델과 비교한 결과, 제안된 하이브리드 증류 방법이 가장 높은 성능을 보였다.
- **Hybrid Distillation**: 39.30 BLEU
- **BiBERT**: 38.61 BLEU
- **Bi-SimCut**: 38.37 BLEU

### 3. 절제 연구 (Ablation Study)
단일 증류 방식과 하이브리드 방식을 비교했을 때, 하이브리드 방식이 두 단일 방식의 성능을 모두 상회하였다.
- 문장 수준 증류: 39.01 BLEU
- 토큰 수준 증류: 39.15 BLEU
- 하이브리드 증류: 39.30 BLEU

## 🧠 Insights & Discussion

### 게이트 값 $G$의 동적 변화 분석
학습 과정 중 게이트 값 $g(x)$의 변화를 분석한 결과, 초기에는 $g(x)$ 값이 낮게 유지되다가 학습이 진행됨에 따라 점진적으로 1에 가깝게 상승하는 패턴이 발견되었다. 이는 모델이 먼저 학습하기 쉬운 '문장 수준의 글로벌 구조'를 익힌 후, 점차 더 어렵고 세밀한 '토큰 수준의 지식'을 학습하는 커리큘럼 학습(Curriculum Learning)과 유사한 양상을 보인다는 것을 의미한다.

### 비판적 해석 및 한계
본 논문은 경험적 실험을 통해 매우 설득력 있는 가설을 제시하고 이를 하이브리드 구조로 해결하였다. 다만, 게이트 함수 $z(x)$가 구체적으로 입력의 어떤 특성을 포착하여 $g(x)$를 변화시키는지에 대한 깊이 있는 해석은 부족하다. 또한, IWSLT14 외의 다른 데이터셋에서 하이브리드 방법의 일반화 성능을 광범위하게 검증한 결과가 더 추가되었다면 제안 방법의 강점이 더 명확해졌을 것이다.

## 📌 TL;DR

본 연구는 NMT의 지식 증류에서 **"토큰 수준 증류는 단순한 시나리오(큰 모델, 깨끗한 텍스트, TF 디코딩)에, 문장 수준 증류는 복잡한 시나리오(작은 모델, 노이즈 텍스트, BS 디코딩)에 적합하다"**는 가설을 입증하였다. 이를 바탕으로 두 방식을 동적으로 결합하는 **게이팅 메커니즘 기반의 하이브리드 KD**를 제안하여 기존 단일 KD 및 최신 베이스라인 모델보다 향상된 번역 성능을 달성하였다. 이 연구는 향후 데이터의 복잡도나 모델 용량에 따라 증류 전략을 유연하게 선택해야 하는 실무적 가이드를 제공한다.
