# Deep Surrogate of Modular Multi Pump using Active Learning

Malathi Murugesan, Kanika Goyal, Laure Barriere, Maura Pasquotti, Giacomo Veneri, Giovanni De Magistris (2022)

## 🧩 Problem to Solve

본 연구가 해결하고자 하는 문제는 해저 에너지 분야에서 사용되는 Modular Multi Pump의 작동 지점(Operating Point), 특히 **Surge Distance(서지 거리)**를 정확하게 추정하는 것이다. 펌프 제어기(Controller)는 펌프의 작동 지점을 파악하여 Surge Limit Line(SLL)으로부터 일정 거리(5-10%)를 유지해야 하며, 이를 통해 펌프 토출 압력이 급격히 증가하여 시스템에 무리를 주는 'Surge' 현상을 방지해야 한다.

이 문제의 핵심적인 어려움은 데이터의 희소성에 있다. 해저에 배치되는 시스템 특성상 센서의 비용이 매우 높고 유지보수가 어렵기 때문에, 설치 가능한 센서의 수가 극히 제한적이다. 따라서 가용한 데이터 양이 적어 기존의 추정 방식으로는 고객사의 요구 수준을 만족하는 정확도를 확보하기 어렵다. 본 논문의 목표는 **Active Learning(능동 학습)** 프레임워크를 적용하여, 최소한의 데이터셋만으로도 작동 지점을 정밀하게 예측하는 Surrogate Model(대리 모델)을 구축하는 것이다.

## ✨ Key Contributions

본 연구의 중심적인 설계 아이디어는 모든 데이터를 무작위로 학습시키는 대신, 모델이 예측에 있어 가장 불확실성을 느끼는 데이터 포인트만을 전략적으로 선택하여 학습시키는 **Active Learning** 기법을 산업용 펌프의 서지 거리 추정에 도입한 것이다. 

특히, 단일 모델이 아닌 **Neural Network Ensemble**을 사용하여 예측값의 분산(Variance)을 측정하고, 이를 불확실성 지표로 활용함으로써 학습에 필요한 데이터의 양을 획기적으로 줄이면서도 높은 예측 성능을 유지하고자 하였다. 이는 실제 산업 현장에서 실험 설계(Design of Experiment, DoE) 비용을 최적화할 수 있는 실질적인 방안을 제시한다.

## 📎 Related Works

논문에서는 실험 설계 및 최적화를 위해 Active Learning이 광범위하게 사용되어 왔음을 언급한다. 구체적으로는 합성 데이터(Synthetic data)를 통한 효율적인 실험 선택 방법(Yu et al., 2006)이나, 실제 애플리케이션을 위한 대규모 데이터셋 수집 프레임워크(Jamieson et al., 2015) 등이 소개되었다.

기존의 접근 방식과의 차별점은, 본 연구가 에너지 분야의 구체적인 물리 시스템인 '다상 펌프(Multiphase Pump)'의 제어라는 실무적인 도메인에 Active Learning을 적용했다는 점이다. 또한, 단순한 데이터 증강이 아니라 물리 모델(PHM, Physical Model) 기반의 데이터셋에서 가장 정보 가치가 높은 샘플을 능동적으로 선택함으로써, 실제 테스트 캠페인의 비용을 낮추는 것에 초점을 맞추었다.

## 🛠️ Methodology

### 1. 전체 파이프라인 및 시스템 구조
본 시스템은 물리 모델(PHM)을 통해 생성된 데이터셋을 바탕으로, 최적의 모델 구조를 선정하고 이를 Active Learning 루프에 통합하는 구조를 가진다.

1.  **모델 선정 단계**: XGBoost, LSTM, MLP 세 가지 모델을 비교하여 가장 성능이 좋은 기본 아키텍처를 결정한다.
2.  **Active Learning 루프**: 
    *   전체 데이터를 Pool Set과 Test Set으로 분리한다.
    *   소량의 초기 데이터로 모델을 학습시킨다.
    *   Pool Set에 대해 모델의 불확실성(분산)을 측정한다.
    *   불확실성이 가장 높은 샘플을 선택하여 학습 세트에 추가하고 모델을 재학습시킨다.

### 2. 데이터 및 주요 방정식
입력 변수는 입구 온도($T_{in}$), 입구 압력($P_{in}$), 회전 속도($N$), 차압($\Delta P$), 총 소비 전력($Power$)이며, 출력 변수는 Surge Distance($SD$)이다. $SD$는 다음과 같은 수식을 통해 계산된다.

$$SD = 100 \frac{\phi - \phi_{surge}}{\phi_{surge}}$$

여기서 $\phi_{surge} = 0.076$이며, $\phi$는 다음과 같이 정의된다.

$$\phi = \frac{Q_{in}}{2.93 \times 10^{-3} N_{speed}}$$

($Q_{in}$은 입구 질량 유량, $N_{speed}$는 회전 속도이다.)

### 3. Active Learning 상세 설계
본 연구에서는 `UQ360` 라이브러리를 활용하여 **Neural Network Ensemble** 기반의 Active Learning을 구현하였다.

*   **아키텍처**: 5개의 신경망으로 구성된 앙상블 모델을 사용한다. 각 네트워크는 입력층(5노드), 3개의 은닉층(각 256 ReLU), 그리고 출력층으로 구성된다. 출력층은 Scaled Hyperbolic Tangent 활성화 함수와 Softplus 활성화 함수를 가진 유닛으로 이루어져 있다.
*   **학습 목표 및 손실 함수**: 가우시안 분포의 음의 로그 가능도(Negative-log-likelihood, NLL)를 손실 함수로 사용한다. 
*   **추론 절차**: 앙상블된 모델들의 예측값의 평균과 로그 분산을 반환하며, 이 중 분산(Variance)이 큰 샘플을 '정보량이 많은 샘플'로 간주하여 선택한다.
*   **최적화**: Adam 옵티마이저를 사용하며, 학습률은 $0.001$에서 시작하여 10 에포크 이후 $0.99$ 비율로 감쇠시킨다.

## 📊 Results

### 1. 실험 설정
*   **데이터셋**: PHM 툴로 생성된 63,116개의 샘플.
*   **비교 모델**: XGBoost, LSTM, MLP.
*   **평가 지표**: $R^2$, RMSE, Max Error, MAPE 및 수용 기준(Acceptance Criteria: 오차 $\pm 4\%$ 이내) 달성률.

### 2. 모델 선정 결과
전체 데이터셋(학습 50,492개, 테스트 12,624개)을 사용한 결과, MLP 모델이 수용 기준 달성률 $61.3\%$로 가장 우수한 성능을 보였다. 반면 XGBoost는 $49.7\%$, LSTM은 $52.5\%$를 기록하였다. 또한, 학습 데이터를 대폭 줄여 실험했을 때도 MLP가 가장 낮은 오차를 유지하였다. 이에 따라 최종 Active Learning 실험에는 MLP 앙상블이 채택되었다.

### 3. Active Learning 결과
제한된 샘플 수($n=2200$) 환경에서 기본 MLP 모델(무작위 선택)과 Active Learning MLP 모델을 비교한 결과는 다음과 같다.

*   **데이터 효율성**: Active Learning 모델은 단 1,000개의 샘플만으로도 매우 높은 성능을 보였으며, 이는 일반 MLP 모델이 무작위 샘플링으로 약 3,000개를 학습했을 때의 성능과 유사한 수준이다.
*   **결론**: AL을 통해 학습 데이터의 양을 3배 가까이 줄이면서도 유사한 정확도를 확보할 수 있음을 정량적으로 확인하였다.

## 🧠 Insights & Discussion

본 논문은 산업 현장에서 데이터 수집 비용이 매우 높은 경우, 모델의 불확실성을 이용한 능동적 샘플링이 매우 효과적인 해결책이 될 수 있음을 입증하였다. 특히 앙상블 모델을 통해 예측값의 분산을 측정함으로써, 어떤 데이터가 모델의 성능 향상에 기여할지를 수학적으로 판단한 점이 강점이다.

다만, 본 연구의 실험은 실제 측정 데이터가 아닌 물리 모델(PHM)에서 생성된 합성 데이터를 기반으로 수행되었다는 한계가 있다. 실제 현장 데이터는 PHM 데이터보다 노이즈가 심하고 분포가 다를 수 있으므로, 합성 데이터에서의 성공이 실제 환경에서 동일하게 나타날지는 추가적인 검증이 필요하다. 그럼에도 불구하고, 실제 적용 시 각 장비마다 모델을 재학습시켜야 하는 비용을 AL이 획기적으로 줄여줄 수 있다는 점은 실무적으로 매우 중요한 시사점을 제공한다.

## 📌 TL;DR

본 연구는 해저 펌프의 Surge Distance를 예측하는 대리 모델을 구축하기 위해 **Neural Network Ensemble 기반의 Active Learning** 기법을 제안하였다. 실험 결과, 무작위 데이터 선택 방식보다 훨씬 적은 양의 데이터(약 1/3 수준)만으로도 동등한 수준의 예측 정확도를 달성할 수 있었다. 이는 고비용의 산업용 센서 데이터 수집 환경에서 실험 설계(DoE) 비용을 최적화하고 모델 구축 효율성을 극대화하는 데 기여할 수 있다.
