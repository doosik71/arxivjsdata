# Analysis of Stopping Active Learning based on Stabilizing Predictions

Michael Bloodgood, John Grothendieck (2013)

## 🧩 Problem to Solve

본 논문은 자연어 처리(NLP) 분야에서 새로운 시스템을 구축할 때 직면하는 가장 큰 병목 현상 중 하나인 데이터 어노테이션 비용 문제를 해결하기 위한 Active Learning(AL)의 중단 시점 결정 문제를 다룬다.

Active Learning의 핵심은 모델 학습에 가장 도움이 되는 예제를 선택적으로 라벨링하여 전체 어노테이션 비용을 줄이는 것이다. 그러나 AL의 실제 효용성을 확보하기 위해서는 '언제 학습을 멈출 것인가'에 대한 정확한 기준이 필요하다. 너무 일찍 학습을 중단하면 모델의 성능이 낮아지고, 너무 늦게 중단하면 불필요한 어노테이션 작업에 인적 자원이 낭비되어 AL의 이점이 사라지게 된다. 따라서 본 논문의 목표는 모델 예측의 안정성을 기반으로 한 중단 방식인 Stabilizing Predictions(SP) 방법론에 대해 최초의 이론적 분석을 제공하고, 그 작동 원리를 수학적으로 증명하는 것이다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 연속적으로 학습된 두 모델이 특정 데이터셋(Stop set)에 대해 내리는 예측의 일치도가 매우 높다면, 추가적인 데이터 학습이 모델의 성능 향상에 기여하는 바가 매우 적을 것이라는 직관에 기반한다.

주요 기여 사항은 다음과 같다.

1. **SP 방법론의 이론적 분석**: Cohen's Kappa 계수를 이용한 모델 간 일치도 분석이 어떻게 성능 변화량의 상한선(Upper bound)을 결정짓는지 이론적으로 규명하였다.
2. **성능 차이의 상한선 증명**: Kappa 일치도 $T$가 특정 임계치를 넘을 때, 두 모델 간의 F-measure 성능 차이 $|\Delta F|$가 $\frac{4(1-T)}{T}$ 이하로 제한됨을 수학적으로 증명하였다.
3. **분산 분석을 통한 Stop set 크기 제안**: Kappa 추정치의 분산이 Stop set의 크기에 반비례함을 분석하여, 실무적으로 신뢰할 수 있는 중단 결정을 내리기 위한 데이터셋 규모에 대한 근거를 제시하였다.

## 📎 Related Works

기존의 Active Learning 연구들은 주로 '어떤 데이터를 선택하여 라벨링할 것인가'라는 선택 전략(Selection methods)에 집중해 왔으며, 중단 시점을 결정하는 문제는 미래의 과제로 미뤄두는 경향이 있었다. 최근 들어 모델의 확신도(Confidence), 에러율, 또는 안정성(Stability)에 기반한 휴리스틱(Heuristic) 방식의 중단 기준들이 제안되었으나, 이러한 방법들은 소수의 데이터셋에서만 실험적으로 성공했을 뿐 이론적 근거가 부족하여 범용적인 적용에 한계가 있었다.

본 논문에서 다루는 SP 방법은 기존의 보수적인(Conservative) 중단 방식과 달리 공격적인(Aggressive) 방식으로 분류된다. 보수적인 방식은 성능 손실을 최소화하기 위해 필요 이상의 데이터를 더 라벨링하는 경향이 있는 반면, SP 방법은 성능 손실 없이 최대한 빠르게 학습을 중단하여 어노테이션 비용을 극대화하여 절감하는 것을 목표로 한다.

## 🛠️ Methodology

### 1. 전체 파이프라인 및 시스템 구조

SP 방법의 전체 흐름은 다음과 같다.

1. 라벨이 없는 대규모의 대표 데이터셋인 **Stop set**을 구성한다.
2. AL의 각 반복(Iteration)마다 현재 모델 $M_t$와 이전 모델 $M_{t-1}$을 Stop set에 적용하여 예측값을 얻는다.
3. 두 모델의 예측값 사이의 **Cohen's Kappa** 일치도를 계산한다.
4. Kappa 값이 미리 정해진 임계치(예: 0.99)를 3회 연속으로 초과하면 학습을 중단한다.

### 2. Cohen's Kappa ($\kappa$) 정의

두 모델의 일치도를 측정하기 위해 단순히 일치하는 비율(Percentage agreement)을 사용하지 않고, 우연히 일치할 확률을 고려한 Cohen's Kappa를 사용한다.
$$\text{agreement} = \frac{A_o - A_e}{1 - A_e}$$
여기서 $A_o$는 관찰된 일치도(Observed agreement)이며, $A_e$는 우연에 의해 기대되는 일치도(Agreement expected by chance)이다. $A_e$는 각 모델이 각 카테고리에 할당할 확률의 합으로 계산된다.

### 3. Kappa 추정치의 분산 분석

실제 적용 시에는 모집단 확률을 알 수 없으므로 표본을 통해 $\hat{K}$를 추정한다. 본 논문은 Delta method를 사용하여 $\hat{K}$의 대표본 분산 $\text{Var}(\hat{K})$을 분석하였다. 분석 결과, 분산은 Stop set의 크기 $n$에 반비례한다. 즉, Stop set의 크기가 클수록 Kappa 추정치의 변동성이 줄어들어 더 안정적인 중단 결정이 가능하며, 이는 라벨이 없는 데이터를 사용하는 Stop set의 특성상 크기를 매우 크게 설정할 수 있다는 점이 SP 방법의 강력한 장점이 된다.

### 4. 성능 차이에 대한 이론적 상한선 (Theoretical Bounds)

본 논문의 핵심은 Kappa 일치도와 F-measure 성능 차이 사이의 관계를 증명한 것이다.

- **최악의 경우 (Worst Case)**:
  두 모델의 Kappa 일치도가 $T$보다 클 경우, 두 모델의 F-measure 성능 차이 $|\Delta F|$는 다음과 같은 상한선을 가진다.
  $$|\Delta F| \le \frac{4(1-T)}{T}$$

- **정밀도(Precision)를 고려한 경우**:
  두 모델이 모두 긍정(Positive)으로 예측한 결과의 정밀도를 $p$라고 가정하면, 상한선은 다음과 같이 더 타이트하게 조정된다.
  $$|\Delta F| \le \frac{4(1-T)}{(p+1)T}$$
  여기서 $p=1$(완벽한 정밀도)일 경우, 상한선은 $\frac{2(1-T)}{T}$로 줄어든다.

## 📊 Results

본 논문은 새로운 실험을 수행하기보다 기존의 실험 결과(Bloodgood and Vijay-Shanker, 2009a)를 이론적으로 뒷받침하는 분석에 집중한다.

- **분석 대상**: NER-DNA, NER-cellType, Reuters, 20 Newsgroups, WebKB 등 다양한 NLP 데이터셋.
- **측정 지표**: $\hat{K}$의 추정 분산 $\text{Var}(\hat{K})$.
- **실험 결과**:
  - Stop set의 크기를 $n=2000$으로 설정했을 때, 대부분의 데이터셋에서 $\hat{K}$의 분산이 매우 낮게 측정되었다(평균 $\approx 0.0000209$).
  - 이는 $n=2000$ 정도의 규모가 Kappa 값을 정밀하게 추정하여 안정적으로 학습을 중단하기에 충분한 크기였음을 시사한다.
  - 또한, 이론적으로 유도된 성능 상한선 공식은 왜 SP 방법이 성능 손실 없이 공격적으로 학습을 중단할 수 있었는지를 수학적으로 설명해 준다.

## 🧠 Insights & Discussion

**강점 및 의의**
본 논문은 그동안 휴리스틱에 의존했던 AL 중단 기준에 대해 최초의 이론적 분석을 제공하였다. 특히 Cohen's Kappa라는 통계적 지표와 모델의 실제 성능 지표인 F-measure를 수학적으로 연결함으로써, "모델 간 예측이 안정화되면 성능 향상도 정체된다"는 직관을 공식화하였다.

**한계 및 비판적 해석**

1. **상한선의 느슨함 (Loose Bound)**: 저자가 명시했듯이, 제시된 $\frac{4(1-T)}{T}$ 상한선은 모든 예측 차이가 한쪽 모델의 성능을 극대화하고 다른 쪽을 최소화하는 '최악의 상황'을 가정한 것이다. 따라서 실제 환경에서의 성능 차이는 이 상한선보다 훨씬 작을 가능성이 높다.
2. **데이터 분포 가정**: Stop set이 모집단을 편향 없이 대표한다는 가정이 전제되어야 한다. 만약 Stop set이 편향되어 있다면, 여기서 계산된 Kappa 값이 실제 테스트 데이터에서의 성능 변화를 보장하지 못할 수 있다.
3. **이진 분류 한정**: 본 분석은 이진 분류(Binary Classification) 상황을 가정하고 진행되었다. 다중 클래스 분류(Multi-class Classification) 상황으로 확장했을 때 Kappa와 F-measure의 관계가 어떻게 변하는지에 대한 논의는 부족하다.

## 📌 TL;DR

이 논문은 Active Learning에서 모델 예측의 안정성(Stabilizing Predictions)을 이용해 학습을 중단하는 방법론을 이론적으로 분석하였다. Cohen's Kappa 일치도가 높을수록 두 모델 간의 F-measure 성능 차이가 수학적으로 제한됨을 증명함으로써, SP 방법이 효율적으로 어노테이션 비용을 줄이면서도 성능을 유지할 수 있음을 보였다. 이 연구는 향후 AL 전략 전환 시점 결정이나 Co-training 중단 시점 연구에 중요한 이론적 토대가 될 것으로 보인다.
