# DECAL: DEployable Clinical Active Learning

Y. Logan, M. Prabhushankar and G. AlRegib (2022)

## 🧩 Problem to Solve

본 논문은 의료 영상 진단 분야에서 기존의 Active Learning(능동 학습) 기법들이 가진 실용적 한계를 해결하고자 한다. 일반적인 자연 이미지 대상의 머신러닝 시스템은 이미지 내의 속성만으로 의사결정을 내린다고 가정한다. 하지만 실제 의료 현장에서의 진단은 의료 영상뿐만 아니라 전자의무기록(Electronic Medical Records, EMR)에 포함된 환자의 상태, 인구통계학적 정보 등 다양한 임상적 맥락을 함께 고려하여 이루어진다.

특히 의료 데이터에서는 동일한 질환이라 하더라도 환자의 연령이나 상태에 따라 시각적 특성이 매우 다르게 나타나는 Intra-class diversity(클래스 내 다양성)가 심하게 나타난다. 기존의 Active Learning 알고리즘들은 이러한 임상적 특성을 반영하지 못하며, 단순히 이미지의 정보량이나 불확실성만을 측정하여 샘플을 선택한다. 이로 인해 특정 환자 군에 편향된 샘플이 선택될 위험이 있으며, 이는 안전이 최우선인 의료 분야에서 치명적인 결과로 이어질 수 있다. 따라서 본 연구의 목표는 EMR 데이터를 통합하여 실제 임상 환경에서 배포 가능한 실용적인 Active Learning 프레임워크인 DECAL을 설계하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 EMR 데이터, 특히 **환자 식별자(Patient Identity)**를 Active Learning의 샘플 선택 과정에 제약 조건으로 추가하는 '플러그인(Plug-in)' 방식의 프레임워크를 제안하는 것이다.

단순히 모델의 불확실성(Uncertainty)이나 다양성(Diversity)만을 따지는 것이 아니라, 선택된 배치(Batch) 내의 샘플들이 서로 다른 환자로부터 추출되도록 강제함으로써 데이터의 대표성을 높이고 클래스 내 다양성을 효과적으로 확보한다. 이는 기존의 어떤 Active Learning 알고리즘과도 결합할 수 있는 범용적인 구조로 설계되어, 기존 알고리즘들이 의료 데이터에서 더 빠르고 정확하게 일반화 성능을 갖도록 돕는다.

## 📎 Related Works

Active Learning은 전문가의 레이블링 비용을 줄이기 위해 가장 정보 가치가 높은 샘플을 선택하는 것을 목표로 한다. 기존 연구들은 주로 다음과 같은 획득 함수(Acquisition Function)를 사용해 왔다.
- **앙상블 학습(Ensemble Learning)** 기반의 정보량 측정
- **확률적 불확실성(Probabilistic Uncertainty)** 측정
- **데이터 표현(Data Representation)** 기반의 다양성 측정

이러한 기법들이 의료 영상 분야에도 일부 적용되었으나, 대부분 자연 이미지 도메인에서 개발된 방식을 그대로 옮겨온 수준이었다. 논문은 기존 방식들이 의료 진단 프로세스의 특성, 즉 이미지 외의 임상 데이터(EMR)가 의사결정에 결정적인 역할을 한다는 점을 간과했다는 점을 지적하며, DECAL이 이러한 임상적 맥락을 통합함으로써 기존 접근 방식과 차별화된다고 설명한다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조
DECAL은 기존의 Active Learning 알고리즘 위에 얹어서 사용하는 플러그인 형태의 프레임워크이다. 전체 과정은 크게 '초기화 단계'와 '샘플 획득 단계'로 나뉜다.

### 주요 구성 요소 및 작동 원리
1. **EMR 기반 초기화 (DECAL Initialization):**
   - 일반적인 Active Learning은 초기 학습 세트를 무작위로 선택한다. 그러나 이는 불균형한 데이터 분포에서 대표성 없는 샘플을 선택할 위험이 있다.
   - DECAL은 unlabeled pool에서 **서로 다른 환자 ID를 가진 이미지들을 우선적으로 선택**하여 초기 학습 세트를 구성한다. 이를 통해 모델이 학습 시작 단계부터 최대한 상이한 이미지들을 접하게 하여 일반화 성능을 높인다.

2. **EMR 제약 기반 샘플 선택 (Patient-Aware Sample Selection):**
   - 모델이 어떤 획득 함수(예: Entropy, BADGE 등)를 통해 정보량이 높은 샘플들을 추천하더라도, DECAL은 여기에 '환자 ID' 제약을 적용한다.
   - 다음 학습 배치에 추가될 샘플들은 **unlabeled pool 내에서 서로 유니크한 환자 ID**를 가져야 한다. 즉, 한 환자로부터 너무 많은 이미지가 선택되는 것을 방지하고 다양한 환자의 데이터를 균형 있게 수집한다.

### 학습 및 추론 절차
- **모델 아키텍처:** ResNet-18, ResNet-50, DenseNet-121을 사용하며, 사전 학습된 모델(Pre-trained model)을 사용하지 않고 처음부터 학습시킨다.
- **최적화:** Adam optimizer를 사용하며, 학습률은 $1.5 \times 10^{-4}$로 설정한다.
- **학습 중단 조건:** 각 라운드마다 ResNet과 DenseNet 모델이 각각 훈련 세트에서 98%와 94%의 정확도에 도달할 때까지 학습시킨다.
- **반복 과정:** 모델 가중치를 리셋하고 새로운 샘플을 추가하며 이 과정을 반복한다.

## 📊 Results

### 실험 설정
- **데이터셋:** 
    - **OCT:** 3가지 망막 질환 데이터 (CNV, DME, Drusen).
    - **X-Ray:** 건강한 상태 및 2가지 유형의 폐렴(Viral, Bacterial) 데이터.
- **비교 대상 (Baselines):** Random sampling, Margin, Least Confidence, Entropy, BADGE.
- **평가 지표:** 테스트 세트에 대한 평균 정확도(Average Accuracy).

### 주요 결과
1. **초기화 전략의 효과:**
   - 대규모 초기 학습 세트를 사용했을 때, DECAL 초기화 방식이 무작위 초기화보다 OCT에서 $5.59\%$, X-Ray에서 $7.02\%$ 더 높은 평균 정확도를 보였다.
   - 소규모 초기 세트에서도 DECAL 초기화가 더 이른 라운드에서 무작위 추측 수준($33\%$)을 벗어나 일반화 성능을 확보함을 확인하였다.

2. **Active Learning 성능 향상:**
   - 20라운드에 걸친 학습 결과, DECAL을 적용했을 때 전반적인 일반화 성능이 약 $4.81\%$ 향상되었다.
   - 학습 곡선(Learning Curve) 분석 결과, DECAL은 모든 베이스라인 알고리즘과 비슷하거나 더 뛰어난 성능을 보였으며, 특히 초기 라운드에서 성능 상승 속도가 더 빨랐다. 이는 DECAL이 더 적은 수의 샘플로도 빠르게 최적의 모델에 도달할 수 있음을 시사한다.

## 🧠 Insights & Discussion

### 강점
본 연구는 복잡한 수학적 모델 수정 없이, '환자 ID'라는 단순한 임상적 제약 조건을 추가하는 것만으로도 의료 영상 Active Learning의 성능을 유의미하게 높일 수 있음을 증명하였다. 특히 의료 데이터의 핵심 난제인 Intra-class diversity 문제를 데이터 선택 단계에서 해결하려 한 점이 돋보인다.

### 한계 및 미해결 질문
본 논문에서 사용한 EMR 데이터는 오직 '환자 식별자(Patient ID)'에 국한되어 있다. 실제 EMR에는 인구통계학적 정보, 다른 검사 결과 등 훨씬 풍부한 정보가 포함되어 있다. 따라서 환자 ID 외의 다른 EMR 속성들을 어떻게 Active Learning 프레임워크에 통합할 것인가는 여전히 해결해야 할 과제로 남아 있다.

### 비판적 해석
DECAL은 매우 실용적인 접근법이지만, 방법론적으로는 기존 알고리즘의 '필터' 역할을 수행하는 것에 가깝다. 즉, 새로운 획득 함수를 제안한 것이 아니라 기존 함수의 선택 결과에 제약을 가한 것이다. 그럼에도 불구하고, 의료 도메인에서는 데이터의 분포와 수집 방식이 특수하므로, 이러한 도메인 지식(Domain Knowledge)을 반영한 제약 조건이 순수 알고리즘적인 개선보다 더 큰 효과를 낼 수 있음을 보여준 유의미한 연구라고 평가할 수 있다.

## 📌 TL;DR

DECAL은 의료 영상 Active Learning 시 EMR의 환자 식별자(Patient ID)를 활용하여 샘플의 다양성을 강제로 확보하는 플러그인 프레임워크이다. 무작위 샘플링의 한계를 극복하고 환자 간 다양성을 보장함으로써, 기존 AL 알고리즘들이 의료 데이터에서 더 빠르고 정확하게 일반화되도록 돕는다. 이 연구는 향후 더 정교한 EMR 데이터를 통합한 지능형 의료 진단 시스템 구축의 기반이 될 가능성이 높다.
