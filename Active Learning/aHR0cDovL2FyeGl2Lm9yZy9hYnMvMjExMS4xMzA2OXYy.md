# Continual Active Learning Using Pseudo-Domains for Limited Labelling Resources and Changing Acquisition Characteristics

Matthias Perkonigg, Johannes Hofmanninger, Christian Herold, Helmut Prosch, Georg Langs (2022)

## 🧩 Problem to Solve

의료 영상 분석 분야에서 딥러닝 모델의 성능과 신뢰성은 스캐너 하드웨어의 변경, 촬영 프로토콜의 변화, 혹은 병원 정책의 변경으로 인한 데이터 획득 특성의 변화(domain shift)에 의해 크게 저하된다. 이러한 환경에서 모델을 지속적으로 업데이트하기 위해 Continual Learning(CL)을 적용할 수 있으나, 의료 영상의 특성상 전문가의 수동 레이블링 작업에 막대한 비용과 시간이 소요된다는 치명적인 한계가 존재한다.

기존의 Active Learning(AL)은 레이블링 비용을 줄이는 데 유용하지만, 대다수의 기존 방법론은 학습 데이터의 분포가 변화하는 상황을 고려하지 않거나, 데이터가 어떤 도메인(예: 특정 제조사의 스캐너)에 속하는지에 대한 정보가 미리 주어져 있다고 가정한다. 그러나 실제 임상 환경에서는 메타데이터 인코딩의 가변성으로 인해 도메인 정보를 명확히 알 수 없는 경우가 많다. 따라서 본 논문은 **도메인 정보가 없는 상태에서 지속적으로 유입되는 데이터 스트림 내의 도메인 변화를 자동으로 감지하고, 제한된 레이블링 예산 내에서 가장 정보 가치가 높은 샘플을 선택하여 모델을 업데이트함으로써 치명적 망각(catastrophic forgetting)을 방지하는 방법론**을 제안하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 **CASA (Continual Active Learning for Scanner Adaptation)** 프레임워크의 제안이다. CASA의 중심적인 설계 아이디어는 다음과 같다.

1. **Pseudo-Domain (가상 도메인) 개념 도입**: 도메인에 대한 사전 지식 없이, 이미지의 스타일 임베딩(style embedding)을 이용하여 유사한 외형을 가진 이미지들을 그룹화하고 이를 '가상 도메인'으로 정의한다.
2. **자동 도메인 감지 및 능동적 샘플링**: 데이터 스트림에서 기존 가상 도메인에 속하지 않는 샘플(outlier)이 일정 수준 이상 모이면 새로운 가상 도메인이 생성된 것으로 판단하고, 해당 도메인의 샘플을 우선적으로 레이블링하여 모델에 반영한다.
3. **균형 잡힌 Rehearsal Memory 구성**: 새로운 도메인을 학습할 때 발생하는 치명적 망각을 막기 위해 Rehearsal Memory를 사용하며, 이때 각 가상 도메인별로 샘플 수를 균형 있게 유지하여 과거의 지식을 보존한다.

## 📎 Related Works

본 논문은 다음의 세 가지 연구 분야를 결합하여 기존의 한계를 극복하고자 한다.

- **Domain Adaptation (DA)**: 서로 다른 획득 설정 간의 차이를 줄이는 연구이다. Harmonization이나 일반적인 DA 방법들은 소스 도메인과 타겟 도메인의 데이터를 동시에 사용할 수 있다고 가정하지만, 실제 임상에서는 데이터가 순차적으로 유입되므로 적용에 한계가 있다.
- **Continual Learning (CL)**: 새로운 지식을 학습하면서 이전 지식을 잊지 않게 하는 연구이다. Rehearsal method(과거 샘플의 일부를 저장하여 함께 학습하는 방식)가 효과적임이 알려져 있으나, 모든 유입 데이터를 레이블링해야 한다는 전제가 있어 비용 효율성이 낮다.
- **Active Learning (AL)**: 가장 정보량이 많은 샘플을 선택해 레이블링 비용을 최소화하는 연구이다. 하지만 기존의 stream-based selective sampling 방식들은 학습 중 데이터 분포가 변하는 도메인 시프트 현상을 적절히 다루지 못한다.

CASA는 이러한 한계들을 통합적으로 해결하기 위해, 도메인 정보를 모르는 상태에서 스타일 기반의 가상 도메인 감지와 균형 잡힌 메모리 관리를 결합한 능동적 지속 학습 방식을 제안한다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

CASA는 크게 **Pseudo-domain module**, **Task module**, 그리고 두 가지 메모리(**Outlier memory $O$**, **Training rehearsal memory $M$**)로 구성된다. 전체 프로세스는 다음과 같다.

1. 특정 스캐너 데이터로 Task network를 사전 학습(Base training)한다.
2. 데이터 스트림 $S$에서 미니배치를 가져와 각 이미지의 스타일 임베딩을 추출한다.
3. Pseudo-domain module이 해당 이미지를 기존 가상 도메인에 할당하거나, 할당 불가능할 경우 Outlier memory $O$로 보낸다.
4. $O$에 샘플이 쌓여 새로운 가상 도메인이 발견되면, 오라클(전문가)에게 레이블을 요청하여 $M$에 저장한다.
5. $M$에서 샘플을 추출하여 Task network를 업데이트한다.

### 2. 주요 구성 요소 및 상세 설명

#### 스타일 임베딩 (Style Embedding)

이미지의 스타일을 수치화하기 위해 사전 학습된 스타일 네트워크를 사용한다. 특정 레이어 $l$에서 피처 맵의 활성화 값 $f_{il}(x)$와 $f_{jl}(x)$의 내적을 통해 Gram Matrix $G^l$을 계산한다.

$$G_{ij}^l(x) = \frac{1}{N^l M^l} f_{il}(x)^T f_{jl}(x)$$

여기서 $N^l$은 피처 맵의 수, $M^l$은 벡터화된 피처 맵의 요소 수이다. 여러 레이어의 Gram Matrix를 결합한 후 PCA(주성분 분석)를 적용하여 고정된 크기의 스타일 임베딩 $e(x)$를 생성한다.

#### 가상 도메인 할당 및 감지 (Pseudo-domain Assignment & Detection)

각 가상 도메인 $d$는 중심점 $c_d$와 최대 거리 임계값 $d_d$를 가진다. 새로운 이미지 $x$의 도메인 $p(x)$는 다음과 같이 결정된다.

$$p(x) = \begin{cases} \arg \min_{d} |e(x) - c_d| & \text{if } |e(x) - c_d| < d_d \\ -1 & \text{otherwise} \end{cases}$$

만약 $p(x) = -1$이면 해당 샘플은 Outlier memory $O$에 저장된다. $O$ 내의 샘플들 간의 쌍별 유클리드 거리를 계산하여 모든 거리값이 임계값 $t$보다 작은 밀집 영역이 발견되면 새로운 가상 도메인이 생성된 것으로 간주하고, 해당 샘플들을 레이블링하여 $M$으로 옮긴다.

#### Rehearsal Memory 관리

치명적 망각을 방지하기 위해 크기가 $M$으로 고정된 메모리를 사용하며, 각 가상 도메인이 최대 $M/D$ ($D$는 도메인 수)개의 샘플을 점유하도록 균형을 맞춘다. 새로운 도메인이 추가되면 기존 도메인의 샘플 중 일부를 삭제하여 공간을 확보한다. 특정 도메인의 샘플을 교체할 때는 스타일 임베딩 거리가 가장 가까운 샘플을 찾아 교체함으로써 도메인 내 다양성을 유지한다.

#### 학습 절차 및 손실 함수

Task module은 $M$에서 샘플링된 미니배치 $T$를 사용하여 표준적인 지도 학습 방식으로 업데이트된다. 레이블링은 주어진 예산 $\beta$ 내에서만 수행되며, 특정 가상 도메인의 성능 지표 $\bar{p}_j$가 임계값 $k$에 도달하면 해당 도메인의 학습이 완료된 것으로 보고 더 이상 레이블을 요청하지 않는다.

## 📊 Results

### 실험 설정

- **평가 작업**: 심장 분할(Cardiac segmentation), 폐 결절 검출(Lung nodule detection), 뇌 연령 추정(Brain age estimation).
- **비교 대상**: Joint Model (JM), Domain Specific Models (DSM), Naive AL (NAL), Uncertainty AL (UAL).
- **지표**: Dice Score (DSC), Average Precision (AP), Mean Absolute Error (MAE).

### 주요 결과

1. **도메인 간 성능**: 세 가지 작업 모두에서 CASA는 NAL과 UAL보다 우수한 성능을 보였다. 특히 폐 결절 검출 작업에서 CASA는 다른 AL 방법론들이 놓쳤던 새로운 도메인(L4)의 데이터를 효과적으로 레이블링하여 베이스 모델보다 월등한 성능 향상을 이끌어냈다.
2. **망각 방지 및 전이 성능**: Backward Transfer (BWT) 값이 0에 가깝게 유지되어 치명적 망각이 거의 발생하지 않았음을 확인하였으며, Forward Transfer (FWT) 측면에서도 기존 지식을 통해 새로운 도메인에 더 빠르게 적응하는 모습을 보였다.
3. **레이블링 예산($\beta$) 및 메모리 크기($M$)의 영향**:
    - 예산 $\beta$가 너무 적으면 새로운 도메인이 나타나기 전에 예산이 소진되어 적응에 실패하는 경우가 발생한다.
    - 메모리 $M$이 너무 작으면 가상 도메인 기반의 균형 잡힌 샘플링의 이점이 사라지며, 너무 크면 새로운 도메인으로의 적응 속도가 느려지는 경향이 있다.
4. **메모리 구성 분석**: t-SNE 시각화 결과, UAL과 NAL은 메모리가 초기 도메인(C1)에 치우쳐 있는 반면, CASA는 모든 가상 도메인에 걸쳐 샘플이 균형 있게 분포되어 있음을 확인하였다.
5. **데이터 스트림 순서의 영향**: 데이터가 순차적이지 않고 무작위(Random stream)로 섞여 유입될 경우, 도메인 시프트 현상이 사라지므로 CASA의 이점이 줄어들며 오히려 NAL이나 UAL이 더 나은 성능을 보이기도 한다. 이는 CASA가 '도메인 시프트가 존재하는 환경'에 최적화된 구조임을 입증한다.

## 🧠 Insights & Discussion

### 강점 및 의의

본 연구는 의료 영상 분야에서 실제 발생 가능한 시나리오(도메인 정보를 모르는 상태에서의 지속적 데이터 유입 및 제한된 레이블링 자원)를 정확히 모델링하였다. 특히 스타일 임베딩을 통한 가상 도메인 구축과 이를 이용한 메모리 밸런싱 전략은 전문가의 개입을 최소화하면서도 모델의 범용성을 유지하는 효과적인 방법임을 보여주었다.

### 한계 및 향후 과제

1. **개인정보 보호 문제**: Rehearsal memory에 실제 환자의 이미지를 저장해야 하므로 데이터 프라이버시 문제가 발생할 수 있다. 저자들은 이를 해결하기 위해 이미지를 직접 저장하지 않는 pseudo-rehearsal 방식과의 결합을 제안한다.
2. **메모리 및 예산의 동적 확장**: 현재는 $M$과 $\beta$가 고정되어 있으나, 실제 환경에서는 데이터 유입량에 따라 이를 유연하게 확장하는 전략이 필요하다.
3. **다중 사이트 환경**: 현재는 단일 스트림을 가정하고 있으나, 여러 병원에서 동시에 데이터가 유입되는 multi-site 환경으로 확장하여 Federated Learning과 결합할 필요가 있다.

## 📌 TL;DR

본 논문은 의료 영상의 도메인 시프트 문제를 해결하기 위해 스타일 기반의 가상 도메인을 자동으로 감지하고, 제한된 예산 내에서 효율적으로 샘플을 선택해 학습하는 **CASA** 프레임워크를 제안한다. CASA는 Rehearsal Memory를 가상 도메인별로 균형 있게 유지함으로써 치명적 망각을 방지하며, 다양한 의료 영상 작업(분할, 검출, 회귀)에서 기존의 능동적 학습 방법론보다 뛰어난 적응 성능을 입증하였다. 이 연구는 향후 의료 AI 모델이 임상 현장에서 지속적으로 배포되고 업데이트되는 파이프라인을 구축하는 데 중요한 기초가 될 것으로 보인다.
