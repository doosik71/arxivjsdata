# Online Active Learning for Soft Sensor Development using Semi-Supervised Autoencoders

Davide Cacciarelli, Murat Kulahci, John Tyssedal (2022)

## 🧩 Problem to Solve

산업 및 화학 공정에서는 직접 측정이 어렵거나 측정 비용이 높은 공정 변수를 실시간으로 예측하기 위해 데이터 기반의 소프트 센서(Soft Sensor)를 광범위하게 사용한다. 이러한 소프트 센서의 회귀 모델을 학습시키기 위해서는 다량의 라벨링된 데이터(Labeled data)가 필요하지만, 실제 산업 현장에서 품질 검사를 통해 라벨 정보를 획득하는 것은 시간과 비용 측면에서 매우 비효율적이다.

기존의 능동 학습(Active Learning, AL) 방법론들이 이러한 비용 문제를 해결하기 위해 정보량이 많은 데이터를 선택적으로 쿼리하는 방식을 제안해 왔으나, 대부분의 연구는 이미 수집된 데이터 풀에서 최적의 샘플을 선택하는 오프라인(Pool-based) 설정에 집중되어 있다. 하지만 고속 생산 공정과 같이 데이터가 빠르게 생성되고 지나간 데이터를 다시 획득할 수 없는 환경에서는 스트림 기반(Stream-based) 능동 학습이 필수적이다. 따라서 본 논문의 목표는 스트림 기반 시나리오에서 효율적으로 정보량이 많은 데이터를 선택하는 기법을 제안하고, 반지도 학습(Semi-supervised learning) 구조를 통해 이를 최적화하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **직교 오토인코더(Orthogonal Autoencoder, OAE)를 이용한 특징 추출과 스트림 기반 능동 학습의 결합**이다.

1. **차원 축소를 통한 효율성 증대**: OAE를 통해 고차원의 입력 변수를 저차원의 핵심 특징 공간으로 투영한다. 이는 회귀 모델이 추정해야 할 파라미터 $\beta$의 수를 줄여, 능동 학습 단계로 진입하기 위해 필요한 초기 라벨 데이터의 양을 감소시키고 모델의 강건성을 높인다.
2. **직교성 제약을 통한 다중공선성 해결**: OAE의 손실 함수에 직교성 규제항(Orthogonality regularization)을 추가하여 학습된 특징들이 서로 상관관계가 낮도록 강제함으로써, 선형 회귀 모델에서 발생하는 다중공선성(Multicollinearity) 문제를 완화한다.
3. **역사적 데이터를 활용한 동적 임계값 설정**: 과거의 라벨 없는 데이터($H$)를 사용하여 각 능동 학습 기준의 분포를 추정하고, 커널 밀도 추정(Kernel Density Estimation, KDE)을 통해 상한 관리 한계(Upper Control Limit, UCL)를 설정함으로써 스트림 상에서 실시간으로 쿼리 여부를 결정하는 체계를 구축한다.

## 📎 Related Works

논문에서는 회귀 모델을 위한 세 가지 대표적인 능동 학습 샘플링 기준을 소개한다.

- **Mahalanobis Distance**: 특징 공간에서의 다양성을 확보하는 방법으로, Hotelling $T^2$ 통계량을 사용하여 새로운 데이터가 기존 학습 세트와 얼마나 다른지를 측정한다. 이는 오직 특징 공간만을 고려한다는 한계가 있다.
- **Query By Committee (QBC)**: 여러 개의 부트스트랩 모델(Ensemble)을 구축하고, 각 모델의 예측값들 사이의 분산(Variance)을 측정하여 불확실성이 높은 데이터를 선택한다. 응답 변수의 불확실성을 고려한다는 점에서 장점이 있다.
- **Expected Model Change (EMC)**: 특정 데이터를 추가했을 때 모델 파라미터 $\beta$의 변화가 가장 클 것으로 예상되는 데이터를 선택한다. 손실 함수의 기울기(Gradient)를 통해 모델의 변화량을 추정한다.

기존 연구들이 주로 Pool-based 방식에 치중했던 것과 달리, 본 연구는 데이터가 순차적으로 입력되는 Stream-based 시나리오에 이러한 기준들을 적용하고, 반지도 학습 기반의 특징 추출 단계를 추가하여 차별성을 둔다.

## 🛠️ Methodology

### 1. 전체 파이프라인

전체 시스템은 **[역사적 데이터 학습 $\rightarrow$ 특징 추출 $\rightarrow$ 스트림 데이터 모니터링 $\rightarrow$ 동적 샘플링 $\rightarrow$ 모델 업데이트]** 순으로 진행된다.

### 2. Semi-supervised Orthogonal Autoencoder (OAE)

먼저 라벨이 없는 역사적 데이터 $H$를 사용하여 OAE를 학습시킨다. OAE는 입력 데이터 $x \in \mathbb{R}^p$를 저차원 특징 $z \in \mathbb{R}^k$로 인코딩하며, 이때 다음과 같은 Ortho-Loss를 최소화한다.
$$ \text{Ortho-Loss} = \text{Reconstruction Loss} + \lambda \cdot \text{Orthogonality Regularization} $$
여기서 $\lambda$는 특징들 간의 상관관계를 줄여 직교성을 높이는 가중치이다. 이렇게 추출된 $z$는 이후 선형 회귀 모델의 예측 변수로 사용된다.

### 3. 스트림 기반 능동 학습 및 임계값 결정

스트림으로 들어오는 데이터 $x$에 대해 샘플링 기준 $J(z)$를 계산한다. 이때, 역사적 데이터 $H$를 통해 $J$의 분포를 Gaussian 커널 기반의 KDE로 추정한 후, 사용자가 정의한 샘플링 비율 $\alpha$에 따라 임계값(UCL)을 결정한다.
$$ P(J(x) \geq \text{UCL}) = \alpha $$
새로운 데이터의 $J(z)$ 값이 $\text{UCL}$보다 크거나 같을 때만 라벨을 쿼리하고 학습 세트에 추가한다.

### 4. 샘플링 기준 $J$의 상세 수식

- **Mahalanobis Distance**:
  $$ T^2(x) = (x - \bar{x})^T S^{-1} (x - \bar{x}) $$
  ($\bar{x}$: 평균 벡터, $S$: 공분산 행렬)
- **QBC (Ambiguity)**:
  $$ a(x) = \frac{1}{K} \sum_{i=1}^K (f_i(x) - y_K(x))^2 $$
  ($f_i$: 앙상블 모델의 예측값, $y_K$: 예측값들의 평균)
- **EMC**:
  부트스트랩 앙상블을 활용하여 파라미터 변화량의 기댓값을 계산한다.
  $$ \text{Criterion} = \frac{1}{K} \sum_{i=1}^K \|(f_i(x) - f(x))x\| $$

## 📊 Results

### 실험 설정

- **데이터셋**: Tennessee Eastman Process (TEP) 시뮬레이션 데이터.
- **입력 변수**: 16개의 제어 변수.
- **목표 변수**: Stream 9E (조성 측정치).
- **평가 지표**: 학습 곡선(Learning Curve)을 통한 예측 오차 감소 속도 측정.
- **실험 규모**: 50회의 시뮬레이션 실행에 대한 평균 및 표준편차 산출.

### 주요 결과

1. **특징 추출의 효과**: 동일한 무작위 샘플링 조건에서, 원본 변수를 사용했을 때보다 OAE로 추출한 특징을 사용했을 때 예측 성능이 더 빠르게 향상되었다. 이는 OAE가 비선형 관계를 효과적으로 포착하고, 추정해야 할 파라미터 수를 줄였기 때문으로 분석된다.
2. **능동 학습 전략 비교**:
   - **EMC와 QBC**는 무작위 샘플링(RND)보다 일관되게 우수한 성능을 보이며 예측 오차를 더 빠르게 감소시켰다.
   - **Mahalanobis Distance (HOT)**는 오히려 성능을 악화시키는 경향을 보였다. 저자들은 이는 $T^2$ 통계량이 높은 데이터가 정보량이 많은 데이터가 아니라 단순한 이상치(Outlier)일 가능성이 크며, 이를 학습 세트에 포함하는 것이 모델을 오염시켰기 때문이라고 해석한다.

## 🧠 Insights & Discussion

### 강점

본 연구는 산업 현장의 실제 제약 사항인 '데이터 획득 비용'과 '실시간 스트림 환경'을 정확히 반영하였다. 특히, 단순히 AL 알고리즘만 적용한 것이 아니라 OAE를 통한 반지도 학습을 결합하여, AL이 효과적으로 작동하기 위해 필요한 '최소 초기 데이터' 문제를 차원 축소로 해결한 점이 매우 실용적인 접근이다.

### 한계 및 비판적 해석

- **이상치 민감도**: Mahalanobis distance가 성능을 저하시켰다는 결과는, 데이터의 분포가 정규분포에서 크게 벗어나거나 노이즈가 많은 산업 데이터의 특성을 보여준다. 단순히 '다양성'만을 추구하는 샘플링은 위험할 수 있음을 시사한다.
- **선형 모델의 한계**: 특징 추출 이후 최종 예측 모델로 단순 선형 회귀를 사용하였다. OAE가 비선형성을 어느 정도 흡수했겠으나, 예측 모델 자체를 비선형 모델(예: Gaussian Process, Random Forest)로 확장했을 때의 AL 효율성에 대한 분석이 누락되어 있다.
- **가정**: 역사적 데이터 $H$가 현재 스트림 데이터의 분포를 충분히 대표한다는 가정이 전제되어 있다. 만약 공정의 개념 드리프트(Concept Drift)가 발생한다면, 고정된 UCL은 더 이상 유효하지 않을 것이다.

## 📌 TL;DR

본 논문은 산업용 소프트 센서 개발을 위해 **직교 오토인코더(OAE) 기반의 특징 추출**과 **스트림 기반 능동 학습(Online AL)**을 결합한 프레임워크를 제안한다. OAE를 통해 데이터의 차원을 축소하고 다중공선성을 제거함으로써 더 적은 라벨로도 빠르게 모델을 수렴시킬 수 있으며, EMC와 QBC 전략이 실시간 환경에서 가장 효율적인 샘플링 성능을 보임을 입증하였다. 이 연구는 고비용 라벨링 환경의 실시간 공정 모니터링 시스템 구축에 중요한 가이드라인을 제공한다.
