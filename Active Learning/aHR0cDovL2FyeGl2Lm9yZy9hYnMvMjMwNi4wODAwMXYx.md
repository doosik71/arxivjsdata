# A MARKOVIAN FORMALISM FOR ACTIVE QUERYING

Sid Ijju (2023)

## 🧩 Problem to Solve

본 논문은 인공지능 분야에서 핵심적인 역할을 하는 능동 학습(Active Learning, AL) 연구들이 서로 매우 다양하게 분산되어 있으며, 이를 관통하는 종합적인 조직적 관점(overall organizing lens)이 부족하다는 문제를 제기한다. 특히 Inverse Reinforcement Learning(IRL)의 경우, 전문가의 시연(expert demonstrations)으로부터 보상 함수(reward function)를 학습하지만, 이 보상 함수가 부정확하거나 사용자의 실제 의도와 일치하지 않는 정렬 문제(alignment problem)가 발생할 수 있다. 이를 해결하기 위해 인간 전문가에게 능동적으로 쿼리를 던져 데이터를 보강하는 방식이 사용되고 있으나, 이러한 다양한 능동 학습 기법들을 체계적으로 통합하여 분석할 수 있는 수학적 틀이 부재한 상태이다. 따라서 본 논문의 목표는 능동 학습의 전체 과정을 마르코프 체계(Markovian formalism)로 정식화하여, 기존의 다양한 연구들을 통합하고 향후 연구 방향을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 능동 학습의 쿼리 과정을 **정보 마르코프 결정 과정(Information Markov Decision Process)**이라는 POMDP(Partially Observable Markov Decision Process) 프레임워크로 정의한 것이다.

중심 아이디어는 로봇이 인간에게 어떤 쿼리를 던지고 그 응답을 통해 지식을 업데이트하는 일련의 과정을 하나의 상태 전이 시스템으로 보는 것이다. 이를 통해 쿼리 생성, 데이터셋 증강, 보상 함수 업데이트 등의 서로 다른 능동 학습 요소들을 메타 상태(meta-states) 간의 전이(transition)로 해석할 수 있게 한다. 결과적으로 정보 도출(information derivation)과 정보 최적화(information optimization)라는 두 가지 연구 방향을 하나의 MDP 최적화 문제로 통합하였다.

## 📎 Related Works

논문은 Inverse Reinforcement Learning(IRL)과 그 한계를 먼저 언급한다. IRL은 전문가의 시연을 통해 보상 함수를 유도하지만, 학습된 보상 함수가 실제 인간의 의도와 어긋날 수 있다는 한계가 있다. 이를 보완하기 위해 시연 간의 선호도(preferences)를 쿼리하는 방식이 제안되었다.

또한, 기존 능동 학습의 쿼리 선택 전략으로 Uncertainty sampling, Query-by-committee, Expected model change 등을 언급한다. 하지만 저자는 이러한 기존 연구들이 주로 '어떤 샘플을 선택할 것인가'라는 개별적인 알고리즘 최적화에 치중해 있으며, 전체 시스템을 관통하는 통합된 정식화(formalism) 없이 파편화되어 있다는 점을 차별점으로 지적한다.

## 🛠️ Methodology

### 1. 기본 환경 모델

먼저 실제 학습 환경을 결정론적이고 완전 관측 가능한 MDP로 정의한다.

- 상태 집합 $S$, 행동 집합 $A$
- 궤적(Trajectory) $\xi \in \Xi$는 상태와 행동의 유한한 시퀀스 $(\left(s_t, a_t\right))_{t=0}^T$로 정의된다.
- 보상 함수 $r: \Xi \to \mathbb{R}$는 특징 함수 $\Phi : \Xi \to \mathbb{R}^d$의 선형 조합으로 가정한다:
$$r(\xi) = \omega^T \Phi(\xi)$$
여기서 $\omega$는 인간의 보상 함수를 설명하는 유한 차원 벡터이다.

### 2. 정보 마르코프 결정 과정 (Information MDP)

능동 학습의 쿼리 프로세스를 정식화하기 위해 다음과 같은 POMDP $M' = (S', A', T', R', \Omega, O, \gamma)$를 제안한다.

- **상태 공간 $S' \in \Xi \times \mathbb{R}$**: 현재 시스템의 상태는 데이터셋 $D$와 현재까지 학습된 보상 함수 $r$의 쌍으로 구성된다.
- **행동 공간 $A'$**: 인간 전문가에게 던질 수 있는 잠재적 쿼리들의 집합이다.
- **관측 공간 $\Omega$**: 쿼리 $a \in A'$에 대한 인간의 응답 집합이다.
- **조건부 관측 확률 $O: \Omega \times S' \times A' \to [0, 1]$**: 상태 $s'$에서 행동 $a$를 취했을 때 응답 $o \in \Omega$를 얻을 확률이다.
- **보상 함수 $R': S' \times A' \to \mathbb{R}$**: 특정 쿼리를 던졌을 때 원래의 보상 함수 $r$을 학습하는 데 얻는 이득을 나타낸다.
- **전이 함수 $T': S' \times A' \to S'$**: 쿼리에 대한 응답을 받은 후 시스템의 정보(데이터셋, 보상 함수)가 어떻게 변하는지를 정의한다.

### 3. 전이 함수의 세부 구조

전이 함수 $T$는 다음과 같이 데이터셋 변환 함수 $F$와 학습 알고리즘 $L$의 조합으로 분해하여 설명할 수 있다.

- $F \in \mathcal{F}: \Xi \to \Xi$: 응답을 통해 데이터셋을 업데이트하는 함수.
- $L \in \mathcal{L}: \Xi \times \mathcal{R} \to \mathcal{R}$: 업데이트된 데이터셋 $D$를 사용하여 보상 함수 $r$을 업데이트하는 학습 알고리즘.

## 📊 Results

본 논문은 새로운 알고리즘을 제안하여 수치적 성능을 측정하는 실험 논문이 아니라, 기존의 방법론들을 통합하는 이론적 프레임워크를 제안하는 논문이다. 따라서 전통적인 의미의 정량적 실험 결과는 제시되지 않는다.

대신, 저자는 제안한 정식화가 기존의 다양한 쿼리 유형들을 어떻게 수용할 수 있는지를 Table 1을 통해 정성적으로 입증한다.

- **Labels**: $\xi \notin D$를 선택하여 라벨을 요청 $\to D' = D \cup \{\xi\}$로 전이.
- **Comparisons**: $(\xi_a, \xi_b)$를 선택하여 선호도를 요청 $\to D$는 유지되나 $L(D, r)$을 통해 $r$만 업데이트.
- **Demonstrations**: 특정 상태 $s \in S$에서 시연을 요청 $\to D' = D \cup \{o\}$로 전이.
- **Feature Labels**: 특정 특징 $f(\xi)_i$의 유용성을 요청 $\to L$의 가중치 업데이트로 전이.
- **Language**: 문장 궤적 $\xi$에 대한 수정을 요청 $\to D' = (D \setminus \{\xi\}) \cup \{o\}$로 전이.

이를 통해 제안된 POMDP 프레임워크가 기존의 파편화된 AL 연구들을 모두 포괄할 수 있는 상위 개념임을 보여준다.

## 🧠 Insights & Discussion

### 강점 및 해석

본 논문은 능동 학습을 단순히 '좋은 샘플을 뽑는 문제'에서 '정보 획득을 위한 MDP 최적화 문제'로 확장시켰다. 특히 인간의 응답 확률을 모델링하는 $O$와 쿼리의 가치를 정의하는 $R'$를 명시함으로써, 향후 연구가 단순한 쿼리 선택 알고리즘($A'$)을 넘어 시스템 전체의 최적화로 나아갈 수 있는 이론적 토대를 마련하였다.

### 한계 및 미래 방향

저자는 현재 대부분의 연구가 행동 공간 $A'$의 선택 전략에만 집중되어 있음을 지적하며, 다음과 같은 미개척 영역을 제안한다.

1. **비결정론적 전이 $T'$**: 쿼리 응답을 데이터셋에 추가할 때 확률적 임계값 $\alpha$를 도입하거나, 학습 알고리즘 $L$ 자체에 무작위성을 부여하여 보상 함수를 분포 형태로 관리하는 방법.
2. **데이터셋 튜닝 (Dataset Tuning)**: 새로운 데이터를 추가하는 것뿐만 아니라, 쿼리 결과를 바탕으로 기존 데이터의 가중치를 낮추거나 수정하는 방식의 전이 함수 설계.
3. **Meta-RL 적용**: "Information MDP" 자체를 최적화하는 Meta-RL 알고리즘을 설계하여, 쿼리의 가치를 보상으로 삼아 최적의 쿼리 정책을 학습하는 방향.

## 📌 TL;DR

본 논문은 파편화되어 있던 능동 학습(Active Learning) 연구들을 **'정보 마르코프 결정 과정(Information MDP)'**이라는 하나의 POMDP 프레임워크로 통합하였다. 이 정식화를 통해 쿼리 유형, 데이터 업데이트, 보상 함수 학습 과정을 상태 전이로 정의할 수 있으며, 이는 향후 Meta-RL이나 비결정론적 전이 모델 등을 적용한 더 지능적인 능동 학습 시스템을 구축하는 데 중요한 이론적 가이드라인이 될 것으로 기대된다.
