# Learning from minimum entropy queries in a large committee machine

Peter Sollich (1996)

## 🧩 Problem to Solve

지도 학습(Supervised Learning)에서 무작위로 추출된 예제(Random examples)들은 많은 중복된 정보를 포함하고 있어 학습 효율이 떨어진다는 문제가 있다. 이를 해결하기 위해 기존 학습 데이터를 기반으로 가장 '유용한' 입력을 선택하여 학습하는 쿼리 학습(Query learning)이 대안으로 제시되었다.

본 논문은 특히 다층 신경망 구조인 Tree-Committee Machine(TCM)에서 최소 엔트로피 쿼리(Minimum entropy queries)를 사용할 때의 일반화 성능(Generalization performance)을 분석하는 것을 목표로 한다. 기존의 단층 퍼셉트론에 대한 연구는 많이 이루어졌으나, 다층 네트워크에서의 평균적인 일반화 성능에 대한 통계역학적 분석은 부족한 상태였다. 따라서 본 연구는 다층 네트워크에서 정보 획득을 최대화하는 쿼리를 통해 일반화 오차를 얼마나 빠르게 줄일 수 있는지 이론적으로 규명하고자 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 다층 신경망인 TCM에서 최소 엔트로피 쿼리를 사용할 경우, 일반화 오차가 학습 예제의 수에 따라 지수적으로 감소(Exponential decay)한다는 것을 증명한 점이다. 이는 무작위 예제를 사용할 때 나타나는 대수적 감소(Algebraic decay)보다 훨씬 빠른 성능 향상을 의미한다.

또한, 계산 비용이 많이 드는 기존의 쿼리 선택 방식(Query by Committee)을 대체할 수 있는 '구성적 알고리즘(Constructive algorithm)'을 제안하였다. 이 알고리즘은 각 은닉 유닛의 엔트로피가 서로 독립적이라는 가정을 바탕으로, 각 유닛의 가중치 벡터의 평균에 수직인 입력을 선택함으로써 매우 효율적으로 최적의 쿼리에 근접한 성능을 낼 수 있음을 보였다.

## 📎 Related Works

기존 연구들은 주로 선형 및 이진 퍼셉트론과 같은 단층 신경망에서의 쿼리 학습 성능을 다루었으며, 이 경우의 일반화 성능은 이미 잘 알려져 있었다. 다층 네트워크의 경우 일부 휴리스틱한 쿼리 학습 알고리즘들이 제안되었으나, 이들은 특정 상황에만 적용 가능하며 이론적 일반화가 어려웠다.

또한, 기존의 다층 네트워크 분석은 주로 PAC(Probably Approximately Correct) 학습 프레임워크 내에서 수행되어 최악의 경우(Worst case)에 대한 경계값만을 제시하였다. 하지만 실제 시스템의 평균적인 동작을 이해하기 위해서는 평균 사례(Average case) 분석이 필요하며, 본 논문은 이를 위해 통계역학(Statistical mechanics) 도구를 도입하여 차별성을 두었다.

## 🛠️ Methodology

### 시스템 구조: Tree-Committee Machine (TCM)

TCM은 $N$개의 입력 유닛, $K$개의 은닉 유닛, 그리고 1개의 출력 유닛으로 구성된 2층 신경망이다. 각 은닉 유닛의 수용 영역(Receptive fields)은 서로 겹치지 않으며, 은닉 층에서 출력 층으로 가는 모든 가중치는 1로 고정되어 있다. 출력 $y$는 다음과 같이 정의된다.

$$y = \text{sgn} \left( \frac{1}{\sqrt{K}} \sum_{i=1}^{K} \sigma_i \right)$$
$$\sigma_i = \text{sgn} \left( \sqrt{K} \mathbf{x}_i^T \mathbf{w}_i \right)$$

여기서 $\mathbf{w}_i$는 은닉 유닛의 가중치 벡터이며, $\mathbf{x}_i$는 해당 유닛이 받는 $N/K$ 차원의 입력 벡터이다. 가중치 벡터는 $\mathbf{w}_i^2 = N/K$로 정규화되어 있다고 가정한다.

### 학습 및 쿼리 선택 절차

본 논문은 모든 학습 예제를 정확하게 예측하는 학생(Student) 네트워크를 무작위로 선택하는 zero-temperature Gibbs learning을 사용한다.

쿼리 학습의 목표는 학생 네트워크의 파라미터 공간에서의 엔트로피 $S$를 최대화하는 입력을 선택하는 것이다. 엔트로피는 버전 공간(Version space, 학습 데이터를 모두 만족하는 가중치 공간)의 부피 $V$의 로그값으로 정의된다.

$$S(\Theta^{(p)}) = \ln V(\Theta^{(p)})$$

최적의 쿼리는 기대 엔트로피 감소량 $\langle \Delta S \rangle$을 최대화하는 입력 $\mathbf{x}_{p+1}$이며, 이는 새로운 입력이 기존 버전 공간을 정확히 이분(Bisect)할 때, 즉 출력값 $y = \pm 1$일 확률이 각각 $1/2$일 때 달성된다.

### 이론적 분석 도구

분석을 위해 열역학적 극한(Thermodynamic limit, $N \to \infty$)과 은닉 유닛의 수가 매우 많은 경우($K \to \infty$)를 가정한다. 복제 대칭성(Replica symmetry) 가정을 사용하여 평균 엔트로피를 계산하며, 일반화 오차 $\epsilon_g$는 다음과 같은 유효 오버랩(Effective overlap) 파라미터 $R_{\text{eff}}$로 표현된다.

$$\epsilon_g = \frac{1}{\pi} \arccos R_{\text{eff}}$$

### 제안하는 구성적 알고리즘 (Constructive Algorithm)

기존의 Query by Committee(QBC) 방식은 무작위 입력 스트림에서 조건을 만족하는 쿼리를 필터링하므로 시간이 매우 오래 걸린다. 이를 해결하기 위해 본 논문은 각 은닉 유닛의 엔트로피가 디커플링(Decoupling)되어 있다는 가정하에, 각 은닉 유닛 $i$에 대해 $\mathbf{x}_{i}^{\mu+1}$가 현재의 평균 가중치 벡터 $\mathbf{\bar{w}}^\mu_i$와 직교하도록 직접 구성하는 방식을 제안한다.

## 📊 Results

### 일반화 오차의 감소 속도

실험 결과, 최소 엔트로피 쿼리를 사용했을 때 일반화 오차 $\epsilon_g$는 학습 예제 수 $\alpha = p/N$에 대해 다음과 같이 지수적으로 감소한다.

$$\epsilon_g \propto \exp(-c\alpha), \quad c = \frac{1}{2} \ln 2$$

이는 무작위 예제를 사용할 때의 $\epsilon_g \propto 1/\alpha$ (대수적 감소)와 비교하여 매우 비약적인 성능 향상이다.

### 엔트로피와 일반화 오차의 관계

이진 퍼셉트론의 경우 엔트로피 $s$와 일반화 오차의 관계가 $s \approx \ln \epsilon_g$인 반면, TCM에서는 다음과 같은 관계가 성립함을 확인하였다.

$$s \approx \ln \epsilon_g^2$$

이는 동일한 일반화 성능을 얻기 위해 TCM이 이진 퍼셉트론보다 이론적으로 두 배 더 많은 예제가 필요함을 시사한다.

### 구성적 알고리즘의 효율성

제안된 구성적 알고리즘을 통해 선택된 쿼리의 성능을 분석한 결과, 정확한 최소 엔트로피 쿼리와 거의 동일한 엔트로피 감소율($ds/d\alpha = -\ln 2$)을 보였다. 특히, 수치적 결과에서는 구성적 알고리즘이 정확한 최소 엔트로피 쿼리보다 약간(약 4%) 더 낮은 일반화 오차를 기록하였다. 이는 엔트로피 감소와 일반화 오차 감소가 강하게 상관되어 있지만, 완전히 일치하는 관계는 아님을 보여준다.

## 🧠 Insights & Discussion

본 연구는 다층 네트워크에서도 능동적인 쿼리 선택을 통해 학습 속도를 획기적으로 높일 수 있음을 이론적으로 입증하였다. 특히, $K \to \infty$ 극한에서 은닉 유닛 간의 상관관계가 약해지기 때문에, 전체 네트워크의 엔트로피를 줄이는 대신 개별 은닉 유닛의 엔트로피를 줄이는 단순한 전략(구성적 알고리즘)이 유효하다는 점이 흥미롭다.

다만, 본 논문은 은닉 유닛의 가중치 벡터가 겹치지 않는 Tree-Committee Machine이라는 특수한 구조를 가정하였다. 모든 입력이 모든 은닉 유닛에 연결된 Fully-connected committee machine으로 확장할 경우, 쿼리를 모든 은닉 유닛의 평균 가중치 벡터들이 이루는 부분 공간(Subspace)에 수직이 되도록 설정함으로써 유사한 효과를 얻을 수 있을 것으로 추측된다.

또한, 출력층의 가중치까지 학습 파라미터로 포함되는 일반적인 2층 신경망에서도 이러한 엔트로피 기반 쿼리 방식이 계산적으로 저렴하게 구현될 수 있을지는 향후 연구 과제로 남아 있다.

## 📌 TL;DR

본 논문은 Tree-Committee Machine(TCM)에서 최소 엔트로피 쿼리를 사용할 때 일반화 오차가 학습 데이터 수에 따라 지수적으로 감소함을 통계역학적으로 분석하였다. 특히, 계산량이 많은 기존 방식 대신 각 은닉 유닛의 평균 가중치에 수직인 입력을 생성하는 '구성적 알고리즘'을 제안하여, 매우 낮은 계산 비용으로 최적의 일반화 성능을 달성할 수 있음을 보였다. 이 연구는 향후 복잡한 다층 신경망의 능동 학습(Active Learning) 효율성을 높이는 데 중요한 이론적 토대를 제공한다.
