# Knowledge-driven Active Learning

Gabriele Ciravegna, Frédéric Precioso, Alessandrio Betti, Kevin Mottin, and Marco Gori (2023)

## 🧩 Problem to Solve

딥러닝(Deep Learning, DL) 모델을 배포함에 있어 가장 큰 걸림돌 중 하나는 대량의 지도 학습 데이터(supervised data)가 필요하다는 점이다. 이를 해결하기 위해 데이터 라벨링 비용을 최소화하면서 모델 성능을 극대화하려는 Active Learning(AL) 전략들이 연구되어 왔다.

기존의 Active Learning 전략들은 주로 모델의 예측 불확실성(uncertainty)에 기반하여 샘플을 선택하거나, 결정 경계(decision boundary) 근처의 샘플들에 집중하는 경향이 있다. 이러한 방식은 이론적으로는 타당하지만, 선택된 샘플이 왜 중요한지에 대한 내용적 이해가 어렵다는 단점이 있다. 결과적으로 비전문가 사용자들에게 딥러닝 모델은 여전히 내부 동작을 알 수 없는 '블랙박스(black-box)'로 인식되며, 도메인 전문가가 가진 지식을 모델 학습 과정에 직접적으로 활용할 방법이 부족하다는 문제가 존재한다.

따라서 본 논문의 목표는 일반적인 도메인 지식(domain-knowledge)을 Active Learning의 샘플 선택 과정에 통합하여, 더 적은 수의 샘플로도 효율적으로 모델을 학습시키고 모델의 투명성을 높이는 Knowledge-driven Active Learning (KAL) 프레임워크를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 도메인 전문가의 지식을 심볼릭한 규칙(symbolic rules) 형태로 정의하고, 모델의 예측이 이 규칙을 위반하는 정도를 측정하여 라벨링할 샘플을 선택하는 것이다.

구체적으로, 1차 논리(First-Order Logic, FOL)로 표현된 도메인 지식을 T-Norm을 통해 수치적 제약 조건(numerical constraints)으로 변환한다. 모델이 예측한 결과가 이 제약 조건을 크게 위반할수록 해당 데이터는 모델이 아직 학습하지 못한 영역이거나 중요한 정보가 포함되어 있을 가능성이 높다고 판단하여, 이를 우선적으로 라벨링 대상으로 선정한다. 이는 기존의 불확실성 기반 방식이 단순히 결정 경계에 매몰되는 것과 달리, 지식에 근거하여 데이터 분포의 빈틈을 찾아낼 수 있게 한다.

## 📎 Related Works

### 기존 Active Learning 접근 방식 및 한계

- **Uncertainty Sampling**: 모델이 가장 확신하지 못하는 샘플(예: Entropy가 높거나 Margin이 좁은 샘플)을 선택한다. 하지만 딥러닝 모델은 종종 과잉 확신(over-confident)하는 경향이 있으며, 결정 경계 근처의 샘플만 반복적으로 선택하여 데이터 불균형을 초래하거나 새로운 데이터 분포를 발견하지 못하는 한계가 있다.
- **Diversity Sampling**: 입력 데이터의 분포를 최대한 커버하기 위해 CoreSet이나 K-Means 같은 방식을 사용하여 샘플을 선택한다. 이 방식은 데이터 분포는 잘 파악하지만, 모델의 예측 성능을 직접적으로 개선하는 결정 경계상의 정밀한 샘플 선택 능력이 부족하다.

### 기존 연구와의 차별점

본 논문은 단순한 통계적 수치나 모델의 내부 상태(uncertainty)에 의존하는 대신, 외부의 '도메인 지식'이라는 심볼릭한 정보를 가이드로 사용한다. 이는 Neuro-symbolic Learning의 관점에서 신경망의 유연성과 논리 체계의 명확성을 결합한 것으로, 특히 전문가의 지식을 모델이 실제로 습득했는지 검증할 수 있다는 점에서 기존 방식과 차별화된다.

## 🛠️ Methodology

### 전체 파이프라인 및 시스템 구조

KAL 프레임워크는 다음과 같은 흐름으로 작동한다:

1. **지식 정의**: 도메인 전문가가 1차 논리(FOL) 형태로 규칙(Rule) 세트 $K$를 정의한다.
2. **수치 변환**: T-Norm을 이용하여 FOL 규칙을 수치적 제약 조건 및 손실 함수 $\phi$로 변환한다.
3. **위반도 측정**: 라벨링되지 않은 데이터 풀 $X_u$에 대해 모델의 예측값 $f(x)$가 규칙을 얼마나 위반하는지 계산한다.
4. **샘플 선택**: 위반도가 가장 높은 샘플을 선택하여 라벨링하고 학습 세트에 추가한다.

### 주요 구성 요소 및 방법론 상세

#### 1. FOL의 수치적 제약 조건 변환

FOL 공식은 T-Norm(Triangular Norms)을 통해 $[0, 1]$ 범위의 수치 값으로 변환된다. 본 논문에서는 Product T-Norm을 사용하며, 논리 연산자는 다음과 같이 매핑된다:

- **Conjunction ($\wedge$)**: 곱셈 ($\cdot$)
- **Negation ($\neg x$)**: $1 - x$
- **Implication ($x \to y$)**: $1 - (x \cdot (1 - y))$

예를 들어, 규칙 $\neg((x_1 \wedge x_2) \wedge \neg f)$는 $1 - (x_1 x_2 (1 - f))$라는 수치적 제약 조건으로 변환되며, 이를 위반하는 정도를 측정하는 손실 함수 $\phi(f(x))$는 $x_1 x_2 (1 - f)$가 된다. 이 값은 규칙이 만족될 때 0이 되고, 위반될수록 커진다.

#### 2. 샘플 선택 알고리즘

매 반복(iteration)마다 KAL은 다음 식을 통해 가장 많은 규칙을 위반하는 샘플 $x^\star$를 선택한다:
$$x^\star = \arg \max_{x \in X_u} \sum_{\phi \in K} \phi(f(x))$$
여기서 $K$는 가용한 모든 FOL 규칙의 집합이며, $\phi(f(x))$는 각 규칙의 위반도이다.

#### 3. 확장된 적용 범위

- **분류(Classification)**: 각 클래스의 확률값 $f_i(x)$를 사용하여 규칙을 적용한다.
- **회귀(Regression)**: 예측값 $f(x)$가 특정 구간($k_1 < f(x) < k_2$)에 있는지 확인하는 규칙을 사용한다.
- **객체 탐지(Object Detection)**: 특정 클래스에 대해 예측된 바운딩 박스들 중 최대 신뢰도 점수를 $f_i(x)$로 정의하여 적용한다.

#### 4. 성능 향상을 위한 추가 기법

- **다양성 확보(Diversity)**: 특정 규칙 하나에만 치우쳐 샘플이 선택되는 것을 방지하기 위해, 한 규칙당 선택할 수 있는 최대 샘플 수 $r$을 제한($r = p/2$)한다.
- **불확실성 유사 규칙(Uncertainty-like rule)**: 지식이 부족한 상황을 보완하기 위해, 모든 예측값이 0 또는 1에 가까워야 한다는 규칙 ($\bigvee_i f_i \oplus \neg f_i$)을 추가하여 불확실성이 높은 샘플을 함께 선택한다.

## 📊 Results

### 실험 설정

- **데이터셋**: XOR-like(인공), IRIS(분류), Insurance(회귀), ANIMALS(이미지 분류), CUB200(세밀한 조류 분류), DOGvsPERSON(객체 인식).
- **비교 대상**: Entropy, Margin, LeastConf (및 이들의 MC Dropout 버전), BALD, ADV(BIM, DeepFool), K-Center, K-Means, SupLoss.
- **평가 지표**: F1 Score, R Score (회귀), mAP (객체 인식), AUBC (Area Under the Budget Curve).

### 주요 결과

1. **성능 우위**: 대부분의 데이터셋에서 KAL은 기존 Active Learning 전략보다 높은 AUBC를 기록했다. 특히 도메인 지식이 풍부한 XOR-like 및 IRIS 데이터셋에서 성능 향상이 뚜렷했다.
2. **데이터 분포 발견 능력**: XOR-like 실험을 통해, 불확실성 기반 전략(BALD, Margin 등)은 초기 샘플링에서 누락된 데이터 영역을 발견하지 못하고 결정 경계 주변만 탐색하는 반면, KAL은 규칙 위반을 통해 누락된 데이터 영역을 즉각적으로 찾아내는 것을 확인했다.
3. **지식 습득 보장**: CUB200 데이터셋 실험에서, 전문가가 제공한 소규모 규칙 집합을 기준으로 측정했을 때 KAL로 학습된 모델이 다른 전략보다 훨씬 더 규칙을 잘 준수(낮은 위반도)함을 보였다.
4. **범용성**: 불확실성 기반 전략을 적용하기 어려운 회귀 및 객체 인식 작업에서도 KAL이 Random sampling이나 SupLoss보다 우수한 성능을 보였다.
5. **계산 효율성**: KAL의 계산 비용은 Random sampling 대비 소폭 증가하는 수준이며, ADV 기반이나 CoreSet 기반 방법들보다 훨씬 적은 계산 자원을 소모한다.
6. **XAI와의 결합**: 도메인 지식이 없는 경우, 학습된 모델에서 XAI(의사결정 나무 등)를 통해 규칙을 추출하여 사용하는 $\text{KAL}_{\text{XAI}}$ 방식 또한 기존 KAL 성능의 1~2% 내외로 근접한 성능을 보였다.

## 🧠 Insights & Discussion

### 강점

KAL은 딥러닝의 '데이터 굶주림' 문제를 해결함과 동시에, '블랙박스' 문제를 완화한다. 전문가의 지식을 샘플 선택의 가이드로 사용함으로써, 모델이 학습해야 할 방향을 명시적으로 제시하며 이는 곧 학습 속도 향상과 데이터 효율성으로 이어진다. 또한, 단순한 성능 수치를 넘어 "전문가의 지식이 모델에 반영되었는가"를 정량적으로 확인할 수 있게 함으로써 모델에 대한 신뢰도를 높인다.

### 한계 및 논의사항

- **지식 의존성**: 본 연구의 결과에서 보듯, 지식이 부족한 경우(ANIMALS 데이터셋) 성능 향상 폭이 적다. 이는 KAL의 성능이 제공된 지식의 질과 양에 직접적으로 비례함을 의미한다.
- **지식 표현의 복잡성**: 현재는 FOL을 사용하고 있으나, 매우 복잡한 도메인 지식은 1차 논리만으로는 충분히 표현되지 않을 수 있으며, 고차 논리(Higher-order logic) 도입이 필요할 수 있다.
- **컴퓨터 비전 적용**: 이미지 데이터의 경우 입력 픽셀에 직접 규칙을 적용하기 어려워 속성(attribute) 기반의 규칙을 사용했다. 하지만 이러한 속성 정보가 없을 경우, 잠재 공간(latent space)에서 개념을 자동으로 추출하는 추가적인 연구가 필요하다.

## 📌 TL;DR

본 논문은 도메인 지식을 FOL 규칙으로 정의하고, 모델의 예측이 이 규칙을 위반하는 정도를 측정하여 라벨링할 샘플을 선택하는 **Knowledge-driven Active Learning (KAL)** 프레임워크를 제안한다. KAL은 기존의 불확실성 기반 방식이 가지는 '결정 경계 매몰' 문제를 해결하여 새로운 데이터 분포를 효율적으로 발견하며, 계산 비용이 낮으면서도 회귀 및 객체 인식 등 다양한 작업에 적용 가능하다. 이 연구는 전문가의 지식을 딥러닝 학습 과정에 직접 주입함으로써 데이터 효율성을 높이고 모델의 투명성과 신뢰성을 확보하는 새로운 방향성을 제시한다.
