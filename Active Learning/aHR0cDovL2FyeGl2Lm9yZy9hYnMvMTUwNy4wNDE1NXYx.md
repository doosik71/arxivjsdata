# ALEVS: Active Learning by Statistical Leverage Sampling

Cem Orhan, Oznur Tastan (2015)

## 🧩 Problem to Solve

본 논문은 지도 학습(Supervised Learning)에서 레이블링 비용을 줄이기 위해 적은 수의 데이터만으로 높은 분류 정확도를 달성하려는 Active Learning(능동 학습) 문제를 다룬다.

전통적인 Passive Learning은 가용한 모든 데이터를 사용하여 모델을 학습시키지만, 실제 환경에서는 데이터를 수집하는 비용보다 레이블을 지정하는 비용이 훨씬 높은 경우가 많다. 따라서 무작위로 데이터를 선택하는 대신, 모델의 성능 향상에 가장 기여도가 높은 '효과적인' 쿼리(Query)를 선택하여 오라클(Oracle, 인간 주석자)에게 레이블을 요청하는 것이 중요하다.

기존의 Active Learning 방법론들은 주로 데이터의 정보량(Informativeness)이나 대표성(Representativeness)에 기반하여 샘플링을 수행해 왔다. 본 논문의 목표는 통계적 레버리지 점수(Statistical Leverage Scores)라는 새로운 기준을 도입하여, 데이터셋 내에서 영향력이 큰 인스턴스를 효율적으로 선택하는 ALEVS 방법론을 제안하고 그 효용성을 검증하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 행렬 분석에서 데이터 포인트의 영향력을 측정하는 지표인 **Statistical Leverage Score**를 Active Learning의 쿼리 기준으로 활용하는 것이다.

레버리지 점수는 원래 회귀 분석의 진단이나 행렬의 저차원 근사(Low-rank approximation) 등에 사용되던 개념으로, 특정 행(데이터 포인트)이 전체 행렬의 구조에 얼마나 큰 영향을 미치는지를 나타낸다. 저자들은 이 직관을 Active Learning에 적용하여, 특성 공간(Feature Space)에서 중요도가 높은 데이터 포인트를 선택함으로써 학습 효율을 극대화하고자 하였다. 특히, 전체 데이터에 대해 점수를 계산하는 것이 아니라 예측된 클래스별로 데이터를 분리하여 레버리지 점수를 계산하는 전략을 취한 것이 주요한 설계 포인트이다.

## 📎 Related Works

논문에서는 Active Learning의 일반적인 프레임워크와 더불어, 레버리지 점수가 활용된 기존의 수치 해석 및 데이터 분석 연구들을 언급한다.

1. **Active Learning 기법**: 기존 방법론들은 주로 불확실성 샘플링(Uncertainty Sampling)과 같이 모델이 확신하지 못하는 지점을 찾거나, 데이터의 분포를 대표하는 지점을 찾는 방식에 집중하였다.
2. **레버리지 점수의 활용**:
   - 회귀 진단(Regression Diagnostics)에서 영향력이 큰 포인트(High influential points)를 탐지하는 데 사용되었다.
   - CUR 분해(CUR Decomposition) 및 Nyström 방법과 같은 무작위 저차원 행렬 근사 알고리즘에서 중요한 열(Column)이나 행(Row)을 샘플링하는 확률 분포로 사용되었다.

본 연구는 이러한 레버리지 점수를 '행렬 근사'라는 수치적 목적이 아닌, '분류기의 성능 향상'이라는 기계 학습의 목적을 위해 Active Learning 쿼리 전략으로 전용했다는 점에서 기존 연구와 차별화된다.

## 🛠️ Methodology

### 전체 파이프라인

ALEVS(Active Learning by Statistical Leverage Sampling)는 반복적인 쿼리-학습 루프를 통해 수행된다. 전체 과정은 다음과 같다.

1. 소량의 레이블링된 데이터 $D_l^t$로 분류기 $h_t$를 학습시킨다.
2. 학습된 $h_t$를 사용하여 레이블이 없는 데이터 $D_u^t$의 클래스를 예측한다.
3. 예측 결과에 따라 데이터를 양성($X_+^t$)과 음성($X_-^t$) 그룹으로 분리한다.
4. 각 그룹에 대해 커널 행렬(Kernel Matrix)을 생성하고 레버리지 점수를 계산한다.
5. 계산된 점수가 가장 높은 인스턴스를 선택하여 오라클에게 레이블을 요청한다.
6. 요청한 데이터를 $D_l$에 추가하고 과정을 반복한다.

### 상세 구성 요소 및 방법

#### 1. 데이터 분리 및 커널 행렬 구성

데이터를 클래스별로 나누는 이유는 각 클래스 내부의 구조적 특징을 더 잘 포착하기 위함이다. $X_+^t$는 실제 양성 레이블 데이터와 $h_t$에 의해 양성으로 예측된 데이터를 포함하며, $X_-^t$는 마찬가지로 음성 데이터를 포함한다. 이후 다음과 같은 커널 함수를 통해 커널 행렬 $K$를 생성한다.

- **Linear Kernel**: $K_{ij} = \langle x_i, x_j \rangle$
- **RBF Kernel**: $K_{ij} = \exp\left(-\frac{\|x_i - x_j\|_2^2}{2\sigma^2}\right)$

#### 2. 레버리지 점수 계산 (Leverage Score Calculation)

SPSD(Symmetric Positive Semi-Definite) 커널 행렬 $K \in \mathbb{R}^{n \times n}$에 대해 다음과 같이 레버리지 점수를 계산한다.

먼저 $K$의 고유값 분해(Eigen-decomposition)를 수행한다:
$$K = U\Sigma U^T$$

여기서 $U$ 행렬의 열 중 상위 $k$개의 고유벡터를 포함하는 행렬을 $U_1$이라고 정의한다:
$$U = \begin{pmatrix} U_1 \\ U_2 \end{pmatrix}$$

$j$번째 데이터 포인트(열)의 레버리지 점수 $\ell_j$는 $U_1$의 $j$번째 행의 유클리드 노름의 제곱으로 정의된다:
$$\ell_j = \|(U_1)_{(j)}\|_2^2$$

#### 3. 쿼리 선택 (Query Selection)

레이블이 없는 데이터 $D_u^t$ 중에서 계산된 레버리지 점수가 가장 높은 샘플 $x_q$를 최종 쿼리 대상으로 선택한다:
$$x_q = \arg \max_{x_j \in D_u^t} \ell_j$$

## 📊 Results

### 실험 설정

- **비교 대상 (Baselines)**:
  - Random Sampling (무작위 선택)
  - Uncertainty Sampling (SVM 기반, Platt's algorithm을 사용하여 가장 불확실한 점 선택)
  - Leverage on all data (클래스 분리 없이 전체 데이터에 대해 레버리지 점수 계산)
- **데이터셋**: digit1, g241c, g241n, letter(D vs P), letter(U vs V), USPS, splice, ringnorm, spambase, MNIST(3 vs 5) 등 총 10종의 이진 분류 데이터셋.
- **분류기**: RBF 커널을 사용하는 SVM.
- **평가 방법**: 초기 레이블 4개로 시작하여 쿼리 횟수를 늘려가며 테스트 셋에 대한 정확도를 측정 (50회 반복 실험의 평균).

### 주요 결과

- **성능 우위**: 10개 데이터셋 중 7개에서 ALEVS가 베이스라인보다 우수한 성능을 보였다. 나머지 3개에서는 Uncertainty Sampling과 대등한 수준이었다.
- **초기 수렴 속도**: 거의 모든 결과에서 ALEVS는 학습 초기 단계(초반 쿼리 횟수가 적을 때)에 매우 효율적인 데이터 포인트를 선택하여 정확도를 빠르게 끌어올리는 경향을 보였다.
- **클래스 분리의 효과**: 'Leverage on All' 방식보다 'ALEVS(클래스 분리)' 방식의 성능이 더 높게 나타났으며, 이는 클래스별로 영향력을 측정하는 것이 유의미함을 시사한다.
- **하이퍼파라미터 $k$의 영향**: 저차원 근사 파라미터인 $k$의 값(20, 40, 60, 80)에 따라 성능 차이가 발생했으며, 특히 USPS와 splice 데이터셋에서 그 영향이 두드러졌다.

## 🧠 Insights & Discussion

### 강점 및 의의

ALEVS는 단순한 모델의 불확실성(Uncertainty)에 의존하는 대신, 데이터 자체의 기하학적/통계적 구조(Leverage Score)를 활용함으로써 더 대표성 있는 샘플을 찾을 수 있음을 보여주었다. 특히 학습 초기 단계에서 매우 강력한 성능을 보인다는 점은 주목할 만하며, 이는 초기 샘플링 전략으로 ALEVS를 사용하고 이후 다른 전략으로 전환하는 하이브리드 방식의 가능성을 제시한다.

### 한계 및 논의사항

1. **계산 복잡도**: 매 반복마다 커널 행렬의 고유값 분해를 수행해야 하므로 데이터 크기가 커질 경우 계산 비용이 매우 높다. 저자들 또한 이후 연구에서 이전 반복 회차의 계산 결과를 재사용하는 효율화 방안을 언급하였다.
2. **파라미터 $k$ 설정**: $k$ 값에 따라 성능 변동이 크지만, 이를 결정하는 체계적인 기준이 논문에 명시되지 않았다. 입력 행렬의 구조적 특성에 기반한 $k$ 설정 방법이 필요하다.
3. **후반부 성능 저하**: 일부 데이터셋(letter 등)에서는 쿼리 횟수가 늘어날수록 Uncertainty Sampling에 역전당하는 모습이 보인다. 이는 레버리지 점수가 데이터의 '대표성'은 잘 잡지만, 결정 경계(Decision Boundary) 근처의 '세밀한 정보'를 잡는 데는 한계가 있을 수 있음을 시사한다.

## 📌 TL;DR

본 논문은 통계적 레버리지 점수(Statistical Leverage Score)를 활용하여 Active Learning의 쿼리 효율을 높이는 **ALEVS** 방법론을 제안하였다. 데이터셋을 예측 클래스별로 나누어 커널 행렬의 상위 고유벡터 노름을 통해 영향력 있는 샘플을 선택하는 방식이며, 실험 결과 다수의 데이터셋에서 기존의 무작위 및 불확실성 샘플링보다 우수한 성능, 특히 빠른 초기 수렴 속도를 보였다. 이 연구는 데이터의 통계적 구조 분석이 능동 학습의 샘플 선택 전략에 효과적으로 적용될 수 있음을 입증하였다.
