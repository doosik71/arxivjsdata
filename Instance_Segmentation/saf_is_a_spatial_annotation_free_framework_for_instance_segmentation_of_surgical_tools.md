# SAF-IS: a Spatial Annotation Free Framework for Instance Segmentation of Surgical Tools

Luca Sestini, Benoit Rosa, Elena De Momi, Giancarlo Ferrigno, Nicolas Padoy

## 🧩 해결하려는 문제

수술 도구의 인스턴스 분할(Instance Segmentation)은 컴퓨터 보조 수술 애플리케이션 개발에 매우 중요한 오랜 연구 문제입니다. 기존의 대부분 접근 방식은 픽셀 수준의 주석(annotation)이 필요한 완전 지도(fully-supervised) 딥러닝 모델 훈련에 의존하며, 이는 시간과 비용이 많이 드는 단점이 있습니다. 본 논문은 훈련 시 공간 주석(spatial annotation)에 의존하지 않는 인스턴스 분할 프레임워크를 개발하는 것을 목표로 합니다.

## ✨ 주요 기여

* **비지도 도구 인스턴스화 (Unsupervised Tool Instantiation):** 이 단계는 바이너리 마스크(binary mask)로부터 각 도구 인스턴스에 대한 개별 바이너리 분할 마스크를 추출하도록 모델을 훈련시킵니다. 픽셀 수준 주석 없이, 연결된 구성 요소(connected component) 인스턴스화에서 가상 지도(pseudo-supervision) 신호를 생성하고, 도구 위치에 대한 간단한 가정을 사용하여 이를 정제합니다. 이 신호는 각 인스턴스 중심의 2D 변위장(displacement field)을 직접 예측하도록 모델을 훈련하는 데 사용됩니다.
* **자기 지도 특징 표현 학습 (Self-Supervised Feature Representation Learning):** 각 도구 인스턴스를 의미론적 특징을 포착하는 압축 벡터 표현으로 인코딩하는 모델을 훈련합니다. 픽셀 수준의 의미론적 주석 없이, 비디오 시퀀스의 고유한 시간 정보를 활용하여 표현을 학습합니다. 특히, 로컬 인스턴스 트래킹(local instance tracking) 기반의 대조 학습(contrastive learning) 접근 방식을 설계하여 긍정 및 부정 샘플을 생성합니다.
* **바이너리 도구 존재 레이블을 이용한 인스턴스 분류 (Instance Classification from Binary Tool Presence Labels):** 학습된 특징 표현을 사용하여 적은 수의 대표적인 인스턴스(실험에서 8개)를 자동으로 선택하고, 인간 작업자가 이들에 대해 도구 유형 레이블을 부여하도록 합니다. 이 정보는 각 훈련 인스턴스를 해당 바이너리 도구 존재 레이블과 매칭시키는 데 사용되며, 이는 도구 인스턴스 분류기 훈련에 효과적인 지도 신호를 제공합니다.
* **추론 (Inference):** 훈련된 아키텍처는 단일 프레임에서 개별 도구 인스턴스를 추출하고, 각 인스턴스를 압축 특징 표현으로 인코딩한 후 개별적으로 분류하여 인스턴스 분할을 수행할 수 있습니다.

## 📎 관련 연구

* **기존의 수술 도구 분할:** 딥러닝 이전에는 색상 분포, 모양, 시야 내 방향과 같은 사전 지식에 의존했습니다.
* **완전 지도 딥러닝 솔루션:** 딥러닝 도입 이후 U-Net 기반의 인코더-디코더 아키텍처는 의미론적 분할(Semantic Segmentation)에 널리 채택되었습니다. 인스턴스 분할(Instance Segmentation)의 경우 Mask-RCNN 기반 접근 방식(예: ISI-Net, Mask-then-classify, Tra-SeTr)이 주로 사용되었으며, 종종 시간 정보를 활용하여 성능을 향상시켰습니다. 그러나 이들은 모두 픽셀 수준의 값비싼 주석을 요구합니다.
* **비완전 지도 솔루션 (Non Fully-Supervised Solutions):**
  * **준 지도(Semi-Supervised) 솔루션:** 레이블이 없는 데이터를 훈련 과정에 통합하며, 소량의 수동 주석 데이터가 필요합니다 (예: Cycle-GAN을 이용한 사전 훈련, 자기 지도 표현 학습, 옵티컬 플로우를 이용한 희소 주석 전파).
  * **약 지도(Weakly-Supervised) 솔루션:** 완전 지도 방식보다 저렴한 주석(예: 스크리블, 바이너리 도구 존재 레이블)을 사용합니다. 주로 경계 상자(bounding-box) 탐지 작업에 국한되었으며, 픽셀 단위 분할에는 한계가 있었습니다. 특히, 프레임별(frame-wise) 레이블에 초점을 맞추어 시퀀스별(sequence-wise) 레이블의 활용은 제한적이었습니다.
  * **사전 지식 기반(Prior knowledge based) 솔루션:** 내시경 프레임의 색상 분포, 도구 위치, 사전 모양 지식과 같은 일반적인 가정들을 정보원으로 활용합니다 (예: FUN-SIS).

본 연구는 이러한 기존 연구의 한계를 극복하기 위해 사전 지식과 바이너리 도구 존재 레이블을 결합하여 수술 도구의 인스턴스 분할을 학습합니다.

## 🛠️ 방법론

제안하는 SAF-IS 프레임워크는 도구 인스턴스화, 특징 표현 학습, 인스턴스 분류의 세 가지 핵심 구성 요소로 명확히 나뉩니다.

1. **도구 인스턴스화 (Tool Instantiation):**
    * 목표: 내시경 이미지 $I$에서 각 개별 도구에 해당하는 바이너리 마스크 $\{M_{Inst_i}\}$ 세트를 예측합니다.
    * 정형화: 각 도구 픽셀을 인스턴스에 고유하게 할당하는 변위장 $D \in \mathbb{R}^{W \times H \times 2}$을 학습합니다. 픽셀 $p=[p_x, p_y]$에 대해 $D|_p$는 해당 인스턴스 $i$의 중심 $[c_{i_x}, c_{i_y}]$를 가리키는 벡터 $[c_{i_x} - p_x, c_{i_y} - p_y]$이거나, 배경일 경우 $[0,0]$입니다.
    * 훈련:
        * 바이너리 마스크 $M_B$만 알려져 있을 때, 도구가 겹치지 않는다는 가정하에 $M_B$의 연결된 구성 요소(CC)로부터 변위장 $D_{CC}$를 얻습니다.
        * **오버랩 문제 완화:** 잠재적으로 겹치는 도구(OV)를 식별하여 손실 계산에서 제외하고, "인스턴스 붙여넣기 증강(AugmPaste, PS)"을 통해 겹치는 인스턴스의 존재를 인위적으로 시뮬레이션합니다.
        * 손실 함수: $L_I = |D_{CC^*} - \tilde{D}|(1 - M_{OV}) + L_{CE}(M_{B^*}, \tilde{M}_B)$
    * 추론: 예측된 변위장 $\tilde{D}$에서 높은 수렴률을 보이는 픽셀 영역을 중심으로 식별하고, 각 도구 픽셀을 가장 가까운 중심에 할당하여 인스턴스 마스크를 얻습니다.

2. **특징 표현 학습 (Feature Representation Learning):**
    * 목표: 각 도구 인스턴스 마스크 $\tilde{M}_{Inst_i}$를 ResNet-50 기반 특징 추출기를 사용하여 의미론적 내용을 포착하는 특징 벡터 $F_i$로 매핑합니다.
    * 자기 지도 대조 학습:
        * **긍정 샘플:** 인스턴스 $i$의 프레임별 트래킹(optical flow를 사용하여 인접 프레임 간 중심을 매칭)을 통해 구축된 "인스턴스 튜브" $T_i$에서 샘플링합니다.
        * **부정 샘플:** 같은 프레임의 다른 튜브 또는 시간상 멀리 떨어진 튜브에서 샘플링합니다.
        * 손실 함수: $L_F = L_{SCL}(\{F^+_i\}, \{F^-_i\})$ (Supervised Contrastive Loss)

3. **인스턴스 분류 (Instance Classification):**
    * 목표: 인스턴스 특징을 도구 유형 클래스 레이블에 매핑하는 분류 모델을 훈련합니다.
    * **프로토타입 레이블링 (Prototype Labelling):**
        * 훈련 세트의 모든 인스턴스에 대해 학습된 특징에 비지도 클러스터링(K-Means++)을 적용하여 $N_{km}$개의 클러스터 중심에 해당하는 "프로토타입 인스턴스"를 선택합니다.
        * 인간 작업자가 이 $N_{km}$개의 프로토타입 인스턴스에 도구 유형 레이블 $S_P$를 부여합니다. 이 레이블은 같은 클러스터에 속하는 모든 인스턴스에 전파됩니다.
    * **바이너리 도구 존재 레이블 통합 (Binary Tool Presence Labels Incorporation):**
        * 프레임과 관련된 바이너리 도구 존재 레이블(약한 레이블, $S^W_i$)을 활용합니다. 이는 프레임별(frame-wise) 또는 시퀀스별(sequence-wise)일 수 있습니다.
        * **Teacher-Student 접근 방식:**
            * Teacher 네트워크는 각 특징 $F_i$를 해당 프로토타입 레이블 $S^P_i$에 매핑하도록 훈련됩니다. ($L_{T_i} = L_{CE}(\tilde{P}^T_i, S^P_i)$)
            * Teacher 네트워크의 예측을 사용하여 주어진 프레임의 약한 레이블 $S^W_i$의 가능한 조합과 순열 중에서 가장 적합한 "정렬된 약한 레이블 세트"를 찾습니다. 이는 각 조합/순열에 대한 할당 비용 $C_{\langle i_C, i_P \rangle}$를 Teacher의 예측과 약한 레이블 간의 평균 교차 엔트로피 손실로 계산하여 최소화하는 것을 목표로 합니다.
            * Student 네트워크는 이 "매칭된 약한 레이블"로 훈련됩니다. ($L_{S_i} = L_{CE}(\tilde{P}^S_i, \bar{S}^W_{\langle i_C, i_P \rangle_i})$)

## 📊 결과

* **데이터셋:** MICCAI 2017 및 2018 EndoVis Robotic Instrument Segmentation Challenge 데이터셋.
* **바이너리 마스크 소스:** 수동 주석된 바이너리 마스크와 비지도 FUN-SIS 예측 마스크를 모두 사용하여 실험을 수행했습니다.
* **도구 인스턴스화:** 제안된 SAF-IS는 Mask-RCNN baseline에 비해 두 데이터셋 모두에서 우수한 성능을 보였습니다. 특히, FUN-SIS 예측 마스크와 같은 잡음이 있는 지도 신호를 사용할 때 +$\Delta$17.5% AP@0.5 (EndoVis 2017)의 큰 성능 차이로 Mask-RCNN을 능가하여, 잡음에 대한 강건성을 입증했습니다.
* **도구 인스턴스 분할:**
  * SAF-IS는 8개의 프로토타입 레이블 (전체 훈련 인스턴스의 약 0.3%)과 바이너리 도구 존재 레이블(프레임별 또는 시퀀스별)만을 사용하여 훈련되었음에도 불구하고, Ternaus, MF-TN, DMF-TN과 같은 완전 지도 및 준 지도 의미론적 분할 솔루션보다 뛰어난 성능을 보였습니다.
  * EndoVis 2017 데이터셋에서 표준 Mask-RCNN보다 우수한 성능을 달성했습니다.
  * 프레임별 및 시퀀스별 바이너리 도구 존재 레이블 간의 성능 차이가 1.2% IoU 미만으로 작아, 자동으로 얻을 수 있는 시퀀스별 레이블의 효과를 입증했습니다.
* **어블레이션 연구:** 도구 인스턴스화 증강 전략(오버랩 마스킹, 인스턴스 붙여넣기)의 효과를 확인했으며, 8개의 프로토타입 레이블이 인스턴스 분류에 충분함을 보여주었습니다.

## 🧠 통찰 및 토론

* **주요 의의:** SAF-IS 프레임워크는 수술 도구 인스턴스 분할에서 값비싼 픽셀 수준의 공간 주석 필요성을 성공적으로 제거했습니다.
* **주석 효율성:** 단 8개의 프로토타입 레이블(전체 훈련 인스턴스의 약 0.26%)이라는 최소한의 인간 주석으로도 강력한 성능을 달성하여, 기존 준 지도 솔루션(예: 픽셀 수준 주석이 필요한 30%의 프레임)을 훨씬 능가합니다.
* **약한 레이블의 활용:** 기존에는 간과되었던 시퀀스별 바이너리 도구 존재 레이블을 효과적으로 통합하여, 수집 비용이 전혀 없는 지도 소스임을 입증했습니다.
* **한계 및 향후 연구:** 최상위 성능의 완전 지도 인스턴스 분할 방법론(추론 시 시간 정보 또는 추가 도구 부분 주석 활용)과의 성능 차이가 아직 존재합니다. 향후 연구는 추론 시 시간 모델링 통합, 약한 정보를 활용한 특징 학습 개선, SAM(Segment Anything Model)과 같은 최신 솔루션과의 연계를 통해 성능을 더욱 향상시킬 수 있습니다.
* **결론적으로,** SAF-IS의 주요 기여는 훈련 데이터의 공간 주석 필요성을 제거하는 능력에 있습니다. 이는 인간 주석 노력을 특히 대표적이거나 도전적인 샘플에 집중시키는 등 새로운 연구 방향을 열 수 있습니다.

## 📌 TL;DR

* **문제:** 수술 도구 인스턴스 분할은 픽셀 단위의 고비용 주석이 필수적이었습니다.
* **제안 방법:** SAF-IS는 공간 주석이 없는 프레임워크로, (1) 바이너리 마스크와 변위장을 이용한 비지도 도구 인스턴스화, (2) 시간 트래킹과 대조 학습을 통한 자기 지도 특징 표현 학습, (3) 최소한의 인간 레이블링된 프로토타입(8개)과 저렴한 바이너리 도구 존재 레이블(프레임별 또는 시퀀스별)을 활용하는 Teacher-Student 모델 기반 인스턴스 분류로 구성됩니다.
* **핵심 발견:** SAF-IS는 상당한 주석 노력(단 8개의 프로토타입 레이블)을 줄이면서도 여러 완전 지도 의미론적 분할 방법과 Mask-RCNN보다 우수한 성능을 달성하며, 자동으로 얻을 수 있는 시퀀스별 레이블의 효과를 입증했습니다.
