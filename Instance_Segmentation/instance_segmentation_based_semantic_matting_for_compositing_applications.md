# Instance Segmentation based Semantic Matting for Compositing Applications

Guanqing Hu, James J. Clark (2019)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 자연스러운 배경(natural scenes)을 가진 이미지에서 전경 객체를 정밀하게 추출하여 새로운 배경과 합성하는 **이미지 합성(Image Compositing)**의 자동화이다.

전통적으로 이미지 합성을 위해서는 전경과 배경을 분리하는 작업이 필수적이다. 스튜디오 환경에서는 크로마키(chroma-keying)를 통해 쉽게 구현할 수 있지만, 복잡한 자연 배경에서는 숙련된 작업자가 수동으로 외곽선을 따는 로토스코핑(rotoscopy) 과정이 필요하여 막대한 시간과 비용이 소모된다.

기존의 자동화 시도들은 다음과 같은 한계가 있었다.

1. **Instance Segmentation**: 딥러닝 기반의 세그멘테이션은 빠르게 발전했으나, 결과물인 마스크가 너무 거칠어(coarse) 고품질 합성에 사용하기에는 경계선 디테일이 부족하다.
2. **Image Matting**: 전경과 배경의 불투명도를 나타내는 알파 매트(alpha matte)를 추정하여 정밀한 추출이 가능하지만, 대부분의 알고리즘이 트리맵(trimap)이나 스트로크(strokes) 같은 사용자 정의 입력에 의존하는 반자동 방식이라는 점이다.

따라서 본 논문의 목표는 사용자 개입 없이 **Instance Segmentation과 Image Matting을 통합하여, 복잡한 자연 배경에서도 고품질의 시맨틱 알파 매트(semantic alpha mattes)를 생성하는 완전 자동화된 파이프라인을 구축**하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Instance Segmentation의 시맨틱 정보(객체 종류 및 위치)를 이용하여 Image Matting에 필요한 트리맵을 자동으로 생성하고, 이를 피드백 루프(feedback loop)를 통해 점진적으로 정교화**하는 것이다.

단순히 두 기술을 이어 붙인 것이 아니라, 세그멘테이션 마스크로부터 트리맵을 유도하고, 여기서 생성된 알파 매트를 다시 트리맵 생성에 반영함으로써 마스크의 거친 경계선을 정밀한 알파 매트로 변환하는 반복적 정제 과정을 제안하였다. 이를 통해 사용자의 수동 입력 없이도 개별 객체 단위의 정밀한 추출과 합성이 가능해졌다.

## 📎 Related Works

### 1. Natural Image Matting

이미지 매팅은 각 픽셀의 불투명도 값인 알파 매트를 추정하는 작업이다.

- **Non-learning-based**: 샘플링 기반, 베이지안 기반, Closed-form Matting 등이 있으며, 주로 트리맵이라는 사용자 입력이 필요하다.
- **Learning-based**: 최근 딥러닝을 이용해 데이터셋으로부터 학습하는 방식이 등장하였다. 특히 Deep Image Matting [3]과 같은 인코더-디코더 구조가 제안되었으며, 이는 색상이 유사한 전경과 배경을 구분하는 능력이 뛰어나다.

### 2. Instance Segmentation

객체 검출(Object Detection)과 시맨틱 세그멘테이션(Semantic Segmentation)을 결합한 형태로, Mask R-CNN [2]이 대표적이다. 이는 객체의 클래스, 바운딩 박스, 그리고 픽셀 단위의 마스크를 제공한다. 하지만 본 논문에서 지적하듯, 이 마스크들은 합성에 쓰기에는 너무 거칠다.

### 3. Soft Segmentation

전통적인 이진 마스크 생성이 아닌, 경계선의 전이(transition)를 정밀하게 추정하는 방식이다. Semantic Soft Segmentation [5]은 DeepLab의 고수준 특징과 spectral decomposition의 저수준 특징을 결합하여 자동화된 소프트 세그멘테이션을 수행한다.

## 🛠️ Methodology

### 1. 전체 파이프라인 구조

제안된 시스템은 크게 **Segmentation $\rightarrow$ Trimap Estimation $\rightarrow$ Matting**의 순서로 진행되며, Matting 결과가 다시 Trimap 생성으로 이어지는 **Feedback Loop** 구조를 가진다.

### 2. 상세 구성 요소 및 절차

#### (1) Segmentation Stage

입력 이미지에서 객체의 바운딩 박스와 인스턴스 마스크를 생성하기 위해 **Mask R-CNN**을 사용한다. 정확도를 높이기 위해 ResNeXt-152-32x8d-FPN 백본을 사용하며, ImageNet-5k로 사전 학습된 모델을 활용한다.

#### (2) Trimap Estimation Stage

트리맵은 전경($\alpha=1$), 배경($\alpha=0$), 그리고 불확실한 경계 영역($\alpha=0.5$)으로 구성된다. 본 논문에서는 Mask R-CNN의 마스크 경계를 기준으로 **팽창(dilation)** 연산을 수행하여 불확실 영역을 정의한다.

- 팽창 정도는 객체의 크기(바운딩 박스의 너비와 높이의 평균)에 비례하여 결정한다.
- 마스크 내부의 깊은 곳은 전경, 마스크 외부의 먼 곳은 배경, 그 사이의 팽창 영역은 불확실 영역으로 설정한다.

#### (3) Matting Stage

**Deep Image Matting** 네트워크(VGG16 기반 인코더-디코더 및 정제 단계)를 사용하여 최종 알파 매트를 생성한다.

- **수식 모델**: 이미지 매팅의 기본 원리는 다음과 같다.
  $$I_i = \alpha_i \times FG_i + (1 - \alpha_i) \times BG_i$$
  여기서 $I$는 입력 이미지, $FG$는 전경, $BG$는 배경, $\alpha$는 추정해야 할 알파 값이다.
- **추론 절차 (Patch-based processing)**: 고해상도 이미지의 디테일 손실을 막기 위해 이미지를 $320 \times 320$ 크기의 패치로 나누어 처리한다. 각 패치는 트리맵의 불확실 영역(gray region)의 중심을 기준으로 크롭되며, 결과 패치들을 다시 합친다.
- **Multiple Sampling**: 경계면의 불연속성과 이상치(outlier)를 제거하기 위해 동일 이미지에 대해 $K=10$번 반복 실행한 후 각 픽셀의 **중앙값(median)**을 취하는 방식을 사용한다.

#### (4) Feedback Loop

초기 마스크 기반 트리맵은 매우 거칠기 때문에 초기 알파 매트의 품질이 낮다. 이를 해결하기 위해 생성된 알파 매트를 이용해 다시 트리맵을 생성하고 매팅 단계로 보내는 과정을 반복한다.

- 반복 횟수가 늘어날수록 트리맵과 알파 매트가 서로를 정교화하며, 이때 트리맵의 팽창률은 점차 줄여나간다. 일반적으로 **4회의 루프**를 수행했을 때 최적의 결과가 도출되었다.

#### (5) 다중 객체 처리

이미지 내에 여러 객체가 있을 경우, 한 번에 하나의 객체만 전경으로 간주하고 나머지는 배경으로 처리하는 방식을 취하여 개별 알파 매트를 생성한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: MS COCO 데이터셋을 사용하였다.
- **평가 방법**: COCO 데이터셋에는 정답(Ground Truth) 알파 매트가 존재하지 않으므로, 정량적 지표 대신 **정성적 분석(Qualitative Analysis)**을 중심으로 수행하였다.
- **비교 대상**: Semantic Soft Segmentation [5], Spectral Matting [10].

### 2. 주요 결과

- **파이프라인 효과**: Figure 4에서 보듯, Feedback Loop를 거치며 거친 마스크 $\rightarrow$ 초기 트리맵 $\rightarrow$ 정제된 알파 매트로 품질이 점진적으로 향상됨을 확인하였다.
- **비교 분석**:
  - 전경과 배경의 색상이 유사한 경우, 제안 방법이 Semantic Soft Segmentation이나 Spectral Matting보다 더 정밀한 경계를 추출하는 경향을 보였다.
  - 특히 **모션 블러(motion blur)**가 있는 영역(예: 움직이는 팔)에서도 배경이 자연스럽게 투영되는 알파 매트를 생성해냈으며, 이는 비교 대상 모델들이 수행하지 못한 부분이다.
  - **인스턴스 레벨 선택**: 시맨틱 세그멘테이션 기반 방법들은 같은 클래스의 여러 객체를 하나로 묶는 경향이 있으나, 본 방법은 인스턴스 단위로 마스크를 생성하므로 특정 객체만 선택적으로 추출하여 합성하는 것이 가능하다.

### 3. 수행 시간

Nvidia Tesla K40 GPU 기준, Mask R-CNN의 트리맵 생성에 평균 39초, 4회 피드백 루프를 포함한 매팅 단계에 10초가 소요되었다.

## 🧠 Insights & Discussion

### 1. 강점

- **완전 자동화**: 사용자 개입 없이 RGB 이미지 하나만으로 고품질 합성이 가능하다.
- **인스턴스 제어력**: 단순한 클래스 분류를 넘어 개별 객체 단위로 알파 매트를 생성하므로, 편집자의 의도에 따라 특정 객체만 추출하는 유연성이 매우 높다.
- **실제적 유용성**: 모션 블러 처리 능력을 통해 실제 영화나 비디오 합성 작업의 효율성을 크게 높일 수 있는 잠재력을 보여주었다.

### 2. 한계 및 비판적 해석

- **투명도 처리의 한계**: 트리맵 생성이 객체의 경계(border)에 국한되어 있어, 객체 내부의 넓은 투명 영역이나 복잡한 불투명도 변화가 있는 경우에는 정확한 트리맵을 생성하지 못해 결과가 부정확할 수 있다.
- **학습 데이터 의존성**: 매팅 네트워크가 학습한 객체 카테고리에 제한이 있어, 학습되지 않은 생소한 객체에 대해서는 경계선이 너무 흐릿하거나 투명하게 처리되는(fuzzy) 경향이 있다.
- **정량적 검증 부족**: GT 데이터가 없는 COCO 데이터셋을 사용했기에 시각적 비교에 의존하고 있다. 실제 합성 품질을 측정할 수 있는 벤치마크 데이터셋에서의 정량적 수치가 제시되지 않은 점은 아쉬운 부분이다.

## 📌 TL;DR

본 논문은 **Mask R-CNN의 인스턴스 세그멘테이션과 Deep Image Matting을 결합한 완전 자동화된 이미지 추출 파이프라인**을 제안한다. 세그멘테이션 마스크로부터 트리맵을 자동 생성하고, **4단계의 피드백 루프**를 통해 알파 매트를 점진적으로 정교화함으로써 사용자 입력 없이도 고품질의 합성이 가능하게 하였다. 특히 인스턴스 단위의 제어와 모션 블러 처리 능력이 뛰어나, 향후 전문적인 영상 편집 및 합성 작업의 시간을 획기적으로 단축시킬 수 있을 것으로 기대된다.
