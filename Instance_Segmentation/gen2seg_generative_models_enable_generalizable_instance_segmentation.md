# GEN2SEG: Generative Models Enable Generalizable Instance Segmentation

Om Khangaonkar, Hamed Pirsiavash (2025)

## 🧩 Problem to Solve

본 논문은 **카테고리 불가지론적 인스턴스 분할(Category-agnostic Instance Segmentation)** 문제, 즉 특정 클래스 정보 없이 이미지 내의 개별 객체들을 분리해내는 문제에 집중한다.

기존의 Zero-shot 분할 모델들은 대규모의 레이블링된 데이터셋(예: SAM의 SA-1B)에 의존하여 광범위한 객체 카테고리와 스타일을 학습함으로써 일반화 성능을 확보하였다. 그러나 저자들은 "모델이 시각적 세계의 매우 좁은 부분(제한된 객체 유형)만 학습하고도, 학습 시 보지 못한 새로운 객체 유형과 이미지 스타일에 대해 일반화할 수 있는가?"라는 질문을 던진다.

이 문제의 핵심은 데이터의 양이나 다양성이 아닌, 모델이 가진 **내재적인 그룹화 메커니즘(Inherent grouping mechanism)**을 통해 인간처럼 처음 보는 객체라도 '하나의 독립된 물체'로 인식하게 만드는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **생성 모델(Generative Models)이 이미지를 합성하는 과정에서 객체의 경계와 장면 구성에 대한 깊은 이해를 내재적으로 학습한다**는 가설에서 출발한다.

구체적으로, Stable Diffusion이나 Masked Autoencoder(MAE)와 같은 모델들은 훼손된 입력이나 텍스트 프롬프트로부터 정교한 이미지를 생성해야 하므로, 객체의 구조와 부분-전체 관계를 모델링하는 능력을 갖추게 된다. 저자들은 이러한 생성적 사전 학습(Generative Pretraining) 지식을 활용하여, 매우 제한된 도메인의 데이터(실내 가구 및 자동차)만으로 파인튜닝하여도 보지 못한 카테고리에 대해 강력한 Zero-shot 일반화 성능을 보임을 입증하였다.

## 📎 Related Works

### 관련 연구 및 한계

- **Perception을 위한 생성 모델:** 과거 GAN이나 Inpainting 기반 연구들이 표현 학습(Representation Learning)에 사용되었으나, 판별적 사전 학습(Discriminative Pretraining) 모델에 비해 성능이 낮았다. 최근 Diffusion 모델의 등장으로 3D 재구성, 시맨틱 분할 등 다양한 인식 작업으로 확장되고 있다.
- **인스턴스 분할:** SAM(Segment Anything Model)은 거대 데이터셋을 통해 일반화 능력을 확보한 대표적인 모델이다. 또한, Contrastive learning 기반의 특징 추출과 Normalized Cuts 등을 이용해 의사 레이블(Pseudo-mask)을 생성하는 방식들이 연구되었다.

### 기존 방식과의 차별점

- **데이터 효율성:** SAM이 수십억 개의 마스크를 학습한 반면, 본 연구의 모델은 매우 좁은 범위의 합성 데이터셋(약 87,000장 이미지)만으로 학습되었으며, 학습 시간과 자원 소모가 훨씬 적다.
- **일반화의 원천:** 데이터의 다양성으로 일반화를 꾀하는 기존 방식과 달리, 생성 모델이 가진 **생성적 사전 학습(Generative Prior)** 자체를 활용하여 일반화 성능을 얻는다.

## 🛠️ Methodology

### 1. 이미지-투-이미지 변환으로서의 분할

본 연구는 인스턴스 분할을 RGB 이미지-투-이미지 변환 문제로 재정의한다. 정답(Ground Truth) 마스크를 RGB 이미지 형태로 인코딩하며, 이때 각 인스턴스는 고유한 색상을 가지고 배경은 검은색($[0, 0, 0]$)으로 처리한다.

### 2. Instance Coloring Loss ($L_{IC}$)

특정 색상을 강제하는 대신, 마스크의 색상 특성(낮은 분산, 인스턴스 간 구분)을 최적화하는 손실 함수를 제안한다.

- **Intra-Instance Variance Loss ($L_{var}$):** 동일한 인스턴스 내부의 픽셀들이 일관된 색상을 갖도록 강제한다. Smooth $\ell_1$ 손실을 사용하여 이상치에 대한 민감도를 낮춘다.
  $$L_{var} = \sum_{i=0}^{n} \frac{1}{|S_i|} \sum_{j \in S_i} \sum_{c=0}^{2} L_s(p_{j,c}, \mu_{i,c})$$
  ($S_i$: 인스턴스 $i$의 픽셀 집합, $p_{j,c}$: 예측 색상, $\mu_{i,c}$: 인스턴스 평균 색상)

- **Inter-Instance Separation Loss ($L_{sep}$):** 인스턴스 외부에 있는 픽셀들의 색상이 해당 인스턴스의 평균 색상으로부터 멀어지도록 밀어낸다. 작은 객체의 중요도를 높이기 위해 $\sqrt{|S_i|}$ 항을 분모에 포함한다.
  $$L_{sep} = \sum_{i=0}^{n} \frac{1}{\sqrt{|S_i||T_i|}} \sum_{j \in T_i} \frac{1}{1 + \sum_{c=0}^{2} (p_{j,c} - \mu_{i,c})^2}$$
  ($T_i$: 인스턴스 $i$ 외부의 픽셀 집합)

- **Mean-Level Separation Loss ($L_{mean}$):** 서로 다른 인스턴스들의 평균 색상(Centroids)끼리 서로 구분되도록 강제한다.
  $$L_{mean} = \frac{1}{n(n+1)} \sum_{0 \le i < j \le n} \frac{1}{1 + \sum_{c=0}^{2} (\mu_{i,c} - \mu_{j,c})^2}$$

최종 손실 함수는 다음과 같다:
$$L_{IC} = L_{var} + \lambda_{sep} L_{sep} + \lambda_{mean} L_{mean}$$

### 3. Point-Promptable Instance Segmentation

학습된 모델이 생성한 특징 맵 $F$를 사용하여 포인트 프롬프트 기반의 이진 마스크를 생성한다. 별도의 마스크 디코더를 학습시키지 않고 특징 맵의 유사도만을 이용한다.

1. 프롬프트 지점 $p$ 주변의 가우시안 가중 평균을 통해 쿼리 벡터 $q_p$를 계산한다.
2. 픽셀별 특징 $F(x,y)$와 $q_p$ 사이의 유사도 맵 $S_p$를 생성한다.
3. Joint Bilateral Filter를 통해 유사도 맵을 부드럽게 다듬고, 임계값(Threshold)을 적용하여 최종 마스크를 생성한다.

## 📊 Results

### 실험 설정

- **학습 데이터:** Hypersim(실내 가구) 및 Virtual Kitti 2(자동차) 합성 데이터셋만 사용.
- **평가 데이터셋:** 학습 시 보지 못한 카테고리가 포함된 $\text{COCO}_{\text{exc}}$, 예술 작품(DRAM), 1인칭 시점(EgoHOS), 복잡한 미세 구조(iShape), X-ray 이미지(PIDRay) 등 5개 도메인.
- **비교 대상:** SAM, SimpleClick(MAE-B 기반), DINO-B(DINO 특징 + VAE 디코더).

### 주요 결과

- **Zero-shot 일반화:** Stable Diffusion(SD) 기반 모델은 학습하지 않은 객체들에 대해서도 SAM과 대등하거나 일부 상회하는 mIoU 성능을 보였다. 특히 미세 구조 분할 작업인 iShape 데이터셋에서 SAM을 크게 능가하였다.
- **경계 정밀도:** BSDS500 데이터셋에서 엣지 검출(Edge Detection) 성능을 평가한 결과, SD 기반 모델이 SAM보다 훨씬 더 정교하고 깨끗한 객체 경계를 생성함을 확인하였다. 이는 생성 모델이 이미지를 합성하기 위해 엣지와 세부 구조를 정밀하게 모델링해야 한다는 특성에서 기인한다.
- **데이터 효율성:** 단 5~10개의 클래스만으로 학습하더라도 전체 데이터셋으로 학습한 것과 유사한 일반화 성능을 보였다. 이는 일반화 능력이 학습 데이터의 다양성이 아닌 생성 모델의 Prior에서 온다는 가설을 뒷받침한다.

## 🧠 Insights & Discussion

### 강점 및 분석

- **Equivariant Representations:** 저자들은 DINO와 같은 판별적 모델이 불변성(Invariance)을 강조하는 반면, 생성 모델은 입력의 변화(스케일, 모양, 구조)에 따라 적절한 이미지를 생성해야 하므로 등변성(Equivariance) 표현을 학습하며, 이것이 인스턴스 분할에 더 유리하다고 분석한다.
- **생성적 사전 학습의 힘:** 단순한 RGB 색상 예측 학습임에도 불구하고, 모델이 이전에 학습한 '물체란 무엇인가'에 대한 생성적 지식이 전이되어 새로운 객체의 형태를 정확히 짚어낸다.

### 한계 및 미해결 과제

- **소형 객체 인식:** Stable Diffusion의 텍스트 조건화 편향(크고 두드러진 객체 강조)이나 MAE의 ImageNet 편향(중앙의 메인 객체 강조)으로 인해 작은 객체 분할 성능이 저하되는 경향이 있다.
- **배경 오분류:** 일부 모델은 구름이나 풀과 같은 영역을 객체가 아닌 배경으로 처리하는 경향이 있으며, 이는 생성 모델의 내재적 편향일 가능성이 있다.

## 📌 TL;DR

본 논문은 **생성 모델(Stable Diffusion, MAE)이 이미지 합성 능력을 통해 이미 객체의 구조와 경계를 이해하고 있다**는 점을 이용해, 아주 적은 양의 특정 카테고리 데이터만으로도 처음 보는 객체를 분할할 수 있는 **Gen2Seg** 프레임워크를 제안한다. 이 모델은 특히 **미세한 구조 분할과 정교한 경계 생성**에서 SAM과 같은 거대 지도학습 모델보다 우수한 성능을 보였으며, 이는 생성적 사전 학습이 일반화 가능한 인식 능력을 구축하는 강력한 방법이 될 수 있음을 시사한다.
