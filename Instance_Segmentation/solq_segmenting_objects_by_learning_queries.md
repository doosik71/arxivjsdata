# SOLQ: Segmenting Objects by Learning Queries

Bin Dong, Fangao Zeng, Tiancai Wang, Xiangyu Zhang, Yichen Wei

## 🧩 Problem to Solve

기존의 인스턴스 세그멘테이션 방법들은 대부분 두 단계(detect-then-segment) 패러다임을 따르며, 이는 검출 브랜치에 크게 의존하여 다중 작업의 공동 학습을 어렵게 합니다. SOLO와 같은 직접 예측 방식도 NMS와 같은 후처리 과정이 필요하고, 작은 객체 성능이 만족스럽지 못하다는 한계가 있었습니다. DETR [1]은 객체 검출을 위한 엔드-투-엔드 솔루션을 제시했지만, 공간 이진 마스크를 임베딩 도메인으로 효과적으로 인코딩하여 인스턴스 세그멘테이션에 확장하는 것은 어려운 문제로 남아있었습니다. 특히, DETR의 FPN 스타일 마스크 생성 방식은 공간 정보 모델링에 취약하며 높은 해상도의 마스크 레이블로 인해 계산 비용이 높다는 문제가 있습니다.

## ✨ Key Contributions

* **엔드-투-엔드 인스턴스 세그멘테이션 프레임워크 제안**: DETR [1]을 기반으로 하며, 객체 쿼리를 학습하여 인스턴스 마스크를 병렬로 예측하는 SOLQ(Segmenting Objects by Learning Queries)를 제안합니다.
* **통합 쿼리 표현(UQR) 도입**: 마스크 표현을 공간 도메인에서 임베딩 도메인으로 변환하여 DETR의 학습 가능한 쿼리 임베딩과 일관성을 유지합니다. UQR은 분류, 박스 회귀, 마스크 인코딩을 하나의 벡터 형태로 동시에 수행합니다.
* **마스크 압축 코딩**: 고해상도 공간 마스크를 저차원 마스크 벡터로 압축하여 효율적인 마스크 표현을 가능하게 하며, 훈련 시 마스크 벡터를 감독하고 추론 시 공간 마스크로 복원할 수 있도록 합니다.
* **우수한 성능 달성**: MS COCO 데이터셋에서 ResNet101 백본으로 40.9% mask AP와 48.7% box AP를 달성하여 기존 SOTA 모델들을 능가합니다. UQR의 공동 학습을 통해 DETR의 검출 성능을 2.0% 향상시켰습니다.
* **Transformer 기반 인스턴스 세그멘테이션의 강력한 베이스라인 제시**: ROIAlign과 같은 두 단계 작업 없이 진정한 원샷(one-shot) 프레임워크를 제공합니다.

## 📎 Related Works

* **인스턴스 세그멘테이션**:
  * **하향식(Top-down)**: Mask R-CNN [2], Cascade Mask R-CNN [7] 등. 객체 검출 후 RoIAlign [2] 또는 동적 컨볼루션 [9]으로 마스크를 분할.
  * **상향식(Bottom-up)**: 픽셀별 임베딩을 학습하여 인스턴스 그룹으로 클러스터링.
  * **직접 예측(Directly-predict)**: PolarMask [29], SOLO [15], SOLOv2 [16]. 바운딩 박스나 임베딩 학습에 의존하지 않고 위치 기반으로 직접 마스크를 출력. QueryInst [24] 및 ISTR [30]은 Sparse RCNN [31]을 확장하여 엔드-투-엔드 인스턴스 세그멘테이션을 수행.
* **비전 분야의 트랜스포머**:
  * Non-local block [33], CCNet [34], DETR [1], Deformable DETR [35], IPT [36], ViT [37] 시리즈 등. DETR은 학습 가능한 쿼리와 트랜스포머 아키텍처를 활용하여 NMS 없이 엔드-투-엔드 객체 검출을 수행.
* **비전 분야의 압축 코딩**:
  * Sparse Coding [41], PCA [42], DCT [43] 등. DCT-Mask [45], MEInst [21], ISTR [30]은 이진 마스크를 고정 차원 벡터로 인코딩하는 데 활용.

## 🛠️ Methodology

SOLQ는 특징 추출 네트워크, 트랜스포머 디코더, 통합 쿼리 표현(UQR)의 세 부분으로 구성됩니다.

1. **특징 추출 네트워크 (Feature Extraction Network)**:
    * ResNet [50] 백본을 사용하여 입력 이미지 $I \in R^{H \times W \times 3}$에서 기본 특징 맵 $x_0 \in R^{C \times \frac{H}{32} \times \frac{W}{32}}$를 추출합니다.
    * $K$개의 트랜스포머 인코더 레이어를 통해 정제된 특징 맵 $x_K \in R^{C \times \frac{HW}{32^2}}$를 얻습니다. 각 인코더 레이어 $F_k(\cdot)$는 멀티헤드 셀프 어텐션(MHSA)과 피드포워드 네트워크(FFN)로 구성됩니다.
2. **트랜스포머 디코더 (Transformer Decoder)**:
    * $J$개의 학습 가능한 객체 쿼리 $q_0 \in R^{J \times C}$를 무작위로 초기화합니다.
    * $K$개의 트랜스포머 디코더 레이어 $H_k(\cdot)$에서 $q_0$는 정제된 특징 맵 $x_K$와 상호작용하여 인스턴스-인식 쿼리 임베딩 $q_K \in R^{J \times C}$를 생성합니다. 각 디코더 레이어는 크로스 어텐션 레이어를 추가로 가집니다.
3. **통합 쿼리 표현 (Unified Query Representation, UQR)**:
    * $q_K$의 각 인스턴스-인식 쿼리 임베딩은 해당 인스턴스의 특징을 나타냅니다.
    * UQR은 분류, 로컬라이제이션, 마스크 세그멘테이션의 세 가지 하위 작업을 병렬로 예측합니다.
        * **분류 (Classification)**: FC 레이어를 통해 클래스 신뢰도 $\hat{c}$를 예측합니다.
        * **로컬라이제이션 (Localization)**: MLP를 통해 4개의 박스 좌표 $\hat{b}$를 예측합니다.
        * **마스크 (Mask)**: MLP를 통해 마스크 벡터 $\hat{v} \in R^{J \times n_k}$를 예측합니다. 여기서 $n_k$는 각 마스크 벡터의 차원입니다.
    * **마스크 압축 코딩 (Mask Compression Coding)**: 2D 공간 이진 마스크 $m \in R^{J \times N \times N}$를 1D 마스크 벡터 $v \in R^{J \times n_k}$로 변환하는 데 사용됩니다.
        * **Sparse Coding [41]**: $m$을 과완전 사전 $D$의 희소 조합으로 압축하여 라쏘(Lasso) 문제로 $v$를 얻습니다. 재구성: $\hat{m} = \hat{v}D$.
        * **PCA [42]**: 행렬 분해를 통해 $m$을 저차원 $v$로 변환합니다. $v=mP$. 재구성: $\hat{m} = \hat{v}P^T$.
        * **DCT [43]**: $m$을 주파수 도메인 $f = AmA^T$로 변환하고 저주파수 컴포넌트를 샘플링하여 $v$를 얻습니다. 재구성: $\hat{m} = A^{-1}\hat{f}(A^T)^{-1}$.
    * **손실 함수 (Loss Function)**:
        $$ L_{inst} = L_{det} + \lambda_{vec} \cdot L_{vec}(v, \hat{v}) $$
        여기서 $L_{det}$는 DETR의 객체 검출 손실($\lambda_{cls} \cdot L_{cls} + \lambda_{L1} \cdot L_{L1} + \lambda_{giou} \cdot L_{giou}$)이며, $L_{vec}$는 마스크 벡터 손실(실제로는 L1 손실)입니다. $\lambda_{vec}$는 마스크 벡터 손실의 가중치입니다. 마스크 손실은 이분 매칭에 포함되지 않습니다.

## 📊 Results

* **SOTA 비교**: COCO 2017 `test-dev` 셋에서 ResNet-101 백본으로 40.9% AP$_{seg}$ 및 48.7% AP$_{box}$를 달성하여 Mask R-CNN, Cascade Mask R-CNN, CondInst, SOLOv2 등 기존 SOTA 방법들을 능가했습니다. 특히, SOLOv2보다 AP$_{seg}$에서 1.2%, AP$_{box}$에서 6.1% 높은 성능을 보였습니다.
* **UQR vs. SQR**: SOLQ의 통합 쿼리 표현(UQR)은 DETR의 개별 쿼리 표현(SQR)에 비해 AP$_{box}$를 ResNet50에서 2.3%, ResNet101에서 2.0% 크게 향상시켰습니다. SQR 방식의 DETR은 AP$_{seg}$ 32.2%에 그친 반면 UQR은 39.5%로 뛰어난 마스크 성능을 보였습니다.
* **마스크 압축 코딩 방법**: DCT가 AP$_{seg}$ 및 AP$_{box}$에서 가장 좋은 성능을 보였습니다. DCT는 실시간으로 각 이미지에 대해 수행될 수 있는 반면, Sparse Coding 및 PCA는 오프라인 방식으로 "주성분" 또는 "사전"을 얻어야 합니다.
* **마스크 벡터 손실 가중치 ($\lambda_{vec}$)**: $\lambda_{vec}=3.0$일 때 최적의 성능을 달성하여 마스크 브랜치와 로컬라이제이션 브랜치 간의 균형을 유지하는 것이 중요함을 보여주었습니다.
* **보조 손실 활성화 디코더 수**: 모든 디코더에 보조 마스크 벡터 손실을 추가하면 AP$_{seg}$ 2.9%, AP$_{box}$ 1.1% 향상을 보였습니다.
* **이진 마스크의 공간 해상도 ($N$)**: $N=128$일 때 AP$_{seg}$가 32.6%로 최적의 성능을 보였으며, 그 이상에서는 성능 향상이 미미했습니다.
* **마스크 벡터의 차원 ($n_k$)**: $N=128$일 때 $n_k=256$이 AP$_{seg}$ 및 AP$_{box}$에서 가장 좋은 성능을 달성했습니다.

## 🧠 Insights & Discussion

* **UQR의 효과**: SOLQ의 통합 쿼리 표현(UQR)은 객체 분류, 위치 파악, 마스크 예측을 하나의 회귀 방식으로 동시 학습함으로써 DETR의 객체 검출 성능까지 크게 향상시켰습니다. 이는 다중 작업 학습의 시너지 효과를 극대화했음을 의미합니다.
* **마스크 압축 코딩의 중요성**: 고해상도 공간 마스크를 저차원 마스크 벡터로 효율적으로 인코딩하고 디코딩하는 방식이 마스크 품질과 모델 학습의 안정성에 결정적인 영향을 미칩니다. 특히 DCT는 작은 객체에 대한 마스크 예측 성능 향상에 기여했습니다.
* **엔드-투-엔드 시스템**: SOLQ는 NMS와 같은 어떤 후처리 과정도 필요 없는 진정한 엔드-투-엔드 인스턴스 세그멘테이션 프레임워크를 제공합니다. 이는 복잡한 파이프라인과 수작업 튜닝을 줄여줍니다.
* **제한 사항**: Boundary-AP 측정에서 대형 객체에 대한 성능이 PointRend나 BMask R-CNN에 비해 상대적으로 낮은 것으로 나타났습니다. 이는 객체 쿼리의 희소한 활성화 및 고정된 코딩 길이가 대형 객체의 광범위한 수용 필드를 포착하는 데 한계가 있기 때문으로 분석됩니다. 향후 다양한 크기의 객체에 적응하기 위한 더 길거나 동적인 쿼리 코딩 길이가 필요할 수 있습니다.

## 📌 TL;DR

SOLQ는 DETR 기반의 엔드-투-엔드 인스턴스 세그멘테이션 프레임워크로, **통합 쿼리 표현(UQR)**을 통해 분류, 바운딩 박스 회귀, 마스크 예측을 단일 벡터 형태로 동시에 수행합니다. 고해상도 공간 마스크는 **DCT 기반의 마스크 압축 코딩**을 통해 저차원 마스크 벡터로 인코딩되어 훈련되고, 추론 시 원래 마스크로 복원됩니다. 이 방식은 MS COCO에서 SOTA 성능을 달성하고 DETR의 검출 성능을 크게 향상시키며, 기존 두 단계 접근법의 한계를 극복하고 NMS와 같은 후처리 없이 동작하는 강력한 Transformer 기반 베이스라인을 제공합니다.
