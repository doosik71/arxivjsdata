# From Forks to Forceps: A New Framework for Instance Segmentation of Surgical Instruments

Britty Baby, Daksh Thapar, Mustafa Chasmai, Tamajit Banerjee, Kunal Dargan, Ashish Suri, Subhashis Banerjee, Chetan Arora

## 🧩 Problem to Solve

최소 침습 수술(Minimally Invasive Surgeries, MIS)에서는 수술 도구의 인스턴스 수준 분류 및 분할이 필수적입니다. 그러나 현재의 문제점은 다음과 같습니다:

* **낮은 클래스 분별력**: 수술 도구는 외형이 유사하고 가늘고 길며 특정 각도로 사용되는 특징이 있어, 자연 이미지로 훈련된 최신(State-Of-The-Art, SOTA) 인스턴스 분할 모델은 도구 클래스를 정확하게 분류하는 데 어려움을 겪습니다.
* **오분류 문제**: 종종 경계 상자와 분할 마스크는 정확하지만, 분류 헤드가 수술 도구의 클래스 레이블을 오분류하는 경우가 발생합니다. 이는 SOTA 모델의 성능 저하의 주된 원인으로 지적됩니다.
* **도구 형상으로 인한 배경 노이즈**: 수술 도구의 긴 종횡비와 비스듬한 배치로 인해, 제안된 경계 상자(bounding box)에 불필요한 배경 특징이 많이 포함되어 분류를 방해합니다.
* **클래스 간 유사성**: 수술 도구는 시각적으로 매우 유사하며, 일반적인 교차 엔트로피 기반 분류기 훈련은 이러한 미세한 차이를 학습하기에 부적합합니다.

## ✨ Key Contributions

이 논문은 수술 도구 인스턴스 분할의 주요 과제인 오분류 문제를 해결하기 위해 다음과 같은 핵심 기여를 합니다:

* **오분류 문제 명확화**: 기존 SOTA 인스턴스 분할 모델이 수술 도구에 대해 정확한 경계 상자와 마스크를 생성하지만, 클래스 레이블을 오분류하는 것이 성능 저하의 주된 원인임을 식별했습니다. 예를 들어, Mask R-CNN의 예측된 클래스 레이블을 정답으로 대체하면 AP$_{50}$ 점수가 0.65에서 0.90으로 크게 향상됨을 보여주었습니다.
* **새로운 3단계 신경망 프레임워크 (S3Net) 제안**: 기존 인스턴스 분할 모델에 분류 모듈을 새로운 단계로 추가하는 프레임워크를 제시했습니다. 이 모듈은 기존 모델이 생성한 도구 마스크의 분류를 개선하는 데 특화되어 있습니다.
* **다중 스케일 마스크 어텐션 (Multi-Scale Mask Attended Classifier, MSMA)**: 제안된 분류 모듈에 마스크 기반 어텐션을 도입하여, 도구 영역에 집중하고 방해되는 배경 특징을 마스킹하여 분류 정확도를 높였습니다. 이는 도구의 긴 형상과 비스듬한 배치 문제를 해결합니다.
* **메트릭 학습 (Metric Learning)과 Arc Loss 활용**: 수술 도구의 낮은 클래스 간 분산(low inter-class variance)을 처리하기 위해 arc loss와 같은 메트릭 학습을 사용하여 분류기 모듈을 훈련하는 방식을 제안했습니다. 이는 클래스 특징 간의 거리를 최대화하여 분류 정확도를 향상시킵니다.
* **SOTA 성능 달성**: EndoVis2017 및 EndoVis2018 벤치마크 데이터셋에서 광범위한 실험을 통해 제안된 방법이 18개 이상의 SOTA 방법을 능가하며, EndoVis2017 챌린지에서 SOTA 성능을 최소 12점 (20%) 향상시켰음을 입증했습니다.

## 📎 Related Works

수술 도구 분할 분야는 주로 다음 두 가지 접근 방식으로 연구되어 왔습니다.

* **의미 분할 (Semantic Segmentation)**: 각 픽셀을 하나의 도구 클래스로 분류합니다. TernausNet (U-Net + VGG), U-NetPlus 등이 있으며, 종종 단일 단계 접근 방식으로 인해 도구를 여러 클래스로 과분할하는 경향이 있습니다. PAANet, MF-TAPNet과 같은 실시간 접근 방식도 있습니다.
* **인스턴스 분할 (Instance Segmentation)**:
  * **자연 이미지 사전 훈련 모델의 미세 조정**: Mask R-CNN 기반 (Kong et al., ISINet) 또는 Transformer 기반 (TraSeTR) 접근 방식이 주로 사용됩니다. ISINet은 시퀀스 데이터의 시간적 일관성을 활용하고, TraSeTR은 트래킹 정보를 통합합니다.
  * **맞춤형 모델**: AP-MTL (실시간 인스턴스 분할을 위한 인코더-디코더), Mask-then-classify (마스크 예측 후 분류를 수행하지만, 마스크 오류에 취약) 등이 있습니다.
이 논문은 경계 상자 기반 인스턴스 분할 방법의 오분류 문제에 초점을 맞춰 이를 해결하기 위한 새로운 분류 모듈을 제안합니다.

## 🛠️ Methodology

이 논문은 기존 인스턴스 분할 모델(예: Mask R-CNN)에 새로운 분류 단계를 추가한 **S3Net (Three Stage Deep Neural Network)** 프레임워크를 제안합니다.

1. **Stage 1 (Box Proposal)**: Region Proposal Network (RPN)를 사용하여 입력 이미지에서 객체 후보 영역(경계 상자)을 생성합니다.
2. **Stage 2 (Mask Prediction)**: Stage 1에서 제안된 경계 상자를 기반으로 각 도구의 마스크와 초기 클래스 레이블 ($\hat{c}$)을 예측합니다.
3. **Stage 3 (MSMA: Multi-Scale Mask Attended Classifier)**: 이 단계는 분류 정확도를 전문적으로 개선하기 위해 추가된 핵심 모듈입니다.
    * **입력**: 원본 RGB 이미지 $I_i$와 Stage 2에서 예측된 마스크 $P_{i,j,\hat{c}}$를 입력으로 받습니다.
    * **특징 추출**: ResNet 백본을 사용하여 $I_i$에서 다중 스케일 특징을 추출합니다.
    * **마스크 어텐션**: 예측된 마스크 $P_{i,j,\hat{c}}$를 각 특징 맵에 곱하여 마스크가 적용된 특징을 생성합니다. 이는 도구 영역에만 집중하고 배경 노이즈를 제거하는 데 도움을 줍니다.
    * **특징 병합 및 임베딩 학습**: 마스크가 적용된 특징들은 1x1 컨볼루션을 통해 병합되고, 각 인스턴스에 대한 임베딩 $E_{i,j}$를 학습합니다.
    * **메트릭 학습을 통한 분류**: MSMA 분류기 헤드는 다음과 같은 Arc Loss를 사용하여 훈련됩니다:
        $$ L = - \frac{1}{C} \sum_{c=1}^{C} \log \frac{e^{\cos(\theta_c+m)}}{e^{\cos(\theta_c+m)} + \sum_{j=1, j \ne c}^{C} e^{\cos\theta_j}} $$
        여기서 $C$는 클래스 수, $m$은 각도 마진, $\theta_j$는 임베딩 $E_j$와 최종 fully connected 레이어의 $j$번째 뉴런의 가중치 벡터 사이의 각도입니다. Arc Loss는 낮은 클래스 간 분산을 가진 데이터셋에 적합하며, 클래스 임베딩 간의 각도 거리를 최대화하여 클래스 분별력을 높입니다.
    * **최종 클래스 레이블 정제**: MSMA는 Stage 2의 초기 클래스 레이블 $\hat{c}$를 더 정확한 레이블 $c$로 정제합니다.

* **수정된 NMS**: 마스크 R-CNN과 같은 기존 모델에서 발생하는 다중 클래스에 걸친 중복 영역 오분류를 줄이기 위해, 클래스 간 중복되는 세그먼트를 제거하는 수정된 비최대 억제(Non-Maximal Suppression, NMS)를 적용합니다.

## 📊 Results

* **EndoVis2017 벤치마크**: 제안된 S3Net은 ChallengeIoU (Ch$_{IoU}$) 지표에서 모든 SOTA 방법을 능가하며, SOTA 대비 최소 12점 (20%) 향상된 성능을 달성했습니다. ISINet 대비 Ch$_{IoU}$에서 30%, mcIoU에서 60% 향상되었고, TraSeTR 대비 Ch$_{IoU}$에서 20%, mcIoU에서 26% 향상되었습니다.
* **EndoVis2018 벤치마크**: S3Net은 ISINet 대비 Ch$_{IoU}$에서 3.8%, mcIoU에서 5.8% 향상된 성능을 보였습니다. TraSeTR보다는 약간 낮은 결과를 보였는데, 이는 이 데이터셋에서는 분류 외에 도구의 시간적 정보가 중요한 요소임을 시사합니다.
* **일반화 능력**: S3Net은 EndoVis2017 및 EndoVis2018 데이터셋 모두에서 잘 일반화되었으며, Mask R-CNN과 Mask2former와 같은 다양한 백본에서도 우수한 성능을 보여주었습니다.
* **정성적 분석**: 희소 클래스(sparse class) 및 겹치는 도구(overlapping instruments)의 경우 더 나은 분류 성능을 보였고, 인스턴스 누락(missed instances)을 줄였습니다.

## 🧠 Insights & Discussion

* **오분류가 주요 병목 현상**: 이 연구는 기존 SOTA 모델의 성능이 마스크 또는 경계 상자 예측의 정확도 부족이 아니라, 수술 도구 클래스 분류의 부정확성 때문에 저해됨을 명확히 밝혔습니다. 간단히 클래스 레이블을 정답으로 교체했을 때 AP$_{50}$ 점수가 크게 향상되는 결과가 이를 뒷받침합니다.
* **도구 특성을 고려한 아키텍처 혁신**: 수술 도구의 독특한 형상(길고 가늘며 비스듬하게 사용)으로 인해 발생하는 배경 노이즈 문제를 마스크 어텐션을 통해 성공적으로 해결했습니다. 이는 모델이 도구 자체의 중요한 특징에 집중하게 합니다.
* **미세 분류를 위한 메트릭 학습의 중요성**: Arc Loss와 같은 메트릭 학습의 도입은 시각적으로 유사한 수술 도구 간의 미세한 차이를 분별하는 데 결정적인 역할을 했습니다. 이는 클래스 간 분산이 낮은 의료 이미지 데이터셋에서 표현 학습과 분류를 분리하는 것이 유리함을 입증합니다.
* **확장성 및 일반화**: 제안된 S3Net 프레임워크는 Mask R-CNN, Mask2former 등 다양한 백본에 유연하게 적용될 수 있으며, EndoVis2017과 EndoVis2018 두 벤치마크 데이터셋 모두에서 강력한 일반화 능력을 보여주었습니다.
* **향후 연구 방향**: 현재 연구는 주로 공간 정보 기반의 분류 정확도 향상에 초점을 맞추었으나, 향후에는 시간적 컨텍스트(temporal context)를 통합하여 마스크 품질과 인스턴스 레이블을 더욱 개선할 수 있을 것입니다.

## 📌 TL;DR

**문제**: 기존 SOTA 인스턴스 분할 모델은 수술 도구의 마스크를 정확히 예측하더라도, 시각적으로 유사한 도구들을 오분류하여 성능이 저해되는 문제가 있었습니다. 이는 취약한 분류 헤드와 경계 상자에 포함된 방해되는 배경 특징 때문입니다.
**방법**: 이 논문은 S3Net이라는 새로운 3단계 프레임워크를 제안합니다. S3Net은 기존 2단계 인스턴스 분할 모델에 다중 스케일 마스크 어텐션(MSMA)을 사용하는 전문화된 분류 모듈을 추가합니다. MSMA는 마스크 기반 어텐션으로 도구 영역에만 집중하고, 낮은 클래스 간 분산을 처리하기 위해 메트릭 학습(arc loss)으로 훈련됩니다.
**결과**: S3Net은 EndoVis2017 벤치마크에서 SOTA 성능을 최소 12점 (20%) 이상 크게 향상시켰으며, EndoVis2018에서도 우수한 일반화 능력을 보였습니다. 이는 미세한 분류가 필요한 수술 도구 분할 작업에서 전문화된 분류 단계와 메트릭 학습의 효과를 입증합니다.
