# DEEP METRIC LEARNING-BASED SEMI-SUPERVISED REGRESSION WITH ALTERNATE LEARNING

Adina Zell, Gencer Sumbul, Begüm Demir (2022)

## 🧩 Problem to Solve

본 논문은 데이터로부터 특정 파라미터를 추정하는 회귀(Regression) 문제에서 학습 데이터(Labeled samples)의 부족으로 인해 발생하는 성능 저하 문제를 해결하고자 한다. 일반적으로 환경 내 일산화탄소 농도 측정이나 당뇨 환자의 혈당 농도 추정 등과 같은 파라미터 추정 문제는 정답 값(Target value)을 얻기 위한 비용이 많이 들고 시간이 오래 걸리기 때문에, 대량의 레이블링된 데이터를 확보하는 것이 매우 어렵다.

레이블링된 데이터의 양이 적고 품질이 낮을 경우 회귀 모델은 데이터의 실제 분포를 충분히 학습하지 못해 일반화 성능이 떨어진다. 따라서 본 논문의 목표는 소량의 레이블링된 데이터와 대량의 레이블링되지 않은 데이터(Unlabeled samples)를 함께 활용하는 준지도 회귀(Semi-supervised Regression, SSR) 방법론을 제안하여, 추가적인 레이블 수집 없이도 파라미터 추정의 정확도를 높이는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 Deep Metric Learning(DML)을 준지도 회귀 프레임워크에 도입하여, 샘플 간의 상대적인 유사도를 학습하는 공간을 구축하는 것이다. 구체적인 설계 아이디어는 다음과 같다.

1. **회귀 문제의 재정의**: 개별 샘플의 타겟 값을 직접 예측하는 대신, 두 샘플 간의 타겟 값 차이(Target value difference)를 예측하도록 모델을 설계하여 데이터 부족 문제를 완화한다.
2. **2단계 학습 구조**: 소량의 레이블링된 데이터로 쌍별 유사도(Pairwise similarity)를 먼저 모델링하고, 이를 바탕으로 대량의 레이블링되지 않은 데이터를 활용해 트리플렛 기반(Triplet-based)의 메트릭 공간을 학습한다.
3. **교차 학습 전략(Alternate Learning Strategy)**: 두 학습 단계가 서로의 가이드 역할을 하도록 에포크(Epoch) 단위로 번갈아 가며 학습시키는 전략을 사용하여 모델의 수렴도와 정확도를 높인다.

## 📎 Related Works

기존의 준지도 회귀(SSR) 연구들은 주로 레이블링되지 않은 데이터를 활용하여 학습 성능을 높이려 했다. 최근에는 Deep Metric Learning(DML)을 SSR에 적용하려는 시도가 있었으며, 대표적으로 다음과 같은 접근 방식이 존재한다.

- **Metric-based SSR (MSSR)**: Siamese Neural Network(SNN)와 Contrastive Loss를 사용하여 쌍별 메트릭 공간을 학습한다. 하지만 이 방식은 단순히 쌍별 유사성만을 고려하므로 메트릭 공간을 정교하게 학습하는 데 한계가 있다.
- **LSTM 기반 SNN 방식**: 먼저 레이블링된 데이터로만 메트릭 공간을 학습한 뒤, 이를 이용해 레이블링되지 않은 데이터에 의사 레이블(Pseudo-labels)을 부여하고 학습하는 2단계 방식을 취한다. 그러나 레이블링된 데이터 자체가 매우 적을 경우, 1단계에서 메트릭 공간이 부정확하게 형성되어 전체 성능이 저하되는 문제가 있다.

제안된 DML-S2R은 이러한 한계를 극복하기 위해 레이블링되지 않은 데이터를 메트릭 공간 학습 단계에서 직접 활용하며, 타겟 값의 차이를 학습함으로써 레이블 부족으로 인한 오버피팅을 방지한다.

## 🛠️ Methodology

### 1. 전체 시스템 구조

DML-S2R은 Siamese Neural Network(SNN)를 기반으로 하며, 크게 '쌍별 유사도 모델링' 단계와 '트리플렛 기반 메트릭 학습' 단계로 구성된다.

### 2. Pairwise Similarity Modeling (1단계)

이 단계에서는 소량의 레이블링된 데이터 $S$를 사용하여 두 샘플 간의 타겟 값 차이를 예측하는 함수 $f_d$를 학습한다.

- **목표**: 두 샘플 $x^l_i, x^l_j$가 입력되었을 때, 그들의 타겟 값 차이 $y^l_i - y^l_j$를 예측한다.
$$f_d(x^l_i, x^l_j) = y^l_i - y^l_j$$
- **구조**: 가중치를 공유하는 두 개의 동일한 서브 네트워크(SNN)가 각각의 샘플에서 특징을 추출하고, 이를 결합(Concatenate)하여 완전 연결 계층(FC layer)을 통해 차이 값을 출력한다.
- **손실 함수**: Pairwise Similarity Modeling Loss ($L_{PSM}$)를 사용하며, 이는 예측된 차이 값과 실제 차이 값 사이의 평균 제곱 오차(MSE)로 정의된다.
$$L_{PSM} = \frac{1}{N(N-1)} \sum_{i=1}^{N} \sum_{j=1, i \neq j}^{N} (z_{ij} - f_d(x^l_i, x^l_j))^2$$
여기서 $z_{ij}$는 실제 타겟 값의 차이이다.

### 3. Triplet-Based Metric Learning (2단계)

1단계에서 학습된 $f_d$를 이용하여, 레이블링되지 않은 데이터 $U$를 포함한 메트릭 공간을 학습한다.

- **트리플렛 구성**: 앵커(Anchor) 샘플 $x^l_a$는 레이블링된 집합 $S$에서 선택하고, 긍정(Positive) 샘플 집합 $P_{x^l_a}$와 부정(Negative) 샘플 집합 $N_{x^l_a}$는 레이블링되지 않은 집합 $U$에서 선택한다.
- **샘플 선택 (Algorithm 1)**: $f_d$를 사용하여 $x^l_a$와 $U$ 내 샘플들 간의 예측 차이 값을 계산한다. 이 차이 값이 가장 작은 $k$개의 샘플을 긍정 집합으로, 가장 큰 $k$개의 샘플을 부정 집합으로 선정한다.
- **손실 함수**: Ranked List Loss ($L_{RLL}$)를 사용하여 긍정 샘플들은 앵커에 가깝게, 부정 샘플들은 멀게 배치한다.
$$L_{RLL} = \frac{1}{2N} \sum_{i=1}^{N} (L_P(x^l_a, P_{x^l_a}) + L_N(x^l_a, N_{x^l_a}))$$
여기서 $L_P, L_N$은 마진 손실(Margin loss)을 포함한 가중치 기반의 손실 함수이다.

### 4. Alternate Learning 및 추론 절차

- **교차 학습**: $L_{PSM}$을 최소화하는 학습을 1에포크 수행한 후, 바로 $L_{RLL}$을 최소화하는 학습을 1에포크 수행하는 방식을 반복한다. 이를 통해 두 단계의 정보가 서로를 가이드하며 상호 보완적으로 학습된다.
- **최종 추정 (Inference)**: 새로운 샘플 $x^*$의 타겟 값 $y^*$는 다음과 같이 모든 레이블링된 샘플과의 예측 차이 값을 평균 내어 계산한다.
$$y^* = \frac{1}{N} \sum_{i=1}^{N} \left( \frac{f_d(x^*, x^l_i) - f_d(x^l_i, x^*)}{2} + y^l_i \right)$$

## 📊 Results

### 1. 실험 설정

- **데이터셋**: Boston Housing (주택 가격), Superconductivity (초전도 임계 온도), Air Quality (벤젠 농도) 3종의 데이터셋을 사용하였다.
- **데이터 구성**: 레이블링된 샘플 수 $|S|$를 10, 20, 50개로 설정하여 데이터 희소성 상황을 시뮬레이션하였다.
- **비교 대상**: COREG(Co-training-style SSR), MSSR(Metric-based SSR) 및 제안 방법의 1단계만 사용한 경우와 비교하였다.
- **평가 지표**: 평균 절대 오차(MAE, Mean Absolute Error)를 사용하였다.

### 2. 정량적 결과

- **제안 방법의 효용성**: 모든 데이터셋에서 DML-S2R이 가장 낮은 MAE를 기록하며 가장 우수한 성능을 보였다. 예를 들어, Superconductivity 데이터셋에서 $|S|=50$일 때, DML-S2R은 MAE 3.3을 기록한 반면 COREG는 9.5, MSSR은 12.7을 기록하였다.
- **단계별 기여도(Ablation Study)**: 1단계(Pairwise Similarity Modeling)만 사용했을 때보다 2단계(Triplet-based Metric Learning)를 함께 적용했을 때 성능이 비약적으로 향상됨을 확인하였다. 이는 레이블링되지 않은 데이터를 통한 메트릭 공간 학습이 타겟 값 추정의 정확도를 크게 높인다는 것을 의미한다.

## 🧠 Insights & Discussion

본 논문은 회귀 문제에서 레이블 데이터의 부족이라는 고질적인 문제를 '상대적 차이 학습'과 '준지도 메트릭 학습'의 결합으로 효과적으로 해결하였다.

특히 주목할 점은 **Alternate Learning** 전략이다. 단순히 순차적으로 학습시키는 것이 아니라, 두 손실 함수를 교차하여 최적화함으로써, 1단계의 차이 예측 능력이 2단계의 샘플 선택(Positive/Negative set selection)을 돕고, 2단계의 정교해진 메트릭 공간이 다시 1단계의 특징 추출 능력을 향상시키는 선순환 구조를 구축하였다.

다만, 본 연구는 정형 데이터(Tabular data) 위주의 데이터셋에서 검증되었다는 한계가 있다. 또한, $k$값(긍정/부정 샘플 수)이나 온도 파라미터 $\tau$와 같은 하이퍼파라미터 설정이 결과에 미치는 영향에 대한 상세한 분석은 부족한 편이다. 하지만 SNN 아키텍처에 독립적인 방법론이므로, 향후 이미지나 오디오와 같은 고차원 데이터 도메인으로의 확장이 가능할 것으로 보인다.

## 📌 TL;DR

본 논문은 레이블링된 데이터가 극히 적은 환경에서 정확한 파라미터 추정을 가능하게 하는 **DML-S2R** 방법론을 제안한다. 이 방법은 **SNN을 통해 샘플 간의 타겟 값 차이를 학습**하고, 이를 이용해 **레이블링되지 않은 데이터로부터 정교한 메트릭 공간을 구축**하는 2단계 구조를 가지며, 두 단계를 **교차 학습(Alternate Learning)** 시킴으로써 성능을 극대화한다. 실험 결과, 기존의 SSR 방법론들보다 훨씬 낮은 오차를 기록하였으며, 이는 향후 이미지 기반의 회귀 문제(나이 예측, 감정 인식 등)로 확장될 가능성이 높다.
