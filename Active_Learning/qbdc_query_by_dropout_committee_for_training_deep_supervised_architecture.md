# QBDC: QUERYBYDROPOUTCOMMITTEE FOR TRAINING DEEP SUPERVISED ARCHITECTURE

Melanie Ducoffe & Frederic Precioso (2016)

## 🧩 Problem to Solve

딥러닝 모델, 특히 심층 신경망의 성능을 높이기 위해서는 네트워크의 깊이를 증가시키는 추세이며, 이에 따라 학습에 필요한 데이터베이스의 크기도 함께 커져야 한다. 최근 방대한 양의 데이터베이스가 구축되고 있으나, 학습 세트를 구축하기 위해 모든 데이터에 레이블(label)을 제공하는 작업은 비용이 매우 많이 드는 작업이다.

본 논문은 이러한 비용 문제를 해결하기 위해, 전체 데이터 중 어떤 샘플을 우선적으로 레이블링할 것인지 온라인 방식으로 선택하는 Active Learning 전략을 제안한다. 즉, 적은 양의 레이블링된 데이터만으로도 전체 데이터를 사용한 것과 유사한 성능을 내는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 Query-By-Committee (QBC) 전략을 딥러닝, 특히 Convolutional Neural Network (CNN)에 효율적으로 적용하는 것이다.

일반적인 QBC는 여러 개의 독립적인 모델(Committee)을 학습시켜 의견 불일치가 큰 샘플을 선택하지만, 딥러닝 모델은 파라미터 수가 너무 많아 여러 개의 모델을 개별적으로 학습시키는 것이 계산적으로 불가능에 가깝다. 이를 해결하기 위해 본 연구에서는 **Batchwise Dropout** 기술을 사용하여 하나의 전체 네트워크(Full Network)로부터 여러 개의 부분 네트워크(Partial CNNs)로 구성된 위원회(Committee)를 생성함으로써 계산 비용을 획기적으로 줄이는 QBDC 방식을 제안한다.

## 📎 Related Works

- **Query By Committee (QBC):** Seung et al. (1992)에 의해 제안된 전략으로, 위원회 구성원들 간의 불일치를 이용해 샘플을 선택하면 일반화 오차가 더 빠르게 감소한다는 것을 입증하였다.
- **Semi-supervised Learning:** 제한된 레이블링 데이터와 대량의 미레이블링 데이터를 함께 사용하여 성능을 높이는 방법이다. 주로 생성 모델(Generative Networks) 기반의 접근 방식이 많으며, 레이블링된 부분집합의 선택 문제에 의존하는 경향이 있다.
- **Active Learning for Deep Architectures:** 감성 분석 등 특정 분야에서 Active Deep Network (ADN)와 같이 신뢰도가 낮은 샘플을 선택하는 방식이 연구되었다.
- **Active Deep Dropout:** Gammelsaeter (2015)가 MLP에 Dropout을 적용해 위원회를 구성하는 방법을 제안하였다. 하지만 본 논문은 다음과 같은 차별점을 갖는다.
  - Gammelsaeter의 방식은 CNN에 적용하기 어렵다.
  - 한 번에 하나의 샘플만 추가하여 학습 샘플 간의 가중치 불균형이 발생한다.
  - 위원회 멤버들에 대한 보정(rectification) 과정이 없어 버전 공간(version space)을 부정확하게 근사할 가능성이 있다.

## 🛠️ Methodology

### 전체 파이프라인

시스템은 다음과 같은 반복적인 루프를 통해 학습 세트를 구축한다.

1. 현재 레이블링된 데이터 $L_j$를 사용하여 전체 네트워크 $N_F$를 학습시킨다.
2. Batchwise Dropout을 통해 $N_F$로부터 $n$개의 부분 네트워크 $C_i$로 구성된 위원회를 생성한다.
3. 각 부분 네트워크의 마지막 층(last layer)을 역전파(backpropagation)를 통해 미세 조정하여 정확도를 높인다.
4. 미레이블링 데이터 $U_j$에서 무작위로 $K$개의 샘플을 추출하여 위원회에 평가를 맡긴다.
5. 위원회 멤버들의 예측 결과가 다수결 결과와 가장 많이 다른(불일치 점수가 높은) 샘플들을 선택하여 오라클(Oracle)에게 레이블링을 요청한다.
6. 선택된 샘플들을 학습 세트에 추가하고 다시 1단계로 돌아간다.

### Batchwise Dropout 및 위원회 구성

일반적인 Dropout은 뉴런을 개별적으로 제거하지만, **Batchwise Dropout**은 미니배치의 각 샘플에 대해 동일한 베르누이 마스크를 적용하여 필터 단위로 뉴런을 제거한다. 이는 CNN 아키텍처의 공간적 일관성을 유지하면서 네트워크의 파라미터 수를 획기적으로 줄여 부분 네트워크를 생성할 수 있게 한다.

단순히 Dropout만 적용하면 일부 멤버의 정확도가 너무 낮아져 현재의 버전 공간을 제대로 대표하지 못할 수 있다. 이를 방지하기 위해 본 논문에서는 각 부분 네트워크의 마지막 층 $W_{i,M}$만을 다시 학습시키는 보정 과정을 거친다.

### 손실 함수 및 방정식

전체 네트워크 $N_F$의 학습과 위원회 멤버 $C_i$의 마지막 층 학습 모두 교차 엔트로피(Cross Entropy) 손실 함수 $H$를 사용한다.

전체 네트워크의 손실 함수:
$$L_F: L(W \in N_F) = H(y_{true}, P_F)$$

위원회 멤버 $C_i$의 마지막 층 $W_{i,M}$에 대한 손실 함수:
$$L_i: L(W_{i,M}) = H(y_{true}, P_i)$$

여기서 $P_F$와 $P_i$는 각각 전체 네트워크와 부분 네트워크의 소프트맥스(softmax) 출력값이다.

### 샘플 선택 기준

샘플의 유용성 점수(Score)는 위원회 구성원 중 다수결 예측 결과와 다르게 예측한 모델의 수로 결정된다. 점수가 높을수록 해당 샘플이 현재 모델이 학습하지 못한 영역에 속해 있을 가능성이 크다고 판단하여 우선적으로 레이블링 대상으로 선택한다.

## 📊 Results

### 실험 설정

- **데이터셋:** MNIST
- **네트워크 구조:** Conv 레이어 2개(필터 수 20, 40, 크기 $3 \times 3$), Pooling ($2 \times 2$), Fully Connected 레이어 2개(각 100 뉴런)
- **최적화:** RMSProp (learning rate 0.001, decay 0.9), 미니배치 크기 200
- **QBDC 설정:** 위원회 크기 $n=3$, 초기 학습 세트 10 미니배치, Dropout 비율 0.5

### 주요 결과

1. **효율성:** QBDC는 무작위 선택(Random Selection)보다 항상 낮은 에러율을 보였으며, 수렴 속도가 더 빨랐다.
2. **데이터 절감:** 전체 학습 데이터의 30% 미만만 사용하고도 평균 에러율 $1.10\%$를 달성하였다. 이는 무작위 선택의 $2.13\%$보다 훨씬 우수하며, 전체 데이터를 사용한 경우와 경쟁 가능한 수준이다.

| METHOD | MEAN error rate | MIN error rate |
| :--- | :---: | :---: |
| QBDC | $1.10$ | $0.99$ |
| RANDOM | $2.13$ | $1.78$ |

1. **Dropout 영향:** 전체 네트워크 학습 시 Dropout을 적용하는 것이 직관적으로 좋아 보일 수 있으나, MNIST와 같은 작은 데이터셋에서는 초기 학습 단계에서 특징(feature)을 제대로 학습하지 못해 오히려 위원회의 선택 능력을 저하시키는 결과가 나타났다.
2. **적대적 공격에 대한 강건성(Adversarial Robustness):** Fast Gradient Sign Method (FGSM)를 통해 테스트한 결과, QBDC로 학습된 네트워크가 전체 데이터를 사용한 네트워크보다 적대적 예제에 약간 더 민감한 경향을 보였다. 다만, 학습 데이터의 70%를 버린 것에 비하면 그 차이는 크지 않았다.

## 🧠 Insights & Discussion

본 연구는 자동화된 샘플 선택 기법을 통해 지도 학습에서 필요한 데이터의 양을 획기적으로 줄이면서도 성능을 유지할 수 있음을 보여주었다.

**강점 및 한계:**

- **강점:** Batchwise Dropout을 통해 딥러닝 모델에서도 계산 비용 효율적인 QBC 구현이 가능함을 입증하였다.
- **한계:** 새로운 미니배치가 추가될 때마다 전체 네트워크를 다시 학습시켜야 한다는 점이 계산적 병목이 된다. 재학습 과정을 생략할 경우 과적합(overfitting)이 발생함을 확인하였다.
- **비판적 해석:** 적대적 예제에 대한 민감도가 약간 상승한 점은, Active Learning이 데이터의 분포를 골고루 학습하기보다 결정 경계(decision boundary) 근처의 모호한 샘플들에 집중하기 때문에 발생하는 현상일 수 있다. 이는 향후 연구에서 데이터 분포의 다양성을 확보하는 전략이 추가되어야 함을 시사한다.

## 📌 TL;DR

본 논문은 CNN 학습 시 레이블링 비용을 줄이기 위해 **Batchwise Dropout을 이용한 위원회 구성 기반의 Active Learning 전략(QBDC)**을 제안한다. 전체 네트워크에서 파생된 부분 네트워크들의 예측 불일치를 이용해 가장 유용한 샘플을 선택하며, 이를 통해 **MNIST 데이터셋에서 30% 미만의 데이터만으로도 전체 데이터를 사용한 것과 유사한 성능을 달성**하였다. 이 연구는 향후 대규모 데이터셋이나 반지도 학습(Semi-supervised learning) 분야에 효율적인 데이터 선택 메커니즘을 제공할 가능성이 크다.
