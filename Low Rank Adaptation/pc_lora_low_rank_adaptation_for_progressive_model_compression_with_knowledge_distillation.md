# PC-LoRA: Low-Rank Adaptation for Progressive Model Compression with Knowledge Distillation

Injoon Hwang, HaeWon Park, Youngwan Lee, Jooyoung Yang, SunJae Maeng (2024)

## 🧩 Problem to Solve

본 논문은 사전 학습된 Transformer 모델의 거대한 크기로 인해 발생하는 배포 및 미세 조정(fine-tuning) 단계의 높은 계산 비용 문제를 해결하고자 한다. 기존의 Low-Rank Adaptation (LoRA)은 미세 조정 단계에서 학습 가능한 파라미터 수를 획기적으로 줄여 메모리 효율성을 제공하지만, 추론(inference) 단계에서는 여전히 동결된 사전 학습 가중치(pre-trained weights)가 필요하다는 한계가 있다. 즉, LoRA는 학습 효율성은 높여주지만 추론 시의 모델 크기나 연산량(FLOPs)을 직접적으로 줄여주지는 않는다.

따라서 본 연구의 목표는 "미세 조정의 최종 단계에서 사전 학습 가중치 없이 오직 LoRA 가중치만으로 충분한 표현력을 가질 수 있는가?"라는 질문에 답하는 것이며, 이를 위해 모델 압축과 미세 조정을 동시에 수행하는 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Progressive Compression (점진적 압축)**이다. 미세 조정 과정 동안 사전 학습된 가중치의 영향력을 점진적으로 감소시켜 결국 완전히 제거하고, 그 공백을 LoRA 어댑터가 메우도록 유도하는 방식이다.

사전 학습 가중치가 서서히 사라짐에 따라 발생하는 정보 손실을 보완하기 위해, LoRA 어댑터는 사전 학습 모델이 가졌던 표현력과 미세 조정을 통해 얻은 새로운 지식을 동시에 학습하게 된다. 결과적으로 학습이 끝나면 거대한 사전 학습 가중치를 완전히 제거하고 저차원(low-rank) 어댑터만으로 모델을 구성하여, 성능 저하를 최소화하면서도 극적인 모델 압축을 달성할 수 있다.

## 📎 Related Works

**1. Low-Rank Adaptation (LoRA):**
LoRA는 기존 가중치 행렬 $W_0$를 고정하고, 두 개의 작은 저차원 행렬 $A$와 $B$의 곱 $\Delta W = BA$를 추가하여 학습하는 방식이다. 하지만 앞서 언급했듯이, 추론 시에는 $W_0 + BA$를 계산해야 하므로 모델 크기가 줄어들지 않는다.

**2. Feature-based Knowledge Distillation (KD):**
지식 증류(Knowledge Distillation)는 교사(Teacher) 모델의 지식을 학생(Student) 모델에게 전달하는 기법이다. 본 논문은 특히 중간 계층의 특징 맵(feature map)을 일치시키는 Feature-based KD를 사용한다. 이는 PC-LoRA가 레이어 단위로 적용되므로, 각 레이어의 세만틱 정보를 보존하는 데 가장 적합하기 때문이다.

## 🛠️ Methodology

### 전체 시스템 구조 및 파이프라인

PC-LoRA는 기존 선형 레이어에 저차원 어댑터를 부착하고, 학습이 진행됨에 따라 기존 가중치의 출력을 감쇠(decay)시키는 구조를 가진다.

### 주요 구성 요소 및 방정식

특정 레이어 $i$에서의 출력 $F_i$는 다음과 같이 계산된다.

$$F_i = D_i + L_i$$

여기서 $D_i$는 감쇠된 사전 학습 모델의 출력이며, $L_i$는 LoRA 어댑터의 출력이다.

1. **사전 학습 모델 출력 ($D_i$):**
   $$D_i = \lambda W_i(x_i)$$
   $\lambda$는 1에서 0으로 감소하는 **Decay Factor (감쇠 계수)**이다. $W_i$는 동결된 사전 학습 가중치이다.

2. **LoRA 어댑터 출력 ($L_i$):**
   $$L_i = B_i(A_i(x_i)) + C_i$$
   $A_i, B_i$는 랭크 $r$을 가진 저차원 행렬이며, $C_i$는 기존 모델의 bias를 대체하기 위해 도입된 학습 가능한 파라미터이다.

학습이 완료되면 $\lambda = 0$이 되어 $D_i$는 사라지고, 최종 출력은 $F_i = L_i$가 된다. 즉, 저차원 어댑터가 전체 모델을 대체하게 된다.

### 학습 목표 및 손실 함수

모델은 Task 손실과 특징 기반 지식 증류 손실의 가중합으로 최적화된다.

$$L_{total} = \alpha \cdot L_{task}(y_i, \hat{y}_i) + (1-\alpha) \cdot L_{featKD}(F_S, F_T)$$

- $L_{task}$: 다운스트림 태스크(예: 분류)를 위한 교차 엔트로피 손실이다.
- $L_{featKD}$: 학생 모델($F_S$, PC-LoRA 적용 모델)과 교사 모델($F_T$, 원래의 사전 학습 모델) 간의 중간 특징 맵에 대한 평균 제곱 오차(MSE)이다.
  $$L_{featKD}(F_S, F_T) = \frac{1}{M} \sum_{m=1}^{M} MSE(F_{S_m}, F_{T_m})$$
  여기서 $M$은 PC-LoRA가 적용된 레이어의 수이다. 이 항은 어댑터가 사전 학습 모델의 표현력을 유지하도록 돕는 정규화 역할을 한다.

### Decay Factor Scheduler

$\lambda(n)$은 전체 반복 횟수 $N$ 중 설정된 지점 $q$까지 다음과 같이 변화한다.

$$\lambda(n) =
\begin{cases}
1 - \text{Selected Decay function}(n, q) & \text{if } n < q \\
0 & \text{if } n \geq q
\end{cases}$$

본 논문에서는 **Sine, 1-Cosine, Linear** 세 가지 함수를 실험하였으며, 기본값으로 **Sine** 함수를 사용하였다. $\lambda$가 1일 때는 사전 학습 가중치가 완전히 보존되며, 0이 되면 완전히 제거된다.

## 📊 Results

### 실험 설정
- **데이터셋 및 작업:** CIFAR-10 (이미지 분류), IMDb (텍스트 분류)
- **대상 모델:** ViT-B, BERT, RoBERTa, ELECTRA 등
- **비교 대상:** Full Fine-Tuning (Full-FT), LoRA Fine-Tuning (LoRA-FT)
- **지표:** 정확도(Accuracy), 파라미터 수(Param), 연산량(GFLOPs)

### 주요 정량적 결과
PC-LoRA는 약간의 정확도 하락을 감수하면서 매우 높은 압축률을 달성하였다.

- **Vision 모델 (ViT-B 기준):**
  - 파라미터 감소율: **94.36%** $\downarrow$
  - GFLOPs 감소율: **89.1%** $\downarrow$
  - 정확도 변화: LoRA-FT 대비 약 $-3.56\%p$ 하락 (ViT-B 기준 98.80% $\rightarrow$ 95.04%)

- **Language 모델 (BERT 기준):**
  - 파라미터 감소율: **93.42%** $\downarrow$
  - GFLOPs 감소율: **84.2%** $\downarrow$

### 분석 및 정성적 결과
1. **모델 크기별 성능:** ViT-B를 PC-LoRA로 압축하여 $r=32$로 만들었을 때, 모델 크기는 ViT-Tiny와 유사하지만 성능은 오히려 ViT-Tiny나 ViT-Small보다 뛰어난 결과를 보였다. 이는 사전 학습된 거대 모델의 지식을 저차원 공간으로 효율적으로 전이시켰음을 시사한다.
2. **Attention Map 시각화:** 압축된 모델의 Attention Map이 Full-FT 모델과 매우 유사한 품질을 유지하고 있음을 확인하였다. 다만, 효과적으로 기여하는 헤드(head)의 수는 줄어든 것으로 나타났다.

## 🧠 Insights & Discussion

### 강점 및 의의
본 연구는 기존 LoRA가 해결하지 못한 '추론 시 효율성' 문제를 '점진적 제거'라는 직관적인 방식으로 해결하였다. 특히 단순히 레이어를 제거하거나 처음부터 작은 모델을 학습시키는 것보다, 거대 모델의 가중치를 점진적으로 어댑터에 이식하는 방식이 훨씬 높은 성능을 유지함을 입증하였다. 또한, 특정 랭크 $r$을 조절함으로써 사용자가 원하는 수준의 모델 크기를 유연하게 결정할 수 있는 확장성을 제공한다.

### 한계 및 미해결 과제
- **정확도 하락:** 압축률은 매우 높지만, Full-FT 대비 수 퍼센트의 성능 하락이 발생한다.
- **초기화 방식:** 현재는 가우스 분포와 0으로 초기화하여 학습을 시작하지만, 이는 최적의 시작점이 아닐 수 있다. 저자들은 향후 SVD(특이값 분해)를 이용한 초기화 방법을 적용하여 성능을 더 높이겠다고 언급하였다.
- **하이퍼파라미터 의존성:** $\alpha$ 값(손실 함수 가중치)과 Decay Scheduler의 종류에 따라 성능 차이가 발생하므로 이에 대한 최적화가 필요하다.

## 📌 TL;DR

PC-LoRA는 미세 조정 과정에서 사전 학습 가중치를 점진적으로 제거하고 그 역할을 저차원 어댑터(LoRA)가 대체하도록 하여, **추론 단계에서 사전 학습 가중치 없이 어댑터만으로 작동하는 초경량 모델**을 생성하는 방법론이다. 비전 모델과 언어 모델 모두에서 90% 이상의 파라미터 압축률을 달성하면서도 기존 소형 모델(ViT-Tiny 등)보다 우수한 성능을 보여주었으며, 이는 향후 거대 모델의 효율적인 배포 및 경량화 연구에 중요한 기여를 할 것으로 보인다.
