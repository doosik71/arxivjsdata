# SARA: Singular-Value Based Adaptive Low-Rank Adaption

Jihao Guan, Shuai Chen, Zelin Wang, Yibo Zhang, Ping Gong (2024)

## 🧩 Problem to Solve

본 논문은 대규모 사전 학습 모델(Large Pre-trained Models)을 특정 다운스트림 태스크에 적응시키기 위해 널리 사용되는 Parameter-Efficient Fine-Tuning(PEFT) 기법, 특히 LoRA(Low-Rank Adaptation)의 한계점을 해결하고자 한다.

LoRA는 모델의 가중치 변화량이 저차원 행렬(Low-rank matrix)로 근사될 수 있다고 가정하며, 모든 레이어에 동일한 랭크 $r$ 값을 하이퍼파라미터로 설정하여 적용한다. 그러나 이러한 방식은 두 가지 핵심적인 문제를 야기한다. 첫째, 각 레이어마다 최적의 랭크 값이 다름에도 불구하고 이를 수동으로 검증하여 설정해야 하는 번거로움이 있다. 둘째, 트랜스포머 구조의 서로 다른 레이어들은 정보의 중요도가 다르다는 특성이 있음에도 불구하고, 일괄적인 랭크 적용은 레이어 간 정보 불일치 문제를 일으켜 모델의 효율성을 저하시킨다.

따라서 본 연구의 목표는 사전 학습된 가중치의 특성을 분석하여 각 레이어에 가장 적합한 랭크를 적응적으로(adaptively) 결정하고, 파라미터 효율성을 극대화하는 새로운 튜닝 방법을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 특이값 분해(Singular Value Decomposition, SVD)를 통해 사전 학습된 가중치 행렬의 '내재적 랭크(intrinsic rank)'를 분석하고, 이를 기반으로 레이어별 최적 랭크를 설정하는 것이다.

주요 기여 사항은 다음과 같다.
1. **SVD 기반 레이어 분석**: SVD를 통해 사전 학습된 파라미터 행렬의 구성 성분을 분석함으로써, 특정 임계값(threshold) 이상의 특이값 합을 가지는 개수 $k$가 해당 레이어의 성능 및 중요도와 상관관계가 있음을 밝혀냈다.
2. **SARA 제안**: 초기화 단계에서 SVD를 수행하여 각 레이어의 적절한 랭크 $k$를 적응적으로 계산하고, 절단된 특이값 행렬(truncated singular value matrix)을 추가하여 튜닝하는 SARA 방식을 제안하였다.
3. **Mo-SARA 제안**: 파라미터 수를 극단적으로 줄이기 위해, Mixture-of-Experts(MoE) 개념을 도입하여 병렬적인 특이값 집합만을 튜닝하고 라우터(router)로 제어하는 Mo-SARA 구조를 제안하였다.

## 📎 Related Works

### PEFT Methods
전통적인 PEFT 방법론은 크게 세 가지로 나뉜다. 모델 내부에 새로운 튜닝 가능 파라미터를 직렬로 연결하는 Adapter 방식, 입력값에 가상 토큰을 추가하는 Prefix-tuning 방식, 그리고 가중치 행렬을 저차원 행렬의 합으로 분해하여 튜닝하는 LoRA 방식이 있다. 특히 LoRA는 추론 시 추가 오버헤드가 없다는 점에서 가장 널리 사용된다.

### LoRA's Variants
LoRA의 성능을 개선하려는 여러 시도가 있었다. AdaLoRA와 DyLoRA는 학습 중에 적절한 랭크를 계산하며, DoRA는 가중치를 크기(weight)와 방향(direction) 성분으로 분해하여 개별적으로 튜닝한다. 또한 VeRA는 랜덤하게 초기화된 up-sampling 및 down-sampling 행렬을 고정하고 두 개의 대각 행렬만을 튜닝하여 파라미터를 줄인다.

SARA는 이러한 기존 연구들과 달리, 학습 시작 전 **초기화 단계에서 SVD를 통해 사전 학습된 가중치의 정보를 직접 활용**하여 레이어별 최적 랭크를 찾아낸다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 1. SARA (Singular-Value Based Adaptive Low-Rank Adaption)

SARA는 사전 학습된 가중치 $W_0$에 대해 SVD를 수행하여, 정보의 중요도를 나타내는 특이값(singular values)의 분포를 분석한다.

**랭크 $k$의 결정 과정**:
SVD를 통해 얻은 특이값 $\Lambda$의 전체 합 중 설정한 임계값(threshold, $m$) 이상의 비율을 차지하는 최소한의 특이값 개수 $k$를 다음과 같이 구한다.
$$\sum_{i=1}^{k} \sigma_i \ge m \times \sum_{i=1}^{d} \sigma_i$$
여기서 $\sigma_i$는 내림차순으로 정렬된 특이값이다.

**모델 구조 및 학습**:
기존 LoRA가 $W_0 + \lambda AB$ 형태를 사용하는 것과 달리, SARA는 SVD 결과물인 왼쪽 특이벡터 $U_k$, 특이값 대각행렬 $\Lambda_k$, 오른쪽 특이벡터 $V_k$를 활용한다. 최종 계산식은 다음과 같다.
$$h = x(W_0 + U_k \Lambda_k V_k)$$
이때 $U_k$와 $V_k$는 고정하고, **절단된 특이값 행렬(truncated singular value matrix) 부분만을 학습 가능하게 설정**한다. 특이값 행렬은 대각 행렬이므로 1차원 벡터 형태로 저장 가능하여 메모리 효율이 매우 높으며, 별도의 스케일링 인자 $\lambda$ 없이 특이값 자체가 세밀한 학습 가능 스케일링 팩터 역할을 수행한다.

### 2. Mo-SARA (Mixture-of-SARA)

Mo-SARA는 파라미터 수를 극단적으로 줄이기 위해 MoE 구조를 접목한 방식이다.

**구조적 특징**:
- $U_k$와 $V_k$는 완전히 고정한다.
- 여러 개의 병렬적인 특이값 대각행렬 $\Lambda_{k,i}$ 세트를 구성하고, 입력값에 따라 어떤 전문가(특이값 세트)를 사용할지 결정하는 라우팅 메커니즘을 도입한다.
- 수렴 속도를 높이기 위해 0으로 초기화된 대각행렬 $v$를 추가한다.

**계산식**:
$$h = xW_0 + x U_k \left( \sum_{i=1}^{m} g_i \Lambda_{k,i} \right) V_k v$$
여기서 $g = [g_1, g_2, \dots, g_m]$은 소프트맥스(softmax)를 통해 계산된 게이팅 값이다.
$$g = \text{softmax}(x U_k \cdot (W_{g1} W_{g2}))$$
라우터는 $x U_k$를 입력으로 받는 작은 MLP(두 개의 1차원 선형 레이어)로 구성되어 파라미터 증가를 최소화한다.

## 📊 Results

### 실험 설정
- **데이터셋**: 수학적 추론(SVAMP, AQuA, GSM8K 등 6종), 상식 추론(ARC, Boolq 등 8종), E2E 벤치마크 등 총 15개 데이터셋 사용.
- **모델**: LLaMA-7B/13B, GPT-J-6B, GPT-2 Medium.
- **비교 대상**: LoRA, Prefix, Adapter, Parallel Adapter, DoRA, VeRA 등.

### 주요 결과
1. **수학적 추론**: SARA는 다양한 모델에서 LoRA 대비 최대 11%의 성능 향상을 보였다. Mo-SARA는 LoRA보다 훨씬 적은 파라미터(약 1~2 order magnitude 감소)를 사용하면서도 LLaMA-13B와 GPT-J 모델에서 모든 베이스라인을 능가하는 결과를 얻었다.
2. **상식 추론**: SARA는 LoRA 대비 최대 5% 성능 향상을 보였으며, DoRA와 유사하거나 더 나은 성능을 기록했다. Mo-SARA 역시 매우 적은 파라미터로 Prefix-tuning 이상의 성능을 냈다.
3. **E2E 벤치마크**: SARA는 더 적은 파라미터로 가장 높은 성능을 보였으며, Mo-SARA는 VeRA보다 적은 파라미터로 더 높은 성능을 달성했다.
4. **레이어별 분석**: SARA는 레이어마다 적절한 랭크를 할당함으로써, LoRA에서 나타났던 레이어 간 성능 편차(variance)를 줄이고 전반적인 효율성을 높였다.

## 🧠 Insights & Discussion

**강점 및 해석**:
SARA의 성공 요인은 사전 학습된 모델의 가중치 행렬이 가진 고유한 정보 구조(SVD 결과)를 초기화 단계에서 반영했다는 점에 있다. 분석 결과, 모델의 하위 레이어는 낮은 내재적 랭크를 필요로 하고 상위 레이어는 높은 랭크를 필요로 하는 경향이 있는데, SARA는 이를 $k$ 값으로 자동 조절하여 '병목 현상'을 방지하였다. 또한 Mo-SARA의 라우팅 히트맵 분석을 통해, 태스크마다 서로 다른 특이값 전문가 세트가 활성화됨을 확인하였으며, 이는 매우 적은 파라미터로도 복잡한 태스크를 수행할 수 있음을 시사한다.

**한계 및 미해결 질문**:
- **초기화 비용**: 학습 시작 전 SVD 분해를 수행해야 하므로 약간의 시간적 비용이 발생한다. 저자들은 향후 SVD 가속화 방법을 연구할 필요가 있다고 언급한다.
- **라우팅 방식**: 현재는 모든 전문가의 가중합을 구하는 soft-routing 방식을 사용하고 있다. 특정 전문가만 선택하는 hard-routing이나 다른 MoE 메커니즘을 적용했을 때의 효과에 대해서는 추가 연구가 필요하다.

## 📌 TL;DR

본 논문은 LoRA의 고정 랭크 설정 문제를 해결하기 위해, 사전 학습된 가중치의 SVD 분석을 통해 레이어별 최적 랭크를 적응적으로 설정하는 **SARA**와, 특이값들만을 MoE 구조로 튜닝하는 **Mo-SARA**를 제안한다. 실험 결과, SARA는 LoRA보다 뛰어난 성능을 보였으며, Mo-SARA는 극소수의 파라미터만으로도 경쟁력 있는 성능을 달성하였다. 이 연구는 PEFT 분야에서 레이어 간 정보 불일치 문제를 해결하는 새로운 관점을 제시하며, 저자원 환경에서의 효율적인 모델 튜닝에 중요한 기여를 할 가능성이 크다.
