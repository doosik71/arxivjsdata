# THE EXPRESSIVE POWER OF LOW-RANK ADAPTATION

Yuchen Zeng, Kangwook Lee (2024)

## 🧩 Problem to Solve

본 논문은 거대 언어 모델(LLM)과 확산 모델(Diffusion Models)의 효율적인 미세 조정(Fine-tuning)을 위해 널리 사용되는 Low-Rank Adaptation(LoRA)의 이론적 기반을 분석한다. LoRA는 실무적으로 매우 성공적인 성과를 거두었음에도 불구하고, 어떠한 이론적 근거로 이러한 성능이 나타나는지에 대한 연구는 부족한 상태였다.

특히 본 연구는 다음과 같은 구체적인 질문에 답하고자 한다.

1. 프리트레인된 모델 $f_0$를 타겟 모델 $f$의 기능과 일치시키기 위해 필요한 LoRA 어댑터의 최소 랭크(Minimum Rank)는 얼마인가?
2. 모델의 구조(깊이, 너비)가 필요한 최소 랭크에 어떠한 영향을 미치는가?
3. 어댑터의 랭크가 이 임계값보다 낮을 때 발생하는 근사 오차(Approximation Error)는 어떻게 정량화되는가?

결과적으로 본 논문의 목표는 Fully Connected Neural Networks(FNN)와 Transformer Networks(TFN) 환경에서 LoRA의 표현력(Expressive Power)을 이론적으로 규명하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 LoRA가 타겟 모델을 정확하게 표현하기 위해 필요한 랭크의 조건을 수학적으로 증명한 것이다.

중심적인 아이디어는 **'효과적인 랭크(Effective Rank)'** 개념이다. 단일 층의 가중치를 업데이트하는 것이 아니라, 여러 층에 걸쳐 저차원 어댑터를 배치함으로써 전체 모델 관점에서는 이들의 랭크 합이 누적되어 더 높은 표현력을 가질 수 있다는 점을 이론적으로 입증하였다. 이를 통해 LoRA가 매우 적은 수의 학습 파라미터만으로도 전체 미세 조정(Full Fine-tuning)에 근접하는 표현력을 가질 수 있음을 보여주었다.

## 📎 Related Works

기존의 신경망 표현력 연구는 주로 학습되지 않은 모델의 보편적 근사 정리(Universal Approximation Theorem)나 너비/깊이가 함수 근사 능력에 미치는 영향에 집중해 왔다. 그러나 이러한 연구들은 이미 프리트레인된 가중치가 고정된(Frozen) 상태에서 특정 어댑터만을 통해 모델을 조정하는 상황을 설명하지 못한다.

최근 정규화 층(Normalization Layer)만을 튜닝하는 방식의 표현력 연구가 있었으나, 현재 가장 지배적인 방법론인 LoRA의 표현력을 다룬 연구는 본 논문이 처음이다. 본 연구는 기존의 선형 모델 분석을 넘어 ReLU 활성화 함수가 포함된 비선형 FNN과 복잡한 구조의 TFN으로 분석 범위를 확장하여 차별점을 갖는다.

## 🛠️ Methodology

### 1. 선형 모델에서의 기초 분석 (Warm-up)

본 논문은 먼저 가장 단순한 선형 모델에서 시작한다. 타겟 모델 $f(x) = Wx$와 $L$개의 층을 가진 고정 모델 $f_0(x) = (\prod_{l=1}^L W_l)x$가 있을 때, 각 층에 랭크 $R$인 어댑터 $\Delta W_l$을 추가한다.
이때 적응된 모델의 가중치 행렬은 $\prod_{l=1}^L (W_l + \Delta W_l)$이 된다. 논문은 이 곱셈 구조에서 각 $\Delta W_l$의 랭크가 $R$일 때, 전체 시스템의 유효 랭크는 $RL$이 됨을 증명하였다. 즉, $R \ge \lceil \text{rank}(W - \prod W_l) / L \rceil$이면 타겟 모델을 정확히 표현할 수 있다.

### 2. FNN (Fully Connected Neural Networks) 분석

비선형성(ReLU)이 포함된 FNN의 경우, 다음과 같은 전략을 사용한다.

- **선형화(Linearization):** 고정된 모델의 앞선 $L-1$개 층의 바이어스(Bias) 값을 충분히 크게 설정하여 모든 ReLU가 활성화 상태(Active)가 되도록 만든다. 이렇게 하면 비선형 층들이 사실상 선형 층처럼 동작하게 되어 앞서 증명한 선형 모델의 결과를 적용할 수 있다.
- **모델 파티셔닝(Model Partitioning):** 타겟 모델의 깊이가 $\bar{L}$이고 고정 모델의 깊이가 $L$일 때, 고정 모델의 층들을 $M = L/\bar{L}$개씩 묶어 타겟 모델의 한 층을 근사하도록 설계한다.
- **랭크 조건:** 타겟 모델의 각 층 $i$에 대해, 필요한 LoRA-rank $R$은 다음과 같은 조건을 만족해야 한다.
$$R \ge \lceil \max_{i \in [\bar{L}]} \text{rank}(W_i - \prod_{l \in P_i} W_l) / M \rceil$$
여기서 $P_i$는 $i$번째 타겟 층을 근사하기 위해 할당된 고정 모델의 층 집합이다.

### 3. TFN (Transformer Networks) 분석

TFN의 경우, 주로 Self-Attention 층의 가중치 행렬에 LoRA를 적용하는 상황을 분석한다.

- **분석 방법:** Softmax와 ReLU로 인한 비선형성을 해결하기 위해, 네트워크를 세그먼트로 나누어 각 세그먼트의 출력을 순차적으로 일치시키는 방식을 취한다.
- **결과:** 특정 랭크 기반 기능 격차(Functionality Gap) $G_i$에 대해 $R \ge \max \lceil G_i / 2 \rceil$를 만족하면, attention weight matrices에 LoRA를 적용함으로써 타겟 모델을 정확히 구현할 수 있음을 증명하였다.

## 📊 Results

### 1. 정량적 실험 (Synthetic Data)

- **설정:** 모델 차원 $D=16$, 다양한 깊이($L$)와 랭크($R$) 설정 하에 MSE(Mean Squared Error)를 측정하였다.
- **결과:** 이론적 임계값인 $R \ge D/M$에서 MSE가 거의 0으로 수렴함을 확인하였다. 또한, 프리트레인된 모델을 사용할 경우 랜덤 모델보다 더 낮은 랭크에서도 빠르게 0에 도달함을 보였다. 이는 $f_0$와 $f$ 사이의 거리가 가까울수록 필요한 랭크가 줄어든다는 이론적 예측과 일치한다.

### 2. 실전 데이터셋 실험 (GLUE Benchmark)

- **비교 대상:** RoBERTa-base (110M) vs RoBERTa-large (340M).
- **결과:**
  - RoBERTa-large 모델이 $R=2$일 때, RoBERTa-base 모델이 $R=6$일 때보다 8개 태스크 중 7개에서 더 높은 성능을 보였다.
  - 이는 모델의 규모(깊이와 너비)가 클수록, 상대적으로 더 낮은 LoRA 랭크만으로도 충분한 표현력을 확보할 수 있음을 실증적으로 입증한 것이다.

### 3. 최종 층 튜닝(Final Layer Tuning)과의 비교

- **결과:** 랜덤하게 초기화된 모델에서 최종 층만 튜닝하는 방식은 파라미터 수를 늘리더라도 LoRA보다 훨씬 낮은 성능을 보였다. 이는 LoRA가 모든 층에 분산되어 가중치를 수정함으로써 초기 층에서 발생하는 정보 손실을 방지하기 때문이다.

## 🧠 Insights & Discussion

본 논문은 LoRA의 성공 요인을 **'깊이에 따른 랭크의 효율적 분산'**으로 해석한다.

**강점 및 통찰:**

- **표현력의 최적성:** LoRA는 학습 가능한 파라미터 수가 매우 적음에도 불구하고, 전체 파라미터를 튜닝하는 것과 비교해 상수 배 정도의 차이만으로도 이론적으로 동일한 표현력을 가질 수 있다.
- **모델 크기의 이점:** 모델이 더 깊고 클수록 타겟 태스크에 적응하기 위한 개별 층의 랭크 요구량은 오히려 낮아진다. 이는 거대 모델일수록 LoRA가 더 효율적으로 작동하는 이유를 설명한다.

**한계 및 논의:**

- **최적화 문제:** 본 논문의 증명은 '최적의 어댑터가 존재함'을 보인 것이지, 실제 Gradient Descent 기반의 최적화 알고리즘이 항상 그 최적점에 도달함을 보장하는 것은 아니다. 실험에서도 Gradient Update 방식이 이론적 구성 방식보다 낮은 랭크에서 더 나은 성능을 보이는 경우가 발견되었다.
- **가정의 단순화:** TFN 분석 시 Skip Connection과 Layer Normalization을 제외하였으므로, 실제 모델 구조에서의 정확한 영향력은 추가 연구가 필요하다.

## 📌 TL;DR

이 논문은 LoRA의 표현력을 수학적으로 분석하여, **모델의 깊이가 깊어질수록 타겟 모델을 모사하기 위해 필요한 LoRA 랭크가 낮아진다**는 것을 증명하였다. 특히 LoRA가 단순히 효율적인 도구를 넘어, 적절한 랭크 설정 시 전체 미세 조정과 이론적으로 대등한 표현력을 가질 수 있음을 보였으며, 이는 거대 모델일수록 LoRA의 효율성이 극대화되는 이론적 근거를 제공한다.
