# Self-Supervised 3D Scene Flow Estimation and Motion Prediction using Local Rigidity Prior

Ruibo Li, Chi Zhang, Zhe Wang, Chunhua Shen, Guosheng Lin (2023)

## 🧩 Problem to Solve

본 논문은 3D 포인트 클라우드(point clouds)를 이용한 **자기지도 학습(self-supervised learning) 기반의 3D 씬 플로우(scene flow) 추정**과 **클래스 불가지론적 모션 예측(class-agnostic motion prediction)** 문제를 해결하고자 한다.

씬 플로우 추정은 동적인 장면에서 객체들의 3D 이동 경로를 생성하는 핵심적인 작업이며, 로봇 공학이나 자율 주행의 경로 계획 및 내비게이션에 필수적이다. 그러나 실제 환경에서 씬 플로우의 정답(ground truth) 데이터를 수집하는 것은 매우 어렵다. 이를 해결하기 위해 기존 연구들은 합성 데이터(synthetic data)를 이용한 지도 학습이나, 전경/배경 마스크와 같은 약한 지도(weak supervision)에 의존해 왔다.

최근의 자기지도 학습 방식들은 주로 소스 포인트 클라우드를 타겟으로 워핑(warp)한 후 두 점군 사이의 거리를 최소화하는 방식을 사용한다. 하지만 이러한 방식은 점 간의 단순한 매칭(nearest neighbor search 등)에 의존하여, 실제 물리적 세계의 객체가 가지는 **국부적 강성(local rigidity)**, 즉 객체의 각 부분이 서로 일정한 간격을 유지하며 함께 움직이는 구조적 특성을 무시한다. 결과적으로 이는 부정확한 감독 신호(supervision signal)를 생성하여 성능 저하를 야기한다. 따라서 본 논문의 목표는 국부적 강성 사전 지식을 활용하여 정확한 의사 라벨(pseudo labels)을 생성하고, 이를 통해 정답 데이터 없이도 높은 성능의 씬 플로우 모델을 학습시키는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **"현실 세계의 장면은 여러 개의 강체(rigid body)로 움직이는 부분들의 집합으로 모델링할 수 있다"**는 직관에 기반한다. 이를 위해 다음과 같은 기여를 제시한다.

1. **RigidFlow++ 제안**: 씬 플로우 학습을 위해 국부적 강성 제약 조건을 활용하여 의사 라벨을 생성하는 새로운 자기지도 학습 프레임워크를 제안한다.
2. **신뢰도 기반 조각적 의사 라벨 생성 모듈**: 소스 포인트 클라우드를 여러 개의 슈퍼복셀(supervoxel)로 분할하고, 각 영역을 강체로 가정하여 조각적 강체 정합(piecewise rigid registration)을 수행한다. 이때 전방-후방 일관성(forward-backward consistency)과 공간적 근접성(spatial proximity)을 이용해 신뢰도 가중치를 부여함으로써 이상치(outlier)의 영향을 억제한다.
3. **유효성 마스크(Validity Mask) 도입**: 생성된 의사 라벨 중 신뢰할 수 없는 부분을 걸러내기 위한 이진 유효성 마스크를 함께 생성하여, 학습 과정에서 유효한 라벨만이 모델을 가이드하도록 설계하였다.
4. **범용적 적용 및 확장**: 제안 방법이 다양한 씬 플로우 백본 네트워크(BiFlowNet, 3DFlow 등)에서 작동함을 입증하였으며, 이를 클래스 불가지론적 모션 예측 작업으로 확장하여 SOTA(state-of-the-art) 성능을 달성하였다.

## 📎 Related Works

### 기존 연구 및 한계

- **지도 학습 기반 방식**: 3DFlow, BiFlowNet 등이 강력한 성능을 보이지만, 실제 데이터의 부족과 합성 데이터와 실제 데이터 간의 도메인 간극(domain gap) 문제로 인해 실무 적용에 한계가 있다.
- **자기지도 학습 기반 방식**: SLIM, PDF-Flow 등이 포인트 매칭이나 확률 밀도 함수를 통해 정답 없이 학습을 시도하였으나, 포인트들의 구조적 움직임(structured motion)을 무시하고 단순한 거리 기반 손실 함수를 사용하여 감독 신호의 정확도가 떨어진다.
- **테스트 시간 최적화(Test-time Optimization) 방식**: NSFP, SCOOP 등은 추론 시점에 최적화를 수행하여 정확도를 높이지만, 반복적인 계산으로 인해 추론 시간이 매우 오래 걸려 실시간 응용 분야에 부적합하다.

### 차별점

본 연구는 단순한 포인트 매칭이 아니라 **영역별 강체 정합(region-wise rigid alignment)**을 통해 의사 라벨을 생성한다. 특히 기존의 RigidFlow에서 나아가 신뢰도 재가중치 메커니즘과 유효성 추론을 추가함으로써, 가려짐(occlusion)이나 노이즈가 많은 실제 환경에서도 강건한 감독 신호를 생성할 수 있다는 점에서 차별화된다.

## 🛠️ Methodology

### 1. 전체 파이프라인

전체 시스템은 다음과 같은 순서로 동작한다.

1. **오버-세그멘테이션(Over-segmentation)**: 소스 포인트 클라우드 $P$를 $K$개의 슈퍼복셀 $\{P^{(1)}, ..., P^{(K)}\}$로 분할한다.
2. **의사 라벨 생성**: 각 슈퍼복셀에 대해 신뢰도 기반의 조각적 강체 정합을 수행하여 의사 씬 플로우 라벨 $\hat{D}$와 유효성 마스크 $C$를 생성한다.
3. **모델 학습**: 생성된 $\hat{D}$와 $C$를 감독 신호로 사용하여 신경망(Neural Networks)을 학습시킨다.

### 2. 신뢰도 기반 조각적 의사 라벨 생성

각 슈퍼복셀 $P^{(k)}$에 대해 타겟 포인트 클라우드 $Q$와의 최적 강체 변환 $[R^*_k, t^*_k]$를 찾는다.

#### 가중치 기반 강체 정합

단순한 최소제곱법은 이상치에 취약하므로, 본 논문은 다음과 같은 가중치 목적 함수를 최소화한다.
$$\arg \min_{[R_k, t_k]} \sum_{i=1}^{N_k} w^{(k)}_i \| R_k p^{(k)}_i + t_k - q_{m^{(k)}_i} \|^2_2 \quad \text{s.t. } R_k \in SO(3)$$
여기서 $w^{(k)}_i$는 점 $p^{(k)}_i$와 대응점 $q_{m^{(k)}_i}$ 사이의 매칭 신뢰도 가중치이다.

#### 신뢰도 및 유효성 추론 절차 (Iterative Process)

다음 세 단계를 반복하여 수렴시킬 때까지 업데이트한다.

1. **가중치 업데이트**: 전방-후방 일관성을 측정하여 가우시안 커널로 신뢰도 점수 $\bar{w}$를 계산하고, 유효성 마스크 $C$를 곱해 최종 가중치 $w$를 결정한다.
    $$\bar{w}^{(k)}_i = \exp\left( -\frac{\| f^{(k)}_i + b^{(k)}_i \|^2_2}{2\theta^2} \right), \quad w^{(k)}_i = \bar{w}^{(k)}_i \cdot C^{(k)}_i$$
2. **강체 변환 업데이트**: 가중치 중심(weighted centroids)과 가중치 교차 공분산 행렬(weighted cross-covariance matrix) $H_k$를 계산한 후, SVD(Singular Value Decomposition)를 통해 최적의 회전 행렬 $R_k$와 이동 벡터 $t_k$를 도출한다.
3. **포인트 매칭 및 유효성 업데이트**: 현재 변환을 적용해 가장 가까운 점을 다시 찾고, 전방-후방 일관성과 공간적 근접성(거리 $\beta_1, \beta_2$ 미만)을 기준으로 유효성 마스크 $C^{(k)}_i$를 업데이트한다.

### 3. 학습 목표 및 손실 함수

최종적으로 생성된 의사 라벨 $\hat{D}$와 유효성 마스크 $C$를 이용하여 다음과 같은 손실 함수로 모델을 학습시킨다.
$$L = \frac{\sum_{i=1}^N C_i \| f_i - \hat{d}_i \|^2}{\sum_{i=1}^N C_i}$$
여기서 $f_i$는 모델이 예측한 씬 플로우이며, $\hat{d}_i$는 생성된 의사 라벨이다. 유효성 마스크 $C_i$는 신뢰할 수 없는 라벨의 영향을 제거하는 역할을 한다.

## 📊 Results

### 실험 설정

- **데이터셋**: FlyingThings3D (FT3D), KITTI 2015, nuScenes.
- **평가 지표**: End Point Error (EPE), Accuracy (AS, AR), Outlier ratio (Out).
- **비교 대상**: 지도 학습 모델(FlowNet3D, BiFlowNet 등) 및 자기지도 학습 모델(SLIM, PDF-Flow, SPFlowNet 등).

### 주요 결과

1. **씬 플로우 추정 성능**:
    - FT3D와 KITTI 데이터셋에서 기존의 모든 자기지도 학습 방법보다 뛰어난 성능을 보였다.
    - 특히, 정답 데이터 없이 학습했음에도 불구하고 일부 지도 학습 모델(FlowNet3D, HPLFlowNet 등)보다 낮은 EPE를 기록하였다.
    - 가려짐(occlusion)이 존재하는 데이터셋에서도 이전 버전인 RigidFlow 대비 EPE를 크게 낮추며 강건함을 입증하였다.
2. **추론 효율성**:
    - 테스트 시간 최적화 방식(NSFP, SCOOP)과 비교했을 때, 추론 속도가 약 15배 이상 빠르면서도 FT3D 데이터셋에서는 더 높은 성능을 보였다.
3. **모션 예측 확장**:
    - nuScenes 데이터셋에서 클래스 불가지론적 모션 예측을 수행한 결과, SOTA 자기지도 학습 방법인 PillarMotion을 모든 지표에서 약 25% 이상 압도하였다.

## 🧠 Insights & Discussion

### 강점 및 분석

- **강성 제약의 유효성**: 단순한 포인트-투-포인트 매칭보다 영역별 강체 정합을 통해 생성된 의사 라벨이 훨씬 정확한 감독 신호를 제공함을 입증하였다.
- **자기 교정 루프(Self-correcting Loop)**: 학습 초기에는 모델의 예측값이 부정확하여 의사 라벨의 질이 낮지만, 학습이 진행됨에 따라 모델의 예측값이 개선되고, 이는 다시 더 정확한 의사 라벨 생성으로 이어지는 선순환 구조를 가진다.
- **범용성**: 특정 네트워크 구조에 종속되지 않고, 3DFlow나 BiFlowNet 같은 다양한 백본에 적용 가능하다는 점이 확인되었다.

### 한계 및 논의

- **슈퍼복셀 의존성**: 본 방법은 소스 포인트 클라우드를 슈퍼복셀로 분할하는 과정에 의존한다. 슈퍼복셀의 개수가 너무 적으면 강체 가정(rigid assumption)이 깨지고, 너무 많으면 강체 제약의 이점이 사라지므로 적절한 하이퍼파라미터 설정($K=30$)이 중요하다.
- **계산 비용**: 학습 단계에서 매 에포크마다 의사 라벨을 생성해야 하므로 학습 시간이 증가한다. 다만, 이는 추론 단계의 속도에는 영향을 주지 않는다.

## 📌 TL;DR

본 논문은 3D 포인트 클라우드의 **국부적 강성(Local Rigidity)** 특성을 활용해 정답 없이도 학습 가능한 **자기지도 씬 플로우 추정 프레임워크(RigidFlow++)**를 제안한다. 장면을 여러 강체 영역(슈퍼복셀)으로 나누고, 신뢰도 기반의 강체 정합을 통해 고품질의 의사 라벨과 유효성 마스크를 생성하여 모델을 학습시킨다. 실험 결과, 기존 자기지도 학습 방법들을 압도하며 일부 지도 학습 모델보다 우수한 성능을 보였으며, 이를 모션 예측 작업으로 확장하여 nuScenes 데이터셋에서 SOTA를 달성하였다. 이 연구는 실제 환경의 물리적 특성을 자기지도 학습의 감독 신호로 변환하는 효율적인 방법을 제시하여, 라벨링 비용이 높은 3D 동적 장면 분석 연구에 크게 기여할 것으로 보인다.
