# Toward Cross-Domain Speech Recognition with End-to-End Models

Thai-Son Nguyen, Sebastian Stüker, Alex Waibel (2020)

## 🧩 Problem to Solve

본 논문은 여러 도메인의 음성 데이터를 혼합하여 학습시켰을 때, 자동 음성 인식(Automatic Speech Recognition, ASR) 시스템이 각 개별 도메인에서 얼마나 잘 일반화되는지를 분석한다.

기존의 Hybrid HMM/ANN 모델은 특정 도메인의 데이터로 학습했을 때 최적의 성능을 보이지만, 서로 다른 음향 조건(Acoustic conditions)을 가진 여러 도메인의 데이터를 함께 학습시킬 경우 오히려 특정 도메인에서의 성능이 저하되는 경향이 있다. 이는 Hybrid 모델이 다양한 도메인의 데이터를 효과적으로 활용하여 일반화된 특징을 학습하는 데 한계가 있음을 의미한다.

따라서 본 연구의 목표는 Hybrid 모델과 End-to-End(E2E) 모델이 다중 도메인(Multi-domain) 학습 환경에서 서로 다르게 동작하는지 실증적으로 분석하고, E2E 모델이 도메인 간 전이 성능(Cross-domain performance)에서 우위를 점하는지 확인하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 시퀀스 기반의 최적화 기준(Sequence-based criterion)을 사용하는 End-to-End 모델이 프레임 기반의 Hybrid 모델보다 다양한 도메인의 데이터로부터 더 잘 일반화된 표현을 학습할 수 있다는 가설을 검증하는 것이다.

구체적으로, E2E 모델은 음향 모델과 언어 모델을 하나의 신경망으로 통합하여 학습하므로, 추론 시 별도의 도메인 적응 언어 모델(Domain-adapted Language Model) 없이도 다양한 도메인에서 강건한 성능을 낼 수 있음을 보여준다. 또한, 도메인 식별자(Domain ID)를 조건으로 입력하는 Domain-conditioned 모델을 통해 도메인 정보를 명시적으로 활용하는 방법의 가능성을 제시한다.

## 📎 Related Works

기존의 다중 도메인 및 도메인 불변(Domain-invariant) 음성 인식 연구는 크게 두 가지 방향으로 진행되었다.

1. **도메인 불변 학습**: 시뮬레이션된 노이즈 데이터를 추가하여 배경 소음이나 샘플링 레이트 변화에 영향을 받지 않는 모델을 구축하는 방식이다.
2. **적응 기법(Adaptation techniques)**: 데이터가 많은 도메인에서 학습된 모델을 데이터가 적은 다른 도메인으로 전이시키는 Student-Teacher 프레임워크나 Transfer Learning 방식이 사용되었다.

그러나 이러한 기존 연구의 대부분은 Hybrid 음향 모델에 기반하고 있었다. 본 논문은 기존의 Hybrid 방식과 최신 E2E 방식의 행동 차이를 직접적으로 비교함으로써, 모델 구조의 변화가 도메인 일반화 능력에 미치는 영향을 분석한다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 1. 데이터셋 구성

연구를 위해 다음과 같은 4가지 서로 다른 도메인의 공개 코퍼스를 결합하여 Multi-domain 데이터셋을 구성하였다.

- **Switchboard**: 전화 통화 (Telephone Conversation)
- **TED-LIUM**: 강연 (Lecture Presentation)
- **Libri Speech**: 낭독 (Read Speech)
- **Hub4**: 방송 뉴스 (Broadcast News)
총 학습 데이터는 1,282시간 규모이며, 각 도메인의 음향적 특성이 명확히 구분됨을 확인하기 위해 도메인 분류기(Domain Classifier)를 통한 분석을 선행하였다.

### 2. Hybrid Acoustic Model

전통적인 $\text{HMM}/\text{ANN}$ 구조를 사용한다.

- **구조**: FFNN(7개 층, 층당 2,000 유닛)과 Bidirectional LSTM(5개 층, 층당 320 유닛) 두 가지 모델을 구축하였다.
- **특징**: 40-채널 log mel filter-bank 특징량을 사용하며, 추론 시에는 각 도메인에 최적화된 언어 모델(LM)을 결합하여 디코딩한다.
- **한계**: 추론 시 입력 도메인을 미리 알고 그에 맞는 LM을 적용해야만 높은 성능을 얻을 수 있다.

### 3. End-to-End Models

언어 모델 없이 음향 신호를 텍스트로 직접 매핑하는 두 가지 모델을 사용한다.

- **Acoustic-to-Word (A2W)**:
  - Connectionist Temporal Classification (CTC) 기준을 사용하여 단어 단위로 직접 예측한다.
  - 5개 층의 LSTM(층당 320 유닛)을 사용하며, 8,000개의 음소(Phoneme) 분류 작업을 보조 작업으로 수행하는 Multi-task learning 방식을 통해 데이터 희소성 문제를 해결하였다.
- **Sequence-to-Sequence (Seq2Seq)**:
  - Encoder-Decoder 구조와 Attention 메커니즘을 사용한다.
  - Encoder는 A2W 모델에서 사전 학습된 LSTM 가중치로 초기화하였으며, Decoder는 단일 방향 LSTM 층을 사용한다.
  - 출력 단위는 문자(Character)와 단어(Word) 두 가지 버전으로 실험하였다.

### 4. Domain-conditioned Model

E2E 모델이 도메인 특성을 학습할 수 있도록, 입력 시퀀스 앞에 도메인 식별자(Domain ID)를 추가하여 학습시킨 모델이다. 이는 모델이 현재 입력되는 음성이 어떤 도메인에 속하는지를 인지하게 함으로써 성능을 최적화하려는 시도이다.

## 📊 Results

### 1. Hybrid 모델 결과

- **도메인 일치 여부**: 특정 도메인 데이터로만 학습한 모델은 해당 도메인에서는 잘 작동하지만, 타 도메인 데이터(Out-of-domain)에 대해서는 성능이 급격히 저하된다.
- **다중 도메인 학습 효과**: 서로 유사한 도메인(예: TED-LIUM과 Hub4)을 섞으면 상호 보완 효과가 있어 성능이 향상되지만, 음향 조건이 매우 다른 도메인(예: Switchboard)을 섞을 경우 개별 도메인의 성능 향상이 거의 나타나지 않는다.
- **LM 의존성**: 도메인에 적합한 LM을 사용하지 않을 경우 WER(Word Error Rate)이 크게 증가한다.

### 2. End-to-End 모델 결과

- **다중 도메인 학습의 이점**: Hybrid 모델과 달리, E2E 모델은 다중 도메인 데이터로 학습했을 때 개별 도메인 전용 모델보다 훨씬 더 높은 성능을 보였다.
- **일반화 능력**: 특히 Hub5'00(전화)과 Libri(낭독) 테스트 셋에서 뚜렷한 성능 향상이 관찰되었으며, 이는 E2E 모델이 서로 다른 도메인의 데이터를 통해 더 강건한 공통 표현을 학습했음을 시사한다.
- **LM 불필요**: 별도의 언어 모델 없이도 도메인 전용 Hybrid 모델과 대등하거나, 적절한 LM이 없는 Hybrid 모델보다 월등한 성능을 기록하였다.

### 3. Domain-conditioned 모델 결과

- 도메인 식별 정확도(DAC)가 매우 높게 나타났으며, 이를 통해 모델이 입력 음성의 도메인을 정확히 구분하고 있음을 확인하였다.
- MSLT 코퍼스와 같은 외부 데이터셋 평가 시, 적절한 도메인 ID(예: SWB)를 부여했을 때 성능이 개선됨을 보였다.

## 🧠 Insights & Discussion

본 논문의 결과는 음성 인식 시스템에서 **프레임 기반의 최적화(Hybrid)**와 **시퀀스 기반의 최적화(E2E)**가 데이터를 처리하는 방식에 근본적인 차이가 있음을 시사한다.

Hybrid 모델은 각 프레임의 상태 확률을 예측하는 데 집중하므로, 음향 조건이 크게 다른 데이터가 섞이면 모델이 혼란을 겪고 일반화에 실패하기 쉽다. 반면, E2E 모델은 전체 입력 시퀀스와 출력 시퀀스의 관계를 학습하므로, 더 장기적인 특징(Long-term features)을 추출할 수 있으며, 이 과정에서 도메인에 구애받지 않는 불변적인 표현을 학습하는 능력이 더 뛰어나다고 해석할 수 있다.

다만, E2E 모델이 다중 도메인 데이터에서 이점을 가지더라도, 여전히 도메인 전용 LM을 사용한 최상위 Hybrid 모델의 성능에는 미치지 못했다는 점은 한계로 남는다. 이는 E2E 모델 내부의 내재적 언어 모델링 능력이 여전히 전문적인 외부 LM의 성능을 완전히 대체하기에는 부족함을 의미한다.

## 📌 TL;DR

본 연구는 Hybrid ASR 모델과 End-to-End 모델이 다중 도메인 학습 데이터에 어떻게 반응하는지 비교 분석하였다. 실험 결과, Hybrid 모델은 데이터 도메인이 다양해질수록 일반화에 어려움을 겪지만, **End-to-End 모델은 오히려 다양한 도메인의 데이터를 학습할 때 개별 도메인의 인식 성능이 향상되는 일반화 능력**을 보였다. 이는 E2E 모델이 복잡한 음향 조건 속에서도 강건한 특징을 학습할 수 있음을 증명하며, 향후 입력 도메인을 알 수 없는 실제 환경의 음성 인식 시스템 구축에 E2E 접근 방식이 훨씬 유리함을 시사한다.
