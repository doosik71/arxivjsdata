# A Comparison of Hybrid and End-to-End Models for Syllable Recognition

Sebastian P. Bayerl and Korbinian Riedhammer (2019)

## 🧩 Problem to Solve

본 논문은 독일어 음절 인식(Syllable Recognition) 작업에서 전통적인 하이브리드(Hybrid) 방식의 음성 인식 시스템과 최신 엔드투엔드(End-to-End, E2E) 모델의 성능을 비교 분석하는 것을 목표로 한다.

일반적으로 단어 단위의 음성 인식 시스템은 발음 사전(Pronunciation Dictionary)과 같은 사전 지식에 의존하지만, 이는 사전에 없는 단어인 OOV(Out-of-Vocabulary) 문제에 취약하다는 한계가 있다. 특히 독일어와 같이 복합어 생성이 빈번하여 어휘집이 거의 무한대에 가까운 언어나, 터키어, 핀란드어와 같은 교착어(Agglutinative languages)의 경우 단어보다 작은 단위인 음절 단위의 인식이 훨씬 효율적이다. 또한, 음절 인식은 키워드 검색 인덱싱, 의료 음성 처리, 그리고 말더듬증과 같은 언어 치료에서의 유창성 측정 등 다양한 하위 작업에서 중요한 역할을 한다.

따라서 본 연구의 핵심 목표는 중규모의 독일어 자발화 말뭉치(Verbmobil corpus)를 사용하여, 사전 지식을 명시적으로 활용하는 하이브리드 구조와 데이터로부터 직접 학습하는 엔드투엔드 구조 중 어느 것이 음절 인식에 더 효과적인지 정량적으로 검증하는 것이다.

## ✨ Key Contributions

본 논문의 주요 기여는 음절 기반의 자동 음성 인식(ASR)을 위해 전통적인 HMM/DNN 하이브리드 시스템과 CTC(Connectionist Temporal Classification)로 학습된 다층 bLSTM(Bidirectional Long Short-Term Memory) 네트워크를 상세히 비교 분석했다는 점이다.

연구의 중심적인 직관은 '사전 지식(Prior Knowledge)의 명시적 모델링이 여전히 유효한가'를 확인하는 것이다. 특히, 음절 기반의 강력한 언어 모델(Language Model, LM)을 결합한 구조화된 접근 방식이 데이터 기반의 엔드투엔드 방식보다 적은 데이터셋 환경에서 더 우수한 성능을 보일 수 있다는 가설을 실험적으로 증명하였다.

## 📎 Related Works

논문에서는 기존의 음성 인식 접근 방식을 두 가지 흐름으로 설명한다.

첫째, 전통적인 하이브리드 시스템은 은닉 마르코프 모델(HMM)과 심층 신경망(DNN)을 결합하여 사용한다. 이러한 시스템은 발음 사전과 언어 모델과 같은 언어적 사전 지식을 명시적으로 주입하여 동작한다.

둘째, 최근 인기를 끌고 있는 엔드투엔드 모델(예: Baidu의 Deep Speech)은 음향 모델(AM)과 언어 모델(LM)을 하나의 통합된 신경망으로 학습하여 사전 지식의 필요성을 줄이려 한다. 대규모 데이터셋을 사용한 연구들에서는 명시적인 언어 모델 없이도 높은 성능을 낼 수 있음이 보고되었다.

그러나 저자는 엔드투엔드 모델이 방대한 학습 데이터와 컴퓨팅 자원을 요구하며, 새로운 도메인이나 어휘집에 적응시키는 비용이 높다는 한계점을 지적한다. 또한, 많은 엔드투엔드 접근 방식이 키워드 검색이나 파라언어 분석에 필수적인 정확한 시간 정렬(Time alignment) 정보가 부족하다는 점을 차별점으로 제시한다.

## 🛠️ Methodology

### 전체 시스템 구조 및 데이터

두 시스템 모두 40차원의 MFCC(Mel Frequency Cepstral Coefficients) 특징량을 사용하며, CMVN(Cepstral Mean and Variance Normalization)을 적용하였다. 데이터셋으로는 독일어 자발화 대화 말뭉치인 Verbmobil(VM) 코퍼스를 사용하였다.

### 하이브리드 시스템 (Hybrid System)

Kaldi 툴킷을 기반으로 한 HMM/DNN 하이브리드 구조를 채택하였다.

- **음향 모델(AM):** TDNN(Time-Delay Neural Network) 아키텍처를 사용하며, 학습 목표 함수로는 Lattice-free MMI(Maximum Mutual Information)를 사용한다.
- **언어 모델(LM):** 음절 기반의 $N$-gram 모델을 사용하며, 실험을 위해 0-gram(사전 지식 없음), 1-gram(약한 사전 지식), 4-gram(강한 사전 지식)의 세 가지 설정을 비교한다.
- **추론 절차:** WFST(Weighted Finite State Transducer)를 통해 음향 모델과 언어 모델을 결합하여 디코딩을 수행한다.

### 엔드투엔드 시스템 (End-to-End System)

TensorFlow를 이용하여 구현된 bLSTM 기반의 구조를 사용한다.

- **아키텍처:** 양방향 LSTM(bLSTM)을 사용하여 문맥의 앞뒤 정보를 모두 활용한다. 세 가지 토폴로지를 실험하였다: 1) 단일 은닉층(256 노드), 2) 3개 은닉층(각 196 노드), 3) 3개 은닉층 + Dropout(50%).
- **학습 목표 및 손실 함수:** CTC(Connectionist Temporal Classification) 손실 함수를 사용한다. CTC는 입력 시퀀스와 출력 시퀀스의 길이가 다를 때, 가능한 모든 정렬(Alignment)을 고려하여 blank 심볼을 도입함으로써 정답 라벨과 입력 데이터 간의 정렬 문제 없이 학습을 가능하게 한다.
- **최적화:** ADAM 옵티마이저를 사용하며, 학습률을 $0.001$에서 $0.00003$까지 점진적으로 감소시키는 Learning rate decay를 적용하였다.

## 📊 Results

실험은 전체 음절 집합(Full set, 2825개)과 빈도수가 높은 상위 199개 음절로 구성된 축소 집합(Reduced set)으로 나누어 진행되었다. 성능 지표로는 음절 기준의 단어 에러율(WER, Word Error Rate)을 사용하였다.

### 하이브리드 시스템 결과

- 언어 모델의 차수가 높을수록 성능이 비약적으로 향상되었다. 0-gram 모델은 성능이 매우 낮았으나, 간단한 1-gram 모델만으로도 큰 개선을 보였다.
- **최적 결과:** 4-gram LM과 전체 음절 집합을 사용했을 때 $\text{WER} = 10.0\%$를 기록하며 전체 실험 중 가장 우수한 성능을 보였다.

### 엔드투엔드 시스템 결과

- 네트워크의 층이 깊어질수록, 그리고 Dropout을 통한 정규화가 적용될수록 성능이 향상되었다.
- **최적 결과:** 3개 은닉층과 Dropout을 적용하고 축소된 음절 집합을 사용했을 때 $\text{WER} = 27.53\%$를 기록하였다.
- 타겟 음절의 수(Full vs Reduced)는 하이브리드 시스템에 비해 성능에 주는 영향이 상대적으로 적었다.

### 종합 비교 (WER 결과 요약)

| 시스템 | 0-gram | 1-gram | 4-gram | 1 Layer | 3 Layer | 3 Layer + Dropout |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| **Reduced Set** | $44.6\%$ | $43.0\%$ | $32.9\%$ | $37.2\%$ | $33.89\%$ | $\mathbf{27.53\%}$ |
| **Full Set** | $58.7\%$ | $35.2\%$ | $\mathbf{10.0\%}$ | $39.36\%$ | $33.96\%$ | $29.76\%$ |

## 🧠 Insights & Discussion

### 주요 분석 및 강점

본 실험 결과는 중규모 데이터셋 환경에서 **명시적인 언어 모델을 결합한 하이브리드 시스템이 엔드투엔드 모델보다 압도적으로 우수함**을 보여준다. 특히 4-gram LM을 사용한 하이브리드 모델의 $\text{WER} (10.0\%)$은 E2E 모델의 최선책 $(27.53\%)$보다 훨씬 낮다. 이는 데이터가 충분하지 않은 상황에서는 언어적 제약 조건을 명시적으로 제공하는 것이 학습 효율을 극대화함을 시사한다.

### 한계 및 비판적 해석

1. **데이터 규모의 영향:** bLSTM 모델의 결과가 기대보다 낮게 나온 원인은 Verbmobil 코퍼스의 크기가 엔드투엔드 모델이 언어 모델을 스스로 학습하기에는 너무 작기 때문으로 해석된다.
2. **E2E 모델의 구성:** 본 실험의 E2E 모델은 출력단에 별도의 언어 모델 기반 Beam Search나 Prefix Beam Search를 적용하지 않았다. 만약 이러한 후처리가 추가되었다면 성능 격차가 줄어들었을 가능성이 크다.
3. **유연성 측면:** 하이브리드 시스템은 새로운 어휘집이나 언어 모델이 필요할 때 전체 모델을 재학습시키지 않고 LM만 빠르게 교체할 수 있어 적응성이 뛰어나다는 실무적 강점이 확인되었다.

## 📌 TL;DR

본 논문은 독일어 음절 인식을 위해 하이브리드 HMM/DNN 시스템과 엔드투엔드 bLSTM-CTC 모델을 비교하였다. 실험 결과, 중규모 데이터셋에서는 강력한 $N$-gram 언어 모델을 사용하는 하이브리드 방식이 엔드투엔드 방식보다 훨씬 낮은 에러율($10.0\%$ vs $27.53\%$)을 보였다. 이는 음성 인식 시스템 구축 시, 특히 데이터가 제한적인 상황에서는 사전 지식을 활용한 구조적 모델링이 여전히 매우 중요함을 입증한다. 향후 연구로 음절-단어 변환(Syllable-to-word transduction)을 통한 효율적인 E2E 시스템 구축 가능성을 제시한다.
