# Pre-training for Spoken Language Understanding with Joint Textual and Phonetic Representation Learning

Qian Chen, Wen Wang, Qinglin Zhang (2021)

## 🧩 Problem to Solve

본 연구가 해결하고자 하는 핵심 문제는 기존의 계층적(cascaded) 구조를 가진 음성 언어 이해(Spoken Language Understanding, SLU) 시스템에서 발생하는 자동 음성 인식(Automatic Speech Recognition, ASR) 오류의 전파 문제이다. 일반적인 SLU 시스템은 ASR이 음성을 텍스트로 변환하고, 이후 자연어 이해(Natural Language Understanding, NLU) 시스템이 해당 텍스트를 바탕으로 의도 분류(Intent Classification, IC)와 슬롯 채우기(Slot Filling, SF)를 수행한다. 그러나 ASR 단계에서 오인식된 단어가 발생할 경우, 이는 하위 NLU 시스템의 성능을 심각하게 저하시키는 결과를 초래한다.

연구의 목표는 ASR의 1-best 결과물(가장 확률이 높은 하나의 텍스트 출력)만을 입력으로 사용하는 상황에서, 음성적 정보(phonetic information)를 활용하여 ASR 오류에 대한 NLU의 강건성(robustness)을 향상시키는 것이다. 특히 원본 오디오 신호에 직접 접근하지 않고도 음성적 특성을 반영할 수 있는 공동 텍스트-음성 표현(joint textual-phonetic representation)을 학습하는 사전 학습(pre-training) 방법론을 제안한다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 텍스트와 음성적 특징인 음소(phoneme) 정보를 단일 스트림(single-stream) 모델 내에서 공동으로 학습함으로써, 소리가 유사하여 ASR이 혼동하기 쉬운 단어들 간의 표현 거리를 좁히는 것이다.

주요 기여 사항은 다음과 같다.
첫째, 텍스트 시퀀스와 음소 시퀀스를 동시에 입력받아 공동의 의미 표현을 학습하는 단일 스트림 사전 학습 모델을 제안하였다.
둘째, 이를 위해 조건부 마스크 언어 모델링(condMLM), 조건부 마스크 음성 모델링(condMSM), 단어-음성 정렬(Word-Speech Alignment, WSA)이라는 세 가지 사전 학습 태스크를 설계하였다.
셋째, 사전 학습뿐만 아니라 미세 조정(fine-tuning) 단계에서도 음소 임베딩을 텍스트 임베딩에 결합하여 성능을 추가적으로 향상시키는 방법을 제시하였다.

## 📎 Related Works

기존의 SLU 연구들은 ASR 오류 문제를 해결하기 위해 다양한 접근 방식을 취해왔다. 일부 연구는 ASR의 N-best 리스트를 재순위화(reranking)하거나 단어 혼동 네트워크(Word Confusion Networks, WCN)를 활용하여 다중 가설을 고려하였다. 또한, 학습 데이터에 인위적으로 ASR 오류를 시뮬레이션하여 추가하는 데이터 증강 기법도 사용되었다. 최근에는 ASR과 NLU를 하나의 모델로 통합하여 오류 전파를 방지하는 엔드-투-엔드(End-to-End, E2E) SLU 모델들이 제안되었으나, 이는 대량의 정렬된 데이터가 필요하며 기존의 계층적 구조를 대체해야 한다는 부담이 있다.

음성적 정보를 활용하려는 시도는 과거에도 있었으나, 단어 임베딩에 음소 임베딩을 단순히 추가하거나 음소 경계를 이용해 특징을 압축하는 수준에 그쳤다. 본 논문은 이러한 단편적인 결합에서 벗어나, 사전 학습 단계에서부터 텍스트와 음소 간의 상호 의존성을 학습하는 공동 표현 학습 방식을 채택함으로써 기존 연구들과 차별점을 둔다.

## 🛠️ Methodology

### 전체 파이프라인 및 아키텍처

제안된 모델은 단일 스트림 구조의 다층 양방향 트랜스포머 인코더(multi-layer bidirectional transformer encoder)를 기반으로 한다. 입력값으로는 텍스트 시퀀스 $W$와 그에 대응하는 음소 시퀀스 $P$가 함께 들어간다. 음소 시퀀스는 CMU 발음 사전을 통해 생성되며, 사전에 없는 단어는 `<UNK>` 토큰으로 처리한다. 입력층에서는 토큰 임베딩, 위치 임베딩, 세그먼트 임베딩의 합이 인코더로 전달되어 텍스트와 음소 정보가 융합된 문맥화된 표현(contextualized representations)을 학습한다.

### 사전 학습 태스크 (Pre-training Tasks)

모델은 다음 세 가지 목적 함수를 통해 학습된다.

1. **조건부 마스크 언어 모델링 (condMLM):**
   텍스트 시퀀스 $W$의 일부 토큰을 $[MASK]$로 치환한다. 모델은 마스킹되지 않은 나머지 텍스트 토큰들과 전체 음소 시퀀스 $P$를 이용하여 마스킹된 텍스트 토큰을 예측하며, 교차 엔트로피 손실 함수 $\mathcal{L}_{condMLM}$을 최소화한다.

2. **조건부 마스크 음성 모델링 (condMSM):**
   반대로 음소 시퀀스 $P$의 일부 토큰을 $[MASK]$로 치환한다. 모델은 마스킹되지 않은 음소들과 전체 텍스트 시퀀스 $W$를 이용하여 마스킹된 음소를 예측하며, 손실 함수 $\mathcal{L}_{condMSM}$을 최소화한다. 이때 단어 단위의 마스킹(whole-word masking)을 적용하여 노이즈를 줄인다.
   - **oneMod 전략:** 샘플마다 텍스트나 음소 중 하나의 모달리티만 선택하여 마스킹한다.
   - **twoMod 전략:** 두 모달리티 모두를 동시에 마스킹한다.

3. **단어-음성 정렬 (Word-Speech Alignment, WSA):**
   텍스트 $W$와 음소 $P$가 서로 일치하는지 판별하는 이진 분류 태스크이다. 정답 쌍은 긍정 샘플로, 무작위로 섞인 쌍은 부정 샘플로 구성하여 $[CLS]$ 토큰의 은닉 상태를 통해 일치 여부를 예측하며, 손실 함수 $\mathcal{L}_{WSA}$를 최소화한다.

전체 학습 목표는 각 샘플의 성격(긍정/부정)에 따라 $\mathcal{L}_{condMLM} + \mathcal{L}_{condMSM} + \mathcal{L}_{WSA}$ (또는 기본 MLM/MSM 조합)의 합을 최소화하는 것이다.

### SLU 미세 조정 및 음소 특징 결합

미세 조정 단계에서는 의도 분류(IC)와 슬롯 채우기(SF)를 공동으로 수행한다. IC는 $[CLS]$ 토큰의 출력을 사용하여 예측하고, SF는 각 토큰의 은닉 상태를 BIO 스킴에 따라 분류한다.

특히, 미세 조정 시 음소 정보를 직접 주입하기 위해 다음과 같은 확장된 입력 임베딩 방식을 제안한다.
$$E[w_i, p_{1i}, \dots, p_{li}] = E[w_i] + \beta * \sum_{j=1}^l E[p_{ji}]$$
여기서 $E[w_i]$는 표준 텍스트 임베딩(토큰+위치+세그먼트)이며, $\sum E[p_{ji}]$는 해당 단어를 구성하는 모든 음소 임베딩의 합이다. $\beta$는 검증 세트에서 최적화되는 하이퍼파라미터이다.

## 📊 Results

### 실험 설정

- **데이터셋:** Fluent Speech Commands (FSC) 및 SNIPS 벤치마크를 사용하였다. FSC는 248개의 명령어 구문으로 구성되며, SNIPS는 개인 음성 비서 데이터이다.
- **지표:** 의도 분류 정확도(ICAcc)와 의미 오류율(semER)을 측정하였다. semER은 슬롯의 삽입, 삭제, 대치 오류와 IC 오류를 종합적으로 평가하는 지표이다.
- **기준선:** 영어 uncased BERT-Base 모델을 기본 baseline으로 설정하였다.

### 정량적 결과

실험 결과, 제안된 사전 학습 방법론과 음소 임베딩 결합 방식이 BERT-Base보다 일관되게 높은 성능을 보였다.

1. **FSC 데이터셋:** ASR 1-best 결과(WER 19.2%)에서 BERT-Base의 ICAcc 87.6% 대비, 제안 모델(oneMod 사전 학습 + 음소 임베딩 결합)은 90.5%로 **2.9%p의 절대적 향상**을 기록하였다.
2. **SNIPS 데이터셋:** WER 42.3%의 매우 높은 오류 상황에서 BERT-Base(ICAcc 82.1%, semER 57.1%) 대비, 제안 모델은 **ICAcc 85.4%, semER 53.4%**를 달성하여 각각 **3.3%p, 3.7%p의 향상**을 보였다.
3. **분석:** 특히 `oneMod` 전략(condMLM 100% + condMSM 100%)이 가장 우수한 성능을 보였으며, 미세 조정 시 음소 임베딩(PE)을 추가하는 것이 성능 향상에 가산적인(additive) 효과를 준다는 점이 확인되었다.

### 모델 분석

음성적으로 혼동하기 쉬운 단어 쌍(confusion word pairs) 20개를 선정하여 입력 임베딩 간의 코사인 거리를 기반으로 MRR(Mean Reciprocal Rank)을 측정한 결과, BERT-Base(0.1012)보다 제안 모델(0.1591)의 수치가 훨씬 높았다. 이는 제안된 사전 학습이 음성적으로 유사한 단어들을 벡터 공간에서 더 가깝게 배치함으로써 ASR 오류에 대한 내성을 키웠음을 입증한다.

## 🧠 Insights & Discussion

본 논문은 ASR 오류라는 고질적인 문제를 해결하기 위해 텍스트와 음소 정보를 통합적으로 학습하는 효율적인 프레임워크를 제시하였다. 특히, 원본 오디오 없이 발음 사전 기반의 음소 시퀀스만을 활용함으로써 계산 복잡도를 낮추면서도 실질적인 성능 향상을 이끌어냈다는 점이 강점이다.

하지만 몇 가지 한계점과 논의 사항이 존재한다. 첫째, WSA(단어-음성 정렬) 태스크가 기대만큼의 성능 향상을 가져오지 않았는데, 이는 텍스트와 음소의 정렬 학습이 너무 단순하여 모델에게 충분한 도전 과제가 되지 않았을 가능성이 크다. 둘째, 본 연구는 CMU 발음 사전이라는 고정된 룰에 의존하고 있어, 사전에 없는 단어($\text{<UNK>}$) 처리에 한계가 있다. 논문에서도 언급되었듯, 향후 forced alignment를 통해 실제 음성 신호에서 추출한 음소 라벨을 사용한다면 더 정교한 표현 학습이 가능할 것으로 보인다.

비판적으로 해석하자면, 제안된 방법론이 'ASR 시스템에 독립적(agnostic)'이라는 점은 매우 큰 장점이나, 특정 ASR 시스템의 오류 패턴을 직접 학습한 Oracle 모델이나 특화된 모델(예: [26, 27] 연구)보다는 성능이 낮다. 이는 범용적인 강건성을 확보하는 것과 특정 시스템의 편향을 교정하는 것 사이의 트레이드-오프를 보여준다.

## 📌 TL;DR

이 논문은 ASR 오류로 인해 발생하는 SLU 성능 저하를 막기 위해, **텍스트와 음소(phoneme) 정보를 동시에 학습하는 단일 스트림 사전 학습 모델**을 제안하였다. $\text{condMLM}$, $\text{condMSM}$, $\text{WSA}$라는 세 가지 태스크를 통해 음성적으로 유사한 단어들의 표현을 가깝게 학습시켰으며, 미세 조정 단계에서 음소 임베딩을 가중 합산하여 추가하는 방식을 도입하였다. 그 결과 FSC와 SNIPS 데이터셋에서 ASR 오류에 대한 강건성이 크게 향상되었으며, 이는 향후 음성 인식 기반 인터페이스의 신뢰성을 높이는 데 기여할 수 있을 것으로 기대된다.
