# Online Continual Learning of End-to-End Speech Recognition Models

Muqiao Yang, Ian Lane, Shinji Watanabe (2022)

## 🧩 Problem to Solve

본 논문은 종단간(end-to-end) 자동 음성 인식(Automatic Speech Recognition, ASR) 모델의 **Online Continual Learning(OCL)** 문제를 해결하고자 한다. 일반적인 기계 학습 모델은 고정된 데이터셋으로 학습되며, 새로운 데이터가 추가될 때마다 최적의 성능을 내기 위해서는 전체 데이터를 사용하여 처음부터 다시 학습(retraining from scratch)해야 한다. 하지만 이 과정은 막대한 계산 비용과 시간을 소모하며, 환경적으로도 탄소 배출 등의 문제를 야기한다.

반면, 새로운 데이터만을 사용하여 모델을 점진적으로 업데이트할 경우, 과거에 학습한 지식을 잊어버리는 **Catastrophic Forgetting(치명적 망각)** 현상이 발생하여 전체적인 인식 정확도가 저하된다. 기존의 ASR 관련 Continual Learning 연구들은 주로 서로 다른 여러 작업(Task)이나 도메인 간의 적응에 초점을 맞추어 왔으며, 이는 명확한 작업 경계(Task Boundary) 정보가 필요하다는 한계가 있었다.

따라서 본 연구의 목표는 단일 ASR 작업에 대해 데이터가 온라인 스트림 형태로 점진적으로 제공되는 환경에서, 작업 경계 정보 없이도 계산 비용을 획기적으로 줄이면서 성능 저하 없이 모델을 지속적으로 업데이트하는 방법을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 단일 ASR 작업을 위한 온라인 지속 학습 설정을 제안하고, 이를 위해 **Online Gradient Episodic Memory (O-GEM)** 방법론과 **Selective Sampling(선택적 샘플링)** 전략을 적용한 것이다.

핵심 아이디어는 과거 데이터의 일부를 메모리에 저장하고, 새로운 데이터를 학습할 때 과거 데이터의 기울기(gradient) 방향을 고려하여 업데이트함으로써 치명적 망각을 방지하는 것이다. 특히, 메모리에서 샘플을 추출할 때 단순히 무작위로 뽑는 것이 아니라, 현재 모델의 업데이트 방향과 유사도가 낮은 샘플을 우선적으로 선택하는 선택적 샘플링을 통해 수렴 속도와 정확도를 향상시켰다.

## 📎 Related Works

Continual Learning 방법론은 크게 세 가지 범주로 나뉜다.

1. **Regularization approaches**: 새로운 작업을 학습할 때 파라미터의 중요도에 따라 추가적인 정규화 항을 도입하여 중요 파라미터의 변화를 억제한다 (예: EWC, MAS).
2. **Architectural approaches**: 작업별로 파라미터를 분리하여 간섭을 최소화하거나 네트워크 구조를 확장한다.
3. **Replay-based approaches**: 과거 데이터의 일부를 메모리에 저장했다가 새로운 학습 시 함께 재현(replay)하여 망각을 방지한다.

기존 ASR의 지속 학습 연구들은 주로 다양한 데이터셋(WSJ, Librispeech 등)을 순차적으로 학습하며 새로운 도메인에 적응하는 방식에 집중하였다. 하지만 이러한 방식들은 데이터셋이 바뀔 때 명확한 작업 경계 정보를 알고 있어야 한다는 전제가 필요했다. 본 논문은 이러한 제약 없이, 동일한 작업 내에서 데이터가 지속적으로 유입되는 온라인 시나리오를 다룬다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 전체 파이프라인

본 연구는 초기 데이터셋 $D_0$로 학습된 시드 모델 $M_0$에서 시작하여, 이후 순차적으로 제공되는 데이터셋 $\{D_i\}_{i=1}^N$을 통해 모델을 업데이트한다. 각 단계 $n$에서 모델 $M_{n-1}$은 새로운 데이터 $D_n$을 학습하며, 이때 과거의 기억을 유지하기 위해 GEM 알고리즘을 사용한다.

### Gradient Episodic Memory (GEM)

GEM은 리플레이 기반 방법으로, 과거 데이터의 샘플을 메모리에 저장한다. 새로운 데이터의 기울기를 $g$, 업데이트하고자 하는 타겟 기울기를 $\omega$라고 할 때, GEM은 다음의 제약 조건을 만족하는 $\omega$를 찾는 문제로 정의된다.

$$ \min_{\omega} \frac{1}{2} \|\omega - g\|_2^2 \quad \text{s.t.} \quad \langle \omega, \tilde{g}_i \rangle \ge 0, \forall i \in \{0, \dots, n-1\} \quad (1) $$

여기서 $\tilde{g}_i$는 과거 $i$번째 작업의 샘플에서 계산된 기울기이다. 즉, 새로운 데이터의 기울기와 최대한 유사하면서도, 과거 데이터의 손실 함수를 증가시키지 않는(내적이 0 이상인) 방향으로 업데이트를 수행하여 망각을 방지한다. 이 문제는 계산 효율성을 위해 다음과 같은 듀얼 형태(dual form)로 변환되어 해결된다.

$$ \min_{v} \frac{1}{2} v^T GG^T v + g^T G^T v \quad \text{s.t.} \quad v \ge 0 \quad (2) $$

여기서 $G = (\tilde{g}_1, \dots, \tilde{g}_{n-1})$이며, 최종 업데이트 기울기는 $\omega = G^T v + g$가 된다.

### Selective Strategy (선택적 샘플링 전략)

온라인 환경에서는 명확한 작업 경계가 없으므로 모든 과거 작업을 일일이 계산하는 것이 불가능하다. 따라서 메모리 버퍼 $M$에서 샘플을 선택해야 하는데, 본 논문은 단순 무작위 샘플링 대신 코사인 유사도를 기반으로 한 선택적 샘플링을 제안한다. 각 샘플 $i$에 대해 다음과 같은 확률 점수 $c_i$를 부여한다.

$$ c_i = 1 - \frac{\langle \omega, \tilde{g}_i \rangle}{\|\omega\| \cdot \|\tilde{g}_i\|} $$

이 점수는 현재 업데이트 방향 $\omega$와 유사도가 낮은(즉, 예상치 못한) 샘플일수록 더 높은 확률로 선택하게 만든다. 이 경우 GEM의 엄격한 제약 조건을 완화하여 다음과 같이 재정의한다.

$$ \min_{\omega} \frac{1}{2} \|\omega - g\|_2^2 \quad \text{s.t.} \quad \langle \omega, \tilde{\omega} \rangle \ge -\xi, \xi \ge 0 \quad (3) $$

여기서 $\tilde{\omega}$는 선택된 메모리의 기울기이며, $\xi$는 슬랙 변수이다. 이 문제는 KKT 정리에 의해 다음과 같은 단순화된 듀얼 형태로 풀 수 있다.

$$ \min_{v} \frac{1}{2} v^2 \tilde{\omega}^T \tilde{\omega} + g^T \tilde{\omega} v \quad \text{s.t.} \quad v \ge 0 \quad (4) $$

최종적으로 업데이트 기울기는 $\omega = v \tilde{\omega} + g$로 계산된다.

## 📊 Results

### 실험 설정

- **데이터셋**: TED-LIUM 1, 2, 3을 사용한다. TED-LIUM 1을 $D_0$로 사용하여 50 epoch 동안 사전 학습하고, 이후 TED-LIUM 2와 3의 차분 데이터셋(2-1, 3-2)을 $N$개의 서브셋으로 나누어 순차적으로 학습한다.
- **기준선(Baselines)**:
  - **All Data**: 매번 전체 데이터를 사용하여 처음부터 다시 학습 (상한선).
  - **New Data**: 새로운 데이터만 사용하여 학습 (하한선, 망각 발생).
  - **EWC, MAS**: 정규화 기반의 지속 학습 방법.
- **평가 지표**: 단어 오류율(Word Error Rate, WER) 및 누적 계산 시간.

### 주요 결과

1. **성능 비교 (Table 2)**: O-GEM은 New Data, EWC, MAS보다 훨씬 낮은 WER을 기록하며, All Data의 성능에 매우 근접한 결과를 보였다. 특히 TED-LIUM 3-2 학습 후 O-GEM의 WER은 8.5~8.6%로, All Data(8.1%)와 경쟁 가능한 수준이었다.
2. **계산 효율성 (Fig 1b)**: O-GEM은 All Data 방식 대비 계산 시간을 4배 이상 단축시켰다. 데이터 양이 증가할수록 All Data 방식은 재학습 비용이 기하급수적으로 증가하지만, O-GEM은 새로운 데이터 양에 비례하는 비용만 발생한다.
3. **샘플링 전략의 효과**: 선택적 샘플링(Selective Sampling)이 무작위 샘플링보다 WER을 추가적으로 개선함을 확인하였다.
4. **분할 수($N$)의 영향 (Table 3)**: 데이터 분할 수 $N$이 커질수록(즉, 더 빈번하게 업데이트할수록) All Data 방식 대비 O-GEM의 계산 시간 절감 효과가 더욱 극명하게 나타났다.
5. **범용성 검증**:
    - **SSL 호환성**: HuBERT를 프론트엔드로 사용했을 때 WER 곡선이 더 빠르게 수렴하며, O-GEM의 효율성이 그대로 유지됨을 확인하였다.
    - **타 데이터셋**: WSJ0, WSJ1 데이터셋에서도 All Data 대비 약 4배 적은 시간으로 유사한 WER을 달성하여 방법론의 범용성을 입증하였다.

## 🧠 Insights & Discussion

본 논문은 ASR 모델을 위한 온라인 지속 학습의 실질적인 프레임워크를 제시하였다. 특히, 기존의 작업 기반(task-based) 지속 학습에서 벗어나, 실제 서비스 환경과 유사한 '단일 작업-연속 데이터 유입' 시나리오를 설정한 점이 매우 현실적이다.

**강점**은 GEM이라는 리플레이 기반 방법론을 ASR에 성공적으로 이식하고, 특히 선택적 샘플링을 통해 계산 효율성과 성능의 균형(Trade-off)을 최적화했다는 점이다. 또한 SSL 모델과의 호환성을 증명함으로써 최신 ASR 트렌드에도 적용 가능함을 보였다.

**한계 및 논의사항**으로는, 메모리 버퍼의 크기가 성능에 미치는 영향에 대한 상세한 분석이 부족하다는 점이 꼽힐 수 있다. 리플레이 기반 방법론은 저장하는 샘플의 수에 따라 메모리 사용량이 결정되므로, 실제 배포 환경에서 메모리 제약 조건과 성능 사이의 상관관계에 대한 추가 연구가 필요할 것으로 보인다. 또한, 본 실험에서는 TED-LIUM과 WSJ라는 비교적 정제된 데이터셋을 사용했으나, 실제 야생(in-the-wild) 데이터의 극심한 분포 변화(Non-stationary distribution) 상황에서도 동일한 강건함을 유지할지는 미지수이다.

## 📌 TL;DR

본 연구는 단일 ASR 작업에서 새로운 데이터가 지속적으로 유입될 때, 전체 재학습 없이 모델을 업데이트하는 **Online Continual Learning** 방법을 제안하였다. **O-GEM(Online Gradient Episodic Memory)**과 **선택적 샘플링 전략**을 통해 치명적 망각을 방지하였으며, 결과적으로 전체 재학습 대비 **계산 비용을 4배 이상 절감하면서도 거의 동일한 인식 성능(WER)**을 달성하였다. 이는 향후 실시간으로 진화하는 음성 인터페이스 및 라이프롱 러닝(Lifelong Learning) ASR 시스템 구축에 중요한 기초 연구가 될 것으로 평가된다.
