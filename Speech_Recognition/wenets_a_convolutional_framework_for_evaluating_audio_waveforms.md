# WENETS: A CONVOLUTIONAL FRAMEWORK FOR EVALUATING AUDIO WAVEFORMS

Andrew A. Catellier and Stephen D. Voran (2019)

## 🧩 Problem to Solve

오디오 신호의 품질(Quality)과 명료도(Intelligibility)를 측정하는 것은 통신 서비스의 품질 유지와 사용자 만족도 향상을 위해 매우 중요하다. 기존의 인지 기반 측정 방식은 크게 Full-Reference(FR) 방식과 No-Reference(NR) 방식으로 나뉜다. PESQ, POLQA, STOI와 같은 FR 방식은 송신 신호와 수신 신호를 모두 비교하여 높은 정확도를 보이지만, 실제 환경에서 송신 신호를 항상 확보하기 어렵다는 치명적인 한계가 있다.

반면, 수신 신호만을 사용하는 NR(또는 Non-intrusive) 방식은 실시간 모니터링과 결함 탐지에 유리하다. 그러나 기존의 NR 접근 방식은 대부분 magnitude spectrogram, Mel-spectral/cepstral features 등 사람이 설계한(hand-crafted) 특징량(feature)을 추출하여 머신러닝 모델에 입력하는 방식을 취한다. 이러한 방식은 특징 추출 과정에서 정보의 손실이 발생하며, 설계자의 가설에 따라 솔루션 공간이 제한된다는 문제가 있다. 따라서 본 논문은 사전 정의된 특징 추출 과정 없이 오디오 파형(waveform)에서 직접 유용한 특징을 학습하여 품질과 명료도를 예측하는 프레임워크를 개발하는 것을 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 오디오 파형을 직접 입력으로 받는 1차원 합성곱 신경망(1D CNN) 기반의 프레임워크인 WEnets를 제안한 것이다. 중심적인 설계 아이디어는 이미지 인식 분야의 VGG 네트워크가 다양한 스케일의 객체를 인식하기 위해 적층된 Convolutional layers와 Pooling을 사용하는 것에서 착안하였다.

오디오 신호는 이미지와 달리 샘플링 레이트가 고정되어 있으므로, 네트워크의 깊이를 통해 스케일 불변성을 학습하기보다는 입력 샘플링 레이트와 추출하고자 하는 특징의 시간 규모(time-scale)에 맞춰 네트워크 구조를 설계하는 전략을 취한다. 이를 통해 모델이 데이터로부터 직접 품질 및 명료도 판단에 중요한 특징을 스스로 학습하게 함으로써, 특징 추출 단계에서의 제약을 제거하고 예측 성능을 극대화하였다.

## 📎 Related Works

기존의 음성 품질 측정 연구는 주로 PESQ, POLQA와 같은 FR 기반 지표들에 의존해 왔다. 최근에는 딥러닝을 활용한 NR 방식의 연구들이 등장하였으나, 이들은 대부분 전처리 단계에서 Mel-filter bank나 spectrogram 등을 통해 특징을 압축하여 입력하는 방식을 사용한다. 예를 들어, Quality-Net은 magnitude spectrum을 사용하며, NISQA는 spectrogram을 생성한 후 Mel filter bank로 처리하여 입력한다.

본 논문이 제안하는 WEnets는 이러한 기존 방식들과 달리, 전처리 과정을 통해 특징을 압축하는 대신 raw waveform을 직접 입력으로 사용하여 CNN이 최적의 특징을 직접 학습하도록 설계되었다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조

WEnets 프레임워크는 크게 두 부분으로 구성된다. 첫 번째는 raw waveform으로부터 특징을 추출하는 **CNN Feature Extractor**이며, 두 번째는 추출된 특징을 바탕으로 최종 품질 점수를 계산하는 **Dense Network**이다. 본 논문에서는 이 프레임워크를 이용해 협대역(Narrowband) 오디오 전용 모델인 NAWEnet을 구현하였다.

### 상세 아키텍처 (NAWEnet)

NAWEnet은 5개의 합성곱 섹션($S_1 \sim S_5$)과 3개의 Fully-connected 레이어($L_1 \sim L_3$)로 구성된다.

1. **CNN Feature Extractor ($S_1 \sim S_5$):**
    * 각 섹션은 1D Convolution $\rightarrow$ Batch Normalization $\rightarrow$ PReLU Activation $\rightarrow$ Pooling 레이어로 구성된다.
    * $S_1$ 섹션은 필터 길이 $f_l=11$을 사용하며, Average Pooling을 통해 다운샘플링을 수행한다. 이는 초기 단계에서 특정 주파수 성분을 강조하거나 감쇄시키는 것이 중요하며, 단일 샘플보다 주변 정보의 평균이 더 유의미하다는 판단에 근거한다.
    * $S_2 \sim S_5$ 섹션은 Max Pooling을 사용하여 가장 강한 특징을 선택적으로 전달하며, 층이 깊어질수록 추상화 수준을 높인다.
    * 최종 섹션 $S_5$의 출력은 512개 채널과 125개의 샘플 길이로 구성되며, 이를 평탄화(flatten)하여 총 64,000차원의 벡터를 생성한다.

2. **Dense Network ($L_1 \sim L_3$):**
    * $L_1$과 $L_2$는 512개의 뉴런을 가지며, PReLU 활성화 함수와 Dropout ($p=0.5$)이 적용되어 과적합을 방지한다.
    * $L_3$는 최종 예측값(PESQ, POLQA, STOI 중 하나)을 출력하는 단일 뉴런으로 구성된다.

### 학습 절차 및 손실 함수

* **입력 데이터:** 8,000Hz 샘플링 레이트의 3초 길이 오디오 세그먼트를 사용하며, $[-1, 1]$ 범위로 정규화한다.
* **데이터 증강:** 위상 반전(Phase Inversion)이 인지 품질에 영향을 주지 않는다는 점을 이용해 Inverse Phase Augmentation (IPA)을 적용하여 학습 데이터를 2배로 확장하였다.
* **타겟 변수:** PESQ, POLQA, STOI 값을 $[-1, 1]$ 범위로 아핀 변환(affine transformation)하여 타겟으로 사용한다.
* **최적화:** Adam optimizer ($\text{learning rate} = 10^{-4}$)와 $L_2$ regularization ($10^{-5}$)을 사용하였으며, 검증 세트의 RMSE가 5 epoch 동안 개선되지 않을 경우 학습률을 $10^{-1}$배로 감소시키는 스케줄링을 적용하였다.

## 📊 Results

### 실험 설정

* **데이터셋:** 총 133.5시간의 학습 데이터와 106.8시간의 테스트 데이터를 사용하였다. 다양한 언어(영어, 이탈리아어, 일본어 등)와 75가지 이상의 왜곡 조건(코덱, 패킷 손실 등)이 포함되었다.
* **평가 지표:** 예측값과 실제 타겟 값 사이의 피어슨 상관계수($\rho_{seg}$)와 평균 제곱근 오차(RMSE)를 사용하였다.

### 정량적 결과

NAWEnet은 각각의 타겟 지표에 대해 다음과 같은 테스트 상관계수를 달성하였다.

* **PESQ:** $\rho_{seg} = 0.95$
* **POLQA:** $\rho_{seg} = 0.92$
* **STOI:** $\rho_{seg} = 0.95$

특히, POLQA 데이터의 경우 분포가 매우 불균형(상위 점수 집중)함에도 불구하고 0.92라는 높은 상관관계를 보였다.

### 비교 분석 및 일반화 성능

* **FR vs NR:** 기존 NR 지표인 ANIQUE+ ($\rho_{seg} = 0.60$)나 P.563 ($\rho_{seg} = 0.53$)에 비해 월등히 높은 성능을 보였으며, FR 지표와 MOS(주관적 품질 점수) 간의 상관관계($\rho_{seg} = 0.81$)보다도 NAWEnet의 예측 성능이 더 높게 나타났다.
* **일반화 능력:** 학습에 사용되지 않은 완전히 새로운 화자와 데이터로 구성된 데이터셋($U_1, U_2$)에 대해서는 상관계수가 다소 하락하였으나, 해당 데이터를 학습 셋에 포함시킨 $S_2$ 설정에서는 성능이 다시 회복되었다. 이는 충분한 데이터가 제공된다면 새로운 화자나 낮은 음성 활동(speech activity) 환경에서도 일반화가 가능함을 시사한다.

## 🧠 Insights & Discussion

본 연구는 오디오 분석에서 사람이 설계한 특징량을 제거하고 raw waveform을 직접 사용했을 때의 이점을 입증하였다. 특히 하이퍼파라미터 분석을 통해 다음과 같은 인사이트를 얻었다.
첫째, PReLU 활성화 함수가 Leaky ReLU보다 학습 속도와 정확도 면에서 우수하였으며, 채널별로 서로 다른 PReLU 파라미터를 학습하게 함으로써 네트워크의 유연성을 높였다.
둘째, 첫 번째 레이어($S_1$)에서 필터 길이를 11로 설정하는 것이 7보다 효과적이었으며, 이는 초기 단계에서 특정 주파수 대역을 정교하게 필터링하는 것이 특징 추출의 핵심임을 의미한다.
셋째, $S_1$에서 Max Pooling 대신 Average Pooling을 사용한 것이 성능 향상으로 이어졌는데, 이는 오디오 신호의 특성상 개별 샘플 하나하나의 값보다는 국소적 평균값이 더 중요한 정보를 담고 있기 때문으로 해석된다.

다만, 완전히 새로운 데이터셋에 대한 일반화 성능이 다소 떨어진다는 점은 여전한 한계로 남으며, 이는 더 방대한 양의 다양한 데이터셋 확보가 필수적임을 보여준다.

## 📌 TL;DR

본 논문은 raw audio waveform을 직접 입력으로 받아 음성 품질 및 명료도를 예측하는 1D CNN 기반의 **WEnets** 프레임워크를 제안한다. 제안된 모델인 **NAWEnet**은 전처리 과정 없이도 PESQ, POLQA, STOI와 같은 전문 지표들을 매우 높은 상관관계($\rho_{seg} \ge 0.92$)로 모사할 수 있음을 보였다. 이 연구는 수동적인 특징 추출 방식의 한계를 극복하고 딥러닝이 오디오 파형에서 직접 최적의 특징을 학습할 수 있음을 증명하였으며, 향후 실시간 통신 품질 모니터링 및 고해상도 오디오 평가 시스템 구축에 중요한 기초가 될 것으로 기대된다.
