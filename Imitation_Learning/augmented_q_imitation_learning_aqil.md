# Augmented Q Imitation Learning (AQIL)

Xiao Lei Zhang, Anish Agarwal (Year Not Specified)

## 🧩 Problem to Solve

본 논문은 심층 강화학습(Deep Reinforcement Learning)의 느린 수렴 속도와 모방 학습(Imitation Learning)의 성능 한계를 동시에 해결하고자 한다. 전통적인 심층 강화학습, 특히 Deep Q-Network(DQN)는 최적 정책(Optimal Policy)에 도달하기까지 매우 많은 양의 훈련 시간이 필요하며, 이는 환경이 복잡해질수록 더욱 심화된다.

반면, 모방 학습은 전문가 시스템의 행동을 복제함으로써 초기 학습 속도를 높일 수 있으나, 학습 결과가 전문가의 성능 수준으로 제한된다는 상한선(Upper Bound)의 문제가 존재한다. 즉, 전문가보다 더 나은 성능을 내기 위해서는 환경과의 직접적인 상호작용을 통한 강화학습이 필수적이다. 따라서 본 연구의 목표는 모방 학습을 통해 초기 학습 과정을 가속화하고, 이후 강화학습으로 전환함으로써 전문가의 성능을 뛰어넘는 최적 정책에 빠르게 도달하는 Augmented Q-Imitation Learning (AQIL) 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 모방 학습과 강화학습의 상호 보완적 관계를 이용하는 것이다. 구체적으로는 전문가 시스템을 모방하는 Q-Imitation Learning을 전행 학습(Precursor) 단계로 배치하여 초기 가중치를 빠르게 최적화하고, 이후 전통적인 Deep Q-learning을 통해 환경으로부터 직접 보상을 받으며 성능을 극대화하는 파이프라인을 설계하였다.

이러한 접근 방식은 강화학습의 고질적인 문제인 느린 초기 수렴 속도를 모방 학습으로 해결하고, 모방 학습의 한계인 전문가 성능의 상한선을 강화학습으로 돌파하려는 시도이다.

## 📎 Related Works

논문에서는 모방 학습을 전문가 행동을 모델링하는 Behavioral Cloning과 전문가의 보상 함수를 추정하는 Inverse Reinforcement Learning으로 구분하여 설명한다. 또한, 본 연구에서 사용한 Q-imitation-learning의 기반이 되는 SQIL(Soft Q-Imitation Learning)을 언급한다.

기존의 SQIL은 Soft Bellman Equation을 사용하여 정규화된 Behavioral Cloning을 수행하지만, 본 논문에서 제안하는 AQIL은 Hard Argmax를 사용하는 표준 Bellman Equation을 적용한다는 점에서 차별점을 가진다. 즉, 복잡한 Soft-max 구조 대신 전통적인 Q-learning의 구조를 유지하면서 보상 함수만을 전문가의 행동 준수 여부에 따라 정의하여 구현의 단순성과 효율성을 꾀하였다.

## 🛠️ Methodology

### 전체 시스템 구조 및 파이프라인

AQIL은 두 단계의 연속적인 훈련 과정으로 구성된다. 첫 번째 단계는 PID 제어기로 구현된 전문가 시스템을 모방하는 Q-Imitation Learning 단계이며, 두 번째 단계는 환경과의 상호작용을 통해 성능을 최적화하는 Deep Reinforcement Learning 단계이다.

### 마르코프 결정 과정 (MDP) 정의

본 연구는 MDP를 $\{S, A, P, R, I\}$로 정의한다. 여기서 $S$는 상태 집합, $A$는 유한한 행동 집합, $P$는 상태 전이 확률, $R$은 보상 함수, $I$는 초기 상태 분포를 의미한다. 또한 정책 $\pi$를 통해 행동을 결정하며, 전문가 정책을 $\pi'$, 최적 정책을 $\pi''$로 정의한다.

### 보상 함수 (Reward Function)

AQIL의 핵심은 단계별로 서로 다른 가우시안(Gaussian) 기반 보상 함수를 사용하는 것이다.

1. **Q-Imitation Learning 단계**: 모델의 행동이 전문가(PID 제어기)의 행동과 일치하고, 폴(Pole)의 각도가 최적 상태($0^\circ$)에 가까울 때 높은 보상을 준다.
   $$R(\theta, a_{PID}, a_{model}) = 0.2e^{-\frac{1}{2}(\frac{\theta_{optimal}-\theta}{\sigma_1})^2} + 0.8e^{-\frac{1}{2}(\frac{a_{PID}-a_{model}}{\sigma_2})^2}$$
   (단, $\theta_{optimal}=0, \sigma_1=10, \sigma_2=0.5$)

2. **Reinforcement Learning 단계**: 전문가의 행동과 무관하게, 오직 폴의 각도가 목표치에 얼마나 근접했는가에 대해서만 보상을 제공한다.
   $$R(\theta) = e^{-\frac{1}{2}(\frac{\theta_{optimal}-\theta}{\sigma_1})^2}$$
   (단, $\theta_{optimal}=0, \sigma_1=10$)

### 학습 절차 및 손실 함수

모델은 Fully Connected Neural Network를 사용하며, 은닉층에는 ReLU 활성화 함수를, 출력층에는 Linear 활성화 함수를 적용하였다. 학습 시에는 Bellman Error를 손실 함수로 사용하여 가중치를 업데이트한다.
$$L_i(\theta_i) = (y_i - Q(s, a; \theta_i))^2$$
여기서 타겟 값 $y_i$는 다음과 같이 계산된다.
$$y_i = r + \gamma \max_{a'} Q(s', a'; \theta_{i-1})$$

## 📊 Results

### 실험 설정

- **환경**: OpenAI Gym의 `CartPole-v1` (폴이 수직을 유지하도록 카트를 움직이는 제어 문제)
- **전문가 시스템**: $P=0.6, I=0.00625, D=0.8$로 튜닝된 PID 제어기
- **비교 대상**:
  - `RL500`: 순수 강화학습만 500 에피소드 수행
  - `IL250`, `IL500`: 순수 모방 학습을 각각 250, 500 에피소드 수행
  - `IL250 + RL250`: 모방 학습 250회 후 강화학습 250회 수행 (AQIL)

### 정량적 결과

실험 결과, AQIL 방식이 동일한 총 에피소드 횟수 내에서 압도적인 성능 향상을 보였다.

| 방법론 | 평균 보상 (Average Reward) | 최고 보상 (Best Reward) |
| :--- | :---: | :---: |
| RL500 | 331.63 | 1949.39 |
| IL250 | 593.1 | 6082.91 |
| IL500 | 681.91 | 4652.33 |
| **IL250 + RL250** | **2000** | **13000** |

결과적으로 `IL250 + RL250` 조합이 평균 및 최고 보상 모두에서 다른 모든 모델을 큰 격차로 앞섰으며, 이는 모방 학습으로 구축한 초기 지식이 강화학습의 수렴을 가속화하고 더 높은 성능 지점으로 이끌었음을 입증한다.

## 🧠 Insights & Discussion

### 강점 및 분석

본 논문은 전문가 데이터가 존재하는 환경에서 강화학습의 초기 Cold-start 문제를 효과적으로 해결할 수 있음을 보여주었다. 특히 단순히 전문가를 따라 하는 것에 그치지 않고, 이를 바탕으로 환경과 상호작용하게 함으로써 전문가의 성능 상한선을 돌파하는 결과를 얻었다는 점이 고무적이다.

### 한계 및 비판적 해석

1. **환경의 단순성**: 실험이 단일 환경인 `CartPole-v1`에서만 수행되었다. CartPole은 상태 공간이 작고 단순한 환경이므로, 고차원 상태 공간을 가진 복잡한 환경에서도 동일한 가속 효과가 나타날지는 추가 검증이 필요하다.
2. **보상 함수의 설계**: 가우시안 기반의 보상 함수가 매우 정교하게 설계되어 있다. 실제 환경에서는 $\sigma$ 값과 같은 하이퍼파라미터를 설정하는 것이 매우 어려울 수 있으며, 이에 대한 민감도 분석이 부족하다.
3. **전문가의 존재 가정**: 본 연구는 PID 제어기라는 완벽한 전문가가 이미 존재한다는 가정하에 진행되었다. 현실 세계에서는 이러한 전문가 모델을 구축하는 것 자체가 또 다른 어려운 문제일 수 있다.

## 📌 TL;DR

본 논문은 전문가의 행동을 모방하는 **Q-Imitation Learning**으로 모델을 초기화한 후, **Deep Q-Learning**으로 최적화를 수행하는 **AQIL** 방법론을 제안한다. 이를 통해 강화학습의 느린 수렴 속도를 해결하고 전문가의 성능을 초과하는 결과를 얻었으며, `CartPole-v1` 실험에서 순수 RL 또는 IL 단독 모델보다 월등히 높은 보상을 달성하였다. 이 연구는 전문가 시스템이 가용한 환경에서 강화학습의 효율성을 극대화하는 실용적인 프레임워크를 제시하였다는 점에서 의의가 있다.
