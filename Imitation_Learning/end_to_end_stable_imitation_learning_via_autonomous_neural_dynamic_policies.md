# End-to-End Stable Imitation Learning via Autonomous Neural Dynamic Policies

Dionis Totsila, Konstantinos Chatzilygeroudis, Denis Hadjivelichkov, Valerio Modugno, Ioannis Hatzilygeroudis, and Dimitrios Kanoulas (2023)

## 🧩 Problem to Solve

본 논문은 로봇 학습, 특히 Imitation Learning(IL)에서 신경망(Neural Network, NN) 기반 정책이 가지는 불안정성 문제를 해결하고자 한다. 최신의 센서모터 학습 알고리즘들은 유연한 NN 기반 정책을 사용하여 복잡한 동작을 학습하지만, 이는 블랙박스(Black-box) 특성을 가지므로 예측 불가능한 상황에서 불안정한 동작을 유발하여 로봇이나 환경에 손상을 줄 위험이 있다.

반면, 전통적인 Dynamical System(DS) 기반 정책은 제어 이론을 통해 점근적 안정성(Asymptotic Stability)을 보장할 수 있다는 장점이 있으나, 유연성이 부족하고 주로 고유 수용성 감각(Proprioceptive sensor states) 데이터만을 입력으로 사용한다는 한계가 있다. 따라서 본 연구의 목표는 NN의 유연한 표현 능력과 DS의 안정성 보장이라는 두 가지 장점을 결합하여, 임의의 입력 공간(예: RGB 이미지)을 수용하면서도 제어 가능한 상태에 대해 항상 안정적인 동작을 생성하는 Autonomous Neural Dynamic Policies(ANDPs)를 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 제어 가능한 상태(Controllable state)에 대해 점근적 안정성을 보장하는 선형 동역학 시스템(Linear DS)들의 가중치 합으로 정책을 구성하고, 이 가중치를 신경망이 결정하게 하는 것이다.

중심적인 설계 직관은 다음과 같다.

1. **상태 분리**: 시스템 상태를 직접 제어 가능한 부분($x_c$)과 관찰만 가능한 부분($x_{nc}$)으로 분리하여, 안정성 보장 범위를 제어 가능한 부분으로 한정한다.
2. **구조적 제약**: 각 기본 선형 DS의 행렬 $A_i$와 가중치 함수 $w_i(x)$에 특정 수학적 제약을 가함으로써, 전체 시스템이 항상 고정된 어트랙터(Attractor) $x^*_c$로 수렴하도록 설계한다.
3. **제약 조건의 재매개변수화**: 복잡한 제약 조건 최적화 문제를 피하기 위해, Cholesky 분해 등을 이용하여 제약 조건을 자동으로 만족하는 형태의 무제약 최적화(Unconstrained optimization) 문제로 변환하여 학습 효율성을 높인다.

## 📎 Related Works

본 논문은 다음과 같은 기존 연구들의 한계점을 지적하며 차별성을 강조한다.

- **NN 기반 정책**: 표현력은 높으나 안정성 보장이 불가능하며 예측 불가능한 궤적을 생성할 수 있다.
- **Dynamic Movement Primitives (DMPs)**: 구조적인 정책 표현을 제공하지만 시간 의존적(Time-dependent)인 특성 때문에 특정 시간 이후의 섭동(Perturbation)에 적절히 대응하지 못하는 경우가 있다.
- **Stable Estimator of Dynamical Systems (SEDS)**: 가우시안 혼합 모델(GMM)을 사용하여 자율적(Autonomous)이고 안정적인 제어를 수행하지만, GMM의 구조적 제약으로 인해 곡률이 높거나 비단조적인(Non-monotonic) 고도의 비선형 동작을 학습하는 데 어려움이 있다.
- **Neural Dynamic Policies (NDPs)**: NN과 DMP를 결합하여 유연성을 높였으나, 일정 단계마다 동역학 시스템을 변경하는 방식이므로 이론적인 안정성 보장이 결여되어 있다.

ANDPs는 이러한 기존 방식들과 달리, 임의의 입력(이미지 포함)을 처리할 수 있는 범용성을 가지면서도 이론적인 점근적 안정성을 수학적으로 보장한다는 점에서 차별화된다.

## 🛠️ Methodology

### 1. 전체 시스템 구조 및 상태 정의

시스템의 상태 $x$를 다음과 같이 두 부분으로 나눈다.
$$x = \begin{bmatrix} x_c \\ x_{nc} \end{bmatrix} \in \mathbb{R}^{d_c + d_{nc}}$$
여기서 $x_c$는 로봇의 말단 장치(End-effector) 위치나 속도와 같이 직접 제어 가능한 부분이며, $x_{nc}$는 장애물이나 대상 물체와 같이 관찰만 가능한 부분이다.

### 2. 정책 수식 (ANDP)

제어 정책 $\pi(x)$는 고정된 어트랙터 $x^*_c$를 향하는 $N$개 기본 선형 DS의 가중치 합으로 정의된다.
$$\dot{x}_c = \pi(x) = \sum_{i=1}^{N} w_i(x) A_i (x^*_c - x_c)$$
여기서 $w_i(x)$는 상태 $x$에 따라 결정되는 가중치 함수이며, $A_i$는 각 기본 시스템의 특성을 결정하는 행렬이다.

### 3. 안정성 조건 및 보장

본 논문은 다음 두 조건이 만족될 때, 제어 가능한 상태 $x_c$가 어트랙터 $x^*_c$로 점근적으로 수렴(Asymptotically stable)함을 증명한다.

1. $\forall i, \quad A_i + A_i^T \succ 0$ (행렬 $A_i$의 대칭 부분이 양의 준정부호(psd)여야 함)
2. $\forall x, \quad w_i(x) > 0$ (가중치 함수가 항상 양수여야 함)

### 4. 신경망을 이용한 구현 및 최적화

위의 수학적 제약 조건을 만족하면서 신경망으로 학습하기 위해 다음과 같은 전략을 사용한다.

- **가중치 함수 $w_i(x)$**: 신경망 $\Psi(x|\psi)$의 마지막 층에 $\text{softmax}$ 레이어를 배치하여 모든 $w_i$가 양수이고 합이 1이 되도록 강제한다.
- **행렬 $A_i$의 제약 해결**: $A_i$를 다음과 같이 분해하여 표현한다.
  $$A_i = B_i + C_i - C_i^T$$
  여기서 $C_i$는 아무런 제약이 없는 행렬이며, $B_i$는 대칭 양의 정부호(SPD) 행렬이다. $B_i$의 성질을 보장하기 위해 Cholesky 분해를 이용하여 $B_i = L_i L_i^T$ (여기서 $L_i$는 하삼각행렬)로 정의한다.
- **학습 파라미터**: 최적화 대상은 신경망의 가중치 $\psi$와 각 기본 시스템의 행렬 $L_i, C_i$가 된다.

## 📊 Results

### 1. 시뮬레이션 실험 (Multi-Task Learning)

DART 시뮬레이터를 사용하여 사인파, 직선, 곡선 운동의 세 가지 태스크를 하나의 정책으로 학습하는 실험을 수행하였다.

- **설정**: $x_{nc}$로 태스크를 구분하는 이미지를 입력으로 사용하였으며, 가중치 함수 $\Psi$를 위해 LeNet 변형 CNN을 사용하였다.
- **결과**: 단일 모델이 이미지 입력에 따라 서로 다른 세 가지 동작을 정확히 수행하였으며, 항상 어트랙터로 수렴하는 안정성을 보였다.
- **반응성 및 강건성**: 실행 도중 이미지를 변경하면 즉각적으로 다른 동작으로 전환하는 반응성(Reactiveness)을 보였으며, 외부 힘에 의한 섭동이 발생해도 다시 궤적으로 복귀하는 강건성을 입증하였다.

### 2. 실제 로봇 실험 (Physical Robot Experiment)

Franka Panda 로봇을 이용하여 컵의 액체를 그릇에 붓는 Pouring task를 수행하였다.

- **설정**: Kinesthetic guidance를 통해 3개의 시연 데이터를 수집하였으며, 상태 $x_c$에는 말단 장치의 위치와 오일러 각(Euler angles) 기반의 방향 정보가 포함되었다.
- **결과**: 로봇이 성공적으로 액체를 부었으며, 서로 다른 초기 설정에서 10회 반복 실험한 결과, 객체가 그릇 내부로 들어간 비율의 중앙값(Median)이 100%로 나타나 높은 성공률과 신뢰성을 보였다.

## 🧠 Insights & Discussion

### 강점 및 의의

본 연구는 신경망의 유연성과 동역학 시스템의 안정성을 성공적으로 결합하였다. 특히, 이미지와 같은 비정형 데이터를 입력으로 받으면서도 제어 출력의 안정성을 수학적으로 보장했다는 점이 매우 고무적이다. 또한, 정책이 기본 선형 DS들의 합으로 구성되어 있어, 기존의 제어 이론 도구들을 통해 정책의 의사결정 과정을 분석할 수 있는 설명 가능성(Explainability)을 내포하고 있다.

### 한계 및 향후 과제

가장 큰 한계는 **고정된 어트랙터($x^*_c$)**를 사용한다는 점이다. 이로 인해 다음과 같은 문제가 발생할 수 있다.

1. **장기 과업 수행의 어려움**: 목표 지점이 고정되어 있어, 복잡하고 긴 시퀀스의 동작을 학습하기 어렵다.
2. **비단조적 동작의 제약**: 현재의 제약 조건 하에서는 어트랙터에서 멀어지는 방향의 동작(Non-monotonic motions)을 생성하는 데 한계가 있다.

저자들은 향후 연구에서 안정성 속성을 유지하면서도 시간에 따라 변하는 **이동 어트랙터(Moving attractor)**를 도입하여 이러한 한계를 극복하고자 한다.

## 📌 TL;DR

본 논문은 신경망 기반 로봇 정책의 불안정성을 해결하기 위해, 안정성이 보장되는 선형 동역학 시스템(DS)을 NN으로 조합하는 **Autonomous Neural Dynamic Policies(ANDPs)**를 제안한다. Cholesky 분해와 Softmax를 통해 안정성 제약 조건을 무제약 최적화 문제로 변환하여 효율적으로 학습시켰으며, 이미지 입력을 통한 다중 태스크 수행 및 실제 로봇의 정밀한 Pouring task에서 그 효과를 입증하였다. 이 연구는 안정성이 필수적인 실제 로봇 환경에서 딥러닝 기반 제어기를 안전하게 적용할 수 있는 가능성을 제시한다.
