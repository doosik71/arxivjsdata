# Novelty Detection in MultiClass Scenarios with Incomplete Set of Class Labels

Nomi Vinokurov and Daphna Weinshall (2016)

## 🧩 Problem to Solve

본 논문은 훈련 세트에서 일부 클래스 레이블이 누락된 다중 클래스(Multi-class) 시나리오에서의 Novelty Detection 문제를 다룬다. 일반적인 Novelty Detection은 주로 데이터의 이상치(Outlier)나 변칙성(Anomaly)을 찾아내는 것에 집중하지만, 본 연구가 해결하고자 하는 문제는 데이터 포인트가 완전히 새로운(훈련 데이터에 존재하지 않는) 클래스에 속하는 경우를 식별하는 것이다.

이 문제는 실제 환경에서 레이블링 프로토콜의 특성상 모든 가능한 클래스를 훈련 단계에서 수집하기 어려운 경우 빈번하게 발생한다. 예를 들어, 조류 생태학 연구에서 관찰자가 지상에서 볼 수 있는 새들만 레이블링할 수 있고, 보이지 않는 곳에서 수집된 데이터는 레이블이 누락될 수 있는 상황이 이에 해당한다. 따라서 본 논문의 목표는 훈련 데이터에 포함되지 않은 '새로운 클래스'에 속하는 테스트 샘플을 효과적으로 탐지하는 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 두 단계의 구조로 구성된다. 첫째, 단순히 가장 높은 확신도(Confidence)를 사용하는 대신, 상위 두 개의 확신도 값의 비율을 이용한 **Raw Novelty Score**를 도입하여 클래스 간의 모호성을 측정한다. 둘째, 훈련 데이터를 인위적으로 '가정된 기지 클래스(Presumed-known)'와 '가정된 새로운 클래스(Presumed-novel)'로 분할하여 학습시킨 **이진 분류기 앙상블(Ensemble of binary classifiers)**을 통해 최종 Novelty Score를 산출한다. 이를 통해 단일 분류기가 가질 수 있는 편향을 줄이고, 기지 클래스와 새로운 클래스를 구분하는 일반적인 특성을 학습시킨다.

## 📎 Related Works

논문에서는 다음과 같은 기존 Novelty Detection 접근 방식들을 소개한다:

- **확률 기반 방법:** 정상 클래스의 밀도 추정(Density Estimation)을 통해 가능도를 측정한다.
- **거리 기반 방법:** 테스트 포인트와 훈련 세트 사이의 거리나 유사성을 측정하며, $k$-NN 기반 방법이 대표적이다.
- **재구성 기반 방법:** 정상 클래스의 모델을 구축하고 재구성 오차(Reconstruction Error)를 점수로 사용한다.
- **판별적 방법:** 정상 클래스 주변에 경계(Boundary)를 구축하는 One-Class SVM(OCSVM) 등이 있다.
- **KNFST (Kernel Null Space):** 기지 클래스의 Null Space로 투영하여 거리를 측정하는 최신 방법이다. 하지만 클래스 수가 많아지면 자유도가 낮아져 성능이 저하되는 한계가 있다.

또한, 다중 클래스 시나리오에서 사용되는 **Reject** 개념(Distance reject 및 Ambiguity reject)을 언급하며, 본 논문의 방법론이 특히 Distance reject와 유사한 맥락에서 개선된 형태임을 명시한다.

## 🛠️ Methodology

### 1. Initial Representation 및 Raw Novelty Score

먼저, CNN이나 다중 클래스 SVM과 같은 분류기를 통해 각 테스트 포인트 $x_j$에 대해 클래스별 확신도 벡터 $u_j \in \mathbb{R}^k$를 얻는다. 테스트 포인트들의 집합 $S$가 있을 때, 이들의 평균 확신도 벡터를 정렬한 것을 $u(S)$라고 정의한다.

**Raw Novelty Score ($\theta_S$)**는 다음과 같이 정의된다:
$$\theta_S = \frac{u(S)_1}{u(S)_2}$$
여기서 $u(S)_1$은 가장 높은 확신도 값이고, $u(S)_2$는 두 번째로 높은 확신도 값이다. 이 비율은 예측된 클래스가 두 번째 후보에 비해 얼마나 확신이 있는지를 나타내며, 새로운 클래스에 속하는 샘플일수록 이 값이 작아지는 경향이 있다. 또한, 각 기지 클래스 $i$에 대해서도 동일한 방식으로 대표값 $\theta_i$를 계산하여 저장한다.

### 2. 이진 분류기 앙상블 학습 절차

단일 $\theta_S$ 값만으로는 부족하므로, 본 연구는 $L$개의 이진 분류기 $h_l$로 구성된 앙상블을 학습시킨다.

- **데이터 분할:** 훈련 클래스 집합 $C$를 $C_K^l$(가정된 기지 클래스)와 $C_N^l$(가정된 새로운 클래스)로 무작위 분할한다. 이때 $|C_N^l| \ll |C_K^l|$로 설정하여 불균형을 둔다.
- **특징 추출:** $C_K^l$만을 사용하여 다중 클래스 분류기를 학습시키고, 이를 통해 $C_K^l$과 $C_N^l$에 속하는 포인트들의 표현을 얻는다.
- **이진 분류기 학습:** 각 포인트에 대해 $\{\theta_S, \theta_{o_S}\}$ 쌍을 입력으로 하여, 해당 포인트가 $C_N^l$에 속하는지(Novel) 아니면 $C_K^l$에 속하는지(Known)를 판별하는 선형 SVM을 학습시킨다. 여기서 $\theta_{o_S}$는 예측된 클래스 $o_S$의 대표값으로, $\theta_S$ 값을 보정(Calibrate)하는 역할을 한다.

### 3. 최종 Novelty Score 산출

테스트 세트 $S$가 입력되면 다음과 같은 절차를 거친다:

1. 전체 훈련 데이터를 사용해 학습된 분류기로 $S$의 예측 클래스 $\hat{o} = o_S$를 구한다.
2. 앙상블 $H$ 내의 분류기 중, $\hat{o}$가 $C_K^l$에 포함되어 있는 분류기들만 선택한다. (예측 클래스를 '새로운 것'으로 학습한 분류기는 제외하여 오류를 줄이기 위함이다.)
3. 선택된 분류기들에 대해 $\{\theta_S, \theta_{\hat{o}}\}$를 입력하여 'Novel'로 판정하는 횟수를 합산한다.
$$\text{Novelty Score} = \sum_{l=1}^{L} \mathbb{I}(P_l(S) = 1)$$
이 최종 점수를 임계값(Threshold)과 비교하여 최종적으로 Novel 여부를 결정한다.

## 📊 Results

### 실험 설정

- **데이터셋:** Caltech-256 (250개 클래스), Cifar-100 (100개 클래스).
- **비교 대상:** One-Class SVM(OCSVM), $k$-NN, Max Confidence, KNFST, 그리고 학습 단계가 없는 Simple Threshold ($\theta_S$만 사용).
- **측정 지표:** EER (Equal Error Rate) 및 AUC (Area Under Curve).
- **검증 방식:** 10-fold cross-validation (훈련 클래스의 10%를 제외하여 Novel 클래스로 설정).

### 주요 결과

- **정량적 성과:** 표 1과 표 2에 따르면, 제안 방법이 모든 비교 대상보다 우수한 성능을 보였다. 특히 집합 $S$의 크기가 커질수록($|S|=1 \rightarrow 5$) 성능 향상 폭이 두드러졌다.
- **Caltech-256 결과:** $S=5$일 때, 제안 방법은 AUC $90\pm 1\%$를 기록하여, KNFST ($64\pm 4\%$)나 OCSVM ($44\pm 2\%$)을 압도하였다.
- **Cifar-100 결과:** $S=1$일 때 이미 타 방법 대비 우수하였으며, $|S|$가 증가함에 따라 EER이 $0.38 \rightarrow 0.11$로 크게 낮아졌다.
- **분석:** 단순히 $\theta_S$만 사용하는 'Threshold' 방법보다 앙상블 학습을 거친 제안 방법의 성능이 더 높게 나타났는데, 이는 학습 단계가 보정 역할을 수행했음을 시사한다.

## 🧠 Insights & Discussion

본 논문의 강점은 Novelty Detection을 단순한 이상치 탐지가 아니라, **'알려지지 않은 클래스의 탐지'**라는 관점에서 접근하여 학습 가능한 프레임워크를 구축했다는 점이다.

- **$\theta_S$의 유효성:** 가장 높은 확신도 값만 사용하는 기존 방식과 달리, 두 번째 값과의 비율을 사용함으로써 결정 경계 근처의 모호성을 더 잘 포착할 수 있었다.
- **앙상블의 필요성:** 그림 2에서 보여주듯, $\{\theta_S, \theta_i\}$ 공간에서 기지와 새로운 클래스가 어느 정도 분리되지만 완벽하지는 않다. 따라서 다수의 이진 분류기를 통해 투표(Voting) 방식을 채택함으로써 개별 분류기의 오류를 상쇄하고 강건함(Robustness)을 확보하였다.
- **한계 및 가정:** 본 방법론은 테스트 시 동일 클래스에 속하는 포인트들의 집합 $S$가 제공된다는 가정을 활용할 수 있다. $|S|=1$일 때도 작동하지만, $|S|$가 클수록 성능이 비약적으로 향상된다는 점은 실제 적용 시 샘플링 전략이 중요함을 의미한다.

## 📌 TL;DR

이 논문은 훈련 세트에 일부 클래스가 누락된 상황에서 새로운 클래스를 탐지하는 방법론을 제안한다. 상위 두 개의 확신도 비율($\theta_S$)을 이용한 Raw Novelty Score를 정의하고, 이를 기반으로 인위적으로 분할된 데이터를 학습한 이진 분류기 앙상블을 구축하여 최종 점수를 산출한다. 실험 결과, Caltech-256 및 Cifar-100 데이터셋에서 OCSVM, $k$-NN 등 기존 방법론보다 월등한 성능을 보였으며, 특히 다수의 샘플을 동시에 평가할 때 그 효과가 극대화된다. 이 연구는 대규모 클래스를 다루는 실제 빅데이터 환경의 Novelty Detection에 중요한 가이드를 제공한다.
