# Noisy Boundaries: Lemon or Lemonade for Semi-supervised Instance Segmentation?

Zhenyu Wang, Yali Li, Shengjin Wang (2021)

## 🧩 Problem to Solve

본 논문은 인스턴스 분할(Instance Segmentation) 분야에서 픽셀 수준의 정밀한 어노테이션(Pixel-level annotation)을 획득하는 데 드는 막대한 비용 문제를 해결하고자 한다. 통계적으로 객체 하나를 분할하는 데 평균 79초가 소요되며, 고품질의 이미지 한 장을 작업하는 데는 1.5시간 이상이 걸리기도 한다. 이러한 고비용 구조는 데이터셋의 규모를 제한하고, 결과적으로 모델의 성능 향상을 저해하는 병목 현상이 된다.

이에 저자들은 라벨이 없는 이미지(Unlabeled images)를 활용하여 성능을 높이는 **준지도 인스턴스 분할(Semi-supervised Instance Segmentation, SSIS)** 작업을 공식적으로 정의하고 해결하고자 한다. 특히, 이 과정에서 발생하는 의사 라벨(Pseudo labels)의 '노이즈 섞인 경계(Noisy boundaries)'가 모델에게 해로운 정보(Lemon)가 될 수도, 혹은 유용한 세부 정보(Lemonade)가 될 수도 있다는 이중적인 특성에 주목하며, 이를 동시에 제어하고 활용하는 것을 목표로 한다.

## ✨ Key Contributions

본 연구의 핵심 아이디어는 의사 라벨의 경계 영역에 존재하는 노이즈를 단순히 제거하는 것이 아니라, **노이즈에 저항하는 능력(Resistance)**과 **세부 경계를 보존하는 능력(Preservation)**을 통합적으로 설계하는 것이다.

1. **Noise-tolerant Mask Head (NTM)**: 마스크의 해상도와 픽셀 수준 노이즈 사이의 부정적 상관관계를 이용하여, 저해상도 특징(Low-resolution features)을 통해 노이즈에 강건한 학습을 수행하고 이를 고해상도 브랜치에 융합한다.
2. **Boundary-preserving Map (BPM)**: 경계 영역의 세부 정보를 학습하되, 노이즈가 극심한 아주 좁은 경계선 영역은 억제하는 가중치 맵을 도입하여 정밀한 경계를 학습한다.
3. **SSIS 프레임워크 구축**: Teacher-Student 구조를 기반으로, 데이터 분포를 고려한 임계값(Threshold) 설정 방식을 도입하여 보다 정확한 의사 라벨을 생성하는 파이프라인을 제안한다.

## 📎 Related Works

인스턴스 분할은 주로 Mask R-CNN과 같은 Detection-based 방식들이 주도해 왔으나, 모든 방법론이 픽셀 수준의 정밀한 라벨을 필요로 한다는 한계가 있다. 이를 해결하기 위해 Bounding box나 Image-level tag만을 사용하는 약지도 학습(Weakly-supervised learning) 연구들이 진행되었으나, 정밀한 마스크를 얻기에는 여전히 성능적 한계가 명확하다.

준지도 학습(Semi-supervised learning)은 분류나 시맨틱 분할, 객체 탐지 분야에서는 일관성 규제(Consistency regularization)나 의사 라벨링(Pseudo labeling)을 통해 널리 적용되었으나, 인스턴스 분할의 경우 객체의 개수와 형태를 동시에 다뤄야 하는 내재적 어려움으로 인해 연구가 미진했다. 본 논문은 기존의 경계 보존 모델(PointRend, BMask R-CNN 등)이 완전 지도 학습에서는 효과적이지만, 준지도 학습 환경에서는 경계의 노이즈까지 함께 증폭시켜 성능이 제한된다는 점을 지적하며 차별성을 둔다.

## 🛠️ Methodology

### 1. 전체 프레임워크 (Semi-supervised Pipeline)

본 논문은 다음과 같은 3단계 파이프라인을 따른다.

- **Step 1: Teacher Model Training**: 라벨링된 데이터만을 사용하여 Teacher 모델(기본 Mask R-CNN)을 학습시킨다.
- **Step 2: Pseudo Label Generation**: 학습된 Teacher 모델로 라벨 없는 이미지에서 의사 라벨을 생성한다. 이때 클래스별 분포를 맞추기 위해 다음과 같은 전략을 사용한다.
  - **Box-level threshold**: 클래스별로 이미지당 평균 인스턴스 수가 라벨 데이터와 일치하도록 설정한다.
  - **Pixel-level threshold**: RoI 내에서 전경(Foreground)과 배경(Background) 픽셀의 비율이 라벨 데이터와 동일하도록 설정하여 클래스 불균형 문제를 완화한다.
- **Step 3: Student Model Training**: 생성된 의사 라벨을 Ground-truth로 삼아 Student 모델을 학습시킨다. 이때 강한 데이터 증강(Strong augmentation: Color transformation, Cutout)을 적용하여 모델의 다양성을 확보한다.

### 2. Noise-tolerant Mask Head (NTM)

의사 라벨의 노이즈는 주로 경계 영역에 집중되어 있으며, 이는 해상도가 높을 때 더 도드라진다. 이를 해결하기 위해 NTM은 두 개의 브랜치를 운용한다.

- **저해상도 브랜치**: 마스크 크기를 $14 \times 14$ 수준으로 낮추어 학습한다. 저해상도에서는 세부 노이즈가 사라지고 깨끗한(Clean) 정보만 남게 되어 노이즈에 저항하는 능력이 생긴다.
- **고해상도 브랜치**: 기존의 정밀한 마스크를 예측하는 브랜치이다.
- **융합**: 저해상도 브랜치에서 학습된 '깨끗한 메시지'를 고해상도 브랜치에 융합하여, 모델이 노이즈에 휘둘리지 않고 전체적인 형태를 먼저 파악하도록 유도한다. 추론 단계에서는 고해상도 브랜치만 사용한다.

### 3. Boundary-preserving Map (BPM)

경계의 세부 정보를 학습하면서도 노이즈를 증폭시키지 않기 위해, 픽셀의 중요도를 재조정하는 BPM을 도입한다.

저자들은 확률 맵 $p$에 대해 라플라스 연산(Laplace operation) $\nabla^2 p$를 적용한 결과가 경계 근처의 정보를 잘 포착하면서도, 노이즈가 극심한 아주 좁은 영역은 억제한다는 점을 발견했다.

$$BPM = \nabla^2 p$$

이 BPM은 마스크 손실 함수(Mask Loss)의 가중치로 사용되어, 모델이 상대적으로 깨끗하면서도 세부 정보가 많은 경계-관련 영역(Boundary-relevant regions)에 더 집중하게 만든다.

## 📊 Results

### 실험 설정

- **데이터셋**: Cityscapes, MS COCO, BDD100K
- **베이스라인**: Mask R-CNN (ResNet-50 + FPN)
- **지표**: Mask AP, Boundary AP ($\text{AP}_{\text{bd}}$)

### 주요 결과

- **Cityscapes**: 라벨링된 데이터의 비율을 5%에서 40%까지 변화시키며 실험한 결과, 제안 방법이 모든 구간에서 지도 학습(Supervised) 및 기존 준지도 객체 탐지/시맨틱 분할 방법론들보다 우수한 성능을 보였다. 특히 라벨 데이터의 30%만 사용하고도 전체 데이터를 사용한 지도 학습과 유사한 성능을 냈으며, 40% 사용 시에는 지도 학습을 능가하는 결과를 얻었다.
- **COCO**: 매우 도전적인 데이터셋임에도 불구하고, 라벨 데이터가 5%~10%인 상황에서 지도 학습 대비 7% 이상의 큰 성능 향상을 기록했다.
- **BDD100K**: 7k의 마스크 라벨과 67k의 박스 라벨이 있는 환경에서, 박스 라벨을 사용하지 않고 오직 준지도 학습 프레임워크만 적용했을 때 기존의 부분 지도 학습(Partially-supervised) 방식보다 더 높은 AP를 달성했다.

### 절제 연구 (Ablation Study)

- **데이터 증강(DA)**: 지도 학습에서는 영향이 미비했으나, 준지도 학습에서는 AP를 28.3%에서 30.2%로 크게 향상시켜 Student 모델 학습에 필수적임을 입증했다.
- **NTM의 효과**: 전반적인 Mask AP를 향상시키며, 특히 누락되었던 객체를 더 잘 검출하게 함으로써 전체적인 판별 능력을 높였다.
- **BPM의 효과**: $\text{AP}_{\text{bd}}$(Boundary AP)를 10.9%에서 11.6%로 유의미하게 끌어올려, 실제 객체의 외곽선(Contour)을 더 정밀하게 생성함을 확인했다.

## 🧠 Insights & Discussion

본 논문은 준지도 인스턴스 분할이라는 새로운 문제를 정의하고, 의사 라벨의 '노이즈'를 어떻게 다룰 것인가에 대한 통찰력 있는 접근법을 제시했다. 특히, 모든 노이즈를 제거하려 하기보다 **저해상도 학습을 통한 노이즈 저항(NTM)**과 **라플라스 연산을 통한 경계 보존(BPM)**이라는 상호 보완적인 전략을 사용한 점이 돋보인다.

**강점 및 의의:**

- 단순한 Teacher-Student 구조를 넘어, 인스턴스 분할 특유의 '경계 노이즈' 문제에 집중하여 이론적/실험적 해결책을 제시했다.
- 라벨 데이터가 극히 적은 상황(예: COCO 5%)에서도 상당한 성능 향상을 이끌어내어 실무적 적용 가능성을 높였다.
- 인간이 작성한 거친 라벨(Coarse labels)보다 모델이 생성한 의사 라벨을 잘 정제하여 사용하는 것이 더 효과적일 수 있음을 보여주었다.

**한계 및 논의사항:**

- Teacher 모델의 초기 성능에 의존하는 경향이 있으며, 만약 Teacher 모델이 매우 낮은 품질의 의사 라벨을 생성할 경우 NTM과 BPM만으로 이를 극복할 수 있을지에 대한 분석은 부족하다.
- 추론 단계에서는 고해상도 브랜치만 사용하는데, 학습 단계에서 융합된 저해상도 특징의 효과가 추론 시점에 직접적으로 반영될 수 있는 구조적 방안이 있다면 더 발전할 가능성이 있다.

## 📌 TL;DR

본 논문은 고비용의 픽셀 라벨 문제를 해결하기 위해 **준지도 인스턴스 분할(SSIS)** 프레임워크를 제안한다. 의사 라벨의 경계 노이즈를 효율적으로 제어하기 위해 **저해상도 특징을 활용한 Noise-tolerant Mask Head(NTM)**와 **라플라스 연산 기반의 Boundary-preserving Map(BPM)**을 도입했다. 실험 결과, Cityscapes, COCO, BDD100K 데이터셋에서 지도 학습의 상한선을 돌파하는 성능을 보였으며, 이는 향후 데이터 효율적인 인스턴스 분할 연구에 중요한 기반이 될 것으로 보인다.
