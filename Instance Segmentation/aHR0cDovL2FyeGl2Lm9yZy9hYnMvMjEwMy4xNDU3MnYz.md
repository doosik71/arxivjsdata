# Sparse Object-level Supervision for Instance Segmentation with Pixel Embeddings

Adrian Wolny, Qin Yu, Constantin Pape, Anna Kreshuk (발행 연도 미명시)

## 🧩 Problem to Solve

본 논문은 인스턴스 분할(Instance Segmentation) 모델 학습 시 발생하는 **밀집 주석(Dense Annotation)의 병목 현상**을 해결하고자 한다. 일반적으로 최신 인스턴스 분할 방법론들은 모든 픽셀에 대해 정밀한 마스크 주석이 포함된 데이터셋을 필요로 한다. 하지만 생의학 영상(Biomedical images) 분야에서는 전문 지식을 가진 전문가가 직접 주석을 달아야 하므로 비용이 매우 높으며, 자연어 처리나 일반 컴퓨터 비전과 달리 사전 학습(Pre-training)에 사용할 수 있는 대규모 공개 데이터셋이 부족하다는 문제가 있다.

따라서 본 연구의 목표는 이미지 내의 모든 객체가 아닌, **일부 객체에 대해서만 마스크 주석이 제공되는 '양성-미분류(Positive-Unlabeled, PU)' 설정**에서도 효과적으로 인스턴스 분할을 수행할 수 있는 방법론을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **비공간적 픽셀 임베딩(Non-spatial Pixel Embeddings)** 공간의 구조를 활용하여 미분 가능한 방식으로 개별 인스턴스를 추출하고, 이를 통해 희소한 주석만으로도 모델을 학습시키는 것이다. 주요 기여 사항은 다음과 같다.

1. **미분 가능한 인스턴스 선택(Differentiable Instance Selection):** 픽셀 임베딩 네트워크에서 개별 인스턴스를 미분 가능한 형태로 추출하여, 인스턴스 수준의 손실 함수(Instance-level loss)를 직접 적용할 수 있게 하였다.
2. **인스턴스 수준의 일관성 손실(Instance-level Consistency Loss):** 주석이 없는(Unlabeled) 영역에서 발생할 수 있는 불안정성을 해결하기 위해, 두 개의 네트워크(f와 g)를 이용한 자기지도 학습 기반의 일관성 손실을 도입하여 배경 및 미분류 객체 영역의 학습을 안정화하였다.
3. **효율적인 클러스터링 알고리즘:** 학습된 임베딩을 최종 인스턴스로 변환하기 위해 메트릭 그래프(Metric Graph)를 분할하는 빠르고 확장 가능한 알고리즘을 제안하였다.

## 📎 Related Works

기존의 인스턴스 분할 방식은 크게 두 가지로 나뉜다. 첫째는 Mask R-CNN과 같은 **제안 기반 방법(Proposal-based methods)**으로, 바운딩 박스 같은 약한 주석으로 학습이 가능하지만 사전 학습된 백본 네트워크가 필수적이며 복잡한 비볼록(Non-convex) 형태의 객체 분할에 어려움이 있다. 둘째는 **시맨틱 분할 후 후처리를 수행하는 방법**으로, 생의학 영상에서 주로 사용되나 후처리 단계가 미분 불가능하여 전체 파이프라인의 최적화가 어렵다.

최근의 **임베딩 네트워크 기반 방법**들은 픽셀을 고차원 공간으로 매핑하여 클러스터링함으로써 객체를 분리한다. 하지만 기존 방법들은 이미지 전체에 대한 완전한 주석이 필요하며, 손실 함수를 이미지 전체 수준에서 계산한다. 본 논문은 여기서 나아가 **PU 학습(Positive Unlabeled Learning)** 개념을 인스턴스 분할에 도입하여, 일부 마스크만으로도 학습이 가능하도록 차별화하였다.

## 🛠️ Methodology

### 전체 시스템 구조

본 방법론은 입력 이미지를 $D$-차원 임베딩 벡터로 매핑하는 U-Net 구조의 네트워크 $f$를 사용한다. 학습은 크게 완전 감독(Full Supervision) 설정과 약한 감독(Weak Supervision) 설정으로 나뉜다.

### 1. 완전 감독 및 미분 가능한 인스턴스 선택

기본적으로 픽셀 임베딩 네트워크는 각 객체의 중심 $\mu_k$로 픽셀들을 끌어당기는 **Pull force**와 객체 중심 간의 거리를 멀게 하는 **Push force**를 사용한다.

$$L_{pull} = \frac{1}{C} \sum_{k=1}^{C} \frac{1}{N_k} \sum_{i=1}^{N_k} [\|\mu_k - e_i\| - \delta_v]_+^2$$
$$L_{push} = \frac{1}{C^2} \sum_{k=1}^{C} \sum_{l=1}^{C} [2\delta_d - \|\mu_k - \mu_l\|]_+^2$$

여기서 $[\cdot]_+ = \max(0, \cdot)$이며, $\delta_v$와 $\delta_d$는 각각 마진 파라미터이다. 본 논문은 여기에 더해 개별 인스턴스 수준의 손실 $L_{obj}$를 추가하기 위해 **미분 가능한 '소프트 마스크(Soft mask)'** $S_k$를 생성한다.

- **절차:** 무작위로 선택한 앵커 포인트 $a_k$를 기준으로 모든 픽셀과의 거리를 계산하고, 가우시안 커널 함수 $\phi$를 적용한다.
$$\phi(e_i, a_k) = \exp\left(-\frac{\|e_i - a_k\|^2}{2\sigma^2}\right)$$
- 이 소프트 마스크 $S_k$와 실제 정답 마스크 $I_k$ 사이의 Dice Loss를 최소화하여 인스턴스 정밀도를 높인다.
$$L_{obj} = \frac{1}{C} \sum_{k=1}^{C} D(S_k, I_k)$$

### 2. 약한 감독 (SPOCO)

주석이 일부만 제공되는 PU 설정에서는 다음 두 가지 손실 함수를 추가하여 학습을 안정화한다.

- **Unlabeled Push Loss ($L_{Upush}$):** 주석이 없는 영역 $U$의 픽셀들을 기존에 라벨링된 객체 중심 $\mu_k$로부터 밀어내어 배경과 객체가 섞이지 않도록 한다.
$$L_{Upush} = \frac{1}{C} \sum_{k=1}^{C} \frac{1}{N_U} \sum_{i=1}^{N_U} [\delta_d - \|\mu_k - e_i\|]_+^2$$

- **Embedding Consistency Loss ($L_{Ucon}$):** 두 개의 네트워크 $f$와 $g$를 사용한다. $g$는 $f$의 지수 이동 평균(EMA, Exponential Moving Average)으로 업데이트되는 모멘텀 네트워크이다. 입력 이미지에 서로 다른 증강(Augmentation)을 적용한 후, 동일한 앵커 포인트에서 추출된 두 소프트 마스크 $S_f$와 $S_g$ 사이의 겹침을 최대화(Dice Loss 최소화)한다.
$$L_{Ucon} = \frac{1}{K} \sum_{k=1}^{K} D(S_{f,k}, S_{g,k})$$

최종 손실 함수 $L_{SSO}$는 다음과 같이 구성된다.
$$L_{SSO} = \hat{L}_{SO} + \gamma \cdot L_{Upush} + \delta \cdot L_{Ucon}$$

### 3. 추론 및 클러스터링

최종 인스턴스 분할은 학습된 임베딩 공간에서 클러스터링을 통해 수행된다. 본 논문은 HDBSCAN, Mean-shift 외에도 **Affinity Graph Partitioning** 방식을 제안한다. 이는 픽셀 간의 유사도를 엣지 가중치로 변환하여 Mutex Watershed 알고리즘으로 분할하는 방식으로, 기존 클러스터링보다 속도가 매우 빠르다.

## 📊 Results

### 실험 설정

- **데이터셋:** CVPPP(식물 잎), Cityscapes(도시 거리), 3D LM(식물 조직), EM(미토콘드리아) 등 다양한 2D/3D 데이터셋을 사용하였다.
- **지표:** Symmetric Best Dice (SBD), Average Precision (AP@0.5), Adapted Rand Error 등을 사용하였다.

### 주요 결과

- **CVPPP:** 완전 감독 설정에서 SBD 0.932를 달성하여 기존의 Harmonic Embedding [28]보다 우수한 성능을 보였으며, 벤치마크 리더보드에서 최상위 성능을 기록하였다. 특히 전체 객체의 10%만 주석을 사용한 $\text{SPOCO@0.1}$ 설정에서도 성능 하락이 적었다.
- **Cityscapes:** $\text{SPOCO@0.4}$(객체의 40% 주석) 설정이 완전 감독 방식의 기본 Discriminative loss [6]보다 더 높은 mAP(0.412 vs 0.387)를 기록하였다. 이는 일관성 손실(Consistency loss)의 강력한 정규화 효과 덕분으로 분석된다.
- **생의학 영상 (LM/EM):** 3D LM 데이터셋에서 복잡한 3단계 파이프라인인 PlantSeg [55]와 유사하거나 더 나은 성능을 보였다. 또한 EM 데이터셋을 이용한 전이 학습(Transfer Learning) 실험에서, 타겟 도메인 객체의 단 1~5%만 주석을으로 사용해도 성능이 비약적으로 향상됨을 확인하였다.

## 🧠 Insights & Discussion

### 강점 및 분석

1. **일관성 손실의 중요성:** 실험 결과, $L_{Ucon}$이 없을 경우 배경 영역에서 가짜 객체(Spurious objects)가 다수 생성되는 문제가 발생한다. 일관성 손실은 배경 픽셀들을 임베딩 공간에서 하나의 밀집된 클러스터로 모으는 효과를 주어 배경 분리 능력을 크게 향상시킨다.
2. **희소 주석의 효율성:** 동일한 수의 객체를 주석 달 때, 소수의 이미지를 밀집하게 주석 다는 것보다 많은 이미지에서 소수의 객체를 희소하게 샘플링하는 것이 일반화 성능이 훨씬 높았다. 이는 네트워크가 더 다양한 데이터 분포에 노출되기 때문이다.
3. **모멘텀 네트워크의 효과:** $g$ 네트워크를 단순 공유나 독립 학습보다 EMA 기반의 모멘텀 방식으로 구현했을 때 수렴 속도가 가장 빠르고 학습이 안정적이었다.

### 한계 및 비판적 해석

- **클러스터링 알고리즘의 의존성:** 임베딩 학습은 성공적이지만, 이를 최종 마스크로 변환하는 클러스터링 단계에서 '범용적으로 완벽한' 알고리즘이 없다는 점이 한계로 지적된다. 데이터셋의 특성에 따라 HDBSCAN이나 Affinity-based 방식 중 선택해야 하는 상황이다.
- **사전 학습의 부재:** 본 연구는 사전 학습 없이 처음부터 학습(train from scratch)하는 것에 집중하였으나, 향후 자기지도 학습(Self-supervised pre-training)을 통한 성능 향상 가능성이 남아 있다.

## 📌 TL;DR

본 논문은 인스턴스 분할에서 발생하는 막대한 주석 비용 문제를 해결하기 위해, 일부 객체만 주석이 제공되는 **양성-미분류(PU) 학습 프레임워크인 SPOCO**를 제안한다. **미분 가능한 인스턴스 선택**과 **EMA 기반의 임베딩 일관성 손실**을 통해 희소한 주석만으로도 고성능의 분할 결과를 얻었으며, 특히 사전 학습된 백본을 사용할 수 없는 생의학 영상 분야에서 매우 효율적인 대안을 제시하였다. 이 연구는 향후 데이터 구축 비용을 획기적으로 줄이면서도 정밀한 인스턴스 분할을 가능케 하는 실용적인 방향성을 제시한다.
