# TRAINING-FREE DATASET PRUNING FOR INSTANCE SEGMENTATION

Yalun Dai, Lingao Xiao, Ivor W. Tsang, Yang He (2025)

## 🧩 Problem to Solve

본 논문은 인스턴스 세그멘테이션(Instance Segmentation) 작업을 위한 데이터셋 프루닝(Dataset Pruning) 문제를 해결하고자 한다. 데이터셋 프루닝은 모델의 성능을 최대한 유지하면서 훈련 데이터의 양을 줄여 학습 시간과 저장 공간을 절약하는 것이 목적이다.

기존의 데이터셋 프루닝 기술은 주로 이미지 분류(Classification) 작업에 집중되어 있으며, 인스턴스 세그멘테이션과 같은 복잡한 작업에 적용하기에는 다음과 같은 세 가지 핵심적인 어려움이 존재한다:
1. **픽셀 수준의 주석(Pixel-level annotations):** 단순한 클래스 라벨이 아니라 각 픽셀마다 라벨이 부여되며, 한 이미지에 여러 객체가 존재한다.
2. **가변적인 인스턴스 영역(Variable instance areas):** 동일한 이미지 내에서도 객체의 크기가 매우 다양하게 나타난다.
3. **클래스 불균형(Class imbalance):** 데이터셋 내에서 특정 클래스의 객체 수는 매우 많고, 일부는 매우 적은 불균형이 심하다.

또한, 기존의 분류 기반 프루닝 방법들은 중요도를 계산하기 위해 모델을 직접 훈련시켜야 하는 '훈련 기반' 방식이다. 이는 전체 학습 시간을 줄이려는 프루닝의 목적과 상충되는 모순(Paradox)을 가지며, 계산 비용이 높고 모델 아키텍처에 따라 결과가 달라지는 일반화 성능 저하 문제를 야기한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 모델 훈련이 전혀 필요 없는 **Training-Free Dataset Pruning (TFDP)** 프레임워크를 제안한 것이다.

중심적인 아이디어는 모델의 예측값이 아닌, 정답(Ground Truth) 마스크가 가진 **기하학적 형상 정보**를 활용하여 샘플의 중요도를 평가하는 것이다. 구체적으로, 객체의 경계선이 복잡할수록 학습하기 어려운 'Hard Sample'일 가능성이 높다는 직관에 기반하여 **Shape Complexity Score (SCS)**를 설계하였다. 여기에 객체 크기에 따른 편향을 제거하는 **Scale-Invariant (SI-SCS)** 기법과 클래스 간 빈도 차이를 보정하는 **Class-Balanced (CB-SCS)** 기법을 추가하여 인스턴스 세그멘테이션의 특수성을 해결하였다.

## 📎 Related Works

### 데이터셋 압축 (Dataset Compression)
- **Coreset Selection:** Entropy, EL2N, AUM 등 특정 기준에 따라 핵심 샘플을 선택하는 연구들이 진행되었다. 그러나 대부분 분류 작업에 치중되어 있으며, 모델의 출력값(Logits)에 의존하므로 계산 비용이 크고 모델 간 일반화 능력이 떨어진다는 한계가 있다.
- **Dataset Distillation:** 합성 데이터셋을 생성하여 성능을 복원하려 하지만, 인스턴스 세그멘테이션에서는 모든 픽셀에 대한 그래디언트 업데이트가 필요하여 훈련 비용이 매우 높다.

### 인스턴스 세그멘테이션 (Instance Segmentation)
- Top-down(Mask R-CNN), Bottom-up, Direct, 그리고 최근의 Transformer 기반(Mask2Former) 방법론들이 제안되었다. 하지만 이러한 모델들을 효율적으로 훈련시키기 위한 데이터셋 프루닝 연구는 거의 이루어지지 않았다.

## 🛠️ Methodology

### 1. 기존 방법론의 인스턴스 세그멘테이션 적응 (Adapted Baselines)
비교를 위해 기존 분류 기반 프루닝 방법을 다음과 같이 수정하여 적용하였다.
- **픽셀 $\to$ 인스턴스:** 각 픽셀의 중요도 점수($s_{a,b,j}$)를 계산한 후, 인스턴스 내의 모든 픽셀 점수를 **평균(Average)** 내어 인스턴스 점수를 구한다. (합계를 사용할 경우 큰 객체가 유리해지는 Area Bias가 발생하기 때문)
- **인스턴스 $\to$ 이미지:** 이미지 내의 모든 인스턴스 점수를 **합산(Sum)** 하여 최종 이미지 중요도 $I_i$를 결정한다.
$$I_i = \sum_{j=1}^{G_i} \left( \frac{1}{H_{i,j}} \sum_{a,b} s_{a,b,j} \right)$$

### 2. Proposed TFDP Pipeline
TFDP는 모델 없이 마스크의 기하학적 특성만으로 점수를 산출하며, 과정은 다음과 같다.

#### (1) Shape Complexity Score (SCS)
마스크의 경계선 복잡도를 측정하기 위해 **둘레(Perimeter)와 면적(Area)의 비율**을 사용한다.
- **계산 절차:** 이진 마스크 생성 $\to$ 주 윤곽선(Primary Contour) 추출 $\to$ 둘레 $P_{i,j}$ 및 면적 $A_{i,j}$ 계산.
- **SCS 정의:**
$$S_{i,j} = \frac{P_{i,j}}{A_{i,j}}$$
둘레 대비 면적이 작을수록(즉, 경계선이 복잡할수록) 점수가 높아진다.

#### (2) Scale-Invariant SCS (SI-SCS)
SCS는 객체의 크기가 작을수록 점수가 높아지는 경향이 있다. 이를 해결하기 위해 면적이 동일할 때 둘레가 최소인 '원(Circle)'의 SCS를 기준으로 정규화한다. 원의 SCS는 $S^{\circ}_{i,j} = \frac{2\pi r}{\pi r^2} = \frac{2}{r} = \frac{2}{\sqrt{\pi A_{i,j}}}$ 이므로, 다음과 같이 정의한다.
$$S'_{i,j} = \frac{S_{i,j}}{S^{\circ}_{i,j}} = \frac{P_{i,j}}{2\sqrt{\pi A_{i,j}}}$$
이 수식은 스케일 인자 $f$가 곱해져도 값이 변하지 않으므로, 크기와 상관없이 순수하게 '형상의 복잡도'만 측정할 수 있다.

#### (3) Class-Balanced SCS (CB-SCS)
특정 클래스가 너무 많아 해당 클래스의 샘플이 이미지 선택을 지배하는 것을 막기 위해 클래스 내 정규화를 수행한다.
$$S''_{i,j} = \frac{S'_{i,j}}{\sum_{i' \in I} \sum_{j' \in G_{i'}} S'_{i',j'}} \quad \text{where } c(i,j) = c(i',j')$$
각 인스턴스의 점수를 해당 클래스의 전체 점수 합으로 나누어, 모든 클래스가 데이터셋 구성에 동등하게 기여하도록 한다.

#### (4) 최종 이미지 점수 (Image-level Score)
최종적으로 이미지 $i$의 중요도는 해당 이미지 내 모든 인스턴스의 정규화된 점수 합으로 계산한다.
$$I_i = \sum_{j=1}^{G_i} S''_{i,j}$$

## 📊 Results

### 실험 설정
- **데이터셋:** MS COCO, VOC 2012, Cityscapes.
- **지표:** mAP, $\text{AP}_{50}$, $\text{AP}_{75}$, 크기별 AP ($\text{AP}_S, \text{AP}_M, \text{AP}_L$).
- **비교 대상:** Random, Entropy, Forgetting, EL2N, AUM, CCS.
- **백본:** Mask R-CNN (ResNet-50), SOLO-v2, QueryInst 등.

### 주요 결과
1. **성능 및 효율성:**
   - COCO 데이터셋에서 50% 프루닝 시, TFDP는 $\text{AP}_{50}$ 기준 53.4%를 달성하여 EL2N(51.2%)이나 Entropy(51.7%)보다 우수하다.
   - 특히 **시간 효율성**이 압도적이다. COCO 기준 샘플 랭킹 시간이 타 방법론 대비 평균 **1349배** 빠르다 (TFDP: 0.014h vs 타 방법론: 12~21h).
2. **일반화 능력:**
   - Mask R-CNN으로 선택한 데이터셋을 SOLO-v2(CNN 기반) 및 QueryInst(Transformer 기반)에 적용했을 때도 일관되게 성능 향상을 보였다. 반면 훈련 기반 방법들은 특정 아키텍처에 편향되어 QueryInst에서는 Random보다 낮은 성능을 보이기도 했다.
3. **확장성:**
   - ResNet-101, ResNeXt-101 등 더 큰 백본을 사용해도 TFDP의 성능 향상 효과가 유효함을 확인하였다.
4. **Ablation Study:**
   - SI-SCS와 CB-SCS를 모두 적용했을 때 가장 높은 성능이 나타났다. 특히 SI-SCS가 없으면 큰 객체($\text{AP}_L$)의 성능이 급격히 떨어진다.

## 🧠 Insights & Discussion

### 강점 및 분석
- **모델 독립성:** TFDP는 특정 모델의 예측값에 의존하지 않으므로, 한 번 프루닝한 데이터셋을 어떤 아키텍처의 모델 학습에도 사용할 수 있는 범용성을 가진다.
- **데이터 품질 개선:** 일부 실험(Cityscapes)에서는 50%의 데이터만 사용했을 때가 전체 데이터를 사용했을 때보다 성능이 높게 나타났다. 이는 TFDP가 단순한 중복 제거를 넘어, 학습에 방해가 되는 저품질 또는 노이즈 데이터를 효과적으로 제거했음을 시사한다.
- **시각적 분석:** 시각화 결과, TFDP는 경계선이 복잡한 객체가 많거나, 매우 희귀한 클래스(예: toaster)가 포함된 이미지를 'Hard' 샘플로 정확히 식별하여 보존하고, 단순한 형태의 객체만 있는 이미지를 'Easy' 샘플로 분류하여 제거함을 확인하였다.

### 한계 및 논의사항
- 본 논문은 마스크의 기하학적 정보에만 의존한다. 이미지의 텍스처, 색상, 또는 배경과의 관계와 같은 시각적 난이도는 고려되지 않았다.
- 다만, 훈련 없이 이 정도의 성능을 낸다는 점은 실무적으로 매우 큰 이점이며, 향후 비디오나 언어 데이터셋으로의 확장 가능성이 높다.

## 📌 TL;DR

본 논문은 인스턴스 세그멘테이션을 위해 **훈련이 필요 없는 데이터셋 프루닝(TFDP)** 방법을 제안한다. 마스크의 둘레-면적 비율을 이용한 **Shape Complexity Score (SCS)**를 핵심으로 하며, 여기에 크기 불변성(SI)과 클래스 균형(CB) 정규화를 추가하여 데이터셋의 핵심 샘플을 선별한다. 이 방법은 모델 훈련 기반의 기존 방식보다 **수천 배 빠르면서도 더 높은 성능과 뛰어난 모델 간 일반화 능력**을 보여주며, 대규모 데이터셋 학습 비용을 획기적으로 줄일 수 있는 실용적인 솔루션이다.
