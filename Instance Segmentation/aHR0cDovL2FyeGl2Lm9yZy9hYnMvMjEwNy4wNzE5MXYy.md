# Deep Learning based Food Instance Segmentation using Synthetic Data

Deokhwan Park, Joosoon Lee, Junseok Lee and Kyoobin Lee (2021)

## 🧩 Problem to Solve

본 논문은 식단 관리 및 헬스케어 로봇 시스템(예: 식사 보조 로봇 팔)을 위한 이미지 내 음식의 인스턴스 분할(Instance Segmentation) 문제를 해결하고자 한다. 음식 인식 기술은 크게 분류(Classification), 검출(Detection), 분할(Segmentation)로 나뉘는데, 정확한 섭취량 계산과 정밀한 조작을 위해서는 음식의 정확한 영역을 픽셀 단위로 구분하는 분할 기술이 가장 필수적이다.

그러나 음식 분할 모델을 학습시키기 위해서는 다음과 같은 세 가지 주요 어려움이 존재한다. 첫째, 음식 분류 데이터셋에 비해 분할을 위한 공개 데이터셋이 매우 부족하며, 공개된 경우에도 접근이 제한적인 경우가 많다. 둘째, 음식의 종류가 너무 다양하여 실제 데이터를 수집하고 픽셀 단위로 라벨링하는 데 막대한 노동 비용이 소요된다. 셋째, 음식은 형태, 부피, 질감, 색상 및 구성의 변화가 매우 커서 딥러닝 모델이 학습하기에 매우 도전적인 과제이다. 따라서 본 연구의 목표는 합성 데이터(Synthetic Data)를 활용하여 실제 환경에서도 적용 가능한 음식 인스턴스 분할 방법론을 제시함으로써 데이터 수집 및 라벨링의 어려움을 극복하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **Sim-to-Real** 기법을 적용하여 가상 환경에서 생성한 합성 데이터로 모델을 먼저 학습시키고, 이를 실제 환경에 적용하는 것이다. 특히 다음과 같은 설계 전략을 사용하였다.

- **Blender 기반 합성 데이터 생성**: 오픈소스 3D 그래픽 소프트웨어인 Blender를 사용하여 다양한 음식 형태의 객체와 식판, 배경을 생성하였다.
- **Domain Randomization 적용**: 실제 환경의 다양성을 반영하기 위해 식판의 텍스처, 배경 패턴, 조명 조건(개수, 위치, 강도) 등을 무작위로 변경하여 모델의 강건성(Robustness)을 높였다.
- **Class-Agnostic Segmentation**: 개별 음식의 종류를 분류하는 대신, 배경과 구분되는 '음식'이라는 단일 카테고리로 인식하는 인스턴스 분할을 수행하여 학습의 효율성을 높이고 미학습 음식(Unseen food)에 대한 일반화 성능을 확보하고자 하였다.

## 📎 Related Works

기존의 음식 분할 연구는 초기에는 Normalized Cuts, Deformable Part Model, Grab Cut, Random Forest와 같은 전통적인 이미지 처리 기법에 의존하였다. 이러한 방식은 데이터셋의 양보다는 알고리즘의 정교함이 더 중요했으나, 복잡한 음식의 특성을 모두 반영하는 데 한계가 있었다.

최근에는 딥러닝의 도입으로 Fast R-CNN을 이용한 국지화 및 인식, Saliency map estimation을 이용한 픽셀 단위 주석 없는 분할 등의 연구가 진행되었다. 하지만 대부분의 기존 연구는 동일한 종류의 음식을 서로 다른 위치에서 구분하지 못하는 시맨틱 분할(Semantic Segmentation)에 머물러 있으며, 여전히 대량의 고품질 라벨링 데이터가 필요하다는 한계가 있다. 본 논문은 이러한 데이터 의존성을 줄이기 위해 로봇 제어 및 시뮬레이션 분야에서 사용되는 Sim-to-Real 기법을 음식 분할 영역에 도입하여 차별성을 갖는다.

## 🛠️ Methodology

### 1. 데이터셋 구성
- **Synthetic Dataset**: Blender를 통해 생성되었다. 음식의 외형을 모사하기 위해 다양한 색상과 질감을 가진 기본 도형(Primitives)들을 그룹화하여 식판 위에 무작위로 배치하였다. 총 28,839장의 RGB 이미지와 대응하는 마스크 이미지를 생성하였으며, 이때 음식 객체 외의 방해 요소(Distractors)는 배경으로 처리하여 인스턴스 분할을 학습하도록 하였다.
- **Real-world Dataset**: 검증을 위해 전문가의 자문을 거쳐 선정된 50종의 한국 음식 데이터를 수집하였다. 데이터 수집 시스템을 구축하여 다양한 각도, 높이, 조명 환경에서 촬영하였으며, 총 229장의 이미지에 대해 정밀한 어노테이션을 수행하였다.

### 2. 네트워크 아키텍처
본 연구에서는 인스턴스 분할에서 널리 사용되는 **Mask R-CNN**을 사용하였다. 전체 구조는 다음과 같은 구성 요소로 이루어져 있다.
- **Backbone Network**: ResNet-50을 기반으로 하며, **Feature Pyramid Network (FPN)**를 결합하여 다양한 레이어의 특징을 피라미드 구조로 추출함으로써 풍부한 시맨틱 정보를 확보한다.
- **Region Proposal Network (RPN)**: 슬라이딩 윈도우 방식으로 이미지를 스캔하여 객체가 존재할 가능성이 높은 영역(Anchors)을 제안한다.
- **RoIAlign & Classifier**: RPN이 제안한 ROI(Region of Interest)를 바탕으로 클래스 분류, 바운딩 박스 회귀, 그리고 픽셀 단위의 마스크 예측을 수행한다.

### 3. 학습 및 평가 절차
- **학습 설정**: PyTorch를 사용하여 SGD 옵티마이저(Learning rate: $0.0001$, Weight decay: $0.00005$, Batch size: $8$)로 학습하였다.
- **학습 전략**: 세 가지 시나리오로 실험을 진행하였다. (1) 합성 데이터만으로 학습, (2) 실제 데이터만으로 학습, (3) 합성 데이터로 사전 학습(Pre-training) 후 실제 데이터로 미세 조정(Fine-tuning).
- **평가 지표**: COCO 데이터셋의 기준을 따라 **mAP (mean Average Precision)**를 사용하였다. 기본적으로 $\text{IoU (Intersection over Union)}$를 계산하며, 수식은 다음과 같다.
$$\text{IoU} = \frac{\text{area of overlap}}{\text{area of union}}$$
- 또한, 정밀도(Precision)와 재현율(Recall)을 통해 mAP를 산출하며, 각 지표의 정의는 다음과 같다.
$$\text{Precision} = \frac{\text{true positive}}{\text{true positive} + \text{false positive}}$$
$$\text{Recall} = \frac{\text{true positive}}{\text{true positive} + \text{false negative}}$$
최종 성능은 $\text{IoU}$ 임계값을 $0.50$에서 $0.95$까지 $0.05$ 간격으로 평균하여 측정하였다.

## 📊 Results

### 1. 자체 수집 데이터셋 결과
배경의 다양성에 따라 Easy, Medium, Hard 세 그룹으로 나누어 테스트한 결과, 다음과 같은 성과를 거두었다.
- **Synthetic Only**: 실제 데이터로 테스트했을 때 Mask AP@all 기준 $52.2\%$의 성능을 보였다. 이는 합성 데이터만으로도 실제 환경의 미학습 음식을 어느 정도 분할할 수 있음을 시사한다.
- **Syn + Real (Fine-tuning)**: 합성 데이터로 사전 학습 후 실제 데이터로 미세 조정을 거치자 성능이 가장 높게 나타났다. 특히 실제 데이터만으로 처음부터 학습시킨 모델(Real Only)과 비교했을 때 약 $+6.4\%p$ (또는 본문 내 언급된 $+8.8\%$)의 성능 향상을 보였다.

### 2. 공개 데이터셋 (UNIMIB2016) 결과
범용 성능 검증을 위해 UNIMIB2016 데이터셋을 사용하여 실험한 결과이다.
- **Synthetic Only**: mAP $32.9\%$를 기록하였다. 합성 데이터의 특성상 한 접시 위의 여러 음식을 하나의 인스턴스로 인식하는 경향이 발견되었다.
- **Syn + Real**: mAP $82.7\%$로 가장 높은 성능을 기록하였다.

| Test Sets | Metric | Syn + Real | Synthetic Only | Real Only |
| :--- | :--- | :---: | :---: | :---: |
| Our test set | BBOX (AP) | 76.1 | 51.4 | 65.6 |
| | SEG (AP) | 79.0 | 52.2 | 72.6 |
| UNIMIB2016 | BBOX (AP) | 80.6 | 35.7 | 79.3 |
| | SEG (AP) | 82.7 | 32.9 | 81.7 |

## 🧠 Insights & Discussion

### 강점 및 분석
본 연구는 Sim-to-Real 기법이 음식 인스턴스 분할에 유효함을 입증하였다. 특히 합성 데이터로의 사전 학습이 일종의 **일반적인 특징 추출기(General feature extractor)** 역할을 하여, 소량의 실제 데이터만으로도 빠르게 task-specific한 적응(Adaptation)을 가능하게 함을 확인하였다.

### 한계 및 비판적 해석
- **액체 모델링의 부재**: 합성 데이터만으로 학습한 모델은 국물 요리와 같은 액체 형태의 음식을 제대로 인식하지 못하는 경향이 있었다. 이는 Blender를 통한 합성 데이터 생성 시 액체 성질에 대한 모델링이 부족했기 때문이며, 이 문제는 실제 데이터로의 미세 조정을 통해서만 해결되었다.
- **인스턴스 구분 능력**: 합성 데이터 기반 모델은 한 접시에 여러 음식이 밀집해 있을 때 이를 하나의 객체로 묶어서 인식하는 경향이 있다. 이는 가상 환경에서 생성된 객체들의 배치 분포가 실제 음식의 밀집도나 접촉 상태를 완벽히 모사하지 못했기 때문으로 분석된다.
- **카테고리 인식 불가**: 본 모델은 '음식'이라는 객체 자체는 잘 찾아내지만, 그것이 구체적으로 어떤 음식인지(예: 김치인지 콩나물인지)는 구분하지 못하는 Class-agnostic 모델이라는 한계가 있다.

## 📌 TL;DR

본 논문은 데이터 수집 및 라벨링 비용이 높은 음식 분할 문제를 해결하기 위해 **Blender를 이용한 합성 데이터 생성 및 Sim-to-Real 학습 전략**을 제안하였다. Mask R-CNN을 사용하여 합성 데이터만으로도 실제 음식 분할이 가능함을 보였으며, 특히 **합성 데이터 사전 학습 후 실제 데이터로 미세 조정하는 방식**이 처음부터 실제 데이터로 학습하는 것보다 더 높은 성능과 강건성을 보임을 입증하였다. 이 연구는 향후 음식 인식 로봇의 정밀한 조작을 위한 기초 기술로 활용될 가능성이 높으며, 향후 음식 카테고리 분류 지능을 결합하는 방향으로 확장이 필요하다.
