# UNIT: Unsupervised Online Instance Segmentation through Time

Corentin Sautier, Gilles Puy, Alexandre Boulch, Renaud Marlet, Vincent Lepetit (2024)

## 🧩 Problem to Solve

본 논문은 Lidar 포인트 클라우드 데이터에서 클래스 구분 없는(class-agnostic) 비지도 방식의 온라인 인스턴스 분할(instance segmentation) 및 추적(tracking) 문제를 해결하고자 한다. 자율 주행 시스템에서 주변 환경의 객체를 정확히 분할하고 추적하는 것은 안전한 의사결정을 위해 필수적이다. 그러나 이러한 작업을 위해 필요한 수동 어노테이션(manual annotation) 비용이 매우 높으며, 특히 희귀 객체나 가혹한 기상 조건과 같은 엣지 케이스를 모두 포함하는 데이터셋을 구축하는 것은 현실적으로 매우 어렵다.

따라서 본 연구의 목표는 수동 레이블 없이 비지도 학습(unsupervised learning)만으로 입력 포인트 클라우드의 각 점을 단일 인스턴스 ID에 할당하고, 이를 시간축으로 일관되게 유지하며 추적할 수 있는 온라인 시스템을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 비지도 학습 기반의 온라인 인스턴스 분할 및 추적 프레임워크인 UNIT을 제안한 것이다. 주요 설계 아이디어는 다음과 같다.

1. **4D Pseudo-labeling**: 오프라인 단계에서 시공간 클러스터링(spatio-temporal clustering)을 통해 시간적으로 일관된 인스턴스 레이블(pseudo-labels)을 생성하여 학습 데이터로 활용한다.
2. **Auto-regressive Architecture**: Mask3D를 기반으로 하되, $t$ 시점의 쿼리 임베딩을 $t+1$ 시점의 입력으로 다시 사용하는 자기회귀(auto-regressive) 구조를 도입하여 추론 시 별도의 odometry 정보 없이도 객체를 추적할 수 있게 한다.
3. **Time Consistency Loss**: 인접한 두 프레임 사이에서 동일 객체에 대한 쿼리 할당 확률 분포가 유지되도록 강제하는 일관성 손실 함수를 제안하여 추적 성능을 높였다.
4. **Query Recycling Mechanism**: 고정된 수의 쿼리로 수많은 객체를 처리하기 위해, 객체의 무게중심(barycenter) 변화를 이용해 사용하지 않는 쿼리를 재활용하는 휴리스틱 방법을 도입하였다.

## 📎 Related Works

기존의 3D 인스턴스 분할 연구들은 주로 다음과 같은 한계를 가지고 있었다.

- **Transformer 기반 방식 (DETR, Mask3D 등)**: 뛰어난 분할 성능을 보이지만, 기본적으로 단일 프레임 단위로 동작하며 시간적 일관성을 유지하는 추적 기능이 결여되어 있다.
- **비지도 인스턴스 분할 (3DUIS, SegContrast, TARL 등)**:
  - 3DUIS는 그래프 컷(graph-cut) 방식을 사용하여 연산 비용이 높고 단일 스캔 단위로 동작한다.
  - TARL 및 SegContrast는 시공간 클러스터링을 사용하지만, 추론 시에도 여러 프레임을 누적(accumulation)해야 하거나 오프라인 방식에 가깝다.
  - AutoInst는 누적 스캔에서 동작하며, 학습 시 동적 객체가 많은 시퀀스를 제외해야 하는 제약이 있다.

UNIT은 이러한 기존 방식과 달리, **추론 시 단일 스캔(single scan)만으로 동작하는 온라인 방식**이며, 누적 스캔이나 외부의 odometry 정보 없이도 시간적 일관성을 유지한다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 전체 파이프라인

UNIT의 전체 과정은 크게 **비지도 의사 레이블 생성(Preprocessing)**과 **자기회귀 네트워크 학습(Training)** 및 **온라인 추론(Inference)**으로 나뉜다.

### 1. 시공간 클러스터링을 통한 의사 레이블링 (4D-Seg)

네트워크 학습을 위해 정답지 역할을 할 의사 레이블을 생성한다.

- **지면 제거**: Patchwork 또는 Patchwork++를 사용하여 지면 포인트를 먼저 제거함으로써 객체들이 고립된 클러스터로 나타나게 한다.
- **시공간 누적**: 40개의 연속된 스캔을 하나의 좌표계로 누적한다. 연산 효율을 위해 복셀 그리드 샘플링(voxel grid sampling)을 적용하여 포인트 수를 줄인다.
- **클러스터링**: HDBSCAN 알고리즘을 적용하여 4D 공간(3D 공간 + 시간축)에서 객체들을 군집화한다. 이때 시간 축의 가중치를 조절하여 시간적 일관성을 확보한다.

### 2. 온라인 인스턴스 분할 네트워크

Mask3D 아키텍처를 기반으로 하며, 다음과 같이 수정되었다.

- **Class-agnostic 설정**: 시맨틱 분류 헤드를 제거하고, 포인트 특징과 쿼리 임베딩의 내적(scalar product)을 통해 마스크를 생성한다.
- **Dense Segmentation**: 추론 시 각 포인트 $p$는 가장 높은 반응을 보이는 쿼리의 ID를 할당받는다.
    $$\text{query}(p) = \arg \max_{q \in Q} \langle \text{feat}(p), q \rangle$$
- **자기회귀 확장**: $t$ 시점의 트랜스포머 디코더 출력 쿼리를 $t+1$ 시점의 입력 쿼리로 사용하여 동일 객체를 계속 추적하게 한다.

### 3. 학습 목표 및 손실 함수

학습은 두 단계(단일 스캔 사전 학습 $\rightarrow$ 연속 스캔 미세 조정)로 진행된다.

**A. 스캔 단위 손실 (Scan-wise Loss)**
쿼리와 객체 인스턴스 간의 일치 정도를 측정하기 위해 Dice Loss와 Binary Cross Entropy(BCE)를 조합한 비용 행렬 $C$를 구성하고, 헝가리안 매칭(Hungarian matching)을 통해 최적의 쿼리를 할당한다.
$$C = \lambda_{\text{dice}} C_{\text{dice}} + \lambda_{\text{BCE}} C_{\text{BCE}}$$

**B. 시간 일관성 손실 (Time Consistency Loss)**
동일한 객체 $o$에 대해 $t$ 시점과 $t+1$ 시점의 쿼리 할당 확률 분포 $H_t^o$와 $H_{t+1}^o$가 유사하도록 KL-Divergence를 최소화한다.
$$L_{\text{cons}}^o = D_{\text{KL}}(H_t^o || H_{t+1}^o)$$
여기서 $H_t^o$는 타겟 분포로 간주하여 gradient 전파를 차단(stop-gradient)한다.

### 4. 쿼리 재활용 메커니즘 (Query Recycling)

고정된 쿼리 수(예: 300개)의 한계를 극복하기 위해, 각 쿼리가 할당된 포인트들의 무게중심(barycenter)을 계산한다. 현재 무게중심과 이전 무게중심 사이의 거리가 $10\text{m}$보다 크면, 해당 쿼리가 새로운 객체를 추적하고 있다고 판단하여 새로운 인스턴스 ID를 부여한다.

## 📊 Results

### 실험 설정

- **데이터셋**: SemanticKITTI, PandaSet-GT, nuScenes.
- **평가 지표**:
  - $S_{\text{temp}}^{\text{assoc}}$: 시간적 일관성을 포함한 연관성 점수.
  - $S_{\text{assoc}}$: 단일 프레임 기반의 연관성 점수.
  - $\text{IoU}^*$: 각 GT 객체에 대해 가장 잘 맞는 예측 세그먼트와의 평균 IoU.

### 주요 결과

- **SemanticKITTI**: UNIT은 모든 베이스라인(3DUIS++, TARL-Seg++, 4D-Seg++)보다 뛰어난 성능을 보였다. 특히 자신의 의사 레이블인 4D-Seg보다도 더 높은 성능을 기록했는데, 이는 네트워크 학습 과정에서 정규화(regularization) 효과가 발생했음을 시사한다.
- **PandaSet-GT**: TARL-Seg보다는 우수하며, 4D-Seg와 대등한 성능을 보였다. 4D-Seg가 약간 우세한 이유는 4D-Seg가 오프라인에서 전체 시퀀스에 접근할 수 있기 때문이다. 반면 UNIT은 온라인 방식으로 동작함에도 매우 근접한 성능을 낸다.
- **nuScenes (저밀도 데이터)**: 포인트 밀도가 낮아 의사 레이블의 품질이 떨어지므로, 4D-Seg보다는 낮은 성능을 보였다. 하지만 여전히 TARL-Seg보다는 우수하며, 온라인 방식의 이점을 가진다.

### 추가 실험: self-supervised pre-training

UNIT으로 사전 학습된 Sparse U-Net 백본을 시맨틱 분할 작업에 파인튜닝한 결과, 많은 양의 레이블 데이터를 사용할 때 최신 비지도 학습 기법들보다 더 높은 mIoU를 달성하였다.

## 🧠 Insights & Discussion

**강점 및 통찰**

- **자기회귀 쿼리의 효율성**: 명시적인 시간 일관성 손실 함수 없이 자기회귀 구조만 사용하더라도 어느 정도의 추적 성능이 나타났다. 이는 이전 프레임의 쿼리 임베딩 자체가 객체의 특성을 유지하는 기억 장치 역할을 함을 의미한다.
- **온라인 추론의 실용성**: odometry 정보나 프레임 누적 없이 단일 스캔만으로 추적을 수행함으로써 실제 자율 주행 시스템에 적용 가능한 실시간성을 확보하였다.

**한계 및 비판적 해석**

- **쿼리 재활용의 단순함**: 무게중심 거리($10\text{m}$) 기반의 재활용 방식은 매우 단순한 휴리스틱이다. 만약 새로운 객체가 이전 객체가 사라진 지점 근처에서 나타난다면 ID 혼동이 발생할 가능성이 크다.
- **의사 레이블 의존성**: nuScenes 결과에서 보듯, 초기 의사 레이블 생성 단계(4D-Seg)의 품질이 최종 모델의 성능 상한선을 결정하는 경향이 있다. 저밀도 포인트 클라우드에서 더 강건한 클러스터링 방법이 필요하다.

## 📌 TL;DR

본 논문은 Lidar 데이터에서 수동 레이블 없이 객체를 분할하고 추적하는 **UNIT**이라는 비지도 온라인 프레임워크를 제안한다. 오프라인 시공간 클러스터링으로 생성한 의사 레이블을 통해 학습하며, **자기회귀(auto-regressive) 쿼리 구조**와 **시간 일관성 손실 함수**를 통해 추론 시 odometry 없이도 온라인 추적이 가능하다. 실험 결과, SemanticKITTI 등 주요 데이터셋에서 기존 비지도 방식들을 능가하는 성능을 보였으며, 비지도 사전 학습 모델로서의 잠재력 또한 입증하였다.
