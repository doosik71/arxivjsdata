# Reinforced Coloring for End-to-End Instance Segmentation

Tuan Tran Anh, Khoa Nguyen-Tuan, Tran Minh Quan, and Won-Ki Jeong (2020)

## 🧩 Problem to Solve

본 논문은 컴퓨터 비전의 핵심 과제 중 하나인 인스턴스 분할(Instance Segmentation) 문제를 해결하고자 한다. 인스턴스 분할은 이미지 내의 객체들을 개별적으로 분리하여 라벨링하는 작업이다. 기존의 딥러닝 기반 Feed-forward 네트워크들은 고품질의 분할 결과를 생성하지만, 다수의 객체가 존재할 때 객체들이 서로 합쳐지거나(merging) 쪼개지는(splitting) 위상적 오류(topological errors)가 빈번하게 발생하며, 이를 해결하기 위해 복잡한 후처리 과정이 필요하다는 단점이 있다.

반면, 기존의 반복적(iterative) 방법들은 후처리에 의존하지 않고 한 번에 하나의 객체를 추출하는 방식을 취하지만, 이미지 내 객체의 수가 많아질수록 처리 시간이 선형적으로 증가하여 확장성(scalability)이 떨어진다는 문제가 있다. 따라서 본 연구의 목표는 기존의 단일 객체 추출 방식의 장점을 유지하면서도, 여러 객체를 병렬적으로 구분할 수 있는 확장성 있는 end-to-end 인스턴스 분할 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 인스턴스 분할 문제를 **반복적인 그래프 이진 색칠(Iterative Graph Binary Coloring) 문제**로 재정의하고, 이를 심층 강화학습(Deep Reinforcement Learning, DRL) 에이전트를 통해 해결하는 것이다.

가장 중점적인 설계 아이디어는 한 번에 하나의 객체를 찾는 것이 아니라, 모든 픽셀의 라벨을 이진수(binary representation) 형태로 구성하여 여러 객체를 동시에 구분하는 것이다. 에이전트는 각 단계 $t$에서 픽셀의 라벨을 결정하는 $t$번째 비트 값을 결정하며, 이를 통해 다수의 객체를 병렬적으로 분리해낸다. 이를 위해 저자들은 픽셀 수준의 멀티 에이전트 시스템인 PixelRL과 A3C(Asynchronous Advantage Actor-Critic) 알고리즘을 도입하였다.

## 📎 Related Works

논문에서는 인스턴스 분할과 관련된 기존 연구를 세 가지 범주로 분류하여 설명한다.

1.  **지식 기반 접근 방식(Knowledge-based approaches):** 픽셀 간의 정량적 관계를 그래프의 에지로 구성하고 min-cut, max-flow 등의 알고리즘을 사용하는 방식이다. 하지만 수동으로 설계된(hand-crafted) 사전 지식이 항상 분할 목표와 일치하지 않는다는 한계가 있다.
2.  **지도 학습 접근 방식(Supervised learning approaches):** FCN, U-Net과 같은 CNN 기반 네트워크들이 성공을 거두었으며, 최근에는 RNN을 이용하여 순차적으로 마스크를 예측하는 방식(예: Ren et al.의 E2E)이 제안되었다. 그러나 이러한 순차적 예측 방식은 객체 수가 많아지면 효율성이 급격히 떨어진다.
3.  **강화 학습 접근 방식(Reinforcement learning approaches):** 복잡한 의사결정 과정을 해결하기 위해 DRL이 도입되었다. Araslanov et al.은 순차적 객체 탐지 및 분할 작업을 RL로 정식화하였으나, 여전히 한 번에 하나의 객체만 처리한다는 점에서 확장성 문제가 남아 있다.

본 연구는 이러한 기존 RL 기반 방식과 달리, **여러 객체를 동시에 구분할 수 있는 병렬적 이진 색칠 방식**을 제안함으로써 차별성을 갖는다.

## 🛠️ Methodology

### 전체 시스템 구조
본 제안 방법은 이미지의 모든 픽셀에 대해 이진 라벨을 반복적으로 할당하는 구조이다. 전체 파이프라인은 상태(State) 입력 $\rightarrow$ 에이전트 네트워크 $\rightarrow$ 액션(Action) 생성 $\rightarrow$ 라벨 업데이트 $\rightarrow$ 보상(Reward) 계산 및 학습의 루프로 구성된다.

### 이진 색칠 과정 (Binary Coloring Process)
인스턴스 분할을 위해 각 픽셀 $v$의 색상(라벨) $C(v)$를 결정한다. 최적의 색상 매핑 함수 $F$를 찾는 것은 NP-Hard 문제이므로, 본 논문에서는 이를 $T$단계의 반복적인 이진 색칠 과정으로 근사한다.

단계 $t$에서 에이전트의 액션 $F^{(t)}(V, C^{(t)}(V))[v] \in \{0, 1\}$는 픽셀 $v$의 라벨을 나타내는 이진수의 $t$번째 비트가 된다. 단계 $t+1$에서의 색상 값은 다음과 같은 방정식으로 업데이트된다.

$$C^{(t+1)}(V)[v] = C^{(t)}(V)[v] + 2^t F^{(t)}(V, C^{(t)}(V))[v]$$

이 과정을 통해 $T$단계가 끝나면, 각 픽셀은 $0$부터 $2^T-1$ 사이의 정수 라벨을 갖게 된다.

### 강화학습 프레임워크 (A3C & PixelRL)
- **에이전트:** 각 픽셀 $v_i$마다 픽셀 수준 에이전트가 할당되어 독립적으로 액션을 취하며, 전체 에이전트들의 평균 보상을 최대화하는 것을 목표로 한다.
- **상태(State):** 입력 이미지 $I$와 현재 단계의 이진 인코딩된 색상 맵 $C^{(t)}$가 결합된 형태이다. 입력 이미지의 채널이 $K$이고 색칠 단계가 $T$라면, 상태 공간의 크기는 $H \times W \times (K+T)$가 된다.
- **액션(Action):** 각 픽셀은 $0$ 또는 $1$ 중 하나의 값을 선택한다.
- **네트워크 아키텍처:** 핵심 네트워크로 Attention U-Net(AttU)을 사용한다. 입력 이미지와 색상 맵은 각각 서로 다른 경로를 통해 처리된 후 Concatenation되어 네트워크에 입력된다.

### 보상 함수 (Reward Function)
보상 함수는 픽셀들이 동일한 객체면 같은 색을 갖고, 다른 객체면 다른 색을 갖도록 유도하며 세 가지 구성 요소로 이루어져 있다.

1.  **배경-전경 분리 보상 ($R_{BF}$):** 첫 번째 단계($t=0$)에서 배경과 전경을 정확히 구분했는지 평가한다.
2.  **분리 보상 (Splitting Reward, $R_S$):** 서로 다른 ground truth(GT) 세그먼트에 속한 픽셀들이 서로 다른 색상을 갖도록 유도한다. 분리 반경 $r$ 내의 픽셀들 간의 에지를 구성하여, 올바르게 분리된 경우($TS$)는 보상을 주고 잘못 합쳐진 경우($FM$)는 감점을 준다.
3.  **합치기 보상 (Merging Reward, $R_M$):** 동일한 GT 세그먼트에 속한 픽셀들이 동일한 색상을 갖도록 유도한다. 특히 객체의 내부 영역(inner region)에 더 높은 가중치를 두어 중심부부터 견고하게 합쳐지도록 설계하였다.

최종 보상 $R^{(t)}(v)$는 다음과 같이 합산된다.
- $t=0$ 일 때: $R^{(0)}(v) = R_{BF}^{(0)}(v)$
- $1 \le t < T$ 일 때: $R^{(t)}(v) = R_{BF}^{(t)}(v) + w_m \sum R_M + w_s \sum R_S$

## 📊 Results

### 실험 설정
- **데이터셋:** CVPPP (식물 잎), KITTI (자동차), CREMI (전자현미경 세포 이미지)
- **비교 대상:** E2E (Ren et al.), AC-Dice (Araslanov et al.) 등
- **평가 지표:** Symmetric Best Dice (SBD), $|DiC|$ (CVPPP), MWCov, MUCov, AvgFP, AvgFN (KITTI), VOI-split, VOI-merge, ARand (CREMI)

### 주요 결과
1.  **CVPPP:** SBD 지표에서 기존의 E2E 및 AC-Dice 방식과 대등한 성능을 보였으며, 객체 수 예측 오차($|DiC|$)에서도 경쟁력 있는 결과를 나타냈다.
2.  **KITTI:** 평균 가중 커버리지(MWCov) 등에서 우수한 성능을 보였으며, 특히 False Positive(AvgFP)와 False Negative(AvgFN) 비율이 기존 순차적 방식들보다 낮게 나타나 일반화 능력이 뛰어남을 입증했다.
3.  **CREMI:** 객체가 매우 밀집된 데이터셋에서 본 방법의 강점이 극명하게 드러났다. E2E 방식은 객체가 너무 많을 때 추적을 놓쳐 배경으로 처리하는 경우가 많았으나, 본 제안 방법은 병렬적 색칠 과정을 통해 정밀하게 분할해냈다. 특히, **추론 시간이 객체 수에 관계없이 일정하게 유지**되어 탁월한 확장성을 보여주었다.

## 🧠 Insights & Discussion

**강점 및 해석:**
본 논문은 인스턴스 분할을 '색칠 문제'로 치환함으로써, 기존의 '하나씩 찾는' 방식의 한계인 시간 복잡도 문제를 해결했다. 특히 CREMI 데이터셋 결과에서 보듯, 객체가 매우 많고 밀집된 환경에서도 성능 저하 없이 일정한 추론 시간을 유지한다는 점은 매우 고무적이다. 또한, 분리 보상($w_s$)과 합치기 보상($w_m$)의 가중치 조절을 통해, 모델이 먼저 쉬운 작업(합치기)을 학습한 후 점진적으로 어려운 작업(분리)을 학습하는 경향을 발견하였다. 실험적으로 $w_s/w_m > 1$일 때 더 좋은 성능이 나왔는데, 이는 객체를 세밀하게 나누는 작업에 더 강한 인센티브가 필요함을 시사한다.

**한계 및 미해결 질문:**
- **후처리 의존성:** 평가 단계에서 리사이징, 작은 세그먼트 제거, 라벨 재인덱싱 등의 후처리를 수행하였다. 이는 완전한 end-to-end라고 하기에는 일부 후처리에 의존하고 있음을 의미한다.
- **T 값의 설정:** 색칠 단계 $T$에 따라 구분 가능한 최대 객체 수가 $2^T$개로 제한된다. 매우 많은 수의 객체가 존재하는 이미지에서 $T$를 어떻게 효율적으로 설정할지에 대한 논의가 부족하다.
- **계산 복잡도:** A3C 기반의 학습 과정이 일반적인 지도 학습 방식보다 학습 시간이 오래 걸릴 가능성이 높다.

## 📌 TL;DR

본 논문은 인스턴스 분할 문제를 **반복적 이진 색칠(Iterative Binary Coloring)** 문제로 정의하고, 이를 **심층 강화학습(A3C, PixelRL)**으로 해결하는 새로운 프레임워크를 제안하였다. 이 방식은 여러 객체를 병렬적으로 구분함으로써, 기존의 순차적 추출 방식이 가졌던 **확장성 문제(객체 수 증가에 따른 추론 시간 증가)를 해결**하였으며, 특히 객체가 밀집된 환경에서 뛰어난 성능과 일정한 추론 시간을 보장한다. 향후 대규모 다중 객체 분할 작업에 적용될 가능성이 매우 높은 연구이다.
