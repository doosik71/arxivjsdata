# FreePoint: Unsupervised Point Cloud Instance Segmentation

Zhikai Zhang, Jian Ding, Li Jiang, Dengxin Dai, Guisong Xia (2024)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 3D 포인트 클라우드(Point Cloud)의 **비지도 클래스 불가지론적 인스턴스 분할(Unsupervised Class-Agnostic Instance Segmentation)**이다. 인스턴스 분할은 장면 내에서 개별 객체를 위치시키고 분리하는 중요한 작업이지만, 이를 달성하기 위해서는 정밀한 픽셀/포인트 단위의 수동 어노테이션이 대량으로 필요하다. 예를 들어 ScanNet 데이터셋의 평균적인 장면 하나를 라벨링하는 데 약 22.3분이 소요될 정도로 비용이 매우 높다.

따라서 본 연구의 목표는 수동 라벨에 대한 의존성을 완전히 제거하고, 어노테이션 없이도 포인트 클라우드에서 객체 인스턴스를 효과적으로 분리할 수 있는 프레임워크인 **FreePoint**를 제안하는 것이다.

## ✨ Key Contributions

FreePoint의 중심 아이디어는 **그래프 파티셔닝을 통한 유사-라벨(Pseudo-label) 생성**과 이를 활용한 **단계적 딥러닝 모델 학습**의 결합이다.

1. **하이브리드 특징 표현(Hybrid Feature Representation):** 좌표(Coordinates), 색상(Colors), 그리고 자기지도학습(Self-supervised) 기반의 딥러닝 특징을 결합하여 포인트 간의 유사도를 더욱 정밀하게 측정한다.
2. **id-as-feature 전략:** 하향식(bottom-up) 멀티컷(multicut) 알고리즘의 무작위성을 줄이기 위해, 여러 번의 알고리즘 실행 결과를 특징으로 다시 사용하는 앙상블 전략을 도입하여 유사-라벨의 품질을 높였다.
3. **약지도 단계적 학습(Weakly-supervised Two-step Training):** 생성된 유사-라벨의 부정확성을 극복하기 위해, 과분할(over-segmented) 마스크로 기본 특징을 학습시킨 후, 저분할(under-segmented) 마스크로 객체를 연결하는 2단계 학습 전략과 이를 보완하는 특수 손실 함수를 제안하였다.

## 📎 Related Works

기존의 포인트 클라우드 인스턴스 분할 연구는 대부분 완전 지도학습(Fully-supervised) 방식에 의존하며, 최근에는 일부 라벨만을 사용하는 약지도(Weakly-supervised) 또는 준지도(Semi-supervised) 방식이 제안되었다. 하지만 이들 역시 여전히 상당량의 바운딩 박스나 포인트 라벨을 필요로 한다.

비지도 학습 분야에서는 2D 이미지에서의 성공을 바탕으로 한 시도들이 있었으며, 3D 분야에서는 최근 **UnScene3D**가 제안되었다. 본 논문은 UnScene3D와 다음과 같은 차별점을 가진다:

- **특징 활용:** UnScene3D가 2D 및 3D 사전 학습 모델의 멀티모달 특징을 사용하는 반면, FreePoint는 오직 3D 색상 및 기하학적 특징만을 사용한다.
- **학습 전략:** 시간이 많이 소요되는 다회차 자기훈련(multi-round self-training) 대신, 효율적인 2단계 학습 전략을 설계하였다.

## 🛠️ Methodology

FreePoint의 전체 파이프라인은 전처리 및 특징 추출, 유사-라벨 생성, 단계적 학습의 세 단계로 구성된다.

### 1. 전처리 및 포인트 특징 추출

먼저 평면 분할(Plane segmentation) 알고리즘을 반복 적용하여 바닥, 벽, 천장과 같은 배경 포인트($V_{bg}$)를 제거하고 전경 포인트($V_{fg}$)만을 추출한다. 이후 계산 효율성을 높이기 위해 Farthest Point Sampling(FPS)을 통해 다운샘플링된 포인트 셋 $V_{sampled}$를 생성한다.

특징 표현을 위해 다음 세 가지 정보를 결합한다:

- **딥러닝 특징:** 자기지도 사전 학습된 백본을 통해 추출한 임베딩.
- **기하학적 특징:** 포인트의 XYZ 좌표.
- **색상 특징:** 포인트의 RGB 값.

### 2. 포인트 클라우드 분할 및 유사-라벨 생성

포인트들을 정점으로, 인접한 $k_1$개의 포인트를 엣지로 하는 무방향 그래프 $G=(V, E, A)$를 구축한다. 엣지의 가중치인 어피니티(Affinity) $A$는 다음과 같이 계산된다.

딥러닝 임베딩 $F$에 대한 코사인 유사도는 다음과 같다:
$$A^{(i,j)}_{emb} = \text{Cos}(F_i, F_j)$$

좌표 $P$와 색상 $C$에 대한 $L_2$ 거리는 다음과 같다:
$$A^{(i,j)}_{xyz} = -\|P_i, P_j\|^2, \quad A^{(i,j)}_{rgb} = -\|C_i, C_j\|^2$$

최종 어피니티는 각 항목을 정규화한 후 가중합으로 계산한다:
$$A = \alpha_1 A_{emb} + \alpha_2 A_{xyz} + \alpha_3 A_{rgb}$$

이후 GPU 기반의 고속 멀티컷 알고리즘인 **RAMA**를 사용하여 그래프를 분할한다. 이때 알고리즘의 무작위성을 해결하기 위해 **id-as-feature** 전략을 사용한다. RAMA를 $T$번 실행하여 각 포인트가 얻은 라벨 ID를 새로운 특징으로 생성하고, 이를 기반으로 다시 RAMA를 실행하여 최종 유사-라벨 $L$을 생성한다. ID 기반 유사도는 다음과 같이 정의된다:
$$A^{(i,j)}_{id} = \frac{1}{T} \sum_{t=1}^T I[IDF_i[t] = IDF_j[t]]$$

또한, 하이퍼파라미터 $\sigma$를 조절하여 **Base masks**(과분할 상태, 객체 위치 파악에 용이)와 **Under-segmented masks**(저분할 상태, 객체 전체 형태 보존에 용이) 두 종류의 마스크를 생성한다.

### 3. 유사-라벨을 이용한 학습

본 논문은 Mask3D를 기본 네트워크로 사용하며, 부정확한 유사-라벨에 대응하기 위해 다음과 같은 손실 함수와 전략을 제안한다.

**약지도 학습 손실 함수:**
포인트 단위의 손실 외에, 예측 마스크 $m$과 타겟 마스크 $m^*$의 중심점(Center)과 바운딩 박스(Bounding Box)의 거리를 최소화하는 손실을 추가한다.

- **중심점 손실:** $\mathcal{L}_{mean} = \text{Euclidean}(\text{avg}(m), \text{avg}(m^*))$
- **바운딩 박스 손실:** $\mathcal{L}_{box} = \sum (\text{Euclidean}(\max(m), \max(m^*)) + \text{Euclidean}(\min(m), \min(m^*)))$

최종 손실 함수는 다음과 같다:
$$\mathcal{L} = \lambda_{dice}\mathcal{L}_{dice} + \lambda_{BCE}\mathcal{L}_{BCE} + \lambda_{mean}\mathcal{L}_{mean} + \lambda_{box}\mathcal{L}_{box}$$

**2단계 학습 전략(Step-training Strategy):**

1. **1단계:** Base masks를 사용하여 학습한다. 모델은 객체의 부분(part)들을 분리하는 능력을 배운다.
2. **2단계:** Under-segmented masks를 사용하여 소수의 에포크 동안 학습한다. 모델은 과분할된 부분들을 하나의 온전한 객체로 연결하는 방법을 배운다. 이때, 예측 마스크보다 유사-라벨 마스크가 지나치게 클 경우 매칭을 무시하는 **undersegmentation-ignore** 설계를 통해 성능 저하를 방지한다.

## 📊 Results

### 실험 설정

- **데이터셋:** ScanNet (실내 장면), S3DIS.
- **지표:** Average Precision (AP, $\text{AP}_{50}, \text{AP}_{25}$). 세만틱 라벨 없이 인스턴스 마스크의 정확도만 측정한다.

### 주요 결과

1. **비지도 인스턴스 분할 (ScanNet):**
    - FreePoint는 $\text{AP} = 18.9\%$를 달성하여, 기존 전통적인 클러스터링 방법들(DBSCAN 등)보다 18.2% 이상 높은 성능을 보였다.
    - 경쟁 모델인 UnScene3D(3D 특징만 사용 시) 대비 5.5% 높은 AP를 기록하며 SOTA 성능을 달성하였다.

2. **사전 학습 작업(Pretext Task)으로서의 활용 (S3DIS):**
    - FreePoint를 사전 학습시킨 후 제한된 라벨(10% masks)로 파인튜닝했을 때, $\text{AP} = 42.9\%$를 기록하여 기존 자기지도 학습 방법인 CSC($37.1\%$)보다 $6.0\%$ 높은 성능을 보였다. 이는 비지도 인스턴스 분할 작업이 유의미한 3D 특징 표현 학습으로 이어진다는 것을 시사한다.

### Ablation Study

- **특징 표현:** 딥러닝 특징과 전통적 특징(XYZ, RGB)을 결합했을 때 가장 높은 성능을 보였다.
- **분할 방법:** top-down 방식(Spectral, FreeMasks)은 복잡한 3D 장면에서 실패하였으나, RAMA의 bottom-up 방식은 효과적이었다.
- **학습 전략:** 2단계 학습 전략을 적용했을 때 $\text{AP}$가 $14.2\% \rightarrow 18.9\%$로 크게 상승하여, 과분할/저분할 마스크의 상호 보완 효과가 입증되었다.

## 🧠 Insights & Discussion

본 연구는 비지도 환경에서 3D 인스턴스 분할이 가능하다는 것을 입증하였으며, 특히 다음의 통찰을 제공한다:

- **Bottom-up 접근법의 우위:** 2D에서 성공적이었던 top-down 방식의 비지도 분할 모델들이 3D의 복잡하고 밀집된 환경에서는 제대로 작동하지 않음을 발견하였다. 이는 3D 데이터의 특성에 맞는 새로운 분할 메커니즘의 필요성을 시사한다.
- **유사-라벨의 노이즈 관리:** 단순히 유사-라벨을 생성해 학습시키는 것이 아니라, 과분할과 저분할의 특성을 이용한 단계적 학습과 중심점/바운딩 박스 기반의 약지도 손실 함수를 통해 노이즈를 효과적으로 억제할 수 있음을 보여주었다.

한계점으로는 여전히 유사-라벨 생성 단계에서 하이퍼파라미터 $\sigma$에 의존한다는 점과, 매우 복잡한 객체 간의 접점 부분에서 완벽한 분리가 어려울 수 있다는 점이 있다.

## 📌 TL;DR

FreePoint는 **[하이브리드 특징 추출 $\rightarrow$ RAMA 기반 유사-라벨 생성 $\rightarrow$ 2단계 약지도 학습]**으로 이어지는 파이프라인을 통해, 어노테이션 없이도 높은 성능의 3D 인스턴스 분할을 수행하는 프레임워크이다. 특히 비지도 학습만으로도 기존 SOTA를 상회하는 성능을 보였으며, 이를 사전 학습 작업으로 활용했을 때 적은 라벨만으로도 지도 학습 성능을 크게 향상시킬 수 있음을 증명하였다. 이 연구는 향후 3D 장면 이해를 위한 데이터 효율적인 학습 방향에 중요한 기여를 할 것으로 보인다.
