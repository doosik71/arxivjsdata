# HUNIS: High-Performance Unsupervised Nuclei Instance Segmentation

Vasileios Magoulianitis, Yijing Yang, C.-C. Jay Kuo (2022)

## 🧩 Problem to Solve

본 논문은 디지털 병리학(digital pathology)의 핵심 과제인 조직학 이미지 내의 핵 인스턴스 분할(nuclei instance segmentation) 문제를 다룬다. 핵 분할은 암 진단 및 평가를 위해 분자 형태학적 정보를 제공하는 필수적인 단계이다.

일반적으로 이러한 작업은 전문가의 수동 주석(manual annotation)에 의존하는데, 이는 매우 많은 시간이 소요될 뿐만 아니라 관찰자 간의 변동성(inter-observer variability)이 크다는 문제가 있다. 또한, 서로 다른 실험실의 프로토콜이나 스캐너에 따라 이미지의 색상과 질감이 달라지므로, 지도 학습(supervised learning) 기반의 딥러닝 모델들은 학습 데이터에 없는 새로운 도메인(unseen organs)에 적용했을 때 일반화 성능이 급격히 떨어진다는 한계가 있다. 따라서 본 연구의 목표는 대량의 주석 데이터 없이도 높은 성능을 낼 수 있는 고성능 비지도 학습 기반의 핵 인스턴스 분할 방법론인 HUNIS를 제안하는 것이다.

## ✨ Key Contributions

HUNIS의 핵심 아이디어는 **2단계 블록 기반 연산(two-stage block-wise operations)**과 **자기 지도 학습(self-supervision) 메커니즘**의 결합이다.

첫 번째 단계에서 적응형 임계값 처리와 형태학적 필터링을 통해 초기 분할 결과를 얻고, 이를 통해 생성된 픽셀 수준의 의사 라벨(pseudo-labels)을 두 번째 단계의 가이드로 사용한다. 이러한 설계는 복잡한 딥러닝 모델의 학습 과정 없이도 데이터 자체의 특성을 이용하여 정밀도를 높이는 효과를 제공하며, 특히 학습 가능한 파라미터가 전혀 없다는 점에서 연산 효율성과 일반화 능력을 동시에 확보하였다.

## 📎 Related Works

기존의 핵 분할 접근 방식은 크게 세 가지로 나뉜다.

첫째, 딥러닝 이전의 전통적인 방식으로는 적응형 임계값 처리(adaptive thresholding), 클러스터링, Active Contours, Graph Cuts 등이 있으며, 특히 Watershed 알고리즘은 후처리 단계에서 널리 사용되었다.

둘째, 최근의 딥러닝 기반 지도 학습 방식들은 다중 스케일 분기나 맞춤형 손실 함수를 통해 높은 성능을 보이지만, 앞서 언급한 대로 데이터 부족 문제와 도메인 일반화(generalization) 문제에 취약하다.

셋째, 비지도 학습 기반의 딥러닝 방식들은 도메인 적응(domain adaptation)이나 GAN을 이용한 데이터 합성 등을 시도하였으나, 여전히 지도 학습 방식에 비해 성능이 크게 떨어진다. 비-딥러닝 방식인 CBM(Color-Based Method)은 투명한 파이프라인을 제공하지만, 본 논문에서 제안하는 HUNIS보다 성능이 낮다.

## 🛠️ Methodology

HUNIS는 크게 두 단계의 파이프라인으로 구성된다. 전체 이미지($1000 \times 1000$)는 국부적 균질성을 보장하기 위해 $50 \times 50$ 크기의 블록으로 분할되어 처리된다.

### Stage-1: Initial Segmentation

첫 번째 단계는 초기 마스크를 생성하고 의사 라벨을 확보하는 과정이다.

1. **Adaptive Thresholding**:
    * RGB 이미지를 Hematoxylin(H) 색상 기저로 투영하고 대비 향상을 적용한다.
    * PCA(주성분 분석)를 통해 RGB 채널 간의 상관관계를 제거하고 첫 번째 주성분만을 남겨 3D 색상 정보를 1D 강도(intensity) 값으로 변환한다.
    * 기본적으로 이봉 분포(bi-modal assumption)를 가정하여 임계값을 설정하지만, 대비가 낮거나 모호한 영역이 있을 경우 다음 식을 통해 임계값을 보정한다.
    $$T' = T_o + \lambda(T_o - T_c)$$
    여기서 $T_o$는 두 피크 사이의 중간점, $T_c$는 두 피크를 잇는 직선의 수평축 절편, $\lambda$는 보정 강도를 조절하는 가중치이다.

2. **Incorporation of Size/Shape Priors**:
    * 임계값 처리 후 발생하는 노이즈를 제거하기 위해 핵의 크기와 모양에 대한 사전 지식을 활용한다.
    * 비정상적으로 작은 인스턴스는 제거하고, Convex Hull 알고리즘을 통해 서로 붙어 있는 핵들을 분리하며, 내부 구멍 채우기(hole filling)를 수행한다.

3. **Removal of False Positive (FP) Instances**:
    * $200 \times 200$ 크기의 타일 단위로 처리하며, 인스턴스를 크기에 따라 기준 집합 $R$(큰 핵)과 쿼리 집합 $Q$(작은 핵)로 나눈다.
    * HSV 색공간의 대비 값을 특징 벡터 $x$로 추출하여, $R$ 집합의 평균 특징 벡터 $x_R = \frac{1}{|R|} \sum_{i \in R} x_i$와 쿼리 샘플 $x_j$ 간의 유사도를 계산한다.
    $$S(x_R, x_j) = e^{-\gamma ||x_j - x_R||^2}$$
    유사도 $S$가 임계값 $T_s$보다 낮으면 거짓 양성(FP)으로 간주하여 제거한다.

### Stage-2: Refinement

두 번째 단계는 Stage-1에서 얻은 의사 라벨을 활용하여 정밀도를 높이는 과정이다.

1. **Self-Supervised Binary Classification**:
    * $200 \times 200$ 타일 내에서 Stage-1의 의사 라벨을 학습 데이터로 사용하여 픽셀 단위 분류기를 학습시킨다.
    * 특히 분류기의 확신도가 낮은 불확실한 픽셀(주로 핵의 경계 영역)에 대해 예측을 수행함으로써 라벨의 노이즈를 제거하고 경계를 정밀하게 조정한다.

2. **Shape Refinement**:
    * 분류 결과에서 발생할 수 있는 형태 왜곡을 방지하기 위해 Stage-1에서 사용한 Convex Hull 및 형태학적 연산을 다시 한번 적용하여 최종 인스턴스를 확정한다.

## 📊 Results

본 연구는 2018 MICCAI MoNuSeg 데이터셋을 사용하여 성능을 평가하였다. 성능 지표로는 인스턴스 수준의 검출과 픽셀 수준의 오류를 동시에 고려하는 Aggregated Jaccard Index (AJI)를 사용하였다.

### 실험 설정

* **Test-1**: 학습 데이터에 포함된 장기들로 구성된 14장의 이미지.
* **Test-2**: 학습 데이터에 없었던 새로운 장기(방광, 결장, 위)로 구성된 6장의 이미지.
* **하이퍼파라미터**: $\lambda=0.3, \gamma=0.1, T_s=0.6$.

### 주요 결과

* **Test-1 결과**: HUNIS는 AJI $0.6387$을 기록하여 모든 비지도 학습 방법(CBM 포함)을 크게 상회하였다. 또한, 지도 학습 기반의 UNet-Atten($0.6498$)과 비교해도 매우 경쟁력 있는 성능을 보였다.
* **Test-2 결과**: 새로운 도메인에 대한 일반화 능력을 테스트한 결과, HUNIS는 AJI $0.6548$을 기록하며 모든 비지도 및 지도 학습 모델(CIA-Net 등)보다 높은 성능을 보였다. 이는 지도 학습 모델이 데이터 부족으로 인해 새로운 장기 이미지에서 성능이 하락하는 것과 대조적이다.
* **Ablation Study**: Stage-1의 FP 제거 모듈과 Stage-2의 전체 공정이 AJI 점수를 단계적으로 유의미하게 향상시킴을 확인하였다.

## 🧠 Insights & Discussion

HUNIS의 가장 큰 강점은 **학습 가능한 파라미터가 전혀 없다**는 점이다. 수백만 개의 파라미터를 가진 딥러닝 모델(예: NucleiSegNet, UNet-Atten)과 달리, HUNIS는 단순한 수학적 연산과 형태학적 규칙에 기반하므로 GPU 없이 모바일이나 엣지 디바이스에서도 구현 가능하다.

특히 새로운 도메인(Test-2)에서 지도 학습 모델보다 뛰어난 성능을 보인 점은, 데이터에 의존적인 학습 대신 이미지 자체의 통계적 특성과 기하학적 사전 지식을 활용한 접근 방식이 의료 영상의 도메인 전이 문제에 더 강건할 수 있음을 시사한다.

다만, 본 논문은 특정 데이터셋(MoNuSeg)에 특화된 하이퍼파라미터를 사용하였으며, $\lambda, \gamma, T_s$와 같은 값들이 다양한 종류의 조직 이미지에서도 동일하게 작동하는지에 대한 범용성 검증은 추가적으로 필요해 보인다. 또한, 딥러닝을 전혀 사용하지 않았기에 복잡한 형태의 중첩된 핵(overlapping nuclei)을 분리하는 능력에 한계가 있을 수 있다는 점이 잠재적 제약 사항이다.

## 📌 TL;DR

HUNIS는 2단계 블록 기반 처리와 자기 지도 학습 메커니즘을 통해 주석 데이터 없이도 고성능의 핵 인스턴스 분할을 수행하는 비지도 학습 방법론이다. 학습 파라미터가 없어 연산 효율이 극도로 높으며, 특히 새로운 장기의 이미지에서도 지도 학습 모델보다 뛰어난 일반화 성능을 보였다. 이는 데이터 확보가 어려운 의료 영상 분석 분야에서 매우 실용적인 '그린 솔루션'이 될 가능성이 크다.
