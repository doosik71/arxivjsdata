# BUILDING HEIGHT PREDICTION WITH INSTANCE SEGMENTATION

Furkan Burak Bağcı, Ahmet Alp Kindiroğlu, Metehan Yalçın, Ufuk Uyan, Mahiye Uluyağmur Öztürk (2022)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 단일 RGB 위성 이미지로부터 개별 건물의 마스크(Mask)와 해당 건물의 높이를 동시에 예측하는 것이다. 일반적으로 건물 높이 추출에는 Lidar나 스테레오 이미지로 생성된 DSM(Digital Surface Models)이 사용되지만, 이러한 데이터는 획득 비용이 높고 가용성이 낮다는 단점이 있다.

단일 RGB 이미지 기반의 높이 예측은 데이터의 부족, 낮은 데이터 품질, 건물 유형의 다양성, 그리고 태양광의 각도에 따른 그림자와 가림 현상(Occlusion) 등으로 인해 매우 어려운 과제이다. 따라서 본 연구의 목표는 Instance Segmentation 기법을 활용하여 개별 건물을 구분함과 동시에, 각 인스턴스별 높이 클래스를 정확하게 분류하는 모델을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 중심적인 아이디어는 Instance Segmentation을 통해 건물을 개별 객체로 분리하고, 이를 높이 예측과 결합하는 것이다. 기존의 Semantic Segmentation 방식은 픽셀 단위의 예측만 가능하여 인접한 건물들을 하나의 덩어리로 인식하는 문제가 있었으나, Instance Segmentation을 적용함으로써 각 건물을 독립적인 개체로 식별하고 개별적인 높이 값을 부여할 수 있게 하였다.

또한, 데이터 부족 및 일반화 성능 저하 문제를 해결하기 위해 대규모 오픈소스 위성 데이터셋을 활용한 Transfer Learning(전이 학습) 전략을 도입하였다. 이는 다양한 지리적 특성과 조명 조건이 반영된 데이터로 모델을 사전 학습시킨 후, 특정 높이 정보가 포함된 데이터셋으로 미세 조정(Fine-tuning)함으로써 모델의 강건성을 높이는 설계이다.

## 📎 Related Works

기존의 건물 추출 연구들은 주로 Semantic Segmentation 방식을 사용하였다. 하지만 이 방식은 동일 클래스 내의 서로 다른 객체를 구분하지 못하며, 이를 해결하기 위해 후처리(Connected Component Analysis 등)를 거쳐야 하는데, 이 과정에서 인접 건물들이 하나로 묶여 3D 모델링 시 오류가 발생할 가능성이 크다.

최근에는 Mask R-CNN과 같은 Instance Segmentation 기법이 건물 검출에 적용되었으나, 대부분 높이 예측 없이 단순한 검출과 분할에 그쳤다. 한편, 높이 예측을 시도한 연구들은 주로 DSM 데이터를 기반으로 UNet 등을 사용하여 픽셀 레벨의 회귀(Regression) 방식으로 높이를 추정하였다. 본 논문은 이러한 기존 연구와 달리, RGB 이미지 기반으로 Instance Segmentation을 통해 개별 건물의 높이 클래스를 예측한다는 점에서 차별점을 가진다.

## 🛠️ Methodology

### 1. 데이터 전처리 (Preprocessing)
원천 데이터셋의 노이즈를 제거하기 위해 다음과 같은 전처리 과정을 수행하였다.

- **중첩 어노테이션 해결 (Overlapping Annotations):** 고층 건물의 경우 높이 값이 다른 여러 어노테이션이 중첩되는 문제가 발생한다. 이를 해결하기 위해 단순한 $\text{IoU}(\text{Intersection Over Union})$ 대신, 교집합 영역을 각 어노테이션의 면적으로 나눈 값 중 최댓값을 기준으로 중첩 여부를 판단하고, 가장 높은 높이 값을 가진 어노테이션으로 통합하였다.
- **잘못된 어노테이션 필터링 (Misaligned and Partial Annotations):** 수만 장의 이미지를 수동으로 검수하는 것은 불가능하므로, 사전 학습된 $\text{Unet++}$ 모델을 통해 예측된 마스크와 실제 어노테이션 사이의 $\text{IoU}$를 계산하였다. 예측값과 실제값의 차이가 $50\%$ 이상인 샘플은 제거하였다.

### 2. 평가 지표 개선 (Over-Under Detections)
단순한 1:1 매칭 방식의 평가로는 실제 비즈니스 케이스에서 허용 가능한 예측을 놓칠 수 있다. 따라서 Over-detection(하나의 객체를 여러 개로 예측)과 Under-detection(여러 객체를 하나로 예측) 사례를 고려한 혼동 행렬(Confusion Matrix) 계산 방식을 도입하여 평가의 정확도를 높였다.

### 3. 학습 파이프라인 및 아키텍처
본 연구는 $\text{Mask R-CNN}$ 프레임워크를 기반으로 하며, $\text{ResNext-101}$을 Backbone 네트워크로 사용하였다. 학습은 다음의 2단계 과정을 거친다.

1. **Pre-training:** 다양한 국가의 건물이 포함된 Miyazaki의 오픈소스 데이터셋을 사용하여 단일 클래스(건물 여부) Instance Segmentation 모델을 학습시킨다. 이때 $\text{MS-COCO}$ 가중치를 시작점으로 하며, $\text{ResNext}$ 인코더의 첫 번째 레이어만 동결(Freeze)하고 나머지를 학습시킨다.
2. **Fine-tuning:** 사전 학습된 가중치를 초기값으로 하여 3개의 높이 클래스($15\text{m}$ 미만, $15\text{m}\sim40\text{m}$, $40\text{m}$ 초과)를 분류하는 학습을 수행한다. 이 단계에서는 인코더를 동결하고 나머지 레이어만 학습시킨다.

특히, 위성 이미지 내 건물의 크기가 $\text{MS-COCO}$ 데이터셋의 객체보다 매우 작다는 점을 고려하여, RPN(Region Proposal Network)의 앵커 박스 스케일(Anchor scales) 파라미터를 기본값 $[8]$에서 $[2, 4, 8, 16, 32]$로 수정하여 작은 객체에 대한 검출 성능을 높였다.

## 📊 Results

### 실험 설정
- **데이터셋:** 사전 학습을 위한 Miyazaki 공개 데이터셋(일본, 태국, 케냐, 모잠비크 4개국 이미지)과 5개 도시의 높이 정보가 포함된 프라이빗 데이터셋을 사용하였다.
- **높이 클래스:** 히스토그램 분석을 통해 클래스 균형을 맞춘 3개 클래스(up to $15\text{m}$, $15\text{m}\text{-}40\text{m}$, over $40\text{m}$)로 정의하였다.
- **평가 데이터:** 무작위로 선택된 1200장의 테스트 이미지를 사용하였다.

### 주요 결과
정량적 결과는 다음과 같다.
- **Bounding Box $\text{mAP}(\text{IoU}=0.5)$:** $59$
- **Segmentation Mask $\text{mAP}(\text{IoU}=0.5)$:** $52.6$
- **클래스별 정확도:**
    - $15\text{m}$ 미만: $73\%$
    - $15\text{m}\sim40\text{m}$: $71\%$
    - $40\text{m}$ 초과: $65\%$
- **종합 정확도:** 약 $70\%$

혼동 행렬 분석 결과, 대부분의 오류는 인접한 클래스 간의 오분류에서 발생하였다. 예를 들어, $15\text{m}$ 미만 건물 중 $8\%$가 $15\text{m}\sim40\text{m}$로 예측되었으나, $40\text{m}$ 초과로 예측된 경우는 없었다.

## 🧠 Insights & Discussion

본 연구는 Instance Segmentation을 통해 위성 이미지에서 개별 건물의 높이를 클래스 단위로 예측할 수 있음을 입증하였다. 특히 Transfer Learning을 통해 지리적 다양성을 확보함으로써 모델의 일반화 성능을 개선한 점이 강점이다. $\text{ResNext-101}$과 수정된 앵커 박스 설정을 통해 작은 건물들에 대한 검출력을 높인 점 또한 실용적인 접근이다.

다만, 단일 RGB 이미지라는 제약으로 인해 빛의 반사, 그림자, 시점의 부재 등으로 인한 한계가 존재한다. 특히 가장 높은 클래스(over $40\text{m}$)의 정확도가 가장 낮게 나타났는데, 이는 초고층 건물의 경우 위성 이미지상에서 투영되는 면적이 실제 높이를 충분히 반영하지 못하거나, 주변 건물과의 관계가 복잡하기 때문으로 해석된다. 또한, 배경을 건물로 오인하는 False Positive 사례가 $15\text{m}$ 미만 클래스에서 빈번하게 발생한다는 점은 향후 개선이 필요한 지점이다.

## 📌 TL;DR

본 논문은 단일 RGB 위성 이미지에서 $\text{Mask R-CNN}$ 기반의 Instance Segmentation을 활용하여 건물의 마스크와 높이 클래스(3단계)를 동시에 예측하는 방법론을 제안한다. 대규모 공개 데이터셋을 통한 사전 학습과 전용 데이터셋의 미세 조정을 통해 일반화 성능을 확보하였으며, 결과적으로 $\text{mAP}$ $52.6$(mask), 평균 정확도 약 $70\%$를 달성하였다. 이 연구는 DSM과 같은 고비용 데이터 없이도 효율적인 도시 높이 지도 제작 가능성을 제시하며, 향후 정밀한 도시 계획 및 통신 인프라 설계에 활용될 가능성이 높다.
