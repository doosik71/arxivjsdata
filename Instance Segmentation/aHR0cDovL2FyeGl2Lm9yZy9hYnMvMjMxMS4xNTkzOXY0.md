# Unleashing the Power of Prompt-driven Nucleus Instance Segmentation

Zhongyi Shui et al. (2024)

## 🧩 Problem to Solve

본 논문은 조직학 이미지(histology images) 내에서 핵 인스턴스 분할(nucleus instance segmentation)을 자동화하는 문제를 해결하고자 한다. 핵의 형태와 구조를 정량적으로 분석하는 것은 암 진단, 치료 계획 수립 및 생존 분석에 있어 필수적이지만, 전체 슬라이드 이미지(WSI)에 포함된 수만 개의 핵을 수동으로 분석하는 것은 매우 많은 시간이 소요되며 관찰자 간의 주관적 해석 차이가 크다는 문제가 있다.

기존의 지배적인 알고리즘들은 주로 핵 프록시 맵(nuclear proxy maps)의 회귀(regression)에 의존하는 bottom-up 방식이다. 그러나 이러한 방식은 추정된 맵으로부터 개별 핵 인스턴스를 분리하기 위해 매우 정교하게 설계된 후처리(post-processing) 과정이 필요하며, 이는 하이퍼파라미터 설정에 민감하고 노이즈에 취약하다는 단점이 있다. 또한, top-down 방식(예: Mask R-CNN)은 경계가 모호한 핵이나 심하게 겹쳐진 핵 클러스터를 처리하는 데 한계가 있으며, 고정된 해상도의 마스크를 리샘플링하는 과정에서 양자화 오차(quantization errors)가 발생하여 정밀한 경계 추출이 어렵다. 따라서 본 논문의 목표는 복잡한 후처리 없이도 겹쳐진 핵을 정확하게 분리할 수 있는 자동화된 핵 인스턴스 분할 프레임워크를 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 Segment Anything Model (SAM)의 프롬프트 가능(promptable) 특성을 활용하여, '하나의 프롬프트가 하나의 핵을 담당하는(one-prompt-one-nucleus)' 체계로 핵 분할을 수행하는 것이다. 이를 위해 다음과 같은 세 가지 주요 기여를 제시한다.

첫째, 자동 프롬프트 생성기인 Nucleus Prompter를 개발하여 추론 단계에서 사용자의 개입 없이 각 핵의 중심점을 나타내는 포인트 프롬프트를 생성한다.
둘째, Prompter의 성능을 높이기 위해 핵 영역 분할(nuclear region segmentation)이라는 보조 작업(auxiliary task)을 도입하여 모델이 전경(foreground) 영역에 집중하게 하고, 예측된 마스크를 통해 거짓 양성(false positive) 프롬프트를 필터링한다.
셋째, 인접한 핵들을 부정 프롬프트(negative prompts)로 사용하여, 경계가 모호하여 발생하는 과분할(over-segmentation) 문제를 해결하고 겹쳐진 핵들을 더 정밀하게 분리한다.

## 📎 Related Works

### SAM의 의료 영상 적용

SAM은 자연 영상에서 뛰어난 일반화 성능을 보이지만, 의료 영상과의 도메인 차이(domain gap)로 인해 성능이 저하된다. 기존 연구들은 SAM을 의료 데이터로 파인튜닝하거나, SAM의 이미지 인코더만을 백본으로 사용하여 프록시 맵 회귀 모델을 구축했다. 하지만 이는 SAM의 통합 아키텍처가 가진 잠재력을 완전히 활용하지 못하거나, 여전히 수동 프롬프트에 의존한다는 한계가 있다.

### 핵 인스턴스 분할 방법론

기존 방법론은 크게 top-down과 bottom-up으로 나뉜다. Top-down 방식은 바운딩 박스를 먼저 예측한 후 내부를 분할하는데, 핵의 심한 중첩으로 인해 최적화가 어렵고 리샘플링 오차가 발생한다. Bottom-up 방식은 거리 맵(distance map)이나 윤곽선 맵(contour map) 등을 예측한 후 후처리를 통해 인스턴스를 분리하며 정확도는 높지만, 후처리 과정의 하이퍼파라미터 튜닝이 매우 까다롭다. 제안된 PromptNucSeg는 center point를 이용해 인스턴스를 대표하는 top-down 계열에 속하지만, SAM의 프롬프트 메커니즘을 통해 복잡한 후처리 없이 인스턴스를 분리함으로써 기존 방식들의 한계를 극복한다.

## 🛠️ Methodology

### 전체 시스템 구조

PromptNucSeg는 크게 두 가지 모듈로 구성된다. **Nucleus Prompter**가 입력 이미지에서 각 핵의 중심점(point prompt)을 예측하면, **Segmentor(Fine-tuned SAM)**가 해당 프롬프트를 입력받아 개별 핵의 마스크를 생성한다.

### 1. SAM의 핵 분할 적응 (Segmentor)

SAM의 이미지 인코더 $F$, 프롬프트 인코더 $P$, 마스크 디코더 $M$ 중 프롬프트 인코더는 고정하고 이미지 인코더와 마스크 디코더를 파인튜닝한다. 훈련 시, 각 핵 인스턴스의 전경 영역에서 무작위로 포인트 프롬프트 $p_z$를 샘플링하며, 예측된 마스크 $\tilde{O}_z$는 다음과 같이 계산된다.

$$\tilde{O}_z = M(F(x), P(\{p_z\}), [mask], [IoU])$$

손실 함수 $\mathcal{L}_{sam}$은 Focal Loss(FL), Dice Loss(DL), 그리고 IoU 예측을 위한 Mean-Square-Error(MSE)의 조합으로 정의된다.

$$\mathcal{L}_{sam} = \omega FL(\tilde{O}_z, O_z) + DL(\tilde{O}_z, O_z) + MSE(\tilde{\nu}, \nu)$$

### 2. Nucleus Prompter 학습

Prompter는 이미지 위에 일정 간격 $\lambda$로 배치된 앵커 포인트 $A = \{a_i\}_{i=1}^M$를 기반으로 중심점 좌표와 카테고리를 예측한다.

- **특징 추출:** 이미지 인코더 $F'$를 통해 계층적 특징 맵 $\{P_j\}_{j=2}^L$을 생성하고, bilinear interpolation을 통해 각 앵커 포인트의 다중 스케일 특징 벡터 $\{f_{i,j}\}$를 추출한다.
- **디코딩:** 추출된 특징을 MLP 헤드에 입력하여 오프셋 $\delta_i$와 클래스 로짓 $q_i$를 예측한다.
- **매칭:** 헝가리안 알고리즘(Hungarian algorithm)을 사용하여 앵커 포인트와 실제 중심점 $b_j$ 간의 최대 가중치 매칭 $\phi$를 수행한다. 가중치 $w_{i,j}$는 다음과 같다.

$$w_{i,j} = q_i(c_j) - \alpha ||\hat{a}_i - b_j||^2$$

여기서 $\hat{a}_i = a_i + \delta_i$는 보정된 위치이다. 최종 손실 함수는 분류 손실 $\mathcal{L}_{cls}$와 회귀 손실 $\mathcal{L}_{reg}$의 합으로 구성된다.

### 3. 보조 작업 및 프롬프트 필터링

핵의 크기와 모양 정보를 학습에 반영하기 위해 핵 영역 분할(nuclear region segmentation) 보조 작업을 추가한다. Conv-BN-ReLU-Conv 구조의 마스크 헤드를 통해 핵 확률 맵 $\hat{S}$를 예측하고 Focal Loss로 학습시킨다. 추론 시에는 이 확률 맵에서 값이 0.5를 초과하는 프롬프트만 유지하여 거짓 양성(false positive)을 제거한다.

### 4. 부정 프롬프트(Negative Prompts) 활용

겹쳐진 핵의 과분할 문제를 해결하기 위해, 대상 핵(ROI) 주변의 인접 핵들을 부정 프롬프트로 지정한다.

- **추론 단계:** 대상 포인트 $\hat{p}_z$에서 가장 가까운 $K$개의 포인트를 부정 프롬프트 $\{n_{z,k}\}_{k=1}^K$로 사용한다.
- **학습 단계:** SAM이 부정 프롬프트의 효과를 잊어버리는 'catastrophic forgetting'을 방지하기 위해, 파인튜닝 단계에서부터 $K$개의 인접 포인트를 함께 입력하여 학습시킨다.

$$\tilde{O}_z = M(F(x), P(\{p_z\} \cup \{n_{z,k}\}_{k=1}^K), [mask], [IoU])$$

### 5. 추론 절차

실제 배포를 위해 SAM의 입력 해상도를 $1024 \times 1024$에서 $256 \times 256$로 축소한다. 입력 이미지가 이보다 클 경우, 위치 임베딩의 보간으로 인한 성능 저하를 막기 위해 슬라이딩 윈도우(sliding window) 방식으로 타일링하여 처리하고, 최종적으로 Non-Maximum Suppression(NMS)을 통해 중복 예측을 제거한다.

## 📊 Results

### 실험 설정

- **데이터셋:** Kumar (30장), CPM-17 (64장), PanNuke (7,899장)
- **평가 지표:** Aggregated Jaccard Index (AJI) 및 Panoptic Quality (PQ). 본 논문은 중첩 영역의 과도한 패널티 문제가 있는 AJI보다 PQ를 주요 지표로 사용한다.

### 정량적 결과

- **PanNuke:** PromptNucSeg-H는 기존 SOTA 모델들보다 bPQ에서 1.1, mPQ에서 1.4 포인트 높은 성능을 기록했다. 특히 5가지 핵 클래스 모두에서 감지(Detection) F1 스코어가 가장 높았으며, 분할(Segmentation)에서도 5개 중 4개 클래스에서 최고 PQ를 달성했다.
- **Kumar & CPM-17:** Kumar 데이터셋에서 AJI 0.622, PQ 0.627을 기록하며 2위 모델을 앞섰으며, 특히 CPM-17에서는 AJI와 PQ에서 각각 1.9, 2.8 포인트라는 상당한 격차로 최고 성능을 보였다.

### 효율성 분석

U-Net 기반의 기존 방법들은 고해상도 프록시 맵을 생성하기 위해 많은 MACs가 소모되고, CPU 집약적인 후처리가 필요하여 추론 속도가 느리다. 반면 PromptNucSeg는 저해상도 특징에서 프롬프트와 마스크를 직접 예측하므로 연산 비용이 저렴하고 추론 속도가 빠르다.

### 절제 연구 (Ablation Study)

- **모듈 효과:** 파인튜닝(FT), 보조 작업(AUX), 프롬프트 필터링(MAPF), 부정 프롬프트(NP)를 순차적으로 적용했을 때 PQ 성능이 지속적으로 향상됨을 확인했다.
- **부정 프롬프트 수:** 실제 예측된 프롬프트를 사용할 때는 $K=1$일 때 성능이 가장 좋았다. 이는 프롬프트 자체의 노이즈 때문에 $K$가 커지면 오히려 성능이 저하될 수 있음을 시사한다. 다만, 정답(GT) 중심점을 사용하는 'Oracle' 실험에서는 $K$가 증가할수록 성능이 계속 향상되어, 시스템의 병목 지점이 프롬프트의 품질에 있음을 입증했다.

## 🧠 Insights & Discussion

본 논문은 SAM이라는 강력한 범용 모델을 핵 인스턴스 분할이라는 특수 도메인에 맞게 성공적으로 적응시켰다. 특히 주목할 점은 **부정 프롬프트의 학습 시점**에 대한 통찰이다. 단순히 추론 단계에서만 부정 프롬프트를 추가하는 것은 효과가 없었으며, 이는 긍정 프롬프트로만 파인튜닝한 모델이 부정 프롬프트의 억제 기능을 잊어버리는 'catastrophic forgetting' 현상 때문임을 밝혀내고, 이를 학습 단계에 통합함으로써 해결했다.

또한, Prompter가 예측하는 포인트의 정밀도가 전체 시스템 성능을 결정짓는 핵심 요소임을 확인했다. 이는 향후 더 적은 비용의 포인트 라벨링 데이터를 통해 Prompter의 정확도를 높인다면, 추가적인 마스크 주석 없이도 분할 성능을 개선할 수 있는 가능성을 제시한다.

한계점으로는 SAM 기반 모델의 특성상 파라미터 수가 많아 모델 크기가 크다는 점이 언급되었다. 이는 저장 공간 및 전송 비용을 증가시키므로, 향후 모델 가지치기(pruning)나 더 경량화된 SAM 변형 모델(예: Semantic-SAM)의 적용이 필요할 것으로 보인다.

## 📌 TL;DR

PromptNucSeg는 자동 프롬프트 생성기(Prompter)와 파인튜닝된 SAM(Segmentor)을 결합하여, 복잡한 후처리 없이 핵 인스턴스를 분할하는 프레임워크이다. 특히 인접한 핵을 부정 프롬프트로 활용하여 겹쳐진 핵 분리 문제를 해결했으며, PanNuke, Kumar, CPM-17 등 3개 벤치마크에서 SOTA 성능을 달성했다. 이 연구는 프롬프트 기반의 접근 방식이 의료 영상의 인스턴스 분할에서 기존의 맵 회귀 방식보다 효율적이고 정확할 수 있음을 입증했다.
