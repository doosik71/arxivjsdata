# Evaluation Study on SAM 2 for Class-agnostic Instance-level Segmentation

Jialun Pei, Zhangjun Zhou, Tiantian Zhang (2024)

## 🧩 Problem to Solve

본 논문은 클래스 정보가 없는 상태에서 인스턴스 단위의 분할을 수행하는 Class-agnostic Instance-level Segmentation 작업에서 최신 기반 모델인 Segment Anything Model 2 (SAM 2)의 제로샷(zero-shot) 성능을 평가하는 것을 목표로 한다.

인스턴스 수준의 분할은 단순히 픽셀을 분류하는 것을 넘어 개별 객체를 식별하고 정교하게 분리해야 하므로 매우 도전적인 작업이다. 특히, 배경과 객체의 구분이 어려운 위장 객체나 그림자, 그리고 매우 세밀한 구조를 가진 객체들의 경우 기존 모델들이 한계를 보였다. SAM 2가 이미지 및 비디오 분할에서 뛰어난 성능을 보였음에도 불구하고, 이러한 특수한 인스턴스 분할 시나리오에서 어느 정도의 일반화 성능을 가지며, 프롬프트 전략에 따라 결과가 어떻게 달라지는지를 정량적·정성적으로 분석할 필요가 있다.

## ✨ Key Contributions

본 연구의 핵심적인 기여는 SAM 2를 네 가지 서로 다른 클래스-불가지론적 인스턴스 분할 시나리오에 적용하여 그 한계와 가능성을 체계적으로 분석했다는 점이다.

1. **다양한 시나리오 평가**: Salient Instance Segmentation (SIS), Camouflaged Instance Segmentation (CIS), Shadow Instance Detection (SID), 그리고 Dichotomous Image Segmentation (DIS)이라는 네 가지 상이한 환경에서 SAM 2의 성능을 검증하였다.
2. **프롬프트 전략 비교**: 자동 모드(Automatic mode)와 정답 바운딩 박스를 사용하는 모드(GT-Bbox mode)를 비교하여, SAM 2가 '객체 위치 찾기(Localization)'와 '마스크 생성(Segmentation)' 중 어느 단계에서 강점과 약점을 가지는지 분석하였다.
3. **세밀한 구조 분석**: 고해상도 DIS 벤치마크를 통해 SAM 2가 아주 세밀한 객체 구조(granular object structures)를 얼마나 정교하게 묘사할 수 있는지 평가하였다.
4. **모델 규모 및 효율성 검토**: SAM 2의 서로 다른 백본(Hiera-Tiny, Base+, Large)과 기존 SAM의 모델 크기별 성능을 비교하여 파라미터 수와 성능 간의 관계를 고찰하였다.

## 📎 Related Works

논문은 기반 모델의 발전, 특히 SAM과 SAM 2의 등장이 이미지 분할 분야에 가져온 혁신을 언급한다. SAM은 포인트나 바운딩 박스 같은 사용자 프롬프트를 통해 높은 정확도의 마스크를 생성하며, SAM 2는 이를 확장하여 비디오 콘텐츠 처리 능력을 개선하고 추론 효율성을 높였다.

기존의 관련 연구들은 SAM 2를 수중 환경, 내시경 및 현미경 이미지, 의료 이미지 등 특정 도메인에 적용하여 평가한 바 있다. 특히 위장 객체 탐지(Camouflaged Object Detection)에 관한 선행 연구에서는, 프롬프트가 없을 때 SAM 2가 SAM보다 성능이 저하되지만, 분할 프롬프트가 제공될 때는 SAM보다 성능이 크게 향상된다는 흥미로운 결과가 보고되었다. 본 논문은 이러한 기존 발견을 바탕으로, 인스턴스 수준의 분할 작업에서도 유사한 경향이 나타나는지 확장하여 분석하고자 한다.

## 🛠️ Methodology

### 전체 시스템 구조 및 설정

본 연구는 SAM 2의 아키텍처 자체를 수정하기보다는, 제공된 공식 코드를 사용하여 다양한 벤치마크 데이터셋에서 제로샷 성능을 측정하는 평가 연구의 성격을 띤다. 모든 입력 이미지는 $1024 \times 1024$ 크기로 리사이즈되어 입력된다.

### 프롬프트 전략 (Prompt Strategies)

SAM 2의 성능을 평가하기 위해 두 가지 실행 모드를 설정하였다.

- **Automatic Mode**: 별도의 외부 가이드 없이 $32 \times 32$ 그리드 형태의 포인트 프롬프트를 기본 설정으로 사용하여 객체를 자동으로 탐색하고 분할한다.
- **GT-Bbox Mode**: 정답(Ground Truth) 바운딩 박스를 프롬프트로 입력하여, 모델이 객체의 위치를 이미 알고 있다는 가정하에 마스크 생성 능력만을 평가한다.

### 분석 대상 모델 및 백본

- **SAM**: ViT-Base, ViT-Large, ViT-Huge 백본을 사용한다.
- **SAM 2**: Hiera-Tiny, Hiera-Base+, Hiera-Large 백본을 사용한다.

### 평가 지표

작업의 특성에 따라 서로 다른 지표를 사용한다.

- **CIS 및 SIS**: $\text{AP}$, $\text{AP}_{50}$, $\text{AP}_{75}$, $\text{AP}_{70}$ 등 COCO 스타일의 정밀도 지표를 사용한다.
- **SID**: 인스턴스 $\text{AP}$ 지표에 집중한다 (SAM 2에 객체-그림자 매칭 메커니즘이 없으므로 SOAP 지표는 제외하였다).
- **DIS**: $\text{F}_{\max}^{\beta}$ (최대 F-measure), $\text{F}_{\omega}^{\beta}$ (가중 F-measure), $\text{MAE}$ (평균 절대 오차), $\text{S}^{\alpha}$ (구조적 측정치), $\text{E}_{\phi}^m$ (강화된 정렬 측정치), $\text{HCE}^{\gamma}$ (인간 수정 노력) 등 6가지 정밀 지표를 사용한다.

*참고: 본 논문은 평가 보고서이므로 새로운 손실 함수나 아키텍처 변경 사항에 대한 방정식은 제시되지 않았다.*

## 📊 Results

### 1. Salient Instance Segmentation (SIS)

- **정량적 결과**: GT-Bbox 설정에서 SAM 2는 SAM 및 기존 전용 모델들보다 우수한 성능을 보였다. 예를 들어, ILSO 데이터셋에서 SAM 2-L은 $\text{AP} 82.2$를 기록하여 SAM-H($79.2$)를 앞섰다. 하지만 자동 모드에서는 SAM 2-L($49.1$)이 SAM-H($72.2$)보다 현저히 낮은 성능을 보였다.
- **정성적 결과**: 자동 모드에서는 전역적인 분할을 수행하여 결과가 파편화되는 경향이 있으나, 바운딩 박스가 제공되면 매우 정밀한 분할이 가능해진다.

### 2. Camouflaged Instance Segmentation (CIS)

- **정량적 결과**: 자동 모드에서는 SAM 2가 SAM이나 전용 모델에 비해 성능이 낮았다. 그러나 GT-Bbox 모드에서는 성능이 비약적으로 상승하여, COD10K 테스트 셋에서 SAM 2-L의 $\text{AP}$가 $10.6$에서 $68.8$로 급증하며 모든 모델 중 최고 성능을 기록하였다.
- **분석**: 이는 SAM 2의 핵심 문제가 '위장된 객체를 찾는 것(Localization)'에 있으며, 위치만 지정되면 매우 정교한 마스크를 생성할 수 있음을 시사한다.

### 3. Shadow Instance Detection (SID)

- **정량적 결과**: GT-Bbox 설정에서 SAM 2-T가 SOBA-challenge($\text{AP} 51.9$)와 SOBA-test($\text{AP} 58.9$)에서 기존 모델들을 압도하였다.
- **특이점**: 백본의 크기를 키우는 것이 반드시 성능 향상으로 이어지지 않았으며, 오히려 큰 백본이 성능을 소폭 저하시키는 경향이 관찰되었다.
- **한계**: 자동 모드에서는 인스턴스는 잘 찾지만 그림자를 정확히 식별하는 데 어려움을 겪는다. 이는 객체-그림자 간의 $\text{IoU}$ 매칭 연산이 부재하기 때문으로 분석된다.

### 4. Dichotomous Image Segmentation (DIS)

- **정량적 결과**: 자동 모드에서 SAM 2는 SAM보다 월등히 높은 성능을 보였으며(예: DIS-VD에서 $\text{F}_{\max}^{\beta}$ 기준 SAM 2-B+ $0.428$ vs SAM-H $0.283$), GT-Bbox 설정에서도 전반적으로 SAM을 앞섰다.
- **한계**: 하지만 전용 지도학습 모델(IS-Net 등)과 비교하면 $\text{HCE}$ 지표가 높게 나타났다. 이는 SAM 2가 객체의 전반적인 영역은 잘 잡지만, 매우 복잡하고 세밀한 구조적 디테일을 묘사하는 능력은 여전히 부족함을 의미한다.

## 🧠 Insights & Discussion

본 연구를 통해 도출된 주요 통찰은 다음과 같다.

첫째, **Localization vs. Segmentation**. SAM 2는 특히 CIS와 같은 난이도 높은 작업에서 객체의 위치를 찾는 능력(Localization)은 부족하지만, 위치가 주어졌을 때의 분할 능력(Segmentation)은 매우 강력하다. 이는 SAM 2가 강력한 마스크 생성기(Mask Generator)로서의 잠재력이 높음을 의미한다.

둘째, **모델 크기의 역설**. 일반적인 딥러닝 모델과 달리, SAM 2는 모든 작업에서 백본의 크기가 커진다고 해서 성능이 비례하여 상승하지 않았다. 특히 자동 모드에서는 오히려 성능이 저하되는 경우가 발견되었는데, 이는 단순한 파라미터 증량보다는 태스크 특화된 어댑터(Adapter)나 튜닝이 필요함을 시사한다.

셋째, **세밀한 구조 표현의 한계**. DIS 실험 결과, SAM 2는 객체의 주된 영역을 인식하는 데는 능숙하지만, 아주 얇은 선이나 복잡한 미세 구조를 정확히 렌더링하는 것에는 한계가 있다. 이는 향후 기반 모델의 고도화 방향이 단순한 일반화 성능을 넘어 '세밀한 구조적 정확도'를 높이는 방향으로 가야 함을 보여준다.

## 📌 TL;DR

본 논문은 SAM 2를 네 가지 클래스-불가지론적 인스턴스 분할 작업(SIS, CIS, SID, DIS)에 적용하여 제로샷 성능을 평가하였다. 실험 결과, SAM 2는 **바운딩 박스 프롬프트가 제공될 때 기존 전용 모델들을 능가하는 강력한 분할 성능**을 보였으나, 프롬프트 없는 자동 모드에서는 객체 탐지(Localization) 능력의 한계로 인해 성능이 크게 저하되었다. 또한, 매우 세밀한 구조의 객체를 분할하는 능력은 여전히 부족함이 확인되었다. 이 연구는 SAM 2를 기반으로 한 **태스크 특화 어댑터 개발**의 필요성을 강조하며, 향후 기반 모델을 인스턴스 분할 작업에 최적화하기 위한 중요한 기초 데이터를 제공한다.
