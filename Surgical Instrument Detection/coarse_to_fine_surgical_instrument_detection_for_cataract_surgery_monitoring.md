# Coarse-to-fine Surgical Instrument Detection for Cataract Surgery Monitoring

Hassan Al Hajj, Gwenol Quellec, Mathieu Lamard, Guy Cazuguel, Béatrice Cochener (2016)

## 🧩 Problem to Solve

본 논문은 백내장 수술(Cataract Surgery)의 실시간 자동 분석 성능을 향상시키기 위해 수술 도구 검출(Surgical Instrument Detection) 문제를 해결하고자 한다. 수술 중 발생하는 비디오 데이터를 분석하여 어떤 도구가 사용되고 있는지 파악하는 것은 수술 보고서 생성, 외과 의사의 숙련도 평가, 그리고 실시간 경고/추천 시스템 구축을 위해 매우 중요하다.

그러나 기존의 현미경 비디오(Microscope video)만을 이용한 분석은 다음과 같은 한계가 있다. 첫째, 수술 도구들의 형태가 매우 다양하며 수술 장면 내에서 부분적으로 가려지는(Partial occlusion) 경우가 많다. 둘째, 많은 도구가 서로 매우 유사한 외형을 가지고 있어 식별이 어렵다. 본 논문의 목표는 현미경 뷰 외에 수술대(Operating table)를 촬영하는 보조 비디오 스트림을 추가하여, 어떤 도구가 수술대에 진입하거나 나갔는지를 파악함으로써 현재 사용 중인 도구를 더욱 정확하게 추정하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 수술대의 상태를 **정적 상태 기술(Static state description)**과 **동적 변화 기술(Dynamic change description)**의 두 단계로 나누어 분석하는 것이다.

1. **Strongly Supervised k-NN Regression**: 각 도구에 대해 개별 모델을 정의하는 대신, 패치(Patch) 단위의 특징 벡터를 추출하고 k-최근접 이웃(k-NN) 알고리즘을 통해 도구 존재 확률을 계산하는 일반적인 지도 학습 방식을 제안한다.
2. **Coarse-to-fine Search Strategy**: 계산 효율성을 높이기 위해 큰 패치에서 시작하여 도구 존재 확률이 일정 수준 이상인 경우에만 패치를 세분화하는 계층적 탐색 전략을 도입하였다.
3. **Motion Compensation for Dynamic Detection**: 도구가 단순히 위치만 이동한 경우와 실제로 수술 영역으로 사라지거나 나타난 경우를 구분하기 위해, '이전' 이미지와 '이후' 이미지 사이의 최적 매칭 패치를 찾는 보상 기법을 적용하였다.

## 📎 Related Works

수술 도구 검출을 위한 기존 연구는 크게 두 가지 범주로 나뉜다.

1. **RFID 기반 식별**: 무선 주파수 식별 방식을 사용하지만, 수술실 내에 추가 장비를 설치해야 하므로 수술 절차에 영향을 줄 수 있다는 단점이 있다.
2. **영상 처리 기반 식별**: 추가 장비 설치가 필요 없다는 장점이 있으나, 대부분의 기존 연구는 형태가 매우 뚜렷하게 구분되는 소수의 도구만을 대상으로 하였다.

본 연구는 이와 달리 매우 유사하게 생긴 다수의 도구를 다루며, 특히 현미경 뷰의 폐색 문제를 해결하기 위해 수술대 뷰라는 추가적인 데이터 소스를 활용한다는 점에서 기존 접근 방식과 차별화된다.

## 🛠️ Methodology

전체 시스템은 수술 시작 시점의 도구들을 검출하는 정적 검출 단계와 수술 중 도구의 변화를 추적하는 동적 검출 단계로 구성된다.

### 1. 정적 도구 검출 (Static Instrument Detection)

수술 시작 시점에 수술대 위에 놓인 도구들을 배경 및 무관한 객체로부터 분리하는 작업이다.

- **특징 추출 (Patch Description)**: 이미지를 패치 단위로 나누고, 각 패치에서 $R, G, B, H, S, V$ 채널의 평균(Mean)과 표준편차(Standard deviation), 그리고 휘도(Luminance) 채널에 Sobel 엣지 검출을 적용한 결과의 평균과 표준편차를 추출한다. 총 14차원의 특징 벡터가 생성된다.
- **k-NN 회귀**: 테스트 패치와 가장 유사한 $k$개의 참조 패치를 찾는다. 각 참조 패치는 전문가가 수동으로 세그멘테이션한 도구 픽셀의 비율(Instrument probability)을 가지고 있으며, 테스트 패치의 확률은 이 $k$개 이웃의 평균값으로 결정된다.
- **Coarse-to-fine 전략**: 계산 속도를 높이기 위해 큰 패치부터 분석하며, 도구 확률이 $0\%$보다 큰 경우에만 해당 영역을 더 작은 패치로 분할하여 정밀하게 분석한다.
- **파라미터 최적화**: $k$, 최소 패치 크기($P_{\min}$), 스케일 인자($\tau$), 스케일 레벨 수($P\text{-levels}$)를 결정하기 위해 이산형 입자 군집 최적화(D-PSO) 알고리즘을 사용하였다.

### 2. 동적 도구 검출 (Dynamic Instrument Detection)

수술 중 움직임이 감지된 시점의 '전(Before)' 이미지와 '후(After)' 이미지를 비교하여 도구의 출현(Appearance)과 소멸(Disappearance)을 검출한다.

- **움직임 보상 (Compensating For Instrument Motions)**: 도구가 단순히 테이블 위에서 이동한 것을 출현/소멸로 오인하는 것을 방지하기 위해, '후' 이미지의 패치 $P_1$에 대해 '전' 이미지의 일정 윈도우 영역 내에서 유클리드 거리가 가장 가까운 특징 벡터를 가진 패치 $P_2$를 찾는다.
- **변화 기술 (Change Description)**: 두 패치 간의 특징 벡터 차이 $(V_2 - V_1)$를 계산한다. 도구가 실제로 나타난 경우 매칭되는 패치를 찾을 수 없거나 차이가 매우 크게 나타난다.
- **분석 절차**: 정적 검출과 동일하게 k-NN 회귀와 Coarse-to-fine 전략을 사용하며, 추가적으로 탐색 윈도우 크기($W\text{-size}$) 파라미터를 최적화한다.

## 📊 Results

### 실험 설정

- **데이터셋**: 2명의 외과 의사가 집도한 36건의 백내장 수술 비디오(수술대 뷰 및 현미경 뷰)를 사용하였다.
- **Ground Truth**: 전문가가 수술 시작 프레임(정적)과 특정 도구 이동 액션 전후 프레임(동적)을 수동으로 세그멘테이션하였다.
- **평가 지표**: ROC 곡선의 아래 면적인 $A_z$ (Area under the ROC curve)를 사용하여 성능을 측정하였다.

### 결과

- **정적 검출**: D-PSO로 최적화된 파라미터를 적용한 결과, $A_z$ 평균 $0.982$ ($\text{std}=0.015$)라는 높은 성능을 보였다. 배경의 타월과 같은 무관한 객체는 제외하고, 전문가가 정의한 도구 및 용기들을 정확하게 분리해냈다.
- **동적 검출**: 그리드 탐색(Grid search)을 통해 $W\text{-size}$를 최적화하였으며, $A_z$ 평균 $0.947$ ($\text{std}=0.045$)를 기록하였다. 도구가 수술대에서 사라지거나 나타나는 상황을 효과적으로 식별하였다.

## 🧠 Insights & Discussion

본 연구는 k-NN 기반의 단순한 특징 추출과 Coarse-to-fine 전략만으로도 수술대 위의 도구를 매우 높은 정확도로 검출할 수 있음을 입증하였다. 특히, 복잡한 딥러닝 모델 없이도 도메인 특화된 특징(RGB, HSV, Sobel)과 효율적인 탐색 전략을 통해 실시간 분석 가능성을 보여준 점이 강점이다.

하지만 다음과 같은 한계점이 존재한다.

1. **시점 변화에 취약**: '전' 이미지와 '후' 이미지에서 도구가 보이는 각도가 크게 다를 경우, 특징 벡터의 차이가 커져 이를 새로운 도구의 출현이나 소멸로 오인하는 경우가 발생한다.
2. **자동화 미비**: 현재는 분석할 '전/후' 이미지를 수동으로 선택하고 있으나, 실제 시스템으로 작동하려면 움직임 감지 및 이미지 선택 과정의 완전한 자동화가 필요하다.
3. **식별 단계의 부재**: 도구의 '존재' 여부(Detection)는 파악했으나, 그것이 정확히 '어떤 도구'인지(Recognition)에 대한 분류 단계는 구현되지 않았다.

## 📌 TL;DR

이 논문은 백내장 수술 모니터링을 위해 수술대 비디오를 활용한 **Coarse-to-fine k-NN 기반 도구 검출 방법**을 제안한다. 정적 상태와 동적 변화를 구분하여 분석하고 움직임 보상 기법을 통해 단순 이동과 실제 출현/소멸을 구분함으로써, 높은 정확도($A_z \approx 0.95 \sim 0.98$)로 도구를 검출하였다. 이 연구는 향후 수술 도구의 자동 인식 및 수술 단계 추적 시스템의 기초 데이터 제공원으로 활용될 가능성이 높다.
