# Automatic Verification of Parameterised Interleaved Multi-Agent Systems

Panagiotis Kouvaros, Alessio Lomuscio (2013)

## 🧩 Problem to Solve

본 논문은 다중 에이전트 시스템(Multi-Agent Systems, MAS)의 검증 과정에서 발생하는 **상태 공간 폭발(State-space explosion)** 문제를 해결하고자 한다. 특히, 시스템에 포함된 에이전트의 수가 증가함에 따라 상태 공간이 지수적으로 증가하여, 소수의 에이전트만 포함된 시스템조차 실질적인 모델 체킹(Model Checking)이 불가능해지는 경우가 많다.

더욱이, 실제 응용 분야(로보틱스, 군집 지능 등)에서는 에이전트의 수가 사전에 정해지지 않은 **매개변수화된 시스템(Parameterised Systems)**을 다루어야 하는 경우가 많다. 임의의 에이전트 수 $n$에 대해 특정 성질이 유지되는지를 확인하려면 이론적으로 무한히 많은 시스템 인스턴스를 검사해야 하므로, 이는 일반적인 모델 체킹 기법으로는 해결할 수 없는 문제이다. 따라서 본 논문의 목표는 임의의 에이전트 수에 대해 성질을 검증할 수 있도록 하는 **컷오프(Cutoff)** 결과—즉, 특정 수 $c$ 이하의 에이전트만 검사하면 모든 $n \ge c$에 대해 성질을 보장할 수 있는 임계값—를 도출하는 기법을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **매개변수화된 인터리브 해석 시스템(Parameterised Interleaved Interpreted Systems, PIIS)**이라는 특수한 클래스와, 이에 대응하는 **인덱스 기반 시제-인식 논리(Indexed Temporal-Epistemic Logic, $\text{IACTL}^*_{K-X}$)**를 정의하여 컷오프를 자동으로 결정하는 것이다.

중심적인 설계 아이디어는 다음과 같다.
1. **Stuttering-insensitive Logic 사용**: 'Next' 연산자($X$)를 제거한 $\text{CTL}^*$의 유니버설 프래그먼트를 사용하여, 상태의 단순 반복(stuttering)에 영향을 받지 않는 논리를 구축함으로써 매개변수화된 검증 문제의 결정 불가능성(undecidability)을 회피한다.
2. **대칭성 및 시뮬레이션 활용**: 동일한 템플릿 에이전트로 구성된 시스템의 대칭성을 이용하여, 모든 에이전트 조합에 대한 검증을 단일 인스턴스 검증으로 축소한다.
3. **최소 컷오프의 도출**: 특정 성질 $\wedge_J \phi(J)$에 대해, 컷오프 값 $c$가 해당 공식에 나타나는 고유 인덱스의 수 $|J|$와 정확히 일치함을 이론적으로 증명한다.

## 📎 Related Works

기존의 매개변수화된 검증 연구는 주로 반응형 시스템(Reactive Systems)과 단순 시제 논리에 국한되어 있었다. 
- **기존 접근 방식의 한계**: 기존의 컷오프 도출 방법론들은 (1) 도출된 컷오프가 최소값임을 보장하지 못하거나, (2) 컷오프의 존재 자체가 보장되지 않아 불완전한 경우가 많았다. 또한, 링(Ring) 토폴로지와 같은 매우 제한적인 구조에서만 작동하는 한계가 있었다.
- **본 연구의 차별점**: 본 논문은 인식 논리(Epistemic Logic)를 통합하여 에이전트의 '지식'에 대한 성질을 다룰 수 있게 하였으며, PIIS 환경에서 **건전성(Soundness)**과 **완전성(Completeness)**을 동시에 갖춘 최소 컷오프 도출 방법을 제시하였다.

## 🛠️ Methodology

### 1. Parameterised Interleaved Interpreted Systems (PIIS)
시스템은 동일한 **템플릿 에이전트(Template Agent)** $T = \langle L, \iota, Act, P, t, h \rangle$의 병렬 조합으로 정의된다.
- **인터리브 의미론(Interleaved Semantics)**: 한 번에 하나의 로컬 액션만 수행된다. 단, **동기식 액션(Synchronous Action, $Act^S$)**이 수행될 때는 모든 에이전트가 동시에 동일한 액션을 수행한다.
- **전역 상태(Global State)**: $n$개 에이전트의 로컬 상태들의 튜플 $g = (l_1, \dots, l_n)$로 표현된다.

### 2. 논리 체계: $\text{IACTL}^*_{K-X}$
본 논문은 인덱스 기반의 시제-인식 논리를 도입한다.
- **구문(Syntax)**: 표준 $\text{CTL}^*$에서 $X$(next) 연산자를 제거하고, 에이전트 인덱스 $v \in VS$를 가진 원자 명제 $p_v$와 인식 모달리티 $K_v$를 추가한다.
- **보편 양화자 $\wedge_J$**: $\wedge_J \phi(J)$는 $|J|$-튜플의 모든 서로 다른 에이전트 조합에 대해 $\phi$가 성립함을 의미한다.
- **의미론(Semantics)**: $g \models K_i \phi$는 에이전트 $i$의 로컬 상태가 동일한 모든 전역 상태 $g'$에서 $\phi$가 성립함을 의미한다.

### 3. 컷오프 도출 및 검증 절차
핵심 이론은 **$J$-Stuttering Simulation**을 통해 증명된다. 두 모델이 $J$-stuttering simulation equivalent 하면, $\text{ACTL}^*_{K J-X}$ 공식에 대해 동일한 진릿값을 가진다.

**주요 정리 (Theorem 3.3):**
성질 $\psi = \wedge_J \phi(J) \in \text{IACTL}^*_{K-X}$에 대하여, 모든 $n \ge |J|$인 시스템 $T(n)$에서 $\psi$가 성립할 필요충분조건은 컷오프 인스턴스 $T(|J|)$에서 $\psi$가 성립하는 것이다.
$$\forall n \ge |J| : T(n) \models \psi \iff T(|J|) \models \psi$$

**검증 알고리즘 흐름:**
1. 검증하려는 공식 $\wedge_J \phi(J)$에서 사용된 고유 인덱스의 수 $|J|$를 계산한다.
2. $|J|$개의 에이전트로 구성된 단일 시스템 인스턴스 $T(|J|)$를 생성한다.
3. 대칭성 축소(Symmetry Reduction)를 통해 $\wedge_J \phi(J)$를 단일 인스턴스 공식 $\phi(\{1, \dots, |J|\})$로 변환한다.
4. BDD(Binary Decision Diagram) 기반의 모델 체커를 사용하여 $T(|J|) \models \phi$를 검증한다.

## 📊 Results

### 1. 실험 설정 및 사례 연구
**Autonomous Robot (AR) 예시**를 통해 이론을 검증하였다.
- **시나리오**: 로봇들이 트랙을 이동하며 센서를 통해 위치를 측정하고 정지(halt)하는 시스템.
- **검증 공식**:
    - $\phi_{AR1} = \wedge_{\{i,j\}} AG(h_i \to K_i g_j)$: "에이전트 $i$가 정지하면, 에이전트 $i$는 다른 모든 에이전트 $j$가 목표 영역($g$) 내에 있음을 안다."
    - $\phi_{AR2}, \phi_{AR3}$: 개별 로봇의 안전성 및 정지 조건 검증.

### 2. 정량적 결과
구현체인 `mcmas-p`를 기존 `mcmas`와 비교 분석하였다. (표 1 결과 요약)
- **상태 공간 증가**: 에이전트 수 $n$이 증가함에 따라 상태 공간이 지수적으로 증가한다. 예를 들어 $n=30$일 때 상태 수가 약 $9.2 \times 10^{27}$에 달한다.
- **검증 시간**: 일반 모델 체킹으로는 $n=90$인 경우 모델 구축 단계에서 타임아웃이 발생한다.
- **컷오프 적용 결과**: `mcmas-p`는 $|J|=2$인 공식 $\phi_{AR1}$에 대해 단 2개의 에이전트만 가진 시스템을 검증함으로써, **0.1초 미만의 시간** 내에 임의의 $n$개 에이전트에 대한 성질을 확정 지었다.

## 🧠 Insights & Discussion

### 강점
- **결정 가능성(Decidability)**: 일반적인 매개변수화된 검증은 결정 불가능하지만, 본 논문은 특정 논리 제약($X$ 제거)과 시스템 구조(PIIS)를 통해 결정 가능한 영역을 성공적으로 정의하였다.
- **최소 컷오프 보장**: 단순한 인덱스 카운팅만으로 최소 컷오프를 찾을 수 있음을 증명하여 실용성을 극대화하였다.

### 한계 및 비판적 해석
- **제한된 모델링 능력**: PIIS 구조상 에이전트들이 환경의 액션에 따라 서로 다르게 진화하는 것을 표현하기 어렵다. 이는 다양한 네트워크 토폴로지를 모델링하는 데 제약이 된다.
- **논리적 제약**: 'Next' 연산자를 사용할 수 없으므로, 정확한 단계(step)를 세어야 하는 성질은 검증할 수 없다.

### 논의사항
본 연구는 에이전트의 '지식'을 다루는 인식 논리를 매개변수화된 검증에 성공적으로 통합했다는 점에서 가치가 크다. 향후 연구에서는 환경과의 상호작용 모델을 확장하여 더 복잡한 토폴로지에서도 컷오프를 도출할 수 있는 방법론이 필요할 것으로 보인다.

## 📌 TL;DR

본 논문은 에이전트 수가 무한히 확장 가능한 다중 에이전트 시스템(MAS)의 검증 문제를 해결하기 위해, **최소 컷오프(Smallest Cutoff)**를 자동으로 결정하는 기법을 제안하였다. $\text{IACTL}^*_{K-X}$ 논리와 PIIS 의미론을 통해, **공식에 등장하는 인덱스의 수만큼의 에이전트만 검증하면 모든 경우의 수를 보장할 수 있음**을 이론적으로 증명하고 구현하였다. 이는 지수적으로 증가하는 상태 공간 문제를 해결하여, 무한한 수의 에이전트를 가진 시스템의 성질을 상수 시간(특정 컷오프 내)에 검증할 수 있게 한다.
