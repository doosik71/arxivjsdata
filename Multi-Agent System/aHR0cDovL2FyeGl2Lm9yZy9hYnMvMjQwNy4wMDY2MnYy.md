# Multi-Agent Training for Pommerman: Curriculum Learning and Population-based Self-Play Approach

Nhat-Minh Huynh, Hoang-Giang Cao, I-Chen Wu (2025)

## 🧩 Problem to Solve

본 논문은 다중 에이전트 환경인 Pommerman에서 에이전트를 효율적으로 학습시키기 위한 방법론을 제시한다. Pommerman은 2vs2 팀 모드 또는 개인전(Free-For-All) 모드로 진행되는 전략 게임으로, 모델 프리 강화학습(Model-free Reinforcement Learning)을 적용할 때 다음과 같은 치명적인 문제점들이 존재한다.

첫째, **희소 보상(Sparse Rewards)** 문제이다. 게임의 최종 승패 결과는 최대 800 타임스텝이 지난 후에나 주어지므로, 에이전트가 어떤 행동이 승리로 이어졌는지 학습하기 매우 어렵다.
둘째, **거짓 양성(False Positives)** 보상 문제이다. 에이전트가 뛰어난 전략을 구사해서가 아니라, 상대방이 스스로 폭탄에 걸려 자살하는 등의 실수를 저질러 승리하는 경우가 발생하며 이는 잘못된 학습 방향을 유도할 수 있다.
셋째, **지연된 행동 효과(Delayed Action Effects)**이다. 폭탄을 설치한 후 실제 폭발까지 10타임스텝의 지연 시간이 존재하므로, 행동과 결과 사이의 인과관계를 파악하는 것이 어렵다.
넷째, **부분 관측 가능성(Partial Observability)**이다. 에이전트의 시야가 주변 $9 \times 9$ 그리드로 제한되어 있어 맵 전체를 탐색하고 적을 찾는 것이 어렵다.

따라서 본 연구의 목표는 Curriculum Learning과 Population-based Self-play를 결합하여 이러한 제약 사항들을 극복하고, 경쟁 환경에서 강력한 성능을 내는 다중 에이전트 시스템을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 단순한 학습 알고리즘 적용을 넘어, 학습의 단계적 구성과 동적인 보상 조절, 그리고 효율적인 매칭 시스템을 설계한 점에 있다.

1. **2단계 학습 파이프라인 설계**: 기초 기술을 습득하는 Curriculum Learning 단계와 전략을 고도화하는 Population-based Self-play 단계를 순차적으로 적용하였다.
2. **적응형 어닐링 계수(Adaptive Annealing Factor)** 제안: 에이전트의 현재 성능에 따라 탐색 보상(Exploration Reward)과 게임 보상(Game Reward)의 비중을 동적으로 조절하는 메커니즘을 도입하여, 학습 초기에는 탐색을 장려하고 후기에는 승리 전략에 집중하게 하였다.
3. **Elo 레이팅 기반 매칭 시스템**: Population-based Self-play 단계에서 에이전트 간의 실력 차이를 고려하여 적절한 상대를 매칭함으로써, 학습의 정체를 방지하고 점진적인 성능 향상을 유도하였다.
4. **비통신 에이전트의 성능 증명**: 팀원 간의 통신(Communication) 기능을 사용하지 않고도 기존의 상위 학습 에이전트 및 룰 기반 에이전트보다 우수한 성능을 보임을 입증하였다.

## 📎 Related Works

논문에서는 기존의 다중 에이전트 경쟁 학습 연구들을 언급하며, 특히 emergent complexity를 다룬 연구[3]와 Pommerman의 하드 탐색 문제를 다룬 연구[2]를 참고하였다. 

기존의 접근 방식 중 하나는 선형 어닐링(Linear Annealing)을 사용하여 탐색 보상을 점진적으로 줄이는 방법이다. 하지만 이는 하이퍼파라미터에 대한 수동 튜닝이 많이 필요하며, 에이전트의 실제 학습 속도나 성능 변화를 반영하지 못한다는 한계가 있다. 본 논문은 이러한 정적인 방식 대신 에이전트의 성능 지표($x$)를 직접 반영하는 적응형 함수를 사용하여 이 문제를 해결하고자 하였다.

## 🛠️ Methodology

### 1. 전체 학습 파이프라인
학습은 **Curriculum Learning $\rightarrow$ Population-based Self-play** 순으로 진행된다.

#### (1) Curriculum Learning Stage
에이전트가 게임에 필요한 필수 기술을 단계적으로 습득하도록 세 가지 난이도의 룰 기반 에이전트와 대결하게 한다.
- **Phase 1 (Static Agent)**: 움직이지 않는 적과 대결하며 맵 탐색, 아이템 획득, 벽 파괴 및 기초적인 폭탄 설치를 학습한다.
- **Phase 2 (Simple Moving Agent)**: 랜덤하게 움직이지만 폭탄은 설치하지 않는 적과 대결하며, 더 효율적인 폭탄 배치 전략을 학습한다.
- **Phase 3 (Simple Bomb Agent)**: 이동과 폭탄 설치를 모두 수행하는 적과 대결하며, 생존을 위한 방어 전략과 공격 전략을 동시에 학습한다.
- **전이 조건**: 현재 단계의 룰 기반 에이전트에 대해 승률 $55\%$를 달성하면 다음 단계로 넘어간다.

#### (2) Adaptive Exploration Reward
탐색 보상 $e_t$와 게임 보상 $R$ 사이의 충돌을 해결하기 위해 다음과 같은 보상 함수를 사용한다.
$$r_t = \alpha_t e_t + (1 - \alpha_t) R$$
여기서 $\alpha$는 적응형 어닐링 계수로, 에이전트의 성능 $x$ (상대방의 평균 사망 수)에 따라 다음과 같이 정의된다.
$$\alpha = 1 - \tanh(k \cdot x)$$
($k=1.2$로 설정). 학습 초기 $x=0$일 때는 $\alpha$가 커서 탐색 보상이 지배적이지만, 성능이 향상되어 $x$가 증가하면 $\alpha$가 감소하며 게임 보상(승리)에 더 집중하게 된다.

#### (3) Population-based Self-play Stage
기초 학습이 끝난 에이전트를 기반으로 8개의 에이전트 풀(Population)을 구성한다. 
- **구성**: 룰 기반 에이전트 3개(망각 방지용) + 학습 기반 에이전트 5개.
- **교체 메커니즘**: 전체 풀 내에서 승률이 $45\%$ 미만인 에이전트는 제거되고, 풀 내의 더 강한 에이전트로 무작위 교체된다.

#### (4) Elo Rating Matchmaking
에이전트 간의 적절한 대전 상대를 찾기 위해 Elo 레이팅 시스템을 도입한다.
- **기대 승률 $E_A$**:
$$E_A = \frac{1}{1 + 10^{(R_B - R_A)/400}}$$
- **레이팅 업데이트**:
$$R'_A = R_A + K(S_A - E_A)$$
- **매칭 확률**: 모든 가능한 상대와의 기대 승률($E$)에 Softmax 함수를 적용하여 매칭 확률을 결정한다. 이를 통해 실력이 비슷한 혹은 약간 더 강한 상대와 대결할 확률을 높여 점진적 학습을 유도한다.

### 2. 네트워크 아키텍처 및 학습 설정
- **알고리즘**: PPO (Proximal Policy Optimization) 기반의 Actor-Critic 구조.
- **입력**: $9 \times 9$ 2D 특징 맵(통로, 벽, 폭탄, 화염, 적/아군 위치 등)과 스칼라 특징(탄약, 폭발 범위, 킥 능력 등).
- **구조**: $\text{Conv Layers (4개)} \rightarrow \text{Flatten} \rightarrow \text{Concatenate with Scalar Features} \rightarrow \text{LSTM (10 steps memory)} \rightarrow \text{FC (128 units)} \rightarrow \text{Policy/Value Heads}$.

## 📊 Results

### 1. Curriculum Learning 결과
- 적응형 어닐링을 적용한 경우, Phase 1을 빠르게 통과하고 Phase 2에서 성능이 일시적으로 하락하더라도 $\alpha$가 다시 상승하며 탐색을 촉진해 결국 Phase 3까지 성공적으로 도달하였다.
- 반면, 선형 어닐링(Linear Annealing)을 사용한 모델은 Phase 2에서 탐색 동기를 잃어버려 학습이 정체되었으며, 결국 Phase 2를 통과하지 못하는 결과가 나타났다. 이는 성능 기반의 동적 보상 조절이 필수적임을 시사한다.

### 2. Self-play 결과
- Self-play 단계에서 에이전트들의 Elo 레이팅은 지속적으로 상승하여, 2억 타임스텝 이후 약 $1160$ 포인트에 도달하였다.
- **최종 성능 평가 (Round-robin 100회)**:
    - **학습 에이전트 대비**: 2018년 상위 학습 에이전트인 Skynet955(승률 $96.23\%$)와 Navocado(승률 $88.33\%$)를 압도하였다.
    - **룰 기반 에이전트 대비**: 2019년 4위 모델인 Neoteric보다 높은 Elo 레이팅을 기록하였다.
    - **기타**: 트리 탐색(Tree-search) 기반 에이전트(dypm)와는 대등한 수준의 Elo를 보였으나, 실제 승률은 $30\%$로 낮았다. 또한 팀원 간 통신을 사용하는 에이전트들에게는 낮은 승률을 기록하였다.

## 🧠 Insights & Discussion

본 연구는 다중 에이전트 환경에서 발생하기 쉬운 '학습 정체' 문제를 Curriculum Learning과 적응형 보상, 그리고 Elo 기반 매칭이라는 세 가지 장치로 효과적으로 해결하였다. 특히 선형 어닐링의 한계를 지적하고 성능 기반 적응형 함수를 제안한 점은 실질적인 학습 안정성을 크게 높인 부분이다.

다만, 실험 결과에서 나타나듯 트리 탐색 기반 모델이나 통신 기능을 활용하는 모델에게는 패배하였다. 이는 본 논문이 다루는 범위가 '통신 없는 모델-프리 강화학습'에 한정되어 있기 때문이며, 향후 연구에서 팀원 간의 통신 프로토콜을 학습하는 메커니즘을 추가한다면 더욱 강력한 에이전트를 구축할 수 있을 것으로 판단된다.

## 📌 TL;DR

본 논문은 Pommerman 환경의 희소 보상과 지연된 효과 문제를 해결하기 위해 **[단계적 커리큘럼 학습 $\rightarrow$ 인구 기반 셀프 플레이]**라는 2단계 학습 체계를 제안한다. 특히 **성능 기반 적응형 어닐링 계수**를 통해 탐색과 승리 전략 사이의 균형을 맞추고, **Elo 레이팅 기반 매칭**으로 학습 효율을 극대화하였다. 그 결과, 통신 기능을 사용하지 않고도 기존의 많은 학습 및 룰 기반 에이전트를 능가하는 성능을 달성하였다.
