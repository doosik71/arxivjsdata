# Refined Hardness of Distance-Optimal Multi-Agent Path Finding

Tzvika Geft and Dan Halperin (2022)

## 🧩 Problem to Solve

본 논문은 Multi-Agent Path Finding (MAPF) 문제의 계산 복잡도, 특히 거리 최적화(distance-optimal) 관점에서의 복잡성을 분석한다. MAPF의 목표는 그래프 $G$와 각 에이전트의 시작점과 목표점이 주어졌을 때, 에이전트 간 충돌이 없는 경로를 찾으면서 모든 에이전트가 이동한 총 거리의 합(total distance traveled)을 최소화하는 것이다.

이 문제는 창고 물류 시스템이나 철도 교통 스케줄링과 같은 실제 다중 로봇 시스템 운영에 필수적이다. 기존의 복잡도 연구들은 매우 제약적인 환경(예: 빈 정점이 단 하나뿐인 그리드)이나 너무 일반적인 환경(예: 일반 평면 그래프)에 집중되어 있었다. 따라서 본 논문의 목표는 가장 단순하고 제약이 적은 그래프 클래스 중 하나인 2D 그리드(2D grid) 환경에서, 여러 개의 빈 정점(multiple empty vertices)이 존재할 때 거리 최적화 MAPF가 여전히 NP-hard한지를 규명하는 것이다. 이는 이전 연구에서 제기된 미해결 문제(open problem)를 해결하는 것을 핵심 목표로 한다.

## ✨ Key Contributions

본 논문의 핵심 기여는 거리 최적화 MAPF가 2D 그리드 환경에서 빈 정점이 여러 개 존재하더라도 NP-hard함을 증명한 것이다. 구체적인 기여 사항은 다음과 같다.

1. **NP-hardness 증명**: 2D 그리드 환경의 거리 최적화 MAPF가 NP-hard임을 보임으로써, 해당 문제가 가장 단순한 형태의 환경에서도 계산적으로 매우 어렵다는 것을 입증하였다.
2. **선형 환원(Linear Reduction) 제시**: 3-SAT 문제로부터 MAPF 인스턴스로의 직접적이고 단순한 선형 환원 방법을 제시하였다. 이는 기존의 복잡한 환원 방식보다 정보 제공력이 높으며, 향후 긍정적인 결과(알고리즘적 해결책)를 찾는 데 기여할 수 있다.
3. **지수적 하한(Exponential Lower Bound) 도출**: 선형 환원을 이용해 Exponential Time Hypothesis (ETH)를 적용함으로써, 해당 문제의 실행 시간에 대한 지수적 하한을 최초로 도출하였다.
4. **Monotone MAPF의 하드니스 증명**: 일반적인 경우의 증명을 위한 징검다리로서, 에이전트들이 중간 정지 없이 순차적으로 움직이는 Monotone MAPF 역시 NP-hard함을 증명하였다.

## 📎 Related Works

거리 최적화 MAPF의 복잡도 연구는 오래전부터 진행되었으나, 최근의 MAPF 정식화(formulation)를 충분히 반영하지 못한 측면이 있었다.

- **초기 연구**: 15-puzzle과 같은 문제에서 시작하여, Goldreich(1984)는 빈 정점이 하나인 일반 그래프에서 NP-hardness를 증명했다. 이후 Ratner와 Warmuth(1990)는 이를 2D 그리드로 확장하였으나, 매우 복잡한 환원 방식을 사용했다.
- **최근 연구**: Yu와 LaValle(2013) 및 Yu(2016)는 평면 그래프(planar graphs)에서 빈 정점이 여러 개인 경우에도 NP-hard함을 보였다. 하지만 평면 그래프는 2D 그리드보다 훨씬 일반적인 범주이므로, 이 결과가 그리드 그래프에서의 하드니스를 직접적으로 의미하지는 않는다.
- **차별점**: 본 논문은 '평면 그래프'라는 일반적인 범주에서 '2D 그리드'라는 구체적이고 제한적인 범주로 복잡도 분석을 정밀화(refine)하였다. 또한, 기존 연구들이 간과했던 '선형 환원'을 통해 ETH 기반의 구체적인 시간 하한을 제시했다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 문제 정의 및 용어
- **거리 비용(Distance Cost)**: 모든 에이전트 $r \in R$이 이동한 경로 길이의 합으로 정의한다. 
- **최적 하한 $\text{d}^*(M)$**: 각 에이전트가 장애물 없이 최단 경로로 이동했을 때의 거리 합을 의미하는 낙관적 하한선이다.
- **Monotone Motion Plan**: 에이전트들이 서로 겹치지 않는 활성 구간(active interval)을 가지며, 한 번에 한 에이전트씩만 움직이는 계획이다.

### Monotone Distance-Optimal MAPF 환원
3-SAT 공식 $\phi$를 MAPF 인스턴스 $M$으로 환원하는 구조를 설계한다.
- **Variable Gadget**: 각 변수에 대응하며, 두 개의 최적 경로(Top path: 긍정 리터럴, Bottom path: 부정 리터럴)를 가진다.
- **Clause Gadget**: 각 절(clause)에 대응하며, 해당 절을 만족시키는 리터럴 에이전트들의 목표 지점과 절 에이전트(clause agent)의 목표 지점을 포함한다.
- **작동 원리**: $\phi$가 만족 가능하면, 리터럴 에이전트들이 변수 가젯을 통과해 목표 지점으로 이동한 후, 절 에이전트들이 최단 경로를 통해 자신의 목표 지점에 도달할 수 있는 계획이 존재한다. 만약 $\phi$가 만족 가능하지 않다면, 최소한 하나의 절 에이전트가 목표 지점에 도달하기 위해 최단 경로보다 더 긴 경로를 택해야 하므로 총 거리가 $\text{d}^*(M)$보다 커지게 된다.

### General Distance-Optimal MAPF 환원 (Blockers 도입)
일반적인 MAPF에서는 에이전트가 중간에 멈출 수 있어, 리터럴 에이전트가 변수 가젯 내에서 '편법(cheat)'을 써서 할당을 우회할 가능성이 있다. 이를 방지하기 위해 **Blocker 에이전트**를 도입한다.
- **Blocker의 역할**: 각 변수 가젯의 입구에서 출구로 이동하는 블로커 에이전트를 배치한다.
- **제약 사항**: 블로커와 다른 에이전트가 동일한 최적 경로를 공유할 수 없으므로, 해당 가젯을 통과하는 모든 에이전트는 블로커가 선택하지 않은 나머지 하나의 경로만을 강제적으로 사용하게 된다. 
- 이를 통해 일반적인 MAPF 환경에서도 Monotone case와 동일한 논리적 제약을 부여하여 NP-hardness를 증명한다.

### ETH를 이용한 하한 분석
- **ETH (Exponential Time Hypothesis)**: 3-SAT 문제를 $2^{o(n)}$ 시간 내에 해결하는 알고리즘이 존재하지 않는다는 가설이다.
- 본 논문에서 제시한 환원은 변수와 절의 개수에 대해 선형적인 크기의 MAPF 인스턴스를 생성한다. 따라서 MAPF를 $2^{o(n)}$ 시간에 해결할 수 있다면 3-SAT 또한 동일한 시간 복잡도로 해결할 수 있게 되므로, ETH가 성립한다면 거리 최적화 MAPF의 시간 하한은 지수적일 수밖에 없다.

## 📊 Results

본 논문은 정량적 실험 수치보다는 이론적 증명을 통해 다음과 같은 결과를 도출하였다.

1. **NP-hardness 확립**: 2D 그리드 그래프에서 빈 정점이 여러 개 존재하는 경우에도 거리 최적화 MAPF는 NP-hard하다.
2. **계산 복잡도 하한**: ETH 가설 하에, 에이전트의 수 $n$ 또는 정점의 수 $|V|$에 대해 $2^{o(n)}$ 또는 $2^{o(|V|)}$ 시간 내에 문제를 해결하는 알고리즘은 존재할 수 없다.
3. **제약 조건의 완화**: 본 증명은 병렬 이동(parallel motion)이나 순차 이동(sequential motion) 여부와 상관없이 성립하며, 심지어 에이전트들이 아래쪽과 오른쪽으로만 이동할 수 있는 극히 제한적인 상황에서도 NP-hard함이 유지됨을 보였다.

## 🧠 Insights & Discussion

### 하드니스의 원천에 대한 새로운 관점
기존 연구(Yu [39])에서는 서로 반대 방향으로 이동하려는 에이전트 집단 간의 충돌(contention)이 문제의 어려움을 유발한다고 보았다. 그러나 본 논문의 구성에서는 모든 에이전트가 대체로 같은 방향(오른쪽)으로 이동한다. 이는 거리 최적화 MAPF의 어려움이 단순히 '반대 방향 이동'에서 오는 것이 아니라, **"다른 에이전트의 시작점과 목표점이 밀집된 경로를 어떻게 협상하며 통과하는가"**에서 발생함을 시사한다.

### 실무적 시사점
- **우선순위 결정**: 경로 상에 다른 에이전트의 목표 지점이 있는 경우, 해당 지점이 점유되기 전에 통과해야 하므로 에이전트 간의 이동 순서 결정이 매우 중요하다.
- **알고리즘 설계**: 경로 상의 목표 지점이 곧 점유될 예정인 에이전트에게 높은 우선순위를 부여하는 전략이 유효할 수 있다.

### 한계 및 향후 연구
- 본 논문은 최악의 경우(worst-case)에 대한 복잡도를 다루었으나, 실제 환경에서는 경로의 중첩도(overlap degree)나 에이전트당 경로 상의 목표 지점 수 등의 파라미터에 따라 복잡도가 달라질 수 있다.
- 현재의 지수적 하한과 실제 알고리즘의 상한(upper bound) 사이에 간극이 존재하며, 이를 좁히기 위한 정밀한 분석이 필요하다.

## 📌 TL;DR

본 논문은 2D 그리드 환경에서 빈 정점이 여러 개 있을 때 거리 최적화 MAPF 문제가 NP-hard함을 증명하여 기존의 미해결 문제를 해결하였다. 특히 3-SAT로부터의 선형 환원을 통해 ETH 기반의 지수적 시간 하한을 최초로 제시하였으며, 문제의 어려움이 반대 방향 이동이 아닌 경로 상의 목표 지점 점유 및 협상 과정에서 발생한다는 새로운 통찰을 제공하였다. 이는 향후 효율적인 다중 로봇 경로 계획 알고리즘을 설계하는 데 있어 이론적 기초가 될 것으로 기대된다.
