# Learning to Learn Neural Networks

Tom Bosc (2016)

## 🧩 Problem to Solve

본 논문은 '학습하는 방법' 자체를 학습하는 Meta-learning(메타 학습) 문제를 다룬다. 특히, 데이터가 순차적으로 입력되고 단 한 번만 제시되는 온라인 학습(on-line learning) 상황에서, 모델의 파라미터를 어떻게 효율적으로 업데이트할 것인가에 집중한다.

일반적인 학습 알고리즘(예: Gradient Descent)은 사람이 설계한 규칙에 따라 파라미터를 업데이트하지만, 본 연구의 목표는 데이터의 특성에 맞게 최적의 업데이트 규칙을 스스로 찾아내는 학습 알고리즘(Learner)을 신경망으로 구현하는 것이다. 이를 통해 특정 데이터 분포에 최적화된 맞춤형 업데이트 규칙을 생성함으로써 학습 성능을 극대화하고자 한다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 LSTM(Long Short-Term Memory) 네트워크를 사용하여 다른 신경망(Target Model)의 파라미터를 실시간으로 계산하고 업데이트하는 구조를 제안한 것이다. 

가장 큰 특징은 타겟 모델의 파라미터 $\theta$를 LSTM의 셀 상태(cell state)에 직접 저장한다는 점이다. 즉, LSTM이 단순히 데이터를 처리하는 것이 아니라, 타겟 모델의 가중치 그 자체가 되어 각 타임스텝마다 최적의 가중치 값을 출력하도록 설계되었다. 또한, 학습 데이터와 테스트 데이터를 구분하는 '분리 플래그(separator flag)'를 도입하여, 학습된 알고리즘이 단순히 데이터를 암기하는 것이 아니라 새로운 데이터에 대해 일반화(generalization) 성능을 갖는지 객관적으로 평가할 수 있는 프레임워크를 제시하였다.

## 📎 Related Works

논문은 Hochreiter 등이 제안한 연구를 언급한다. 기존 연구에서는 RNN을 사용하여 감독 학습 하에 이차 함수(quadratic functions)의 계수를 학습하는 것이 가능하다함을 보여주었다. 당시 RNN은 현재 타임스텝의 입력과 이전 타겟 값을 입력받아 내부 상태를 업데이트함으로써 예측 성능을 향상시켰으며, LSTM 블록과 출력층 사이에 추가적인 은닉층을 배치한 구조를 사용하였다.

본 연구는 이러한 기존 접근 방식을 확장하여, 단순한 함수 계수 추정이 아닌 다층 퍼셉트론(MLP)과 같은 더 복잡한 신경망의 가중치를 업데이트하는 메타 학습기로 발전시켰다. 특히, 훈련 세트와 테스트 세트를 명확히 구분하여 학습된 알고리즘의 성능을 기존의 수작업 알고리즘(hand-made algorithms)과 동일한 조건에서 비교했다는 점이 차별점이다.

## 🛠️ Methodology

### 전체 시스템 구조
시스템은 크게 두 부분으로 나뉜다: 타겟 모델(Model)과 학습기(Learner)이다.

1.  **모델(Model)**: 입력 $x_t$가 주어졌을 때 타겟 $y_t$의 추정치 $o_t$를 계산한다.
    $$o_t = f_{\theta_t}(x_t)$$
    여기서 $\theta_t$는 타임스텝 $t$에서의 파라미터 추정치이며, 이는 RNN(LSTM)의 은닉 상태(hidden state)에 해당한다.

2.  **학습기(Learner)**: 모델의 파라미터 $\theta_t$를 업데이트하여 $\theta_{t+1}$을 생성하는 메커니즘이다.
    $$\theta_{t+1} = g_\alpha(x_t, y_t * 1_{t < \tau}, 1_{t < \tau}, o_t)$$
    학습기는 파라미터 $\alpha$와 초기 상태 $\theta_1$에 의해 정의된다. 여기서 $1_{t < \tau}$는 현재 타임스텝이 훈련 세트에 속하는지를 나타내는 지시 함수(indicator function)로, 테스트 단계에서는 타겟 $y_t$가 제공되지 않으므로 학습기가 이를 오해하지 않도록 구분해주는 역할을 한다.

### 메타 학습 목표 (Meta-learning Objective)
메타 학습은 $\alpha$와 $\theta_1$을 최적화하는 과정이다. 저자는 두 가지 손실 함수를 제시한다.

- **평가용 목적 함수**: 훈련 데이터($t < \tau$)를 통해 업데이트된 모델이 테스트 데이터($t \ge \tau$)에서 보이는 평균 손실을 최소화한다.
  $$\text{argmin}_{\alpha, \theta_1} \frac{1}{N} \sum_{i=1}^{N} \frac{1}{n_i - \tau_i + 1} \left( \sum_{t=\tau_i}^{n_i} L(f_{\theta_t}(x_t^{(i)}), y_t^{(i)}) \right)$$
- **훈련용 목적 함수**: 모든 샘플(훈련 및 테스트 포함)에 대한 평균 손실을 최소화한다.
  $$\text{argmin}_{\alpha, \theta_1} \frac{1}{N} \sum_{i=1}^{N} \frac{1}{n_i} \sum_{t=1}^{n_i} L(f_{\theta_t}(x_t^{(i)}), y_t^{(i)})$$
실제 메타 학습 시에는 최적화의 어려움을 줄이기 위해 후자의 함수를 사용하고, 알고리즘 간의 성능 비교에는 전자의 함수를 사용한다.

### 세부 구현 (LSTM-based Learner)
학습기 $g_\alpha$는 LSTM 블록으로 구현되었다. 입력 벡터 $[x_t, y_t * 1_{t < \tau}, 1_{t < \tau}, o_t]$는 먼저 여러 개의 완전 연결 층(Fully Connected layers)을 통과하여 $x^*$를 생성하며, 이후 LSTM의 게이트 파라미터들을 계산한다.

$$x^* = g_1 \circ \dots \circ g_n([x_t, y_t * 1_{t < \tau}, 1_{t < \tau}, o_t])$$
$$z_t = W_z x^* + b_z, \quad i_t = \sigma(W_i x^* + b_i), \quad f_t = \sigma(W_f x^* + b_f)$$

최종적인 파라미터 업데이트 식은 다음과 같다.
$$\theta_{t+1} = i_t * z_t + f_t * \theta_t$$
여기서 $\theta$는 LSTM의 셀 상태이며, 별도의 출력 게이트 없이 이 상태 값이 곧 모델의 가중치로 사용된다.

## 📊 Results

### 실험 설정
- **타겟 모델**: 1-은닉층 MLP (입력 5, 은닉 유닛 32, 출력 1). ReLU 및 Sigmoid 활성화 함수 사용. 총 파라미터 수는 225개이다.
- **학습기**: 두 개의 FC 층(크기 128, 256)과 LSTM 블록으로 구성. 총 메타-학습 가능 파라미터는 207,651개이다.
- **데이터셋**: 무작위로 생성된 비선형 분리 불가능한 이진 분류 데이터. 다변량 가우시안 분포와 노이즈, XOR 연산을 통해 생성되었다.
- **최적화**: SMORMS3 알고리즘을 사용하였으며 학습률은 $10^{-3}$이다.
- **비교 대상**: Logistic Regression (L1/L2 규제), Linear SVM, RBF SVM.

### 정량적 결과
평가 지표인 평균 교차 엔트로피(Mean Cross-Entropy, MCE) 결과는 다음과 같다.

| 알고리즘 | $\mu_{MCE}$ | $\sigma_{MCE}$ |
| :--- | :---: | :---: |
| **LSTM (Proposed)** | **0.540** | **0.139** |
| Logistic Regression | 0.574 | 0.208 |
| SVM (linear) | 0.573 | 0.159 |
| SVM (RBF) | 0.507 | 0.164 |

제안된 LSTM 기반 학습 알고리즘은 선형 모델들(Logistic Regression, Linear SVM)보다 유의미하게 낮은 손실 값을 기록하여, 비선형 데이터에 대해 일반화 능력이 있음을 증명하였다. 다만, RBF 커널을 사용한 SVM보다는 성능이 낮게 나타났다.

## 🧠 Insights & Discussion

### 학습 및 메타 학습의 트레이드-오프
저자는 가중치 행렬의 변화를 분석하여 흥미로운 사실을 발견하였다. 학습 과정에서 일부 파라미터는 학습기에 의해 계속 업데이트되는 반면, 일부 파라미터는 거의 변하지 않고 고정된 상태를 유지한다. 이는 모델이 모든 가중치를 동적으로 업데이트하는 대신, 메타 학습 과정을 통해 최적의 초기값을 가진 일부 가중치는 고정시키고(Meta-learning), 나머지 가중치들만 데이터에 맞게 업데이트(Learning)하는 전략을 취하고 있음을 시사한다.

### 분석 및 한계
- **일반화 능력**: 학습된 알고리즘이 단순한 암기가 아니라 비선형 데이터에 대해 적절한 업데이트를 수행한다는 점이 확인되었다.
- **테스트 단계의 정체**: 실험 결과, 훈련 샘플을 모두 본 후 테스트 샘플 단계로 진입하면 성능 변화가 멈추고 가중치 업데이트가 거의 일어나지 않는 현상이 관찰되었다. 이는 학습기가 테스트 단계에서도 지속적으로 파라미터를 샘플링하거나 조정하도록 유도하는 추가 연구가 필요함을 의미한다.
- **모델 비교의 한계**: 현재 결과는 '학습 알고리즘' 간의 비교이며, 동일한 MLP 구조를 사용하되 다른 학습 알고리즘을 적용했을 때의 엄격한 비교는 향후 과제로 남겨두었다.

## 📌 TL;DR

본 논문은 LSTM의 셀 상태를 다른 신경망의 가중치로 직접 활용하여, 데이터에 따라 가중치를 실시간으로 업데이트하는 '학습하는 방법'을 학습하는 메타 학습 프레임워크를 제안한다. 실험을 통해 제안된 방식이 선형 모델보다 비선형 데이터셋에서 우수한 성능을 보이며, 메타 학습을 통해 일부 가중치는 고정하고 일부는 업데이트하는 효율적인 전략을 스스로 습득함을 보여주었다. 이 연구는 신경망을 이용해 최적의 최적화 알고리즘을 설계할 수 있는 가능성을 제시하며, 향후 맞춤형 최적화 도구 개발에 기여할 수 있다.
