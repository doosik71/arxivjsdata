# Large Pre-trained time series models for cross-domain Time series analysis tasks

Harshavardhan Kamarthi, B. Aditya Prakash (2024)

## 🧩 Problem to Solve

본 논문은 서로 다른 도메인(예: 전염병, 에너지, 경제, 교통 등)에서 발생하는 이질적인 시계열 데이터들을 통합하여 학습하고 적용할 수 있는 파운데이션 모델(Foundation Model) 구축의 어려움을 해결하고자 한다.

기존의 시계열 분석은 주로 특정 도메인의 데이터와 전문가 지식을 활용해 개별 모델을 처음부터 학습시키는 방식에 의존해 왔다. 최근 언어 및 비전 분야의 성공에 힘입어 시계열 분야에서도 사전 학습된 모델들이 제안되고 있으나, 다음과 같은 핵심적인 문제에 직면해 있다.

1. **데이터의 이질성(Heterogeneity):** 도메인마다 샘플링 속도(Sampling rate), 생성 역학(Generative dynamics), 노이즈 수준이 다르다. 예를 들어, GDP 데이터는 샘플링 주기가 길어 긴 세그먼트가 필요하지만, 심박수 센서 데이터는 밀리초 단위로 수집되므로 훨씬 짧은 세그먼트가 필요하다.
2. **토큰화(Tokenization)의 경직성:** 기존의 Transformer 기반 시계열 모델들은 주로 고정된 길이의 세그먼트로 데이터를 나누는 uniform segmentation 방식을 사용한다. 이는 시계열의 국소적 패턴이나 시간적 변화(예: 전염병의 비수기와 유행기 간의 복잡도 차이)를 유연하게 포착하지 못하게 한다.

따라서 본 논문의 목표는 다양한 도메인의 시계열 데이터로부터 의미 있는 토큰을 자동으로 추출할 수 있는 **적응형 세그멘테이션(Adaptive Segmentation)** 메커니즘을 도입하여, 제로샷(Zero-shot) 및 퓨샷(Few-shot) 설정에서도 우수한 성능을 내는 범용 시계열 파운데이션 모델인 LPTM(Large Pre-trained Time-series Models)을 구축하는 것이다.

## ✨ Key Contributions

본 논문의 핵심적인 기여는 시계열 데이터를 모델의 입력으로 변환하는 과정에서 **도메인별 최적의 세그멘테이션 전략을 자동으로 학습하는 적응형 세그멘테이션 모듈**을 제안한 것이다.

단순히 고정된 크기의 윈도우를 사용하는 대신, 자기지도학습(Self-Supervised Learning, SSL) 손실을 최소화하는 방향으로 세그먼트의 경계를 결정한다. 이를 통해 모델은 데이터의 복잡도가 높은 구간에서는 세밀한(fine-grained) 세그멘테이션을, 단순한 구간에서는 성긴(coarse-grained) 세그멘테이션을 수행함으로써 각 도메인의 고유한 특성을 효율적으로 포착할 수 있다.

## 📎 Related Works

기존의 시계열 분석 접근 방식은 크게 세 가지로 분류된다.

1. **도메인 특화 모델:** DeepAR, Informer, Autoformer 등은 특정 작업에 최적화되어 있으나, 매번 처음부터 학습시켜야 하므로 데이터 소모가 크고 범용성이 떨어진다.
2. **자기지도학습(SSL) 기반 표현 학습:** TS2Vec, TS-TCC 등은 대조 학습(Contrastive Learning) 등을 통해 표현력을 높였으나, 주로 단일 데이터셋 내에서 수행되어 멀티 도메인 환경으로의 확장성이 부족하다.
3. **LLM 기반 시계열 모델:** Chronos, TimesFM 등 최근 LLM의 구조를 차용한 모델들이 등장하였다. 이들은 주로 개별 타임스텝을 입력으로 쓰거나 고정된 세그먼트를 사용한다. LPTM은 이러한 LLM 기반 모델들보다 파라미터 규모가 훨씬 작음에도 불구하고(2~10배 작음), 적응형 세그멘테이션을 통해 더 효율적인 성능을 달성하였다.

## 🛠️ Methodology

### 1. 전체 파이프라인

LPTM은 **적응형 세그멘테이션 모듈 $\rightarrow$ Transformer Encoder $\rightarrow$ 다운스트림 태스크 헤드**로 구성된다. 사전 학습 단계에서는 멀티 도메인 데이터셋 $D_{pre}$를 사용하여 자기지도학습을 수행하며, 이후 특정 태스크에 맞춰 파인튜닝하거나 제로샷으로 배포한다.

### 2. 적응형 세그멘테이션 모듈 (Adaptive Segmentation Module)

이 모듈은 각 도메인에 최적화된 토큰화 전략을 찾는다.

- **인코딩:** 입력 시계열 $y^{(1...t)}$를 GRU에 통과시켜 시간적 패턴을 담은 은닉 임베딩 $\{z^{(i)}\}_{i=1}^t$를 생성한다.
  $$\{z^{(i)}\}_{i=1}^t = \text{GRU}(\{y^{(i)}\}_{i=1}^t)$$
- **스코어링:** 임의의 두 시점 $i$와 $j$ 사이의 세그먼트가 얼마나 유용한지를 나타내는 스코어 함수 $s(i,j)$를 정의한다.
  $$s(i,j) = v^T \tanh(W_1 z_i + W_2 z_j + b)$$
- **세그먼트 선택 및 프루닝:**
  1. 각 시작점 $i$에 대해 가장 높은 스코어를 가진 종료점 $h(i) = \arg \max_j s(i,j)$를 찾아 세그먼트 집합 $\hat{S}$를 구성한다.
  2. 전체 시계열을 커버한다는 조건 하에, 스코어가 가장 낮은 세그먼트를 반복적으로 제거(Pruning)하여 최종 세그먼트 집합 $S$를 결정한다.
- **토큰 임베딩 생성:** 선택된 세그먼트 $(i,j)$ 내의 임베딩들을 Self-Attention 레이어에 통과시켜 집계(Aggregate)한다.
  $$\hat{e}(i,j) = \sum_{k=i}^j \text{Self-Atten}\{z^{(i)}, \dots, z^{(j)}\}$$
  여기에 세그먼트의 시작 위치 $\text{pos}(i)$와 세그먼트 길이 $\text{pos}(j-i)$에 대한 위치 인코딩을 결합하여 최종 토큰 $\text{e}_i$를 생성한다.

### 3. 자기지도학습(SSL) 태스크

모델은 다음 두 가지 마스킹 기반 태스크를 통해 사전 학습된다.

- **Random Masking (RANDMASK):** 입력 세그먼트 토큰을 확률 $\gamma$로 마스킹하고, 이를 복원하도록 학습한다. (보간 및 외삽 능력 학습)
- **Last Token Masking (LASTMASK):** 시계열의 마지막 $\gamma$ 비율의 세그먼트들을 마스킹하여 미래 값을 예측하도록 학습한다. (예측 능력 학습)
손실 함수로는 복원된 값과 실제 값 사이의 평균 제곱 오차(MSE)인 $L_{SSL}$을 사용한다.

### 4. 학습 절차 및 최적화

- **REVIN:** 데이터 도메인 간의 분포 차이를 해결하기 위해 Reversible Instance Normalization을 사용하여 입력을 정규화하고 출력 시 다시 복원한다.
- **스코어 함수 학습:** 세그먼트 선택 과정이 이산적(discrete)이어서 그래디언트 전파가 불가능하므로, 선택된 세그먼트들의 스코어 합이 SSL 손실의 음의 로그값과 일치하도록 하는 별도의 손실 함수 $L_g$를 정의하여 학습한다.
  $$L_g = \left( \sum_{(i,j) \in S} s(i,j) + \log(L_{SSL}) \right)^2$$
- **파인튜닝:** Linear Probing(마지막 레이어만 학습) 후 전체 파라미터를 학습시키는 2단계 전략을 사용하여 OOD(Out-of-Distribution) 데이터에 대한 성능 저하를 방지한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋:** 전염병(Project Tycho), 전력(ETT), 교통(PEM-Bays, METR-LA), 수요(NY-Bike/Taxi), 주식(Nasdaq, S&P 500), M4 경쟁 데이터, 센서/행동 데이터 등 매우 광범위한 도메인을 사용하였다.
- **평가 지표:** 예측(Forecasting) 태스크에서는 RMSE를, 분류(Classification) 태스크에서는 Accuracy를 사용하였다.

### 2. 주요 결과

- **제로샷 예측:** LPTM은 TimesFM, Chronos, MOIRAI 등 최신 파운데이션 모델들을 유의미하게 능가하였다. 이는 적응형 세그멘테이션이 도메인 간 일반화 성능을 크게 향상시켰음을 시사한다.
- **파인튜닝 예측:** 도메인 특화 모델(EpiFNP, STEP 등)과 비교했을 때, LPTM은 대부분의 벤치마크에서 1위 또는 2위의 성능을 기록하였다. 특히 전력, 교통, M4 데이터셋에서는 기존 SOTA를 경신하였다.
- **시계열 분류:** 35개의 센서 및 행동 데이터셋에 대해 실험한 결과, LPTM이 가장 높은 평균 순위(Mean Rank)를 기록하며 범용적인 표현 학습 능력을 입증하였다.
- **데이터 및 학습 효율성:** LPTM은 SOTA 모델들과 유사하거나 더 나은 성능을 내면서도, 학습 데이터의 80% 미만만으로도 충분했으며, 학습 시간 또한 크게 단축되었다.

## 🧠 Insights & Discussion

### 1. 강점 및 분석

LPTM의 가장 큰 성공 요인은 **데이터의 국소적 복잡도에 따라 토큰화 해상도를 조절**한 점에 있다. 시각화 결과, 전염병의 정점(Peak)과 같이 변동성이 큰 구간에서는 세그먼트 크기가 작아지고, 단순한 추세를 보이는 구간에서는 세그먼트 크기가 커지는 것이 확인되었다. 이는 모델이 데이터의 의미적 중요도에 따라 집중도를 자동으로 조절하고 있음을 보여준다.

### 2. 한계 및 비판적 해석

- **확장성:** 본 모델은 단변량(Univariate) 시계열 분석에 집중되어 있으며, 다변량(Multivariate) 시계열이나 확률적 예측(Probabilistic forecasting)에 대한 처리 능력은 명시적으로 다루지 않았다.
- **복잡도:** 적응형 세그멘테이션을 위해 GRU와 스코어링 함수를 추가로 사용하므로, 단순한 고정 세그먼트 방식보다는 추론 시 오버헤드가 발생할 수 있다. 다만, 논문에서는 전체 파라미터 수가 LLM 기반 모델보다 훨씬 적음을 강조하여 이를 상쇄하고 있다.

## 📌 TL;DR

본 논문은 서로 다른 도메인의 시계열 데이터를 통합 학습하기 위해, 데이터의 특성에 따라 세그먼트 길이를 자동으로 조절하는 **적응형 세그멘테이션(Adaptive Segmentation)** 기반의 파운데이션 모델 **LPTM**을 제안한다. LPTM은 적은 양의 데이터와 학습 시간만으로도 제로샷 및 파인튜닝 설정에서 기존의 거대 LLM 기반 시계열 모델과 도메인 특화 모델들을 압도하는 성능을 보여주었다. 이 연구는 시계열 데이터의 '토큰화' 문제가 범용 모델 구축의 핵심임을 밝혔으며, 향후 멀티모달 데이터 결합이나 불확실성 추정 연구로 확장될 가능성이 높다.
