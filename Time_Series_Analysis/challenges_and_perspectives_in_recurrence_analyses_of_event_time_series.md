# Challenges and perspectives in recurrence analyses of event time series

Norbert Marwan (2024)

## 🧩 Problem to Solve

본 논문은 이벤트 시계열(Event Time Series) 데이터 분석에서 발생하는 근본적인 어려움과 이를 해결하기 위한 방법론적 접근 방식을 다룬다. 이벤트 시계열은 금융 거래, 심장 박동(Cardiac events), 극한 강수량, 산사태와 같은 자연 현상 등 다양한 분야에서 나타나며, 일반적인 시계열 데이터와 달리 다음과 같은 특징을 가진다.

첫째, 이벤트는 불규칙한 간격으로 발생하므로 데이터가 불규칙하게 샘플링(Irregularly sampled)되는 경향이 있다. 둘째, 이벤트 사이의 간격이 매우 넓어 데이터의 대부분이 0으로 채워지는 희소성(Sparsity) 문제가 발생한다. 기존의 표준 시계열 분석 도구들은 등간격 샘플링을 가정하거나 연속적인 값의 변화를 분석하는 데 최적화되어 있어, 이러한 이벤트 기반 데이터의 동역학을 분석하거나 서로 다른 이벤트 시계열 간의 유사성을 측정하는 데 한계가 있다. 따라서 본 논문의 목표는 비선형 시계열 분석의 강력한 도구인 재귀 분석(Recurrence Analysis)을 이벤트 데이터에 적용하여 이러한 문제를 해결하고, 향후 연구 방향을 제시하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 이벤트 시계열의 특성을 반영하여 재귀 분석(Recurrence Analysis)의 프레임워크를 확장한 점이다. 특히, 연속적인 상태 값의 유클리드 거리(Euclidean distance) 대신, 이벤트 시퀀스 간의 변환 비용을 측정하는 Edit Distance(편집 거리)를 유사성 측정 지표로 도입하였다. 이를 통해 불규칙하게 샘플링된 이벤트 데이터에서도 시스템의 재귀적 구조를 시각화하고 정량화할 수 있는 기반을 마련하였다. 또한, 이벤트 기반 RP(Recurrence Plot)와 연속 시계열 기반 RP를 결합하여 서로 다른 성격의 데이터 간 동기화(Synchronisation)를 분석할 수 있는 가능성을 제시하였다.

## 📎 Related Works

기존의 재귀 분석은 주로 연속적인 상태 변수를 가진 동역학 시스템을 대상으로 하며, 상태 공간에서의 거리 $d(\vec{x}_i, \vec{x}_j)$가 임계값 $\epsilon$보다 작을 때 재귀가 일어난다고 정의한다. 범주형 데이터(Categorical data)의 경우 크로네커 델타 함수(Kronecker delta function)를 사용하여 동일한 상태인지를 판별하는 방식이 사용되어 왔다.

이벤트 데이터 분석을 위해 신경과학 분야에서 제안된 Event Synchronisation이나 Levenshtein distance 기반의 Edit distance 등이 존재하지만, 이를 재귀 분석의 전체 파이프라인(RP 생성 $\rightarrow$ RQA 정량 분석)에 통합하여 다양한 과학 분야에 적용하려는 시도는 상대적으로 부족했다. 본 논문은 이러한 기존의 거리 측정 방식들을 재귀 분석의 프레임워크 내로 통합함으로써, 단순한 유사도 측정을 넘어 시스템의 전반적인 동역학적 특성을 분석할 수 있도록 확장하였다.

## 🛠️ Methodology

### 전체 파이프라인

이벤트 시계열 분석을 위한 재귀 분석의 기본 흐름은 다음과 같다: 이벤트 시계열 $\rightarrow$ 시간 창(Time Window) 분할 $\rightarrow$ 시퀀스 간 Edit Distance 계산 $\rightarrow$ 재귀 행렬 생성 $\rightarrow$ RP 시각화 및 RQA 정량 분석 순으로 진행된다.

### 재귀 행렬의 정의

기본적인 재귀 행렬 $R$은 다음과 같이 정의된다.
$$R_{i,j} = \Theta (\epsilon - d(\vec{x}_i, \vec{x}_j))$$
여기서 $\Theta(\cdot)$는 헤비사이드 함수(Heaviside function)이며, 유사성 측정값 $d(\cdot, \cdot)$가 임계값 $\epsilon$보다 작을 때 $R_{i,j} = 1$이 된다.

### 이벤트 데이터를 위한 Edit Distance

이벤트 시계열에서는 유클리드 거리 대신 Edit Distance를 사용한다. 이벤트 시계열을 길이 $T_w$의 시간 창으로 나누어 시퀀스 $S_i$와 $S_j$를 생성하고, 한 시퀀스를 다른 시퀀스로 변환하는 데 필요한 최소 비용을 거리로 정의한다.

비용 함수는 다음과 같이 정의된다:
$$d(S_i, S_j) = \min \left( \lambda_s (N_i + N_j - 2N_{(i,j)}) + \sum_{a,b \in C} \lambda_0 \|t^{(i)}_a - t^{(j)}_b\| \right)$$

- $N_i, N_j$: 각 시퀀스 $S_i, S_j$에 포함된 이벤트의 수이다.
- $N_{(i,j)}$: 시프트(Shift) 연산이 적용되는 이벤트의 쌍의 수이다.
- $\lambda_s$: 이벤트의 추가(Addition) 또는 삭제(Deletion)에 할당되는 비용이다.
- $\lambda_0$: 이벤트의 시간적 위치를 이동시키는 시프트 연산에 할당되는 비용이다.
- $C$: 시프트 연산이 적용되는 이벤트들의 집합이다.

이 수식의 첫 번째 항은 이벤트 개수의 차이로 인한 삽입/삭제 비용을 의미하며, 두 번째 항은 발생 시간의 미세한 차이로 인한 조정 비용을 의미한다.

### 혼합 데이터 분석 (Joint-RP)

이벤트 시계열 $x$와 연속 시계열 $y$를 동시에 분석하기 위해 Joint-RP를 다음과 같이 구성할 수 있다.
$$JR_{x,y}^{i,j} = d(S^x_i, S^x_j) \circ \Theta (\epsilon - d(\vec{y}_i, \vec{y}_j))$$
이는 이벤트 데이터의 재귀 구조와 연속 데이터의 재귀 구조를 아다마르 곱(Hadamard product)으로 결합하여 두 시스템 간의 상호작용을 분석하는 방법이다.

## 📊 Results

### 실험 설정

논문에서는 Rössler 시스템의 $x$ 변수에서 발생하는 극댓값(Maxima)을 추출하여 희소한 이벤트 데이터를 모사하였다. 분석을 위해 시간 창 $T_w = 15\text{s}$, 이동 간격 $s = 1\text{s}$를 설정하였으며, 재귀율(Recurrence Rate)이 15%가 되도록 임계값 $\epsilon$을 선택하였다.

### 결과 및 해석

- **연속 데이터 RP vs 이벤트 데이터 RP**: Rössler 시스템의 전체 상태 변수 $(x, y, z)$를 이용해 그린 RP와, 오직 극댓값 이벤트만을 추출하여 Edit Distance 기반으로 그린 RP를 비교하였다.
- **분석 결과**: 이벤트 기반 RP에서도 원본 시스템의 주기적인 특성을 나타내는 대각선 구조(Period line structures)가 명확하게 나타났다. 이는 Edit Distance 기반의 재귀 분석이 데이터의 희소성에도 불구하고 시스템의 핵심적인 동역학적 정보를 보존하고 있음을 보여준다.
- **특이 사항**: 이벤트가 발생하지 않는 구간(Empty bars)에서는 재귀 포인트가 나타나지 않아, 시스템의 갑작스러운 변화나 정지 구간을 시각적으로 쉽게 식별할 수 있었다.

## 🧠 Insights & Discussion

### 강점 및 가능성

본 방법론은 데이터의 샘플링 주기나 형태에 구애받지 않고, '상태의 재귀'라는 관점에서 서로 다른 도메인의 데이터를 통합 분석할 수 있다는 강력한 장점이 있다. 특히 머신러닝과의 결합 가능성이 높다. RP를 이미지로 변환하여 CNN(Convolutional Neural Networks)의 입력값으로 사용함으로써 이벤트 시계열의 분류 및 예측 성능을 높일 수 있을 것으로 기대된다.

### 한계 및 해결 과제

1. **데이터 결측치 문제**: 산사태 데이터와 같이 특정 지점에서만 수집된 데이터는 분석 결과에 편향(Bias)을 일으킬 수 있으며, 이벤트 데이터에 적합한 보간법이나 재구성 방법이 필요하다.
2. **비정상성(Non-stationarity)**: 이벤트의 발생 빈도가 시간에 따라 변하는 경우, 시간 창 $T_w$의 선택이 매우 까다롭다. $T_w$가 너무 작으면 빈 시퀀스가 발생하여 거리를 정의할 수 없는 문제가 생긴다.
3. **Timing Jitter**: 실제 데이터에서 이벤트 발생 시간의 정밀한 측정은 어렵다. 이러한 시간적 불확실성(Jitter)이 분석 결과에 미치는 영향을 평가하기 위해 몬테카를로 앙상블이나 베이지안 접근법의 도입이 필요하다.
4. **연속-이벤트 데이터 매칭**: 두 데이터의 샘플링 포인트 수가 크게 다르기 때문에, Joint-RP를 적용하기 위한 coarse-graining이나 보간법 등의 정교한 매칭 전략이 요구된다.

## 📌 TL;DR

본 논문은 불규칙하고 희소한 **이벤트 시계열 데이터를 분석하기 위해 Edit Distance를 결합한 재귀 분석(Recurrence Analysis) 프레임워크를 제안**한다. 이를 통해 기존 도구로 분석하기 어려웠던 극치 이벤트(Extreme events)의 동역학적 패턴을 시각화하고 정량화할 수 있으며, 향후 머신러닝 결합 및 인과관계 추론, 연속 데이터와의 통합 분석으로 확장될 가능성이 매우 높다.
