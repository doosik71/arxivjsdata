# From Sleep Staging to Spindle Detection: Evaluating End-to-End Automated Sleep Analysis

Niklas Grieger, Siamak Mehrkanoon, Philipp Ritter, Stephan Bialonski (2025)

## 🧩 Problem to Solve

수면 분석은 크게 수면 단계(Sleep Stages)를 구분하는 거시 구조적(Macrostructural) 분석과 수면 방추파(Sleep Spindles)와 같은 특정 파형을 찾는 미시 구조적(Microstructural) 분석으로 나뉜다. 기존 연구들은 수면 단계 분류나 방추파 검출과 같은 개별 단계의 자동화에는 성공했으나, 이 두 단계를 결합한 전체 파이프라인(End-to-End)의 자동화 가능성과 신뢰성에 대해서는 명확히 검증되지 않았다.

특히 수면 방추파 검출은 생물학적으로 주로 N2 수면 단계에서 발생하므로, 방추파 검출 모델의 정확도는 앞선 단계인 수면 단계 분류 모델의 성능에 크게 의존한다. 만약 수면 단계 분류가 잘못된다면, 방추파가 나타날 수 없는 REM 수면이나 각성(Wake) 상태에서 방추파가 검출되는 오류가 발생할 수 있다. 따라서 본 논문의 목표는 최신 딥러닝 모델들을 이용해 수면 분석의 전체 과정을 자동화하고, 이것이 실제 전문가가 수행한 임상 연구 결과(조울증 환자의 수면 분석)를 동일하게 재현할 수 있는지 평가하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 개별적으로 최적화된 SOTA 모델인 RobustSleepNet(수면 단계 분류)과 SUMOv2(방추파 검출)를 통합하여 전체 자동화 수면 분석 파이프라인을 구축하고 그 효용성을 입증한 것이다.

주요 설계 아이디어는 모델의 성능을 단순히 정답셋과의 일치도로 평가하는 것이 아니라, 전문가들 사이에서도 발생하는 합의 수준(Inter-rater agreement)과 비교함으로써 모델이 인간 전문가 수준의 성능을 내는지 검증하는 것이다. 또한, 조울증(Bipolar Disorder) 환자와 건강한 대조군의 데이터를 분석하여 전문가 기반의 기존 연구 결과를 정성적으로 재현함으로써, 수개월이 걸리던 수동 분석 작업을 단 몇 분 만에 처리할 수 있음을 보여주었다.

## 📎 Related Works

기존의 자동 수면 분석은 특정 주파수 대역의 전력(Power)과 같은 특징량을 정의하여 사용하던 방식에서, 최근에는 특징을 스스로 학습하는 심층 신경망(Deep Neural Network) 기반으로 발전하였다.

- **수면 단계 분류:** RobustSleepNet과 같은 모델들이 다양한 데이터셋에서 높은 성능을 보였으나, 주로 단일 작업의 정확도에 집중했다.
- **방추파 검출:** SUMO와 같은 모델들이 SOTA 성능을 달성했으나, 대부분 전문가가 이미 구분해 놓은 N2 단계 데이터만을 대상으로 테스트되었다.
- **한계점:** 기존 연구들은 각 단계의 독립적인 성능만을 다루었으며, 수면 단계 분류 오류가 방추파 검출에 미치는 영향 등 단계 간의 상호의존성을 고려한 엔드 투 엔드(End-to-End) 평가는 부족했다.

## 🛠️ Methodology

본 연구의 전체 파이프라인은 **[RobustSleepNet을 통한 수면 단계 분류 $\rightarrow$ N2 단계 추출 $\rightarrow$ SUMOv2를 통한 방추파 검출]** 순으로 진행된다.

### 1. 자동 수면 단계 분류 (Sleep Staging)

수면 단계 분류에는 **RobustSleepNet (RSN)** 모델이 사용된다. 이 모델은 PSG 몽타주(채널 구성)의 수나 순서에 상관없이 작동하도록 설계되었다.

- **입력 및 처리:** 21개의 수면 에포크(Epoch, 총 10.5분)를 하나의 시퀀스로 입력받아 각 에포크의 수면 단계 확률을 출력한다. 데이터는 4차 Butterworth 밴드패스 필터(0.3–30 Hz)를 거쳐 60 Hz로 다운샘플링되며, 중앙값(Median)과 사분위수 범위(IQR)를 이용해 정규화된다.
- **최종 예측:** 슬라이딩 윈도우 방식으로 얻은 21개의 예측 확률값들에 대해 기하 평균(Geometric Mean)을 계산하여 최종 단계를 결정한다.

### 2. 자동 수면 방추파 검출 (Sleep Spindle Detection)

방추파 검출에는 U-Net 구조 기반의 **SUMOv2** 모델이 사용된다.

- **아키텍처:** 2개의 인코더 블록과 2개의 디코더 블록으로 구성된 U-Net을 사용하여, 입력 신호의 각 시점(Data point)에 대해 방추파 존재 여부를 나타내는 세그멘테이션 마스크(Segmentation Mask)를 생성한다.
- **손실 함수:** 클래스 불균형 문제(방추파가 없는 구간이 훨씬 많음)를 해결하기 위해 일반화된 다이스 손실(Generalized Dice Loss)을 사용한다.
- **학습 전략 (Robustness):** 기존 SUMO 모델이 진폭(Amplitude) 변화에 민감한 문제를 해결하기 위해, 학습 과정에서 진폭을 무작위로 0.5배에서 2배까지 조절하는 **Random Rescaling** 데이터 증강 기법을 적용하였다.
- **추론 절차:** RSN이 N2 단계로 분류한 구간에 대해서만 SUMOv2를 적용하며, 검출된 방추파 중 0.3초 미만 혹은 2.5초 초과인 것은 제거하고, 매우 인접한 방추파들은 병합하는 후처리를 수행한다.

## 📊 Results

### 1. 모델 성능 평가 (Model-Expert Agreement)

모델의 성능을 전문가 간 합의도(Inter-rater agreement)와 비교하여 평가하였다.

- **수면 단계 분류:** RSN 모델과 전문가의 일치도는 Macro F1 스코어 기준 평균 $0.73$으로, 전문가 쌍(Pair) 간의 평균 일치도인 $0.67$보다 통계적으로 유의미하게 높았다 ($p < 0.001$).
- **방추파 검출:** SUMOv2는 전문가가 지정한 N2 구간에서 IoU-F1 스코어 $0.60$을 기록했으며, RSN이 자동으로 분류한 N2 구간에서는 $0.54$로 약간 감소했다. 이는 MODA 데이터셋의 전문가 간 평균 일치도($0.59$)와 유사한 수준이다.

### 2. 조울증(BD) 연구 재현 실험

건강한 대조군(HC, 25명)과 조울증 환자(BP, 23명)의 데이터를 분석하여 기존 전문가 기반 연구 결과를 재현했다.

- **정량적 결과:** 자동 분석 결과, 조울증 환자군은 건강한 대조군에 비해 **빠른 방추파(Fast Spindles, $>13\text{Hz}$)의 밀도(Density)**가 유의미하게 낮게 나타났다.
  - 전두엽(Frontal) 채널: BP $2.16\text{ SPM}$ vs HC $3.41\text{ SPM}$
  - 중앙(Central) 채널: BP $2.79\text{ SPM}$ vs HC $4.37\text{ SPM}$
- **재현성:** 이는 조울증 환자의 빠른 방추파 밀도가 낮다는 기존 전문가 연구의 핵심 발견을 정성적으로 완벽히 재현한 것이며, 분석 시간은 수개월에서 단 몇 분으로 단축되었다.

## 🧠 Insights & Discussion

본 연구는 딥러닝 모델을 이용한 엔드 투 엔드 수면 분석이 전문가 수준의 성능을 낼 수 있으며, 실제 임상적 특징을 추출하는 데 유효함을 입증하였다.

**강점 및 해석:**

- 모델-전문가 간 일치도가 전문가-전문가 간 일치도와 비슷하거나 높다는 점은, 모델의 결과가 전문가의 주관적 편향(Bias)을 넘어 객관적인 기준을 제공할 가능성이 있음을 시사한다.
- 특히 조울증 환자의 방추파 특성을 정확히 잡아낸 것은, 이 파이프라인이 향후 정신 질환의 생체 지표(Biomarker) 발굴에 활용될 수 있음을 보여준다.

**한계 및 비판적 논의:**

- **정량적 수치의 차이:** 자동 분석 결과가 전문가의 수치보다 전반적으로 방추파 밀도를 높게 측정하는 경향이 있었다. 이는 모델의 편향일 수도 있으나, 논문에서는 방추파를 집계(Aggregation)하는 방식의 차이에서 기인했을 가능성이 크다고 분석한다.
- **단순 순차 구조:** 본 연구는 [분류 $\rightarrow$ 검출]의 순차적 구조를 가졌다. 두 작업을 동시에 최적화하는 Joint Optimization 방식이 더 효율적일 수 있으나, 이를 위해서는 두 가지가 모두 라벨링된 대규모 데이터셋이 필요하다는 현실적인 제약이 있다.
- **아티팩트 처리:** EEG 신호의 노이즈(Artifact)를 명시적으로 제거하는 단계가 부족하다. 다만, 최신 모델들이 학습 과정에서 아티팩트가 포함된 데이터를 통해 어느 정도 내성을 갖췄을 것으로 추측한다.

## 📌 TL;DR

이 논문은 **RobustSleepNet(수면 단계 분류)**과 **SUMOv2(방추파 검출)**를 결합하여, 전문가의 개입 없이 수면 분석을 수행하는 **엔드 투 엔드 자동화 파이프라인**을 제안하고 검증하였다. 실험 결과, 이 시스템은 전문가 간의 합의 수준에 달하는 성능을 보였으며, 조울증 환자의 수면 방추파 밀도가 낮다는 기존의 임상 연구 결과를 정성적으로 성공적으로 재현하였다. 이는 대규모 수면 연구의 비용과 시간을 획기적으로 줄일 수 있는 가능성을 제시하며, 이를 위해 **SomnoBot**이라는 프라이버시 보존형 분석 플랫폼을 함께 공개하였다.
