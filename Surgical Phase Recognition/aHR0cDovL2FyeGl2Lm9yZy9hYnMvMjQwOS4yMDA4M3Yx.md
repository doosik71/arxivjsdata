# SurgPETL: Parameter-Efficient Image-to-Surgical-Video Transfer Learning for Surgical Phase Recognition

Shu Yang, Zhiyuan Cai, Luyang Luo, Ning Ma, Shuchang Xu and Hao Chen (2024)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 거대한 이미지 수준의 사전 학습 모델(Image-level pre-trained models)을 정밀한 수술 단계 인식(Surgical Phase Recognition) 작업에 효율적으로 적응시키는 것이다.

일반적으로 고차원 비디오 데이터를 처리하기 위해 "이미지 사전 학습 후 비디오 미세 조정(Fine-tuning)" 패러다임을 사용하지만, 이는 다음과 같은 세 가지 주요 병목 현상을 야기한다:

1. **계산 비용의 급증**: 최신 파운데이션 모델의 크기가 매우 커짐에 따라 모든 파라미터를 미세 조정하는 것은 계산적으로 불가능에 가깝다.
2. **데이터 부족 및 과적합**: 수술 도메인의 데이터셋은 전문의의 수동 어노테이션에 의존하므로 규모가 매우 작다. 모든 파라미터를 학습시킬 경우, 제한된 데이터로 인해 모델이 과적합(Overfitting)될 가능성이 매우 높다.
3. **기존 파이프라인의 한계**: 기존의 수술 단계 인식은 공간 특징을 먼저 추출하고 이후에 시간 특징을 추출하는 '2단계 학습 패러다임(Two-stage paradigm)'을 주로 사용한다. 이는 공간-시간 통합 모델링을 방해하며, 사전 학습된 이미지 모델의 잠재력을 충분히 활용하지 못하게 한다.

따라서 본 연구의 목표는 적은 수의 파라미터만 학습시키면서도(Parameter-Efficient), 이미지-비디오 간의 도메인 간극을 메우고 정밀한 프레임 단위의 수술 단계 인식을 수행하는 프레임워크를 개발하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 기여는 사전 학습된 이미지 모델을 수술 비디오 분석에 효율적으로 전이시키기 위한 **SurgPETL** 프레임워크와 **STA(Spatial-Temporal Adaptation)** 모듈을 제안한 것이다.

중심적인 설계 아이디어는 다음과 같다:

- **End-to-End 구조로의 전환**: 기존의 2단계 파이프라인을 탈피하여, 고정된(Frozen) 이미지 백본에 가벼운 어댑터를 추가함으로써 공간-시간 특징을 동시에 학습하는 엔드-투-엔드 구조를 채택하였다.
- **STA 모듈의 도입**: 단순한 공간 어댑터를 넘어, 프레임 간의 협업과 암시적 모션 정보를 캡처할 수 있는 **Temporal Adapter**와 두 어댑터의 기여도를 조절하는 **Spatial-Temporal Router**를 통합하여 정밀한 시간적 모델링을 구현하였다.
- **SurgPETL 벤치마크 구축**: 다양한 규모의 ViT 모델과 다양한 사전 학습 데이터셋(자연 이미지 및 의료 이미지)을 사용하여 파라미터 효율적 전이 학습의 효과를 체계적으로 검증할 수 있는 기준을 제시하였다.

## 📎 Related Works

### 1. 이미지 사전 학습 모델 및 PETL

ViT와 같은 트랜스포머 기반 모델은 강력한 일반화 능력을 보여주며, 이를 효율적으로 전이하기 위해 Adapters, LoRA, Prefix Tuning과 같은 **Parameter-Efficient Transfer Learning (PETL)** 기법들이 제안되었다. 최근에는 이를 비디오 영역으로 확장한 **PEIVTL (Parameter-Efficient Image-to-Video Transfer Learning)** 연구(예: AIM, ST-Adapter, Dual-Path)들이 등장하였다.

### 2. 기존 접근 방식의 한계

기존의 PEIVTL 방법론들은 주로 자연 비디오의 '액션 인식(Action Recognition)'에 집중되어 있다. 액션 인식은 비디오 전체의 카테고리를 분류하는 비디오 레벨 분석인 반면, 수술 단계 인식은 각 프레임에 라벨을 부여해야 하는 **정밀한 프레임 레벨 분석**이다. 따라서 기존 방법들은 거친(Coarse) 전역적 시간 정보만 캡처할 뿐, 수술 단계의 미세한 전환을 감지하는 데 필요한 정밀한 시간적 모델링 능력이 부족하다.

### 3. 수술 단계 인식 연구

기존의 수술 단계 인식 모델(TeCNO, Trans-SVNet, SKiT 등)은 대부분 2단계 학습 방식을 사용하여 공간 특징 추출 후 TCN이나 Transformer로 시간 정보를 처리하였다. 이는 학습 비용이 높고 공간-시간 정보를 통합적으로 학습하는 데 한계가 있다.

## 🛠️ Methodology

### 1. 전체 시스템 구조 및 SurgPETL

본 연구는 기본적으로 ViT(Vision Transformer) 아키텍처를 사용한다. ViT의 기본 블록은 다음과 같이 정의된다:
$$X' = X + \text{MSA}(\text{LN}(X))$$
$$X'' = X' + \text{MLP}(\text{LN}(X'))$$

SurgPETL은 이미지 백본을 고정한 채, 학습 가능한 어댑터만을 추가하여 엔드-투-엔드로 학습하는 구조이다. 수술 비디오에서 샘플링된 프레임들을 패치 단위의 토큰으로 변환하여 입력하며, 공간적 특징과 시간적 특징을 동시에 추출하여 최종 단계를 예측한다.

### 2. Spatial-Temporal Adaptation (STA) 모듈

본 논문의 핵심인 STA 모듈은 기존 AIM 모델의 시간적 모델링 부족 문제를 해결하기 위해 설계되었다. STA는 다음과 같은 구성 요소로 이루어져 있다:

- **Spatial Adapter**: 각 프레임 내의 세부 공간 특징을 정제하고 강화한다.
- **Temporal Adapter (Feature Re-embedding, FR)**: 프레임 간의 상관관계를 강화하여 암시적인 모션 정보를 학습한다.
- **Spatial-Temporal Router**: 공간 어댑터와 시간 어댑터에서 나온 특징들의 가중치를 조절하여 중복되거나 잘못된 정보를 억제하고 변별력 있는 특징을 추출한다.

STA 모듈의 연산 과정은 다음과 같이 수식화된다:
$$H = \text{T-MSA}(\text{LN}(X))$$
$$H' = H + \alpha \cdot g_s(HW_s^d)W_s^u + \beta \cdot g_t(HW_t^d)W_t^u$$
여기서 $\alpha, \beta$는 Router의 학습 가능한 파라미터이며, $W_d, W_u$는 차원 축소(down-projection) 및 복원(up-projection)을 위한 선형 층이다. $g_s$는 GELU 활성화 함수를, $g_t$는 **Feature Re-embedding(FR)** 메커니즘을 의미한다.

### 3. Feature Re-embedding (FR) 상세 설명

FR은 시간적 차원의 표현력을 높이기 위해 다음과 같은 절차를 거친다:

1. 시간적 시퀀스를 윈도우 크기 $w$를 기준으로 $k$개의 윈도우로 분할한다.
2. 윈도우 내에서 서로 다른 시간 위치에 있는 프레임들의 특징을 채널 차원에서 결합(Concatenate)한다.
3. 이를 통해 인접 프레임 간의 픽셀 차이를 시간적 정보로 해석하여, 모델이 암시적인 움직임을 포착할 수 있도록 유도한다.

### 4. 학습 절차

- **백본**: ViT-B 또는 ViT-L을 사용하며, WIT-400M 등 다양한 데이터셋으로 사전 학습된 가중치를 초기값으로 사용한다.
- **최적화**: AdamW 옵티마이저를 사용하며, 코사인 학습률 스케줄러와 웜업(Warm-up) 전략을 적용한다.
- **디코더**: I3D를 디코더로 사용하여 최종 단계 예측을 수행한다.

## 📊 Results

### 1. 실험 설정

- **데이터셋**: Cholec80(담낭 절제술), AutoLaparo(자궁 적출술), Cataract-101(백내장 수술)의 3가지 공개 데이터셋을 사용하였다.
- **측정 지표**: 비디오 레벨 정확도(Accuracy), 단계 레벨 정밀도(Precision), 재현율(Recall), Jaccard 지수 등을 사용하였다. Cholec80의 경우 10초의 오차를 허용하는 'Relaxed' 평가와 엄격한 'Unrelaxed' 평가를 모두 수행하였다.

### 2. 주요 결과

- **PEIVTL 벤치마크 결과**: AIM, ST-Adapter, Dual-Path 세 가지 방법 중 **AIM**이 가장 적은 파라미터로 가장 높은 성능을 보였다. 특히 Dual-Path는 비디오 레벨 작업에는 강하나, 정밀한 프레임 레벨 작업인 수술 단계 인식에서는 성능이 낮게 나타났다.
- **사전 학습 데이터의 영향**: 자연 이미지 데이터셋보다 **수술 도메인 전용 데이터셋(SURGERY-23M)**으로 사전 학습된 모델을 사용할 때 성능이 비약적으로 향상되었다. 이는 수술 도메인의 특수한 특징이 전이 학습에 매우 중요함을 시사한다.
- **SurgPETL-STA의 성능**: 제안된 STA 모듈을 적용한 결과, 기존의 PEIVTL 방법들은 물론, 모든 파라미터를 학습시킨 SOTA(State-of-the-Art) 모델인 **SKiT**보다 우수하거나 대등한 성능을 보였다.
  - **Cholec80 (Unrelaxed)**: SurgPETL-STA가 Jaccard 지수 등에서 기존 모델들을 상회하는 결과를 보였다.
  - **AutoLaparo**: 특히 시간적 모델링이 중요한 이 데이터셋에서 SKiT 대비 Recall과 Jaccard 지수가 각각 4.6%, 6.2% 상승하는 등 뚜렷한 성능 향상을 보였다.
  - **Cataract-101**: 데이터 규모가 작은 이 데이터셋에서도 과적합 없이 SOTA 모델(GLSFormer)과 대등한 성능을 달성하였다.

## 🧠 Insights & Discussion

### 1. 강점 및 해석

- **파라미터 효율성**: 모든 파라미터를 미세 조정하는 Full Fine-tuning 방식(예: TimeSformer)보다 훨씬 적은 파라미터만 학습시키고도 더 높은 성능을 냈다. 이는 파운데이션 모델의 지식을 보존하면서 특정 도메인에 적응시키는 PETL 방식이 수술 비디오와 같이 데이터가 부족한 환경에서 매우 효과적임을 입증한다.
- **시간적 모델링의 중요성**: 단순한 어댑터는 각 프레임을 독립적으로 처리하는 경향이 있어 시간적 맥락을 놓치기 쉽다. 본 논문의 FR(Feature Re-embedding) 기법은 프레임 간의 상관관계를 명시적으로 모델링함으로써 이 문제를 해결하였다.

### 2. 한계 및 논의사항

- **데이터셋의 다양성**: 세 가지 데이터셋에서 검증하였으나, 수술 종류가 매우 다양하므로 더 많은 종류의 수술 비디오에 대한 일반화 성능 검증이 필요하다.
- **계산 비용의 트레이드-오프**: 파라미터 수는 줄였으나, 추론 시에 추가적인 어댑터 연산이 포함되므로 실제 실시간 수술 시스템에 적용하기 위한 추론 속도 최적화에 대한 분석이 추가될 필요가 있다.

## 📌 TL;DR

본 논문은 수술 단계 인식의 고질적인 문제인 **데이터 부족으로 인한 과적합**과 **막대한 계산 비용**을 해결하기 위해, 이미지 사전 학습 모델을 효율적으로 전이시키는 **SurgPETL** 프레임워크를 제안한다. 특히 공간과 시간 특징을 동시에 정제하는 **STA(Spatial-Temporal Adaptation)** 모듈과 프레임 간 모션을 캡처하는 **Feature Re-embedding** 기법을 통해, 적은 파라미터 업데이트만으로도 기존의 Full Fine-tuning 기반 SOTA 모델들을 뛰어넘는 성능을 달성하였다. 이 연구는 향후 거대 시각 파운데이션 모델을 의료 영상 분석의 하위 작업에 효율적으로 적용하는 중요한 방법론적 근거를 제시한다.
