# Paired 3D Model Generation with Conditional Generative Adversarial Networks

Cihan Ongün and Alptekin Temizel (2019)

## 🧩 Problem to Solve

본 논문이 해결하고자 하는 핵심 문제는 Conditional GAN(CGAN)이 주어진 조건(Condition)에 따라 샘플을 생성할 수는 있지만, 서로 다른 조건에서도 '동일한 정체성을 가진' 객체를 생성하지 못한다는 점이다.

일반적으로 CGAN에서는 입력 노이즈 $z$와 조건 값 $y$를 함께 입력으로 사용한다. 하지만 $z$와 $y$ 사이의 표현이 복잡하게 얽혀(Entanglement) 있기 때문에, 입력 노이즈 $z$를 고정하더라도 조건 $y$를 변경하면 완전히 다른 형태의 객체가 생성되는 문제가 발생한다. 예를 들어, 3D 의자 모델 생성에서 회전 각도를 조건으로 설정했을 때, $0^\circ$에서는 A라는 형태의 의자가 생성되고 $180^\circ$에서는 B라는 전혀 다른 형태의 의자가 생성되는 식이다.

이러한 문제는 서로 다른 조건의 샘플들을 하나의 객체로 병합하거나 정밀하게 제어해야 하는 후속 작업에서 심각한 제약이 된다. 따라서 본 연구의 목표는 표준 CGAN 아키텍처를 수정하지 않고도, 동일한 입력 노이즈에 대해 서로 다른 조건에서도 동일한 객체가 생성되도록 가이드하는 새로운 학습 방법을 제안하는 것이다.

## ✨ Key Contributions

본 논문의 핵심 아이디어는 **'조건별 생성 샘플들의 병합(Merging) 결과물을 판별자(Discriminator)의 입력으로 사용하여 생성기의 일관성을 강제하는 것'**이다.

기존의 CGAN이 개별 샘플의 현실성(Realism)만을 학습했다면, 제안된 방법은 동일한 노이즈 $z$로부터 생성된 서로 다른 조건의 샘플들을 하나로 합쳤을 때, 그 결과물이 여전히 현실적인 객체의 형태를 유지해야 한다는 제약 조건을 추가한다. 만약 생성된 샘플들이 서로 다른 객체라면 병합 결과는 노이즈가 심하거나 의미 없는 형태가 될 것이며, 판별자는 이를 '가짜(Fake)'로 판별하여 생성기에게 부정적인 피드백을 준다. 반면, 샘플들이 동일한 객체의 서로 다른 회전 상태라면 병합 결과는 깨끗한 하나의 객체 형태가 되며, 판별자는 이를 '진짜(Real)'로 판별하여 생성기가 일관된 샘플을 만들도록 유도한다.

## 📎 Related Works

논문에서는 다음과 같은 관련 연구들을 언급하며 차별점을 제시한다.

- **GAN 및 CGAN**: 표준 GAN은 무작위 샘플을 생성하며, CGAN은 특정 클래스나 조건을 부여하여 생성할 수 있게 한다. 하지만 앞서 언급한 정체성 유지(Pair generation) 문제는 여전히 해결되지 않았다.
- **Entanglement 해결 노력**: InfoGAN과 같이 입력 노이즈와 출력 특징 사이의 의미적 연결을 학습하려는 시도가 있었으나, 본 논문이 다루는 다중 조건 하의 쌍(Pair) 생성과는 목적이 다르다.
- **Joint Distribution 학습**: CoGAN, DiscoGAN, CycleGAN 등 여러 도메인 간의 관계를 학습하여 쌍을 생성하려는 연구들이 존재한다. 그러나 이러한 방법들은 대부분 복잡한 아키텍처 수정이나 쌍으로 이루어진 학습 데이터(Paired training data)를 필요로 한다.
- **3D 생성 모델**: 3D Voxel 기반의 생성 모델이나 Point Cloud, Mesh 기반의 연구들이 진행되었으나, 본 논문은 3D Voxel 데이터에 집중하며 아키텍처 변경 없이 학습 단계의 추가만으로 문제를 해결하려 한다는 점에서 차별성을 가진다.

## 🛠️ Methodology

### 전체 시스템 구조 및 파이프라인

본 방법론은 표준 CGAN 아키텍처를 그대로 유지하며, 학습 루프 내에 추가적인 단계(Additional training step)를 삽입하는 방식으로 동작한다.

1. **표준 CGAN 학습**: 일반적인 CGAN 방식대로 실제 데이터와 생성 데이터를 판별자가 구분하고 생성기가 이를 속이도록 학습한다.
2. **일관성 가이드 학습 (Proposed Method)**:
   - 동일한 입력 노이즈 $z$를 사용하여 서로 다른 $n$개의 조건 $y_1, y_2, \dots, y_n$에 대한 샘플 $S_0, S_1, \dots, S_n$을 생성한다.
   - 생성된 샘플들을 기준 조건($S_0$)에 맞게 정렬(Align)한 뒤, 도메인 특화 병합 연산자 $M$을 통해 하나의 샘플로 합친다.
   - 이 병합된 샘플 $M(G(z|y_n))$을 판별자 $D$에 입력하여 현실성을 평가받고, 이 결과로 생성기 $G$를 업데이트한다.

### 주요 방정식 및 손실 함수

제안된 방법이 적용된 전체 목적 함수는 다음과 같이 정의된다.

$$ \min_{G} \max_{D} V(D, G) = \mathbb{E}_{x \sim p_{data}(x)}[\log D(x|y)] + \mathbb{E}_{z \sim p_z(z)}[\log(1 - D(G(z|y)))] + \mathbb{E}_{z \sim p_z(z)}[\log(1 - D(M(G(z|y_n))))] $$

여기서 세 번째 항이 본 논문에서 제안한 추가 학습 단계에 해당한다. $M(G(z|y_n))$은 서로 다른 조건에서 생성된 샘플들의 병합 결과이며, 생성기는 이 결과물이 판별자로부터 '진짜'라는 판정을 받도록(즉, $\log(1-D(\cdot))$ 값을 최소화하도록) 학습된다.

### 3D Voxel 병합 절차 (Merge Operation)

3D Voxel 데이터의 경우, 병합 연산 $M$은 다음과 같이 수행된다.

- **정렬(Alignment)**: 서로 다른 회전 각도로 생성된 모델들을 동일한 기준 각도로 회전시켜 위치를 맞춘다.
- **평균화(Averaging)**: 정렬된 3D 행렬들의 각 요소(Voxel 값)에 대해 산술 평균을 구한다.
- **직관**: 두 모델이 동일한 형태라면 평균화된 결과도 명확한 형태를 유지하지만, 서로 다른 형태라면 서로 상쇄되어 빈 공간이 많아지거나 무의미한 형태가 된다.

## 📊 Results

### 실험 설정

- **데이터셋**: ModelNet 데이터셋의 Chair, Bed, Sofa 클래스를 사용하였으며, $32 \times 32 \times 32$ 해상도의 Voxel 데이터로 구성하였다.
- **조건**: 2-조건($0^\circ, 180^\circ$) 및 4-조건($0^\circ, 90^\circ, 180^\circ, 270^\circ$) 케이스를 실험하였다.
- **평가 지표**:
  - **AAD (Average Absolute Difference)**: 생성된 샘플들과 병합된 모델 사이의 평균 절대 차이. 0에 가까울수록 일관성이 높음을 의미한다.
  - **AVAR (Average Voxel Agreement Ratio)**: 이진화된 샘플들과 병합 모델 간의 Voxel 일치 비율. 1에 가까울수록 일관성이 높음을 의미한다.

### 주요 결과

정량적 평가 결과, 제안된 방법이 Baseline인 표준 CGAN보다 월등한 성능을 보였다.

- **2-조건 실험**: Chair 클래스에서 AAD가 약 3배 감소하고 AVAR가 $0.32 \rightarrow 0.79$로 크게 상승하였다. Bed와 Sofa 역시 유의미한 개선을 보였으며, 특히 Sofa의 AVAR는 0.95에 도달하였다.
- **4-조건 실험**: 문제의 복잡도가 증가함에 따라 2-조건 때보다는 개선 폭이 적었으나, 여전히 Baseline 대비 높은 일관성을 보였다.
- **정성적 결과**: 시각화 결과, Baseline은 동일 노이즈에 대해 서로 다른 모양의 의자를 생성했지만, 제안 방법은 동일한 모양의 의자를 서로 다른 각도로 생성하는 데 성공하였다. 또한 병합된 결과물(Intersection)이 노이즈 없이 깨끗하게 생성됨을 확인하였다.

## 🧠 Insights & Discussion

### 강점

본 논문은 복잡한 네트워크 구조 변경 없이 오직 **학습 절차(Training Procedure)**의 추가만으로 CGAN의 고질적인 문제인 Entanglement 문제를 완화하고 Paired 샘플 생성을 가능하게 했다는 점에서 실용성이 매우 높다. 특히, 병합 연산 $M$이 도메인에 따라 정의될 수 있으므로 3D 모델뿐만 아니라 2D 이미지 등 다른 데이터 도메인으로의 확장 가능성이 크다.

### 한계 및 논의사항

- **조건 수 증가에 따른 성능 저하**: 4-조건 실험에서 보았듯이, 조건의 수가 늘어날수록 생성된 샘플에 노이즈가 증가하고 해상도가 낮아지는 경향이 있다. 이는 문제의 복잡도가 증가함에 따라 더 많은 학습 데이터나 더 많은 학습 횟수(Epochs)가 필요함을 시사한다.
- **판별자 업데이트의 영향**: 저자들은 병합 샘플 단계에서 판별자를 함께 업데이트할 경우 오버트레이닝이 발생하여 시스템이 불안정해짐을 발견하고, 오직 생성기만 업데이트하는 전략을 취했다. 이는 생성-판별 간의 균형이 매우 민감한 GAN의 특성을 보여준다.
- **데이터 의존성**: 결과가 객체 클래스(Chair vs Bed/Sofa)에 따라 다르게 나타난 점은, 객체 자체의 형태적 다양성(Intra-class variance)이 학습 난이도에 큰 영향을 미침을 의미한다.

## 📌 TL;DR

이 논문은 Conditional GAN이 동일한 입력 노이즈에 대해 서로 다른 조건을 주었을 때 서로 다른 객체를 생성하는 문제를 해결하기 위해, **'다양한 조건의 생성물을 병합하여 판별자에게 평가받게 하는 추가 학습 단계'**를 제안한다. 이를 통해 아키텍처 수정 없이도 정체성이 유지되는 3D 모델 쌍(Paired models)을 생성할 수 있음을 입증하였으며, 이는 향후 다양한 도메인에서 일관성 있는 데이터 생성이 필요한 연구에 중요한 기초 아이디어를 제공한다.
